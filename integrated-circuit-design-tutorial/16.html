第16章 运算放大器设计

大家好，我是你们的讲师。上一章我们学习了模拟IC设计的基础知识——电流镜、单级放大器、频率响应和噪声。如果你认真消化了那些内容，那你已经拥有了设计模拟电路最基本的能力。现在，我们要把这些"砖块"拼在一起，搭建整个模拟IC设计中最核心、最通用的模块——运算放大器（Operational Amplifier，简称op-amp或运放）。

运放是模拟设计里最常见的电路模块。你用运放可以做有源滤波器、ADC采样保持电路、DAC输出缓冲器、稳压器、比较器、仪表放大器……基本上任何模拟信号处理功能都离不开它。在一个中等规模的混合信号芯片中，你可能找到几十个运放——每个都针对特定的功耗、噪声、带宽和精度要求做了定制。学会设计运放，你就掌握了模拟IC设计的"核心技能"。

16.1 运放性能指标

在学习运放电路之前，我们需要先建立一套"语言"来描述运放的性能。以下是最关键的指标：

DC增益（A0）——运放的开环直流增益，通常用dB表示。一个单级运放的增益约在30-40dB，两级运放可以达到60-80dB，三级或增益增强型（gain-boosted）运放可以达到100dB以上。A0决定了闭环电路的精度——在反馈系统中，环路增益越高，闭环误差越小。

单位增益带宽（GBW）——开环增益下降到0dB时的频率。GBW = gm₁ / (2π·Cc)（其中Cc是补偿电容）。这是运放"速度"的度量。如果你需要信号在10MHz频率下还能获得60dB的增益，那就需要10MHz×1000 = 10GHz的GBW——这不是一件容易的事。

相位裕度（Phase Margin, PM）——在GBW频率处，运放输出信号相对于输入信号的相移与-180°之间的差值。通常要求PM ≥ 60°以确保闭环稳定性。PM不足会导致闭环响应出现峰值（peaking），PM为负值则意味着振荡。

压摆率（Slew Rate, SR）——运放输出端电压的最大变化速率，单位V/µs。压摆率受限于尾电流源的电流和内部节点的电容。当输入信号太大（超过运放的小信号线性范围），运放进入大信号行为，压摆率成为限制因素。SR = Iss / Cc（对Miller补偿的两级运放而言）。

共模抑制比（CMRR）——差模增益与共模增益之比（以dB表示）。CMRR衡量运放抑制输入端共模信号（如电源噪声、地弹）的能力。在设计差分信号链时，高CMRR至关重要。

电源抑制比（PSRR）——运放抑制电源电压变化的能力。一个设计良好的运放，在低频时PSRR应大于80dB——这意味着电源上1V的纹波只会产生不到100µV的输出扰动。

其他重要指标还包括：输入失调电压（Input Offset）——理想运放在两个输入端电压相等时输出应为0，但实际上由于器件失配会存在几十微伏到几毫伏的失调；输入共模范围（ICMR）——运放所有输入晶体管都工作在饱和区时允许的输入电压范围；输出摆幅（Output Swing）——运放输出端能达到的电压范围；以及上面一章介绍的输入参考噪声。

关于"指标选择"的建议：很多初学者问："这个运放的设计目标是什么？"答案是——取决于应用。一个用在音频ADC的采样保持电路中的运放，你需要关注的是噪声和失真；一个用在开关电源误差放大器中的运放，你需要关注的是GBW和SR；一个用在精密仪表放大器中的运放，你需要关注的是失调和CMRR。没有"万能运放"——芯片设计本来就是一系列权衡决策组成的链条。

16.2 两级运放设计

运放的拓扑种类繁多，但最经典、最常用的结构当属Miller补偿两级运放。它的结构简洁而优雅：第一级是差分对提供高增益和CMRR，第二级是共源放大器提供额外的增益和输出摆幅，Miller电容Cc连接两级之间提供频率补偿。

在此之前，我们先讨论一个常见的"中间"结构——折叠式共源共栅（Folded Cascode）运放。为什么需要折叠？因为经典telescopic cascode虽然增益高、速度快，但输出摆幅受限——输入共模范围和输出摆幅之间有严重的冲突。Folded cascode通过"折叠"电流路径来解决这个问题：输入差分对的尾电流源折叠并叠加在cascode负载结构上，使输出摆幅不再受制于输入共模范围。代价是——增加了一条额外的电流支路，功耗更高、噪声略大。

回到两级运放：Miller补偿两级运放的系统性设计方法如下：

第一步，根据GBW确定第一级差分对的跨导gm₁：GBW = gm₁ / (2π·Cc)。选择补偿电容Cc（通常取负载电容CL的0.2-0.5倍），反推gm₁，进而确定差分对管的宽长比W/L和尾电流Iss。

第二步，根据SR确定尾电流：SR = Iss / Cc。如果SR指标不满足，需要调整Iss或Cc。

第三步，设计第二级放大器。第二级的gm₂决定了输出极点位置（p₂ = gm₂ / (2π·CL)）。为了确保相位裕度，p₂需要足够远——通常要求p₂ ≥ 2.2·GBW。

第四步，确定Miller补偿电阻Rz（也称为"调零电阻"）。Miller电容在前馈路径中会引入一个RHP零点（因为电流可以从输入端通过Cc直接流向输出端，绕过主增益级），这个RHP零点的存在会严重恶化相位裕度。在Cc串联一个电阻Rz，可以将RHP零点移到LHP或将其推到无穷远频率。Rz的最优值约为1/gm₂。

第五步，检查功耗、噪声、摆幅等约束，必要时执行迭代。

这套基于gm/ID或平方律（square-law）的设计方法，是模拟IC工程师的"标准操作流程"。业界倾向于使用gm/ID方法，因为它更准确地考虑了短沟道效应和速度饱和，而传统的平方律公式在深亚微米工艺下误差较大。

16.3 共模反馈（CMFB）

在差分输出运放（fully differential op-amp）中，有一个数字设计中没有的概念——共模反馈（Common-Mode Feedback, CMFB）。为什么需要它？因为差分输出运放的输出共模电平是无法通过差分反馈来稳定的——差分反馈只控制差分输出，而共模输出是悬空的。如果没有CMFB，输出共模电平会随着工艺、温度和电源电压的变化而漂移，最终使某些晶体管进入线性区（三极管区），放大器功能崩溃。

CMFB的思路是：检测输出端的共模电平，与参考电压比较，产生一个反馈信号来调整尾电流或负载电流，从而稳定共模电平。常见的CMFB实现方式包括：

开关电容CMFB（Switched-Capacitor CMFB）——利用电容采样输出共模电压，与参考电压比较后产生校正信号。这种实现方式不消耗静态功耗（只在开关切换时消耗动态功耗），在高性能ADC中非常常见。缺点是开关时钟可能通过寄生电容耦合到输出端，产生毛刺。

连续时间CMFB（Continuous-Time CMFB）——使用一个额外的差分对或电阻分压网络检测共模电压，通过一个额外的误差放大器输出控制信号。这种实现方式连续工作，没有开关噪声，但增加了静态功耗和噪声。

设计CMFB时最容易忽略的问题是共模环路的稳定性。CMFB环路有自己的频率响应和相位裕度——如果CMFB环路不稳定，输出端会出现共模振荡，而这种振荡有时不会立即表现为"输出抖动"或"起振"——它可能只是表现为噪声增加、THD恶化。这种"软"失效在测试中极难定位。

真实的"踩坑"故事：2021年参与一个Pipelined ADC项目，芯片回来后测试发现SNR比仿真值差了4dB——采样后信号有间歇性畸变。团队花了六周来做故障定位：我们测试了每个子ADC的输出，发现第二级MDAC的运放输出在特定条件下（输入信号幅度约-6dBFS时）会有微弱的共模振荡——振荡幅度只有10mV，频率约200MHz，但恰好被ADC采样时"折叠"到基带，产生了谐波损耗。根源是CMFB环路的相位裕度只有42°——仿真中没发现，因为CMFB环路的模型缺少了版图寄生电容，而版图寄生导致CMFB环路多了一个次级极点。修复方案是将CMFB的补偿电容增大30%。一个小小的稳定性余量不足问题，导致整整三个月的工程回溯和一次"改版"（re-spin）。

16.4 频率补偿技术

多级运放的频率补偿是模拟IC设计中最考验功力的部分。每一级放大器贡献一个极点——三级运放就有三个极点。要让整个环路稳定，我们必须"安排"这些极点的位置。

Miller补偿——这是最经典的补偿方法。用一个电容Cc连接在第二级（高频增益级）的输入和输出之间。由于Miller效应，Cc在输入端表现为一个等效电容Cc·(1+A₂)，将主极点推向低频（"分裂极点"效应）。同时输出端的有效负载电容不受影响，所以次级极点可以保持在较高频率。结果就是主极点被压低、次级极点被推高——相位裕度因此改善。

调零电阻（Nulling Resistor Rz）——上面提到过，在Cc的路径中串联一个电阻可以消除RHP零点。Rz的最佳值取决于第二级的跨导gm₂。在实际实现中，Rz常由工作在线性区的MOS管构成，使阻值可调。

Ahuja补偿——Miller电容的另一种接法：把Miller电容接地，而不是接在输入和输出之间。这种补偿方式可以消除前馈路径，从而消除RHP零点。Ahuja补偿在低功耗设计中很有价值，因为不需要额外的调零电阻。

前馈补偿（Feedforward Compensation）——通过添加一条并行的高频通路绕过增益级，在高频时直接传递信号。这种方式可以在不降低低频增益的前提下提高带宽。在三级或更多级运放中，前馈补偿常与Miller补偿配合使用。

NMC（嵌套米勒补偿，Nested Miller Compensation）——用于三级或更多级运放。每一级之间都有一层Miller补偿，形成一个嵌套结构。NMC的设计挑战在于，Miller电容之间的相互作用使得补偿分析变得非常复杂，而且外层的Miller电容需要足够大才能"压住"内层的极点，导致芯片面积显著增加。

16.5 先进运放结构

在许多实际应用中，传统的两级运放无法满足所有设计约束——这时候就需要更先进的结构。

Rail-to-Rail输入——当电源电压降低到1.8V甚至1.2V以下时，传统的PMOS差分对或NMOS差分对的输入共模范围都无法覆盖整个电源轨。解决方案是使用互补输入对——一个PMOS差分对（处理低共模输入）和一个NMOS差分对（处理高共模输入），两个输入对并联。问题是，当输入共模电压在中间区域时两个输入对都工作，导致总跨导gm翻倍——这意味着GBW和频率响应会随输入共模电平变化。解决方案是设计"gm恒定"电路：通过电流开关、1:1电流复制等方法，确保任意输入共模电平下总gm保持恒定。

Rail-to-Rail输出——传统的输出级在负载较重时无法真正摆动到电源轨。Class-AB输出级解决了这个问题：它结合了Class-A的低失真和Class-B的高效率，在静态时消耗较小的静态电流，在大信号时可以提供较大的峰值电流来驱动负载。

增益增强（Gain Boosting）——在cascode结构的基础上，用一个额外的辅助放大器来进一步"提升"cascode管的输出电阻。辅助放大器通过负反馈稳定cascode管的漏极电压，使输出电阻从gm·ro²提升到(gm·ro)²·ro，也就是增益得到了约一个运放增益量级的提升。这种结构可以达到100dB以上的增益而不牺牲太多输出摆幅。

SPICE .MEASURE 命令示例 —— 两级运放AC特性测量

* 测量DC增益（低频开环增益）
.MEAS AC A0 MAX VDB(VOUT)

* 测量单位增益带宽（增益=0dB的频率）
.MEAS AC GBW WHEN VDB(VOUT)=0

* 测量相位裕度（在GBW处的相位 + 180°）
.MEAS AC PM FIND VP(VOUT) WHEN VDB(VOUT)=0
.MEAS AC PHASE_MARGIN PARAM='180+PM'

* 测量-3dB带宽
.MEAS AC BW_3DB WHEN VDB(VOUT)=A0-3

* 测量60°相位裕度时的最大闭环增益
.MEAS AC MAX_CLOSED_GAIN PARAM='A0' WHEN VP(VOUT)=-120

* 测量压摆率（瞬态）
.TRAN 0.1n 100n
.MEAS TRAN SR_DERIV DERIV V(VOUT)
.MEAS TRAN SLEW_RATE MAX SR_DERIV FROM=10n TO=50n

* 测量输入失调
.MEAS DC VOS WHEN V(VOUT)=0

16.6 本章小结

这一章我们深入了运算放大器设计的世界。我们先建立了运放性能指标的完整"语言"——从A0、GBW、相位裕度到压摆率、CMRR、PSRR。然后我们学习了经典的两级Miller补偿运放的系统性设计方法——从GM/ID方法到逐步迭代。我们理解了为什么fully differential运放需要CMFB，以及CMFB环路稳定性为何容易被忽视。我们分析了多种频率补偿技术——Miller补偿、调零电阻、Ahuja补偿、前馈补偿和NMC。最后，我们看了几种先进运放结构——Rail-to-Rail输入/输出、Class-AB输出级和增益增强技术。

这些内容量很大，但请相信——运放设计是模拟IC设计中"投入产出比"最高的技能。你能设计运放，就意味着你能设计大部分模拟模块。下一章我们将进入另一个重要的模拟技术领域——数据转换器（Data Converters）。你会看到运放如何被用在ADC和DAC的核心电路中，以及数据转换器设计独有的挑战——如量化噪声、INL/DNL、采样开关非线性等。准备好迎接一个新的挑战吧。
