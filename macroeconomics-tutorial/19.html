第19章 开放经济宏观经济学

2015年8月11日，中国人民银行宣布调整人民币中间价报价机制，人民币一次性贬值近2%。消息一出，全球金融市场剧烈震荡——欧美股市齐跌，新兴市场货币竞相贬值，大宗商品价格暴跌。一个国家的汇率调整，为什么能引发全球范围的连锁反应？答案很简单：今天的世界上，没有任何一个经济体是孤岛。

大家好，欢迎来到开放经济宏观经济学这一章。在前十八章中，为了简化分析，我们一直默认中国是一个"封闭经济"——即不与外界发生贸易和资本往来。但现实显然不是这样：你用的iPhone可能是在郑州富士康组装、但设计来自美国加州；你买的原油价格受中东局势和OPEC决策的影响；你股票账户里的外资流向随时可能因为美联储加息而逆转。从这一章开始，我们把"世界"请进模型，看看国际贸易和资本流动如何改变宏观经济运行的逻辑。

1. 为什么要关心"外部"：从封闭经济到开放经济

让我们从一个基本问题开始：为什么国家之间要相互做生意？答案可以追溯到亚当·斯密和大卫·李嘉图——比较优势理论。简单说就是：每个国家都有自己相对擅长生产的东西，分工合作能让总产出变得更大。巴西种咖啡、沙特采石油、中国制造电子产品、德国造精密仪器——没有哪个国家能包揽一切，也没有哪个国家需要包揽一切。

但"开放"的意义远不止于买卖商品。在开放经济中，一个国家的宏观经济运行通过三个渠道与外部世界紧密啮合：

第一个渠道：贸易流动。你买日本汽车、用韩国手机、吃美国牛肉——这是商品和服务的跨境流动。在全球化的巅峰时期（2010年代），全球贸易总额占GDP的比例超过60%。虽然近年来贸易保护主义有所抬头，但贸易仍然是连接各国经济的最主要纽带。

第二个渠道：资本流动。中国购买美国国债、日本投资者在上海买公寓、沙特主权基金投资硅谷创业公司——这是资本的跨境流动。资本流动的规模甚至比贸易更大：全球外汇市场每天的交易量超过6万亿美元，是全球商品和服务贸易总额的几十倍。

第三个渠道：信息与预期。美联储主席一讲话，A股就会跟着波动；中东一打仗，全球油价就会飙升；英国一脱欧，日元就会升值。在信息时代，预期的跨境传导几乎是瞬时的。这三条渠道共同决定了：任何国家的宏观经济政策，都必须在"外部约束"下进行。

下面这张表对比了封闭经济模型和开放经济模型的差别：

维度 封闭经济模型 开放经济模型
产品市场均衡 Y = C + I + G Y = C + I + G + NX
储蓄与投资关系 S = I（国内储蓄=国内投资） S = I + NX（储蓄可用于国外投资）
货币政策 利率是国内决定的 利率受国际资本流动影响
财政政策效果 完全的挤出效应 溢出效应（影响贸易伙伴）
约束条件 无外部约束 国际收支平衡是硬约束

表中的NX代表净出口（Net Exports），即出口减去进口。当把"净出口"这个新变量加入等式，我们发现整个经济的运行逻辑都变了——一个国家的储蓄不再必须等于本国的投资，因为储蓄可以"出国"去投资别国的资产。这正是美国长期"双赤字"（贸易赤字+财政赤字）和中国的"双顺差"可以同时存在的根本原因。我们接下来就深入拆解。

2. 国际收支平衡表（BOP）

既然开放经济涉及大量的跨境交易，那总得有个"账本"来记录这些交易。这个账本就叫国际收支平衡表（Balance of Payments，简称BOP）。它是宏观经济学的"国民收入核算"在开放经济中的翻版——记录一个国家与全世界之间所有经济往来的统计报表。

BOP的核心理念可以用一句话说清楚：你给我钱，我给你东西。你给我的钱，和我给你的东西，必须相等。这里的"钱"和"东西"被分成了三个账户来记录：

经常账户（Current Account）：记录商品和服务的进出口（贸易差额）、投资收益的跨境支付（比如中国持有的美债利息）、以及跨国汇款（比如华侨寄回家的钱）。这是BOP中最受媒体关注的账户，因为"贸易顺差"和"贸易逆差"的概念就来自这里。

资本账户（Capital Account）：记录非生产性、非金融资产的转移，以及资本转移（比如债务减免、移民携带的资产）。这个账户的规模通常很小，在一般分析中经常被合并到金融账户中讨论。

金融账户（Financial Account）：记录金融资产的跨境交易——包括直接投资（FDI，比如特斯拉在上海建厂）、证券投资（比如外资买入A股）、其他投资（银行贷款、存款）和储备资产（央行持有的外汇储备）。

这三个账户加在一起，总差额必须为零。如果有差额，会计上会用"错误与遗漏"来平衡。用公式说就是：

经常账户 + 资本账户 + 金融账户 + 错误与遗漏 = 0

换个更直观的角度：
经常账户顺差 = 金融账户逆差
经常账户逆差 = 金融账户顺差

推导逻辑：
- 经常账户顺差 = 挣了外国人的钱但没有花掉→这些钱变成资本流出去
- 经常账户逆差 = 花了外国人的钱但没有挣够→差额需要资本流进来弥补

这个等式含义很深。我们拿中国的情况来理解。

中国的"双顺差"时代（2000-2014年）

按照会计逻辑，经常账户顺差意味着金融账户应该是逆差（资本净流出）。但中国在很长一段时间呈现的是"双顺差"——经常账户和金融账户都是正数。这是怎么回事？

答案藏在"错误与遗漏"和"央行干预"中。中国央行为维持汇率稳定，大量买入流入的外汇，将其变为外汇储备。这笔交易记在"储备资产"项下，导致金融账户的"储备资产"子项大幅增加（储备增加在BOP中记负值），从而在统计上让金融账户看起来是顺差。实质上，中国等于把经常账户顺差挣来的美元"存"进了央行，变成了外汇储备投资美国国债。

双顺差是一个特殊时代的产物——它反映了中国在特定发展阶段"出口导向+资本管制+强势央行干预"的独特组合。随着中国经济转型和人民币国际化，双顺差在2015年以后逐渐消失。这个变化本身就是一个值得深思的宏观经济学案例。

以下是中国某年的BOP简化表（示意数据）：

项目 金额（亿美元） 方向
货物出口 +24,900 资金流入
货物进口 -20,700 资金流出
服务净额 -2,400 资金流出
初次收入净额 -1,300 资金流出
经常账户合计 +500 顺差
直接投资净额 +500 资金流入
证券投资净额 +600 资金流入
其他投资 -800 资金流出
储备资产变动 -1,000 储备增加
金融账户合计 -700 逆差

注意金融账户合计为逆差（-700亿），加上经常账户顺差（+500亿），加上资本账户和误差项，合起来为零。这正好印证了"会计恒等式"——有借必有贷，借贷必相等。

3. 汇率制度的选择

讲完了BOP这个"账本"，我们接下来讨论开放经济中最重要的价格——汇率。准确地说，是一个国家选择什么样的汇率制度。这是每个国家央行的"顶层设计"问题之一，其重要性不亚于货币政策框架本身。

汇率制度大致分为三类：

第一类：固定汇率制（Fixed Exchange Rate）。央行承诺把本国货币的汇率盯住某个外币（通常是美元或一篮子货币）。香港的联系汇率制度就是最典型的例子——港元锚定美元，稳定在7.75-7.85的窄幅区间内。固定汇率的优点是可以消除汇率波动的风险，促进贸易和投资。缺点是需要央行有足够的外汇储备来维持盯住，一旦市场对汇率产生怀疑，就可能引发"投机性攻击"——1997年亚洲金融危机中，泰国央行动用了几乎全部外汇储备也没能守住泰铢，最终被迫贬值，损失惨重。

第二类：浮动汇率制（Floating Exchange Rate）。汇率完全由市场供求决定，央行不干预。主要发达国家（美国、欧元区、日本、英国）都采用这种制度。浮动汇率的优点在于它可以自动调节国际收支——贸易逆差时本币贬值，出口竞争力增强，逆差自动缩小。缺点是汇率波动会给进出口企业带来不确定性，也容易引发汇率过度波动（"超调"现象）。

第三类：中间汇率制（Intermediate Regimes）。介于固定和浮动之间，包括"管理浮动"（有管理的浮动汇率，央行偶尔干预）、"爬行盯住"（定期微调盯住的汇率）、"汇率目标区"（划定一个波动区间）等。中国在2005年汇改之后采取的就是"以市场供求为基础、参考一篮子货币进行调节、有管理的浮动汇率制度"——可以说是中间汇率制的一个复杂版本。

理解汇率制度的关键框架：不可能三角（三元悖论）

国际金融学中最著名的"铁律"之一，由经济学家蒙代尔和弗莱明提出。它说：在开放经济中，一个国家不可能同时实现以下三个目标，最多只能选两个：

① 独立的货币政策——可以根据国内需要自主调整利率和货币供应量
② 自由资本流动——资本可以自由进出，不受管制
③ 固定的汇率——汇率保持稳定，不自由浮动

举例来说：
· 美国选择了①+②，放弃③（浮动汇率）：美联储可以自主加息，资本自由进出，但汇率自由浮动。
· 香港选择了②+③，放弃①（联系汇率）：资本自由进出，汇率固定，但香港紧跟美联储利率，无独立货币政策。
· 中国长期选择了①+③，放弃②：央行自主决定利率，汇率相对稳定，但实行资本管制。

这个"三角"是理解各国汇率制度选择的最佳分析工具。注意，不存在"完美的选择"——每个"角"都有代价。

4. 蒙代尔-弗莱明模型

现在我们把第9-11章学过的IS-LM模型搬到开放经济的背景下，就得到了蒙代尔-弗莱明模型（Mundell-Fleming Model）。这个模型是IS-LM的开放经济版本，它的核心贡献在于揭示了一个深刻结论：同样的政策，在固定汇率和浮动汇率下的效果截然不同。

蒙代尔-弗莱明模型的基本设定是这样的：

  • IS曲线变成了开放经济版本：Y = C + I + G + NX(e)，这里的NX(e)表示净出口依赖于汇率e。本币贬值（e上升）→净出口增加→总需求增加。
  • LM曲线不变：M/P = L(Y, r)——货币供求决定，r是国内利率。
  • BP曲线（国际收支平衡线）：BOP = 经常账户 + 资本账户 = 0。当资本完全自由流动时，国内利率r必须等于世界利率r*，否则资本会大规模流入或流出。

核心结论分两种情形：

情形一：浮动汇率制下。假设政府实施扩张性财政政策（增加G或减税）。IS曲线右移→利率上升→资本流入→本币升值→净出口下降→IS曲线又向左回移一部分。最终结果是：扩张性财政政策的效果被部分或完全抵消。在资本完全流动的极端情况下，财政政策完全无效——产出不变，只有汇率变了。

情形二：固定汇率制下。假设同样是扩张性财政政策。IS曲线右移→利率上升→资本流入→本币有升值压力→央行为了维持固定汇率，必须买入外币、卖出本币→增加基础货币供应→LM曲线右移→利率回落。最终结果是：财政政策的扩张效果得到货币政策的"被动"配合，效果被放大。

反过来看货币政策：

  • 浮动汇率下：扩张性货币政策（央行降息）→LM右移→利率下降→资本流出→本币贬值→净出口增加→IS右移。货币政策通过利率和汇率双重渠道发挥作用。货币政策非常有效。
  • 固定汇率下：扩张性货币政策→利率下降→资本流出→本币有贬值压力→央行为保固定汇率，卖出储备买入本币→基础货币收缩→LM左移回原处。最终产出不变。货币政策完全无效。

下面这张表总结了不同汇率制度下的政策效果：

政策类型 浮动汇率 固定汇率
扩张性财政政策 部分或完全无效（挤出净出口） 非常有效（货币被动扩张配合）
扩张性货币政策 非常有效（利率+汇率双渠道） 完全无效（货币收缩抵消）
资本管制 政策独立性增强 政策空间增大
典型代表 美国、日本、欧元区 香港、沙特、保加利亚

这个模型的政策涵义非常清晰：如果你想用财政政策来刺激经济，最好采用固定汇率（或至少让汇率不自由浮动）；如果你想用货币政策来调节经济，最好让汇率自由浮动。这正是为什么中国在2008年金融危机时期坚持人民币汇率基本稳定（固定汇率倾向），同时大规模推出"四万亿"财政刺激计划——在当时的汇率制度下，财政政策是最有效的工具。

5. 全球失衡

2005年，时任美联储主席伯南克（Ben Bernanke）提出了一个引起广泛讨论的概念——"全球储蓄过剩"（Global Savings Glut）。这个假说尝试解释一个令人困惑的现象：为什么21世纪初全球利率持续走低，同时美国经常账户赤字不断扩大，而新兴市场（尤其是中国）积累了巨额的外汇储备？

故事是这样的：

1997-1998年亚洲金融危机之后，东亚国家（中国、韩国、东南亚各国）对"资本突然外逃"深有恐惧。它们得出的教训是：必须建立足够的外汇储备作为"防波堤"。于是，这些国家开始系统性地积累外汇储备——出口挣来的美元不再花掉，而是"存起来"。

同时，石油出口国（沙特、俄罗斯、挪威）在2000年代油价高涨时也积累了巨额"石油美元"。这些新兴市场的"过剩储蓄"变成了对美国国债等安全资产的需求，从而压低了全球利率。

低利率又刺激了美国的消费和房地产投资——美国消费者可以以很低的利率贷款买房、买汽车。美国的储蓄率持续下降，经常账户赤字不断扩大。于是，全球的贫富格局变成了这样：

"双循环"的全球失衡

亚洲（特别是中国）→ 出口商品到美国 → 赚取美元 → 用美元买美国国债 → 美国国债利率走低

美国 → 从亚洲进口便宜商品 → 消费者获益 → 低利率鼓励借贷消费 → 贸易赤字扩大 → 继续发债让亚洲买

表面上这是一个"完美"的循环：亚洲生产者、美国消费者皆大欢喜。但这种模式隐含了巨大的风险：

· 中国积累了巨额美元资产，相当于把"超国民储蓄"投向了收益率并不高的美国国债
· 美国形成了过度依赖消费和外债的增长模式
· 全球利率被人为压低，鼓励了过度冒险和金融泡沫

2008年全球金融危机的爆发，某种程度上就是这种全球失衡的一次强制性调整。危机后，中国的储蓄率有所下降，美国的储蓄率有所上升，全球失衡程度有所缓解，但远未消失。时至今日，全球贸易摩擦、中美竞争的深层根源，实际上都与这种结构性失衡密切相关。

从储蓄-投资的角度来看全球失衡：

基本恒等式：S - I = NX = CA

其中：
  S = 国民储蓄
  I = 国内投资
  NX = 净出口
  CA = 经常账户余额

含义：
  S > I → NX > 0 → 经常账户顺差（储蓄过剩，出口＞进口）
  S < I → NX < 0 → 经常账户逆差（投资过度，进口＞出口）

全球加总：∑S = ∑I（全球整体储蓄必须等于全球整体投资）
           ∑CA = 0（一个国家的顺差必是另一个国家的逆差）

这个恒等式告诉我们一个反直觉的结论：一个国家的贸易顺差不是"贸易竞争力"的简单反映，而是国内储蓄与投资失衡的反映。中国的高顺差本质上是中国的高储蓄率（居民、企业和政府都在存钱）超过了国内投资需求的结果。而美国的逆差则是低储蓄率（甚至负储蓄）的结果。

要纠正全球失衡，关键不是通过关税战去压缩某个国家的顺差——因为关税即使减少了进口，也可能会压制出口和投资，反而得不偿失。真正的解决之道在于结构性改革：中国需要完善社保体系来降低预防性储蓄，扩大内需；美国需要增加公共投资和改善财政纪律来提高国民储蓄。这些调整都不是一朝一夕能完成的。

一个常见的误解：很多人认为"贸易顺差是好事，逆差是坏事"。这是一个极其流行但极其片面的观点。顺差意味着本国在"借钱给外国"——中国长期以较低的利率借钱给美国，实际上是在补贴美国消费者。逆差意味着外国在"借钱给本国"——美国享受了来自全世界的廉价商品和廉价融资。没有绝对的好坏。关键要看：你借出去的钱是不是得到了合理的回报？你借来的钱是不是投入了生产性用途？

好，第19章的内容就到这里。我们从"为什么开放经济如此重要"开始讲起，认识了国际收支平衡表这个"全球化账本"，讨论了汇率制度的选择以及"不可能三角"，学习了蒙代尔-弗莱明模型的核心逻辑，最后分析了全球失衡的成因和含义。开放经济将以前面十八章的封闭经济框架为基础，引入了一个全新的维度——也是当下最现实的维度。
