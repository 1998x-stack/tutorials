为什么需要多项式逼近

大家好，我是你们的讲师。让我先问你一个问题：sin 1° 等于多少？如果你没有计算器，你能算出它大概是多少吗？

手动算 sin 1° 是一件可怕的事情——因为正弦函数不是我们熟悉的加减乘除运算能直接表达的。但泰勒公式告诉我们一个惊人的事实：世界上几乎所有光滑函数，都可以用多项式来逼近。而多项式——无非就是 x, x², x³ 这些"简单东西"的加减乘除。

我在大二上数值分析课时深刻体会到了这一点。老师在黑板上写出了 sin x 的泰勒展开，然后说："你们用的计算器，里面就是这么算 sin 值的。"那一刻我意识到，泰勒公式不仅仅是一个数学定理——它是现实世界中计算机做近似计算的理论基础。

从微分（线性近似）到泰勒公式（高阶多项式近似），本质上是一脉相承的思想：用简单的函数（多项式）来逼近复杂的函数。区别只在于，泰勒公式把这个思想发挥到了极致——展开到 n 阶，精度可以达到 (x - x₀)^(n+1) 的量级。

泰勒公式的系数推导思路

假设我们想用 n 次多项式 Pn(x) 来逼近点 x₀ 附近的函数 f(x)：

设 Pn(x) = a₀ + a₁(x-x₀) + a₂(x-x₀)² + ... + aₙ(x-x₀)ⁿ

我们希望多项式在 x₀ 处的各阶导数与原函数完全相等。因为多项式越光滑，在 x₀ 附近的行为就越由它的各阶导数决定。所以：

Pn(x₀) = f(x₀)       → a₀     = f(x₀)
        Pn'(x₀) = f'(x₀)      → a₁     = f'(x₀) / 1!
        Pn''(x₀) = f''(x₀)    → a₂     = f''(x₀) / 2!
        Pn'''(x₀) = f'''(x₀)  → a₃     = f'''(x₀) / 3!
        ...
        Pn⁽ⁿ⁾(x₀) = f⁽ⁿ⁾(x₀) → aₙ     = f⁽ⁿ⁾(x₀) / n!

于是我们得到：

f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)/2! · (x-x₀)² + ...
                + f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ + Rn(x)

其中 Pn(x) 是泰勒多项式，Rn(x) 是余项（误差）。

为什么系数分母有阶乘？
因为对 xⁿ 求 n 次导得到 n!。所以要让 Pn(x) 的 n 阶导数等于 f⁽ⁿ⁾(x₀)，系数必须设为 f⁽ⁿ⁾(x₀)/n!。这是一个"反过来"的推导：先想好导数要相等，再反推出系数的值。

皮亚诺余项与拉格朗日余项的区别

余项 Rn(x) = f(x) - Pn(x) 刻画了近似误差。有两种常见形式：

余项类型 表达式 适用场景 优点
皮亚诺余项 Rn(x) = o((x-x₀)ⁿ) 求极限 形式简单，只关心阶数
拉格朗日余项 Rn(x) = f⁽ⁿ⁺¹⁾(ξ)/(n+1)! · (x-x₀)ⁿ⁺¹ 近似计算、误差估计 给出误差的具体上界

皮亚诺余项只告诉你"误差是比 (x-x₀)ⁿ 更高阶的无穷小"——当 x → x₀ 时，误差消失得比 (x-x₀)ⁿ 更快。这用在求极限中已经足够。

拉格朗日余项则给出了误差的精确形式，其中 ξ 是介于 x 和 x₀ 之间的某个点（不知道具体在哪）。通过估计 f⁽ⁿ⁺¹⁾(ξ) 的范围，我们可以算出误差的最大可能值。

特别地，当 n = 0 时，泰勒公式的拉格朗日余项形式变为：

f(x) = f(x₀) + f'(ξ)(x - x₀)，其中 ξ 在 x 和 x₀ 之间

        这正是拉格朗日中值定理！

所以泰勒公式是拉格朗日中值定理的高阶推广——从"一阶近似"扩展到"n 阶近似"。

五个必背麦克劳林展开

当 x₀ = 0 时，泰勒公式称为麦克劳林公式（Maclaurin formula）。下面五个展开是微积分考试的"常客"，必须背熟：

五个必背麦克劳林展开

e^x = 1 + x + x²/2! + x³/3! + ... + xⁿ/n! + o(xⁿ)

sin x = x - x³/3! + x⁵/5! - x⁷/7! + ... + (-1)ᵏ x²ᵏ⁺¹/(2k+1)! + o(x²ᵏ⁺¹)

cos x = 1 - x²/2! + x⁴/4! - x⁶/6! + ... + (-1)ᵏ x²ᵏ/(2k)! + o(x²ᵏ)

ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ... + (-1)ⁿ⁻¹ xⁿ/n + o(xⁿ)

(1+x)ᵃ = 1 + a x + a(a-1)x²/2! + ... + C(a,n)xⁿ + o(xⁿ)

记忆规律：

  • e^x：所有阶乘在分母，正号到底——最简单。
  • sin x：只有奇次项（1, 3, 5...），正负交替（先正后负）。
  • cos x：只有偶次项（0, 2, 4...），正负交替（先正后负）。
  • ln(1+x)：分母没有阶乘（就是 n），正负交替，收敛域 (-1, 1]。
  • (1+x)ᵃ：就是二项式定理的推广（牛顿广义二项式定理）。

另外，还有一个非常实用的派生展开：

1/(1-x) = 1 + x + x² + x³ + ... + xⁿ + o(xⁿ)   （|x| < 1）

        这是等比级数的求和公式，也是很多分式展开的起点。

用泰勒公式求极限（展开到同阶）

泰勒公式在求极限中的核心优势是：把复杂函数展开成多项式，极限计算变成多项式极限——任何 0/0 型未定式都可以这样处理。

例：求 lim_{x→0} (sin x - x) / x³。

sin x 的麦克劳林展开到 3 阶：
        sin x = x - x³/6 + o(x³)

        代入原式：
        (sin x - x) / x³ = (x - x³/6 + o(x³) - x) / x³
                         = (-x³/6 + o(x³)) / x³
                         = -1/6 + o(1) → -1/6   ✅

        如果用洛必达法则，你需要连续求三次导——泰勒展开一步到位。

例 2：求 lim_{x→0} (e^x - 1 - x) / x²。

e^x 展开到 2 阶：e^x = 1 + x + x²/2 + o(x²)

        代入：
        (e^x - 1 - x) / x² = (x²/2 + o(x²)) / x² = 1/2 + o(1) → 1/2   ✅

注意：这里必须展开到 x² 项。如果只展开到 x 项（e^x ≈ 1 + x），分子就变成了 0，无法判断。这引出了下面要讲的阶数选择问题。

泰勒展开的阶数选择策略

在实际使用中，确定展开到多少阶是一个关键问题。下面几条原则帮你判断：

原则 1：分母定阶法

如果分母是 xᵏ，分子至少要展开到 xᵏ 项。这是最常用的规则。

例：lim_{x→0} (tan x - sin x) / x³
        分母是 x³，分子要展到 x³ 项。
        tan x = x + x³/3 + o(x³)
        sin x = x - x³/6 + o(x³)
        tan x - sin x = (x + x³/3) - (x - x³/6) + o(x³) = x³/2 + o(x³)
        极限 = (x³/2)/x³ = 1/2   ✅

原则 2：不抵消原则

如果展开后，低次项会互相抵消，就要展到"不会完全抵消"的那一阶。

例：lim_{x→0} (e^x - 1 - x - x²/2) / x³
        如果只把 e^x 展到 x²，分子 = 0。所以必须展到 x³：
        e^x = 1 + x + x²/2 + x³/6 + o(x³)
        分子 = x³/6 + o(x³)，极限 = 1/6   ✅

原则 3：复合函数展开法

对复合函数 f(g(x)) 做泰勒展开时，先展开内层 g(x)，再代入外层 f(u) 的展开，每一步都要保持相同阶数。

一个翻车案例：展开阶数不够
当年我做助教时，一个学生用泰勒展开求 lim_{x→0} (cos x - 1 + x²/2) / x⁴，他把 cos x 展到 x² 项，得到 cos x ≈ 1 - x²/2，然后分子 = 0，结果为 0。但正确答案是 1/24！

他错误的原因是：cos x 的展开中，x²/2 和 -1 抵消了，但 x⁴ 项才是"主角"。正确的做法是把 cos x 展到 x⁴ 项：cos x = 1 - x²/2 + x⁴/24 + o(x⁴)，这样分子 = x⁴/24，极限 = 1/24。

记住：如果你的展开导致分子完全为 0（而不是 0 + 高阶无穷小），说明展开阶数不够，需要继续往下展。
展开到多少阶的一个实用技巧
在极限计算中，先把分子分母中最低阶的主要项写出来，然后每次多展一阶，看看分子是否从 0 变成了非零。如果变成了非零，那就是需要的阶数。

还有一种更直接的方法：如果分母是 xᵏ，那么分子展到 xᵏ 项就够了——但前提是分子确实有 xᵏ 项而且没有被更低阶项完全掩盖。

本章我们学习了微积分中最重要的公式之一——泰勒公式。它的核心思想是"用多项式逼近函数"，系数由各阶导数决定。我们区分了两种余项（皮亚诺用于求极限，拉格朗日用于误差估计），背诵了五个基本函数的麦克劳林展开，并掌握了用泰勒公式求极限的方法和阶数选择策略。
