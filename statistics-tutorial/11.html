第11章：常见假设检验

大家好，我是你们的讲师。上一章我们学习了假设检验的基本逻辑——零假设、备择假设、p值、显著性水平。但实际应用中，你会发现不同的问题需要使用不同的检验方法。你想比较两个班的平均分？用t检验。你想看用户性别与购买偏好是否有关？用卡方检验。你想比较三个不同算法的方差？用F检验。

选择错误的检验方法是一个非常常见的错误——就像用扳手拧螺丝一样，虽然也能拧进去，但很可能把螺丝拧坏。本章我们来系统学习四种最常见的假设检验方法，并给你一个实用的决策指南。

11.1 z检验：已知总体标准差

z检验是最基础的假设检验，但它的使用条件比较苛刻：你需要已知总体的标准差 σ。在实际工作中，我们几乎永远不知道 σ 的确切值——这更像是一个教学工具。不过在质量控制的场景中，如果流程已经稳定运行了很久，σ 确实可以近似已知。

z统计量的计算公式：

z = (x̄ - μ₀) / (σ / √n)

其中 μ₀ 是零假设中的总体均值。计算出 z 值后，我们查标准正态分布表得到 p 值。

实用提示：在大样本（n > 30）下，即使 σ 未知，用样本标准差 s 代替 σ 做 z 检验，结果和 t 检验非常接近。所以在大样本场景下，z检验和t检验其实差别不大。但在小样本下，请务必使用 t 检验——t分布对 σ 的不确定性做了更合理的补偿。

11.2 t检验：单样本、双样本、配对

t检验是实践中用得最多的假设检验。它的发明者正是我们上一章提到的威廉·戈塞特（笔名Student），当时他在吉尼斯啤酒厂工作，需要在小样本下做质量控制。

t检验有三种常见形式：

  • 单样本t检验：比较样本均值是否等于某个定值。例如：今年的平均气温是否显著高于历史平均的20°C？
  • 独立样本t检验：比较两个独立组的均值是否存在差异。例如：男生的平均身高是否与女生不同？
  • 配对t检验：比较同一组对象前后的变化。例如：同一批患者治疗前后的血压变化。

11.3 卡方检验：分类变量的独立性

卡方检验（chi-square test）用于分析分类变量之间的关系。比如：用户的性别（男/女）与购买偏好（品类A/B/C）是否独立？

卡方检验的核心是比较观测频数和期望频数。如果两者差异很大，就说明变量间可能存在关联。卡方统计量计算如下：

χ² = Σ (Oᵢ - Eᵢ)² / Eᵢ

其中 Oᵢ 是观测频数，Eᵢ 是在"独立"假设下的期望频数。

实战经验：卡方检验有一个容易被忽略的假设——期望频数应至少为5。如果你的列联表中有些单元格的期望频数小于5，卡方检验的结果就不可靠。这种情况下，可以考虑用 Fisher 精确检验（Fisher's exact test）。我见过很多初学者直接把原始数据丢进卡方检验，结果做了个寂寞。

11.4 F检验：比较多个方差

F检验（F-test）用于比较两个或多个总体的方差是否相等。它也是方差分析（ANOVA）的基础。F统计量是两个方差的比值：

F = s₁² / s₂²

如果两个方差相等，F值应该在1附近。F值太大或太小都说明方差不等。

F检验对正态性假设非常敏感——如果数据不是正态分布，F检验的结果可能完全不可靠。所以在使用F检验前，最好先对数据进行正态性检验（如 Shapiro-Wilk 检验）。

下面用 Python 演示四种检验方法：

import numpy as np
from scipy import stats

# 准备数据
np.random.seed(42)
group_a = np.random.normal(100, 15, 40)
group_b = np.random.normal(108, 15, 40)
group_c = np.random.normal(105, 15, 40)

# === 1. z检验（大样本近似）===
from statsmodels.stats.weightstats import ztest
z_stat, z_p = ztest(group_a, group_b)
print(f"z检验: z = {z_stat:.3f}, p = {z_p:.4f}")

# === 2. t检验 ===
# 独立样本t检验
t_stat, t_p = stats.ttest_ind(group_a, group_b)
print(f"独立样本t检验: t = {t_stat:.3f}, p = {t_p:.4f}")

# 配对t检验（模拟治疗前后的数据）
before = np.random.normal(130, 10, 30)
after = before - np.random.normal(5, 3, 30)  # 平均降低5
paired_t, paired_p = stats.ttest_rel(before, after)
print(f"配对t检验: t = {paired_t:.3f}, p = {paired_p:.4f}")

# === 3. 卡方检验 ===
# 构造列联表：性别 × 购买类别
observed = np.array([[45, 30, 25],   # 男性
                     [35, 40, 25]])  # 女性
chi2_stat, chi2_p, dof, expected = stats.chi2_contingency(observed)
print(f"卡方检验: χ² = {chi2_stat:.3f}, p = {chi2_p:.4f}, df = {dof}")
print(f"期望频数:\n{expected}")

# === 4. F检验（比较方差）===
f_stat = np.var(group_a, ddof=1) / np.var(group_b, ddof=1)
df1 = len(group_a) - 1
df2 = len(group_b) - 1
f_p = 2 * min(stats.f.cdf(f_stat, df1, df2),
              1 - stats.f.cdf(f_stat, df1, df2))
print(f"F检验: F = {f_stat:.3f}, p = {f_p:.4f}")

11.5 如何选择合适的检验方法

面对不同的数据类型和研究问题，如何快速选择正确的检验方法是——个非常实用的问题。下面这个表格可以作为你的快速参考：

数据类型（因变量）研究问题自变量 / 分组推荐检验
连续（正态）均值 vs 常数单组单样本t检验
连续（正态）两组均值比较二分类（独立）独立样本t检验
连续（正态）前后变化同一组对象配对t检验
连续（非正态）两组比较二分类Mann-Whitney U检验
分类变量间是否独立两个分类变量卡方检验
连续（正态）三组及以上比较多分类ANOVA（F检验）
连续（正态）方差是否相等多组方差Levene检验
踩坑故事：有位数据分析师在做A/B测试分析时，面对三组不同版本的测试结果，分别做了三组两两t检验（A vs B、A vs C、B vs C）。结果B组看起来显著优于其他组，于是决定上线B版本。但问题是——做了三次比较没有做多重比较校正！每组比较的α=0.05，三次比较的总体Type I错误率已经膨胀到了 1 - (0.95)³ ≈ 14.3%。实际上B版本的"显著优势"很可能只是随机波动。正确的做法是用ANOVA先做整体检验，或者对p值做Bonferroni校正。

还有一个重要的原则：检验方法应该先于数据收集确定。在收集数据之前就明确你要用什么方法分析，而不是先看数据再选择检验方法——后者是p-hacking（p值操纵）的常见形式。如果在看到数据后才决定检验方法，你对结果的主观操控空间就太大了。

小建议：如果你不确定该用什么检验，可以考虑用自助法（bootstrap）做一个近似检验。自助法不对数据的分布做任何假设，通过反复重抽样来估计统计量的分布。虽然它不能完全替代传统检验方法，但在很多复杂场景下是一个非常好的备用方案。

掌握了这些常用的假设检验方法，你就能应对大多数数据分析场景了。但还有一个重要的概念我们还没讨论——当你发现"没有显著差异"时，是真的没有差异，还是因为样本量太小导致检测不到？下一章我们将讨论统计功效与样本量的关系。
