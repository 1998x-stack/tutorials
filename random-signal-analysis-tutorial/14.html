开篇：从连续到离散的跨越

大家好，我是你们的讲师。前面六章我们一直在连续时间域中讨论随机信号——连续的时间索引 t、连续的积分、连续的傅里叶变换。但现实世界中，绝大多数信号处理系统是数字的：ADC 以固定采样率将模拟信号转换为数字序列，DSP 芯片或 FPGA 对这些离散样本进行处理，然后 DAC 将结果转换回模拟信号。

这就引出了本章的主题：离散时间随机信号（随机序列）及其通过离散 LTI 系统的分析。好消息是：前两章的所有结论都有直接的离散版本——均值→序列均值，自相关→自相关序列，卷积→离散卷积，PSD→功率谱密度函数。所有公式在形式上几乎一模一样，只是积分变成了求和，傅里叶变换变成了 DTFT（或 z 变换）。

但离散域也有它特有的新工具——本章的重头戏是AR、MA 和 ARMA 模型，它们是现代时间序列分析和谱估计的基石。

离散时间随机信号

采样与平稳序列

一个连续时间随机信号 X(t) 经过理想采样（采样周期 T_s）后，得到离散时间随机序列：

X[n] = X(nT_s)，n = 0, ±1, ±2, ...

离散时间 WSS 的定义与连续版本完全对应：

  • 均值恒定：E[X[n]] = μ_X，对所有 n 成立。
  • 自相关只依赖于时间差：R_X[n₁, n₂] = E[X[n₁] X[n₂]] = R_X[m]，其中 m = n₂ - n₁。

离散自相关函数 R_X[m] 具有以下性质：

  • R_X[0] = E[|X[n]|²] ≥ 0（总平均功率）
  • |R_X[m]| ≤ R_X[0]（柯西-施瓦茨不等式）
  • R_X[-m] = R_X[m]（实序列的对称性）

离散维纳-辛钦定理

就像连续时间一样，WSS 序列的自相关与功率谱密度构成一对离散时间傅里叶变换对（DTFT）：

S_X(e^{jω}) = Σ_{m=-∞}^{∞} R_X[m] e^{-jωm}

R_X[m] = (1/2π) ∫_{-π}^{π} S_X(e^{jω}) e^{jωm} dω

注意几个关键差异：

  • 频率 ω 是归一化角频率，范围 [−π, π]（对应物理频率 [−f_s/2, f_s/2]）。
  • DTFT 是周期为 2π 的周期函数——这是离散时间信号在频域中的根本特征。
  • 积分限从 -∞ 到 ∞ 变成了 -π 到 π。
提醒：在使用离散 PSD 时，务必注意归一化频率与实际物理频率的换算关系。例如，采样率 f_s = 1000 Hz 时，归一化频率 ω = π 对应物理频率 500 Hz（奈奎斯特频率）。很多初学者在仿真中设定了 f_s 后，却用错归一化频率，导致滤波器设计偏差。记住：数字频率 ω = 模拟频率 f · (2π / f_s)。

离散 LTI 系统响应

离散版本的核心公式

考虑一个离散 LTI 系统，单位脉冲响应为 h[n]，z 变换为 H(z)。输入为 WSS 序列 X[n]，输出 Y[n] = X[n] ∗ h[n] = Σ_{k=-∞}^{∞} h[k] X[n-k]。

所有连续时间的结论在离散域中都有对应：

统计量连续域公式离散域公式
输出均值μ_Y = μ_X · H(0)μ_Y = μ_X · H(e^{j0}) = μ_X · Σ h[n]
互相关R_{XY}(τ) = R_X(τ) ∗ h(τ)R_{XY}[m] = R_X[m] ∗ h[m]
输出自相关R_Y(τ) = R_X(τ) ∗ h(τ) ∗ h(-τ)R_Y[m] = R_X[m] ∗ h[m] ∗ h[-m]
输出 PSDS_Y(ω) = |H(ω)|² · S_X(ω)S_Y(e^{jω}) = |H(e^{jω})|² · S_X(e^{jω})
总功率∫ S_Y(ω) dω / (2π)(1/2π) ∫_{-π}^{π} S_Y(e^{jω}) dω

离散域的优势在于：我们可以在数字系统中精确实现任意响应 h[n]（满足因果关系），而不受模拟元件容差的限制。这就是为什么现代信号处理几乎全部在数字域中完成。

AR、MA、ARMA 模型

从白噪声到参数化模型

现在我们进入本章最核心的内容——参数化随机序列模型。这些模型的核心思想是：一个复杂的随机序列可以用白噪声通过一个线性系统来生成。如果系统的结构足够简单（只有少数几个参数），那么我们可以用这些参数来紧凑地刻画随机序列的统计特性。

考虑一个白噪声序列 ε[n]（均值为零、方差为 σ²），通过一个线性时不变系统 H(z) 后得到输出 X[n]：

X[n] = Σ_{k=0}^{∞} h[k] ε[n-k]

如果 H(z) 是有理函数（分子分母都是 z 的多项式），那么 X[n] 满足一个线性差分方程。根据方程的形式，分为三类模型：

MA 模型（滑动平均模型）

MA(q) 模型——q 阶滑动平均（Moving Average）：

X[n] = ε[n] + b₁ ε[n-1] + b₂ ε[n-2] + ... + b_q ε[n-q]

MA(q) 的关键性质：

  • 自相关截尾（Autocorrelation Cut-off）：R_X[m] = 0 对所有 |m| > q 成立。这是 MA 模型最显著的特征——自相关函数在滞后超过 q 后突然降为零。
  • PSD：S_X(e^{jω}) = σ² · |B(e^{jω})|²，其中 B(z) = 1 + b₁z⁻¹ + ... + b_q z^{-q}。
  • MA 模型总是平稳的，因为它只涉及白噪声的有限个延迟的线性组合。

AR 模型（自回归模型）

AR(p) 模型——p 阶自回归（Autoregressive）：

X[n] = a₁ X[n-1] + a₂ X[n-2] + ... + a_p X[n-p] + ε[n]

AR(p) 的关键性质：

  • 偏自相关截尾（Partial Autocorrelation Cut-off）：PACF 在滞后超过 p 后为零。
  • PSD：S_X(e^{jω}) = σ² / |A(e^{jω})|²，其中 A(z) = 1 - a₁z⁻¹ - ... - a_p z^{-p}。
  • 平稳性条件：A(z) 的所有根必须在单位圆外（即 |z| > 1）。
  • AR 模型的自相关函数是无限延续的（永远不会突然截断），以指数衰减。

AR 模型最重要的工具是Yule-Walker 方程：

R_X[0] = a₁ R_X[1] + a₂ R_X[2] + ... + a_p R_X[p] + σ²
R_X[k] = a₁ R_X[k-1] + a₂ R_X[k-2] + ... + a_p R_X[k-p],  k ≥ 1

这个方程告诉我们：AR(p) 的自相关系数满足一个线性递归关系，前 p 个自相关值加上噪声方差 σ² 就完全决定了整个自相关序列。反过来，给定自相关估计值，我们可以解线性方程组求出 AR 参数——这就是现代谱估计的基础方法之一。

ARMA(p,q) 模型

ARMA(p,q) 模型是 AR 和 MA 的结合：

X[n] = Σ_{i=1}^{p} a_i X[n-i] + ε[n] + Σ_{j=1}^{q} b_j ε[n-j]

ARMA(p,q) 的 PSD 为：

S_X(e^{jω}) = σ² · |B(e^{jω})|² / |A(e^{jω})|²

这里 A(z) 是 AR 部分的多项式，B(z) 是 MA 部分的多项式。ARMA 模型用更少的参数就能描述更复杂的谱形状——因为分子和分母多项式共同塑造频谱。

模型选择指南：MA 适合自相关截尾的情况（如量化噪声、有限记忆信道）；AR 适合谱峰尖锐的情况（如语音信号中的共振峰、雷达回波）；ARMA 适合谱峰和谱谷都存在的情况（如某些噪声干扰场景）。记住一个重要的原则：不要追求"最精确"的模型，而是追求"充分简洁"的模型——过高的阶数会导致过拟合。

ARMA 模型的应用

语音编码中的 LPC（线性预测编码）

AR 模型最经典的应用是语音编码中的线性预测编码（Linear Predictive Coding, LPC）。人的声道可以被建模为一个 AR 系统：声带产生的脉冲序列（激励）经过声道（一个全极点滤波器）输出语音信号。声道滤波器正是 AR 模型：

S[n] = Σ_{i=1}^{p} a_i S[n-i] + G · u[n]

其中 u[n] 是激励信号（对于浊音是周期脉冲，对于清音是白噪声），G 是增益因子。LPC 的核心思想是：从语音信号中估计出 AR 参数 a_i（通常 p = 10-16 对 8 kHz 采样率就够了），然后只传输这些参数和激励信息——码率可以从 64 kbps（PCM）压缩到 2-8 kbps（LPC）而保持可懂度。

这个应用完美展示了 AR 模型的威力：一个 10 阶 AR 模型（11 个参数）就可以相当精确地描述语音信号的频谱包络（共振峰结构）。

系统辨识

ARMA 模型在系统辨识中的应用同样重要。假设我们有一个未知系统 H(z)，输入是白噪声（或已知信号），输出是 Y[n]。通过辨识输出序列的 ARMA 模型参数，我们可以估计出系统的传递函数：

  • 输出 Y[n] 的自相关 → 系统 PSD：Y[n] 的 PSD = σ² · |H(e^{jω})|²，因此从 Y[n] 的自相关可以估计出系统幅频响应的平方。
  • 输入-输出互相关 → 系统冲激响应：如果输入是白噪声，R_{XY}[m] ∝ h[m]。
  • Yule-Walker 参数提取：从 Y[n] 的自相关中解出 AR 参数，再用 AR 参数反推系统极点位置。

这些方法在结构健康监测、机械故障诊断、地震勘探等领域广泛应用。

踩坑故事：AR 模型阶数选错导致的谱估计失真

这个故事来自我参与的一个雷达目标检测项目。我们需要从雷达回波中提取目标的微多普勒特征，用于区分不同类型的空中目标（固定翼飞机 vs. 直升机）。原始回波信号包含目标散射体的振动/旋转调制，这些调制在频谱上表现为目标主体频率周围的边带。团队决定使用 AR 模型来做高分辨率的功率谱估计，以获得清晰的微多普勒特征。

一名学生先用 AIC（Akaike Information Criterion）准则估计了 AR 模型的最佳阶数，AIC 给出的阶数是 p = 28。然而另一个同学认为"更高阶的模型谱分辨率更好"，坚持将阶数设为 p = 64。他们使用 Burg 算法估计了 AR 参数，然后计算了 PSD。

结果令人困惑：p = 28 的谱显示出了三个清晰的微多普勒边带，与理论预期一致；而 p = 64 的谱却"分裂"出了七八个虚假的谱峰，看起来像是存在多个目标，但实际上只是 AR 模型过拟合了噪声。更糟糕的是，这些虚假谱峰的位置和幅度在不同的数据段中完全不一致——因为 AR 模型的极点被噪声推到了单位圆附近，产生了尖锐但不真实的共振峰。

团队后来发现，AIC（Akaike Information Criterion）准则给出的 p = 28 并不是随便选的——它是基于理论和实测数据验证后得到的、在这类雷达回波分析中的标准经验值。过高的阶数引入了太多自由度，模型开始"学习"噪声的随机波动而不是信号的真实谱结构。

教训：AR 模型阶数选择是一个偏差-方差权衡（Bias-Variance Tradeoff）问题。阶数太低：谱过于平滑，丢失细节（高偏倚）；阶数太高：谱出现虚假峰，方差大（过拟合）。几个实用建议：(1) 使用 AIC 或 BIC 准则给出参考阶数——它们本身就是为模型选择设计的；(2) 从低阶开始逐步增加，观察谱的稳定性；(3) 如果不同阶数得到的谱差异过大，那说明你的数据可能不满足 AR 假设（比如包含强周期成分或非平稳性）；(4) 特别注意——AR 谱的"尖峰"看起来诱人，但如果那个峰对于阶数选择极其敏感，它很可能就是假的。记住：模型越复杂，越容易拟合噪声，而不是信号。

本章小结

这一章我们将随机信号分析从连续时间推广到了离散时间域。离散 WSS 序列与连续 WSS 信号有着平行的理论体系——离散维纳-辛钦定理、离散 LTI 系统的响应公式——但离散域带来了更强大的参数化建模工具。我们学习了 MA(q)、AR(p) 和 ARMA(p,q) 三类参数模型：MA 的自相关截尾、AR 的 Yule-Walker 方程、ARMA 的综合建模能力。在应用层面，介绍了 AR 模型在语音编码（LPC）和系统辨识中的经典用例。最后通过雷达微多普勒分析中 AR 阶数选择的教训，警示了过拟合导致虚假谱峰的风险。

下一章我们将进入一个新的主题——高斯过程（Gaussian Process）。虽然前几章我们已经多次提到高斯分布，但现在我们要系统性地研究"高斯随机过程"这个特殊而重要的类别。你会发现：高斯过程在信号处理中如此受欢迎，不仅是因为它是常见的噪声模型，更因为它的数学优美性——均值和协方差函数完全决定了它的全部统计特性。
