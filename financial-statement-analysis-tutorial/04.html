大家好，我是你们的讲师。前两章咱们学了资产负债表（底子）和利润表（面子），今天咱们来学三张表中最"诚实"的一张——现金流量表（Cash Flow Statement）。

如果让我在三张表中只能选一张来判断一家公司，我选现金流量表。为什么？因为利润可以"做出来"，但现金流很难。利润表基于权责发生制——货发了、发票开了就可以确认收入，哪怕钱还没收到。而现金流量表基于收付实现制——钱到账了才算流入，钱划走了才算流出。一张是"会计利润"，一张是"真实现金"。

说得直白一点：利润表是公司想让你看到的"故事"，现金流量表是公司遮不住的"真相"。

为什么现金流比利润更真实

咱们用一个小例子来说明权责发生制和收付实现制的区别：

假设你开了一家小公司，今年签了一个100万的合同，货发了、票开了、客户说"明年付款"。按照权责发生制（利润表），今年你就确认了100万收入和相应的利润。但如果按照收付实现制（现金流量表），今年你一分钱现金都没收到——你的"利润"是纸面上的，你的银行账户余额并没有增加100万。

这就是为什么：一家公司可以每年盈利，但依然破产。因为破产不是因为"亏损"，而是因为"没钱了"。这种现象在商业史上反复上演——安然公司（Enron）在破产前一年的利润表看起来还不错，但现金流早已枯竭。

核心认知：权责发生制给了管理层调节利润的空间（提前确认收入、推迟确认费用），而收付实现制没有这个空间。所以现金流量表是检验利润质量的试金石。当一家公司的净利润连年增长但经营性现金流持续为负，这叫"纸面富贵"——看似富贵，实则危险。

三大现金流板块

现金流量表把企业的现金活动分为三类：经营活动（Operating）、投资活动（Investing）、筹资活动（Financing）。

  • 经营活动现金流——公司主营业务产生的现金流入和流出。包括销售商品收到的现金、支付给供应商的现金、支付给员工的工资、支付的税费等。这是最重要的部分——长跑型好公司的经营现金流一定是正的，而且应该持续增长。
  • 投资活动现金流——购买或出售固定资产（资本支出）、并购其他公司、购买或赎回理财产品的现金流动。成长型公司通常投资活动现金流为负（持续投入资本），而收缩型公司可能为正（出售资产换钱）。
  • 筹资活动现金流——借款、还债、增发股票、分红、回购产生的现金流动。筹资活动现金流为正（借的钱比还的多）说明公司在扩张债务或股权融资；为负（还的钱比借的多）说明公司在偿还债务或回馈股东。

自由现金流的计算

自由现金流（Free Cash Flow, FCF）是衡量公司"真金白银赚钱能力"的核心指标。它回答了这个问题：公司在维持现有经营规模的前提下，每年能产生多少可供股东自由支配的现金？

FCF的三种常见算法：

算法一（最常用）：
FCF = 经营活动现金流净额 - 资本支出（Capital Expenditure）

算法二（从利润出发）：
FCF = 净利润 + 折旧摊销 - 营运资本增加 - 资本支出

算法三（从EBITDA出发）：
FCF = EBITDA - 所得税 - 资本支出 - 营运资本增加

其中：
资本支出 = 购买固定资产、无形资产等支付的现金（在投资活动现金流中）
营运资本增加 = 应收账款增加 + 存货增加 - 应付账款增加

如果一家公司的FCF持续为正且不断增长，说明它不依赖外部融资也能自我造血——这是最优质的商业模式特征。相反，如果一家公司净利润不错但FCF常年为负，说明它需要不断借钱或融资才能活下去——这种模式在经济下行时会非常脆弱。

现金流组合的健康诊断

三大现金流板块的正负号组合，可以快速判断一家公司所处的生命周期阶段和财务状况。8种组合各有含义：

经营活动 投资活动 筹资活动 企业类型 诊断结论
+ - - 优质成长型 主业赚钱，持续投资，还债或分红。最理想的组合——如茅台、美的
+ - + 扩张型 主业赚钱，扩大投资，但投资资金不够，需要外部融资补充。成长阶段常见
+ + - 收缩型 主业赚钱，同时在出售资产（如卖子公司），并用现金还债。可能正在战略收缩
+ + + 异常型 主业赚钱，卖资产还赚钱，同时还融资——这么多钱要去哪？需关注资金用途
- - + 初创/困境型 主业不赚钱，还要投资，全靠融资续命。初创企业正常，成熟企业则危险
- - - 全面失血型 所有板块都在烧钱。极度危险——公司很可能撑不过12个月
- + + 变卖家产型 主业不赚钱，靠卖资产和融资维持。不可持续——资产总有卖完的一天
- + - 回光返照型 主业亏损，卖资产还债。资产卖完就没了，非常危险
快速诊断法：先看经营现金流是正是负——如果是负的，不用再看其他两项了，这家公司大概率有问题（除非是高速成长的初创企业）。经营现金流是企业的"造血能力"，造血能力不足的公司都不是好标的。

踩坑故事——康美药业的现金流谎言

A股历史上最大的财务造假案之一——康美药业（现已退市），是现金流量表分析价值的经典反面教材。

在暴雷之前，康美药业的利润表看起来非常漂亮：营收从2015年的180亿增长到2018年的近200亿，净利润稳定在30-40亿的水平，ROE持续在15%以上。只看利润表的话，这是一家非常优质的医药公司。

但是，现金流量表早就露出了马脚：2017年，康美药业年报显示净利润40.9亿，但经营活动现金流净额只有18.4亿，净现比（经营现金流/净利润）仅为0.45。2018年上半年，净利润26.6亿，而经营现金流净额居然是负4.1亿！净利润和经营现金流出现了严重的背离。

也就是说，康美药业赚了26.6亿的"利润"，但实际上现金净流出了4.1亿。这是怎么做到的？虚构交易、虚增收入、虚增货币资金——这些后来都被证监会查实了。如果当时投资者认真看现金流量表，看到经营现金流和净利润持续背离，就能提前嗅到危险的气息。

关键指标：净现比（CFO/Net Profit）——即经营活动现金流净额除以净利润。正常公司应该>0.7，优秀公司>1.0。如果净现比连续多年<0.5，一定要高度警惕。这是识别"纸面利润"的最简单、最有效的指标。

为了帮助你形成分析习惯，这里是一个现金流量表分析速查清单：

现金流量表分析速查清单：
1. 经营现金流是否为正？（是否≥净利润的70%？）
2. 净现比（经营现金流/净利润）趋势如何？
3. 自由现金流（FCF）是否为正？
4. 资本支出占经营现金流比例是否合理？（<50%为佳）
5. 三大现金流组合属于哪种类型？（对照8种组合表）
6. 筹资活动现金流是否依赖持续借款？（借新还旧不可持续）
7. 经营现金流增速是否与营收增速匹配？
8. 是否存在"经营现金流为负+投资现金流为正"的组合？（卖资产填窟窿）

好了，本章咱们掌握了现金流量表的核心分析方法。记住：利润可以调节，但现金流很难伪装。当你下次看到一家公司宣布"业绩大增"时，第一件事不是去看利润增长了多少，而是去翻现金流量表——看看它的经营现金流是不是真的在增长。下一章咱们要进入一个很多投资者忽略、但极其重要的领域——报表附注。魔鬼在细节里，咱们下篇见！
