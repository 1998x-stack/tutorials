第03章 对偶理论与灵敏度分析

大家好，我是你们的讲师。上一章我们理解了线性规划的建模和单纯形法求解。但有一个问题你可能已经在心里想了：求出最优解之后呢？如果原材料价格上涨、如果客户追加订单、如果新工艺上线——最优解会怎么变化？

这正是对偶理论（Duality Theory）和灵敏度分析（Sensitivity Analysis）要回答的问题。如果说单纯形法是告诉你"最优解是什么"，那么对偶理论就是告诉你"为什么这是最优的"，灵敏度分析则是告诉你"如果情况变了怎么办"。这三者加在一起，才构成线性规划的完整武器库。

我们先从对偶理论讲起——这是整个优化理论中最优美的部分之一。

对偶问题的构造 — 原问题与对偶问题的对称关系

每一个线性规划问题（原问题，Primal），都对应着另一个线性规划问题（对偶问题，Dual）。两者之间有着深刻的数学联系。

还是用上一章家具厂的例子。原问题是：

原问题（Primal）：
  max  Z = 80x₁ + 50x₂
  s.t. 4x₁ + 2x₂ ≤ 60  (木工)
       2x₁ + 2x₂ ≤ 40  (油漆)
       x₂ - 2x₁ ≤ 0    (市场)
       x₁, x₂ ≥ 0

它的对偶问题是：

对偶问题（Dual）：
  min  W = 60y₁ + 40y₂ + 0·y₃
  s.t. 4y₁ + 2y₂ - 2y₃ ≥ 80
       2y₁ + 2y₂ + y₃ ≥ 50
       y₁, y₂, y₃ ≥ 0

看到这组转化，你可能觉得莫名其妙。别急，我们来看转换规则。

对称型与非对称型对偶的构造规则

原问题（Primal） 对偶问题（Dual）
最大化目标（max） 最小化目标（min）
约束 ≤ 类型 变量 ≥ 0
约束 = 类型 变量无符号限制
变量 ≥ 0 约束 ≥ 类型
变量无符号限制 约束 = 类型
约束的右端项 目标函数系数
目标函数系数 约束的右端项
约束矩阵 A 约束矩阵 Aᵀ（转置）

用一句话记住核心规律：原问题的约束对应偶问题的变量，原问题的变量对应偶问题的约束；max 变成 min，方向转置。

对偶的经济解释：影子价格

对偶变量 y₁, y₂, y₃ 有非常直观的经济含义。在上一章的最终单纯形表中，松弛变量的检验数分别是 -15 和 -5。这些数值的绝对值——15 和 5——就是影子价格（Shadow Price）。

影子价格的含义：约束右端项每增加一个单位，最优目标值将改进多少。

在我们的例子中：

  • y₁ = 15：木工工时每增加 1 小时，总利润增加 15 元。
  • y₂ = 5：油漆工时每增加 1 小时，总利润增加 5 元。
  • y₃ = 0：市场限制每放宽一单位，对利润没有影响——因为该约束在最优解处不是紧的（s₃ = 10 > 0）。

这告诉厂长一个清晰的决策方向：优先扩木工产能，因为它的边际贡献最高。这就是对偶理论在商业决策中的直接价值。

影子价格的商业价值：我在一家化工厂做过生产优化项目。通过影子价格分析，我们发现某一中间产品的储罐容量是"最紧的瓶颈"——它的影子价格高达每立方米 320 元。而储罐扩容的一次性投资成本约 50 万元，按日产量计算，不到半年就能收回投资。这个 ROI 分析直接说服了管理层立项扩容。没有影子价格，这种投资决策只能凭直觉。

互补松弛定理 — 最优解的充要条件

对偶理论最优雅的结论之一是互补松弛定理（Complementary Slackness）。它给出了一个解是最优解的充要条件。

定理内容：设 x 是原问题的可行解，y 是对偶问题的可行解，则它们同时是最优解当且仅当：

yᵢ · (bᵢ - Aᵢx) = 0 对所有 i 成立
(cⱼ - yᵀAⱼ) · xⱼ = 0 对所有 j 成立

通俗地说：要么约束是紧的（松弛变量为零），要么对应的对偶变量是零，不能两者都"宽松"。

回顾家具厂的最优解 x = (10, 10)：

  • 木工工时约束：4×10 + 2×10 = 60，松弛为 0 → y₁ = 15（可以非零）✓
  • 油漆工时约束：2×10 + 2×10 = 40，松弛为 0 → y₂ = 5（可以非零）✓
  • 市场限制约束：10 - 2×10 = -10，松弛 s₃ = 10 > 0 → y₃ = 0（必须为零）✓

完全满足互补松弛条件，验证了 (10, 10) 确实是最优解。

互补松弛的实用价值在于：可以用它来快速验证一个候选解的 optimality，甚至在求解之前就推导出最优解的结构。这在调度问题中非常有用——有时候约束的数量远少于变量的数量，通过对偶可以大幅缩小搜索空间。

互补松弛验证示例：

给定原问题：
  max  4x₁ + 3x₂
  s.t. x₁ + x₂ ≤ 5
       2x₁ + x₂ ≤ 8
       x₁, x₂ ≥ 0

假设有人给出了一个候选解 x* = (3, 2)：
  检验可行性：3+2=5≤5 ✓, 6+2=8≤8 ✓, 可行。

构造对偶问题：
  min  5y₁ + 8y₂
  s.t. y₁ + 2y₂ ≥ 4
       y₁ + y₂ ≥ 3
       y₁, y₂ ≥ 0

代入互补松弛：
  第1约束：3+2=5 → 松弛=0 → y₁ 可以非零
  第2约束：6+2=8 → 松弛=0 → y₂ 可以非零
  第1原变量：c₁ - yᵀA₁ = 4 - y₁ - 2y₂ = ? ...
  通过求解得 y₁=2, y₂=1，代入 x 检验：全部满足 → x* 是最优解。

灵敏度分析四场景

在真实世界中，模型的参数很少是固定不变的。原材料价格会波动，客户订单会变化，新的产品线可能上线。灵敏度分析回答的核心问题是：当输入数据变化时，最优解会发生什么变化？

场景一：价值系数变化

假设桌子的利润从 80 元涨到了 90 元。最优解会变吗？答案是：在一定的变化范围内，最优基不变——即同样的约束仍然是紧约束，但最优解的值本身会变化。这个范围称为目标系数的允许变化区间。当变化超出这个区间，最优基会发生改变，需要重新求解。

场景二：右端项变化

这就是影子价格直接应用的场景。只要右端项变化在允许范围内，影子价格就有效。超出范围后，基可能改变，影子价格也需要重新计算。

场景三：新增变量

比如工厂考虑增加第三种产品（书柜）。我们需要回答：这个新产品值得生产吗？

判断标准：计算新产品的 reduced cost（检验数）。如果 reduced cost ≤ 0（在最大化问题中），说明纳入新产品不会改善目标；如果 > 0，值得纳入。

计算公式：reduced cost = 利润 - (对偶变量 · 资源消耗)。对于书柜，假设利润 100 元，消耗木工 5 工时、油漆 3 工时，则 reduced cost = 100 - (15×5 + 5×3) = 100 - 90 = 10 > 0 → 应该生产。

场景四：新增约束

比如环保部门要求每天总工时不超过 90 小时。我们检查当前最优解是否满足新约束：4×10 + 2×10 + 2×10 + 2×10 = 100 > 90 ——不满足，所以需要重新求解。如果当前解已经满足，新增约束就是冗余的，最优解不变。

100% 规则速记：当多个参数同时变化时，计算每个参数变化量占其允许变化范围的百分比，如果总和不超过 100%，则可以保证最优基不变。这是快速判断"要不要重新求解"的实用工具。

实战踩坑：在一次供应链优化项目中，我们建立了包含 2000+ 约束和 8000+ 变量的 LP 模型。客户问："如果运费上涨 15%，我们该调整哪条路径？"我们跑完灵敏度分析，发现 80% 的路径都还在允许变化区间内，但有一条路径的 reduced cost 已经接近临界点。最终我们提前锁定了替代运输方案，在运费正式上涨后只花了 2 周就完成了切换——而竞争对手花了 3 个月。这个案例让我深刻体会到：灵敏度分析不是学术游戏，它是企业应对不确定性的"预警系统"。

对偶理论和灵敏度分析给了我们一双"透视眼"——不仅能看到最优解本身，还能理解最优解如何随环境变化。在调度问题中，这尤其重要：车间的加工时间、交货期、设备可用性随时在变，一个没有灵敏度分析支持的调度方案是僵化的。
