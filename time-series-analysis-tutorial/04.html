第4章 自相关分析

大家好，我是你们的讲师。这一章我们要聊一个非常实用的概念——自相关（Autocorrelation）。名字听起来有点吓人，但核心思想很简单：今天的自己和昨天的自己有多像。

想象一下你每天记录自己的体重。如果今天你体重超标了，明天大概率也超标——因为体重不会一天之内大变。这就是"今天的值与昨天的值相关"，也就是自相关。在时间序列里，这几乎是一个普遍存在的现象：今天的温度受昨天影响，今天的股价受昨天影响（尽管更随机），今天的用电量受昨天影响。理解自相关就是理解时间序列的"记忆"。而自相关函数（ACF）和偏自相关函数（PACF）就是测量这种记忆的工具。

为什么要专门花一章来讲它？因为后面所有的 AR、MA、ARIMA 模型，选什么参数、定什么阶数，全靠对 ACF 和 PACF 的解读。这一章学好了，后面模型选择你就有了一双火眼金睛。

4.1 ACF：自相关函数

ACF（Autocorrelation Function）衡量的是序列 Y_t 和它的滞后值 Y_{t-k} 之间的相关系数。定义：

ρ_k = Corr(Y_t, Y_{t-k}) = Cov(Y_t, Y_{t-k}) / Var(Y_t)

其中 k 是滞后期数（lag）。ρ_0 永远是1，因为自己和自己完全相关。ρ_1 衡量相邻两点的相关性，ρ_2 衡量间隔两期的相关性，以此类推。

ACF 图（也叫相关图，Correlogram）就是把不同滞后的 ρ_k 画成柱状图。它会告诉你两个关键信息：

  • 衰减速度——如果 ACF 缓慢衰减，序列可能有趋势，非平稳
  • 周期模式——如果 ACF 在某个滞后期出现峰值，说明存在季节性或周期性

4.2 PACF：偏自相关函数

PACF（Partial Autocorrelation Function）比 ACF 稍微难理解一点。它衡量的是扣除中间所有滞后变量的影响后Y_t 与 Y_{t-k} 的直接相关性。

举个例子：假设今天是周三。周三的气温和周一的气温可能相关，但这种相关是因为周二的气温同时影响了周一和周三。PACF 会去掉"周二"的间接影响，只看周一直接对周三的影响。

数学上，PACF 是 Y_t 对 Y_{t-1}, Y_{t-2}, ..., Y_{t-k+1} 做回归后，Y_{t-k} 的系数。

ACF 和 PACF 的组合模式是判断 AR 和 MA 阶数的核心依据。这是时间序列领域最重要的技能之一，值得反复练习。

4.4 白噪声与 Ljung-Box 检验

白噪声（White Noise）是时间序列中的"纯净物"——每个时间点的值独立同分布，均值为0，方差恒定。如果模型拟合后的残差是白噪声，说明模型已经捕捉了所有数据中的结构信息，剩下的只是随机误差。

我们怎么统计上检验残差是不是白噪声？用 Ljung-Box 检验：

  • 原假设 H₀：前 m 个滞后的自相关系数全部为 0（即白噪声）
  • 备择假设 H₁：至少有一个不为 0
  • p 值 > 0.05：不能拒绝原假设，残差是白噪声（好消息）
  • p 值 < 0.05：残差中还有结构，模型需要改进
面试考点：Ljung-Box 的 m 怎么选？太大会降低检验力（被不显著的大滞后稀释），太小会漏掉重要的长周期自相关。经验法则是取 m ≈ min(10, n/5) 或者 m = ln(n)。对月度数据可以取 12 或 24，检查年度季节性是否被捕捉。

4.5 AIC 与 BIC：模型选择的信息准则

ACF 和 PACF 帮我们初步判断模型阶数，但最后的精确选择要靠信息准则。最常用的是 AIC（Akaike Information Criterion） 和 BIC（Bayesian Information Criterion）。

AIC = -2 * ln(L) + 2 * k
BIC = -2 * ln(L) + k * ln(n)

其中 L 是似然值（越大模型拟合越好），k 是参数个数，n 是样本量。

公式逻辑很简单：第一项奖励拟合优度（似然越大越好），第二项惩罚参数过多（防止过拟合）。AIC 和 BIC 的区别在于惩罚力度——BIC 对复杂模型的惩罚更重，选择的模型通常更简洁。

使用建议：

  • AIC 更优的场景——预测为主，愿意容忍稍微复杂的模型来换取更好的预测精度
  • BIC 更优的场景——模型解释为主，希望找到真实的生成过程（假设其包含在候选模型中）
  • 实际工作中，两个都看，如果它们选不一样的模型，倾向于选更简单的那个
真实案例：有一次我帮一个学生诊断他的 ARIMA 模型。他用 ACF 和 PACF 识图后发现两个都在前 3 阶有显著值，后面缓慢衰减。根据我们的口诀，这像是 ARMA(3,3)——但参数太多了，拟合后很多系数不显著。我们改用 AIC 在 0-3 阶范围内搜索，最终选择了 ARMA(1,1)，不仅系数全部显著，预测效果也更好。记住：ACF/PACF 是"指南针"，不是"GPS"，最终定阶要结合信息准则。

4.6 用 Python 绘制相关图

import pandas as pd
import numpy as np
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
from statsmodels.stats.diagnostic import acorr_ljungbox
import matplotlib.pyplot as plt

# 假设 data 是已平稳的序列
# 绘制 ACF 和 PACF
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 6))

plot_acf(data, lags=20, ax=ax1, alpha=0.05)
plot_pacf(data, lags=20, ax=ax2, alpha=0.05)

plt.tight_layout()
plt.show()

# Ljung-Box 检验
lb_test = acorr_ljungbox(data, lags=[10, 20, 30], return_df=True)
print(lb_test)

# 自动搜索最佳 AR 阶数（用 AIC）
from statsmodels.tsa.ar_model import AutoReg
best_aic = np.inf
best_lag = 0
for lag in range(1, 11):
    model = AutoReg(data, lags=lag)
    result = model.fit()
    if result.aic < best_aic:
        best_aic = result.aic
        best_lag = lag
print(f"Best AR lag by AIC: {best_lag}")

到这里，你已经掌握了时间序列分析最核心的"望闻问切"工具——ACF、PACF、白噪声检验、信息准则。你会发现，用这些工具去看任何一条时间序列，就像戴上了 X 光眼镜，数据的内部结构一下变得清晰起来。下一章我们将学习第一种真正的预测方法——指数平滑。它不依赖复杂的模型定阶，而是通过加权历史数据来预测未来，简单却出奇地有效。
