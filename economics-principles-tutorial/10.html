10.1 完全竞争：理想市场的四个条件

前几章我们建立了需求和供给的概念，但并没有深究"市场"本身的结构。这一章开始，我们将进入市场结构的核心地带。首先登场的是完全竞争市场（Perfect Competition）——它是所有市场结构的基准模型，就像物理学中的"无摩擦"理想状态。

一个市场要被称为完全竞争，需要同时满足四个条件：

  • 价格接受者（Price Taker）：每个买家和卖家都太小了，无法影响市场价格。没有人有"市场力量"。
  • 产品同质（Homogeneous Product）：所有卖家的产品完全一样。买家不会因为品牌差异而偏好某家。
  • 自由进入与退出（Free Entry and Exit）：企业可以毫无障碍地进入或退出市场，没有法律、技术或资金壁垒。
  • 完全信息（Perfect Information）：所有市场参与者都知道价格、质量和生产技术的一切信息。

现实中，哪个市场最接近完全竞争？农产品市场是经典例子——你买的大米就是大米，张三家的和李四家的没有本质区别，而且有无数农户在种。股票市场也很接近——一股腾讯股票和另一股完全一样，且你无法用个人力量影响股价。

10.2 利润最大化：MR = MC 的黄金法则

在完全竞争中，企业面临的需求曲线是水平的——因为企业是价格接受者，无论生产多少，单价都等于市场价格P。这意味着：

  • 边际收益（MR）= P，因为每多卖一单位就多收入P元
  • 平均收益（AR）= P，显然

企业的利润最大化条件是：MR = MC，即P = MC。这个公式会贯穿整个市场结构篇。逻辑很简单：如果MR > MC，多生产一单位增加的收入大于成本，应增产；反之应减产；只有当两者相等时，利润达到最大。

计算利润最大化的过程类似这样：

# 利润最大化计算
# 假设某完全竞争企业的成本函数: TC(Q) = 100 + 20Q + Q²
# 市场价格: P = 80

def profit_maximization(P):
    """给定市场价格，计算利润最大化的产量和最大利润"""
    # MC = d(TC)/dQ = 20 + 2Q
    # 利润最大化条件: P = MC
    # 80 = 20 + 2Q
    Q_star = (P - 20) / 2  # Q* = 30

    # 计算总成本和总收益
    TC = 100 + 20*Q_star + Q_star**2  # TC = 100 + 600 + 900 = 1600
    TR = P * Q_star                    # TR = 80 * 30 = 2400
    profit = TR - TC                   # 利润 = 2400 - 1600 = 800

    # 验证MR=MC条件
    MC_at_star = 20 + 2*Q_star         # MC(30) = 80 = P ✓

    return Q_star, profit, TC

Q_star, profit, TC = profit_maximization(80)
print(f"最优产量: Q* = {Q_star:.0f}")
print(f"总收益: TR = {80 * Q_star:.0f}")
print(f"总成本: TC = {TC:.0f}")
print(f"最大经济利润: π = {profit:.0f}")
print("验证: MR=MC=80，利润最大化条件成立 ✓")
# 输出:
# 最优产量: Q* = 30
# 总收益: TR = 2400
# 总成本: TC = 1600
# 最大经济利润: π = 800

10.3 短期停产决策：当P低于AVC时

赚到800块利润当然开心，但如果市场价格下跌了呢？假设P跌到30，企业还继续生产吗？

这里有一个关键概念——沉没成本（Sunk Cost）。固定成本在短期内已经支付且不可收回，因此它不应该影响短期决策。企业真正应该关注的是：收入能否覆盖可变成本。

  • 如果P >= AVC：继续生产。虽然可能亏损（如果P < ATC），但至少收入覆盖了可变成本并"补贴"了一部分固定成本。
  • 如果P < AVC：立即停产！因为收入连可变成本都覆盖不了，多生产一天多亏一天。此时停产只亏固定成本。

因此，企业的短期供给曲线就是MC曲线在AVC最低点以上的部分。当P低于AVC最低点时，供给量为零。

🔑 关键思维：亏损并不等于应该停产。企业要区分"暂时的亏损"和"不可持续的亏损"。只要P > AVC，企业就应该继续生产，因为它在为固定成本"承担义务"——每多生产一单位，虽然可能仍在亏损，但亏损的幅度在减小。只有当P < AVC时，生产本身不创造任何价值，才应该停产。

10.4 长期均衡：P = 最低ATC

长期和短期的关键区别在于：长期中企业可以自由进出。这使得完全竞争市场具备了一种强大的调节机制：

  • 当经济利润 > 0（P > ATC）：新企业看到利润被吸引进入市场，市场供给增加，价格下降，直到利润归零。
  • 当经济利润 < 0（P < ATC）：企业亏损并退出市场，市场供给减少，价格回升，直到不再有企业退出。
  • 长期均衡：P = MC = 最低ATC，经济利润为零。

10.5 故事：中国光伏产业的兴衰

没有比中国光伏产业更能说明"完全竞争"残酷性的案例了。2000年代初，光伏产业被认为是"朝阳产业"，利润丰厚。大量企业蜂拥进入，2007年中国有近千家光伏企业。到2010年，多晶硅价格从每公斤400美元暴跌到50美元以下。

发生了什么？经济学家会说：这是完全竞争的经典剧本。产品高度同质（多晶硅就是多晶硅），技术门槛逐渐降低，资本自由涌入——生产规模急剧扩大，供应远超需求，价格被压到成本线以下。2011-2012年，尚德电力、赛维LDK等龙头企业相继陷入困境，整个行业经历了一次大规模洗牌。

但故事还没完。经过残酷的市场出清，存活下来的企业（隆基、晶科等）通过技术创新持续降低成本，最终实现了"平价上网"——光伏发电成本低于火电。这正是完全竞争的美丽之处：长期中，竞争迫使企业在最低成本处生产，将效率提升到极致，消费者以最低价格获得产品。

到2023年，中国光伏产业占据了全球80%以上的产能，组件价格下降超过90%。消费者（电网企业和终端用户）是最大的受益者——而这正是完全竞争理论所预言的。

💡 思考：完全竞争的"零利润"听起来可怕，但它恰恰是资源有效配置的标志。如果某个行业长期存在超额利润，说明资源没有流向最需要它的地方。竞争的美妙之处就在于此——它迫使利润归零，而其中的效率收益最终归于消费者。

10.6 从完美到不完美

完全竞争是一个理想化的基准，但它揭示了市场机制的核心逻辑：竞争驱使企业降低成本、提高效率，最终让消费者受益。然而，现实生活中大多数市场并不满足完全竞争的条件。当一个企业有能力影响价格时会发生什么？那就是下一章的主题——垄断。当市场只有一个卖家时，价格和效率会发生怎样的变化？我们下一章见。
