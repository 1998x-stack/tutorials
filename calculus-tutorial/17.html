分部积分公式的推导：来自乘积求导

大家好，我是你们的讲师。

前两章我们学习了直接积分法和换元积合法。直接积分法解决基本形式的积分，换元法解决复合函数结构的积分。但这两种方法都无法处理一类非常重要的积分——两个不同类型函数的乘积。比如 ∫ x·e^x dx：它是幂函数与指数函数的乘积，既不是基本公式能直接解决的，也不能通过简单的凑微分来解（因为 x 和 e^x 之间没有互为导数的关系）。

那怎么办？既然直接积分的路走不通，我们能不能从导数中找灵感？前面说过，不定积分是求导的逆运算。那么我们来看看，什么函数的导数会同时包含两种不同类型的函数？

答案是乘积求导法则（product rule）。回顾一下：

(u · v)' = u' · v + u · v'

两边对 x 积分：

∫ (u · v)' dx = ∫ u'·v dx + ∫ u·v' dx

        左边就是 u · v（积分抵消导数），所以：
        u · v = ∫ u'·v dx + ∫ u·v' dx

整理一下，就得到了分部积分公式：

∫ u·v' dx = u·v - ∫ u'·v dx

        更常见的形式是：
        ∫ u dv = u·v - ∫ v du

这个公式的妙处在于：它把一个难积的 ∫ u dv 转换成了一个可能更容易的 ∫ v du。代价是出现了 u·v 这一项——实际上是把积分的难度从一部分转移到了另一部分。

让我们用分部积分法计算 ∫ x·e^x dx：

设 u = x，dv = e^x dx
        则 du = dx，v = ∫ e^x dx = e^x

        代入公式：
        ∫ x·e^x dx = x·e^x - ∫ e^x dx = x·e^x - e^x + C = e^x(x - 1) + C

验证一下：d/dx [e^x(x - 1)] = e^x(x - 1) + e^x·1 = x·e^x，正确！

u 和 dv 的选择策略：LIATE 原则

分部积分法的关键挑战是：面对一个积分 ∫ f(x) g(x) dx，到底应该把谁设为 u，把谁设为 dv？选择了不同的分配方式，计算难度可能天差地别。

来看一个反例。如果上面那道题我们反过来设：

设 u = e^x，dv = x dx
        则 du = e^x dx，v = x^2/2

        ∫ x·e^x dx = e^x·(x^2/2) - ∫ (x^2/2)·e^x dx

        新的积分比原来的还复杂！x 的幂次从 1 升到了 2。

所以 u 的选择原则是：选择那些求导后会变"简单"的函数作为 u。而 dv 的选择原则是：选择那些积分后不会变"复杂"的函数作为 dv。

根据这个原则，人们总结出了一个非常实用的选择优先级——LIATE 原则：

优先级 字母 含义 举例 为什么优先设为 u
最高 L Logarithmic（对数函数） ln x, log_a x 求导后变成幂函数，更简单
↑ I Inverse trig（反三角） arcsin x, arctan x 求导后变成代数函数
↑ A Algebraic（代数函数） x^n, 多项式 求导后幂次降低，逐步简化
↑ T Trigonometric（三角函数） sin x, cos x 可轮换，积分后仍是三角函数
最低 E Exponential（指数函数） e^x, a^x 求导和积分后形式不变

按照 LIATE 顺序，排在前面的函数优先设为 u，排在后面的设为 dv。所以：

  • ∫ x·e^x dx：A 在 E 之前，所以 u = x（代数），dv = e^x dx（指数）
  • ∫ ln x dx：L 在 A 之前（这里 A 是 1），所以 u = ln x，dv = dx
  • ∫ x·sin x dx：A 在 T 之前，所以 u = x，dv = sin x dx
  • ∫ e^x·sin x dx：E 和 T 相邻，任选一个为 u，但需要循环使用分部积分
LIATE 只是一个经验法则，不是定理。对于 90% 的题目它都管用，但偶尔会有例外。我建议你把它当作"首选策略"而非"唯一策略"。如果按 LIATE 选择后越算越复杂，果断换另一种分配方式试试。

循环分部积分技巧

有些积分在使用分部积分后，右边依然出现和左边相同形式的积分。这时我们把它看作一个方程，可以通过代数等式求解。这就是"循环分部积分"（integration by parts with a循环）。

例：∫ e^x · sin x dx

        设 u = sin x，dv = e^x dx
        则 du = cos x dx，v = e^x
        I = ∫ e^x sin x dx = e^x sin x - ∫ e^x cos x dx

        对 ∫ e^x cos x dx 再次分部积分：
        设 u = cos x，dv = e^x dx
        则 du = -sin x dx，v = e^x
        ∫ e^x cos x dx = e^x cos x - ∫ e^x (-sin x) dx = e^x cos x + ∫ e^x sin x dx

        代入原式：
        I = e^x sin x - (e^x cos x + I)
        I = e^x sin x - e^x cos x - I
        2I = e^x(sin x - cos x)
        I = (1/2)e^x(sin x - cos x) + C

这种技巧也常用于 ∫ e^(ax)·sin(bx) dx 和 ∫ e^(ax)·cos(bx) dx 的积分。诀窍是：两次分部积分之后，右边会出现原始积分本身，然后把它当作未知数求解。

我再给大家展示一个看似简单但极易出错的经典例子——∫ ln x dx：

∫ ln x dx

        这里的被积函数只有一个"ln x"，我们把它看作 ln x · 1。
        按 LIATE：L 优先，所以 u = ln x，dv = dx
        du = 1/x dx，v = x

        ∫ ln x dx = x·ln x - ∫ x·(1/x) dx = x·ln x - ∫ 1 dx = x·ln x - x + C

我第一次看到这个结果时，有点意外——原来自然对数的不定积分这么简洁！后来在工作中处理信息熵相关的计算时，经常用到这个结果。信息熵公式 H = -∑ p·log p 的连续版本就涉及 ∫ p(x)·ln p(x) dx 形式的积分。如果不熟悉分部积分法，这些应用根本无从下手。

有理函数的真分式分解

让我们转向另一个大主题——有理函数的积分。有理函数是两个多项式之比 P(x)/Q(x)。如果分子的次数低于分母的次数，就称为真分式（proper rational function）；否则是假分式，需要通过多项式除法化简为"多项式 + 真分式"的形式。

有理函数积分的基本思想是：任何有理函数都可以分解为一系列部分分式（partial fractions）的和，而每个部分分式都可以用基本积分公式或简单换元法积分。

具体来说，如果分母 Q(x) 可以分解成一次和二次因式的乘积：

Q(x) = (a₁x + b₁)^k₁ · (a₂x² + b₂x + c₂)^k₂ · ...

那么真分式 P(x)/Q(x) 就可以分解为以下四种基本分式的和：

类型 部分分式形式 对应的积分
单重一次因子 A/(ax + b) (A/a) ln|ax + b| + C
重一次因子 A_k/(ax + b)^k A_k · (ax + b)^(1-k) / (a(1-k)) + C（k > 1）
单重二次因子 (Ax + B)/(ax² + bx + c) arctan 或 ln 形式的组合
重二次因子 (A_k x + B_k)/(ax² + bx + c)^k 需递推公式，较为复杂

待定系数法

部分分式分解中最核心的操作就是确定每个分式前的系数。最常用的方法就是待定系数法（method of undetermined coefficients）。

来看一个例子：

求 ∫ 1/(x² - 3x + 2) dx

        第一步：分解分母
        x² - 3x + 2 = (x - 1)(x - 2)

        第二步：设部分分式
        1/[(x-1)(x-2)] = A/(x-1) + B/(x-2)

        第三步：两边乘以分母
        1 = A(x-2) + B(x-1)

        第四步：待定系数
        展开：1 = (A+B)x + (-2A - B)
        比较系数：
        { A + B = 0
        { -2A - B = 1
        解得 A = -1，B = 1

        第五步：积分
        ∫ 1/(x²-3x+2) dx = ∫ [-1/(x-1) + 1/(x-2)] dx
        = -ln|x-1| + ln|x-2| + C
        = ln|(x-2)/(x-1)| + C
常见错误：部分分式分解中最常见的错误来自重根的处理。如果分母有 (ax + b)² 这样的重一次因子，你需要设两个项：A/(ax+b) + B/(ax+b)²，而不是只设一项。很多初学者在这里犯错，导致分解不完整，积分结果自然也就错了。我在本科第一次学这部分时，因为漏掉了重根的高次项，算了半天都对不上答案，后来发现这个错误后懊恼不已。

四种基本分式的积分

在完成部分分式分解后，我们最终需要积分的无非是以下四类：

类型一：∫ 1/(ax + b) dx

直接凑微分即可：= (1/a) ln|ax + b| + C

类型二：∫ 1/(ax + b)^k dx（k > 1）

幂函数积分：= 1/(a(1-k)) · (ax + b)^(1-k) + C

类型三：∫ (Ax + B)/(x² + px + q) dx（分母判别式 < 0）

这是最常用的类型。配方后拆成两项：

以 ∫ (2x + 3)/(x² + 4x + 13) dx 为例：

        分母：x² + 4x + 13 = (x+2)² + 9

        拆分分子：2x + 3 = 2(x+2) - 1
        （目的是让一部分是分母的导数 2x+4 的形式）

        原式 = ∫ [2(x+2) - 1]/[(x+2)² + 9] dx
        = ∫ 2(x+2)/[(x+2)² + 9] dx + ∫ (-1)/[(x+2)² + 9] dx
        = ln|(x+2)² + 9| - (1/3) arctan((x+2)/3) + C

类型四：∫ (Ax + B)/(x² + px + q)^k dx（k > 1）

需要利用递推公式（reduction formula）。这个比较复杂，在实际题目中出现频率不高，但考研和竞赛中偶尔会见到。

一个工程实践中的故事：我在做控制系统分析时遇到过有理函数积分的应用。控制系统的传递函数通常是两个多项式的比值，而系统的时域响应就是它的拉普拉斯逆变换，本质上就是在做有理函数的部分分式分解和积分。有一次我在设计一个 PID 控制器时，需要计算某个复杂传递函数的单位阶跃响应。手动计算部分分式分解花了我整整一个下午。后来我花了半天时间写了一个 Python 脚本来自动做部分分式分解——这让我深刻体会到：理解部分分式分解的原理比实际手动计算更重要，因为在工程实践中，大部分这样的计算都由计算机代劳了。但如果你不理解原理，你连怎么调用 sympy 的 apart() 函数都不知道，更不用说判断结果是否正确了。

本章我们学习了两种强大的积分工具：分部积分法专门用于处理两类不同函数的乘积，而部分分式分解则是有理函数积分的系统化方法。掌握了这两把武器，再加上前面的直接积分法和换元法，你就能够解决微积分课程中 95% 以上的不定积分问题了。

下一章我们将进入到积分学的另一个里程碑——定积分。前面三章我们一直在学习的都是不定积分（一族函数），而定积分将会给出一个具体的数值——曲边梯形的面积。更重要的是，下一章我们将会揭开微积分中最迷人的公式——牛顿-莱布尼茨公式，它告诉我们：定积分可以通过原函数在端点处的差值来计算。正是这个公式，把微分和积分联系成了一个统一的整体，这就是"微积分"这个名字的真正含义。
