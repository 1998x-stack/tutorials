开篇：当傅里叶变换失效时

大家好，我是你们的讲师。在"信号与系统"课程中，傅里叶变换是我们最强大的武器：任何一个确定性信号 x(t)，只要它绝对可积（∫|x(t)|dt < ∞），就能被分解为不同频率的正弦波之和。频谱 X(ω) 清晰地告诉我们"信号在不同频率上的成分大小"。

但到了随机信号这里，这套方法碰壁了。随机信号 X(t) 在时间轴上无限延伸，它的一个典型样本不满足绝对可积条件——∫|x(t)|dt 是发散的。即使我们强行计算傅里叶变换，结果也是一团糟（事实上，随机过程的傅里叶变换在经典意义下不存在）。那怎么办？我们需要的不是"信号的频谱"，而是信号功率的频谱——即信号的平均功率在频率上是如何分布的。这就是功率谱密度（Power Spectral Density, PSD）的由来。

从傅里叶变换到功率谱

确定性信号的频谱 vs. 随机信号的功率谱

先回顾一下确定性信号的处理方式。对于能量有限信号（∫|x(t)|²dt < ∞），帕塞瓦尔定理（Parseval's Theorem）告诉我们：

∫|x(t)|² dt = (1/2π) ∫|X(ω)|² dω

左边是时域的总能量，右边是频域的"能量谱密度"|X(ω)|² 的积分。确定性信号分析到此完美——能量守恒，一目了然。

但随机信号是功率信号而非能量信号——它在无限时间区间上的能量是无穷大，但它的平均功率是有穷的。这迫使我们转换思路：与其研究信号"一次性释放的总能量"，不如研究它"持续释放的平均功率"。

所以问题转化为：如何定义一个量，可以描述随机信号的平均功率在频率轴上的分布？答案是先从自相关函数入手，再通过傅里叶变换转回频域——这就是维纳-辛钦定理的核心思想。

功率谱密度的定义

PSD的物理意义

功率谱密度 S_X(ω) 的物理意义非常直观：它告诉我们信号 X(t) 的平均功率在频率轴上是如何分布的。具体来说，信号在频率区间 [ω, ω + dω] 内的平均功率为 S_X(ω) · dω/(2π)。信号的总平均功率等于 PSD 在全频率上的积分：

E[|X(t)|²] = (1/2π) ∫_{-∞}^{∞} S_X(ω) dω

注意这里的 E[|X(t)|²] 就是信号的总平均功率（对 WSS 信号来说，它不依赖于 t）。

如果这个公式让你联想到帕塞瓦尔定理，那就对了——PSD 就是随机信号版本的"能量谱密度"，只不过处理的对象从能量变成了平均功率。

为什么自相关是关键

一个深刻的洞察是：信号在不同频率上的功率分布，本质上是由信号在不同时刻之间的相关性结构决定的。如果信号变化缓慢（相邻时刻高度相关），那么它的功率主要集中在低频；如果信号剧烈振荡（相邻时刻几乎不相关），那么它的功率散布在整个频带上。而这个相关性结构，恰好由自相关函数 R_X(τ) 完整刻画。

这就引出了本章的核心定理。

维纳-辛钦定理

定理陈述

维纳-辛钦定理（Wiener-Khinchin Theorem）是随机信号分析的"桥梁定理"。它建立了时域和频域之间的等价关系：

对于一个宽平稳随机信号 X(t)，其功率谱密度 S_X(ω) 与自相关函数 R_X(τ) 构成一对傅里叶变换对：

S_X(ω) = ∫_{-∞}^{∞} R_X(τ) e^{-jωτ} dτ

R_X(τ) = (1/2π) ∫_{-∞}^{∞} S_X(ω) e^{jωτ} dω

这个定理的优美之处在于：它告诉我们时域的自相关和频域的功率谱密度包含着完全相同的统计信息——只是用不同的语言（时域语言 vs 频域语言）表达了同一个事实。你可以选择在时域分析（用自相关），也可以选择在频域分析（用功率谱），取决于哪个视角更方便。

推导思路

维纳-辛钦定理的严格证明需要一些数学技巧，但核心思路可以这样理解：

  1. 截断：将无限长的随机信号 X(t) 截断为一个有限区间 [-T, T]，得到 X_T(t)。截断后的信号是能量有限的，其傅里叶变换 X_T(ω) 存在。
  2. 傅里叶变换：计算截断信号的"能量谱" |X_T(ω)|²。
  3. 取期望：对 |X_T(ω)|² 取统计平均——因为我们关心的是平均功率。
  4. 取极限：让 T → ∞，将截断区间恢复到无限长。在取极限的过程中，出人意料的事情发生了——极限不存在于样本路径层面，但存在于期望层面。这个极限就是 PSD。
直观证明：更直接的理解方式是：R_X(τ) 反映了信号在间隔 τ 上的相关性。如果我们在频域中查看这种相关性，不同频率成分的功率贡献恰好是 R_X(τ) 的傅里叶变换。这就像棱镜将白光分解为彩虹——自相关是"白光"，PSD 是"彩虹光谱"，维纳-辛钦定理就是那个"棱镜"。

PSD的性质

基本性质

由维纳-辛钦定理可以直接推导出 PSD 的一系列重要性质：

  • 非负性：S_X(ω) ≥ 0 对所有 ω 成立。功率不能为负！这是物理可实现性的必要条件。
  • 偶函数（实信号）：对于实值的随机信号，R_X(τ) 是实偶函数，因此 S_X(ω) 也是实的偶函数：S_X(ω) = S_X(-ω)。这也是为什么工程中经常只画 ω ≥ 0 一侧的单边谱。
  • 总面积 = 平均功率：PSD 曲线下的总面积等于信号的总平均功率 E[|X(t)|²]。
  • 非负定性：自相关函数 R_X(τ) 是一个非负定函数——这意味着它的傅里叶变换（PSD）必须非负。反过来，任意非负定的函数都可以是某个 WSS 信号的自相关函数。

互功率谱密度（CPSD）

类似地，对于两个联合平稳的随机信号 X(t) 和 Y(t)，我们可以定义它们的互功率谱密度（Cross Power Spectral Density, CPSD）：

S_{XY}(ω) = ∫_{-∞}^{∞} R_{XY}(τ) e^{-jωτ} dτ

CPSD 反映了两个信号在不同频率上的相互关系。注意 CPSD 不一定是实的——即使 X 和 Y 都是实信号，它们的互相关 R_{XY}(τ) 可能不是偶函数，因此 S_{XY}(ω) 可能是复数。它的幅值表示两个信号在对应频率上的相关强度，相位表示相位差。

观察上表你会发现一个模式：自相关衰减得越慢（例如指数衰减的 α 很小），PSD 在低频越集中；自相关衰减得越快（例如白噪声的 δ 函数），PSD 越平坦。这是时-频不确定关系的体现——一个域的"宽度"与另一个域的"宽度"成反比。

踩坑故事：直接对随机信号做FFT的灾难

几年前，我帮一个做无线通信的团队调试一个频谱感知算法。他们的想法很简单：用软件无线电（SDR）采集一段频谱数据，对采样序列直接做 FFT，然后把 FFT 幅度的平方当作功率谱密度来用，再设置一个门限来判断某个频段是否被占用。

实验的初始结果看起来不错——算法能够检测到已知的 Wi-Fi 信号。但当团队把系统搬到真实的办公楼环境中测试时，性能急剧下降：虚警率（错误地判断频段被占用）飙升到 30% 以上。他们百思不得其解，反复检查硬件和实现代码都没有发现问题。

后来我仔细看了他们的数据处理流程，一下子就发现了问题：他们直接对原始采样序列的 FFT 幅度平方做平均，没有加窗，没有做任何谱平滑处理。直接用 FFT 幅度平方估计 PSD（即周期图法）是有偏的——它的方差不会随着数据长度的增加而降低！这就是所谓的"周期图不一致性"问题。即使信号是纯白噪声，周期图估计出的 PSD 也是剧烈震荡的，完全不像平坦的谱线。

换句话说，他们犯的错误相当于：用一个随机变量的单次实现来估计它的均值——理论上可以做，但估计量的方差太大了。而且这个问题不通过加窗、分段平均或频谱平滑（即 Welch 方法）是无法解决的。

教训：FFT 的幅度平方（|X(ω)|²）不是 PSD 的好估计！维纳-辛钦定理告诉我们 PSD 是自相关函数的傅里叶变换，而不是直接对信号做 FFT 再取平方。正确的做法是：(1) 用 Welch 方法（分段加窗平均）来降低估计方差；(2) 或者先估计自相关函数再变换到频域。前者是现代谱分析的标配，避免了"谱泄露"和"高方差"两大问题。记住：一个样本的 FFT 平方不叫功率谱，叫"容易被误导"。

本章小结

这一章我们学习了随机信号频域分析的核心工具——功率谱密度（PSD）。我们从傅里叶变换在随机信号上的失效谈起，引入了 PSD 的概念与物理意义，然后详细推导和讲解了维纳-辛钦定理——这座时域与频域之间的桥梁。PSD 的非负性、偶函数等性质，以及常见信号的 PSD 例子，都帮助你建立"时域自相关 ⇔ 频域功率谱"的双向直觉。最后用一个周期图估计的踩坑故事警示了不遵循维纳-辛钦定理直接做 FFT 谱分析的危害。

下一章我们将学习随机信号分析中最常用的噪声模型——白噪声与热噪声。你会发现，PSD 恒定的白噪声是一个理想化的模型，它虽然物理上不可实现，但却是理解所有更复杂噪声的起点。而热噪声——你电阻两端那个永远存在的微小电压——正是白噪声的物理代言人。
