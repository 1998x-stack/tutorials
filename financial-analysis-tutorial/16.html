第16章 投资组合理论

大家好，我是你们的讲师。假设你面前有三支基金：A基金去年涨了60%，B基金涨了15%，C基金跌了5%。你会选哪个？如果你毫不犹豫地选了A，恭喜你——你和绝大多数人的直觉一样。但这个直觉大概率是错的。

为什么？因为投资考量的不是单支资产的回报，而是整个组合的风险收益特征。一个由"60%的平庸资产+40%的现金"构成的组合，在风险调整后的表现可能远远好于"100%的明星基金"。这听起来反直觉，却是过去七十年金融学最重要的发现之一。

我在刚入行的时候，曾经帮一位老前辈管理一个很小的个人账户。他告诉我："小明，你只要记住一件事——不要把鸡蛋放在一个篮子里，也不要放在太多篮子里以至于忘了哪些篮子有鸡蛋。"后面这句话是我后来自己加的，但前半句，就是投资组合理论最朴素、最核心的智慧。

16.1 马科维茨均值-方差优化

1952年，一位名叫哈里·马科维茨（Harry Markowitz）的年轻博士生在《金融期刊》上发表了一篇只有14页的论文，题目是《投资组合选择》（Portfolio Selection）。这篇论文彻底改变了金融学的面貌，最终为马科维茨赢得了诺贝尔经济学奖。

马科维茨的核心思想非常优雅：投资者不应该单独评估每一项资产，而应该考虑它们之间的相关性。不同资产组合在一起的整体风险，并不是单个资产风险的简单加权平均——因为资产之间的协方差可以抵消部分风险。

从数学上看，对于一个由N支资产构成的组合，其方差可以表示为：

  • 组合方差 = Σ(wi²σi²) + 2ΣΣ(wiwjCov(i,j))，其中 i < j
  • wi 为资产i的权重，σi² 为资产i的方差，Cov(i,j) 为资产i和j的协方差

当资产之间呈负相关时，Cov(i,j) < 0，组合的方差就会小于单个资产方差的加权平均。这就是"分散化"的数学本质——不是降低每个资产的波动，而是利用资产之间的低相关性来抵消波动。

有效前沿（Efficient Frontier）是均值-方差框架的核心概念。对于一组给定的资产，通过调整各资产的配置权重，可以绘制出一条"在给定风险水平下收益最大"的曲线。所有在这条曲线上的组合都是"有效的"——它们以最小风险实现了给定收益，或者以给定风险实现了最大收益。

投资者根据自己的风险偏好（即无差异曲线 indifference curve 的形状），在有效前沿上选择一个"最优点"。风险厌恶型投资者的最优组合会偏向左侧（低风险、低收益），风险偏好型则偏向右侧。

数学之美：有效前沿为什么是"凸"的？直觉上理解：当你在有效前沿上向右侧移动（承担更多风险）时，每单位额外风险带来的额外收益是递减的。这就像吃东西——饿的时候第一口最爽，吃到后面每一口的满足感都在下降。

下面用Python和PyPortfolioOpt库来展示如何计算有效前沿：

import akshare as ak
import pandas as pd
import numpy as np
from pypfopt import expected_returns, risk_models, EfficientFrontier

# 获取三支代表性资产的日行情
tickers = ['600519', '000858', '159915']  # 茅台、五粮液、创业板ETF
data = {}
for t in tickers:
    df = ak.stock_zh_a_hist(symbol=t, period="daily",
                            start_date="20240101", end_date="20241231")
    data[t] = df['收盘']

prices = pd.DataFrame(data)

# 计算预期收益和协方差矩阵
mu = expected_returns.mean_historical_return(prices)
S = risk_models.sample_cov(prices)

# 寻找最大夏普比率组合
ef = EfficientFrontier(mu, S)
weights = ef.max_sharpe()
cleaned_weights = ef.clean_weights()

print("最优权重分配（最大化夏普比率）：")
for asset, w in cleaned_weights.items():
    print(f"  {asset}: {w:.2%}")

print(f"\n预期年化收益率: {ef.portfolio_performance()[0]:.2%}")
print(f"预期年化波动率: {ef.portfolio_performance()[1]:.2%}")
print(f"夏普比率: {ef.portfolio_performance()[2]:.2f}")

这段代码展示了标的的选择、协方差矩阵的估计和有效前沿上的最优组合选择三个步骤。实际应用中，均值-方差优化对输入参数极其敏感——期望收益估计微调1%，最优权重可能发生20%的变化。这也是马科维茨框架在实践中最大的软肋。

16.2 CAPM与Beta

在马科维茨框架的基础上，威廉·夏普（William Sharpe）等学者在20世纪60年代提出了资本资产定价模型（Capital Asset Pricing Model, CAPM）。CAPM的洞察在于：它将风险分解为两个部分——系统性风险（systematic risk，又称市场风险）和非系统性风险（idiosyncratic risk）。

系统性风险是无法通过分散化消除的——比如经济衰退、利率变动、地缘政治冲击等影响所有资产的因素。非系统性风险是单个公司或行业特有的——比如管理层丑闻、产品召回、原材料涨价——这些可以通过持有多样化的资产组合来分散掉。

CAPM的核心公式：

  • E(Ri) = Rf + βi × [E(Rm) - Rf]
  • 其中 E(Ri) 为资产i的预期收益率，Rf 为无风险利率，E(Rm) 为市场组合的预期收益率，βi 为资产i的Beta系数

Beta（β）衡量的是个股相对于整个市场的敏感度。β = 1.5意味着：市场涨1%，该股票平均涨1.5%（反之亦然）。β > 1 的股票被称为"进攻型"（如券商股、科技股），β < 1 的股票则为"防御型"（如公用事业、消费品）。

CAPM的一个重要推论是：高收益只能来自承担高系统性风险。如果你买的股票跑赢了市场，那只是因为它的Beta更高——而不是因为你的选股能力强。当然，事实远非如此简单——这也正是后面"Alpha"概念的意义所在。

Alpha（α）衡量的是实际收益超过CAPM模型预期收益的部分。α > 0意味着基金经理创造了"超额收益"——这可能是运气，也可能确实来自卓越的投资能力。

核心洞见：我个人认为，理解"Alpha vs Beta"的区分是金融分析中最基本的思维框架之一。它让你在做投资决策时能够清晰地知道：你是在承担系统性风险（Beta）赚市场的钱，还是在利用信息优势或分析能力（Alpha）赚超额收益。这两个赚钱逻辑完全不同，需要的技能和工具也截然不同。

需要注意的是，CAPM虽然概念简洁优美，但学术界对其有效性一直存在激烈争论。Fama-French的三因子模型、Carhart四因子模型，乃至后续的多因子模型，都是在CAPM基础上的改进——我们将在第19章详细讨论。

16.3 风险平价与Black-Litterman

经典的均值-方差优化有一个致命的实践问题：它极度依赖预期收益率的估计，而收益率的预测是出了名的不准确。这导致了一个尴尬的局面——优化出来的权重往往极不稳定，而且经常集中在少数几类资产上。为了解决这个问题，业界发展出了两种重要的替代方案。

风险平价（Risk Parity）

风险平价策略放弃了"预测收益"这个环节，转而聚焦于风险贡献的均等化。其核心思想是：与其让各类资产按资金权重分配，不如让各类资产对组合总风险的贡献度相等。

举个简单的例子：假设你的组合中有60%的股票和40%的债券。但股票的风险（波动率）是债券的三倍。实际上，组合中大约95%的风险来自股票——债券虽然占了40%的资金，却几乎没有为组合带来任何风险上的分散化。风险平价策略的做法是：降低股票的权重、增加债券的杠杆（通常通过期货等工具），直到两者的风险贡献大致相等。

桥水基金（Bridgewater Associates）的"全天候策略"（All Weather Strategy）是风险平价理念最著名的实践者。该策略将经济环境划分为四个象限（增长上升/下降 × 通胀上升/下降），在每个象限中配置合适的资产，确保无论经济处于哪个阶段，组合中总有资产在"工作"。桥水的全天候策略在过去三十年间实现了稳健的绝对回报，最大回撤远低于传统股债组合。

学习建议：风险平价的理念对个人投资者同样有价值。即便你无法使用杠杆来放大债券的风险贡献，至少可以审视一下自己的组合："这20个持仓到底是在分散风险，还是20个同涨同跌的Beta？"

Black-Litterman模型

Black-Litterman模型（由Fischer Black和Robert Litterman在高盛期间提出）是另一种应对均值-方差优化"输入敏感"问题的方法。它的思路非常巧妙：从市场均衡（CAPM隐含权重）出发，然后结合投资者的主观观点进行修正。

具体来说：

  • 先根据市场组合的市值权重反推出"隐含的预期收益率"——这相当于一个中性的、无观点的基准
  • 投资者可以输入自己对某些资产的观点（例如"茅台未来一年将跑赢行业平均5%"），并附带信心水平
  • 模型使用贝叶斯统计将中性预期和主观观点进行"融合"，得到修正后的预期收益率
  • 再将修正后的预期收益率输入均值-方差优化，得到最终的组合权重

Black-Litterman模型的优雅之处在于：当投资者没有任何观点时，它的输出就是市场组合本身（即被动投资的"最优解"）；当投资者非常有信心时，它的输出会向投资者的方向倾斜。这给了主动管理一个理论上的"容身之处"——你可以有观点，但必须知道自己在和"市场共识"对着干。

16.4 绩效归因体系

做出一个投资组合后，你需要回答一个灵魂拷问：它为什么赚钱（或亏钱）？是资产配置做对了，还是个股选择优秀，抑或纯粹是运气好赶上了牛市？绩效归因（Performance Attribution）就是回答这个问题的方法论。

最经典的方法是Brinson归因（由Gary Brinson在1986年提出），它将超额收益分解为三个部分：配置效应（allocation effect）、选择效应（selection effect）和交互效应（interaction effect）。

效应类型 定义 计算公式 解读
配置效应 大类资产配置偏离基准带来的超额收益 Σ(Wpi - Wbi) × Rbi 是否在正确的赛道下注
选择效应 在每类资产内部选股/选券的能力 ΣWbi × (Rpi - Rbi) 是否在赛道中挑到了好选手
交互效应 配置和选择的协同影响 Σ(Wpi - Wbi) × (Rpi - Rbi) 配置和选择是否产生了共振

其中 Wpi 为组合在资产i上的权重，Wbi 为基准中资产i的权重，Rpi 为组合在资产i上的收益率，Rbi 为基准中资产i的收益率。

来看一个具体的例子。假设你的基准是"60%沪深300 + 40%中证全债"，但你选择了"70%沪深300 + 30%中证全债"的配置，并且你买的股票跑赢了沪深300指数3%。那么：

  • 配置效应 = (70% - 60%) × 12% + (30% - 40%) × 4% = 1.2% - 0.4% = +0.8%（因为超配了表现更好的股票）
  • 选择效应 = 60% × 3% + 40% × 0% = +1.8%（选股能力贡献了1.8%）
  • 交互效应 = (70%-60%) × 3% + (30%-40%) × 0% = +0.3%

这个分析揭示了一个重要的信息：你的超额收益主要来自选股能力（+1.8%），配置决策贡献相对较小（+0.8%）。如果结果反过来——配置效应很大但选择效应为负——那就说明你选对了方向但选错了具体的标的。

核心洞见：Brinson归因的最大价值不在于精确计算到小数点后几位，而在于迫使你持续追问"我的优势到底在哪里"。如果你发现自己每次赚钱都是靠"配置效应"——即市场Beta——而选择效应长期为负，或许你应该考虑转型做指数投资，或者至少在选股环节投更多精力。

下面是用常用绩效指标来评估组合表现的汇总表：

指标 公式 含义 理想值
夏普比率 (Rp - Rf) / σp 每单位总风险的超额回报 > 1（良好），> 2（优秀）
索提诺比率 (Rp - Rf) / σ_downside 每单位下行风险的超额回报 > 1.5
卡玛比率 (Rp - Rf) / MDD 每单位最大回撤的超额回报 > 1（良好），> 2（优秀）
信息比率 (Rp - Rb) / TE 每单位跟踪误差的超额回报 > 0.5（良好），> 1（优秀）

其中 Rp 为组合收益率，Rf 为无风险利率，σp 为组合波动率，σ_downside 为下行波动率，MDD 为最大回撤，Rb 为基准收益率，TE 为跟踪误差（Tracking Error）。

踩坑警示：注意，这些指标都是基于历史数据计算的，而历史不代表未来。我见过一个私募产品，过去三年的夏普比率高达2.8（非常惊人），但第四年就腰斩了——因为它的高夏普来自持续做空波动率（short volatility），而波动率终有一天会爆发。任何看起来"完美无瑕"的指标，都可能隐藏着你没有识别的风险。指标是工具，不是信仰。

投资组合理论是一个深度和广度都极其丰富的领域。马科维茨给了我们分析框架，CAPM提供了定价基准，风险平价和Black-Litterman弥补了实践中的缺陷，而Brinson归因则帮我们理解收益的来源。掌握这些理论，你就拥有了从"拍脑袋配置"升级到"科学化管理组合"的基础能力。
