第六章：弹性——衡量市场反应

大家好！前几章我们建立了供需框架，学会了如何分析一个外部变化会让均衡往哪个方向走。但学过物理的人都知道，"知道方向"和"知道大小"是两回事。今天我们就来学那个"大小"的工具——弹性 elasticity。

弹性可能是整个微观经济学中最实用的概念。定价策略、税收政策、市场预测——不管你将来做产品、做投资、还是做政策分析，弹性都是你每天都会用到的东西。

弹性到底是什么

弹性衡量的是：当一个变量变化1%时，另一个变量会变化百分之几。用最直观的话说——弹性就是"敏感度"。

一个关键区别：弹性不是斜率。斜率取决于单位（元/斤、美元/桶），换了单位斜率就变。但弹性是百分比之比，没有单位，因此可以在不同商品、不同市场之间直接做比较。

# 需求价格弹性 Price Elasticity of Demand 的通用公式
# 弧弹性（中点法）——最常用的版本

P1 = 初始价格   Q1 = 初始需求量
P2 = 新价格     Q2 = 新需求量

需求价格弹性 = | (Q2 - Q1) / ((Q2 + Q1)/2) ÷ (P2 - P1) / ((P2 + P1)/2) |

# 为什么用中点法？因为从A到B和从B到A得到同样的弹性值
# 不用中点法的话，方向不同会算出不同的结果

# 数值示例：
# 咖啡从25元涨到30元，需求量从100杯降到80杯

价格变化百分比 = (30-25) / ((30+25)/2) = 5/27.5 = 18.18%
需求变化百分比 = (80-100) / ((80+100)/2) = -20/90 = -22.22%
弹性 = | -22.22% / 18.18% | = 1.22

# 结果：|1.22| > 1 → 弹性需求（消费者对价格敏感）

记忆技巧：弹性大于1叫"弹性"（敏感），小于1叫"缺乏弹性"（不敏感），等于1叫"单位弹性"。绝对值符号很重要——经济学家讨论弹性时默认带上绝对值，只说"弹性的绝对值是多少"。

决定需求价格弹性大小的因素有很多：

  • 替代品的多少——替代品越多，弹性越大。牛肉的弹性大（猪肉鸡肉都可以替代），胰岛素弹性几乎为零（没有替代品）。
  • 商品占预算的比例——占预算比例高，弹性大。汽车弹性大（买一辆车是重大支出），火柴弹性小（花不了几个钱）。
  • 时间长短——时间越长，弹性越大。短期内汽油涨价你只能忍受，长期内你可以换电动车或搬家离公司更近。

弹性决定定价策略

这是弹性概念最直接、最实用的应用——总收益测试 total revenue test。

总收益 Total Revenue = 价格 × 销售量。当价格变化时，销售量也会反向变化。你的总收益最终是升还是降，取决于哪个方向的变化幅度更大——而"幅度"就是弹性。

需求弹性 涨价 → 总收益 降价 → 总收益 商业策略启示
弹性（>1） 下降（量跌得比价涨得快） 上升（量涨得比价跌得快） 不要轻易涨价，考虑促销降价
缺乏弹性（<1） 上升（量跌得少） 下降（量涨得少） 涨价是好的策略，降价是"自残"
单位弹性（=1） 不变（量和价格等比例变化） 不变 价格变化不影响总收益

这解释了为什么：

  • 奢侈品品牌（LV、Chanel）经常涨价——核心客群对价格不敏感，需求缺乏弹性，涨价直接增加收入。
  • 航空公司的最后一分钟机票折扣巨大——商务旅客已买完，剩下来的都是休闲旅客，对价格高度敏感（弹性高），打折才能填满座位。
  • 互联网软件为什么拼命做免费策略——数字产品的边际成本近乎为零，价格从零开始。一旦用户习惯了免费，需求变得极其弹性，任何收费都会导致大量流失。所以它们的变现方式是广告和增值服务。

关键洞察：亚马逊在2010年代做过一个著名的实验——他们用算法发现某些品类的需求弹性极高，小幅降价带来销量暴涨，总收益反而大幅增加。这促成了亚马逊"天天低价"策略的数学基础。弹性数据，就是定价权的"GPS"。

收入弹性与交叉弹性

价格弹性是最常用的，但不是唯一的。下面这两种弹性对商业分析同样重要。

收入弹性 Income Elasticity of Demand：衡量的是需求变化对收入变化的敏感度。

收入弹性 = 需求量变化百分比 / 收入变化百分比

结果解读:
  收入弹性 > 0     → 正常品（收入增加需求也增加）
  收入弹性 > 1     → 奢侈品（精神消费、高端旅游——收入涨了买得更多）
  0 < 收入弹性 < 1 → 必需品（米面油——收入涨了也不会多买很多）
  收入弹性 < 0     → 劣等品（收入涨了反而不买了，如方便面）

交叉弹性 Cross-Price Elasticity of Demand：衡量的是商品A的需求变化对商品B价格变化的敏感度。

交叉弹性 = A需求量变化百分比 / B价格变化百分比

结果解读:
  交叉弹性 > 0 → A和B是替代品（咖啡涨价 → 茶需求增加）
  交叉弹性 < 0 → A和B是互补品（咖啡涨价 → 咖啡伴侣需求减少）
  交叉弹性 = 0 → A和B不相关

这两种弹性在产品管理和市场竞争分析中极其重要：

  • 如果你的产品收入弹性高（奢侈品），经济下行期就要准备增长放缓甚至下滑
  • 如果你的产品和竞争对手的交叉弹性很高（强替代），就不能忽视对手的定价策略

丰收悖论：谷贱伤农

现在我们来应用弹性知识，分析一个著名的经济学谜题——丰收悖论 paradox of good harvest。

如果今年风调雨顺，农民大丰收，你猜农民的总收入是增加还是减少？

直觉告诉大多数人：丰收了，产量多了，收入应该增加吧？但实际情况往往相反——丰收让农民的总收入下降了。

为什么？因为农产品（粮食、蔬菜、水果）的需求通常是缺乏弹性的——人们对粮食的需求量不会因为价格下降而大幅增加。你不可能因为大米降价了一倍就一天吃五顿饭。

所以丰收的结果是：供给曲线大幅右移 → 价格大幅下降（因为需求缺乏弹性，需要大幅降价才能吸收掉增加的量） → 价格降幅远远超过销量增幅 → 总收益下降。

这就是中国老话"谷贱伤农"的经济学解释。

真实案例：美国大萧条时期的农业危机：1929年大萧条爆发后，农产品价格暴跌。美国农民试图通过扩大生产来弥补收入损失（"种更多来赚更多"），结果生产越多价格跌得越狠。到1933年，美国农业收入比1929年下降了约60%。最终美国政府在1933年通过了《农业调整法》，采取"休耕补贴"政策——让农民减少种植面积来维持农产品价格。这不是反市场，而是精准针对农产品需求弹性低这个特征做的政策设计。直到今天，各国农业补贴政策的底层逻辑依然与此一致。

本章小结

今天我们给供需框架装上了"刻度尺"——弹性。回顾核心要点：

  • 弹性：衡量一个变量对另一个变量变化的敏感度，是"百分比之比"，没有单位，可以跨商品比较。
  • 需求价格弹性：用中点法计算。替代品多少、预算占比、时间长短是三大决定因素。
  • 总收益测试：弹性 > 1 时降价增加收入，弹性 < 1 时涨价增加收入。企业定价策略的核心依据。
  • 收入弹性与交叉弹性：区分正常品/奢侈品/劣等品的工具，判断商品之间是替代还是互补关系的指标。
  • 丰收悖论：农产品需求缺乏弹性，增产反而导致总收入下降。"谷贱伤农"有了数学解释。

下一章我们将利用弹性这个工具来分析政府政策——当政府通过税收、补贴、价格管制等方式干预市场时，谁真正承担了代价？弹性高的群体还是弹性低的群体？这些问题的答案将非常清晰地告诉我们：好心不一定能办好事。
