第13章 完全竞争市场（上）：利润最大化

大家好，我是你们的讲师。从这一章开始，我们把前面学的所有工具——需求、供给、成本、效用——组合起来，分析不同类型的市场结构 market structure。

市场结构有很多种，我们从最"理想"的开始：完全竞争市场 perfect competition。之所以说它"理想"，不是因为它在现实中很常见——恰恰相反，完全符合条件的情况很少。但它是经济学最重要的基准模型 benchmark model，就像物理中的"真空中的球形鸡"——虽然不存在，但我们用它来理解现实中的市场到底偏离了理想状态多远。

由于内容较多，完全竞争分两章讲。本章聚焦于企业的利润最大化决策，下一章讲市场均衡与效率。

13.1 完全竞争的四个条件

一个市场要被称为"完全竞争"，必须同时满足四个条件：

条件 含义 现实例子 如果违反则变成
1. 大量买家和卖家 每家企业的市场份额极小，无法影响市场价格 农产品市场：成千上万的农民 寡头/垄断
2. 产品同质化 所有企业生产的产品完全相同，消费者无法区分 小麦、玉米、原油 垄断竞争
3. 自由进入与退出 企业可以无成本地进入或退出市场 网约车司机可以随时开始或停止接单 存在进入壁垒
4. 完全信息 所有买家和卖家都知道价格、质量等全部信息 股票市场（接近） 信息不对称

这四个条件缺一不可。现实中完全没有摩擦的市场几乎是找不到的，但有些市场非常接近——比如农产品期货市场、股票市场（尤其是高频交易场景）、某些标准化原材料的现货市场。

关键洞察：完全竞争不是对"现实世界是什么样的"的描述，它是一个思想实验中的理想状态。有了这个基准，我们才能回答"如果市场是完美竞争的，资源配置会怎样？"然后从理想出发，逐步加入现实世界的摩擦——产品差异化（垄断竞争）、进入壁垒（寡头/垄断）、信息不对称——来分析每个摩擦带来的效率和福利影响。

13.2 价格接受者

在完全竞争市场中，每个企业都是价格接受者 price taker。这意味着什么呢？

假设一个农民种植小麦，他种的小麦和市场上其他农民的小麦完全一样。如果他试图以高于市场价格出售，消费者会直接去其他农民那里买——他的小麦会一颗也卖不出去。反过来，他也没必要以低于市场的价格出售——因为以市场价格他就能卖出全部产量。

所以对于完全竞争企业来说：

  • 价格 P 是固定的——由整个市场的供需决定，单个企业无法影响。
  • 需求曲线是一条水平线——在价格P上，企业可以卖出任意数量；如果提价一点，销售量为零。
  • 边际收益 = 价格——每多卖一单位产品带来的额外收入就是市场的价格。

这个"水平需求曲线"的特征非常重要。在市场层面，需求曲线是向下倾斜的（符合需求定律）；但在单个企业的层面，需求曲线是水平的——因为企业对市场价格的影响力为零。

💡 厘清关系：有一个常见的混淆——"完全竞争企业的需求曲线是水平的，是不是说市场不需要需求定律了？"完全不是。整个市场的需求曲线仍然是向下倾斜的。每个企业面临的水平需求曲线只是说明它太小了，它的产量变化不影响市场总供给。打个比方：你在一场万人演唱会上喊一声，不会影响整体音量——但这不代表音量没有物理规律。

13.3 MR = MC：企业决策的万能法则

不管市场结构如何——完全竞争、垄断、寡头——所有追求利润最大化的企业都遵循一个黄金法则：把产量定在边际收益等于边际成本的位置上，即 MR = MC。

直觉非常简单：

  • 如果多生产一单位的边际收益 MR > 边际成本 MC——这一单位给你带来正利润，继续生产！
  • 如果多生产一单位的边际收益 MR < 边际成本 MC——这一单位给你带来亏损，减少生产！
  • 如果 MR = MC——多生产一单位和不生产这一单位没区别——这就是利润最大化的产量。

在完全竞争市场中，由于P = MR（价格等于边际收益），MR = MC法则简化为：

完全竞争企业的利润最大化条件:
  P = MC

  推导:
  TR  = P × Q（因为企业是价格接受者）
  MR  = ΔTR / ΔQ = P
  利润最大化条件: MR = MC → P = MC

  例子: 假设市场价格P = 50元
  企业的MC函数: MC = 10 + 2Q
  利润最大化: 50 = 10 + 2Q → Q = 20
  企业最优产量 = 20单位

注意：P = MC 只在完全竞争下成立。对于其他市场结构（如垄断），MR ≠ P，所以条件仍然是MR = MC（而不是P = MC）。这个区别在对比不同市场结构时非常重要。

13.4 短期利润最大化

画图分析企业在短期中的利润最大化决策。假设市场价格为P*，企业根据P = MC原则找到最优产量Q*。在Q*处，企业的利润是：

  • 总收益 TR = P* × Q*——价格乘以产量。
  • 总成本 TC = ATC(Q*) × Q*——平均总成本乘以产量。
  • 利润 = TR - TC = (P* - ATC(Q*)) × Q*。

所以利润取决于P*和ATC(Q*)的相对大小：

  • P* > ATC(Q*) → 经济利润为正 → 企业赚钱
  • P* = ATC(Q*) → 经济利润为零 → 正常利润
  • P* < ATC(Q*) → 经济利润为负 → 企业亏损

这里有一个关键的问题：如果企业在短期中亏损，它应该立即停产吗？不一定！

停产条件 shutdown condition：企业应该在P < AVC（价格低于平均可变成本）时停产。为什么？

  • 如果P > AVC但P < ATC：企业虽然在亏损，但收入覆盖了所有可变成本和一部分固定成本。因为固定成本在短期中已经发生了，不生产也得支付，所以继续生产的亏损小于停产（停产要承担全部固定成本）。
  • 如果P < AVC：收入连可变成本都覆盖不了，每多生产一单位就多亏一分钱。此时停产损失更小。

这是一个非常重要的细分知识点，很多初学者会误以为"亏损就应该停产"——但在短期中，只要P > AVC，继续生产比停产更好。

关键洞察：这个逻辑和沉没成本的概念完美衔接。在短期中，固定成本就是沉没成本——已经发生且无法收回。决策时不应该考虑固定成本，只应该看"继续生产的边际收益（P）是否能覆盖边际成本（MC）和平均可变成本（AVC）"。这才是向前看的决策思维。

13.5 企业的短期供给曲线

在前面分析的基础上，我们得到一个重要的结论：完全竞争企业的短期供给曲线就是其MC曲线在AVC最低点以上的部分。

为什么？

  • 当P > min AVC时：企业根据P = MC决定产量——对于每个可能的价格P，MC曲线决定了一个对应的产量Q。这就是"供给"的定义。
  • 当P < min AVC时：企业停产，产量为零。
  • 当P = min AVC时：企业无所谓——生产和停产一样（"关门点 shutdown point"）。

所以供给曲线不是整条MC曲线，而是从AVC最低点向上的那一段。这解释了为什么在短期中，市场价格下降时企业的供给量会减少——沿着MC曲线向左移动。而当价格跌破AVC最低点时，企业直接退出市场，供给量降为零。

完全竞争企业的短期供给决策总结:
  1. 如果 P >= min AVC:
     → 企业在 P = MC 处决定产量
     → 供给曲线 = MC 曲线在 min AVC 以上的部分
  2. 如果 P < min AVC:
     → 企业停产 (Q = 0)
     → 供给量 = 0

  行业的短期供给曲线 = 所有企业供给曲线的水平加总

💡 易错提醒：不要把"停产点"和"盈亏平衡点"搞混。停产点看的是P是否高于AVC最低点，决定的是"生不生产"；盈亏平衡点看的是P是否高于ATC最低点，决定的是"赚不赚钱"。一家企业可以在亏钱的状态下继续生产（当P在AVC和ATC之间时），这不是非理性的——因为停产亏得更多。

13.6 本章小结

好，完全竞争的第一部分就到这里。总结一下核心内容：

  • 完全竞争的四个条件——大量买卖双方、同质化产品、自由进入退出、完全信息。理想基准模型。
  • 价格接受者——企业面临水平的需求曲线，P = MR。
  • MR = MC 法则——所有利润最大化企业都遵循的黄金法则。在完全竞争中简化为 P = MC。
  • 短期利润最大化与停产条件——P > AVC时继续生产（即使亏损），P < AVC时停产。
  • 短期供给曲线——企业短期供给曲线是MC曲线在AVC最低点以上的部分。

下一章，我们将把目光从单个企业转向整个市场——看看完全竞争市场在长期中是如何通过企业的进入和退出达到均衡的。我们会得到一个惊人的结论：在完全竞争的长期均衡中，经济利润为零。但这其实是好事，而不是坏事。敬请期待。
