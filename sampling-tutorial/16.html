第16章 非抽样误差

大家好，我是你们的讲师。第15章我们提到了总误差框架中除了抽样误差之外的另一个重要部分——非抽样误差（Non-sampling Error）。如果说抽样误差是一个"已知的敌人"（你能计算、能预测、能控制），那非抽样误差就是一个"潜伏的刺客"——它可能比你想象的更严重，而且经常在你最意想不到的地方给你致命一击。

这里有一个触目惊心的趋势。美国著名的皮尤研究中心（Pew Research Center）发布了一份长期追踪数据：1997年，美国的电话调查应答率大约在36%左右——也就是说，每联系三个家庭，大概有一个会完成调查。到2010年代，这个数字降到了个位数，大约6%。想象一下：你拨出100通电话，只有6个人愿意参与。剩下的94人中，拒绝参与的理由千差万别——有些人太忙，有些人厌恶骚扰电话，还有人恰好对调查主题持强烈不同意见才愿意接听。这6%的人能代表那94%吗？大概率不能。这就是非抽样误差中最棘手的无回答偏差（Nonresponse Bias）。

关键认知：非抽样误差有一个非常危险的特质——它不会随着样本量的增大而自动消失。抽样误差会，但非抽样误差不会。这意味着你在一个糟糕的调查设计上堆积再多样本，也只是在放大系统性的偏差。你花100万收集了10万份样本，如果问卷本身有暗示性措辞，那10万份数据只是在精确地测量一个错误的答案。

抽样误差 vs 非抽样误差

在深入具体类型之前，先看一张对比表格，帮你建立一个清晰的全局认知。

维度 抽样误差（Sampling Error） 非抽样误差（Non-sampling Error）
产生原因 只调查了部分而非全部总体 抽样框、测量、无回答、数据处理等
随样本量增大 减小（与√n成反比） 通常不会自动减小
可计算性 可用统计公式精确计算 通常难以直接量化
偏差方向 随机，可正可负，期望为零 通常有系统方向，期望不为零
主要控制手段 增大样本量、优化设计 改进流程、培训、质量监控
在大型调查中的相对重要性 通常较小（样本量大时） 通常更大——是误差的主要来源

抽样框误差（Frame Error）

抽样框（Sampling Frame）是你用来抽取样本的那份名单。如果这份名单和真实总体之间存在差异，就产生了抽样框误差。这是1936年《文学文摘》翻车的根本原因——他们用电话用户和汽车注册名单作为抽样框，但这部分人群在当时并不能代表全体选民。

抽样框误差有三种常见表现形式：

涵盖不全（Undercoverage）

抽样框遗漏了部分目标总体。例如用固定电话名单调查全国居民——这会系统性地遗漏只有手机的家庭（年轻人、低收入群体）。《文学文摘》就是一个经典的涵盖不全案例：电话和汽车只覆盖了富裕阶层，穷人和农民基本没有入选机会。

过时（Outdated Frame）

抽样框的信息没有及时更新。比如用三年前的企业目录来调查新公司——很多当年的公司已经倒闭了，而新成立的公司根本没有出现在名单上。在企业调查中，过时的抽样框会导致"幸存者偏差"（Survivorship Bias），让你高估行业的整体健康状况。

重叠（Overlapping / Duplication）

同一个人在抽样框中出现了多次。例如用多个来源拼凑的地址列表，同一个家庭可能在两个不同的渠道都被登记了，被抽中的概率实际上翻倍了。如果不做去重处理，这种隐含的不等概率会影响估计的无偏性。

无回答误差（Nonresponse Error）

无回答是当代调查面临的最大挑战。它的核心机制是：参与调查的人和拒绝参与调查的人，在关键特征上很可能不同。如果差异与调查主题相关，就会产生偏差。

单位无回答（Unit Nonresponse）

整个个体（调查单位）完全没有参与。原因可能是联系不上、拒绝参与、语言障碍、身体条件不允许等。单位无回答率在逐十年升高——这是全球性的趋势。

项目无回答（Item Nonresponse）

个体参与了调查，但跳过了某些问题。最常见的情况是收入问题、年龄问题、敏感行为问题被跳过。项目无回答的处理通常涉及插补（Imputation）方法，我们会在第17章详细讨论。

# 模拟无回答偏差
import numpy as np

# 真实总体：收入均值50000
population_income = np.random.normal(50000, 15000, 10000)

# 假设低收入者更不愿意回答
# 回答概率随收入增加而增加
response_prob = 1 / (1 + np.exp(-(population_income - 45000) / 5000))
responded = np.random.random(10000) < response_prob

# 只看回答者的数据
respondent_mean = population_income[responded].mean()
true_mean = population_income.mean()

print(f"真实总体均值: {true_mean:.0f}")
print(f"回答者均值:   {respondent_mean:.0f}")
print(f"偏差:         {respondent_mean - true_mean:.0f}")
# 输出显示：回答者均值系统性地高于真实总体均值

计量误差（Measurement Error）

计量误差是回答者给出的答案与其真实值之间的差异。它可能源于以下任何一个环节：

  • 问题措辞（Question Wording）：有引导性的问题会影响回答。"您是否同意政府采取的环保措施？"远比"您对政府环保措施有什么看法？"更容易得到肯定回答。微小的措辞差异可能带来5-10个百分点的差异。
  • 调查员效应（Interviewer Effect）：面对访问员时，受访者会倾向于给出"社会期望"的回答——比如隐瞒吸烟、饮酒等行为，夸大阅读量、运动频率等积极行为。
  • 记忆偏差（Recall Bias）：让人们准确回忆三个月前吃了多少次火锅，几乎是不可能的。回忆问题的时间窗口越长，误差越大。
  • 社会期望偏差（Social Desirability Bias）：这是最难处理的一类。选民投票率调查中，实际投票率约60%，但调查回答"我投了票"的比例通常在80%以上——因为不投票是一种社会不期望行为。
一个我自己踩过的坑：在做某次产品满意度调查时，问卷中使用了"您对我们的产品有多满意？"的5分制，结果98%的分数集中在4-5分。这个数据对产品改进几乎毫无价值。后来我们改用净推荐值（NPS）和开放式问题，才真正获得了可操作的反馈。原因是：直接的满意度问题容易引发"礼貌性好评"，而具体的行为导向问题能获得更真实的信息。

离群值（Outliers）

离群值是极端值，可能是真实存在的极端个体（比如一位年收入上亿的CEO出现在一般家庭调查中），也可能是数据录入错误（比如年龄填成了200岁）。离群值对均值估计的影响极大——一个极端值就可能把均值拉偏十几个百分点。

识别和处理离群值的一般流程：

  • 识别：箱线图（Boxplot）、Z分数（Z-score）、IQR方法
  • 核查：返回原始问卷或录音确认是否为记录错误
  • 处理：如果是录入错误则修正；如果是真实极端值，考虑截尾处理（Winsorization）或使用稳健估计量（如中位数代替均值）

非抽样误差的控制策略

既然非抽样误差不随样本量增大而减小，如何应对？以下是几条经过实践检验的核心策略：

  • 预防优于纠正：在问卷设计阶段做认知访谈（Cognitive Interviewing）和预测试（Pilot Test），提前发现措辞问题。请经验丰富的调查员培训师来培训访问员队伍。
  • 多源数据验证：将调查数据与其他来源（行政记录、普查数据）进行比对，检查是否存在系统性偏差。
  • 追踪无回答者：对初始无回答者进行二次接触（使用不同渠道，如电话换邮件），分析无回答者和回答者在关键特征上的差异。
  • 质量控制与复访：随机抽取5%-10%的已完成问卷进行电话回访，验证关键信息。对访问员的通话录音进行随机抽查。
  • 事后校正：利用加权调整、插补、多重插补等手段在分析阶段尽量减小偏差——但这不能替代预防。
一个昂贵但真实的教训：2010年某大型政府调查投入了上亿元预算。在数据分析阶段才发现，由于访问员培训不到位，某些访问员为赶进度在家自行填答了超过30%的问卷（curb-stoning / 假调查）。最终不得不丢弃整批数据，项目重做。事后复盘发现，如果在执行过程中加入10%的录音抽查和GPS定位核对，这个悲剧完全可以避免。在调查中，质量控制不是成本，而是投资。

本章小结

非抽样误差是调查误差中更容易被忽视但往往影响更大的部分。我们从抽样框误差、无回答误差、计量误差和离群值四个维度展开了讨论，核心结论是：非抽样误差不会随样本量增大而自动消失，它需要从设计阶段就开始系统性地预防、在执行过程中持续监控，并在分析阶段通过统计方法进行补救。记住：不管你的样本量多大、抽样设计多精妙，如果问卷本身有引导性、如果访问员没有经过充分培训、如果大量受访者拒绝参与——这些非抽样误差足以摧毁整个调查的可靠性。

那么问题就来了：当我们发现了数据中存在偏差，有没有一套系统的方法来诊断偏差的严重程度、并尽可能地进行校正？这正是第17章的目标——偏差诊断与校正。
