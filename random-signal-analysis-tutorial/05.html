随机变量函数的分布

大家好，我是你们的讲师。在工程中，我们很少直接观测到我们关心的随机变量——更多时候我们观测到的是它的一个函数。雷达接收机收到的是回波信号的幅度（它是目标距离、雷达截面积等随机变量的函数）；通信接收机对信号做检波后得到的是包络（它是 I/Q 两路高斯变量的函数）。这就引出一个核心问题：已知 X 的分布，Y = g(X) 的分布是什么？

换句话说：随机变量经过一个确定的函数变换后，新变量的分布如何推导？

问题：已知X的PDF，Y=g(X)的PDF是什么？

最通用的方法是"两步走"策略：先用 CDF 法——写出 F_Y(y) = P(Y ≤ y) = P(g(X) ≤ y)，再转化为关于 X 的事件，最后对 PDF 积分求得 CDF，求导得 PDF。这个思路在理论推导中非常通用，但计算上可能繁琐。

当 g 是严格单调的可微函数时，有一个简洁的直接公式。

单调函数变换公式：fY(y) = fX(g⁻¹(y))·|d(g⁻¹)/dy|

如果 Y = g(X)，且 g 是严格单调的可微函数，则 Y 的 PDF 为：

f_Y(y) = f_X( g^{-1}(y) ) · | d(g^{-1}(y)) / dy |

这个公式的直观理解：通过变换 g，x 轴上的一个小区间 dx 被"拉伸"或"压缩"到 y 轴上的区间 dy。概率质量守恒要求 f_X(x)dx = f_Y(y)dy，所以 f_Y(y) = f_X(x) · |dx/dy|。绝对值保证 PDF 始终非负（当 g 单调递减时 dx/dy 为负，取绝对值确保 PDF 为正）。

记忆口诀："原 PDF 乘上雅可比行列式的绝对值"——这是更一般的多维变换公式在一维的特例。雅可比项 |dx/dy| 的作用就是"调整尺度"，因为变换后 y 轴上的"单位长度"对应 x 轴上的不同长度，概率密度需要相应缩放。

非单调函数：分段处理

如果 g 不是单调的（例如 Y = X²），则同一个 y 可能对应多个 x 值。需要用"分段法"：找出所有满足 g(x)=y 的 x 值，对每一段分别应用变换公式再求和：

f_Y(y) = Σ_i f_X(x_i) / |g'(x_i)|,  其中 x_i = g_i^{-1}(y) 是第 i 个分支的反函数

以 Y = X² 为例，当 y > 0 时，有 x₁ = √y 和 x₂ = -√y 两个解，且 g'(x) = 2x，所以：

f_Y(y) = f_X(√y)/(2√y) + f_X(-√y)/(2√y)

这个公式在信号处理中非常实用——例如求高斯随机变量经过平方律检波器后的分布。

瑞利分布与莱斯分布

在通信和雷达系统中，包络是最常测量的物理量之一。无论你是在接收无线信号还是分析雷达回波，你第一步通常都是提取信号的包络（幅度）。而包络的分布，恰好可以从高斯分布通过函数变换推导出来。

从高斯到瑞利：X,Y~N(0,σ²)独立，R=√(X²+Y²)服从瑞利分布

设 X 和 Y 是独立同分布的高斯随机变量，均值为 0，方差为 σ²。定义 R = √(X² + Y²) 为"幅度"。通过二维随机变量变换（从直角坐标到极坐标），可以推导出 R 的 PDF：

f_R(r) = (r/σ²) · exp(-r²/(2σ²))，r ≥ 0

这就是瑞利分布 (Rayleigh distribution)。它的形状是一个从 0 开始上升到峰值再缓慢下降的曲线。峰值出现在 r = σ 处，均值为 σ√(π/2) ≈ 1.253σ。

在通信中，瑞利分布描述的是：当没有直视路径（只有多径散射）时，接收信号包络的分布。这也是为什么"瑞利衰落信道"是无线通信中最基本的衰落模型。

从高斯到莱斯：加上非零均值（正弦信号+噪声的包络）

现在考虑更一般的情况：X 和 Y 仍然是独立同方差的高斯变量，但均值不为零——X~N(A, σ²)，Y~N(0, σ²)。这对应于一个幅度为 A 的正弦信号加上高斯噪声的正交分量。此时 R = √(X² + Y²) 的PDF为：

f_R(r) = (r/σ²) · exp(-(r²+A²)/(2σ²)) · I₀(rA/σ²)，r ≥ 0

其中 I₀(·) 是第一类零阶修正贝塞尔函数。这就是莱斯分布 (Rician distribution)。

特征瑞利分布莱斯分布
物理场景纯噪声/纯多径散射信号 + 噪声（有直视分量）
参数σ（尺度参数）σ（尺度参数），A（直视幅度）
均值σ√(π/2)> σ√(π/2)，随 A 增大
莱斯因子 KK=0（退化为瑞利）K = A²/(2σ²)（信号功率/噪声功率）

通信工程意义：瑞利—纯噪声包络、莱斯—信号+噪声包络

这两个分布的工程意义怎么强调都不过分：

  • 瑞利：当你只看到噪声（或只有散射径）时，包络的分布是瑞利。雷达在没有目标的场景中，接收机输出的包络服从瑞利分布——这就是设置检测门限的依据。
  • 莱斯：当噪声中混有确定信号时，包络的分布是莱斯。雷达从杂波中检测目标——有目标时包络服从莱斯分布，无目标时服从瑞利分布。检测问题本质上就是区分这两个分布。
一个关键参数：莱斯因子 K = A²/(2σ²) 是直视路径功率与散射路径功率之比。K 很大时（强直视路径），莱斯分布近似为高斯分布（均值 ≈ A，方差 ≈ σ²）；K = 0 时退化为瑞利分布。在卫星通信中，K 可以大到 20 dB 以上；在密集城市环境中的非直视链路，K ≈ 0（即瑞利衰落）。

特征函数的妙用

独立随机变量之和的特征函数 = 特征函数的乘积

上一章我们介绍了特征函数 Φ(ω) = E[e^{jωX}]。它最重要的应用在于处理独立随机变量之和。设 Z = X + Y，其中 X 与 Y 独立，则：

Φ_Z(ω) = E[e^{jω(X+Y)}] = E[e^{jωX} · e^{jωY}]
         = E[e^{jωX}] · E[e^{jωY}]   (由独立性)
         = Φ_X(ω) · Φ_Y(ω)

这个乘法性质极其强大：独立和的特征函数等于各特征函数的乘积。对 n 个独立变量之和，就是 n 个特征函数连乘。这比在时域做 n 重卷积要简单得多——因为 特征函数把卷积变成了乘积，这正是傅里叶变换的魔法。

用特征函数证明：独立高斯之和仍为高斯

来看一个经典证明。设 X ~ N(μ₁, σ₁²)，Y ~ N(μ₂, σ₂²)，且 X 与 Y 独立。高斯分布的特征函数为：

Φ_X(ω) = exp(jμ₁ω - σ₁²ω²/2)
Φ_Y(ω) = exp(jμ₂ω - σ₂²ω²/2)

则 Z = X + Y 的特征函数：
Φ_Z(ω) = Φ_X(ω) · Φ_Y(ω)
        = exp(j(μ₁+μ₂)ω - (σ₁²+σ₂²)ω²/2)

看！这就是均值为 μ₁+μ₂、方差为 σ₁²+σ₂² 的高斯分布的特征函数。因此 Z 也是高斯分布——独立高斯之和仍为高斯。这个性质使得高斯过程在经过线性系统后仍然是高斯的，是信号处理中不可替代的便利性。

矩生成：对特征函数求导得矩

特征函数的另一个实用功能是"矩生成"。将特征函数在 ω=0 处做泰勒展开：

Φ_X(ω) = 1 + jω·E[X] + (jω)²/2! · E[X²] + ... + (jω)ⁿ/n! · E[Xⁿ] + ...

因此，n 阶原点矩可以通过对特征函数在 ω=0 处求 n 阶导数得到：

E[Xⁿ] = (-j)ⁿ · dⁿΦ_X/dωⁿ |_{ω=0}

这在计算高阶矩时非常方便——你只需要对特征函数求导，而无需直接计算涉及复杂被积函数的期望积分。

大数定律与中心极限定理

最后，我们来谈概率论中最深刻的两个定理。它们解释了为什么"统计"是可靠的，又为什么"高斯"无处不在。

弱大数定律：样本均值依概率收敛到期望

设 X₁, X₂, ..., Xₙ 是独立同分布的随机变量，均值为 μ，方差有限。定义样本均值 X̄ₙ = (1/n)Σᵢ Xᵢ。弱大数定律 (Weak Law of Large Numbers, WLLN) 指出：

对任意 ε > 0，lim_{n→∞} P(|X̄ₙ - μ| > ε) = 0

用大白话说：当样本数量足够大时，样本均值 依概率收敛到真实均值。你抛硬币 1000 次得到的正面比例，会非常接近真实概率 0.5——而且随着抛掷次数增加，偏差超过任意小量 ε 的概率趋于零。这就是大数定律给你的承诺。

大数定律是所有统计估计方法的基础。在随机信号分析中，它保障了我们可以用时间平均来估计统计平均——这正是第 11 章"各态历经性"的理论前提。

中心极限定理：无论原始分布如何，样本均值近似高斯

中心极限定理 (Central Limit Theorem, CLT) 是概率论中"皇冠上的明珠"。它说：

设 X₁, X₂, ..., Xₙ 是独立同分布的随机变量，均值为 μ，方差为 σ²。则当 n 很大时，

标准化和：Zₙ = (X̄ₙ - μ) / (σ/√n)  →  近似服从标准正态分布 N(0,1)

这个定理极其震撼：无论原始分布是均匀、泊松、指数还是任何"奇怪"的形状，经过大量独立同分布相加后，它们的和的分布趋向于高斯。这是人类为数不多的"从无序中涌现有序"的数学定理。

用特征函数来证明的思路很简洁：独立同分布和的特征函数是 [Φ(ω)]ⁿ。对 n 取极限，取对数后做泰勒展开，保留到二阶项，结果正是高斯分布的特征函数。这个证明虽然只有几行，但它背后的深刻含义——吸引域 (domain of attraction) 的概念——却酝酿了近两个世纪的研究。

蒙特卡罗方法的理论基础

大数定律和 CLT 共同构成了蒙特卡罗方法 (Monte Carlo method) 的理论支柱：

  • 大数定律保证：用随机采样去估计期望值是可行的（收敛的）
  • CLT 告诉我们收敛速率：估计误差的标准差 ≈ σ/√n，与 1/√n 成正比

这意味着：要提高估计精度一个数量级（误差除以 10），你需要将样本量增加 100 倍。这个"平方根法则"是所有仿真工程师必须牢记在心的——你在 MATLAB 中跑 10,000 次蒙特卡罗仿真，误差标准差大约是 1%；要降到 0.1%，你需要跑 1,000,000 次。

踩坑故事：用CLT做假设检验时忽略了样本量的要求

一个真实发生的事件。某团队开发了一个 A/B 测试平台，用于评估产品改动的效果。他们的统计引擎使用了基于 CLT 的 z 检验。在一次测试中，对照组只有 15 个用户，实验组只有 12 个用户，他们用 z 检验算出 p 值 = 0.03，认定改动有统计显著性并发布了产品。

但实际上，CTR（点击率）的原始分布严重偏斜——90% 的用户根本不点击，10% 的用户贡献了所有点击。样本量 n=12 时，样本均值的分布远远不是高斯（仍然是严重偏斜的离散分布）。他们用了 z 检验（假设高斯近似），但实际上应该用 Fisher 精确检验或 bootstrap 方法。

教训：CLT 的收敛速率取决于原始分布的形状。对称、薄尾的分布只需要 n≈30 就能有不错的高斯近似；但严重偏斜或厚尾的分布可能需要 n 达到几百甚至上千。一个安全经验法则：在应用 CLT 之前，检查原始分布的偏度和峰度；如果偏度绝对值大于 1 或峰度大于 5，就需要更多样本。不要默认"n>30 就安全"——这是一个危险的神话。

本章小结与衔接

本章我们走过了随机变量函数变换的三种方法（CDF法、单调公式、分段处理），然后从高斯分布推导出了通信工程中举足轻重的瑞利分布和莱斯分布。接着我们看到了特征函数的神奇——把卷积变为乘积——并用它证明了"独立高斯之和仍是高斯"这一核心性质。最后，大数定律和中心极限定理为统计估计提供了理论基础，也警示了我们盲目使用 CLT 的风险。

从下一章开始，我们将正式进入随机信号的世界。随机变量是静态的（一个时间点），而随机信号是动态的（随时间演化）。我们将把本章学到的所有工具——分布、数字特征、变换、极限定理——应用到随时间变化的随机过程中去。
