第5章：隔板法与整数解

大家好，我是你们的讲师。今天我们来学习组合数学中最"形象化"的一个技巧——隔板法。隔板法解决的问题看起来很简单：把 n 个相同的物品分给 k 个人，有多少种分法？但就是这么简单的问题，在竞赛中可以演变出上百种变体。

我见过很多学生在这类题目上丢分——不是因为他们不会"插隔板"，而是他们不清楚在不同的约束条件（正数、非负数、有上界、有下界）下，隔板法的结果会有什么不同。更糟糕的是，他们还常常把"球不同"和"盒子不同"的组合搞混。今天我们就来彻底解决这些问题。

5.1 隔板法的三种经典模型

隔板法解决的核心问题可以用一个标准形式来描述：

求方程 x₁ + x₂ + ... + x_k = n 的整数解个数

其中 x_i 的取值范围不同，对应不同的情况。

5.1.1 模型一：正整数解（x_i ≥ 1）

有 n 个完全相同的物品，要分给 k 个人，每人至少分到一个。把 n 个物品排成一排，它们之间有 n-1 个空隙。在 n-1 个空隙中选 k-1 个位置插入隔板，就把 n 个物品分成了 k 份。

物品排列： ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ (n个)
空隙位置：  ↑ ↑ ↑ ↑ ↑ ↑ ↑ ↑ ↑  (n-1个)
插入 k-1 块隔板后： ○|○○○|○|○○|○○○ (分给k个人)

所以正整数解的数量为 C(n-1, k-1)。

用方程的语言：方程 x₁ + x₂ + ... + x_k = n，x_i ≥ 1 的整数解个数为 C(n-1, k-1)。

5.1.2 模型二：非负整数解（x_i ≥ 0）

当 x_i 可以为零时，情况稍有不同。我们不能在"零个物品"之间插隔板——因为零个物品不存在空隙。

解决方法非常巧妙：做一个变量替换。令 y_i = x_i + 1，则 y_i ≥ 1，且：

y₁ + y₂ + ... + y_k = (x₁+1) + (x₂+1) + ... + (x_k+1) = n + k

这样就把"非负整数解"问题转化成了"正整数解"问题！

所以 x_i ≥ 0 时方程 x₁+...+x_k=n 的解数为 C(n+k-1, k-1)。

🔥 两种模型的对比：

条件解数公式记忆方法
正整数解 x_i ≥ 1C(n-1, k-1)n 个物品，k-1 块隔板插在 n-1 个空隙
非负整数解 x_i ≥ 0C(n+k-1, k-1)先把每个变量"借"一个，变成 n+k 个物品

5.1.3 模型三：下限约束（x_i ≥ a_i）

更一般地，如果每个 x_i 都有一个不同的下界 x_i ≥ a_i（其中 a_i 是整数），处理方法同样是变量替换：

令 y_i = x_i - a_i + 1，则 y_i ≥ 1
∑ y_i = ∑ (x_i - a_i + 1) = n - ∑ a_i + k

所以解数为 C(n - ∑a_i + k - 1, k - 1) / 或等价为：

更直接的做法：先给每个 x_i 分配 a_i 个（确保下界满足），然后对剩下的 n - ∑a_i 个物品用"非负整数解"模型。

解数 = C((n - ∑a_i) + k - 1, k - 1) = C(n - ∑a_i + k - 1, k - 1)

5.2 从方程到分配——不同模型间的转化

隔板法的威力在于，它可以把各种"分配问题"统一成"方程整数解问题"。下面这张表总结了常见的对应关系：

分配问题描述 对应的方程 解数
n 个相同的球放入 k 个不同的盒子，盒子不空 x₁+...+x_k=n, x_i≥1 C(n-1, k-1)
n 个相同的球放入 k 个不同的盒子，允许空盒 x₁+...+x_k=n, x_i≥0 C(n+k-1, k-1)
n 个不同的球放入 k 个相同的盒子，盒子不空 （Stirling数，非隔板法） S(n,k)
n 个不同的球放入 k 个不同的盒子，允许空盒 （每球有 k 种选择） kⁿ

注意第三行——当球不同且盒子不同时，问题不再是隔板法能处理的了。这是竞赛中一个常见的混淆点。

⚠️ 踩坑实录：我见过太多学生在考试中把"球不同"的问题当成了"球相同"来做。一个经典问题："10 本不同的书分给 3 个人，每人至少一本，有多少种分法？"很多学生直接套用隔板法写了 C(9,2) = 36，但正确答案是 3¹⁰ - 3×2¹⁰ + 3（为什么？用容斥原理！）。36 和这个答案差了不止两个数量级。

为什么隔板法不行？因为隔板法的前提是"物品相同"——书是不同的，隔板法把不同的书当成了相同的物品，自然会严重少算。记住：看到"不同的"三个字，立刻放弃隔板法。

5.3 可重组合数的组合解释

C(n+k-1, k-1) 还有另一层重要的身份——它是可重组合数。

什么叫做"可重组合"？从 n 种物品中选 k 个（每种物品可以重复选取，不计顺序），不同的选法数量就是 C(n+k-1, k)。注意这里是"从 n 种选 k 个"，和"非负整数解"中的 n 和 k 角色互换。

举例：从 3 种水果（苹果、香蕉、橙子）中选 5 个（允许重复），有多少种选法？

C(3+5-1, 5) = C(7,5) = 21 种

这 21 种方案对应的方程是：设 x₁=苹果个数、x₂=香蕉个数、x₃=橙子个数，则 x₁+x₂+x₃=5，x_i≥0——这不就是隔板法模型二嘛！

所以可重组合数和非负整数解数是同一个问题的两面。

💡 直观理解：为什么从 n 种物品中可重复地选 k 个是 C(n+k-1, k) 而不是 nᵏ？因为 nᵏ 是有顺序的（如 AAAB 和 BAAA 算两种不同的序列），而可重组合不计顺序（AAAB 和 BAAA 是同一个组合）。除以顺序重复后，就得到了 C(n+k-1, k)。

5.4 上界约束与容斥原理的结合

当变量既有下界又有上界时——也就是 "a_i ≤ x_i ≤ b_i"——单纯的隔板法就不够了，需要结合容斥原理。

5.4.1 例1：带上下界的整数解

求方程 x₁ + x₂ + x₃ + x₄ = 15
满足 1 ≤ x₁ ≤ 6, 0 ≤ x₂ ≤ 5, 2 ≤ x₃ ≤ 7, 0 ≤ x₄ ≤ 8 的整数解个数。

解法：

  1. 先处理下界：令 y₁ = x₁-1, y₂ = x₂, y₃ = x₃-2, y₄ = x₄，则 y_i ≥ 0。
  2. 方程变为 y₁ + y₂ + y₃ + y₄ = 15 - 1 - 2 = 12。
  3. 现在代入上界约束：y₁ ≤ 5, y₂ ≤ 5, y₃ ≤ 5, y₄ ≤ 8。
  4. 用容斥原理处理上界。设 A₁ = {y₁ ≥ 6}, A₂ = {y₂ ≥ 6}, A₃ = {y₃ ≥ 6}, A₄ = {y₄ ≥ 9}。
  5. 然后用容斥计算"没有任何上界被违反"的解数。

具体的计算留给读者作为练习——关键在于理解先下界后上界的流程。

5.4.2 例2：不等式的整数解

有时候问题不是方程，而是不等式：

求 x₁ + x₂ + x₃ ≤ 10 的非负整数解个数。

技巧：引入松弛变量 s ≥ 0，将不等式转化为等式：

x₁ + x₂ + x₃ + s = 10，所有变量 ≥ 0

然后用非负整数解公式：C(10+4-1, 4-1) = C(13,3) = 286。

这个方法在组合优化和线性规划中极为常见。

5.5 真正的踩坑故事——竞赛中的血泪教训

有一年我带队参加全国高中数学联赛，队里一个平时成绩很好的同学遇到这样一道题：

"把 20 个相同的乒乓球分给甲、乙、丙、丁四个班，要求甲班至少 3 个，乙班至少 2 个，丙班至少 1 个，丁班至少 1 个。且每个班分到的球数不超过 8 个。有多少种分法？"

他的思路：先处理下界，分配 3+2+1+1=7 个，还剩 13 个。然后考虑上界每个班 ≤ 8。用隔板法，13 个球分给 4 个班，允许空盒——C(13+4-1, 4-1) = C(16,3) = 560。然后他再用容斥减去有人超过 8 的情况。

计算过程：假设甲班超过 8（即 ≥ 9），令 y₁ = x₁ - 8，则问题变为至少分配 3+2+1+1=7，然后 y₁≥1（因为要在 8 的基础上再加 1，才能说超过 8），剩余 12 个分给 4 个班——C(12+4-1, 4-1) = C(15,3) = 455。同理乙、丙、丁各 455。再减两两交……

算到一半他发现自己漏了一个关键点：甲班上界是 ≤ 8，但下界已经是 3。所以甲班不能超过 8，意味着 x₁ ≤ 8。 在处理上界违反时，他计算的是"x₁ ≥ 9"的情况，但他忘了——原始约束中的下界是 3，而他在第一步已经分配了 3 个给甲班。现在所说的"超过 8"，是指分配后的总数超过 8，而不是"再分配的部分超过 8"。

他的混乱来源于没有分清楚"原始变量"和"变换后的变量"。

⚠️ 踩坑解析：这个问题的正确做法是：

  • 设分配给甲班的球数为 x₁（原始变量），已知 3 ≤ x₁ ≤ 8。
  • 先处理下界：令 y₁ = x₁-3，则 0 ≤ y₁ ≤ 5。同理 y₂ = x₂-2 (0≤y₂≤6)，y₃ = x₃-1 (0≤y₃≤7)，y₄ = x₄-1 (0≤y₄≤7)。
  • 方程变为 y₁+y₂+y₃+y₄ = 20-3-2-1-1 = 13。
  • 现在用"非负整数解"公式得 C(13+4-1, 4-1)=C(16,3)=560。
  • 然后用容斥减去违反上界的情况——但注意这里的"违反上界"是 y₁ ≥ 6（不是 y₁ ≥ 9！）。

区别就在这一步！因为 y₁ 是变换后的变量，它的上界是 5，所以违反上界就是 y₁ ≥ 6。他错把 y₁ ≥ 9 当成了违反条件——这是出于对变量替换的生搬硬套。教训：做了变量替换后，所有的约束条件也要相应地替换，不能抱着原始变量不放。

5.6 本章小结与过渡

今天我们学习了隔板法的三种经典模型——正整数解、非负整数解和带下界约束的整数解，理解了它们之间的转化关系。我们还讨论了可重组合数与隔板法的内在联系，以及隔板法和容斥原理结合处理带上下界的整数解问题。

隔板法看起来简单——不就是插几块板子吗？但真正在竞赛中使用时，关键的难点在于：第一，判断物品是否相同（相同才可用隔板法）；第二，选择正确的模型（正数/非负数/带约束）；第三，变量替换后不要忘记更新所有约束。此外，当变量有上界时，隔板法需要和容斥原理配合使用——它们俩是组合计数中最经典的拍档。

下一章我们将学习递推与归纳法——不同于前面几章"直接计算"，递推法的思路是：把一个大问题降解成若干个同类型的小问题，通过找到问题之间的数量关系来求解。这是组合数学中另一种完全不同的思想范式，也是解决复杂计数问题的最重要方法论之一。我们第六章见。
