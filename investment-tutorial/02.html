第2章 股票

大家好，欢迎来到第2章！如果你问一个普通人"投资是什么"，十有八九他会回答"买股票"。股票确实是普通人最熟悉的投资品种——但你有没有想过，当你说"买股票"的时候，你实际上在买什么？

股票的本质是公司所有权凭证。你买入一股茅台，就成了茅台的股东之一，理论上对公司的重大决策有投票权，每年还能分到一部分利润（如果公司决定分红的话）。当然，在现实中，如果你只买了100股茅台（大约十几万元），你的投票权微乎其微，但你的股东权益是真实存在的——这是理解一切股票知识的基础。

一个真实的故事：腾讯的"奶茶"员工福利

2018年，腾讯有一位叫Judy的早期员工在Facebook上发了一张照片——她卖掉了自己持有的1000股腾讯股票，用这笔钱开了一家奶茶店。她是腾讯第50号员工，在2004年腾讯上市时就以3.7港元的IPO价格获得了员工配股。到2018年，腾讯股价最高涨到475港元。她的1000股从3,700港元变成了475,000港元——仅仅14年，翻了128倍。

Judy的奶茶店生意不错，但如果我们看这笔"机会成本"：那1000股如果持有到2021年腾讯股价最高点（约700港元），价值会是700,000港元。如果她在2024年腾讯股价低谷（约250港元）再买回来，她可以用卖奶茶店的钱买回几千股。这个故事告诉我们两个道理：第一，长期持有优质公司的股权是最暴利的投资方式之一；第二，在你还不需要用钱的时候，不要轻易卖出核心资产。

核心启示：股票投资的本质是分享优秀公司的成长。如果你看好腾讯的未来，持有它的股票比开一家奶茶店更可能帮你实现财富增长——因为腾讯的利润增长不受你每天营业时间的限制。

2.1 普通股与优先股

股票并非只有一种。按股东权利的不同，股票分为普通股（Common Stock）和优先股（Preferred Stock）。

普通股

普通股是最常见的股票形式。普通股股东享有以下权利：

  • 投票权：每持有一股普通股，通常享有一票投票权。重大事项如选举董事会成员、合并收购、修改公司章程等需要股东投票决定。A股实行"一股一票"原则，但实践中中小股东的投票参与率很低。
  • 股息权：当公司宣布派发股息时，普通股股东有权按持股比例获得股息。但股息不是强制的——公司可以不派息，将利润用于再投资。
  • 剩余索取权：如果公司破产清算，普通股股东是最后一个拿到钱的。在所有债权人、优先股股东都得到偿付后，剩下的才分给普通股股东。这也意味着普通股的风险最高。

优先股

优先股是介于普通股和债券之间的"杂交工具"：

  • 固定股息：优先股通常有固定的股息率，且在公司派息时优先于普通股。如果公司某年利润不足未能支付优先股股息，累计优先股（Cumulative Preferred Stock）会累积欠付的股息。
  • 优先清偿：公司破产时，优先股股东的清偿顺序在债权人之后、普通股股东之前。
  • 无（或有限）投票权：优先股通常没有投票权，除非公司长期拖欠股息。

A股市场上优先股的数量很少，但在美国市场，优先股是机构投资者常用的工具。2022年巴菲特就曾通过优先股投资了美国银行（Bank of America），获得了每年6%的固定股息收益。

特征普通股优先股
投票权有（一股一票）通常无
股息不固定，由董事会决定固定股息率，优先支付
风险高中
收益潜力高（股价上涨空间大）中（类似固收产品）
清算顺序最后优先于普通股

2.2 股息政策

公司赚了钱之后，要不要分给股东？分多少？这是公司的股息政策（Dividend Policy）问题。

现金股息

公司直接向股东发放现金，按每股多少元计算。例如，工商银行每年每股派发现金股息约0.3元，如果你持有1万股，就能拿到3,000元。A股的现金分红通常一年一次，但一些优质公司（如双汇发展、中国神华）会一年分红两次，甚至特别分红。

小贴士：A股持有满1年以上的股息收入免征个人所得税，持有1个月至1年按10%征税，持有不满1个月按20%征税。这个税收政策鼓励长期持股——做时间的朋友！

股票股息

公司用股票代替现金发股息，俗称"送股"或"转增股本"。比如"10送10"，就是你原来持股100股，送股后变成200股。但注意：总市值不变——股价会相应下调（除权）。所以送股不是"白送你钱"，而是把一块蛋糕切成了两块，总量没变。

股息再投资计划

很多公司提供股息再投资计划（DRIP, Dividend Reinvestment Plan），允许股东将获得的现金股息自动用于购买更多股票，而且通常免收佣金。长期来看，复利效应惊人——标普500从1926年至今的总回报中，约40%来自于股息再投资的复利增长。

2.3 股票拆分与回购

股票拆分

当股价太高时，公司可以把一股拆成多股。比如1拆10：原来1股1000元变成10股各100元。总市值不变，但每股价格降低了，流动性更好，更多小投资者可以参与。

最著名的例子：1999年巴菲特通过发行B类股将伯克希尔·哈撒韦的股价从8万美元拆到约3,000美元，就是为了让普通投资者也能买得起。2021年，特斯拉宣布1拆5，股价从约700美元降到约140美元，引发了新一轮购买潮。A股也有类似操作，如2024年贵州茅台有传闻要拆股（但尚未实施），因为每股2,000多元的门槛确实让很多散户望而却步。

注意：拆股本身不创造价值。好比你有一张100元钞票换成了两张50元——钱的总数没变。但拆股往往被视为公司管理层对股价前景有信心的信号，因此短期可能推动股价上涨（心理学效应）。

股票回购

公司用自有资金从市场上买回自己的股票，这些股票成为库藏股（Treasury Stock）。回购的好处：

  • 减少流通股数量，提升每股收益（EPS）
  • 表明管理层认为股价被低估
  • 比现金股息更有税收效率（美国市场尤为明显）

A股的股票回购在2024-2025年显著增加。随着新"国九条"鼓励上市公司提升股东回报，越来越多的A股公司（如比亚迪、宁德时代、美的集团）开始大规模回购股份并注销——这意味着每股对应的利润增加了，对长期持有者是实实在在的好处。

2.4 股票估值基础

我们终于来到了股票投资最核心的问题：一只股票到底值多少钱？

股利贴现模型（DDM）

最简单且最经典的估值模型。原理非常直观：股票的价值等于它未来所有现金股息的现值之和。

# 股利贴现模型（DDM）— 零增长模型
# 假设每年股息 D = 2.00 元，必要收益率 r = 8%
# 零增长模型：V = D / r
V = 2.00 / 0.08  # = 25.00 元

# 戈登增长模型（固定增长）
# 假设股息 D0 = 2.00，增长率 g = 5%
V = D0 * (1 + g) / (r - g)
V = 2.00 * 1.05 / (0.08 - 0.05)  # = 70.00 元

# 不同增长率假设下的理论估值差异巨大！
# 为什么"估值是一门艺术"——因为增长率 g 只差1%，估值结果可能差30%。

相对估值法

现实中，投资者更常用相对估值指标来快速判断贵贱：

指标全称含义应用场景
PE市盈率 = 股价 / 每股收益几年回本盈利稳定的成熟公司
PB市净率 = 股价 / 每股净资产股价与账面价值之比银行、保险等金融股
PS市销率 = 股价 / 每股营收每元收入对应的市值亏损但高增长的公司

实战中，你需要比较同类公司的估值水平。比如食品饮料行业的PE通常是20-30倍，而银行股只有5-8倍。如果一家消费公司的PE只有12倍，可能被低估；但如果一家银行股的PE到了15倍，反而可能被高估。估值永远是横向比较 + 纵向比较的艺术。

初学者最容易犯的错误：看到PE低的股票就认为是"便宜货"。一只股票PE低可能是因为市场预期它的盈利即将下滑（"价值陷阱"）。反过来，高PE不一定贵——如果公司的盈利增长足够快，高PE可能是"合理的贵"。

2.5 中国A股特色

从审批制到注册制

2023年2月，中国资本市场迎来历史性时刻——全面注册制正式落地。在此之前，A股实行的是审批制/核准制。简单说：过去公司能不能上市，由证监会"审批"决定，排队等个两三年是常态。现在改为"注册制"，监管机构只做形式审查，价值判断交给市场。这带来了几个重大变化：

  • 上市速度大幅加快——2023年A股新增IPO超过300家
  • "壳价值"消失——过去价值几十亿元的"壳资源"变成废纸
  • 退市常态化——2024年退市公司数量创历史新高
  • 新股破发成为常态——过去"打新必赚"的"无风险套利"时代终结

北京证券交易所

2021年9月成立，定位是服务"专精特新"中小企业（专业化、精细化、特色化、新颖化）。北交所的上市标准更灵活，适合早期阶段的科技创新企业。到2025年，北交所上市公司已超过300家，总市值超过1.5万亿元。对于投资者来说，北交所的股票波动更大、流动性更差，但潜在回报也更高——类似中国版的"纳斯达克微型股"。

本章要点回顾：
  • 普通股享有投票权和剩余索取权，优先股享受固定股息和优先清偿
  • 股息政策包括现金股息、股票股息和股息再投资计划
  • 股票拆分增加流通性但不改变市值，回购减少流通股提升EPS
  • DDM模型是股息贴现估值的理论基础，PE/PB/PS是实战中最常用的相对估值指标
  • A股已进入全面注册制时代，北交所为"专精特新"企业提供了新的融资平台

本章我们认识了股票——投资世界中最核心的资产类别。但投资者不能把所有的钱都放在股票上，因为风险太大了。下一章我们来学习一个更稳健的工具——债券。你会发现，债券看似"无聊"，但它在资产配置中的重要性绝不亚于股票。让我们继续前进！
