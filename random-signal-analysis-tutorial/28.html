项目目标

本项目要完成的任务是：设计并实现一个数字通信信号的端到端接收处理系统。具体来说，系统接收一段被噪声污染的BPSK调制信号，依次完成功率谱分析、噪声抑制、信号检测和参数跟踪，最终输出解调后的比特流并评估系统性能。

分步实现

Step 1：生成BPSK调制信号 + AWGN噪声

我们首先生成一段 BPSK（二进制相移键控）调制信号。BPSK 用两个相位（0 和 π）分别代表比特 1 和比特 0。信号经过 AWGN 信道后，接收端收到的信号是：r(t) = s(t) + w(t)。

import numpy as np
import matplotlib.pyplot as plt
from scipy import signal as sp_signal

# --- 参数配置 ---
fs = 10000       # 采样率 (Hz)
fc = 1000        # 载波频率 (Hz)
Rb = 200         # 比特率 (bps)
Ns = int(fs / Rb) # 每比特采样数
N_bits = 200     # 发送比特数
SNR_dB = 6       # 信噪比 (dB)

# --- Step 1: 生成 BPSK 基带信号 ---
bits = np.random.randint(0, 2, N_bits)
symbols = 2 * bits - 1               # 0→-1, 1→+1
baseband = np.repeat(symbols, Ns)     # 基带信号（矩形脉冲）

t = np.arange(len(baseband)) / fs
carrier = np.cos(2 * np.pi * fc * t)
tx_signal = baseband * carrier        # BPSK 调制

# 添加 AWGN 噪声
E_b = np.sum(tx_signal**2) / N_bits
N0 = E_b / (10**(SNR_dB/10))
noise = np.sqrt(N0/2) * np.random.randn(len(tx_signal))
rx_signal = tx_signal + noise

Step 3：自适应噪声抵消

使用LMS自适应滤波器从接收信号中消除噪声。我们假设有一个参考噪声输入（相关噪声），LMS滤波器自适应地估计噪声路径并抵消它：

# --- Step 3: LMS 自适应噪声抵消 ---
M = 32           # 滤波器阶数
mu = 0.01        # 步长
w = np.zeros(M)  # 滤波器权重初始化

# 假设参考噪声（与接收信号中的噪声相关）
ref_noise = 0.5 * noise + 0.1 * np.random.randn(len(noise))

y = np.zeros(len(rx_signal))
e = np.zeros(len(rx_signal))

for n in range(M, len(rx_signal)):
    x_n = ref_noise[n-M:n][::-1]
    y[n] = np.dot(w, x_n)           # 估计噪声
    e[n] = rx_signal[n] - y[n]      # 误差 = 去噪后的信号
    w += 2 * mu * e[n] * x_n        # LMS 权重更新

# 计算 SNR 改善量
SNR_before = 10 * np.log10(np.var(tx_signal) / np.var(noise))
SNR_after  = 10 * np.log10(np.var(tx_signal) / np.var(e - tx_signal))
print(f"SNR 改善: {SNR_before:.1f} dB → {SNR_after:.1f} dB")

Step 4：匹配滤波器检测

使用匹配滤波器检测去噪后信号中的 BPSK 符号，并与发送比特对比计算误码率：

# --- Step 4: 匹配滤波器检测 ---
# 基带脉冲形状（能量归一化）
pulse = np.ones(Ns) / np.sqrt(Ns)
matched_filter = pulse[::-1]

# 匹配滤波
detected = np.convolve(e * carrier, matched_filter, mode='same')

# 在最佳采样点判决（每比特中间）
sample_idx = np.arange(Ns//2, len(detected), Ns)
decisions = np.sign(detected[sample_idx])
detected_bits = (decisions + 1) // 2  # 映射回 0/1

# 计算 BER
ber = np.sum(detected_bits != bits) / N_bits
print(f"误码率 (BER) = {ber:.4f}")

Step 5：卡尔曼滤波跟踪时变参数

假设信道增益是时变的（例如移动通信中的衰落信道），我们使用卡尔曼滤波实时跟踪信道增益的变化：

# --- Step 5: 卡尔曼滤波跟踪时变信道增益 ---
# 状态空间模型: 信道增益 h(t) 由一阶AR过程驱动
# 状态方程: h[n] = a * h[n-1] + v[n],  v ~ N(0, Q)
# 观测方程: y[n] = s_baseband[n] * h[n] + w[n], w ~ N(0, R)

a = 0.99          # AR 系数（慢变化信道）
Q = 0.01          # 过程噪声方差
R = N0 / 2        # 观测噪声方差

h_est = np.zeros(N_bits)  # 估计的信道增益
h_err = np.zeros(N_bits)  # 估计误差方差

# 初始化
h_hat = 1.0
P = 1.0

for n in range(N_bits):
    # 预测
    h_pred = a * h_hat
    P_pred = a**2 * P + Q

    # 更新
    K = P_pred * symbols[n] / (symbols[n]**2 * P_pred + R)
    # 简化：假设已知发送符号（导频辅助）
    y_obs = rx_signal[n*Ns + Ns//2] * np.cos(2*np.pi*fc*t[n*Ns + Ns//2])
    h_hat = h_pred + K * (y_obs - symbols[n] * h_pred)
    P = (1 - K * symbols[n]) * P_pred

    h_est[n] = h_hat
    h_err[n] = P

# 绘制卡尔曼滤波的收敛过程
plt.figure()
plt.plot(h_est, label='估计信道增益')
plt.fill_between(range(N_bits), h_est - 2*np.sqrt(h_err),
                  h_est + 2*np.sqrt(h_err), alpha=0.3, label='±2σ 置信区间')
plt.xlabel('符号序号')
plt.ylabel('信道增益')
plt.legend()
plt.title('卡尔曼滤波跟踪信道增益')
plt.show()

Step 6：性能评估

最后，我们改变 SNR 运行多次仿真，绘制 BER vs Eb/N0 曲线，观察理论与仿真的吻合程度：

# --- Step 6: BER vs Eb/N0 曲线 ---
SNR_range = np.arange(0, 14, 2)
ber_sim = []

for snr in SNR_range:
    EbN0 = 10**(snr/10)
    N0_sim = 1 / EbN0
    noise_sim = np.sqrt(N0_sim/2) * np.random.randn(len(tx_signal))
    rx_sim = tx_signal + noise_sim

    # 简化的匹配滤波+BPSK判决（省略重复细节）
    detected_sim = np.convolve(rx_sim * carrier, pulse[::-1], mode='same')
    decisions_sim = np.sign(detected_sim[Ns//2::Ns])
    ber_sim.append(np.sum(decisions_sim != symbols) / N_bits)

# 理论 BPSK BER: 0.5 * erfc(sqrt(Eb/N0))
ber_theory = 0.5 * np.special.erfc(np.sqrt(10**(SNR_range/10)))

plt.figure()
plt.semilogy(SNR_range, ber_sim, 'o-', label='仿真')
plt.semilogy(SNR_range, ber_theory, 's--', label='理论')
plt.xlabel('Eb/N0 (dB)')
plt.ylabel('误码率 (BER)')
plt.legend()
plt.grid(True, alpha=0.3)
plt.title('BPSK 误码率性能')
plt.show()

从项目到工程：一些额外的思考

完成这个项目后，我想分享几点工程经验：

  • 模块化第一：每个模块独立测试通过后再串联。如果整个系统出了问题，用"二分法"定位——逐一半屏蔽各模块，找到问题源头。
  • 可视化是调试利器：不要只盯着最终BER看——中间过程的星座图、误差曲线、功率谱、滤波器权重轨迹，每一个都是诊断系统健康的"生命体征"。
  • 理论是地图，仿真才是实地：理论公式给出了在理想假设下的性能界，但真实系统的性能往往受限于非理想因素（定时同步误差、载波频偏、滤波器非理想响应）。能把理论性能仿真到和公式一致，说明你理解了模型；能在实际硬件上接近理论性能，说明你理解了工程。
