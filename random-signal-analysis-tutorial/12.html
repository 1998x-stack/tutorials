开篇：随机信号穿越"黑箱"

大家好，我是你们的讲师。一个随机信号 X(t) 进入一个线性时不变系统（LTI 系统），出来变成 Y(t)。已知 X(t) 的统计特性，如何求出 Y(t) 的统计特性？这个问题在工程中无处不在：电子噪声通过放大器、无线信号通过多径信道、股票价格通过移动平均滤波器……本质上都是同一个数学结构。

好消息是，如果你已经掌握了确定性信号通过 LTI 系统的分析方法，那么随机版本并不困难——我们只需要把经典卷积运算与统计期望结合起来。本章和下一章将完整推导这套理论，本章专注于时域分析，下一章进入频域。

问题的提出

系统模型

考虑一个线性时不变系统，其冲激响应为 h(t)，频率响应为 H(ω)。输入是随机信号 X(t)，输出为：

Y(t) = ∫_{-∞}^{∞} h(τ) X(t - τ) dτ = h(t) ∗ X(t)

问题：已知输入随机信号 X(t) 的统计特性（均值 μ_X(t)、自相关 R_X(t₁, t₂)），以及系统的冲激响应 h(t)，求输出 Y(t) 的统计特性（均值 μ_Y(t)、自相关 R_Y(t₁, t₂)）以及输入-输出的互相关 R_{XY}(t₁, t₂)。

注意，这里一开始不假设输入是平稳的——公式推导对一般的随机信号都成立，然后再看平稳情况下的简化。

核心思想：卷积是线性运算，而期望也是线性运算——两者可以交换顺序。这个"期望与卷积可交换"的性质，是整个随机信号通过 LTI 系统分析的基石。正是因为这两个线性算子的可交换性，我们才能从输入的统计特性推导出输出的统计特性。

输出均值

一般公式

输出 Y(t) 的均值为：

μ_Y(t) = E[Y(t)] = E[∫ h(τ) X(t-τ) dτ]
        = ∫ h(τ) E[X(t-τ)] dτ        （交换期望与积分）
        = ∫ h(τ) μ_X(t-τ) dτ
        = μ_X(t) ∗ h(t)

漂亮！输出均值 = 输入均值与冲激响应的卷积。这完全类似于确定性信号分析的输入-输出关系——只不过这里的"输入"是统计均值函数。

平稳输入的特殊情况

如果 X(t) 是 WSS 的，那么 μ_X(t) = μ_X 是常数：

μ_Y(t) = μ_X · ∫_{-∞}^{∞} h(τ) dτ = μ_X · H(0)

其中 H(0) = ∫ h(τ) dτ 是系统的直流增益——频率响应在 ω = 0 处的值。这个结果非常直观：平稳信号的直流分量经过系统后，被系统的直流增益缩放。如果 H(0) = 0（例如交流耦合放大器），输出均值为零，不管输入均值是多少。

更重要的是：平稳输入 ⟹ 输出均值也是常数，所以输出至少是"均值平稳"的。后面我们会看到，输出自相关也只依赖于 τ，因此 WSS 输入通过 LTI 系统的输出也是 WSS 的——这个性质非常关键。

输入输出互相关

定义与推导

输入与输出之间的互相关函数 R_{XY}(t₁, t₂) 定义为：

R_{XY}(t₁, t₂) = E[X(t₁) Y(t₂)]
                = E[X(t₁) · ∫ h(τ) X(t₂-τ) dτ]
                = ∫ h(τ) E[X(t₁) X(t₂-τ)] dτ
                = ∫ h(τ) R_X(t₁, t₂-τ) dτ
                = R_X(t₁, t₂) ∗_{t₂} h(t₂)

注意这里的卷积是关于 t₂ 变量的（第二个时间参数）。直观含义：互相关 = 输入自相关与系统冲激响应的卷积。

平稳情况下的简化

如果 X(t) 是 WSS 的，R_X(t₁, t₂) = R_X(t₂ - t₁) = R_X(τ)，其中 τ = t₂ - t₁：

R_{XY}(τ) = ∫ h(u) R_X(τ - u) du = R_X(τ) ∗ h(τ)

所以输入输出互相关也只依赖于时延 τ！这意味着 X(t) 和 Y(t) 是联合 WSS 的。这是一个重要的性质：WSS 输入通过 LTI 系统 ⟹ 输入输出联合 WSS。

这个结果在系统辨识中有重要应用：如果我们能在系统输入端注入一个已知统计特性的测试信号（如白噪声），然后测量输入输出互相关，就可以估计出系统的冲激响应——因为当输入为白噪声时，R_X(τ) = (N₀/2)δ(τ)，于是 R_{XY}(τ) ∝ h(τ)。这就是相关法系统辨识的基本原理。

系统辨识的简单思路：给待测系统输入一个宽带宽的白噪声信号，同时记录输入和输出。计算输入输出的互相关函数——理论上它正比于系统的冲激响应 h(τ)。这个方法的优点：(1) 不需要断开系统；(2) 白噪声不影响系统的正常工作；(3) 可以平均多次来提高估计精度。这是工程中测量未知系统冲激响应的首选方法之一。

输出自相关（时域）

双重卷积公式

现在来到最重要的部分——输出自相关 R_Y(t₁, t₂)：

R_Y(t₁, t₂) = E[Y(t₁) Y(t₂)]
             = E[∫ h(α) X(t₁-α) dα · ∫ h(β) X(t₂-β) dβ]
             = ∫∫ h(α) h(β) E[X(t₁-α) X(t₂-β)] dα dβ
             = ∫∫ h(α) h(β) R_X(t₁-α, t₂-β) dα dβ
             = R_X(t₁, t₂) ∗_{t₁} h(t₁) ∗_{t₂} h(t₂)

这就是输出自相关的双重卷积公式——输入自相关分别对两个时间参数与系统冲激响应做卷积。

平稳情况的关键简化

当 X(t) 是 WSS 时，令 τ = t₂ - t₁：

R_Y(τ) = ∫∫ h(α) h(β) R_X(τ + α - β) dα dβ

进一步可以简化为更方便的形式。先对 β 进行变量替换 u = β - α：

R_Y(τ) = ∫ h(α) ∫ h(β) R_X(τ + α - β) dβ dα
        = ∫ h(α) [R_X(τ) ∗ h(-τ)] 在 (τ+α) 处取值  dα

最常用的形式是引入"冲激响应的自相关" ϕ_h(τ) = ∫ h(t) h(t+τ) dt = h(τ) ∗ h(-τ)，于是：

R_Y(τ) = R_X(τ) ∗ ϕ_h(τ) = R_X(τ) ∗ h(τ) ∗ h(-τ)

也就是说：输出自相关 = 输入自相关与系统冲激响应自相关的卷积。这个公式优雅而强大——它将系统的影响完全封装在 ϕ_h(τ) 中。

直观理解：ϕ_h(τ) = h(τ) ∗ h(-τ) 是系统冲激响应的自相关函数，它反映了系统的"记忆特性"——好的滤波器（窄带）有很长的 ϕ_h(τ)，意味着输出信号在不同时刻之间有很强的相关性；而宽带系统（短冲激响应）的 ϕ_h(τ) 衰减很快，意味着输出信号的相关性弱。

观察上表你会发现一个模式：时域的公式涉及卷积，比较繁琐；但频域的公式（见下一章）是简单的乘法。这就是为什么在实际工程中，大多数随机信号通过 LTI 系统的分析都是在频域中完成的。

计算示例：RC低通滤波器的输出自相关

考虑一个一阶 RC 低通滤波器，其冲激响应 h(t) = (1/RC) · e^{-t/RC} · u(t)。输入是 PSD 为 N₀/2 的白噪声。求输出自相关。

由 ϕ_h(τ) = h(τ) ∗ h(-τ)：

ϕ_h(τ) = (1/(2RC)) · e^{-|τ|/RC}

R_Y(τ) = (N₀/2) · ϕ_h(τ)
       = (N₀/4RC) · e^{-|τ|/RC}

输出平均功率：σ²_Y = R_Y(0) = N₀/(4RC)。有趣的是，时间常数 RC 越大（滤波器越窄），输出平均功率越小——因为更多的噪声功率被滤波器滤除了。这正是直觉的数学表达。

踩坑故事：忘记双重卷积的代价

我的一个同事在做 MEMS（微机电系统）加速度计的噪声建模时，遇到了一个经典错误。加速度计可以建模为一个二阶线性系统（质量-弹簧-阻尼系统），输入是外部加速度，输出是检测到的位移。系统中的电子噪声可以建模为输入端的白噪声。

他需要计算输出噪声的功率（即位移检测的噪声方差），以便评估加速度计的灵敏度极限。他正确地计算了白噪声通过系统的 PSD 传递（这个我们下章会讲），然后积分得到了噪声功率——结果与实验测量值差了 2 倍。

排查了两天后，他发现——在计算输出信号的总功率时，他使用了公式：

σ²_Y = ∫ S_X(ω) · |H(ω)|² dω     ← 正确

但在检查时域结果时，他直接用了 R_Y(0) = R_X(0) ∗ h(0) ∗ h(0)，即只做了一次而不是双重卷积。这个错误相当于只对输入自相关在一个时间轴上做了滤波，忘记了对另一个时间轴的处理——结果少了一个数量级的噪声功率。

教训：输出自相关的双重卷积公式 R_Y = R_X ∗ h ∗ h 不是打字打重复了，而是两次卷积缺一不可。第一次卷积对应输出信号的时间索引 t₁，第二次对应 t₂。如果你只做一次卷积，你得到的是 R_{XY}（互相关），而不是 R_Y（输出自相关）。一个简单的记忆方法："输入自相关要过两次系统，一次给 t₁，一次给 t₂；互相关只过一次，因为另一个时间点在输入侧"。别把互相关当自相关！

本章小结

这一章我们从时域角度完整分析了随机信号通过 LTI 系统的统计特性传递。我们推导了三个核心公式：输出均值（μ_Y = μ_X ∗ h）、输入输出互相关（R_{XY} = R_X ∗ h）、输出自相关（R_Y = R_X ∗ h ∗ h）。对于 WSS 输入，输出也是 WSS 的，且输入输出联合平稳——这个性质是所有频域分析的基础。我们还展示了一个 RC 低通滤波器的完整计算例子，并通过 MEMS 噪声建模的踩坑故事警示了忘记双重卷积的常见错误。

下一章我们将进入频域视角——你会发现，同样的关系在频域中变得异常简洁：S_Y(ω) = |H(ω)|² · S_X(ω)。频域分析不仅公式更干净，还引入了等效噪声带宽等重要概念。准备好迎接"乘法时代"吧。
