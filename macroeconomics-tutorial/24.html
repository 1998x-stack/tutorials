第24章 预期与宏观经济学

想象你正在开车，但后视镜被完全遮住了，你只能看到挡风玻璃外的路况。你能安全驾驶吗？不能——因为你无法判断后面有没有车、侧后方有没有危险。现在反过来：遮住挡风玻璃，只能看后视镜开车。你还会觉得安全吗？当然不——你只能看到已经走过的路，看不到前方的弯道和障碍。

这个简单的类比暗含着宏观经济学过去半个世纪最深刻的一场革命。在1970年代之前，经济学家几乎都是在"只看后视镜"的状态下做预测和制定政策。他们用过去的数据来预测未来，并假设人们会延续过去的行为模式。直到一位名叫罗伯特·卢卡斯（Robert Lucas）的芝加哥大学教授站出来说：你们错了。人们不是傻瓜——他们会根据政策的变化来调整自己的预期。预测政策的效果，不能只看历史数据，还要看人们对政策的反应。

大家好，我是你们的讲师。这一章讨论一个贯穿现代宏观经济学所有分支的概念——预期（Expectations）。我们将从原始的适应性预期开始，历经理性预期的革命，了解著名的卢卡斯批判，探讨时间不一致性问题，最后落到央行如何管理预期。你很快就会意识到：预期不是经济学的配角，它就是经济本身。

1. 适应性预期：用"后视镜"开车

什么是适应性预期（Adaptive Expectations）？简单说就是：用过去推断未来。

如果你想知道明年的通货膨胀率，最简单的办法是看今年的通胀率，然后做一点微调。假设今年的通胀是5%，你的预测就是"大概还是5%左右"。如果实际发生了偏差——比如变成了7%——那么你下次会修正你的预期，往上调一些。这就是适应性预期的核心逻辑：预期 = 过去的预期 + 对过去预测误差的部分修正。

这个想法在1950-1960年代非常流行，由菲利普·卡根（Phillip Cagan）和米尔顿·弗里德曼等人发展和推广。弗里德曼在解释菲利普斯曲线时就用到了它：他认为，短期内工人可能会被未预期到的通胀"愚弄"，以为工资上涨是实际购买力的提升，从而增加劳动供给。但长期来看，工人会逐步调整预期，最终失业率会回到"自然率"水平——不存在通胀与失业的长期权衡。

适应性预期看起来挺合理的。毕竟人类本来就有"惯性思维"的倾向。而且在不那么复杂的环境中，它确实能给出不错的预测。但它的致命缺陷在于：它允许系统性的预测错误持续存在。

适应性预期的数学形式与缺陷：

公式：E_t[P_{t+1}] = E_{t-1}[P_t] + λ(P_t - E_{t-1}[P_t])

其中：
  E_t[P_{t+1}] = 本期对下一期价格的预期
  E_{t-1}[P_t] = 上期对本期价格的预期
  P_t = 实际价格
  λ = 调整速度（0 < λ ≤ 1）

情景模拟：假设通胀从3%上升到10%，并且持续在10%
  第1年：预期3%，实际10%，误差=7个点
  第2年：预期=3%+0.5×7%=6.5%，实际10%，误差=3.5个点
  第3年：预期=6.5%+0.5×3.5%=8.25%，实际10%，误差=1.75个点
  第4年：预期≈9.1%，实际10%，误差≈0.9个点

发现了吗？即使通胀稳定在10%，适应性预期也需要数年
才能"追上"现实。在这段时间里，人们的预期一直是
系统性地偏低的——这给政策制定者留下了"欺骗"的空间。

这种"系统性被愚弄"的可能性，在理论上意味着政府可以通过制造意外的通胀来暂时降低失业率。但正是这一点——政府可以系统性地"骗"过公众——在1970年代遭到了根本性的挑战。

2. 理性预期革命：Muth的种子与Lucas的收获

1961年，一位名叫约翰·穆斯（John Muth）的经济学家发表了一篇不长的文章，标题很不起眼：《理性预期与价格变动理论》。这篇文章在发表后的前十年几乎没有引起什么关注。但后来它成了整个宏观经济学史上被引用最多的论文之一——因为它在里面埋下了一颗种子。

穆斯的想法极其简单但又极其深刻：人们在做决策时，会不会比适应性预期更聪明？如果你知道政府有制造通胀的倾向，你为什么还会年复一年地犯同样的预测错误？你会上网查数据、你会看央行的报告、你会结合所有可得信息——最终，你的预期应该不会系统性地偏离现实。

这就是理性预期（Rational Expectations）的核心定义：经济主体利用所有可得信息来形成预期，他们的预期误差是随机的、均值为零的——也就是说，没有系统性的预期错误。

注意：理性预期不是说每个人都能准确预测未来。它说的是——你无法持续地、系统地"骗"过经济主体。你可能有运气好猜对一次两次，但你不能年复一年地让他们犯同样的方向性错误。

穆斯的思想在1970年代被罗伯特·卢卡斯（Robert Lucas）带入了宏观经济学的主流。卢卡斯是芝加哥大学的教授，也是理性预期革命的旗手。他把理性预期与市场出清（价格和工资完全灵活）结合起来，发展出了新古典宏观经济学（New Classical Macroeconomics）。

这个学派最著名的结论是政策无效性命题（Policy Ineffectiveness Proposition）：如果人们的预期是理性的，而且价格和工资是灵活的，那么任何被预期到的货币政策都无法影响实际产出和就业——它只会改变价格水平。只有未被预期到的"意外"才有效，但这种意外在长期中是随机的，不可能被系统性地利用。

理性预期的核心含义：政策制定者必须把公众当作"对手"
在适应性预期时代，政策制定者就像在跟一群小孩玩"剪刀石头布"——小孩每次都出"剪刀"，你只需要每次都出"石头"就能赢。但在理性预期下，公众升级成了"同级别的对手"——他们会观察你的策略，调整自己的行为。如果你想制造通胀来刺激就业，公众会看穿你的意图，提前调整工资和价格要求，让你的政策落空。

这就引出了一个看似悲观的结论：政府越是想"管理"经济，经济就越不听话。因为市场参与者不是被动等待的棋子，他们是主动思考的策略家。

3. 卢卡斯批判：为什么用历史数据评价政策是自欺欺人

1976年，卢卡斯发表了另一篇里程碑式的论文——《经济计量政策评价：一个批判》。这篇文章后来被简称为卢卡斯批判（Lucas Critique），它在宏观经济学界引发了一场地震。

卢卡斯批判的核心论点可以这样理解：在1970年代之前，宏观经济学家和政府部门使用大规模的宏观计量经济模型来评估政策效果。这些模型由几十甚至上百个方程组成，里面参数都是根据历史数据估计出来的。假设政府想评估"减税对GDP的影响"，模型就会把历史数据中的"减税与GDP关系"提取出来做预测。

卢卡斯说：这么做是荒谬的。

为什么？因为历史数据中观测到的行为参数（比如"边际消费倾向""货币需求弹性"），都是在当时政策环境下人们的最优反应。当你改变政策时，人们也会改变他们的最优反应——那些参数也会随之改变。用不变的历史参数去预测政策变动后的世界，就像用牛顿力学去计算量子世界——方法论从根本上就错了。

方面 卢卡斯批判之前 卢卡斯批判之后
经济模型的基础 统计"发现"变量间的关系（约化形式） 从个体最优化行为推导（结构形式）
参数的含义 固定的统计相关系数 政策环境决定的"深度参数"
政策评估方法 把历史数据代入模型推算 考虑政策变化后行为参数的变化
对IS-LM的态度 IS-LM是核心分析框架 IS-LM缺乏微观基础，参数不稳定
对凯恩斯主义的影响 占主导地位 遭遇严重危机，推动新凯恩斯主义转型

卢卡斯批判直接动摇了IS-LM模型的根基。希克斯在1937年把凯恩斯的理论概括为IS-LM模型之后，这个模型成了宏观教学的标配，也是很多政策分析的基础工具。但IS-LM的参数是统计上拟合的，不是从微观主体的最优化行为推导出来的。卢卡斯说：这个模型在政策不变的环境下做短期的趋势描述也许还行，但用它来评估"政策变动后的效果"是完全不可靠的。

卢卡斯批判的后果是什么？宏观经济学进入了一个新的阶段——经济学家们开始从微观基础出发重新构建宏观模型。这就是下一章要讲的DSGE模型的起源。可以说，没有卢卡斯批判，就没有现代宏观经济学。

课堂思考：卢卡斯批判对现实生活的启示
卢卡斯批判不仅是经济学理论，它也是一个深刻的思维模型。想想看：当公司推出新的考核制度时，员工的反应一定与旧制度下的行为不同——用过去的绩效数据预测新考核制度的效果必然不准确。当政府出台新的限购政策时，房地产市场的参与者的行为会发生改变——用过去的房价数据预测新政后的房价走势同样不可靠。

无论在哪个领域——一旦规则改变，参与者的行为就会改变。这是卢卡斯批判教给我们的最普适的智慧。

4. 时间不一致性与央行独立性：为什么"承诺"比"相机抉择"好

1977年，Kydland和Prescott（是的，就是我们在上一章认识的那两位RBC理论的奠基人）发表了另一篇影响深远的论文——《规则而非相机抉择：最优计划的不一致性》。这篇论文提出了著名的时间不一致性（Time Inconsistency）问题，并因此获得了2004年的诺贝尔奖。

什么叫"时间不一致性"？我们用一个故事来说明它。

假设你是一个央行行长。你宣布："今年的通胀目标定在2%，我不会试图用意外的通胀来刺激就业。"企业和工人们相信了你，于是他们按照2%的通胀率来签订工资合同（接受2%的工资增长率）。好了，现在合同签了，你不能反悔了吧？——等等，你发现一个诱惑：既然合同已经签了，现在制造5%的通胀会怎样？实际工资会下降，企业会多雇佣工人，失业率会暂时降低。这多好呀！

但是，聪明的企业和工会知道你有这种动机——他们不是傻子。所以他们一开始就不相信你的2%承诺，要求工资合同包含5%甚至更高的通胀补偿。结果呢？你被迫接受了"高通胀+自然失业率"的组合——通胀很高，但失业率并没有降低。这就是通胀偏差（Inflation Bias）：你试图"欺骗"公众的成本，最终以更高的均衡通胀率体现出来。

时间不一致性的博弈树：

第1步：央行宣布"通胀目标=2%"
第2步：公众决定是否相信
  ├─ 如果相信 → 按照2%签订工资合同
  │   └─ 央行面临诱惑：制造5%通胀 → 降低实际工资 → 增加就业
  │       → 公众被"骗"了 → 下次不再相信
  └─ 如果不信 → 按照5%签订工资合同
      └─ 央行只能制造5%通胀（制造更高通胀也没好处）
          → 结果：高通胀，无额外就业

纳什均衡：公众不信，央行制造5%通胀
           两个都输——这就是没有承诺机制的结果

解法是什么？Kydland和Prescott的答案非常简单有力：用规则代替相机抉择。把货币政策交给一个独立的、以价格稳定为单一目标的央行，以法律形式明确其职责，让央行行长无法（或没有动机）去追求短期的政治利益。这就是央行独立性（Central Bank Independence）的核心理念。

这个理论在现实中得到了有力的验证。1990年代，新西兰、英国、加拿大、瑞典等国先后实行了通胀目标制（Inflation Targeting）——央行被明确要求将通胀控制在某个区间内（通常是2%左右），如果做不到就要公开解释。这些国家的通胀率确实从之前的高水平降下来了，而且长期维持稳定。全球央行的独立性也在过去的三十年里得到了实质性的提升。

5. 前瞻指引与预期管理：央行如何"用嘴加息"

经历了理性预期革命和时间一致性讨论之后，现代央行行长们深刻理解了管理预期的重要性。既然公众会根据对未来的预期来做今天的决策，那么央行能不能通过"说"来影响经济，而不需要真的"做"？

答案是：能。这就是前瞻指引（Forward Guidance）。

前瞻指引是指央行就未来的货币政策路径向公众提供信息。它有三种形式，从弱到强：

第一种：定性指引。央行说"我们将在相当长一段时间内保持宽松"。这很模糊——"相当长"是多长？"宽松"到什么程度？但市场仍然会从中读取信号。

第二种：时间指引。央行说"我们承诺在2023年底之前不会加息"。这个更具体——市场可以直接据此做出决策。比如2020年美联储就承诺"在实现充分就业和2%通胀之前不会加息"，后来通胀飙升到8%以上才被迫修改了措辞。

第三种：状态指引。央行说"在失业率降至6.5%以下之前，我们不会加息"。这个把承诺条件化——你清楚地知道触发政策变动的门槛。伯南克在担任美联储主席时就喜欢用这种形式。

前瞻指引为什么有效？因为长期利率不是由今天的短期利率决定的，而是由市场对未来短期利率路径的预期决定的。如果央行成功让市场相信"未来三年都不会加息"，那么长期利率就会保持在低位，从而刺激投资和消费。哪怕央行一天都没买过债券，它已经实现了"降息"的效果——这就是"用嘴加息"（或者说"用嘴降息"）。

工具 传统机制 预期管理机制 优势
政策利率 调整基准利率影响借贷成本 通过利率信号传递政策意图 直接、精准，但有零下限约束
前瞻指引 — 告知未来政策路径，影响长期利率预期 零下限下仍有效，不需实际动用工具
量化宽松 购买资产压低长端利率 向市场传递"我们决心对抗萧条"的信号 信号效应可能比实际购买更重要
通胀目标 — 锚定长期通胀预期，防止自我实现的通缩 建立信用，减少时间不一致性

预期管理当然不是万能的。它有明显的边界：信用是预期管理的基石。如果央行的承诺跟实际行为长期不一致（比如说了不加息但加了），市场的信任就会丧失。一旦信任崩塌，前瞻指引就变得毫无用处——你说什么市场都不信了。2022年美联储在"通胀是暂时的"说法被现实证伪之后，其前瞻指引的可信度确实受到了一定影响。

前瞻指引的一个经典翻车案例
2013年6月，时任美联储主席伯南克在国会听证会上突然暗示"可能会在未来几次会议上开始缩减QE规模"。市场完全没有准备——债券收益率瞬间飙升，全球股市暴跌，新兴市场资本外逃。这就是著名的"缩减恐慌"（Taper Tantrum）。

这件事说明了一个深刻的道理：前瞻指引是一把双刃剑。它既能安抚市场，也能惊吓市场。不是所有的"沟通"都是好事——沟通不当本身就是一种冲击。现代央行行长需要学习的不只是经济学，还包括沟通的艺术。

好的，第二十四章的内容就到这里了。我们走完了一段从适应性预期到理性预期、从卢卡斯批判到时间不一致性、再到前瞻指引的理论之旅。你可能会觉得有点抽象——但这些理念直接塑造了今天所有中央银行的运作方式。没有理性预期的洞见，就没有独立央行；没有时间不一致性的分析，就没有通胀目标制；没有对预期管理的理解，就没有量化宽松和前瞻指引这些非常规货币政策工具。
