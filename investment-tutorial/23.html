第23章 Python投资分析入门

我有个朋友叫小张，在某券商研究所做研究员。他入行第一年，师傅丢给他一个任务："把过去五年所有A股的财务数据拉出来，算一下每个行业的ROE中位数变化趋势。"小张一听，打开了Excel，准备手动复制粘贴——然后他对着3000多只股票的数据量沉默了。当晚，他花了8个小时，终于用Excel的VLOOKUP和透视表搞定了。但第二天师傅轻飘飘地说了一句："哦对了，每个月要更新一次。"小张当场崩溃。后来他学了Python，原来同样的工作只需要30行代码、3分钟就跑完了。

这个故事不是段子，是每天都在发生的事情。在2025年的投资研究中，Excel早已无法应对海量数据的处理需求。Python凭借其丰富的数据科学生态系统（pandas、numpy、matplotlib、scipy），已经成为投资分析领域的"通用语言"。无论你是做基本面研究、量化策略开发，还是机器学习模型，Python都是绕不开的工具。本章将带你从一个零基础的投资者视角，建立起Python投资分析的完整工具箱。

23.1 环境搭建：开始你的Python之旅

在写第一行代码之前，先把"厨房"准备好。对于没有编程背景的投资者，我强烈推荐以下方案：

Anaconda：一站式数据科学平台

Anaconda是目前最省心的Python发行版，它内置了200多个数据科学常用库，还附带了一个叫Conda的包管理器。你不需要像纯Python用户那样手动装pip、装库、处理依赖冲突——Anaconda全帮你搞定了。下载安装后，打开Anaconda Navigator，你就能看到一个图形化界面。

Jupyter Notebook：交互式分析的利器

传统的Python IDE（如PyCharm）适合写大型工程代码，但投资分析更多是"探索性的"——你拿到数据，先看看长什么样，算几个统计量，画几张图，再决定下一步怎么做。Jupyter Notebook完美适配这种工作流：你可以把代码、运行结果、图表、文字说明全都放在一个文件里。这也是全球量化研究员中使用率最高的开发工具。

23.2 数据获取：从市场拿到原始材料

数据是投资分析的"原材料"。没有数据，一切分析都是空中楼阁。对于中国投资者来说，最关心的通常是两类数据：A股数据和美股数据。

A股数据：AkShare（推荐） vs Tushare

AkShare是一个开源的A股数据接口，完全免费，无需注册即可使用。它封装了东方财富、同花顺等主流财经网站的数据接口，覆盖股票行情、财务数据、宏观经济指标、行业板块等。Tushare也是老牌库，但需要注册Token，部分数据需要积分。对初学者来说，AkShare更友好。

import akshare as ak
import pandas as pd

# 获取日线数据
stock_zh_a_hist_df = ak.stock_zh_a_hist(
    symbol="600519",
    period="daily",
    start_date="20200101",
    end_date="20250630",
    adjust="qfq"  # 前复权
)

print(stock_zh_a_hist_df.head())
print(f"数据行数: {len(stock_zh_a_hist_df)}")

yfinance是Yahoo Finance的非官方Python接口，可以免费获取美股、ETF、指数等行情数据。对于研究美股或全球ETF（如标普500、纳斯达克100）的投资者来说，它是首选工具。

23.3 收益率计算：量化分析的第一步

拿到价格数据后，第一件事就是把价格转化为收益率。收益率是投资分析的"燃料"——几乎所有后续分析（波动率、夏普比率、Beta、VaR）都基于收益率序列。

单期收益率 vs 多期收益率

简单收益率（Simple Return）是最直观的计算方式：R_t = (P_t - P_{t-1}) / P_{t-1}。但如果你需要计算多期总收益，简单收益率不能直接相加，需要连乘：R_total = (1+R_1)*(1+R_2)*...*(1+R_n) - 1。

对数收益率：量化分析的标配

对数收益率 r_t = ln(P_t / P_{t-1}) 在学术和量化实战中更常用，原因有三：第一，对数收益率具有时间可加性（多期收益率直接相加）；第二，对数收益率的分布更接近正态分布，方便统计建模；第三，在计算复合收益率时更精确。

# 收益率计算实战
import pandas as pd
import numpy as np

# 假设df包含'Close'列
df = stock_zh_a_hist_df.copy()
df['returns'] = df['收盘'].pct_change()           # 简单收益率
df['log_returns'] = np.log(df['收盘'] / df['收盘'].shift(1))  # 对数收益率

# 年化统计指标
daily_vol = df['returns'].std()
annual_vol = daily_vol * np.sqrt(252)   # 年化波动率
annual_return = df['returns'].mean() * 252  # 年化收益率
sharpe = annual_return / annual_vol     # 简单夏普比（假设无风险利率为0）

print(f"年化收益率: {annual_return:.2%}")
print(f"年化波动率: {annual_vol:.2%}")
print(f"夏普比率: {sharpe:.2f}")

# 最大回撤计算
df['cummax'] = df['收盘'].cummax()
df['drawdown'] = (df['收盘'] - df['cummax']) / df['cummax']
max_drawdown = df['drawdown'].min()
print(f"最大回撤: {max_drawdown:.2%}")

23.4 组合优化实现：从理论到代码

还记得第8章学过的马科维茨有效前沿吗？当时我们用了数学公式推导。现在，让我们用Python把它真正实现出来。

最小方差组合与最大夏普组合

给定一组资产的历史收益率数据，我们可以通过scipy.optimize来求解最优权重。具体来说：最小方差组合就是让组合波动率最小的权重分配；最大夏普组合是在给定无风险利率下，让单位风险获得的超额收益最大。

# 组合优化：有效前沿计算
import numpy as np
import pandas as pd
from scipy.optimize import minimize

# 假设我们有4只股票的收益率数据
# 实际中从AkShare获取多只股票数据后拼接
np.random.seed(42)
n_assets = 4
n_days = 1000
returns_df = pd.DataFrame(
    np.random.randn(n_days, n_assets) * 0.02 + 0.0008,
    columns=['茅台', '腾讯', '招商银行', '宁德时代']
)

mean_returns = returns_df.mean() * 252
cov_matrix = returns_df.cov() * 252

def portfolio_stats(weights):
    """计算组合的收益率、波动率、夏普比"""
    port_return = np.dot(weights, mean_returns)
    port_vol = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
    sharpe = port_return / port_vol
    return port_return, port_vol, sharpe

def min_variance_portfolio():
    """最小方差组合"""
    n = n_assets
    args = (mean_returns, cov_matrix)
    constraints = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})
    bounds = tuple((0, 1) for _ in range(n))
    initial = n * [1. / n]

    result = minimize(
        lambda w, mr, cov: portfolio_stats(w)[1],
        initial, args=args, method='SLSQP',
        bounds=bounds, constraints=constraints
    )
    return result.x

def max_sharpe_portfolio():
    """最大夏普组合"""
    n = n_assets
    constraints = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})
    bounds = tuple((0, 1) for _ in range(n))
    initial = n * [1. / n]

    def neg_sharpe(w):
        return -portfolio_stats(w)[2]

    result = minimize(
        neg_sharpe, initial, method='SLSQP',
        bounds=bounds, constraints=constraints
    )
    return result.x

min_var_w = min_variance_portfolio()
max_sharpe_w = max_sharpe_portfolio()

print("最小方差组合权重:", dict(zip(returns_df.columns, min_var_w.round(4))))
print("最大夏普组合权重:", dict(zip(returns_df.columns, max_sharpe_w.round(4))))

有效前沿的完整绘制

通过遍历不同的目标收益率，找到每个收益率水平下的最小方差组合，就能画出完整的有效前沿曲线。

实战经验：用历史数据做组合优化的最大陷阱是"过拟合"。你优化出来的"最优权重"是过去的最优，不代表未来。一个常见的改进是使用"收缩估计"（Shrinkage Estimation）来修正协方差矩阵，或者使用Black-Litterman模型将主观观点融入先验分布。简单说：不要迷信优化器的输出，把它当作参考而不是圣杯。

23.5 回测框架简单搭建

回测（Backtesting）是验证投资策略有效性的核心手段。一个完整的回测流程包括：信号生成 → 持仓管理 → 绩效评估。让我们用不到100行代码搭建一个简易的回测框架。

# 简易双均线策略回测
import pandas as pd
import numpy as np

def backtest_ma_strategy(df, short_win=20, long_win=60):
    """
    双均线策略回测
    策略逻辑：短均线上穿长均线时买入，下穿时卖出
    """
    data = df.copy()

    # 计算均线信号
    data['MA_short'] = data['收盘'].rolling(short_win).mean()
    data['MA_long'] = data['收盘'].rolling(long_win).mean()
    data['signal'] = 0
    data.loc[data['MA_short'] > data['MA_long'], 'signal'] = 1
    data['position'] = data['signal'].shift(1)  # 避免未来函数

    # 计算策略收益
    data['strategy_return'] = data['position'] * data['returns']
    data['strategy_cum'] = (1 + data['strategy_return']).cumprod()
    data['buy_hold_cum'] = (1 + data['returns']).cumprod()

    # 绩效评估
    total_return = data['strategy_cum'].iloc[-1] - 1
    buy_hold_return = data['buy_hold_cum'].iloc[-1] - 1
    strategy_vol = data['strategy_return'].std() * np.sqrt(252)
    strategy_sharpe = (data['strategy_return'].mean() * 252) / strategy_vol

    print(f"策略总收益: {total_return:.2%}")
    print(f"买入持有收益: {buy_hold_return:.2%}")
    print(f"策略年化波动率: {strategy_vol:.2%}")
    print(f"策略夏普比率: {strategy_sharpe:.2f}")

    return data

23.6 Python实战代码：滚动Beta与因子分析

代码是学出来的，不是看出来的。下面给出两个在投资分析中非常实用的代码片段，建议你照着敲一遍。

用pandas计算滚动Beta

Beta是衡量个股相对市场敏感度的核心指标。滚动Beta可以观察Beta的时变性——你会发现大多数股票的Beta并不是稳定的。

# 计算滚动Beta（个股vs沪深300）
def calc_rolling_beta(stock_returns, market_returns, window=252):
    """
    计算滚动Beta系数
    stock_returns: 个股日收益率序列
    market_returns: 市场指数日收益率序列（如沪深300）
    window: 滚动窗口，默认252个交易日
    """
    import pandas as pd

    df = pd.DataFrame({
        'stock': stock_returns,
        'market': market_returns
    }).dropna()

    # 滚动协方差 / 滚动市场方差
    rolling_cov = df['stock'].rolling(window).cov(df['market'])
    rolling_mkt_var = df['market'].rolling(window).var()
    rolling_beta = rolling_cov / rolling_mkt_var

    return rolling_beta

用scipy做组合优化（带约束条件）

实战中的组合优化往往有额外约束：行业集中度限制、单只股票权重上限、跟踪误差约束等。scipy.optimize可以灵活处理这些约束条件。

# 带约束的组合优化
from scipy.optimize import minimize

def constrained_portfolio(mean_ret, cov, max_weight=0.15):
    """
    带权重上限约束的组合优化
    max_weight: 单只股票最大权重（防止过度集中）
    """
    n = len(mean_ret)

    # 约束条件：权重之和=1
    constraints = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})

    # 权重边界：每只股票0到max_weight
    bounds = tuple((0, max_weight) for _ in range(n))

    # 目标函数：最大化夏普比率的负值
    def objective(w):
        port_ret = np.dot(w, mean_ret)
        port_vol = np.sqrt(np.dot(w.T, np.dot(cov, w)))
        return -port_ret / port_vol

    # 初始权重：等权
    x0 = np.array([1./n] * n)

    result = minimize(objective, x0, method='SLSQP',
                      bounds=bounds, constraints=constraints)
    return result.x
