第21章 电磁场数值方法

能解析求解的电磁问题少之又少——球体、圆柱体、无限大平面，这是麦克斯韦方程组仅有的"宠儿"。现实世界中的 PCB 不规则走线、人体在 MRI 中的电磁响应、手机天线的近场分布，都无法写出闭式解。数值方法把麦克斯韦方程组从"只能欣赏"变成了"可以计算"。本章介绍三种最基本的电磁场数值方法。

为什么需要数值方法

回顾前面 20 章的内容，我们推导了大量解析公式——库仑定律、毕奥-萨伐尔定律、平面波解……但这些公式全部建立在特定的几何对称性之上。实际问题往往不具备这种对称性：

  • 一块 PCB 上数百条走线交织，没有对称性可言
  • 人体头部在 MRI 线圈中的电磁场分布极度不规则
  • 智能手机天线紧邻金属外壳、电池、摄像头模组，几何结构极其复杂
  • 汽车发动机舱内的线束耦合问题涉及大量非规则导体

对于这些真实问题，我们的选择只有：做无法接受的近似简化，或者用数值方法求解。后者已经成为现代电磁工程的标准做法。

有限差分时域法（FDTD）

FDTD（Finite-Difference Time-Domain）由 Kane Yee 于 1966 年提出，是最直观的电磁场数值方法。它的核心思想是：直接用有限差分近似麦克斯韦方程组中的时间和空间偏导数。

Yee 网格是 FDTD 的基石。Yee 将计算区域划分为矩形网格，在每个网格单元上交错放置电场分量 E（网格棱边中点）和磁场分量 H（网格面心）。这种交错排布使得法拉第定律和安培定律的旋度可以自然地用中心差分近似。

蛙跳时间推进（Leapfrog）：FDTD 在时间上交替更新 E 和 H：

  1. 在 t = nΔt 时刻，根据旋度方程和 Hn-1/2 更新 En
  2. 在 t = (n+1/2)Δt 时刻，根据旋度方程和 En 更新 Hn+1/2
  3. 重复以上两步，不断推进时间。

稳定性条件（CFL 条件）：时间步长必须满足：

Δt ≤ 1 / (c · √(1/Δx² + 1/Δy² + 1/Δz²))

其中 c 是光速，Δx、Δy、Δz 是空间步长。这个条件本质上是说：在一个时间步内，信息传播不能超过一个网格单元。

吸收边界条件：FDTD 无法模拟无限大自由空间——必须在计算区域边界截断。PML（Perfectly Matched Layer）是最常用的吸收边界，它通过在边界区域添加非物理的吸收材料来吸收外向波而不产生反射。

FDTD 的优势与局限：优点是算法简单、天然支持宽频带（一次仿真得到全频段响应）、适合并行计算（GPU 加速效果极好）。缺点是结构化网格无法精确拟合弯曲表面，且三维 FDTD 的内存需求为 O(N³)，大尺寸问题受限。

一维 FDTD 示例代码

import numpy as np
import matplotlib.pyplot as plt

# 参数设置
N = 200           # 网格数
c = 3e8            # 光速
dx = 0.01          # 空间步长 (m)
dt = dx / c         # 时间步长 (CFL = 1)
STEPS = 500        # 仿真步数

# 场量初始化
Ez = np.zeros(N)
Hy = np.zeros(N)
# 材料参数（真空）
epsilon_r = np.ones(N)
sigma = np.zeros(N)

# 主 FDTD 循环
for t in range(STEPS):
    # 更新 H 场: Hy[n] = Hy[n] + dt/(mu0*dx) * (Ez[n+1] - Ez[n])
    for n in range(N-1):
        Hy[n] += dt / (4 * np.pi * 1e-7 * dx) * (Ez[n+1] - Ez[n])

    # 更新 E 场: Ez[n] = Ez[n] + dt/(epsilon0*dx) * (Hy[n] - Hy[n-1])
    for n in range(1, N):
        Ez[n] += dt / (8.854e-12 * dx) * (Hy[n] - Hy[n-1])

    # 源: 高斯脉冲激励
    Ez[50] += np.exp(-((t - 50) / 20)**2)

# 此时 Ez 和 Hy 包含了时域波形
print("FDTD 仿真完成")
print(f"最后时刻 Ez 的最大值: {np.max(np.abs(Ez)):.4f} V/m")

有限元法（FEM）

有限元法（Finite Element Method, FEM）是工程中最通用的数值方法，不仅适用于电磁场，还适用于结构力学、热传导、流体力学等领域。

FEM 的数学流程：

  1. 变分形式：将偏微分方程转换为等效的变分问题（弱形式）。例如，静电场问题由 Poisson 方程 ∇²φ = −ρ/ε 变为 ∫Ω (ε∇φ·∇v − ρv) dΩ = 0，其中 v 是测试函数。
  2. 离散化：将求解区域划分为有限个单元（四面体、六面体、棱柱等）。
  3. 基函数展开：在单元内用插值函数（基函数）近似场量：φ ≈ Σ Ni · φi
  4. 组装矩阵：将所有单元的贡献组装成全局矩阵方程：K·u = b
  5. 求解：求解线性方程组，得到节点上的场量数值。

FEM 的优势在于非结构化网格——四面体可以精确拟合任何复杂几何形状。Ansys HFSS 和 COMSOL Multiphysics 都采用 FEM 作为核心求解器。对于高频电磁问题，FEM 通常配合矢量基函数（边缘元）使用，以避免伪解（spurious modes）问题。

矩量法（MoM）

矩量法（Method of Moments, MoM）采用完全不同的策略——它基于积分方程而不是微分方程。这意味着 MoM 不需要对自由空间进行网格划分，只对导体或介质表面/体积进行离散。

以理想导体的散射问题为例：入射场 Einc 在导体表面 S 上感应出表面电流 Js，后者产生的散射场满足边界条件——总切向电场在导体表面为零。这导致电场积分方程（EFIE）：

−Einctan = [jkη ∫S Js G dS']tan

其中 G = e−jkR/(4πR) 是自由空间格林函数。MoM 将表面离散为基函数，通过加权残量法构建线性方程组
Z · I = V。

MoM 的核心优势：只有散射体表面需要网格划分，自由空间不需要。这意味着对于"小目标在大空间"的问题（例如天线辐射、雷达散射），MoM 的计算量远小于 FDTD 或 FEM。代价是矩阵 Z 是稠密矩阵（元素非零），存储复杂度 O(N²)。FEKO 是采用 MoM 的代表性商用软件。

如何选择方法？如果你仿真的是物体与自由空间交互（天线、RCS），首选 MoM。如果几何极其复杂且有精细结构（射频连接器、变压器），FEM 最灵活。如果需要宽频带数据或需要看瞬态过程（ESD、雷击），FDTD 最合适。工程实践中，复杂问题往往需要混合方法——例如用 MoM+FEM 处理天线罩内的天线问题。

边界条件与误差控制

数值仿真的可靠性高度依赖边界条件的正确处理。常见陷阱包括：

  • 截断边界太近：FDTD 的 PML 需要足够厚度（通常 8~16 层）才能吸收低频分量。FEM 的一阶吸收边界在距离目标 λ/4 以内时反射过大。
  • 网格不够密：FEM 和 FDTD 要求最大网格尺寸小于 λ/10（一阶精度）或 λ/20（较高精度）。对于精细结构，可能需要自适应网格细化。
  • 色散误差：FDTD 中数值色散会使波速随频率和传播方向变化，大尺寸问题中累积相位误差可观。
  • 迟滞收敛：FEM 迭代求解器的收敛阈值设置过大可能漏掉弱耦合效应。
仿真中的常见错误：有工程师用 FDTD 仿真手机天线时忘记了设置 PML 吸收边界，结果远场方向图中出现了明显的伪反射波纹——这些波纹并不是真实的辐射特征，而是计算区域边界将外向波反射回来的伪迹。正确的边界条件设置是数值仿真中最重要的基础工作，也是最有经验的仿真工程师反复检查的地方。
