央行的三大职能——发行的银行、银行的银行、政府的银行

如果说商业银行是金融体系的"毛细血管"，那中央银行就是"心脏"。央行不跟普通老百姓直接打交道，但它的一举一动——加息还是降息、放水还是收水——深刻影响着每一个人的钱袋子。

中央银行的三大经典职能，是理解央行一切行为的钥匙：

第一，发行的银行。央行拥有货币发行的垄断权。在中国，人民币的发行权属于中国人民银行——这意味着市场上流通的所有现金（M0）都是央行"印"出来的（当然现在更多是"数字发行"）。这是央行最根本的职能，也是其他职能的基础。

第二，银行的银行。央行是商业银行的"最后靠山"。商业银行在央行开设准备金账户，办理清算结算。当商业银行之间资金头寸紧张时，央行可以提供贷款——但前提是商业银行要提供合格的抵押品。央行还负责监管商业银行的经营行为，确保它们不会冒险过头。

第三，政府的银行。央行代理国库收支，管理国家外汇储备，制定和执行货币政策。央行还是政府债券的主要承销商之一。但这里有一个微妙的关系——央行是"政府的银行"，不等于央行是"政府的下属"。央行的独立性，正是下一节要讨论的核心问题。

💡 趣味历史： 世界上最早的中央银行是瑞典央行（Sveriges Riksbank），成立于1668年。但真正意义上的"现代央行鼻祖"是1694年成立的英格兰银行。有趣的是，英格兰银行最初是一家私人商业银行，后来才逐渐获得了货币发行的垄断权和"最后贷款人"的角色。直到1946年被国有化之前，它居然是一家"私人央行"——股东们每年还能分到股息！

央行的资产负债表——读懂央行"家底"

央行也有资产负债表。读懂央行的资产负债表，就像读懂一个人的体检报告——你能看出央行的政策意图、操作方向和面临的压力。咱们来看看简化版的结构：

// 中国人民银行资产负债表结构（简化）
// 基于2025年末公开数据

const pbocBalanceSheet = {
  // 资产端
  assets: {
    "国外资产（外汇占款为主）": "22.3万亿",
    "对存款性公司债权（MLF/PSL等）": "16.8万亿",
    "对政府债权": "1.5万亿",
    "其他资产": "2.4万亿",
    // 资产总计
    total: "43.0万亿"
  },
  // 负债端
  liabilities: {
    "储备货币（M0+准备金）": "32.5万亿",
    "政府存款": "4.2万亿",
    "发行债券（央票）": "1.8万亿",
    "其他负债": "4.5万亿",
    // 负债总计
    total: "43.0万亿"
  }
};

// 资产 = 负债 + 所有者权益
// 央行资产负债表的核心等式
console.log(`资产总计: ${pbocBalanceSheet.assets.total}`);
console.log(`负债总计: ${pbocBalanceSheet.liabilities.total}`);
// 净值为央行资本金，通常很小

看懂这张表了吗？有几个关键点值得一提：

  • 外汇占款（国外资产）曾经是央行最大的资产项，占比一度超过80%。这是因为过去中国大量积累外汇储备，央行每买入1美元就要吐出等值的人民币。但近些年外汇占款占比下降，央行通过MLF、PSL等工具主动投放流动性的比重上升。
  • 储备货币（负债端）是央行最大的负债，包括流通中的现金（M0）和商业银行在央行的准备金存款（包括法定准备金和超额准备金）。
  • 政府存款——政府的钱也存在央行账户上。当政府增加支出时，这部分存款减少，实际上是在向市场注入流动性。

理解央行资产负债表的关键在于：央行可以通过增减资产来调节负债，从而控制市场上的货币供应量。比如央行买国债（资产增加），对应的是商业银行在央行的准备金增加（负债增加），市场上的钱就多了——这就是量化宽松（QE）的底层逻辑。

央行独立性问题

央行应不应该独立于政府？这个问题在学术界和政策圈争论了几十年。我们来看看全球三大央行的独立性差异：

维度 美联储（Fed） 中国人民银行（PBoC） 欧央行（ECB）
独立性程度 高度独立 相对独立（受国务院领导） 极高独立
目标设定 法律授权：最大化就业+稳定物价（双重使命） 国务院确定，央行执行 条约规定：物价稳定优先（单一使命）

财政融资 禁止直接购买国债（1913年《联邦储备法》禁止） 《中国人民银行法》禁止直接认购国债 禁止直接购买成员国国债

央行独立性的核心争论在于：如果央行听命于政府，政府可能会在选举前要求央行降息刺激经济，造成"政治性通胀周期"。经验研究表明，独立性越高的央行，其通胀控制效果越好。但独立性也不是绝对的——在金融危机或疫情期间，央行需要与财政政策协调配合，完全独立反而可能成为障碍。

🔑 重要辨析： 央行的独立性分为两种——"目标独立性"（能否自己决定政策目标）和"工具独立性"（能否自己选择实现目标的手段）。中国人民银行的独立性属于"工具独立"而非"目标独立"——政策目标由国务院确定，但央行自主选择使用什么工具来达成目标。2025年《中国人民银行法》修订讨论中，一个核心议题就是如何在法律层面进一步明确央行的职责和权限边界。

最后贷款人角色——白芝浩原则

1873年，英国记者兼经济学家白芝浩（Walter Bagehot）出版了一本经典著作《伦巴第街》，提出了至今仍然被奉为圭臬的"最后贷款人"原则：

当银行体系出现流动性危机时，央行应该——"自由放贷、惩罚利率、优质抵押"。

翻译一下：当一家银行暂时缺钱但有优质抵押品时，央行应该毫不犹豫地借钱给它。但有两个条件：利息要足够高（惩罚利率，让银行吸取教训），抵押品要足够优质（防止央行承担损失）。白芝浩的本意是：央行出手不是为了救傻子，而是为了防止单体流动性危机演变成系统性支付危机。

这套原则指导了全球央行过去150年的危机应对。但2008年金融危机之后，央行们的做法已经远远超越了白芝浩原则——它们不仅对商业银行提供贷款，还直接购买债券（量化宽松）、甚至向企业提供贷款。2020年新冠疫情中，美联储更是直接下场购买公司债和高收益债ETF。这些操作被称为"最后做市商"——央行已经不满足于做"贷款人"，而是直接做"市场参与者"。

⚠️ 踩坑故事：2013年"钱荒"——央行不注资导致隔夜SHIBOR飙升至13%

2013年6月，中国金融市场经历了一场教科书级的流动性危机，被称为"钱荒"。2013年6月20日，银行间市场隔夜拆借利率（SHIBOR）飙升至13.44%，创下历史最高纪录！这意味着什么？银行之间借钱要付13.44%的年化利息——比很多贷款的利率都高。有些银行甚至愿意出30%的年息来借隔夜资金，但依然借不到。

事情的起因：2013年上半年，银行通过"影子银行"（表外理财、同业业务）大量扩张信用，导致资金"脱实向虚"。央行决心"敲打"一下这些激进的银行，于是连续几周没有做逆回购操作——也就是不给市场注入流动性。央行想传递的信号是："你们太放纵了，该收敛了。"

但事情超出了央行的预期。流动性紧张迅速演变成恐慌。6月20日那天，某大型国有银行甚至出现了"违约"——到期无法支付给对手方，这在银行间市场是极其罕见的。下午两点半左右，央行不得不紧急出手，向部分银行提供了流动性支持，市场利率才逐渐回落。

事后看来，央行那次"压力测试"让整个市场深刻认识到了流动性的重要性。2013年之后，银行开始重视流动性管理，央行也建立了更完善的常备借贷便利（SLF）和中期借贷便利（MLF）工具，为银行提供更稳定的流动性支持通道。但钱荒的教训一直留在市场参与者的记忆中——在金融市场中，流动性不是万能的，但没有流动性是万万不能的。

钱荒的故事说明了一个核心道理：央行的"最后贷款人"角色是一把双刃剑。出手太慢，市场恐慌蔓延；出手太快，银行会产生依赖（道德风险）。央行要在"防范道德风险"和"防止系统性风险"之间走钢丝——这可能是央行行长最难的工作。

理解了央行的角色和操作逻辑，咱们就能更好地理解它调控市场的具体工具了。下一章进入金融市场中最重要的板块——货币市场。这是央行进行日常流动性管理的主战场，也是所有金融机构每天都要打交道的市场。下一章：货币市场。
