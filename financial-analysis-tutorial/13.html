第13章 技术指标详解

大家好，我是你们的讲师。上一章我们学习了技术分析的基础——K线形态、趋势线、成交量。这些都是"定性"的工具——你需要用眼睛去看、用经验去判断。但人类的判断难免有主观偏差：同样的图表，两个人可能画出完全不同的趋势线。这就是为什么在实战中，绝大多数技术分析者都会搭配量化的技术指标——用数学公式来消除主观性，让买卖信号变得清晰可执行。

技术指标从功能上可以分为三大类：趋势类（告诉你方向）、动量类（告诉你力度）、波动类（告诉你空间）。成熟的交易者通常会从每类中选择一两个指标组合使用，而不是在几十个指标中不停地切换。记住一个重要的原则：指标不是越多越好，关键是理解你使用的指标在测量什么。

13.1 趋势类指标

趋势类指标的核心功能是识别和确认趋势的方向，并在趋势可能发生转变时给出信号。

移动平均线（MA / SMA / EMA）

移动平均线是最基础、最常用的趋势指标。它平滑了价格数据，让趋势变得一目了然。

  • SMA（简单移动平均）——过去N个收盘价的算术平均。例如20日SMA = 过去20天收盘价之和/20。
  • EMA（指数移动平均）——加权平均，近期的价格权重更高，对价格变化反应更灵敏。

实践中常用的均线组合：

均线周期 常用名称 用途
5日 / 10日 短期均线 短线交易者的参考线，反映近期趋势
20日 / 30日 中期均线 月线级别趋势，常用于判断波段方向
60日 / 120日 长期均线（季度线/半年线） 中期趋势的生命线，机构投资者重要参考
250日 年线 "牛熊分界线"——站上年线算牛市，跌破算熊市

均线的经典交易信号：

  • 金叉（Golden Cross）——短期均线上穿长期均线，趋势转向看涨的信号。
  • 死叉（Death Cross）——短期均线下穿长期均线，趋势转向看跌的信号。
  • 价格回踩均线不破——上升趋势中，价格回调到某条重要均线（比如20日/60日）后企稳回升，是加仓点。
踩坑警示：移动平均线有严重的滞后性（Lagging）。趋势走得越久，均线信号越可靠——但当你看到金叉时，可能已经涨了一大段了。反过来，在震荡行情中均线会反复产生"假信号"——今天金叉明天死叉，专打追涨杀跌的交易者。所以均线最好在有明确趋势的市场中使用，在横盘震荡中尽量少参考。

MACD（指数平滑异同移动平均线）

MACD是技术分析中应用最广泛的指标之一，由杰拉尔德·阿佩尔（Gerald Appel）在1970年代发明。它由三部分组成：

  • DIF线（快线）——12日EMA减去26日EMA，反映短期和长期均线的"距离"。
  • DEA线（慢线）——DIF的9日EMA，是DIF的平滑版本。
  • MACD柱（柱状线）——DIF减去DEA的差值 * 2，直观显示快慢线的距离和方向变化。

MACD的核心信号：

  • DIF上穿DEA——买入信号（相当于均线的"金叉"）
  • DIF下穿DEA——卖出信号（相当于均线的"死叉"）
  • 顶背离——价格创新高但MACD未创新高 → 趋势可能见顶
  • 底背离——价格创新低但MACD未创新低 → 趋势可能见底
实操建议：MACD背离信号在日线和周线级别的可靠性远高于小时线。建议大家先从日线MACD开始训练背离识别——当价格和MACD柱线（或DIF线）出现方向不一致时，在图表上用箭头标注出来，然后回溯验证。大约练习30-50张图表后，你就能建立对背离的"直觉"。

13.2 动量类指标

动量指标的核心思想是：趋势的力度可以通过速度的变化来测量。好比一辆车在加速时说明动力充足，在减速时说明动力在衰竭。

RSI（相对强弱指标）

RSI由韦尔斯·怀尔德（Welles Wilder）在1978年提出，衡量的是一段时期内价格上涨幅度占总波动幅度的比例。默认参数是14天：

RSI = 100 - 100 / (1 + 平均上涨幅度 / 平均下跌幅度)

RSI的取值范围是0-100。传统的解读规则：

  • RSI > 70——超买（Overbought），价格可能回调
  • RSI < 30——超卖（Oversold），价格可能反弹
  • RSI = 50——多空均衡，没有明显方向

但这里有一个非常重要的进阶用法：在强势上涨趋势中，RSI可以长时间停留在70以上——此时卖空是极其危险的（这就是"强者恒强"）。同样，在强势下跌趋势中，RSI可以在30以下持续。所以更可靠的做法是找RSI的背离而非绝对值。

KDJ随机指标

KDJ（Stochastic Oscillator）与RSI类似但也测量价格位置，但计算方法有所不同。它比较当前收盘价与过去N天（通常9天）的价格区间，判断价格处于区间的什么位置。K线是快线，D线是K线的移动平均，J线是K和D的差值放大。

KDJ的核心信号：

  • K > 80——超买；K < 20——超卖
  • K线上穿D线——买入信号（在低位时更可靠）
  • K线下穿D线——卖出信号（在高位时更可靠）

13.3 波动类指标

波动类指标衡量的不是价格的方向，而是价格波动的大小。它们帮助交易者判断当前市场是"活跃"还是"平静"，从而选择合适的策略。

布林带（Bollinger Bands）

由约翰·布林格（John Bollinger）在1980年代发明，布林带包含三条线：

  • 中轨——20日SMA（基准线）
  • 上轨——中轨 + 2倍标准差
  • 下轨——中轨 - 2倍标准差

布林带的工作原理基于一个统计事实：如果价格服从正态分布，约95%的价格数据会落在均值±2个标准差之内。因此当价格触及上轨或下轨时，统计学上属于"极端"情况。

布林带的核心用法：

  • 带宽收缩（Squeeze）——上下轨收窄至极限，暗示市场即将爆发大行情（但无法预知方向）。这是布林带最有价值的信号之一。
  • 价格沿上轨运行——强势上涨趋势，此时"卖出"是不明智的。
  • 价格沿下轨运行——强势下跌趋势。
  • 价格触及上轨后回调——可能的卖出点（结合其他指标确认）。

ATR（平均真实波幅）

ATR由怀尔德（与RSI同一作者）发明，用来衡量市场波动的绝对幅度。它考虑了跳空缺口的因素，计算的是N天（通常14天）内的平均波动范围。ATR值越高说明市场波动越大。

ATR的实际用途：

  • 设定止损位——比如设置止损为"入场价 - 1.5倍ATR"，这样止损会随着波动率自动调整。
  • 头寸规模管理——ATR大的时候减少头寸，ATR小的时候增加头寸，保持每笔交易的风险金额一致。

13.4 多重时间框架分析

这是从入门到进阶必须跨越的一道门槛。新手通常只盯着一个时间框架（比如只看日线），但成熟的交易者一定会同时观察三个时间框架——大周期定方向，小周期找买点。

具体做法（以日线交易为例）：

  • 周线（大周期）——判断宏观趋势：是上升趋势、下降趋势还是震荡？如果周线是上升趋势，你只做多或中性，避免做空。
  • 日线（中周期）——确定交易计划：在周线方向的基础上，等待日线级别的回调或突破信号。
  • 60分钟/30分钟（小周期）——寻找精确入场点：在日线信号出现后，用小周期K线确认具体买/卖点位。

如果三个时间框架的信号方向一致（比如周线上升、日线回调结束、60分钟出现反转信号），这就是"共振"——胜率最高的交易机会。如果信号矛盾（周线上升但日线出现顶背离），你应该保持观望或者至少降低仓位。

下表总结了三类技术指标的对比：

类别 代表指标 测量对象 最佳使用场景 局限性
趋势类 MA, MACD 趋势方向和强度 趋势明显市场（单边行情） 在震荡市中频繁假信号
动量类 RSI, KDJ 超买超卖状态 识别趋势衰竭和反转 强趋势中会长时间钝化
波动类 布林带, ATR 波动幅度和范围 判断爆发时机和设定止损 不提供方向信息

下面是用Python+TA-Lib计算常用技术指标的示例：

import akshare as ak
import pandas as pd
import talib
import numpy as np

# 获取贵州茅台日线数据
df = ak.stock_zh_a_hist(
    symbol="600519", period="daily",
    start_date="20240101", end_date="20241231"
)

# 准备数据（TA-Lib要求numpy数组）
close = df['收盘'].values
high = df['最高'].values
low = df['最低'].values
volume = df['成交量'].values

# 计算MACD
macd, macd_signal, macd_hist = talib.MACD(
    close, fastperiod=12, slowperiod=26, signalperiod=9
)

# 计算RSI(14)
rsi = talib.RSI(close, timeperiod=14)

# 计算布林带(20,2)
upper, middle, lower = talib.BBANDS(
    close, timeperiod=20, nbdevup=2, nbdevdn=2
)

# 加入DataFrame
df['MACD'] = macd
df['MACD_Signal'] = macd_signal
df['MACD_Hist'] = macd_hist
df['RSI'] = rsi
df['BB_Upper'] = upper
df['BB_Middle'] = middle
df['BB_Lower'] = lower

# 查看最新5条数据
print(df[['日期', '收盘', 'MACD', 'RSI', 'BB_Upper', 'BB_Lower']].tail(5))

# 简单的信号生成：RSI < 30 且 收盘价接近布林下轨
df['Signal'] = np.where(
    (df['RSI'] < 30) & (df['收盘'] <= df['BB_Lower'] * 1.02),
    '买入关注', ''
)
print("\n买入信号：")
print(df[df['Signal'] == '买入关注'][['日期', '收盘', 'RSI']])

这段代码可以帮你快速生成任意A股股票的MACD、RSI和布林带数据。注意，TA-Lib的安装可能需要一些系统依赖（在Linux上可以通过apt安装ta-lib库），但一旦装好，它的计算速度和稳定性是手工实现无法比拟的。

核心洞见：技术指标是对价格数据的数学变形——它们不"预测"未来，只是以不同的"视角"展示已经发生的价格行为。当你看到MACD金叉时，它只是在告诉你"短期均线跑到了长期均线上面"——至于这之后价格是涨是跌，取决于未来的市场行为。所以使用技术指标的正确心态是：把它当作决策的工具，而不是真理的化身。

至此，我们完成了技术分析的初步学习——从K线形态到趋势线，从MACD到RSI到布林带。这套工具体系在短线交易中非常实用。但从下一章开始，我们将会把视野拉高——从个股的走势图，拉升到整个经济的基本面。是的，我们将进入宏观经济分析的世界——你会发现，很多时候个股的涨跌，根源不在公司本身，而在于宏观经济的潮汐。
