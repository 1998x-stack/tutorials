开场：对称性的力量

上一章我们用毕奥-萨伐尔定律计算了各种电流构型的磁场。它的通用性无可置疑，但每次都要做矢量积分，未免有些笨重。如果系统具有高度对称性——比如无限长导线、无限长螺线管、环形螺线管——有没有一种更简洁的方法？

答案是肯定的。正如高斯定理之于库仑定律的关系，安培环路定理（Ampere's Circuital Law）就是毕奥-萨伐尔定律的"对称性捷径"。如果说毕奥-萨伐尔定律是"积分算力士"，那安培环路定理就是"对称性剑客"。

在本章中，我们还将介绍磁高斯定理——一个看似简单但意义深远的事实：自然界中不存在磁单极子。

安培环路定理

安培环路定理表述如下：

∮C B · dl = μ₀ Ienclosed

也就是说，磁感应强度 B 沿任意闭合回路 C 的环量，等于穿过该回路所围曲面的总电流乘以真空磁导率 μ₀。

这一定理可以从毕奥-萨伐尔定律严格推导出来。其核心思想是：磁场是有旋无源场，电流是磁场的"涡旋源"。环量积分（∮B·dl）正是衡量这种旋涡效应的积分形式。

使用安培环路定理求解磁场需要满足对称性条件：在选取的积分回路上，B 的大小必须是常数（或可以分段处理），且 B 的方向必须与 dl 平行（或可以明确确定）。典型的可解系统包括：

  • 无限长直导线（柱对称）
  • 无限长圆柱导体（均匀载流）
  • 无限长螺线管（平移对称）
  • 环形螺线管（环形对称）
  • 无限大电流平面（面对称）

安培环路定理应用

环形螺线管（Toroid）

环形螺线管是一个绕在环形骨架上的密绕线圈，总匝数为 N，载有电流 I。由对称性可知，其内部磁感线是同心圆（沿环面方向）：

B(r) · (2π r) = μ₀ N I

因此：B(r) = μ₀ N I / (2π r)（仅限环内，r 为到环中心的径向距离）

注意：环形螺线管的磁场在环外为零——因为环外任何闭合回路包围的净电流都为零。这一性质使环形螺线管成为理想的磁屏蔽元件。

无限长螺线管

由对称性可知，无限长螺线管内部 B 沿轴向且均匀，外部 B = 0。取矩形回路，两条边在管内、两条边跨过管壁：

B · L = μ₀ (nL) I

因此：B内部 = μ₀ n I——与位置无关，完全均匀。

💡 安培环路定理的回路选择技巧
选用安培环路定理时，积分回路的选择决定了计算的难度。基本原则：
1. 回路应沿着 B 的方向（使 B·dl = B dl）
2. 回路上 B 的大小应是常数（可以提出积分号）
3. 如果 B 方向未知，选择使 B·dl = 0 的部分规避未知分量
记住：回路的选取不影响最终结果，但聪明的选择可以极大简化计算。

同轴电缆

同轴电缆是最经典的安培环路定理应用案例。它由内导体（半径 a）和外导体（内半径 b，外半径 c）组成，电流 I 从内导体流出、从外导体流回。

分区域讨论：

  • r < a（内导体内部）：B(r) = (μ₀ I r) / (2π a²)——随 r 线性增大
  • a < r < b（内导体与外壳之间）：B(r) = μ₀ I / (2π r)——随 1/r 衰减
  • b < r < c（外导体内部）：B(r) = (μ₀ I / (2π r)) · (c² - r²)/(c² - b²)——逐渐减为零
  • r > c（电缆外部）：B(r) = 0——内外电流完全抵消
⚠ 常见犯错：安培环路中的 Ienclosed
这是学生最容易出错的地方。安培环路定理中的 Ienclosed 是穿过以回路为边界的任意曲面的净电流，不是回路附近的全部电流。
关键规则：电流在回路内部被计入，在外部不被计入。
以同轴电缆为例，取 r > c 的回路：内导体的电流 +I 被计入，外导体的电流 −I 也被计入（因为方向相反，回路同样包围了外导体），净电流为零，所以 B=0。初学者常常只算了内导体电流就认为外部有磁场——这是错误的！

磁高斯定理

磁高斯定理是电磁学中最简洁也最深远的定理之一：

∮S B · dS = 0  或微分形式  ∇ · B = 0

这个公式说：穿过任意闭合曲面的磁通量总为零。对比电场的高斯定理 ∮S E · dS = Qenclosed/ε₀——电场可以有净通量（因为存在正负电荷），但磁场的净通量永远是零。

这意味着什么？物理上极其深刻：不存在磁单极子（magnetic monopole）。

电场线从正电荷出发，终止于负电荷——电场有"源"和"汇"。磁场线（即 B 线）却没有起点也没有终点，它们永远是闭合的曲线。从一个闭合曲面流入的磁通量等于流出的磁通量，净通量为零。

性质 静电场 静磁场
基本源 电荷（单极） 电流（偶极 / 闭合环）
是否有单极子 是（正电荷、负电荷） 否（实验尚未发现）
散度 ∇·E = ρ/ε₀ ∇·B = 0
旋度（静态） ∇×E = 0 ∇×B = μ₀J
场线形态 从正到负（有始有终） 闭合曲线（无始无终）
高斯通量 ∮E·dS = Q/ε₀ ∮B·dS = 0
环路积分（保守性） ∮E·dl = 0（保守场） ∮B·dl = μ₀Ienc（非保守）
🔑 核心洞察
"不存在磁单极子"是一个实验事实，而不是逻辑必然。狄拉克（1931年）指出，如果存在磁单极子，它就能解释电荷的量子化——为什么所有带电粒子的电荷量都是基本电荷 e 的整数倍。
至今，物理学家们在月岩、陨石、加速器碰撞产物中搜索磁单极子，仍无果。但理论物理从未放弃——大统一理论（GUT）和超弦理论都预言了磁单极子的存在。如果某一天被证实，整个电磁学教科书都要重写！

磁矢势 A

由磁高斯定理 ∇·B = 0，我们知道 B 是一个无散场。在向量场论中，任何无散场都可以表示为某个矢量场的旋度：

B = ∇ × A

其中 A 称为磁矢势（magnetic vector potential）。

A 与标势 φ 的类比：

  • 电场：E = −∇φ（无旋场用标势表示）
  • 磁场：B = ∇ × A（无散场用矢势表示）

引入磁矢势有什么好处？

  • 自动满足 ∇·B = 0：因为 ∇·(∇×A) ≡ 0，这是向量恒等式。
  • 简化了磁场计算：对于某些问题，计算 A 比直接计算 B 更容易。例如对于载流回路：A(r) = (μ₀ I / 4π) ∮ dl' / |r − r'|。再取旋度即得 B。
  • 量子力学的关键角色：在量子力学中，AB 效应（Aharonov-Bohm Effect）证明即使 B=0 的区域，A 也会影响电子波的相位——这说明 A 具有比 B 更根本的物理实在性。
💡 磁矢势的规范自由度
如果 A 是某个磁场的矢势，那么 A' = A + ∇Λ（其中 Λ 是任意标量函数）描述的是同一个 B——因为 ∇×(∇Λ) ≡ 0。
这种自由度称为规范自由度（gauge freedom）。在处理具体问题时，我们会施加一个额外的"规范条件"来固定 A，最常见的两种是：
— 库仑规范：∇·A = 0（便于静电-静磁场分离）
— 洛伦兹规范：∇·A + (1/c²)∂φ/∂t = 0（保持相对论协变性，为电磁波做准备）

Python 计算：同轴电缆的磁场分布

import numpy as np
import matplotlib.pyplot as plt

mu0 = 4*np.pi * 1e-7
I = 1.0          # 总电流 (A)
a = 1e-3         # 内导体半径 (m)
b = 3e-3         # 外导体内半径 (m)
c = 4e-3         # 外导体外半径 (m)

r_vals = np.linspace(0, 5e-3, 1000)
B_vals = np.zeros_like(r_vals)

for i, r in enumerate(r_vals):
    if r < a:
        # 内导体内部：电流均匀分布
        B_vals[i] = mu0 * I * r / (2 * np.pi * a**2)
    elif r < b:
        # 内导体与外壳之间
        B_vals[i] = mu0 * I / (2 * np.pi * r)
    elif r < c:
        # 外导体内部：电流反向，均匀分布
        I_enclosed = I - I * (r**2 - b**2) / (c**2 - b**2)
        B_vals[i] = mu0 * I_enclosed / (2 * np.pi * r)
    else:
        # 电缆外部：净电流为零
        B_vals[i] = 0.0

plt.figure(figsize=(8, 5))
plt.plot(r_vals * 1000, B_vals * 1000)  # 转换为 mm 和 mT
plt.axvline(a*1000, color='gray', ls='--', label=f'内导体半径 a={a*1000:.1f}mm')
plt.axvline(b*1000, color='gray', ls=':', label=f'外导体内半径 b={b*1000:.1f}mm')
plt.axvline(c*1000, color='gray', ls='--', label=f'外导体外半径 c={c*1000:.1f}mm')
plt.xlabel('径向距离 r (mm)')
 plt.ylabel('磁感应强度 B (mT)')
plt.title('同轴电缆的磁场分布')
plt.legend()
plt.grid(alpha=0.3)
plt.show()

从计算结果可以清晰地看到：磁场只存在于导体之间的区域（a < r < b）以及导体内部，外部完全为零。这正是同轴电缆能够实现电磁屏蔽的根源——外部设备不受电缆信号的干扰，电缆内部信号也不辐射到外部。

本章小结

  • 安培环路定理：∮B·dl = μ₀ Ienclosed，是高对称性系统求 B 的高效工具。
  • 安培环路定理只能用于 B 的对称性足够高的系统——此时选取合适的积分回路将积分简化为乘法。
  • 磁高斯定理：∮B·dS = 0 → ∇·B = 0，宣告自然界没有磁单极子，磁感线永远闭合。
  • 磁矢势 A：B = ∇×A，自动满足 ∇·B=0，在量子力学中具有更根本的意义。
