第21章 固定收益分析

大家好，我是你们的讲师。上一章我们讨论了衍生品——那些让风险可以被独立定价和交易的金融工具。但这又引出一个问题：衍生品的定价基础（无风险利率、折现因子）是从哪里来的？答案来自本章要讲的内容——固定收益分析（Fixed Income Analysis）。

固定收益市场是全球资本市场中规模最大、影响最深远但同时也最容易被个人投资者忽略的市场。截至2025年底，中国债券市场存量规模已经突破180万亿元人民币，是A股总市值的近两倍。全球债券市场规模更是超过130万亿美元——比全球股票市场还要大。然而，大多数个人投资者对债券的理解仍然停留在"利息比存款高一点"的层面。

固定收益分析看似枯燥——票面利率、到期收益率、久期、信用利差……这些术语听起来像是算命的"术语"。但当你真正理解之后，你会发现债券市场的逻辑远比股票市场更加严谨和可量化。债券的价格有精确的数学公式（不像股票，你说它值多少钱可以有不同的看法）；风险有明确的度量指标（久期告诉你利率每变动1%，债券价格变动多少）。这种"精确性"让固定收益分析成为整个金融分析体系中最具有"工程感"的分支。

21.1 债券定价基础

债券的本质就是一张"借条"（IOU）。发行人（政府或企业）向你借钱，承诺在未来某个时间（到期日）偿还本金，并在此期间定期支付利息（票息）。但债券有比普通借条复杂得多的细节。

核心概念

  • 票面利率（Coupon Rate）——债券面值上印着的年化利率。比如一张面值100元、票面利率3%的债券，每年支付3元利息。
  • 面值（Face Value / Par Value）——到期时发行人要偿还的金额。中国债券通常为100元。
  • 到期收益率（Yield to Maturity, YTM）——如果你按当前价格买入并持有到期，获得的年化内部收益率。它是债券价格"倒推"出来的综合收益率。
  • 净价与全价（Clean Price vs Dirty Price）——中国债券市场上报价通常使用净价（不含应计利息），但实际交易结算时使用全价（净价+应计利息）。
  • 应计利息（Accrued Interest）——自上次付息日以来累计但尚未支付的利息。如果你在两次付息日之间买入债券，你需要补偿卖方这段时间应得的利息。

债券定价公式

债券价格等于未来所有现金流（票息+本金）按YTM折现的现值之和：

P = Σ [C / (1 + r)ᵗ] + F / (1 + r)ⁿ

其中，P是价格，C是每期票息，r是YTM，F是面值，t是期数，n是总期数。

从这个公式可以看出一个关键关系：债券价格与YTM呈反向关系——收益率上升，价格下跌；收益率下降，价格上涨。这个关系是固定收益分析的"牛顿定律"。

中国市场实务：在中国债券市场，YTM的计算通常采用"实际/365"的计息基准（银行间市场）或"实际/360"（交易所市场）。如果你做跨市场分析，一定要注意计息基准的差异——在一个市场算出来的YTM，可能在另一个市场被算错。

21.2 收益率曲线分析

收益率曲线（Yield Curve）是将同一信用等级的债券（通常是国债）的不同期限收益率连接起来形成的曲线。它是固定收益市场最重要的"基准"——不仅决定了所有其他债券的定价锚，而且是宏观经济的重要预测指标。

三种形态

  • 正常曲线（Normal / Upward Sloping）——短期收益率低，长期收益率高。反映了"期限越长，不确定性越大，因此需要更高的收益率"。这是最常见的形态。
  • 平坦曲线（Flat）——长短端收益率差异很小。通常出现在货币政策紧缩周期末期或经济转型期。
  • 倒挂曲线（Inverted / Downward Sloping）——短期收益率高于长期收益率。这是最引人关注的形态——在历史上，美国国债收益率曲线倒挂几乎总是领先于经济衰退（recession），是市场公认的"衰退预警信号"。

骑乘策略

骑乘收益率曲线（Riding the Yield Curve）是一种经典的债券投资策略。它的逻辑是：在正常形态的收益率曲线下，随着时间推移，债券的剩余期限缩短，其收益率会沿着收益率曲线"下滑"到更低的水平，从而带来资本利得（收益率下降→价格上涨）。

举例来说，如果3年期国债收益率为2.5%，2年期为2.0%，那么买入3年期并持有一年，不但获得了票息收益，还因为收益率从2.5%下降到2.0%而获得了额外的资本利得。这就是"骑乘"的收益来源。

平坦化与陡峭化交易

专业投资者会基于对收益率曲线形态变化的判断进行交易：

  • 陡峭化交易（Steepener）——赌长端收益率上升快于（或下降慢于）短端。例如，买入短期国债同时卖空长期国债。
  • 平坦化交易（Flattener）——赌长端收益率下降快于短端，或者短端上升快于长端。
核心洞见：收益率曲线的形态变化背后是三种力量的博弈：对未来利率的预期（expectations theory）、流动性溢价（liquidity preference）和市场分割效应（market segmentation）。没有哪一个理论能完全解释曲线形态——但在实践中，如果你能判断出当前曲线主要受哪种力量驱动，你就找到了交易的优势所在。

21.3 久期与凸性

久期和凸性是固定收益分析中最重要的两个风险度量指标。它们是理解"利率变动如何影响债券价格"的核心工具。

麦考利久期

麦考利久期（Macaulay Duration）以债券所有现金流回收的"加权平均时间"来度量——权重是每笔现金流现值占总现值的比例。举例来说，一张面值100元、票息3%、3年到期的债券，其麦考利久期大约为2.85年——意味着你收回全部现金流（本金和利息）的平均时间大约是2.85年。

修正久期

修正久期（Modified Duration）是麦考利久期除以（1+YTM）。它的直接含义是：YTM每变动1个百分点，债券价格变动的百分比。修正久期为5的债券，如果收益率上升1%，理论上价格下跌约5%。

有效久期

有效久期（Effective Duration）适用于含权债券（如可转债、可赎回债券）。它通过小幅上调和下调收益率，分别计算债券价格的变化来近似久期，因为含权债券的现金流会随着利率变化而变化，普通久期公式不再适用。

凸性：久期的"非线性"修正

久期假设债券价格与收益率的变化呈线性关系，但实际上这种关系是凸性的（convex）——即收益率下降时价格上升的幅度，大于收益率上升时价格下跌的幅度。这就是凸性效应（Convexity）。

重要理解：凸性是债券持有人喜欢的性质——它意味着"跌得少，涨得多"。但天下没有免费的午餐：高凸性的债券通常收益率更低（价格更高），因为市场已经在为这种"不对称保护"定价了。所以，你的选择是：要么接受低收益率换取凸性保护，要么放弃凸性追求更高收益——这是一场权衡。

下面是用Python计算债券价格、YTM和久期的示例：

import numpy as np
from scipy.optimize import newton

def bond_price(face, coupon, rate, years, freq=2):
    """计算债券价格"""
    periods = int(years * freq)
    coupon_pmt = face * coupon / freq
    rates = [(rate / freq) + 1 for _ in range(periods)]
    cf = [coupon_pmt] * periods
    cf[-1] += face  # 最后一期加上本金
    pv = sum(c / (1 + rate/freq)**(t+1) for t, c in enumerate(cf))
    return pv

def ytm_calc(price, face, coupon, years, freq=2):
    """用牛顿法反解YTM"""
    def ytm_func(r):
        return bond_price(face, coupon, r, years, freq) - price
    return newton(ytm_func, 0.03)  # 初始猜测3%

def macaulay_duration(face, coupon, ytm, years, freq=2):
    """计算麦考利久期"""
    periods = int(years * freq)
    coupon_pmt = face * coupon / freq
    cf = [coupon_pmt] * periods
    cf[-1] += face
    disc = [(1 + ytm/freq)**(t+1) for t in range(periods)]
    pv = sum(c / d for c, d in zip(cf, disc))
    weights = [(c / d) / pv for c, d in zip(cf, disc)]
    duration = sum((t+1)/freq * w for t, w in enumerate(weights))
    return duration

# 示例：面值100元，票息3.5%，YTM 3.2%，5年期，年付息
price = bond_price(100, 0.035, 0.032, 5, freq=1)
dur = macaulay_duration(100, 0.035, 0.032, 5, freq=1)
mod_dur = dur / (1 + 0.032)
print(f"价格: {price:.4f}")
print(f"麦考利久期: {dur:.4f} 年")
print(f"修正久期: {mod_dur:.4f}")
print(f"收益率上升1%时价格近似变化: -{mod_dur:.2f}%")

21.4 信用分析框架

国债通常被认为是"无风险"资产，但企业债、地方债等信用债则存在违约风险。信用分析的核心任务就是评估和定价这种风险。

中国债券市场主要品种

品种 发行人 信用评级 流动性 典型期限
国债 财政部 最高（无风险） 极好 1~30年
地方政府债 省级政府 接近国债 较好 3~30年
政策性银行债 国开行/农发行/进出口行 接近国债 好 1~10年
企业债/公司债 各类企业 AAA到C 差异大 1~10年
中期票据 非金融企业 AAA到C 中等 2~5年
可转债 上市公司 取决于发行人 较好 通常6年
资产支持证券 SPV 取决于基础资产 差 1~10年

信用利差

信用利差（Credit Spread）是指信用债收益率与同期限国债收益率的差值。它是对投资者承担的违约风险和流动性风险的补偿。信用利差的变化受三方面因素影响：

  • 宏观因素——经济衰退时，信用利差整体扩大（flight to quality）；经济繁荣时收窄。
  • 行业因素——行业面临政策调整或技术颠覆时（如房地产调控、煤炭去产能），该行业债券的信用利差会系统性走阔。
  • 个体因素——公司基本面恶化（盈利能力下降、负债率上升、现金流恶化）会导致其债券的信用利差扩大。

Altman Z-Score

Altman Z-Score是纽约大学教授爱德华·奥特曼（Edward Altman）在1968年提出的破产预测模型。它利用五个财务比率加权计算出一个综合得分，用于判断企业是否接近破产边缘：

Z = 1.2X₁ + 1.4X₂ + 3.3X₃ + 0.6X₄ + 0.999X₅

其中：

  • X₁ = 营运资本 / 总资产（流动性）
  • X₂ = 留存收益 / 总资产（盈利积累）
  • X₃ = 息税前利润 / 总资产（盈利能力）
  • X₄ = 权益市值 / 负债账面值（杠杆水平）
  • X₅ = 销售收入 / 总资产（资产周转率）

Z值判断标准：Z > 2.99 —— 安全区；1.81 < Z < 2.99 —— 灰色区；Z < 1.81 —— 破产风险区。

踩坑警示：Altman Z-Score最初是基于美国制造业上市公司开发的，直接套用到中国非制造业企业可能产生误判。中国市场实践中，很多分析师对模型做了本土化改造——比如调整X₄的权重（因为中国企业的股权融资习惯与美国差异很大），或者用EBITDA替代EBIT。永远记住：模型来自特定样本，贸然套用到不同市场就是给自己挖坑。

信用违约互换（CDS）

CDS（Credit Default Swap）是一种为信用风险"上保险"的衍生品。CDS的买方定期支付保费（spread），卖方则在参考实体发生信用事件（违约、破产等）时进行赔付。CDS spread直接反映了市场对该实体信用风险的最新定价——如果CDS spread飙升，意味着市场认为该实体违约风险在急剧上升。

中国从2018年开始推出了信用风险缓释工具（CRM），本质上就是中国版的CDS。2018年民企"违约潮"期间，CRM的推出对稳定债券市场信心起到了关键作用。

学习建议：如果你想从事固定收益相关工作，建议养成每天看收益率曲线的习惯——中国的国债收益率曲线可以在中国债券信息网（chinabond.com.cn）上查到。连续跟踪三个月，你就能对收益率曲线的"正常"与"异常"状态形成直觉。这种"直觉"比任何数学模型都值钱。

固定收益分析为我们提供了评估"确定性"和"风险溢价"的精确工具。但在现代金融分析中，越来越多决策者开始将目光投向传统财务数据之外的"另类数据"——下一章我们将探讨卫星图像、舆情数据、供应链网络等另类数据如何为投资决策提供全新的信号。
