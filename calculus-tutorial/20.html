从一元到多元：变化率的维度升级

大家好，我是你们的讲师。欢迎来到多元函数微积分的世界。

回忆一元函数的情形：给你一个函数 y = f(x)，导数 dy/dx 表示函数值随自变量的变化率。问题很单纯——只有一个自变量 x，一个因变量 y，你关心的是"x 变了一点点，y 变了多少"。

但现实中的问题几乎从来不是这样简单的。比如，你想预测一个地区的房价——它取决于面积、房龄、楼层、地段评分、周边配套等多个因素。每个因素都是一个自变量。房价 P 是这些自变量的函数：P = f(area, age, floor, location_score, ...)。这时候问"房价的变化率"是什么意思？你是说面积增加一平米房价涨多少（保持其它因素不变），还是说房龄增加一年房价跌多少？

这就是偏导数（partial derivative）要回答的问题：当函数依赖于多个变量时，每次只看一个变量的影响，其余变量暂时视为常数。换句话说，偏导数是多元版本的"控制变量法"——一次只让一个因素变化，别的都冻结住。

我当年第一次学偏导数时，最大的困惑是："把其它变量看作常数"听起来很简单，但实际操作中经常忘记哪些是变量、哪些是常数。有一次做热力学作业，计算内能 U(T, V) 对温度 T 的偏导时，我下意识地把 V 也当作 T 的函数一起求了导——结果算出了一个长得离谱的表达式。老师改完后给了个大红叉，批注说："V 是独立变量，不是 T 的函数。∂/∂T 的意思是只对 T 求导，V 不变。"从那以后，我每次写偏导符号 ∂ 时都会在心里默念一遍："只动这一个，别的别碰。"

二元函数的概念与定义域

先从一个变量开始。最简单的多元函数是二元函数（function of two variables）：z = f(x, y)。两个自变量 x 和 y（独立变化），一个因变量 z。

二元函数的定义域（domain）不再是数轴上的一个区间，而是xy 平面上的一个区域。例如：

  • f(x, y) = √(1 - x² - y²) 的定义域是 x² + y² ≤ 1，即单位圆盘
  • f(x, y) = 1/(x - y) 的定义域是 x ≠ y 的整个平面（挖掉直线 y = x）
  • f(x, y) = ln(xy) 的定义域是第一象限和第三象限（xy > 0）
定义域的几何直观
二元函数的定义域是平面上的区域。当你画出这个区域时，你就能直观地看到函数在"哪里"有定义。判断定义域时，仍然遵循和一元函数相同的规则：分母不为零、根号内非负、对数的真数为正等等。只不过现在这些条件描述的是平面区域，而不是数轴上的区间。

函数 z = f(x, y) 的图像是一个曲面（surface）——xy 平面上的每个点对应一个 z 值，所有这些点在三维空间中拼成一个曲面。这是理解多元函数的关键几何图像：你看到的不是一条曲线，而是一张"毯子"铺在 xy 平面上方（或下方）。

偏导数的定义与几何意义

偏导数的定义

函数 z = f(x, y) 在点 (x₀, y₀) 处对 x 的偏导数定义为：

∂f/∂x |(x₀,y₀) = limΔx→0 [f(x₀ + Δx, y₀) - f(x₀, y₀)] / Δx

对 y 的偏导数类似，只是把 Δx 换成 Δy：

∂f/∂y |(x₀,y₀) = limΔx→0 [f(x₀, y₀ + Δy) - f(x₀, y₀)] / Δy

注意符号 ∂（读作"partial"或"偏"）代替了一元导数的 d。这个符号提醒你：这是一个"有偏见的"导数——只关心一个方向上的变化。

几何意义：截线的斜率

∂f/∂x 在几何上是什么意思？在点 (x₀, y₀) 处，把 y 固定在 y₀，让 x 变化。这意味着你用平面 y = y₀ 去截曲面 z = f(x, y)——截出来的是一条平面曲线（在 y = y₀ 平面上）。∂f/∂x 就是这条截线在 x₀ 处的斜率。

同样，∂f/∂y 是用平面 x = x₀ 截曲面得到的截线在 y₀ 处的斜率。

我的理解方法是这样的：站在曲面上的某一点，沿着 x 方向走一步，曲面上升（或下降）得有多陡？这就是 ∂f/∂x。沿着 y 方向走一步呢？这就是 ∂f/∂y。两个偏导数合起来，告诉你在这一点处曲面往各个方向的倾斜程度。

计算偏导数时，方法非常简单：把其它变量当作常数，对目标变量用一元求导法则即可。例如：

f(x, y) = x² sin y + y³

        ∂f/∂x = 2x · sin y  （把 sin y 看作常数）
        ∂f/∂y = x² · cos y + 3y²  （把 x² 看作常数）

就这么简单。你把 y 当作一个常数，就像对待数字 5 一样，然后对 x 正常求导。反过来的情况也一样。

高阶偏导数与混合偏导的相等条件

和一元函数类似，多元函数的偏导数也可以继续求偏导，得到高阶偏导数（higher-order partial derivatives）。

二元函数 f(x, y) 的二阶偏导数有四个：

记号 含义 读法
∂²f/∂x² 先对 x 求偏导，再对 x 求一次 "偏 x 平方"
∂²f/∂y² 先对 y 求偏导，再对 y 求一次 "偏 y 平方"
∂²f/∂x∂y 先对 y 求偏导，再对 x 求偏导 "偏 y 偏 x"（注意顺序！）
∂²f/∂y∂x 先对 x 求偏导，再对 y 求偏导 "偏 x 偏 y"

后两个被称为混合偏导数（mixed partial derivatives）。一个重要的定理是：如果混合偏导数在某个区域内连续，则它们与求导顺序无关——即 ∂²f/∂x∂y = ∂²f/∂y∂x。

混合偏导相等的条件（Clairaut 定理）
如果 f(x, y) 在点 (a, b) 的某个邻域内具有连续的混合偏导数 ∂²f/∂x∂y 和 ∂²f/∂y∂x，那么它们在 (a, b) 处相等。

这个定理在实际计算中非常实用——你可以选择更容易的计算顺序。比如先对 y 求导表达式很简洁，那就先对 y 求导，再对 x 求导。但需要记住前提：混合偏导数必须在区域内连续。如果函数不够光滑（比如在某点处有尖点或间断），顺序交换可能会得到不同的结果。

全微分：多元版的"以直代曲"

在一元函数中，微分 dy = f'(x) dx 是用切线来近似曲线。在多元函数中，全微分（total differential）是用切平面来近似曲面。

对于 z = f(x, y)，全微分定义为：

dz = (∂f/∂x) dx + (∂f/∂y) dy

它的含义非常直观：如果 x 变动了 dx，y 变动了 dy，那么 z 的总变化量 dz 由两部分组成——x 方向的变化贡献 (∂f/∂x) dx，y 方向的变化贡献 (∂f/∂y) dy。两者线性叠加就是总变化量。

全微分的用处在于近似计算。例如，你要测量一个圆柱形储油罐的容积。你测出底面半径 r ≈ 5m，高度 h ≈ 10m，测量误差 dr = ±0.05m，dh = ±0.05m。容积 V = πr²h，利用全微分：

dV = (∂V/∂r) dr + (∂V/∂h) dh
        = (2πrh) dr + (πr²) dh
        = (2π · 5 · 10)(±0.05) + (π · 25)(±0.05)
        = (±5π) + (±1.25π) = ±6.25π ≈ ±19.63 m³

你就知道容积的测量误差大约在 20 m³ 以内。有了这个估算，你可以决定这个测量精度是否满足工程要求——是否需要换更精密的仪器来减小误差。

小心：全微分假设各变量独立变化
全微分公式 dz = (∂f/∂x) dx + (∂f/∂y) dy 成立的前提是 x 和 y 是独立变量。如果 x 和 y 之间存在依赖关系（比如 y 是 x 的函数），那么总变化就不能简单叠加了——你需要使用链式法则（下一节）。

我见过不少同学在误差分析中重复计算了相关变量的误差，导致结果被放大了一倍。如果你测量半径 r 和周长 C = 2πr，用它们计算某个量——那 r 和 C 就不是独立的，它们的误差也不能独立处理。

多元复合函数的链式法则

一元函数的链式法则告诉我们：如果 y = f(g(x))，那么 dy/dx = f'(g(x)) · g'(x)——"外层导数乘以内层导数"。

多元复合函数的链式法则（chain rule for multivariable functions）更丰富，因为它有多种"复合"方式。最常见的情形是：

情形一：z = f(x, y)，其中 x = x(t)，y = y(t)

dz/dt = (∂f/∂x) · (dx/dt) + (∂f/∂y) · (dy/dt)

这是最基础的"路径"情形——你沿着一条参数曲线运动，z 的总变化率是 x 方向变化率和 y 方向变化率的加权和，权重分别是 x 和 y 对 t 的变化率。

情形二：z = f(x, y)，其中 x = x(s, t)，y = y(s, t)

∂z/∂s = (∂f/∂x) · (∂x/∂s) + (∂f/∂y) · (∂y/∂s)
        ∂z/∂t = (∂f/∂x) · (∂x/∂t) + (∂f/∂y) · (∂y/∂t)

这是更一般的情形，也是偏导版本链式法则的标准形式。它本质上是一个依赖关系图：z 依赖于 x 和 y，x 和 y 各自依赖于 s 和 t。∂z/∂s 就是从 z 到 s 的所有路径的偏导乘积之和。

链式法则的树形图记忆法：

            z
           / \
          x   y
         / \ / \
        s   t   s   t

        ∂z/∂s = (∂z/∂x)(∂x/∂s) + (∂z/∂y)(∂y/∂s)

对于更复杂的依赖链，这个树形图可以任意扩展。你只需要记住：从根（最终变量）到叶子（最底层自变量）的每条路径，把路径上偏导乘起来，再把所有路径的结果相加。

链式法则最容易犯的错误：漏项
当变量依赖关系复杂时，非常容易漏掉某条路径。我自己的习惯是在草稿纸上先画出依赖树，然后逐条路径遍历，每条路径上的偏导相乘，最后求和。每写完一条路径就在树上画一条线标记，避免重复或遗漏。

隐函数的偏导数

一元函数中，方程 F(x, y) = 0 可以隐式定义 y 为 x 的函数，隐函数求导公式为 dy/dx = -F_x / F_y（其中 F_y ≠ 0）。

多元版本完全类似。方程 F(x, y, z) = 0 定义了 z 为 x 和 y 的隐函数，那么：

∂z/∂x = -F_x / F_z    （F_z ≠ 0）
        ∂z/∂y = -F_y / F_z    （F_z ≠ 0）

怎样推导的？对 F(x, y, z(x, y)) = 0 两边同时对 x 求偏导（记住 z 是 x 和 y 的函数，需要用链式法则）：

∂F/∂x + (∂F/∂z)(∂z/∂x) = 0
        → F_x + F_z · (∂z/∂x) = 0
        → ∂z/∂x = -F_x / F_z

这个公式非常实用。例如给定方程 x² + y² + z² = 1（单位球面），求 ∂z/∂x。写成 F(x, y, z) = x² + y² + z² - 1 = 0，则 F_x = 2x，F_z = 2z，因此 ∂z/∂x = -x/z。这个结果在几何上非常合理：球面上某一点 (x, y, z) 处，z 随 x 的变化率就是 -x/z——如果在赤道附近 (z 很小)，x 方向的微小变化会引起 z 的剧烈变化。

场景 公式 备注
单个隐函数 F(x, y) = 0 dy/dx = -F_x / F_y 一元隐函数求导
两个变量隐函数 F(x, y, z) = 0 ∂z/∂x = -F_x / F_z
∂z/∂y = -F_y / F_z		 z 被隐式定义为 x, y 的函数
方程组情形 需要用 Jacobi 行列式 更复杂，研究生课程中展开

使用隐函数求导公式时，有个细节必须注意：分母 F_z 不能为零。如果 F_z = 0，说明在该点处隐函数定理的条件不满足——你可能需要换一个变量作为因变量，或者该点处根本不存在隐函数。

关于隐函数，我印象最深的是有一次做三维曲面重建的项目。给定一个点云数据（大量三维散点），我们需要用一个二次曲面去拟合。一开始我总想显式地解出 z = f(x, y)，但有些曲面是"竖直"的——同一个 (x, y) 对应多个 z 值（比如球面的上半部分和下半部分）。这时候隐式形式 F(x, y, z) = 0 反而更自然。我意识到：隐函数不是数学家的任性，而是现实世界复杂关系的必然需求。一对多的关系在现实中比比皆是，显式函数根本描述不了。

偏导数是多元微积分的基石。掌握了它，你就可以分析任意多元函数在单个方向上的变化率。下一章我们将在此基础上，讨论如何寻找多元函数的极值（最大值和最小值）——这是一元函数"导数为零找极值"的自然推广，但情况会变得更加丰富：不仅有 Hessian 矩阵这种二阶判别工具，还有拉格朗日乘数法这种处理约束优化问题的利器。
