第13章 设计效应

大家好，我是你们的讲师。上一章我们反复提到了一个概念——设计效应（Design Effect, Deff）。当时我说，样本量的计算要乘以Deff，复杂抽样的Deff通常大于1。但Deff的本质是什么？有哪些因素在影响它？它在实际工作中如何帮我们做决策？这一章我们就来彻底搞清楚这些问题。

先讲一个故事。我早年参与一个公共卫生项目时，合作方拿着一份调查报告来找我，语气中带着困惑和焦虑。他们的调查样本量是5000人，使用了典型的多阶段整群设计。他们用SRS公式计算，认为精度应该非常高——标准误在±1%以内。但实际数据分析时却发现，估计值的波动远大于预期，标准误大约是±2.5%。他们百思不得其解："我们不是抽了5000个样本吗？为什么精度好像只相当于SRS下的800个样本？"

这个谜题的答案就是设计效应。5000 ÷ 2.5² ≈ 800——他们的复杂设计将有效样本量从5000降到了800。这不是抽样出了问题，而是对抽样设计的效率缺乏理解。

Deff的定义

设计效应的正式定义出自抽样调查的泰斗——Leslie Kish（1975年）：

设计效应的定义

Deff = V_Complex / V_SRS

其中：
V_Complex = 在给定复杂抽样设计下估计量的实际方差
V_SRS = 在相同样本量下简单随机抽样的方差

解读：
- Deff = 1：与SRS效率相同
- Deff > 1：效率低于SRS（如整群抽样、多阶段抽样）
- Deff < 1：效率高于SRS（如高效的分层抽样）

另一个有用的概念是"有效样本量"：
n_eff = n_actual / Deff

例如 n=5000, Deff=2.5 → n_eff = 2000
意思是：这5000个复杂抽样的样本，
其信息量只相当于2000个SRS样本。

有效样本量（Effective Sample Size）是一个非常直观的概念。它回答的是"你的复杂样本等效于多少个简单随机样本"。当别人问你"你的样本多大"时，有效样本量比名义样本量更能反映真实的精度。

影响Deff的因素

设计效应不是天上掉下来的，它由几个关键因素决定。理解这些因素，你就知道如何主动控制Deff。

因素一：群内相关系数 ρ

群内相关系数（Intra-cluster Correlation Coefficient, ICC 或 ρ）衡量的是同一群内个体之间的相似程度。ρ越大，群内个体越同质，整群抽样的效率就越低（因为一个群内的人互相"重复"信息）。

  • ρ接近0：群内个体完全不相关（如随机分组的实验），整群抽样与SRS几乎一样有效
  • ρ接近1：群内个体几乎相同（如家庭成员的政治倾向），整群抽样非常低效
  • ρ通常在0.01到0.20之间——在社会调查中，典型的ρ值约为0.02-0.05

因素二：群的大小 m

这是每个群（PSU）中被抽取的样本数量。在其他条件相同的情况下，m越大，Deff越大。直观解释：你从同一个群里抽的人越多，他们之间的"信息重复"就越多。

因素三：权重的变异性

当权重（w_i）差异很大时，也会增加方差——因为权重大的样本对估计量的影响更大，引入了更多的不确定性。这一点在Kish的Deff公式中也有体现。

Kish公式

Leslie Kish提出了一个经典公式，将设计效应分解为两个主要来源：由聚类造成的效率损失，和由权重不等造成的效率损失。

Kish公式（约化形式）

Deff = 1 + (m - 1) × ρ + CV²(w)

其中：
- (m - 1) × ρ 是聚类效应（cluster effect）
  m = 每群平均样本量
  ρ = 群内相关系数

- CV²(w) 是权重变异效应（weighting effect）
  CV(w) = 权重的变异系数 = SD(w) / Mean(w)

示例1：整群抽样
  m=30, ρ=0.05, 自加权（CV=0）
  Deff = 1 + 29 × 0.05 + 0 = 2.45

示例2：整群抽样 + 不等权
  m=30, ρ=0.05, CV(w)=0.30
  Deff = 1 + 29 × 0.05 + 0.09 = 2.54

示例3：分层抽样（高效）
  ρ可能为负（层内更异质），或Deff < 1
  Deff ≈ 0.8（取决于层间差异大小）
一个重要的启示：当ρ很小时（比如0.01），即使m很大（比如100），聚类效应也只有(100-1)×0.01=0.99，Deff≈2——这还可以接受。当ρ很大时（比如0.2），即使m只有10，聚类效应也是(10-1)×0.2=1.8，Deff≈2.8——效率更低。这说明：控制Deff最有效的手段是降低ρ和m中的较大者。如果ρ无法降低（因为群体确实同质），那就减少每群样本量m；如果m受成本约束无法减少，那就应该尽量选择异质性高的群。

不同抽样方法的Deff典型范围

下表列出了常见抽样设计的Deff典型范围，供你在规划调查时参考：

抽样设计 典型 Deff 说明
SRS 1.0 基准
分层抽样（比例分配） 0.8 - 0.95 层间差异大时可能更小
分层抽样（最优分配） 0.5 - 0.9 取决于层间方差占比
系统抽样 0.9 - 1.1 接近SRS（排列随机时）
整群抽样（小群，m=10） 1.2 - 2.0 取决于ρ
整群抽样（大群，m=50） 2.0 - 5.0 ρ=0.02-0.08
多阶段抽样 2.0 - 8.0 阶段越多、ρ越大，Deff越大
PPS + 自加权 1.5 - 4.0 比等概率多阶段略优

Deff在实践中的应用

设计效应不是一个纯学术概念——它在抽样调查的每一个环节都有实际用途。

用途一：样本量规划

这是最常见的应用。在调查设计阶段，你需要根据预期的Deff来放大样本量：

n_planned = n_SRS × Deff_expected

例如：你想达到SRS下n=1000的精度，
但计划使用多阶段设计，预估Deff=3
则你需要 n = 1000 × 3 = 3000 个样本

用途二：结果报告

在报告调查结果时，应该报告设计效应和有效样本量，而不只是名义样本量。这能让读者正确理解你的估计精度：

好报告 vs 差报告

差报告："我们调查了5000人，结果置信区间为±2%"
好报告："我们调查了5000人（多阶段设计），
        设计效应Deff=2.5，有效样本量n_eff=2000，
        相当于SRS下±2.2%的置信区间"

用途三：设计优化

通过比较不同设计方案下的预期Deff，你可以选择效率最高的方案。例如：方案A（更多PSU、更少SSU）和方案B（更少PSU、更多SSU），在相同成本下比较哪个方案的Deff更小。

分析时的注意事项：很多统计软件（如Stata的svy命令、R的survey包）会自动计算复杂设计下的标准误，并隐含地考虑了设计效应。但如果你用普通软件（如Excel、SPSS默认设置）来分析复杂抽样数据，得到的结果会倾向于低估标准误——也就是说，你会觉得自己"很准"，但实际上没有那么准。这是所有入门者最容易犯的错误之一。

真实案例：2002年中国居民营养调查中的Deff

还记得我们在第8章提到的2002年中国居民营养与健康状况调查吗？这个调查的设计师们在论文中公布了各层（六类地区）的Deff值，其范围令人瞠目结舌。

2002年中国居民营养调查——各层Deff

┌──────────────┬───────┬────────┐
│ 地区类型      │  Deff  │ 备注   │
├──────────────┼───────┼────────┤
│ 大城市        │ 30.9  │ 极高！ │
│ 中小城市      │ 22.5  │ 很高   │
│ 一类农村      │ 14.2  │ 高     │
│ 二类农村      │ 16.8  │ 高     │
│ 三类农村      │ 12.4  │ 较高   │
│ 四类农村      │ 15.1  │ 高     │
└──────────────┴───────┴────────┘

这是什么概念？
大城市层的Deff=30.9 意味着：
  n_eff = n_actual / 30.9

如果在大城市层实际调查了3000人，
有效样本量只有 3000/30.9 ≈ 97 人！
也就是精度只相当于SRS下的97个样本。

为什么Deff会这么高？原因在于"营养与健康状况"在同一个城市内的不同家庭之间具有高度同质性：同一个城市的居民饮食习惯相似，水质相似，空气污染水平相似，甚至医疗可及性也相似。这就导致城市间的差异巨大，而城市内的差异很小——整群抽样的效率自然极低。

Deff=30.9意味着什么？简单计算一下：如果要达到SRS下n=1000的精度，在大城市层你需要调查1000 × 30.9 ≈ 30900人。而实际上该调查在大城市层大概只调查了几千人，所以精度其实相当有限——尤其是在做城市间比较的时候。

这个案例教给我们的教训是：如果你做整群或多阶段抽样，一定要对Deff有心理准备。特别是在公共卫生和流行病学调查中，ρ值往往比一般社会调查高得多（比如同一社区的家庭，营养状况和健康行为的相关系数很容易达到0.1-0.3），导致Deff可能高达10以上。

不要被Deff吓到就不做复杂抽样了：看到Deff=30.9，你是不是想说"那我干脆用SRS算了"？但别忘了，SRS在大型全国调查中根本不可行——你无法获得全国13亿人的完整名单，也无法在全国范围内等概率地抽取个体进行调查。复杂抽样虽然效率低，但它是唯一可行的选择。关键在于：事先认识到这种低效率，在样本量规划时充分放大。

Deff的局限性

设计效应是一个非常有用的概念，但它也有局限性：

  • Deff依赖于具体的目标变量：同一个抽样设计，对"性别比例"的Deff可能接近1（因为性别在群内分布均匀），但对"收入水平"的Deff可能很大（因为收入在群内高度同质）。所以报告Deff时应该说明是针对哪个变量的。
  • Deff的估计本身有不确定性：特别是在设计阶段，你只能根据过往研究或预调查来"猜测"Deff——这个猜测可能偏差很大。
  • Deff忽略了无回答的影响：Deff只衡量抽样设计造成的方差变化，但不考虑无回答带来的偏差。一个Deff很小但存在严重无回答偏差的调查，结果同样不可靠。

本章小结

设计效应是连接抽样设计和统计精度的桥梁。Deff的核心公式——Deff = 1 + (m-1)ρ + CV²(w)——用三个简洁的参数抓住了影响设计效率的关键因素。在样本量规划时用Deff调整，在分析结果时报告有效样本量，在设计方案时通过减少m或ρ来降低Deff——这些都是你有能力主动管理的。

但我们一直回避了一个技术问题：当你有了复杂抽样数据后，到底如何正确地计算方差？SRS的方差公式不能随便用了，那应该用什么？Taylor线性化、Jackknife、BRR、Bootstrap——这些方法各有什么优劣？这正是我们下一章的主题。
