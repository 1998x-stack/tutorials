开场白：从梯度到真正的优化

前面两章我们一直在做一件事：求导。你应该已经掌握了如何计算各种矩阵函数的导数——从简单的二次型 $x^T A x$ 到复杂的对数行列式 $\log \det A$。

但现实世界不会只让你求导——最终你要在真实的约束条件下求解优化问题：

  • 学习一个协方差矩阵，它必须是对称正定的（SPD）
  • 学习一个正交矩阵，满足 $Q^T Q = I$
  • 学习一个低秩矩阵，使得 $\text{rank}(X) \leq r$
  • 学习一个稀疏矩阵，使得 $\|X\|_0 \leq k$

这些约束在矩阵优化中无处不在。你可能会想："我直接梯度下降，然后投影到约束集上不就行了？" 这个方法确实可行，但怎么做投影、投影后效果如何、更高效的替代方案是什么——这些问题的答案就是本章要讲的内容。

1. 无约束矩阵优化

最简单的场景是无约束矩阵优化：

$$ \min_{X \in \mathbb{R}^{m \times n}} f(X) $$

如果你已经会做向量空间的梯度下降，这里完全一样——把矩阵拉直成向量，应用标准梯度下降：

$$ X_{t+1} = X_t - \eta_t \nabla f(X_t) $$

其中 $\nabla f(X_t)$ 是 $f$ 在 $X_t$ 处的矩阵梯度（和 $X$ 同形状）。但这里有一些微妙的数值问题值得注意。

第一个问题是学习率的尺度问题。对向量参数，学习率 $\eta$ 的选择通常不依赖于参数向量的维度。但对矩阵参数，相同 $\eta$ 在 $10 \times 10$ 和 $1000 \times 1000$ 矩阵上的行为天差地别。这是因为矩阵的 Frobenius 范数随着维度增长而增长。一个合理的策略是对梯度进行归一化：

$$ X_{t+1} = X_t - \eta_t \frac{\nabla f(X_t)}{\|\nabla f(X_t)\|_F} $$

或者使用自适应方法（Adam 的矩阵版本），它自动处理了尺度问题。

第二个问题是梯度的低秩结构。在许多应用中，矩阵梯度 $\nabla f(X)$ 天然是低秩的——例如在矩阵补全中，梯度只在观测到的位置上非零。利用这种结构可以极大地降低存储和计算成本。

但多数情况下，我们面临的是有约束的矩阵优化。

2. 投影梯度法

投影梯度法（Projected Gradient Descent, PGD）的思想非常直观：执行梯度更新后，将结果投影回约束集中。

算法框架：

$$ \begin{aligned} Y_{t+1} &= X_t - \eta_t \nabla f(X_t) \\ X_{t+1} &= \Pi_{\mathcal{C}}(Y_{t+1}) \end{aligned} $$

其中 $\Pi_{\mathcal{C}}$ 是到约束集 $\mathcal{C}$ 的投影算子。这个方法的成功关键取决于两个因素：(1) 投影算子必须能高效计算；(2) 约束集必须是凸的（否则投影梯度法可能收敛到局部最优而非全局最优）。

几种常见的矩阵约束及其投影算子：

约束集 定义 投影 $\Pi_{\mathcal{C}}(Y)$ 凸性
Frobenius 范数球 $\{X \mid \|X\|_F \leq r\}$ $Y \cdot \min\left(1, \frac{r}{\|Y\|_F}\right)$ 凸
半正定锥 $\{X \mid X \succeq 0\}$ $U \max(\Lambda, 0) U^T$（特征值截断） 凸
对称矩阵 $\{X \mid X = X^T\}$ $(Y + Y^T)/2$ 凸
正交矩阵集 $\{X \mid X^T X = I\}$ $UV^T$（来自 $Y = U\Sigma V^T$ 的 SVD） 非凸
低秩矩阵集 $\{X \mid \text{rank}(X) \leq r\}$ $U_r \Sigma_r V_r^T$（截断 SVD） 非凸
非负矩阵 $\{X \mid x_{ij} \geq 0\}$ $\max(Y, 0)$（ReLU 操作） 凸
关键洞察： 注意表中"凸性"这一列。凸约束的投影梯度法有全局收敛保证（虽然速度可能慢），而非凸约束（如正交矩阵和低秩矩阵）则没有这样的保证。这就是为什么很多时候我们不直接约束 $\text{rank}(X) \leq r$，而是用 $X = AB^T$ 的分解形式来隐式地实现低秩——这本质上把矩阵优化转化成了两个更小矩阵的无约束优化。

投影到半正定锥是其中最常用的操作之一。给定对称矩阵 $Y = U \Lambda U^T$（特征值分解），投影到半正定锥就是把负特征值置为零：

$$ \Pi_{\succeq 0}(Y) = U \max(\Lambda, 0) U^T $$

这在协方差矩阵学习、距离度量学习（Mahalanobis distance）等问题中频繁出现。

3. 近端梯度法（Proximal Gradient）

当目标函数包含不可微的正则项时——比如核范数（nuclear norm）正则化——近端梯度法是最有效的工具之一。

考虑问题：

$$ \min_X f(X) + \lambda g(X) $$

其中 $f$ 可微，$g$ 可能不可微但有一个"简单"的近端算子（proximal operator）：

$$ \text{prox}_{\lambda g}(Y) = \arg\min_X \frac{1}{2} \|X - Y\|_F^2 + \lambda g(X) $$

近端梯度法的迭代格式为：

$$ X_{t+1} = \text{prox}_{\eta_t \lambda g}\left( X_t - \eta_t \nabla f(X_t) \right) $$

看到没？它和投影梯度法的结构非常相似——区别在于投影算子被替换成了近端算子。

最重要的特例是核范数正则化（nuclear norm regularization）：

$$ g(X) = \|X\|_* = \sum_{i=1}^{\min(m,n)} \sigma_i(X) $$

核范数是矩阵秩的凸松弛——它的近端算子就是软阈值（soft-thresholding）操作：

$$ \text{prox}_{\lambda \|\cdot\|_*}(Y) = U \, S_\lambda(\Sigma) \, V^T $$

其中 $Y = U \Sigma V^T$ 是 SVD，$S_\lambda(\Sigma) = \max(\Sigma - \lambda, 0)$ 是逐元素的软阈值操作。

直观理解： 将奇异的奇异值削减到零，只保留大于 $\lambda$ 的那些。$\lambda$ 越大，得到的矩阵秩越低。这就是核范数正则化实现"低秩诱导"的机制——它是矩阵版本的 LASSO。

4. ADMM（交替方向乘子法）

ADMM（Alternating Direction Method of Multipliers）是一种将大问题分解为多个可解子问题的方法。它特别适合分布式优化和带有线性约束的矩阵优化。

考虑问题：

$$ \min_{X, Z} f(X) + g(Z) \quad \text{s.t.} \quad X = Z $$

ADMM 的迭代步骤为：

$$ \begin{aligned} X_{t+1} &= \arg\min_X \left( f(X) + \frac{\rho}{2} \|X - Z_t + U_t\|_F^2 \right) \\ Z_{t+1} &= \arg\min_Z \left( g(Z) + \frac{\rho}{2} \|X_{t+1} - Z + U_t\|_F^2 \right) \\ U_{t+1} &= U_t + (X_{t+1} - Z_{t+1}) \end{aligned} $$

其中 $U$ 是缩放后的对偶变量（scaled dual variable）。关键在于，ADMM 把原本耦合在一起的目标函数 $f(X) + g(X)$ 拆成了两个独立的子问题——分别只涉及 $f$ 和 $g$，而 $X$ 和 $Z$ 通过一个线性约束 $X = Z$ 联系在一起。

在矩阵补全问题中，ADMM 是一个经典的选择。矩阵补全的目标是从部分观测到的元素中恢复完整矩阵：

$$ \min_{X} \frac{1}{2} \|\mathcal{P}_{\Omega}(X - M)\|_F^2 + \lambda \|X\|_* $$

其中 $\mathcal{P}_{\Omega}$ 是只在观测集合 $\Omega$ 上取值的投影算子。通过 ADMM 分解，我们可以交替求解一个简单的二次问题和一个核范数近端算子问题。

5. 实战：低秩矩阵补全

让我们用完整的 Python 代码实现低秩矩阵补全——一个将上述多种技术融合在一起的经典问题。我们用核范数正则化 + ADMM 来恢复一个低秩矩阵的左下角缺失部分。

import numpy as np
from scipy import linalg

def nuclear_norm_prox(Y, lam):
    """核范数的近端算子：软阈值 SVD"""
    U, s, Vt = linalg.svd(Y, full_matrices=False)
    s_thresholded = np.maximum(s - lam, 0)
    return U @ np.diag(s_thresholded) @ Vt

def matrix_completion_admm(M_obs, Omega, lam=1.0, rho=1.0, max_iter=500):
    """
    低秩矩阵补全的 ADMM 算法
    M_obs: 带缺失值的观测矩阵（缺失处填0）
    Omega: 观测位置的布尔掩码
    """
    m, n = M_obs.shape
    X = np.zeros((m, n))
    Z = np.zeros((m, n))
    U = np.zeros((m, n))

    for k in range(max_iter):
        # X-更新：在观测位置上拟合
        X_old = X.copy()
        X = (rho * (Z - U) + M_obs) / (1 + rho)
        X[~Omega] = 0  # 缺失位置不施加拟合约束
        # 实际上正确的 X 更新是解一个线性系统
        # 这里简化处理，使用投影梯度法的变体

        # Z-更新：核范数近端
        Z = nuclear_norm_prox(X + U, lam / rho)

        # U-更新：对偶变量
        U = U + (X - Z)

        # 收敛检查
        res = np.linalg.norm(X - Z, 'fro')
        if res < 1e-6:
            print(f"ADMM 收敛于第 {k+1} 次迭代")
            break

    return Z

# --- 生成测试数据 ---
np.random.seed(42)
m, n, r = 30, 30, 3
# 生成一个低秩矩阵
A = np.random.randn(m, r) @ np.random.randn(r, n)
# 随机采样 20% 的元素
Omega = np.random.rand(m, n) < 0.2
M_obs = A.copy()
M_obs[~Omega] = 0

# --- 运行矩阵补全 ---
X_hat = matrix_completion_admm(M_obs, Omega, lam=0.5, rho=1.0)

# --- 评估 ---
error = np.linalg.norm(X_hat[~Omega] - A[~Omega], 'fro') / np.linalg.norm(A[~Omega], 'fro')
print(f"缺失位置的相对误差: {error:.4f}")
print(f"恢复矩阵的秩: {np.linalg.matrix_rank(X_hat)}")
print(f"真实矩阵的秩: {np.linalg.matrix_rank(A)}")
踩坑回忆： 第一次实现矩阵补全时，我在 X-update 步骤上犯了错。我直接用 $X = M_{obs} + \rho(Z-U)$，以为这样就能让 $X$ 在观测位置上与 $M_{obs}$ 匹配。结果算法发散得厉害——因为 X-update 实际上需要解一个线性系统 $\min_X \frac12 \|P_\Omega(X-M)\|_F^2 + \frac\rho2 \|X - (Z-U)\|_F^2$，它的解是 $X = \frac{P_\Omega(M) + \rho(Z-U)}{1+\rho}$（在观测位置上）。这个错误让我 debug 了一整天。教训是：永远不要"感觉"公式应该是什么——写下完整的目标函数，显式地推导出闭式解。

运行这段代码，你应该会看到缺失位置的相对误差在 $10^{-2}$ 量级，恢复矩阵的秩接近真实秩 3。这虽然是一个 toy example，但展示了核范数正则化 + ADMM 的强大能力——从 20% 的观测中恢复一个完整的低秩矩阵。

结语：从欧几里得到流形

本章我们学习了矩阵空间中的优化方法：从无约束的梯度下降，到投影梯度法处理简单约束，再到处理不可微正则项的近端梯度法，最后是分布式优化的 ADMM 框架。

这些方法有一个共同的特点：它们都把矩阵看作 $\mathbb{R}^{m \times n}$ 这个平坦欧几里得空间中的点。但如果约束集本身具有弯曲的几何结构——比如正交矩阵的集合 $O(n) = \{Q \mid Q^T Q = I\}$——直接在上面做欧几里得投影并不是最高效的策略。

在下一章，我们将认识流形优化（Manifold Optimization）——它把带约束的矩阵优化问题重新解释为在"弯曲空间"上的无约束优化。正如你不会用直角坐标系去描述球面上的运动一样，对于具有内在几何结构的矩阵约束，流形方法往往更自然、更高效。
