第8章 简单国民收入决定

假设你是一家小餐馆的老板，这个月生意不太好，你在犹豫要不要裁员。你算了一笔账：裁掉一个服务员，每月能省3000块工资。但你马上意识到——那个服务员被裁之后，他就不会再来你的餐馆吃饭了，你每个月会少赚500块。更麻烦的是，被他服务的熟客因为他不在而体验变差，来的次数也减少了。你看，一个简单的裁员决定，引发了一连串的经济涟漪。

这正是国民收入决定要回答的核心问题：一个经济体的总产出是怎么决定的？为什么有时候经济会陷入"越不敢花钱、大家就越穷"的恶性循环？凯恩斯在1936年给出的答案，直到今天仍然是宏观经济分析的基石——总需求决定短期产出。这一章，我们就从最简单的"凯恩斯交叉图"出发，搭建起理解整个经济波动的框架。

1. 凯恩斯交叉图：45度线的核心直觉

凯恩斯有一个极其简单但又极其深刻的想法：一个经济体的产出（Y），最终是由总支出（AE, Aggregate Expenditure）决定的。企业生产多少东西，取决于人们打算买多少东西。如果大家想买的东西比企业生产的多，企业就会增加生产；如果大家想买的少，企业就会减产甚至裁员。

这个想法画出来就是凯恩斯交叉图。横轴是国民收入Y（也就是总产出），纵轴是总支出AE。关键的"45度线"代表一个恒等关系：AE = Y，也就是所有产出都被花掉了。总支出线AE是一条向上倾斜的线——收入越高，人们的消费支出越多。两条线的交点，就是经济的均衡产出。

这个图最反直觉的地方在于：均衡产出 不一定等于充分就业产出。如果大家的"消费意愿"普遍偏低（AE线偏低），均衡就会落在低于充分就业的位置——经济就出现了"非自愿失业"的资源闲置。这就是凯恩斯之前几乎所有经济学家都否认的可能性：他们认为市场会自动出清，不可能长期存在闲置资源。

核心直觉：需求创造供给
凯恩斯之前的古典经济学认为"供给创造自己的需求"（萨伊定律）。凯恩斯把它翻了个个儿——在短期内，是需求决定供给。如果没人花钱，企业就不会生产；企业不生产，工人就没有收入；工人没有收入，就更没人花钱。这就是"需求不足型衰退"的根本逻辑。

2. 消费函数：凯恩斯的心理定律

既然总支出是决定产出的关键，我们就得搞清楚总支出是怎么构成的。在一个最简单的两部门经济（只有家庭和企业）中，总支出AE = 消费C + 投资I。而消费，是其中最大的一块——通常占到GDP的60%-70%。

凯恩斯提出了一个大胆的假设：消费是当期可支配收入的函数。他称之为"基本心理定律"——人们倾向于随着收入增加而增加消费，但消费的增加量 小于收入的增加量。写成数学形式就是：

C = a + b · Y

参数说明：
  a  = 自主性消费（autonomous consumption）——即使收入为零也必须维持的基本消费
  b  = 边际消费倾向 MPC（marginal propensity to consume）——每增加1元收入，会有多少用于消费
  Y  = 可支配收入

典型取值：
  · 发达国家的MPC通常在0.5到0.8之间
  · 中国城镇居民的MPC大约为0.6-0.7（即每多赚100元，花掉60-70元）
  · 低收入群体的MPC显著高于高收入群体（约0.8 vs 0.3）

与MPC对应的概念是边际储蓄倾向MPS（marginal propensity to save）。因为收入要么消费、要么储蓄，所以MPC + MPS = 1。如果你的MPC是0.7，那么MPS就是0.3——每多赚100元，你存下30元。

下面这张表总结了消费函数的各个参数及其经济含义：

概念 符号 定义 典型值 经济含义
自主性消费 a 收入为零时的消费 取决于基本生存需要 最低生活保障线的宏观反映
边际消费倾向 MPC = b ΔC / ΔY 0.5 - 0.8 收入增加1元，消费增加b元
边际储蓄倾向 MPS = 1-b ΔS / ΔY 0.2 - 0.5 收入增加1元，储蓄增加(1-b)元
平均消费倾向 APC C / Y 随收入上升而下降 富人储蓄率更高，穷人消费倾向更高

3. 乘数效应：一块钱如何变成三块钱

现在到了本章最精彩的部分。假设政府决定多修一条高速公路，投入了1000万给建筑公司。这1000万不会就此消失——建筑公司拿到钱后，会给工人发工资、买钢筋水泥。工人和建材商拿到钱后，会去餐馆吃饭、买新手机、给孩子交学费。餐馆老板和手机店员的收入增加了，他们又会把钱花出去……

这就是乘数效应（multiplier effect）：最初的一笔支出，通过经济体内的消费循环，最终产生了数倍于初始支出的总需求增加。乘数的大小取决于MPC：MPC越高，每一轮"漏出"到储蓄的就越少，乘数就越大。

我们来推导一下。设初始支出增加ΔI，第一轮直接产生ΔI的收入。第二轮，人们把其中MPC·ΔI用于消费，产生新的收入。第三轮，MPC·(MPC·ΔI) = MPC²·ΔI……如此无限循环下去：

总收入增加 ΔY = ΔI + MPC·ΔI + MPC²·ΔI + MPC³·ΔI + ...
                = ΔI · (1 + MPC + MPC² + MPC³ + ...)
                = ΔI · 1/(1 - MPC)

所以乘数 k = ΔY/ΔI = 1/(1 - MPC) = 1/MPS

举例：
  如果 MPC = 0.8，乘数 = 1/(1-0.8) = 1/0.2 = 5
  政府每花1元，最终GDP增加5元

  如果 MPC = 0.5，乘数 = 1/(1-0.5) = 2
  政府每花1元，最终GDP增加2元

不同类型的支出乘数有细微差别：

乘数类型 公式 说明
投资乘数 ΔY/ΔI = 1/(1-MPC) 企业投资增加对GDP的放大效应
政府支出乘数 ΔY/ΔG = 1/(1-MPC) 与投资乘数相同，因为政府支出直接构成总需求
税收乘数 ΔY/ΔT = -MPC/(1-MPC) 减税的经济刺激力度——绝对值小于支出乘数
平衡预算乘数 ΔY = ΔG (当ΔG = ΔT时) 政府同时等额增支和增税，GDP仍然增加一个ΔG
学习提示：注意税收乘数为什么小于政府支出乘数。政府支出增加1元是"直接进入总需求"，而减税1元要经过"先变成居民可支配收入→居民决定消费多少→消费部分进入总需求"的链条，所以效果打了折扣。这也是为什么经济学家普遍认为"花钱"比"减税"的短期刺激效果更强——虽然减税在长期可能有供给侧的好处。

4. 节俭悖论：为什么"大家一起省"反而更穷

踩坑故事：樱花小区的储蓄竞赛
2015年，江南某市一个叫"樱花小区"的中档社区，几位退休老人发起了一个"储蓄竞赛"活动：到年底，哪家储蓄率最高，就获得"勤俭持家模范家庭"的锦旗和2000元奖金。活动一开始反响热烈，邻居们纷纷减少外出就餐、推迟换家电的计划、少买新衣服。

两个月后，怪事发生了。小区门口的小卖部老板——老李——第一个撑不住了。以前每天有几十个人来买烟酒零食，现在一天不到十个人。他只能裁员，把帮工的小王辞退了。小王没有了收入，连小区门都不怎么进了。很快，小区的理发店、早餐摊、水果店，一个接一个门可罗雀。到年底，整个小区的商业活动几乎停摆。

最讽刺的是什么呢？那些拼命存钱的居民，年底一算账——确实存下了更多的钱。但是他们的房子贬值了（因为周边商业环境恶化），社区的租金收入下降了（如果有出租房产的话），甚至有些人的工作也受到了影响。个人"理性"的储蓄行为，合在一起导致了集体的非理性结果：每个人都想多存钱，结果每个人都变得更穷。

这就是凯恩斯在《通论》中提出的节俭悖论（Paradox of Thrift）。它解释了为什么在经济衰退时期，鼓励储蓄反而可能是有害的——你存下的每一块钱，就是别人失去的一块钱收入。当我们都在拼命存钱的时候，我们其实是在把经济推向更深的衰退。

但这并不意味着储蓄是坏事情！请注意这个结论的条件：它只在经济处于衰退、资源闲置时才成立。在经济过热时，储蓄反而是美德——它能抑制过度的总需求、为投资提供资金。节俭悖论再次提醒我们：对个人正确的事，对集体不一定正确。这就是第1章讲过的"合成谬误"（fallacy of composition）最经典的例子。

5. 从短期到长期的桥梁：为什么乘数只在资源闲置时有效

你一定注意到了一个问题：如果乘数效应这么强大，为什么我们不一直让政府花钱刺激经济呢？答案在于一个关键的区分——短期 vs 长期。

乘数效应成立的前提是：经济中存在闲置资源（Idle Resources）。失业的工人、闲置的机器、空置的厂房——只有在这种条件下，增加需求才能转化为增加产出。如果经济已经处于充分就业（或接近充分就业），需求增加带来的不是更多产出，而是更高的物价。

想象一个已经满座的电影院。你拿着1000张免费票走进来，想多拉一些人来看电影——但座位已经坐满了。你不可能让更多人进来，只能让票价上涨。乘数效应需要一个"有空位的电影院"。当经济满负荷运转时，它就是那个没有空位的电影院。

这个区分完美地连接了宏观经济学两大传统：

短期（凯恩斯） 长期（古典/新古典）
核心问题 总需求不足导致闲置资源 供给能力决定潜在产出
价格调整 价格粘性，主要调整数量 价格灵活，充分调整
政策含义 财政货币刺激有效 刺激只会推高物价，不影响实际产出
适用情景 衰退、萧条、流动性陷阱 经济过热、高通胀时期

用一句形象的话来总结：短期看需求，长期看供给。当经济处于低谷时，要记住凯恩斯的教导——需求不足是敌人，政府该出手时就出手。当经济恢复、产能趋于饱和时，要回到古典经济学——回归供给侧的改革和增长策略。一个优秀的经济政策制定者，知道什么时候该用凯恩斯工具箱，什么时候该把它收起来。

这一章，我们从一个简单的45度线模型出发，认识了消费函数、乘数效应和节俭悖论。这些概念是理解之后所有宏观经济模型的基石。下一章，我们将把投资从"简单的外生变量"升级为"由利率决定的内生变量"，从而进入产品市场与IS曲线——这是搭建完整IS-LM模型的第一步。
