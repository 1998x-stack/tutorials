第10章 数值积分基础

大家好，我是你们的讲师。今天我们要进入数值分析中一个非常经典的话题——数值积分。大家从微积分课上都学过 Newton-Leibniz 公式：只要能找到原函数，积分就迎刃而解。但现实世界中有多少函数能找到原函数呢？我来问问大家：$\int_0^1 e^{-x^2} dx$ 是多少？$\int_0^\pi \sin(\sin x) dx$ 呢？这些看似简单的积分，它们的原函数都无法用初等函数表示。更不用说实际工程中我们往往只有离散的采样数据，连解析表达式都没有。这就是数值积分存在的意义。

我读博时有个师兄做计算电磁学，需要计算一个极其复杂的积分——被积函数来自有限元仿真，每次求值需要 0.1 秒。他一开始天真地用了 Mathematica 的符号积分，跑了两个小时没跑出来。后来换成数值积分，用自适应 Simpson 公式，总共只需要 200 次函数求值，20 秒搞定。他在组会上感慨："有时候，近似答案就是正确答案。"

10.1 数值积分的基本思想

数值积分的核心思想可以用一句话概括：用多项式近似被积函数，然后精确积分这个多项式。

更具体地说，所有数值积分公式（也称求积公式，quadrature rules）都具有以下形式：

∫_a^b f(x) dx ≈ Σ_{i=0}^n w_i f(x_i)

其中 $x_i$ 是求积节点，$w_i$ 是求积权重。不同求积公式的区别就在于如何选择节点和权重。

最简单的推导方法是通过插值：用 Lagrange 插值多项式 $L_n(x)$ 近似 $f(x)$，然后对 $L_n(x)$ 精确积分：

∫_a^b f(x) dx ≈ ∫_a^b L_n(x) dx = ∫_a^b Σ_i f(x_i) l_i(x) dx = Σ_i f(x_i) ∫_a^b l_i(x) dx

这样就自然地得到 $w_i = \int_a^b l_i(x) dx$。这个思路在后续章节中会反复出现。

10.2 代数精度

如何衡量一个求积公式的"好坏"？最重要的概念是代数精度（degree of precision, DoP）。

定义：如果一个求积公式对所有次数不超过 $m$ 的多项式都精确成立，但对 $m+1$ 次多项式不精确，则称它具有 $m$ 次代数精度。

检验方法：要测试一个求积公式的代数精度，只需依次代入 $f(x) = 1, x, x^2, x^3, \dots$ 计算。第一个不精确的幂次减 1 就是代数精度。例如，若公式对 $1, x, x^2$ 都精确但对 $x^3$ 不精确，则代数精度为 2。

下表列出了常见 Newton-Cotes 求积公式及其代数精度：

求积公式节点数节点位置代数精度
中点公式 (矩形)1中点1
梯形公式2端点1
Simpson 公式3等距三节点3
Simpson 3/8 公式4等距四节点3
Boole 公式5等距五节点5

一个关键的观察：$n+1$ 个节点的插值型求积公式的代数精度至少为 $n$。但 Simpson 公式用了 3 个节点却达到了 3 次代数精度（比 $n=2$ 多了 1 次），这是因为对称性使得 $x^3$ 的积分正好被精确计算。这种"意外惊喜"在数值积分中时有发生。

10.3 插值型求积公式（Newton-Cotes 公式）

Newton-Cotes 公式是插值型求积公式中最基础的一类——节点在 $[a,b]$ 上等距分布。记 $h = (b-a)/n$，节点 $x_i = a + ih$，$i = 0,1,\dots,n$。

封闭型（closed form）：节点包含端点 $a$ 和 $b$。

开放型（open form）：节点不包含端点，在区间内部。

封闭型 Newton-Cotes 公式的一般形式为：

∫_a^b f(x) dx ≈ (b-a) Σ_{i=0}^n C_i^{(n)} f(x_i)

其中 $C_i^{(n)}$ 是 Cotes 系数。对于 $n=1$，Cotes 系数为 $C_0^{(1)} = C_1^{(1)} = 1/2$，这就是梯形公式。对于 $n=2$，$C_0^{(2)} = C_2^{(2)} = 1/6$，$C_1^{(2)} = 2/3$，这就是 Simpson 公式。

10.4 梯形公式与 Simpson 公式

我们从几何角度来理解这两个最基本的公式。

梯形公式用一条直线段连接两个端点 $f(a)$ 和 $f(b)$，用梯形的面积近似曲线下的面积：

T(f) = (b-a) · [f(a) + f(b)] / 2

Simpson 公式用一条抛物线穿过三个点 $(a, f(a))$、$(\frac{a+b}{2}, f(\frac{a+b}{2}))$、$(b, f(b))$，用抛物线下的面积近似：

S(f) = (b-a) · [f(a) + 4f((a+b)/2) + f(b)] / 6

误差分析告诉我们：

梯形公式误差：E_trap = -(b-a)³ / 12 × f''(ξ),  ξ ∈ (a,b)
Simpson 公式误差：E_simp = -(b-a)⁵ / 2880 × f''''(ξ),  ξ ∈ (a,b)

这意味着如果被积函数的二阶导数很大，梯形公式的误差就会很大；Simpson 公式对三次多项式是精确的，这比梯形公式强得多。

下面是一个 Python 对比：

import numpy as np

def trapezoidal(f, a, b):
    """梯形公式"""
    return (b - a) * (f(a) + f(b)) / 2

def simpson(f, a, b):
    """Simpson 公式"""
    c = (a + b) / 2
    return (b - a) * (f(a) + 4*f(c) + f(b)) / 6

# 测试 f(x) = sin(x) 在 [0, π] 上的积分
# 精确值 = 2.0
f = np.sin
exact = 2.0

t = trapezoidal(f, 0, np.pi)
s = simpson(f, 0, np.pi)

print(f"精确积分值:  {exact:.10f}")
print(f"梯形公式:     {t:.10f}, 误差 = {abs(t-exact):.2e}")
print(f"Simpson公式:  {s:.10f}, 误差 = {abs(s-exact):.2e}")

# 对于 f(x) = x³，Simpson 公式应该精确到机器精度
def cubic(x):
    return x**3

exact_cubic = (1**4 - 0**4) / 4  # = 0.25
s_cubic = simpson(cubic, 0, 1)
print(f"\n测试 f(x)=x³, Simpson 公式: {s_cubic:.15f}, 精确: {exact_cubic:.15f}")
print(f"误差: {abs(s_cubic-exact_cubic):.2e} (应该是机器精度)")

踩坑故事：我有个学生在做物理实验数据处理时，用梯形公式计算 $\int_0^{100} \sin(x^2) dx$。这个函数振荡极快，他在整个区间上只用一个梯形公式——误差大到离谱！他跑来问我说为什么数值积分不准确。我说："你把函数画出来看看。"他看到 $\sin(x^2)$ 在 $[0,100]$ 上振荡了上千次，一条直线怎么可能拟合上千次振荡？这个教训告诉我们：在应用求积公式前，先了解被积函数的行为。对于快速振荡的函数，要么加密节点（复合求积），要么用其他方法。

本章我们学习了数值积分的基本思想——用插值多项式近似被积函数然后积分。我们介绍了代数精度的概念，以及最基础的梯形公式和 Simpson 公式。这两个公式在实际中很少单独使用（因为精度太差），但它们构成了更高级方法的基础。下一章我们将学习如何把它们组合成复合求积公式，以及如何用 Richardson 外推技术来大幅提高精度。敬请期待！
