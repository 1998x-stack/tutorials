开场：为什么铁能被磁铁吸引？

你可能从幼儿园起就知道磁铁能吸铁。但有没有想过：为什么是铁？为什么木头不行？为什么有些磁铁是"永久的"而有些只是"临时的"？

答案藏在原子的微观世界中。要理解磁介质，我们必须进入比原子还小的尺度——电子的轨道运动和自旋——并看到它们如何共同产生宏观的磁性。今天，我们将揭开磁化（magnetization）的神秘面纱，并学习描述磁性材料的重要概念：磁滞回线、磁导率、磁路。

磁性的微观起源

材料的磁性来源于原子内部运动电荷产生的微小磁矩。有两个主要贡献：

电子的轨道磁矩

电子绕原子核运动，本质上就是一个微小的电流环。根据经典电磁学，电流环的磁矩 μ = I·A（电流 × 面积）。对电子轨道运动：

μ轨道 = −(e / 2m) · L

其中 L 是电子的轨道角动量。负号表示磁矩方向与角动量相反（因为电子带负电）。

电子的自旋磁矩

除了轨道运动，电子还有一个内禀的自旋（spin）属性。自旋磁矩为：

μ自旋 ≈ −(e / m) · S

自旋磁矩大约比轨道磁矩大一个因子 2（这就是"g因子" ≈ 2的来由）。自旋是铁磁性的主要来源。

基本磁矩的天然单位是玻尔磁子（Bohr magneton）：

μB = e ħ / (2m) = 9.274 × 10⁻²⁴ J/T

每个电子的自旋磁矩约等于 1 μB。

三种磁性

根据原子磁矩在外磁场下的行为，材料表现出三种截然不同的磁性：

类型 磁化率 χ 典型数值 物理机制 举例
抗磁性 χ < 0，小 ≈ −10⁻⁵ 外磁场感应出反向磁矩 Cu, Au, Bi, 水, 有机物
顺磁性 χ > 0，小 ≈ +10⁻⁵ ~ +10⁻³ 未配对电子自旋取向 Al, Pt, O₂（液态）
铁磁性 χ >> 0，大 10³ ~ 10⁶ 交换耦合使自旋整齐排列 Fe, Co, Ni, 钕铁硼

抗磁性（Diamagnetism）

所有材料都具有抗磁性——这是外磁场作用下原子中电子轨道运动被微扰的结果。楞次定律在原子尺度的体现：感应电流产生的磁矩总是对抗外磁场的变化。因此抗磁性的磁化率 χ 为负值，但非常小（约 −10⁻⁵），通常被更强的顺磁性或铁磁性掩盖。

有趣的现象：超导体是完美抗磁体（χ = −1），能将磁场完全排出体外——这就是迈斯纳效应（Meissner Effect）。

顺磁性（Paramagnetism）

当原子或分子中含有未配对电子时（如过渡金属离子、自由基），原子具有永久磁矩。无外磁场时，热运动使磁矩随机取向，净磁化为零。施加外磁场后，磁矩趋向于沿磁场方向排列——但热运动不断对抗这种有序化。

顺磁磁化率遵循居里定律（Curie's Law）：χ = C / T，其中 C 是居里常数。温度越高，热运动越剧烈，磁矩越难整齐排列，磁化率越小。

铁磁性（Ferromagnetism）

铁磁性材料的相邻原子之间通过交换相互作用（exchange interaction）——一种纯粹的量子力学效应——使电子自旋自发地平行排列。这种量子力学耦合如此之强，以至于即使在无外磁场时，材料也能保持宏观磁化。

但铁磁体并非整体均匀磁化，而是分为许多磁畴（magnetic domains）。每个磁畴内部的磁矩方向一致，但不同磁畴的方向不同。施加外磁场时，磁畴壁移动、与磁场方向一致的磁畴扩大、方向不利的磁畴缩小，最终实现宏观磁化。

🔑 核心洞察
铁磁性是一个纯粹的量子力学效应——经典物理学无法解释为什么铁原子的磁矩会自发整齐排列。
海森堡 1928 年提出的海森堡模型（Heisenberg Model）用交换积分 J 描述相邻自旋之间的相互作用能量：E = −2J S₁·S₂。当 J > 0 时，平行排列能量更低——这就是铁磁性的微观起源。
在此之前，人们只知道"铁能被吸引"，但不知道为什么。量子力学给出了答案。

磁化强度 M 与磁场 H

为了描述介质的磁化状态，我们引入磁化强度 M（magnetization），定义为：

M = 单位体积内的净磁矩

单位：A/m（安培/米）。磁化强度 M 与磁感应强度 B 之间通过磁场强度 H 联系：

B = μ₀ (H + M)

对于线性、均匀、各向同性的磁介质：

M = χm H ⇒ B = μ₀ (1 + χm) H = μ₀ μr H

其中：

  • χm —— 磁化率（magnetic susceptibility）
  • μr = 1 + χm —— 相对磁导率（relative permeability）
  • μ = μ₀ μr —— 绝对磁导率（permeability）
材料 相对磁导率 μr 分类
真空 1（精确值） —
铜（Cu） 0.99999 抗磁
铝（Al） 1.00002 顺磁
铁（Fe） ≈ 5000 铁磁
硅钢（变压器用） ≈ 10⁴ 铁磁
坡莫合金（Supermalloy） ≈ 10⁶ 铁磁

铁磁体的磁滞回线

铁磁材料的 M-H 关系不是线性的，而是具有磁滞（hysteresis）现象。当外磁场 H 从零增大时，磁化强度沿一条曲线上升；当 H 减小回零时，M 并不沿原路返回——而是留下"记忆"。

磁滞回线的关键参数：

  • 饱和磁化强度 Ms：所有磁畴方向与 H 完全一致，再增大 H 也无法增加 M。
  • 剩余磁化强度（剩磁）Br：H 降为零后残留的磁感应强度。
  • 矫顽力 Hc：将 B 降低到零所需的反向 H 的大小。

根据矫顽力大小，铁磁材料分为：

  • 软磁材料（Hc 小，≈ 1-100 A/m）：容易磁化、容易退磁。用于变压器、电机铁芯。代表：硅钢、坡莫合金。
  • 硬磁材料（Hc 大，≈ 10⁴-10⁶ A/m）：难以退磁，适合做永磁体。代表：钕铁硼（NdFeB）、钐钴（SmCo）、铁氧体磁铁。
💡 如何选择软磁还是硬磁？
- 变压器铁芯：必须用软磁。变压器铁芯需要交变磁化，响应快、损耗小。软磁的磁滞回线窄，面积小（损耗 ∝ 回线面积）。
- 永磁电机转子：必须用硬磁。转子需要维持恒定的磁场——高剩磁、高矫顽力。
- 电磁屏蔽：用高 μr 的软磁（如坡莫合金），将磁场"吸入"屏蔽材料中。
记住一个简单的判断：如果系统需要频繁开关或交变工作，用软磁；如果系统需要保持恒定磁场，用硬磁。

磁路设计

磁路（magnetic circuit）的概念是电路理论的类比，在工程设计中极其有用：

电路量 磁路对应量
电动势 EMF（V） 磁动势 MMF = N I（A·匝）
电流 I（A） 磁通 Φ（Wb）
电阻 R = L/(σA)（Ω） 磁阻 ℛ = L/(μA)（H⁻¹）
欧姆定律 V = I R 霍普金森定律 MMF = Φ ℛ

霍普金森定律（Hopkinson's Law）：MMF = N I = Φ ℛ，磁阻 ℛ = L / (μ A)。这与电阻公式高度相似——只是电导率 σ 换成了磁导率 μ。

磁路分析在以下场景中非常实用：

  • 变压器铁芯磁通计算
  • 电感器气隙设计（加入气隙可降低有效磁导率，防止饱和）
  • 电机磁路设计
  • MRI 超导磁体设计

Python 模拟：磁滞回线

import numpy as np
import matplotlib.pyplot as plt

def hysteresis(H, Ms, a, alpha, c):
    """
    简化的 J-A 磁滞模型 (Jiles-Atherton) 的单调分支。

    参数:
        H: 外磁场数组
        Ms: 饱和磁化强度
        a: 磁畴壁参数
        alpha: 磁畴间耦合
        c: 可逆磁化比例

    返回:
        M: 磁化强度
    """
    M = np.zeros_like(H)
    for i in range(1, len(H)):
        dH = H[i] - H[i-1]
        if abs(dH) < 1e-10:
            M[i] = M[i-1]
            continue
        # 无磁滞磁化 (Langevin 函数)
        He = H[i-1] + alpha * M[i-1]
        Man = Ms * (np.cosh(He/a)/np.sinh(He/a) - a/He) if He != 0 else 0
        Man = np.clip(Man, -Ms, Ms)

        # 不可逆磁化分量
        delta = np.sign(dH)
        k = a  # 简化：损耗系数
        dMirr = (Man - M[i-1]) / (delta * k - alpha * (Man - M[i-1])) * dH

        # 可逆磁化分量
        dMrev = c * (Man - M[i-1])

        M[i] = M[i-1] + dMirr + dMrev
        M[i] = np.clip(M[i], -Ms, Ms)

    return M

# 生成数据
H = np.linspace(-2000, 2000, 1000)
Ms, a, alpha, c = 1.6e6, 500, 1e-3, 0.1

M_up = hysteresis(H, Ms, a, alpha, c)
M_down = hysteresis(H[::-1], Ms, a, alpha, c)[::-1]

plt.figure(figsize=(8, 6))
plt.plot(H, M_up, 'b-', label='上升支')
plt.plot(H, M_down, 'r--', label='下降支')
plt.axhline(0, color='gray', linewidth=0.5)
plt.axvline(0, color='gray', linewidth=0.5)
plt.xlabel('磁场强度 H (A/m)')
plt.ylabel('磁化强度 M (A/m)')
plt.title('Jiles-Atherton 磁滞回线模型')
plt.legend()
plt.grid(alpha=0.3)
plt.show()
⚠ 工程警示：高 μr 不等于高效率
变压器铁芯选择硅钢而非纯铁的原因很值得思考：纯铁 μr 更高，为什么不用？
因为变压器工作在交变磁场中，涡流损耗不可忽略。纯铁的电阻率低（≈ 10⁻⁷ Ω·m），涡流大；硅钢掺入 3% 硅后电阻率提升约 4 倍，且叠片结构进一步抑制涡流。
工程设计永远是多目标权衡：高 μr 减少励磁电流，但电阻率影响涡流损耗，矫顽力影响磁滞损耗，饱和磁感应强度限制工作范围——没有完美的材料，只有适合的工程选择。

本章小结

  • 磁性的微观起源是电子的轨道磁矩和自旋磁矩，玻尔磁子 μB 是磁矩的基本单位。
  • 三种磁性：抗磁性（所有材料，χ < 0）、顺磁性（未配对电子，χ > 0 且 χ ∝ 1/T）、铁磁性（交换耦合，χ >> 0）。
  • B = μ₀(H + M)；磁化率 χm 和相对磁导率 μr 描述材料对磁场的响应。
  • 铁磁体有磁滞回线：饱和磁化、剩磁、矫顽力；软磁和硬磁各有用途。
  • 磁路设计（霍普金森定律）是电路理论在磁学中的直接类比，变压器和电感器设计的基础。

至此，我们已经建立了静电场和静磁场的完整图像——包括场在真空中的行为、在场源（电荷、电流）附近的分布，以及物质如何响应这些场。接下来，让磁场开始变化吧——电磁感应，这才是电磁学真正闪耀的领域。
