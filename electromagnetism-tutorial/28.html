项目A：Qi无线充电线圈设计与仿真

Qi无线充电标准已发展到Qi 1.3版本（2025年发布），工作频率110-205kHz，支持5-15W功率传输。本项目将设计一个符合Qi标准的基本充电线圈系统。

设计规格：螺旋线圈，电感值 L≈24μH，在100kHz处串联谐振（配谐振电容 C=100nF），基板FR4。使用COMSOL AC/DC模块进行二维轴对称或三维仿真。

仿真流程：

  1. 构建线圈几何结构——设定匝数（通常8-12匝）、线圈内外半径、线径和间距。
  2. 设置材料参数——铜线圈（电导率 σ=5.8×10⁷ S/m）、FR4基板（εr=4.4, tanδ=0.02）、空气域（包围线圈，设置PML边界）。
  3. 添加物理场——磁场（mf）接口，选定频域研究，在100kHz激励。
  4. 仿真得到磁场B分布——观察磁感线从发射线圈延伸到接收线圈的路径，确认耦合。
  5. 参数扫描——改变线圈间距（2mm、5mm、10mm、15mm、20mm），计算耦合系数 k = M/√(L₁L₂)。
  6. 分析负载调谐曲线——在不同负载电阻下计算传输效率 η = Pout/Pin。

关键分析：线圈对齐误差对传输效率的影响。Qi标准要求在线圈偏移达到线圈直径的50%时仍能保持至少5W的传输功率。仿真可以量化的数据包括：轴向偏移（水平错位）和角度偏移（倾斜角）对耦合系数 k 的衰减曲线。

import numpy as np

# 螺旋线圈电感近似计算 (Wheeler公式)
# N: 匝数, R: 平均半径(m), a: 线圈截面半径(m)
def spiral_inductor(N, R_avg_mm, a_mm):
    """计算圆形螺旋线圈电感 (μH)"""
    R = R_avg_mm * 1e-3  # 转米
    a = a_mm * 1e-3
    # Wheeler公式: L = μ₀ * N² * R² / (0.8*R + 0.2*a)
    mu0 = 4 * np.pi * 1e-7
    L = mu0 * N**2 * R**2 / (0.8 * R + 0.2 * a)
    return L * 1e6  # 转 μH

# 设计参数试算
for N in range(8, 13):
    L = spiral_inductor(N, R_avg_mm=15.0, a_mm=1.0)
    print(f"N={N:2d} → L={L:.2f} μH  → C={1/(4*np.pi**2*100e3**2*L*1e-6)*1e9:.1f} nF")

# 互感和耦合系数
def mutual_inductance(L1, L2, k):
    return k * np.sqrt(L1 * L2)

# 效率估算 (谐振电容匹配)
def efficiency(k, Q1=100, Q2=100):
    """临界耦合条件下的最大传输效率"""
    eta = (k**2 * Q1 * Q2) / (1 + np.sqrt(1 + k**2 * Q1 * Q2))**2
    return eta * 100  # 百分比

for k in [0.1, 0.2, 0.3, 0.5, 0.7]:
    print(f"k={k:.1f} → η={efficiency(k):.1f}%")

项目B：2.4GHz微带天线阵

WiFi 6E和未来的WiFi 7对天线增益和方向图提出了更高要求。本项目设计一个工作在2.4-2.48GHz频段的1×4微带贴片天线阵列。

设计规格：频率范围2.4-2.48GHz，增益>12dBi，输入反射系数S₁₁<−15dB，极化方式为线极化。

设计步骤：

  1. 单贴片设计：使用传输线模型或腔模模型估算贴片宽度 W = c/(2f)√(2/(εr+1)) 和长度 L = c/(2f√(εeff)) − 2ΔL（边缘场修正）。在HFSS中建立单贴片模型，优化贴片宽度使工作频率落在2.45GHz。
  2. 馈电设计：采用微带线侧面馈电或同轴探针馈电。对于50Ω端口匹配，设计合适的馈入深度。常用方法：嵌入缝隙馈电（inset feed），通过调整缝隙深度 d 控制输入阻抗。
  3. 阵列排布：1×4阵元间距选择 dsp = 0.7λ₀ ≈ 86mm（@2.45GHz）。过密的间距导致互耦增加，过疏产生栅瓣。采用等幅等相馈电（corporate feed网络）。
  4. 馈电网络设计：用T型功分器将输入功率均分为4路，每路微带线的长度需保持相位一致（电长度相同）。
  5. 全波仿真验证：仿真S参数（确认S₁₁<−15dB且所有端口匹配）、3D方向图（确认最大增益方向、副瓣电平）和表面电流分布。

优化参量：贴片宽度控制谐振频率，嵌入馈电深度控制阻抗匹配，阵元间距平衡阵列增益和栅瓣抑制。可以利用第27章的AI方法加速参数扫描。

项目C：X波段超表面隐身

本项目将超构材料与天线工程结合，设计一个在X波段（8-12GHz）实现雷达散射截面（RCS）减缩的超表面隐身罩。

概念：传统Salisbury屏通过在金属表面λ/4处放置电阻层实现谐振吸波，但带宽极窄。而梯度超表面通过相位梯度将入射波的能量偏转到非镜面反射方向，等效于降低了后向RCS。

设计流程：

  1. 单元设计：设计一个"方形环"或"耶路撒冷十字"结构的单元贴片，在CST/COMSOL中仿真单单元的S参数，提取其在8-12GHz的反射相位 φ(f)。
  2. 相位梯度设计：设计4-8个不同尺寸的单元，使其反射相位覆盖0-2π范围。按照相位梯度 dΦ/dx 的要求排列单元。例如，要在10GHz实现反射波偏转30°，需要的相位梯度为 dΦ/dx = (2π/λ)sin30° × 2（系数2来自反射双程路径）。
  3. 超表面构建：在HFSS/CST中构建一个包含50-100个单元的超表面阵列（例如10×10），设置Floquet端口和周期性边界。
  4. RCS验证：用平面波照射超表面，计算单站RCS（monostatic RCS）相对于同尺寸金属平板的减缩量。目标：在8-12GHz内实现>10dB的RCS减缩。

挑战：带宽与RCS减缩深度之间的权衡。薄型超表面的带宽受限于其Q值，通常需要多层设计或电路模拟吸收体（CAA）技术来扩展带宽。建议从窄带目标起步（如10%带宽、中心频率10GHz），逐步迭代。

从仿真到实物

仿真只是第一步。真正考验一名电磁工程师的，是让实物与仿真吻合的过程。

原型制作：对于PCB微带天线和阵列，推荐JLCPCB或OSH Park进行打样。关键注意事项：（1）确认板材的介电常数 εr 和损耗角正切 tanδ 是否与仿真中使用的参数一致——不同批次FR4的εr可以在4.2到4.8之间变化；（2）铜箔厚度（通常1oz=35μm）对微带线特性阻抗有显著影响；（3）SMA连接器的焊盘寄生电容将影响高频匹配。

测试：矢量网络分析仪（VNA）是核心测试工具。基本测试步骤：

  • 用校准件（Open/Short/Load/Thru）对VNA进行校准，将参考平面推到连接器端面。
  • 测量S₁₁和S₂₁（对于多端口网络），频率范围覆盖设计频段的2倍以上。
  • 对于天线，还需要在微波暗室中测量方向图和增益。

仿真 vs 实测——不吻合时的调试清单：

  1. 检查板材εr和tanδ——用带状线法或谐振环法实测介电常数。
  2. 检查SMA连接器模型——是否在仿真中包含了连接器的等效寄生参数？
  3. 检查制造公差——PCB蚀刻过刻会导致线宽减小，高频影响显著。
  4. 检查测试环境——屏蔽/接地不良、电缆移动导致的相位漂移、人体接近效应。
⚠ 经验之谈：理论与工程之间的距离
作者第一次独立设计微带天线时，仿真结果堪称完美：S₁₁=−28dB @ 2.45GHz，增益7.2dBi，方向图对称。满怀信心地制作了PCB，焊上SMA连接器，迫不及待地上VNA测试——结果S₁₁只有−8dB，谐振频率漂到了2.65GHz。检查了三天，最终发现问题出在两个地方：（1）仿真中使用了理想εr=4.4的FR4，而实际PCB板材的εr在2.4GHz附近约为4.2；（2）SMA连接器的内针到微带线的过渡区域产生了寄生电感，未在仿真中建模。修正这两个因素后，重仿真与实测高度吻合。理论可以完美，工程不能完美——在仿真和实测之间，误差就是经验积累的空间。

结语：场与光的诗篇

28章，从库仑定律到AI电磁建模，我们走过了一条贯穿300年物理学的道路。库仑在1785年测量的扭秤上的微小偏转，麦克斯韦在1865年写下的那组光辉的方程组，赫兹在1887年第一次产生的无线电波，Pendry和Smith在2000年创造的超构材料，以及在2026年将AI与电磁场建模深度融合的最新研究——这是人类理解和驾驭电磁现象的故事，也是从好奇到技术、从纸面方程到改变世界的发明的缩影。

费曼（Richard Feynman）曾在他著名的《物理学讲义》中这样评价麦克斯韦方程组："从人类历史的更长视角来看——比如说一万年后回头看——毫无疑问，19世纪最重要的事件是麦克斯韦发现了电动力学定律。美国内战在同一时期相比之下会显得微不足道。"这个评价绝非夸张。你正在学习的是人类文明最伟大也最实用的理论体系之一。

作为本课程的最后一页，我想对你说三句话：

  • 理论是优美的，但实践才能让你真正理解它。选一个本章的项目，动手做一遍仿真，如果能，做一块样板并测试。你会发现在动手过程中遇到的问题比读十本书学到的还多。
  • 不要停止追问"为什么"。最好的电磁工程师不是靠背公式，而是靠直觉——那种你看到一个问题就能在脑海里"看见"电场线和磁感线如何分布的能力。这个直觉来自于持续不断地质疑和思考。
  • 这28章只是起点，不是终点。电磁学是一个有300年历史但仍在飞速发展的学科——超构材料、太赫兹技术、量子电磁学、光电融合、AI驱动的设计自动化——未来的机会远比过去多。你正在正确的时间学习正确的知识。
