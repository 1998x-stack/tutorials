13.1 惠更斯-菲涅尔原理与绕射

大家好，我是你们的讲师。上一章我们研究了地面反射——电磁波像光遇到镜面一样被弹开。但如果遇到的是像山峰、楼顶这样的尖锐障碍物呢？它不会完全反射，也不会完全被挡住——电波会"绕过"障碍物继续传播。这就是绕射（diffraction）。

你可能亲身经历过绕射现象：在离WiFi路由器一段距离的位置，中间隔着一堵墙——虽然信号弱了很多，但你没被完全"切断"。这其中有部分信号是穿透墙壁过来的，也有相当一部分是绕射——电波在墙的边缘弯折进了阴影区。

理解绕射的基础是惠更斯-菲涅尔原理（Huygens-Fresnel principle）：

  • 波前上的每一个点都可以看作是一个次级球面波源（惠更斯原理，1678年）
  • 这些次级波在空间中的叠加干涉形成了新的波前（菲涅尔补充，1818年）

当遇到障碍物时，障碍物遮挡了一部分次级波源，未被遮挡的波源继续辐射——于是波"绕"进了阴影区。你不能说绕射是"波拐了个弯"——更准确地说，是未被遮挡的惠更斯次级波源在障碍物后面重构了波前。

对于刃峰绕射（knife-edge diffraction），核心参数是菲涅尔-基尔霍夫绕射参数v：

v = h · √[2(d₁ + d₂) / (λ d₁ d₂)]

其中h是障碍物相对于视距连线的突出高度（正值为遮挡，负值为视线清除），d₁和d₂分别是障碍物到发射点和接收点的距离。v是绕射损耗计算中的"万能参量"。

物理含义：v其实就是障碍物高度相对于第一菲涅尔区半径的归一化——回顾第11章，第一菲涅尔区半径r₁ = √(λd₁d₂/d)，所以v = h·√2/r₁。当v = 0时，障碍物刚好与视线平齐；当v = -1时，障碍物低于视线恰好一个第一菲涅尔区半径——此时绕射损耗几乎为0。

13.2 刃峰绕射模型

单个刃峰（理想化的无限薄、无限长的吸收屏）引起的绕射损耗由菲涅尔积分计算：

J(v) = -20log₁₀{ |[1 - j]/2 · ∫v∞ exp(-jπt²/2) dt| }

看起来很吓人，但工程上有一组简单的近似公式：

v 条件 绕射损耗 (dB) 物理解释
v < -1 视线远高于障碍物 0 dB 自由空间传播，绕射可忽略
-1 ≤ v < 0 视线略高于障碍物 0 - 6 dB 部分菲涅尔区被遮挡
v = 0 障碍物与视线平齐 6 dB 第一菲涅尔区一半被挡
0 < v ≤ 1 障碍物略高于视线 6 - 13 dB 轻微遮挡
v > 1 障碍物显著高于视线 13 + 20log₁₀(v) dB "深阴影区"，损耗快速增加

当v > 1时，近似公式J(v) ≈ 13 + 20log₁₀(v)非常准确——这是每个链路设计工程师都应该背下来的经验公式。

故事：阿尔卑斯山的奇迹——1930年代，瑞士邮政电话电报局（Swiss PTT）需要在阿尔卑斯山区建立VHF中继通信网络。阿尔卑斯山峰峦叠嶂，视线传播几乎不可能。工程师们发现：即使发射机和接收机分别在两个山谷、中间隔着巨大的山峰，信号仍然可以通过刃峰绕射保持通信。瑞士工程师率先系统地利用绕射原理设计VHF中继网络——当需要翻越一座山峰时，将天线架设在山峰附近的侧面，利用山峰的"刃峰绕射"来覆盖背面的山谷。这个经验后来被写入了ITU-R P.526标准，成为全球微波链路设计的经典方法。

多重刃峰绕射是更贴近实际的模型——信号路径上通常不止一座山或一个障碍物。工程中有三种主要方法：

  • Bullington方法（1946年）：将所有障碍物用一个单一的"等效刃峰"代替。优点：极其简单。缺点：过于乐观，在深遮挡条件下误差很大。
  • Deygout方法（1966年）：找到主要障碍物（遮挡最严重的），然后递归地考虑次要障碍物（在主要障碍物两侧分别重新计算）。优点：物理直觉清晰。缺点：对某些几何配置过估计损耗。
  • Giovaneli方法（1980s）：改进了Deygout方法，修正了过估计问题，在ITU-R P.526中被推荐使用。

13.3 球形地面绕射

刃峰模型假设障碍物是无限薄的屏。但地球本身是一个巨大的球体——当信号传播距离远超视距时（比如海上或沙漠），地球曲率本身成为最主要的"障碍物"。

球面绕射（spherical earth diffraction）的数学求解比刃峰绕射复杂得多——它需要求解球体表面的绕射问题（Watson变换，1918年）。ITU-R P.526提供了基于绕射损耗因子G的工程计算模型，其中包含了地球曲率、大气折射、极化和频率的影响。

当链路工作在超视距（beyond line-of-sight, BLOS）条件时，除了球面绕射外，还有一种"额外"的传播机制——对流层散射（tropospheric scatter）。

对流层散射的原理是：在对流层中（距地面约10-15 km高度），大气湍流造成局部折射率的不均匀性——这些不均匀体就像微小的散射体，将入射电磁波的能量散射到各个方向，其中有一部分能量被散射到地平线以下的地面接收点。

下表总结了各频段常见的超视距传播机制：

频段 频率范围 主要超视距机制 典型跨距 (km) 可靠性
MF (中波) 300-3000 kHz 地表波、天波（电离层D/E层） 100-2000 高（白天地波+夜间天波）
HF (短波) 3-30 MHz 电离层天波反射 500-4000 中等（受太阳活动影响）
VHF (米波) 30-300 MHz 对流层散射、绕射 100-800 低-中等
UHF (分米波) 300-3000 MHz 绕射（球面+刃峰） 50-300 低（需高增益天线）
SHF (厘米波) > 3 GHz 3-30 GHz 雨散射、大气波导 30-200 很低（依赖气象条件）
对流层散射不是免费的午餐：虽然它实现了超视距，但信号损耗巨大——典型的超视距散射链路损耗比自由空间高出60-100 dB。一个60 km的散射链路，路径损耗可能达到200 dB以上，需要数千瓦的发射功率和巨型天线。历史上，冷战期间北约在格陵兰、冰岛和英国之间建立了"冰岛-法罗群岛-英国"对流层散射链路（ACE High系统），支援功率达10 kW。今天，对流层散射仍被用于某些军事和偏远地区的通信。

13.4 多径传播与衰落

绕射和反射讨论的都是单一传播机制。但在真实的无线环境中——特别是城市、室内等复杂场景——信号沿多个不同的路径到达接收端：可能有直达波、地面反射波、建筑物侧面反射波、屋顶绕射波等等。这就是多径传播（multipath propagation）。

多径的后果是衰落（fading）——接收信号幅度随时间或位置剧烈波动。分为两大类：

平坦衰落（flat fading）：所有频率分量经历相同的衰落。通常发生在当信号带宽远小于信道相干带宽时。特点是：信号整体上下浮动，但频谱形状不变。

频率选择性衰落（frequency-selective fading）：不同频率分量经历不同的衰减。发生在信号带宽大于信道相干带宽时。特点是：某些频率被严重衰减，信号失真。这是在宽带通信中（如OFDM/5G NR）需要特别处理的问题。

描述多径信道的关键参数是时延扩展（delay spread） σ_τ。它是不同路径到达时间差异的标准差。时延扩展的倒数乘以一个系数可以得到相干带宽（coherence bandwidth）：

Bc ≈ 1 / (5στ)

这告诉我们：如果信道的时延扩展很大（例如在山区或城市峡谷中），相干带宽就很小——意味着即使一个带宽不宽的信号也可能经历频率选择性衰落。

工程经验值：典型的室内环境σ_τ ≈ 10-50 ns，对应相干带宽4-20 MHz（所以WiFi在20 MHz带宽下需要OFDM来对抗选择性衰落）。室外城市环境σ_τ ≈ 0.5-5 μs，对应相干带宽约40-400 kHz。所以LTE/5G的子载波间隔（15-30 kHz）远小于相干带宽——每个子载波经历平坦衰落，大大简化了均衡器的设计。

13.5 移动环境中的散射

除了反射和绕射，散射（scattering）是另一种重要机制。当电磁波遇到尺寸远小于波长的物体时——例如雨滴、树叶、路面碎石——会发生散射。散射将能量重新分配到各个方向。

描述单个散射体引起的散射信号强度的工具是双站雷达方程（bistatic radar equation）：

Pr = PtGt/(4πd₁²) · σRCS · Grλ²/(4π)² · 1/(4πd₂²)

其中σ_RCS是目标物体的雷达散射截面（Radar Cross Section, RCS），d₁和d₂分别是发射机和接收机到散射体的距离。

城市环境中的常见散射体包括：

  • 建筑物：外墙的窗户、金属框架、不规则表面都产生大量散射
  • 车辆：运动的金属体，RCS可达10-100 m²（在UHF频段），是移动通信中主要的动态散射源
  • 植被：树叶和树枝在风中晃动，产生随时间变化的散射
  • 人体：特别是在室内/手持通信中，人体本身就是强散射体（含水量高，介电常数大），对天线近场有显著影响

散射的一个重要工程影响是角度扩展（angular spread）——到达接收端的信号分布在多个方向上，而不是单一方向。角度扩展越大，天线阵列的空间分集和MIMO信道容量就越大。某种意义上，散射是MIMO的"朋友"——没有散射，MIMO的信道矩阵将是秩为1的，无法支持多流传输。

下面用Python来模拟双射线模型下的干涉模式和路径损耗变化：

import numpy as np

def two_ray_path_loss(d, h_tx, h_rx, f):
    """
    双射线模型路径损耗计算
    返回距离数组对应的路径损耗
    """
    c = 3e8
    lam = c / f
    # 直达路径长度
    d_direct = np.sqrt(d**2 + (h_tx - h_rx)**2)
    # 反射路径长度 (假设地面为理想导体，反射系数 = -1)
    d_reflect = np.sqrt(d**2 + (h_tx + h_rx)**2)
    # 路径差
    delta_d = d_reflect - d_direct
    # 相位差
    phi = 2 * np.pi * delta_d / lam
    # 场强叠加 (理想反射: 反射幅度=1, 相位反转pi)
    # 接收场强 E_r = |E_direct - E_reflect * exp(-j*phi)|
    E_direct = 1.0 / d_direct
    E_reflect = 1.0 / d_reflect
    E_r = np.abs(E_direct - E_reflect * np.exp(-1j * phi))
    # 路径损耗 (相对于1m处自由空间场强归一化)
    PL = -20 * np.log10(E_r * d[0] if len(d) > 0 else 1)
    return PL

def simulate_interference_pattern():
    """模拟双射线干涉与自由空间的对比"""
    f = 2.4e9   # WiFi频段
    h_tx = 2.0  # 路由器高度 2m
    h_rx = 1.2  # 接收天线高度 1.2m (桌面)
    d = np.linspace(1, 50, 500)  # 距离1-50m

    PL_two_ray = two_ray_path_loss(d, h_tx, h_rx, f)

    # 自由空间路径损耗作为参考
    PL_fs = 20 * np.log10(4 * np.pi * d * f / 3e8)

    print("距离(m) | 双射线PL(dB) | 自由空间PL(dB) | 差值(dB)")
    print("-" * 55)
    for dist in [5, 10, 15, 20, 25, 30]:
        i = np.argmin(np.abs(d - dist))
        diff = PL_two_ray[i] - PL_fs[i]
        print(f" {dist:4d}   | {PL_two_ray[i]:10.1f}  | {PL_fs[i]:10.1f}      | {diff:+.1f}")
    # 输出示例:
    # 距离(m) | 双射线PL(dB) | 自由空间PL(dB) | 差值(dB)
    # --------------------------------------------------------
    #     5   |       52.3   |       54.0      | -1.7
    #    10   |       63.4   |       60.0      | +3.4
    #    15   |       78.6   |       65.6      | +13.0
    #    20   |       84.2   |       68.0      | +16.2
    #    25   |       85.1   |       70.4      | +14.7
    #    30   |       94.2   |       72.0      | +22.2

    # 结论: 双射线模型的损耗远大于自由空间，且不是单调的
    # 在某些距离上干涉增强(差值负)，在其他距离上干涉抵消(差值正)

simulate_interference_pattern()
MIMO与散射的关系：你可能听说过MIMO（多输入多输出）天线技术——它是4G/5G的核心技术之一。MIMO的吞吐量增益来源于空间复用（spatial multiplexing）：多个数据流同时在同一频段发送，通过不同的空间路径被接收机区分。而这一技术成立的前提是散射环境足够丰富，使得不同收发天线对之间的信道衰落系数相互独立。如果散射体太少（比如沙漠或海上），MIMO的信道矩阵是病态的，空间复用的效果会大打折扣。这就是为什么MIMO在密集城区效果最好、在开阔郊区效果一般。

绕射让电波"翻山越岭"进入阴影区，散射把能量散布到四面八方，多径则让接收信号像水面上的涟漪一样起伏不定。这一章我们学会了如何用刃峰模型估算绕射损耗，了解了对流层散射如何实现超视距通信，也看到了多径传播如何导致衰落——从时延扩展到相干带宽，从平坦衰落到频率选择性衰落。

以上三章（自由空间、地面反射、绕射散射）的传播机制，在实际的通信系统设计中是需要综合使用的。下一章我们将把所有传播机制整合起来，介绍移动通信信道模型——从大尺度路径损耗模型、阴影衰落到小尺度衰落，直到3GPP/ITU的标准信道模型。你将看到，前面学到的所有传播机制是如何被凝结为标准化的工程模型的。
