从噪声中提取信号的"形"与"相"

大家好，我是你们的讲师。上一章我们学习了窄带信号的I/Q表示和复包络，掌握了将高频信号"搬"到基带处理的技术。但有一个问题我们还没有回答：当窄带信号经过无线信道到达接收端时，它的幅度和相位发生了什么？特别是，当信号本身是高斯随机过程时，它的包络和相位服从什么分布？这可不是一个理论游戏——它直接决定了无线通信系统的误码率、信道容量和链路预算。

想象一下：你的手机距离基站1公里，信号强度大致是-80dBm。但如果你移动10厘米——半个波长——信号强度可能突然掉到-90dBm甚至更低。这就是多径衰落的结果：电磁波经过不同的反射路径到达接收天线，相互叠加或抵消。要想定量描述这种衰落，我们就必须研究窄带高斯信号的包络特性。本章将介绍三个极其重要的分布：瑞利分布（纯噪声/纯衰落场景）、均匀分布（随机相位）和莱斯分布（信号+噪声场景）。这些分布是无线通信信道建模的基石。

窄带高斯噪声的数学模型

设 n(t) 是一个零均值平稳窄带高斯随机过程，其中心频率为 f₀。根据上一章的结论，我们可以将它写成：

n(t) = I(t) · cos(2πf₀t) - Q(t) · sin(2πf₀t)
     = A(t) · cos(2πf₀t + Φ(t))

其中：
A(t) = √(I²(t) + Q²(t))  — 包络（envelope）
Φ(t) = arctan(Q(t)/I(t)) — 相位（phase）
I(t) 是同相分量，Q(t) 是正交分量

关键事实：对于零均值平稳窄带高斯过程，它的同相分量 I(t) 和正交分量 Q(t) 是独立的零均值高斯随机过程，且具有相同的方差 σ²：

I(t) ~ N(0, σ²),   Q(t) ~ N(0, σ²)
E[I(t)] = E[Q(t)] = 0
Var[I(t)] = Var[Q(t)] = σ²
E[I(t)·Q(t)] = 0   （同一时刻不相关，对高斯而言等价于独立）

这个结论在数学上可以这样理解：窄带高斯过程的正频率部分被搬移到基带后，其实部和虚部（即I和Q）在任意时刻是相互独立的高斯随机变量。这是后续所有推导的出发点。

包络的瑞利分布

纯噪声：包络的分布

在只有噪声（没有信号）的情况下，包络 R = √(I² + Q²) 的分布是什么？这等价于问：两个独立同分布零均值高斯随机变量的模长服从什么分布？答案是瑞利分布（Rayleigh distribution）。

设 X = I ~ N(0, σ²), Y = Q ~ N(0, σ²)，且独立
则 R = √(X² + Y²) 的概率密度函数为：

f_R(r) = (r/σ²) · exp(-r²/(2σ²)),   r ≥ 0

累积分布函数：F_R(r) = 1 - exp(-r²/(2σ²)),   r ≥ 0

瑞利分布的PDF形状很有意思：它在 r=0 处值为0，然后迅速上升到峰值，再缓慢衰减——是一种"单峰右偏"的分布。它的关键数字特征为：

  • 均值：E[R] = σ·√(π/2) ≈ 1.253σ
  • 中位数：R_med = σ·√(2ln2) ≈ 1.177σ
  • 方差：Var[R] = (2 - π/2)σ² ≈ 0.429σ²
  • 众数（峰值位置）：R_mode = σ

注意中位数小于均值——这符合右偏分布的特征。有趣的是，包络的均值 1.253σ 大于各分量的标准差 σ，这意味着"向量的长度"在统计上大于"分量的分散程度"。

物理直觉：瑞利分布描述的是"多个随机散射路径叠加"后的信号幅度。如果你在室内用一个简单的接收机测量信号强度，当发射机和接收机都固定不动时，信号幅度是稳定的。但只要你在房间里走几步——哪怕只是移动几十厘米——接收到的信号幅度就会发生剧烈变化，这种变化的统计规律就是瑞利分布。这就是为什么瑞利分布也被称为瑞利衰落（Rayleigh fading）的物理基础。

瑞利衰落信道的物理基础

在无线通信中，当发射机和接收机之间没有直射路径（line-of-sight, LOS），只有通过建筑物、地面、树木等反射的非视距（NLOS）路径时，接收信号的包络服从瑞利分布。这是因为大量散射路径的叠加效果，在复平面上的投影等价于两个正交的独立高斯随机变量——即同相和正交分量。

瑞利衰落是无线信道的最基本模型，它的"恶劣程度"可以作为其他信道模型的基准。在瑞利衰落信道中，接收信号的信噪比（SNR）服从指数分布——这意味着有相当大的概率出现深度衰落（信号强度远低于平均值的状态）。

相位的均匀分布

纯噪声：相位均匀分布

对于纯窄带高斯噪声，相位 Φ = arctan(Q/I) 服从 [0, 2π) 上的均匀分布：

f_Φ(φ) = 1/(2π),   φ ∈ [0, 2π)

这个结论的直觉非常简单：由于 I 和 Q 是独立同分布的零均值高斯随机变量，它们的联合分布在 (I,Q) 平面上是圆对称的（等高线是同心圆）。在这种圆对称下，角度 Φ 自然就是均匀分布的——没有任何方向是特殊的。

包络与相位独立

更重要的一个结论是：在纯噪声情况下，包络 R 和相位 Φ 是相互独立的随机变量。这个独立性可以直接从联合分布分解中得出：

f_{I,Q}(i,q) = (1/(2πσ²)) · exp(-(i²+q²)/(2σ²))

作变量代换：i = r·cos(φ), q = r·sin(φ)，雅可比行列式为 r，得到：
f_{R,Φ}(r,φ) = r · f_{I,Q}(r·cos(φ), r·sin(φ))
             = (r/(2πσ²)) · exp(-r²/(2σ²))
             = [r/σ² · exp(-r²/(2σ²))] · [1/(2π)]
             = f_R(r) · f_Φ(φ)

证毕：包络与相位独立！

这个独立性在实际分析中非常有用——意味着我们可以分别处理幅度衰落和随机相位，而不必担心两者之间的耦合。

信号+噪声：莱斯分布

正弦信号加窄带噪声

现在考虑更现实的情况：接收信号中既包含有用信号（通常是正弦波），又包含窄带高斯噪声。设：

x(t) = s(t) + n(t)
     = A·cos(2πf₀t) + I(t)·cos(2πf₀t) - Q(t)·sin(2πf₀t)
     = [A + I(t)]·cos(2πf₀t) - Q(t)·sin(2πf₀t)
     = R(t)·cos(2πf₀t + Φ(t))

此时 I 分量的均值不再是零（因为信号分量贡献了直流偏置 A），而 Q 分量仍然保持零均值。包络的分布变为莱斯分布（Rice distribution），也称为广义瑞利分布。

包络 R = √((A+I)² + Q²)，其PDF为：

f_R(r) = (r/σ²) · exp(-(r² + A²)/(2σ²)) · I₀(rA/σ²),   r ≥ 0

其中 I₀(·) 是第一类零阶修正贝塞尔函数，
A 是信号的幅度，σ² 是噪声方差（每分量）。

莱斯因子

莱斯分布的关键参数是莱斯因子（Rice factor）K：

K = A²/(2σ²) = 信号功率/噪声功率（每分量）

- 当 K = 0（即 A = 0，无信号）：莱斯分布退化为瑞利分布
- 当 K → ∞（噪声可忽略）：包络趋近于一个确定值 A，但仍有随机相位
- 典型值：室外微蜂窝 K ≈ 6~12dB，室内环境 K ≈ 4~10dB

莱斯因子 K 是信道质量的"晴雨表"：K 越高，说明直射路径的能量远大于散射路径的总能量，信道质量越好；K 越低，说明信道接近瑞利衰落，通信可靠性越差。

特性瑞利分布（K=0）莱斯分布（K>0）
PDF公式f(r)=r/σ²·exp(-r²/(2σ²))f(r)=r/σ²·exp(-(r²+A²)/(2σ²))·I₀(rA/σ²)
峰值位置r=σ靠近 A（随K增大趋于A）
形状单峰右偏单峰，K越大越对称
均值σ·√(π/2)σ·√(π/2)·₁F₁(-½;1;-K)（合流超几何函数）
应用场景NLOS、室内、城市密集区LOS、卫星通信、微蜂窝
衰落深度深衰落概率高深衰落概率低（K越大越低）
极限行为K=0的特例K→∞ 包络趋近于确定值A
实用技巧：在MATLAB或Python中生成瑞利和莱斯衰落信道非常容易。瑞利：r = sigma * sqrt(randn² + randn²)。莱斯：r = sqrt((A+sigma*randn)² + (sigma*randn)²)。这些简单几行代码就可以模拟出真实的无线信道衰落效应——很多通信系统仿真都是从这开始的。

包络分布的应用：误码率分析

有了包络的分布，我们就可以计算无线通信系统的平均误码率（BER）。以BPSK调制为例，在加性高斯白噪声（AWGN）信道中，误码率为 P_e = Q(√(2γ))，其中 γ = E_b/N₀ 是信噪比。但在瑞利衰落信道中，γ 本身是一个随机变量（服从指数分布），因此平均误码率需要对 γ 的分布积分：

P_e_avg = ∫₀^∞ Q(√(2γ)) · f_Γ(γ) dγ
         = ½ · (1 - √(γ̄/(1+γ̄)))
         ≈ 1/(4γ̄)   （当 γ̄ ≫ 1 时）

其中 γ̄ 是平均信噪比，f_Γ(γ) = (1/γ̄)·exp(-γ/γ̄) 是指数分布。

从这个公式可以看出一个残酷的事实：在瑞利衰落信道中，平均误码率随信噪比线性下降（1/γ̄），而在AWGN信道中，误码率是指数下降（e^{-γ}）。这意味着，要达到同样的误码率，瑞利衰落信道需要的发射功率要高得多——这就是"衰落代价"（fading penalty）。

踩坑故事 — 低信噪比下误把瑞利当莱斯

某卫星物联网初创公司设计了一个窄带卫星IoT终端，用于在偏远地区收集环境传感器数据。他们的链路预算分析采用了一个莱斯信道模型，假设存在较强的直射路径分量（K=10dB），并据此计算了终端的发射功率需求和电池寿命。

产品原型测试的第一轮结果惨不忍睹：终端的连接成功率只有设计值的60%，并且在阴天条件下几乎完全失联。团队最初怀疑是天线效率问题，换了三种天线设计毫无改善。最终在实地信道测量中发现：设备部署的实际环境中，由于密集的树冠遮挡和建筑物反射，实际信道条件更接近瑞利衰落（K接近0dB），而非设计假定的莱斯信道（K=10dB）。

莱斯因子高估了10dB，导致链路预算中系统余量（link margin）严重不足。更致命的是，莱斯分布的尾部（深衰概率）远小于瑞利分布——在K=10dB的假设下，深度衰落（低于均值20dB）的概率几乎为零，而在实际瑞利信道中，深度衰落的概率高达约2%，这就是连接失败的直接原因。

教训：信道模型的选取直接影响系统设计决策。在不确定是否存在稳定直射路径时，最稳妥的做法是采用瑞利模型（最坏情况）或至少进行实地信道探测。对莱斯因子K的乐观估计会带来系统余量不足——而在物联网场景中，链路预算不足意味着数十万台的设备可能在某些天气条件下集体失联。永远不要用桌面假设替代实地测量。

本章小结与衔接

本章将高斯过程和窄带信号分析结合起来，推导了窄带高斯信号包络和相位的统计分布。你学到了：纯噪声的包络服从瑞利分布（无线衰落信道的基准模型）、相位服从均匀分布（且包络与相位独立）、有信号存在时包络服从莱斯分布（莱斯因子K衡量直射路径与散射路径的功率比）。最关键的是，我们通过一个踩坑故事警示：错误地假设信道模型会导致系统设计中的灾难性错误。
