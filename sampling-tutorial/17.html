第17章 偏差诊断与校正

大家好，我是你们的讲师。上一章我们了解了非抽样误差的各种类型，但有一个问题悬而未决：当我们拿到一份已经收集好的数据时，如何判断它是否存在偏差？偏差有多大？能不能通过事后手段进行修正？

先给大家讲一个真实却令人痛心的案例。2020年新冠肺炎疫情初期，全球各地的研究人员争分夺秒地了解这种新型病毒。其中一项研究通过社交媒体（Facebook和Twitter）发布了一份在线问卷，采用滚雪球方式扩散，最终收集了3309份有效回复。研究得出的结论是："某种症状在感染者中的比例约为XX%。" 这项研究被多家媒体报道，也影响了早期的临床认知。然而，后续基于大规模系统抽样和有完整检测数据的研究表明，这3309份滚雪球样本与真实总体的偏差极为显著——研究人员后来估算，自选（self-selected）样本仅代表了总体的2.55%。而且最棘手的是：你无法从数据内部判断偏差的方向和大小，因为滚雪球样本的结构是完全未知的。正如后来评论者所说："这数据不可挽回地有偏，再多的统计方法也没法完全校正。"

不可挽回的有偏样本：COVID滚雪球调查的核心问题在于——参与者的特征（活跃的社交媒体用户、有意愿分享健康信息的人群、有网络访问条件的人群）与总体存在系统性差异。这种自选偏差（Self-selection Bias）是最危险的偏差之一，因为它不是随机发生的，而是与调查主题本身相关。如果偏差来自已知变量（如年龄、性别），我们可以用加权来校正；但如果偏差来自与结局变量相关的不可观测因素（比如更关注自身健康的人既更可能参与调查也更容易报告症状），那么偏差就是不可校正的。

选择偏差的识别

偏差识别是校正的第一步。如果不能识别，当然也无从校正。以下是几种最实用的偏差识别方法：

与已知总体分布对比

如果你知道总体的人口学分布（年龄、性别、地区、教育程度等），可以将样本的分布与之进行对比。这是最直观也最常用的方法。

# 比较样本分布与总体分布
import pandas as pd

# 总体年龄分布（来自普查数据）
pop_age_dist = {"18-24": 15, "25-34": 20, "35-44": 18,
                "45-54": 17, "55-64": 15, "65+": 15}

# 样本中的年龄分布
sample_age_dist = {"18-24": 8, "25-34": 25, "35-44": 22,
                   "45-54": 20, "55-64": 15, "65+": 10}

# 计算偏差
for age_group in pop_age_dist:
    diff = sample_age_dist[age_group] - pop_age_dist[age_group]
    print(f"{age_group}: 样本 {sample_age_dist[age_group]}% "
          f"vs 总体 {pop_age_dist[age_group]}%, 偏差 {diff:+.1f}%")
# 结果显示：年轻人占比偏低，社会经验丰富的群体占比略高

R-Indicator（代表性指标）

R-Indicator是衡量样本代表性的一个综合指标。它基于估计的应答倾向得分（Response Propensity）的变异程度。如果应答概率高度变异——即有些人非常愿意回答、另一些人几乎不可能回答——说明样本的代表性差。R-Indicator越接近1，代表性越好；越接近0，代表性越差。

F-检验与卡方检验

对于分类变量，可以用卡方检验来判断样本分布与总体分布是否存在统计上的显著差异。对于连续变量，可以用t检验或F检验。但需要注意：统计显著并不意味着实际重要——在大样本中，微小的偏差也可能被检测为显著，这时需要结合效应量（Effect Size）来综合判断。

诊断方法 适用场景 优点 局限
分布对比 已知总体边缘分布 直观、易于沟通 只能覆盖已知变量
R-Indicator 评估整体代表性 单一指标，便于比较 依赖倾向得分模型
卡方/F检验 假设检验 统计上有依据 大样本下过于敏感
与基准调查比较 可信赖的外部参考 最可靠的基准 外部数据可能不可得

灵敏度分析（Sensitivity Analysis）

当你无法确知偏差的大小时，灵敏度分析是一种评估"偏差对结论有多大影响"的系统方法。它的核心思路是：假设偏差存在一定的方向和大小，然后重新估算结果，观察结论是否改变。

具体操作步骤：

  • Step 1：定义偏差参数——假设未观测到的子群体的均值与观测到的子群体均值相差δ
  • Step 2：设定δ的取值范围（如δ=-0.5, -0.3, 0, 0.3, 0.5个标准差）
  • Step 3：对每个δ值，重新计算校正后的估计值
  • Step 4：观察结论在哪个δ值下会发生翻转
实际操作建议：灵敏度分析的一个重要产出是"Fragility Index"（脆弱性指数）——即你需要多大的偏差才能推翻当前结论。如果只需要很小的偏差就能让结论翻转，说明你的结论非常"脆弱"，对潜在的偏差高度敏感。在报告这类结果时，一定要坦率地呈现灵敏度分析的结果，让读者自己判断。

多重插补（Multiple Imputation）

对于项目无回答（Item Nonresponse），多重插补是目前最主流的处理方法。它的核心思想是：不填补一个值，而是填补多个可能的值，然后合并分析结果，使得标准误中反映填补的不确定性。

多重插补的基本流程：

  1. 插补阶段（Imputation）：建立预测模型（如回归、随机森林），基于回答者的数据预测缺失值的分布。从该分布中抽取m组（通常m=5~20）不同的填补值，生成m个完整数据集。
  2. 分析阶段（Analysis）：对每个填补后的数据集分别进行分析（如计算均值、回归系数等），得到m组估计结果。
  3. 合并阶段（Pooling）：用Rubin规则将m组结果合并为一个最终估计值，并计算考虑插补不确定性的标准误。
# 使用R的mice包进行多重插补（伪代码示意）
library(mice)

# 假设data有缺失值
imputed_data <- mice(data, m=5, method='pmm', seed=123)
# method='pmm'使用预测均值匹配（Predictive Mean Matching）

# 分析每个填补数据集
fit <- with(imputed_data, lm(y ~ x1 + x2))

# 合并结果
pooled <- pool(fit)
summary(pooled)
重要提醒：多重插补有一个关键假设——数据是随机缺失（Missing at Random, MAR）的。MAR意味着缺失的概率只依赖于观测到的变量，不依赖于缺失值本身。如果数据是"非随机缺失"（Missing Not at Random, MNAR），比如高收入人群故意隐瞒收入，那么多重插补仍然会有偏差。在实际应用中，你永远无法完全验证MAR假设——但你可以通过灵敏度分析来评估偏离MAR假设的后果。

倾向得分方法（Propensity Score Methods）

倾向得分是处理选择偏差和无回答偏差的另一个有力工具。它的核心思想是：估计每个个体被选入样本（或回答）的概率，然后用这个概率的倒数作为权重来校正偏差。

倾向得分估计

倾向得分通常用逻辑回归（Logistic Regression）来估计。因变量是一个二值变量（1=回答/参与了调查，0=未回答/未参与），自变量是那些可能影响回答概率的变量。

# Python倾向得分估计示例
import numpy as np
from sklearn.linear_model import LogisticRegression

# X是协变量（年龄、教育、地区等），R是是否回答（0/1）
X = np.array([[...], [...], ...])  # n × p 矩阵
R = np.array([0, 1, 1, 0, ...])    # 回答状态

model = LogisticRegression()
model.fit(X, R)
propensity = model.predict_proba(X)[:, 1]  # P(回答|X)

# 逆概加权（Inverse Probability Weighting, IPW）
weights = 1.0 / propensity
# 在回答者上使用这些权重进行加权估计

逆概率加权（Inverse Probability Weighting, IPW）

IPW的基本思想很直观：一个具有某特征的人只有40%的概率回答，那么他的回答就代表了"他"以及"另外1.5个与他特征相似但没回答的人"，因此权重为1/0.4=2.5。通过这种方式，回答者被加权"代表"那些没有回答的人，从而校正无回答偏差。

倾向得分的陷阱：使用倾向得分方法时有几个常见的坑。第一，如果倾向得分接近0（即某些人几乎不可能回答），IPW权重会变得极大（1/0.01=100），导致方差爆炸。实践中常对极端倾向得分进行截尾处理（如限制在[0.05, 0.95]范围内）。第二，倾向得分模型必须包含正确的协变量——如果遗漏了同时影响回答概率和结局变量的关键变量，偏差仍然存在。第三，倾向得分只能校正基于观测变量的选择偏差，不能校正基于未观测变量的选择偏差。

一个完整的偏差诊断与校正工作流

在实际项目中，偏差诊断与校正通常遵循以下流程：

  1. 第一步——数据审计：计算回答率、检查缺失模式、识别异常值
  2. 第二步——偏差诊断：将样本分布与总体已知分布进行对比，计算R-Indicator或进行统计检验
  3. 第三步——无回答分析：比较早期回答者与晚期回答者（"continuum of resistance"假设——晚期回答者更接近无回答者），分析无回答者的特征推断
  4. 第四步——选择校正方法：根据偏差来源选择加权调整、多重插补、倾向得分方法或组合方法
  5. 第五步——灵敏度分析：评估不同校正方法对结果的影响，测试对未观测偏差的敏感性
  6. 第六步——透明报告：报告回答率、校正方法和关键假设，让读者能够评估结果的可靠性

本章小结

偏差诊断与校正是抽样实践中不可回避的核心环节。我们从COVID滚雪球调查的惨痛教训出发，学到了最重要的一课：预防永远优于纠正——如果样本在源头就存在不可观测的选择偏差，再多的统计方法也无法完全修复。对于可处理的偏差，我们介绍了分布对比、R-Indicator等诊断方法，多重插补和倾向得分加权等校正方法，以及灵敏度分析这一评估工具。关键是要记住：校正方法都有其前提假设（如MAR、positivity），在实际使用中需要用透明和谨慎的态度来处理。

学完偏差诊断和校正，我们的工具箱已经相当丰富了。但还有一种特殊情况——当调查主题本身非常敏感时（比如吸毒、性行为、逃税），直接询问几乎不可能得到真实回答。这就引出了第18章的内容：敏感问题抽样（Randomized Response Technique and Beyond）。
