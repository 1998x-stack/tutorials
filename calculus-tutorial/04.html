无穷小：微积分的基本零件

大家好，我是你们的讲师。前面两章我们建立了极限的严格定义。现在，我们来看一个贯穿整个微积分的关键概念——无穷小。

"无穷小"这个词听起来有点神秘，甚至可能让你想起历史上关于"幽灵般的数量"的争论。但实际上，它在现代微积分中有非常精确的定义：如果一个函数 f(x) 满足 lim f(x) = 0（x → x₀ 或 x → ∞），就称 f(x) 是无穷小量，简称无穷小。

注意一个容易混淆的点：无穷小不是"很小的数"。0.0000001 虽然很小，但它是一个固定的数，不是无穷小。无穷小描述的是函数的一种变化趋势——当自变量变化时，函数值趋向于 0。用中文说就是："要多小有多小"。同样地，如果 lim f(x) = ∞，称 f(x) 是无穷大量。

无穷小和无穷大的关系很简单：互为倒数。如果 f(x) 是无穷小（且不恒为 0），那么 1/f(x) 就是无穷大；反之亦然。这个关系虽然简单，在极限计算中却是一个非常有用的转化技巧——有时把"除零"的问题转化为"乘无穷小"的问题，会让计算简单很多。

高阶、同阶与等价无穷小

同样是趋于 0 的函数，它们趋于 0 的速度是不同的。比如当 x → 0 时：

  • x^2 比 x 趋于 0 更快（平方让数值衰减更快）
  • x^3 比 x^2 更快
  • sin x 和 x 趋于 0 的速度差不多

为了精确比较这种"趋于 0 的速度"，我们引入了阶的概念。设有两个无穷小 α(x) 和 β(x)，考察 lim α(x)/β(x)：

条件 结论 记法 通俗理解
lim α/β = 0 α 是比 β 高阶的无穷小 α = o(β) α 比 β 更快趋于 0
lim α/β = C ≠ 0 α 与 β 同阶无穷小 α = O(β) α 和 β 趋于 0 速度相当
lim α/β = 1 α 与 β 等价无穷小 α ~ β α 和 β 几乎一样
lim α/β = ∞ α 是比 β 低阶的无穷小 β = o(α) α 更慢趋于 0

例：当 x → 0 时：

  • x^2 是 x 的高阶无穷小，因为 x^2/x = x → 0，记作 x^2 = o(x)
  • 3x 与 x 同阶，因为 3x/x = 3 → 3 ≠ 0
  • sin x 与 x 等价，因为 sin x / x → 1（这就是第一个重要极限）

等价无穷小替换法则

在极限计算中，替换法则是实战中用到最多的技巧之一。它的核心思想是：在乘除运算中，可以把一个无穷小替换为它的等价无穷小，极限值不变。

等价无穷小替换法则
如果 α ~ α' 且 β ~ β'，那么 lim (α/β) = lim (α'/β')（前提是比值极限存在，且分母不为 0）。

特别提醒：这个法则只适用于乘除法中的因式替换。在加减法中直接替换等价无穷小可能出错——这是考试中最大的扣分点之一！

常用的等价无穷小（当 x → 0 时，这是一个需要记忆的清单）：

常用等价无穷小（x → 0）：

        sin x ~ x
        tan x ~ x
        arcsin x ~ x
        arctan x ~ x
        ln(1 + x) ~ x
        e^x - 1 ~ x
        1 - cos x ~ (1/2)x^2
        (1 + x)^a - 1 ~ a x      【考研常考！】

        这些公式不需要死记硬背。随着课程深入，你会在导数公式
        中找到它们的来源——本质上每个等价关系都对应一个导数。
血泪教训：加减法中的等价替换
一位学生在考试中遇到 lim_{x→0} (sin x - x)/x^3，他直接把 sin x 替换为 x，得到 (x - x)/x^3 = 0——他以为算出来了，交卷走人了。结果是 0 分。

正确答案是多少？通过后面学的泰勒展开或洛必达法则可以算出，这个极限等于 -1/6。为什么直接替换错了？因为 sin x ~ x 意味着 sin x = x + o(x)，在减法 sin x - x 中，主要部分 x - x 抵消了，剩下的不是 0 而是更高阶的无穷小。等价替换把高阶信息丢掉了。

结论：等价无穷小替换只能用在乘除因式上，不能在加减法里面直接替换。这是微积分极限计算中最常见的错误，没有之一。

第一个重要极限：lim sin x / x = 1

这个极限是三角学与微积分的交汇点。它揭示了当角度极小时，正弦值和角度本身几乎相等——这正是等价无穷小 sin x ~ x 的来源。

几何证明（夹逼法）

考虑单位圆 x² + y² = 1，设圆心角为 x（弧度制）。在图中构造三个面积：

  • 三角形 OAB（顶点在圆心和圆弧两端点），面积 = (1/2) sin x
  • 扇形 OAB，面积 = (1/2) x
  • 大三角形 OAC（C 是过 B 的切线与 OA 延长线的交点），面积 = (1/2) tan x

由几何关系知：(1/2) sin x < (1/2) x < (1/2) tan x。化简得 sin x < x < tan x。

除以 sin x（正数）：1 < x/sin x < 1/cos x。

取倒数：cos x < sin x/x < 1。

当 x → 0⁺ 时，cos x → 1，由夹逼定理：lim_{x→0⁺} sin x/x = 1。由对称性，x → 0⁻ 时也成立。所以 lim_{x→0} sin x/x = 1。

应用举例

例1：求 lim_{x→0} sin(3x)/x

        令 t = 3x，则 x → 0 时 t → 0，
        sin(3x)/x = 3·sin(t)/t → 3·1 = 3

        例2：求 lim_{x→0} tan x/x

        tan x/x = (sin x/x) · (1/cos x) → 1 · 1 = 1

        例3：求 lim_{x→0} (1 - cos x)/x^2

        利用 1 - cos x = 2 sin^2(x/2)，
        原式 = 2 sin^2(x/2)/x^2
            = (1/2)·[sin(x/2)/(x/2)]^2
            → (1/2)·1^2 = 1/2
        这也解释了为什么 1 - cos x ~ (1/2)x^2

第二个重要极限：lim (1 + 1/x)^x = e

如果说第一个重要极限来源于几何，第二个重要极限就来源于金融——虽然发明者雅各布·伯努利可能没想到他在研究一个改变整个数学的数。

从复利模型到 e

故事是这样的：假如你在银行存款 1 元，年利率 100%。按年计息，一年后你有 2 元。如果改为半年计息一次，年利率不变，半年利率 50%，那么一年后你有 (1 + 0.5)^2 = 2.25 元。如果每月计息一次，(1 + 1/12)^12 ≈ 2.613 元。每天计息一次，(1 + 1/365)^365 ≈ 2.7146 元。

如果每秒计息一次呢？你会无限趋于某个数——这个数就是 e ≈ 2.71828...。

e 的直观理解
e 是"连续复利"的极限值。当你把计息时间无限细分时，最终本息和趋向的极限值就是 e。

更广泛地：任何以指数方式增长的量，如果增长率与当前量成正比，那么最终都会和 e 打交道。放射性衰变、人口增长、电容充放电——这些看似不相关的现象背后，都有 e 的身影。

数学形式与证明思路

第二个重要极限有多种等价形式：

  • lim_{x→∞} (1 + 1/x)^x = e
  • lim_{x→0} (1 + x)^(1/x) = e（令 t = 1/x 即可转化）

证明的核心思路是利用单调有界准则证明数列 (1 + 1/n)^n 收敛。可以证明：

  • 这个数列单调增加
  • 有上界（实际上可以证明上界是 3）

因此它收敛，其极限定义为 e。由数列极限通过海涅定理推出函数极限，从而得到上述形式。

应用举例

例1：求 lim_{x→∞} (1 + 1/(2x))^x

        原式 = lim [(1 + 1/(2x))^(2x)]^(1/2)
            = e^(1/2) = √e

        例2：求 lim_{x→0} (1 + 2x)^(1/x)

        原式 = lim [(1 + 2x)^(1/(2x))]^2
            = e^2

        例3：求 lim_{x→∞} ((x+1)/(x-1))^x

        原式 = lim [(1 + 1/x)/(1 - 1/x)]^x
            = e / e^(-1) = e^2

两个重要极限的变形

实际考试中很少直接考标准形式。绝大多数题目都是经过变形后才能套用。你需要掌握的核心技能是"凑"出标准形式。

标准形式 常见变形 做法
sin x / x sin(ax) / (bx) 分子分母同乘 a，化为 (a/b) · sin(ax)/(ax)
sin x / x tan x / x 拆为 (sin x / x)·(1/cos x)
(1+1/x)^x (1 + a/x)^x 化为 [(1 + a/x)^(x/a)]^a → e^a
(1+1/x)^x (1 + 1/x)^(kx) 化为 [(1 + 1/x)^x]^k → e^k

两个重要极限的变形是后续导数与积分计算的基础。请花时间反复练习"凑形式"的过程，直到你能一眼看出一个极限题是否需要用到这两个工具。

在本章中，我们理解了无穷小的精确含义和阶的比较，掌握了等价无穷小替换法则（及其关键限制），并学习和证明了微积分中最著名的两个极限。这两个极限就像微积分工具箱里的"两把万能扳手"——碰上三角函数的极限，先试试 sin x/x；碰上幂指型极限，先试试 (1 + 1/x)^x。
