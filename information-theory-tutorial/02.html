大家好，我是你们的讲师。今天我们来给信息论打一个坚实的数学地基。

概率论是信息论的母语。 没有概率，就没有办法量化不确定性；而不确定性正是信息论要处理的核心对象。你回想一下第1章我们讲的内容——香农对信息的定义是"不确定性的减少"。要度量"不确定性减少了多少"，你首先得能够度量不确定性本身。要度量不确定性，你需要概率论。

对许多读者来说，概率论可能已经是"远古记忆"了——你可能还记得一些公式，但对它们为什么重要缺乏直觉。这一章我们不追求全面的概率论教程，而是从信息论的视角，快速回顾那些最核心、最直接相关的概念。如果你已经熟练掌握了概率论，可以快速浏览；但如果有些概念让你觉得"当时就没太搞懂"，这一章会帮你理清思路。

我当年在学信息论时，教授说了一句话让我印象深刻："信息论就是用概率论的语言，问关于通信的问题。"没有概率论，信息论就失去了一切表达能力。所以，让我们从最基础的概念开始。

随机变量与概率空间

所有概率论的起点都是一个随机试验：抛硬币、掷骰子、明天是否下雨——这些都有一个共同特征：在试验之前，我们无法确定结果。

样本空间 Ω 是所有可能结果的集合。抛一枚硬币，Ω = {正面, 反面}。掷一个六面骰子，Ω = {1, 2, 3, 4, 5, 6}。明天是否下雨，Ω = {下雨, 不下雨}。

事件是样本空间的子集。"抛硬币得到正面"是一个事件，"掷骰子得到偶数点"也是一个事件（它是{2,4,6}）。事件可以用集合运算组合：A ∩ B（A和B同时发生）、A ∪ B（A或B发生）、Aᶜ（A不发生）。

概率测度 P 是一个从事件到[0,1]的映射，必须满足三条概率公理：

  1. 非负性：P(A) ≥ 0，对任意事件A
  2. 规范性：P(Ω) = 1——整个样本空间的概率为1
  3. 可列可加性：如果A₁, A₂, ...是互不相交的事件序列，那么P(∪ᵢ Aᵢ) = ∑ᵢ P(Aᵢ)

这三条公理构成了整个概率论的基石。你学过的所有概率公式——全概率公式、贝叶斯公式、大数定律——都是从这三条公理推导出来的。它就像欧几里得几何的五条公理一样，极简但力量无穷。

随机变量 X 是一个从样本空间 Ω 到实数 ℝ 的函数。也就是说，它对每一个试验结果赋予一个数字。抛硬币时，我们可以定义 X(正面) = 1，X(反面) = 0。掷骰子时，X 就是骰子的点数本身。随机变量不是"随机的变量"——它是一个从样本空间到实数的确定性的函数，只不过它的输入（样本点）是随机选中的。

重要区分：很多初学者把"随机变量"理解为"它的值是随机变化的"。更准确的视角是：样本空间是随机的，但随机变量本身是一个确定的函数。一旦样本点固定了（比如硬币落地了），随机变量的值也就确定了。这个视角在信息论的推导中非常有用。

随机变量分为两大类：

  • 离散随机变量：取值是有限个或可数无穷个。比如硬币结果、骰子点数、一个字符集中的字母。
  • 连续随机变量：取值是不可数无穷个。比如人的身高、温度读数、信号的幅度。

离散随机变量用概率质量函数（PMF）描述：p(x) = P(X = x)。连续随机变量用概率密度函数（PDF）描述：f(x)，概率通过积分求得：P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx。

条件概率与贝叶斯公式

概率论中最强大的概念之一就是条件概率：已知一些信息后，我们对某个事件的认识会发生什么变化？这正是信息论要度量的东西。

条件概率的定义很直观：已知事件B发生的条件下，事件A发生的概率为：

P(A|B) = P(A ∩ B) / P(B)

前提是 P(B) > 0。可以这样理解：当我们知道B发生了，样本空间就缩小到了B——原来的所有可能性都被过滤了，只剩下B中的那些样本点。概率需要重新"归一化"，使得B内部的概率之和为1。除以P(B)就是做这个归一化。

从这个定义出发，可以得到概率论中最优美的公式之一——贝叶斯公式：

P(A|B) = P(B|A) · P(A) / P(B)

贝叶斯公式的魅力在于它做了一件非常"反直觉"的事情：它让我们从结果反推原因。如果A是原因，B是结果——那么P(B|A)是我们已知原因时看到结果的概率（这通常容易建模），而P(A|B)是我们看到结果后对原因的信念更新（这通常是我们要计算的）。

真实故事： 我在做垃圾邮件过滤的项目时，就深刻体会了贝叶斯公式的威力。问题很简单：收到一封包含"免费"和"点击"两个词的邮件，它是垃圾邮件的概率有多大？用贝叶斯公式，我们需要知道三件事：垃圾邮件中这些词出现的频率（P(词|垃圾)）、正常邮件中出现这些词的频率（P(词|正常)），以及垃圾邮件的总体比例（P(垃圾)）。然后计算：

P(垃圾|词) = P(词|垃圾) · P(垃圾) / [P(词|垃圾) · P(垃圾) + P(词|正常) · P(正常)]

这个简单的公式在1990年代末催生了第一代高效的垃圾邮件过滤器——直到今天，朴素贝叶斯分类器仍然是文本分类的强基线模型。

贝叶斯公式在信息论中同样无处不在。在信道解码时，我们观察到接收到的信号，要推断发送的原始符号——这正是贝叶斯推断。在信源编码中，我们需要知道符号的概率分布——这也是贝叶斯视角。实际上，整个信息论可以看作"贝叶斯推断的通信版本"。

期望、方差与矩

一个概率分布包含了很多信息——有时我们需要一些"摘要统计量"来抓住分布的核心特征。

期望 E[X] 是分布的"中心"或"重心"。对于离散随机变量，E[X] = ∑ x · p(x)；对于连续随机变量，E[X] = ∫ x · f(x) dx。期望是线性算子：E[aX + b] = aE[X] + b。

但期望只告诉我们"中心在哪"，没有告诉我们"分布有多分散"。两个分布可能有相同的均值但形状完全不同。

方差 Var(X) = E[(X - E[X])²] = E[X²] - (E[X])² 度量的是分布的"离散程度"。方差越大，数据越分散。标准差σ = √Var(X) 让这个度量回到原始单位。

更一般地，矩（moment）是分布的更广义的描述：

  • 原点矩：E[Xᵏ]，k阶原点矩
  • 中心矩：E[(X - E[X])ᵏ]，k阶中心矩。一阶中心矩为0，二阶中心矩就是方差
  • 三阶中心矩度量偏度（skewness），四阶中心矩度量峰度（kurtosis）

为什么矩在信息论中重要？因为许多最大熵分布（第6章会讲）是通过固定某些矩来确定的。比如，给定均值和方差时，熵最大的分布是正态分布——这就是为什么高斯分布如此普遍。

来看一个Python示例：

import numpy as np

# 定义一个离散分布
# X 取值 1, 2, 3, 4, 5, 6，对应偏倚骰子
values = np.array([1, 2, 3, 4, 5, 6])
probs = np.array([0.1, 0.1, 0.2, 0.3, 0.2, 0.1])

# 期望
expectation = np.sum(values * probs)
print(f"期望 E[X] = {expectation:.2f}")
# 输出：期望 E[X] = 3.70

# 方差（定义法）
variance = np.sum((values - expectation)**2 * probs)
print(f"方差 Var(X) = {variance:.2f}")
# 输出：方差 Var(X) = 1.81

# 方差（快捷公式：E[X²] - (E[X])²）
second_moment = np.sum(values**2 * probs)
variance_alt = second_moment - expectation**2
print(f"方差(快捷法) = {variance_alt:.2f}")
# 输出：方差(快捷法) = 1.81

# 三阶矩（偏度相关）
third_central_moment = np.sum((values - expectation)**3 * probs)
print(f"三阶中心矩 = {third_central_moment:.2f}")
# 输出：三阶中心矩 = -0.33

Jensen不等式与凸函数

现在，我们来到本章中对信息论最重要的一个数学概念——Jensen不等式。

先回忆一下什么是凸函数：一个函数 f 被称为凸函数，如果对于任意 x₁, x₂ 和 λ ∈ [0,1]，有：

f(λx₁ + (1-λ)x₂) ≤ λf(x₁) + (1-λ)f(x₂)

直观理解：函数图像上任意两点的连线，位于函数图像上方。反过来，如果连线在图像下方，则是凹函数。

常见的凸函数有：f(x) = x², f(x) = eˣ, f(x) = -log x（在正数域上）。常见的凹函数有：f(x) = √x, f(x) = log x。

Jensen不等式说：如果 f 是凸函数，那么 f(E[X]) ≤ E[f(X)]。如果 f 是凹函数，不等号方向相反：f(E[X]) ≥ E[f(X)]。

这个不等式看起来简洁，但它贯穿了整部信息论。我们来感受一下它的力量：

  • f(x) = x² 是凸函数，所以 (E[X])² ≤ E[X²] —— 这意味着方差 Var(X) = E[X²] - (E[X])² ≥ 0。成立。
  • f(x) = -log x 是凸函数，所以 -log(E[X]) ≤ E[-log X]，即 log(E[X]) ≥ E[log X]。这个不等式将在熵、互信息、KL散度的几乎所有性质推导中反复出现。
  • 信息论中最常见的技巧之一：看公式里有没有 log 和期望，大概率要上 Jensen 不等式。
Jensen不等式是信息论的"力学定律"： 如果说信息论的三大定理是摩天大楼，Jensen不等式就是支撑它们的地基。在接下来的每一章中，你都会看到它的身影——熵的非负性、互信息的非负性、KL散度的非负性、数据处理不等式——它们的证明都依赖Jensen不等式。

直觉上，Jensen不等式告诉我们：函数值的平均 ≥ 平均值的函数值（对凸函数）。你可以这样记忆：由于凸函数的图像是"向上弯曲"的，函数会在"极端值"处被放大，所以取完函数再平均，会比先平均再取函数得到更大的值。

我们来对比离散和连续随机变量的核心概念：

性质 离散随机变量 连续随机变量
描述方式 概率质量函数 PMF: p(x) 概率密度函数 PDF: f(x)
概率计算 P(X=x) = p(x) P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx
归一化条件 ∑ p(x) = 1 ∫ f(x)dx = 1
期望 E[X] = ∑ x · p(x) E[X] = ∫ x · f(x)dx
常见例子 伯努利分布、二项分布、泊松分布 均匀分布、正态分布、指数分布
在信息论中 香农熵 H(X) = -∑ p(x)log p(x) 微分熵 h(X) = -∫ f(x)log f(x)dx

这一章我们快速回顾了概率论的几个核心概念：概率空间的三条公理、条件概率与贝叶斯公式、期望与方差、以及信息论最重要的数学工具——Jensen不等式。

关键 takeaways：

  • 概率论是信息论的数学语言——没有概率，就无法量化不确定性
  • 条件概率和贝叶斯公式是"从结果推断原因"的强大工具，在信道解码和机器学习中无处不在
  • 期望是"重心"，方差是"分散程度"——它们是对分布最基础的描述
  • Jensen不等式是信息论最核心的数学工具——f(E[X]) ≤ E[f(X)]（对凸函数），几乎所有信息论量的非负性和极值性质都依赖它
