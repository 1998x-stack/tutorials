10.1 口径天线的共同原理

大家好，我是你们的讲师。在前三章中，我们学习了线天线（偶极子、单极子）、微带贴片天线和宽带天线——这些都是通过导体上的电流直接向外辐射电磁波的天线。今天我们要转向另一大类天线——口径天线（aperture antennas）。

什么是"口径"？简单来说，口径就是一个经过精心设计的开口（opening），电磁波通过这个开口辐射到自由空间中去。想象一下：光束从手电筒的反射镜前射出——那个圆形的开口就是"口径"。在无线电世界里，喇叭天线（horn antenna）就是波导末端张开的口，抛物面天线（parabolic dish）就是反射面汇聚波束的那个"虚拟口径"。

口径天线的理论基础是场等效原理（field equivalence principle）——它是惠更斯原理（Huygens principle）在电磁学中的严格数学表述。核心思想是：

  • 在一个封闭曲面（例如口径面）上，我们可以用等效面电流和面磁流来代替实际的天线源
  • 这些面电流/磁流在曲面外部产生的场与原天线产生的场完全相同
  • 这意味着，我们不需要知道天线内部发生了啥——只要知道口径面上的场分布，就能计算出远场

更进一步，远场与口径场分布之间是傅里叶变换关系：

E(θ, φ) ∝ ℱ{Eaperture(x, y)}

也就是说：口径面上的场分布经过一个二维傅里叶变换，就得到了远场方向图。这个关系极具实践意义——它告诉我们：

  • 均匀分布（uniform taper）：口径面上场强恒定 → 最大直接性和最窄的主瓣，但副瓣高（第一副瓣约-13.2 dB）
  • 锥削分布（tapered distribution）：口径中心场强大、边缘弱 → 副瓣降低、波束展宽、直接性下降
工程权衡：口径效率（aperture efficiency）与副瓣电平之间存在不可调和的矛盾。追求最大增益就用均匀分布，牺牲副瓣；追求低副瓣就用锥削分布，牺牲增益。在雷达中，低副瓣优先级高于增益（低副瓣意味着更好的杂波抑制）。在卫星通信中，往往优先优化增益。

10.2 喇叭天线

喇叭天线（horn antenna）是最简单的口径天线——你可以把它理解为"波导的喇叭口"。波导中传播的电磁波在喇叭口处逐渐过渡到自由空间，实现了阻抗匹配和定向辐射。

喇叭天线按张口的形状分为几种类型：

  • E面扇形喇叭（E-plane sectoral horn）：仅在电场方向（E面）展开
  • H面扇形喇叭（H-plane sectoral horn）：仅在磁场方向（H面）展开
  • 角锥喇叭（Pyramidal horn）：两个方向都展开，最常见的类型
  • 圆锥喇叭（Conical horn）：圆形波导末端圆锥过渡
  • 波纹喇叭（Corrugated horn）：内壁加工了环形波纹槽

角锥喇叭的设计有一个经典的最优化问题：给定工作频率和增益目标，需要确定喇叭长度和开口尺寸，使得相位误差（喇叭口面边缘与中心的相位差）不超过可接受的范围。典型的最优设计使最大相位差约为λ/4，此时增益接近理论最大值。

最佳喇叭设计（Optimum horn）：对于角锥喇叭，当E面和H面的相位差均为约0.25λ时，增益达到最大值。如果喇叭继续拉长，增益变化不大；如果喇叭缩短，相位误差增大，增益下降。所以"最佳喇叭"就是增益-尺寸权衡的甜蜜点。

波纹喇叭是一个特别值得讲的故事。在波纹喇叭出现之前，普通喇叭天线的E面和H面方向图宽度不一样（E面窄，H面宽），且交叉极化电平较高。这在卫星通信双极化复用中是一个大问题——交叉极化会恶化极化隔离度。

1970年代，Thomas Minnett等人发现：如果在喇叭内壁加工一圈圈的环形波纹（λ/4深度），边界条件会被改变——E面和H面的方向图变得几乎完全对称，交叉极化极低（<-30 dB）。这个发现使波纹喇叭成为卫星通信馈源的首选。到2026年的今天，绝大多数卫星通信地面站和射电天文望远镜的馈源都是波纹喇叭。

一个划时代的"意外"发现：1964年，美国Bell Labs的两位研究人员Arno Penzias和Robert Wilson正在测试一个用于卫星通信的喇叭天线（Holmdel Horn Antenna）。他们发现无论如何消除干扰，总有一种微弱的"嘶嘶"声无法去除——甚至在清除鸽子和鸽子粪后依然存在。他们最终意识到，这个信号来自宇宙的每一个方向，强度相当于约3 K的黑体辐射——这正是宇宙微波背景辐射（CMB），大爆炸理论的直接证据。1978年，Penzias和Wilson因此获得诺贝尔物理学奖。而那个"惹事"的喇叭天线，至今仍是科学史上最重要的天线之一。

10.3 抛物面反射天线

抛物面反射天线（parabolic reflector antenna）是电磁工程中最经典的"大天线"。它的原理来自中学几何：抛物面将焦点处发出的所有光线反射成平行光线（反之亦然）。在电磁学中，这个性质意味着：

  • 从焦点处馈源（feed）发出的球面波经抛物面反射后变成平面波→形成高度定向的波束
  • 反过来，平行入射的平面波被抛物面汇聚到焦点处的馈源中→实现高增益接收

抛物面天线的关键几何参数是焦径比 F/D（焦距与口径直径之比）：

  • 大F/D（长焦距）：馈源离反射面远，馈源照射更均匀，交叉极化更低，但结构更长
  • 小F/D（短焦距）：结构紧凑，但馈源照射角度大、边缘照射弱、交叉极化增大
  • 典型的F/D范围：0.3 - 0.8

馈源遮挡（feed blockage）是前馈抛物面天线的主要问题。馈源及其支撑结构位于波束路径上，会遮挡一部分口径，导致：损失约1-2 dB的增益，增加副瓣电平。解决方法是偏馈（offset feed）——只使用抛物面的一部分，馈源放在反射面的侧面，完全避开波束路径。偏馈抛物面天线广泛用于家用卫星电视接收（俗称"小锅"），因为它的轮廓线在门窗外墙上看起来不那么突兀。

10.4 卡塞格伦与格里高利天线

以上介绍的都是前馈（front-feed）抛物面天线。但对于大口径高增益天线（比如射电望远镜），把沉重的馈源和接收机放到焦点处非常不方便——你需要在数十米高的地方安装和维护电子设备。因此，工程师们借鉴了光学望远镜的设计思路。

卡塞格伦天线（Cassegrain antenna）

卡塞格伦天线使用双反射面系统：主反射面是抛物面，副反射面是双曲面（hyperboloid）。副反射面位于抛物面的焦点和顶点之间。馈源也不再在焦点处，而是穿过主反射面的中心孔安放在主反射面后方。

工作方式：馈源发出的球面波先经双曲面副镜反射→看起来像是从抛物面焦点处发出的→再经主抛物面反射成为平面波。这样，馈源和接收机都被移到了主反射面后方——方便维护、易于屏蔽、也缩短了馈线长度。

噪声温度的改善：卡塞格伦天线有一个更隐形的优势——馈源指向副镜，副镜正对的是冷空（cold sky），而不是前馈天线那样直接看到热地面（hot ground）。这意味着接收系统的噪声温度更低。对于射电天文和深空通信，每降低10 K的噪声温度都是巨大的收益。

格里高利天线（Gregorian antenna）

格里高利天线是卡塞格伦的"表亲"——它用椭球面（ellipsoid）作为副反射面而不是双曲面。椭球面的好处是：焦点的位置在反射面之前，使得总体结构和布局更加灵活。格里高利天线在某些卫星通信地面站中广泛使用，尤其要求非常低副瓣的应用场合。

10.5 大口径天线的工程挑战

当天线的口径增大到数十米甚至数百米时，一系列新问题出现了。

表面精度是第一大挑战。实际反射面的表面与理想抛物面之间的偏差（RMS误差）会造成增益损失。经典的Ruze公式给出了估计：

G/G₀ = exp[-(4πσ/λ)²]

其中σ是表面RMS误差。当σ = λ/16时，增益损失约0.5 dB；σ = λ/8时，损失约2 dB。工程上的经验准则是：表面精度必须优于λ/16。

对于10 GHz的工作频率（λ=3 cm），这就要求表面RMS误差小于1.9 mm——对一座50米口径的钢结构天线而言，这是一个极其苛刻的精度要求！

重力和热变形是另一个大问题。天线在不同仰角时，重力分布发生变化，反射面形状会发生重力变形（gravitational deformation）。温度变化（特别是日照不均）也会导致热变形。大型天线通常采用以下措施应对：

  • 等刚度设计：采用碳纤维复合材料或空间桁架结构使变形最小化
  • 主动面补偿：实时调整反射面面板的位置（如FAST望远镜的主动反射面）
  • 重力模型校正：事先测量不同仰角下的变形，用数学模型进行补偿
世界著名的巨型天线：
  • FAST（五百米口径球面射电望远镜，中国贵州）：500 m口径，世界最大单口径射电望远镜。利用喀斯特地貌天然凹坑建造，主动反射面实时调节。2020年正式运行，灵敏度是Arecibo的2.5倍。
  • Arecibo（阿雷西博，波多黎各）：305 m口径，曾是全球最大的单口径射电望远镜。2020年12月因年久失修发生灾难性坍塌，结束了57年的科学使命。它曾是电影《接触》（Contact）的取景地——也是人类搜寻地外文明（SETI）的重要阵地。
  • Goldstone DSN 70m（美国加州）：NASA深空网络（Deep Space Network）的三座70 m天线之一。用于与旅行者号、新视野号等深空探测器通信。旅行者1号距离地球超过240亿公里——这个信号从发出到被Goldstone接收需要超过22小时。

最后，用一段Python代码来展示如何根据口径尺寸估算抛物面天线的增益：

import numpy as np

def parabolic_dish_gain(D, f, efficiency=0.55):
    """
    计算抛物面天线的增益

    参数:
        D         : 口径直径 (m)
        f         : 工作频率 (Hz)
        efficiency: 口径效率 (典型值 0.50-0.65)
    返回:
        G         : 增益 (dBi)
        HPBW      : 半功率波束宽度 (度)
    """
    c = 3e8
    lam = c / f
    A = np.pi * (D/2)**2        # 口径物理面积
    Ae = efficiency * A          # 有效口径面积
    G_linear = (4 * np.pi * Ae) / (lam**2)
    G_dBi = 10 * np.log10(G_linear)
    # 半功率波束宽度的近似公式
    HPBW = 70 * lam / D          # 度 (适用于均匀照射)
    # 对于锥削照射，实际HPBW约为 65-75 * lam/D
    return G_dBi, HPBW

# 示例：计算几个经典天线的参数
antennas = [
    ("家用卫星锅",     0.6,   12.5e9, 0.60),
    ("VSAT终端",       1.8,   14.25e9,0.58),
    ("C波段地面站",    4.5,   6.0e9,  0.55),
    ("DSN深空站",     70.0,   8.4e9,  0.50),
]

print("天线类型    | 口径(m) | 频率(GHz) | 增益(dBi) | HPBW(°)")
print("-" * 65)
for name, D, f, eff in antennas:
    G, HPBW = parabolic_dish_gain(D, f, eff)
    print(f"{name:8s} | {D:5.1f}   | {f/1e9:6.2f}   | {G:6.1f}    | {HPBW:.2f}")
# 输出示例:
# 天线类型    | 口径(m) | 频率(GHz) | 增益(dBi) | HPBW(°)
# ----------------------------------------------------------------
# 家用卫星锅  | 0.6    | 12.50    | 33.6     | 1.12
# VSAT终端    | 1.8    | 14.25    | 44.1     | 0.41
# C波段地面站 | 4.5    | 6.00     | 42.3     | 0.78
# DSN深空站   | 70.0   | 8.40     | 80.2     | 0.01
关于口径效率：上面代码中我们用efficiency=0.5-0.6作为典型值，但实际工程中的效率受多种因素影响——馈源照射效率（illumination efficiency，锥削导致）、溢出效率（spillover efficiency，部分能量漏到反射面外）、遮挡效率（blockage efficiency，馈源和支架遮挡）、表面公差效率（Ruze损失）、相位效率（馈源相位中心偏移）等。一个好的抛物面天线设计师需要精心平衡这五个效率分量。

在这一章中，我们从场等效原理出发，探讨了天线中的"大口径方案"：喇叭天线、抛物面反射天线、卡塞格伦天线。我们看到，口径天线的核心在于将电磁波汇聚到一个平面上，利用傅里叶变换关系来设计方向图。从Bell Labs那具发现宇宙微波背景辐射的喇叭天线，到中国贵州的FAST 500m"天眼"——口径天线承载着人类对宇宙最深处的探索。
