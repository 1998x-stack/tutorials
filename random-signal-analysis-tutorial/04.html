期望：随机变量的"重心"

大家好，我是你们的讲师。前两章我们学会了用 PDF/CDF 完整地描述一个随机变量——这好比用一张全身照片来认识一个人。但在很多工程问题中，我们并不需要了解全部的细节，几个"关键数字"就足以指导我们的决策。比如设计一个放大器，你关心的是信号的平均功率和动态范围，而不需要知道每一个采样点的精确值。

数字特征就是用来回答这类问题的：随机变量的中心在哪里（期望）？它围绕中心散布的程度如何（方差）？两个变量之间的关联有多强（协方差）？

离散期望：Σx·p(x)；连续期望：∫x·f(x)dx

数学期望（简称期望或均值）是随机变量最核心的数字特征。它是随机变量所有可能取值的概率加权平均：

  • 离散型：E[X] = Σᵢ xᵢ · p_X(xᵢ)
  • 连续型：E[X] = ∫_{-∞}^{+∞} x · f_X(x) dx

期望的物理直觉是"重心"——如果把概率质量看作分布在数轴上的物质，期望就是质心所在的位置。注意：期望不一定是随机变量可能取的值（例如掷骰子的期望是 3.5，但 3.5 永远不会出现）。

期望的线性性质：E[aX+bY] = aE[X]+bE[Y]（永远成立）

期望最重要的性质是线性性：对任意随机变量 X、Y 和常数 a、b，有：

E[aX + bY] = a·E[X] + b·E[Y]

这条性质永远成立——不管 X 和 Y 是否独立，是否存在相关，甚至不管分布是否存在。只需要期望本身存在（有限）。线性性极大地简化了复杂系统的分析：一个由多个随机分量加权组合的信号，其平均功率就是各分量期望的加权和。

更一般地，对任意函数 g，E[g(X)] 不总是等于 g(E[X])——这个等式只在 g 是线性函数时成立。初学者常犯的错误就是假设 E[X²] = (E[X])²，实际上 E[X²] ≥ (E[X])²（詹森不等式）。

条件期望与全期望公式：E[X] = E[E[X|Y]]

条件期望 E[X|Y] 是在给定 Y 的值之后 X 的条件均值——它是 Y 的函数，因此本身也是一个随机变量。全期望公式（又称为双重期望定律）给出了无条件期望与条件期望之间的关系：

E[X] = E[E[X|Y]]

这个公式看起来像是同义反复，但在工程中极其有用。它的思想是：如果你不方便直接计算 E[X]，可以先"以 Y 为条件"计算条件期望，然后再对 Y 取平均。

工程用例：假设一个通信系统有两种信道状态——"好"（概率 0.9）和"差"（概率 0.1）。在好信道下，误码率的条件期望是 10^{-6}；在差信道下，条件期望是 10^{-2}。那么整体的平均误码率 E[BER] = 0.9×10^{-6} + 0.1×10^{-2} ≈ 10^{-3}。这个分情况讨论再取平均的思路，就是用全期望公式对信道状态作条件期望。

方差与标准差

Var(X) = E[(X-μ)²] = E[X²]-μ²

期望告诉我们随机变量的"中心"，而方差告诉我们随机变量在期望周围的"散布程度"：

Var(X) = E[(X - μ)²] = E[X²] - μ²

第二种形式 E[X²] - μ² 在计算上更便利——你只需要计算两个期望，而不需要先知道 μ 再去算偏差平方的期望。

方差的一些基本性质：

  • Var(aX + b) = a²·Var(X) —— 平移不变（加常数不影响方差），缩放会平方放大
  • Var(X + Y) = Var(X) + Var(Y) + 2·Cov(X,Y) —— 独立时协方差项为零
  • 方差总是非负的，方差为零当且仅当 X 以概率 1 取常数

标准差与数据同单位，更直观

方差有一个"缺点"——它的单位是原随机变量单位的平方。如果你在测量电压（单位 V），方差的单位就是 V²，物理含义不太直观。标准差 σ = √Var(X) 则与原变量同单位：

设 X 是噪声电压（µV），方差 Var(X) = 4 µV²，那么标准差 σ = 2 µV——"噪声的典型振幅在 2 µV 左右"。这个说法在工程直觉上比"均方振幅 4 µV²"直观得多。

切比雪夫不等式：无论什么分布，大部分数据在μ±kσ内

切比雪夫不等式是概率论中最"普适"的不等式之一——它对任何分布（只要方差存在）都成立：

P(|X - μ| ≥ kσ) ≤ 1/k²

或者说，P(|X - μ| < kσ) ≥ 1 - 1/k²。例如：

  • k=2：至少 75% 的数据落在 μ ± 2σ 内（1-1/4=0.75）
  • k=3：至少 88.9% 的数据落在 μ ± 3σ 内（1-1/9≈0.889）
  • k=5：至少 96% 的数据落在 μ ± 5σ 内

对比上一章高斯分布的 3σ 法则（99.73%），切比雪夫给出的界限（88.9%）要宽松得多——这就是"普适性"的代价：它不依赖任何分布假设，所以界限不会太紧。但在你不知道分布形式时，切比雪夫不等式仍能给你一个可靠的粗略上界。

实用场景：假设你需要设计一个 ADC（模数转换器），输入信号的最大幅度必须在一定范围内以避免削波。你知道信号均值 μ=0，标准差 σ=0.5V。如果不用切比雪夫，你需要假设信号分布才能计算削波概率。用切比雪夫，你至少知道：当量程设为 ±5σ=±2.5V 时，削波概率不超过 1/25=4%。这个"最坏情况"的估计在设计初期非常有用。

协方差与相关系数

Cov(X,Y) = E[(X-μx)(Y-μy)]

期望和方差描述单个随机变量的特征。协方差描述两个随机变量之间的线性关联程度：

Cov(X,Y) = E[(X - μ_X)(Y - μ_Y)] = E[XY] - μ_X·μ_Y

协方差的正负号告诉方向：Cov > 0 意味着 X 增大时 Y 也倾向于增大；Cov < 0 意味着 X 增大时 Y 倾向于减小。协方差的绝对值大小受变量尺度的影响——如果 X 的单位从米改为毫米，协方差会放大 1000 倍。

相关系数ρ ∈ [-1,1]：+1线性正相关、-1线性负相关、0不相关

为了消除尺度影响，我们使用相关系数（也称 Pearson 相关系数）：

ρ_{XY} = Cov(X,Y) / (σ_X · σ_Y)

相关系数是一个无量纲的标量，取值范围在 [-1, 1] 之间：

  • ρ = +1：完全线性正相关，Y = aX + b 且 a > 0
  • ρ = -1：完全线性负相关，Y = aX + b 且 a < 0
  • ρ = 0：不相关——但注意，这不意味着独立！
  • |ρ| 越接近 1，线性关系越强；越接近 0，线性关系越弱

不相关 ≠ 独立！一道经典反例（X~N(0,1), Y=X²）

这是概率论中最经典的误区之一。考虑：

令 X ~ N(0,1)，Y = X²。显然 Y 完全由 X 决定——X 和 Y 高度依赖，绝不是独立的。但计算 Cov(X,Y)：

Cov(X,Y) = E[X·X²] - E[X]·E[X²]
         = E[X³] - 0 · 1
         = 0  （因为标准正态的三阶矩为零）

所以 ρ = 0，X 和 Y 不相关！

为什么会这样？因为 X 和 Y 的关系是二次的（抛物线），而不是线性的。相关系数只能捕捉线性关系。X 取正值时 Y 增大（正相关区域），但 X 取负值时 Y 也增大（又是正相关区域）——两者在协方差积分中相互抵消，导致协方差为零。

重要原则：不相关（ρ=0）只意味着"没有线性关系"，但两个变量之间可能存在任何形式的非线性依赖（平方、指数、周期性、条件依赖等）。反过来，独立（联合分布可分解）则必然不相关。用逻辑语言：独立 ⇒ 不相关，但不相关 ⇏ 独立。这个单向箭头是无数工程"翻车"的根源——我们在第1章已经讲过一个 MIMO 通信的案例，就是尾部依赖导致的 outage。

注意，高斯分布是一个重要的例外：如果 X 和 Y 是联合高斯的，则不相关 ⇔ 独立。这是因为高斯分布的所有依赖结构都被协方差矩阵完全刻画——没有"隐藏的非线性关系"。这也是高斯分布在数学上如此"友好"的原因之一。

矩与特征函数

n阶原点矩与中心矩

期望和方差可以统一到一个更大的框架——矩：

  • n 阶原点矩：m_n = E[X^n] ——期望就是一阶原点矩
  • n 阶中心矩：μ_n = E[(X - μ)^n] ——方差就是二阶中心矩

矩这个名称来自物理学中的"力矩"——概率质量分布在数轴上，各阶矩描述了分布的不同形状特征。

偏度（不对称性）和峰度（尾部厚度）

三阶和四阶中心矩经过归一化后，给出了描述分布"形状"的两个重要指标：

名称定义含义
偏度 (Skewness)μ₃ / σ³分布的不对称性：正偏↔右尾长，负偏↔左尾长
峰度 (Kurtosis)μ₄ / σ⁴分布的尾部厚度：峰度>3→厚尾（比高斯更极端），<3→薄尾

峰度的"参考值"3 来自标准正态分布（峰度=3）。超过 3 的分布（如 t 分布）比高斯有更厚的尾部——极端值出现的概率更大，这在金融风险管理和通信中断概率分析中非常重要。偏度则在非对称信道（如光纤通信中的非线性效应）分析中有用。

特征函数：Φ(ω)=E[e^{jωX}]，概率论中的"傅里叶变换"

特征函数把概率论和傅里叶分析连接了起来：

Φ_X(ω) = E[e^{jωX}] = ∫_{-∞}^{+∞} f_X(x) · e^{jωx} dx

如果你注意到了：特征函数就是 PDF 的傅里叶变换（差一个符号约定）。这个联系极其深刻：

  • 唯一性：特征函数与 PDF 一一对应——不同的 PDF 有不同的特征函数，反之亦然
  • 矩生成：对特征函数在 ω=0 处求导，可得各阶矩：E[Xⁿ] = (-j)ⁿ · dⁿΦ/dωⁿ |_{ω=0}
  • 独立和：独立随机变量之和的特征函数等于各自特征函数的乘积——这是下一章证明中心极限定理的关键工具

为什么要引入特征函数？因为在很多情况下，PDF 的表达式很复杂，但特征函数的形式很简单。例如柯西分布（没有有限方差的厚尾分布）的 PDF 形式复杂，但其特征函数优雅简洁——这让我们能用特征函数处理 PDF 难以直接分析的问题。

本章小结与衔接

本章我们从"重心"期望出发，学习了方差（散布程度）、协方差与相关系数（线性相关程度），以及更高阶的矩和特征函数。最重要的三件事：期望的线性性（永远成立）、不相关 ≠ 独立（牢记反例）、特征函数是 PDF 的傅里叶变换（将概率论与信号处理的核心工具连接）。

有了这些数字特征作为工具，下一章我们探讨一个有趣的问题：当我们对随机变量施加一个函数变换（比如取平方、取对数、求包络），新随机变量的分布是什么？这就是"随机变量函数"的话题，它会把我们引向瑞利分布、莱斯分布和概率论中最重要的两个极限定理。
