第六章：循环神经网络（RNN）

大家好！前面几章我们把词变成了向量，但还有一个关键问题没解决：句子是有顺序的。"我打你"和"你打我"用的词一样，意思完全不同。前馈神经网络（把词向量拼接起来送进全连接层）无法处理变长序列，每次输入长度不一样，网络结构就得变。

循环神经网络（Recurrent Neural Network, RNN）就是为解决序列问题而生的。它的核心思想极其优雅：用同一个网络反复处理序列中的每个元素，同时维护一个"记忆"状态。我2014年第一次理解RNN时，感觉就像打开了新世界的大门。

6.1 序列数据的本质

在NLP中，文本天然就是序列：词一个接一个，靠前后关系产生意义。序列数据有三个关键特性：

特性说明例子
顺序依赖元素的顺序改变整个含义"狗咬人"vs"人咬狗"
变长序列长度不固定一条微博vs一篇论文
长距离依赖远距离的词之间可能有关系"我出生在...所以我热爱那个城市"

传统的前馈网络要求固定大小的输入，而且没有记忆——处理"我"和处理"爱"用的是不同的参数，无法共享对"这是一句话"这个事实的理解。

6.2 RNN的核心机制与展开

RNN的数学核心——共享参数+隐藏状态。每个时间步，RNN做两件事：用当前输入更新隐藏状态，用隐藏状态产生输出：

# RNN 的核心公式
# h_t = tanh(W_hh × h_{t-1} + W_xh × x_t + b_h)
# y_t = W_hy × h_t + b_y

# 参数数量：W_hh + W_xh + W_hy（三个矩阵，与序列长度无关！）
# 这就是RNN能处理任意长度序列的秘诀

import torch
import torch.nn as nn

rnn = nn.RNN(input_size=300,   # 词向量维度
             hidden_size=128,  # 隐藏状态维度
             num_layers=1,
             batch_first=True)

# 输入: (batch, seq_len, 300)
# 输出: (batch, seq_len, 128)
# 隐藏状态: (1, batch, 128)

RNN在时间上"展开"（Unroll）后，本质上是一个深度为序列长度的前馈网络——每一层对应一个时间步，但所有层共享同一组参数。这个共享是RNN精巧之处的核心：无论序列多长，参数量是固定的。

6.3 反向传播与梯度消失

BPTT（Back-Propagation Through Time）是把标准的反向传播应用到"展开"后的RNN上。这里有一个致命的数学问题：

# BPTT中的梯度计算（简化）
# ∂L/∂h_1 = ∂L/∂h_T × ∂h_T/∂h_{T-1} × ... × ∂h_2/∂h_1
# 每一项 ∂h_t/∂h_{t-1} 都包含 tanh 的导数

# tanh'(x) ∈ (0, 1]
# 当序列很长时：连乘会让梯度趋近于0
# 例如：0.9^100 ≈ 2.65e-5 — 几乎为0！

# 这就是梯度消失：早期时间步的梯度几乎为零
# 模型无法学习长距离依赖

⚠️ 踩坑记录：我2016年用纯RNN做机器翻译，训练了三天loss一动不动。百思不得其解，后来发现是BPTT展开步数设太大（100步），梯度全消失了。截断BPTT到20步后立刻开始收敛。但这也意味着——模型只能看到前20个词，再远的依赖就学不到了。

6.4 双向RNN

单向RNN只能利用"前文"信息。但在很多NLP任务中（如NER、情感分析），"后文"同样重要。双向RNN（Bi-RNN）就是同时跑两个RNN——一个从左到右，一个从右到左——然后把两个方向的隐藏状态拼接起来。

# 双向RNN
# h_forward[t] = RNN_forward(x[t], h_forward[t-1])
# h_backward[t] = RNN_backward(x[t], h_backward[t+1])
# h[t] = [h_forward[t]; h_backward[t]]  # 拼接

# 例子：识别"苹果"的词义
# 前向："我 今天 买了 一个 苹果 ..." → 前面有"买"，暗示是水果
# 后向："... 苹果 手机 很好 用" → 后面有"手机"，暗示是品牌
# 双向拼接 → 同时看到前后文，正确消歧

需要注意：双向RNN不能用于语言模型（预测下一个词）——因为语言模型不允许"看未来"。但用于编码器（如BERT、文本分类），双向是标配。

关键总结：RNN解决了序列建模的三个核心问题——变长输入、参数共享、序列记忆。但它有一个致命缺陷：梯度消失导致无法学习长距离依赖。下一章的LSTM和GRU，就是专门为此设计的改进版。

理解了RNN的痛——梯度消失让模型"记不住"早期信息——你就能真正理解LSTM门控机制的设计动机。下一章，我们来剖析LSTM和GRU这两个"记忆力超强"的RNN变体。
