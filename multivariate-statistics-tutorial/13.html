开篇：因子求出之后

大家好，欢迎继续因子分析的学习。到上一章为止，我们已经学会了如何估计因子载荷矩阵，以及如何通过旋转让它变得清晰可读。假设现在你得到了一个漂亮的旋转后载荷矩阵——每个因子有清楚的变量归属，共同度也理想。这时你会面临两个新问题：

第一，每个样本在这个因子结构中的"位置"在哪里？——即因子得分 (factor scores)。第二，这些抽象的因子到底代表什么现实含义？——即因子命名 (factor naming)。

这两个问题看似简单，但处理不好会让你前功尽弃。比如说，你辛苦提取了3个因子，但第三个因子的高载荷变量是"我喜欢吃辣"和"我常去旅行"——这该怎么命名？它是真的有意义，还是纯属随机噪声？

本章我们来解决这两个问题，并用一个完整的心理学量表案例把因子分析全流程走一遍。

因子得分的概念

回忆因子模型：X = μ + LF + ε。对第 j 个样本来说：

xⱼ - μ = L fⱼ + εⱼ

其中：
xⱼ: 第 j 个样本的观测向量（p 维）
fⱼ: 第 j 个样本的因子得分向量（m 维），不可观测
εⱼ: 第 j 个样本的特殊因子（p 维），不可观测
L: 因子载荷矩阵（p × m），已知（已估计出）

因子得分 fⱼ 告诉我们每个样本在每个潜变量上的"取值"。比如在心理学量表中，因子得分可以表示每个被试在"外向性"因子上的得分高低。这在后续分析中非常有用——例如你可以比较不同人群的平均因子得分差异、用因子得分做聚类等。

但这里有一个根本性的问题：与主成分得分不同，因子得分是不确定的。在 PCA 中，主成分得分是观测数据的确定性线性函数：z = X w。但在 FA 中，fⱼ 不是 X 的确定性函数——因为模型中有误差项 ε，所以同一组观测值可能对应不同的 fⱼ 值。我们需要用估计的方法从数据中"猜测"因子得分。

PCA 得分 vs FA 得分的根本区别：PCA 得分是"算出来"的（确定性的线性组合），FA 得分是"估计出来"的（有一定的误差）。从实用角度看，这不影响我们使用它们——但当你在学术报告中报告因子得分时，需要指出这一不确定性。

回归法估计因子得分

回归法 (Regression Method) 是最常用的因子得分估计方法，也称为Thompson 法。它基于一个简单的想法：把因子 f 看作"潜变量"，用观测变量 x 来预测 f——即找到一组权重，使预测的因子得分与真实因子之间的均方误差最小。

在正交因子模型下，回归法的因子得分计算公式为：

回归法因子得分：

f̂ = Lᵀ R⁻¹ x （标准化后的数据）

其中：
L: p × m 因子载荷矩阵
R: p × p 相关矩阵（对于标准化数据）
x: 标准化后的观测向量（p × 1）
f̂: 估计的因子得分向量（m × 1）

用 Python 计算：
from factor_analyzer.factor_analyzer import FactorAnalyzer

fa = FactorAnalyzer(n_factors=3, rotation='varimax')
fa.fit(X)

# 回归法因子得分（默认）
scores = fa.transform(X)

# 或者手动计算：
import numpy as np
L = fa.loadings_              # 载荷矩阵 p × m
R = np.corrcoef(X.T)          # 相关矩阵 p × p
scores = X @ L @ np.linalg.inv(L.T @ L)  # 近似公式

回归法得到的因子得分是有偏的——它们的方差会被压缩（向均值收缩）。但它们的均方误差最小，而且与真实因子的相关性最高。在大多数实际应用中，回归法是一个合理的选择。

此外还有 Anderson-Rubin 法，它是回归法的一种修正——保证估计的因子得分是标准化的（均值为0、方差为1），并且在正交因子模型中保持不相关。这在后续分析（如回归）中比较方便。

Bartlett法

Bartlett 法采取了与回归法不同的思路。它不追求"预测误差最小"，而是追求"特殊因子 ε 的加权平方和最小"——即找到一组 f 值，使得在给定观测 x 的条件下，特殊因子的"贡献"尽可能小。

Bartlett 得分的公式为：

Bartlett 因子得分：

f̂ = (Lᵀ Ψ⁻¹ L)⁻¹ Lᵀ Ψ⁻¹ x

其中 Ψ = diag(ψ₁, ..., ψₚ) 是特殊方差矩阵

Bartlett 得分是无偏的——它们的期望等于真实因子值。
但它们的方差比回归法大（因为无偏估计通常方差更大）。

在 Python 中指定 Bartlett 法：
scores_bartlett = fa.transform(X, method='bartlett')

回归法与 Bartlett 法的选择取决于你的目标：

维度回归法 (Thompson)Bartlett 法
偏差有偏（方差收缩）无偏
均方误差更小（整体更好）更大
与真实因子的相关性更高略低
因子间不相关性不保证保持（正交模型下）
推荐场景大多数通用分析需要无偏估计时
选择建议：如果你后续要用因子得分做回归分析（作为自变量），建议用 Bartlett 法——因为无偏性可以保证回归系数估计的一致性。如果你只是做描述性分析（如比较各组均值的差异），回归法更合适——它的均方误差更小，个体得分的精度更高。

因子命名的艺术

因子的命名可能是因子分析中最需要"人的智慧"而非"数学技巧"的环节。一个命名得当的因子可以让整篇论文的分析站得住脚；一个命名不当的因子则可能被审稿人一句话打回："你对第三个因子的解释缺乏理论依据。"

因子命名的基本原则：

  • 看高载荷变量：一般在每个因子上找载荷绝对值最大的 3-5 个变量，看它们共同描述了哪个概念
  • 考虑变量的符号：如果某个变量在因子上有负载荷，命名时需要反映这种反向关系（例如"焦虑"因子下，"我感到平静"这个题目有负载荷是合理的）
  • 查阅理论文献：看看已有的理论和量表是如何命名类似因子结构的
  • 保持简洁：因子名不要太长，2-4 个汉字或 1-3 个英文单词最佳
  • 不要强行命名：如果一组变量实在没有共同的含义，宁可承认"无法解释"也不要生造一个牵强的名字

来看一些实际命名的例子：

高载荷变量可能的因子名说明
销售额、利润、市场份额企业绩效三个指标都反映企业经营成果
心率、血压、皮质醇水平生理应激反应典型的生理压力指标
我喜欢社交 (+)、我享受独处 (-)外向性正负载荷反映了概念的两端
存款金额、投资比例、理财频率理财行为三个行为共同反映理财倾向

完整案例：心理学量表因子分析

现在我们把前面所学的所有内容串联起来，用一个完整的心理学量表案例走一遍因子分析的全流程。假设我们对 500 名被试施测了一份 12 个题目的工作倦怠量表，想验证它是否真的包含三个子维度（情绪耗竭、去人格化、低成就感）。

# Python：完整因子分析流程
import pandas as pd
import numpy as np
from factor_analyzer import FactorAnalyzer, calculate_kmo, calculate_bartlett_sphericity
from factor_analyzer import Rotator

# 1. 加载数据（假设有 12 列、500 行）
# 列名: EE1-EE4 (情绪耗竭), DP1-DP4 (去人格化), PA1-PA4 (低成就感)
data = pd.read_csv('burnout_scale.csv')

# 2. 适用性检验
kmo_all, kmo_total = calculate_kmo(data)
chi2, p = calculate_bartlett_sphericity(data)
print(f"KMO = {kmo_total:.3f}")   # 假设输出 0.87 → 良好
print(f"Bartlett p = {p:.4f}")    # 假设输出 0.000 → 显著

# 3. 决定因子数（平行分析）
# 可以使用 fa_parallel 函数（来自 psych 包的心理测量学传统）
# 这里假设平行分析建议 3 个因子

# 4. 提取因子并进行旋转
fa = FactorAnalyzer(n_factors=3, rotation='varimax')
fa.fit(data)

# 5. 检查结果
ev, v = fa.get_eigenvalues()
print(f"方差贡献率: {fa.get_factor_variance()}")

# 6. 载荷矩阵（已旋转）
loadings = pd.DataFrame(
    fa.loadings_,
    index=data.columns,
    columns=['因子1', '因子2', '因子3']
)
print(loadings.round(3))

# 假设输出如下（理想情况）：
#       因子1    因子2    因子3
# EE1  0.82   -0.02   -0.01
# EE2  0.78    0.05    0.03
# EE3  0.85    0.01   -0.02
# EE4  0.76    0.08    0.04
# DP1  0.03    0.79   -0.01
# DP2 -0.01    0.82    0.02
# DP3  0.05    0.74    0.06
# DP4 -0.02    0.80   -0.03
# PA1  0.04   -0.01    0.71
# PA2 -0.03    0.02    0.78
# PA3  0.01    0.05    0.74
# PA4  0.06   -0.03    0.68

来看看这个输出。载荷矩阵呈现出近乎完美的"简单结构"：EE1-EE4 只在因子1上有高载荷，DP1-DP4 只在因子2上有高载荷，PA1-PA4 只在因子3上有高载荷。交叉载荷（变量在其他因子上的载荷）都接近于 0。这正是因子分析希望达到的理想状态。

现在我们可以命名了：因子1 = "情绪耗竭"，因子2 = "去人格化"，因子3 = "低成就感"。三个因子一起解释了总方差的 68%。每个变量的共同度 h² 都在 0.5 以上——说明三个因子对每个变量都有良好的解释力。

结果验证：注意，我们"发现"的因子结构恰好与量表设计时的理论维度一致——这在量表开发和验证中叫作结构效度 (construct validity) 验证。如果实际数据跑出来的因子结构与理论预期不符，则需要反思：是理论有问题？量表题目设计有缺陷？还是被试群体的特殊性导致的？因子分析在社科研究中常用于这个目的——验证一个测量工具是否真的测量了它声称要测量的概念。

因子分析报告规范

如果你在学术论文或正式报告中使用因子分析，有一组标准的报告内容需要包含。下面是一个推荐的报告清单：

报告内容说明示例
样本量标明有效样本数N = 486
KMO 值总体和每个变量的 KMOKMO = 0.87
Bartlett 检验卡方值和 p 值χ² = 2843.2, p < 0.001
提取方法PCF、MLFA、PAF 等极大似然估计 (MLFA)
旋转方法Varimax、Promax 等Promax 斜交旋转 (k=4)
因子数确定依据平行分析、Kaiser 规则等平行分析建议 3 因子
方差解释率每个因子和累计三因子累计解释 68.3%
载荷矩阵表含共同度，≥0.4 的载荷加粗见表 1
因子命名基于高载荷变量命名因子1：情绪耗竭
因子相关矩阵斜交旋转时需要报告r = 0.35（因子1与因子2）
表格呈现小技巧：在论文的载荷矩阵表中，通常只显示绝对值 ≥ 0.4 的载荷，低于这个阈值的显示为空或 "."。这样做有两个好处：一是突出了高载荷变量（读者一眼就能看出每个因子有哪些变量），二是避免了读者被细小的交叉载荷分散注意力。很多软件（如 SPSS）在输出时默认提供这个选项。

本章小结与衔接

本章我们完成了因子分析的最后两个步骤：因子得分估计和因子命名解释。我们学习了回归法（最小均方误差，有偏）和 Bartlett 法（无偏，方差较大）两种得分估计方法，以及它们的取舍原则。在因子命名方面，我们讨论了基于高载荷变量进行命名的原则和技巧。然后我们用一个 12 题心理学量表的完整案例，从适用性检验到因子提取、旋转、命名、报告一气呵成。最后给出了学术论文中因子分析结果的规范报告清单。

到这里，你可能会觉得"因子分析和 PCA 好像差不多啊"——都会得出载荷矩阵，都可以做旋转，都可以计算得分。确实，它们在操作层面有很多相似之处，但在哲学层面、数学假设层面和应用场景层面有着本质的区别。下一章，也就是本课程关于 PCA 与 FA 的最后一章，我们将把这两套方法放在一起做一次全面的比较——什么时候用 PCA、什么时候用 FA、两者结果的差异从何而来。用一个踩坑故事和一个决策流程图帮你彻底理清这个"统计学的世纪之争"。
