21.1 对流层的大气结构

大家好，我是你们的讲师。你有没有想过这样一个问题：为什么在天朗气清的日子里，手机基站的信号可以传得很远，但在大雾弥漫的早晨，信号反而时好时坏？为什么海边的雷达能看到地平线以下的目标？这些问题的答案都藏在与我们朝夕相处却常被忽视的大气层中。

对流层（Troposphere）是大气层的最低一层，从地面延伸到约12公里的高度（在赤道附近可达17公里，在极地约8公里）。它包含了大气中约75%的质量和几乎全部的水蒸气。所有的气象现象——风、云、雨、雪、雾——都发生在这里。更重要的是，这一层对无线电波的传播有着深远的影响。

对流层对电波的影响主要通过大气折射率来刻画。折射率n非常接近1，因此通常用折射率差N来表示：

N = (n - 1) × 10⁶

其中:
N = 77.6 · P / T + 3.73 × 10⁵ · e / T²

P : 大气压 (hPa, 即百帕)
T : 绝对温度 (K)
e : 水汽压 (hPa)

在标准大气条件下，N随高度下降的速率约为40 N-units/km。这意味着什么？折射率随高度降低，意味着电磁波在对流层中传播时会向折射率更大的方向弯曲——即向地面弯曲。这就是所谓的大气折射。

国际标准参考：ITU-R P.453建议书提供了全球参考大气折射率模型，包括标准大气剖面、地图化的月平均N₀（地面折射率）以及年变化统计。在做任何地面微波链路设计时，P.453都是第一个需要查阅的ITU文件。

21.2 大气折射与等效地球半径

为了简化大气折射的分析，工程师们提出了一个天才的等效方法：与其处理弯曲的射线和分层的大气，不如把地球的半径"等效放大"，同时假设射线沿直线传播。

k因子（等效地球半径因子）的定义：

a_e = k · a

其中:
k = 1 / (1 + a · dn/dh)
a  = 真实地球半径 (6370 km)
dn/dh = 折射率梯度

在标准大气下，dn/dh ≈ -40 × 10⁻⁶ / km，代入可得 k ≈ 4/3，等效地球半径 a_e ≈ 8500 km。这意味着一对相距50 km的收发天线，在k=4/3的条件下，其"等效视线距离"比真实几何视线远了约15%。

非标准折射条件

折射条件并非总是标准的。以下情况经常在工程实践中遇到：

  • 次折射（Sub-refraction），k < 1：折射率梯度大于标准值（即N下降太慢甚至上升）。射线向上弯曲——通信距离缩短，主要发生在冷空气经过暖地面时。
  • 超折射（Super-refraction），k > 1：射线向下弯曲——通信距离超出几何地平线。这在暖空气经过冷海面时很常见。
  • 波导条件（Ducting），k → ∞：当N下降速率超过约157 N-units/km时，射线完全被"捕获"在大气薄层中——我们将在下一节详细讨论。
k值 dn/dh (N-units/km) 条件 对通信的影响
k < 1 > 0 次折射：冷空气/暖地面 链路距离缩短，可能中断
k = 1 0 无折射（均匀大气） 射线沿直线传播
k = 4/3 -40 标准大气 正常传播，略有延伸
k = 2 ~ ∞ -80 ~ -157 超折射：暖空气/冷地面 传播距离显著延长
k < 0 (波导) < -157 大气波导 信号被捕获，超远距离传播
实测踩坑故事：某次在沿海城市进行5G毫米波覆盖测试，团队发现一个距离基站4公里的测试点信号异常强——比2公里处的功率还高出5 dB。所有人百思不得其解，怀疑设备故障。重新检查气象日志后发现：测试当天海面温度低于气温约6°C，形成了强烈的超折射条件，信号被"弯曲"到了原本的阴影区域。这个案例提醒我们：任何野外测试，必须先记录气象数据——不对传播条件做记录，根本无法解释测试结果的异常。

21.3 大气波导效应

大气波导（Atmospheric Duct）是对流层中最壮观的传播现象之一。当N的下降速率超过约157 N-units/km时，射线向下弯曲的曲率超过了地球表面的曲率——电磁波被困在一个"大气波导"中传播，就像光在光纤中传播一样。

蒸发波导（Evaporation Duct）

这是最常见的大气波导类型。在海面上，海水蒸发使得近海面（通常10-20米高度内）的水汽压急剧下降，形成一个非常陡峭的N梯度。蒸发波导几乎总是存在于全球的海洋上（尽管高度和强度随季节和天气变化）。对于海上雷达和海事通信系统来说，蒸发波导是必须考虑的因素——它既可以让雷达"看到"地平线以下的目标，也可以让雷达信号被意外捕获而看不到原本在视距内的目标。

地表波导与高空波导

地表波导（Surface Duct）从地面开始向上延伸一定高度（通常几十到几百米）。它可以将信号困在地表附近传播数百公里。这对地面微波链路既是好事（延长传输距离）也是坏事（增加与同频系统的干扰概率）。

高空波导（Elevated Duct）发生在离地一定高度处（通常500-2000米），上下边界都是空气。这种波导会影响机载雷达和卫星地面站的信号——下行信号可能被波导捕获而无法到达地面，或者上行信号被意外偏折到错误的位置。

历史故事：二战期间，英国海岸雷达操作员经常在屏幕上发现远在200公里外的目标——远超过雷达的理论最大作用距离。最初，他们以为设备出了故障，甚至怀疑是德军的"秘密干扰技术"。后来的分析表明，这其实是英吉利海峡上空的蒸发波导将雷达波捕获并传输到了超远距离——德国军舰在不知不觉中被"看不见的波导"出卖了。这是科学史上一个著名的"由意外发现到理论完善"的案例。

在今天的蜂窝网络中，大气波导依然在制造麻烦。当夏季出现稳定的高压天气系统时，相距数百公里的两个基站可能通过波导形成干扰——尤其在同频组网的LTE/5G网络中，这会表现为奇怪的"远距离同频干扰"，定位起来极为困难。

21.4 大气气体吸收与雨衰

大气分子吸收

大气中的氧气（O₂）和水蒸气（H₂O）分子在特定频率上具有分子谐振——这些谐振频率上的电磁波会被大量吸收。对无线电通信最重要的吸收峰是：

  • O₂谐振：60 GHz（吸收高达5-15 dB/km！）和118 GHz
  • H₂O谐振：22 GHz和183 GHz

ITU-R P.676建议书提供了详细的大气气体衰减模型。对于地面链路，你可以粗略估算：在10 GHz以下，大气吸收可以忽略；在10-30 GHz，吸收大约0.01-0.1 dB/km（视水汽含量而定）；到了60 GHz的氧气吸收峰，传输几乎不可能超过1-2公里——这也是为什么60 GHz只能用于短距离通信（如WiGig）。

雨衰

雨衰（Rain Attenuation）是10 GHz以上无线通信的最大敌人。雨滴对电磁波的散射和吸收随频率升高而急剧增加。ITU-R P.838给出了雨衰率的计算模型：

γ_R = k · R^α   (dB/km)

其中 R 是降雨率 (mm/h)
k 和 α 由频率、极化和仰角决定（P.838给出表值）

例如，在28 GHz（5G毫米波），水平极化时的k ≈ 0.15, α ≈ 1.14。这意味着一场50 mm/h的大雨会造成约12 dB/km的衰减——如果链路长度是500米，那就是6 dB的额外损耗，足以让链路余量告急。

频率 10 mm/h (小雨) 50 mm/h (大雨) 100 mm/h (暴雨)
10 GHz 0.01 dB/km 0.1 dB/km 0.3 dB/km
28 GHz 0.2 dB/km 2.8 dB/km 8.2 dB/km
60 GHz 0.8 dB/km 10 dB/km 28 dB/km
100 GHz 2.5 dB/km 30 dB/km 85 dB/km

对于5G毫米波（28 GHz）小站来说，一个好消息是站间距通常只有200米左右。在200米的路径上，即使是大雨造成的衰减也是可控的（约0.5-1 dB）。而对于卫星通信（常用20/30 GHz频段）来说，地面站到卫星的路径长度很长，雨衰会非常显著——因此自适应编码调制（ACM）和上行功率控制是Ka波段卫星系统的标配功能。

工程设计建议：ITU-R P.530建议书提供了地面视距链路的雨衰中断概率预测方法。设计链路时，需要根据本地降雨率统计（来自P.837建议书）确定年度可用度目标——比如99.99%可用度（即每年中断不超过53分钟）。这会直接决定链路预算中需要预留多少"雨衰余量"。

21.5 对流层散射通信

在完成了大气折射和吸收的讨论后，我们来介绍一种在雷达和卫星时代几乎被遗忘、但至今仍在军工领域发挥关键作用的通信方式——对流层散射通信（Troposcatter Communication）。

工作原理

对流层中存在着大量由于大气湍流造成的折射率随机起伏（即"大气气泡"）。当一束强无线电波射向对流层时，这些折射率不均匀体会将一部分能量向各个方向散射——其中一小部分会向前方地面方向传播，从而被远处的接收天线捕获。这就是"前向散射"。

常规工作频段为300 MHz - 5 GHz，单跳通信距离为100-500公里。与卫星通信相比，它不需要卫星；与地面微波接力相比，它不需要中继站——对于跨越山脉、海洋、沙漠等无人区非常理想。

巨大的路径损耗

对流层散射的代价是：路径损耗极大。典型数值：

# ITU-R P.617 对流层散射路径损耗估算

import math

def troposcatter_loss(d, f, h_tx=50, h_rx=50, N_s=350):
    """
    估算对流层散射路径损耗 (dB)

    参数:
        d     : 通信距离 (km)  — 100-500 km
        f     : 频率 (MHz)     — 300-5000 MHz
        h_tx  : 发射天线高度 (m)
        h_rx  : 接收天线高度 (m)
        N_s   : 地面折射率 (N-units)

    返回:
        L     : 路径损耗 (dB)
    """
    # 散射损耗 - P.617 简化模型
    L_bs = 20 * math.log10(f) + 30 * math.log10(d) - 20 * math.log10(N_s) + 48.5
    # 天线耦合修正（高度越高，耦合越好）
    L_c = 0.07 * math.exp(0.055 * (h_tx + h_rx))
    # 气候因子
    L_cf = 10  # 温带气候默认值
    L = L_bs + L_c + L_cf
    return L

# 示例：300 km, 2 GHz
loss = troposcatter_loss(300, 2000)
print(f"对流层散射路径损耗 (300 km, 2 GHz): {loss:.1f} dB")
# 输出约 195-205 dB — 极其巨大！

# 作为对比：
fspl_300km = 20 * math.log10(300e3) + 20 * math.log10(2e9) + 20 * math.log10(4*math.pi/3e8)
print(f"同等距离自由空间损耗: {fspl_300km:.1f} dB")
# 输出约 128 dB — 两者相差约 70-80 dB！

巨大的损耗意味着对流层散射链路需要：极高的发射功率（数百瓦到数千瓦）、非常大的天线（十几米口径的抛物面天线）以及多种分集技术（空间分集、频率分集、时间分集）。这是无线通信领域少有的"大力出奇迹"的场景。

现代化进展

你可能觉得这项技术已经过时了——但恰恰相反，它正在经历一场数字革命。传统的模拟对流层散射链路只能传输几十路语音。而现代数字对流层散射系统采用自适应均衡、Turbo/LDPC编码、OFDM调制和MIMO技术，在同样的100-300公里距离上已经实现了50 Mbps以上的数据速率。中国、俄罗斯和美国都在边境监控、海岛通信和军事战术网络中广泛使用现代化的对流层散射系统。

军事价值：对流层散射通信的最大优势是抗毁性和低截获概率。它不依赖卫星（无法被反卫星武器摧毁），不依赖地面中继（无法被定点摧毁），且信号极难被截获和干扰（因为散射信号本身的信噪比极低，必须在接收端通过分集和大天线才能"拼凑"出信号——拦截者做不到这一点）。在电子战环境日益严峻的今天，对流层散射通信的价值正在被重新评估。

本章我们深入探索了对流层对电波传播的多种影响机制——从大气折射的k因子概念，到波导效应的意外远距离传播，再到气体吸收和雨衰对高频段（特别是5G毫米波和卫星通信）的严峻考验，最后是对流层散射这种"暴力美学"式的超视距通信方式。对流层是一个极其丰富的传播环境，也是无线工程师最需要关注的大气层。

第21章集中在12公里以下的对流层，但电离层——从60公里延伸到1000公里的高层大气——才是电波传播中最令人惊叹的舞台。在下一章中，我们将探讨电离层电波传播，了解短波通信如何利用电离层的反射实现全球通信，以及电离层闪烁如何影响GPS导航的精确性。
