第二十一章 多自由度体系振动

如果单自由度体系是一把小提琴——只有一根弦、一个音。那么多自由度体系就是一支交响乐团——几十种乐器、几十个声部，它们各自振动，又相互影响，共同奏出复杂的乐章。

2004年，台湾台北101大厦竣工，成为当时世界第一高楼。这座509米高的摩天大楼在设计过程中面临一个巨大的挑战：如何在台湾频繁的台风和地震中保持稳定和舒适？答案是一种称为调谐质量阻尼器（Tuned Mass Damper, TMD） 的装置——一个660吨重的巨大钢球用钢索悬挂在第87至92层之间。当大楼被风吹动时，TMD会以相反的相位运动，像一个"反向钟摆"一样吸收振动能量。要设计这样一个系统，工程师需要对大楼的多阶振型有深刻的理解——因为TMD只针对某一特定振型（通常是一阶弯曲振型）进行调谐，如果调错了振型，它不仅不吸振，反而会让情况更糟。

多自由度体系（MDOF，Multi-Degree-of-Freedom）的分析比SDOF复杂得多，但核心思想是一致的：找到结构"天生"的振动模式（振型和自振频率），然后把复杂的响应分解到这些"天然"的模式上。这就是本章要学的——特征值问题、振型正交性、振型叠加法，以及实用的近似计算方法。

多自由度体系的运动方程

考虑一个N层剪切型框架——假设楼盖为刚性，每层的质量集中在楼盖位置，层间刚度分别为k₁, k₂, ..., k_N。这就是一个N自由度的"串联质量-弹簧"系统。

    N自由度剪切型框架的运动方程（矩阵形式）：

    M·ü(t) + C·u̇(t) + K·u(t) = P(t)

    其中：
    M —— 质量矩阵（Mass Matrix），N×N对角矩阵
    C —— 阻尼矩阵（Damping Matrix），N×N矩阵
    K —— 刚度矩阵（Stiffness Matrix），N×N矩阵
    u(t) —— 位移向量（Displacement Vector），N×1
    P(t) —— 荷载向量（Load Vector），N×1

对于三层剪切型框架，具体形式为：

    ┌ m₁ 0  0 ┐   ┌ ü₁ ┐   ┌ c₁+c₂  -c₂   0 ┐   ┌ u̇₁ ┐   ┌ k₁+k₂  -k₂   0 ┐   ┌ u₁ ┐   ┌ p₁(t) ┐
    │ 0  m₂ 0 │ · │ ü₂ │ + │  -c₂  c₂+c₃ -c₃ │ · │ u̇₂ │ + │  -k₂  k₂+k₃ -k₃ │ · │ u₂ │ = │ p₂(t) │
    └ 0  0  m₃┘   └ ü₃ ┘   └   0   -c₃   c₃ ┘   └ u̇₃ ┘   └   0   -k₃   k₃ ┘   └ u₃ ┘   └ p₃(t) ┘

注意刚度矩阵的形态——它是一个三对角矩阵（Tridiagonal Matrix）。这是因为只有相邻楼层之间有直接的刚度耦合：第i层的力平衡方程中只涉及u_{i-1}、u_i和u_{i+1}三个位移。这种"近邻耦合"是串联质量-弹簧系统的共性特征。

特征值问题与自振频率

和SDOF一样，MDOF的自由振动分析从无阻尼自由振动方程开始：

    M·ü(t) + K·u(t) = 0

    假设简谐振动形式 u(t) = φ·sin(ωt + α)，代入得：
    (K - ω²·M) · φ = 0

    这就是广义特征值问题（Generalized Eigenvalue Problem）。
    其中：
    ω² —— 特征值（Eigenvalue），对应自振圆频率的平方
    φ  —— 特征向量（Eigenvector），对应振型（Mode Shape）

因为φ不能为零向量，所以系数矩阵的行列式必须为零：

    det(K - ω²·M) = 0

    展开后得到关于ω²的N次代数方程——频率方程（Frequency Equation）。
    求解后得到N个自振频率：
    ω₁ < ω₂ < ... < ω_N

    每个频率对应一个振型 φ₁, φ₂, ..., φ_N
物理意义：N自由度体系有N个自振频率，每个频率对应一种"天然"的振动形态。一阶频率（基频）ω₁最低，对应的振型通常是最简单的弯曲或剪切变形。高阶频率的振型越来越复杂——结点（位移为零的点）越来越多，就像一根琴弦的高次谐波。在实际工程中，通常只有前几阶振型对响应有显著贡献——高阶振型的质量参与系数很小，可以忽略。

三层框架的手算示例

假设一个三层剪切型框架，每层质量相同m=100t，每层刚度相同k=200MN/m：

    质量矩阵 M = m·I，刚度矩阵 K = k·[1 -1 0; -1 2 -1; 0 -1 1]

    特征值问题：
    (K - ω²·M)φ = 0  →  (K - λ·M)φ = 0，λ = ω²

    频率方程（令 λ̄ = mλ/k = mω²/k）：
    det ┌ 1-λ̄   -1     0 ┐
        │ -1   2-λ̄   -1 │ = 0
        └ 0    -1    1-λ̄┘

    展开：(1-λ̄)[(2-λ̄)(1-λ̄)-1] - (-1)[(-1)(1-λ̄)-0] = 0
    化简：λ̄(λ̄² - 4λ̄ + 3) = 0
    解得：λ̄₁ = 0, λ̄₂ = 1, λ̄₃ = 3

    自振频率：
    ω₁ = 0 rad/s  （刚体位移，无弹性恢复力）
    ω₂ = √(k/m) = √(2×10⁶/s²) ≈ 44.7 rad/s，f₂ = 7.12 Hz
    ω₃ = √(3k/m) ≈ 77.5 rad/s，f₃ = 12.3 Hz

    对应振型：
    φ₁ = [1, 1, 1]ᵀ      —— 所有楼层同向等幅平移（刚体模态）
    φ₂ = [1, 0, -1]ᵀ     —— 顶层和底层反向，中间层不动
    φ₃ = [1, -2, 1]ᵀ     —— 顶层和底层同向，中间层反向（S形）
振型阶数 自振频率 ω (rad/s) 自振周期 T (s) 振型描述 振型形状
1阶（刚体） 0 ∞ 整体平动（无弹性变形） [1, 1, 1]ᵀ
2阶 44.7 0.14 上下反向，中间为节点 [1, 0, -1]ᵀ
3阶 77.5 0.08 S形弯曲 [1, -2, 1]ᵀ
一个有趣的观察：振型并不需要"归一化"——它的绝对值大小没有意义（因为它是特征向量，尺度是任意的），但各分量之间的相对比例是有意义的。比如振型[1,0,-1]ᵀ和[2,0,-2]ᵀ代表同一种振动形态。在实际应用中，我们通常会将振型归一化——按最大位移归一或按质量矩阵归一（使φᵀMφ=1）。

主振型的正交性

振型正交性（Orthogonality of Mode Shapes）是MDOF分析中最优美也最重要的数学性质。它说的是：不同阶振型在"加权内积"的意义下是正交的。

    φᵢᵀ · M · φⱼ = 0      当 i ≠ j
    φᵢᵀ · K · φⱼ = 0      当 i ≠ j

    这意味着振型之间"互不相干"——第i阶振型的运动不会激发第j阶振型。

这个性质为什么如此重要？因为它让我们能够把耦合的MDOF方程组解耦（Decouple）成N个独立的SDOF方程。具体做法是：

    假设 u(t) = Φ · q(t) = Σᵢ φᵢ · qᵢ(t)

    其中 Φ = [φ₁, φ₂, ..., φ_N] 是振型矩阵
    q(t) = [q₁, q₂, ..., q_N]ᵀ 是广义坐标（模态坐标）向量

    将 u = Φ·q 代入运动方程，前乘以 Φᵀ：
    ΦᵀMΦ·q̈ + ΦᵀCΦ·q̇ + ΦᵀKΦ·q = Φᵀ·P

    利用正交性，ΦᵀMΦ和ΦᵀKΦ变为对角矩阵：
    M̃·q̈ + C̃·q̇ + K̃·q = P̃(t)

    其中 M̃ᵢᵢ = φᵢᵀ·M·φᵢ（模态质量）
         K̃ᵢᵢ = φᵢᵀ·K·φᵢ = M̃ᵢᵢ·ωᵢ²（模态刚度）
         P̃ᵢ(t) = φᵢᵀ·P(t)（模态荷载）

    于是每个广义坐标qᵢ满足独立的SDOF方程：
    M̃ᵢ·q̈ᵢ + C̃ᵢ·q̇ᵢ + K̃ᵢ·qᵢ = P̃ᵢ(t)

这就是振型叠加法（Mode Superposition Method）的核心思想——把复杂的MDOF振动拆解为若干个SDOF振动的叠加。

初学者的常见误区：有些同学认为"振型叠加法只是一种数学技巧"。不对！它的物理基础是振型的正交性——不同振型之间确实不会交换能量。你可以做一个思想实验：把一个结构按第一阶振型的形状掰弯然后松手——它会以第一阶频率振动，而不会"自动"产生第二阶振型的成分。只有当外荷载的空间分布形式与某阶振型"有重叠"时，该阶振型才会被激发。这个物理直觉对理解结构的风振和地震响应非常关键。

振型叠加法的实施步骤

振型叠加法的完整步骤如下：

    步骤1：求解特征值问题 (K - ω²M)φ = 0
            → 得到 ω₁, ω₂, ..., ω_N 和 φ₁, φ₂, ..., φ_N

    步骤2：计算模态质量、模态刚度、模态荷载
            M̃ᵢ = φᵢᵀ·M·φᵢ
            K̃ᵢ = φᵢᵀ·K·φᵢ = ωᵢ²·M̃ᵢ
            P̃ᵢ(t) = φᵢᵀ·P(t)

    步骤3：解耦后的SDOF方程（必要时考虑阻尼）
            M̃ᵢ·q̈ᵢ + C̃ᵢ·q̇ᵢ + K̃ᵢ·qᵢ = P̃ᵢ(t)
            或者：
            q̈ᵢ + 2ζᵢωᵢ·q̇ᵢ + ωᵢ²·qᵢ = P̃ᵢ(t)/M̃ᵢ

    步骤4：对每阶振型求解qᵢ(t)（可用杜哈梅积分）

    步骤5：叠加得到物理坐标下的响应
            u(t) = Σᵢ φᵢ · qᵢ(t)

在实际工程中，我们通常只需要叠加前几阶振型——因为高阶振型的模态质量小、模态阻尼大，对总响应的贡献往往可以忽略。对于一般的高层建筑，叠加前3~5阶振型就能得到足够精确的结果。对于规则结构，有时甚至只取前1~2阶（基频为主的结构，如剪切型框架）。

近似计算：邓克利法与瑞利法

对于复杂结构，精确求解特征值问题可能很耗时——尤其是在手算时代。好在我们有一些巧妙的近似方法，可以在不求解完整特征值问题的情况下，快速估计结构的自振频率。这里介绍两种最经典的方法。

邓克利法（Dunkerley's Method）

邓克利法用于估计多自由度体系的最低自振频率（基频）。它的思想是：把多自由度体系的基频的倒数平方，近似为各"子体系"基频倒数平方之和。

    邓克利公式：
    1/ω₁² ≈ 1/ω₁₁² + 1/ω₂₂² + ... + 1/ω_NN²

    其中 ωᵢᵢ 是"只有第i个质量存在、其他质量为零"时体系的基频。
    或者说，ωᵢᵢ是只有第i个自由度时（其他自由度假定被锁住）的自振频率。

    对于悬臂梁端部有N个集中质量的情况：
    ωᵢᵢ = √(kᵢ/mᵢ)，其中kᵢ是第i个质量处的柔度系数的倒数。

邓克利法给出的是基频的下限估计（即估算值比真实值偏小），这在工程上是偏于保守的（安全侧）。

瑞利法（Rayleigh's Method）

瑞利法建立在能量守恒原理之上。对于无阻尼自由振动，体系的最大动能等于最大势能（应变能）。

    瑞利商（Rayleigh Quotient）：

    ω² = (φᵀ·K·φ) / (φᵀ·M·φ) = R(φ)

    其中φ是假定的振型形状（试探函数）。

    关键性质：如果φ恰好是真实的第i阶振型，那么R(φ)=ωᵢ²。
    如果φ是任意形状（接近真实振型），那么R(φ)给出的ω²
    是真实基频的**上限估计**（比真实值偏大）。
    而且，试探振型越接近真实振型，估算越精确。

瑞利法的实用步骤：

    1. 假设一个合理的振型形状 φ̂ —— 可以从荷载作用下的静力变形曲线入手
    2. 计算势能（应变能）：U_max = (1/2)·φ̂ᵀ·K·φ̂
    3. 计算动能：T_max = (1/2)·ω²·φ̂ᵀ·M·φ̂
    4. 令 U_max = T_max，解出 ω²

    对于悬臂梁，通常假设 φ̂(x) = 1 - cos(πx/(2L)) 或 φ̂(x) = (x/L)²
    即可得到非常精确的基频估计。
方法 适用范围 估计方向 精度 需要的输入
邓克利法 基频（最低频率） 下限（偏安全） 中等 各单自由度频率
瑞利法 基频（可推广到高阶） 上限 较高（取决于试探函数） 试探振型形状
精确解 所有频率和振型 精确值 精确 完整特征值求解
实用建议：邓克利法和瑞利法可以配合使用——邓克利给出下限，瑞利给出上限，真实基频一定介于两者之间。如果两个方法给出的结果很接近（比如误差在5%以内），那你就可以很有信心地说"基频算准了"。在没有计算机的年代，这是结构工程师赖以生存的"两手绝活"。

高阶振型的截断问题

在使用振型叠加法时，我们通常只取前若干个振型。那么这个"若干个"到底是多少？判断标准有二：

  • 振型质量参与系数（Modal Mass Participation Factor）：每一阶振型贡献的有效质量占总质量的比例。规范通常要求参与质量之和达到总质量的90%以上。
  • 荷载的空间分布：如果荷载的空间分布与某阶振型正交（即φᵢᵀ·P=0），那么该阶振型根本不会被激发，即使它的频率很低也可以忽略。
一个实用的经验法则：对于地震作用下的规则结构，通常前3阶振型就能贡献90%以上的质量参与系数。对于风荷载下的高层建筑，通常前1~2阶振型占主导（因为风荷载的分布形状接近一阶弯曲振型）。但对于局部荷载（如某层楼面堆载），高阶振型的贡献可能不可忽略——一个局部冲击可能激发很多高阶振型，这时截断误差需要谨慎评估。

在这一章中，我们学习了多自由度体系振动的核心内容：
