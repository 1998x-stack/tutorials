从离散到连续：为什么需要函数极限

大家好，我是你们的讲师。上一章我们学习了数列的极限——数列是一系列离散的点：a₁, a₂, a₃, ...，自变量 n 只能取正整数。但现实世界中的变化是连续的：时间从 0 到 1 秒，中间经过了每一个实数时刻，而不是只经过"第 1 秒、第 2 秒"这样的离散点。

想想看：一辆汽车在 0 到 1 秒之间的速度是连续变化的。如果你想问"在 t = 0.5 秒这一瞬间的速度是多少"，数列极限帮不上忙——因为数列只描述了整时刻的值，无法处理"无限接近某个点"的情况。于是，我们需要把极限从离散的数列推广到连续的函数——也就是 函数的极限。

函数的极限回答的是这样一个问题：当自变量 x 无限接近某个值 x₀ 时，函数值 f(x) 会无限接近哪个值？注意关键——我们关心的是"无限接近"这个过程，而不是 x 在 x₀ 处本身的值。f(x) 在 x₀ 处甚至可能没有定义，但这丝毫不影响极限的存在。

极限的核心思想
函数在 x₀ 点的极限只关心"x 靠近 x₀ 时 f(x) 的趋势"，完全不关心 f(x₀) 本身是多少——甚至不关心 f(x₀) 是否存在。

这是刚开始学极限时最容易混淆的概念。之后学习函数的连续性时，你会发现"连续"和"有极限"的区别正在于此。

x→x₀ 时的极限：ε-δ 语言

数列极限用 ε-N 语言描述的是"当 n 足够大时，a_n 与 L 任意接近"。函数极限则需要一种新的精确语言——ε-δ 语言，同样由魏尔斯特拉斯完成严格化。

函数极限的 ε-δ 定义
设函数 f(x) 在 x₀ 的某去心邻域内有定义。如果对于任意给定的正数 ε，都存在一个正数 δ，使得当 0 < |x - x₀| < δ 时，总有 |f(x) - L| < ε，那么我们就说 lim_{x→x₀} f(x) = L。

让我们把这个定义和数列的 ε-N 定义做个对比：

概念 数列极限 ε-N 函数极限 ε-δ
自变量 n（正整数） x（实数）
趋近方式 n → ∞ x → x₀（或 ∞）
误差控制的"回应" 存在 N 存在 δ
条件 n > N 0 < |x - x₀| < δ

两种定义的逻辑结构完全一致：你给我一个容忍度 ε，我给出一个保证范围（N 或 δ），让在此范围外的所有点都在 ε 容忍度内。这就是"任意给定—存在"的极限语言框架。

例：证明 lim_{x→2} (3x - 1) = 5

对任意 ε > 0，我们需要找到 δ > 0，使得当 0 < |x - 2| < δ 时，|(3x - 1) - 5| < ε。

先算：|(3x - 1) - 5| = |3x - 6| = 3|x - 2|。

要使 3|x - 2| < ε，只需 |x - 2| < ε/3。取 δ = ε/3 即可。

ε-δ 证明标准格式（lim_{x→2} (3x - 1) = 5）：

        对 ∀ ε > 0，取 δ = ε/3，
        则当 0 < |x - 2| < δ 时，有
        |(3x - 1) - 5| = |3x - 6|
                     = 3|x - 2|
                     < 3δ
                     = 3·(ε/3) = ε
        因此 lim_{x→2} (3x - 1) = 5

左右极限与极限存在的条件

函数可以从 x₀ 的左侧靠近，也可以从右侧靠近。这就引出了左极限和右极限的概念：

  • 左极限：x 从小于 x₀ 的方向趋近，记作 lim_{x→x₀⁻} f(x)
  • 右极限：x 从大于 x₀ 的方向趋近，记作 lim_{x→x₀⁺} f(x)

函数在某点的极限存在，当且仅当左极限和右极限都存在且相等。这个条件看似简单，但却是判断分段函数在分段点处是否存在极限的关键工具。

例：判断 f(x) = |x|/x 在 x = 0 处的极限

这个函数在 x = 0 处没有定义（分母为0），但这不影响我们考察极限。

  • x > 0 时，|x|/x = x/x = 1，所以右极限 lim_{x→0⁺} |x|/x = 1
  • x < 0 时，|x|/x = -x/x = -1，所以左极限 lim_{x→0⁻} |x|/x = -1

左右不相等，所以 lim_{x→0} |x|/x 不存在。

踩坑记录：一个"看起来连续"的陷阱
曾经有一个同学问我：函数 f(x) = x·sin(1/x) 在 x = 0 处有没有极限？他观察了图像，觉得 sin(1/x) 震荡得太快了，看起来乱糟糟的，应该没有极限。

但实际计算 lim_{x→0} x·sin(1/x) = 0。为什么？因为 |sin(1/x)| ≤ 1，所以 |x·sin(1/x)| ≤ |x|。当 x → 0 时，|x| → 0，由夹逼定理，极限就是 0。虽然 sin(1/x) 本身在剧烈震荡，但乘以 x 这个"趋于 0 的衰减因子"后，震荡被压制住了。

这件事告诉我们：不要凭图像的"直观感觉"判断极限。极限是精确的数学概念，需要用数学工具（ε-δ、夹逼、四则运算）来精确判断。

x→∞ 时的极限

除了 x → x₀，函数极限的另一种重要情形是 x → ∞（或 x → -∞）。这时我们关心的是当自变量无限增大时，函数值趋向何处。

x→∞ 时函数极限的 ε-N 型定义
如果对于任意 ε > 0，存在 X > 0，使得当 x > X 时，|f(x) - L| < ε，那么 lim_{x→∞} f(x) = L。

这个定义本质上和数列极限相同——只是把"n > N"换成"x > X"。实际上，数列就是定义在自然数上的特殊函数，所以数列极限是函数极限的特例。

例：lim_{x→∞} 1/x = 0

这个例子和数列极限 lim 1/n = 0 几乎一样，差别只在于 x 可以在两个整数之间取任意实数值。但极限结果是一样的。

常见的 x→∞ 极限类型：

  • lim_{x→∞} 1/x^k = 0（k > 0）——幂函数的衰减
  • lim_{x→∞} a^x：如果 0 < a < 1，极限为 0；a > 1，极限为 ∞
  • lim_{x→∞} arctan x = π/2——反三角函数的水平渐近线

特别要注意的是，x→∞ 时的极限和 x→x₀ 时的极限在几何上有不同的含义：前者给出了水平渐近线的信息，后者则描述了函数在某个点附近的局部行为。

极限的四则运算法则

和数列极限一样，函数极限也满足四则运算法则——这是实际计算中用到最多的工具。

如果 lim f(x) = A 且 lim g(x) = B（A、B 为有限实数），那么：

运算 公式 条件
加法 lim (f + g)(x) = A + B 无
减法 lim (f - g)(x) = A - B 无
乘法 lim (f·g)(x) = A·B 无
除法 lim (f/g)(x) = A/B B ≠ 0

这些法则极大地简化了极限计算。比如：

lim_{x→1} (x^2 + 2x - 3) / (x + 1)

        分子：lim (x^2 + 2x - 3) = 1^2 + 2·1 - 3 = 0
        分母：lim (x + 1) = 2
        所以：lim = 0/2 = 0

但要注意除法的条件——分母的极限不能是 0。如果分母趋于 0，情况就复杂了，这就是我们将在第7章讨论的"未定式"问题。

函数极限与数列极限的关系：海涅定理

函数极限和数列极限之间有一个深刻的联系——海涅定理（Heine's Theorem）：

海涅定理
lim_{x→x₀} f(x) = L 当且仅当：对任意满足 x_n → x₀ 且 x_n ≠ x₀ 的数列 {x_n}，都有 lim f(x_n) = L。

这个定理的意义在于：它把函数极限的问题归结为数列极限的问题。换句话说，只要你能证明"无论数列 {x_n} 以什么方式趋近 x₀，对应的 f(x_n) 都趋近 L"，那么函数极限就存在且为 L。

海涅定理的一个特别有用的应用是证明某个极限不存在。如果你能找到两个不同的数列 {x_n} 和 {y_n}，它们都趋近 x₀，但 f(x_n) 和 f(y_n) 趋近于不同的值，那么函数极限就不存在。

例：用海涅定理证明 lim_{x→0} sin(1/x) 不存在

取 x_n = 1/(nπ)，则 x_n → 0，而 sin(1/x_n) = sin(nπ) = 0，所以 lim f(x_n) = 0。

再取 y_n = 1/(2nπ + π/2)，则 y_n → 0，而 sin(1/y_n) = sin(2nπ + π/2) = 1，所以 lim f(y_n) = 1。

两个数列对应的函数值极限不同，所以 lim_{x→0} sin(1/x) 不存在。

海涅定理的直观理解
想象你从四面八方走向同一个点 x₀。如果无论你从什么路径走向 x₀，f(x) 都趋向同一个 L，那么函数极限就存在。但如果你走一条路时 f(x) 趋向 0，走另一条路时 f(x) 趋向 1，那说明函数在这个点附近的行为受到路径影响——此时函数极限不存在。

这个思想在后面学习多元函数时会更重要——在二维平面上，趋近一个点有无穷多种"方向"，函数极限是否存在就复杂多了。

本章我们从数列的离散极限过渡到了函数的连续极限，掌握了 ε-δ 定义和左右极限的概念，理解了极限四则运算法则及其使用条件，并学习了通过海涅定理将函数极限问题转化为数列极限问题的思想。函数极限是微积分真正的发动机——导数、积分、连续性分析都离不开它。
