第三章：需求曲线——支付意愿与需求定律

大家好！上一章我们画了PPF曲线，学了比较优势，理解了为什么专业化分工和贸易能让所有人变好。从这一章开始，我们要进入市场分析的核心——需求和供给。这可能是你整个微观经济学课程中最重要的一对概念。

今天咱们先从需求一侧讲起。别急，我知道"需求曲线"听起来像高中政治课本里的内容。但相信我，当你真正理解了需求曲线背后的人性逻辑，你会发现它解释的不只是市场——它还解释了你为什么会买那杯45块钱的奶茶，为什么会为了省5块钱绕路去另一家超市，以及为什么那些"限时抢购"的倒计时总能让你冲动下单。

需求定律

需求定律 Law of Demand是经济学中最基础的规律之一。它的表述很简单：在其他条件不变的情况下，当一种商品的价格上升时，需求量减少；当价格下降时，需求量增加。

换句话说，价格和需求量之间是反向关系 inverse relationship。这听起来像废话——谁会反对呢？但真正有意思的是"为什么"。

需求曲线向下倾斜有三个层面的原因：

  • 边际效用递减——你消费越多，每多一个单位的满足感就越低，所以你愿意付的钱也越少。第一杯水值10块，第十杯可能只值1毛了。
  • 替代效应——当咖啡涨价的时候，你可能会换成茶。当牛肉涨价了，你会多买鸡肉。价格上升使该商品相对于替代品更贵，人们就会转向替代品。
  • 收入效应——价格上涨让你的实际购买力下降了，相当于你"变穷了"——所以你会减少所有正常商品的需求，不仅仅是被涨价的这个。

我们先用一个简单的需求表来感受一下：

一杯奶茶的价格（元） 小张每周买的杯数 解释
30 0 太贵了，不买
20 1 "偶尔犒劳自己"的心理价位
12 3 和同事拼单，一周买几次
5 7 几乎每天一杯

每个价格对应一个需求量。当价格从20降到12，小张从买1杯变成买3杯——这就是需求定律在生活中的表现。

从个体需求到市场需求

了解了单个消费者的需求曲线之后，下一步是把所有人的需求加总起来，得到市场需求曲线 market demand curve。

方法出奇地简单：水平加总 horizontal summation。对于每一个价格，把所有消费者在该价格下的需求量加起来，得到的就是这个价格下的"市场需求量"。

用一段伪代码来表示就是：

# 市场需求 = 所有消费者需求量的水平加总
def market_demand(price):
    total_quantity = 0
    for consumer in all_consumers:
        total_quantity += consumer.demand(price)
    return total_quantity

# 示例：三种商品价格下的加总
price = 10:
  小张: 4杯, 小李: 2杯, 小王: 3杯
  市场总需求 = 4 + 2 + 3 = 9杯

price = 20:
  小张: 1杯, 小李: 0杯, 小王: 1杯
  市场总需求 = 1 + 0 + 1 = 2杯

市场上每个人的需求曲线都是向下倾斜的，加总之后仍然是向下倾斜的。市场的需求曲线就是这个经济体中无数消费者"投票"的结果——每个购买决定就是一票，价格越低，投"购买"票的人就越多。

思考练习：你是否注意到，扫码点餐系统会在大促时段显示"已有xx人下单"？这不是心理暗示，这是需求信号——商家在实时收集需求数据来调整定价和库存。大数据的出现让经济学家几十年前提出的"需求曲线"从一个理论概念变成了可以实时计算的实际工具。

替代效应与收入效应直觉

刚才提到了需求曲线向下倾斜的两个核心机制——替代效应 substitution effect 和 收入效应 income effect。这两个概念在第10章我们会用数学和无差异曲线详细推导，但这里先用直觉理解它们。

替代效应是指：当一种商品的价格相对其他商品变得更贵时，消费者会减少该商品的消费、增加替代品的消费。

  • 可口可乐涨价 → 你开始买百事可乐
  • 苹果手机涨价 → 你考虑安卓手机
  • 猪肉涨价 → 你多吃鸡肉和鱼肉

收入效应是指：当一种商品涨价时，你的实际购买力（真实收入）下降，所以你会减少所有商品的消费（对正常商品而言）。

  • 房租大涨 → 你发现每个月可自由支配的钱变少了 → 不仅租同样的房子更贵了，你连出去吃饭、看电影的次数也减少了

两种效应同时起作用，让需求曲线坚定不移地向下倾斜。绝大多数情况下，两种效应方向一致。唯一可能出岔子的特殊情况我们会在后面讲到——那就是吉芬商品。

需求曲线的移动 vs 沿曲线移动

这是初学经济学时最容易混淆的概念，没有之一。请认真看下面这一段。

沿着需求曲线的移动和需求曲线本身的移动，是两个完全不同的概念：

类型 原因 图示 例子
沿曲线移动（量变） 该商品自身价格变化 同一条曲线上的不同点 咖啡从25降到20 → 你买3杯而不是2杯
曲线移动（质变） 非价格因素变化 整条曲线左移或右移 你升职加薪了 → 所有价格下都买更多咖啡

哪些因素会让需求曲线整体移动（而不是沿着曲线滑动）？

  • 收入变化——赚钱多了，对正常品的需求曲线右移；收入下降则左移
  • 相关商品价格变化——咖啡机降价 → 咖啡需求曲线右移（互补品），茶涨价 → 咖啡需求曲线右移（替代品）
  • 偏好 taste / preferences——某个网红带货了 → 需求右移；食品安全丑闻 → 需求左移
  • 预期 expectations——你觉得下周要涨价 → 今天先囤货 → 需求右移
  • 买者数量——城市人口增长 → 所有商品需求右移

踩坑案例：某品牌误判"需求移动"：2018年，一个国产运动品牌看到销量连续三个季度下滑，认为是产品定价太高（沿曲线移动），于是大幅降价促销。结果销量仍然上不去——后来调研发现，真正的原因是消费者的偏好从"运动休闲风"转向了"机能风"，整条需求曲线已经左移了。降价的策略在一个移动了的需求曲线上毫无作用。这个品牌白白损失了利润空间，还损伤了品牌形象。记住：分清"沿着曲线走"还是"曲线本身动了"，这是市场分析的第一个分水岭。

正常品与劣等品

上面提到"收入变化会让需求曲线移动"，但方向并不总是相同的。这取决于商品是正常品 normal good还是劣等品 inferior good。

  • 正常品：收入增加 → 需求增加（曲线右移）。大多数商品都是正常品——衣服、电子产品、出去吃饭、旅游。
  • 劣等品：收入增加 → 需求减少（曲线左移）。注意"劣等"不是指品质差，而是指在收入高的时候你会转向更好的替代品。

经典的劣等品例子：

  • 方便面——大学刚毕业工资低，天天吃方便面。工作几年后收入上来了，你开始叫外卖、下馆子，方便面的需求就下降了。
  • 公共巴士——收入低的时候坐公交，收入高了买私家车或者打车，公交的需求下降。
  • 二手衣物——收入低的时候买二手，收入提高后买新衣服。

区分正常品和劣等品在商业分析中非常实用——如果你做的是劣等品生意（比如平价速食），你的增长天花板会受到整体经济走势的影响。经济下行时大家收入减少，反而可能利好你的生意——这就是为什么2022年经济承压时，方便面企业的股价反而上涨了。

吉芬商品：需求定律的例外

最后来讲一个经济学里的传说——吉芬商品 Giffen good。

前面我们说需求定律"几乎所有情况下都成立"，但这个"几乎"留下了悬念。有没有一种商品，价格越高，需求量反而越大？这听起来完全违背直觉，但在理论上确实存在——而且有一个著名的历史案例。

爱尔兰马铃薯大饥荒 Irish Potato Famine（1845-1852年）。故事是这样的：马铃薯是当时爱尔兰穷人的主食，占他们食物预算的很大一部分。那几年马铃薯歉收，价格暴涨。

按理说，价格涨了，需求量应该下降，对吧？但经济学家罗伯特·吉芬 Robert Giffen 观察到：马铃薯涨价后，贫穷的爱尔兰农民反而买了更多的马铃薯。

为什么？因为马铃薯涨价导致农民的实际收入大幅缩水（收入效应巨大）。他们变得更穷了，买不起肉、奶酪等其他食品了，于是只能把剩下的有限预算全部用来买马铃薯——即使马铃薯变贵了，他们也只能买更多马铃薯来填饱肚子。

换句话说，马铃薯在这里既是"劣等品"（穷人吃的），又占据了极大比例的预算。在极端贫困的情况下，收入效应压倒了替代效应，导致价格上涨反而带来了需求量上升。

关键洞察：吉芬商品在现实中极其罕见甚至可以说几乎不存在。因为它需要同时满足三个条件：① 必须是劣等品；② 必须占消费者预算的极大比例；③ 必须有足够强的收入效应来压倒替代效应。这也是为什么大多数经济学教材只在脚注里提到它——它是一个理论可能性，而不是你需要日常担心的商业场景。如果你觉得自己公司在卖吉芬商品，大概率是你把"涨价后销量上升"归因到了错误的原因（比如可能是品牌升级带来的认知变化）。

本章小结

今天的内容是微观经济学中最基础但也最容易被误解的部分。我们快速过一遍：

  • 需求定律：价格和需求量呈反向关系，因为边际效用递减、替代效应和收入效应三者共同作用。
  • 从个体到市场：市场需求曲线就是所有个体需求曲线的水平加总。
  • 移动类型区分：价格变化导致"沿着曲线移动"；收入、偏好、相关品价格、预期、买者数量变化导致"曲线本身移动"。这两个概念绝对不能混淆。
  • 正常品 vs 劣等品：收入增加时需求增加的是正常品，需求减少的是劣等品。
  • 吉芬商品：需求定律的罕见例外，爱尔兰马铃薯饥荒是经典案例，但在现实中几乎不需要担心。

下一章，我们走到市场的另一侧——供给曲线。如果需求曲线反映的是消费者的"意愿"，供给曲线反映的就是生产者的"意愿"。这两个力量加在一起，市场才会真正运转起来。
