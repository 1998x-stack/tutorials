26.1 6G愿景与天线传播新需求

6G的KPI表是令人瞠目结舌的：峰值速率1 Tbps（是5G的50倍）、空口时延0.1 ms（是5G的十分之一）、连接密度107 devices/km²（是5G的10倍）、厘米级定位精度。这些数字不只是空谈，它们对天线和传播工程提出了根本性的新要求。

最直观的变化是新频谱的开放。5G NR已经涵盖了FR1（sub-6 GHz）和FR2（mmWave, 24-52 GHz）。6G将在此基础上大幅拓展：

  • FR3（7-24 GHz）：又称"黄金频段"，介于sub-6 GHz的覆盖优势和mmWave的大带宽之间。这个频段的天线需要在尺寸、效率和带宽之间精细平衡。
  • mmWave扩展（100+ GHz）：从5G的24-52 GHz推进到100 GHz以上，大气吸收开始成为重要因素。
  • sub-THz（100-300 GHz）：分子吸收线变得显著，水分子（H₂O）和氧气（O₂）的共振吸收峰开始分割频谱。
  • THz频段（300 GHz - 3 THz）：这是真正的"无人区"——极宽的带宽（以百GHz计），但传播距离极度受限。

这对天线和传播设计的含义是深刻的：

  • 近场主导：随着频率升高和阵列尺寸增大，Fraunhofer距离不断延伸，用户越来越频繁地工作在阵列的近场区。这不再是特殊情况——而是常态。
  • 稀疏散射：高频电磁波的行为越来越像光——反射占主导，绕射微弱，散射体数量减少。这让传统统计信道模型的适用性受到挑战。
  • 分子吸收峰：大气中的水蒸气和氧气分子在某些频率上有强烈的吸收共振。这些吸收峰之间的"窗口"频谱成为优选工作频段。

关键洞察：6G的天线设计不再是"天线本身的设计"，而是"天线-传播-信号处理"的联合设计。天线工程师需要同时理解传播物理和信号处理算法——跨学科能力将成为核心竞争力。

26.2 超大尺度天线阵列（ELDAA/xMIMO）

Massive MIMO是5G的标志性技术——典型配置是64T64R（64发射64接收）。6G将把这个数字推到1024甚至10000+。这就是ELDAA（Extremely Large Aperture Array Antenna）或xMIMO的概念。

但这里有一个"不寻常"的现象：当天线阵列变得极大时，传统的远场假设开始崩溃。

回忆一下第7章学过的Fraunhofer距离（远场距离）：

import numpy as np

def fraunhofer_distance(D, f):
    """
    计算Fraunhofer距离 (远场边界)
    参数:
        D : 天线最大尺寸 (m)
        f : 频率 (Hz)
    返回:
        d_f : Fraunhofer距离 (m)
    """
    c = 3e8
    lambd = c / f
    d_f = 2 * D**2 / lambd
    return d_f

# 案例：5G基站阵列 vs 6G ELDAA
print("--- 5G Massive MIMO (D=0.5m, 3.5 GHz) ---")
d_f_5g = fraunhofer_distance(0.5, 3.5e9)
print(f"Fraunhofer距离: {d_f_5g:.1f} m")

print("\n--- 6G ELDAA (D=1m, 30 GHz) ---")
d_f_6g = fraunhofer_distance(1.0, 30e9)
print(f"Fraunhofer距离: {d_f_6g:.1f} m")

print("\n--- 6G ELDAA (D=2m, 100 GHz) ---")
d_f_6g2 = fraunhofer_distance(2.0, 100e9)
print(f"Fraunhofer距离: {d_f_6g2:.1f} m")

输出结果令人震惊：对于D=1m、30 GHz的系统，Fraunhofer距离为200米——这意味着绝大多数用户都在近场区内！在近场中，波前不再是平面波，而是球面波。这带来一个根本性的变化：

波束聚焦 vs 波束转向。在远场中，我们只能控制波束的方向（波束转向，beam steering）。而在近场中，我们可以同时控制波束的方向和距离——这就是波束聚焦（beam focusing）。你在不同距离上可以形成不同的波束，这为空间复用增加了一个全新的维度。

更妙的是，研究人员发现近场实际上可以增加系统的自由度（DOF）。我给你们讲一个故事：

2021年，几个意大利和瑞典的研究者在IEEE TSP上发表了一篇论文，题目很不起眼——"近场信道的自由度增加"。他们严格证明了一个反直觉的结论：对于同样的阵列孔径，球面波前相对于平面波前提供了更多的空间自由度。原因是球面波前的相位曲率携带了距离信息——这相当于从2D（角度域）扩展到了3D（角度-距离域）。起初审稿人都不太相信——"近场不是一直被认为是个麻烦事吗？"但数学证明无懈可击。今天，近场MIMO已经成为一个独立的研究子领域，每年有数百篇论文发表。

学术小贴士：如果你对这个方向感兴趣，推荐阅读以下里程碑文献——M. Cui等人的"Near-Field MIMO Communications"（IEEE JSAC, 2023）和H. Zhang等人的"Holographic MIMO Surfaces"（IEEE TSP, 2022）。它们从信息论角度重新定义了大型阵列的理论极限。

除了近场效应，ELDAA还面临空间非平稳性（spatial non-stationarity）的挑战。简单来说，阵列太大了，阵列不同部分看到的传播环境可能完全不同——左侧单元看到的是来自东边的散射簇，右侧单元看到的是西边的。阵列的尺寸超过了平稳区域（stationarity region）。这意味着我们不能假设所有单元经历相同的信道统计特性，传统的信道模型需要彻底重写。

ELDAA的信道建模至少需要处理以下四个新的复杂性：球面波前（而非平面波）、空间非平稳性、单元间互耦（大阵列中尤其严重）、以及双移动性（基站和用户都可能高速移动）。这是一个正在积极研究中的开放问题。

26.3 Pinching天线系统

在2022年，日本运营商NTT DOCOMO展示了一种令人好奇的天线概念——你拿一根看起来像普通电缆的东西，用手指在某处"捏"一下，电磁波就从那里辐射出来了。这就是Pinching天线（"捏合"天线）的基本想法。

让我们从物理上理解这是怎么做到的。Pinching天线的基本结构是一根低损耗介质波导——某种特制的介电材料条带，RF信号沿着它传播，损耗很低。正常情况下，信号沿着波导走，几乎不向外辐射（就像光在光纤中传播一样）。但是，如果你在波导表面引入一个扰动——比如机械地压一下、或者在表面贴一个金属片——波导中的传播模式就会在这个扰动点发生泄漏，产生辐射。

这就好比你在花园水管上用力捏一下，水就会从捏的地方的缝隙中喷出来。Pinching天线的名字正是由此而来。

Pinching天线的核心优势在于传播损耗极低。在介质波导中，信号以表面波形式传播，损耗大约3.2 dB/m（60 GHz），而同样的距离在自由空间传播损耗是68 dB/m——相差20倍以上。这意味着你可以把RF信号"运输"到很远的位置再辐射出去，效率远高于传统的馈电网络。

2026年的最新进展是电子式Pinching（E-Pinching）。早期的Pinching天线需要机械扰动（比如用微型马达压下波导表面），这使得响应速度和可靠性都有限。E-Pinching使用变容二极管（varactor）或PIN二极管在波导表面产生可调的电抗性扰动。通过改变偏置电压，可以精确控制辐射幅度和相位，实现9-10 dB的重构范围。这意味着Pinching天线可以从一个固定的介质波导变成一个可编程的辐射表面。

Pinching天线的潜在应用非常诱人：

  • 用户中心覆盖：可以沿着波导在任意位置产生辐射点，适配用户的位置变化。
  • 室内遮挡缓解：在走廊或开放办公室中，用一根波导覆盖整层，"捏"出多个辐射点来避开柱子或隔断。
  • 同时LoS和NLoS服务：不同"捏合点"可以配置为指向不同方向，同时服务视距和非视距用户。
工程提醒：Pinching天线虽然概念优雅，但尚处于早期研究阶段。主要挑战包括：介质波导的材料损耗（尤其是100 GHz以上）、多个捏合点之间的互耦、以及如何高效率地同时激励多个辐射点。目前还没有商用产品，但各大厂商（爱立信、华为、NTT）都在积极布局专利。

26.4 通信感知一体化（ISAC）

想象一下：你的基站在给手机发数据的同时，还能像雷达一样"看见"周围的环境——检测到一辆车正在靠近、一个人正在过马路、一架无人机正在禁飞区边缘徘徊。这不是科幻，这是通信感知一体化（Integrated Sensing and Communication, ISAC）——6G的杀手级应用之一。

ISAC的理念是让通信和雷达共享频谱、硬件和波形。过去，通信和雷达是两个各自独立的系统：通信关心比特传输，雷达关心目标探测。但它们的物理层本质上是相同的——都要发射电磁波、分析回波或信道响应。为什么不能合二为一呢？

从天线角度看，ISAC提出了独特的要求：

  • 共享孔径：通信和感知功能共用同一组天线单元，需要精心设计馈电网络来在两种模式间切换或同时工作。
  • 双极化：感知需要极化分集来识别目标形状，通信需要极化复用来提高容量。
  • 宽带能力：感知的距离分辨率正比于带宽：ΔR = c / (2B)。要实现厘米级分辨率，需要GHz级的带宽。
  • 波束赋形兼顾：通信波束需要宽（覆盖用户），感知波束需要窄（高角度分辨力）。如何在两者间优化是一个难题。

传播建模方面，ISAC需要同时描述通信信道和目标散射。这意味着传播模型需要扩展到包含目标的雷达散射截面（RCS）、微多普勒特征、以及杂波统计。传统上，通信信道建模和雷达目标建模是两个几乎不交流的领域——ISAC正在推动它们的融合。

波形设计上，OFDM成为了一个自然的候选，因为5G/NR已经基于OFDM。OFDM天然支持频域处理，可以通过对通信符号进行距离-多普勒处理来提取感知信息。一个典型的ISAC接收机结构是：通信接收机提取比特，同时雷达处理模块从相同的OFDM符号中计算距离-多普勒图。

功能 天线需求 传播模型需求 波形需求

目标探测与定位 低副瓣、精确相位控制 目标RCS + 杂波统计 大带宽、良好的模糊函数
目标成像 大孔径、双极化、宽角扫描 环境散射体分布 波形分集、MIMO雷达波形

值得关注：3GPP在Rel-20中已经将ISAC列为6G研究课题，IEEE 802.11bf（WLAN Sensing）也在推进类似标准化。预计第一批ISAC商用产品将在2029-2030年出现——正好赶上6G的第一波部署。如果你正在寻找博士或研究方向，ISAC绝对是一个热门赛道。

26.5 太赫兹通信的天线与传播挑战

如果说mmWave是"可用但有点挑战"，那么THz通信就是"能用简直是奇迹"。我们来看看为什么。

首先，分子吸收是THz传播的头号敌人。THz频段中，水蒸气（H₂O）和氧气（O₂）分子有大量的旋转和振动吸收线，它们像梳子一样把THz频谱分割成一系列"窗口"。目前关注的主要窗口包括：140 GHz、220 GHz、340 GHz、410 GHz、670 GHz和850 GHz附近。在这些窗口中，大气衰减相对较低（可能低至几个dB/km），但在吸收线上，衰减可以高达数百dB/km——通信不可能进行。

其次，散射极其稀疏。THz波的波长在0.1-1 mm之间，而大多数日常物体的表面粗糙度与波长相当甚至更大。这意味着THz波在粗糙表面的散射是漫反射，而不是镜面反射。绕射更是微乎其微——一个几厘米的障碍物就能产生极大的绕射损耗。THz传播几乎完全依赖LoS（视距）和少量镜面反射。它的行为已经非常接近可见光了。

第三，波束跟踪在THz频段是噩梦级的挑战。典型的THz波束宽度在0.1-1°之间——想象一下，你拿着一支激光笔对着100米外的一个硬币大小的目标，要保持稳定不晃动，即使轻微的震动或目标移动都会导致信号中断。在移动场景中，预测性波束跟踪（predictive beam tracking）和多连接（multi-connectivity）成为必需——基站必须提前预测用户位置并准备备选波束。

THz天线技术本身也是一个巨大的挑战。传统意义上的"天线"在THz频段已经不太合适了。当前的主流方向包括：

  • 片上天线（On-chip antennas）：直接在SiGe或InP芯片上集成天线，用芯片的金属层做辐射结构。优点是集成度高，缺点是效率低（衬底损耗大）。
  • 透镜耦合天线（Lens-coupled antennas）：在硅透镜表面做小天线，用透镜来聚焦波束。效率较高，但体积较大。
  • 漏波天线（Leaky-wave antennas）：利用周期性结构产生空间谐波辐射，天然适合频率扫描。
  • 石墨烯等离子体天线（Graphene-based plasmonic antennas）：石墨烯的表面等离子体效应可以在THz频段实现亚波长束缚和可调谐性——但尚处于实验室阶段。

我来分享一个让人印象深刻的故事。2018年，日本NICT（国家信息与通信技术研究所）的研究团队在300 GHz上实现了100+ Gbps的无线传输，距离仅10米。他们使用了超导SIS混频器和精心设计的透镜天线——整个系统占了半个实验室的台面。相比之下，今天的商用5G基站已经可以在28 GHz上实现几百Mbps到几Gbps的实用系统，设备大小与一个冰箱相当。十年后，THz系统能否做到同样的小型化和可靠性？这是悬而未决的问题。从实验室到产品，THz通信还有很长的路要走。

从6G天线技术前沿的讨论中，一个清晰的趋势是：天线系统的复杂度已经远远超出了传统解析方法和经验法则能处理的范围。随着我们进入近场区、面对非平稳信道、需要同时完成通信和感知功能，传统"试错+仿真"的设计方法开始显得力不从心。这正是人工智能介入的最佳时机。在第27章中，我们将深入探讨AI如何彻底改变天线设计的方式——从神经网络替代模型加速仿真，到生成模型自动设计天线几何结构。

本章我们预览了6G時代天线与传播技术的5个前沿方向。从ELDAA超越Fraunhofer距离带来的近场波束聚焦革命，到Pinching天线用介质波导"捏"出可编程辐射点，从ISAC让基站既能说话又能"看"，到THz频段极端的传播限制和天线挑战——每一个方向都在重新定义我们这门学科的传统边界。

这些前沿技术的共同特点是：复杂度极高、设计空间极高、传统方法日益不够用。第27章将为你介绍这个困局的答案——AI驱动的天线设计与优化。我们不仅会学遗传算法、粒子群优化、神经网络替代模型，还会看到深度学习如何从根本上改变信道估计和波束管理。
