开场：电感与电容——电路中的对偶精灵

电容器储存电场能量，电感器储存磁场能量——二者在电路理论中是对偶的，但物理本质截然不同。电容的储能来自电荷分离（电场），电感的储能来自电流建立的磁场。在交流电路中，正是这对"对偶精灵"决定了系统的谐振频率、阻抗特性以及瞬态响应。

更深刻地，电感不仅是电路元件，它反映了磁场能量（magnetic field energy）的集中表现。今天我们就从自感和互感的定义出发，深入理解LR电路的暂态过程、磁场能量的计算，以及有趣的虚功原理。

自感

当一个线圈中的电流变化时，电流产生的磁场也随之变化，从而在线圈自身中感应出反向电动势——这就是自感（self-inductance）现象。

自感系数（电感量）L 定义为：

L = N Φ / I

其中 N 是线圈匝数，Φ 是单匝穿过的磁通量，I 是电流。电感量的单位是亨利（Henry, H），1 H = 1 Wb/A = 1 V·s/A。

自感电动势：εL = −L dI/dt

常见构型的电感计算

  • 长直螺线管（长 l，截面积 A，匝数 N）：L = μ N² A / l
  • 环形螺线管（内半径 a，外半径 b，高 h，匝数 N）：L = μ N² h · ln(b/a) / (2π)
  • 同轴电缆（内半径 a，外半径 b，单位长度）：L' = (μ₀ / 2π) ln(b/a) —— 只有内外导体之间的区域贡献电感

互感

当一个线圈中的电流变化时，其产生的变化磁场会在邻近的另一个线圈中感应出电动势——这就是互感（mutual inductance）。互感系数 M 定义为：

M₁₂ = N₂ Φ₁₂ / I₁ = M₂₁ = N₁ Φ₂₁ / I₂

其中 Φ₁₂ 是线圈 1 的电流 I₁ 在线圈 2 中产生的磁通量。互感的互易性（M₁₂ = M₂₁）是电磁学中一条深刻且实用的性质。

耦合系数 k 表示两个线圈磁耦合的紧密程度：

k = M / √(L₁ L₂)  0 ≤ k ≤ 1

  • k = 1：完美耦合（所有磁通都穿过两个线圈，理想变压器）
  • k ≈ 0.9-0.99：紧密耦合（变压器铁芯）
  • k ≈ 0.1-0.5：松耦合（无线充电、RFID）
  • k ≈ 0：无耦合（间距很大的线圈）

LR 电路暂态

电感在电路中的核心特征是抵抗电流的变化——这不是阻碍电流流动，而是让电流不能突变。

电流建立（RL 串联接电源）

电路方程：ε = L dI/dt + R I

解（初始条件 I(0)=0）：I(t) = (ε/R) (1 − e−tR/L)

电流从零开始，以指数形式趋近于稳态值 ε/R。当 t = τ = L/R 时，电流上升到最终值的 63.2%。

电流衰减（电源短接）

电路方程：0 = L dI/dt + R I

解（初始条件 I(0)=I₀）：I(t) = I₀ e−tR/L

电流从 I₀ 按指数衰减到零。在 t = τ 时，衰减到初始值的 36.8%。

时间常数 τ = L/R 是 LR 电路最重要的参数。τ 越大，暂态过程越长。对于电源设计，通常认为经过 5τ 后暂态结束（电流达到稳态的 99.3%）。

储能元件 储能公式 能量密度 时间常数 对偶量
电容器（C） W = ½ C V² uE = ½ ε E² τ = RC 电压不能突变
电感器（L） W = ½ L I² uB = ½ B·H = B²/(2μ) τ = L/R 电流不能突变
电阻器（R） P = I² R（耗散） — — 消耗能量

磁场能量

电感器储存的能量来源于电流建立过程中电源克服自感电动势所做的功：

W = ∫ P dt = ∫ L I dI = ½ L I²

但问题来了：能量"储存在哪里"？

在我们建立电流的过程中，磁场也在空间中建立起来。更深刻的答案是：能量储存在磁场中，而不是在电感器里。电感器只是磁场能量的集中表现。

磁场能量密度：uB = ½ B · H = B² / (2μ)（对线性介质）

这与电场能量密度 uE = ½ ε E² = ½ D · E 完美对称。

整个空间的磁场总能量：W = ∫V ½ B · H dV

这一观点在工程中有重要的实际意义：

  • 设计变压器时，磁芯中的气隙会改变磁场能量分布——气隙中 B 相同但 μ 较小，能量密度更高。
  • 在电感器设计中，储能能力决定了器件的饱和电流。
💡 工程警示：电感饱和
电感设计中最常见的问题就是饱和（saturation）。当电感器中的电流增大到一定程度，磁芯材料达到饱和磁化强度 Bsat，此时 μr 急剧下降（从数千降为接近 1），电感量也随之暴跌。
后果是什么？在开关电源中，电感饱和意味着对电流的"阻力"突然消失，电流急剧上升——这会烧毁开关管！
经验法则：选型时不仅要注意电感量 L，更要关注额定饱和电流 Isat ——确保工作电流的最大值 < 80% Isat。

磁力与虚功原理

磁场不仅能储存能量，还能产生力——这是电机、继电器、电磁阀的物理基础。

虚功原理

有两种情况：

  • 恒磁链条件（Φ = const，如永磁体周围的铁磁体移动）：F = −∂W / ∂x（能量减少的方向受力）
  • 恒电流条件（I = const，如电磁铁通电时衔铁被吸引）：F = +∂W / ∂x（外部电源补充能量，力为磁能的增长率）

电磁铁吸力

一个典型例子：电磁铁吸引衔铁。气隙中的磁场产生吸引力：

F = B² A / (2μ₀)

其中 A 是磁极面积，B 是气隙磁感应强度。这个公式说明：磁场对铁磁体的吸引力正比于 B²——类似于电场力正比于 E²。

🔑 核心洞察
磁力与电力的对称性是电磁学中最美的主题之一：
电场能量密度 uE = ½ ε E² → 电场对导体的力 ∝ E²
磁场能量密度 uB = ½ μ H² → 磁场对磁体的力 ∝ H²
这种对称性不是偶然的——它是麦克斯韦方程组在静态极限下的必然结果。当电场和磁场在电磁波中交融时，这种对称性变得完美无缺。

Python 模拟：RL 电路暂态

import numpy as np
import matplotlib.pyplot as plt

# 参数
R = 10.0         # 电阻 (Ω)
L = 50e-3        # 电感 (H)
V = 5.0          # 电源电压 (V)
T = 20e-3        # 仿真时长 (s)
dt = 1e-6
N = int(T / dt)

t = np.linspace(0, T, N)
tau = L / R      # 时间常数

# 解析解：电流上升
I_rise = (V / R) * (1 - np.exp(-t / tau))
# 解析解：电流衰减（从稳态开始）
I0 = V / R
I_decay = I0 * np.exp(-t / tau)

plt.figure(figsize=(10, 6))
plt.plot(t * 1000, I_rise * 1000, 'b-', label='电流建立 (接电源)')
plt.plot(t * 1000, I_decay * 1000, 'r--', label='电流衰减 (短接)')

# 标注时间常数
plt.axvline(tau * 1000, color='gray', ls=':', label=f'τ = L/R = {tau*1000:.1f} ms')
plt.axhline(0.632 * V/R * 1000, color='green', ls=':', alpha=0.5)
plt.axhline(0.368 * V/R * 1000, color='orange', ls=':', alpha=0.5)

plt.xlabel('时间 (ms)')
plt.ylabel('电流 (mA)')
plt.title('RL 电路暂态响应')
plt.legend()
plt.grid(alpha=0.3)
plt.show()

# 计算储存的磁场能量
W_steady = 0.5 * L * (V/R)**2
print(f"稳态磁场能量: W = ½LI² = {W_steady*1e6:.2f} μJ")

本章小结

  • 自感 L：线圈抵抗电流变化的度量。L = NΦ/I，自感电动势 ε = −L dI/dt。
  • 互感 M：两个线圈之间的磁耦合。M = k√(L₁L₂)，耦合系数 k 范围 0-1。
  • LR 暂态：时间常数 τ = L/R，电流以指数规律建立或衰减。
  • 磁场能量：W = ½ L I² = ∫ ½ B·H dV，能量密度 uB = B²/(2μ)。
  • 磁场能量是一种场能量——能量分布在空间中的磁场里，而不是"困"在电感器中。
