第21章 国际投资

如果你只投资A股，意味着你放弃了全球超过80%的可投资资产。美国有苹果和英伟达，日本有丰田和索尼，欧洲有LVMH和SAP，印度有Reliance和Infosys。每一个市场都有其独特的产业优势和增长动力。国际投资的核心问题不是"要不要出去"，而是"怎么出去"。

这一章我们会探讨国际投资的收益来源、独特的风险因素（汇率、政治、法律）、以及如何通过全球配置真正实现分散化。我们还会重点讨论中国A股的国际化和全球市场的实战配置逻辑。

日本"渡边太太"的汇率噩梦

20世纪90年代末到21世纪初，日本的家庭主妇们发现了一个"稳赚不赔"的策略——日元利率接近零，而澳元、新西兰元利率高达5-7%。她们借入日元，换成澳元或新西兰元购买高息债券。这个群体被称为"渡边太太"（Mrs. Watanabe），她们一度控制了日本家庭储蓄的相当大一部分。

2008年金融危机期间，全球投资者恐慌性抛售风险资产，日元作为避险货币大幅升值。美元兑日元从2007年的124跌到2008年的87，跌幅超过30%。那些持有澳元/新西兰元多头头寸的渡边太太们面临双重打击：高息货币贬值 + 日元升值使债务成本飙升。许多人被迫平仓，损失惨重。

核心教训："渡边太太"的失败在于完全忽视了汇率风险。高息差带来的收益在汇率波动面前可能微不足道。国际投资中，汇率是最大但也是最容易被忽视的风险因子。

21.1 国际组合投资的收益来源

国际投资的回报可以分解为三个部分：

  • 本地收益（Local Return）：以当地货币计价的资产价格变动和股息/利息收入
  • 汇率变动（Currency Return）：外币兑本币的汇率变化
  • 协方差效应：本地收益和汇率变动的交互影响
# 国际投资总收益分解
# 假设你投资美股标普500指数基金
# 以人民币计价的年化收益率 = ?

# 分解公式：
# R_CNY = (1 + R_local) × (1 + R_fx) - 1
# 其中：
# R_local = USD计价的收益率（如10%）
# R_fx = 美元兑人民币的年升值率（如2%）

# R_CNY = (1 + 0.10) × (1 + 0.02) - 1 = 0.122 = 12.2%

# 如果人民币升值（R_fx为负值），则会侵蚀收益
# 假设美元兑人民币贬值3%：
# R_CNY = (1 + 0.10) × (1 - 0.03) - 1 = 0.067 = 6.7%

从分解中可以看到，即使本地收益完全相同，不同货币计价的回报可能天差地别。这也是为什么国际投资者必须管理汇率风险。

21.2 汇率风险

汇率决定理论是国际金融的基石，这里我们关注两个在投资中最实用的模型。

购买力平价（PPP）：认为长期来看，汇率应该调整到使得一篮子商品在不同国家的价格相同。如果中国一个巨无霸卖20元，美国卖5美元，那么"合理"汇率应该是4。实际汇率偏离PPP的程度可以用来判断货币是高估还是低估。《经济学人》杂志的"巨无霸指数"就是基于这个原理。

利率平价（IRP）：认为两国的利差应该等于远期汇率和即期汇率的差异。如果美国利率5%，中国利率2%，那么美元远期应该贴水（贬值）大约3%。否则就存在套利机会（即"套息交易"）。

对冲 vs 不对冲：对国际投资者来说，是否对冲汇率风险是一个关键决策：

  • 对冲（Hedged）：使用远期外汇合约锁定汇率，消除汇率波动的影响。适合短期投资者或对汇率没有明确观点的投资者。
  • 不对冲（Unhedged）：暴露在汇率风险中，享受潜在的汇率收益（也承担损失）。适合长期投资者——PPP理论暗示长期汇率会回归均值。
业界实践：大多数全球资产管理公司（如Vanguard、BlackRock）对发达市场的股票投资通常不进行汇率对冲，因为长期来看汇率波动会相互抵消。但对债券投资则几乎一定会对冲，因为债券的本地波动本身就小于汇率波动，不对冲的话"债券投资变成了汇率投机"。

21.3 国家风险

国家风险（Country Risk）是国际投资区别于国内投资的核心维度。它包括：

主权评级：标普、穆迪、惠誉三大评级机构对各国政府信用的评级直接影响了该国所有资产的风险溢价。一个主权评级下调的国家，其股市和债市通常会同步下跌。

政治风险：包括政权更迭、政策突变、地缘冲突等。2022年俄罗斯入侵乌克兰后，俄罗斯股市暴跌，卢布贬值超过50%，西方国家对俄罗斯实施金融制裁，俄罗斯资产几乎变得不可交易。这是一个极端但真实的案例。

法律风险：不同国家的法律体系对投资者保护程度不同。普通法系国家（美、英、香港）通常对少数股东保护更好；民法法系国家保护较弱。

征收风险（Expropriation Risk）：政府直接没收外国投资者的资产。虽然20世纪70年代后大规模的征收已很少见，但"隐性征收"——如突然增加税收、取消牌照、强制国有化——仍然存在。委内瑞拉和阿根廷是近年来的典型案例。

21.4 国际分散化收益

国际分散化的核心逻辑是：不同国家的股票市场并非完全同步波动。当一个市场下跌时，另一个市场可能上涨或持平，从而降低组合的整体波动。

跨国相关性：全球金融一体化使得各国股市的相关性在过去30年显著上升。在危机期间，相关性会进一步飙升——几乎全球所有市场都会同步下跌（如2008年、2020年）。这意味着危机时期的"分散化"效果会大打折扣。这被称为"危机相关性"或"传染效应"。

新兴市场的独立性：尽管相关性在上升，新兴市场（中国、印度、巴西等）与发达市场的相关性仍然显著低于发达国家之间的相关性。中国A股与美股的相关性大约在0.3-0.4左右——这意味着将A股纳入全球组合仍能提供显著的分散化收益。

数据说话：根据MSCI的长期研究，一个纯美股组合的年化波动率约为15-16%。加入20%的EAFE（欧洲、澳洲、远东指数）后，波动率降至13-14%。再加入10%的新兴市场，波动率进一步降至12-13%。但收益并未同步降低——这就是所谓的"免费午餐"效应。不过，随着全球一体化加深，这顿"免费午餐"越来越小。

21.5 中国A股国际化

中国A股进入全球投资者的视野，是过去十年最重要的资本市场事件之一。

沪港通/深港通（2014/2016年）：这是中国资本市场开放里程碑式的制度安排。香港和海外投资者可以通过香港交易所直接买卖A股（北向交易），内地投资者也可以通过内地券商直接买卖港股（南向交易）。沪港通和深港通已成为外资进入A股的主要渠道。

MSCI纳入（2018年开始）：2018年6月，MSCI首次将A股纳入其新兴市场指数，初始纳入因子为5%。此后逐步提高至20%。2024年，MSCI A股纳入因子维持在20%。富时罗素和标普道琼斯也相继纳入A股。据估算，纳入MSCI为A股带来了超过500亿美元的被动资金流入。

QFII/RQFII：合格境外机构投资者（QFII）和人民币合格境外机构投资者（RQFII）制度是外资进入中国市场的早期渠道。2020年，中国取消了QFII/RQFII的投资额度限制，进一步降低了外资进入门槛。

投资者视角：对中国内地投资者来说，"国际化"意味着两个方向——"引进来"（外资流入A股改变了定价生态）和"走出去"（通过QDII基金、港股通投资海外）。截至2025年，QDII基金规模已超过6000亿元人民币，覆盖美股、欧股、日股、印度股等多个市场。对普通投资者来说，QDII基金是最便捷的全球配置工具。

21.6 全球配置实践

不同市场的风险收益特征差异显著，理解这些差异是构建全球组合的前提。

美股：全球最大、最成熟、流动性最好的股市。标普500指数长期年化回报约10%（1926-2025年）。以科技股为主导，Magnificent Seven（苹果、微软、英伟达、谷歌、亚马逊、Meta、特斯拉）占比超过25%。2023-2025年的AI浪潮主要由美股带动。

欧股：欧洲股市以金融、奢侈品、工业为主导。代表性指数包括Euro Stoxx 50、德国DAX、法国CAC 40。价值股占比高，分红率普遍优于美股。欧洲市场受欧元区政治和货币政策影响较大。

日股：2023-2025年，日股走出了一轮历史性牛市。日经225指数在2024年突破40000点，超过1989年的历史高点。驱动因素包括：东京证券交易所的公司治理改革、巴菲特增持日本商社股、日元贬值利好出口企业。

新兴市场：印度是当前最受关注的新兴市场——GDP增速超过6%，人口结构年轻，莫迪政府的"印度制造"政策推动制造业发展。巴西和墨西哥受益于"近岸外包"（Nearshoring）趋势。但新兴市场普遍具有高波动、高政治风险、弱法治环境的共性。

全球组合参考配置：对一位中国普通投资者，一个"起步级"的全球配置可以是：40% A股/港股 + 30% 美股（通过QDII）+ 15% 日股/欧股 + 10% 新兴市场 + 5% 全球债券。这个组合在中国的QDII框架下完全可以实现。当然，具体的配置比例要根据你的风险承受能力和投资目标来决定。

国际投资不仅仅是"多买几个国家的股票"这么简单。它要求你理解汇率、政治、法律、税收等多重维度的复杂性。但回报也是丰厚的——你不再把所有希望寄托在单一国家的经济命运上，而是分享全球创新的红利。

下一章，我们将探索传统股票债券之外的另类投资世界——私募股权、房地产、大宗商品、艺术品、甚至加密货币。这些资产类别正在重塑现代投资组合的边界。
