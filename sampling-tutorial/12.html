第12章 样本量确定

大家好，我是你们的讲师。从事抽样调查工作的人，被问得最多的一个问题就是："我要抽多少样本才够？"

这个问题的答案看似简单，实则复杂。你可能会在网上看到各种"经验法则"：有人说400就够了（因为正态分布在n=30以后就差不多了，400更稳妥），有人说1000是"标准民调样本量"，还有人说什么"10%法则"——抽总体的10%就够了。这些说法不能说完全错误，但它们都忽略了一个关键变量：你要求的精度是多少？

让我问你三个问题：你要估计的是均值还是比例？你能容忍多大的误差？你有多少预算？这三个问题决定了样本量的答案。本章我们就来系统梳理样本量确定的数学工具和实践技巧。

SRS的样本量公式

一切从简单随机抽样开始。样本量计算公式是你必须背下来的基本功。

估计均值时的样本量

SRS下估计总体均值的样本量

假设检验水平 α=0.05（即95%置信度），允许的绝对误差为 d。

不重复抽样（有限总体修正）：
  n = n₀ / (1 + n₀/N)
  其中 n₀ = (z² × σ²) / d²

当 N 很大时，n₀ ≈ n，常用简化公式：
  n₀ = (z² × σ²) / d²

其中 z = 1.96（对应95%置信度），
σ² 是总体方差（通常用样本方差或历史数据估计），
d 是你能接受的最大误差（半宽度）。

例如：你想估计某城市居民平均月收入，希望误差不超过±500元，已知历史数据估算标准差σ≈8000元。则n₀ = (1.96² × 8000²) / 500² ≈ 983。如果该城市人口100万，则n ≈ 983 / (1 + 983/1000000) ≈ 982——有限总体修正几乎没影响。

估计比例时的样本量

SRS下估计总体比例的样本量

  n₀ = (z² × p × q) / d²

其中 p 是预期的比例，q = 1-p。
当 p 未知时，取 p=0.5 使 p×q 最大 = 0.25，
得到最保守的样本量：
  n₀ = (z² × 0.25) / d² = 0.9604 / d²

取 z=1.96，则：
  d=±0.03 → n₀ = 1067
  d=±0.05 → n₀ = 384
  d=±0.01 → n₀ = 9604

这就是为什么那么多民调都取n≈1000左右——在±3%的误差下，最保守估计就是1067个样本。不管总体是100万还是1亿，这个数都差不多。很多新手惊讶于这个事实："什么？中国14亿人，我只要抽1000个就够了？"是的——如果抽样是真正随机的，1000个样本确实足以在±3%的误差内估计全国比例。

但"真正随机"是这里的关键前提：1000个SRS样本确实够用。但现实中，大多数调查用的不是SRS——而是多阶段分层整群设计，设计效应（Deff）通常在2-5之间。所以实际需要的样本量是 1000 × Deff = 2000-5000。而且还要考虑无回答（通常应答率只有30-60%），那么初始样本量还得再翻倍。这就是"理论样本量"和"实际发放样本量"之间的巨大差距。

各抽样方法的样本量公式

不同抽样方法需要不同的样本量公式。下表汇总了主要方法的计算公式：

抽样方法 样本量公式（估计均值） 关键参数
SRS n = z²σ² / d² 总体方差 σ²
分层抽样（比例分配） n ≈ z²σ²_w / d² 层内方差 σ²_w
分层抽样（最优分配） n = (ΣN_hσ_h)² / (N²d²/z² + ΣN_hσ²_h) 各层方差 σ²_h
整群抽样 n_cluster = n_SRS × Deff 群内相关系数 ρ
多阶段抽样 n = n_SRS × Deff 各阶段方差分量

注意一个重要的规律：分层抽样可以"降低"所需样本量（如果层内方差小于总体方差），而整群/多阶段抽样通常"增加"所需样本量（因为设计效应大于1）。所以在选择抽样方法时，你需要权衡操作成本（多阶段更省钱）和样本量需求（分层更节约样本）。

精度与费用的权衡

样本量确定不是纯粹的统计计算——它本质上是一个优化问题。你需要在给定预算下最大化精度，或者在给定精度下最小化费用。

成本函数的一般形式

总成本 C = C₀ + Σ(C₁_i × n_i)

其中 C₀ 是固定成本（设计、培训、管理等），
C₁_i 是第i个阶段的单位成本，
n_i 是第i阶段的样本量。

对于两阶段抽样：
C = C₀ + C₁ × n_PSU + C₂ × Σm_i

约束条件：预算最多 C_max
优化目标：在 C ≤ C_max 下使方差最小，或
在 V ≤ V_target 下使 C 最小。
实用法则：记得上一章我们说过"第一阶段PSU数量决定精度"吗？在成本优化中，这个规律表现为：固定预算下，增加PSU数量并减少每个PSU内的SSU数量，通常优于"少PSU、多SSU"的方案。我自己经手的一个调查中，从每个PSU抽30人改为抽20人，把节省下来的成本用来多抽2个PSU，结果总方差降低了约15%。这就是"广撒网"的经济学依据。

考虑设计效应

如果你使用的是复杂抽样设计（而不再是SRS），那么样本量的计算公式需要乘以设计效应：

考虑设计效应的样本量

n_actual = n_SRS × Deff

其中 Deff（设计效应）反映的是你的抽样设计
相对于SRS的效率损失。

Deff = 1 + (m - 1) × ρ

m = 每群平均样本量
ρ = 群内相关系数（intra-cluster correlation）

例如，整群抽样中 m=30, ρ=0.05:
  Deff = 1 + 29 × 0.05 = 2.45
  这意味着你需要 1067 × 2.45 ≈ 2614 个样本，
  才能达到与SRS下1067个样本相同的精度。

关于设计效应，我们下一章会有更深入的分析。这里你只需要记住：复杂抽样的样本量总是大于SRS（因为Deff > 1，除非你用了非常高效的分层方案）。

还要考虑无回答和不可及的情况：

实际发放样本量

initial_n = n_actual / (r_response × r_eligibility)

r_response = 预计应答率
r_eligibility = 预计合格率（eligible rate）

假设 n_actual = 2500, r_response = 40%, r_eligibility = 95%:
  initial_n = 2500 / (0.40 × 0.95) ≈ 6579

也就是说，你需要向约6600人发出邀请，
才能最终获得2500份有效问卷。

真实案例：Deff≈5的全国调查

让我分享一个我亲身参与过的案例。这是一个全国性的农村金融可及性调查，目标是在95%置信度下，估计农村居民获得正规金融服务的比例，误差控制在±3%以内。

第一步：计算SRS所需样本量。p=50%的保守估计，d=0.03：

n_SRS = (1.96² × 0.5 × 0.5) / 0.03² ≈ 1067

第二步：考虑设计效应。我们的方案是"分层（按省）+ 多阶段（县→乡镇→村）"设计。根据过往同类调查的经验，Deff大约在4-6之间。保守取Deff=5：

n_complex = 1067 × 5 = 5335

第三步：考虑无回答和不可及率。在农村地区入户调查，预计应答率约70%，合格率约98%：

initial_n = 5335 / (0.70 × 0.98) ≈ 7776

看起来7800个样本似乎远远超过"400够用"的经验法则。但这就是复杂设计下真实的样本量需求。如果你只按SRS公式算出1067就去执行，最终得到的数据精度只有你预期的大约40%——因为设计效应吞噬了大部分的精度。

这个调查最后的执行结果是：成功访问了6800户，加权后有效样本量（按Deff折算回SRS当量）约为1360，实际误差约±2.7%，略好于预期目标。但这个结果建立在事前对Deff的合理估计之上——如果当初错误地用了SRS公式，最终误差会达到±6%以上，几乎不具备政策参考价值。

一个让人心疼的案例：我曾经帮某高校的一个课题组审阅他们的调查设计方案。他们要做一项全国大学生就业意向调查，样本量定了800人——因为他们看到"全国民调1000人够了"的说法，觉得自己比民调还要简单就定了800。方案中没有任何分层、整群或多阶段设计。我问他："你怎么保证这800人能代表全国所有高校、所有专业、所有年级的学生？" 他愣住了。我建议他采用分层（按学校层次）×多阶段（学校→院系→班级）设计，但预算只有5万元。最终的妥协方案是：在10个城市、30所高校各抽2-3个班，总样本量约2400人，Deff约2.8，有效样本量≈860。虽然不完美，但至少在设计上给出了可以量化的误差范围。

软件辅助

手动计算样本量虽然能帮你建立直觉，但实际工作中建议使用专门的软件或在线工具：

  • R包 pwr：基础样本量计算，支持t检验、比例检验、ANOVA等。
  • R包 samplesize4surveys：专为调查抽样设计的样本量计算，支持SRS、分层、整群等。
  • Stata的 power 命令：功能强大的样本量和功效分析。
  • 在线计算器：如Creative Research Systems、Raosoft等网站提供免费的样本量计算器。

但无论用什么工具，最关键的一步永远是：你输入的参数（σ²、p、d、Deff、r_response）是不是合理的？Garbage in, garbage out——这个原则在样本量确定中同样适用。

本章小结

样本量确定的"三部曲"是：第一，根据SRS公式算出理论所需样本量；第二，乘以设计效应Deff得到复杂设计的实际需要样本量；第三，除以预期应答率和合格率得到最终需要发放的样本量。关键参数（σ²、p、ρ、r_response）的估计需要依靠历史数据、预调查或同类研究的经验。

但Deff究竟怎么计算、不同因素对Deff的影响有多大，这些细节我们还没有展开。事实上，设计效应是连接"样本量规划"和"实际精度"之间的桥梁——理解它就是理解抽样设计效率的关键。下一章，我们就全身心投入设计效应的世界里。
