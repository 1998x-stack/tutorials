第15章 现代投资组合理论

大家好，我是你们的讲师。经历前面14章的学习，你已经学会了看财报、估估值、画K线、用指标。但有一个更重要的问题摆在我们面前：如果你手里不是只有一只股票，而是有一个投资组合，该怎么管理？单独看好某只股票很容易，但如何把多只股票组合在一起，让整体收益更稳定、风险更可控？

这就引出了现代投资组合理论（Modern Portfolio Theory, MPT）。1952年，一个叫哈里·马科维茨（Harry Markowitz）的年轻人发表了《投资组合选择》这篇只有14页的论文，彻底改变了金融世界。此后的CAPM、多因子模型、量化投资，都建立在他的思想基础之上。在这章里，我们一层层拆解这个体系——从分散化的直觉，到有效前沿的数学之美，再到Beta和Alpha的实战运用。

15.1 不要把所有鸡蛋放在一个篮子里

分散化的数学原理

"不要把鸡蛋放在同一个篮子里"——这句话你肯定听过。但为什么分散化能降低风险？两个关键因素：资产数量和资产之间的相关性。

当你持有N只股票时，组合的方差由两部分组成：每只股票自身的方差 + 股票之间的协方差。随着N增大，每只股票自身的贡献被摊薄，但协方差的贡献不会消失。换句话说：分散化可以消除个股特有的非系统性风险，但无法消除所有股票共有的系统性风险。

实证研究表明：当组合从1只股票增加到10只时，波动率大约能降低40%-50%；从10只到20只，再降低约10%；超过30只之后，继续分散的效果就很有限了。A股市场上，持有15-20只来自不同行业的股票，基本就能消除大部分非系统性风险。

但这里有一个关键：你分散的必须是"不同篮子"，而不是"同一个篮子上的不同把手"。 如果你买了10只白酒股，2012年塑化剂事件一来，全部一起暴跌。这时候分散化不降低风险——因为它们高度相关。真正有意义的是跨行业、跨风格的分散。

相关性矩阵的意义

相关性的取值范围是[-1, 1]。相关系数为1意味着两只股票同涨同跌；为-1意味着完全反向；为0意味着完全不相关。在现实中，绝大多数A股之间的相关性在0.3-0.6之间——它们共享同一个宏观经济环境和市场情绪，所以有一定正相关，但又不完全同步。

构建投资组合的核心目标之一，就是找到预期收益高但彼此相关性低的资产。比如股票+债券（A股和国债的相关性长期在0.1-0.2左右），或者A股+美股（相关性约0.3-0.4），都能有效降低组合整体波动。

15.2 马科维茨均值-方差模型

期望收益 vs 波动率

马科维茨把投资组合的选择问题简化为两个维度：预期收益（Expected Return）和波动率（Volatility）。每个投资者都想最大化收益、最小化风险，但这两者之间存在trade-off。均值-方差模型的核心就是：给定一个期望收益率，找到风险最小的组合；或者给定一个风险水平，找到收益最大的组合。

具体来说，组合的预期收益是个股预期收益的加权平均，但组合方差的计算就没这么简单了——它需要用到协方差矩阵：

σ²_p = ΣᵢΣⱼ wᵢwⱼCov(rᵢ, rⱼ)

        其中：
        σ²_p = 组合方差
        wᵢ, wⱼ = 股票i和j的权重
        Cov(rᵢ, rⱼ) = 股票i和j的协方差 = ρᵢⱼσᵢσⱼ

        举例——两只股票的组合方差：
        σ²_p = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρ₁₂σ₁σ₂

        如果σ₁=30%, σ₂=25%, ρ₁₂=0.3, w₁=w₂=0.5：
        σ²_p = 0.25×0.09 + 0.25×0.0625 + 2×0.5×0.5×0.3×0.3×0.25
            = 0.0225 + 0.0156 + 0.0113
            = 0.0494
        σ_p = √0.0494 = 22.2%

        对比20只股票的完整计算需要协方差矩阵，代码实现：
        import numpy as np
        weights = np.array([w₁, w₂, ..., w₂₀])
        cov_matrix = np.cov(returns.T)
        portfolio_variance = weights.T @ cov_matrix @ weights

有效前沿（Efficient Frontier）

通过改变各资产的权重，我们可以得到无数个组合。在"预期收益-波动率"的二维平面上，所有可能的组合构成一个区域。这个区域的左上边界就是有效前沿——在同等风险下收益最高的组合，或者在同等收益下风险最低的组合。

有效前沿上的每一个点都代表一个"最优"组合。理性的投资者应该从有效前沿上选择符合自己风险偏好的组合。保守型选左边的点（低风险低收益），激进型选右边的点（高风险高收益）。

但这里有一个巨大的坑：有效前沿极度依赖输入参数。 只要预期收益率、波动率或相关系数发生微小变化，有效前沿的形态就会大变。而这些参数本身就是估计值——你只能根据历史数据去"猜"未来的收益和风险。这就引出了一个经典问题：Garbage In, Garbage Out。马科维茨模型是理论上的杰作，但在实战中直接使用历史数据来构建有效前沿，效果往往不尽如人意。

⚠️ 实战警示：有效前沿对输入参数极度敏感。收益率估计的微小误差会被优化过程放大，导致"最优组合"在样本外表现远不如预期。一个常见的改良方案是使用"收缩估计"（Shrinkage Estimation）或贝叶斯方法，来减少参数估计误差对组合优化的影响。另一个务实的方法是直接使用等权重组合（1/N）——出人意料的是，在许多实证研究中，简单的1/N组合表现优于复杂的均值-方差优化。

15.3 CAPM：一个因子统治一切？

Beta的含义

马科维茨的理论很美，但实际运用中需要估计太多参数（N只股票就需要估计N个收益率、N个方差和N(N-1)/2个协方差）。1964年，威廉·夏普（William Sharpe）在马科维茨的基础上提出了资本资产定价模型（Capital Asset Pricing Model, CAPM），将问题大大简化。

CAPM的核心思想：任何资产的预期收益，只取决于它对市场组合的敏感度——也就是Beta。

E(rᵢ) = r_f + βᵢ × (E(r_m) - r_f)

        其中：
        r_f = 无风险利率（通常用10年期国债收益率）
        βᵢ = Cov(rᵢ, r_m) / σ²_m，衡量股票i对市场的敏感度
        E(r_m) - r_f = 市场风险溢价

        例子：假设r_f=2.5%, E(r_m)=10%, β=1.2
        E(rᵢ) = 2.5% + 1.2 × (10% - 2.5%) = 2.5% + 9.0% = 11.5%

        如果β=0.8：
        E(rᵢ) = 2.5% + 0.8 × 7.5% = 2.5% + 6.0% = 8.5%

Beta=1表示股票跟市场同涨同跌；Beta>1表示比市场更"激进"（牛市涨更多，熊市跌更多）；Beta<1表示比市场更"稳健"。比如券商股的Beta通常在1.5以上，而公用事业股的Beta在0.5-0.8之间。

SML证券市场线

CAPM的图形表达就是证券市场线（Security Market Line, SML）。横轴是Beta，纵轴是预期收益。SML是一条从无风险利率出发、斜率为市场风险溢价的直线。在CAPM的世界里，所有资产都应该落在这条线上。如果某只股票在SML上方，说明它的预期收益高于其风险对应的水平——也就是"被低估"；反之则"被高估"。

CAPM在A股的实证

CAPM在A股市场表现如何？大量实证研究给出了一个让人尴尬的结论：CAPM在A股的解释力很弱。 具体来说：

  • 低Beta股票的收益率高于CAPM的预测，高Beta股票的收益率低于CAPM的预测——这就是著名的"低Beta异象"。
  • 在A股，用Beta来预测未来收益基本不显著。2020-2025年的数据显示，低Beta组合的年化收益率甚至略高于高Beta组合，而波动率显著更低，夏普比率高出近一倍。
  • 这意味着：CAPM不是完全"错"的，但它遗漏了太多重要的解释变量。Beta一个因子远远不够。

15.4 从CAPM到多因子

Fama-French三因子与五因子

1993年，尤金·法玛（Eugene Fama）和肯尼斯·弗伦奇（Kenneth French）发表了划时代的论文，在CAPM的基础上增加了两个因子：规模因子（SMB, Small Minus Big）和价值因子（HML, High Minus Low）。这就是著名的Fama-French三因子模型。2015年，他们又增加了盈利因子（RMW）和投资因子（CMA），扩展为五因子模型。

模型 因子 提出时间 在A股的解释力
CAPM 市场因子（Market） 1964 弱 — R²约30%-50%
Fama-French 三因子 市场 + 规模（SMB）+ 价值（HML） 1993 较强 — R²约60%-75%
Carhart 四因子 市场 + 规模 + 价值 + 动量（MOM） 1997 中等 — 动量在A股效果弱
Fama-French 五因子 市场 + 规模 + 价值 + 盈利（RMW）+ 投资（CMA） 2015 较强 — R²约65%-80%
中国市场四因子 市场 + 规模 + 价值 + 换手率 2017 适应A股特征，加入换手率因子提升显著

A股规模因子和价值因子的表现

在A股市场，规模因子（小盘股相对于大盘股的超额收益）在2000-2016年期间非常显著——小盘股长期跑赢大盘股。但在2017年之后，随着外资流入加速和注册制改革，龙头白马股的估值得到系统性重估，大盘股开始跑赢小盘股。这说明因子的表现不是永恒的，它会随着市场结构和参与者行为的变化而改变。

价值因子（低市盈率/低市净率股票相对于高估值股票的超额收益）在A股的历史表现也不稳定。2009-2015年价值因子表现一般，2016-2018年价值因子大放异彩（核心资产行情），2019-2023年成长因子又明显跑赢价值。因子的"风水轮流转"是常态。

15.5 Alpha与Beta的分离

Beta是市场给你的，Alpha是你自己赚的

在多因子模型的框架下，一个组合的收益可以被分解为：

组合收益 = 无风险收益 + Σ(β_factor × factor_return) + Alpha

        Alpha = 实际收益 - 预期收益（基于因子暴露）

        如果α > 0：组合跑赢了因子模型的预期——你创造了超额收益
        如果α < 0：组合跑输了——你"亏"掉了本该赚到的收益
        如果α ≈ 0：组合只是靠因子暴露赚钱——你啥也没贡献

这个概念非常重要。Beta是市场给你的，Alpha才是你自己赚的。 如果你买了一只高Beta的股票，牛市里涨得多，那不是你的本事，是市场送给你的。同样，熊市里跌得惨，也不全是你选股的问题——那是Beta在起作用。真正的Alpha，是排除了所有因子暴露之后，剩下的、属于你独特洞察力的那部分收益。

2025年A股量化超额来源分析

截至2025年，A股量化私募的总管理规模已超过2万亿元人民币。这些量化产品在扣除管理费和业绩报酬后，仍然能获得相对沪深300指数约10%-20%的年化超额收益。这些超额来自哪里？

  • 高频因子：利用订单流、资金流向等高频数据捕捉短期定价偏差。这是A股量化超额的主要来源，但随着量化规模扩大，这部分收益正在快速衰减。
  • 另类数据：卫星图像（监测工厂开工率）、电商数据（跟踪消费品销量）、招聘数据（评估公司扩张意愿）等非传统数据来源。2024-2025年，另类数据贡献的Alpha占比显著提升。
  • AI模型：深度学习模型（LSTM、Transformer）捕捉非线性关系。传统线性多因子模型解释不了的复杂模式，AI模型可以学到。
  • 执行优化：算法交易减少冲击成本，这部分虽然不算严格意义上的Alpha，但对实际收益的贡献不可忽视。

核心框架：投资的终极能力，就是找到真正属于自己的Alpha来源。无论你是通过深研财报发现被低估的股票，还是通过技术指标捕捉趋势信号，抑或是用量化模型发掘定价规律——只要你能稳定地获得正的Alpha，你就是一个成功的投资者。Beta可以被ETF低成本复制，但Alpha才是真正的稀缺资源。

踩坑故事：过度优化导致的样本外失效

2018年我曾参与过一个量化研究项目，团队里一个研究员用了100多个候选因子，在2013-2017年的A股数据上反复筛选、回测。最终他找到了一个"完美"的10因子模型——在回测期间夏普比率高达3.0，最大回撤不到5%，简直是无敌的圣杯。

但当我们把这个模型部署到实盘交易中，2018年全年表现惨不忍睹——跑输沪深300近15个百分点。问题出在哪里？过度的数据挖掘（Data Snooping）。 100多个因子在5年的数据上筛选，总有一组因子能恰好拟合历史的噪音。那些漂亮的回测数字，本质上是对历史数据的"过度拟合"（Overfitting），而不是真正找到了稳定的规律。

这个教训让整个团队改变了对因子研究的认知。后来我们建立了一套严格的多因子研究流程：所有因子必须通过逻辑检验（先有经济直觉再上数据验证）、样本外检验（保留最近2年数据不碰）、以及跨市场验证（在美股/港股上也跑一遍）。从那以后，我们的模型上线后的表现就稳定多了。记住：如果某个策略的回测好得不像真的，那它大概率就不是真的。

小结与预习

这一章我们从分散化的直觉出发，走过了马科维茨均值-方差模型的数学之美，理解了CAPM和Beta的含义，看到了从单因子到多因子的进化，最后理清了Alpha和Beta的分离——这是投资世界里最重要的一对概念。

到这里你会发现，构建投资组合不仅仅是选股的问题，它更是一个风险收益的权衡与优化的问题。但有了组合之后，怎么管理它的风险？什么时候应该减仓？怎么设置止损？下一章我们就来聊聊风险管理体系——那些在赚钱之前必须先学会的"保命"技能。
