第27章 实战：搭建完整 Coding Agent 🏗️

大家好，我是你们的讲师。欢迎来到这门课程最重要的一章。

过去 26 章，我们做了三件事：第一，理解了 Claude Code 的架构设计；第二，拆解了它的每一个核心模块；第三，在每一章中动手实现了这些模块的最小版本。现在，是时候把所有零件组装起来了。这一章的目标只有一个：让你们手中拥有一个真正完整可用的 AI Coding Agent。

第 3 章我们搭了一个"能跑"的 Agent Loop，但那只是一个骨架。经过二十多章的逐个模块实现，我们现在有了肌肉、血管、神经——Read 工具、Write 工具、Edit 工具、Bash 工具、Todo 规划器、上下文管理器、Sub-Agent 调度器、权限链、MCP 集成、Skills 加载器、日志与追踪系统。今天，我们要把它们缝合在一起，让这个 Agent 真正站起来走路。

如果你是按章节顺序一路走下来的，你手里应该有 10 个左右的独立 Python 模块。如果你跳过了某些实现章节，也不用担心——我会在代码中通过清晰的接口定义和注释来保证你可以跟上。关键是理解集成的逻辑，而不是每一行代码。

27.2 CLI 交互界面实现

Chat 界面是用户和 Agent 之间的第一层接触。我们不追求漂亮的 GUI——对于 Coding Agent，终端界面（Terminal UI）是最自然、最高效的交互形式。使用 Python 标准库的 readline 和 shutil，我们可以实现一个可用的命令行界面，支持命令历史、流式输出显示和基本的 Markdown 渲染：

# CLI 交互界面（内置于 main.py）
import readline
import shutil
import sys
from typing import Callable

def get_terminal_width() -> int:
    return shutil.get_terminal_size().columns

def print_separator(char: str = "="):
    print(char * get_terminal_width())

def stream_output(text: str, end: str = "\n"):
    """流式输出，兼容富文本显示"""
    print(text, end=end, flush=True)

def render_markdown_simple(text: str):
    """简易终端 Markdown 渲染（表格、代码块、粗体）"""
    lines = text.split("\n")
    in_code_block = False
    for line in lines:
        if line.startswith("```"):
            in_code_block = not in_code_block
            if in_code_block:
                print(f"\033[36m{'─' * get_terminal_width()}\033[0m")
            else:
                print(f"\033[36m{'─' * get_terminal_width()}\033[0m")
            continue
        if in_code_block:
            print(f"  {line}")
        elif line.startswith("##"):
            print(f"\033[1;34m{line}\033[0m")
        elif line.startswith("- ") or line.startswith("* "):
            print(f"  \033[33m•\033[0m {line[2:]}")
        elif "|" in line and line.count("|") >= 3:
            cells = [c.strip() for c in line.split("|")[1:-1]]
            print(f"\033[90m{' | '.join(cells)}\033[0m")
        else:
            print(line)

def interactive_cli(task_callback: Callable[[str], None]):
    """交互式 CLI 主循环"""
    print(f"\033[1;32mCoding Agent v1.0\033[0m  |  "
          f"输入任务描述，或输入 /quit 退出")
    print_separator()
    while True:
        try:
            task = input("\n\033[1;36m>>>\033[0m ").strip()
            if not task:
                continue
            if task.lower() in ("/quit", "/exit", "/q"):
                print("Goodbye!")
                break
            task_callback(task)
        except (KeyboardInterrupt, EOFError):
            print("\nGoodbye!")
            break

这个实现虽然简单，但足够实用了。在实际使用中，你还可能想加入进度条（rich.progress 或 tqdm）、彩色 diff 输出、以及并行的流式输出管理。但核心交互哲学是一样的：Agent 的输出应该是可阅读的、可操作的信息，而不是原始的 JSON dump。

27.3 配置文件系统

一个真正可用的 Agent 必须有灵活的配置能力。我们的设计采用四层配置覆盖：

  • 默认值：代码内部硬编码的默认参数
  • 项目配置：YAML 文件（agent_config.yaml），存放在项目根目录
  • CLI 参数：--model、--max-turns 等命令行选项
  • 环境变量：ANTHROPIC_API_KEY、AGENT_MAX_TURNS 等

优先级从低到高。任何层都可以覆盖低层的值。注意：API Key 永远不应出现在配置文件中。始终通过环境变量注入。这样做的好处是配置文件和代码可以安全地提交到 Git，而敏感信息依赖运行时的环境配置。

# agent_config.yaml
model: claude-sonnet-4-20250514
max_turns: 100
context_budget: 180000
tool_timeout: 120

permissions:
  allow_all: false
  allowed_commands:
    - ls
    - cat
    - python
    - pip
    - git
  denied_commands:
    - rm -rf /
    - sudo

mcp_servers:
  - name: filesystem
    command: npx -y @modelcontextprotocol/server-filesystem

skill_dirs:
  - .claude/skills/

log_level: INFO

配置文件在启动时被 main.py 读取并验证。验证的内容包括：模型名是否合法、最大轮次是否为正整数、工具超时是否在合理范围内、MCP 服务器命令是否可以解析。任何验证失败都会在 Agent 开始工作之前报错退出——这样可以避免 Agent 运行到一半才发现配置问题。

27.4 模块集成：从零到完整 Agent

下面是整个集成过程的核心。我们将 10 个模块按依赖顺序依次集成到 CodingAgent 类中。每一步都验证集成是否正常，然后再进行下一步。

Step 1：Agent Loop（第3章）
首先是骨架中的骨架——Agent 主循环。我们直接从第 3 章引入 AgentLoop 类。此时它只有一个空的工具注册表和一个 mock 模型调用。验证方法：传入 "hello world" 任务，确认 Agent 能完成一个完整的 think-act-observe 循环并正确退出。

Step 2：四件套工具 + 注册中心（第6章）
添加 ReadTool、WriteTool、EditTool、BashTool 到 ToolRegistry。验证方法：让 Agent 读取一个已知文件、写一个新文件、编辑已有文件、执行一个简单命令。检查每一步的输出是否和预期一致。

Step 3：Todo 规划器（第9章）
让 TodoPlanner 在 Agent Loop 开始之前创建任务计划。验证方法：输入一个多步骤任务（"创建一个 Flask app，包含两个路由"），检查规划器是否生成了合理的步骤分解。

Step 4：上下文管理器（第11章）
在 Agent Loop 内部集成 ContextManager：每次工具调用后检查上下文 token 计数，当超过阈值时触发压缩策略。验证方法：设置一个很小的上下文预算（比如 5K tokens），输入需要多轮交互的任务，确认压缩机制被正确触发。

Step 5：Sub-Agent 系统（第13章）
添加 SubAgentManager，让主 Agent 在遇到可并行任务时（比如同时修改多个独立的文件）可以生成子 Agent。验证方法：输入一个包含两个不相关子任务的任务，观察是否生成了两个子 Agent 且它们各自完成自己的工作。

Step 6：权限系统（第16章）
在 BashTool 中注入 PermissionChain，确保所有 Bash 执行在运行前经过权限检查。验证方法：在配置中禁止 curl 命令，然后要求 Agent "用 curl 下载一个文件"——确认 Agent 收到权限拒绝通知并且不会执行。

Step 7：MCP 集成（第19章）
添加 MCPIntegration 模块，连接到外部 MCP 服务器。这为 Agent 打开了无限扩展可能——数据库查询、Web 搜索、文件系统操作——全都可以通过 MCP 协议接入。验证方法：连接到一个 MCP filesystem 服务器，让 Agent 列出某个目录中的文件。

Step 8：Skills 系统（第21章）
添加 SkillLoader，让 Agent 可以从 .claude/skills/ 目录加载领域特定的知识。验证方法：在 skills 目录中放置一个包含"本项目使用 Poetry 管理依赖"的 skill 文件，然后问 Agent "如何安装依赖"——确认它正确引用了 Poetry 而不是 pip。

Step 9：日志与可观测性（第23章）
将 Tracer 和 Logger 集成到所有模块的关键路径中。验证方法：运行一个完整任务后，检查 agent.log 文件是否包含了预期的关键事件（工具调用、权限决策、Sub-Agent 生命周期、上下文压缩事件）。

Step 10：配置系统 + CLI
最后封裝 YAML 配置加载、CLI 参数解析、交互式命令行界面。这是让 Agent 从"一个 Python 类"变成"一个可运行的应用"的最后一步。验证方法：运行 python main.py "hello" 确认程序正常启动，然后运行一个简单的任务确认端到端流程完整。

集成测试的最佳实践：每完成一个集成步骤，用一个最小可逆任务进行验证。最小意味着花最少的时间和 token 来确认功能正常；可逆意味着这个任务不会损坏项目——比如创建一个临时文件然后删除它。永远不要在你的生产代码库上第一次测试新集成的 Agent。

27.6 最终代码走读：main.py

下面是我们完整 Agent 的入口文件 main.py。这是所有模块的胶水代码——它不包含复杂的业务逻辑，但它决定了所有模块如何协作。我们逐段来看。

#!/usr/bin/env python3
"""
AI Coding Agent - Main Entry Point
整合了 28 章课程所有模块的完整 Coding Agent

Usage:
    python main.py "your task description"
    python main.py --config my_config.yaml "your task"
    python main.py --model claude-sonnet-4-20250514 "your task"
"""

import os
import sys
import json
import yaml
import logging
import argparse
import readline
import shutil
from typing import Optional, Dict, Any, List

# ---- 模块导入 ----
# 我们将每个模块放在独立的文件中，通过清晰的接口导入
from core.loop import AgentLoop
from core.tools import ToolRegistry
from core.tools.read import ReadTool
from core.tools.write import WriteTool
from core.tools.edit import EditTool
from core.tools.bash import BashTool
from core.planning import TodoPlanner
from core.context import ContextManager
from core.sub_agent import SubAgentManager
from core.permissions import PermissionChain, PermissionRule
from core.mcp import MCPIntegration
from core.skills import SkillLoader
from core.observability import Logger, Tracer


class CodingAgent:
    """完整 AI Coding Agent 主类"""

    def __init__(self, config_path: Optional[str] = None):
        self.config = self._load_config(config_path)
        self._setup_logging()
        self._init_modules()
        self.logger.info("Coding Agent initialized successfully")

    # ── 配置加载（四层覆盖） ──────────────────────────────

    def _load_config(self, config_path: Optional[str] = None) -> Dict:
        """加载配置：默认值 → 项目配置 → 环境变量"""
        config = {
            "model": "claude-sonnet-4-20250514",
            "max_turns": 100,
            "context_budget": 180_000,
            "tool_timeout": 120,
            "permissions": {
                "allow_all": False,
                "allowed_commands": ["ls", "cat", "python", "pip", "git"],
                "denied_commands": ["rm -rf /", "sudo"],
            },
            "mcp_servers": [],
            "skill_dirs": [".claude/skills/"],
            "log_level": "INFO",
        }
        if config_path and os.path.exists(config_path):
            with open(config_path) as f:
                file_config = yaml.safe_load(f) or {}
            self._deep_merge(config, file_config)
        if os.getenv("ANTHROPIC_API_KEY"):
            config["api_key"] = os.getenv("ANTHROPIC_API_KEY")
        if os.getenv("AGENT_MAX_TURNS"):
            config["max_turns"] = int(os.getenv("AGENT_MAX_TURNS"))
        self._validate_config(config)
        return config

    def _validate_config(self, config: Dict):
        """启动时验证配置完整性"""
        assert "api_key" in config, "ANTHROPIC_API_KEY 必须设置"
        assert config["max_turns"] > 0, "max_turns 必须为正数"
        assert config["context_budget"] > 1000, "context_budget 太小"

    def _deep_merge(self, base: Dict, override: Dict):
        for key, value in override.items():
            if key in base and isinstance(base[key], dict) and isinstance(value, dict):
                self._deep_merge(base[key], value)
            else:
                base[key] = value

    # ── 日志设置 ─────────────────────────────────────────

    def _setup_logging(self):
        logging.basicConfig(
            level=getattr(logging, self.config["log_level"].upper()),
            format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
            handlers=[
                logging.FileHandler("agent.log"),
                logging.StreamHandler(sys.stdout),
            ],
        )
        self.logger = logging.getLogger("coding-agent")

    # ── 模块初始化（10 步集成） ─────────────────────────

    def _init_modules(self):
        """初始化所有核心模块"""
        # Step 2: 工具注册中心 + 四件套
        self.tool_registry = ToolRegistry()
        self.tool_registry.register(ReadTool())
        self.tool_registry.register(WriteTool())
        self.tool_registry.register(EditTool())
        self.tool_registry.register(BashTool(
            timeout=self.config["tool_timeout"],
        ))

        # Step 6: 权限链
        self.permissions = PermissionChain()
        if not self.config["permissions"].get("allow_all", False):
            allowed = self.config["permissions"].get("allowed_commands", [])
            denied = self.config["permissions"].get("denied_commands", [])

            def _check_bash(args: Dict) -> bool:
                cmd = args.get("command", "")
                for d in denied:
                    if cmd.startswith(d):
                        return False
                if allowed:
                    return any(cmd.startswith(a) for a in allowed)
                return True

            self.permissions.add_rule(
                PermissionRule("deny", "Bash", _check_bash)
            )

        # Step 4: 上下文管理器
        self.context = ContextManager(
            budget=self.config["context_budget"]
        )

        # Step 3: Todo 规划器
        self.planner = TodoPlanner()

        # Step 5: Sub-Agent 管理器
        self.sub_agents = SubAgentManager(max_parallel=3)

        # Step 7: MCP 集成
        self.mcp = MCPIntegration()
        for sc in self.config.get("mcp_servers", []):
            self.mcp.add_server(sc["name"], sc["command"])
        self.mcp.start_all()

        # Step 8: Skills 加载器
        self.skills = SkillLoader(dirs=self.config["skill_dirs"])

        # Step 9: 可观测性
        self.tracer = Tracer()
        self.logger = Logger()

        # Step 1: Agent 主循环（依赖以上所有模块）
        self.loop = AgentLoop(
            model=self.config["model"],
            api_key=self.config["api_key"],
            tool_registry=self.tool_registry,
            context_manager=self.context,
            planner=self.planner,
            sub_agent_manager=self.sub_agents,
            permission_chain=self.permissions,
            mcp_integration=self.mcp,
            skill_loader=self.skills,
            tracer=self.tracer,
            max_turns=self.config["max_turns"],
        )

    # ── 任务执行 ─────────────────────────────────────────

    def run(self, task: str) -> Dict[str, Any]:
        """执行一个完整任务"""
        self.logger.info(f"Starting task: {task[:120]}...")
        self.tracer.start("task")

        # 规划
        self.logger.info("Phase 1: Planning")
        plan = self.planner.create_plan(task)
        if not plan.steps:
            self.logger.warning("Planner returned no steps")

        # 执行（Agent Loop 自动完成 think-act-observe）
        self.logger.info("Phase 2: Execution")
        result = self.loop.run(task, plan=plan)

        # 完成
        self.logger.info(f"Phase 3: Complete — {result['status']}")
        self.tracer.end("task")

        return {
            "status": result["status"],
            "summary": result.get("summary", ""),
            "turns_used": result.get("turns_used", 0),
            "tokens_used": result.get("tokens_used", 0),
            "sub_agent_calls": result.get("sub_agent_calls", 0),
            "errors": result.get("errors", []),
        }

def parse_args() -> argparse.Namespace:
    parser = argparse.ArgumentParser(
        description="AI Coding Agent — 从零搭建的完整 Coding Agent",
    )
    parser.add_argument("task", nargs="*", help="任务描述")
    parser.add_argument("--config", "-c", default="agent_config.yaml",
                        help="配置文件路径")
    parser.add_argument("--model", "-m", help="指定模型")
    parser.add_argument("--max-turns", type=int, help="最大交互轮次")
    parser.add_argument("--verbose", "-v", action="store_true",
                        help="详细日志输出")
    return parser.parse_args()


def display_report(result: Dict[str, Any]):
    """显示任务执行报告"""
    width = shutil.get_terminal_size().columns
    print("\n" + "=" * width)
    status_icon = "✅" if result["status"] == "success" else "❌"
    print(f"{status_icon} 任务状态: {result['status']}")
    print(f"\U0001f504 交互轮次: {result['turns_used']}")
    print(f"\U0001f4a0 Token 消耗: {result['tokens_used']:,}")
    if result.get("sub_agent_calls"):
        print(f"\U0001f916 Sub-Agent 调用: {result['sub_agent_calls']}")
    if result.get("errors"):
        print(f"⚠️ 错误: {len(result['errors'])}")
        for err in result["errors"][:3]:
            print(f"  • {err}")
    if result.get("summary"):
        print(f"\n\U0001f4dd 摘要: {result['summary']}")
    print("=" * width)


def main():
    args = parse_args()
    task = " ".join(args.task) if args.task else ""

    # 交互模式或单次运行模式
    if not task:
        # 交互模式
        agent = CodingAgent(config_path=args.config)
        print(f"\033[1;32mCoding Agent v1.0\033[0m  |  输入任务或 /quit 退出")
        print("=" * shutil.get_terminal_size().columns)
        while True:
            try:
                task = input("\n\033[1;36m>>>\033[0m ").strip()
                if not task:
                    continue
                if task.lower() in ("/quit", "/exit", "/q"):
                    print("Goodbye!")
                    break
                result = agent.run(task)
                display_report(result)
            except (KeyboardInterrupt, EOFError):
                print("\nGoodbye!")
                break
    else:
        # 单次运行模式
        agent = CodingAgent(config_path=args.config)
        if args.model:
            agent.config["model"] = args.model
        if args.max_turns:
            agent.config["max_turns"] = args.max_turns
        if args.verbose:
            agent.config["log_level"] = "DEBUG"
            logging.getLogger().setLevel(logging.DEBUG)
        result = agent.run(task)
        display_report(result)


if __name__ == "__main__":
    main()
