第10章 货币市场与LM曲线

你有没有注意过一个奇怪的现象：明明我们生活在一个越来越"无现金"的时代——微信支付、支付宝、银行卡——大家对"钱"本身的需求却从未消失。你的工资到账后，你总是留一部分在活期账户里而不是全部存定期或买理财，为什么？因为你随时可能需要用钱。这种"以备不时之需"的心理，就是凯恩斯所说的流动性偏好。

上一章我们研究了产品市场（IS曲线）。现在我们把目光转向货币市场。什么是利率？在凯恩斯看来，利率不是"储蓄的报酬"，而是放弃流动性的报酬——你把钱借给别人、放弃了随时支配现金的权利，所以需要获得利息作为补偿。这一章我们将推导LM曲线，它刻画的是货币市场均衡时利率与收入的关系，与IS曲线一起构成宏观经济学最经典的框架。

1. 凯恩斯的流动性偏好理论：货币需求的三大动机

凯恩斯在《通论》中提出了人们持有货币的三大动机。理解这三个动机，就理解了货币需求的全部逻辑。

交易动机（Transactions Motive）：你需要用钱来买东西。从交房租水电到买早餐，日常交易离不开货币。收入越高，交易量越大，你需要的货币就越多。所以交易性货币需求与收入Y正相关。

预防动机（Precautionary Motive）：你永远不知道明天会发生什么——可能生病、可能失业、可能突然要修车。为了应对这些不确定性，人们会额外持有一部分现金。预防性货币需求也与收入正相关——收入越高，你"防一手"的预算就越充足。

投机动机（Speculative Motive）：这是凯恩斯最独到的贡献。人们持有货币不仅仅是为了交易和预防，还为了等待机会——等债券价格下跌时买进、等利率升高时锁定收益。当利率很低时，债券价格很高，人们预期未来利率会上升（债券价格会下跌），因此宁愿持有现金而不是债券。这就是为什么投机性货币需求与利率负相关。

将三大动机合并，就得到了凯恩斯的货币需求函数：

货币需求：L = L₁(Y) + L₂(r)

简化形式：L = kY - h·r

参数说明：
  L  = 实际货币需求（剔除了物价因素）
  k  = 货币需求的收入敏感度（kY代表交易和预防动机）
  h  = 货币需求的利率敏感度（h·r代表投机动机，利率越低货币需求越大）

均衡条件：货币供给 M/P = 货币需求 L = kY - h·r

其中 M 是名义货币供给，P 是物价水平（在短期分析中通常假设P固定）

2. LM曲线的推导：货币市场均衡到LM

LM曲线的推导逻辑与IS曲线正好对称。IS曲线的逻辑是"利率→投资→产出"，LM曲线的逻辑是"产出→货币需求→利率"。

推导步骤：

第一步，国民收入Y上升，增加了交易性和预防性货币需求（因为买卖更多、不确定性更高）。第二步，在货币供给M/P不变的情况下，货币需求增加推高了利率——人们为了获得更多现金会卖出债券，导致债券价格下跌、利率上升。结果是：更高的产出 → 更高的利率。在利率-产出平面上，这条向上倾斜的曲线就是LM曲线。

LM曲线名称的由来
"LM"代表"Liquidity preference = Money supply"（流动性偏好等于货币供给）。货币市场均衡的条件就是人们愿意持有的货币量（L）恰好等于央行提供的货币量（M/P）。当产出Y上升导致货币需求L增加时，利率r必须上升来"挤出"一部分投机性货币需求，从而使总货币需求重新等于给定的货币供给。

从数学上推导：

货币市场均衡：M/P = kY - h·r

整理出r：h·r = kY - M/P

         r = (k/h)·Y - (1/h)·(M/P)

这就是LM曲线的方程：r = (k/h)·Y - (1/h)·(M/P)

LM曲线的斜率 = k/h
  · k越大（货币需求的收入敏感度高），LM越陡——收入增加导致利率大幅上升
  · h越大（货币需求的利率敏感度高），LM越平缓——收入增加只需利率微调
LM曲线形态 条件 经济含义
陡峭 h小（投机性货币需求对利率不敏感） 产出变动引起利率大幅波动，货币政策效果强
平缓 h大（投机性货币需求对利率高度敏感） 产出变动引起利率微小变动，货币政策效果弱
水平（极端） h → ∞ 流动性陷阱——货币政策完全无效（见下节）
垂直（极端） h → 0 货币需求完全由收入决定——古典情形，货币政策效果最强

3. 流动性陷阱：利率低到"无路可走"

这是凯恩斯理论中最富洞见也最具争议的概念之一。流动性陷阱（Liquidity Trap）是指：当利率已经降到极低水平（接近零）时，所有人都预期利率只会上升不会下降，因此人们愿意持有任何数量的货币，而不愿意去买债券（因为债券价格已经极高，未来大概率会下跌）。此时，LM曲线变成了一条水平线。

在流动性陷阱中，央行增发货币完全无效——因为增发的货币全部被人们持有在手，没有进入实体经济。就像往一个已经满水的杯子里倒水，水只会溢出来。货币政策失去了刺激经济的能力。

踩坑故事：日本零利率的二十年
1990年代初，日本的资产泡沫破裂。房价跌了60%，股市跌了80%，银行坏账堆积如山。日本央行迅速把利率从8%降到0.5%，想刺激经济。到了1995年，隔夜拆借利率已经降到0.5%以下——几乎为零。

但奇怪的事情发生了：不管央行怎么放水，企业就是不借钱，居民就是不消费。大家继续存钱，继续还债，经济继续停滞。日本央行陷入了流动性陷阱。到1999年，日本央行推出了"零利率政策"（ZIRP），但效果甚微。随后二十年，日本经历了"失落的三十年"——零利率、低增长、通缩如影随形。

更值得关注的是21世纪20年代的全球反思。2020年新冠疫情后，欧美主要经济体也把利率降到了零附近，又一次激活了流动性陷阱的讨论。但与日本不同的是，欧美央行配合了大规模财政刺激（直接给居民发钱），结果避免了日本的通缩命运。这个对比说明：流动性陷阱中货币无效，但财政仍然有效——甚至可以说，越是在流动性陷阱中，财政政策越是唯一有效的工具。

4. 从LM到MP曲线：现代中央银行的利率规则

这里我要告诉大家一个重要的"行业秘密"：在当代宏观经济学研究和央行的实际操作中，传统的LM曲线基本已经不再使用了。为什么？因为现代央行不再把货币供给量作为操作目标，而是直接设定利率。

大多数央行（美联储、欧央行、中国人民银行）现在通过设定政策利率（联邦基金利率、MLF利率等）来执行货币政策，而不是通过控制货币供给量M。央行公开宣布"我们将利率定在r*"，然后通过公开市场操作来确保市场利率吻合这个目标。这意味着货币供给变成了"内生"的——它被调整到恰好使市场利率等于目标利率的水平。

这一转变催生了MP曲线（Monetary Policy Curve）：一条水平的利率线，代表央行设定的政策利率。MP曲线取代了向上倾斜的LM曲线，成为现代宏观教科书（如曼昆、布兰查德的最新版本）的标准工具。

为什么还要学LM曲线？
你可能要问：既然LM曲线已经"过时"了，为什么还要花一章来学它？有四个理由：第一，LM曲线帮助你理解"货币市场均衡"这个核心概念，这在任何框架中都不会过时；第二，在分析历史上1970年代以前的经济事件时，LM曲线仍然是最合适的工具——因为当时的央行确实以货币供给量为目标；第三，流动性陷阱的概念只有通过LM曲线才能得到最直观的演示；第四，也是最重要的一点——从IS-LM到IS-MP的转变本身就是极好的"经济学理论如何进化"的案例。

5. 货币政策如何移动LM(MP)

无论你用LM曲线还是MP曲线，货币政策的作用机制是相同的。

扩张性货币政策（央行增发货币/降低政策利率）：

在传统的LM框架中：央行买入国债（公开市场操作）→ 货币供给M增加 → 在每一产出水平上货币供给都超过需求 → 利率下降 → 直到货币市场重新均衡。结果是整条LM曲线向右移动。

在MP框架中更简单：央行直接宣布降低政策利率 → MP曲线下移。

紧缩性货币政策（央行回收货币/提高政策利率）：

反之亦然。央行卖出国债 → 货币供给减少 → 利率上升 → LM曲线向左移动；或者央行直接加息 → MP曲线上移。

政策操作 LM曲线 MP曲线 利率
央行买入国债（扩表） 右移 下移 ↓
央行卖出国债（缩表） 左移 上移 ↑
降低准备金率 右移 下移 ↓
提高政策利率 左移（间接） 上移（直接） ↑
量化宽松（QE） 大规模右移 下移（特殊时期） ↓
思考题：如果你发现央行的资产负债表大幅扩张（货币供给激增），但市场利率并没有显著下降——这说明什么？答案是：经济可能正处于流动性陷阱中。这正是2020年美国的状况——美联储资产从4万亿飙升至9万亿美元，但联邦基金利率纹丝不动地趴在0-0.25%的底部。这个观察也印证了上面日本案例的道理：在利率触及零下限时，常规货币政策工具已经失效。

这一章，我们完成了拼图的另一块。你不仅学会了货币需求三大动机、LM曲线的推导和移动，还了解了流动性陷阱这个极端但重要的情形，以及现代央行实践中从LM到MP的范式转变。下一章，我们将把IS曲线和LM曲线放在同一个坐标系中——这就是宏观经济学中最著名的IS-LM模型。当产品市场和货币市场同时达到均衡时，我们就能全面分析财政政策和货币政策的完整传导路径了。
