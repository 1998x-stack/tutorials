第21章 双重抽样与捕获-再捕获

大家好，我是你们的讲师。有时在抽样中你会遇到一个很实际的困境：你想收集的详细信息（如生物样本、深度访谈）成本极高，但基本信息（如人口学变量、联系信息）却很容易获取。你该怎么办？一个聪明的策略是先抽一个大样本收集廉价信息，再从中抽一个子样本收集昂贵信息。这就是双重抽样（Two-phase Sampling / Double Sampling）的核心思想。

本章还要介绍另一个与"两遍"思路相关但目标完全不同的方法——捕获-再捕获（Capture-Recapture）。这个方法的起源很有意思：它最初是生态学家用来估计野生动物种群数量的。生物学家在一片湖中撒网捕获一批鱼，标记后放回，过一段时间再撒一次网，通过两次捕获的重叠数量来估计湖中鱼的总数。后来社会科学家发现：这个方法也可以用来估计"难以计数"的人类群体——比如某个城市的吸毒人数、无家可归者数量、或者漏报的犯罪案件数。一个来自生态学的工具，竟然成了社会调查中的利器。

一个巧妙的连接：双重抽样和捕获-再捕获虽然出发点不同，但共享同一个核心思想——利用两轮（或更多轮）的数据收集来提高效率或解决枚举问题。双重抽样关注的是"如何用两轮数据收集来节省成本"，而捕获-再捕获关注的是"如何用两轮捕获来估算总体大小"。两种方法都在现实调查中有着广泛的应用。

两相抽样（Two-phase Sampling）

两相抽样（也常称为双重抽样）的流程非常直观：

  • 第一相（Phase I）：从总体中抽取一个大样本，只收集容易获取的信息。这些信息可能包括人口学变量、分层变量、联系方式等。这个样本一般规模很大，但成本很低。
  • 第二相（Phase II）：从第一相样本中抽取一个子样本，收集详细的、昂贵的信息。子样本通常规模小得多，但每份数据的价值极高。

为什么两相抽样比一次性的"中等规模、中等信息量"抽样更有效？因为第一相的大样本可以用极低成本提供精确的分层信息，让第二相的子样本可以设计得更高效（比如在层内最优分配）。

两相抽样的典型应用

两相抽样的典型应用场景

        场景1：分层抽样中的两相设计
        Phase I: 抽取 n'=10000 人，询问年龄、性别、地区
        → 用这些信息将样本划分到各层
        Phase II: 从各层中按最优分配抽取 n=500 人，做详细面访
        → 相比直接分层设计（需要完整的总体分层信息），
        两相设计可以在总体分层信息不可得时使用

        场景2：昂贵检测
        Phase I: 抽取 n'=5000 人，做基础健康问卷
        Phase II: 从子样本中抽取 n=200 人，做基因测序
        → 用Phase I的数据来修正Phase II的选择偏差

两相抽样的估计

两相抽样下的总体均值估计需要考虑两层的抽样权重：

两相抽样下的Horvitz-Thompson估计

        w_i = w_1i × w_2i

        其中：
        w_1i = 第一相入选概率的倒数
        w_2i = 第二相入选概率的倒数（条件于第一相已被选中）

        最终权重 w_i 是两阶段权重的乘积。

        例如：如果第一相从100000人中抽了10000人（w_1=10），
        第二相从10000人中抽了500人（w_2=20），
        则最终权重 w = 10 × 20 = 200
特征 两相抽样 多阶段抽样
核心目的 先用廉价信息筛选/分层，再抽取详细样本 适应地理分散、缺乏完整抽样框
抽样单元 同一层级的个体（两相都在个体级） 逐级降维（县→街道→家庭）
抽样框需求 Phase I需要完整抽样框或大规模筛查 每个阶段只需要当前层级的名录
效率提升机制 利用Phase I的信息优化Phase II的设计 降低实地成本（集中访问）

捕获-再捕获（Capture-Recapture）

捕获-再捕获方法（也常被称为"标记-再捕获"，mark-recapture）要回答的问题完全不同：总体到底有多大？ 当你面对一个无法直接枚举的总体（比如一个湖里的鱼、一座城市中的流浪人口）时，你无法构造抽样框，当然也无法进行传统的概率抽样。捕获-再捕获方法给出了一个精巧的解决方案。

Lincoln-Petersen估计

这是捕获-再捕获中最经典的估计方法，适用于两次捕获的场景：

Lincoln-Petersen估计

        第一次捕获：捕获并标记 M 个个体，然后放回
        第二次捕获：捕获 n 个个体，其中 m 个是带标记的（即重捕）

        假设：
        - 两次捕获之间总体是封闭的（没有新增或死亡）
        - 所有个体被捕获的概率相等
        - 标记不会影响第二次被捕获的概率

        总体规模的估计：
        N̂ = (M × n) / m

        直觉：如果标记个体在总体中占 M/N 的比例，
        那么在第二次捕获中，标记个体应该占同样的比例：
        m / n ≈ M / N  →  N ≈ M × n / m

        示例：M = 100（第一次标记了100条鱼）
        n = 80（第二次捕了80条）
        m = 16（其中有16条带有标记）
        N̂ = 100 × 80 / 16 = 500 条鱼

        方差的近似估计：
        Var(N̂) ≈ M × n × (M - m) × (n - m) / m³
Lincoln-Petersen的"生日问题"类比：估计总体大小有点像"生日配对"问题。第一次捕获你可以理解为给个体发"身份证"；第二次捕获你看有多少个体既有"身份证"又被再次抓住。如果第二次抓获的个体中有很高比例带有标记（比如50%），说明总体不大（因为你标记的个体在总体中占比高）；反之，如果第二次抓获中只有极少数带有标记（比如1%），说明总体非常大。这个比例决定了你能估计的总体规模。

捕-再捕获的关键假设

Lincoln-Petersen估计的质量高度依赖于几个关键假设是否成立：

  • 封闭总体（Closed Population）：在两次捕获之间，总体没有发生数量变化（没有出生、死亡、迁入、迁出）。这个假设在野生动物研究中很难完全满足，但在短时间窗口的人类调查中相对可控。
  • 等可捕获性（Equal Catchability）：每个个体在每次捕获中有相等的概率被捕获。如果一些鱼更"贪吃"（更容易上钩），它们被两次捕获的概率都更高，会导致估计偏低。
  • 标记不影响再捕获：标记本身不会改变个体的行为。如果一条鱼因为被标记而变得更谨慎从而难以上钩，第二次捕获的标记比例就会偏低，导致总体规模被高估。
一个关于等可捕获性的真实教训：在野生动物生态学中有一个著名的案例——生物学家用一种固定网眼尺寸的渔网来估计某湖泊的鱼群数量。结果发现估计值严重偏高。后来调查发现：这种渔网的网眼大小固定，导致小鱼可以穿过而不被捕获。也就是说，小鱼被捕获的概率远低于大鱼——违反了等可捕获性假设。修正方法是用多种网眼尺寸或改用其他捕获方式。在人类调查中也有类似的"陷阱"：比如用社交媒体广告来"捕获"吸毒人群，有些人可能根本不使用社交媒体，导致覆盖不全。

捕获-再捕获在人类群体估计中的应用

这项技术从生态学"跨界"到社会科学后，主要应用于以下场景：

漏报人口估计

人口普查后，各国统计机构都会评估普查的覆盖程度——有多少人被漏报了？方法是进行"普查后调查"（Post-Enumeration Survey, PES），这本质上是一个大规模的捕获-再捕获项目。第一次"捕获"是人口普查本身；第二次"捕获"是一个独立的、高质量的样本调查。通过匹配两份数据中的重叠个体，就可以估计普查的遗漏率和总人口。

隐秘群体的规模估计

这是最让人拍案叫绝的应用之一。比如要估计某城市中吸毒者的总数：

  • 第一次捕获：从戒毒所的登记名单中获得M个吸毒者
  • 第二次捕获：从警方抓捕记录中获得n个吸毒者
  • 重叠部分m：既在戒毒所登记又在警方记录中的吸毒者

用Lincoln-Petersen估计就可以估算全市吸毒者总数。当然，前提是两个来源对"吸毒者"的定义一致，且两个来源的覆盖有一定独立性。

# Python：Lincoln-Petersen估计示例
        import numpy as np

        # 场景：估计某城市吸毒人数
        M = 1200   # 戒毒所登记的吸毒人数（第一次捕获）
        n = 800    # 警方记录的吸毒人数（第二次捕获）
        m = 150    # 同时在两个名单中的人数（重叠）

        # Lincoln-Petersen估计
        N_hat = (M * n) / m
        print(f"估计吸毒人数: {N_hat:.0f}")

        # 方差估计（Seber公式）
        var_N = M * n * (M - m) * (n - m) / (m**3)
        se_N = np.sqrt(var_N)
        print(f"标准误: {se_N:.0f}")
        print(f"95%置信区间: [{N_hat - 1.96*se_N:.0f}, {N_hat + 1.96*se_N:.0f}]")

        # 输出示例：
        # 估计吸毒人数: 6400
        # 标准误: 462
        # 95%置信区间: [5494, 7306]

一个生动故事：从鱼群到街头毒贩

20世纪50年代，生物学家在一座大型湖泊中使用捕获-再捕获方法估计鱼群数量。他们第一次捕获了500条鱼，做好标记后放回；一周后再次捕获了400条鱼，其中80条带标记。于是估计湖中大约有500×400/80 = 2500条鱼。这个结果与后来用排水法实测的结果比较接近，验证了方法的有效性。

几十年后，社会科学家将这个思路应用到了社会调查中。在上述城市吸毒人数的案例中，他们面临的挑战比生物学家多得多：两个"捕获"来源（戒毒所和警方）并不是独立的——一个人被戒毒所登记后，可能会被警方特别关注（违反独立性假设）；吸毒者的流动性很高（违反了封闭总体假设）；而且两个数据源对"吸毒者"的定义可能不完全一致。因此，使用捕获-再捕获方法估计隐秘群体规模时，通常需要更复杂的模型（如多源捕获-再捕获、Log-linear模型）来放松经典假设。但即使存在这些复杂因素，捕获-再捕获仍然是估计隐秘群体规模时最主流的方法之一。

从生态学到社会科学的跨界启示：捕获-再捕获的故事告诉我们，好的方法论是可以跨领域迁移的。生物学家研究鱼群的思路，经过适当的调整和改进，可以用来研究人类社会中难以触及的群体。这启发我们在遇到棘手的抽样问题时，不妨看看其他学科是怎么解决的——或许答案不在统计学教科书里，而在生态学、流行病学或工程学的方法库中。

本章小结

本章介绍了两个"两轮"方法。两相抽样解决的是"成本-精度"权衡问题——先用廉价的大样本收集辅助信息，再用小样本收集精细数据，是大型调查中极为实用的成本节约策略。捕获-再捕获解决的是"总体规模估计"问题——通过两次"捕获"的重叠比例来推断总体大小，是从野生动物保护到隐秘群体估计都广泛使用的经典方法。两者的共同哲学是：用两轮数据交换信息，来突破一轮数据无法解决的问题。

至此，我们讨论的所有方法都假设调查者主动设计了某种抽样方案。但现实中有大量数据并非来自概率抽样——社交媒体数据、网络问卷、便利样本等——如果硬要用传统方法分析，结果很可能是有偏的。下一章我们将讨论一个非常现实的问题：非概率抽样的校正（Non-probability Sampling Calibration）——如何让"不太干净"的数据变得"可用"。
