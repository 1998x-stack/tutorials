开场白：给矩阵一把"尺子"

大家好，我是你们的讲师。前几章我们讨论了线性空间的结构和特殊矩阵的"性格"——这些都是定性的分析。但从今天开始，我们要引入定量的工具：范数（norm）和内积（inner product）。

想象一下你站在一片广阔的土地上。如果你没有测量工具，你只能说"这里很大"、"那里有点远"。但如果你有了一把尺子，就可以说"距离原点 100 米"、"面积 500 平方米"。范数和内积就是矩阵分析中的"尺子"——它们让我们能精确地度量向量和矩阵的大小、方向、角度和距离。

为什么这很重要？因为精确的度量是误差分析和收敛性证明的基础。你说"这个算法收敛了"——收敛的速度多快？误差多大？不引入范数，你就无法回答这些问题。而不引入内积，你也无法定义"夹角"和"正交"——而正交性恰恰是 Fourier 分析、最小二乘和 SVD 的基础。

1. 向量范数

向量空间 $V$ 上的范数是一个函数 $\|\cdot\|: V \to \mathbb{R}_{\ge 0}$，满足三条公理：

  1. 正定性： $\|\mathbf{x}\| \ge 0$，且 $\|\mathbf{x}\| = 0 \iff \mathbf{x} = \mathbf{0}$
  2. 齐次性： $\|\alpha\mathbf{x}\| = |\alpha| \|\mathbf{x}\|$
  3. 三角不等式： $\|\mathbf{x} + \mathbf{y}\| \le \|\mathbf{x}\| + \|\mathbf{y}\|$

最常见的是 $\ell_p$ 范数族：

$$ \|\mathbf{x}\|_p = \left( \sum_{i=1}^n |x_i|^p \right)^{1/p}, \quad p \ge 1 $$

其中几个特殊的情形尤为重要：

范数 公式 几何意义 常见用途
$\ell_1$ 范数 $\|\mathbf{x}\|_1 = \sum_i |x_i|$ Manhattan 距离（菱形球） 稀疏正则化（Lasso）
$\ell_2$ 范数 $\|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}$ Euclidean 距离（球形） 最小二乘、PCA
$\ell_\infty$ 范数 $\|\mathbf{x}\|_\infty = \max_i |x_i|$ Chebyshev 距离（立方体） 一致收敛、对抗鲁棒性
$\ell_p$ 范数 $(\sum_i |x_i|^p)^{1/p}$ $p$-球体 弹性网（elastic net）

从几何上看，$\|\mathbf{x}\|_p = 1$ 确定的"单位球"形状随 $p$ 的变化而变化：$p=1$ 是菱形（钻石形），$p=2$ 是圆形，$p \to \infty$ 趋向于正方形。这种形状差异直接影响了 $\ell_1$ 和 $\ell_2$ 正则化在机器学习中的行为差异——$\ell_1$ 的"尖角"更容易产生稀疏解。

2. 矩阵范数

矩阵既然也是向量空间中的元素，自然也可以有范数。但矩阵范数多了一层约束——它们需要与矩阵乘法相容（submultiplicative）：

$$ \|AB\| \le \|A\| \|B\| $$

这个性质在很多证明中不可或缺。下面介绍三种最重要的矩阵范数。

Frobenius 范数

Frobenius 范数（也叫 Euclidean 范数、Schur 范数、Hilbert-Schmidt 范数）是把矩阵当成一个长向量来对待：

$$ \|A\|_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^n |a_{ij}|^2} = \sqrt{\operatorname{tr}(A^* A)} $$

它是所有奇异值的平方和再开方：$\|A\|_F = \sqrt{\sum_{i=1}^r \sigma_i^2}$，其中 $r = \operatorname{rank}(A)$，$\sigma_i$ 是奇异值。Frobenius 范数满足相容性：$\|AB\|_F \le \|A\|_F \|B\|_F$。

谱范数

谱范数（spectral norm）是矩阵作为线性算子从 $\ell_2$ 到 $\ell_2$ 的诱导范数（induced norm）：

$$ \|A\|_2 = \max_{\|\mathbf{x}\|_2 = 1} \|A\mathbf{x}\|_2 = \sigma_{\max}(A) $$

谱范数等于矩阵的最大奇异值。它衡量的是矩阵能最大放大一个单位向量的倍数。谱范数在分析数值稳定性时至关重要——条件数 $\kappa(A) = \|A\|_2 \|A^{-1}\|_2 = \sigma_{\max} / \sigma_{\min}$ 正是用谱范数定义的。

其他诱导范数

一般来说，对于向量 $p$-范数可以诱导出矩阵范数：

$$ \|A\|_p = \max_{\|\mathbf{x}\|_p = 1} \|A\mathbf{x}\|_p $$

三个常用的特例：

  • $\|A\|_1 = \max_{1 \le j \le n} \sum_{i=1}^m |a_{ij}|$（最大列和）
  • $\|A\|_\infty = \max_{1 \le i \le m} \sum_{j=1}^n |a_{ij}|$（最大行和）
  • $\|A\|_2 = \sigma_{\max}(A)$（最大奇异值）

来看一个用 Python 验证范数性质的例子：

import numpy as np

A = np.array([[3, 1], [1, 2]])
B = np.array([[0, 1], [1, 0]])

# Frobenius 范数
fro_A = np.linalg.norm(A, 'fro')
fro_B = np.linalg.norm(B, 'fro')
fro_AB = np.linalg.norm(A @ B, 'fro')
print(f"Frobenius: ||A||_F = {fro_A:.4f}")
print(f"相容性: ||AB||_F = {fro_AB:.4f} ≤ {fro_A * fro_B:.4f}")

# 谱范数
spec_A = np.linalg.norm(A, 2)
spec_B = np.linalg.norm(B, 2)
spec_AB = np.linalg.norm(A @ B, 2)
print(f"\n谱范数: ||A||_2 = {spec_A:.4f} (最大奇异值)")
print(f"奇异值: {np.linalg.svd(A)[1]}")
print(f"相容性: ||AB||_2 = {spec_AB:.4f} ≤ {spec_A * spec_B:.4f}")

# 条件数
cond_A = np.linalg.cond(A)
print(f"\n条件数 κ(A) = {cond_A:.4f}")
print(f"谱范数验证: ||A||_2 * ||A^{-1}||_2 = {spec_A * np.linalg.norm(np.linalg.inv(A), 2):.4f}")
关键洞察： 选哪个范数取决于你的问题。在误差分析中谱范数最自然（因为它直接对应"最大放大倍数"），在计算中 Frobenius 范数最容易求导（因为它是平方和的根号）。两者的区别决定了算法设计和理论分析的走向。

3. 内积空间

内积（inner product）是 $\ell_2$ 范数的一般化。向量空间 $V$ 上的内积是一个函数 $\langle\cdot,\cdot\rangle: V \times V \to \mathbb{F}$，满足：

  1. 共轭对称性： $\langle \mathbf{x}, \mathbf{y} \rangle = \overline{\langle \mathbf{y}, \mathbf{x} \rangle}$
  2. 对第一变元线性： $\langle \alpha\mathbf{x} + \beta\mathbf{y}, \mathbf{z} \rangle = \alpha\langle \mathbf{x},\mathbf{z} \rangle + \beta\langle \mathbf{y},\mathbf{z} \rangle$
  3. 正定性： $\langle \mathbf{x}, \mathbf{x} \rangle \ge 0$，且 $\langle \mathbf{x}, \mathbf{x} \rangle = 0 \iff \mathbf{x} = \mathbf{0}$

带有内积的向量空间叫内积空间（inner product space）。在 $\mathbb{R}^n$ 中，标准内积是 $\langle \mathbf{x}, \mathbf{y} \rangle = \mathbf{x}^T \mathbf{y} = \sum_i x_i y_i$。在 $\mathbb{C}^n$ 中，标准内积是 $\langle \mathbf{x}, \mathbf{y} \rangle = \mathbf{x}^* \mathbf{y}$。

内积自动导出范数：$\|\mathbf{x}\| = \sqrt{\langle \mathbf{x}, \mathbf{x} \rangle}$。但不是所有范数都来自内积——比如 $\ell_1$ 和 $\ell_\infty$ 范数就不满足平行四边形法则。

Cauchy-Schwarz 不等式

这是数学中最重要的不等式之一：

Cauchy-Schwarz 不等式： $$ |\langle \mathbf{x}, \mathbf{y} \rangle| \le \|\mathbf{x}\| \, \|\mathbf{y}\| $$ 等号成立当且仅当 $\mathbf{x}$ 与 $\mathbf{y}$ 线性相关。

它告诉我们：两个向量的内积的绝对值，不会超过各自长度的乘积。这个不等式在几乎所有涉及内积的领域中出现——从概率论（$\operatorname{Cov}(X,Y) \le \sqrt{\operatorname{Var}(X)\operatorname{Var}(Y)}$）到量子力学（不确定关系）到信号处理（相关检测）。

有了 Cauchy-Schwarz 不等式，我们可以定义向量之间的夹角：

$$ \cos\theta = \frac{\langle \mathbf{x}, \mathbf{y} \rangle}{\|\mathbf{x}\| \|\mathbf{y}\|} $$

当 $\langle \mathbf{x}, \mathbf{y} \rangle = 0$ 时，我们说 $\mathbf{x}$ 与 $\mathbf{y}$ 正交（orthogonal），记作 $\mathbf{x} \perp \mathbf{y}$。

Gram-Schmidt 正交化

给定一组线性无关的向量 $\{\mathbf{v}_1, \dots, \mathbf{v}_k\}$，Gram-Schmidt 过程可以构造出一组正交基 $\{\mathbf{u}_1, \dots, \mathbf{u}_k\}$：

$$ \mathbf{u}_1 = \mathbf{v}_1 $$ $$ \mathbf{u}_2 = \mathbf{v}_2 - \frac{\langle \mathbf{v}_2, \mathbf{u}_1 \rangle}{\langle \mathbf{u}_1, \mathbf{u}_1 \rangle} \mathbf{u}_1 $$ $$ \mathbf{u}_j = \mathbf{v}_j - \sum_{i=1}^{j-1} \frac{\langle \mathbf{v}_j, \mathbf{u}_i \rangle}{\langle \mathbf{u}_i, \mathbf{u}_i \rangle} \mathbf{u}_i $$

这本质上就是 QR 分解的数学基础——$A = QR$ 中的 $Q$ 就是 Gram-Schmidt 过程得到的正交矩阵。

4. 对偶范数

每个范数 $\|\cdot\|$ 都有一个伴生的对偶范数（dual norm）$\|\cdot\|_*$，定义为：

$$ \|\mathbf{y}\|_* = \max_{\|\mathbf{x}\| \le 1} \langle \mathbf{x}, \mathbf{y} \rangle $$

对偶范数的几何含义是：单位球的"支撑函数"在给定方向上的最大值。一些重要的对偶关系：

范数 对偶范数
$\ell_1$ $\ell_\infty$
$\ell_2$ $\ell_2$（自对偶）
$\ell_p$ ($1 < p < \infty$) $\ell_q$，其中 $1/p + 1/q = 1$

对偶范数的桥梁是 Hölder 不等式：

$$ \sum_{i=1}^n |x_i y_i| \le \|\mathbf{x}\|_p \|\mathbf{y}\|_q, \quad \frac{1}{p} + \frac{1}{q} = 1 $$

当 $p = q = 2$ 时，这就是 Cauchy-Schwarz 不等式。对偶范数在凸优化、正则化理论和机器学习中频繁出现——比如 $\ell_1$ 正则化的对偶问题天然涉及 $\ell_\infty$ 约束。

5. 范数等价性与有限维空间的紧性

在有限维空间中（比如 $\mathbb{R}^n$），一个极其重要的结论是：所有范数都是等价的。这意味着对于 $\mathbb{R}^n$ 上的任意两个范数 $\|\cdot\|_\alpha$ 和 $\|\cdot\|_\beta$，存在常数 $c, C > 0$ 使得：

$$ c \|\mathbf{x}\|_\beta \le \|\mathbf{x}\|_\alpha \le C \|\mathbf{x}\|_\beta, \quad \forall \mathbf{x} \in \mathbb{R}^n $$

比如对于 $\ell_1$ 和 $\ell_2$ 范数：

$$ \|\mathbf{x}\|_2 \le \|\mathbf{x}\|_1 \le \sqrt{n} \|\mathbf{x}\|_2 $$

这个结论的直观理解是：在有限维空间中，任何范数定义的拓扑（即"什么序列收敛"）都完全相同。你不需要担心选错范数会影响"收敛与否"的判断——只影响收敛的速度和常数。

等价性的一个深刻推论是有限维空间中的紧性（compactness）：在有限维空间中，有界闭集是紧的（Heine-Borel 定理）。这意味着任何有界序列都有收敛子序列——这对许多存在性证明至关重要。

踩坑回忆： 读研时我曾尝试把 $\ell_2$ 范数梯度下降的收敛性证明直接"搬"到 $\ell_1$ 正则化问题上，结果发现收敛速度差了好几个数量级。我当时完全困惑——"范数等价性不是说差不多吗？"后来才意识到：范数等价说的是拓扑（收敛与否），不是度量（收敛速度）。等价性保证了"如果梯度 $\to 0$，在两种范数下都会 $\to 0$"，但 $\ell_1$ 和 $\ell_2$ 的常数因子 $\sqrt{n}$ 会显著影响实际收敛界的紧度。在 $n=10^6$ 的深度学习场景中，$\sqrt{n} \approx 1000$——这个因子足以让收敛分析变得毫无意义。这个教训让我深入学习了对偶范数自适应的优化算法，比如 mirror descent。

结语：从定性到定量的飞跃

范数和内积给矩阵分析装上了"度量器"。有了它们，我们可以：精确地讨论"误差有多小"、"近似有多好"、"收敛有多快"。向量范数告诉我们如何衡量向量的大小，矩阵范数（尤其是谱范数和 Frobenius 范数）告诉我们如何衡量线性变换的"强度"和"敏感性"，内积则打开了"正交"和"投影"这扇大门——这正是最小二乘、Fourier 分析和 SVD 的数学基石。

下一章我们将进入矩阵分析的核心主题——特征值与特征向量。有了范数工具，我们不仅能计算特征值，还能定量地分析它们对扰动的敏感性和分布范围——这是从"会求特征值"到"懂特征值"的关键一步。
