第十九章：图论基础（一）

大家好，我是你们的讲师。今天开始我们进入整个组合数学竞赛中最"有画面感"的一个分支——图论。

从这一章开始，我们将连续用四章的篇幅系统学习图论及相关专题。为什么花这么多时间？原因很简单：在CMO和IMO中，图论题的出场率几乎接近100%。每年至少一道，有时甚至两道。而且图论题的特点是非常"两极分化"——理解了就很简单，没思路就完全无从下手。今天我们从最基础的定义和性质开始，把图论的语言体系建立起来。

很多同学觉得图论就是画点连线，像是数学里的"画画"。但我要告诉你——图论的严格性丝毫不亚于代数和分析。一张图虽然可以画出来，但它的定义、性质和定理都是极其精确的。你很快就会看到，仅仅靠"握手引理"这一个结论，就能推出让人意想不到的深刻结果。

19.1 图的基本概念

19.1.1 图的定义

图（Graph）是一个有序对 G = (V, E)，其中 V 是顶点（vertex）的集合，E 是边（edge）的集合。每条边连接两个顶点。如果边有方向，称为有向图（directed graph）；如果边没有方向，称为无向图（undirected graph）。本章如无特别说明，讨论的都是无向图。

几个关键术语：

  • 阶（order）：图的顶点数，记为 |V| 或 n
  • 边数（size）：图的边数，记为 |E| 或 m
  • 相邻（adjacent）：两个顶点之间有边连接
  • 关联（incident）：边和它的端点之间的关系
  • 环（loop）：连接同一个顶点的边（竞赛中几乎从不出现）
  • 重边（multiple edges）：两个顶点之间有多条边

在竞赛组合中，我们绝大多数情况下讨论的是简单图（simple graph）——无环、无重边的无向图。

19.1.2 度（Degree）

顶点 v 的度记为 deg(v)，是与 v 相关联的边的数目。在简单图中，就是与 v 相邻的顶点数。

度这个概念看起来简单，但它是图论中最有力的分析工具之一。为什么？因为下面这个定理：

定理 19.1（握手引理——Handshaking Lemma）

对任意无向图 G = (V, E)，有：

∑_{v∈V} deg(v) = 2|E|

即所有顶点的度数之和等于边数的两倍。

证明：每条边对两个端点的度数各贡献1，因此每条边在度数和中被计数了2次。证毕。

这个引理简单到难以置信，但它的推论非常深刻：

推论 19.2：任意图中，奇度顶点的个数一定是偶数。

证明：设 V奇 为奇度顶点集合，V偶 为偶度顶点集合。由握手引理：

∑_{v∈V} deg(v) = ∑_{v∈V_奇} deg(v) + ∑_{v∈V_偶} deg(v) = 2|E|

右边是偶数，左边∑_{偶}deg(v)是偶数之和，必为偶数。因此∑_{奇}deg(v)也必须为偶数。奇度数的和要为偶数，必须有偶数个奇数项。因此奇度顶点个数必为偶数。

这个推论看起来也很简单，但它在竞赛中有着惊人的威力。来看一个例子：

例1：证明在任何一次握手聚会中，握过奇数次手的人数总是偶数。

把每个人看作一个顶点，两个人之间如果握过手就连接一条边。那么每个人的握手次数就是他的度数。由推论19.2，奇度顶点（即握过奇数次手的人）必为偶数。证毕。

握手引理是图论中最常用的工具之一，几乎每一道图论题都会间接用到它。下面我讲一个真实的故事：

📖 踩坑实录：有一年我带学生参加CMO集训，模拟题中出了一道这样的题："某次会议共有n个人参加，已知每个人至少认识其他一人，且认识奇数个人的人数恰好是奇数。问：这种情形是否可能？"

班上最快交卷的一个同学毫不犹豫地回答："可能！比如n=1，他认识0个人，但要求"每个人至少认识其他一人"，所以n=1不满足条件。那n=3呢？"他画了半天图没画出来。后来他发现——这个问题本身就有矛盾！由推论19.2，奇度顶点必须是偶数个，而"认识奇数个人的人数恰好是奇数"意味着奇度顶点有奇数个，矛盾。所以这种情形不可能存在。

这个故事告诉我们两个教训：第一，图论的推论看似简单，但威力巨大；第二，考场上不要急着动手画图——先想想有没有基本的定理可以否定整个问题。很多时候一道图论题的"解法"就是一两个基本推论的应用。

19.1.3 路径、圈与连通性

几个基本概念的严格定义：

  • 路径（path）：顶点序列 v0, v1, ..., vk，其中每对相邻顶点之间有边，且顶点不重复。路径的长度是边的数目 k。
  • 圈（cycle）：起点和终点相同且中间顶点不重复的路径，长度至少为3（简单图中）。长度为 k 的圈记作 Ck。
  • 连通图（connected graph）：任意两个顶点之间都存在路径。
  • 连通分量（connected component）：极大连通子图。

关于连通性有一个非常重要的极值结论：

定理 19.3：n 个顶点的简单图中，如果边数 m > (n-1)(n-2)/2，则图一定连通。

证明：反证法。假设图不连通，则至少有两个连通分量。设其中一个分量的顶点数为 k（1≤k≤n-1），另一个分量为 n-k。在简单图中，一个 k 个顶点的完全图最多有 C(k,2) 条边，另一个最多有 C(n-k,2) 条边。因此总边数最多为：

C(k,2) + C(n-k,2) = k(k-1)/2 + (n-k)(n-k-1)/2
                    = (k² - k + n² - 2nk + k² - n + k)/2
                    = (2k² - 2nk + n² - n)/2
                    = k² - nk + n(n-1)/2
                    = (k - n/2)² - n²/4 + n(n-1)/2
                    = (k - n/2)² + n(n-2)/4

这个式子在 k=1 或 k=n-1 时取最大值 (n-1)(n-2)/2。因此 m ≤ (n-1)(n-2)/2，与条件矛盾。证毕。

这个结论的实用含义：如果边数足够多，图就必然连通。竞赛中常用来解决"是否存在孤立顶点"这一类问题。

19.2 树（Tree）

树是图论中最重要的特殊图之一。它不仅是竞赛中的常客，也是计算机科学中最基础的数据结构之一。先看它的等价定义：

编号 性质 说明
T1 连通无圈 最基本的定义
T2 任意两顶点之间有唯一路径 树的最重要特征
T3 n 个顶点有 n-1 条边 最常用的计数特征
T4 连通且删去任意一条边就变得不连通 极小的连通图
T5 无圈且添加任意一条边就产生唯一圈 极大的无圈图

这些性质在竞赛中可以互相转化使用。你看到"n个顶点n-1条边"的条件，就要马上想到"这很可能是一棵树"。

19.2.1 树的叶子与度数和

树的叶子（leaf）就是度为1的顶点。关于叶子，有一个非常基本的结论：

定理 19.4：任何至少有2个顶点的树，至少有两个叶子。

证明：设树有 n 个顶点（n≥2），m = n-1 条边。由握手引理，∑deg(v) = 2(n-1) = 2n-2。如果至多有一个叶子（即至多一个顶点度为1），那么其余 n-1 个顶点度数至少为2，则 ∑deg(v) ≥ 1 + 2(n-1) = 2n-1 > 2n-2，矛盾。因此至少有2个叶子。

这个结论在归纳法证明中非常有用——从树上"摘掉"一个叶子，剩下的还是一棵树，然后就可以用归纳假设了。

19.2.2 Cayley公式

Cayley公式是图论和组合计数结合的一个经典结果：

定理 19.5（Cayley公式）

n 个标号顶点可以构成的不同树的数目为 nn-2。

这个结果初看令人震惊——为什么恰好是 nn-2？这个数字是怎么来的？我们来看一个优美的证明思路：

Prüfer序列法：这是最经典的证明方法。n 个标号顶点的树与长度为 n-2 的序列（序列中每个元素取自 {1,2,...,n}）之间存在一一对应。步骤如下：

构造Prüfer序列（从树到序列）：
1. 找到当前编号最小的叶子（度为1的顶点）
2. 记录它的邻居编号
3. 删除这个叶子
4. 重复直到只剩2个顶点

从Prüfer序列还原树：
1. 序列第一个数为 x₁，找到不在序列中的最小数 y₁
2. 连接 x₁ 和 y₁
3. 从序列中删除 x₁，从编号集中删除 y₁
4. 重复操作，最后连接剩余的两个顶点

因为长度为 n-2 的序列中每个位置有 n 种选择，所以共有 nn-2 个可能的序列，从而对应 nn-2 棵不同的树。

Cayley公式是组合计数的"天花板"级别的应用——它不仅把图和序列巧妙地联系了起来，而且在后续"概率方法"等高级专题中也会反复用到。

19.3 二分图（Bipartite Graph）

19.3.1 定义与判定

二分图 G = (X, Y, E)：顶点集可以划分为两个非空子集 X 和 Y，使得每条边的两个端点分别属于 X 和 Y（即所有边都在 X 和 Y 之间）。

关于二分图，最重要的定理是：

定理 19.6（二分图判定定理）

一个简单图是二分图当且仅当它不包含奇圈（长度为奇数的圈）。

证明思路：必要性——如果图是二分图，任何回路必须在X和Y之间来回交替，因此长度必为偶数。充分性——任取一个顶点 v，将与其距离为偶数的顶点归入 X，距离为奇数的归入 Y。如果没有奇圈，这个划分不会产生矛盾（即不存在 X 内部或 Y 内部的边）。

这个定理的用途极为广泛。竞赛中遇到"证明某种结构可以分成两类"的问题，首先就应该想到二分图。

19.3.2 竞赛中的应用

例2：给定一个由 n 个人组成的群体，其中某些人之间互相认识。已知不存在三个人两两互相认识。证明：可以将这些人分成两组，使得每组内部没有人互相认识。

分析：以人为顶点，认识关系为边。条件"不存在三个人两两互相认识"意味着图中没有三角形（3-圈）。但要注意，这并不直接说明图没有奇圈——可能有5-圈、7-圈等。但再仔细想，这个条件实际上是指图中没有长度为3的圈，但并没有排除其他奇圈。

等等——这个条件其实不足以直接得出二分图的结论。正确的思路是：如果图中存在奇圈，取其中一个最短奇圈。如果它的长度大于3，连接该圈上的两个距离为2的顶点（它们由两条路径可达，形成三角形），就会产生一个三角形，与条件矛盾。因此该图无奇圈，由定理19.6可知是二分图。

二分图的判定也是竞赛中的常见考点。来看一个更直接的例子：

例3：在一个国际象棋棋盘上，马从某格出发，能否不重复地走遍所有格子并返回起点？

分析：将棋盘格子按坐标和 (i+j) 的奇偶性染色——所有"黑格"（i+j为奇数）和"白格"（i+j为偶数）。马的走法是从黑格跳到白格或从白格跳到黑格（即马的移动总是改变格子颜色）。因此马在棋盘上的移动路径就是一个二分图。每次移动都要更换颜色。经过偶数步回到原格是可能的，但"走遍所有格子再返回"要求步数为格子总数——棋盘有64格，需要64步，是偶数，所以不矛盾。

但如果换成是5×5棋盘（25格），马要遍历所有格并返回起点就需要25步（奇数），而马每步都改变颜色，走奇数步后颜色必然不同，因此不可能回到起点。所以5×5棋盘不存在马的Hamilton回路。

19.4 图论在竞赛中的定位

在结束本章之前，我想和大家聊一聊图论在CMO和IMO中的定位。

首先，近十年的IMO中，每年至少有一道组合题可以用图论的语言来描述。虽然并不是每道题都需要用图论定理来解，但学会用图论的语言来"翻译"问题，本身就是一个重要的技能。很多看起来是"关系型"的问题——比如认识关系、比赛胜负关系、连接关系——都可以自然地用图来表示。

其次，图论题在竞赛中有一个显著的特点：图论题的"天花板"不依赖于大量的公式记忆，而依赖于思维方式的转换。你可能记住了几十个组合恒等式也不一定能做出一道图论题，但如果你理解了握手引理和二分图判定的本质，很多难题就能迎刃而解。

重要提示：从应考策略的角度，我建议大家对图论题采取这样的态度——先尝试用图论的语言重新描述问题，然后问自己三个问题：

  1. 这个图的度数和有什么性质？（握手引理）
  2. 这个图有奇圈吗？（二分图判定）
  3. 这个图的边数有多"密"？（连通性极值定理）

这三个问题能解决80%以上的竞赛图论基础题。
