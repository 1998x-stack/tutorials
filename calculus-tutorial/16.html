换元法的核心思想：反向链式法则

大家好，我是你们的讲师。

上一章我们学习了不定积分的基本概念和直接积分法。你很快就会发现，直接积分法能解决的积分非常有限。想想看，∫ cos(2x) dx 这样的积分——你的积分表里有 cos(2x) 的公式吗？没有。积分表里只有 ∫ cos x dx = sin x + C。但直觉告诉你，sin(2x) 的导数是什么呢？d/dx [sin(2x)] = 2 cos(2x)。所以 ∫ cos(2x) dx = (1/2) sin(2x) + C。你刚刚无意识地完成了一次换元积分！

这个简单的例子揭示了换元法的本质：链式法则的逆运算。求导时我们用链式法则处理复合函数；积分时，我们把链式法则反过来用，就得到了换元法。如果你能认出被积函数是某个复合函数的导数（乘以一个常数因子），那么你就可以通过变量替换来"撤销"链式法则，让积分简化为基本公式。

换元法分为两大类：第一类换元法（凑微分法）和第二类换元法（主要是三角代换）。两类方法的本质相同——都是做变量替换——但应用场景和操作技巧有显著区别。我们先从最简单的凑微分法讲起。

第一类换元：凑微分法

凑微分法的核心思想是：把被积函数的一部分"塞进"微分号 d 里面，使得整个积分变成 ∫ f(u) du 这种基本形式。

形式上，如果 F'(u) = f(u)，且 u = g(x)，那么：

∫ f(g(x)) · g'(x) dx = ∫ f(u) du = F(u) + C = F(g(x)) + C

关键在于，等号左边你要能认出 g'(x) dx 正好可以写成 du。凑微分法的操作步骤如下：

  1. 观察被积函数，寻找一个"内层函数" u = g(x)，使得 g'(x) 也在被积函数中出现（可能差一个常数因子）
  2. 把 g'(x) dx 凑成 du
  3. 积分变成 ∫ f(u) du，套用基本积分公式
  4. 代回 u = g(x) 得到最终结果

来看几个例子：

例1：∫ 2x e^(x^2) dx

        令 u = x^2，则 du = 2x dx
        原式 = ∫ e^u du = e^u + C = e^(x^2) + C

        例2：∫ cos(3x + 5) dx

        令 u = 3x + 5，则 du = 3 dx，即 dx = du/3
        原式 = ∫ cos u · (du/3) = (1/3) ∫ cos u du
             = (1/3) sin u + C = (1/3) sin(3x + 5) + C

        例3：∫ x / (x^2 + 1) dx

        令 u = x^2 + 1，则 du = 2x dx，即 x dx = du/2
        原式 = ∫ (1/u) · (du/2) = (1/2) ∫ 1/u du
             = (1/2) ln|u| + C = (1/2) ln|x^2 + 1| + C
小技巧：凑微分法的关键是"找出内层函数 u = g(x)"。一个常用策略是：观察被积函数中"复杂"的部分，看看它是不是某个更简单函数的复合。例如 e^(x^2) 中的 x^2、sin(3x+5) 中的 3x+5、1/(x^2+1) 中的 x^2+1 ——这些"里层"通常就是我们的 u。然后检查导数因子是否出现（差一个常数倍也可以）。如果导数因子完全没出现，那这个方法就行不通，你需要改用其他技巧。

常见凑微分模式总结

经过大量练习，我总结了几种最常见的凑微分模式。初学者如果能熟记这些模式，做题速度会大幅提升。

积分形式 换元 结果
∫ f(ax + b) dx u = ax + b (1/a) ∫ f(u) du
∫ x·f(x^2) dx u = x^2 (1/2) ∫ f(u) du
∫ f(g(x))·g'(x) dx u = g(x) ∫ f(u) du
∫ e^(g(x))·g'(x) dx u = g(x) e^u + C
∫ g'(x)/g(x) dx u = g(x) ln|u| + C
∫ sin(g(x))·g'(x) dx u = g(x) -cos u + C
∫ cos(g(x))·g'(x) dx u = g(x) sin u + C

当然，这只是一个起点。实际做题时，被积函数的形式可能不会这么整齐。有时候你需要先做代数变形，才能套用凑微分模式。比如：

∫ tan x dx = ∫ sin x / cos x dx

        令 u = cos x，则 du = -sin x dx
        原式 = ∫ -du / u = -ln|u| + C = -ln|cos x| + C = ln|sec x| + C

很多教材直接把 ∫ tan x dx = ln|sec x| + C 当作公式让你背。但如果你理解了凑微分的思路，你根本不需要背——每次遇到 tan x 的积分，你自然就会把它写成 sin x / cos x，然后凑微分。

一个真实的故事：我曾经辅导过一位学生，他问了我一个很有意思的问题："为什么换元积分被叫做'凑微分'？难道这些步骤是靠'凑'出来的？"我的回答是——没错，就是靠"凑"的。所谓"凑微分"，就是有意识地调整被积函数的形式，让它匹配已知的积分模式。这需要经验的积累，就像拼图高手能一眼看出哪块拼图该放哪里一样。但好消息是，常见的凑微分模式就那么十几种，反复练习之后，你也会拥有这种"直觉"。

第二类换元：三角代换

第一类换元法（凑微分）适用于"内层函数的导数恰好出现在被积函数中"的情况。但还有一类积分，它们的主要障碍是根号——被积函数中包含 √(a^2 - x^2)、√(a^2 + x^2) 或 √(x^2 - a^2) 这类表达式。根号的存在使得我们无法直接应用基本积分公式，即使凑微分也往往无能为力。

这时第二类换元法——三角代换——就派上用场了。核心思路是：利用三角恒等式 sin^2 θ + cos^2 θ = 1 及其变形，把根号下的表达式变成完全平方，从而去掉根号。

三种常见的三角代换：

根号形式 代换 化简结果 对应三角形
√(a^2 - x^2) x = a sin θ √(a^2 - a^2 sin^2 θ) = a cos θ 对边/斜边
√(a^2 + x^2) x = a tan θ √(a^2 + a^2 tan^2 θ) = a sec θ 对边/邻边
√(x^2 - a^2) x = a sec θ √(a^2 sec^2 θ - a^2) = a tan θ 斜边/邻边

看得更仔细一些：

例：∫ √(4 - x^2) dx

        令 x = 2 sin θ，则 dx = 2 cos θ dθ
        √(4 - x^2) = √(4 - 4 sin^2 θ) = 2 cos θ

        原式 = ∫ (2 cos θ)(2 cos θ dθ) = 4 ∫ cos^2 θ dθ

        接下来用倍角公式 cos^2 θ = (1 + cos 2θ)/2：
        4 ∫ (1 + cos 2θ)/2 dθ = 2 ∫ (1 + cos 2θ) dθ
        = 2(θ + (1/2) sin 2θ) + C
        = 2θ + sin 2θ + C

        最后把 θ 代回 x。因为 x = 2 sin θ，所以 sin θ = x/2，θ = arcsin(x/2)。
        sin 2θ = 2 sin θ cos θ = 2 · (x/2) · √(4 - x^2)/2 = x·√(4 - x^2)/2
        代入得：
        原式 = 2 arcsin(x/2) + x·√(4 - x^2)/2 + C

三角代换的步骤总结：

  1. 识别根号的形式，选择合适的三角代换
  2. 代入后化简，去掉根号
  3. 用三角恒等式对被积函数化简
  4. 对三角函数的表达式积分
  5. 把结果用原变量 x 表示（用反三角函数或构造直角三角形）
最容易在这里犯错的地方：很多人在最后一步"把 θ 代回 x"时会卡住。记住：你不需要总是用反三角函数来代回。画一个直角三角形就可以了——从代换关系 x = a sin θ 得出 sin θ = x/a，那么 θ 的对边就是 x，斜边是 a，邻边就是 √(a^2 - x^2)。然后任何 cos θ、tan θ 都可以从这个三角形里读出来，不用死记硬背公式。

倒代换技巧

除了三角代换，另一种非常实用的第二类换元是倒代换（reciprocal substitution），即令 x = 1/t。这种代换主要适用于被积函数中分母的幂次远高于分子的情况。

例：∫ 1 / (x^2 · √(x^2 + 1)) dx

        令 x = 1/t，则 dx = -1/t^2 dt
        原式 = ∫ 1 / ((1/t^2) · √(1/t^2 + 1)) · (-1/t^2) dt
             = ∫ 1 / (1/t^2 · √((1 + t^2)/t^2)) · (-1/t^2) dt
             = ∫ 1 / (1/t^2 · √(1 + t^2)/|t|) · (-1/t^2) dt
        当 t > 0 时：
             = ∫ -1 / (1/t^2 · √(1 + t^2)/t) · (1/t^2) dt
             = -∫ t / √(1 + t^2) dt
        令 u = 1 + t^2，du = 2t dt
             = -(1/2) ∫ 1/√u du = -(1/2) · 2√u + C = -√(1 + t^2) + C
             = -√(1 + 1/x^2) + C = -√(x^2 + 1)/x + C

倒代换的效果是"翻转"分子和分母的角色，有时能让积分结构变得简单。它特别适合处理有理函数积分中分母幂次明显偏高的情况。

如何选择换元策略

对于初学者来说，最头痛的问题就是："这道题该用哪种换元法？"这里提供一个决策框架：

  1. 先看能不能凑微分——找内层函数 u = g(x)，检查 g'(x) 是否出现在被积函数中（可能差一个常数倍）。这是最简单、最常用的方法，永远优先尝试。
  2. 再看有没有根号——如果被积函数中有 √(a^2 - x^2)、√(a^2 + x^2) 或 √(x^2 - a^2)，考虑三角代换。
  3. 分母幂次明显偏高——考虑倒代换 x = 1/t。
  4. 什么方法都试过了还是不行——可能该用分部积分法了（下一章的内容）。

这个决策框架不是万能的，但对于初学者来说，它能帮你走完 80% 的路。剩下的 20% 需要你通过大量练习积累经验，逐步培养出"看题就知道用什么方法"的直觉。

换元法和做菜的关系：我经常跟我的学生说，换元积分法和做菜很像。做菜的流程是——看食材（被积函数）、决定烹饪方法（换元策略）、处理食材（代数变形）、下锅（积分）、出锅摆盘（代回变量并加 C）。初学者做菜时会反复翻看菜谱（积分公式表），但熟练之后，看到一块五花肉，脑子里自然就知道该红烧还是回锅。换元积分法也是一样：熟能生巧，无他，唯手熟尔。

下一章我们将学习积分学的另一个重武器——分部积分法。当被积函数是两个不同类型函数的乘积时（比如 x·e^x 或 x·sin x），分部积分法就是你的最佳选择。同时我们还会学习如何对有理函数进行积分——也就是如何处理两个多项式相除的形式。这些都是积分计算中最实用、也最考验功力的技巧。
