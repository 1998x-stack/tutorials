第2章 金融市场基础

大家好，我是你们的讲师。在上一章中我们讨论了金融分析的定义和框架，今天我们把目光投向分析对象所处的"舞台"——金融市场。想象一个场景：你是一家初创公司的CEO，公司经过三年研发，终于拿出了一款有竞争力的产品，现在需要资金扩大生产。你的选择有哪些？是找银行借钱，还是引入风险投资？抑或是筹备IPO上市？每一个选择的背后，都对应着金融市场的不同板块和不同规则。

金融市场就像一个巨大的生态系统，不同的参与者携带不同的资本、在不同板块中进行交易，共同构成了现代经济体的"血液循环系统"。理解这个系统的结构、参与者行为特征以及运行的规则，是进行任何有意义金融分析的前提。本章我们将从市场分类、交易品种、参与者和A股交易规则四个维度，为你搭建金融市场的基础认知框架。

2.1 金融市场的分类

金融市场按照不同的维度可以有多种分类方式，但最核心、最具有实用价值的是按"发行与流通"和按"组织形式"两个维度划分。

一级市场与二级市场

一级市场（Primary Market）是证券首次发行的市场。企业通过IPO（Initial Public Offering，首次公开发行）或再融资（增发、配股）向投资者出售新证券，募集到的资金直接进入企业账户，用于生产经营。这是金融市场最原始、最基础的功能——将储蓄转化为投资。

二级市场（Secondary Market）则是投资者之间交易已发行证券的市场。你在股票软件上买卖贵州茅台，背后就是在二级市场交易。二级市场的价格波动不直接影响上市公司的资产负债表，但它决定了企业的市场估值，进而影响其再融资能力和股权激励效果。

一级市场和二级市场之间有一个非常重要的互动关系：二级市场的活跃度和定价效率，直接影响一级市场的融资功能。如果二级市场持续低迷、估值偏低，企业就不愿意在一级市场发行新股（因为融不到足够的钱），整个资本市场的资源配置功能就会受损——这正是2024年前后A股市场的真实写照。

核心洞见：一级市场是"造血"，二级市场是"输血"。没有健康的二级市场，一级市场也难以独善其身。分析市场时，永远要关注这两个市场的联动关系。

场内市场与场外市场

场内市场（Exchange-Traded Market）是指在有组织的交易所内进行交易的市场，有标准化的合约、集中的竞价机制和严格的监管。A股的上海证券交易所、深圳证券交易所、北京证券交易所都是典型的场内市场。

场外市场（Over-the-Counter, OTC）则是交易所以外进行的交易，交易方式更为灵活，但透明度较低、流动性较差。中国的全国中小企业股份转让系统（新三板）、美国的OTC Markets Group是典型的场外市场。此外，银行间债券市场在本质上也是一个场外市场——尽管规模巨大，但交易通过一对一询价而非集中竞价完成。

一个实用的判断方法是：如果交易价格是通过"撮合"（买一卖一匹配）形成的，那就是场内市场；如果是通过"谈判"（双方商量价格）形成的，那就是场外市场。

特征 场内市场 场外市场
交易机制 集中竞价、自动撮合 一对一询价、谈判
产品标准化 高度标准化 灵活定制
流动性 高 低
透明度 高（信息充分披露） 低（信息不对称）
监管强度 严格 相对宽松
典型市场 沪深交易所、纳斯达克 新三板、银行间债市

2.2 主要交易品种

金融市场中的交易品种五花八门，但对分析师生涯影响最大的就是以下五大类。每一类品种的风险收益特征差异极大，适合不同的投资目标和风险偏好。

品种 本质 风险等级 预期回报 流动性 投资门槛
股票 公司所有权凭证 高 高（长期） 高 低
债券 债务债权凭证 低-中 中低（固定利息） 中 中（银行间市门槛高）
衍生品 基于标的资产的合约 极高 极高（杠杆） 中-高 中（需验资）
外汇 国家货币之间的兑换 中-高 中（波动率低） 极高 低（零售）
基金 集合投资工具 取决于底层资产 取决于底层资产 高（开放型） 极低

对于金融分析师而言，股票和债券是基本面分析的核心战场。衍生品更多用于对冲和风险管理（虽然投机者将其用于放大收益）。外汇分析通常与宏观经济研究紧密相关。基金则是一种"打包"产品——分析基金其实是在分析其管理人能力和底层资产配置。

学习建议：作为初学者，建议从股票分析入手。原因有三：第一，股票数据最为公开透明，获取成本极低；第二，股票分析涉及的财报解读、估值建模等技能可以平滑迁移到债券和基金分析；第三，A股市场目前有超过5000只股票，提供了丰富的分析样本。

2.3 市场参与者

金融市场不是抽象的概念，而是由形形色色的人和组织组成的。理解每个参与者的利益诉求和行为模式，对于判断市场走势和交易机会至关重要。

散户投资者

个人投资者，在A股市场贡献了约60-70%的交易量（尽管只持有约20-30%的市值）。散户的特点是资金规模小、信息获取能力弱、情绪化交易倾向明显。一个经典的散户行为特征是"追涨杀跌"——在市场上涨时买入，在市场下跌时恐慌抛售。行为金融学对此有深入的分析，我们将在第18章展开讨论。

机构投资者

包括公募基金、私募基金、保险资金、社保基金、券商自营、养老金等。机构通常拥有专业的投研团队、系统的分析流程和严格的风控体系。截至2026年初，A股市场中机构投资者的持股市值占比已经超过50%，且仍在持续提升。这个趋势对分析方法有深远的影响——当市场的主导力量从散户转向机构时，基本面研究的价值会更加凸显，单纯的技术图形分析的效力则会下降。

做市商

做市商（Market Maker）的作用是为市场提供流动性——它们同时挂出买入价和卖出价，从买卖价差中赚取利润。在A股市场，做市商制度主要存在于科创板、ETF和债券市场。做市商的利益在于维持平稳的交易环境（不暴涨暴跌），而非方向性投机。关注做市商的行为变化，有时可以提前发现流动性风险的苗头。

监管机构

在中国，主要的金融市场监管机构包括中国证监会（CSRC，负责证券期货市场）、国家金融监督管理总局（负责银行业和保险业）和中国人民银行（负责货币政策和宏观审慎管理）。监管机构通过制定规则、实施检查和执行处罚来维护市场秩序。2024年新"国九条"出台后，A股市场的监管趋严态势进一步升级，退市制度常态化、"长牙带刺"的监管风格对市场生态产生了深远影响。

参与者 核心目标 投资期限 信息优势 对市场的影响
散户 财富增值 短期 弱 提供流动性、放大波动

分析师视角：当你做投资决策时，想清楚你的交易对手是谁。你的买入对手可能是正在减仓的机构，你的卖出对手可能是正在建仓的保险资金。理解对手方的行为逻辑，能帮你做出更理性的判断。

2.4 A股市场交易规则

每个市场都有自己的"游戏规则"。A股市场的交易规则与美股、港股有显著差异，这些差异直接影响分析策略和交易执行。以下是几个最需要关注的核心规则。

T+1 结算制度

A股实行T+1结算——当天买入的股票，下一个交易日才能卖出。这与美股的T+0（当天可卖）和港股的T+2（两个工作日后交收）都不同。T+1意味着如果你在上午追高买入，下午股价跳水，你无法止损出局——只能等到第二天。这一规则降低了日内投机频率，但也增加了隔夜风险。对于分析师来说，T+1意味着收盘前30分钟是全天最重要的交易时段，因为想要当天离场的资金必须在此时段内完成卖出。

涨跌停板制度

A股主板股票的单日涨跌幅限制为±10%，创业板和科创板为±20%，北交所为±30%。涨跌停板在正常市场条件下可以防止过度波动，但在极端行情下会引发严重的流动性危机——当大量卖盘涌出而买盘不足时，股票直接跌停，持有者无法卖出，恐慌情绪在无法宣泄的情况下进一步加剧，形成"跌停-恐慌-更猛烈的跌停"的恶性循环。

踩坑警示：2015年A股股灾是涨跌停板制度下流动性危机的典型案例。2015年6月15日至7月8日，上证指数从5178点暴跌至3421点，跌幅超过34%。期间多次出现"千股跌停"的极端局面——6月26日超过2000只股票跌停，7月7日更是超过1700只股票跌停，同时有超过1400只股票申请停牌（"千股停牌"）。停牌意味着彻底失去了流动性——想卖都卖不掉。这个教训告诉我们：流动性风险是金融市场中最容易被低估的风险，尤其是当你看到别人都在"逃命"而自己却被关在里面的时候。

融资融券

融资（Margin Buying）是向券商借钱买股票，融券（Short Selling）是向券商借股票卖出。融资融券本质上引入了杠杆，放大了收益和亏损。A股的融资余额通常远大于融券余额（2026年约为融券余额的50倍以上），这意味着市场天然具有"做多容易做空难"的结构性特征。对于分析师来说，融资余额的变化是一个重要的市场情绪指标——融资余额持续攀升意味着市场乐观情绪高涨，反之则意味着悲观。

注册制全面落地（2026年）

2026年是A股注册制改革全面落地的关键年份。与过去的核准制不同，注册制下监管机构只对申报材料的完整性和合规性进行审核，不对企业价值进行实质性判断。这意味着：第一，上市门槛降低，新股供应量增加；第二，退市制度更加严格，"壳价值"大幅缩水；第三，对投资者的专业能力要求显著提高——不再有审批制下的"监管背书"。

注册制对金融分析的影响是结构性的：分析的价值被大大提升了。在核准制时代，能够上市的公司在某种意义上已经被"官方认证"了质量；注册制时代，辨别优劣的职责完全落在投资者和分析师自己身上。那些缺乏基本面支撑的公司，上市后很快会被边缘化甚至退市。

下面是一个简单的Python代码片段，帮助你快速查看当前A股市场的基本信息——不同的上市板块和交易状态：

import akshare as ak
import pandas as pd

# 获取A股所有股票实时行情
df = ak.stock_zh_a_spot_em()

# 查看各板块股票数量
print("板块分布：")
print(df['板块'].value_counts())

# 查看涨跌停股票
zt = df[df['涨跌幅'] >= 9.5]
dt = df[df['涨跌幅'] <= -9.5]
print(f"\n当前涨停个股: {len(zt)} 只")
print(f"当前跌停个股: {len(dt)} 只")

# 查看成交额前十
top_vol = df.nlargest(10, '成交额')[['名称', '最新价', '涨跌幅', '成交额']]
print("\n成交额 TOP 10:")
print(top_vol)

这段代码虽然简单，但能帮你快速掌握市场全貌——哪些板块最活跃，市场情绪是偏多还是偏空，资金在往哪些方向流动。在第23章中，我们将系统学习更多实用的Python金融数据分析工具。

2.5 市场有效性的思考

在结束本章之前，我们有必要讨论一个贯穿整个金融分析的理论问题：市场到底有多"有效"？

有效市场假说（Efficient Market Hypothesis, EMH）认为，市场价格已经反映了所有可获得的信息，因此没有人能够持续获得超额收益。如果EMH完全成立，那么金融分析就是徒劳的——反正价格已经对了，分析也白分析。但现实显然不是这样。

行为金融学的研究表明，市场中存在大量由认知偏差导致的定价错误。A股市场作为一个以散户为主导的新兴市场，信息传导效率远低于美股等成熟市场，定价错误的频率和幅度都更大——这意味着在A股市场，基本面分析有更大的用武之地。但也不得不承认，随着机构化进程加速和市场效率提升，寻找定价错误的难度在逐年增加。

核心洞见：对于大多数投资者而言，接受"市场在大部分时间是有效的，但在某些时刻会犯错"这一中间立场可能最为实用。与其花精力判断市场是否有效，不如把精力放在识别那些"市场可能犯错"的情境上——比如极端情绪驱动的涨跌、重大事件被过度反应或反应不足的时候。
