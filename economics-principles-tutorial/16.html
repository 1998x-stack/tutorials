第16章 公共物品与公共资源

16.1 物品的两种属性：排他性与竞争性

大家好，我是你们的讲师。上一章我们讨论了外部性——当你的行为影响了别人的福利，而你没有为此付出或得到补偿。本章我们从另一个角度来理解市场失灵：为什么有些东西市场根本不愿提供？为什么有些资源被过度使用？

要回答这些问题，我们需要先理解物品的两种基本属性：

  • 排他性（Excludability）：能否阻止不付费的人使用。一件衣服——可以，不付钱就拿不走；一条无线电视信号——很难，任何有接收器的人都能看
  • 竞争性（Rivalry）：一个人使用了是否减少其他人可用的量。一个苹果——是，我吃了你就没得吃了；一部电影——不是，你看了一部电影，并不妨碍别人也看同一部

将这两个维度组合，我们可以把物品分为四大类。理解这个分类框架，是理解公共经济学的基础。

表：四种物品类型

排他性（可以阻止使用）非排他性（无法阻止使用）
竞争性（一个人使用减少他人用量）私人物品（Private Goods）
例子：食物、衣服、汽车、手机
由市场有效提供		公共资源（Common Resources）
例子：公共渔场、公共牧场、清洁空气
容易过度使用——公地悲剧
非竞争性（使用不减少他人用量）俱乐部物品（Club Goods）
例子：有线电视、收费高速公路、会员制健身房
自然垄断倾向		公共物品（Public Goods）
例子：国防、灯塔、基础科学研究
市场无法充分提供——搭便车问题

这四种物品需要不同的供给机制。私人物品市场可以高效提供。俱乐部物品需要定价机制来回收成本。公共资源和公共物品则是市场失灵的典型领域——也是本章的重点。

16.2 公共物品与搭便车问题

公共物品的两个特征

公共物品（Public Goods）同时具备非排他性和非竞争性。国防是最经典的例子：国家提供导弹防御系统，保护了所有公民——你无法让某个公民不享受保护（非排他），而多保护一个人并不会增加额外成本（非竞争）。

灯塔是经济学教材中另一个经典案例。灯塔照亮海面，所有经过的船只都能受益。灯塔所有者很难向每一艘受益的船只收费（非排他），一艘船利用了灯塔的光芒并不妨碍另一艘船也利用它（非竞争）。因此，私人几乎没有动力建造灯塔——这就是搭便车问题（Free Rider Problem）。

搭便车问题

当人们能够不付费就享受某种物品的好处时，理性选择就是"搭便车"——等待别人付费，自己免费享用。但如果每个人都这么想，就没人付费了，公共物品就不会被提供。这就是为什么公共物品通常由政府通过税收来提供。

一个有趣的实验：假设你和9个同学想请一位保洁阿姨打扫宿舍走廊。每人最多愿意出20元，但请阿姨需要100元。如果每个人都说"大家凑钱吧"，但心里盘算着"别人会出钱的，我不用出钱也能享受干净的走廊"——结果钱永远凑不齐。这就是搭便车问题在微观层面的体现。

16.3 公地悲剧：人人享有的资源，人人都不珍惜

哈丁的经典论文

1968年，生态学家加勒特·哈丁（Garrett Hardin）在《科学》杂志上发表了那篇著名的文章《公地的悲剧》（The Tragedy of the Commons）。他的故事是这样的：一片公共牧场对所有牧民开放。每个牧民都理性地增加自己的牲畜数量——多一头牛自己获得全部收益，而过度放牧的成本由所有牧民共同承担。最终，每个牧民都不断增加牲畜，牧场不堪重负走向毁灭。

哈丁总结道："在一个共享资源的社会中，每个人追求自身利益最大化，最终会导致所有人的毁灭。"这句话后来成为环境经济学的核心引文之一。

16.4 真实故事：内蒙古草原退化的公地悲剧

让理论落地。从20世纪80年代到21世纪初，中国内蒙古草原经历了一场教科书式的公地悲剧。

1980年代，内蒙古牧区推行"草场公有、牲畜私有"的"双权一制"改革。草场归集体所有，但牲畜归牧民个人。这几乎完美复制了哈丁的场景：每个牧民都希望自己的羊群更大。多养一只羊的收益100%归自己，但多消耗的草场资源由全嘎查（村）共同承担。结果是——草原上的羊群数量从1980年的约3000万只激增到2000年的超过6000万只。草场严重超载，大片草原沙化退化，沙尘暴越来越频繁。2000年春季，北京经历了12次大范围的沙尘暴，"风沙紧逼北京城"成为头条新闻。

2002年后，中国启动了退牧还草工程和草畜平衡制度。核心措施包括：划定禁牧区和休牧期、以草定畜（根据草场的产草量确定允许的牲畜数量）、对减畜牧民给予补贴、推行草场承包到户（明确产权）。这些措施本质上是用"产权私有化+政府管制"的组合来解决公地悲剧。到2015年，内蒙古草原植被覆盖度从最低时的不足30%恢复到约44%。

经济学启示：内蒙古草原的故事完美展示了解决公地悲剧的几种思路——明确产权（科斯定理的应用）、政府直接管制（禁牧休牧）、市场激励（补贴）。没有单一方案是万能的，需要因地制宜的组合。

16.5 埃莉诺·奥斯特罗姆的八项原则

难道"公有"一定等于"悲剧"吗？2009年诺贝尔经济学奖得主埃莉诺·奥斯特罗姆（Elinor Ostrom）给出了否定的答案。她花了几十年研究世界各地的社区如何成功管理公共资源——从瑞士的阿尔卑斯牧场到日本的森林共有地。

奥斯特罗姆发现，成功的公共资源管理通常具备以下八项原则：

  1. 清晰界定的边界：谁有权使用资源，界限明确
  2. 与当地条件匹配的规则：使用规则适应当地的具体条件
  3. 集体选择安排：受规则影响的人可以参与修改规则
  4. 监督：有监督者（往往是使用者自己）检查遵守情况
  5. 分级制裁：违规者受到与其违规严重程度相符的制裁
  6. 冲突解决机制：有低成本、便捷的渠道解决冲突
  7. 对组织权的最低程度认可：社区的自组织权利被外部政府认可
  8. 嵌套式企业：对大型公共资源的管理是多层次的，从基层到区域

奥斯特罗姆的研究证明：公有≠悲剧。当社区有能力自主组织、制定规则并监督执行时，公共资源可以被可持续地管理。这一发现打破了"要么私有化、要么政府管制"的二元思维。

16.6 数字时代的公共物品

在21世纪，公共物品的概念有了全新的载体——数字公共物品。

开源软件

Linux内核、Python解释器、MySQL数据库——这些软件的使用是非竞争性的（你下载一份并不妨碍别人也一样用）。理论上它们也可以是排他性的（闭源授权），但开源社区选择开放代码。这构成了一个有趣的经济学悖论：全球最有价值的软件基础设施之一，是由成千上万的志愿者和公司贡献的"公共物品"。

那么为什么开源项目没有遭遇搭便车问题的毁灭？原因在于：贡献者的动机不仅仅是直接的金钱回报——声誉、学习、开发者的自我实现、企业对基础设施的共同维护需求——这些"私人收益"足以支撑公共物品的持续提供。

维基百科

维基百科是另一个惊人的公共物品成功案例。世界上最大的百科全书，完全免费、无广告、由志愿者撰写和编辑。2023年，维基百科月活跃用户超过15亿。它的运营依靠极少量全职员工（不到700人）和数百万志愿者的贡献。经济学家还在研究：究竟是什么激励机制让这么多人"免费"创造了如此巨大的价值？

数字公共物品的脆弱性：数字公共物品也面临风险。开源项目可能维护者 burnout、被恶意收购或注入后门代码。维基百科面临虚假信息、编辑战和政治操纵。公共物品的成功不是理所当然的——它需要持续的制度维护。

16.7 最优公共物品供给：萨缪尔森条件

公共物品应该提供多少？这是公共经济学的一个核心问题。与私人物品不同，公共物品的总社会收益是所有个人收益的加总（因为人人共享，而非竞争）。

经济学家保罗·萨缪尔森（Paul Samuelson）在1954年提出了萨缪尔森条件（Samuelson Condition）：公共物品的最优供给量满足——所有个人的边际收益之和等于提供该物品的边际成本。

/* 萨缪尔森条件：公共物品的最优供给 */

/* 假设一条高速公路的照明系统升级项目 */
/* 每隔多远设置一盏路灯(N)，决定了总成本 */

边际成本(MC) = 2000 元/每盏路灯
受益者：3个小区，各小区的边际收益不同

小区A（靠近入口）的边际收益: MBA = 1200 - 10N
小区B（中间位置）的边际收益:  MBB = 800 - 5N
小区C（靠近出口）的边际收益: MBC = 600 - 5N

/* 萨缪尔森条件：MBA + MBB + MBC = MC */
(1200 - 10N) + (800 - 5N) + (600 - 5N) = 2000
2600 - 20N = 2000
20N = 600
N = 30 盏路灯 ← 最优数量

/* 检验：每盏路灯的社会边际收益 */
N=30时: MBA = 1200 - 300 = 900
        MBB = 800 - 150 = 650
        MBC = 600 - 150 = 450
总社会边际收益 = 900 + 650 + 450 = 2000 = MC

/* 如果只靠市场供给 */
小区A愿意支付900元，但只有它自己出钱，不足以覆盖2000元的成本
——这就是市场无法充分提供公共物品的原因 */
与私人物品的对比：私人物品中，总量由个人需求曲线横向加总（每个消费者都是价格接受者，数量加总）。公共物品中，社会总价值通过纵向加总（每个消费者支付自己愿意支付的价格，价格加总）。这正是萨缪尔森条件的精妙所在。

16.8 本章小结

排他性和竞争性构成了物品分类的坐标系。公共物品（非排他+非竞争）面临搭便车问题，私人市场无法充分提供。公共资源（非排他+竞争）面临公地悲剧，容易过度使用。奥斯特罗姆告诉我们，在合适的制度条件下，社区可以自主管理公共资源。而萨缪尔森条件为公共物品的最优供给提供了理论基准。
