第28章 综合实战：像宏观经济学家一样思考

2026年6月30日，北京金融街的一间办公室里，一位宏观经济分析师打开工作邮箱，看到了国家统计局刚刚发布的2026年上半年宏观经济数据。屏幕上是一串串数字——GDP增速、CPI、失业率、社融规模、PMI——每一个数字都像一个拼图碎片。她需要做的，是像已经做了上千次那样，把这些碎片拼成一张完整的宏观经济图景。

如果今天坐在这张办公桌前的人是你，你该怎么办？从哪里开始？用什么框架来分析？最后要得出什么判断？

这一章没有新的理论，没有新的模型。它是整个二十八章课程的收网之作——我们要把所有学过的工具放到一张"分析地图"上，然后用两个完整的案例来演练：一次形势诊断、一次政策模拟。准备好了吗？我们开始了。

1. 宏观分析框架总结：把28章串成一张分析地图

面对一组宏观数据，初学者会感到眼花缭乱——GDP、CPI、PPI、PMI、社融、M2、LPR、失业率、外汇储备……该从哪个看起？我们先把整个分析框架浓缩成一张"四层递进"的流程图：

宏观形势分析的四层递进框架

        ┌──────────────────────────────────────────┐
        │  第一层：数据画像                          │
        │  · 增长：实际GDP增速 → 是快是慢？          │
        │  · 通胀：CPI/核心CPI/PCE → 是冷是热？     │
        │  · 就业：失业率/新增就业 → 是紧是松？      │
        │  · 外部：贸易余额/汇率/资本流动 → 顺逆差？ │
        └──────────────────┬───────────────────────┘
                           ▼
        ┌──────────────────────────────────────────┐
        │  第二层：周期定位                          │
        │  · 用产出缺口判断：实际GDP vs 潜在GDP     │
        │  · 用菲利普斯曲线判断：通胀vs失业的位置    │
        │  · 用库存周期/信贷周期辅助判断             │
        │  · 问题：经济在繁荣/衰退/复苏/萧条哪一阶段？│

        │  · 货币面：量(M2/社融)和价(利率/汇率)     │
        │  · 预期面：消费者信心/企业信心/PMI → 情绪  │

        ┌──────────────────────────────────────────┐
        │  第四层：政策应对                          │
        │  · 货币政策应该松还是紧？空间有多大？      │
        │  · 财政政策应该积极还是审慎？债务空间？    │
        │  · 结构性改革：哪些制度障碍需要破除？      │
        │  · 最终判断：当前政策组合够不够？要不要加码？│
        └──────────────────────────────────────────┘

这四层不是线性的，实际上你需要在它们之间来回跳跃。比如：在政策应对时发现空间不足，可能需要回到第二层重新判断经济"病得有多重"。但作为一个起手式，从第一层到第四层逐步推进是最可靠的习惯。

分析层次 核心问题 关键变量 对应章节
数据画像 经济目前是什么状态？ GDP、CPI、失业率、贸易差额 第2-7章
周期定位 经济处在周期的什么位置？ 产出缺口、NAIRU、库存 第12、17、23章
驱动拆解 哪些力量在推动或拖累经济？ C/I/G/NX分项、M2、PMI 第8-11、15-16、19章
政策应对 应该用什么政策？力度多大？ 利率空间、财政空间、汇率 第15-18、24-25章
长期视角 增长的可持续性如何？ TFP、人口、制度质量 第21-22章
分析师的工作习惯：好的宏观经济分析师从来不是"用一次框架"就得出答案。TA会先用数据和图表形成直觉判断，然后用模型和框架来验证或证伪这个直觉。一个最有经验的宏观分析师是这样工作的：先用30分钟扫完所有关键数据，形成"直觉故事"；然后用1-2小时深入数据验证，修正故事；最后用30分钟写出结论。思考和验证的时间远远多于"写结论"的时间。你在刚开始练习的时候，这个流程可能需要一整天——不要着急，熟练了自然会快起来。

2. 案例一：完整的宏观形势诊断

现在是2026年7月初。假设你收到了以下2026年上半年中国宏观经济数据。请写出你的分析报告。

背景数据：2026年上半年中国宏观经济主要指标

        指标                    2025H2       2026H1        变动
        ─────────────────────────────────────────────────────────
        实际GDP同比增速          5.1%         4.8%          ↓0.3pp
        名义GDP同比增速          6.5%         5.2%          ↓1.3pp
        GDP平减指数              1.4%         0.4%          ↓1.0pp

        CPI同比                  0.3%         0.1%          ↓0.2pp
        核心CPI同比              0.6%         0.4%          ↓0.2pp
        PPI同比                 -1.2%        -1.5%          ↓0.3pp

        城镇调查失业率            5.1%         5.3%          ↑0.2pp
        16-24岁青年失业率        14.2%        15.8%          ↑1.6pp

        社会消费品零售总额增速    4.8%         4.2%          ↓0.6pp
        固定资产投资累计增速      5.5%         5.0%          ↓0.5pp
        其中：房地产开发投资      -8.5%        -9.2%         ↓0.7pp
        其中：制造业投资          9.2%         8.8%          ↓0.4pp
        其中：基础设施投资        5.0%         5.6%          ↑0.6pp

        出口同比增速              6.5%         5.0%          ↓1.5pp
        进口同比增速              3.0%         2.5%          ↓0.5pp
        贸易顺差（亿美元/月）     650          600            ↓50

        M2同比增速                9.0%         8.2%          ↓0.8pp
        社会融资规模存量增速      9.5%         8.8%          ↓0.7pp
        1年期LPR                  3.10%        2.85%         ↓0.25pp
        5年期LPR                  3.60%        3.30%         ↓0.30pp

        人民币兑美元（期末）      7.18         7.25          ↓0.7%

现在，让我们按照"四层递进"框架进行分析：

第一层判断：经济仍在扩张，但动能显著放缓。实际GDP增速4.8%在全球主要经济体中不算低，但相比2025年下半年已下降0.3个百分点。更重要的是，名义GDP增速从6.5%骤降到5.2%，GDP平减指数从1.4%降到0.4%——这说明经济正在从"微温"滑向"偏冷"。物价信号非常令人担忧：CPI只有0.1%，PPI连续为负，通缩压力正在加大。

第二层判断：产出缺口可能已经转负。潜在增长率估算在5.0%-5.2%左右，实际增速4.8%意味着经济可能运行在潜在产出以下。失业率上升（尤其是青年失业率突破15%）进一步确认了"经济低于潜在水平"的判断。菲利普斯曲线正在低通胀一侧运行——通胀持续走低，失业上升，符合经济处于"衰退初期"或"复苏中断"的特征。

第三层判断：需求侧全面偏弱，旧动能仍是主要拖累。

  • 消费（C）：社零增速4.2%，低于GDP增速，说明消费没有发挥"压舱石"作用。背后是就业市场疲软和房价下跌的负财富效应。
  • 投资（I）：房地产投资仍在大幅收缩（-9.2%），制造业投资增速也在放缓。唯一的亮点是基建投资小幅加速（+5.6%），体现了财政发力。
  • 政府支出（G）：基建投资加速说明财政正在发力，但从广义赤字率看，力度尚不足以完全对冲房地产的下行。
  • 净出口（NX）：贸易顺差还在，但增速在收窄。关税战的影响正在持续释放。

第四层判断：政策已放松，但传导不畅。央行在2026年上半年连续降息降准——1年期LPR下降25bp、5年期LPR下降30bp——但社融增速却在下降（从9.5%到8.8%）。这是典型的"宽货币、紧信用"格局：银行有钱想贷，但企业和居民不想借。这说明货币政策遇到了"流动性陷阱"式的困境——利率降了，但信心不足，信贷需求起不来。货币政策的极限在接近，财政政策需要更加积极。

结论：2026年上半年中国经济处于"动能放缓、通缩压力加大、信心不足"的状态。政策应对的方向是清晰的——货币政策继续宽松+财政政策显著加码+结构性改革破除传导障碍。但关键的问题不是"该不该做"，而是"能不能做"——货币政策空间在缩小（利率已接近零下限），财政空间受制于债务约束。中国需要在大规模财政刺激（解决短期需求）和财政可持续性（维护长期信用）之间找到平衡。预计下半年政策组合将更加积极地偏向财政扩张，包括增加中央政府杠杆率、扩大消费补贴范围、加速推进"十四五"规划中的重大项目。

3. 案例二：政策模拟

假设你是政策制定者的经济顾问。现在，你面临以下决策情景：

政策方案：央行降息50个基点（1年期LPR从2.85%降至2.35%，5年期LPR从3.30%降至2.80%），同时财政部宣布新增1万亿人民币一般公共预算赤字（主要用于消费补贴和民生支出）。

请用IS-LM和AD-AS框架分析这一政策组合的传导机制和预期效果。

政策模拟的边界条件——永远记住三件事：
第一，模型是简化，不是现实。IS-LM框架假设价格水平不变、没有预期因素，但现实中预期变化会显著影响政策效果。如果企业和居民对未来的信心没有改善，降息带来的信贷扩张可能非常有限。

第二，政策有滞后性。降息的效果通常需要6-12个月才能完全传导到实体经济。财政支出的拉动效果也需要3-6个月。今天的政策决策，效果要到明年才能充分体现。这要求政策制定者要有"前瞻性"——不能等到经济已经明显恶化再出手。

第三，政策的"组合效应"不等于各部分之和。单独降息可能效果有限，单独财政扩张可能面临挤出效应，但两者同时实施会产生协同效应（synergy）：财政扩张提升了信心，让降息后的信贷渠道更通畅；降息降低了财政融资成本，让财政扩张更可持续。这正是"宏观经济治理"优于"单一工具"的核心理念。

4. 宏观经济学家的工具箱

二十八章的课程就要结束了，但真正的学习才刚刚开始。宏观经济学是一门必须与实践相结合的学科。这里给你整理一份"毕业后"的自我修炼清单——如果你希望在宏观分析的道路上持续进阶，以下资源值得长期关注。

数据库——练好数据基本功：

  • 中国数据：国家统计局（stats.gov.cn）、人民银行（pbc.gov.cn）、Wind/CEIC等金融终端。每天花15分钟浏览主要指标的习惯，比任何理论课程都更有价值。
  • 全球数据：IMF数据映射器（data.imf.org）、世界银行公开数据（data.worldbank.org）、BIS国际清算银行（bis.org）。关注跨境资本流动和全球债务数据。
  • 高频数据：日度/周度的高频数据（如30城房地产成交、电厂耗煤、港口集装箱吞吐量）可以帮助你"提前"感知经济动向。在中国，这些高频指标通常比月度的官方统计早2-4周。

学术前沿——保持认知的上限：

  • 期刊：《American Economic Review》、《Quarterly Journal of Economics》、《Journal of Political Economy》、《Econometrica》。如果你觉得全文太难，至少读摘要和结论。
  • 学者Twitter/Bluesky：保罗·克鲁格曼（Paul Krugman）、拉里·萨默斯（Larry Summers）、努里埃尔·鲁比尼（Nouriel Roubini）、迈克尔·佩蒂斯（Michael Pettis）——关注他们的政策评论，但不盲从。
  • 中文资源：中国金融四十人论坛（CF40）、北大国发院、清华五道口——中国本土最活跃的宏观政策讨论平台。
一条学习忠告：在这个信息爆炸的时代，你可以每天花三小时阅读宏观报告和新闻。但我的建议是——少看短期预测，多练分析框架。90%的短期宏观经济预测（比如下个月的CPI是多少、下季度的GDP多少）都会被很快证伪或修正。真正有价值的是你的分析框架——当新数据出现时，你的框架能帮你比别人更快地理解"这意味着什么"，而不是"数字是多少"。框架才是你的护城河。

5. 结束语：宏观经济学不是预测术，而是理解世界的思维框架

二十八章，从IS-LM到AD-AS，从索洛模型到DSGE，从货币传导到金融加速器——我们走过了一段漫长的旅程。在最后这一小节，我想和你分享三个我个人认为最重要的"元认知"——它们不是经济学知识，但比任何知识都重要。

第一，谦逊——接受不确定性。经济学不是物理学。宏观经济学家无法像物理学家预测行星轨道那样预测经济走向。2008年金融危机之前，几乎没有人预测到了它的规模和后果；2020年的疫情冲击更是完全出人意料。好的宏观经济学家从不声称自己能"准确预测"，他们的价值在于：为决策提供了一种系统性的思考方式，帮助他们理解不同情景下的概率和代价。如果你学完这门课只记住一件事，我希望是：对任何声称"肯定"的宏观经济预测保持怀疑。

第二，多元模型思维——不执着于单一框架。凯恩斯主义者、货币主义者、新古典学派、奥地利学派——每个理论都有它的洞见，也都有它的盲区。学完宏观经济学之后最危险的阶段是"锤子陷阱"——你学会了IS-LM模型，于是所有问题看起来都像是IS-LM问题。真正优秀的宏观经济学家会在不同的模型之间灵活切换：分析短期波动用IS-LM和AD-AS，分析增长用索洛和内生增长框架，分析金融危机用明斯基的金融不稳定性假说，分析长期停滞用萨默斯的框架。保持工具箱的多样性，是避免"拿着锤子看什么都像钉子"的最好方法。

第三，保持好奇心——宏观经济学的未解之谜。我们的旅程结束了，但宏观经济学本身的旅程还在继续。今天仍然有一系列重大问题没有定论：数字货币会如何改变货币政策传导？AI对全要素生产率的长期影响到底是革命性的还是渐进性的？人口负增长的长期后果我们真的理解吗？气候变化带来的宏观经济成本如何量化？这些问题中没有一个是这门课程能给出最终答案的——但你所学到的工具和分析框架，足够让你去思考它们、去理解围绕它们的争论、并最终形成你自己的判断。

最后送你一句话：
美国经济学家肯尼斯·博尔丁（Kenneth Boulding）曾经说过一句著名的自嘲："经济学是证明'昨天预测的今天会发生什么'为什么发生在昨天的科学。"这句话既是对经济学预测能力的辛辣讽刺，也提醒我们经济学的真正价值——它不是算命，而是一种理解世界的思维框架。

保持谦逊，持续学习。宏观经济学的魅力不在于它给了你多少确定的答案，而在于它教会了你如何提出好的问题。这二十八章只是一个开始。真正的课堂，是每一天都在发生的现实世界。
