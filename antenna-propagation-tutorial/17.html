大家好，我是你们的讲师。在上一章我们学会了如何测量天线——但测量是为了验证设计。现在我们要问一个更本质的问题：如果我们需要一个特定形状的方向图，应该怎样设计天线阵列？

想象一下你是一位雷达系统工程师，军方要求你设计一个能在300公里外探测到RCS为1 m²目标的预警雷达天线。这意味着你需要极低的副瓣来抑制地物杂波，还要在特定方向上设置零点来对抗干扰。或者，你是一位5G基站工程师，需要为Massive MIMO天线设计一个能在水平 ±60° 范围内提供恒定增益的"扇形波束"。这些需求不是"碰运气"能解决的——你需要一套系统的方法，从目标方向图出发，计算出阵列各单元所需的激励系数。这就是阵列综合（Array Synthesis）要解决的问题。

17.1 阵列综合问题定义

阵列综合是天线阵列的"逆问题"：

正问题（分析）： 给定阵列几何和激励系数 → 计算方向图
逆问题（综合）： 给定目标方向图 → 计算激励系数

设一个N单元等间距线阵的阵因子为：

AF(θ) = Σ_{n=0}^{N-1} w_n · e^{j · n · k · d · (cosθ - cosθ₀)}

其中 wn = an · ejφn 是第n个单元的复加权系数，d是阵元间距，θ₀是主波束指向。综合问题的"输入"是目标方向图（如期望的波束宽度、副瓣电平、零点位置等），"输出"是一组 wn。

阵列综合的典型设计目标包括：

  • 低副瓣（Low Sidelobes）：降低旁瓣区域的辐射功率，抑制干扰和杂波
  • 窄波束（Narrow Beamwidth）：提高角分辨率，通常与阵列口径成正比
  • 零点控制（Null Placement）：在干扰方向设置零点
  • 波束赋形（Beam Shaping）：如余割平方形、平顶形等特定方向图形状

17.2 Dolph-Chebyshev综合法

原理：将阵因子映射到Chebyshev多项式

1946年，美国数学家C. L. Dolph发表了一篇里程碑式的论文，提出了一种在给定副瓣电平（SLL）下实现最窄主瓣宽度的最优综合方法。其核心思想是利用Chebyshev多项式的等波纹特性来构造阵因子。

Chebyshev多项式 Tm(x) 在 |x| ≤ 1 范围内等幅振荡（值在 -1 到 1 之间），在 |x| > 1 范围内快速增长。Dolph天才地将阵因子 AF(θ) 映射为 Chebyshev 多项式 TN-1(x)，使得：

  • 主瓣区域对应 x > 1 的快速增长区
  • 副瓣区域对应 |x| ≤ 1 的等波纹区 → 所有副瓣幅度相等

这种方法保证：在给定副瓣电平的条件下，Dolph-Chebyshev阵列的主瓣宽度最窄；反过来，在给定主瓣宽度的条件下，其副瓣电平最低。这就是所谓的最优（optimal）阵列综合。

数学魔术： Chebyshev多项式有一个惊人的性质——在所有次数为 N-1 且首项系数相同的多项式中，TN-1(x) 在 |x| ≤ 1 区间内具有最小的最大绝对值。这个"极小化极大"（minimax）性质是Dolph方法最优性的数学根源。

设计步骤与参数对比

Dolph-Chebyshev综合的设计步骤：

  1. 确定阵元数 N、单元间距 d 和期望副瓣电平 SLL（dB）
  2. 由 SLL 计算 Chebyshev 多项式在 x = x₀ 处的值：TN-1(x₀) = 10SLL/20
  3. 通过反余弦计算 x₀
  4. 使用 Chebyshev 多项式的根或系数递推公式计算各单元的激励幅度
  5. 激励相位由主波束指向 θ₀ 决定

下表展示了不同副瓣电平要求下的阵列性能对比（N=16, d=λ/2）：

副瓣电平 (dB) 主瓣宽度 (度) 边缘单元与中心单元幅度比 锥削效率 (%)
-20 6.8 0.43 94.3
-25 7.3 0.27 88.1
-30 7.8 0.16 79.8
-35 8.4 0.09 69.5
-40 9.0 0.04 57.3

可以看到，更低的副瓣需要更剧烈的幅度锥削（taper）——即边缘单元的激励幅度远小于中心单元。代价是主瓣变宽和锥削效率下降（即阵列的天线增益降低）。这就是天线阵列设计中一条基本规律：低副瓣是以增益和波束宽度为代价换来的。

实践中的教训： 我曾经参与过一个雷达天线的设计，理论仿真时副瓣设计为-35 dB，一切完美。但实际加工的阵列中，由于各通道的幅度误差和相位误差（源于移相器和衰减器的量化误差），实测副瓣只达到了-28 dB。这个教训让我深刻认识到：理论综合的副瓣电平必须在实际硬件精度的基础上留有裕量。通常建议保留5-10 dB的设计裕量。

17.3 Taylor综合法

Chebyshev方法的问题

Dolph-Chebyshev阵列虽然最优，但有一个实际缺陷：所有副瓣都被强制为等幅。这意味着远离主瓣的远端副瓣（far-out sidelobes）也被设定为和紧邻主瓣的近端副瓣（near-in sidelobes）一样高。然而在实际系统中，远端副瓣通常不是主要问题——它们会被阵列单元的方向图自然衰减所抑制。更重要的是，等幅的远端副瓣意味着阵列的激励幅度分布会在边缘处出现"突起"（即边缘单元的激励幅度比相邻单元大），这在实际馈电网络中很难实现。

Taylor的改进

1955年，T. T. Taylor提出了一个更实用的综合方法：只将靠近主瓣的前若干对副瓣控制在指定电平，而让更远的副瓣按照 1/u 的规律自然衰减。这种方法的阵列激励分布更加平滑，并且

  • 避免了Dolph-Chebyshev阵列中激励幅度在边缘的"突起"现象
  • 实际可实现性更好，对馈电网络的幅度误差不那么敏感
  • 方向图在远端以 1/u 衰减，更接近真实物理天线的行为

Taylor综合中有一个参数 n̄（n-bar），它控制着有多少个近端副瓣被"压低"到指定电平。n̄ 越大，方向图越接近Dolph-Chebyshev（但激励幅度的边缘突变更严重）；n̄ 越小，越接近均匀分布。设计者需要根据实际系统需求在两者之间取折中。

Taylor综合法广泛应用于雷达天线和卫星通信天线设计中——这些场景要求低近端副瓣以抑制杂波或邻星干扰，但对远端副瓣不太敏感。

17.4 傅里叶综合法与Woodward-Lawson法

Woodward-Lawson采样法

Woodward-Lawson法（1950年代提出）是一种直观的波束赋形方法，特别适用于形状波束（Shaped Beam）的设计：

  1. 在角度域以特定间隔采样目标方向图
  2. 每个采样点对应一个"方向性基函数"——即一个指向该方向且宽度为 2π/(Nd) 的 sinc 波束
  3. 将所有 sinc 波束按采样值加权叠加，即可重构目标方向图

每个sinc波束的权值就是目标方向图在该采样点处的幅值。这种方法实现简单、物理直观，但缺点是合成的方向图在采样点之间可能有较大波纹，且需要大量单元（N较大）才能精确重构复杂形状。

Woodward-Lawson法常用于设计余割平方波束（Cosecant-squared Beam）（用于搜索雷达，使地面目标在不同距离处接收到的功率恒定）、平顶波束（Flat-top Beam）（用于卫星通信的全球覆盖或区域覆盖天线）等。

FFT综合法

对于大型平面阵列（如64×64单元），Woodward-Lawson的时域采样思路可以推广到二维，并借助快速傅里叶变换（FFT）高效实现。阵因子与激励系数是一对傅里叶变换关系，因此：

  • 正向：激励系数 → FFT → 方向图
  • 反向：目标方向图 → IFFT → 激励系数

FFT综合的优点是计算效率极高，能处理超大型阵列。缺点是IFFT得到的激励系数可能不满足工程约束（如最大幅度动态范围、相位量化精度等），需要迭代修正。

Python实现：Dolph-Chebyshev与Taylor综合

import numpy as np
import matplotlib.pyplot as plt

def dolph_chebyshev(N, SLL_dB):
    """计算Dolph-Chebyshev阵列的激励幅度（N为偶数）"""
    R = 10**(SLL_dB / 20)
    x0 = np.cosh(np.arccosh(R) / (N - 1))

    # 计算激励幅度（利用Chebyshev多项式的根）
    n = np.arange(1, N//2 + 1)
    # 通过多项式根计算幅度系数
    I = np.ones(N//2)
    for i in range(1, N//2 + 1):
        s = 0
        for m in range(1, N//2 + 1):
            t1 = (N - 1) * np.arccos(x0 * np.cos((2*m - 1) * np.pi / (2*N)))
            s += np.cos(2 * np.pi * (i - 0.5) * m / N) * np.cos(t1)
        I[i-1] = s
    I = I / np.max(I)
    # 对称排列
    w = np.concatenate([I[::-1], I])
    return w

def taylor_nbar(N, SLL_dB, nbar):
    """计算Taylor n-bar分布的激励幅度"""
    R = 10**(SLL_dB / 20)
    A = np.arccosh(R) / np.pi

    # Taylor零点位置
    sigma = nbar / np.sqrt(A**2 + (nbar - 0.5)**2)
    mu = np.arange(1, N)
    zn = np.zeros(N - 1)
    for n in range(1, N):
        if n <= nbar:
            zn[n-1] = sigma * np.sqrt(A**2 + (n - 0.5)**2)
        else:
            zn[n-1] = n

    # 计算激励（通过零点乘积）
    p = np.arange(N)
    I = np.zeros(N)
    for m in range(N):
        I[m] = 1
        for n in range(N - 1):
            if zn[n] != 0 and m > 0:
                I[m] *= (1 - (m**2 / zn[n]**2))
    I = I / np.max(I)
    return I

# 示例：16单元，-30 dB副瓣，Taylor nbar=4
N, SLL = 16, -30
w_dolph = dolph_chebyshev(N, SLL)
w_taylor = taylor_nbar(N, SLL, nbar=4)

theta = np.linspace(0, np.pi, 1000)
k, d = 2*np.pi, 0.5
AF_dolph = np.abs(np.sum(w_dolph[:, None] *
    np.exp(1j * np.arange(N)[:, None] * k * d * np.cos(theta)), axis=0))
AF_taylor = np.abs(np.sum(w_taylor[:, None] *
    np.exp(1j * np.arange(N)[:, None] * k * d * np.cos(theta)), axis=0))

# 归一化到dB
AF_dolph_dB = 20 * np.log10(AF_dolph / np.max(AF_dolph) + 1e-12)
AF_taylor_dB = 20 * np.log10(AF_taylor / np.max(AF_taylor) + 1e-12)

plt.figure(figsize=(10, 5))
plt.plot(np.degrees(theta), AF_dolph_dB, 'b-', label='Dolph-Chebyshev')
plt.plot(np.degrees(theta), AF_taylor_dB, 'r--', label='Taylor n̄=4')
plt.axhline(-30, color='gray', linestyle=':', label='目标SLL')
plt.ylim(-50, 0); plt.xlim(0, 180)
plt.xlabel('角度 (度)'); plt.ylabel('归一化方向图 (dB)')
plt.title(f'Dolph-Chebyshev vs Taylor 综合 (N={N}, SLL={SLL} dB)')
plt.legend(); plt.grid(True); plt.show()
代码解读： 上面这段Python代码实现了Dolph-Chebyshev和Taylor两种综合方法。注意Taylor综合的激励幅度比Chebyshev更平滑（没有边缘突起），代价是远端副瓣衰减更快但近端副瓣略高。你可以调整 nbar 参数观察方向图的变化——nbar越小，方向图越接近均匀分布；nbar越大，方向图越接近Chebyshev。

17.5 自适应波束赋形与零点控制

在干扰方向放置零点

前面的综合方法都是"开环"的——预先计算好激励系数，使用时固定不变。但在实际场景中，干扰环境是动态变化的。这时我们需要自适应波束赋形（Adaptive Beamforming）：根据接收到的信号，实时调整阵列权值，在干扰方向自动形成零点，同时在期望信号方向保持高增益。

Schelkunoff单位圆表示法

理解零点控制的数学工具是Schelkunoff单位圆表示法。它将阵因子多项式在单位圆上表示：

AF(z) = Σ_{n=0}^{N-1} w_n · z^n,  其中 z = e^{j·k·d·cosθ}

多项式的 N-1 个根（零点）在单位圆上的位置决定了方向图的零点角度。通过在单位圆上移动零点位置，我们可以精确控制方向图零点。

这个框架给设计者提供了强大的直觉：

  • 在单位圆上均匀分布的零点 → 均匀分布/低副瓣方向图
  • 将零点聚集在某一区域 → 在该区域形成"零陷"（nulling region）
  • 在单位圆特定角度放置一对共轭零点 → 在对应方向上形成深零点

MVDR与LCMV波束赋形器

MVDR（Minimum Variance Distortionless Response，最小方差无失真响应）波束赋形器（又称Capon波束赋形器）是自适应波束赋形的经典方法。它在保证期望信号方向增益不变的约束下，最小化输出功率（即最小化干扰加噪声功率）：

min w^H R w,   subject to w^H a(θ₀) = 1

其中 R 是接收信号的协方差矩阵，a(θ₀) 是期望方向的导向矢量。解为：

w = R^{-1} a(θ₀) / (a(θ₀)^H R^{-1} a(θ₀))

LCMV（Linearly Constrained Minimum Variance，线性约束最小方差）是MVDR的推广，允许多个线性约束（如多个主瓣方向约束、多个零点方向约束）。

凸优化方法

现代阵列综合越来越倾向于使用凸优化（Convex Optimization）方法，特别是利用CVX等工具包。凸优化的优势在于：

  • 可以灵活添加任意约束（如副瓣包络、零点深度、激励幅度动态范围）
  • 保证找到全局最优解（如果问题是凸的）
  • 可以处理非均匀阵列、共形阵列等复杂几何
知识迁移： 从经典综合（Chebyshev/Taylor）到自适应波束赋形（MVDR/LCMV）再到凸优化综合，体现了天线阵列设计从"闭式解析解"到"数值优化"的演进。今天的阵列设计者需要同时掌握经典综合的数学之美和现代优化的计算能力——两者互为补充，而不是谁取代谁。

本章我们系统地学习了天线阵列综合的核心方法：从Dolph-Chebyshev的最优低副瓣综合，到Taylor的实用化改进；从Woodward-Lawson的形状波束赋形，到自适应波束赋形和凸优化方法。我们还通过Python代码实现了两种经典的阵列综合算法。阵列综合告诉我们"给多少个单元，每个单元应该用多大的幅度和相位"才能实现目标方向图。

综合出激励系数之后，下一步是如何在硬件中实现这些幅度和相位控制——尤其是相位的动态调节。这就需要我们将目光投向相控阵天线（Phased Array Antenna）技术。第18章将深入相控阵的硬件核心：移相器、T/R组件、数字波束形成，以及支撑5G通信的Massive MIMO有源天线系统。我们一起看看，如何将第17章综合出来的理论权值，变成真实工作的硬件系统。
