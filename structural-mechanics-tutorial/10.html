引言：换一个角度看世界

同学们好，力法的旅程暂告一段落。力法教会了我们一个经典的策略：把超静定问题转化为静定问题，再用变形协调条件来求解。但不知道你们有没有想过一个问题——力法用起来其实很"累"：每次都要画好几组单位弯矩图，然后用图乘法算一大堆柔度系数。超静定次数一高（比如5次以上），手算量就大到让人想放弃。

但如果我们换一个角度呢？前面我们一直在问：多余约束力是多少？现在换一个问题：在荷载作用下，结构的结点会发生多少位移？如果我们能求出所有结点的位移——每一根杆件的两端位移都知道了——那杆件的内力是不是也就知道了？毕竟，有了位移就能算应变，有了应变就能算应力，有了应力就能算内力。这就是位移法（Displacement Method）的核心思想。

我在工作第一年的时候，参与了一个大型商业综合体项目的设计复核。主结构是一个6层钢框架，用SAP2000计算。当时有一位老总工来检查我们的计算结果。他没有看那些花花绿绿的应力比云图，而是直接问："首层角柱的层间位移是多少？"我翻了一下报告，说："2.8cm。"他点点头，又问："二层那根大跨度梁端的转角呢？"我愣住了——我从没注意过那个数据。老总工说了一句话让我至今印象深刻："位移是结构行为的'指纹'——内力的规律可以被复杂化，但位移的分布永远直接反映结构的刚度缺陷。"他通过位移的结果，很快就指出我们模型中几处刚度设置不合理的地方。

这件事让我意识到：位移不止是一个计算结果——它是一个更直观、更本质的结构行为指标。位移法之所以重要，不只是因为它能求解超静定结构，更因为它揭示了"位移-内力"之间的因果关系。

10.1 位移法的基本思想

位移法的出发点和力法截然相反：

对比 力法 位移法
基本未知量 多余约束力（力） 结点位移（位移）
基本体系 去掉多余约束的静定结构 施加附加约束（阻止所有结点位移）的结构
建立条件 变形协调条件 静力平衡条件
方程意义 位移为零（或已知值） 附加约束上的反力为零
适用场景 低次超静定手算 高次超静定、易程序化

位移法的基本思路可以概括为三个步骤：

  1. 锁定——在所有可以产生结点位移的地方施加附加约束（附加刚臂阻止转动、附加链杆阻止平动），使所有结点位移为零。此时结构变成一组独立的"单跨超静定梁"。
  2. 加载——施加实际荷载，此时附加约束上会产生反力（或反力矩）。
  3. 放松——让结点产生实际的位移，使附加约束上的反力恰好被"抵消"——即附加约束上的总反力为零。这给出了位移法方程。
一个形象的类比：想象一个转角的会议室门。你用手把门推到一个角度（位移），然后门上的弹簧合页（刚度）会产生一个与你手力相反的力矩。位移法解决的问题就是：在给定的外荷载下，门的最终平衡位置（位移）在哪里？位移法方程的本质就是"结点平衡"——每个结点上的所有杆端力矩加起来为零。

10.2 基本未知量的确定

位移法的基本未知量是独立的结点位移，包括：

  • 结点角位移（Node Rotation）——每个刚结点有一个未知的转角。注意：铰结点和定向支座的转角不是独立未知量（它们由杆件变形自动确定）。
  • 结点线位移（Node Translation）——每个结点可能有水平位移和竖向位移。但独立的线位移个数需要通过几何分析确定。

确定基本未知量个数的原则：

位移法基本未知量数 = 刚结点数 + 独立线位移数

独立线位移数可以通过"铰化法"确定——把所有的刚结点都变成铰结点，然后判断这个铰结体系的几何可变性。这个铰结体系有多少个自由度，原结构就有多少个独立线位移。这本质上回到了第2章的几何组成分析。

例：一个两层刚架

        ↓↓↓↓              ↓↓↓↓
   ──┬────┬──         ──┬────┬──
     │    │             │    │
   ══╪════╪══         ══╪════╪══
     │    │      →      │    │
     │    │             │    │
   ══╪════╪══         ══╪════╪══
     │    │             │    │
   ──┴────┴──         ──┴────┴──
   固定端              固定端

刚结点：上层2个 + 下层2个 = 4个角位移
独立线位移：
  把刚结点变成铰结点后，上层可以水平滑动
  → 1个独立水平线位移（每层一个）

总基本未知量：
  4（角位移）+ 2（线位移）= 6
简化技巧：如果结构中存在"无穷刚度杆"（EI → ∞），或者某些杆件的轴向变形可以忽略，则相应的线位移自由度会减少。在实际工程中，通常忽略杆件的轴向变形——这意味着杆件长度不变，结点的线位移之间会存在几何约束关系，独立线位移数会减少。

10.3 转角位移方程

位移法的核心工具是转角位移方程（Slope-Deflection Equation）。它建立了杆端位移（两端转角和相对线位移）与杆端内力（杆端弯矩和杆端剪力）之间的关系。

对于一根等截面直杆AB（EI = 常数），在忽略轴向变形的前提下，转角位移方程为：

\[ \begin{aligned} M_{AB} &= \frac{4EI}{l} \theta_A + \frac{2EI}{l} \theta_B - \frac{6EI}{l^2} \Delta_{AB} + M_{AB}^F \\ M_{BA} &= \frac{2EI}{l} \theta_A + \frac{4EI}{l} \theta_B - \frac{6EI}{l^2} \Delta_{AB} + M_{BA}^F \end{aligned} \]

其中：

  • \(M_{AB}\)、\(M_{BA}\)——杆端弯矩（顺时针为正）
  • \(\theta_A\)、\(\theta_B\)——杆端转角（顺时针为正）
  • \(\Delta_{AB}\)——杆件两端垂直于杆轴方向的相对线位移（使杆件顺时针转动时为正）
  • \(M_{AB}^F\)、\(M_{BA}^F\)——固端弯矩（Fixed-End Moment, FEM），即把杆件两端完全固定时，由跨间荷载产生的杆端弯矩

注意上述公式中的正负号约定非常重要：所有角度和弯矩均以顺时针为正。这和材料力学中常用的"使梁下凸为正"的弯矩符号约定不同——务必要注意。

常见荷载的固端弯矩：

荷载类型 图示 \(M_{AB}^F\) \(M_{BA}^F\)
均布荷载 q 两端固定梁受均布q \(-\frac{ql^2}{12}\) \(+\frac{ql^2}{12}\)
跨中集中力 P 两端固定梁跨中受P \(-\frac{Pl}{8}\) \(+\frac{Pl}{8}\)
任意位置集中力P 距离左端a、右端b \(-\frac{Pab^2}{l^2}\) \(+\frac{Pa^2b}{l^2}\)
常见错误提醒：转角位移方程中的正负号是最容易搞混的地方。我读书时有一个同学考试时12分的位移法大题得了0分——他方程列对了，代入的数字也对了，但所有正负号都反了。为什么？因为他把"顺时针为正"的约定和"弯矩图绘在受拉侧"的习惯混在了一起。请记住：位移法计算时用的正负号约定（顺时针为正）和画弯矩图时的约定（画在受拉侧）是两个不同的"坐标系"。计算时用计算规则，画图时用画图规则——不要混为一谈。

10.4 基本体系与位移法方程

与力法类似，位移法也有基本体系的概念。位移法的基本体系是：在原结构的所有独立结点位移方向上施加附加约束，使所有结点位移为零。这样，原结构被分解为一组独立的单跨超静定梁。

在基本体系上施加荷载后，每个附加约束上都会产生反力（或反力矩），记为 \(R_{1P}, R_{2P}, \dots, R_{nP}\)。让结点分别产生单位位移 \(Z_1=1, Z_2=1, \dots, Z_n=1\)，可以得到刚度系数 \(r_{ij}\)——表示由 \(Z_j=1\) 引起的在第i个附加约束方向上的反力。

\[ \begin{cases} r_{11}Z_1 + r_{12}Z_2 + \cdots + r_{1n}Z_n + R_{1P} = 0 \\ r_{21}Z_1 + r_{22}Z_2 + \cdots + r_{2n}Z_n + R_{2P} = 0 \\ \vdots \\ r_{n1}Z_1 + r_{n2}Z_2 + \cdots + r_{nn}Z_n + R_{nP} = 0 \end{cases} \]

这就是位移法典型方程（Canonical Equations of Displacement Method）。注意：

  • \(r_{ij}\)——刚度系数（Stiffness Coefficient），\(r_{ii} > 0\)，且 \(r_{ij} = r_{ji}\)（反力互等定理）
  • \(R_{iP}\)——荷载在基本体系中引起的附加约束反力
  • 方程的物理意义是：每个附加约束上的总反力为零——结点平衡
力法与位移法的对偶性：注意到没有？力法方程是 \(\delta_{ij}X_j + \Delta_{iP} = 0\)，位移法方程是 \(r_{ij}Z_j + R_{iP} = 0\)。它们的形式一模一样，只是"力"与"位移"互换了位置。\(\delta\)（柔度系数）和 \(r\)（刚度系数）在数学上是互逆矩阵的关系——\([r] = [\delta]^{-1}\)。这正是结构力学中著名的对偶原理（Duality Principle）。

10.5 无侧移刚架的计算

所谓无侧移刚架（Non-sway Frame），是指没有水平结点位移的刚架。这种刚架只有结点转角这一种基本未知量，是最简单的位移法应用场景。

无侧移刚架的典型特征：

  • 对称结构在对称荷载作用下（水平位移为零）；
  • 有足够的侧向支撑（如剪力墙、支撑体系）；
  • 柱两端都有水平约束。

计算步骤如下：

  1. 确定基本未知量（就是所有刚结点的转角）；
  2. 写出每根杆件的转角位移方程（含固端弯矩）；
  3. 对每个刚结点列力矩平衡方程（\(\sum M = 0\)）；
  4. 解方程组得到结点转角；
  5. 回代到转角位移方程，得到各杆端弯矩；
  6. 画弯矩图。
例：无侧移刚架（所有横梁受均布荷载 q）

  q ↓↓↓↓↓↓       q ↓↓↓↓↓↓
  ──┬────┬──   ──┬────┬──
    │    │       │    │
  ══╪════╪══   ══╪════╪══
    │    │       │    │
    │    │       │    │
  ══╪════╪══   ══╪════╪══
    │    │       │    │
  ──┴────┴──   ──┴────┴──

  所有柱子的上下两端都没有水平位移，
  只有横梁两端有转角位移。

  基本未知量：横梁与柱连接处的结点转角（共4个角位移）

  对每个刚结点列力矩平衡方程：
  结点B: M_{BA} + M_{BE} = 0
  结点C: M_{CD} + M_{CF} = 0
  结点E: M_{EB} + M_{EG} + M_{EF} = 0
  结点F: M_{FC} + M_{FH} + M_{FE} = 0

  其中每个 M 都用转角位移方程表达，
  代入具体参数后解4个线性方程即可。
小技巧：对于无侧移刚架，如果横梁的线刚度（EI/l）远大于柱子（比如大10倍以上），横梁两端的转角会非常接近，近似为零。这时可以粗略地将横梁两端视为固定端，按单跨梁计算。这种近似估计法在初步设计阶段非常有用——手算可以快速给出大致的内力数据。

本章小结与下章预告

本章我们学习了位移法的基础内容：

  • 位移法的基本思想——以结点位移为基本未知量，通过结点平衡建立方程；
  • 基本未知量的确定——刚结点数 + 独立线位移数；
  • 转角位移方程的推导——\(M = f(\theta, \Delta) + M^F\)，这是位移法的"本构关系"；
  • 位移法典型方程——\(r_{ij}Z_j + R_{iP} = 0\)，其物理意义是结点平衡；
  • 无侧移刚架的计算——只有结点转角的最简情形。

位移法打开了结构分析的"新世界"——你不再被超静定次数所束缚。一根再复杂的刚架，只要它的结点位移是有限的，位移法就能处理。更重要的是，位移法的思路更接近计算机的思维方式——"先确定所有自由度，再建立方程统一求解"。这正是我们后面要学的矩阵位移法的雏形。
