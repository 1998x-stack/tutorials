第17章 协整与误差修正模型 (VECM)

想象一下遛狗的场景：人和狗各走各的路线，但狗绳始终把他们连在一起。人走远了，狗会被拉回来；狗跑远了，人也会被拽过去。他们之间的距离会围绕某个均值来回波动，但永远不会无限分离。

这就是协整（Cointegration）的核心思想——两个或多个非平稳时间序列各自随机漫步，但它们之间存在某种长期均衡关系，把它们"拴"在了一起。这是时间序列分析中最重要也最容易被误解的概念之一。

本章核心目标：理解协整的经济含义，掌握 Engle-Granger 两步法和 Johansen 检验，能用 VECM 同时建模短期动态和长期均衡。

17.1 伪回归的教训

1974年，两位统计学家 Clive Granger 和 Paul Newbold 做了一个著名的实验：他们生成了两个完全独立的随机游走序列，然后对它们做回归。结果显示：t 统计量在 5% 水平上显著的比例高达 75% 左右！

换句话说，两个毫无关系的时间序列，仅仅因为都是非平稳的，就在传统回归分析中表现出"高度显著"的关系——这就是经典的伪回归（Spurious Regression）问题。

⚠ 伪回归的警示：当两个独立非平稳序列回归时，R² 很高、t 统计量显著，但 DW 统计量接近零（残差高度自相关）。这是识别伪回归的快速信号。千万别看到高 R² 就以为发现了什么重要关系！

但如果两个非平稳序列的线性组合变成了平稳序列呢？那它们就是协整的。协整为"伪回归困局"提供了一条出路：当变量协整时，回归结果是可信的，因为残差是平稳的。

17.2 协整的直观理解

从数学上说，如果两个序列 x_t 和 y_t 都是 I(1) 单整（一阶差分后平稳），且存在某个系数 β 使 y_t - β·x_t 是平稳的 I(0)，那么 x_t 和 y_t 就是协整的。

现实中的经典例子：

协整关系直觉
短期利率 ↔ 长期利率利差不会无限扩大，否则套利机会出现
股价 A ↔ 股价 B（同一行业）基本面相似的股票价差应围绕合理值波动
消费 ↔ 收入长期来看消费不会持续偏离收入水平
即期汇率 ↔ 远期汇率抛补利率平价确保二者不会长期脱钩

17.3 Engle-Granger 两步法

1987年，Engle 和 Granger 提出了最直观的协整检验方法，分两步走：

第一步：估计长期均衡关系

y_t = α + β·x_t + e_t

用 OLS 估计出 β，然后计算残差序列 e_t = y_t - α - β·x_t

第二步：检验残差的平稳性

对残差 e_t 做 ADF 检验。
如果 e_t 是平稳的 —— x_t 和 y_t 协整。
如果 e_t 有单位根 —— 伪回归，不存在协整。
注意：第二步的 ADF 检验不能使用标准临界值，因为残差是从第一步估计中生成的（生成的回归量问题），需要使用 Engle-Granger 专门的临界值表。Python 的 statsmodels 会自动处理这一点。 
import statsmodels.api as sm
from statsmodels.tsa.stattools import adfuller
import numpy as np

# 假设有两个价格序列
# 第一步：估计长期关系
x = sm.add_constant(price_b)
result_ols = sm.OLS(price_a, x).fit()
residuals = result_ols.resid

# 第二步：检验残差的平稳性
adf_result = adfuller(residuals, autolag='AIC')
print(f"ADF 统计量: {adf_result[0]:.4f}")
print(f"p 值: {adf_result[1]:.4f}")

if adf_result[1] < 0.05:
    print("残差平稳 → 存在协整关系")
else:
    print("残差有单位根 → 不存在协整")

17.4 Johansen 检验

Engle-Granger 方法有两个局限：第一，只能处理两个变量；第二，不能处理多个协整关系（系统中可能存在不止一个协整向量）。Johansen 检验解决了这两个问题。

Johansen 检验在 VAR 框架下进行，使用两种检验统计量：

  • 迹检验（Trace Test）：H0: 最多有 r 个协整向量 ↔ H1: 有大于 r 个
  • 最大特征值检验（Max Eigenvalue Test）：H0: 有 r 个 ↔ H1: 有 r+1 个
from statsmodels.tsa.vector_ar.vecm import coint_johansen

# Johansen 检验
johansen_result = coint_johansen(df, det_order=0, k_ar_diff=1)
print("特征值:", johansen_result.lr1)
print("迹统计量:", johansen_result.lr1)
print("5% 临界值:", johansen_result.cvt)

# 判断协整秩
rank = None
for r in range(len(johansen_result.lr1)):
    if johansen_result.lr1[r] > johansen_result.cvt[r, 1]:
        rank = r + 1
print(f"估计协整秩: r = {rank}")

协整秩 r 就是协整关系的数量。例如 r=0 表示没有协整关系，r=1 表示存在一个线性组合使变量平稳，r=2 表示存在两个独立的协整关系。

17.5 VECM：当协整遇上 VAR

向量误差修正模型（Vector Error Correction Model, VECM）把 VAR 和协整统一在了一起。它同时包含两个部分：

  • 长期均衡项（误差修正项）：上一期的偏离被下一期修正回来
  • 短期动态项：各变量的差分滞后项

VECM 的一般形式：

Δy_t = α·EC_{t-1} + Γ₁·Δy_{t-1} + ... + Γ_{p-1}·Δy_{t-p+1} + ε_t

其中 EC_{t-1} = y_{t-1} - β·x_{t-1} 是上一期的偏离
α 是调整速度（有多快回到均衡）
Γ_i 是短期动态系数
from statsmodels.tsa.vector_ar.vecm import VECM

# 拟合 VECM (需要先知道协整秩)
vecm_model = VECM(df, k_ar_diff=2, coint_rank=1)
vecm_result = vecm_model.fit()

print(vecm_result.summary())

# 查看误差修正系数
alpha = vecm_result.alpha  # 调整速度
print("调整速度 α:", alpha)

α 系数是 VECM 中最关键的部分。α 为负值且统计显著，说明当变量偏离均衡时，系统会以多快的速度"拉回来"。α 越接近 -1，修正越快；α 接近 0，则偏离后的恢复非常缓慢。

17.6 实战应用：配对交易

协整在量化金融中最经典的应用就是配对交易（Pairs Trading）。基本思路：

  1. 找到两只协整的股票（如同行业龙头股 A 和 B）
  2. 当它们的价差偏离均值超过阈值（比如 2 倍标准差）时建仓
  3. 做多被低估的那只，做空被高估的那只
  4. 等待价差回归均值时平仓获利
# 配对交易的简单模拟
spread = price_a - coint_coef * price_b
z_score = (spread - spread.mean()) / spread.std()

# 交易信号
entry_threshold = 2.0
long_signal = z_score < -entry_threshold
short_signal = z_score > entry_threshold
要点总结：协整是伪回归的"解药"，它连接了非平稳序列的长期均衡和短期动态。Engle-Granger 法适合两变量系统，Johansen 检验适合多变量。VECM 统一了协整和 VAR，是最完整的分析框架。

17.7 本章小结

本章的核心知识点：

  • 伪回归：独立非平稳序列的回归产生虚假的显著关系
  • 协整：非平稳序列的线性组合变成平稳序列，意味着长期均衡
  • Engle-Granger 两步法：先回归再检验残差平稳性
  • Johansen 检验：处理多变量和多协整关系
  • VECM = 协整 + VAR，同时建模长期修正和短期动态

Granger 和 Newbold 的"伪回归"发现彻底改变了时间序列计量经济学，Granger 本人也因此获得了 2003 年诺贝尔经济学奖。协整和 VECM 为处理非平稳多变量系统提供了严谨的框架。下一章我们将彻底换一个视角——从状态空间模型和卡尔曼滤波出发，将之前学到的 ARIMA、指数平滑等模型统一到一个更通用的框架中。
