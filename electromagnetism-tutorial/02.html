大家好，我是你们的讲师。"电"这个字，中文来自"闪电"，而在希腊语中它叫 ηλεκτρον（elektron），意思是琥珀。古希腊哲学家泰勒斯在公元前600年左右发现，用毛皮摩擦过的琥珀能吸引羽毛、木屑等轻小物体。这是人类最早关于电现象的记载。两千多年后，我们终于系统性地理解了这种神秘的力量——从库仑定律开始。

电荷的原子论基础

物质为什么会带电？答案藏在原子内部。原子由带正电的原子核和带负电的电子组成。通常原子中正负电荷数量相等，对外呈电中性。当两个物体摩擦时，电子可能从一个物体转移到另一个物体——得到电子的物体带负电，失去电子的物体带正电。这就是摩擦起电的本质：不是创造了电荷，而是电荷发生了转移。

根据材料导电能力的强弱，我们可以将它们分为三类：

  • 导体（Conductor）：如铜、铝、金——内部有大量自由电子，电荷可以在其中自由移动。
  • 绝缘体（Insulator）：如橡胶、玻璃、塑料——电子被束缚在原子附近，几乎不能移动。
  • 半导体（Semiconductor）：如硅、锗——导电能力介于导体和绝缘体之间，可通过掺杂精确调控。

电荷有一个极其重要的性质：电荷是量子化的。任何带电体的电荷量都是元电荷 e 的整数倍：

q = ne,   n = 0, ±1, ±2, ...

其中元电荷 e = 1.602 × 10⁻¹⁹ C（库仑）。这是罗伯特·密立根在1909年通过著名的油滴实验测定的。质子的电荷为 +e，电子的电荷为 −e。一个库仑大约对应 6.24 × 10¹⁸ 个电子的总电荷量——这个数字大到难以想象，这也是为什么我们日常中很少感受到静电的惊人力量：你身上的静电荷通常只有微库仑（µC）量级，但其中已经包含了天文数字般的电子数量。

🔑 核心洞察：库仑——一个极大的单位
1 库仑有多大？两个相距 1 米的 1 C 点电荷之间的作用力约为 9 × 10⁹ N，相当于约 90 万吨物体的重量！这就是为什么你从来没见过"1库仑"的静电荷在日常生活中出现。闪电中转移的电荷量也只有 15-30 C 左右，但已经足以产生毁灭性的能量。我们日常静电实验中涉及的电荷量通常在纳库仑（nC）到微库仑（µC）量级。

库仑定律

1785年，法国物理学家查尔斯-奥古斯丁·库仑（Charles-Augustin de Coulomb）用他发明的扭秤（Torsion Balance）精确测量了两个带电小球之间的作用力。实验原理非常巧妙：将一个小球固定在扭秤的一端，另一个带电小球靠近它，通过测量悬丝扭转的角度来推算力的大小。库仑发现：

F = k · |q₁q₂| / r²

其中比例常数 k ≈ 8.988 × 10⁹ N·m²/C²。在SI单位制中，我们通常将 k 写作 1/(4πε₀)，其中 ε₀ = 8.854 × 10⁻¹² C²/(N·m²) 称为真空介电常数。所以库仑定律最常见的写法是：

F = (1/4πε₀) · q₁q₂ / r² · r̂

其中 r̂ 是从 q₁ 指向 q₂ 的单位矢量，方向决定力的方向——同号电荷相斥，异号电荷相吸。

库仑定律的形式与万有引力定律 F = G·m₁m₂/r² 惊人地相似，都是平方反比律。但两者有本质区别：

性质 万有引力 静电力
公式 F = G·m₁m₂/r² F = k·q₁q₂/r²
比例常数 G = 6.67 × 10⁻¹¹ k = 8.99 × 10⁹
作用 总是引力 引力或斥力
相对强度 弱（~10⁻⁴¹ × 强力） 强（~10⁻² × 强力）
屏蔽 不能屏蔽 可被导体屏蔽

这里有一个至关重要的原理——叠加原理（Superposition Principle）：多个点电荷对某一个点电荷的作用力，等于每个点电荷单独作用时的矢量之和。也就是说，静电力满足线性叠加：

F_total = F₁ + F₂ + ... + F_N

叠加原理看似简单，但它是整个电磁学大厦的基石之一。有了它，我们可以从两个点电荷的简单情况出发，逐步构建任意复杂电荷分布产生的电场。

电场强度

库仑定律描述的是两个电荷之间的"超距作用"。但法拉第提出了一个革命性的概念：电场（Electric Field）。电荷并不直接作用于远处的另一个电荷，而是在它周围的空间中产生电场，电场再对进入其中的电荷施力。这个观点被称作"近距作用"（Action at a distance through field），它彻底改变了物理学的思维方式。

电场强度 E 的定义是：单位正电荷在电场中所受的力：

E = F / q₀

单位是 N/C 或 V/m。有了电场的概念，库仑定律可以重新表述为：一个点电荷 q 在距离 r 处产生的电场为：

E = (1/4πε₀) · q / r² · r̂

注意这是一个矢量场——空间中每一点都有一个矢量，其大小随距离平方衰减，方向从正电荷指向外（或指向负电荷）。

对于连续分布的电荷，我们需要将电荷分布离散化，然后积分：

E(r) = (1/4πε₀) ∫ (ρ(r') / |r − r'|²) · r̂ dV'

其中 ρ 是电荷体密度。类似地，对于线电荷分布（线密度 λ）和面电荷分布（面密度 σ），只需要将体积分分别换为线积分和面积分。这些连续电荷分布的场计算是电磁学的基本功。

# 计算均匀带电圆环轴线上的电场
import numpy as np
import matplotlib.pyplot as plt

epsilon0 = 8.854e-12
k = 1 / (4 * np.pi * epsilon0)

def E_ring_on_axis(R, Q, z):
    """
    均匀带电圆环在轴线上的电场
    R: 圆环半径
    Q: 总电荷量
    z: 轴线上距离圆环平面的距离
    """
    E_z = (k * Q * z) / (z**2 + R**2)**1.5
    return E_z

z_vals = np.linspace(-3, 3, 100)  # 单位：R
Q, R = 1e-6, 1.0
E_vals = E_ring_on_axis(R, Q, z_vals)

plt.plot(z_vals, E_vals)
plt.xlabel('z / R')
plt.ylabel('E(z)')
plt.title('均匀带电圆环轴线电场分布')
plt.grid(True)
plt.show()

电力线与电场可视化

电场是一个抽象的矢量场——你看不见它，但可以用电力线（Electric Field Lines）来可视化。法拉第最早提出了这种直观的表示方法。电力线有以下规则：

  • 从正电荷出发，终止于负电荷。
  • 电力线的切线方向 = 该点的电场方向。
  • 电力线的密度 = 电场强度的大小（线越密，场越强）。
  • 电力线永不交叉。
⚠ 经典踩坑：电力线 = 粒子轨迹？
有一个在电磁学教学中流传甚广的故事：某次考试中，教授出了一道题，问一个带电粒子在点电荷电场中的运动轨迹。一位同学自信满满地画了一条电力线，说粒子就沿电力线运动。结果当然是全错。
原因在于：电力线不是粒子的运动轨迹。电力线给出的是瞬时加速度方向（F = qE, a = F/m），但粒子有速度，它的实际轨迹取决于初速度和力的共同作用。只有在少数特殊情况下（比如粒子初速度为零且电力线是直线），粒子才会沿电力线运动。记住：电力线是"加速度线"，不是"速度线"。

几个经典电场计算

让我们通过几个经典例子来巩固电场计算的基本方法。

1. 均匀带电圆环的轴线电场

半径为 R、总电荷为 Q 的均匀带电圆环，在轴线上距离圆环平面 z 处的电场：

E(z) = (1/4πε₀) · (Q z) / (z² + R²)^{3/2} · ẑ

当 z >> R 时，E ≈ (1/4πε₀) · Q/z² ——退化为点电荷场。当 z = 0 时，E = 0（圆环中心电场为零，对称性使各分量抵消）。在 z = ±R/√2 处电场达到极值。

2. 无限长均匀带电直线

线密度为 λ 的无限长直线，在距离 r 处的电场：

E = (1/2πε₀) · (λ / r) · r̂

注意这里随 r 的一次方反比衰减，而不是平方反比——这是因为线电荷在几何上是"一维无限"的。

3. 均匀带电圆盘

半径为 R、面密度为 σ 的均匀带电圆盘，在轴线上 z 处的电场：

E(z) = (σ / 2ε₀) · [1 − z / √(z² + R²)] · ẑ

当 R → ∞（无限大平面）时，取极限得到：

E = σ / 2ε₀ · n̂

这是一个非常重要的结果：无限大均匀带电平面产生的电场是匀强电场——空间中各点的电场大小和方向都相同，与距离无关！这个结果将在下一章用高斯定理更简洁地导出。

💡 匀强电场的秘密
你可能会觉得奇怪：为什么无限大平面的电场与距离无关？想象你站在这个平面附近：当你离远时，单个电荷元的贡献确实按平方反比衰减，但你"看到"的电荷元数量增加了（你的视角覆盖了更大的立体角）。衰减和增加恰好抵消，结果是与距离无关的恒定电场。这也是平行板电容器内部为什么近似是匀强电场的原因——只要板尺寸远大于板间距，边缘效应可以忽略。

库仑定律的局限性——引出高斯定理

库仑定律是静电学的基础，但它有一个实际困难：对复杂电荷分布做矢量积分往往极其繁琐。回想一下，即使是一个简单的均匀带电圆盘，我们需要在圆盘上做二重积分。对于具有对称性的问题，是否有更简洁的方法？

答案是肯定的。高斯定理（Gauss's Law）是库仑定律的积分形式，它把电通量与闭合曲面内的总电荷量联系起来。在某些对称性条件下，高斯定理能将三维积分简化为简单的代数运算。下一章，我们将深入探讨高斯定理以及与之密切相关的电势概念——从"力"的视角转向"能量"的视角，你会发现电磁学的风景截然不同。
