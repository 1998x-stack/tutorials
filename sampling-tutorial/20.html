第20章 小域估计

大家好，我是你们的讲师。想象这样一个场景：你正在为一个全国性调查设计抽样方案。样本量是1万个家庭，目标是估计全国平均收入——这没什么问题，1万个样本足够用了。但紧接着客户提出了一个需求："我们还需要每个县的估计。" 全中国有近3000个县级行政区，1万个样本平均分到每个县大约只有3-4个样本。天哪，3个样本怎么可能估计一个县的收入水平？这就是本章要解决的问题——小域估计（Small Area Estimation, SAE）。

小域估计不是一个小众的学术话题，它直接关系到数百亿美元的政府资金分配。美国人口普查局每年发布的小域贫困率估计（SAIPE项目）直接影响联邦政府向各县级地区拨付的教育、医疗等专项资金。如果这些估计不准，就意味着贫困县可能拿不到应得的资金，或者富裕县拿到过多的钱。这就是小域估计的实际分量——它不只是一个统计方法，更关乎社会公平和公共资源的合理分配。

核心矛盾：小域估计面对的根本矛盾是——域越小，越需要精确的估计；但域越小，可用的直接样本越少（因而直接估计的方差越大）。要解决这个矛盾，必须放弃"只用本域数据"的固有思路，转而"借用"相关信息和模型的力量。

小域问题与直接估计的困境

在正式讨论方法之前，我们先把"小域"的定义搞清楚。"域"（Domain / Area）是你希望做出估计的任意子总体——可以是一个地理区域（县、街道），也可以是一个人口群体（少数族裔、老年人）。当一个域的样本量不足以支持"直接估计"（即只用该域样本做的估计）时，它就是一个小域。

直接估计的两个问题

对于小域来说，直接估计面临着双重困境：

  • 方差过大：如果某个县只分配到10个样本，那么它的样本均值虽然是无偏的，但标准误会非常大。一个标准误为5000元的收入估计，几乎没有什么实用价值——95%置信区间的宽度接近2万元。
  • 不可靠性：更糟糕的是，有些小域可能完全没有样本（比如在抽样框中没有命中某个少数民族群体）。此时直接估计甚至无法计算。这在多维度交叉分组时非常常见——比如"农村地区、65岁以上、少数民族女性"——几乎必然没有样本。
一个具体例子：美国当前人口调查（CPS）每月调查约6万户家庭，用于估计全国失业率非常可靠。但当你要估计"西弗吉尼亚州某个县的非洲裔男性失业率"时，样本量可能为零。即使有样本，也微不足道。这时CPS的发布策略是：要么不发布这个县的估计，要么使用小域模型来"借力"其他数据源。

间接估计：借力的艺术

既然直接估计不行，那就"借用"信息。间接估计的核心思想是从相关域或全局模型中借用强度（strength）来提高小域估计的精度。

域比率估计（Domain Ratio Estimator）

如果已知小域的某个辅助变量Xi（如上年度的税收数据、人口普查数据），可以利用全样本拟合的比率R = Σy / ΣX来估计小域均值：

域比率估计

        假设全样本中 y 与 X 满足线性关系：
        R = (Σ y) / (Σ X)   （全局比率）

        小域i的均值估计：
        ȳ_i_ratio = R × X̄_i
        其中 X̄_i 是小域i的辅助变量均值（通常来自普查或行政记录）

        精度提升的来源：X̄_i是已知的或可以从大样本精确估计，
        而R的估计利用了所有域的数据（总样本量很大），
        因此方差远小于直接使用小域样本中的y

合成估计（Synthetic Estimator）

合成估计是一种更强力的借力方式：假设所有域的回归关系是相同的，然后利用全局模型来预测每个小域的值。它的偏差风险也相应更高——如果小域之间的差异不能完全由辅助变量解释，合成估计就会产生偏差。

估计方法 偏差 方差 借力程度
直接估计 无偏 极大 不借力
域比率估计 小偏（模型近似成立时） 较小 借用全局比率
合成估计 可能有较大偏差 很小 完全借用全局模型
Fay-Herriot模型 可控偏差 中等 折中——最优结合

Fay-Herriot模型：小域估计的经典之作

1979年，美国人口普查局的统计学家Robert Fay和Roger Herriot发表了一篇里程碑式的论文，提出了后来以他们名字命名的Fay-Herriot模型。这至今仍是小域估计领域中使用最广泛的模型。它本质上是一个两层模型（hierarchical model），巧妙地结合了直接估计和模型预测。

模型结构

Fay-Herriot模型

        第一层（抽样模型）：
        ȳ_i = θ_i + e_i,   e_i ~ N(0, V_i)

        ȳ_i 是小域i的直接估计（可能有大的方差）
        θ_i 是小域i的真实均值（未知）
        V_i 是直接估计的抽样方差（已知，从数据中计算得到）
        e_i 是抽样误差

        第二层（链接模型）：
        θ_i = x_i^T β + u_i,   u_i ~ N(0, σ_u^2)

        x_i 是辅助变量（如普查数据、行政记录）
        β 是固定效应系数
        u_i 是域随机效应，反映辅助变量无法解释的域间差异
        σ_u^2 是域间方差

        合并后的模型：
        ȳ_i = x_i^T β + u_i + e_i

        Fay-Herriot估计（BLUP）：
        θ̂_i_FH = γ_i × ȳ_i + (1 - γ_i) × x_i^T β̂

        其中 γ_i = σ_u^2 / (σ_u^2 + V_i)  被称为"收缩因子"（shrinkage factor）

        当 V_i 很大（小域样本极少）→ γ_i → 0 → 估计更依赖模型预测
        当 V_i 很小（大域样本充足）→ γ_i → 1 → 估计更依赖直接估计
收缩估计的洞察：Fay-Herriot模型产出的估计是直接估计和模型预测的加权平均——权重γ由数据自身决定。对样本量大的域，估计更"信任"直接数据；对样本量小的域，估计更"信任"模型。这种"数据自适应"的收缩（shrinkage）机制正是Fay-Herriot模型的精髓。它自动实现了偏差-方差的权衡（bias-variance tradeoff）：小域虽然带入了少量模型偏差，但方差被大幅压缩，从均方误差（MSE）的角度看整体更优。

实际故事：美国SAIPE项目

美国人口普查局的小域收入与贫困估计（Small Area Income and Poverty Estimates, SAIPE）项目是小域估计最著名的应用之一。它使用Fay-Herriot模型来估计全国每个县和每个学区的贫困率，这些估计直接影响联邦政府每年超过700亿美元的教育和扶贫资金的分配。

SAIPE项目中使用的辅助变量包括：

  • 行政记录数据（个人所得税申报、补充营养援助计划SNAP参与数据）
  • 美国社区调查（ACS）的直接估计及其标准误
  • 人口普查数据（人口构成、住房特征）
  • 上年度的贫困率估计

这个项目的一个关键洞见：即使ACS本身的直接估计对于大部分县来说都太不稳定，但通过Fay-Herriot模型将ACS数据与行政记录数据结合起来，就能得到足够可靠的县级贫困率估计。模型的MSE通常比直接估计的方差小50%~80%，这对于资金分配的公平性来说是巨大的改进。

小域估计的一种常见失败模式——模型失效：我曾经参与过一个小域估计项目，目标是估计某省各市的小微企业存活率。我们用了Fay-Herriot模型，辅助变量包括GDP增长率、税收优惠政策和历史存活率。结果模型在一个工业转型城市上完全失效——因为该市正在进行大规模产业升级，历史数据与现状的关系完全断裂了（模型假设的线性关系不复存在）。这个案例告诉我们：模型越是依赖"借力"，当辅助变量与目标变量的关系被外部冲击改变时，偏差风险就越大。实践中一定要做模型诊断（残差分析、影响诊断）和灵敏度分析。

小域估计的实践流程

在实际项目中实施小域估计，通常遵循以下步骤：

  1. 数据准备：整理小域的直接估计（ȳi）和相应的抽样方差（Vi），收集辅助变量（xi）
  2. 模型拟合：使用最大似然（ML）或限制最大似然（REML）估计模型参数（β, σu²）
  3. 模型诊断：检查残差、异常域、影响点。计算条件数、拟合优度等诊断指标
  4. 计算估计量：对每个小域计算Fay-Herriot估计θ̂i,FH
  5. 计算MSE：估计每个小域的均方误差（MSE），这是评估估计可靠性和报告置信区间的基础
  6. 结果验证：如果可能，将聚合到更高层级的小域估计与可靠的外部数据做比较验证
# R：使用sae包进行Fay-Herriot模型拟合（伪代码示意）

实操建议：在小域估计中，"辅助变量的选择"是决定模型成败的关键。好的辅助变量应该满足：(1) 与目标变量强相关；(2) 对所有小域都可用（包括普查数据或行政记录）；(3) 来自可靠的大型数据源（避免了小域样本问题）。此外，辅助变量最好与目标变量同一年份或相近——滞后太久的辅助变量会降低模型质量。常用的辅助变量包括上一轮普查数据、税收记录、社保参保数据等。

本章小结

小域估计是抽样理论中一个非常"实用"的分支——它解决了一个几乎所有大规模调查都会遇到的问题：客户想要子总体的估计，而子总体的样本量又远远不够。Fay-Herriot模型通过巧妙的收缩估计机制，将直接估计（高方差、可能无偏）与模型预测（低方差、可能有偏）加权结合，在偏差-方差之间取得了最优平衡。在这个领域，辅助数据的质量和模型假设的合理性比样本量本身更关键。小域估计的精髓可以归结为一句话：在承认直接数据不够用的前提下，用模型的力量来弥补。

下一章我们将转向另一个完全不同的抽样策略——双重抽样与捕获-再捕获（Two-phase Sampling and Capture-Recapture）。这两种方法都涉及"抽两遍"的思路，但应用场景截然不同。它们将帮助我们处理那些无法直接枚举总体或需要修正抽样框缺陷的复杂情况。
