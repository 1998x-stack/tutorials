大家好，我是你们的讲师。先问大家一个问题：你花了两周仿真的天线，方向图仿真结果增益高达12 dBi，副瓣低于-25 dB——但你凭什么相信这个结果？

在微波工程领域流传着一句话："如果你没有测量过，你只是在猜。"电磁仿真软件（HFSS、CST）基于数值算法求解Maxwell方程组，理论上精度很高——但前提是你正确地设置了材料参数、边界条件、端口激励和网格剖分。一旦有一项设置不准确（比如材料的介电常数实际值比手册标称值高了0.2，或者SMA接头的寄生效应没有被建模），仿真和实测之间就可能出现显著差异。

天线测量是天线工程的"照妖镜"——你设计得好不好，测量结果说了算。本章将系统地介绍天线测量的核心方法，从阻抗测量到方向图、增益和极化测量，以及各类测试环境的优缺点。

16.1 天线测量的核心挑战

你没法把30米的短波天线放进暗室

天线测量面临一个根本性困境：天线的工作环境与测量环境不同。好的测量应该能排除环境的影响，只测量天线本身的性能。但在实际操作中这极其困难。

举个极端的例子：一个工作于5 MHz（波长60米）的短波通信天线，其辐射体长度可能超过30米。你不可能把它搬进微波暗室——没有这么大的暗室！实际上，HF频段（3-30 MHz）的通信天线绝大多数是在实际场地上进行测量的，测量结果强烈依赖于场地环境（地面电导率、附近建筑物、树木等）。

因此，天线测量的核心挑战可以归纳为三个方面：

  • 物理尺度：低频天线尺寸太大，无法在受控环境中测量
  • 环境去耦：需要将天线本身的性能与测量环境的影响分离开
  • 远场条件：方向图测量需要满足远场距离 d > 2D²/λ，对大口径天线意味着极大的空间需求

天线测量通常分为四大类别：阻抗测量（输入阻抗、VSWR）、辐射方向图测量（幅度和相位随角度变化）、增益测量（绝对和相对增益）和极化测量（轴比、交叉极化鉴别度）。

16.2 阻抗测量与矢量网络分析仪

VNA基础与校准

矢量网络分析仪（Vector Network Analyzer, VNA）是天馈测量中最核心的仪器。它通过向被测件（DUT）发射已知的射频信号并测量反射和传输信号，精确计算出S参数。

对于单端口测量（如天线输入阻抗），我们关注的是S11（即反射系数）。VNA的主要测量流程如下：

  1. 仪器预热：VNA需要30分钟以上的预热时间以保证内部温度稳定。
  2. 校准（Calibration）：使用校准件（Cal Kit）进行SOLT校准——分别测量Short（短路）、Open（开路）、Load（50 Ω匹配负载）和Thru（直通，仅双端口需要），以消除系统误差。
  3. 连接被测天线：使用高质量稳相电缆连接被测天线，尽量减少电缆弯折。
  4. 测量：读取S11幅度和相位，换算为阻抗、VSWR或回波损耗。
校准的重要性：没有校准的VNA数据其实没有意义。我见过太多初学者跳过校准步骤直接测量——结果S11曲线看起来"挺好的"，但实际上包含了电缆、转接器和测试夹具的全部误差。一次完整的SOLT校准只需3-5分钟，而这5分钟决定了你的数据是"工程数据"还是"垃圾数据"。

去嵌入与时域门

在实际测量中，天线不能直接连接到VNA的校准参考面——中间必然有电缆、转接器甚至测试夹具。这些"额外"的传输线会引入相位偏移和少量损耗。去嵌入（De-embedding）就是将这些寄生效应从测量结果中去除的过程。

时域门（Time-domain Gating）是VNA的一项实用功能。VNA可以在频域测量完成后，通过逆傅里叶变换将数据转换到时域，然后设置一个"时间窗"（门）来选择特定的反射信号，排除其他信号。比如，当你测量一个安装在金属杆上的天线时，VNA可以区分出天线本身的反射和远处金属杆的反射——前者来自你想要的天线，后者是环境造成的干扰。通过时域门选通，你能得到"仿佛在自由空间中"的天线阻抗。

下面是VNA测量的关键参数设置指南：

参数 典型值 目的 注意事项
IF带宽 100 Hz - 1 kHz 控制测量噪声底 越小噪声越低，但扫描越慢
扫描点数 201 - 1601 频率分辨率 窄带测量用201点即可；宽带需更多点
输出功率 -10 ~ 0 dBm 激励天线 功率过大会使有源天线饱和
平均次数 3 - 16 降低随机噪声 每次平均扫描时间加倍
时域门跨度 以天线物理尺寸估算 隔离环境反射 门宽过窄会截断有用信息

16.3 方向图测量方法

远场测量法

最直接的方向图测量方法是远场法：将待测天线（AUT, Antenna Under Test）置于远场距离处，用一个固定的标准源天线照射AUT，然后旋转AUT记录接收功率随角度的变化。

远场距离的判定标准为：

d > 2D² / λ

其中 D 是天线最大口径尺寸，λ 是工作波长。例如，一个1米口径的抛物面天线在10 GHz（λ = 3 cm）时，远场距离为 d > 2 × 1² / 0.03 ≈ 66.7米。

远场法的优点是直截了当——测量结果就是天线在实际远场条件下的表现，无需数学变换。缺点是需要足够大的场地——对于低频段或大尺寸天线，往往难以实现。

高架法（Elevated Range）

当无法在微波暗室内进行远场测量时，工程师会在室外场地使用高架法：将AUT和源天线架设在足够高的塔上，使地面反射波与直射波之间形成足够大的路径差，从而利用天线方向图本身的副瓣电平抑制地面反射的影响。

高架法的关键在于天线高度和间距的设计，通常需要借助几何光学分析来确定最佳架设方案。这种方法受天气影响大（雨、雪、风都会影响测量精度），但却是大型天线（如基站天线、卫星地面站天线）唯一的现场测量方案。

紧缩场法（CATR）

紧缩天线测试场（Compact Antenna Test Range, CATR）是解决远场距离问题的聪明办法。它使用一个大型抛物面反射器（或两个反射器）在近距离（通常几米到十几米）内模拟出一个平面波前，相当于在室内"压缩"了远场条件。

CATR的核心是反射器边缘的锯齿或卷边设计，用于抑制边缘绕射对静区（Quiet Zone）平面波质量的影响。静区的幅度锥削通常要求 < 0.5 dB，相位波纹 < ±5°。CATR的优点是占地面积小、全天候可用，但反射器的加工精度要求极高（表面公差 < λ/100），造价昂贵。

近场扫描法——现代测量的黄金标准

球面近场扫描（Spherical Near-Field Scanning）是目前天线测量的黄金标准。其原理是：在距离天线仅几个波长的近场区域内的一个球面上，用探针扫描测量电磁场的幅度和相位，然后通过近场-远场变换（NF-FF Transform）——本质上是一个基于矢量球谐函数展开的数学算法——计算出天线的远场方向图。

对于高定向天线（如反射面天线、相控阵），则更多使用平面近场扫描（Planar Near-Field Scanning）：探针在一个平面上逐点扫描，通过平面波谱展开（Fourier变换）得到远场方向图。

技术前沿： 现代5G Massive MIMO天线的测量是一个新的挑战——天线有几十到上百个端口，每个端口都需要单独激励并测量其辐射方向图，还要在波束赋形状态下测量EIRP（等效全向辐射功率）和EVM（误差矢量幅度）。这催生了"多探头球面近场系统"——在球面上布置上百个探头同时测量，将测量时间从数小时缩短到几分钟。

16.4 增益测量

双天线法与三天线法

增益测量的基本思路是Friis传输公式：在已知距离和频率的条件下，测量发射和接收功率，然后反推天线增益。但这里有一个"鸡生蛋"的问题：要测量天线A的增益，你需要一个已知增益的天线B作为参考；而要标定B的增益，你又需要另一个已知增益的天线……

双天线法要求两支天线完全相同。由Friis公式可得：

G = (4πd / λ) · √(P_r / P_t)

前提是你必须知道两支天线的增益相等，这通常只在制作了完全相同的两副天线时才成立。

三天线法则巧妙地绕开了这个前提：取三副天线A、B、C，测量全部三对组合（A-B, A-C, B-C）的传输系数，建立三个方程、三个未知数，联立求解即可得到每副天线的绝对增益。这是最基础的绝对增益测量方法。

增益比较法

在实际工程中，最常用的是增益比较法（Gain-Transfer / Gain-Comparison Method）：使用一个标准增益喇叭（Standard-Gain Horn, SGH）作为参考。标准增益喇叭的增益经过精确校准（通常不确定度 < ±0.2 dB）。方法如下：

  1. 将标准喇叭接到信号源端，AUT接到接收端，记录接收功率 Pref
  2. 用AUT替换信号源端的标准喇叭（即两天线互换），记录接收功率 Paut
  3. 计算增益差：Gaut - Gref = 10·log(Paut / Pref)

为什么你的"12 dBi"WiFi天线只有9 dBi？

踩坑故事——消费级天线市场的"虚标文化"： 你在某宝上搜索"高增益WiFi天线"，会看到大量标称12 dBi、15 dBi甚至20 dBi的廉价全向天线。实际测量下来，大多数12 dBi标称的天线真实增益只有8-9 dBi，某些劣质产品甚至不到6 dBi。厂家为何虚标？原因有二：一是大部分消费者没有能力验证增益指标，二是多卖几个dB在心理上让产品显得"性能更强"。有经验的工程师会告诉你：一个长度仅30 cm的垂直全向天线，在2.4 GHz频段物理上不可能达到12 dBi——理论极限就在10 dBi左右，且需要复杂的共线阵列设计。面对这类产品，建议查看实测报告或自己测量确认。

16.5 暗室与测试环境

微波暗室

微波暗室（Anechoic Chamber）是天线测量的黄金环境。暗室的墙壁、天花板和地板上覆盖着吸波材料（Absorber）——通常是碳粉浸渍的锥形泡沫金字塔或铁氧体瓦片。吸波材料的作用是将入射电磁波吸收掉（反射率通常低于-40 dB），从而模拟自由空间条件。

暗室的核心指标是静区（Quiet Zone, QZ）——暗室中央一块区域，在该区域内入射波的幅度和相位不均匀度满足测试要求。一个设计良好的全暗室（fully anechoic chamber）在静区内可实现幅度锥削 < 0.25 dB、相位波纹 < ±5°。

吸波材料的性能与频率密切相关：低频吸波体需要更长的锥体（因为波长更长），所以低频暗室需要更大的空间。一个覆盖100 MHz的暗室可能需要锥体长度超过1.5米，导致暗室整体尺寸极大。

混响室

与暗室模拟"自由空间"的理念相反，混响室（Reverberation Chamber, RC）刻意营造一个多径反射的环境——室内没有吸波材料，金属壁面使电磁波不断反射，形成统计均匀的场分布。混响室特别适合测量移动电话等终端的TRP（Total Radiated Power，总辐射功率）和TIS（Total Isotropic Sensitivity，总全向灵敏度）。

使用混响室测量TRP/TIS比在暗室中测量快得多——不需要扫描角度，一次测量即可得到球面积分结果。3GPP标准中明确规定了使用混响室进行OTA（Over-The-Air）测试的方法。

外场测试

很多通信系统最终要在外场（Outdoor Range）进行验证。外场测量的主要挑战包括：

  • 天气因素：温度变化导致电缆相位漂移，风导致天线指向抖动，雨雪增加路径损耗
  • 地面反射：需要精心选择天线架高和极化以抑制多径
  • 电磁干扰：外场中存在各种无意发射（广播、电视、手机基站）干扰测量

测量不确定度分析是每一个专业天线测量必须完成的功课。对于暗室测量，增益的不确定度通常在 ±0.5 dB 以内（精心校准下可到 ±0.2 dB）。外场测量的不确定度通常在 ±1-2 dB 甚至更高。理解你的测量不确定度，才能对你给出的数据有正确的信心——这不是在找借口，这是科学精神。

用Python分析实测数据

现代天线测量系统通常以Touchstone（.snp）格式或CSV格式导出数据。下面是一个简单的Python示例，读取VNA测量的S11数据并计算VSWR和匹配带宽：

import numpy as np
import matplotlib.pyplot as plt

def s11_to_vswr(s11_mag):
    """将S11幅度转换为VSWR"""
    return (1 + np.abs(s11_mag)) / (1 - np.abs(s11_mag))

# 模拟一组实测S11数据（带噪声和陷波）
freq = np.linspace(2.0, 3.0, 501) * 1e9  # 2-3 GHz
fc = 2.45e9
BW = 80e6
# 理想带通响应 + 噪声
s11_mag = 0.05 + 0.90 / (1 + ((freq - fc) / (BW/2))**2)
s11_mag += np.random.normal(0, 0.02, len(freq))  # 测量噪声
s11_mag = np.clip(s11_mag, 0, 1)

vswr = s11_to_vswr(s11_mag)
rl = -20 * np.log10(np.clip(s11_mag, 1e-6, 1))

# 找到VSWR < 2.0的带宽
below_thresh = vswr < 2.0
if np.any(below_thresh):
    idx = np.where(below_thresh)[0]
    bw_10pct = (freq[idx[-1]] - freq[idx[0]]) / 1e6
    print(f"VSWR < 2.0 带宽: {bw_10pct:.1f} MHz")

# 绘图
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 6))
ax1.plot(freq/1e9, vswr, 'b-')
ax1.axhline(2.0, color='r', linestyle='--')
ax1.set_ylabel('VSWR')
ax1.grid(True)

ax2.plot(freq/1e9, rl, 'g-')
ax2.axhline(10, color='r', linestyle='--')
ax2.set_xlabel('频率 (GHz)')
ax2.set_ylabel('回波损耗 (dB)')
ax2.grid(True)
plt.tight_layout()
plt.show()
工程核心观念： 天线测量不是"插上VNA看结果"那么简单。从校准质量到电缆稳定性，从环境和加载效应到数据处理，每个环节都影响着最终数据的可信度。好的天线工程师不仅会设计天线，更要会"测量天线"——包括理解测量系统的误差源，并能正确解读测量结果。正如前辈们常说的一句"名言"：不做不确定度分析的测量报告，本质上是"一张好看的图"而已。

现在我们已经掌握了单天线设计和测量的能力。但在许多应用场景中，仅仅一个天线是不够的——我们需要多个天线协同工作，精确地控制辐射能量的空间分布。第17章我们将进入天线阵列综合与波束赋形的世界：给定一个目标方向图（比如窄波束、低副瓣、特定形状），我们如何计算出阵列中每个单元的激励幅度和相位？这就是经典而优美的阵列综合理论。
