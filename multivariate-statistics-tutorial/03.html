为什么正态分布无处不在？

大家好，欢迎继续学习。在自然界和人类社会中，正态分布几乎无处不在——人的身高、测量误差、考试分数、金融资产收益率（严格来说是"肥尾"的，但先不计较这个），都近似服从正态分布。为什么？中心极限定理告诉了我们答案：大量独立随机变量的和趋近于正态分布，无论这些变量本身是什么分布。在多元的世界里，事情也是类似的——多元正态分布 (multivariate normal distribution) 是多元统计的"主场分布"。本章我们将深入理解它的定义、性质和在实际分析中的应用。

多元正态分布的定义

一个 p 维随机向量 X 服从均值为 μ、协方差矩阵为 Σ 的多元正态分布，记作 X ~ Np(μ, Σ)，其概率密度函数为：

f(x) = (2π)^{-p/2} |Σ|^{-1/2} exp(-½ (x - μ)ᵀ Σ⁻¹ (x - μ))

这个公式看起来很复杂，但我们来拆解一下：

  • (2π)^{-p/2}：归一化常数，保证积分等于1。p 越大，这个常数越小。
  • |Σ|^{-1/2}：协方差矩阵行列式平方根的倒数。行列式越大，数据越"分散"，密度峰值越低。
  • (x - μ)ᵀ Σ⁻¹ (x - μ)：这就是我们上一章提到的马氏距离的平方。它衡量了 x 到中心 μ 的"统计距离"。

仔细看这个公式：马氏距离是二次型，Σ⁻¹ 起到了"去相关"和"归一化"的作用。如果 Σ = σ²I（即各变量独立同方差），那么马氏距离退化为 (x - μ)ᵀ(x - μ)/σ²，就是欧氏距离平方除以方差。

标准多元正态分布是 μ = 0、Σ = I 的特例，记作 z ~ Np(0, I)。它的密度函数简化成了 p 个独立标准正态变量的乘积：f(z) = (2π)^{-p/2} exp(-½ zᵀz)。这个形式优美而简单——因为各分量独立，联合密度就是边缘密度的乘积。

几何直觉：想象 p=2 的情况。标准二元正态分布的密度函数像一个钟形曲面，其等高线是同心圆。如果 Σ 不是单位矩阵（比如变量间有相关性），等高线就变成了椭圆。椭圆轴的方向由 Σ 的特征向量决定，轴的长度由特征值决定。这个"椭圆"的几何直觉，就是理解 PCA 的关键！

重要性质

多元正态分布之所以在统计中如此核心，是因为它有一系列优美的性质：

性质1：线性变换保持正态性

如果 X ~ Np(μ, Σ)，那么对任意常数矩阵 A（q × p）和向量 b（q × 1），有：

Y = A X + b  ~  N_q(Aμ + b, AΣAᵀ)

这是多元正态分布最重要的性质。它意味着：

  • 每个分量单独看都是一元正态分布（取 A 为行向量）
  • 任意线性组合 aᵀX 是一元正态分布
  • 对数据做任何线性变换（如 PCA 中的投影），结果仍然是正态分布

性质2：零相关等价于独立

对于多元正态分布来说，如果两个分量不相关（协方差为0），那么它们一定独立。这个性质对任意其他分布不一定成立——例如两个 t 分布的变量可以不相关但非独立。但在正态的框架下，"不相关"和"独立"是完全等价的。这大大简化了理论分析。

性质3：边缘分布也是正态的

将 X 分为两部分：X = (X1ᵀ, X2ᵀ)ᵀ，对应的均值向量和协方差矩阵也做相应分割：

μ = (μ₁, μ₂)ᵀ,   Σ = [Σ₁₁  Σ₁₂]
                   [Σ₂₁  Σ₂₂]

则边缘分布为：
X₁ ~ N(μ₁, Σ₁₁)
X₂ ~ N(μ₂, Σ₂₂)

也就是说，任意子向量的分布仍然是多元正态——你只需"取"对应的均值和协方差矩阵的子块。

条件分布与偏相关

这一节是重点中的重点——在给定 X₂ 的条件下，X₁ 的条件分布还是正态的：

X₁ | X₂ = x₂  ~  N(μ₁|₂, Σ₁|₂)

其中：
条件均值：μ₁|₂ = μ₁ + Σ₁₂Σ₂₂⁻¹ (x₂ - μ₂)
条件协方差：Σ₁|₂ = Σ₁₁ - Σ₁₂Σ₂₂⁻¹ Σ₂₁

这组公式在多元统计中怎么强调都不过分。让我们仔细看看：

  • 条件均值 μ₁|₂ 是 x₂ 的线性函数。这意味着在多元正态分布下，最优预测就是线性预测——这正是线性回归的基石！回归系数就是 Σ₁₂Σ₂₂⁻¹。
  • 条件协方差 Σ₁|₂ 与 x₂ 的值无关。这是一个很强的假设：无论 x₂ 取什么值，X₁ 的"不确定程度"都一样。这称为同方差性 (homoscedasticity)。

从条件协方差公式，我们可以引出偏相关系数 (partial correlation coefficient)。偏相关系数衡量的是在消除其他变量影响后，两个变量之间的"净"相关关系。简单相关系数可能因为共同影响者而呈现虚假的相关——比如"冰淇淋销量"和"溺水人数"高度正相关，但控制"气温"这个变量后，两者的偏相关系数接近零。偏相关系数可以从 Σ₁|₂ 中提取：

ρ_{ij·其余} = - (Σ₁|₂)_{ij} / sqrt((Σ₁|₂)_{ii} (Σ₁|₂)_{jj})

或者等价地，从精度矩阵（协方差矩阵的逆）中直接得到。用 . 后面的变量表示"控制了哪些变量"。

应用场景：偏相关在基因网络分析中特别有用。基因表达数据中，成千上万个基因之间存在复杂的相关性。简单相关系数会给出大量"虚假连接"——两个基因看起来相关，但实际上是因为第三个基因调节它们导致的。偏相关系数可以揭示"直接调控关系"，这是构建基因调控网络 (gene regulatory network) 的基础。

极大似然估计

假设我们有 n 个独立同分布的样本 x₁, x₂, ..., xn ~ Np(μ, Σ)。我们要估计参数 μ 和 Σ。

极大似然估计 (MLE) 的结果非常直观：

μ̂ = x̄ = (1/n) Σᵢ₌₁ⁿ xᵢ         （样本均值向量）

Σ̂ = (1/n) Σᵢ₌₁ⁿ (xᵢ - x̄)(xᵢ - x̄)ᵀ  （有偏估计）

无偏估计：S = (1/(n-1)) Σᵢ₌₁ⁿ (xᵢ - x̄)(xᵢ - x̄)ᵀ

其中：

  • x̄ 是 μ 的 MLE 和无偏估计
  • Σ̂ 是 Σ 的 MLE（有偏，分母用 n）
  • S 是 Σ 的无偏估计（分母用 n-1）
  • S 的抽样分布是 Wishart 分布——我们在下一节讲

估计量的抽样分布：

x̄ ~ N(μ, Σ/n)            （均值估计的精度随 n 增大而提高）

(n-1)S ~ W_p(n-1, Σ)     （S 的分布是 Wishart 分布）

x̄ 和 S 是独立的          （这个性质在构建检验统计量时至关重要）
数值隐患：当 p 接近 n 时，样本协方差矩阵 S 的估计误差会非常大。当 p > n 时，S 是奇异的——我们已经在上章末尾的踩坑故事中体验过这一点了。实际问题中，统计学家们建议 n/p 至少大于 5，甚至 10，S 才比较可靠。如果达不到，就需要考虑正则化或收缩估计方法。

Wishart 分布

在一元统计中，样本方差 s² 的抽样分布与 χ² 分布有关：(n-1)s²/σ² ~ χ²(n-1)。在多元统计中，样本协方差矩阵 S 的抽样分布就是 Wishart 分布 (Wishart distribution) 的自然推广。

定义：如果 Z₁, ..., Zm 是独立同分布的 p 维标准正态向量 N(0, Σ)，那么：

W = Σᵢ₌₁ᵐ Zᵢ Zᵢᵀ  ~  W_p(m, Σ)

其中 m 是自由度，Σ 是尺度矩阵。Wishart 分布有几个重要性质：

  • 如果 A ~ Wp(m, Σ)，那么 CACAᵀ ~ Wq(m, CΣCᵀ)——线性变换保持 Wishart 性
  • 期望 E[W] = m Σ
  • 如果两个 Wishart 矩阵独立且自由度可加：W1 + W2 ~ W(m₁+m₂, Σ)
  • 当 p=1 时，Wishart 分布退化为 χ² 分布乘以 σ²

Wishart 分布是多元假设检验的基石——Hotelling T²、Wilks Λ 等检验统计量的分布都可以从 Wishart 分布推导出来。

多元正态性检验

许多多元统计方法（如 Hotelling T²、MANOVA、因子分析）的推导都基于多元正态假设。如果数据严重偏离正态，结论可能不可靠。因此，检验数据是否服从多元正态分布是分析前的重要步骤。

Q-Q 图

最常见的方法。将马氏距离的平方 D² = (x - x̄)ᵀS⁻¹(x - x̄) 排序后，与 χ²(p) 分布的分位数做散点图。如果点大致落在对角线上，就不能拒绝正态性假设。这是最直观的方法，但判读有一定主观性。

Mardia 检验

基于偏度 (skewness) 和峰度 (kurtosis) 的联合检验。多元偏度衡量分布的对称性，多元峰度衡量分布的"尾部厚度"。标准正态分布的偏度为0，峰度为 p(p+2)。Mardia 检验的零假设是数据来自多元正态分布。

Henze-Zirkler 检验

基于特征函数的非参数检验，在统计功效（power）上通常优于 Mardia 检验，特别对尾部偏差敏感。R 的 MVN 包和 Python 的 pingouin 库都提供了该检验的实现。

经验法则：在实际应用中，"近似"正态通常就够用了。严格的正态性检验在大样本下容易拒绝（因为实际数据几乎没有完美的正态分布），在小样本下又不容易检测到偏离。我的建议是：结合 Q-Q 图和正式检验一起判断，同时注意你的方法对正态偏离的稳健性。像 Hotelling T² 在大样本下对偏离是相当稳健的，但因子分析和结构方程模型就不那么"宽容"了。

踩坑故事："相关性不等于因果"的正态版本

最后分享一个经典的教学案例。有研究者分析了某地区多年的气象数据，发现"冰淇淋销量"和"溺水人数"的相关系数高达 0.87。于是有人开玩笑说"吃冰淇淋会导致溺水"——当然这是荒谬的。但用我们刚学的偏相关的角度来分析就清楚了：控制"气温"这个变量后，冰淇淋销量和溺水人数的偏相关系数几乎为零。气温同时影响了两者——天气热，更多人吃冰淇淋，也更多人游泳（从而溺水风险增加）。这就是著名的混杂变量 (confounding variable) 问题。

这个故事的多元统计版本是：当你看到两个变量显著相关时，不要急于下结论。用偏相关系数"控制"其他可能的混杂变量，往往能揭示真实的关系。在基因表达分析中，我们经常发现成百上千个基因之间都有显著的相关性，但很多都是"间接调控"导致的虚假关联。偏相关分析能帮你聚焦在"直接关联"上。

动手练习：打开 Python，生成三组数据：(1) X₁ ~ N(0,1)，(2) X₂ = X₁ × 0.5 + ε（ε 独立于 X₁），(3) X₃ 也依赖于 X₁。然后计算 X₂ 和 X₃ 的简单相关系数，再计算控制 X₁ 后的偏相关系数。你会看到简单相关可能显著，但偏相关几乎为零。这就是"虚假相关"的典型例子。

本章小结与衔接

这一章我们从多元正态分布的密度函数开始，理解到它的"指数部分"就是马氏距离的平方，等高线是椭圆——椭圆形状由协方差矩阵的特征结构决定。然后学习了三个核心性质：线性变换保持正态性（这是所有后续推导的基础）、零相关等价于独立（正态分布独有的便利）、边缘和条件分布仍然是正态的。条件分布公式是线性回归的数学起源，"最优预测就是线性预测"这一结论直接来自多元正态。接着我们讨论了 MLE——样本均值向量和协方差矩阵的估计及其抽样分布（Wishart 分布是一元 χ² 的多元推广）。最后介绍了几种正态性检验方法。

现在，你已经掌握了多元正态分布这个强大的理论工具。下一章，我们将用它来构建均值向量和协方差矩阵的假设检验：如何同时检验多个变量的均值是否相等？如何比较多个组的协方差结构？这既是理论的自然延伸，也是实际数据分析中最常见的需求之一。
