第九章：抽屉原理（二）

大家好，我是你们的讲师。在上一章中，我们学习了抽屉原理的基础——三种表述和各种构造抽屉的方法。但说实话，竞赛中的抽屉原理远不止"把m个东西放进n个抽屉"那么简单。真正有挑战的问题是：当"抽屉"本身也需要分层次时，你该怎么办？

今天这章我们继续深入，学习嵌套抽屉、面积抽屉、同余类抽屉的高级用法，以及Ramsey型的抽屉思想。这些工具合在一起，会让"存在性"问题的解决能力上升一个台阶。

9.1 构造嵌套抽屉的技巧

所谓"嵌套抽屉"，就是在一个抽屉内部再细分抽屉。当问题涉及多个条件同时需要满足时，往往需要多层抽屉的配合。

9.1.1 区间划分嵌套

例1：从 1,2,...,99 中任意选出 51 个数。证明：其中必有两个数，它们的差为 50。

分析：这是一个经典的一层抽屉题——把数按 {1,51}, {2,52}, ..., {49,99}, {50} 分成50组。选51个数，必有两个在同一组，差为50。

但如果题目改成"证明其中必有两个数，它们的差为 50 或 49"呢？这时就需要嵌套抽屉了。

考虑将所有数按模 50 的余数分类，每类内部再按数值大小划分。更简单的方法是直接构造抽屉：

构造50个抽屉：{1,51}, {2,52}, ..., {49,99}, {50}。把51个数放入这些抽屉，由抽屉原理，至少有一个抽屉有至少2个数。如果这个抽屉是 {1,51}...{49,99} 之一，差为50，结论成立。但如果唯一的重复落在 {50} 这个抽屉呢？{50}里只有一个数，不可能有2个！所以这不能保证。

我们需要更精细地构造。考虑把 1~99 分成 50 个区间：[1,2], [3,4], ..., [97,98], [99]。每个区间包含2个相邻数（最后一个只含99）。选51个数放入50个区间，至少有一个区间有2个相邻数，差为1。但这和50无关。

看来直接构造不行。换一种思路：把选出的51个数 排序 为 a₁ < a₂ < ... < a₅₁。考虑它们与50的差：a₁+50, a₂+50, ..., a₅₁+50。这51个数都在 51~149 之间。原选的51个数 a₁~a₅₁ 在 1~99 之间。总共有 51+51 = 102 个数在 1~149 之间，但 1~149 只有 149 个数——不对，不是这个证法。

我们再仔细想：原题改成了"差为50或49"，这个"或"字给了我们更多的灵活性。实际上，我们可以证明更强的结论。

实际上，更一般的结论是：从 1~2n 中选 n+1 个数，必有两个数的差为 n。这在上一章已经证过了。而"差为 n 或 n-1"的版本，证明思路是类似的——排序后考虑互补对。

在竞赛中，当一层抽屉不够用时，考虑：

  1. 先按大条件分第一层抽屉（比如按范围、按余数模某数）
  2. 再在每层抽屉内部按小条件细分
  3. 有时候需要选两次抽屉（先确定大抽屉，再在大抽屉内部找小抽屉）

9.1.2 模分类的嵌套应用

例2：任意给定 5 个整数，证明其中必有 3 个整数，它们的和能被 3 整除。（我们在上一章已经用分类讨论解决了。现在用嵌套抽屉再解一遍。）

嵌套解法：按模3余数，5个数放入3个抽屉。由 ⌈5/3⌉ = 2，至少有一个抽屉有至少2个数。但我们还需要3个数，所以一层不够。仔细观察：

  • 如果某个抽屉有至少3个数：这3个数的和 ≡ 3r ≡ 0 (mod 3)，已解决。
  • 如果每个抽屉最多2个数：那么由于总数5，抽屉数3，所有抽屉的分布只能是 (2,2,1) 或 (2,1,2) 或 (1,2,2) 等。每种分布下，三个抽屉各取一个数，其和 ≡ 0+1+2 ≡ 0 (mod 3)。

这里的"嵌套"体现在：第一层是余数抽屉，第二层是"看这个抽屉有多少个"的条件分类。

9.2 面积抽屉原理

当抽屉从离散的"盒子"变成连续的"面积"时，抽屉原理就升级成了面积形式。

9.2.1 Dirichlet 逼近定理

这是抽屉原理在数论中最经典的应用，由 Dirichlet 本人提出。

Dirichlet 逼近定理：给定任意实数 α 和正整数 N，存在整数 p,q（1 ≤ q ≤ N）使得：

|qα - p| < 1/N

换句话说，存在分母不超过 N 的有理数 p/q 逼近 α，误差小于 1/(qN)。

证明：考虑 N+1 个数：{0·α}, {1·α}, {2·α}, ..., {N·α}，其中 {x} = x - ⌊x⌋ 表示 x 的小数部分。这些小数部分都在 [0,1) 区间内。

把 [0,1) 分成 N 个等长子区间：[0, 1/N), [1/N, 2/N), ..., [(N-1)/N, 1)。

有 N+1 个数放入 N 个区间，由抽屉原理，至少有一个区间包含至少两个小数部分，设为 {iα} 和 {jα}（设 i > j）。它们在同一区间内，所以它们的差小于 1/N：

|{iα} - {jα}| = |(i-j)α - (⌊iα⌋ - ⌊jα⌋)| < 1/N

令 q = i-j（1 ≤ q ≤ N），p = ⌊iα⌋ - ⌊jα⌋，即得 |qα - p| < 1/N。

这个证明的美妙之处：Dirichlet 把无限的问题（小数部分可以无限精细）变成了有限的问题（N个子区间）。这是"离散化"的经典范例——用有限区间去逼近无限可能。这个思想后来发展为 Kronecker 逼近定理和三维的 Minkowski 定理，是数论和几何数论的基石。

9.2.2 图形面积抽屉

例3：在一个半径为1的圆内任意放 9 个点。证明：其中必有两点的距离不超过 1。

分析：将圆等分成 8 个扇形（每个圆心角 45°）。9 个点放入 8 个扇形，至少有一个扇形有至少 2 个点。但问题是——同一个扇形中的两点距离不一定小（扇形可以很尖，两点可以在弧的两端）。

所以仅仅按角度划分不行。我们需要更好的划分方式。

更好的方法：将大圆内接一个正八边形，将圆分成 8 个全等的"扇区+内接多边形"区域，每个区域外接圆半径为1。通过几何计算，每个区域中任意两点的最大距离 ≤ 1（实际上等于一个正八边形的边长对应的弦长）。但这样计算量较大。

最简单的解：考虑圆的内接正六边形，每个边对应的圆心角60°，边长为1。实际上，整个圆可以覆盖在 7 个直径为 1 的小圆下——由抽屉原理，9 个点中必有 2 个落在同一个小圆内，距离 ≤ 1。

但这并不是最优的覆盖——这又引出了另一个组合几何问题。在竞赛中，如果你需要证明"存在两点距离 ≤ d"，通常是把区域划分成若干个小区域，使每个小区域的直径（最大距离）不超过 d。然后点数 > 区域数即可。

⚠️ 面积抽屉的关键：面积抽屉的难点在于"划分"——要保证每块的"直径"不超过目标值 d，同时块的数目要尽可能少。块数越少，需要的物体数就越少，结论就越强。考试时如果划分不理想，可以尝试不同的划分方式，比如改用正六边形网格代替正方形网格，往往能得到更好的界。

9.3 同余类抽屉的高级应用

同余类抽屉在数论组合问题中扮演着桥梁角色。我们通过一个综合例题来展示它的威力。

例4：证明：对任意正整数 n，存在一个由 0 和 1 组成的 n 位数（首位可以是0），使得这个 n 位数是 n 的倍数。

分析：考虑 n+1 个数：1, 11, 111, ..., 111...1（n+1个1）。每个数模 n 取余数，得到 n+1 个余数。余数只有 n 种可能（0,1,...,n-1），由抽屉原理，至少有两个数的余数相同。

设这两个数分别是 A = 111...1（a个1）和 B = 111...1（b个1），a > b。那么 A - B = 111...1000...0（a-b个1后跟b个0）= 111...1 × 10^b。这个数模 n 余 0。

但是 10^b 可能与 n 不互质，所以不能简单地说 111...1（a-b个）就是 n 的倍数。不过，我们可以换一个思路——

实际上，我们可以用更简单的方法：考虑 n 个数：1, 11, 111, ..., 111...1（n个1）。它们模 n 的余数。如果有余数为 0 的，直接找到。如果没有余数为 0 的，那么 n 个余数在 1,...,n-1 这 n-1 种可能中，由抽屉原理，必有两个相等——差就是 n 的倍数，且形如 111...1000...0。

这个问题是经典的"全部由1组成的数中必有n的倍数"的变体，核心就是用同余类做抽屉。

9.4 "存在两个..."型问题的套路总结

做了这么多抽屉题，我们来做一个系统性的套路总结。所有"存在两个..."的问题，都可以纳入以下框架：

步骤操作思考方向
1. 识别题目是否要求"存在某两个对象满足某种关系"关键词：存在、必有、保证、无论...
2. 定抽屉确定"状态空间"或"分类标准"什么属性有限？余数？区间？分组？
3. 算数量计算物体数和抽屉数物体数 > 抽屉数？⌈物体/抽屉⌉ ≥ 某个数？
4. 用原理套用抽屉原理得到重合至少同一个抽屉有两个
5. 往回推证明"同一抽屉 → 所需关系"两个在同一抽屉的东西为什么满足要求？

例5（综合练习）：从 1 到 10 中选出 6 个数。证明：其中必有两个数的和是 11。

按上述框架：

  • 识别：存在两个数，和为11。
  • 定抽屉：把和为11的数配对：(1,10), (2,9), (3,8), (4,7), (5,6)。这5对就是5个抽屉。
  • 算数量：6个数 → 5个抽屉，6 > 5。
  • 用原理：至少有一个抽屉有2个数。
  • 往回推：这个抽屉中的两个数和为11。结论成立。

简单吧？但竞赛题可不会这么直接地给你配好对。它们会问"从 1 到 2n 中选 n+1 个数"或"从 1 到 99 中选 50 个数使得没有两个数的和是 100"等变形。配对的模式不变，但你需要自己发现"哪些两个数加在一起是定值"。

9.5 综合例题：多个抽屉联合使用

现在来看一个需要同时使用多个抽屉原理的高级题。

例6：给定 7 个不同的整数。证明：从中必能选出两个整数，使得它们的和或差是 10 的倍数。

分析：每个整数模 10 的余数在 {0,1,...,9} 中。我们考虑"和是10的倍数"和"差是10的倍数"两种情况：

  • 和是10的倍数：两个数模10余数分别为 r 和 10-r（或都是0或都是5）。
  • 差是10的倍数：两个数模10余数相同。

所以问题转化为：7个余数，要证明要么有两个相同（差是10倍数），要么有两个关于10互补（和是10倍数）。

把余数分成以下6组抽屉：{0}, {1,9}, {2,8}, {3,7}, {4,6}, {5}。每个抽屉中的任意两个数，要么和是10的倍数（互补对），要么...等等！{0} 组中两个数的差是10的倍数？不，两个余数0的数差是0，确实是10的倍数。{5} 组同理，两个余数5的数差是0，也是10的倍数。

等等让我重新组织：把余数分成以下6组：{0}, {1,9}, {2,8}, {3,7}, {4,6}, {5}。7个余数放入6组，至少有一组有至少2个余数。如果这组是 {0} 或 {5}，那么两个数模10同余，差是10的倍数。如果这组是其他组（比如 {1,9}），那么这两个数互为补数，和是10的倍数。

所以结论成立。

解题范式总结：这个题的关键是发现了"差是10倍数 = 同余；和是10倍数 = 互补"。然后把剩余类重新分组，使得每个组内要么全相等（差倍数），要么互补对（和倍数）。这样抽屉就同时覆盖了两种情况。这种"合并抽屉"的技巧在解决多个条件的"存在性"问题时非常有效。

例7（IMO 级别的思路）：任意给定 5 个格点（坐标为整数的点），证明存在一个格点位于以其中某三个格点为顶点的三角形内部或边上。

提示：考虑格点坐标的奇偶性。每个格点有4种可能的奇偶类型（偶偶、偶奇、奇偶、奇奇）。5个点放入4个类型，至少有两个类型相同——它们的中点也是格点。但这还不够，我们需要三个点。

继续分析：5个点 → 4种奇偶类 → 至少一类有2个点。如果有一类有3个点，则这3个点构成三角形的重心也是格点？不一定。但如果是5个点分布在4个类中，有可能每个类最多2个点。由抽屉原理的精确形式：⌈5/4⌉=2，所以最多可能有1个类有2个点，另外3个类各1个点。但这也不对——可能是(2,1,1,1)的分布。

如果有一个类有2个点，它们的中点M是格点。再看剩下的3个点——如果这3个点有一个点与M组成格点三角形？不对。

让我们换一种思路：考虑所有点的奇偶类型，如果有三个点属于同一类型，以它们为顶点的三角形的重心就是格点，所以这三个点中任意两个的中点也是格点，但这不直接说明"三角形包含格点"。

实际上，这个题的正确解是：5个格点按奇偶分成4类，必有2个同类的点。取这一对点，再取剩下3个点中的任意一个作为第三个顶点——构成三角形。由 Pick 定理或简单计算可得，这个三角形的重心或边上必有一个格点。

其实更简单的做法是：根据组合几何中的结论，任意5个格点中，必有一个以其中三点为顶点的三角形的重心是格点——而这个重心正好位于三条中线的交点，也在三角形内部。

竞赛方法论：当问题同时涉及"和"和"差"条件时，不要分开考虑。尝试把两种条件对应的抽屉"合并"成一个统一的分类体系，然后在这个体系中应用抽屉原理。合并后的抽屉数越少，需要的物体数就越少，结论就越漂亮。

9.6 本章小结与过渡

本章我们深入学习了抽屉原理的进阶技巧：嵌套抽屉应对多层次条件、面积抽屉处理连续性问题、同余类抽屉连接数论与组合、以及多个抽屉联合使用的方法。我们还系统总结了"存在两个..."型问题的解题套路。

抽屉原理在整个组合数学中的地位，类似于"公理"在几何中的地位——它简单到无法被证明，但又强大到可以解决最复杂的问题。熟练掌握抽屉原理的各种形式，是走向组合高手的第一步。

下一章我们将进入一个全新的主题——母函数。如果说抽屉原理是"定性"地证明存在性，母函数就是"定量"地计算有多少种可能。将数列转化为幂级数的系数，这种"代数化"的思路会让很多看似复杂的计数问题变得无比优雅。
