第19章 以太坊与智能合约 — 世界计算机

比特币的故事我们已经讲了很多章了。到了这里，你可能会想："比特币能转账、能存储价值，挺厉害了——但还能干点别的吗？"

这正是2013年一个19岁的加拿大天才头脑里冒出的问题。这个年轻人叫Vitalik Buterin（我们圈子里都叫他"V神"），他当时是比特币杂志（Bitcoin Magazine）的联合创始人。他写了一篇白皮书，提出了一个大胆的想法：如果区块链不只能转账，还能运行程序呢？

这个想法后来变成了以太坊（Ethereum）——到今天已是仅次于比特币的第二大区块链平台，承载着数千亿美元的资产价值。这一章，我们就来理解它到底做了什么、为什么重要。

19.1 为什么需要以太坊

比特币的设计极其简洁——它只做一件事，而且做得很好：价值转移。比特币的脚本语言（Script）是故意限制的——没有循环，没有复杂逻辑，只能做基本的签名验证和时间锁。这不是设计缺陷，而是安全考虑。中本聪想要一个确定性的、可预测的系统。

但这个限制意味着：你无法在比特币上构建复杂的金融应用。比如你想做一个去中心化的交易所，让两个人交换代币而无需信任第三方——比特币的脚本语言做不到。

Vitalik看到的正是这个机会。他在2013年底发布了以太坊白皮书（Ethereum Whitepaper），提出了一种"图灵完备"的区块链——一个能运行任意复杂程序的"世界计算机"（World Computer）。

核心思想：以太坊不是一个"币"，它是一个去中心化的全球计算平台。比特币是"1.0"，以太坊是"2.0"——它把区块链从"账本"变成了"操作系统"。

19.2 EVM与图灵完备

以太坊最核心的创新是以太坊虚拟机（Ethereum Virtual Machine, EVM）。你可以把EVM想象成一台"全球共享的虚拟电脑"——任何人都可以往上面部署程序，网络中的每个节点都会执行它。

一个关键特征：EVM是图灵完备（Turing-complete）的。这意味着理论上它可以计算任何可计算的问题（当然受限于现实资源）。这对比特币脚本是一个质的飞跃。

但"图灵完备"也带来一个麻烦：如果程序里有无限循环怎么办？在普通计算机上，你会强制终止它。但在去中心化的网络上，谁来决定终止？

解决方案是Gas（燃料）机制：你发布的每一段代码，每一步运算都要消耗Gas。如果Gas用完了，程序自动终止。这确保了"世界计算机"不会被恶意程序"跑死"——攻击者在理论上可以把整个网络撑爆，但需要支付天文数字的Gas费用。

// 一个简单的Gas消耗示例
// 你发送交易时需要指定:
{
  "gasLimit": 21000,    // 你愿意支付的"燃料上限"
  "maxFeePerGas": 50,   // 每单位燃料的"最高单价" (gwei)
  "maxPriorityFee": 2   // 给验证者的小费 (gwei)
}

// 一次简单转账消耗 21000 Gas
// 如果是复杂合约调用，可能消耗 200000+ Gas
// 如果Gas用完 → 交易失败，但已消耗的Gas不退
Gas的小秘密：写以太坊合约有点像开一辆"按公里计费"的租车——代码写得越高效，用户花的钱越少。这就是为什么以太坊开发者社区极度重视gas optimization，一段优化不当的循环可能让用户多付几百美元的Gas费。

19.3 智能合约入门

在以太坊上运行的程序叫智能合约（Smart Contract）。这个术语由计算机科学家Nick Szabo在1994年提出，但直到以太坊出现才真正落地。

智能合约的核心思想是"代码即法律"（Code is Law）——合约的条款被写成代码，部署到链上后，任何人都无法干预它的执行。没有法官，没有仲裁，代码怎么写的，就怎么执行。

最常见的智能合约语言是Solidity。让我们来看一个最简单的智能合约——一个"数字存钱罐"：

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract PiggyBank {
    address public owner;
    uint256 public targetAmount;
    bool public isBroken;

    constructor(uint256 _target) {
        owner = msg.sender;
        targetAmount = _target;
        isBroken = false;
    }

    function deposit() external payable {
        require(!isBroken, "存钱罐已碎！");
        require(msg.value > 0, "存点钱吧！");
    }

    function smash() external {
        require(msg.sender == owner, "不是你存的钱罐！");
        require(address(this).balance >= targetAmount, "还没存够！");
        isBroken = true;
        payable(owner).transfer(address(this).balance);
    }
}

这个合约的逻辑很直观：设定一个储蓄目标，往里面存钱，存够了才能"砸碎"取出来。一旦部署到以太坊上，没有任何人——包括你自己——能在钱没存够之前把钱取走。这就是智能合约的力量。

血的教训：2016年，一个叫"The DAO"的智能合约被黑客利用重入攻击（reentrancy attack）盗走了价值约6000万美元的ETH。核心原因是合约代码没有"先更新余额再转账"。这直接导致了以太坊的硬分叉——分裂成了今天的以太坊（ETH）和以太坊经典（ETC）。Code is Law？但当法律有bug时，社区只能硬着头皮"修法"。

19.4 ERC-20与代币标准

以太坊上最重要的创新之一，是它让任何人都能发行自己的数字资产。在比特币上，你只能有BTC。但在以太坊上，你可以发行"自己的币"。

这得益于ERC-20标准（Ethereum Request for Comments 20）。ERC-20定义了一组通用接口：一个代币合约必须实现哪些函数（transfer、balanceOf、approve等）。因为有了标准，钱包（如MetaMask）、交易所（如币安）可以统一接入任何ERC-20代币。

函数功能说明
totalSupply()总供应量这个代币总共发行了多少
balanceOf(address)查询余额某地址有多少
transfer(to, amount)转账把代币发送给另一个地址
approve(spender, amount)授权授权另一个地址花你的代币
transferFrom(from, to, amount)代转被授权的地址执行转账

你平时听到的"山寨币"——USDT、USDC、UNI、LINK、SHIB——绝大多数都是基于以太坊ERC-20标准发行的代币。它们共享以太坊的安全保障和去中心化网络，但各自带着不同的规则和用途。

有意思的是：USDT（泰达币）最初发行在Omni（比特币）协议上，后来全面迁移到了以太坊ERC-20标准上。截至2026年，ERC-20上的USDT流通量已经超过1000亿美元——以太坊成了稳定币的核心结算层。

19.5 以太坊vs比特币对比

很多新人会问："有了比特币为什么还要以太坊？它们不是竞争关系吗？"从技术角度看，它们完全不在同一个赛道上：

维度比特币（Bitcoin）以太坊（Ethereum）
定位数字黄金，价值存储世界计算机，去中心化应用平台
共识机制PoW（工作量证明） → 纯PoWPoS（权益证明） → 2022年The Merge后
区块时间~10分钟~12秒
可编程性有限（Script非图灵完备）图灵完备（EVM）
供应模型固定2100万，减半机制无硬上限，有销毁机制（EIP-1559）
区块浏览器仅记录UTXO交易记录交易 + 合约状态 + logs
主要用途支付、储值DeFi、NFT、DAO、游戏、身份…

简单来说：比特币想成为"更好的黄金"，以太坊想成为"去中心化的互联网"。它们不是竞争对手，而是不同赛道的领军者。

19.6 2026以太坊生态

截至2026年，以太坊已经是全球最大的应用层区块链平台。几个关键趋势值得关注：

Pectra升级（2026年）：以太坊的最新硬分叉升级，整合了Prague和Electra两个阶段的改进。核心变化包括：改进账户抽象（Account Abstraction）简化用户体验、提升blob数据容量以支持Layer 2扩容、以及改进验证者提款机制。这次升级进一步巩固了以太坊作为L1结算层的地位。

Layer 2爆发：2025年，以太坊L2的交易量首次超过了L1主网。Arbitrum、Optimism、Base、zkSync等L2生态已经成熟，用户在这些L2上支付不到0.01美元的手续费，同时享受以太坊主网的安全性。

机构入场：2024年美国以太坊现货ETF获批后，传统金融机构大量涌入。BlackRock、Fidelity等资管巨头不仅在以太坊上投资，还直接在链上发行代币化基金（如BlackRock的BUIDL基金）。截至2026年，以太坊上锁定的总价值（DeFi TVL）超过1500亿美元。

说到ETF：2024年7月，美国SEC批准了以太坊现货ETF（Ethereum ETF），这比很多人的预期早得多。有分析师认为，这背后的政治推力是拜登政府的"加密友好转向"——毕竟2024年大选中，加密选民已经成了一个不可忽视的力量。

回到Vitalik那个19岁时的疯狂想法："如果区块链不只是个账本呢？"十年后，以太坊已经承载了数千亿美金的经济活动、数百万个智能合约（包括但不限于我们下一章要讲到的DeFi协议）。它确实成了世界计算机——虽然有点慢、有点贵，但正在通过Layer 2等技术变得更快、更便宜。
