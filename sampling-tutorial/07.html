第7章 整群抽样

大家好，我是你们的讲师。今天我们进入一个非常实用但也经常被误解的抽样方法——整群抽样（Cluster Sampling）。

在之前学习的SRS中，我们假设你能获取到总体中每一个个体的信息——这需要一份完整的抽样框（Sampling Frame）。但真实世界并不总是配合你：如果你要调查某市所有小学生的视力状况，你手头可能没有每个小学生的名单，但你肯定知道全市有哪些学校。如果你要调查一个省份的农村居民收入，你不可能列出全省所有农户的名字，但你能列出所有行政村。

这就是整群抽样大显身手的地方：当个体名录不可得，但群体（群）的名录可得时，你可以先抽群，再调查群内的所有（或部分）个体。它的核心思想与分层抽样正好相反——整群抽样追求"群内尽可能异质"（让一个群充分代表总体的多样性），而分层抽样追求"层内尽可能同质"（每一层都是总体的一个同质子集）。

群的定义

在整群抽样中，"群"（Cluster）可以是任何天然存在或人为划分的单元集合：

  • 自然分群：学校（学生群）、社区（居民群）、工厂（工人群）、医院（患者群）、班级、村落、街道等。
  • 人为分群：为了操作方便而人为划分的群体。比如把一个大城市按照地图格网划分为若干区块作为群，每一块内的居民就是一个人为群。
  • 地理分群：初级抽样单元（PSU）通常是地理单元，如县、街道、普查区块等。这是大规模政府调查中最常见的形式。

在定义群时，有一个重要的权衡：

  • 群越大（每个群包含更多个体），则群数越少——这意味着抽样方差可能增大（因为样本量更集中于少数群），但操作成本更低。
  • 群越小（每个群包含更少个体），则群数越多——抽样方差更小，但操作成本更高。

在极端情况下，当每个群只包含一个个体时，整群抽样就退化成了SRS。

整群抽样的成本优势：整群抽样最大的优点是降低调查成本。如果一个调查团队要做入户访问，采用SRS可能意味着他们需要跑遍全市的各个角落——从城东到城西、从城南到城北。而如果先抽几个社区（群），然后在每个社区内集中访问——交通成本和时间成本就会大幅降低。在很多实地调查中，这种成本差异可以达到10倍甚至更多。成本的降低又反过来允许你用同样的预算调查更多的个体，从而部分抵消整群抽样的效率损失。

群内相关系数

整群抽样的效率高度依赖于群内相关系数（Intra-cluster Correlation Coefficient, ρ或ICC）。这个指标衡量的是：同一个群内的个体之间，在目标变量上的相似程度有多高。

ρ的取值通常在0到1之间：

  • ρ = 0：群内个体之间完全不相关——同一个群内的个体和总体的随机个体没有区别。这是理想情况，但现实中很少见。
  • ρ > 0：群内个体之间存在正相关——这是最常见的情况。住在同一个社区的居民往往有相似的收入水平；同一个班级的学生往往成绩接近。ρ越大，整群抽样的效率越低。
  • ρ = 1：群内个体完全一样——一个群里的所有人都有相同的目标变量值。这是最坏的情况：调查一个群里的100个人，和只调查1个人没有任何区别。

ρ的估计公式为：

ρ = σ²群间 / (σ²群间 + σ²群内)

其中σ²群间是群均值之间的方差，σ²群内是群内个体之间的方差。ρ越大，说明群间差异相对于群内差异越大——也就是同一个群里的个体"长得越像"。

在社会科学调查中，ρ的典型值通常在0.01到0.20之间。教育领域中的ICC通常较高（同一学校的学生成绩高度相关），而消费行为调查中的ICC通常较低。

一个反直觉的事实：很多人以为"整群抽样就是把一群人都调查了，样本量这么大，应该很准吧？"——但事实恰恰相反。如果群内相关系数ρ=0.2，群大小m=50，那么设计效应Deff = 1 + (50-1)×0.2 = 10.8——这意味着你需要调查10.8倍的个体才能达到和SRS相同的精度！换句话说，在一个群里调查50个人，在统计效率上只相当于在SRS中调查不到5个人。这是整群抽样最容易被忽视的代价。

整群抽样 vs 分层抽样

初学者最容易混淆的就是整群抽样和分层抽样。它们看起来都是"先把总体分组，再抽样"，但哲学完全不同：

维度 分层抽样 整群抽样
分组原则 层内同质、层间异质 群内异质、群间同质
理想情况 每个层内部尽可能相似 每个群内部尽可能多样（代表总体）
抽样方式 每一层都抽样 抽取一部分群，被抽中的群全查
主要目标 提高精度（降低方差） 降低操作成本
效率 vs SRS 通常更高效（Deff < 1） 通常更低效（Deff > 1）
所需信息 需要知道每个个体属于哪一层 只需要知道每个个体属于哪个群
抽样框 需要完整的个体名录 只需要群体名录
典型场景 有完备名录、追求高精度 有名录不全、成本受限

这张表的信息量很大，我再强调一下核心区别：

  • 分层抽样 = 每层都抽，但每层只抽一部分。它的目的是"消灭组间差异"，只留下组内差异作为抽样误差。
  • 整群抽样 = 只抽一部分群，但被抽中的群全查。它的目的是"降低操作成本"，牺牲一部分统计效率来换取可行性。

如果你混淆了这两种方法，可能会做出非常糟糕的设计。想象一下：按学校分层，然后在每所学校内抽一个班级——这不是分层抽样！这是分层与整群的混合（实际上是一种两阶段抽样）。如果你误以为这是分层抽样并按分层抽样的公式来计算方差，你会严重低估真实的抽样误差。

设计效应 Deff

我们在第5章第一次提到了设计效应（Design Effect, Deff）。整群抽样是理解Deff的最经典场景。对于规模为m的等规模群（每群包含m个个体），整群抽样的Deff为：

Deff整群 = 1 + (m - 1) × ρ

这个公式揭示了整群抽样的效率损失来自两个因素：

  • 群大小 m：群越大，效率损失越大。同一个群里调查的人越多，获得的新信息就越少（因为他们之间本来就相似）。
  • 群内相关系数 ρ：ρ越大，效率损失越大。群内个体越像，调查越多同质个体就越浪费样本。

假设ρ=0.1：

  • m=10时，Deff = 1 + 9×0.1 = 1.9——你需要调查1.9倍的人才能达到SRS的精度
  • m=50时，Deff = 1 + 49×0.1 = 5.9——你需要调查5.9倍的人
  • m=100时，Deff = 1 + 99×0.1 = 10.9——你需要调查近11倍的人！

这就是为什么在实际调查中，我们通常选择较小的群（如每个群调查10-20人），而不是在一个群里调查几百人。边际收益递减——最开始调查的几个人带来了大量新信息，但后续的人提供的新信息越来越少。

def cluster_deff(m, rho):
    """
    计算整群抽样的设计效应
    m: 群大小（每个群平均调查的人数）
    rho: 群内相关系数
    """
    return 1 + (m - 1) * rho

def effective_sample_size(n, m, rho):
    """
    整群抽样的有效样本量
    n: 实际调查的总人数
    m: 平均群大小
    rho: 群内相关系数
    """
    deff = cluster_deff(m, rho)
    return n / deff

# 示例：调查了500人，分布在25个班级（每班20人），ρ=0.15
n_actual = 500
m = 20
rho = 0.15
deff = cluster_deff(m, rho)
n_eff = effective_sample_size(n_actual, m, rho)
print(f"设计效应 Deff = {deff:.2f}")
print(f"实际样本量 = {n_actual}")
print(f"有效样本量 = {n_eff:.0f} 人")
print(f"精度相当于SRS下的 {n_eff:.0f} 人")
整群抽样最常见的错误：很多初学者把整群抽样获得的数据当作SRS数据来分析——完全忽视群内相关性。当你这样做时，计算出的标准误会偏小、置信区间的覆盖率会低于名义水平（比如本应是95%的置信区间，实际覆盖率只有85%甚至更低）。这是非常危险的人为错误——你报告的结果看起来比实际要精确得多。务必记住：在分析整群抽样数据时，必须使用能够处理相依数据的方法（如稳健标准误、多层次模型、或专门针对复杂抽样设计的软件函数）。

真实案例：中学教育质量调查

某市教育局计划开展一项中学生学习习惯调查。全市有50所中学，共约50,000名学生。预算和时间都非常有限——需要在一周内完成调查。

如果用SRS：需要从全市5万名学生中随机抽取1000人，但这意味着调查员需要跑遍全市50所学校，每所学校可能只抽到20个学生——交通和时间成本极高。而且，不同学校的排课时间不同，访问一个学生可能需要重新协调时间——一周内根本完不成。

最终方案：采用整群抽样——从50所学校中随机抽取10所，被抽中学校的所有学生都参与调查。

这个方案的成本优势非常明显：

  • 只需要联系10所学校，而不是50所
  • 在每个学校内集中施测（可以安排在课间操或班会时间），效率极高
  • 交通和时间成本降低约70%

但代价是什么？

假设平均每校有1000名学生，r=0.10（学习成绩的习惯具有一定的群内相关性——同一学校的学生受相似的校园文化和师资影响）。

Deff = 1 + (1000 - 1) × 0.10 ≈ 101

实际调查了10,000人（10所学校×1000人），但有效样本量只有 10,000 / 101 ≈ 99 人。也就是说，从统计精度来看，这次大规模的调查只相当于SRS下调查了99个人！

这个结果让人震惊——但这就是整群抽样的现实。调查团队在做决策时需要在操作成本和统计精度之间做出权衡。他们可能会选择折中方案：抽取25所学校，每所学校只调查1-2个班级（每班约40-50人），这样既降低了操作成本（只需要联系25所学校），又控制了群大小（m=40-50），从而控制了设计效应。

关键启示：整群抽样不是"免费的午餐"——它在降低操作成本的同时，显著降低了统计效率。在设计整群抽样时，你需要回答三个问题：第一，群内相关系数ρ是多少？（可以通过文献、历史数据或预调查来估计）；第二，你能接受的群大小m是多少？（通常建议m在10-50之间，不要贪多）；第三，在给定预算和精度要求下，群数（抽多少个群）和群内样本量（每个群调查多少人）的最佳组合是什么？这个优化问题被称为"整群抽样的最优设计问题"——我们会在后续章节中进一步讨论。

整群抽样的种类

根据抽样阶段的不同，整群抽样可以分为以下几类：

  • 单阶段整群抽样（One-stage Cluster Sampling）：抽中群后，调查群内的所有个体。这是最纯粹的整群抽样形式。适用于群规模较小的情况。
  • 两阶段整群抽样（Two-stage Cluster Sampling）：先抽群（第一阶段），然后在被抽中的群内再抽一部分个体（第二阶段）。这是大规模调查中最常见的形式——比如先抽社区，再在社区内抽一部分家庭。两阶段抽样可以控制群大小m，降低设计效应。
  • 多阶段整群抽样（Multi-stage Cluster Sampling）：多个层次的抽样。比如全国调查中：先抽县（第一阶段）、再抽街道（第二阶段）、再抽社区（第三阶段）、最后抽家庭（第四阶段）。国家统计局的劳动力调查、CPI调查等都是典型的多阶段整群抽样。

本章小结

整群抽样是解决"没有完整抽样框"困境的重要工具。它的核心权衡是：用统计效率的损失换取操作可行性和成本降低。群内相关系数ρ决定了效率损失的严重程度——ρ越大，损失越大。设计效应公式 Deff = 1 + (m-1)ρ 精确量化了这种损失。与分层抽样相比，整群抽样的哲学恰好相反——它追求群内异质、群间同质，通过"只抽部分群但全查"来降低成本。

下一章我们将进入一个更复杂但也更接近真实调查实践的方法——多阶段抽样（Multi-stage Sampling）。多阶段抽样结合了整群抽样的可行性和分层抽样的效率，是现代大规模调查的主流设计。我们下章见！
