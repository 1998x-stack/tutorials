第4章 Newton插值与均差

大家好，我是你们的讲师。今天我们来给插值法"升级"。

上一章我们学了Lagrange插值。它优雅、简洁、理论完美——但有一个工程上的硬伤：如果你已经算好了 n 个点的插值多项式，突然又来了一个新的数据点，用Lagrange方法你必须从头重新计算所有的 Lᵢ(x)。这就像你已经排好了一列多米诺骨牌，想加一张新牌就必须全部重新摆。而在工业界——比如传感器实时数据流中——每秒钟可能有上千个新数据点到来，这种"全部重算"的做法是不可接受的。

💡 场景想象： 你正在做自动驾驶的轨迹规划，车辆每毫秒收到一个新的GPS定位点，需要实时更新插值轨迹。如果每来一个点都 O(n²) 重算一遍——等你算完，车已经开出去十米了。

Newton插值法解决了这个问题。它的核心是引入均差（divided difference）的概念，使得增加一个新节点时，只需要在已有的多项式上加一项，计算量从 O(n²) 降到 O(n)。

4.1 均差的定义与性质

给定函数 f 在节点 x₀, x₁, ..., xₙ 上的值，定义：

  • 零阶均差： f[xᵢ] = f(xᵢ)
  • 一阶均差： f[xᵢ, xⱼ] = (f(xⱼ) − f(xᵢ)) / (xⱼ − xᵢ)
  • k阶均差： f[xᵢ, xᵢ₊₁, ..., xᵢ₊ₖ] = (f[xᵢ₊₁, ..., xᵢ₊ₖ] − f[xᵢ, ..., xᵢ₊ₖ₋₁]) / (xᵢ₊ₖ − xᵢ)

均差的一个关键性质是对称性：f[xᵢ, xⱼ, xₖ] 的值与节点的排列顺序无关。另外，均差与导数的关系也很重要——当节点趋于同一点时，均差趋于导数：

f[x₀, x₁, ..., xₙ] = f⁽ⁿ⁾(ξ) / n! 对于某个 ξ 在节点区间内。

均差可以通过表格形式高效计算：

xᵢ f(xᵢ) 一阶均差 二阶均差 三阶均差
x₀ f[x₀]
f[x₀, x₁]
x₁ f[x₁] f[x₀, x₁, x₂]
f[x₁, x₂] f[x₀, x₁, x₂, x₃]
x₂ f[x₂] f[x₁, x₂, x₃]
f[x₂, x₃]
x₃ f[x₃]

4.2 Newton插值多项式

利用均差，Newton插值多项式可以写成如下形式：

Pₙ(x) = f[x₀] + f[x₀, x₁](x − x₀) + f[x₀, x₁, x₂](x − x₀)(x − x₁) + ... + f[x₀, x₁, ..., xₙ](x − x₀)(x − x₁)...(x − xₙ₋₁)

这个形式的精妙之处在于：当你需要增加一个新节点 xₙ₊₁ 时，你只需要计算新的均差 f[x₀, ..., xₙ₊₁]，然后在现有多项式后面加上一项：

Pₙ₊₁(x) = Pₙ(x) + f[x₀, ..., xₙ₊₁](x − x₀)(x − x₁)...(x − xₙ)

之前的项一个都不用改。这是"增量更新"的数学基础。

import numpy as np

def divided_diff(x_points, y_points):
    """计算均差表"""
    n = len(x_points)
    F = np.zeros((n, n))
    F[:, 0] = y_points
    for j in range(1, n):
        for i in range(n - j):
            F[i, j] = (F[i+1, j-1] - F[i, j-1]) / (x_points[i+j] - x_points[i])
    return F

def newton_interp(x_points, y_points, x):
    """Newton插值"""
    n = len(x_points)
    F = divided_diff(x_points, y_points)
    result = F[0, 0]
    term = 1.0
    for i in range(1, n):
        term *= (x - x_points[i-1])
        result += F[0, i] * term
    return result

def lagrange_interp(x_points, y_points, x):
    """Lagrange插值（对比用）"""
    n = len(x_points)
    result = 0.0
    for i in range(n):
        term = y_points[i]
        for j in range(n):
            if i != j:
                term *= (x - x_points[j]) / (x_points[i] - x_points[j])
        result += term
    return result

# 验证等价性
x_pts = np.array([0.0, 1.0, 2.0, 4.0])
y_pts = np.array([1.0, 3.0, 7.0, 19.0])

x_test = 3.0
print(f"Newton:  {newton_interp(x_pts, y_pts, x_test):.6f}")
print(f"Lagrange: {lagrange_interp(x_pts, y_pts, x_test):.6f}")

# 增量添加节点
print("\n增量添加节点演示：")
x_pts_base = np.array([0.0, 1.0, 2.0])
y_pts_base = np.array([1.0, 3.0, 7.0])
x_new = 4.0; y_new = 19.0

P3 = newton_interp(x_pts_base, y_pts_base, x_test)
print(f"3点插值结果: {P3:.6f}")

x_pts_new = np.append(x_pts_base, x_new)
y_pts_new = np.append(y_pts_base, y_new)
P4 = newton_interp(x_pts_new, y_pts_new, x_test)
print(f"增量为4点插值: {P4:.6f}")
🔍 计算量对比： 添加一个节点时，Lagrange需要 O(n²) 重新计算所有基函数；Newton只需要 O(n) —— 计算新的均差并追加一项。当你处理几百个节点时，这意味着几万次乘法和几百次乘法的差距。

4.3 差分与等距节点

当节点是等距的，即 h = xᵢ₊₁ − xᵢ 是常数时，我们可以定义差分（difference）来简化计算。

  • 向前差分： Δf(xᵢ) = f(xᵢ₊₁) − f(xᵢ)，Δᵏf(xᵢ) = Δᵏ⁻¹f(xᵢ₊₁) − Δᵏ⁻¹f(xᵢ)
  • 向后差分： ∇f(xᵢ) = f(xᵢ) − f(xᵢ₋₁)，∇ᵏf(xᵢ) = ∇ᵏ⁻¹f(xᵢ) − ∇ᵏ⁻¹f(xᵢ₋₁)
  • 中心差分： δf(xᵢ) = f(xᵢ₊₁/₂) − f(xᵢ₋₁/₂)

差分的优势在于计算只涉及加减法，没有除法（因为 h 是常数，均差公式中的分母简化为 hᵏ 的倍数）。

等距节点下，均差与向前差分的关系为：

f[xᵢ, xᵢ₊₁, ..., xᵢ₊ₖ] = Δᵏf(xᵢ) / (k! · hᵏ)

4.4 Newton vs Lagrange 对比

对比维度 Lagrange插值 Newton插值
表达式 基函数的线性组合 均差的幂级数形式
增量添加节点 O(n²)，全部重算 O(n)，追加一项
计算稳定性 对舍入误差敏感 均差表也受累积误差影响
理论分析 便于推导余项公式 便于程序实现
节点分布 任意节点 任意节点（等距时可用差分简化）
💡 工程实践建议： 如果你需要固定的一组节点做一次插值，用Lagrange在纸上算更快。如果你写代码实现一个插值函数，并且数据会动态增加——用Newton。工业级库（如SciPy的 scipy.interpolate）内部用的是更高级的 barycentric Lagrange 形式，它结合了两者的优点。
⚠️ 踩坑故事： 一位做金融数据分析的工程师用Lagrange插值做收益率曲线构建。每次美联储发布新的经济数据，他都要把整个插值过程重新跑一遍。当数据点从20个增加到100个时，运行时间从0.1秒飙升到了8秒——高频交易系统中8秒意味着几百万美元的滑点损失。换成Newton插值后，添加一个新数据点的时间从O(n²)降到了O(n)，问题迎刃而解。

下一章我们将学习Hermite插值——当你不仅知道函数值，还知道导数值时该如何插值。你会看到，均差的概念在那里会发生一个有趣的推广：当节点重合时，均差退化为导数。
