连续：极限等于函数值的那一刻

大家好，我是你们的讲师。前面三章我们都在和"极限"打交道。但你可能一直在想一个问题：极限考察的是"趋近但不等于"——我们允许函数在 x₀ 点没有定义，仍然可以讨论极限。但现实世界中，大部分物理量是连续变化的：温度不会从 20°C 瞬间跳到 100°C，汽车的速度不会从 0 突然变成 60 km/h（加速度可以很大，但不可能无穷大）。

那么，在数学上如何刻画"没有突变"这一性质呢？答案就是连续性。

连续性跟极限只有一字之差：如果说极限关心的是"x 趋近 x₀ 时 f(x) 的趋势"，那么连续性就是在极限的基础上附加了一个条件——这个趋势必须等于 f(x₀) 本身。换句话说，连续就是"极限等于函数值"。

连续函数的三要素

一个函数 f(x) 在点 x₀ 处连续，需要同时满足三个条件：

  1. f(x₀) 有定义——函数在 x₀ 处存在
  2. lim_{x→x₀} f(x) 存在——极限存在
  3. 极限等于函数值——lim f(x) = f(x₀)

三者缺一不可。其中第二条又需要左极限和右极限都存在且相等。

用 ε-δ 语言来写连续的定义也很简洁：对任意 ε > 0，存在 δ > 0，使得当 |x - x₀| < δ 时，总有 |f(x) - f(x₀)| < ε。注意这里的条件和极限定义的差别：极限定义要求 0 < |x - x₀| < δ（去心邻域，排除了 x₀ 本身），而连续定义允许 |x - x₀| < δ（包含 x₀）。这是因为连续要求 f(x₀) 本身有意义且参与比较。

极限 vs. 连续：一字之差
极限：0 < |x - x₀| < δ → |f(x) - L| < ε（不关心 x₀ 点）
连续：|x - x₀| < δ → |f(x) - f(x₀)| < ε（包含 x₀ 点本身）

一个经典的记忆法：连续 = 极限存在 + 函数值存在 + 两者相等。

左右连续

与左极限、右极限对应，我们也有左连续和右连续的概念：

  • 左连续：lim_{x→x₀⁻} f(x) = f(x₀)
  • 右连续：lim_{x→x₀⁺} f(x) = f(x₀)

函数在 x₀ 处连续当且仅当它在 x₀ 处左连续且右连续。这个条件在处理分段函数时非常重要。

例：设 f(x) = { x^2, x < 1 ; ax + b, x ≥ 1 }，问 a、b 取何值时 f(x) 在 x = 1 处连续？

解：

  • 左极限：lim_{x→1⁻} x^2 = 1
  • 右极限：lim_{x→1⁺} (ax + b) = a + b
  • 函数值：f(1) = a + b

连续要求 1 = a + b，即 b = 1 - a。这时 a 可以是任意值，只要满足这个关系，函数就在 x = 1 处连续。

第一类间断点

如果函数在某点不连续，该点就是一个间断点。间断点按其性质分为两大类，先看第一类间断点：左极限和右极限都存在。

可去间断点

左右极限相等，但不等于函数值（或函数无定义）。"可去"意味着你可以通过重新定义或补充定义该点的函数值来让它连续。

例：f(x) = sin x / x 在 x = 0 处无定义。但 lim_{x→0} sin x/x = 1，所以 x = 0 是可去间断点。如果我们定义 f(0) = 1，函数就变成连续的了。

可去间断点的直觉
想象一个函数的图像上有一个小洞。你在 x₀ 处把笔抬起来，然后在原地放下去，画了一个孤零零的点。这就是可去间断点——你只需要把那个"孤点"补上，图像就连续了。

很多初学者会好奇：为什么要专门给这种情况起名字？因为在微积分里，可去间断点往往意味着极限存在，后续可以用极限来分析函数的局部行为。这对导数的存在性有直接影响。

跳跃间断点

左右极限存在但不相等。函数从左边趋近一个值，跳到另一个值。

例：f(x) = |x|/x 在 x = 0 处，左极限为 -1，右极限为 1，极限不存在——这是一个跳跃间断点。

最经典的跳跃间断点是取整函数 f(x) = [x]（不超过 x 的最大整数）。在每一个整数点处，左极限是 n-1，右极限是 n，函数值是 n，左边跳上来。

第一类间断点 左右极限关系 典型例子 可否修复
可去间断点 左右极限相等 sin x / x 在 x = 0 可补充定义修复
跳跃间断点 左右极限不相等 [x] 在整数点 无法修复

第二类间断点

第二类间断点更"暴力"——左极限和右极限至少有一个不存在。常见有两种：

无穷间断点

极限趋向无穷大。例如 f(x) = 1/x 在 x = 0 处：lim_{x→0⁺} 1/x = +∞，lim_{x→0⁻} 1/x = -∞，左右极限都不存在（无穷大不是极限）。

振荡间断点

函数值在某个范围内来回震荡，不趋向任何确定的值。最经典的例子就是 f(x) = sin(1/x) 在 x = 0 处——当 x → 0 时，1/x → ∞，sin(1/x) 在 -1 和 1 之间疯狂振荡，永远停不下来。

间断点分类一览：

        间断点
        ├── 第一类（左右极限都存在）
        │   ├── 可去：左右极限相等 ≠ 函数值
        │   └── 跳跃：左右极限不相等
        └── 第二类（至少一个极限不存在）
            ├── 无穷：极限为 ∞
            └── 振荡：极限不趋向任何值

        区分口诀："第一类看左右，第二类全靠不住"
一个真实的踩坑经历
我教过一个学生，他在编程实现一个数值积分函数时，传入的被积函数在某个点有一个可去间断点。他的积分算法没有处理这个情况，结果每次算到这个点附近，结果就出现莫名其妙的尖峰。他调试了两天，一度怀疑是浮点数精度问题，最后才发现是函数定义中有一处 if x == 0: return None，硬生生造了一个可去间断点出来。

修复方法很简单：把 None 改为正确的极限值，问题立刻消失了。

这个教训是：间断点不仅仅是纯数学概念——在数值计算中，它们是实实在在的 bug 来源。如果你的数值算法在某点附近表现异常，先检查被操作函数在该点是否连续。

连续函数的四则运算与复合

连续性在四则运算和复合下表现良好：

  • 如果 f 和 g 都在 x₀ 处连续，那么 f ± g、f·g、f/g（g(x₀) ≠ 0）都在 x₀ 处连续。
  • 如果 g 在 x₀ 处连续，f 在 g(x₀) 处连续，那么复合函数 f(g(x)) 在 x₀ 处连续。

这些结论的证明都可以直接从极限的四则运算法则推出。它们的实际含义是：你可以在连续函数上放心地进行运算和组合，结果函数仍然是连续的（除分母为 0 的点外）。

特别地，多项式和有理函数在其定义域内都是连续的。这得益于幂函数 x^n 的连续性和四则运算法则的保持性。

初等函数的连续性定理

现在我们可以给出一个非常重要的结论——初等函数的连续性定理：

初等函数的连续性定理
一切初等函数在其定义区间（即定义域内的区间）上都是连续的。

这意味着：只要一个函数是由六大基本初等函数通过有限次四则运算和复合得到的，那么它在有定义的任何区间上处处连续。

这个定理的威力怎么强调都不为过。它意味着对于绝大多数你在微积分课中遇到的函数，你不需要每次都去验证它是否连续——你知道它是连续的。而在连续性已知的前提下，求极限就简化为"直接代入"：

如果 f(x) 是初等函数，且 x₀ 属于定义区间，那么

        lim_{x→x₀} f(x) = f(x₀)

        例：
        lim_{x→2} (x^3 + 3x^2 - 1) = 2^3 + 3·2^2 - 1 = 8 + 12 - 1 = 19
        lim_{x→0} e^x = e^0 = 1
        lim_{x→π/2} sin x = sin(π/2) = 1

这正是初等函数连续性定理给我们的"红利"：不必每次都用 ε-δ 证明极限，直接代入即可。

但有一点要注意：定理说的是"定义区间"上连续，而非"定义域"上连续。比如 f(x) = 1/x 的定义域是 (-∞, 0) ∪ (0, +∞)，这是一个不连续的"区域"——因为整个定义域不是一个完整的区间。但在 (-∞, 0) 上连续，在 (0, +∞) 上也连续，这没问题。x = 0 处的间断不在定义域内，不影响连续性结论。

本章我们从极限迈向了连续性，理解了连续函数三要素和间断点的完整分类。连续与间断不是抽象的理论游戏——可去间断点可以通过"重新定义"修复，跳跃间断点意味着函数在该点有"突变"，而无穷与振荡间断点则意味着函数行为在该点彻底失控。连续性也是下一章讨论闭区间上连续函数特殊性质的前提——那些性质（最值定理、介值定理）只有在连续的前提下才成立。

下一章，我们将把连续函数限制在闭区间上，研究这一特殊场景下的几个重要定理。你会发现：一旦函数在闭区间上连续，它会展现出一些全局性的、出乎意料的好性质。这些性质不仅是理论分析的基石，在实际应用中（如方程求根、数值逼近）也有重要意义。
