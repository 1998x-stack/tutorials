第14章 权益估值

2015年牛市顶峰时，一位刚入市的投资者问他的券商客户经理："贵州茅台现在200块一股，贵不贵？"客户经理想了想说："已经翻了一倍了，可能有点贵了——推荐你买这只10块钱的低价股，上升空间大。"后来的故事大家都知道：茅台在2021年最高涨到了2600元，而那只推荐的低价股后来跌到了2块钱以下。

这个真实故事揭示了估值中最核心的困境："便宜"不等于"好投资"，"贵"也不等于"坏投资"。估值不是看股价的绝对值，而是看价格相对于公司创造价值的能力。这一章，我们把"给公司估值"这件事从入门到精通地过一遍——从最经典的股利贴现模型，到中国A股市场的估值实践。

本章核心：估值不是"计算一个数字"，而是"讲一个关于公司未来现金流的故事，并把故事翻译成一个数字"。同一个数字背后，两个分析师可能有两个完全不同的故事。这才是估值真正有趣的地方。

14.1 股利贴现模型（DDM）

股利贴现模型（Dividend Discount Model, DDM）是最古老也是最直觉的估值方法：股票的价值等于其未来所有股利的现值之和。

Gordon增长模型

这是最简单的DDM形式，假设股利以一个恒定的增长率永远持续下去：

# Gordon增长模型（永续增长模型）
# P₀ = D₁ / (r - g)

# 其中：
# P₀ = 当前股票价值
# D₁ = 预期下一期股利
# r = 权益资本成本（通常用CAPM估算）
# g = 股利永久增长率

# 例：某公司下一年股利 = 3元，r = 10%，g = 5%
# P₀ = 3 / (0.10 - 0.05) = 60元

# 如果公司不分红怎么办？DDM失效——需要用其他方法
Gordon模型的致命陷阱：Gordon模型对r和g的微小差异极其敏感。如果r = 10%、g = 5%，合理估值是60元。但如果把g上调到6%，估值就变成75元（上涨25%）。如果同时把r下调到9%，估值达到100元（暴涨67%）。很多分析师正是通过"微调"这些参数来"制造"出任何他们想要的目标价。所以当你看到一份基于DDM的目标价报告时，永远要看它用的是什么样的假设。

多阶段DDM

现实中，公司的股利增长率会变化。典型的做法是分成2-3个阶段：

# 三阶段DDM
# 阶段1（高速增长期）：g = 15%，持续5年
# 阶段2（过渡期）：g从15%线性下降到5%，持续5年
# 阶段3（成熟期/永续期）：g = 5%，永远

# 价值 = 阶段1各年股利现值之和
#      + 阶段2各年股利现值之和
#      + 阶段3的终值现值

# 这种模型更接近现实，但对参数的敏感性仍然很高

14.2 自由现金流模型

对于不分红或分红不稳定的公司，DDM不适用。自由现金流模型（Free Cash Flow Model）是更通用的替代方案。

FCFF（公司自由现金流）估值

FCFF（Free Cash Flow to Firm）是公司在支付了所有经营费用、税款和资本支出后，可供所有资本提供者（股东+债权人）分配的现金流。

# FCFF估值两阶段模型

# 第一步：计算FCFF
# FCFF = EBIT × (1 - 税率) + 折旧摊销 - 资本支出 - 营运资本增加

# 第二步：预测未来FCFF
# 假设未来5年FCFF如下（单位：亿元）：
# 年份    1    2    3    4    5
# FCFF   10   12   14   16   19

# 第三步：计算终值
# 假设永续增长率g = 3%，WACC（加权平均资本成本）= 9%
# 终值（Terminal Value）= FCFF_5 × (1 + g) / (WACC - g)
#                        = 19 × 1.03 / (0.09 - 0.03) = 326.2亿元

# 第四步：将FCFF和终值折现
# 企业价值（EV）= Σ FCFF_t / (1+WACC)^t + TV / (1+WACC)^5
# 股权价值 = 企业价值 - 净债务（总债务 - 现金及等价物）

# 每股价值 = 股权价值 / 总股数

FCFE（股权自由现金流）估值

FCFE = FCFF - 利息支出 × (1 - 税率) + 净新增借款。FCFE直接度量了"属于股东的那部分自由现金流"，用权益资本成本（r）而不是WACC来折现。

专业建议：在实际工作中，FCFF模型比FCFE模型更常用，因为FCFF不受公司资本结构变化的影响（WACC已经考虑了债务成本）。对于高杠杆或资本结构频繁变化的公司（如杠杆收购、高增长科技公司），FCFF估值更加稳健。

14.3 相对估值法

绝对估值模型（DDM、FCFE）理论上严谨，但操作复杂、假设太多。在实际工作中，绝大多数分析师使用相对估值法——通过与可比公司的估值倍数比较来判断一家公司是否被高估或低估。

指标公式适用场景常见陷阱
PE（市盈率）股价 / 每股收益盈利稳定的成熟企业周期性行业在利润顶点PE最低（看似便宜实则贵）
PB（市净率）股价 / 每股净资产金融、保险、房地产资产质量不同时不可比（如银行的不良贷款率差异）
PS（市销率）股价 / 每股营收高增长、尚未盈利的科技公司毛利率不同时PS差异大（亏损但高毛利vs亏本赚吆喝）
EV/EBITDA企业价值 / EBITDA跨行业比较、并购估值需要调整非经常性项目、维护性资本支出差异
PEGPE / 盈利增长率成长股估值增长率能否持续？PEG<1就是低估？不一定

PE的深度理解

PE是最常用也是最容易被滥用的指标。理解PE的关键是把它拆解：

# PE的驱动因子分解
# P/E = (1 - b) / (r - g)

# 其中：
# b = 留存收益比率（1 - 股利支付率）
# r = 权益资本成本
# g = 增长率

# 这个公式说明：
# - 增长率越高 → PE越高（成长股高PE有其合理性）
# - 资本成本越低 → PE越高（低利率环境下"高PE"更合理）
# - 股利支付率越高 → PE越高（但支付率高也意味着投资机会少）

# 当前A股主要指数的PE水平（2025年数据）：
# 上证50: 约10.5倍
# 沪深300: 约12.5倍
# 中证500: 约20倍
# 创业板指: 约32倍
# 科创50: 约45倍
估值不可比的真相：很多投资者喜欢说"A股比美股贵"——沪深300的PE约12.5倍 vs 标普500的约22倍，看起来A股还更"便宜"。但估值差异背后的原因很复杂：行业结构不同（A股金融和周期股占比高）、增长前景不同、无风险利率不同、资本成本和风险溢价不同。单纯比较PE数字，就像比较"北京房价vs纽约房价"——数字本身没有意义，需要理解背后各自的经济逻辑。

14.4 剩余收益模型

剩余收益模型（Residual Income Model, RIM）是近年来重新受到重视的估值方法。它的核心思想是：公司的价值 = 当前账面价值 + 未来超常收益的现值。

超常收益（或经济增加值，EVA）是指公司赚取的超过其资本成本的收益。一个公司只有在赚取高于其资本成本的回报时，才真正创造了价值。

# 剩余收益模型（RIM / EVA模型）
# V₀ = B₀ + Σ RI_t / (1+r)^t

# 其中：
# B₀ = 当前每股账面价值
# RI_t = 第t期的剩余收益 = (ROE - r) × B_{t-1}

# 例：某公司B₀ = 20元，ROE = 15%，r = 10%
# 未来5年剩余收益：
# 第1年：RI₁ = (0.15 - 0.10) × 20 = 1.0元
# 第2年：B₁ = 20 + 1.0 = 21元
#      RI₂ = 0.05 × 21 = 1.05元
# ...
# 假设5年后剩余收益永续增长g = 3%

# 这是一个"质量度量"极佳的模型——
# 它直接衡量公司是否为股东创造了超过资本成本的回报
EVA框架的实践意义：EVA/剩余收益模型对投资者的最大价值不一定是输出一个精确的"目标价"，而是提供了一个思考框架：高ROE且稳定的公司，即使PE看起来有点高，可能仍然是好投资；低ROE的公司，即使PB很低（"破净"），也可能是一个"价值陷阱"——因为它可能永远无法创造超过资本成本的回报。

14.5 中国A股估值实践

最后，我们来讨论在中国A股市场上进行估值需要特别注意的几个问题：

成长股 vs 价值股

A股的成长股（科技、新能源、医药等）往往享受显著的估值溢价，而传统价值股（银行、地产、建筑）长期处于低估值状态。2020-2025年的数据显示：

  • 成长股的PE中枢在30-50倍之间，PEG通常在1.0-2.0之间
  • 价值股的PE中枢在6-12倍之间，PB经常破净
  • 成长股的高PE反映了对未来高增长的预期——但一旦增长不及预期，估值收缩带来的"戴维斯双杀"非常惨烈
  • 价值股的低PE不能简单视为"低估"——它可能反映了市场对这些行业的结构性悲观

周期性行业估值

这是新手最容易踩坑的地方。钢铁、煤炭、有色、化工等周期性行业的特点是：盈利高点时PE最低（看起来很"便宜"），盈利低点时PE最高（看起来很"贵"）。在周期顶部用PE估值会让你以最高价买入，在周期底部用PE会错过最佳买入时机。

对周期股，更好的方法是使用周期调整市盈率（CAPE或Shiller PE），或者直接用PB + 行业供需周期分析来判断。

行业类型首选估值方法辅助方法需要关注的关键变量
消费白马（茅台、伊利）FCFF/DDMPE、PEG品牌力、提价能力、渠道库存
科技成长PS、PEG用户数、ARPU研发费用率、客户留存率
金融（银行、保险）PBPE、股息率不良贷款率、净息差、偿付能力
周期（钢铁、有色）PB + 周期位置判断EV/EBITDA产能利用率、库存周期、需求预期
房地产NAV（净资产价值）PB土储质量、去化率、融资成本
亏损公司PS可比交易法现金消耗率、到达盈利的时间
踩坑实录：2021年初，一位分析师推荐某只新能源龙头股，理由是"PE只有40倍，而行业增速超过50%，PEG < 1，明显低估"。投资者买入后，该公司在2022年确实实现了50%的利润增长，但PE从40倍压缩到了25倍——股价反而下跌了。为什么？因为市场认为行业竞争加剧，未来增速会放缓到30%。PEG模型的隐含假设是"增长率会持续"，但市场定价的核心恰恰是"增长率的变化"。这个案例说明：估值倍数本身是结果，不是原因——原因是对未来增长预期的变化。
本章要点回顾：
  • DDM（Gordon模型）是最经典的估值方法，但对增长率和折现率的微小变化极其敏感
  • 自由现金流模型（FCFF/FCFE）是更通用的替代方案，适用于不分红的公司
  • 相对估值法（PE、PB、PS、EV/EBITDA）是实务中最常用的方法，但需要理解每种倍数的适用场景和陷阱
  • 剩余收益模型（EVA）从"价值创造"的角度评估公司质量
  • A股估值的核心挑战：成长vs价值的估值分化、周期行业的PE陷阱、不同行业需要不同的估值方法

学会了如何给一家公司估值，你就有了一把"价值之尺"。但估值只是投资决策的一半——另一半是如何将这些估值结果转化为一个可执行的组合管理流程。从下一章开始，我们进入投资学的"实务操作"部分，首先学习投资组合管理流程：从投资目标设定、战略资产配置到绩效归因分析的全过程。
