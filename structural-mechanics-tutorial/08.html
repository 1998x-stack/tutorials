引言：从静定到超静定——现实世界没有"恰好"

同学们好，欢迎来到结构力学真正的"硬核"部分。在前七章中，我们处理的所有结构都有一个共同点：它们是静定结构。也就是说，仅仅依靠三个平衡方程（\(\sum F_x = 0\)、\(\sum F_y = 0\)、\(\sum M = 0\)），我们就能确定所有的支座反力和杆件内力。这是一个非常"舒服"的世界——一个方程、一个答案、清晰明了。

但现实世界从来不是一个"恰好"的世界。同学们有没有想过一个问题：为什么实际工程中几乎不存在静定结构？你看一座连续梁桥——它有好几个桥墩，每一跨都延续过去。一个简支梁桥，支座坏了桥就塌了；但连续梁桥的一个支座沉降了，其他支座会分担荷载，结构不一定倒塌。这就是超静定结构的冗余度（Redundancy）——它有多余的约束，因而有更强的安全性。工程规范通常要求主要承重结构具有一定的冗余度，就是基于这个道理。

2018年，美国佛罗里达州迈阿密的弗罗里达国际大学（FIU）一座正在施工的人行天桥突然坍塌，造成6人死亡。事后调查发现，该桥设计为静定的"彩虹拱"结构——一根拉杆失效后，整个结构没有任何荷载重分配的路径，瞬间倒塌。调查报告明确指出：缺乏足够的结构冗余度是事故的关键原因之一。这个惨痛的案例告诉我们：超静定结构不仅是一个力学概念，更是一种安全保障。

从本章开始，我们将正式进入超静定结构（Statically Indeterminate Structure）的分析。而我们要学的第一种方法，就是历史最悠久、逻辑最根本的——力法（Force Method）。

8.1 超静定结构的概念

先给一个严格的定义：超静定结构是指结构的未知力（支座反力 + 杆件内力）个数超过了独立静力平衡方程的个数，仅靠平衡方程无法求解的结构。

我们用一个非常直观的例子来理解。一根简支梁，两端各有一个支座。两个未知反力（一个固定铰有两个分力，一个活动铰有一个分力），共3个未知数。平面力系有3个平衡方程，解得出来——这是静定结构。现在把右端的活动铰也换成固定铰——于是两个固定铰支座给出4个未知反力（每个支座两个分力），但平衡方程还是3个。第4个未知数从哪里来？这多出来的约束，就是"多余约束"。超静定结构就是有多余约束（Redundant Constraints）的结构。

关键理解："多余"并不意味着"没用"。在力学意义上，"多余"是指这个约束在维持结构几何不变性方面不是必需的——去掉它，结构仍然是几何不变的。但从工程角度看，这些"多余"约束恰恰提供了安全冗余，是结构韧性的来源。

8.2 超静定次数的确定

在力法中，第一个实操步骤就是：确定超静定次数（Degree of Indeterminacy）。它告诉你：需要多少个额外的方程才能求解。

定义与公式

超静定次数 = 未知力总数 - 独立静力平衡方程总数

对于平面体系，每个构件有3个独立平衡方程。更系统的做法是利用几何组成分析的结论：

n = W + m

其中，n为超静定次数，W为计算自由度（第2章的内容），m为多余约束数。但更常用的方法是直接"数": 把结构中所有多余约束的数量加起来。

常见结构的超静定次数判定方法：

结构类型 判定方法 示例
连续梁 多余支座数 = 支座数 - 2 4跨连续梁有5个支座，超静定次数 = 5 - 2 = 3次
单跨刚架 支座约束总数 - 3 两端固定刚架：每端3个约束，共6，超静定次数 = 6 - 3 = 3次
封闭框 每个封闭框有3次超静定 一个矩形框：3次超静定
铰接桁架 m - (2j - 3)，m为杆数，j为结点数 m=15, j=9, 超静定次数 = 15 - (18-3) = 0次（静定桁架）

更通用的计算公式为：

n = 3m - (3g + 2h + r)

其中 m 为刚片（杆件）数，g 为刚结点数，h 为铰结点数（注意：连接n根杆的铰相当于n-1个单铰），r 为支座链杆数。这个公式来自几何组成分析的"搭积木"思想——每根杆件引入3个自由度，每个约束扣除相应的自由度数。

几个课堂例题

来看几个典型例子：

例1：两端固定梁（无中间支承）
  ┌──────────────────────┐
  │                      │
  ╞══════════════════════╡
  Δ                      Δ
固定端                  固定端
支座约束：左端3个 + 右端3个 = 6
独立方程数：3（平面力系）
超静定次数：6 - 3 = 3次

例2：两跨连续梁
  ┌──────────────────────┐
  │                      │
  △        △            △
铰支      铰支          铰支
支座约束：3个铰支座 = 3 + 2 + 2 = 7
独立方程数：3
超静定次数：7 - 3 = 4次
——但等等！这里的"梁"没有中间的刚结点，
铰支座竖向反力可通过简单的梁平衡求出部分未知量，
重新分析更合理——实际上是1次超静定
踩坑故事：我曾见过一位研究生在分析一个3跨连续梁时，把每个支座的反力都当作独立未知量，数出了"6次超静定"的结论——因为他把每个铰支座都算成2个未知力。他忘了：对于连续梁来说，中间铰支座的水平方向是自由滑动的（活动铰），不提供水平约束。这个错误导致他列出的力法方程数比实际多了一倍，足足多算了3个多余未知量。所以，判定超静定次数前，一定要先搞清楚每个支座的约束类型——固定铰还是活动铰，不能想当然。

8.3 力法的基本原理

确定了超静定次数之后，我们要问：如何求解这些"多余"的内力或反力？

力法的核心思想非常朴素：去掉多余约束，代之以未知力，使原超静定结构变成静定结构，然后利用"去掉约束处的位移与原结构一致"这一条件来求解这些未知力。

说白了就是三步曲：

  • 第一步：拆——去掉多余约束，得到一个静定结构，称为基本体系（Primary System）或力法基本结构。
  • 第二步：补——把去掉的多余约束以"未知力"的形式加回去，设为 \(X_1, X_2, \dots, X_n\)。
  • 第三步：等——基本体系在荷载和这些未知力的共同作用下，在去掉约束处的位移应该等于原结构在该处的实际位移（一般是零，除非有支座移动）。由此建立方程。
力法的灵魂：力法本质上是在"恢复"原结构的变形状态。你拆掉的约束越多，基本体系就越"软"（静定），它在荷载下的变形就越大。你要找的那些多余未知力 \(X_i\)，就是那些"恰好把变形拉回到原结构状态"的力。这就是著名的变形协调条件（Compatibility Condition）。

8.4 基本体系与原结构

基本体系的选择是力法中非常关键的一步。选择不同的多余约束，会得到不同的基本体系——但最终结果是一样的。这就像解方程组时，你可以消去不同变量，但最终的解是唯一的。

选择基本体系的原则：

  • 去掉的约束必须是"多余"的——去掉后结构仍然是几何不变的（不能变成机构）。
  • 基本结构最好是静定结构，以便于计算内力（如简支梁、悬臂梁等）。
  • 尽量使基本结构简单、对称，以简化计算。

举个例子——一个两端固定的单跨梁（3次超静定）：

原结构：两端固定梁（3次超静定）
  ┌──────────────────────┐
  │                      │
  ╞══════════════════════╡
  Δ                      Δ

选择1：去掉左端转动约束 → 左端改为铰支
  保留3个约束 → 变成1次超静定，不够简单

选择2：去掉两端转动约束 → 两端改为铰支
  再释放轴向约束 → 得到简支梁（静定结构）
  多余未知力：梁端弯矩 M_A, M_B, 轴力 F_AB
  基本体系的变形条件：
  θ_A = 0, θ_B = 0, Δ_AB = 0

8.5 力法典型方程

现在我们来建立力法的核心方程。假设一个结构有n次超静定，我们去掉n个多余约束，得到基本体系。在基本体系中，多余未知力记为 \(X_1, X_2, \dots, X_n\)。

变形协调条件是：在荷载和所有多余未知力共同作用下，基本体系沿每个 \(X_i\) 方向的位移等于原结构在该方向的位移（通常为零）。

利用叠加原理（Superposition Principle），我们可以写出：

\[ \begin{cases} \delta_{11}X_1 + \delta_{12}X_2 + \cdots + \delta_{1n}X_n + \Delta_{1P} = 0 \\ \delta_{21}X_1 + \delta_{22}X_2 + \cdots + \delta_{2n}X_n + \Delta_{2P} = 0 \\ \vdots \\ \delta_{n1}X_1 + \delta_{n2}X_2 + \cdots + \delta_{nn}X_n + \Delta_{nP} = 0 \end{cases} \]

这就是力法典型方程（Canonical Equations of Force Method）。其中：

  • \(\delta_{ij}\)——柔度系数（Flexibility Coefficient），表示基本体系中，由 \(X_j = 1\) 引起的沿 \(X_i\) 方向的位移。\(\delta_{ii}\) 称为主系数，恒为正。根据位移互等定理，\(\delta_{ij} = \delta_{ji}\)。
  • \(\Delta_{iP}\)——荷载项（Load Term），表示基本体系中，由实际荷载引起的沿 \(X_i\) 方向的位移。
记住这个特点：力法方程是位移方程——方程两边都是"位移"，单位也是"位移"。这和后面要学的位移法方程（平衡方程）有本质区别。力法方程的右边是零（或者已知位移值），表示"变形协调"。

所有系数和自由项都可以通过我们第7章学过的单位荷载法或图乘法来计算：

\(\delta_{ij} = \int \frac{\bar{M}_i \bar{M}_j}{EI} \mathrm{d}s + \int \frac{k \bar{F}_{Qi} \bar{F}_{Qj}}{GA} \mathrm{d}s + \int \frac{\bar{F}_{Ni} \bar{F}_{Nj}}{EA} \mathrm{d}s\)

对于梁和刚架，弯曲变形占主导，通常只保留弯矩项。

8.6 力法计算步骤与例题

标准计算步骤

力法分析一个超静定结构的标准流程如下：

  1. 确定超静定次数——数清楚有多少个多余约束。
  2. 选择基本体系——去掉多余约束，代之以未知力。
  3. 列出力法典型方程——n次超静定就有n个方程。
  4. 计算系数和自由项——画单位内力图（\(\bar{M}_i, \bar{F}_{Qi}, \bar{F}_{Ni}\)）和荷载内力图（\(M_P, F_{QP}, F_{NP}\)），用图乘法或积分求\(\delta_{ij}\)和\(\Delta_{iP}\)。
  5. 求解多余未知力——解线性方程组。
  6. 求最终内力——利用叠加公式：\(M = \bar{M}_1 X_1 + \bar{M}_2 X_2 + \cdots + \bar{M}_n X_n + M_P\)。
  7. 画最终内力图——弯矩图、剪力图、轴力图。

完整例题：一次超静定梁

计算如图所示一次超静定梁的内力（EI = 常数）：

  ┌──────────────────────────────────┐
  │  q = 10 kN/m                     │
  │                                  │
  △──────────────────────────────────△
  A                                  C  B
  ├──────────── 6m ──────────────────┤
                                      ▲
                                      │
                                   活动铰

第一步：确定超静定次数

该梁有2个固定铰支座（A、C各2个约束）和1个活动铰支座（B处1个约束）= 共5个约束。平衡方程3个，超静定次数 = 5 - 3 = 2次？且慢——这是一个连续梁，C支座为活动铰。实际上是A处固定铰（2约束）+ B、C处活动铰（各1约束） = 4个约束，超静定次数 = 4 - 3 = 1次。

第二步：选择基本体系

去掉B支座处的竖向约束，代之以未知力 \(X_1\)，得到基本体系：

基本体系：简支梁（在A和C处铰支）+ 未知力 X₁
  ┌──────────────────────────────────┐
  │  q = 10 kN/m            X₁ ↑     │
  │                                  │
  △──────────────────────────────────△
  A                                  C

第三步：力法方程

变形协调条件：基本体系在B点的竖向位移 = 0（原结构B支座没有沉降）。

\(\delta_{11} X_1 + \Delta_{1P} = 0\)

第四步：计算系数

先画单位弯矩图 \(\bar{M}_1\)（在B点加单位力 \(X_1 = 1\) 时的弯矩图）：

单位力在B点，作简支梁在单位集中力下的弯矩图。利用图乘法：

\(\delta_{11} = \int \frac{\bar{M}_1^2}{EI} \mathrm{d}s = \frac{1}{EI} \times (\text{弯矩图自乘})\)

等截面梁，通过图乘法计算得：\(\delta_{11} = \frac{4l^3}{243EI}\)（其中l为总跨度）

再画荷载弯矩图 \(M_P\)（只有荷载q作用下的弯矩图）：

\(\Delta_{1P} = \int \frac{M_P \bar{M}_1}{EI} \mathrm{d}s = \frac{1}{EI} \times (\text{两图相乘})\)

第五步：求 \(X_1\)

\(X_1 = -\Delta_{1P} / \delta_{11}\)

代入具体数值计算即可得到B支座反力。注意符号：如果 \(\Delta_{1P}\) 为正（向下位移），则 \(X_1\) 为负（向上），表示B支座实际承受向上的压力。

第六步：叠加求最终弯矩

\(M = \bar{M}_1 X_1 + M_P\)，即可画出梁的最终弯矩图。

力法计算的一个关键细节：很多初学者算出 \(X_1\) 之后就直接下结论了。请一定记住——\(X_1\) 只是多余约束力（在这里是B支座的竖向反力），你还需要用叠加公式 \(M = \bar{M}_1 X_1 + M_P\) 计算出每个截面的总弯矩，画出最终的弯矩图。没有弯矩图，你无法判断哪个截面需要多大的配筋。而画弯矩图是结构分析的核心产出之一。

8.7 力法的适用范围

力法虽然经典，但并非万能。它的特点和适用场景如下：

特点 说明
优点 物理概念清晰，直接以"力"为未知量，适合低次超静定结构的手算
优点 能处理温度变化、支座移动等非荷载因素引起的结构内力
优点 适合分析连续梁、简单刚架
局限 超静定次数高时，方程数量大，手算极为繁琐
局限 对复杂刚架，基本体系的选择对计算量影响巨大——选择不当会使计算量指数增加
局限 难以实现程序化——不同结构需要选择不同的基本体系，不具备通用算法结构

本章我们学习了力法的核心内容：

  • 超静定结构的概念——有多余约束，仅靠平衡方程无法求解；
  • 超静定次数的判定——数清楚多余约束的数量，这是力法的第一步；
  • 力法的基本原理——"拆除多余约束、代以未知力、利用变形协调条件"；
  • 基本体系与原结构——基本体系的选择方法；
  • 力法典型方程——\(\delta_{ij} X_j + \Delta_{iP} = 0\)，以及柔度系数的概念；
  • 力法标准计算步骤——从确定次数到画内力图；
  • 一个完整的一次超静定梁例题。

力法是结构力学中逻辑最优雅的方法之一——它把超静定问题"降维"为静定问题，再用变形协调条件把超静定效应"补偿"回来。但力法有一个明显的弱点：超静定次数一高，手工计算量就变得难以承受。
