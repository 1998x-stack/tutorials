第22章 相对论与电磁学

爱因斯坦 16 岁时问了自己一个问题："如果我以光速追着一束光跑，会看到什么？"10 年后，他用狭义相对论回答了这个问题——而电磁学正是相对论的"出生地"。没有麦克斯韦方程组与经典力学的矛盾，就不会有相对论。从某种意义上说，相对论是从电磁学的内在矛盾中诞生的。

电磁学中的矛盾

19 世纪末的物理学家面临一个深刻的问题：电磁学定律与牛顿力学的伽利略相对性原理似乎不相容。

法拉第悖论

考虑一个经典的电磁感应实验：

  • 情况 A：磁铁运动，线圈静止。线圈中感应出电流。解释：变化的磁场产生电场。
  • 情况 B：磁铁静止，线圈运动。线圈中同样感应出电流。解释：洛伦兹力 F = q(v × B) 推动电荷。

实验结果是两种情况下的感应电流完全相同——但按照 19 世纪的理论，这是两种完全不同的物理机制。为什么两种机制给出完全相同的结果？这只是巧合吗？

以太问题

麦克斯韦方程组预言了光速 c = 1/√(μ₀ε₀) ≈ 3×10⁸ m/s。但"光速是相对于什么参考系的？"按照经典物理，所有波都需要介质——声波需要空气，水波需要水。当时物理学家假设存在一种叫做"以太"（ether）的介质作为光的载体。光速应该是相对于以太的。

1887 年，Michelson 和 Morley 设计了精巧的干涉实验试图测量地球穿过以太的运动，结果是：零。不管地球怎么运动，光速在所有方向上都相同。这个"零结果"是经典物理学史上最著名的实验结果——它直接摧毁了以太假说。

洛伦兹变换

1905 年，爱因斯坦发表了狭义相对论，以两个简洁的公设彻底解决了上述矛盾：

  1. 相对性原理：物理定律在所有惯性参考系中形式相同。
  2. 光速不变原理：真空中的光速在所有惯性参考系中均为常数 c。

从这两个公设可以推导出联系两个惯性系的洛伦兹变换：

x' = γ(x − vt)
t' = γ(t − vx/c²)
γ = 1 / √(1 − v²/c²)

洛伦兹变换取代了经典物理中的伽利略变换，带来了革命性的物理意义：

概念伽利略变换（经典）洛伦兹变换（相对论）
时间绝对，t' = t相对，t' = γ(t − vx/c²)
空间绝对，x' = x − vt相对，x' = γ(x − vt)
速度叠加u' = u − vu' = (u − v)/(1 − uv/c²)
光速可变 u' = c − v不变 u' = c
同时性绝对相对（取决于参考系）

相对论速度叠加公式：

def relativistic_add(u, v):
    """
    相对论速度叠加 (一维)

    参数:
        u: 物体在 S 系中的速度 (单位: c, 即 u/c)
        v: S' 系相对于 S 系的速度 (单位: c)

    返回:
        u_prime: 物体在 S' 系中的速度 (单位: c)
    """
    return (u - v) / (1 - u * v)

# 示例: 在 S 系中 u = 0.9c, S' 系以 v = 0.8c 运动
u_prime = relativistic_add(0.9, 0.8)
print(f"u' = {u_prime:.4f}c")
# 输出约 0.357c, 而不是经典结果的 0.1c

# 验证: 光速不变
print(f"光: u' = {relativistic_add(1.0, 0.8):.4f}c")
# 输出 1.0c — 光速不变!
核心物理洞察：时间膨胀和长度收缩并不是"幻觉"或数学技巧——它们是时空本身的几何性质。在相对论中，时空是一个统一的四维流形（闵可夫斯基时空），时间和空间不再是独立的实体。

电磁场的相对论变换

狭义相对论最深刻的成就之一是揭示了电场和磁场不是独立的物理实体，而是同一个电磁场张量的不同分量。换言之，E 和 B 的区分取决于观察者的运动状态。

电磁场从一个惯性系变换到另一个惯性系的变换规则：

  • 平行于运动方向的分量：不变
    E'∥ = E∥，B'∥ = B∥
  • 垂直于运动方向的分量：混合变换
    E'⊥ = γ(E + v × B)⊥
    B'⊥ = γ(B − v × E / c²)⊥

这些变换公式表明：在一个参考系中只有电场、没有磁场的物理情境，在另一个参考系中会同时出现电场和磁场——反过来也一样。这就是为什么"运动磁铁产生电场"这个看似矛盾的现象在相对论中变得自然。

一个简单的记忆方法：电场和磁场的关系类似于时间和空间的关系。就像"过去和未来"取决于观察者的运动状态一样，"电场和磁场"也取决于观察者的运动状态。E 和 B 是同一个物理实体——电磁场——在不同"截面"上的投影。

重新理解"动生电动势"

有了电磁场变换，我们可以重新审视本章开头提到的法拉第悖论。考虑一个在均匀磁场 B 中以速度 v 运动的导体棒：

  • 实验室系：导体棒中电荷受到洛伦兹力 F = q(v × B)，正负电荷分别积累在棒两端，产生动生电动势。
  • 导体棒静止系：导体棒静止，磁场变换后出现了一个电场 E' ≈ v × B（当 v ≪ c 时 γ ≈ 1）。正是这个变换出的电场推动电荷分离，产生电动势。

两种视角计算的电动势完全相同。动生电动势不是"导线切割磁感线"的魔术，而是电磁场在不同参考系下的正确变换。

电磁场的四维形式

相对论将电磁学提升到了前所未有的优雅和统一。引入四维时空坐标 xμ = (ct, x, y, z)，电磁学可以写成极其紧凑的四维形式：

四维势：

Aμ = (V/c, A)

其中 V 是标量势，A 是矢量势。四维势在洛伦兹变换下像一个四维矢量一样变换。

电磁场张量：

Fμν = ∂μAν − ∂νAμ

这是一个 4×4 反对称张量。其分量正好对应电场和磁场：

Fμν = [0, −Ex/c, −Ey/c, −Ez/c;
 Ex/c, 0, −Bz, By;
 Ey/c, Bz, 0, −Bx;
 Ez/c, −By, Bx, 0]

麦克斯韦方程组的协变形式：

∂μFμν = μ₀ Jν （由电荷电流产生场）
∂μ̃Fμν = 0 （无磁单极）

其中 Jν = (cρ, J) 是四维电流密度，̃Fμν 是对偶张量。这四个方程（ν = 0,1,2,3 各一个分量）等价于麦克斯韦方程组原来的四个方程。

极致的美：原本需要四个矢量方程的麦克斯韦方程组，在四维时空中只需要两行优雅的协变方程。这不仅是一个数学简化——它揭示了电磁学与时空结构的深刻联系。正如物理学家所说："磁场是电场的相对论修正——数量级 v²/c²"。当 v ≪ c 时，磁效应可以视为电效应的一阶修正；当 v → c 时，电场和磁场完全融合为一体。

电磁学与相对论的意义

相对论对电磁学的深刻影响可以总结为三个层面：

  1. 统一性：电场和磁场不是两种不同的力，而是同一个电磁场的不同投影。这就像一个人的正面和背面——你看到哪个取决于你站的位置。
  2. 一致性：麦克斯韦方程组自动满足狭义相对论的要求——它们天生就是洛伦兹协变的。经典力学则不然，需要修改（相对论力学）。这意味着电磁学"猜对了"时空结构，而牛顿力学没有。
  3. 局限性：经典电磁学描述的是宏观尺度上的电磁现象。对于永磁体中的铁磁性，根源在于电子自旋——这是一个纯粹的量子力学效应，不能仅用经典相对论解释。
真实故事：一位研究生在学完电磁学后困惑地问教授："如果磁场只是电场的相对论效应，那为什么永磁体有磁场？"教授的回答是："要理解铁磁体内部电子自旋的集体排列，你需要的是量子力学，不是相对论。"这是一个经典的误解：相对论告诉我们磁场如何变换，但没有告诉我们为什么存在永磁体。后者的答案在于电子自旋——一个无法用经典物理学解释的纯量子现象。
