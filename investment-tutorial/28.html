第28章 综合实战项目

2015年6月12日，上证指数站上5178点。一位北京的投资者老王在朋友的怂恿下，将家里的300万积蓄全部投入了股市——"国家需要牛市"、"万点不是梦"。他没有资产配置，没有风险预算，甚至不知道PE是什么。两个月后，指数跌到了2850点，300万变成了120万。

老王的问题不在于"选错了股票"。他买的其实是一些不错的蓝筹股。他的问题在于：没有计划。没有投资目标（赚了就跑还是长期持有？），没有风险预算（最多能亏多少？），没有再平衡纪律（跌了加仓还是割肉？）。如果他在入市前花一周时间写好一份投资计划书（IPS），那一年的大熊市中他可能不仅不会亏钱，反而能抓住低位建仓的机会。

本章就是教你如何成为"老王的反面"——一个有计划的投资者。我们将模拟管理一个1000万人民币的投资组合，期限3年，从撰写IPS到归因分析，完整走完投资组合管理的全部流程。

项目设定：你是一名个人投资者，于2026年7月获得了一笔1000万人民币的闲置资金（可能是出售房产、年终奖金或遗产继承），投资期限3年（2026年7月 - 2029年6月）。你的目标是：在严格控制下行风险的前提下，实现超越通胀的稳健增值。

步骤1：撰写投资政策声明（IPS）

投资政策声明（Investment Policy Statement, IPS）是投资组合管理的"宪法"。好的IPS让你在市场恐慌时有据可依，而不是凭情绪决策。

收益率目标

  • 名义年化收益率目标：5-7%（跑赢2026年预期的GDP增速6%左右）
  • 实际年化收益率目标：≥3%（扣除了约2.5%的预期通胀后，实现真实财富增长）

风险预算

  • 最大回撤容忍度：不超过15%
  • 年化波动率目标：8-12%
  • VaR（95%，1个月）：不超过投资组合价值的5%

约束条件

  • 流动性：至少10%的资产保持高流动性（货币基金、国债逆回购），以应对突发资金需求
  • 投资范围限制：不投资单一股票（通过基金/ETF配置权益），不投资杠杆产品（期货、期权仅用于对冲），不投资非持牌金融机构的产品
  • 税收考虑：A股个人投资者的股息红利和资本利得目前免税，但需关注政策变化
IPS模板下载：你可以创建一个自己的IPS文档，包含以上所有要素。当市场大跌10%时重新翻开IPS——如果你的组合仍然在风险预算范围内，冷静持有；如果超出了风险预算，再考虑调整。

步骤2：资本市场假设

资本市场假设（Capital Market Assumptions, CMA）是你对各类资产未来3年预期收益、风险和相关系数的判断。以下是基于2026年6月市场环境的假设示例：

资产类别预期年化收益预期年化波动率配置比例上限
沪深300指数（A股）8%18%60%
中证500指数（中小盘）10%22%20%
中证国债指数2.5%3%40%
中证信用债指数3.5%5%30%
黄金（人民币计价）5%12%15%
货币基金1.8%0.5%20%

资产之间的相关系数矩阵（简化版）：

# 相关系数矩阵（简化版）
assets = ['沪深300', '中证500', '国债', '信用债', '黄金', '货币']
corr_matrix = np.array([
    [1.00, 0.75, -0.20, -0.10, 0.15, 0.00],  # 沪深300
    [0.75, 1.00, -0.15, -0.05, 0.20, 0.00],  # 中证500
    [-0.20, -0.15, 1.00, 0.60, 0.10, 0.05],  # 国债
    [-0.10, -0.05, 0.60, 1.00, 0.15, 0.05],  # 信用债
    [0.15, 0.20, 0.10, 0.15, 1.00, 0.00],    # 黄金
    [0.00, 0.00, 0.05, 0.05, 0.00, 1.00],    # 货币基金
])

# 预期收益向量（小数）
expected_returns = np.array([0.08, 0.10, 0.025, 0.035, 0.05, 0.018])

# 预期波动率向量（小数）
expected_vol = np.array([0.18, 0.22, 0.03, 0.05, 0.12, 0.005])

# 协方差矩阵 = 相关系数 * 波动率外积
cov_matrix = corr_matrix * np.outer(expected_vol, expected_vol)

print("协方差矩阵构建完成。接下来进行约束条件下的最优化...")

步骤3：战略资产配置

现在我们有收益预期、风险参数和相关系数，下一步是用马科维茨均值-方差优化确定最优权重。但经典的有效前沿有众所周知的缺陷：对输入参数过于敏感，经常给出极端权重（如200%债券、-100%股票）。因此我们施加了约束条件：

  • 每个资产类别的权重在0%到上限之间（见上表）
  • 权益类总权重 ≤ 70%
  • 固定收益类总权重 ≤ 50%
  • 所有权重之和 = 100%

在约束条件下优化，我们得到了以下战略资产配置方案（SAA）：

资产类别目标权重波动率贡献功能定位
沪深30035%63%核心权益仓位，受益于中国慢牛格局
中证50010%22%中小盘弹性配置，增强收益
国债ETF25%6%稳定性压舱石，对冲股市风险
信用债/企业债ETF10%4%票息增强，适度信用风险暴露
黄金ETF10%4%尾部风险对冲，通胀对冲
货币基金10%1%流动性管理，抄底备用金

这个组合的预期年化收益约为5.8%，预期年化波动率约为10.5%，预期最大回撤约12%（在95%的置信区间内），符合IPS设定的风险预算。

重要提醒：量化优化只是起点，不是终点！上面的数字依赖于你输入的假设——如果实际利率走势与假设不同，组合的表现也会不同。因此，每年至少重新评估一次CMA假设。

步骤4：策略实施

SAA确定后，进入具体的执行阶段。

选股/选基策略

  • 权益端（45%）：采用"核心-卫星"策略。核心部分（30%）配置宽基指数ETF（沪深300+中证500），卫星部分（15%）配置3-5只主动管理型基金，选择标准：基金经理任职超5年、年化超额收益超4%、最大回撤小于市场
  • 债券端（35%）：全部使用ETF——国债ETF（25%）+信用债ETF（10%），利用ETF的免税优势和低费率
  • 黄金（10%）：黄金ETF（如华安黄金ETF），交易方便、流动性好
  • 货币基金（10%）：场内货币ETF，T+0交易，可随时用于补仓或应急

组合监控与再平衡规则

再平衡是投资组合管理中"最反人性但最有效"的操作。

  • 定期再平衡：每半年（6月和12月）将权重调回目标比例。如果股市大涨导致权益权重超标，卖出部分股票买入债券——这是在"高卖低买"
  • 阈值再平衡：任何单类资产的权重偏离超过5个百分点时，立即触发再平衡。例如，如果股市大跌导致沪深300从35%降到27%，立即从货币基金中挪出资金补仓
  • 异常情况处理：如果最大回撤达到12%（接近15%的警戒线），召开"组合评审会"——重新评估CMA假设，判断是否需要调降权益仓位
实战技巧：把再平衡的日期设为日历提醒，强制执行。在2020年3月市场最恐慌的时候做了再平衡的人（卖出债券、买入股票），在随后的反弹中获得了显著的超额收益。再平衡的真正价值在于——它强迫你在别人恐惧时贪婪。

步骤5：归因分析

三年后（2029年6月），组合到期。我们需要回答两个问题：赚了多少？为什么赚/亏？这就是绩效归因分析的工作。

Brinson归因分解

Brinson模型将超额收益分解为三个部分：配置效应（Asset Allocation）、选股效应（Security Selection）和交互效应（Interaction）。

# Brinson归因模型简化示例（假设数据）
# 基准：50%沪深300 + 30%国债 + 10%黄金 + 10%货币

actual_returns = {  # 3年累计实际收益
    '沪深300': 0.25, '国债': 0.08, '黄金': 0.15, '货币': 0.05
}
benchmark_returns = {  # 3年基准收益
    '沪深300': 0.22, '国债': 0.07, '黄金': 0.12, '货币': 0.05
}
actual_weights = {'沪深300': 0.38, '国债': 0.28, '黄金': 0.12, '货币': 0.22}
benchmark_weights = {'沪深300': 0.50, '国债': 0.30, '黄金': 0.10, '货币': 0.10}

# 配置效应 = sum( (实际权重 - 基准权重) * 基准收益 )
allocation_effect = sum(
    (actual_weights[a] - benchmark_weights[a]) * benchmark_returns[a]
    for a in actual_weights
)  # 结果：+0.7%（超配国债和黄金起了正面作用）

# 选股效应 = sum( 基准权重 * (实际收益 - 基准收益) )
selection_effect = sum(
    benchmark_weights[a] * (actual_returns[a] - benchmark_returns[a])
    for a in actual_weights
)  # 结果：+1.2%（沪深300和黄金选基优于基准）

# 总超额收益 = 配置效应 + 选股效应 + 交互效应
# 交互效应 = sum( (实际权重-基准权重) * (实际收益-基准收益) )

风险归因

除了收益归因，还要回答：组合的风险来自哪里？使用第16章学过的风险分解方法，我们可以计算每类资产对组合总风险的贡献。假设三年总波动率为11.2%，风险归因如下：

资产类别权重标准差贡献占总风险比例
沪深30038%6.8%61%
中证50012%2.5%22%
债券组合30%1.0%9%
黄金12%0.7%6%
货币基金8%0.2%2%

这个结果告诉我们：虽然权益只占50%的权重，但它贡献了超过80%的风险——这很正常，因为权益资产的波动率远高于其他资产。如果我们想进一步降低风险，唯一有效的办法是降低权益权重。

报告撰写

一份合格的季度/年度投资报告应包括以下部分：

  1. 业绩摘要：组合总收益、与基准比较、期间最大回撤
  2. 市场回顾：本季度/年度市场的宏观环境和主要驱动因素
  3. 归因分析：Brinson分解（配置/选股效应）
  4. 风险报告：当前VaR、波动率、最大回撤，与IPS风险预算对比
  5. 组合持仓明细：各资产类别、各基金/ETF的持仓和收益
  6. 下一步计划：下一期的再平衡安排、调整理由
实战项目最终成果：假设我们的组合在3年内实现了17.6%的累计收益（对应年化约5.55%，扣除管理费和交易成本后约4.8%），最大回撤11.3%（未触发15%的警戒线），夏普比率0.82。这个结果不完全符合年初的"最佳预期"，但它是一个可执行的、有纪律的、在风险预算内的投资计划。对于一个个人投资者来说，这已经是一个合格的成绩单了。

但我想用三个超越知识的观念来结束这门课：

第一，投资是认知的变现，更是纪律的考验。你不需要成为最聪明的投资者——你只需要成为最有纪律的投资者。每一条IPS中的规则，都是在情绪最激动的时候拉住你的那根绳子。巴菲特说"在别人恐惧时贪婪"——但如果没有IPS和再平衡纪律，你根本分不清什么时候是"别人恐惧"、什么时候是"自己恐惧"。

第二，永远不要停止学习。2026年的市场与2016年截然不同，2036年又会与今天截然不同。投资学是一个"终身学习"的领域——新的工具（AI、区块链）、新的数据（另类数据、高频数据）、新的市场环境（低利率、高波动）都在不断涌现。但你的知识体系（资产配置、风险管理、行为金融）会持续生效——就像马科维茨1952年的论文，到今天仍然是智能投顾的核心算法。

第三，投资不是人生的全部。如果你为了省下一顿午饭的钱而焦虑地盯着K线图，或者因为市场大跌而整夜失眠——你可能"投资"得太多了。投资的目标不是成为最富有的人，而是成为财务上最自由的人。真正的自由不是你拥有多少钱，而是你可以在不被金钱绑架的前提下，过你想要的生活。

终身学习建议：
  • 书籍：继续阅读《聪明的投资者》（本杰明·格雷厄姆）、《漫步华尔街》（伯顿·马尔基尔）、《黑天鹅》（纳西姆·塔勒布）
  • 期刊：关注《Journal of Finance》、《金融研究》等学术期刊的最新研究
  • 实践：用小额资金开始实盘操作，但始终遵守写的IPS
  • 圈子：加入投资者社区，与同行交流，但保持独立判断
  • 反思：每季度写一份"投资日记"，记录每一笔投资决策的理由和结果

感谢你读完这28章课程。从第1章Susan的故事到本章老王的故事，从马科维茨的有效前沿到2025年的PAPM模型，从传统股票配置到DFPM动态因子模型——你学到了投资学的全部核心内容。

现在，该轮到你上场了。打开你的笔记本，写下你的IPS，确定你的资产配置，然后——去执行它。投资从来不是"开始行动"这件事最难，难的是一辈子都坚持正确的方法。但这正是我们学完28章之后，已经准备好的事情。
