第22章 非概率抽样校正

大家好，我是你们的讲师。如果说前21章我们一直在"理想世界"里讨论抽样——我们有概率抽样、设计良好的抽样框、已知的入选概率——那么从这一章开始，我们不得不面对"真实世界"的残酷现实。在真实世界中，大量可用的数据来自非概率抽样（Non-probability Sampling）：网络问卷、社交媒体数据、客户数据库、便利样本……这些数据虽然量大、便宜、获取快，但由于不知道入选概率，无法直接做无偏推断。

让我们把时钟拨回到2020年初。新冠肺炎疫情在全球爆发，公共卫生决策者急需了解病毒的传播特征、人群的感染率和症状分布。在短短几周内，全球数十个研究团队发起了在线调查。这些调查通过社交媒体传播，采用便利样本（convenience sample）和滚雪球方式扩散。其中一项著名的研究收集了数万份回复，得出了"发热是最常见症状"等结论。但问题是：这些样本能代表全球人口吗？显然不能——只有有网络访问条件、有闲暇时间填写问卷、且对疫情足够关注的人才会参与。这群人更可能是年轻人、城市居民、高学历者。幸运的是，统计学家们在过去几十年里已经发展出了一套方法来尝试"校正"这类非概率样本的偏差。本章就是关于这些方法的。

非概率抽样校正的"免责声明"：在开始之前，我必须坦诚地说——非概率抽样的校正是一门"补救"科学，而不是"保险"科学。没有任何事后校正方法能保证将一个有偏的非概率样本变成一个无偏的代表性样本。校正的效果取决于：(1) 是否有高质量、可靠的参考数据（如概率调查或普查数据）；(2) 偏差的来源是否可以被观测到的变量所解释；(3) 非概率样本的覆盖范围是否足够广。如果这三点都不能满足，再花哨的统计方法也无法挽回。所以，这一章教你的不是"如何用非概率数据替代概率抽样"，而是"在不得已使用非概率数据时，如何尽量降低偏差"。

配额抽样（Quota Sampling）

配额抽样是最古老的非概率抽样方法之一。它的操作方式是：先把总体按某些特征（如年龄、性别、地区）分层，然后按照总体比例给每层分配一个"配额"，访问员在每层中选取达到配额数量的样本。

表面上看，配额抽样和分层抽样很像，但它们有本质区别——分层抽样在层内是随机抽取的，而配额抽样是"便利选取"的。这意味着即使年龄性别的分布与总体一致，层内的样本在其他特征上可能有严重的选择偏差。比如，一个按年龄配额抽样的街头访问，年轻人可能更多是在商业区闲逛的学生而非在职人员，这就引入了与就业状态相关的偏差。

配额抽样的"假代表性"陷阱：很多调查的初学者看到配额样本在年龄、性别上与总体分布一致，就觉得"好，这个样本有代表性了"。这是危险的误解。配额只保证了配额变量上的代表性，但在其他所有变量上——收入、教育、健康态度、政治观点等——毫无保障。1936年《文学文摘》的翻车和同年盖洛普用配额抽样的成功，让配额抽样名声大噪，但后来的大量研究证实：配额抽样的质量高度依赖配额变量的选择和访问员的执行纪律，总体而言它的偏差通常比简单随机抽样大得多。

滚雪球抽样（Snowball Sampling）

滚雪球抽样是一种特别适合稀有群体的非概率抽样方法。它通过已有的样本去招募新的样本——"你的朋友也是我们想调查的人吗？可以邀请他/她一起参与吗？"——像滚雪球一样越滚越大。

滚雪球抽样的典型应用场景：

  • 吸毒人群、性工作者、流浪者等隐秘群体
  • 罕见病患者或特定职业人群（如外太空航天员）
  • 网络社群中的"意见领袖"群体
非概率方法 选择机制 偏差风险 适用场景
便利抽样 谁方便选谁 高（不可控） 预测试、快速探索
配额抽样 按配额指标选取 中等（取决于配额变量） 市场调查、民意预估
滚雪球抽样 通过社交网络推荐 高（同质性偏差） 隐秘群体、稀有群体
判断抽样 由专家主观选取 取决于专家水平 定性研究、案例选择

滚雪球抽样最大的偏差来源是同质性偏差（Homophily Bias）——"朋友的朋友"在特征上往往和初始样本相似。这意味着滚雪球样本的"多样性"远低于总体，最终结果严重依赖于初始样本（"种子"）的选择。如果种子选错了，整个雪球就偏了。这也是为什么第17章讲的那个COVID滚雪球调查会得出不可靠的结论。

倾向得分加权校正（Propensity Score Weighting）

倾向得分加权是校正非概率样本偏差最常用的方法之一。它的核心思想是：估计每个非概率样本单元"如果被一个概率样本抽中会怎样"的概率，然后用这个概率的倒数来加权。

具体操作步骤：

  1. 数据整合：将非概率样本（NPS）与一个参考的概率样本（如普查数据或权威调查数据）合并。创建一个指示变量 Z=1（来自非概率样本）和 Z=0（来自概率样本）。
  2. 估计倾向得分：用逻辑回归（或更灵活的机器学习模型）来估计每个个体出现在非概率样本中的概率 P(Z=1|X)，其中X是两组数据中都有的共同变量。
  3. 加权：对非概率样本中的每个个体赋予权重 w = 1 / P(Z=1|X)（或变体权重），然后用这些权重来估计总体参数。
# Python：倾向得分加权校正示例
        import numpy as np
        import pandas as pd
        from sklearn.linear_model import LogisticRegression

        # 假设：
        # nps_data = 非概率样本（如网络问卷，N=5000）
        # ref_data = 参考概率样本（如普查数据，N=20000）

        # Step 1: 合并数据，创建指示变量
        nps_data['Z'] = 1  # 非概率样本
        ref_data['Z'] = 0  # 概率参考样本

        combined = pd.concat([nps_data[['age','gender','education','Z']],
                              ref_data[['age','gender','education','Z']]])

        # Step 2: 拟合倾向得分模型
        X = combined[['age','gender','education']]
        y = combined['Z']

        # 处理分类变量（这里简化为已经编码）
        model = LogisticRegression(max_iter=1000)
        model.fit(X, y)

        # 预测非概率样本中每个个体的倾向得分
        X_nps = nps_data[['age','gender','education']]
        propensity = model.predict_proba(X_nps)[:, 1]  # P(Z=1|X)

        # Step 3: 计算权重（逆概加权）
        # 注意：倾向得分接近0时权重会极大，通常需要截尾
        propensity_clipped = np.clip(propensity, 0.05, 0.95)
        weights = 1.0 / propensity_clipped

        # Step 4: 用加权估计目标变量
        # 假设y_var是我们要估计的目标变量
        weighted_mean = np.average(nps_data['y_var'], weights=weights)
        print(f"加权估计值: {weighted_mean:.2f}")
倾向得分加权在疫情研究中的应用：2020年，卡内基梅隆大学的的德尔菲研究组（Delphi Group）开展了一项名为COVIDcast的项目，其中大量使用非概率数据（如Facebook问卷、Google搜索趋势、在线症状报告）来估计新冠感染率。他们用倾向得分加权将非概率数据向CDC的概率调查数据"校准"，结果在多个时间点上与官方数据相当接近。但关键前提是：他们有高质量的概率调查作为"锚"——如果没有这个锚，校正就无从谈起。

校准方法（Calibration Methods）

校准方法（在调查统计中也常被称为"事后分层"或"边缘加权"的推广）是另一类主流的非概率样本校正技术。它的核心逻辑是：调整样本权重，使得加权后的样本在某些已知总体特征上恰好与总体匹配。

最经典的校准方法有：

  • 事后分层（Post-stratification）：将样本划分成与总体分布对齐的层，每层内使用统一权重使加权后的层比例与总体一致。
  • Raking（迭代比例拟合）：当有多个分类变量需要同时校准且样本量不足以支撑完全的交叉分层时，通过迭代调整权重来依次满足每个边缘分布。这是市场调查和民意调查中最常用的校准方法。
  • 广义回归估计（Generalized Regression Estimation, GREG）：在线性模型框架下同时校准多个辅助变量，是Raking的推广形式。
# Raking校准的直观理解（R语言pseudo示例）

        # 已知总体分布：
        # 男性 48%，女性 52%
        # 18-29岁 20%，30-44岁 25%，45-59岁 30%，60+ 25%

        # 样本（n=1000）原始分布：
        # 男性 42%，女性 58%  ← 女性过多
        # 18-29岁 15%，30-44岁 28%，45-59岁 35%，60+ 22%  ← 年轻人偏少

        # Raking迭代步骤：
        # 第1轮：将性别权重调整为48%/52%（调整后样本的年龄分布变了）
        # 第2轮：将年龄权重调整为20%/25%/30%/25%（调整后性别又偏了）
        # 第3轮：再次调整性别……如此反复直到收敛

        # 最终得到的权重同时满足了性别和年龄的总体分布
校准时的一个关键问题——权重变异：校准方法虽然能很好地匹配边缘分布，但可能产生极端权重。如果某个群体在非概率样本中严重缺失（比如老年人），为了匹配总体比例，校准会给这些样本非常大的权重——一个80岁老人可能被赋予10倍于年轻人的权重。这会导致方差膨胀。一个好的做法是：设定权重上限（如不超过中位数的3倍），并在报告中披露权重的变异程度（如权重的变异系数CV）。

实际故事：疫情中的便利样本校正

2020年3月，当武汉封城后，全国多个研究团队迅速发起了在线问卷调查，以了解公众对疫情的认知、防护行为和心理健康状况。这些问卷通过微信、微博等社交平台传播，几天之内就收集了上万份回复。但一个问题立即浮现：这些数据能代表全国人民吗？

我们团队当时也参与了其中一项调查的数据分析工作。当我们把样本的人口结构与2019年国家统计局的全国数据对比时，发现存在明显偏差：样本中女性占68%（总体约50%），大专及以上学历占75%（总体约15%），一线和新一线城市居民占80%（总体约30%）。可以说，这是一个典型的便利样本——它更接近"经常上网、关心疫情、有一定空闲时间的城市女性高学历人群"，而非全体国民。

我们采取了以下校正策略：首先，利用全国人口普查数据作为参考，将样本按性别、年龄、教育程度、地区四个变量进行Raking校准。其次，使用倾向得分加权方法——将便利样本与同期一项全国性的概率电话调查数据合并，估计每个便利样本单元出现在网络调查中的倾向得分。两种方法得到的校正结果比较接近。最终，我们发表的论文同时呈现了校正前后的对比结果，并以高度透明的方式报告了校正方法、权重分布和灵敏度分析结果。审稿人特别赞赏了这种透明度——因为在面对非概率数据时，完全消除偏差是不可能的，但至少要让读者知道偏差有多大、校正做了什么。

非概率样本校正的"三大纪律"：

第一：永远不要声称校正后的非概率样本具有与概率样本同等的可靠性。如果必须使用非概率数据，请使用"准估计"（quasi-estimate）等谨慎表述。

第二：必须有一组可靠的参考数据（普查数据、行政记录或高质量的概率调查数据）。没有参考数据的校正是无源之水。

第三：必须进行灵敏度分析——比如改变权重截尾阈值、使用不同的参考数据、尝试不同的模型规范，看看结论是否稳健。如果结论对方法选择非常敏感，那就说明数据质量不足以支持可靠的推断。

本章小结

非概率抽样校正是一个"在泥潭中跳舞"的话题。我们介绍了配额抽样和滚雪球抽样这两种常用但偏差风险较高的非概率方法，以及倾向得分加权和校准方法这两类主流的校正技术。核心的认知是：校正不是万能的，但不校正是万万不能的——当你不得不使用非概率数据时，系统性的校正虽然不能创造奇迹，但总比完全不做任何处理要好得多。透明报告校正方法、权重分布和灵敏度分析结果，是学术界和实践界对非概率数据使用者的基本要求。
