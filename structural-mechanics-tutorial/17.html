弹性不是终点，塑性才是极限

在前15章中，我们所有的分析都建立在一个共同的假设上：材料是线弹性的，应力与应变成正比，荷载去掉后变形完全恢复。这个假设让我们能够用简洁的数学工具来分析结构——但问题是，真实世界的结构在破坏之前，远远超出了弹性范围。

想想你手中的一个回形针。你轻轻弯它，它弹回来——这是弹性。你再用力弯它——它弯过去了，不再弹回来——这是塑性。你用更大的力气——它断了——这是破坏。回形针从弹性到塑性到破坏，经历了三个阶段。真正的结构也是这样。

那么我们之前学到的弹性分析给出了什么？它给出了结构在"弹性极限"之内的行为。如果荷载很小，弹性分析完全够用。但如果荷载接近结构的极限承载力呢？忽略塑性意味着我们对结构的"安全储备"一无所知。事实上，一个结构在第一个截面屈服之后，并不会立即倒塌——它还有很长的路要走。这条路，就是塑性分析（Plastic Analysis）要探索的领域。

一个问题：为什么全世界的钢结构设计规范都允许利用塑性？答案很直接——因为经济性。如果只允许结构在弹性范围内工作，那意味着我们给结构设计的承载力最多只能达到第一个截面的屈服荷载。但事实上，结构在这个荷载下远远没有达到真正的极限。利用塑性分析，我们可以释放结构的"真实承载力"，让材料物尽其用。据估算，采用塑性设计方法比纯弹性设计可以节省15%~30%的钢材用量。

材料的塑性性能与理想弹塑性模型

真实材料的塑性行为

以建筑工程中最常用的钢材为例，它在单向拉伸试验中表现出典型的弹塑性行为：

  • 弹性阶段（OA段）：应力与应变成正比（\(\sigma = E\varepsilon\)），卸载后变形完全恢复。
  • 屈服阶段（AB段）：应力达到屈服强度 \(f_y\) 后，应变显著增加而应力基本不变——这就是"屈服平台"。在这个阶段，材料在外观上发生了不可恢复的塑性变形。
  • 强化阶段（BC段）：屈服平台结束后，材料重新获得承载能力，应力继续增加直至抗拉强度 \(f_u\)。强化阶段的硬化模量约为弹性模量的1%~2%。
  • 颈缩阶段（CD段）：应力达到峰值后，试件出现局部截面收缩（颈缩），应力下降直至断裂。

在结构塑性分析中，我们通常采用理想弹塑性模型（Ideal Elastic-Plastic Model）——忽略强化阶段，将材料的应力-应变关系简化为两条直线：弹性直线和水平塑性直线。这个模型在数学上非常简洁，又足够准确地反映了结构在极限状态下的主要特征。

材料模型 本构关系 适用场景 数学复杂度
线弹性模型 \(\sigma = E\varepsilon\)，一直到底 正常工作状态分析 简单
理想弹塑性模型 \(\sigma = E\varepsilon\) (\(\varepsilon \le \varepsilon_y\))；\(\sigma = f_y\) (\(\varepsilon > \varepsilon_y\)) 极限荷载分析（塑性分析） 中等
弹塑性强化模型 弹性段 + 屈服平台 + 线性强化 精确分析（有限元） 高
实际材料的选用：塑性分析方法主要适用于钢材等具有良好延性的材料。混凝土材料虽然也有塑性，但它的塑性行为与钢材有很大不同——混凝土受压时表现出塑性，但受拉时是脆性的。钢筋混凝土结构的塑性分析比纯钢结构的塑性分析要复杂得多，通常需要考虑裂缝、钢筋屈服、混凝土压碎等多种失效模式。

极限弯矩与塑性铰

从弹性弯矩到极限弯矩

考虑一个矩形截面的梁，宽度 \(b\)，高度 \(h\)，采用理想弹塑性材料。当梁受弯时，截面上的应力分布经历了三个阶段：

阶段一：全截面弹性。弯矩较小时，整个截面的应力都小于 \(f_y\)，应力呈三角形（线性）分布。这是我们在材料力学和此前结构力学中一直使用的情况。

阶段二：弹塑性状态。弯矩继续增加，截面最外缘的应力首先达到 \(f_y\)——这称为屈服弯矩（Yield Moment） \(M_y\)：

M_y = f_y · W

其中 W = bh²/6 是弹性截面模量（Elastic Section Modulus）

弯矩超过 \(M_y\) 后，塑性区域从外缘向内扩展——外层纤维应力保持 \(f_y\)（不再增加），内层纤维应力继续线性增长直到也达到 \(f_y\)。截面上出现了一个"弹性核"和两个"塑性区"。

阶段三：全截面塑性。当弯矩足够大时，整个截面的应力都达到 \(f_y\)——弹性核消失，应力分布呈现两个矩形块（受拉区 \(f_y\)，受压区 \(-f_y\)）。这时的弯矩称为极限弯矩（Ultimate Moment / Plastic Moment） \(M_u\) 或 \(M_p\)：

M_p = f_y · S

其中 S = bh²/4 是塑性截面模量（Plastic Section Modulus）

对于矩形截面，\(S = bh²/4\)，而 \(W = bh²/6\)，所以：

M_p / M_y = S / W = (bh²/4) / (bh²/6) = 1.5

这意味着矩形截面的极限弯矩是屈服弯矩的1.5倍——这就是该截面的"塑性储备"。对于不同截面形状，这个比值（称为截面形状系数 \(\nu = S/W\)）不同：

截面形状 弹性模量 W 塑性模量 S 形状系数 \(\nu = S/W\)
矩形 \(bh²/6\) \(bh²/4\) 1.50
圆形 \(\pi d³/32\) \(d³/6\) 约1.70
薄壁圆管 \(\pi r²t\) \(4r²t\) 约1.27
工字钢（常见） 查表 查表 1.05~1.20
一个容易犯的错误：初学者常常以为某截面达到了极限弯矩，这个截面就"坏"了、完全不能受力了——这个理解是不对的。达到极限弯矩的截面并没有"断裂"，它只是失去继续抵抗弯矩增长的能力，但仍可以维持极限弯矩 \(M_p\) 不变，同时截面两侧可以发生相对转动。这个现象，就是我们接下来要讲的塑性铰。

塑性铰的概念

当梁的某个截面达到极限弯矩 \(M_p\) 后，该截面的曲率可以无限增大而弯矩不再增加——这就像一个"铰"，但它不是真正的铰（真正的铰不传递弯矩），而是一个能传递恒定弯矩 \(M_p\) 的铰。我们把这种截面称为塑性铰（Plastic Hinge）。

塑性铰与普通铰的关键区别：

  • 普通铰不传递弯矩（\(M = 0\)），塑性铰传递恒定弯矩 \(M_p\)；
  • 普通铰可以自由转动（无约束），塑性铰的转动是"有代价的"——每转动一点，都在做塑性功、耗散能量；
  • 普通铰是"永久性"的，塑性铰如果卸载（弯矩减小），截面会恢复弹性，塑性铰的特性消失（不再是铰了）。

塑性铰的出现是结构从"弹性体系"转向"塑性体系"的标志。当结构中形成足够多的塑性铰，使结构变成机构（mechanism，即几何可变体系）时，结构就达到了其极限承载力。

极限荷载的计算方法

确定结构的极限荷载 \(P_u\)，有两条基本的分析路径。

静力法

静力法（Static Method），也称"下限法"。它的思路是：构造一个满足以下三个条件的弯矩分布：

  • 满足静力平衡条件（平衡方程）；
  • 所有截面的弯矩不超过极限弯矩（\(|M| \le M_p\)）；
  • 边界条件与荷载一致。

满足这些条件的荷载称为"静力容许荷载"。静力法的下限定理说：真正的极限荷载是所有静力容许荷载中的最大值。换句话说，静力法给出的结果总是≤真实极限荷载。

机动法

机动法（Kinematic Method），也称"上限法"。它的思路是：假设结构发生某种破坏机构（mechanism），通过虚功原理计算对应的荷载。机动法的上限定理说：真正的极限荷载是所有机动容许荷载中的最小值。

用虚功原理表达：

外力虚功 = ∑(P_i · δ_i)
内力虚功 = ∑(M_p · θ_i) —— 各塑性铰处的极限弯矩乘以转角

由 W_ext = W_int，解得 P_u

在解题时，通常的做法是：

  • 尝试所有可能的破坏机构（不同位置出现塑性铰的不同组合）；
  • 对每种机构用机动法计算 \(P_u\)；
  • 取所有计算结果中的最小值作为真实极限荷载。
直观理解上下限定理：可以把极限荷载想象成一个"藏起来的宝藏"。静力法是从下面往上挖（从安全侧逼近），永远不大于宝藏的真实位置；机动法是从上面往下挖（从危险侧逼近），永远不小于宝藏的真实位置。当上下限重合时，我们就找到了精确解。这个对偶关系在数学上非常优美，也反映了一种"逼近-校正"的工程思维。

示例：简支梁的极限荷载

一根简支梁，跨度 \(l\)，跨中作用集中荷载 \(P\)。截面极限弯矩为 \(M_p\)。

用静力法：平衡条件要求跨中弯矩 \(M_{max} = Pl/4\)。极限状态时 \(M_{max} = M_p\)，所以 \(P_u = 4M_p/l\)。此时跨中截面形成一个塑性铰，梁变成机构（两根直杆在跨中铰接）。

用机动法：假设跨中出现塑性铰，两侧梁段绕端部转动。设跨中虚位移为 \(\delta\)，则梁段转角 \(\theta = 2\delta/l\)。外力虚功 = \(P \cdot \delta\)，内力虚功 = \(M_p \cdot (\theta + \theta) = 2M_p\theta = 4M_p\delta/l\)。由 \(P\delta = 4M_p\delta/l\)，得 \(P_u = 4M_p/l\)。两种方法结果一致，因为这是静定结构，极限荷载唯一确定。

超静定梁的极限荷载

超静定梁的极限荷载分析更有意思——因为超静定梁发生"第一个"塑性铰后并没有破坏，它只是变成了一个静定结构（减少了多余约束），仍能继续承载。直到出现足够多的塑性铰使它变成机构，才达到极限状态。

考虑一根两端固定的梁，跨度 \(l\)，截面极限弯矩 \(M_p\)，跨中作用集中荷载 \(P\)。

弹性分析告诉我们：在两端固定梁中，端部弯矩 = \(Pl/8\)，跨中弯矩 = \(Pl/8\)（均布荷载下数值相同，但集中荷载下不同）。实际计算表明，对于集中荷载，固定端弯矩 = \(Pl/8\)，跨中弯矩 = \(Pl/8\)。当荷载增大时，固定端首先达到 \(M_p\)，形成第一个塑性铰。此时梁变成"一端固定、一端铰支"的超静定梁——并没有变成机构！

继续加载，跨中弯矩继续增长。当跨中弯矩也达到 \(M_p\) 时，跨中形成第二个塑性铰。此时，梁的两端和跨中各有一个塑性铰——整个梁变成机构（三铰共线），达到极限状态。

下面用机动法计算：

破坏机构：在两端A、B和跨中C各有一个塑性铰
设跨中虚位移为 δ
A端转角：θ_A = δ/(l/2) = 2δ/l
B端转角：θ_B = δ/(l/2) = 2δ/l
C端两侧相对转角：θ_C左 + θ_C右 = 2δ/l + 2δ/l = 4δ/l

内力虚功 = M_p · θ_A + M_p · θ_B + M_p · θ_C
         = M_p · (2δ/l + 2δ/l + 4δ/l) = 8M_pδ/l

外力虚功 = P · δ

由 Pδ = 8M_pδ/l，得 P_u = 8M_p/l

对比弹性分析：弹性分析中，当固定端达到屈服弯矩 \(M_y\)（这里假设 \(M_y = M_p\)，即忽略塑性储备）时，\(P_y = 8M_p/l\)。但这时跨中弯矩仅为 \(M_p/2\)，整个结构远未达到极限。如果我们按弹性方法设计，取 \(P_y\) 为容许荷载，那么我们"浪费"了结构从第一个铰形成到变成机构之间的巨大承载力。塑性分析告诉我们，这个结构的真实极限荷载恰好是弹性屈服荷载的2倍（如果 \(M_y = M_p\)）——这就是塑性储备的力量！

塑性内力重分布：超静定结构的塑性分析揭示了一个非常重要的现象——塑性内力重分布（Plastic Redistribution of Internal Forces）。当一个截面形成塑性铰后，它不再能承受更多的弯矩，但结构整体的承载力并没有耗尽——荷载增量产生的附加内力会"绕开"这个塑性铰，向其他尚未屈服的截面传递。这意味着超静定结构天生就比静定结构有更好的"韧劲"。在钢筋混凝土结构设计中，我们经常利用塑性内力重分布来调整配筋——这就是"调幅法"的基础。

塑性分析与弹性分析的对比

现在我们用一张表来总结弹性分析和塑性分析的关键区别：

对比维度 弹性分析 塑性分析（极限分析）
材料模型 理想弹性（应力无限增长） 理想弹塑性（有限屈服强度）
分析目标 工作状态（容许应力）下的内力与变形 结构达到破坏时的极限荷载
荷载-位移曲线关注点 弹性范围内的线性关系 从弹性直到破坏的全过程
截面检查 最大应力 ≤ 容许应力 最大弯矩 ≤ 极限弯矩
破坏判定 任一截面达到屈服 形成足够塑性铰使结构变为机构
安全系数定义 屈服强度/容许应力（单一系数） 极限荷载/工作荷载（荷载系数）
内力分布 由刚度（EI）决定，与荷载路径无关 塑性铰出现后发生内力重分布
适用范围 所有结构（正常工作状态） 延性材料结构（钢、钢筋混凝土）
塑性分析的局限性：不是所有结构都适合用塑性分析。铸钢、铸铁、高强混凝土等脆性材料在达到屈服前就断裂了——它们没有塑性储备。另外，塑性分析不能用于疲劳问题、不能用于需要严格控制变形的结构（比如精密仪器平台）、也不适用于动力响应分析。在结构设计中，塑性分析通常被用于极限状态设计（检查结构在极限荷载下的安全性），而弹性分析被用于正常使用状态设计（检查结构在正常工作荷载下的变形和裂缝）。两者是互补关系，而非替代关系。

我亲身经历的"塑性"故事

我在读研究生期间参与过一个钢框架的极限承载力试验。试验对象是一个两层两跨的缩尺钢框架，通过千斤顶在顶层施加水平荷载模拟地震作用。

一开始，荷载-位移曲线是笔直的——框架处于弹性状态。当荷载达到约120kN时，我们听到"咔"的一声——首层右侧柱脚形成了第一个塑性铰。这时我们都很紧张，以为框架要塌了。但老师很淡定地说："继续加载。"

果然，荷载继续上升。接着底层左柱脚出现第二个塑性铰，然后是底层梁端——一个接一个，像多米诺骨牌一样形成。但每次形成一个新的塑性铰，框架都没有垮——荷载还在增加！最后，当底层所有柱脚和所有梁端都形成了塑性铰，整个底层变成了一个机构——荷载再也加不上去了。这时极限荷载已经达到了210kN——几乎是"第一个铰"时荷载的1.75倍。

试验后的框架面目全非——柱脚处有明显的塑性弯曲（我们工程师称之为"胖了"），梁端翼缘也出现了明显的局部屈曲。但框架从头到尾都没有倒塌——它用"变形"代替了"断裂"，用"塑性铰"的逐个形成向我们展示了钢材令人惊叹的延性。

这个试验让我深刻理解了："第一个屈服"不等于"破坏"，弹性分析的终点，恰恰是塑性分析的起点。

学到这里，我们已经完成了结构力学所有核心理论和方法的旅程——从静定到超静定，从力法到位移法，从弹性分析到塑性分析。但你可能要问：学了这么多理论，它们在实际工程中到底怎么用？
