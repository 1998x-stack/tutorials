第10章 套利定价与多因子模型

2019年，一家国内量化私募的基金经理在路演中自信满满地说："我们用的就是Barra多因子模型，覆盖了200多个因子。"台下一位机构投资者问："那你们的因子是怎么筛选出来的？"基金经理愣了一下，说："我们把所有能想到的因子都试了一遍，回测效果好的就留下。"

这个故事不是个例。从2010年代开始，因子投资从学术界火到了华尔街，又传到了中国。到今天，"因子"已经成了量化投资中的高频词汇。但因子真的有那么多吗？为什么有的因子今天有效、明天就失效了？这一章，我们从套利定价理论（APT）出发，一路走到"因子动物园"，看看这些问题的答案。

核心洞见：CAPM告诉我们"市场风险是一切"，但实证告诉我们"不够"。多因子模型承认资产收益受多个系统性风险来源驱动——而这些风险因子，就是我们理解市场定价逻辑的钥匙。

10.1 套利定价理论（APT）

1976年，Stephen Ross提出了套利定价理论（Arbitrage Pricing Theory, APT），挑战了CAPM的统治地位。与CAPM不同，APT不假设"所有投资者都使用均值-方差优化"，而只基于一个更朴素的逻辑：如果两个资产在所有风险维度上完全一致，但价格不同，就存在套利机会。

# APT的多因子表达式
E(R_i) = R_f + β_i1 × λ_1 + β_i2 × λ_2 + ... + β_ik × λ_k

# 其中：
# β_ik = 资产i对因子k的暴露（Factor Loading）
# λ_k = 因子k的风险溢价（Risk Premium / Factor Return）
# 每个λ_k代表承担一单位因子k风险所获得的额外收益

# 例如：一个三因子APT模型
# E(R_i) = 1.8% + 0.9 × 4.0% + 0.3 × 2.5% + (-0.2) × 3.0%
#        = 1.8% + 3.6% + 0.75% - 0.6%
#        = 5.55%

APT相对于CAPM的优势在于：

  • 假设更弱：不要求市场组合可观测，不要求同质预期
  • 框架更灵活：因子可以是任何"系统性风险来源"——宏观因子、风格因子、行业因子等
  • 可检验性更强：不需要找到真正的市场组合

但APT也有一个明显的弱点：它不告诉你哪些是因子。理论只说"存在一组因子"，但不说是哪些。这就好比有人告诉你"做菜需要加调料"，但不告诉你到底要加盐还是加糖。

10.2 Fama-French三因子模型

1992年，Eugene Fama和Kenneth French发表了那篇改变了整个金融学术界的论文《The Cross-Section of Expected Stock Returns》。他们发现，除了市场β之外，还有两个变量对股票收益有系统性的解释力：市值和账面市值比。

# Fama-French三因子模型
E(R_i) - R_f = β_mkt × (R_m - R_f) + β_smb × SMB + β_hml × HML + α_i

# 因子定义：
# SMB（Small Minus Big）= 小盘股收益 - 大盘股收益
# HML（High Minus Low）= 高账面市值比（价值股）收益 - 低账面市值比（成长股）收益

# 例如：某只小盘价值股的因子暴露
# 假设：β_mkt = 1.05, β_smb = 0.60, β_hml = 0.40
# 市场溢价 = 6%, SMB = 2.5%, HML = 3.5%, R_f = 1.8%
# E(R_i) = 1.8% + 1.05×6% + 0.60×2.5% + 0.40×3.5%
#        = 1.8% + 6.3% + 1.5% + 1.4% = 11.0%
因子经济含义中国A股历史溢价逻辑解释
市场（MKT）整体市场风险6-8%承担市场系统性风险的补偿
规模（SMB）小盘股额外风险3-5%小盘股流动性差、经营风险高、信息不透明
价值（HML）价值股额外风险2-4%价值股陷入困境的概率更高，需要额外补偿
中国A股发现：三因子模型在中国市场上的表现与美股有显著差异。中国市场的规模效应非常强——小盘股的超额收益远大于美国，但波动也极大。而价值效应在A股较弱，甚至在2019-2024年的"赛道股"行情中完全失效——低市盈率的"便宜货"并不一定涨得更好。这提醒我们：因子溢价具有市场特异性，不能简单套用海外经验。

10.3 Fama-French五因子与Carhart四因子

Carhart四因子：动量

1997年，Mark Carhart在三因子的基础上加入了动量因子（Momentum, MOM）。动量效应指的是：过去12个月涨幅最大的股票（赢家），未来会继续跑赢过去12个月跌幅最大的股票（输家）。

# Carhart四因子模型
E(R_i) - R_f = β_mkt × MKT + β_smb × SMB + β_hml × HML + β_mom × MOM + α_i

# MOM（Momentum）= 过去12个月赢家组合收益 - 输家组合收益
# 在美国市场，动量因子年化溢价约8-10%，是最强的单一异象之一
踩坑实录：动量因子在A股市场非常"调皮"——它在2013-2015年的中小创牛市中极其有效，但在2016-2017年的"熔断行情"中连续20个月失效，最大回撤超过30%。很多2015年底成立的中性策略产品因为重仓动量因子，在2016年被打穿了止损线。教训是：每个因子都有"灾难期"，分散化不仅适用于资产，也适用于因子。

五因子模型：盈利能力与投资水平

2015年，Fama和French本人对三因子模型做了升级，加入了两个新因子：

  • RMW（Robust Minus Weak）：高盈利能力公司减去低盈利能力公司。赚钱的公司应该估值更高。
  • CMA（Conservative Minus Aggressive）：低投资（保守）公司减去高投资（激进）公司。投资激进的公司往往容易过度扩张从而毁灭价值。

有趣的是，加入两个新因子后，原三因子中的HML因子变得不再显著——说明价值效应（HML）的收益来源，在很大程度上被盈利能力和投资水平所解释。

10.4 因子动物园

从APT框架诞生以来，学者们发现了成百上千个"显著的因子"。这就是所谓的因子动物园（Factor Zoo）。2020年的一篇综述论文统计了超过400个已发表的因子，覆盖了估值、动量、质量、波动率、红利、增长、情绪等各个维度。

因子挖掘的方法论问题

这些因子真的都"存在"吗？学术界越来越达成共识：大部分显著因子是数据挖掘的产物。以下是一些典型的问题：

  • 多重假设检验（Multiple Testing）：如果你独立测试1000个因子，在5%的显著性水平下，期望会有50个因子"恰好显著"。很多学者只报告了显著的因子，不报告不显著的——这就是发表偏倚（Publication Bias）。
  • 回测过拟合（Backtest Overfitting）：在大量变量中寻找"最佳策略"，本质上是在拟合噪声。
  • 数据窥视（Data Snooping）：反复使用同一套数据集测试不同的假设。
# 多重假设检验的Bonferroni修正
# 如果测试100个因子，要保持整体显著性水平为5%
# 单个因子的p值门槛应为：0.05 / 100 = 0.0005

# 更先进的修正方法：
# - Holm-Bonferroni
# - Benjamini-Hochberg（FDR控制）
# - 随机化测试（Randomization Test）
业界应对：以AQR、Two Sigma为代表的顶级量化机构，对因子筛选有严格的要求：样本外测试（至少5年的样本外数据）、经济直觉（不能只是一个统计模式）、跨市场验证（有效应在多个市场上有效）、稳健性检验（改变参数后仍有效）。任何一个新因子，如果不符合以上几条，就不值得投入实盘资金。

10.5 中国A股因子特征

最后，让我们聚焦中国A股市场，看看这里有哪些独特的因子特征：

市值效应：A股的"小盘神话"

A股的规模效应远强于美股。2005-2025年，中证500和中证1000的年化收益显著高于沪深300，但波动也更大。然而，2021年以来，小盘股的超额收益有所收敛，原因是注册制改革后股票供给大幅增加，"壳价值"缩水。

价值效应在中国：水土不服？

价值因子在A股的表现时好时坏。2017-2020年，"核心资产"行情下高估值成长股持续跑赢低估值的银行、地产、建筑。2021-2024年，随着"赛道股"泡沫破裂，价值股又开始跑赢。这说明在中国市场，价值因子的溢价与宏观流动性高度相关——在流动性宽松时成长股占优，在流动性收紧时价值股占优。

A股特有的因子

因子描述有效性区间
低换手率换手率低的股票长期跑赢2005-2025持续有效
高股息高分红股票获得稳定超额收益2018-2025显著增强
股权集中度大股东持股比例适中的公司表现更好取决于监管周期
分析师上调分析师集中上调盈利预测的股票短期跑赢3-6个月窗口有效
北向资金流入北向资金持续买入的个股短期有正超额2017-2025持续有效
给投资者的建议：与其追逐"因子动物园"中的数百个因子，不如聚焦3-5个有坚实经济逻辑支撑的核心因子：价值、质量、动量（趋势）、低波动、规模。将这些因子做因子配置（Factor Allocation）和因子择时（Factor Timing），远胜于试图抓住所有因子。
本章要点回顾：
  • APT理论提供了一套比CAPM更灵活的定价框架，但不指定具体的因子
  • Fama-French三因子（市场、规模、价值）是最经典的多因子模型
  • Carhart加入动量因子，Fama-French后来扩展为五因子（加入盈利能力和投资水平）
  • "因子动物园"揭示了因子挖掘中的多重假设检验和过拟合问题
  • 中国A股因子有其独特性：规模效应强、价值效应不稳定、高股息和低换手率有效

多因子模型给了我们理解资产收益的工具箱，但它们与CAPM一样，都假设市场是有效的——价格会迅速反映所有可用信息。但事实真的是这样吗？如果市场是有效的，为什么会有这么多"异象"？下一章，我们将直面这个问题，探讨有效市场假说（EMH）——它可能是金融学中最受争议的理论。
