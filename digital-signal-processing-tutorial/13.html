第13章 多速率信号处理 (二)

上一章我们学会了抽取、内插和多相分解——多速率信号处理的三大基础工具。但实际工程中，你很少会只用"一个滤波器+一种抽取"的简单方案。现实世界的高倍率采样率转换（比如从122.88MHz降到48kHz，超过2560倍）需要的是多级级联的策略：每一级做一小部分抽取，用最适合该级速率的滤波器结构。

什么滤波器最适合做高倍率抽取？答案是本章的主角们：CIC滤波器（无乘法器、仅加法和延迟）、半带滤波器（一半系数为零），以及它们构成的多级抽取链。最后我们还会介绍一个重要的多速率系统概念——正交镜像滤波器组（QMF）。

💡 核心洞察：多速率信号处理的最大智慧不在于"怎么降采样"，而在于"怎么把降采样流程拆成多级以最小化总计算量"。一个256倍抽取如果用单级方案可能需要几千阶FIR，总计算量巨大；但如果分解成CIC(×16)→Half-Band(×4)→FIR(×4)的三级方案，总计算量可能不到单级的1%。这是多速率工程最令人惊叹的地方。

13.1 CIC滤波器：无乘法器的奇迹

CIC（Cascaded Integrator-Comb，级联积分梳状）滤波器是多速率信号处理中最"优雅"的发明之一，最早由Eugene Hogenauer在1981年提出。它的最大特点是：不需要任何乘法器，只需要加法和延迟——乘法器是FPGA和ASIC上最昂贵的资源（面积和功耗），而CIC在逻辑上几乎"免费"。

CIC滤波器的结构分为两部分：

  • 积分器（Integrator）部分：y[n] = y[n-1] + x[n]——就是一个累加器。频率响应是低通的，增益在零频处最大。
  • 梳状（Comb）部分：y[n] = x[n] - x[n-M]——一个延迟M个样本的差分器。频率响应是周期性的陷波形状。

单级CIC的传输函数：

H(z) = (1 - z^{-M}) / (1 - z^{-1}) = Σ_{k=0}^{M-1} z^{-k}

有趣的是：单级CIC等价于一个长度为M的移动平均滤波器（Moving Average）。它的频率响应是|sin(ωM/2) / sin(ω/2)|的形状——在零频处增益为M，第一零点在ω=2π/M处。

在实际工程中，几乎不会使用单级CIC——因为它的阻带衰减太差了（只有约13dB）。通常的做法是级联N级CIC，阻带衰减可以提升到大约N×13dB。比如4级CIC提供了约52dB的阻带衰减，已经可以满足大多数应用的需求。

# N级CIC抽取器的结构：
x[n] → [积分器]×N → [↓M] → [梳状]×N → y[n]
       (高速率端)          (低速率端)

积分器工作在高速率端（ADC采样率），梳状器工作在低速率端（抽取后）。这种"把一半运算放在低速率端"的安排，正是多速率设计思想的体现。

🔧 工程案例：在ΣΔ ADC（如TI的ADS127L01）中，CIC抽取滤波器是标配。ΣΔ调制器以极高过采样率（如64×或128×）产生1位数据流，然后CIC抽取器把这些高速1位流降采样到奈奎斯特率。为什么用CIC？因为1位流乘以系数只需要符号选择——乘法器完全多余，加法器和寄存器足矣。CIC也是软件无线电（SDR）前端的标准滤波器。

13.2 半带滤波器（Half-Band Filter）

半带滤波器是一种特殊的FIR滤波器，设计用于2倍抽取（M=2）。它的特殊性质：

  • 近一半的系数为零：h[奇数索引] = 0（除了中心系数）；
  • 通带和阻带对称：通带截止频率ω_p和阻带起始频率ω_s关于π/2对称，且ω_p + ω_s = π；
  • 通带波纹和阻带波纹相等。

由于近一半系数为零，半带滤波器的计算量只有同等长度FIR的一半。加上多相分解后，2倍抽取的半带滤波器可以做到只有约N/4次乘加/输出样本——非常高效。

半带滤波器在2的幂次抽取链（8倍、16倍、32倍等）中应用极广。一个典型的链路是：先用CIC做高倍率粗抽取，然后用半带滤波器做后续的2倍抽取，最后用FIR做精抽取。每一级的抽取倍数都不大（通常是2倍），但滤波器设计简单、计算量小。

# Python设计半带滤波器示例
from scipy import signal

# 设计一个半带低通滤波器，归一化截止频率0.45
# 半带要求截止频率在0.25-0.5之间，且通带阻带对称
taps = signal.firwin(31, 0.45, window='hamming')
# 验证：偶数索引的系数应当基本为零
print(taps[1::2])  # 应该接近0

13.3 多级抽取系统设计

现在让我们把CIC和半带滤波器组合起来，设计一个完整的多级抽取系统。以SDR接收机中常见的抽取链为例：

目标：从122.88MHz ADC采样率，降采样到48kHz基带采样率，总抽取因子为2560。如果只用单级抽取（一个滤波器+2560倍下采样），抗混叠滤波器的过渡带将极其陡峭，需要上万阶FIR——物理上不可能实现。

多级方案：

级 滤波器类型 抽取倍数 输出采样率 滤波器长度 计算量占比
1 CIC（4级） ×64 1.92 MHz 无乘法器 ~5%
2 半带FIR ×2 960 kHz ~16阶 ~10%
3 半带FIR ×2 480 kHz ~24阶 ~15%
4 半带FIR ×2 240 kHz ~32阶 ~20%
5 FIR通道均衡 ×2 120 kHz ~64阶 ~30%
6 FIR精抽取 ×2.5 48 kHz ~128阶 ~20%

总计算量约为单级方案的1/500。而且CIC级完全不消耗乘法器，半带级因为半数为零系数效率加倍。这就是多级设计的威力。

⚠️ 踩坑实录：曾经有一个团队在设计CIC+Half-Band抽取链时，忽略了CIC滤波器的增益补偿。CIC的直流增益为M^N（M为抽取倍数，N为级数）。以M=64、N=4为例，直流增益是64^4 = 16,777,216。如果不做数字增益补偿（用移位或乘法），后续几级的信号会直接溢出。更隐蔽的问题是：CIC的幅频响应在通带内并不平坦（有滚降），所以CIC后面通常需要接一个CIC补偿滤波器（CIC Compensation Filter）来矫正通带滚降。忘记补偿导致结果信号带内平坦度严重超标——这在通信系统中会导致星座图畸变。

13.4 两通道正交镜像滤波器组（QMF）

QMF（Quadrature Mirror Filter）是多通道信号分解和合成的核心概念。一个两通道QMF系统包含：

  • 分析滤波器组（Analysis Bank）：H₀(z)（低通）和H₁(z)（高通），将输入信号分为低频和高频子带。
  • 下采样：每个子带做2倍抽取。
  • 上采样：处理完后做2倍内插。
  • 综合滤波器组（Synthesis Bank）：G₀(z)（低通）和G₁(z)（高通），合并子带。

QMF的条件：H₁(z) = H₀(-z)（即H₁在频域上是对H₀的镜像）。综合滤波器的选择要使得整个系统实现完美重构（Perfect Reconstruction）——输出等于输入的延迟版本，没有混叠失真。

分析端：             综合端：
x[n] → [H₀(z)] → [↓2] → ...处理... → [↑2] → [G₀(z)] → ┐
      → [H₁(z)] → [↓2] → ...处理... → [↑2] → [G₁(z)] → ├→ ŷ[n] ≈ x[n-Δ]

完美重构的条件（忽略延迟）：

G₀(z) = H₁(-z),  G₁(z) = -H₀(-z)
H₀(z)·H₁(-z) + H₁(z)·H₀(-z) = 0      （混叠消除条件）
H₀(z)·G₀(z) + H₁(z)·G₁(z) = 2z^{-D}  （无失真条件）

QMF组的主要应用：

  • 子带编码（Subband Coding）：MPEG音频编码器将音频信号分解为32个子带，每个子带独立量化——人耳对不同频率的量化噪声敏感度不同，子带编码可以利用这个特性优化比特分配。
  • 数字信道化（Digital Channelization）：SDR接收机中，用一个滤波器组把宽带信号一次分解为多个窄带通道，同时解调多个信道。
  • 小波变换：经典的离散小波变换（DWT）本质上就是多级QMF结构的级联。
🔧 知识延伸：上面讲的两通道QMF只是最简单的滤波器组。在更高级的应用中（如MPEG-AAC、Opus音频编码），系统会使用MDCT（改进型离散余弦变换）滤波器组，它是余弦调制的滤波器组（Cosine-Modulated Filter Bank），通道数高达1024，并且支持窗口切换。这些内容已经进入了更深的音频编码领域——如果你想深入学习，第23章"音频信号处理"会进一步展开。
