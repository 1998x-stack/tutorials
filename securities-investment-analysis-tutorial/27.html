第27章 宏观对冲与全球配置

大家好，我是你们的讲师。前面26章我们主要聚焦在A股市场的个股分析和选股策略。但如果你是一个真正意义上的投资者，迟早会面临一个绕不开的问题——你的视野不能永远只停留在单一资产、单一市场里。

2025-2026年的全球市场给我们上了生动的一课。A股在2025年经历了一轮"先扬后抑"的行情——年初DeepSeek和机器人概念引爆的AI牛市中，沪指一度突破4,200点，但下半年随着美国大选尘埃落定、中美贸易摩擦再度升级，市场又经历了显著回调。与此同时，美国纳斯达克在AI算力股的带动下连创新高，黄金突破3,500美元/盎司的历史关口，比特币在2025年底突破20万美元。如果你只持有A股股票，你的账户在2025年下半年大概率是负收益。但如果你做了全球配置——哪怕只是配了一点黄金和美股——情况可能完全不同。这就是这一章要讲的核心内容：从宏观视角审视市场，做跨资产、跨市场的配置与对冲。

27.1 大类资产配置的底层逻辑

不只看股票

很多散户投资者有一个共同的习惯——"满仓股票"。这其实是一种非常极端的配置方式。大类资产的世界远比股票丰富得多：

  • 股票：长期收益最高（年化8%-12%），但波动最大。
  • 债券：收益相对稳定（3%-5%），在衰退期可以作为避险资产，与股票呈负相关。
  • 商品：包括黄金、原油、工业金属等。抗通胀属性强，与股票的相关性因周期而异。
  • 现金及货币基金：流动性最佳，收益最低（1%-2%），但在市场恐慌时提供了"弹药"。
  • REITs（房地产信托基金）：介于股票和债券之间的收益风险特征，每年有稳定的分红。
  • 另类资产：私募股权、对冲基金、加密资产——流动性较差但潜在收益空间大。

资产配置的核心逻辑在于相关性：不同资产之间的价格走势往往不同步。当股票大跌时，债券可能上涨，商品可能横盘。通过组合低相关性的资产，可以在不牺牲太多预期收益的情况下显著降低组合波动。

核心原则：资产配置决定了投资组合90%以上的收益波动。选股和择时的影响加起来不超过10%。这是Brinson、Hood和Beebower在1986年发表的经典研究中的结论——30多年过去了，这个结论在A股、港股、美股市场上都得到了反复验证。

27.2 美林时钟与它的A股改良版

经典四象限

美林证券在2004年提出了"投资时钟"框架，将经济周期划分为四个阶段，每个阶段有对应的最优资产配置：

经济阶段 GDP增速 通胀水平 最优资产 行业配置
复苏（Recovery） 上升 低位/下降 股票 科技、可选消费、金融
过热（Overheat） 高位/见顶 上升 商品 能源、材料、工业
滞胀（Stagflation） 下降 高位 现金 防御性行业（医药、公用事业）
衰退（Recession） 下降/触底 下降 债券 国债、高评级信用债

中国特色改造：五阶段模型

美林时钟在美国市场上表现良好，但直接套用A股会发现一个问题——中国有非常明显的政策驱动特征。

A股历史上多次行情的核心驱动力并非经济基本面的自发变化，而是政策的主动调控。比如2014-2015年的杠杆牛、2024年的"9·24"政策组合拳、2025年的AI产业政策集中爆发，每一次都与政策面高度相关，传统的"经济-通胀"二维框架不足以解释。

因此，在实践中很多国内机构将美林时钟扩展为"五阶段模型"，加入了"政策宽松"作为独立的过渡阶段：

  • 政策宽松期：降准降息、财政刺激、产业政策密集出台——权益资产开始上涨，债券进入牛市。
  • 复苏期：政策效果显现，经济数据边际改善——股票尤其是周期股和成长股领涨。
  • 过热期：经济增速接近上限，通胀和资产价格泡沫风险上升——转向商品和低估值价值股。
  • 紧缩期：政策收紧，加息、严监管、去杠杆——避险为主，债券开始有配置价值。
  • 衰退期：经济增速显著放缓，企业盈利下滑——债券最佳，防御股相对抗跌。

2025-2026年的中国市场大致处于"温和复苏+低利率"环境：GDP增速在4.5%-5%区间，CPI在0.5%-1%的低位徘徊，十年期国债收益率在2.0%-2.3%的历史低位。这个组合下，按照美林时钟的框架，股票和债券都有配置价值——事实上2025年初的债牛和下半年的AI股行情正是这种宏观背景下的产物。

27.3 全天候策略：达利欧的智慧

桥水基金的创始人雷·达利欧在1996年提出了"全天候策略"（All Weather Strategy），核心理念是：不去预测未来哪个经济环境会占主导，而是构建一个在任何经济环境下都能稳健表现的组合。

具体来说，全天候策略将经济环境分为四个象限：增长高于/低于预期，通胀高于/低于预期。每个象限配置25%的风险。然后： - 增长超预期+通胀超预期：配置商品、黄金、新兴市场股票 - 增长超预期+通胀低于预期：配置股票、公司信用债 - 增长低于预期+通胀超预期：配置通胀挂钩债券、黄金 - 增长低于预期+通胀低于预期：配置国债、TIPS（通胀保护债券）

全天候策略与传统资产配置的最大区别在于：它按"风险贡献"来分配资金，而不是按资金规模。 股票的波动率远高于债券，所以在全天候策略中，股票的配置比例可能只有30%，但贡献了25%左右的风险——剩下的70%资金买了债券和商品，这部分贡献了75%的风险。

下面是一个简化的风险平价权重计算示例：

# 风险平价权重计算（Python伪代码）
        import numpy as np

        # 假设三个资产的历史年化波动率
        vol_equity = 0.20    # 股票 20%
        vol_bond   = 0.05    # 债券 5%
        vol_gold   = 0.15    # 黄金 15%

        # 目标：每个资产贡献相同比例的风险
        # 风险贡献 ∝ 权重 × 波动率
        # 约束：w1 + w2 + w3 = 1

        # 解析解（不考虑相关性时）：
        # w_i ∝ 1/σ_i

        inv_vol_equity = 1 / vol_equity   # 5.0
        inv_vol_bond   = 1 / vol_bond     # 20.0
        inv_vol_gold   = 1 / vol_gold     # 6.67

        total = inv_vol_equity + inv_vol_bond + inv_vol_gold  # 31.67

        w_equity = inv_vol_equity / total  # 约 15.8%
        w_bond   = inv_vol_bond / total    # 约 63.2%
        w_gold   = inv_vol_gold / total    # 约 21.0%

        print(f"股票权重：{w_equity:.1%}")
        print(f"债券权重：{w_bond:.1%}")
        print(f"黄金权重：{w_gold:.1%}")

        # 验证风险贡献
        risk_equity = w_equity * vol_equity  # 3.16%
        risk_bond   = w_bond * vol_bond      # 3.16%
        risk_gold   = w_gold * vol_gold      # 3.16%
        print("三个资产等风险贡献！")

A股版全天候策略怎么做？考虑到中国市场的特点：用沪深300代替美股权益、用中债国债指数代替美债、用上海黄金交易所的黄金合约代替国际黄金、再加入一点点南华商品指数来暴露商品风险。回测显示，这个简化版全天候组合在2013-2025年间实现了年化6.8%的收益，最大回撤仅8.2%，夏普比率约1.1——虽然收益率不如满仓沪深300（年化约9%，但最大回撤超40%），但持有体验好太多了。

27.4 跨境投资的实战操作

主要渠道：QDII、港股通

中国居民合法进行海外投资的渠道有限，但已经比以前丰富多了：

  • QDII基金：投资境外股票、债券、商品的公募基金。到现在QDII额度已超过2,000亿美元，覆盖美股、港股、欧洲、日本、新兴市场以及石油、黄金等商品。购买门槛低（最低10元），适合普通投资者。
  • 港股通：内地投资者可以直接通过沪港通和深港通交易港股通标的（约500只港股）。可作为配置海外资产的起点。
  • QDLP/QDIE：合格境内有限合伙人/合格境内投资企业，面向高净值个人和机构，投资范围更广（可投对冲基金、私募股权等）。
  • 沪伦通/中瑞通：存托凭证（GDR/CDR）互挂，让A股投资者可间接投资伦敦、瑞士等交易所的标的。

汇率：被忽略的收益杀手

这里我必须郑重提醒一个很多投资者都忽略的问题——汇率风险。

我有个朋友在2022年买入了一只投资美股的QDII基金，当时人民币兑美元汇率大约是6.30。到2024年中，汇率贬值到了7.20左右，也就是说他持有的美元资产仅仅因为汇率变动就获得了约14%的额外收益。他很开心。但2025年人民币企稳回升，到2026年6月汇率回到了6.80附近——之前的部分汇率收益回吐了。

而反过来，他同期还买了一只投资日本股市的QDII基金。日股本身涨得不错（日经指数两年涨了约30%），但他完全忽略了日元汇率。2022-2025年间日元对人民币贬值了超过25%！结果他的日股QDII虽然日元计价涨了30%，但换成人民币后收益几乎被汇率吃掉——两年下来总收益只有不到5%。

惨痛教训：跨市场投资时，你暴露在"双重风险"之中——资产价格风险 + 汇率风险。汇率波动在短期可能完全掩盖资产本身的收益。对冲方案包括：①选择有汇率对冲份额的QDII基金（通常带"对冲"或"Hedged"字样）；②利用银行远期结售汇锁定汇率；③对于港币资产，由于港币与美元挂钩的联系汇率制度，可以相对放心一些。

跨境投资的税务问题

跨境投资的税务处理是一个复杂但无法回避的话题。简单来说：

  • A股投资者投资港股通：股息税20%（H股）或10%（非H股），资本利得暂免。
  • QDII基金：分红按10%代扣代缴，基金本身交易的资本利得暂免。
  • 个人直接买卖美股：需自行申报，有双向税务协定的条件下，股息预扣税10%-30%，资本利得税取决于是否是美国税务居民。
  • CRS（共同申报准则）已经全面实施：个人境外金融账户信息会与国内税务机关自动交换，不要想着"海外账户不申报"。

27.5 全球宏观对冲框架

真正的宏观对冲不只是"买点美股分散风险"，而是有完整框架的。

经典宏观交易策略

  • 多空配对（Pair Trading）：做多一个经济体/行业的同时做空另一个。比如"做多中国AI产业链 + 做空传统地产——对冲国内经济结构转型风险"。
  • 国家轮动（Country Rotation）：在不同国家的股票指数之间轮动。2025年典型的轮动路径：美国AI牛→日本半导体周期复苏→印度制造业崛起→欧洲国防开支扩张。
  • 货币套利（Currency Carry）：借入低利率货币（日元、瑞郎），投资高利率货币（澳元、巴西里尔）的资产，赚取利差。但要注意，2024-2025年日元套利交易在日央行加息时出现了剧烈波动。

外资巨头的宏观交易案例

我们来看两个经典案例：

老虎基金（Tiger Management）：朱利安·罗伯逊在1990年代的操作堪称宏观交易的教科书。他预见到日本资产泡沫的破裂，大规模做空日本股市和做多日本国债。同时，他认为美国科技股将成为下一个时代的引擎，重仓科技股。1998年亚洲金融危机期间，他正确做空了东南亚货币和股票市场。一笔交易的核心是——对宏观经济格局的深刻理解 + 对全球资本流向的敏锐感知。

量子基金（Quantum Fund）：乔治·索罗斯最著名的宏观交易当然是1992年狙击英镑——做空英镑、做多德国马克，单笔获利10亿美元。这笔交易建立在他对欧洲汇率机制（ERM）不可持续性的判断上。索罗斯的"反身性理论"认为：市场参与者的偏见会影响市场本身，从而形成正反馈循环。这个理论在宏观交易中尤其适用。

当然，对于普通投资者来说，不必也没有能力复制这些顶级宏观基金的操作。但我们可以学习他们的思维方式：不要只盯着K线图，要把目光拉高到宏观经济、资本流向、政策变化、汇率走势的层面。

给个人投资者的全球配置建议：①核心组合（70%）：A股优质宽基指增+中债国债+黄金ETF；②卫星组合（20%）：QDII基金（美股科技+新兴市场+商品）；③现金武器（10%）：随时可以动用的流动性。这个框架不需要复杂的操作，但已经能抓住全球资产配置的精髓。
