开篇：当判别分析遇上降维

大家好，欢迎继续学习。上一章我们从距离判别、LDA/QDA 到贝叶斯判别，建立了判别分析的基础框架。你可能会注意到一个关键问题：当我们有几十甚至上百个特征变量时，该怎么处理？直接把所有变量扔进 LDA 并不是一个好主意——多余的特征会引入噪声，降低判别准确率，而且某些变量之间可能存在高度的相关性（多重共线性）。

这一章我们沿着两条线深入：第一，回到 Fisher 判别的原点，理解它的几何本质——它本质上是一个"降维+分类"的组合策略，将高维数据投影到一个低维判别空间中再分类。第二，学习逐步判别分析——一种自动筛选重要特征的方法，帮你从杂乱无章的变量集合中找到真正有区分能力的"关键指标"。

在这一章结束时，判别分析的方法论就完整了。下一章我们将开启课程的新篇章——典型相关分析。

Fisher判别的几何本质

费希尔在1936年提出的判别分析方法，其核心观点和上一章的统计 LDA 有所不同。Fisher 的出发点是一个纯粹的几何问题：找到一条直线（或一个低维投影方向），把数据投影上去之后，不同类别的样本尽可能分开。

具体来说，Fisher 定义了两个衡量指标：

  • 类间离散度 (between-class scatter)：投影后各类均值之间的差异——越大越好
  • 类内离散度 (within-class scatter)：投影后各类内部样本的散布——越小越好

Fisher 的优化目标就是最大化两者的比值，这个比值被称为瑞利商 (Rayleigh quotient)：

# Fisher 判别的优化目标
max_a  J(a) = (aᵀ S_B a) / (aᵀ S_W a)

# S_B = 类间散布矩阵 (between-class scatter)
# S_W = 类内散布矩阵 (within-class scatter)
# a = 投影向量（我们要求的方向）

# 求解：这是一个广义特征值问题
# S_B a = λ S_W a
# 最优的 a 是 S_W⁻¹ S_B 的最大特征值对应的特征向量

# 对于二分类问题，S_W⁻¹ S_B 只有一个非零特征值
# 投影方向为：
# a = S_W⁻¹ (μ₁ - μ₂)
# 这和我们上一章 LDA 的判别函数方向完全一致！

所以 Fisher 判别的几何本质是——在原始 p 维空间中找一个一维投影，使得"类间方差 / 类内方差"最大。这个投影方向不一定与连接两个类中心的向量方向相同——因为它还要考虑类内散布的"形状"（由 S_W 编码）。

直觉理解：想象两个椭球形的类，它们的中心连线方向是东北-西南。但东北-西南方向恰好也是椭圆的长轴方向——沿着这个方向投影，两个类内部的样本散布很大，反而可能重叠。Fisher 判别找到的方向，会"避开"高变异方向，选择那些能"拉开"不同类、同时"压缩"同类散布的方向。这就是为什么 S_W⁻¹ 起到了"白化"的作用——它把散布大的方向"压扁"了。

多类别Fisher判别

当类别数 K > 2 时，一个方向（一条直线）不足以充分展示所有类别之间的差异。Fisher 判别的推广是：寻找一组正交的投影方向，逐步捕获各类之间的差异。

一个重要的结论是：对于 K 个类的判别问题，最多可以找到 min(K-1, p) 个非零的判别方向（判别函数）。这些判别函数按照特征值从大到小排列，每个反映了"类间/类内"比值递减的投影方向。

# 多类别 Fisher 判别

# S_B 的秩最多为 K-1（因为 Σ nₖ(μₖ - μ) = 0，各均值向量线性相关）
# 所以 S_W⁻¹ S_B 最多只有 K-1 个非零特征值

# 对于 iris 数据 (K=3, p=4)：
# - 最多 min(3-1, 4) = 2 个判别函数
# - 第一个判别函数解释了约 99% 的类间差异
# - 第二个判别函数解释了剩余的 1%

# 判别空间可视化：
# 将原始 4 维数据投影到 2 维判别空间
# 在这个空间中，三个鸢尾花物种的分离情况一目了然

在多类别场景中，Fisher 判别不仅是一种分类方法，更是一种判别式的降维技术——类似于 PCA，但有监督。PCA 寻找的是"方差最大"的方向（不管类别标签），而 Fisher 判别寻找的是"类间分离最好"的方向（利用了标签信息）。两者的对比很有意思：

维度PCAFisher判别
学习范式无监督有监督
优化目标最大化投影方差最大化类间/类内方差比
利用标签否是
降维上限p 个主成分K-1 个判别方向
最适合数据探索、压缩分类前的降维

典型判别函数

在多类别 Fisher 判别中，我们得到的每个投影方向对应的判别函数称为典型判别函数 (canonical discriminant functions)。这些函数有几个重要概念：

特征值 (eigenvalue)：每个典型判别函数对应的特征值 λₖ 代表"该方向的类间方差 / 类内方差"比值。特征值越大，该判别函数越重要。

贡献率：第 k 个判别函数的贡献率 = λₖ / Σ λᵢ。它衡量了该函数在"解释类间差异"中的相对重要性。特征值贡献率可以用来判断需要保留多少个判别函数——和 PCA 中的"累积方差贡献率"思路类似。

典型相关系数 (canonical correlation)：它衡量的是"判别分数"与"类别成员指示变量"之间的相关性。值在 0-1 之间，越大说明判别效果越好。

# 典型判别分析的输出解读

# 以 iris 数据为例：
# 判别函数1：特征值 = 32.19, 贡献率 = 98.9%
# 判别函数2：特征值 = 0.29, 贡献率 = 1.1%

# 这意味着：
# - 几乎所有的类间差异都被第一个判别函数捕获了
# - 第二个判别函数贡献很小，可以忽略
# - 所以我们只需要一个判别函数（一维投影）就能很好地区分三种鸢尾花

# 判别函数系数（原始变量的权重）：
# LD1 = 0.83 × Sepal.Length + 1.53 × Sepal.Width
#      - 2.20 × Petal.Length - 2.81 × Petal.Width
# 可以看到，花瓣相关的变量权重最大（绝对值）
# 这和我们的直觉一致：花瓣长度和宽度是区分物种的最关键特征

典型判别函数的得分可以用于可视化：将样本的原始特征代入判别函数，得到每个样本在判别空间中的坐标，然后绘制散点图。一张漂亮的判别空间图，往往能直观地展示出不同类别之间的分离程度。

逐步判别分析

当你面前有几十个候选变量时，如何选出对分类"最有贡献"的那几个？这就是逐步判别分析 (Stepwise Discriminant Analysis) 要解决的问题。

逐步判别分析的核心思想与逐步回归类似：它按照一定的统计准则，逐步将"有用"的变量加入模型（向前选择），或将"无用"的变量从模型中剔除（向后剔除）。筛选变量的统计准则常用 Wilks' Λ (Wilks' Lambda, 威尔克斯 Lambda)：

# Wilks' Λ 统计量
Λ = |S_W| / |S_T|

# S_W = 类内散布矩阵
# S_T = 总散布矩阵 = S_B + S_W
# Λ 的取值范围：[0, 1]
# Λ 越小 → 类内散布相对于总散布越小 → 判别效果越好

# 在逐步判别中，我们计算"变量加入/剔除后 Λ 的变化"
# 用 F 统计量检验这个变化是否显著

逐步判别的具体步骤：

算法：逐步判别分析

1. 初始化：模型不含任何变量
2. 向前选择：
   a. 遍历所有不在模型中的变量
   b. 计算每个变量加入后 Wilks' Λ 的下降量
   c. 选择使 Λ 下降最多的变量
   d. 如果对应的 F-to-enter > F_enter_threshold，加入该变量
3. 向后剔除：
   a. 遍历所有已在模型中的变量
   b. 计算剔除每个变量后 Wilks' Λ 的上升量
   c. 如果最小的 F-to-remove < F_remove_threshold，剔除该变量
4. 重复步骤 2-3，直到没有变量可以加入或剔除

# F-to-enter / F-to-remove 是用户指定的阈值
# 常用阈值：F_enter = 3.84, F_remove = 2.71 (对应 α = 0.05 的近似值)

向前选择 (forward selection) 和向后剔除 (backward elimination) 可以组合使用，形成"双向逐步" (stepwise) 策略——每一步先尝试加入一个新变量，然后检查是否有已加入的变量"变得不重要"需要剔除。双向逐步通常比单纯的向前选择更稳定。

小心逐步判别的陷阱：逐步判别虽然方便，但它有几个众所周知的缺点。第一，它基于变量的"边际贡献"做选择，但变量之间的交互作用可能被忽略——两个单独来看微不足道的变量，结合起来可能非常有用。第二，逐步选择的顺序会影响最终选出的变量集——不同的进入/剔除阈值可能导致完全不同的结果。第三，也是最危险的——逐步选择过程本身就会引入选择和多重比较偏差，选出的变量在训练集上的表现往往显著好于它们在测试集上的真实表现。我们稍后在踩坑故事中会详细展开这一点。

判别分析的前提假设

判别分析（特别是 LDA 和 QDA）有一些重要的前提假设，在应用时需要注意检验：

多元正态性

LDA 和 QDA 都假设各类的条件分布服从多元正态分布。这个假设在实际应用中很难完全满足，但 LDA 对此具有一定的稳健性 (robustness)——即使数据轻微偏离正态，LDA 的表现通常也不会太差。但如果数据严重偏态（比如长尾分布或双峰分布），就需要考虑数据变换（如 Box-Cox 变换）后再做判别分析。

方差齐性 (Homoscedasticity)

LDA 假设各类的协方差矩阵相等。这个假设可以用 Box's M 检验 来验证。如果检验显著（p 值小），说明协方差矩阵不等，应考虑使用 QDA 或对数据进行某种变换。Box's M 检验对正态性假设比较敏感——如果数据不是正态的，这个检验的结果只能作为参考。

多重共线性

当变量之间存在高度相关性（多重共线性）时，协方差矩阵 Σ 接近奇异（行列式接近 0），求逆变得不稳定，导致判别函数系数的估计方差急剧膨胀。在这种情况下，应考虑使用 PCA 先降维，或者使用正则化的判别分析方法。

假设检验方法违反后的对策
多元正态性Mardia偏度/峰度检验、Q-Q图数据变换、使用非参数方法
协方差齐性Box's M 检验改用 QDA、正则化判别
无多重共线性方差膨胀因子 (VIF)PCA 降维、逐步筛选、岭判别

模型评估

评估判别模型的表现，光靠训练集上的准确率是不够的——过拟合效应会给你"模型表现极好"的假象。以下是三种可靠评估策略：

留一交叉验证 (LOOCV: Leave-One-Out Cross-Validation)：每次留出一个样本作为验证，用其余 n-1 个样本训练模型，预测留出的样本。重复 n 次，统计正确分类的比例。LOOCV 的偏差很小（几乎无偏估计），但计算量大（n 次拟合）。不过对于判别分析而言，LOOCV 有一个高效的"捷径公式"：不需要真正拟合 n 次，只需要一次拟合就可以计算出每个样本的"留一"预测结果。

K 折交叉验证 (K-fold CV)：将数据分成 K 份（通常 K=5 或 10），每次用 K-1 份训练、1 份验证，重复 K 次。K 折 CV 的方差比 LOOCV 小，计算量也小很多，是实践中广泛使用的选择。

自助法 (Bootstrap)：从原始数据中有放回地抽取 B 个自助样本（每个样本量与原始数据相同），在自助样本上训练模型，然后在原始数据上评估。自助法的特殊变种——.632 自助法——专门用于处理有放回抽样带来的"训练集和测试集的样本重叠"问题，评估结果比较准确。

from sklearn.model_selection import cross_val_score, LeaveOneOut
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
import numpy as np

# 假设 X, y 已经准备好
lda = LinearDiscriminantAnalysis()

# 10折交叉验证
scores_10fold = cross_val_score(lda, X, y, cv=10)
print(f"10折CV准确率: {scores_10fold.mean():.3f} ± {scores_10fold.std():.3f}")

# 留一交叉验证
loo = LeaveOneOut()
scores_loo = cross_val_score(lda, X, y, cv=loo)
print(f"LOOCV准确率: {scores_loo.mean():.3f}")

# 注意：LOOCV 的结果是每个样本预测对（1）或错（0）
# 所以准确率就是正确预测的比例

踩坑故事：逐步判别的过度乐观

这个教训来自我一位朋友的真实经历。他在一家医疗科技公司工作，任务是建立一个判别模型，根据患者的一系列血液化验指标来区分两种类型的糖尿病患者。原始数据有大约 50 个化验指标，但样本只有 200 例。

他用了逐步判别分析，设置了 F-to-enter 和 F-to-remove 的标准阈值，最终选出了 8 个"显著"的指标。在这 200 个样本上回代检验，模型的准确率达到了 95%——一个令人振奋的结果！他把结果汇报给医学团队，准备进入临床验证阶段。

问题在于：他没有做交叉验证。模型中所有这些变量都是通过"在全部 200 个样本上反复测试"而被选中的。逐步选择过程本身已经"看过"了全部数据，等价于在训练集上做了很多次假设检验。其中一些变量"显著"纯粹是样本内的偶然模式——放到新数据上就会消失。后来我的朋友在医学团队的要求下重新做了留一交叉验证，结果准确率跌到了 72%，和随机选 8 个变量做 LDA 的结果差不多。这 23 个百分点的差距，就是"逐步判别 + 不做交叉验证"带来的过度乐观。

教训：逐步判别必须在交叉验证的框架内使用。具体做法是：在每一折的训练集内部执行变量筛选，然后在对应的验证集上评估筛选后的模型。这样得到的准确率才是无偏的。永远不要先在全部数据上做逐步筛选、再在全部数据上做交叉验证——这会让交叉验证失效，因为筛选过程已经"窥视"了验证数据。另一个更安全的策略是：用岭判别 (regularized discriminant analysis, RDA) 或 LASSO 逻辑回归替代逐步判别——正则化方法可以在所有变量上同时施加约束，避免"硬选择"带来的不稳定性。

本章小结与衔接

在这章中，我们从 Fisher 判别的几何本质出发，理解了它通过最大化"类间/类内方差比"来寻找最佳投影方向的原理。然后我们将这一思想推广到多类别情形，学习了典型判别函数——它既是一种分类方法，也是一种有监督的降维技术。之后我们介绍了逐步判别分析，它通过 Wilks' Λ 统计量来评估每个变量的判别贡献，逐步筛选出"有用"的变量组合。我们讨论了判别分析的三项关键前提假设（正态性、方差齐性、无多重共线性）以及违反时的对策。最后，我们用一个真实的踩坑故事说明了逐步判别不做交叉验证的可怕后果。

至此，关于判别分析的两章内容就全部结束了。我们覆盖了从距离判别到贝叶斯判别、从 LDA/QDA 到 Fisher 判别和逐步判别的完整方法体系。从下一章开始，我们将开启课程中第五大核心方法的学习——典型相关分析 (Canonical Correlation Analysis, CCA)。CCA 要回答的问题是：当你有两组变量时（比如一组是学生的各科成绩，另一组是他们的心理测评指标），如何衡量这两组变量之间的整体相关性？这比简单计算两两相关系数要深刻得多。我们下一章见！
