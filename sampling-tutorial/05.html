第5章 分层抽样（下）——最优分配与实际应用

大家好，我是你们的讲师。上一章我们学习了分层抽样的基本原理和比例分配——每层的样本量与该层在总体中的规模成正比。比例分配虽然简单，但它有一个隐含假设：各层的方差相同。但在现实中，不同层的变异性往往相差很大。

举个极端的例子：估计全国居民收入，如果你把"北上广深"作为一层、"五线县城"作为另一层——前者内部的收入差距（从外卖骑手到投行高管）远比后者大得多。在这样的情况下，比例分配并不是最有效率的方案。我们需要一种更聪明的分配方式：把更多的样本分配到方差大的层中，以最大限度地降低总体方差。这就是本章要讲解的最优分配（Optimal Allocation）和奈曼分配（Neyman Allocation）。

最优分配的基本原理

问题设定

在分层抽样中，给定总样本量n，我们需要决定每层的样本量n₁, n₂, ..., nH，使得总体估计量的方差最小化。这就是一个带约束的优化问题：

目标：最小化 Var(x̄st) = Σ (Nₕ/N)² × (σₕ²/nₕ)（忽略FPC）

约束：Σ nₕ = n

利用拉格朗日乘数法（Lagrange Multiplier），我们可以推导出最优分配的一般公式：

nₕ = n × (Nₕ × σₕ) / Σ (Nk × σk)

这就是奈曼分配（Neyman Allocation），以统计学家Jerzy Neyman命名。它的直觉非常简单：每层的样本量与该层的规模Nₕ和层内标准差σₕ的乘积成正比。层越大、层内变异越大，就给这个层分配越多的样本。

奈曼分配 vs 比例分配：比例分配是奈曼分配的一个特例——当各层的标准差σₕ都相等时，奈曼分配就退化成了比例分配。这也意味着：如果各层的方差确实差不多，那比例分配就是最优的，没必要用更复杂的分配方案。在实践中，我建议你先估计各层的标准差（可以用历史数据或小规模预调查），然后根据σₕ的差异程度来决定是否值得使用奈曼分配。

考虑费用的最优分配

现实世界还有一个重要的约束：成本。不同层的调查成本可能相差很大。比如：

  • 对城市居民的调查成本较低（交通方便、入户容易）
  • 对偏远农村居民的调查成本较高（交通不便、分散）
  • 对某些专业群体的调查可能需要更高的"出场费"或激励

当各层单位调查成本cₕ不同时，目标函数变成：在给定总预算C（而非总样本量n）的条件下，最小化方差。其最优分配公式为：

nₕ = (C / cₕ) × (Nₕ × σₕ / √cₕ) / Σ (Nk × σk / √ck)

这个公式告诉我们：方差的层多抽，成本高的层少抽。注意这里的"多抽"和"少抽"不是任意的——它精确平衡了"降低方差带来的收益"和"增加样本带来的成本"。在给定预算下，这种分配方式能让你获得最高的统计精度。

一个深刻的洞察：在考虑费用的最优分配中，你其实是在做一个投资组合优化（Portfolio Optimization）——把有限的"样本预算"分配到不同的"投资标的"（各层）上，使得"整体收益"（估计精度）最大化。这和金融学里把资金分配到不同股票上的逻辑是相通的：方差大的层就像高波动股票，需要更多的投资（样本）来获得稳定的回报；成本高的层就像有手续费的基金，你需要权衡投入和产出。

三种分配方式的效率对比

为了直观理解三种分配方式的差异，我们假设一个简单的场景：总体分为3层，参数如下：

层 Nₕ σₕ cₕ（元）
A 1000 5 10
B 2000 20 20
C 7000 10 15

总样本量n=500。三种分配方式的结果：

方法 nA nB nC Var(x̄st) 总费用（元）
比例分配 50 100 350 0.082 7,750
奈曼分配 18 146 336 0.055 8,180
费用最优分配 24 125 351 0.058 7,715

关键观察：

  • 比例分配方差最大（0.082），因为它没有考虑B层方差远大于其他层的事实
  • 奈曼分配把最多的样本分配给了方差最大的B层（146人），方差降低了33%（0.082→0.055）
  • 费用最优分配在几乎相同的费用下（7,715 vs 7,750），方差比比例分配降低了29%

这个例子清楚地展示了：当各层方差差异较大时，合理的分配方式可以大幅提升效率。

def neyman_allocation(N_h, std_h, n):
    """
    奈曼分配：计算各层最优样本量
    N_h: 各层总体大小列表
    std_h: 各层标准差列表
    n: 总样本量
    """
    weighted = [N_h[i] * std_h[i] for i in range(len(N_h))]
    total_weight = sum(weighted)
    return [round(n * w / total_weight) for w in weighted]

def cost_optimal_allocation(N_h, std_h, cost_h, budget):
    """
    费用约束下的最优分配
    cost_h: 各层单位调查成本
    budget: 总预算
    """
    factor = [N_h[i] * std_h[i] / cost_h[i]**0.5 for i in range(len(N_h))]
    total_factor = sum(factor)
    return [round(budget / cost_h[i] * f / total_factor) for i, f in enumerate(factor)]

N_h = [1000, 2000, 7000]
std_h = [5, 20, 10]
n_total = 500
cost_h = [10, 20, 15]
budget = 7750

n_prop = [round(500 * N / 10000) for N in N_h]
n_neyman = neyman_allocation(N_h, std_h, n_total)
n_cost_opt = cost_optimal_allocation(N_h, std_h, cost_h, budget)

print(f"比例分配: {n_prop}")
print(f"奈曼分配: {n_neyman}")
print(f"费用最优: {n_cost_opt}")

设计效应（Design Effect）

为了比较不同抽样设计的效率，统计学家引入了设计效应（Design Effect, Deff）的概念。它是目标设计的方差与相同样本量的SRS方差之比：

Deff = Var(当前设计) / Var(SRS)

对于分层抽样：

Deff分层 = 1 - (σ²组间 / σ²)

由于σ²组间 ≥ 0，所以Deff ≤ 1——分层抽样的效率总是高于（或等于）SRS。层间差异越大，Deff越小，效率提升越明显。

分层抽样的陷阱：虽然分层抽样理论上总是优于或等于SRS，但这有个前提——你必须在抽样之前就知道每个个体属于哪一层。如果你在做完调查之后才拿到分层信息，那就不是真正的分层抽样，而是"事后分层"（Post-stratification）。事后分层也能在一定程度上提高效率，但它的性质比事先分层要复杂得多。另一个陷阱是：如果你分层过多（比如分了20层以上），某些层内的样本量过小，反而会降低估计的稳定性。一般建议每层至少要有10-20个样本。

真实案例：全国教育水平调查

让我们来看一个真实的案例。某研究团队计划在全国范围内进行一项基础教育水平评估（类似PISA测试的规模）。总体包含城市学校、县城学校和农村学校三类。从预调查和历史数据中，他们得到了各类学校的参数：

  • 城市学校（约占25%）：学生成绩标准差σ=50，调查成本约200元/校
  • 县城学校（约占35%）：学生成绩标准差σ=70，调查成本约350元/校
  • 农村学校（约占40%）：学生成绩标准差σ=100，调查成本约500元/校

注意农村学校的标准差远大于城市学校——因为农村学校之间的教学质量差异极大，从非常差的偏远村小到还不错的乡镇中心校都有。如果使用比例分配，农村学校虽然占总体40%，但40%的样本量不足以精确估计这样一个高度异质的群体。

团队最终采用了奈曼分配：

  • 城市学校：占总样本的 0.25×50 / (0.25×50 + 0.35×70 + 0.40×100) ≈ 13.5%
  • 县城学校：占总样本的 0.35×70 / ... ≈ 26.5%
  • 农村学校：占总样本的 0.40×100 / ... ≈ 60.0%

与比例分配相比（城市25%、县城35%、农村40%），奈曼分配把农村学校的样本量从40%大幅提升到了60%。在总样本量不变的情况下，总体估计的方差降低了约20%。这些额外的样本被投入到了最需要它们的地方——方差最大的农村学校。

关键启示：这个案例说明了一个非常重要的原则——"公平分配"不等于"等比例分配"。如果某些子群体的异质性远大于其他群体，为了精确估计整体参数，你需要对这些群体进行"过抽样"（Oversampling）。这在医学研究中也很常见：在研究罕见病时，你通常会对病例组进行过抽样，以确保有足够的统计功效。这看起来"不公平"（病例组占比远高于它在总体中的比例），但从统计效率的角度看是完全合理的。

实践中的注意事项

在实际运用分层抽样时，有几点值得特别注意：

  • 层数不宜过多：每层的样本量至少需要10-20个才能有可靠的方差估计。20层以上就要小心了。
  • 需要先验信息：奈曼分配和费用最优分配都需要知道各层的σₕ和cₕ——这些通常来自历史数据、预调查或专家判断。如果估计不准，实际效率可能低于理论值。
  • 层量（Nₕ）必须已知：计算加权估计时，各层的规模Nₕ必须是已知的（或至少能准确估计）。这看起来简单，但在实际操作中往往是一个挑战——比如你知道某市有约30%的居民住农村，但具体到街道层面的精确数字可能不可靠。
  • 交叉分层：如果同时按地区、性别、年龄段多层分层，层数会指数级增长。这时需要在精度和可操作性之间做权衡。

本章小结

在这一章中，我们学习了分层抽样的进阶内容。奈曼分配通过在方差大的层中分配更多样本来最大化统计效率；费用最优分配在给定预算下找到最佳样本分配方案；设计效应提供了一个衡量不同抽样设计效率的通用指标。实际案例表明，当各层方差差异较大时，最优分配可以显著提高效率。
