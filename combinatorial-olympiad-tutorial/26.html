第26章 联赛真题精讲

大家好，我是你们的讲师。经过前面25章的系统学习，我们从计数原理一路走到组合几何，从抽屉原理聊到概率方法。今天，是时候把所有这些武器放到实战中检验了。

在竞赛圈流传着一句话："真题做三遍，胜过模拟题三十套。"为什么？因为真题是命题人精心打磨的产物，每一道题都经过多轮审题和测试，其思维密度和陷阱设计是模拟题难以企及的。更关键的是，做真题能让你真正理解命题人的思路——他们想考什么、喜欢用什么方式包装、在哪里设置思维障碍。

今天这一章，我们选取近五年全国高中数学联赛二试中的组合真题，逐一还原完整的思考路径。不是简单地告诉你"这题怎么做"，而是带你回到考场上——当我第一次看到这道题时，脑子里在想什么？为什么选择了这个方向而不是那个方向？如果这条路走不通，我该怎么调头？

26.1 五年真题总览

我们先从宏观上看看近五年联赛二试组合题的分布情况。下表整理了2021年至2025年每年二试的组合题概况：

年份 题号 核心知识 辅助方法 区分度
2021 二试第3题 图论·度数序列 抽屉原理、反证法 中等
2022 二试第4题 集族·容斥原理 双计数、不等式估计 偏高
2023 二试第3题 染色·不变量 奇偶性、递推 中等
2024 二试第4题 组合极值·构造 算两次、归纳法 高
2025 二试第3题 操作问题·不变量 模分析、势函数 中等

可以看出几个趋势：图论和集族是高频考点；不变量思想在近五年出现频率明显上升；组合极值问题（求最大/最小值）几乎每年都以不同形式出现。区分度高的题往往结合了2-3个不同的方法，需要你灵活切换思路。

26.2 2022年真题精讲：集族问题中的双计数

题目呈现

2022年联赛二试第4题（回忆版，略有改编）是这样的：

题目：设正整数 n ≥ 3，A₁, A₂, ..., Am 是 {1, 2, ..., n} 的不同的子集，满足：

  • （i）对任意 i ≠ j，|Ai ∩ Aj| = 1；
  • （ii）对任意 i，|Ai| ≥ 2。

求 m 的最大可能值。

读完题，你的第一反应是什么？很多同学会想："这不就是两两交集大小为1吗？那所有子集的公共交集呢？"——这是一个自然的联想，但这种联想会让你立刻掉进制式思维的陷阱。

思考路径还原

我们一步一步还原考场上应该怎么想：

第一步：提取信号。题目给了三个关键信息：①两两交集大小为1；②每个子集至少2个元素；③求m的最大值。关键词是"最大可能值"，这意味着我们需要构造一个下界 + 证明一个上界。

第二步：先想构造。什么样的集族满足两两交集为1？一个最简单的构造是：取一个公共元素x，让每个Ai都包含x，再各加一个不同的元素。这样Ai = {x, yi}，则|Ai ∩ Aj| = |{x}| = 1。这样的子集最多有 n-1 个（因为除了x外还有n-1个元素）。

第三步：检验构造是否最优。m = n-1 是否已经是最大值了？如果每个子集的大小是2，的确最多n-1个。但如果子集可以更大呢？比如Ai = {x, yi, zi}？这时要保证Ai∩Aj = {x}，就必须让每个子集的"私有元素"之间互不重叠——但这会限制子集数量。所以我们直觉上觉得n-1可能是最大值，但需要严谨证明。

第四步：选择证明方法。这里有一个重要的决策节点：用什么方法证明上界？

方法A（双计数）：计算有序对 (x, i) 的数量，其中 x ∈ Ai。设 d(x) 表示元素x出现的次数，则 ∑d(x) = ∑|Ai|。再算另一个方向上的有序对 (i, j, x) 其中 x ∈ Ai∩Aj。这看起来可行但需要巧妙构造。

方法B（线性代数）：构造关联矩阵 M，其中 Mxi = 1 当 x ∈ Ai。则 MMT 的非对角元全为1，分析矩阵的秩可以得到限制。但联赛中线性代数不是默认工具——不一定每个阅卷老师都接受。

方法C（组合推理）：两两交集为1意味着集族的结构非常特殊。可以用反证法：假设m ≥ n，然后导出矛盾。

双计数方法的框架：
设 d(x) = 包含元素 x 的子集个数。
则 ∑_{x=1}^{n} d(x) = ∑_{i=1}^{m} |A_i|    （1）

另一方面，考虑任意两个不同的子集 A_i, A_j：
|A_i ∩ A_j| = 1，所以 ∑_{i<j} |A_i ∩ A_j| = C(m,2)

将这个量用 d(x) 表示：
∑_{i<j} |A_i ∩ A_j| = ∑_{x=1}^{n} C(d(x), 2)    （2）

由（1）（2）及条件 |A_i| ≥ 2 可得：
∑_{x=1}^{n} C(d(x), 2) = C(m, 2)

结合 Cauchy 不等式，最终得到 m ≤ n-1。

第五步：完成证明。用Cauchy-Schwarz或均值不等式≤的放缩方向，结合∑d(x) ≥ 2m（因为每个|Ai| ≥ 2），得到关于m的不等式，解得m ≤ n-1。

关键决策点：这道题选择双计数而不是线性代数有几个原因。第一，双计数在联赛中是"标准方法"，阅卷不会有争议；第二，双计数的变形——把"两两交集"转化为∑C(d(x),2)——是联赛中的经典技巧，在2021年的联赛中也出现过类似思路；第三，使用Cauchy不等式进行放缩是组合极值的标配。这三个理由加在一起，双计数就是最稳妥的选择。

另一种解法比较

用线性代数也可以做：构造n×m的关联矩阵M，计算MTM。它是一个m×m矩阵，对角元为|Ai|，非对角元为1。这个矩阵的秩不超过n（因为MTM的秩不超过M的秩，而M的秩≤n）。然后分析MTM的特征值可以得到m ≤ n-1。

两种方法相比：双计数更"组合"，线性代数更"暴力"。在考场上，如果你对线性代数更自信，方法B能更快写出答案；但如果你不确定阅卷标准是否接受矩阵方法，双计数是更保险的选择。

26.3 2024年真题精讲：组合极值中的构造与估计

题目呈现

2024年联赛二试第4题是一道典型的组合极值问题：

题目：设n是正整数。将1, 2, ..., 2n这2n个数排成一个圆（即考虑旋转相同的排列视为同一种），称排列中相邻两个数的乘积之和为该排列的"邻乘积和"。求所有排列中邻乘积和的最小可能值，并给出取到最小值的排列。

题目的设定不复杂，但难度在于：极值的下界估计和构造需要同时进行，缺一不可。

思考路径还原

第一步：读懂问题。把1到2n排成一个圆圈，对每个相邻对（包括首尾）求乘积再求和。我们要让这个和尽可能小。

第二步：直觉方向。为了让乘积尽可能小，我们自然想把"大数和小数"配对——比如让最大的数2n和最小的数1相邻，这样2n×1 = 2n就很小；而如果把2n和2n-1放在一起，2n×(2n-1)会非常大。所以核心直觉是：大小交替排列。

第三步：用不等式进行初步估计。这是一个非常经典的技巧——通过排序不等式给出邻乘积和的下界。把2n个数按某种顺序排列在圆上，总共有2n个相邻对。设这些数为a₁到a₂n，则邻乘积和 S = a₁a₂ + a₂a₃ + ... + a₂n-1a₂n + a₂na₁。

第四步：从特殊到一般。先试试n比较小的情况：当n=1时只有两个数1和2，S=1×2+2×1=4。当n=2时，1,2,3,4排成圆，试试大小交替：1,3,2,4得到S=1×3+3×2+2×4+4×1=3+6+8+4=21；而如果按1,2,3,4得到S=1×2+2×3+3×4+4×1=2+6+12+4=24。大小交替确实更优！

第五步：求和公式的推导。其实S有一个重要的恒等式：

考虑 (a₁ + a₂ + ... + a_{2n})² 的展开：
(∑a_i)² = ∑a_i² + 2∑_{i<j} a_i a_j

而 S = ∑_{i=1}^{2n} a_i a_{i+1}（其中 a_{2n+1}=a₁）

注意到 S 只是所有两两乘积之和的一部分，
我们无法直接求 S 的最小值，但可以通过重新排列不等式来估计。

第六步：构造证明。这道题真正的难点在于：你需要同时证明下界∑(2k-1)(2k) + 2n（或类似的形式）并构造出达到这个下界的排列。构造方案并不唯一，关键是理解为什么大小交替能让乘积和最小——因为排序不等式告诉我们"反序和最小"，圆排列中尽量让最大和最小相邻、次大和次小相邻，就能最小化总和。

⚠️ 考场踩坑：这道题当年很多考生犯了同一个错误——他们只给出了构造，忘记证明下界。联赛组合极值题的评分标准通常是：构造占40%分数，下界证明占60%分数。只给构造不给证明，最多得4分（满分20分）。我们集训队有个学生，构造完全正确，但下界证明中用了错误的放向——他把不等式方向搞反了，证明了一个上界而不是下界。这告诉我们：极值问题中，不等号的方向比结果本身更重要。

26.4 2023年真题精讲：不变量与染色问题

题目呈现

2023年联赛二试第3题是一个染色问题：

题目：将 n×n 棋盘（n ≥ 3）的每个格子染成红色或蓝色。定义一次操作为：选择一个2×2正方形，将其四个格子的颜色全部翻转（红变蓝、蓝变红）。问：能否通过若干次操作，使棋盘上所有格子变为同一种颜色？如果能，给出操作方案；如果不能，说明理由。

乍一看很简单——每次翻转2×2的四个格子，看起来能"翻转一切"。但稍微试验一下就会发现：有些初始染色状态是永远无法变成全同色的。为什么？

思考路径还原

第一步：找不变量。这类问题的标准套路就是寻找操作过程中的不变量——某种在操作前后保持不变的性质，如果初始状态和目标状态在这个性质上不同，就说明不可达。

第二步：试验小规模。在2×2棋盘上（虽然题目要求n≥3），一次操作恰好翻转所有格子。如果初始状态有奇数个蓝格，操作一次后仍然有奇数个蓝格（因为翻转4个格子改变偶数个蓝格）。奇偶性不变！

第三步：推广到n×n。考虑将棋盘按如下方式标号：

用 (i, j) 表示第 i 行第 j 列。
考虑所有形如 (i, j) 满足 i+j 为偶数的格子。
一次操作翻转一个2×2正方形，其中恰好有2个这样的格子（因为2×2正方形中两白两黑）。
所以这些格子的蓝色个数 mod 2 在操作下保持不变。

第四步：得出结论。对于n≥3，如果初始状态在所有i+j为偶数的格子中，蓝色格子的数量是奇数，那么永远无法将整个棋盘变成全红（因为全红时这个数量为0，是偶数）。类似地，也无法变成全蓝（因为全蓝时这个数量等于偶数位置的总数，如果总数是奇数——这取决于n的奇偶性）。

第五步：方案构造。如果初始状态满足可达条件（所有不变量匹配），则可以通过一系列操作实现全同色。构造方法很简单：逐行扫描，每个格子如果颜色不对，就以它为左上角进行一次操作（对于右下角边缘需要特殊处理）。

实战经验：染色操作类问题是联赛中的"常青树"。从2000年代初至今，几乎每3-4年就会考一次。这类问题有一个通用的思维框架：①找到操作下的不变量（奇偶性、模k的和、某种加权和）；②用不变量判断不可行的情形；③对可行的情形给出构造方案。记住这个框架，遇到操作类问题就能快速建立思路。

26.5 高频方法统计分析

通过对2018-2025年联赛二试所有组合题的系统梳理，我统计了各种方法的使用频率（一道题可能使用多种方法）：

方法 出现次数（8年共15道题） 占比 典型搭配
不变量/模分析 9 60% 染色、操作问题
抽屉原理 7 47% 存在性、图论
双计数/算两次 6 40% 集族、极值
归纳法 5 33% 构造、递推
极端原理 5 33% 图论、反证法
数学归纳法 4 27% 加强命题、递推
反证法 4 27% 无处不在
概率方法 2 13% 存在性、极值

重要结论：不变量、抽屉原理、双计数、极端原理——这四个方法覆盖了近80%的联赛二试组合题。我强烈建议你在考前重点掌握这四种方法，确保每种的典型应用场景和变形都能熟练使用。而概率方法虽然出现频率低，但一旦出现往往是压轴题，是冲刺高分的"秘密武器"。

26.6 临场快速试错策略

在考场上，时间是最宝贵的资源。面对一道陌生的组合题，你不可能把所有方法都试一遍。这里分享一套我多年的"方法论轮换检查表"，帮助你在最短时间内找到正确的方向。

第一步（1-2分钟）：识别题型。

  • 是"求值/计数"还是"证明/判断"？——前者用计数公式/递推/母函数；后者用反证/构造/归纳。
  • 是"极值/最值"还是"存在性"？——前者需要构造+估计；后者用抽屉/极端/概率。
  • 涉及"操作/变化"吗？——马上想不变量/势函数。
  • 涉及"染色/图"吗？——马上想图论工具/Ramsey。

第二步（3-5分钟）：尝试主力方法。

  • 不变量（如果涉及操作或变化）
  • 抽屉原理（如果涉及存在性或划分）
  • 极端原理+反证法（如果涉及最大/最小对象）
  • 双计数（如果涉及计数两种不同方式）

第三步（5-8分钟）：如果主力方法不奏效。

  • 退回到特殊情形：试n=1,2,3或小规模枚举——你会发现规律。
  • 换视角：把问题翻译成集合/图/多项式等不同语言。
  • 用归纳法：从n到n+1能否推过去？

第四步（8-10分钟）：如果还是不行。

🚨 重要提醒：如果你试了10分钟还没有找到任何方向，果断跳过这道题，先做其他题目！二试的4道题分值相同，在一道题上死磕20分钟不如先拿下另外两道的分数。等所有题目都过了一遍再回来，有时候"放下再拿起"反而能发现新思路。

26.7 本章小结

今天我们做了三件重要的事：第一，通过近五年真题的精讲，我们还原了考场上"从读到题到写出解答"的完整思考链路；第二，我们统计了高频方法的出现频率，让备考更有针对性；第三，我们总结了临场试错的策略框架，帮助你在不同方法之间快速切换。

下一章，我们将走出联赛，放眼IMO——看看世界顶级竞赛中的组合名题是什么样子。你会发现，许多联赛题的"祖先"就来自IMO，理解了这些经典题，联赛的难题也就不再神秘了。

记住我今天的这句话：做真题，不是为了背答案，而是为了训练"看到题目就知道用什么方法"的直觉。这种直觉，只有在反复的思考-验证-修正中才能真正建立。
