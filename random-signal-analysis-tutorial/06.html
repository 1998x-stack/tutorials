什么是随机信号

大家好，我是你们的讲师。经过前五章的概率论基础准备——从随机变量到分布函数、数字特征再到极限定理——我们已经搭建好了所有必要的数学工具。从这一章开始，我们要正式进入这门课的主题：随机信号。

或许你会问：前五章我们花了这么多篇幅学随机变量，和随机信号有什么区别？区别在于一个关键词——时间。随机变量是一个"快照"：在某个固定的时刻 t₀，信号的幅度 X(t₀) 是一个随机变量。但信号是随时间演化的——我们关心的是整个时间历程上的统计规律，而不仅仅是单个时刻。

定义：随时间变化、取值由概率法则决定的信号

一个随机信号 (random signal)，在数学上等同于一个随机过程 (stochastic process)。它是一个定义在概率空间 (Ω, ℱ, P) 上的函数族 {X(t, ω), t ∈ T, ω ∈ Ω}，其中 t 是时间（或更一般的索引参数），ω 是随机性来源的样本点。

等价地，随机信号可以从两个视角来理解：

  • 固定 t：X(t) 是一个随机变量——描述 t 时刻信号幅度的不确定性
  • 固定 ω：X(ω) 是一个确定的时间函数——称为一个样本函数 (sample function) 或实现 (realization)

随机信号是随机过程的样本函数（实现/轨道）

让我们用通信实验来建立直觉。假设你在实验室中搭建了一个接收机来记录某个频率上的信号。你打开接收机，记录了 10 秒的数据。然后关掉再打开，再记录 10 秒——重复 100 次。

你得到了 100 条不同的波形。每一条波形都是"可能的世界"中的一个实例——这就是一个样本函数。这 100 条波形构成的整体（理论上说，当重复次数 → ∞）就是随机过程 {X(t)}，而每一条具体波形就是该过程的一个实现。

核心直觉：随机过程 = 所有可能波形的集合 + 每条波形出现的概率规则。你永远只能观测到有限个实现，但理论分析是在整个集合上进行的。打个比方：确定信号是一张固定的照片；随机信号是一部电影——每放映一次都会有一些随机的不同，但你关心的是所有"可能的放映版本"的统计规律。

确定信号 vs 随机信号

用同样的初始条件，确定信号可以精确复现——同一个实验室、同一台设备、同一组参数，得到的波形完全相同。随机信号则不同：即使所有条件相同，每次记录的波形都不同。差别不在于"我们不知道"，而在于本质上的不可重复性。

举个例子，一个函数发生器输出 sin(2π·1000t)——这是确定信号。你开 100 次机，每次都得到一模一样的正弦波。但电阻的热噪声电压——即使电阻相同、温度相同、时间长度相同——每次记录的电压波形都不同。这就是随机信号的本质特征：每一次实现都是不同的，但所有实现遵循相同的统计规律。

随机信号的概率描述

如何完整地描述一个随机信号？既然每个时刻的取值是随机变量，我们需要描述这些随机变量之间跨时间的依赖关系。

一阶分布：单个时刻的幅度分布

固定时间 t，X(t) 是一个随机变量，它的分布称为随机信号的一阶分布：

F_X(x; t) = P(X(t) ≤ x)

一阶分布只告诉你"在 t 时刻，信号幅度不超过 x 的概率"，它不包含任何关于"不同时刻之间如何关联"的信息。这好比只给你看了电影中某一帧的统计信息——你不知道画面之间如何变化。

二阶联合分布：两个时刻的联合统计特性

一阶分布丢失了时间维度的信息——而时间相关性恰恰是信号的灵魂。我们需要二阶联合分布：

F_X(x₁, x₂; t₁, t₂) = P(X(t₁) ≤ x₁, X(t₂) ≤ x₂)

这描述了 t₁ 和 t₂ 两个时刻的联合分布。它包含了信号的"变化快慢"信息：如果 t₁ 和 t₂ 相距很近时 X(t₁) 和 X(t₂) 总是很接近，说明信号变化缓慢（低频为主）；如果相距很近就已经不相关了，说明信号变化剧烈（高频为主）。

n阶有限维分布族：完全描述一个随机过程

严格地讲，要完全描述一个随机过程，我们需要知道所有 n 和所有时间组合 (t₁,...,tₙ) 的 n 维联合分布——这称为过程的有限维分布族 (finite-dimensional distributions)。

柯尔莫哥洛夫扩展定理 (Kolmogorov Extension Theorem) 告诉我们：如果这些有限维分布满足一致性条件（对称性和相容性），那么存在一个随机过程在所有这些时间点上的分布与给定的有限维分布族一致。这个定理在数学上保证了"用有限维分布来定义随机过程"是可行的——我们不需要直接定义一个不可数无限维的对象。

幸运的是，在工程实践中我们几乎从不使用完整的 n 维分布族。绝大多数情况下，描述到二阶统计量（即均值和自相关函数）就够了——这就是接下来要讨论的。

随机信号的数字特征

正如随机变量可以用数字特征来概括，随机信号也可以用基于时间的数字特征来描述。

均值函数 μX(t) = E[X(t)]

均值函数描述了信号在每个时刻 t 的平均水平：

μ_X(t) = E[X(t)]

这个均值一般是 t 的函数——信号可以是时变的。热噪声的均值为零（不随 t 变化），但一个接收到的脉冲信号的均值在脉冲到达时刻附近会发生变化。当均值不随时间变化时，我们称信号是一阶平稳的。

方差函数 σ²X(t) = E[(X(t)-μX(t))²]

方差函数描述了 t 时刻信号偏离均值的程度：

σ²_X(t) = E[(X(t) - μ_X(t))²]

在工程中，方差函数反映了信号的瞬时功率（扣除直流分量后）。一个静止的噪声源，其方差通常是常数——这意味着噪声的功率是平稳的。

自相关函数 RX(t₁,t₂) = E[X(t₁)X(t₂)]

自相关函数 (autocorrelation function) 是随机信号最重要的数字特征——没有之一。它定义为：

R_X(t₁, t₂) = E[X(t₁) · X(t₂)]

自相关函数衡量了信号在 t₁ 和 t₂ 两个时刻取值的相似程度：

  • 如果 |R_X| 很大 → 两个时刻的信号取值在统计上很接近（正相关）或相反（负相关）
  • 如果 R_X 接近零 → 两个时刻的取值在统计上不关联

自协方差函数 CX(t₁,t₂) = RX(t₁,t₂)-μX(t₁)μX(t₂)

自协方差函数 (autocovariance function) 是减去均值影响后的自相关：

C_X(t₁, t₂) = E[(X(t₁) - μ_X(t₁))(X(t₂) - μ_X(t₂))] = R_X(t₁, t₂) - μ_X(t₁)·μ_X(t₂)

当信号均值为零时，自相关和自协方差是同一个东西。对于零均值的信号（如通信系统中常见的情形），我们可以直接使用自相关函数来刻画信号的二阶统计特性。

为什么自相关如此重要？因为：第一，自相关函数直接决定信号的功率谱密度（第9章的维纳-辛钦定理）；第二，线性系统对随机信号的响应完全由输入的自相关函数和系统的冲激响应决定（第12-13章）；第三，许多最优滤波器（维纳滤波器）的设计只需要信号的二阶统计量。可以说，掌握自相关函数，就掌握了随机信号分析的半壁江山。

典型随机信号实例

抽象的定义讲完了，让我们通过几个具体例子来建立直觉。这些例子在后续章节中会反复出现。

随机正弦信号：X(t)=A·cos(ωt+Θ)，Θ~U(0,2π)

这是最简单的随机信号模型之一：幅度 A 和角频率 ω 固定，但相位 Θ 是在 [0, 2π) 上均匀分布的随机变量。

为什么相位是随机的？因为实际通信中，接收信号的相位取决于信道时延，而时延是未知的——你不知道信号具体在什么相位到达接收机。通过把相位建模为随机变量，我们就能分析所有可能相位的平均行为。

这个信号的均值函数：

E[X(t)] = E[A·cos(ωt + Θ)]
         = A · ∫_0^{2π} cos(ωt + θ) · (1/(2π)) dθ
         = 0

自相关函数：

R_X(t₁, t₂) = E[A²·cos(ωt₁+Θ)·cos(ωt₂+Θ)]
              = (A²/2) · cos(ω(t₂ - t₁))

（推导过程用到了积化和差公式以及 E[cos(ω(t₂+t₁)+2Θ)] = 0）

注意：自相关函数只依赖于时间差 τ = t₂ - t₁，而不依赖于绝对时间——这意味着该信号是广义平稳的，这一点我们下一章会详细讨论。

伯努利随机序列：独立的±1序列

考虑一个时间离散的随机信号：在每个整数时刻 n，Xₙ 独立地以等概率取 +1 或 -1。这被称为伯努利随机序列（也称为二进制对称序列）。它的数字特征：

均值：E[Xₙ] = 0
方差：Var(Xₙ) = E[Xₙ²] = 1
自相关：R_X(m,n) = E[XₘXₙ]
         当 m=n 时：R_X(n,n) = E[Xₙ²] = 1
         当 m≠n 时：R_X(m,n) = E[Xₘ]·E[Xₙ] = 0 (由独立性)

这个序列是白噪声的离散版本——不同时刻之间完全不相关。虽然简单，它是通信系统中基带数字信号的基础模型。

半随机二进制传输信号

在数字通信中，每个比特持续时间 T_b 内发送一个恒定的幅度（+A 或 -A），幅度在每个比特间隔内独立随机选择。这就产生了半随机二进制传输信号：

X(t) = Σₙ Aₙ · rect((t - nT_b)/T_b)

其中 Aₙ ∈ {+A, -A} 等概率独立，rect(·) 是矩形脉冲

这个信号的非平稳性体现在：在同一个比特间隔内的两个时刻，它们的取值完全相关（因为幅度是常数）；跨比特间隔的时刻则不相关（因为 Aₙ 独立）。这样的信号是循环平稳的——统计特性以 T_b 为周期重复。循环平稳是通信信号的一个核心特征，也是许多信号处理算法利用的线索。

高斯随机信号

最后介绍最重要的一个类别——高斯随机信号 (Gaussian random signal)：任意有限 n 个时间点 (t₁, t₂, ..., tₙ) 上的联合分布都是 n 维高斯分布。

高斯信号完全由均值函数 μ_X(t) 和自相关（或自协方差）函数 R_X(t₁, t₂) 决定——因为高斯分布只需要一阶矩和二阶矩就完全指定了。这意味着对于高斯信号，"二阶统计量 = 整个过程的完整描述"。这个性质使得高斯信号异常"好处理"，是所有信号处理理论的天然试验场。

信号类型均值函数自相关函数典型应用
随机正弦波0(A²/2)·cos(ωτ)带通信号建模

本章小结与衔接

本章我们从静态的随机变量跨入了动态的随机信号世界。核心概念是：随机信号 = 一族随时间索引的随机变量，它可以看作所有可能波形的集合加上概率规则。我们学习了随机信号的三种层次描述——有限维分布族（完整但复杂）、数字特征（实用且常用）、典型实例（建立直觉）。其中自相关函数是重中之重，它刻画了信号在不同时刻之间的关联性。

下一章，我们将更深入地挖掘自相关函数的含义，学习互相关函数，澄清"独立、不相关、正交"三个概念的区别，并探索随机信号的均方微积分——是的，我们可以对随机信号做微分和积分。
