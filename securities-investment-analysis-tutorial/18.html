第18章 投资策略流派

大家好，我是你们的讲师。前面各章我们讲"怎么看公司"（基本面分析），讲"怎么看价格"（技术分析），讲"怎么组合"（现代投资组合理论），讲"怎么保命"（风险管理），讲"怎么别被自己坑"（行为金融学）。是时候把这些知识整合成一套具体的投资策略了。

这一章我们来梳理投资史上的几大经典流派。它们各有各的哲学基础、选股标准和适用场景。更重要的问题是：你适合哪种策略？ 投资是一个高度个性化的事情——巴菲特的策略适合他那种精力充沛到90岁还能跳着踢踏舞去上班的人，但不一定适合你。搞清楚每种策略的核心逻辑和适用条件，才能找到真正属于自己的路。

18.1 价值投资：用40美分买1美元

格雷厄姆的烟蒂股策略

本杰明·格雷厄姆（Benjamin Graham）——价值投资的创始人，著有《证券分析》和《聪明的投资者》。他的核心思想极度简单：买入价格低于内在价值的股票。 他发明了"烟蒂股"（Cigar Butt）的概念：就像大街上捡到的烟蒂只剩最后一口，但那一口是免费的。一家公司的股价远低于其净流动资产价值（Net Current Asset Value），即使公司经营一般，最后的清算价值也高于股价。

格雷厄姆有一个著名的选股标准——格雷厄姆数（Graham Number）：

格雷厄姆数 = sqrt(22.5 x EPS x BVPS)

        其中：
        EPS = 每股收益（Earnings Per Share）
        BVPS = 每股净资产（Book Value Per Share）
        22.5 = 15 x 1.5 （15倍PE x 1.5倍PB，格雷厄姆认为合理的上限）

        举例：某公司EPS=3元，BVPS=20元
        格雷厄姆数 = sqrt(22.5 x 3 x 20) = sqrt(1350) ~ 36.7元

        如果当前股价 < 36.7元，则股票被低估（按格雷厄姆的标准）
        如果当前股价 < 36.7 x 2/3 ~ 24.5元，则具有显著安全边际

格雷厄姆要求足够的安全边际（Margin of Safety）：买入价至少比内在价值低1/3。这个安全边际不是为了精确估值——它是因为你一定会犯错。市场不会因为你"算得很认真"就给你好结果，安全边际是你唯一能依赖的保护。

巴菲特的护城河升级

巴菲特早期完全遵循格雷厄姆的烟蒂策略，但后来在查理·芒格（Charlie Munger）的影响下完成了进化。巴菲特的升级版本：用合理的价格买入优质公司，而不是用极低的价格买入普通公司。

巴菲特提出了"护城河"（Moat）的概念——寻找那些具有持久竞争优势的公司：品牌溢价（可口可乐）、网络效应（Visa）、转换成本（穆迪）、成本优势（GEICO）。护城河越宽，公司越能在竞争中保护自己的利润，也越值得长期持有。

巴菲特持股最长的公司——可口可乐（1988年买入，持有至今超过35年）、美国运通、穆迪。他在2008年金融危机时买入高盛和通用电气的优先股，2020年疫情时减持航空股——这些操作都体现了价值投资的核心理念：市场恐慌时你是买家，市场狂热时你是卖家。

18.2 成长投资：为未来买单

成长投资和价值投资看似对立，实则同源——都是寻找价格低于价值的公司。区别在于：价值投资寻找的是"当前资产的价值被低估"，成长投资寻找的是"未来现金流的价值被低估"。

费雪：寻找超级成长股

菲利普·费雪（Philip Fisher）是成长投资的奠基人，著有《怎样选择成长股》。他提出了一个著名的"15要点"选股框架，核心关注：公司是否有可持续的增长潜力、管理层是否优秀、研发投入是否充足。 费雪不关心股价的短期波动——如果一家公司足够优秀，他愿意持有10年甚至更久。

彼得·林奇：买你懂的

彼得·林奇（Peter Lynch）管理富达麦哲伦基金（Fidelity Magellan Fund）的13年间，年化收益率达到29%，是公募基金历史上最辉煌的记录之一。他的名言："买你懂的"（Buy what you know）。

林奇把股票分成六类，每类有对应的管理策略：

  • 缓慢增长型（Slow Growers）：大型成熟公司，分红是主要回报来源，不要期待高增长。
  • 稳健增长型（Stalwarts）：如可口可乐、宝洁，每年增长10%-12%，可以在市场下跌时作为避风港。
  • 快速增长型（Fast Growers）：年增长20%-30%的小公司，林奇最喜欢的类型。关键要判断增长可持续性，不要为过高的增长预期支付过高价格。
  • 周期型（Cyclicals）：汽车、钢铁、化工，在周期底部买入、顶部卖出。
  • 困境反转型（Turnarounds）：已经破产或濒临破产的公司，风险极高但回报也极高。
  • 资产隐蔽型（Asset Plays）：公司拥有潜在价值未被市场认识的资产，如土地、品牌、子公司股权。

18.3 动量策略：顺势而为

Jegadeesh-Titman经典研究

1993年，Narasimhan Jegadeesh和Sheridan Titman发表了一篇影响深远的论文：买入过去6个月表现最好的股票、卖出过去6个月表现最差的股票，持有6个月，可以获得约1%的月均超额收益。这就是动量策略（Momentum Strategy）的起源。

动量的逻辑基础有两个：一是信息传播缓慢（好消息需要时间被市场充分理解）；二是投资者反应不足（趋势一旦形成，自我加强的惯性会让它延续一段时间）。

A股动量效应的特殊性

动量策略在美股市场表现极好，但在A股却效果不佳。大量研究表明：A股的中短期动量效应很弱，反而反转效应（Reversal Effect）非常强。 也就是说，过去涨得多的股票接下来更可能下跌，过去跌得多的股票更可能反弹。

为什么会这样？几个可能的原因：

  • 散户主导的投资者结构：散户的追涨杀跌行为导致股价过度反应，然后回归均值。
  • A股的高波动特性：高波动环境下，动量策略容易被剧烈回调吃掉利润。
  • 政策和事件驱动：A股受政策和消息面影响很大，技术性的趋势容易被突发事件打断。

因此，在A股做趋势策略，要么做极短期的（日内或1-2天的短线动量），要么做长期的（3-5年的反转）；中间的月度/季度动量，效果很不稳定。

18.4 索罗斯反身性理论

乔治·索罗斯（George Soros）可能是这个名单里最与众不同的一个——他不承认自己属于任何传统流派。他的反身性理论（Theory of Reflexivity）对市场有一套完全不同的理解。

传统金融学假设市场是"被动反映基本面"的。索罗斯说：不对，市场的价格可以反过来改变基本面。 股价上涨让公司更容易融资、可以并购对手、扩张业务、招聘顶尖人才——基本面真的变好了，股价进一步上涨。这是一个正反馈循环。

2015年A股的杠杆牛市是反身性的经典案例。股价上涨让投资者用股票抵押融资、加杠杆买入更多、股价进一步上涨。这个正反馈在上涨时极度猛烈，但一旦反向——股价下跌触发平仓、强制卖出、股价进一步下跌——踩踏的惨烈程度同样惊人。

索罗斯的核心理念：找到"反身性循环"的起点和拐点，在大多数人还没意识到正反馈已经开始时入场，在正反馈即将逆转时离场。 这需要极强的宏观判断能力和对市场心理的敏锐感知，不是普通投资者轻易能掌握的。

建议：不要强迫自己成为"纯种"的某一流派投资者。很多成功的投资者是"混合型"的——以价值投资的理念选股（安全边际+护城河），以成长投资的眼光看未来（高ROE+可持续增长），以动量策略的时间框架思考买卖（不接下跌的飞刀），以反身性的警觉观察市场（警惕正反馈循环的顶点）。找到最适合你的那个组合，然后严格执行。

踩坑故事：在A股机械照搬格雷厄姆烟蒂策略的教训

2017年，我一个做价值投资的同行开始系统性地在A股实施格雷厄姆的"净流动资产"策略——买入价格低于每股净流动资产（NCAV）的股票。他在美股回测中发现这个策略的历史年化收益超过20%，于是满怀信心地移植到A股。

筛选结果出来了：符合NCAV标准的股票，全是钢铁、煤炭、纺织等传统行业的公司，而且绝大多数是国有企业的"老破小"。他买了十几只这样的股票，满心期待价值回归。

结果呢？这些股票在2017-2019年期间几乎没有任何上涨。同期茅台、五粮液、恒瑞医药等白马股翻了好几倍。他的组合不仅严重跑输市场，而且那些"烟蒂股"的基本面还在持续恶化——有的亏损扩大，有的被ST，有的甚至面临退市。

问题出在哪里？格雷厄姆的烟蒂策略是在20世纪30-50年代的美股市场被验证的。那个时代的美国，法律对中小股东的保护不完善，信息不透明，确实有很多被严重低估的"烟蒂"。但近90年后的A股市场环境完全不同：

  • A股的小市值"烟蒂股"往往缺乏公司治理改善的驱动力（大股东没有提高股东回报的意愿）
  • A股上市公司的壳价值在注册制改革后退化，没有"并购重组"这个价值实现路径
  • 重资产行业的资产价值可能被高估（过时的设备、老化的厂房，实际清算价值远低于账面价值）

这个教训告诉我：投资策略必须适配市场环境。 格雷厄姆的思想（安全边际、市场先生）是永恒的，但他的操作方法（买NCAV以下）必须结合具体的市场条件来调整。后来这位同行在A股改良了策略——保留"安全边际"的思想，但把筛选标准改成了"低估值+高ROE+分红率>3%"，效果就好多了。

小结与预习

这一章我们横跨了投资史上最经典的几大流派：格雷厄姆和巴菲特的底线思维、费雪和林奇的成长视角、Jegadeesh-Titman的动量验证、索罗斯对市场本质的独特洞察。它们没有对错之分，只有适合与否。重要的是理解每一种策略背后的底层逻辑和适用条件，然后找到属于你自己的那条路。
