第10章 抽样权重（上）

大家好，我是你们的讲师。到目前为止，我们的课程中反复出现了一个短语——"自加权设计"，也就是每个样本的入选概率相同。但现实世界远比理想复杂。在绝大多数实际的抽样调查中，不同个体的入选概率是不相等的：分层抽样中某些层被过度抽样、多阶段PPS抽样中各阶段概率乘积不凑巧、或者某些群体因为难以接触而不得不提高抽样强度。

当一个调查中不同的人被选中的概率不同时，不加权就分析数据，结果会有偏。这不是一个学术问题，而是一个会直接导致错误结论的实际问题。抽样权重（Sampling Weight）就是用来解决这个问题的数学工具。

一句话核心：抽样权重w = 1/π，其中π是某个个体被选入样本的概率。权重告诉你的不是"这个个体有多重要"，而是"这个个体代表了总体中的多少人"。入选概率低的个体权重更大，因为它在总体中"替"更多没被选中的同伴发声。

基础权重的定义

抽样权重的基本公式极其简单：

抽样权重 = 1 / 入选概率

w_i = 1 / π_i

其中 π_i 是第i个样本单元的入选概率（inclusion probability）。

举例：
- 简单随机抽样：从N=10000中抽n=500
  π_i = 500/10000 = 0.05
  w_i = 1/0.05 = 20
  含义：每个样本代表总体中的20个人。

- 分层抽样：某层N_h=2000，该层抽n_h=200
  π_i = 200/2000 = 0.10
  w_i = 1/0.10 = 10
  含义：该层每个样本代表10个人。

如果说得更直白一点：权重就是"放大系数"。你只调查了1个人，但他代表了总体中的20个人，所以你对他的回答要"乘以20"来求和——这就是加权估计的本质。

加权样本均值的含义

加权样本均值（Weighted Sample Mean）的计算公式如下：

加权样本均值

设有n个样本，第i个样本的值为y_i，权重为w_i。

加权均值：ȳ_w = Σ(w_i × y_i) / Σw_i

这个公式的意思是：
- 分子：每个观察值按它代表的"人数"加权求和
- 分母：所有权重之和 = 估计的总体大小

不加权均值（普通平均）：ȳ_unw = Σy_i / n
（这个假设所有人同等重要——但如果不加权真的合理吗？）
直观理解：想象你有10个农村样本和10个城市样本。但农村人口占总体的80%，城市只占20%。不加权的话，城市和农村各占50%的"发言权"——城市被严重高估。加权后，农村样本的权重更大（因为代表更多的人），城市样本的权重更小，加权均值才会接近总体真实值。

不同抽样方法下的权重计算

权重不是凭空拍脑袋想出来的，它是抽样设计内生的——每种抽样方法都天然地决定了一套权重。

抽样方法 入选概率 π 基础权重 w = 1/π
SRS n / N N / n（常数）
分层抽样（等额分配） n_h / N_h（各层可能不同） N_h / n_h（各层不同）
分层抽样（比例分配） n / N（各层相同） N / n（常数，自加权）
PPS（两阶段，有放回） f₁_i × f₂_i 1 / (f₁_i × f₂_i)
自加权两阶段设计 常数 常数

从这张表中你可以看出：自加权设计的权重是常数，这是它"方便"的根本原因。但注意，"方便"不代表"更准"——有时候故意设计成不等概率（比如为了改善某特定子总体的估计精度），然后通过加权来修正，反而能得到更优的整体效果。

加权与不加权的后果

也许你会问："权重这个东西听起来很麻烦，我就不加权直接用原始数据算个平均值，会差多少？" 让我用一个计算例子来回答你。

假设总体中有两个层：

  • 层A（农村）：N_A=80000人，真实人均收入=5000元/月
  • 层B（城市）：N_B=20000人，真实人均收入=15000元/月

总体真实人均收入 = (80000×5000 + 20000×15000) / 100000 = 7000元/月

现在做一个调查，从每层各抽100人：

不加权： (5000 + 15000) / 2 = 10000元/月 ❌ （高估43%！）
加权：
  w_A = 80000/100 = 800
  w_B = 20000/100 = 200
  加权均值 = (800×5000 + 200×15000) / (800+200)
          = (400万 + 300万) / 1000 = 7000元/月 ✅

在分层等额分配（每层抽相同人数）的情况下，不加权的结果差了3000元——对政策制定者来说，这个偏差足以导致完全不同的决策。而只用一个简单的加权修正，就得到了无偏的估计。

刻骨铭心的教训：我曾经见过一个省级健康调查，分析报告直接用了未加权的数据，得出"全省居民慢性病患病率32.5%"的结论。但检查原始设计后才发现——调查对城市居民过度抽样了（因为城市容易入户），农村居民抽样不足。加权后重新计算的患病率其实只有24.8%，差了近8个百分点。报告如果发表出去，不仅会误导卫生资源配置，还会让该省在全国排名中处于一个完全不准确的位置。幸好及时发现了这个问题。

权重的两个重要性质

学习权重时，有两个性质值得你特别留意：

性质一：权重之和估计总体量

所有样本的权重之和 Σw_i 是对总体大小的估计。在无无回答的理想情况下，Σw_i 应该等于N。如果Σw_i明显偏离N，说明权重的计算可能有误。

性质二：加权均值是总体均值的无偏估计

在概率抽样下，只要权重正确（即w_i = 1/π_i），加权样本均值就是总体均值的无偏估计。这是概率抽样的数学基石。需要满足的条件是：π_i > 0 对所有个体成立（每个个体都有非零的入选概率），且所有个体的响应概率都已知。

但请注意，"无偏"≠"准确"。权重大的个体如果在某个关键变量上恰好是极值，可能会让估计量的方差变得很大。这就是"方差-偏差权衡"的经典问题——我们会在下一章讨论极端权重的处理。

真实案例：城市偏向的偏差修正

2000年代初，某城市开展了一项"居民生活质量调查"，目的比较城市居民和农村居民的生活满意度。抽样设计很简单：在全市范围内采用分层随机抽样，按行政区划将总体分为城区和郊区两层，计划每层各抽1000人。

但在实际执行中出现了问题：城区的入户成功率远高于郊区——城区居民住在集中住宅区，访问员一天能访问20户；郊区居民住得分散，山路多，一天只能访问5-6户。最终只完成了800份农村问卷和1200份城市问卷。

城区人口占全市的45%，农村占55%。但最终样本中农村只占40%，城市占60%。如果不加权重，城市居民的满意度会被过度放大，导致全市满意度被高估。修正方案：

权重调整：
  农村：w_rural = (0.55) / (800/2000) = 0.55/0.40 = 1.375
  城市：w_urban  = (0.45) / (1200/2000) = 0.45/0.60 = 0.750

加权后的"农村代表性"从40%恢复到55%，城市从60%降到45%。
加权结果与当年的普查数据（人口普查补充调查）基本一致，
证实了权重修正的有效性。
重要提醒：这个案例中使用的权重是"事后"调整的（因为执行过程中样本分布偏离了原计划）。但更规范的做法是在设计阶段就规划好权重——不应该等到数据收回来才发现城乡比例失调，而应该事先设计好各层的目标样本量，并在执行中严格监控配额达成情况。

本章小结

这一章我们学习了抽样权重的基础知识。权重的公式很简单——w = 1/π——但其背后的意义深远：权重是个体"代表性"的量化。不同抽样方法天然地决定了不同的权重结构，而加权操作就是将样本的分布"还原"到总体分布的过程。不加权的分析在不等概率设计下会直接导致有偏估计——这个错误在实践中非常常见，但完全可以通过正确的加权来避免。

但基础权重只是一个起点。现实调查中还会遇到两个棘手的问题：一是无回答（有些人被选中但不回应），二是辅助信息校准（我们可以利用已知总体信息让权重更"准"）。这两件事正是我们下一章的重点——抽样权重的进阶应用。
