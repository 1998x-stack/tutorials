第6章 三次样条插值

大家好，我是你们的讲师。今天我们来学插值法的"王炸"——三次样条（Cubic Spline）。

先讲一个有趣的历史故事。"样条"（spline）这个词来自造船业。在计算机出现之前，船舶设计师在绘制船体曲线时，会用一根有弹性的细木条（样条）穿过一系列固定的金属压铁（被称为"ducks"）。这根细木条在自然状态下会形成一个尽可能光滑的曲线——它的物理原理是弹性梁的最小弯曲能原理。而这个数学形状，恰好就是三次样条。

🔍 数学与物理的巧合： 弹性梁在点约束下的平衡曲线满足 y⁽⁴⁾(x) = 0（除了约束点），这正是三次多项式满足的微分方程。物理世界"天然"选择了数学上最优的插值方案。

6.1 样条函数的定义

给定区间 [a, b] 上的一个划分 a = x₀ < x₁ < ... < xₙ = b，函数 S(x) 称为 三次样条，如果满足：

  • 在每个子区间 [xᵢ, xᵢ₊₁] 上，S(x) 是一个三次多项式；
  • S(x) 在节点 xᵢ 处具有 C² 连续性，即 S(xᵢ⁻) = S(xᵢ⁺)，S'(xᵢ⁻) = S'(xᵢ⁺)，S''(xᵢ⁻) = S''(xᵢ⁺)；
  • S(xᵢ) = f(xᵢ)，i = 0, 1, ..., n（插值条件）。

每个子区间上有 4 个未知系数，n 个子区间共 4n 个未知数。约束条件包括：

  • n+1 个插值条件；
  • n-1 个内部节点的 C¹ 连续条件；
  • n-1 个内部节点的 C² 连续条件；
  • 还需要 2 个额外的边界条件来闭合系统。

总计：4n 个未知数，4n 个方程——唯一可解。

6.2 三弯矩法

样条构造中最优雅的方法是三弯矩法（method of moments）。核心思想是用二阶导数 S''(xᵢ) 作为未知量（记为 Mᵢ），因为三次多项式的二阶导数是线性的。

推导过程简要概括：

  1. 在每个子区间上，S''(x) 是线性的，由端点 Mᵢ 和 Mᵢ₊₁ 唯一确定。
  2. 对 S''(x) 积分两次，得到 S(x) 的表达式，其中包含两个积分常数。
  3. 利用插值条件 S(xᵢ) = f(xᵢ) 和 S(xᵢ₊₁) = f(xᵢ₊₁) 消去积分常数。
  4. 利用 C¹ 连续条件 S'(xᵢ⁻) = S'(xᵢ⁺) 得到关于 Mᵢ 的方程。

最终得到三对角方程组：

μᵢ Mᵢ₋₁ + 2Mᵢ + λᵢ Mᵢ₊₁ = dᵢ, i = 1, 2, ..., n-1

其中 μᵢ = hᵢ / (hᵢ + hᵢ₋₁)，λᵢ = 1 − μᵢ，而 dᵢ 由已知函数值计算得到。hᵢ = xᵢ₊₁ − xᵢ 是步长。

这个方程组是三对角的、严格对角占优的——这意味着它有唯一解，而且可以用高效的追赶法求解。

💡 为什么叫"三弯矩"？ 在力学中，梁的弯矩与二阶导数成正比。Mᵢ 正是节点处的弯矩值，而三对角方程反映了相邻弯矩之间的平衡关系。这个名字完美地揭示了样条与弹性梁的物理联系。

6.3 边界条件

三弯矩方程组需要两个额外的边界条件才能闭合。常见的边界条件有：

类型 条件 适用场景 特点
自然边界 (Natural) S''(x₀) = S''(xₙ) = 0 端点为自由状态，无额外导数信息 最常用，端点处"无约束"，形状可能偏离
固支边界 (Clamped) S'(x₀) = f'(x₀), S'(xₙ) = f'(xₙ) 已知端点处的切线方向 精度最高，但需要额外导数数据
周期边界 (Periodic) S'(x₀) = S'(xₙ), S''(x₀) = S''(xₙ) 封闭曲线，如环形轨迹 一阶和二阶导数在端点处匹配
非结节边界 (Not-a-Knot) S'''(x₁⁻) = S'''(x₁⁺), S'''(xₙ₋₁⁻) = S'''(xₙ₋₁⁺) 无额外信息，但希望更精确 在前两个子区间使用同一三次多项式
⚠️ 踩坑故事： 某团队在开发火箭轨迹规划系统时，使用了自然边界的三次样条来插值遥测数据点。他们没注意到火箭在发射塔架附近有明确的初始出塔角度（即端点导数已知）。自然样条在起始段的轨迹明显偏离了实际发射路径，导致后续制导参数全部算偏。换用固支边界后，样条完美吻合了物理约束。这个故事告诉我们：边界条件不是数学装饰，而是物理信息。

6.4 求解过程与代码

三次样条的标准求解流程分三步：

  1. 计算步长 hᵢ = xᵢ₊₁ − xᵢ，以及右端项 dᵢ。
  2. 组装三对角矩阵，应用边界条件。
  3. 用追赶法（Thomas algorithm）求解三对角方程组得到 Mᵢ。
  4. 利用 Mᵢ 在每个子区间上构造三次多项式，进行求值。
import numpy as np

def cubic_spline(x_points, y_points, bc_type='natural'):
    """
    构造三次样条插值函数
    bc_type: 'natural' | 'clamped' | 'not-a-knot'
    返回一个函数 s(x)，可在任意 x 处求值
    """
    n = len(x_points)
    h = np.diff(x_points)

    # 计算右端项 d (用于内部节点 i=1,...,n-2)
    d = np.zeros(n)
    for i in range(1, n-1):
        d[i] = 3 * ((y_points[i+1] - y_points[i]) / h[i] -
                     (y_points[i] - y_points[i-1]) / h[i-1])

    # 组装三对角矩阵系数
    mu = np.zeros(n)
    lam = np.zeros(n)
    for i in range(1, n-1):
        mu[i] = h[i-1] / (h[i-1] + h[i])
        lam[i] = h[i] / (h[i-1] + h[i])

    # 应用边界条件
    A = np.zeros((n, n))
    b = np.zeros(n)

    if bc_type == 'natural':
        A[0, 0] = 1; A[n-1, n-1] = 1
        b[0] = 0; b[n-1] = 0
    elif bc_type == 'clamped':
        # 需要额外传入端点导数值，此处简化
        A[0, 0] = 2; A[0, 1] = 1
        A[n-1, n-2] = 1; A[n-1, n-1] = 2
        # b[0] 和 b[n-1] 需要根据端点导数计算
    else:  # not-a-knot
        A[0, 0] = -h[0]; A[0, 1] = h[0]
        A[n-1, n-2] = -h[n-2]; A[n-1, n-1] = h[n-2]
        b[0] = 0; b[n-1] = 0

    for i in range(1, n-1):
        A[i, i-1] = mu[i]
        A[i, i] = 2
        A[i, i+1] = lam[i]
        b[i] = d[i]

    # 追赶法求解三对角系统
    M = thomas_algorithm(A, b)
    return M, x_points, y_points, h

def thomas_algorithm(A, b):
    """追赶法求解三对角方程组"""
    n = len(b)
    a = np.diag(A, -1).copy()
    c = np.diag(A, 1).copy()
    d = np.diag(A).copy()
    x = b.copy()

    # 前推
    for i in range(1, n):
        w = a[i-1] / d[i-1]
        d[i] -= w * c[i-1]
        x[i] -= w * x[i-1]

    # 回代
    x[n-1] /= d[n-1]
    for i in range(n-2, -1, -1):
        x[i] = (x[i] - c[i] * x[i+1]) / d[i]
    return x

# 示例：插值 Runge 函数
def runge(x):
    return 1.0 / (1.0 + 25.0 * x**2)

# 使用不等距节点（在端点处加密）
n_nodes = 15
x_pts = -np.cos(np.linspace(0, np.pi, n_nodes))  # Chebyshev-like
y_pts = runge(x_pts)

M, xp, yp, h = cubic_spline(x_pts, y_pts, bc_type='natural')

# 测试插值精度
x_test = 0.95
# 找到 x_test 所属的子区间
idx = np.searchsorted(xp, x_test) - 1
idx = max(0, min(idx, len(xp)-2))
print(f"x={x_test:.2f}: 样条 vs 真值")
print(f"  共有 {n_nodes} 个节点, 三对角规模 {n_nodes}x{n_nodes}")
print(f"  追赶法 O(n) 完成求解")
🔍 追赶法的妙处： 三对角方程组的追赶法（Thomas algorithm）只需要 O(n) 次运算——而非一般高斯消去的 O(n³)。这是因为样条问题天然产生一个稀疏、对角占优的线性系统。这是数值分析中"利用问题结构来降低计算复杂度"的绝佳范例。
💡 实际使用： 在工程中，你通常不需要自己实现样条算法。SciPy提供了完善的 CubicSpline 接口： 
from scipy.interpolate import CubicSpline
cs = CubicSpline(x, y, bc_type='natural')
x_fine = np.linspace(-1, 1, 200)
y_fine = cs(x_fine)
底层使用的正是三弯矩法和追赶法。

三次样条是插值法的集大成者。它结合了分段插值的稳定性和全局高次多项式的光滑性——C² 连续保证曲率变化平滑，分段三次保证无Runge现象。在计算机图形学、CAD、字体设计（TrueType字体使用的就是二次B样条，PostScript使用三次Bézier曲线）、数值微分和积分中，样条无处不在。

下一章，我们将进入函数逼近（approximation theory）的世界。如果说插值追求的是"完美穿过每一个点"，那么逼近追求的是"在整体上尽可能接近"——当数据有噪声时，后者往往比前者更有用。
