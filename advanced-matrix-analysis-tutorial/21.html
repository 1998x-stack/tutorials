开场白：少于Nyquist的采样可能吗？

教科书告诉我们：要完美重建一个信号，采样频率必须至少是信号最高频率的两倍——这是 Nyquist 采样定理。但 2006 年左右，Candès、Donoho、Tao 等人发现了一个颠覆性的结论：如果信号本身是稀疏的，用远低于 Nyquist 率的采样就能完美重建。这就是压缩感知（Compressed Sensing, CS）的核心。

这个发现改变了整个信号处理领域的面貌，并在 MRI 成像、天文图像处理、无线通信、甚至计算摄影中产生了深远的影响。而它的数学基础，全在矩阵分析和优化之中。

本章我们将围绕一个核心问题展开：如何从欠定的线性系统 $y = Ax$ 中恢复出稀疏的 $x$？

21.1 稀疏性的数学刻画

一个向量 $x \in \mathbb{R}^n$ 称为 $k$-稀疏 的，如果它最多有 $k$ 个非零元素：

$$ \|x\|_0 := |\{i : x_i \neq 0\}| \leq k $$

其中 $\|x\|_0$ 是 $\ell_0$ 伪范数。那么欠定系统 $y = Ax$（$A \in \mathbb{R}^{m \times n}$，$m \ll n$）在有稀疏先验时应该怎样求解？最直接的问题是：

$$ \min_x \|x\|_0 \quad \text{s.t.} \quad Ax = y $$

但 $\ell_0$ 极小化是 NP 难的——它需要枚举所有可能的非零位置组合。这是组合优化的噩梦。

$\ell_1$ 凸松弛： 奇迹发生在将 $\ell_0$ 替换为 $\ell_1$ 范数时： $$ \min_x \|x\|_1 \quad \text{s.t.} \quad Ax = y $$ 在 $A$ 满足一定条件时（稍后讨论），这两个问题等价！而且 $\ell_1$ 问题是凸的，可以在多项式时间内求解。这是压缩感知的核心发现。

为什么 $\ell_1$ 能诱导稀疏性？几何解释：$\ell_1$ 球的形状是"有角"的菱形（在 $\mathbb{R}^2$ 中是正方形旋转 45 度），它与线性约束的交点往往落在坐标轴上，从而产生稀疏解。而 $\ell_2$ 球是圆形的，交点几乎一定不在坐标轴上。

21.2 受限等距性质（RIP）

什么时候 $\ell_0$ 和 $\ell_1$ 等价？这由测量矩阵 $A$ 的性质决定。受限等距性质（Restricted Isometry Property, RIP）是核心概念：

$$ (1 - \delta_k)\|x\|_2^2 \leq \|A x\|_2^2 \leq (1 + \delta_k)\|x\|_2^2 $$

对任意 $k$-稀疏向量 $x$ 成立。参数 $\delta_k \in (0,1)$ 称为 $A$ 的 $k$-阶受限等距常数。RIP 条件意味着 $A$ 在任何 $k$ 维子空间上都近似保持欧氏距离——等价于 $A$ 的任意 $k$ 列近似正交。

Candès 和 Tao 证明了：如果 $\delta_{2k} < \sqrt{2} - 1$，那么 $\ell_1$ 极小化的解与 $\ell_0$ 极小化的解完全相同。实践中，随机矩阵（如 Gaussian 随机矩阵、部分随机 Fourier 矩阵）以极高概率满足 RIP 条件。

21.3 稀疏编码算法

正交匹配追踪（OMP）

OMP 是一种贪婪算法，思想非常简单：

  1. 初始化残差 $r_0 = y$，支撑集 $S_0 = \emptyset$
  2. 在第 $t$ 步，找到与残差最相关的列：$i_t = \arg\max_i |\langle r_{t-1}, A_i \rangle|$
  3. 将 $i_t$ 加入支撑集 $S_t = S_{t-1} \cup \{i_t\}$
  4. 用最小二乘更新系数：$x_t = \arg\min_{x: \text{supp}(x) \subseteq S_t} \|Ax - y\|_2$
  5. 更新残差：$r_t = y - A x_t$
  6. 重复直到残差足够小或达到预设稀疏度 $k$

OMP 简单高效，但不保证全局最优。

LASSO

LASSO（Least Absolute Shrinkage and Selection Operator）是 $\ell_1$ 正则化的最小二乘：

$$ \min_x \frac{1}{2} \|Ax - y\|_2^2 + \lambda \|x\|_1 $$

正则化参数 $\lambda$ 控制稀疏度：$\lambda$ 越大，解越稀疏。LASSO 是凸优化问题，可以通过坐标下降法高效求解。

ISTA与FISTA

ISTA（Iterative Shrinkage-Thresholding Algorithm）是一种近端梯度法，每步迭代为：

$$ x^{k+1} = \mathcal{S}_{\lambda t}\!\left(x^k - t A^T (A x^k - y)\right) $$

其中 $\mathcal{S}_\alpha(z) = \operatorname{sign}(z) \max(|z| - \alpha, 0)$ 是软阈值算子（soft-thresholding）。FISTA 是 ISTA 的加速版本，引入 Nesterov 动量，收敛速度从 $O(1/k)$ 提升到 $O(1/k^2)$。

经验之谈： 在做大规模稀疏编码时（如图像去噪、压缩感知MRI），FISTA 通常是首选——它既有收敛速度保证，又实现简单。但如果问题规模不大且精度要求极高，用 OMP 可能更直接。

21.4 矩阵补全（Matrix Completion）

矩阵补全是稀疏表示的推广——从向量推广到矩阵。问题设定：已知矩阵 $M \in \mathbb{R}^{m \times n}$ 的部分观测值 $M_{ij}, (i,j) \in \Omega$，想恢复整个矩阵。这看起来不可能——未知数远多于方程数——除非矩阵是低秩的。

Netflix Prize 是矩阵补全最著名的应用场景：已知用户对部分电影的评分，预测未评分的电影。假设评分矩阵是低秩的（用户偏好可以被少数潜在因子解释），那么：

Candès-Recht 洞察（2009）： "只要矩阵是低秩的且其奇异向量与标准基是'非相干'的（incoherent），那么从几乎线性的少量观测中完美恢复是完全可能的。" 具体地，只需要 $O(r (m+n) \log^2(m+n))$ 个观测即可恢复秩为 $r$ 的 $m \times n$ 矩阵——这远比矩阵的总元素数 $mn$ 小。

矩阵补全的优化形式为：

$$ \min_X \|X\|_* \quad \text{s.t.} \quad X_{ij} = M_{ij},\; (i,j) \in \Omega $$

其中 $\|X\|_*$ 是核范数（nuclear norm），即奇异值之和。核范数是矩阵秩的凸松弛——就像 $\ell_1$ 范数是向量 $\ell_0$ 的凸松弛一样。

21.5 鲁棒PCA（Robust PCA）

传统 PCA 对异常值（outliers）非常敏感——一个离群点就能把主成分方向带偏。Candès 等人（2011）提出了鲁棒 PCA：将观测矩阵分解为低秩部分$L$和稀疏部分$S$：

$$ M = L + S, \quad \text{rank}(L) \leq r, \; \|S\|_0 \text{ small} $$

优化问题为：

$$ \min_{L, S} \|L\|_* + \lambda \|S\|_1 \quad \text{s.t.} \quad L + S = M $$

这就是所谓的主成分追踪（Principal Component Pursuit, PCP）。以下是它的 ADMM 求解实现：

import numpy as np
from scipy.linalg import svd

def robust_pca(X, lam=None, max_iter=1000, tol=1e-7):
    """鲁棒PCA的ADMM实现
    X: m×n 观测矩阵
    返回 L (低秩), S (稀疏)
    """
    m, n = X.shape
    if lam is None:
        lam = 1.0 / np.sqrt(max(m, n))

    # 初始化
    L = np.zeros((m, n))
    S = np.zeros((m, n))
    Y = np.zeros((m, n))  # 对偶变量
    mu = 1.25 / np.linalg.norm(X, 2)  # 步长参数
    rho = 1.5  # 步长增长因子

    for i in range(max_iter):
        # 更新 L: 奇异值软阈值
        U, s, Vt = svd(X - S + Y / mu, full_matrices=False)
        s = np.maximum(s - 1.0 / mu, 0)
        L_new = U @ np.diag(s) @ Vt

        # 更新 S: 元素软阈值
        diff = X - L_new + Y / mu
        S_new = np.sign(diff) * np.maximum(np.abs(diff) - lam / mu, 0)

        # 更新对偶变量 Y
        Y += mu * (X - L_new - S_new)

        # 收敛检查
        res = np.linalg.norm(X - L_new - S_new, 'fro')
        if res < tol * np.linalg.norm(X, 'fro'):
            break

        L, S = L_new, S_new
        mu *= rho  # 逐步增大步长

    return L, S
鲁棒PCA应用： 视频监控中的背景建模——将视频帧的矩阵分解为低秩背景（$L$）和稀疏前景（$S$，即移动物体）。在 2025 年，这个思想已被推广到深度异常检测中。

小结与展望

本章我们从 $\ell_0$ 范数的 NP 难出发，沿着 $\ell_1$ 凸松弛的思路，依次学习了 RIP 条件、OMP/LASSO/FISTA 等稀疏编码算法、矩阵补全的核范数方法，以及鲁棒 PCA 的 ADMM 实现。贯穿始终的主线是：稀疏性和低秩性是在欠定条件下恢复信号的两大支柱。

下一章，我们将进入一个截然不同的世界——随机矩阵理论。如果说稀疏表示关心的是矩阵的结构（哪些位置是零），随机矩阵理论关心的则是矩阵的统计（大矩阵的奇异值谱是什么分布）。这两个视角是互补的，共同构成了现代高维数据分析的数学基础。
