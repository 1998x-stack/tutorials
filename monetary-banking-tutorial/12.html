股票市场——企业"上市"到底是在干什么

上一章咱们讲了货币市场——短期资金融通的场所。但企业要盖工厂、搞研发、收并购，需要的可不是几个月的短期资金，而是长期甚至永久性的资本。这就轮到了资本市场登场。

资本市场分为两大类：股票市场（股权融资）和债券市场（债权融资）。股票是你"买企业的一部分"，没有到期日，赚了分红、亏了承担。债券是"借钱给企业"，有固定的利息和到期日。两者对企业和投资者来说，意义完全不同。

IPO与再融资

IPO（Initial Public Offering，首次公开募股）是一家公司第一次面向公众发行股票。IPO是企业发展的重要里程碑——不仅是融资手段，更是品牌背书、股权激励和后续融资的基础。

再融资则是已上市公司通过增发、配股等方式再次从资本市场融资。2025年，A股市场IPO和再融资合计规模超过1.5万亿元。

注册制改革——中国资本市场的"成人礼"

2023年2月，中国资本市场迎来了一个历史性时刻——全面注册制改革正式落地。这是中国资本市场三十多年来最深刻的制度变革。

注册制的核心是：把上市判断权从监管机构还给市场。过去是"核准制"——证监会要审你的盈利能力、持续经营能力等，决定"让不让你上"。现在是注册制——交易所审核信息披露的真假和充分性，能不能卖出去、以什么价格卖，由市场说了算。

注册制的影响是深远的：上市效率提高了（审核周期从1-2年缩短到3-6个月），壳价值消失了（垃圾公司不再值钱），退市常态化（2024年A股退市超过50家，创历史新高）。

中国多层次资本市场

板块 定位 上市标准（核心） 投资者门槛

这五个板块构成了中国多层次资本市场的"金字塔"——塔基是海量的中小企业（新三板），塔尖是成熟的大蓝筹（主板），中间是各具特色的细分市场。企业可以从塔基往塔尖"转板"——比如从北交所转到科创板。这套体系的目标是：让每一类企业都能找到适合自己的资本市场入口。

债券市场——比股票市场更庞大

你可能天天看股票行情，但你可能不知道：中国债券市场的规模（约160万亿元）远大于股票市场（约80万亿元）。全球范围内，债券市场也是最大的金融市场。

中国债券市场三分天下

  • 国债：中央政府发行，以国家信用担保。无风险收益率的主要参考。2025年末余额约32万亿元。
  • 地方债：省级政府发行，用于基础设施建设。余额已超国债，成为第一大债券品种（约42万亿元）。
  • 信用债：企业发行的债券，包括企业债、公司债、中期票据、短期融资券等。蕴含信用风险，收益率高于国债。

债券通

2017年开通的"债券通"（Bond Connect）让境外投资者可以通过香港买卖内地债券。这是中国资本市场对外开放的重要一步。截至2025年末，境外机构持有中国债券规模约4万亿元，主要持有国债和政策性银行债。

城投债——地方政府的"隐性债务"问题

城投债是由地方政府融资平台公司发行的债券，名义上是企业债（信用债），但市场一直认为有"地方政府隐性担保"——这就是所谓的"城投信仰"。然而，2023-2025年，随着地方政府债务化解（化债）工作的推进，城投债的"刚兑信仰"正在被打破。一些弱资质城投开始出现技术性违约或展期，城投债的定价正在从"信仰定价"向"信用定价"转变。

🔑 重要概念：收益率曲线 债券市场最重要的分析工具是收益率曲线（Yield Curve）——将同一信用等级、不同期限债券的收益率画在坐标图上形成的曲线。正常形态是"向上倾斜"（期限越长、收益越高，补偿流动性风险）。当出现"倒挂"（短期收益率高于长期）时，通常预示着经济衰退。2024年中美都出现了收益率曲线倒挂——这在历史上往往是经济放缓的先行指标。

咱们来算算债券收益率（YTM，到期收益率）——这是债券投资最核心的计算：

// 债券到期收益率(YTM)的近似计算
// YTM使债券未来所有现金流贴现值等于当前价格

function calcYtm(price, faceValue, couponRate, yearsToMaturity, freq = 2) {
  // price: 当前净价（元，面值100）
  // faceValue: 面值（通常100元）
  // couponRate: 票面利率（如0.025表示2.5%）
  // yearsToMaturity: 剩余期限（年）
  // freq: 每年付息次数（通常2次）

  const annualCoupon = faceValue * couponRate;
  const couponPerPeriod = annualCoupon / freq;
  const totalPeriods = yearsToMaturity * freq;
  const periodRate = 0.03 / freq; // 初始猜测3%年化

  // 简化近似：使用公式近似计算YTM
  // 精确YTM需要迭代求解，这里用近似公式
  const avgAnnualReturn = (annualCoupon + (faceValue - price) / yearsToMaturity);
  const avgInvestment = (price + faceValue) / 2;
  const ytm = avgAnnualReturn / avgInvestment;

  return ytm * 100; // 返回百分比
}

const bond = {
  price: 98.5,           // 买入价98.5元（折价）
  faceValue: 100,        // 面值100元
  couponRate: 0.028,     // 票面利率2.8%
  maturity: 5            // 5年期
};

const ytm = calcYtm(bond.price, bond.faceValue,
                    bond.couponRate, bond.maturity);
const currentYield = (bond.faceValue * bond.couponRate) / bond.price * 100;

console.log(`债券价格: ${bond.price}元`);
console.log(`票面利率: ${bond.couponRate * 100}%`);
console.log(`当期收益率: ${currentYield.toFixed(2)}%`);
console.log(`到期收益率(YTM): ${ytm.toFixed(2)}%`);

// 折价买入时，YTM > 当期收益率 > 票面利率
// 溢价买入时则相反
console.log(`判断: ${bond.price < bond.faceValue ?
  '折价买入 ✓ 到期收益高于票息' :
  '溢价买入 ✗ 到期收益低于票息'}`);

资产证券化——把"未来的钱"变成今天的现金

资产证券化（ABS, Asset-Backed Securities）是个听起来很高大上的概念，但底层逻辑其实很简单：你有一堆贷款（房贷、车贷、信用卡应收款），这些贷款每个月都会产生现金流（借款人还款）。把这些贷款打包成一个"资产池"，然后以这个资产池的未来现金流为支撑发行债券——这就是资产证券化。

资产证券化的参与者包括：发起人（银行等原始债权人）、特殊目的载体（SPV，接收资产的法律实体）、信用增级机构（提供担保，提高债券评级）、投资者（购买ABS的各类机构）。

常见的ABS类型包括：MBS（住房抵押贷款支持证券，美国08年金融危机的核心产品）、ABS（广义资产支持证券，包括车贷ABS、信用卡ABS等）、CLO（贷款抵押债券，以企业贷款为底层资产）。

⚠️ 踩坑故事：2020年永煤违约——AAA国企债违约引爆信用债市场

2020年11月10日，永煤控股（河南一家大型国企）的一笔10亿元超短期融资券到期违约。这个消息让整个债券市场炸了锅——因为永煤是AAA评级的省级国企，违约发生前不到一个月还在市场上成功发行了新债。

为什么市场反应如此剧烈？因为中国债券市场一直存在一个潜规则——"国企不会违约，尤其是AAA国企"。永煤违约彻底打破了"国企刚兑信仰"。违约后发生了什么？信用债市场整个冻结了——企业发不出新债（因为没人买了），基金被大量赎回（投资者恐慌），二级市场信用债收益率飙升。

更让人气愤的是，永煤违约被发现存在"逃废债"嫌疑——违约前不久，永煤将大量优质资产（包括持有的中原银行股权等）无偿划转给了母公司，导致上市公司变成了一个空壳。这种"金蝉脱壳"的行为引发了监管的严厉整治。交易商协会对永煤及相关中介机构进行了严重警告和业务暂停处罚。

永煤违约的深远影响是：信用债市场的"国企信仰"从此瓦解。2020年之后，信用债的定价越来越基于企业的基本面（盈利能力、现金流、负债率）而不是"出身"（国企or民企）。信用分化成为市场最显著的特征——好企业发债利率越来越低，差企业发不出债，或者要付很高的溢价。从长远看，这是市场走向成熟的必经之路。

2010年代，中国资产证券化市场快速发展。截至2025年末，中国ABS市场规模超过5万亿元。但资产证券化也蕴含风险——2008年全球金融危机就是因为美国的次级MBS（次级房贷抵押债券）大规模违约引爆的。当时华尔街把一堆垃圾房贷打包成AAA评级的MBS卖给全球投资者，房地产泡沫破裂后，整个金融体系跟多米诺骨牌一样倒塌。这个教训告诉我们：资产证券化本身不是坏事，坏的是底层资产的质量被掩盖、风险被错误定价。

股票市场和债券市场构成了资本市场的两大支柱。但咱们现在还差一个重要市场——汇率和外币资产交易的市场。下一章：外汇市场与国际金融。在这个全球化的时代，汇率变动和跨国资本流动对各国经济的影响越来越大。
