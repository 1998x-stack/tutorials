第十一章：母函数（二）

大家好，我是你们的讲师。在上一章中，我们学习了普通母函数（OGF）——它将数列对应为 xⁿ 的系数，适用于组合型问题（顺序无关）。但如果你遇到的是"排列"问题（顺序有关），普通母函数就不太方便了。举个例子：从n个不同元素中选k个排列，如果要用母函数表达，系数应该是 P(n,k) 而不是 C(n,k)。而 P(n,k) = n!/(n-k)!，用普通幂级数很难简洁表达。

这时候就需要 指数型母函数（Exponential Generating Function，EGF）登场了。它的核心思想是：把 n! 吸收到分母中，使得排列数在级数中表现为简洁的形式。

这一章，我们就来学习指数型母函数的定义、经典应用，以及如何与普通母函数配合使用。

11.1 指数型母函数的定义与使用场景

数列 {aₙ} 的指数型母函数定义为：

EGF(x) = Σ aₙ · xⁿ/n!
       = a₀ + a₁x/1! + a₂x²/2! + a₃x³/3! + ...

对比普通母函数 OGF(x) = Σ aₙxⁿ，指数型母函数多了一个除以 n!。这个看似微小的变化带来了巨大的差异。

什么时候用普通母函数，什么时候用指数型母函数？有一个简单原则：

普通母函数 OGF指数型母函数 EGF
适用场景组合（顺序无关）排列（顺序有关）
典型计数整数拆分、硬币找零排列数、错排、集合划分
乘积含义组合的卷积排列的卷积（带二项式系数）
公式Σ aₙxⁿΣ aₙxⁿ/n!
标记对象相同物品的分组不同物品的排列

指数型母函数的乘积是什么含义？设 EGF(x) = Σ aₙxⁿ/n! 和 F(x) = Σ bₙxⁿ/n!，它们的乘积为：

EGF(x)·F(x) = Σ cₙxⁿ/n!, 其中 cₙ = Σ C(n,k) aₖ bₙ₋ₖ

多了一个二项式系数 C(n,k)！这是因为在排列问题中，选择 k 个元素作为第一组、n-k 个作为第二组时，需要先选出哪些元素进入第一组（C(n,k) 种方式），再分别排列。

关键区别：

  • 普通母函数的乘法对应"无标签的组合"：cₙ = Σ aₖbₙ₋ₖ
  • 指数型母函数的乘法对应"有标签的排列"：cₙ = Σ C(n,k) aₖbₙ₋ₖ

11.2 错排数的母函数推导

错排（Derangement）是指一个排列中没有任何元素在它原来的位置上。n 个元素的错排数记作 Dₙ 或 !n。

我们知道 Dₙ 的通项公式为 Dₙ = n! · Σₖ₌₀ⁿ (-1)ᵏ/k!。这个公式通常用容斥原理推导。现在来欣赏指数型母函数的推导——它更加优雅。

错排的指数型母函数：

设 D(x) = Σ Dₙ · xⁿ/n!

考虑"全排列"的指数型函数：全排列数 n! 对应 EGF 为：
Σ n! · xⁿ/n! = Σ xⁿ = 1/(1-x)

另一方面，一个排列由"不动点"（即位置不变的元素）和"错排部分"组成。
对于有 k 个不动点的排列，相当于从 n 个元素中选 k 个作为不动点（C(n,k) 种），
剩下的 n-k 个元素构成一个错排（Dₙ₋ₖ 种）。

所以全排列数满足：n! = Σ C(n,k) Dₙ₋ₖ

这正是指数型母函数乘积的形式！所以：
1/(1-x) = eˣ · D(x)    （因为不动点的 EGF 是 eˣ = Σ 1·xⁿ/n!）

因此 D(x) = e⁻ˣ / (1-x)

展开：D(x) = (Σ (-1)ⁿxⁿ/n!) · (Σ xⁿ)
       = Σ (Σₖ₌₀ⁿ (-1)ᵏ/k!) xⁿ
       = Σ n! · (Σₖ₌₀ⁿ (-1)ᵏ/k!) xⁿ/n!

所以 Dₙ = n! · Σₖ₌₀ⁿ (-1)ᵏ/k!

完美！整个推导只需要三行——从"不动点"的组合结构出发，利用指数型母函数的乘积性质，直接得到结果。而用容斥原理的推导至少需要七八步。

指数型母函数的好处：它自动处理了元素之间的"标签"区分。在普通母函数中，"选择 k 个不动点"这一步的 C(n,k) 需要手动处理；在指数型母函数中，乘积的定义自动包含了这个二项式系数。这就是为什么对排列问题要用指数型母函数。

11.3 Catalan 数的生成函数

Catalan 数是组合数学中出现频率最高的数列之一，它的生成函数推导是必须掌握的经典。Catalan 数 Cₙ 满足：

C₀ = 1
Cₙ = Σₖ₌₀ⁿ⁻¹ Cₖ Cₙ₋₁₋ₖ  (n ≥ 1)

这个卷积结构强烈暗示了 Catalan 数的母函数满足一个二次方程。

设 Catalan 数的普通母函数为 C(x) = Σ Cₙxⁿ。那么：

C(x) = Σ Cₙxⁿ = C₀ + Σ_{n≥1} Cₙxⁿ
     = 1 + Σ_{n≥1} (Σₖ₌₀ⁿ⁻¹ Cₖ Cₙ₋₁₋ₖ) xⁿ
     = 1 + x Σ_{n≥1} (Σₖ₌₀ⁿ⁻¹ Cₖ Cₙ₋₁₋ₖ) xⁿ⁻¹
     = 1 + x · C(x)²     (因为 C(x)² 展开后 xⁿ⁻¹ 的系数正是 Σ CₖCₙ₋₁₋ₖ)

所以 C(x) = 1 + x · C(x)²
即 x·C(x)² - C(x) + 1 = 0

这是一个关于 C(x) 的二次方程，用求根公式：

C(x) = (1 ± √(1 - 4x)) / (2x)

由于 C(0) = C₀ = 1，当 x → 0 时：
若取加号：(1 + √1) / 0 → 无穷大，排除
所以 C(x) = (1 - √(1 - 4x)) / (2x)

这就是 Catalan 数生成函数的封闭形式。可以用广义二项式定理展开得到通项公式：

√(1-4x) = 1 - 2Σ C(2n-2, n-1)/n · xⁿ

代入并化简得：
Cₙ = C(2n, n) / (n+1) = (2n)! / (n!(n+1)!)

这就是 Catalan 数的经典公式。

方法论的启示：Catalan 数递推关系的卷积形式直接提示了它的母函数满足二次方程——这是递归结构 → 代数方程 → 封闭形式 → 通项公式的标准路径。这条路径在组合数学中反复出现：

  1. 识别递推关系中的卷积结构
  2. 转化为母函数的代数方程
  3. 解出封闭形式
  4. 用展开得到通项

11.4 Bell 数与集合划分

Bell 数 Bₙ 表示将 n 个不同元素划分成任意多个非空子集（无序组）的方式数。比如 B₀ = 1（空划分），B₁ = 1（{1}），B₂ = 2（{1,2} 和 {1}{2}），B₃ = 5。

Bell 数也可以用指数型母函数得到优美的表达式。

考虑"将 n 个元素分成若干非空子集"与"将 n 个元素分成 k 个非空子集"的关系。后者正是 Stirling 数 S(n,k)（第二类 Stirling 数）。Bell 数是 Bₙ = Σ S(n,k)。

指数型母函数的方法：将一个集合的划分看作是由"块"组成的。每个非空子集（块）的指数型母函数是 eˣ - 1（因为非空意味着去掉空集项）。而整个划分是由任意多个块组成的，所以：

Bell 数的 EGF：Bell(x) = Σ Bₙ · xⁿ/n! = e^(eˣ - 1)

这个结果可以从指数型母函数的组合意义得到：
一个划分 = 将集合分成任意多个"非空块"
"非空块"的 EGF = eˣ - 1
任意多个块的组合 = exp(eˣ - 1)

展开 e^(eˣ - 1) 的前几项：

e^(eˣ - 1) = e⁻¹ · e^(eˣ)
            = e⁻¹ · Σ e^(nx)/n!
            = e⁻¹ · Σ (1/n!) Σ (nx)ᵏ/k!
            = ...

更直接地，我们可以逐个展开：
eˣ - 1 = x + x²/2! + x³/3! + x⁴/4! + ...
e^(eˣ - 1) = 1 + (x + x²/2! + ...) + (x + x²/2! + ...)²/2! + ...

x 的系数：1        → B₁ = 1
x²/2! 的系数：1 + 1/2! × 2! = 1+1 = 2  → B₂ = 2
x³/3! 的系数：1 + 3 + 1 = 5  → B₃ = 5

Bell 数的指数型母函数 e^(eˣ - 1) 简洁得出奇。同样地，第二类 Stirling 数的指数型母函数是：

Σ S(n,k) · xⁿ/n! = (eˣ - 1)ᵏ/k!

⚠️ 一个容易混淆的点：很多同学会问——为什么普通母函数中 1/(1-x) 对应着"任意多个组合"，而指数型母函数中 eˣ 对应着"任意多个排列"？

答案是：因为组合的"任意多个"是 Σ (物品组)ⁿ = 1/(1-物品组)，而排列的"任意多个"是 Σ (物品组)ⁿ/n! = e^(物品组)。前者是几何级数，后者是指数级数——这种对应关系是由两种母函数的乘法定义决定的。

简单记忆法：OGF 中"重复"是几何级数 1/(1-x)；EGF 中"重复"是指数级数 eˣ。

11.5 普通母函数与指数型母函数的选择原则

现在我们来做一个系统性的总结。面对一个计数问题，如何选择母函数类型？

判断依据用 OG用 EGF
物品是否"相同"物品不可区分（如硬币、数字）物品可区分（如人、元素）
结构是否"无序"子集、组合、整数拆分排列、序列、有序对
典型关键词"选几个""分成几组""排列""顺序""队伍"
乘法对应Σₖ aₖ bₙ₋ₖΣₖ C(n,k) aₖ bₙ₋ₖ

更具体地说，在做题时问自己两个问题：

  1. 元素之间是不是不一样的？ 如果元素是"有标签"的（比如不同的人、不同的字母），优先考虑 EGF。
  2. 顺序是否重要？ 如果"先选再排"中的排列顺序重要，用 EGF；如果只是一堆东西放在一起，用 OGF。

例1（选型练习）：

  • "将 10 个相同的球放到 3 个不同的盒子中" → 球相同，盒子有标签但只关心组合 → OGF，1/(1-x)³ 的 x¹⁰ 系数
  • "将 5 个不同的人分成 3 个非空小组" → 人不同，小组无标签 → EGF，需要用 S(5,3)
  • "用 1 元、2 元、5 元硬币凑 100 元" → 硬币面额虽不同但同面额硬币相同 → OGF
  • "n 个人排成一排，其中某些人必须不相邻" → 人不同且顺序重要 → EGF

11.6 综合例题：母函数在竞赛中的实战

例2：求将 n 拆分成互不相同的正整数的拆分方式数 q(n) 的母函数。

解：每个正整数 k 最多使用一次，所以因子为 1 + xᵏ。母函数为：

Q(x) = (1+x)(1+x²)(1+x³)(1+x⁴)··· = Π (1+xᵏ)

著名的 Euler 恒等式 说：

Π (1+xᵏ) = Π 1/(1-x²ᵏ⁻¹)

即"拆分成互不相同的正整数的方式数 = 拆分成奇正整数的方式数"。这可以用组合直觉证明：每个偶数 2m 可以拆成 m+m，但要求互不相同的限制禁止了这种重复。而奇正整数拆分没有这个限制。Euler 的母函数推导证明了这两个看似不同的拆分问题实际上是一样的。

这就是母函数最迷人的地方——你能"看出"两个不同组合问题之间的等价性，仅仅因为它们的母函数相同。

实战建议：在竞赛中，母函数主要用于：

  1. 求递推通项（如 Fibonacci、Catalan）
  2. 证明组合恒等式（通过比较系数）
  3. 处理带限制的计数（通过修改因子）
  4. 证明两个组合问题等价（通过化简到同一个母函数）

前两种最为常见，后两种出现在较难的题目中。

例3：证明恒等式 Σ C(n,k)² = C(2n, n)。

解：考虑 (1+x)²ⁿ = (1+x)ⁿ(1+x)ⁿ。左边展开 C(2n,n) 是 xⁿ 的系数。右边 (1+x)ⁿ 的展开是 Σ C(n,k)xᵏ，两个相乘后 xⁿ 的系数正是 Σ C(n,k)C(n, n-k) = Σ C(n,k)²。得证。

这个证明虽然用的是二项式定理而非母函数的核心框架，但它展示了母函数思想的精髓——通过一个代数对象的两种展开方式来证明组合等式。这就是"算两次"的思路在母函数中的体现，也是我们下一章的主题。

11.7 本章小结与过渡

本章我们学习了指数型母函数的定义、与普通母函数的区别、以及它们在错排数、Catalan 数、Bell 数和 Stirling 数中的优雅应用。指数型母函数的精髓在于用分母 n! 吸收排列的计数因子，使得乘积自动处理元素标签。

母函数的两章内容合在一起，构建了一个强大的代数化组合工具。但母函数不是万能的——某些类型的组合问题更适合用"计数"的思路而非"代数"的思路。

下一章我们将学习一种更直接的计数技巧——算两次与不变量。算两次的核心思想是：用两种不同的方法计算同一个量，然后让它们相等——从而推导出组合恒等式或解题关键。这个方法的奇妙之处在于，它不需要任何高深的数学工具，只需要你有一双敏锐的眼睛，看出"同一个东西可以从两个角度看"。
