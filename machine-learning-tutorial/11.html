11.1 画一条线把两类分开——但画哪条最好？

假设平面上有两类点，红点和蓝点。你可以画无数条直线把它们分开。哪条最好？SVM的回答是：到两边最近点的距离（间隔）最大的那条。这个直觉非常几何化——不只要分对，还要让分界线离每个类都尽可能远，这样新的点进来时不容易被错分。

我一直在想一个问题：为什么SVM在深度学习兴起之后依然没有被淘汰？答案是：在小样本、高维度的场景下，SVM的表现经常优于神经网络。比如基因表达数据（几百个样本、几万个特征），SVM配合合适的核函数依然是SOTA。

11.2 最大间隔分类器

SVM的目标是最大化margin（间隔），即分类边界到最近样本点的距离。这些"最近的点"决定了边界的位置，它们就是支持向量（Support Vectors）——SVM名字的由来。

关键性质：只有支持向量影响决策边界。你把非支持向量的训练样本移动一点或删掉，边界完全不变。这使得SVM天然对远离边界的样本具有鲁棒性。

11.3 软间隔：允许犯一点错

现实中数据往往不是线性可分的——总有几个点混在对方阵营里。硬间隔SVM会为了完美分开它们而画出一个极其扭曲的边界，过拟合。软间隔引入参数C，允许一些点"越过"边界：

  • C很大：几乎不允许犯错 → 边界很"硬" → 过拟合风险
  • C很小：允许较多犯错 → 边界更"软" → 更平滑，泛化更好
💡 C怎么记：C是"惩罚力度"。C大 = 对犯错惩罚大 = 不允许犯错 = 硬边界。C小 = 对犯错宽容 = 允许犯错 = 软边界。记住：C和正则化强度λ是反比关系（C ≈ 1/λ）。

11.4 核技巧：把"不可能"变成"可能"

这是SVM最优雅的部分。如果数据在原空间无法线性分开怎么办？把它们映射到高维空间！

比如二维平面上，红点围成一圈，蓝点在圈里。没有直线能分开它们。但如果加一个维度 z = x² + y²，红点在高处（大的z），蓝点在低处（小的z）——在高维空间一个平面就能分开了。

核技巧的精髓：你不需要真的算高维映射的结果，只需要知道两个样本在高维空间的内积。核函数K(x, x')就是这个内积。

核函数公式（直觉）适用场景
线性核x·x'高维稀疏数据（文本分类）
RBF核exp(-γ||x-x'||²)不知道用什么的时候的默认选项
多项式核(γx·x' + r)^d已知特征交互是多项式关系
Sigmoid核tanh(γx·x' + r)历史上使用，现在很少用

11.5 γ参数：RBF核的关键

RBF核中有两个核心参数——C和γ（gamma）：

  • γ大：每个支持向量的影响范围小 → 边界非常弯曲 → 容易过拟合
  • γ小：每个支持向量的影响范围大 → 边界更平滑 → 可能欠拟合
from sklearn.svm import SVC
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

# SVM对特征尺度极其敏感，必须标准化！
svm_pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('svm', SVC(kernel='rbf', C=1.0, gamma='scale', probability=True))
])
svm_pipe.fit(X_train, y_train)

# 调参：C和gamma通常在对数尺度上搜索
from sklearn.model_selection import GridSearchCV
param_grid = {
    'svm__C': [0.1, 1, 10, 100],
    'svm__gamma': [0.001, 0.01, 0.1, 1, 'scale', 'auto'],
}
grid = GridSearchCV(svm_pipe, param_grid, cv=5, scoring='f1')
grid.fit(X_train, y_train)
print(f"最佳参数: {grid.best_params_}")
⚠️ 两个SVM最常见的坑：(1) 没做标准化就训练——特征尺度差异会导致核函数计算被大尺度特征主导，小尺度特征形同虚设。(2) 默认参数不调参——C=1.0和gamma='scale'只是默认值，不一定是适合你数据的最优值。SVM对参数相当敏感，一定要做网格搜索。

11.6 SVM的优缺点总结

优点缺点
小样本高维数据表现优秀大数据集训练慢（O(n²)~O(n³)）
核技巧非线性灵活对参数敏感，需要仔细调参
全局最优（凸优化）难以解释（黑盒，不像树和线性模型）
泛化理论扎实不直接输出概率（需要额外校准）
🔑 什么场景选SVM？样本量 < 10万 + 特征维度适中 + 需要非线性决策边界。典型应用：手写数字识别（SVM + RBF核在MNIST上表现极好）、文本分类（线性SVM在大规模稀疏文本上速度极快）、生物信息学（基因表达数据分类）。如果是百万级样本的表格数据，请用XGBoost；如果是图像/音频，请用深度学习。
