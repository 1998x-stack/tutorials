从定义到法则：求导的效率革命

大家好，我是你们的讲师。上一章里，我们用定义（差商极限）手工求了几个简单函数的导数。做那种计算就像用刀劈柴——每次都要全力以赴。但假如你需要求 f(x) = sin(e^x) · ln(1 + x^2) / (x^3 + 1) 的导数，用定义去求？那将是一场噩梦。

幸运的是，微积分的先驱者们已经替我们铺好了路。他们总结出了一套求导法则（differentiation rules），把求导变成了一个机械化的流程——就像把"徒手劈柴"升级成了"用电锯锯木"。你不再需要每次回头去算极限，而是可以像搭积木一样，把已知的导数公式通过运算法则组合起来。

我一直记得我本科时第一次学会链式法则后的感慨：原来复杂的求导可以像剥洋葱一样，一层一层地拆解。之前用定义求 sin x 的导数花了整整四行计算，而有了这些法则之后——看到就能写出答案。这节课我们要做的就是把这套"电锯"交到你手上。

基本初等函数的导数公式表

以下是所有基本初等函数的导数公式，这是你需要烂熟于心的内容——就像一个程序员必须知道变量声明怎么写一样，这是微积分世界的"语法"。

函数 导数 备注
f(x) = C（常数） f'(x) = 0 常数不看，一导为零
f(x) = x^n（n 为实数） f'(x) = n·x^(n-1) 幂函数求导：指数提前，指数减一
f(x) = e^x f'(x) = e^x 自然指数的导数是它自己
f(x) = a^x（a > 0, a ≠ 1） f'(x) = a^x · ln a 一般指数：多乘 lna
f(x) = ln x f'(x) = 1/x 自然对数的导数最简洁
f(x) = log_a x f'(x) = 1/(x ln a) 一般对数，换底即可推
f(x) = sin x f'(x) = cos x 正弦变余弦
f(x) = cos x f'(x) = -sin x 余弦变负正弦
f(x) = tan x f'(x) = sec^2 x 正切的导数
f(x) = arcsin x f'(x) = 1/√(1-x^2) 注意定义域 (-1,1)
f(x) = arctan x f'(x) = 1/(1+x^2) 形式优美，常出现在积分中
记忆口诀
幂函数：指数提前当好汉，指数位置减个一。
指数函数：自然指数最神奇，求导还是它自己；一般指数别大意，乘上 ln 记心里。
对数函数：自然对数 x 分之一，一般对数值换底。
三角函数：正弦变余弦，余弦变负弦，正切割方站。

四则求导法则

当你需要求导的函数是两个或多个基本函数的组合时，下面的法则就派上用场了。

和差法则

[f(x) ± g(x)]' = f'(x) ± g'(x)

和差的导数等于导数的和差——这是最简单的法则，直接拆开求导即可。

乘法法则（乘积法则）

[f(x) · g(x)]' = f'(x)·g(x) + f(x)·g'(x)

这是最容易被记错的法则之一！很多初学者的第一反应是 (fg)' = f'g'，但这是错误的。想想看：如果乘法的导数等于导数的乘积，那 1 秒钟后我们还要学链式法则干什么？

乘法法则的"速度"理解
设想一个矩形的面积 A = 长 × 宽。如果长和宽都在变化，面积的变化率 = 长的变化率 × 当前宽 + 当前长 × 宽的变化率。这正是乘法法则的物理意义！(fg)' 不是两个变化率的乘积，而是一个"各自变化、分别贡献"的过程。

除法法则（商法则）

[f(x) / g(x)]' = [f'(x)·g(x) - f(x)·g'(x)] / [g(x)]^2,  其中 g(x) ≠ 0

记忆口诀："上导下不导减去上不导下导，分母平方放下面"。

例子：综合运用

求 y = x^2 · e^x 的导数：

y' = (x^2)' · e^x + x^2 · (e^x)'
   = 2x · e^x + x^2 · e^x
   = e^x (x^2 + 2x)
求 y = (3x^2 + 1) / (x + 2) 的导数：

y' = [(6x)(x+2) - (3x^2+1)(1)] / (x+2)^2
   = [6x^2 + 12x - 3x^2 - 1] / (x+2)^2
   = [3x^2 + 12x - 1] / (x+2)^2

链式法则：复合函数求导的核心

链式法则（chain rule）是微积分里最重要的求导法则，没有之一。它告诉我们如何求复合函数的导数：

如果 y = f(g(x))，则
dy/dx = f'(g(x)) · g'(x)

或者用莱布尼茨记号：
dy/dx = dy/du · du/dx,  其中 u = g(x)

"洋葱"理解法

我特别喜欢用一个比喻来教链式法则：复合函数就像一颗洋葱。你一层一层地剥开它，每次只剥一层，从外到内。

例如 y = sin(x^2 + 3x)：

  • 最外层是 sin(□)，导数 = cos(□)
  • 里面一层是 □ = x^2 + 3x，导数 = 2x + 3
  • 所以 y' = cos(x^2 + 3x) · (2x + 3)

如果你需要剥三层：y = e^(sin(x^2))

  • 最外层：e^(□)，导数 = e^(□)
  • 中间层：sin(○)，导数 = cos(○)
  • 最内层：○ = x^2，导数 = 2x
  • 所以 y' = e^(sin(x^2)) · cos(x^2) · 2x
链式法则的一般形式（多层复合）：
如果 y = f1(f2(f3(...fn(x)...)))，则
dy/dx = f1' · f2' · f3' · ... · fn'

每一层的导数相乘——这就是"链"的含义。
链式法则最常见的错误：遗漏内层导数
这是一个几乎每个学微积分的人都会踩的坑。比如求 y = sin(2x) 的导数，错误答案是 cos(2x)——漏掉了内层 2x 的导数 2。正确答案应该是 2cos(2x)。

我的一个同学在考试时连续三道题都犯了这个错误——他每次都是"只看外层，忘了内层"。考试后他说了一句话我一直记得："我知道链式法则是要乘的，但做题做着做着就忘了内层。"这个问题的根源在于：练习量不够，肌肉记忆没有形成。链式法则必须练到"看到复合函数，自动拆层"的程度。

链式法则的实战例子

例1：y = e^(3x)
y' = e^(3x) · 3 = 3e^(3x)

例2：y = ln(1 + x^2)
y' = 1/(1 + x^2) · 2x = 2x/(1 + x^2)

例3：y = (x^2 + 1)^10
y' = 10(x^2 + 1)^9 · 2x = 20x(x^2 + 1)^9

反函数的求导法则

如果 y = f(x) 在区间内可导且 f'(x) ≠ 0，那么它的反函数 x = f^(-1)(y) 也可导，且：

[f^(-1)]'(y) = 1 / f'(x),  其中 y = f(x)

或者更直观地：
如果 y = f(x) 且 x = f^(-1)(y)，则
dx/dy = 1 / (dy/dx)

这个法则的几何意义非常直观：原函数的切线斜率和反函数的切线斜率互为倒数。

例如，如何求 arctan x 的导数？设 y = arctan x，则 x = tan y，dx/dy = sec^2 y，那么：

dy/dx = 1 / (dx/dy) = 1 / sec^2 y = 1 / (1 + tan^2 y) = 1 / (1 + x^2)

看，这正是导数公式表里的结果！

综合求导练习

让我们综合运用所有法则，求一个略复杂的函数的导数：

求 y = e^x · sin(x^2 + 1) / (x^3 + ln x) 的导数

第一步：这是三个函数的商——实际上是 (e^x · sin(x^2+1)) / (x^3 + ln x)
分别记 u = e^x · sin(x^2+1), v = x^3 + ln x

第二步：u 本身又是两个函数的乘积
u' = (e^x)' · sin(x^2+1) + e^x · [sin(x^2+1)]'
   = e^x · sin(x^2+1) + e^x · cos(x^2+1) · 2x
   = e^x [sin(x^2+1) + 2x cos(x^2+1)]

第三步：v' = 3x^2 + 1/x

第四步：用除法法则
y' = (u'v - uv') / v^2
   = {e^x[sin(x^2+1) + 2x cos(x^2+1)] · (x^3 + ln x)
      - e^x · sin(x^2+1) · (3x^2 + 1/x)} / (x^3 + ln x)^2

做完这种题目的时候，你的感受可能是"好麻烦"，但请对比一下：如果用导数的定义（差商极限）来求这个函数——那几乎是不可能完成的任务。而你刚才只用了几步标准的法则就得到了结果。这就是求导法则的力量：把复杂的求导问题拆解成一系列简单步骤的组合。

求导练习建议
初学者容易犯的一个错误是"一做题就想看答案"。建议你用这样的方法练习：拿到一个求导题后，先遮住答案，自己在纸上写出三个要点——1）这个函数由哪些基本初等函数组成？2）外层是什么？内层是什么？（如果是复合函数）3）应该用什么法则组合？

把这三个问题想清楚了再动笔，你会发现正确率大幅提升。

本章小结

本章我们建立了一套完整的求导体系：基本初等函数的导数公式表是"砖块"，四则求导法则是"水泥"，链式法则是"脚手架"。有了这三件法宝，你可以求任何初等函数的导数。记得那个"剥洋葱"的比喻吗？从外到内，一层一层，每层乘上这层的导数——只要做到这一点，链式法则就永远不会出错。

下一章我们将进入更实际的场景：隐函数和参数方程的求导。当函数关系不是显式写成的 y = f(x) 形式时，我们该如何求导？比如圆的方程 x^2 + y^2 = 1，这里 y 并没有被显式解出来——这就是隐函数求导的用武之地。
