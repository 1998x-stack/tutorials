大家好，我是你们的讲师。上一章咱们学了DCF模型——把未来的现金流折现到今天，算出公司的内在价值。逻辑上，DCF是最"正确"的估值方法。但现实是：在华尔街和A股的机构投资者中，使用频率最高的估值方法并不是DCF，而是相对估值法。

为什么？因为DCF需要太多的假设——增长率、利润率、资本支出、WACC、永续增长率——任何一个假设偏差5%，最终结果可能差30%。而相对估值法就简单直接多了：找到一家可比公司，看看市场给了它多少倍的PE（市盈率），然后乘以目标公司的净利润——得出估值。虽然没有DCF那么"精确"，但它在实操中更接地气、更高效。

但问题来了：各个估值指标都有它的"适用场景"。用错了指标，估值就会全盘皆输。今天咱们就系统地把相对估值法的几大指标讲透。

PE市盈率的正确用法

PE = 股价 / 每股收益（EPS）。它回答的是"以当前的盈利水平，多少年能回本"这个问题。如果PE=20，意味着保持当前利润不变，需要20年才能通过利润赚回你的投资。

但PE有几种不同的"口味"，你一定要分清楚：

  • 静态PE——用最新年报的EPS计算。数据最确定，但反映的是去年的盈利水平，可能已经过时了。
  • 动态PE——用预测的当年EPS计算。前瞻性最强，但预测可能不准。
  • TTM PE（Trailing Twelve Months）——用最近四个季度的EPS计算。在A股中最常用，既反映了最新情况，又不需要依赖预测。你可以这样理解：TTM是"当下"的盈利水平。

哪些公司不适合用PE？

  • 周期股——钢铁、煤炭、有色金属这类强周期行业。在行业顶峰时，利润最高、PE最低（看起来最"便宜"），但恰恰是最高点；在行业低谷时，利润最低甚至亏损、PE最高或为负（看起来最"贵"），但恰恰是最低点。用PE给周期股估值，你会"买在顶峰、卖在谷底"——完全做反。
  • 亏损公司——净利润为负，PE为负值或无穷大，失去意义。比如很多早期创新药企业，连年亏损但管线价值巨大，PE完全无法反映其真实价值。
  • 金融股的特殊处理——银行股的PE普遍在4-8倍之间，看起来"很便宜"，但这是因为高杠杆经营放大了利润基数（后面第21章会详细讲）。
实战口诀：PE用在稳定盈利的消费股、医药股上最有效；用在周期股和亏损股上会误导决策。对于后两类，需要用PB、PS或EV/EBITDA。

PB市净率的适用场景

PB = 股价 / 每股净资产。它回答的是"如果公司现在就破产清算，每股能分到多少钱"。PB<1叫做"破净"——意味着市场认为公司的资产质量很差，或者资产价值被高估了。

PB最适用的行业：

  • 银行股——银行的资产主要是贷款和金融资产，账面价值相对接近真实价值。银行股当前普遍在0.5-0.8倍PB，长期破净。但注意：银行的不良贷款可能侵蚀账面净资产，所以实际PB可能比看起来的要高。
  • 保险公司——保险公司用内含价值（Embedded Value, EV）替代账面净资产进行估值，但底层逻辑和PB一致。
  • 重资产行业——钢铁、化工、公用事业等。这些公司的价值主要来自厂房、设备等实物资产，用PB比用PE更可靠。

破净股分析：A股中有大量上市公司长期处于破净状态（PB<1）。低PB不等于"便宜货"——你需要判断：

  • 资产是不是"真资产"（有没有水分？应收账款能收回来吗？存货能卖掉吗？）
  • 公司能不能持续盈利？如果持续亏损，净资产会逐年减少，现在看起来低的PB会越来越高
  • 行业是否处于下行周期？银行股破净往往反映了市场对不良贷款的担忧

PS市销率和EV/EBITDA

除了PE和PB，还有两个常见的估值倍数，在某些场景下比PE更管用。

PS（市销率）= 市值 / 营业收入

PS最大的优点：营业收入很难被操纵（相比于净利润）。而且对于亏损公司来说，当PE失效时，PS仍然有效。

典型的PS应用场景：

  • SaaS公司——早期可能大幅亏损，但只要收入在快速增长，市场愿意给高PS倍数。例如一家SaaS公司年收入5亿，PS=10意味着市场给它50亿估值，相当于"市场为每1元收入支付10元"。
  • 电商公司——京东常年亏损但收入巨大，PS比PE更有意义。
  • 零售企业——连锁超市、百货公司的收入体量巨大，利润率很低但稳定，PS估值有参考价值。

EV/EBITDA（企业价值倍数）

这是机构投资者最爱的估值指标之一。EV（企业价值）= 市值 + 负债总额 - 现金及等价物。EBITDA = 息税前利润 + 折旧摊销。

EV/EBITDA为什么好？因为它：

  • 消除了资本结构（负债/股权比例）的影响——两个公司即使负债率不同，也可以直接用EV/EBITDA比较
  • 消除了折旧摊销政策的影响——不同公司折旧政策不同会扭曲净利润，但EBITDA不受折旧影响
  • 包含了负债——PE只考虑了股权价值，EV/EBITDA把负债也纳入了估值框架
机构内部的"潜规则"：很多买方研究员私下说，"PE是给散户看的，EV/EBITDA是给自己用的"。因为EV/EBITDA排除了资本结构的影响，更真实地反映了公司的"运营价值"。但EV/EBITDA也有不适用的时候——对于固定资产极少的轻资产公司（如互联网平台），折旧很少，EBITDA和净利润差别不大，EV/EBITDA的优势就不明显了。

分位数分析与历史估值

任何估值倍数的一个数字本身没有意义——PE=15到底贵不贵？你得知道它相对于谁。两个最重要的参照系是：历史的自己和行业的同行。

历史分位数：把一家公司过去5年或10年的PE/PB数据从低到高排序，看看当前估值处于什么位置。

  • 如果当前PE处于历史90%分位数——意味着过去90%的时间里估值都比现在便宜，那现在大概率是偏贵的
  • 如果当前PE处于历史10%分位数——意味着过去90%的时间里估值都比现在贵，那现在可能是低估了

但注意：历史分位数不是万能的。基本面变差了，历史低分位数可能不是"低估"，而是"价值陷阱"——看起来便宜，但实际上公司出了问题，合理估值就应该更低。

行业分位数对比：在同一行业中，把所有公司的PE/PB排序，看看目标公司在行业中处于什么位置。例如，白酒行业平均PE=25，如果某白酒龙头PE=18且在行业中处于25%分位数，说明它相对同行可能被低估了。

踩坑故事——净利润暴增1015%，股价却跌了26%

2023年，A股有一只股票引发了市场的广泛关注：正丹股份。这一年，它的净利润从上一年的约0.2亿元暴增至约2.2亿元，同比增幅高达1015%！看到这个数字，很多散户投资者开始兴奋了——"净利润翻了10倍，股价至少也该翻倍吧？"

于是不少人冲进去买入。结果呢？公司的股价不但没有涨，反而从高点一路下跌，跌幅超过26%。很多投资者想不通：业绩这么好，为什么股价不涨反跌？

原因在于：市场早就预期到了这个业绩。在正丹股份发布业绩预告之前，股价已经从底部涨了将近4倍。等到利好真正落地时（"买预期，卖事实"的经典演绎），机构投资者已经开始获利了结。更关键的是，分析公司历史估值分位数就会发现：在业绩兑现之前，公司的PE已经从之前的40多倍跌到了个位数——看起来PE很低是不是？但真正的逻辑是：在股价高位买入时，你支付的是"业绩已经充分price in"的价格。后面即使业绩达标，也没有新的预期差来推动股价继续上涨。

这就是"PE陷阱"的典型表现——静态PE看起来便宜，但市场已经提前把利润增长"贴现"到股价中了。

教训：不要只看PE的绝对值，更要看估值分位数和市场预期。低PE不一定意味着低估——可能是因为市场认为这个高利润不可持续。高PE也不一定意味着高估——可能是因为市场预期未来利润会大幅增长。永远要问：价格里已经"包含"了多高的预期？

下面这个表格总结了各个估值指标的适用行业和注意事项：

估值指标 适用行业 不适用行业 注意事项
PE（市盈率） 消费、医药、科技（盈利稳定） 周期股、亏损股、金融股 分清静态/动态/TTM；结合增长看PEG
PB（市净率） 银行、保险、钢铁、化工 科技(轻资产)、互联网 注意资产质量；<1破净不等于便宜
PS（市销率） SaaS、电商、零售 高利润率的奢侈品、品牌消费 要结合毛利率看；收入不等于利润
EV/EBITDA 资本密集型行业、高杠杆公司 轻资产公司（折旧少） 机构最常用；排除资本结构影响
PEG 高成长公司 负增长或稳定增长公司 PEG<1可能低估；增长率的持续性很关键

最后，咱们来看一个简单的估值分位数计算逻辑，你可以用Python或SQL来实现：

# 估值分位数计算逻辑（Python伪代码）
import numpy as np

def calculate_percentile(current_value, historical_values):
    """
    计算当前估值在历史数据中的分位数位置

    参数:
        current_value: 当前的PE/PB/PS值
        historical_values: 过去N年的历史估值数据列表
    返回:
        percentile: 当前值在历史数据中的百分位 (0-100)
    """
    sorted_values = sorted(historical_values)
    count_below = sum(1 for v in sorted_values if v < current_value)
    percentile = (count_below / len(sorted_values)) * 100
    return percentile

# 示例：计算某白酒股PE的历史分位数
pe_history_5y = [
    22.1, 23.5, 24.0, 25.3, 26.8,  # 2019年 每月末数据
    28.2, 30.1, 31.5, 29.8, 27.3,
    25.6, 24.8,  # ... 等等共60个月数据
]

current_pe = 26.5
result = calculate_percentile(current_pe, pe_history_5y)
print(f"当前PE={current_pe}，处于历史{result:.1f}%分位数")
# 输出: 当前PE=26.5，处于历史XX%分位数

# 行业分位数对比
def industry_percentile(company_pe, industry_pe_list):
    """计算目标公司在同行业中的估值分位数"""
    sorted_industry = sorted(industry_pe_list)
    # 排除负PE（亏损公司）
    sorted_industry = [p for p in sorted_industry if p > 0]
    position = sum(1 for p in sorted_industry if p < company_pe)
    return (position / len(sorted_industry)) * 100

# 使用示例：
# 白酒行业PE列表
baijiu_pe_list = [18.5, 22.3, 25.1, 28.7, 31.2, 35.8, 40.6]
target_pe = 25.1
result = industry_percentile(target_pe, baijiu_pe_list)
print(f"目标公司在行业中处于{result:.1f}%分位数")

好了，本章咱们深入学习了相对估值法的四大核心指标——PE、PB、PS、EV/EBITDA，以及历史分位数和行业分位数的分析逻辑。每一种指标都有自己的"主场"和"禁区"，用对指标是估值分析的第一步。
