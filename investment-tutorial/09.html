第9章 资本资产定价模型

想象这样一个场景：你和朋友讨论两只股票——茅台和某家刚上市的科技公司。茅台的预期年化收益大概在12%左右，科技公司可能高达20%。你的朋友说："科技公司收益更高，说明它更有价值。"你本能地觉得这个推理有问题，但说不清问题出在哪里。

资本资产定价模型（Capital Asset Pricing Model, CAPM）就是用来回答这个问题的。它告诉我们：一只股票的预期收益，不取决于它自身的波动有多大，而取决于它对市场整体波动的敏感程度。换句话说，你承担的风险中，只有"无法分散掉的那部分"才能获得补偿。这是金融学中最重要的模型之一，也是威廉·夏普（William Sharpe）获得诺贝尔奖的原因。这一章我们就来吃透它。

开篇格言：CAPM用一个简单到令人难以置信的公式，描述了风险与收益之间的基本关系：E(R_i) = R_f + β_i × [E(R_m) - R_f]。简洁，优美，但充满争议。

9.1 从组合理论到CAPM

模型推导：从微观到宏观

上一章我们学习了马科维茨的组合理论——每个投资者都构建自己的有效前沿，选择最优组合。现在，我们把这个逻辑推到极致：如果所有投资者都使用同样的信息、同样的分析方法，会怎样？

CAPM在这个思想实验之上增加了几个关键假设：

  • 所有投资者都可以按无风险利率自由借贷
  • 所有投资者对资产收益的预期完全一致（同质预期，Homogeneous Expectations）
  • 没有交易成本、没有税收、没有信息不对称
  • 所有资产都可以无限细分，所有投资者都是价格接受者

在这些假设下，所有投资者会计算出同样的最优风险组合——这个组合就是市场组合（Market Portfolio）。市场组合包含了所有风险资产，每种资产的权重等于其市值占全部市值的比例。对应的CAL就是资本市场线（Capital Market Line, CML）：

# 资本市场线（CML）
E(R_p) = R_f + [E(R_m) - R_f] / σ_m × σ_p

# 其中：
# E(R_m) = 市场组合的预期收益
# σ_m = 市场组合的标准差

# CML只能用于描述有效组合的收益-风险关系
# 对于单个资产，我们需要证券市场线（SML）

β系数的诞生

单个资产的风险由两部分构成：系统性风险（Systematic Risk）和非系统性风险（Idiosyncratic Risk）。非系统性风险可以通过分散化消除，因此市场不会为它提供溢价补偿。CAPM的核心洞见就是：资产的预期收益只与系统性风险相关。

# CAPM的核心公式：证券市场线（SML）
E(R_i) = R_f + β_i × (E(R_m) - R_f)

# β_i = Cov(R_i, R_m) / Var(R_m)
# β衡量了资产i对市场整体波动的敏感度

# 例如：假设R_f = 1.8%, E(R_m) = 8%, β = 1.2
# E(R_i) = 1.8% + 1.2 × (8% - 1.8%) = 1.8% + 7.44% = 9.24%

# 如果β = 0.8
# E(R_i) = 1.8% + 0.8 × (8% - 1.8%) = 1.8% + 4.96% = 6.76%

9.2 β系数：风险的第一把标尺

β的估计与稳定性

在实际工作中，β通常用历史收益率数据做线性回归来估计。但β的稳定性是一个大问题：

估计窗口中信证券β贵州茅台β长江电力β
60个月（5年）1.350.850.35
36个月（3年）1.280.920.30
12个月（1年）1.520.760.42

可以看到，同一只股票的β会随着选取的时间窗口不同而显著变化。这就是为什么行业实践中常采用调整β（Adjusted Beta）——将原始β向1.0"收缩"，以提高预测能力。Bloomberg和Barra等数据供应商都提供调整后的β。

行业实务：中国A股的β系数有显著的行业特征。科技股、券商股的β通常在1.2-1.8之间，消费白马股在0.7-1.0之间，公用事业股（如水电、高速公路）在0.3-0.6之间。β的行业差异本质上反映了不同行业盈利对宏观经济的敏感度。

9.3 证券市场线与均衡定价

SML是一条穿过市场组合和β坐标的直线。当市场处于均衡状态时，所有资产都应该落在SML上。如果一个点在SML上方，说明它的预期收益高于理论值——被低估（买入机会）；如果在SML下方，则被高估（卖出或回避）。

SML的斜率就是市场风险溢价（Market Risk Premium），即E(R_m) - R_f。这个参数的大小在全球范围内存在明显差异：

  • 美国市场：历史市场风险溢价约5-7%（1926-2023年数据）
  • 中国市场：历史市场风险溢价约6-9%（但波动极大）
  • 新兴市场：通常更高（8-12%），反映了更高的政治风险和流动性风险

9.4 CAPM实证检验：争议与挑战

CAPM在理论上无懈可击，但在实证中却困难重重。我们来梳理主要的批评和发现：

Roll批评（Roll's Critique）

1977年，Richard Roll发表了一篇著名论文，指出CAPM实际上不可检验。因为真正的"市场组合"应该包含所有资产——股票、债券、房地产、人力资本、艺术品等等，但这样的组合在现实中根本无法观测。任何实证检验都只能使用市场组合的代理变量（如沪深300指数、标普500指数），而使用不同代理变量会得到截然不同的结论。

β的预测能力之争

Fama和French在1992年发表了一篇里程碑式的论文，发现1963-1990年间，美国股市的β对横截面收益差异几乎没有任何解释力。反而是市值和账面市值比等变量更具预测能力。这个发现直接催生了Fama-French三因子模型（下一章讲述）。

但也有后续研究（如Baker, Bradley和Wurgler, 2011）发现，如果区分β高估组和低估组，低β股票实际上实现了比CAPM预测更高的收益——这就是著名的低β异象（Low Beta Anomaly）。

低β异象的实践意义：2018年的一项研究显示，如果从A股市场选取β最低的100只股票构建等权组合，2005-2023年间的年化收益达到12.3%，夏普比率0.68，显著优于高β组合（年化8.1%，夏普比率0.42）。这说明承担更低风险可能获得更高收益——听起来匪夷所思，但数据确实如此。这背后的原因包括：机构投资者被迫追逐高β股票（因业绩比较基准）、高β股票被过度定价等。

横截面收益异象

除了β之外，学者们发现了大量CAPM无法解释的异象（Anomalies）：

  • 规模效应：小盘股长期跑赢大盘股
  • 价值效应：低估值股票跑赢高估值股票
  • 动量效应：过去收益高的股票未来继续高收益
  • 低波动异象：低波动股票跑赢高波动股票
  • 应计异象：低应记利润的公司跑赢高应记利润的

这些异象的存在意味着要么CAPM的假设是错误的，要么市场不是有效的——这正是我们在第11章和第12章要深入探讨的内容。

9.5 2025-2026年CAPM的最新发展

CAPM虽然诞生于1960年代，但学术界从未停止对其改进和扩展的步伐。近年来最值得关注的两个突破：

模糊厌恶扩展（Ambiguity Aversion CAPM）

传统CAPM假设投资者知道收益率的确切概率分布（风险），但现实中我们连概率分布本身都不确定（模糊性，Ambiguity）。2025年，一篇发表在《Journal of Financial Economics》上的论文提出了考虑模糊厌恶的CAPM扩展：

# 模糊厌恶CAPM（AA-CAPM）
E(R_i) = R_f + β_i × (E(R_m) - R_f) + λ × FA_i

# 其中：
# FA_i = 资产的模糊暴露（Ambiguity Exposure）
# λ = 模糊的市场价格（Ambiguity Premium）

# 研究表明在高不确定性时期（如新冠、地缘冲突），
# 模糊溢价可以占到总溢价的30-40%

PAPM模型与Markowitz奖

2025年，基于前景理论（Prospect Theory）的PAPM（Prospect Theory Asset Pricing Model）获得了著名的Markowitz奖。与传统CAPM不同，PAPM将投资者的参考点依赖（Reference Dependence）和损失厌恶（Loss Aversion）纳入定价框架。它成功解释了多个CAPM无法解释的异象，包括低β异象和"价值股溢价"。PAPM的出现标志着行为金融学与传统资产定价理论的融合进入了一个新阶段。

实务启示：虽然CAPM在实证中表现不佳，但它仍然是最广泛使用的权益资本成本估算工具。根据Duff & Phelps 2025年调查报告，超过80%的CFO仍然使用CAPM估算公司资本成本。原因是：它足够简单、容易沟通、被监管机构承认（如公用事业定价）。只是在使用时，建议结合行业β的经验调整和宏观经济情景分析来综合判断。
模型核心因子优势局限
经典CAPMβ（市场因子）简单、直觉忽略其他定价因子
条件CAPM时变β、时变风险溢价适应市场状态变化实现复杂
AA-CAPMβ + 模糊暴露解释不确定性溢价模糊难以量化
PAPM参考点 + 损失厌恶解释异象、行为基础参数多、校准难
实用警告：如果你在券商的研报中看到"我们使用CAPM模型计算目标价"，请保持警惕。很多研究员给出的目标价本质上是一个"故事"，CAPM只是用来给这个数字披上"科学的外衣"。记住：模型的输出质量不可能超过输入质量。当他们随便拍了一个β值、随便选了一个市场风险溢价时，最终的"理论目标价"和拍脑袋没有本质区别。
本章要点回顾：
  • CAPM将组合理论推广到市场均衡层面，指出资产的预期收益只与系统性风险（β）相关
  • β衡量了资产对市场整体波动的敏感度，但估计结果对时间窗口选择敏感
  • SML是资产定价的基准线，偏离SML意味着可能的错误定价
  • Roll批评、β预测能力弱、大量异象的存在说明CAPM并非完美
  • 2025-2026年的最新发展包括模糊厌恶CAPM和获得Markowitz奖的PAPM模型

CAPM简洁而优雅，但实证证据告诉我们——市场比模型复杂得多。为了解释那些CAPM无法解释的收益来源，我们需要引入更多的因子。这就是下一章的主题：套利定价理论与多因子模型。从Fama-French的三因子到"因子动物园"，我们将探索定价模型的扩展前沿。
