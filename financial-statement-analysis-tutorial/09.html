大家好，我是你们的讲师。上一章咱们聊了盈利能力——公司能赚多少钱。但一家公司能赚钱不等于它不会破产。事实上，很多看起来很赚钱的公司恰恰是因为债务问题倒下的。为什么？因为利润是"慢慢亏"的，但债务是"马上还"的。

举个极端例子：你开了一家餐厅，月利润5万，看起来经营得不错。但如果你欠了供应商100万货款，下个月到期，你的现金只有30万——那你就得破产。哪怕你的餐厅未来还能赚很多钱，但债主不会等你。这就是偿债能力分析的意义——判断公司在面临债务到期时，有没有足够的"弹药"来应对。

偿债能力分析分为短期偿债能力和长期偿债能力两个层面。咱们还要特别关注一个问题：隐性债务——那些不在资产负债表上直接体现，但可能随时引爆的债务炸弹。

短期偿债能力——流动比率、速动比率、现金比率

短期偿债能力关注的是公司在一年以内能否偿还到期的债务。核心指标有三个：

流动比率 = 流动资产 / 流动负债，衡量的是公司用流动资产偿还流动负债的能力。一般认为流动比率在1.5-2.0之间比较健康。如果低于1，说明公司的流动资产连流动负债都覆盖不了——这是一个危险信号。但也不是越高越好，流动比率过高可能说明资金利用效率低。

速动比率 = (流动资产 - 存货) / 流动负债，比流动比率更保守。因为存货变现需要时间，而且可能存在跌价风险。速动比率剔除了存货，只考虑货币资金、应收账款、交易性金融资产等"速动资产"。一般认为速动比率在1.0左右比较合理。

现金比率 = (货币资金 + 交易性金融资产) / 流动负债，这是最保守的指标，只考虑最直接的支付能力。现金比率低于0.2通常意味着公司面临较大的短期偿债压力。

实战技巧：分析短期偿债能力时，不仅要看这些比率的绝对水平，还要关注债务的期限结构。比如公司的流动负债中有多少是短期借款、多少是应付账款。短期借款需要刚性偿还，应付账款还可以和供应商协商展期——风险程度完全不同。

长期偿债能力——资产负债率、债务-EBITDA倍数

长期偿债能力关注的是公司长期（一年以上）能否按时还本付息。核心指标包括：

资产负债率 = 总负债 / 总资产，这是最常用的杠杆指标。一般认为40%-60%比较合理。低于40%说明公司经营保守，但可能没有充分利用财务杠杆；高于70%则需要警惕，尤其是对于周期性行业。

不过，不同行业的资产负债率差异很大。房地产行业的资产负债率普遍在70%-85%，因为这是行业特性（高杠杆、重资产模式）；而轻资产的高科技公司资产负债率通常在20%-40%。所以判断资产负债率是否合理，必须结合行业特点。

净债务 / EBITDA是机构投资者最看重的长期偿债能力指标之一。EBITDA（息税折旧摊销前利润）近似代表了公司的经营性现金流能力。净债务 / EBITDA倍数反映了公司用经营收益偿还债务需要多少年。一般认为：

  • 低于1.5倍：非常健康
  • 1.5 - 3.0倍：正常范围
  • 3.0 - 4.5倍：需要关注
  • 高于4.5倍：高风险
注意：净债务/EBITDA不适用于所有行业。对于高成长、尚未盈利的科技公司，EBITDA可能为负，这个指标就失去了意义。此外，EBITDA是会计利润而非真正的现金流，还需要结合经营性现金流来验证。

利息保障倍数——利润覆盖利息的能力

利息保障倍数 = 息税前利润（EBIT） / 利息费用，衡量的是公司赚到的利润能够覆盖利息支出的多少倍。如果这个倍数低于1，说明公司连利息都赚不回来——基本上是在"借新还旧"，这是非常危险的信号。

一般认为利息保障倍数在3倍以上比较安全。但同样需要结合行业和趋势来看：

偿债能力指标 计算公式 安全阈值 警戒线 危险线
流动比率 流动资产/流动负债 1.5 - 2.0 1.0 - 1.5 < 1.0
速动比率 (流动资产-存货)/流动负债 0.8 - 1.2 0.5 - 0.8 < 0.5
资产负债率 总负债/总资产 40% - 60% 60% - 70% > 70%
利息保障倍数 EBIT/利息费用 > 5 3 - 5 < 3
净债务/EBITDA (总负债-货币资金)/EBITDA < 1.5 1.5 - 3.0 > 3.0 
偿债能力指标计算公式速查：

1. 流动比率 = 流动资产 / 流动负债

2. 速动比率 = (流动资产 - 存货) / 流动负债

3. 现金比率 = (货币资金 + 交易性金融资产) / 流动负债

4. 资产负债率 = 总负债 / 总资产 × 100%

5. 有形资产负债率 = 总负债 / (总资产 - 无形资产 - 商誉) × 100%
   （剔除无形资产和商誉，更保守地衡量真实负债水平）

6. 利息保障倍数 = EBIT / 利息费用

7. 净债务 / EBITDA = (总负债 - 货币资金 - 交易性金融资产) / EBITDA

8. EBITDA = 净利润 + 所得税 + 利息费用 + 折旧 + 摊销

隐性债务识别——表外负债和经营租赁

最危险的不是看得见的债务，而是看不见的债务。很多公司的资产负债表上负债率看起来很低，但实际上通过表外手段承担了大量隐性债务。以下是几个常见的隐性债务雷区：

  • 对外担保：公司为关联方或第三方提供的担保。一旦被担保方违约，担保责任就会变成公司的实际负债。A股历史上很多暴雷案例都涉及巨额对外担保。
  • 未决诉讼：年报附注中的"或有事项"部分会披露未决诉讼。如果公司涉及重大诉讼且败诉概率很高，潜在的赔偿金额可能远远超出公司的承受能力。
  • 经营租赁（新准则后已部分入表）：2018年新租赁准则实施后，经营租赁需要确认"使用权资产"和"租赁负债"，但很多公司仍在利用准则过渡期规则。
  • 永续债和优先股：有些公司发行的永续债虽然在会计上被归类为"权益工具"，但实际上具有强制付息义务和赎回条款——本质上就是债务。
  • 票据融资和供应链金融：通过商业承兑汇票、保理等方式进行的融资，不一定全部反映在短期借款中。
分析技巧：看偿债能力，至少要看"三步"——第一步看表内指标（流动比率、资产负债率等），第二步看附注中的或有事项（对外担保、未决诉讼），第三步查外部信息（法院执行信息、信用评级变动、债券价格异常）。很多隐性问题在表内是完全看不出来的。

真实案例：表内低负债、表外大担保的暴雷故事

2019年，A股一家上市公司H公司引起了我的注意。从表面数据看，H公司的资产负债率只有52%，净债务/EBITDA也只有2.1倍，各项偿债指标都在安全范围内。很多券商分析师给它的评级是"低风险、稳健经营"。

但我仔细翻阅年报附注后发现了一个问题：H公司为其子公司和关联方提供了总额高达38亿的对外担保，而它自己的净资产才60亿。也就是说，担保金额占净资产的63%以上。我算了一笔账：如果这些被担保方中有20%违约，H公司就要承担7.6亿的担保损失——相当于其当年净利润的1.5倍。

更令人担忧的是，这些被担保方大部分是H公司的上下游企业，业务高度关联。一旦行业下行，很可能出现"多米诺骨牌"效应——一家违约引发连锁反应。

果然，2020年H公司的一主要客户出现经营困难，无法偿还银行贷款，银行随即向担保方H公司追偿。H公司被迫承担了4.3亿的代偿损失，当年净利润从盈利3亿转为亏损1.3亿，股价半年内跌去了70%。

关键教训：永远不要只看表内负债率。对外担保、未决诉讼、永续债——这些"隐性债务"才是真正的杀手。买入一只股票前，至少花10分钟读一遍年报的"或有事项"和"关联交易"附注部分。
