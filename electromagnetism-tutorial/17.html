开场：当导线不再只是导线

在低频电路（如50Hz电力线或1kHz音频线）中，导线就是导线——它仅仅是连接元器件的导电通路。你不需要考虑信号在导线上的"延迟"，因为电压和电流在整个电路中的变化几乎是瞬时的（相比信号周期）。这就是集总参数（Lumped Element）模型的适用范围。

但是当频率升高到微波频段（比如5GHz的5G信号），情况完全不同了。信号波长只有6厘米，一段10厘米长的PCB走线就比1.5个波长还长！此时，导线上不同点的电压和电流相位大不相同，你不能再把它们当作"同一节点"。导线不再是无辜的连线——它变成了一个分布参数系统（Distributed Parameter System）。

传输线理论正是为了分析和设计这样的系统而生。从同轴电缆到微带线，从波导到共面波导，理解传输线是射频和微波工程的第一道门槛。本章我们将从传输线方程开始，一路走到波导理论和实际PCB设计。

集总参数 vs 分布参数

判断一个电路是否需要用分布参数模型，有一个简单的工程标准：当电路尺寸 l 大于信号波长的十分之一（l > λ/10）时，分布参数效应不可忽略。

为什么是 λ/10？因为当传输线长度达到波长量级时，信号在传输线上传播的过程中相位变化可达数十度甚至上百度——这已经足以显著影响系统的阻抗、反射和匹配特性。

在分布参数模型中，传输线被视为无数个微分段（dz）的级联，每个微分段包含：

  • 串联电阻 R（Ω/m）：导体的有限电导率带来的欧姆损耗。
  • 串联电感 L（H/m）：电流流过导体产生的环绕磁场。
  • 并联电导 G（S/m）：介质中的漏电流产生的损耗。
  • 并联电容 C（F/m）：两导体之间的电场存储。

从基尔霍夫定律作用于微分段，可以得到著名的电报方程（Telegraph Equations）：

−∂v(z,t)/∂z = R i(z,t) + L ∂i(z,t)/∂t

−∂i(z,t)/∂z = G v(z,t) + C ∂v(z,t)/∂t

在时谐稳态（正弦信号）下，电报方程简化为：

−dV/dz = (R + jωL) I

−dI/dz = (G + jωC) V

特性阻抗

将两个一阶方程合并，得到电压和电流分别满足的波动方程：

d²V/dz² = γ² V, d²I/dz² = γ² I

其中 γ = √((R + jωL)(G + jωC)) = α + jβ 称为传播常数。

波动方程的解为 V(z) = V₀⁺e^{−γz} + V₀⁻e^{γz}，对应的电流为 I(z) = (V₀⁺e^{−γz} − V₀⁻e^{γz}) / Z₀。这里 Z₀——特性阻抗——是最最关键的参数：

Z₀ = √((R + jωL) / (G + jωC))

对于无损传输线（R=0, G=0）：

Z₀ = √(L / C)

这意味着 Z₀ 完全由传输线的几何结构和介质材料决定，与传输线的长度无关！常见的特性阻抗值：

  • 同轴电缆：50Ω（射频通用标准）或75Ω（电视/广播）。为什么是50Ω？历史原因是兼顾最小损耗（≈77Ω）和最大功率容量（≈30Ω）之间的折中。
  • 双绞线：100–150Ω，用于以太网和差分信号。
  • 平行双线：300Ω，用于FM天线和电视天线。
  • 微带线（PCB）：50Ω或75Ω，最常用的PCB互连阻抗标准。

传输线终端效应

传输线的核心问题之一是终端不匹配导致的反射。当传输线端接的负载阻抗 Z_L 不等于特性阻抗 Z₀ 时，一部分能量被反射回源端。

在负载端（z=0），电压和电流满足 V(0)/I(0) = Z_L。由此可推导出反射系数：

Γ = V₀⁻/V₀⁺ = (Z_L − Z₀) / (Z_L + Z₀)

几种特殊情形：

  • 匹配：Z_L = Z₀ → Γ = 0，无反射，所有功率传输到负载。
  • 开路：Z_L = ∞ → Γ = +1，全反射，电压反射波相位与入射波相同（在开路端电压加倍）。
  • 短路：Z_L = 0 → Γ = −1，全反射，电压反射波相位翻转（在短路端电压为零、电流加倍）。
  • 纯电抗：Z_L = jX → |Γ| = 1，全反射但相位连续可调。

反射使传输线上形成驻波（Standing Wave）——入射波和反射波叠加产生固定的波腹和波节分布。驻波比 VSWR 由反射系数幅度决定：

VSWR = (1 + |Γ|) / (1 − |Γ|)

理想的 VSWR = 1（完全匹配），实际工程中 VSWR < 1.5 通常认为是良好匹配。

阻抗匹配是解决不匹配问题的关键技术。常用方法包括：

  • L型网络：两个电抗元件（一个串联、一个并联）将任意阻抗变换到 Z₀。
  • λ/4阻抗变换器：插入一段特性阻抗为 Z₀' = √(Z₀·Z_L) 的四分之一波长传输线。
  • 单/双枝节匹配（Stub Tuning）：在传输线上并联特定长度的开路或短路枝节。
⚠ 阻抗失配的真实案例——5G基站覆盖失效
在某5G基站的实际部署中，调试团队发现整个扇区的覆盖范围突然从300米锐减到不足100米，且用户频繁掉线。现场仪表测得的VSWR高达3:1。排查后发现：基站到天线的馈线接头中，有一个N型连接器因为安装扭矩不足而松动，引入了一个小间隙。
这个小间隙在高功率下打火（电弧放电），等效于在传输线中串联了一个不稳定的电阻和电容。功放检测到反射功率过大（|Γ|² ≈ 25%，即25%的功率被反射回来），触发保护机制不断降低发射功率。
教训：在高频系统中，每一个连接器、每一段微带线拐角、每一个过孔都是潜在的阻抗不连续点。一个1mm的间隙在3.5GHz下就是λ/85，足以引起显著反射。射频工程师花在"连接"上的时间往往比花在"设计"上的还多。

Smith圆图

Smith圆图（Smith Chart）是射频工程中最经典的可视化工具，由 Phillip Smith 于1939年发明。它将复反射系数 Γ 平面和阻抗/导纳平面巧妙地映射在一起，使得阻抗匹配网络的设计变得直观而优雅。

Smith 圆图的主要特征：

  • 所有无源负载的 Γ 都落在单位圆内（|Γ| ≤ 1）。
  • 等电阻 r 圆是一族圆心在实轴上的圆，圆心从 (r/(1+r), 0) 到 (0,0) 连续变化。
  • 等电抗 x 圆是一族圆心在虚轴上的圆弧，圆心在 (1, 1/x)。
  • 圆图上方的半圆对应感性电抗（+jx），下方对应容性电抗（−jx）。
  • 沿着传输线移动对应沿等 |Γ| 圆旋转，每走 λ/4 旋转180°。

使用 Smith 圆图进行匹配设计的基本步骤：

  1. 在圆图上标出负载阻抗 Z_L 的归一化值 z_L = Z_L/Z₀。
  2. 沿等 |Γ| 圆向源端（顺时针）旋转到与 r=1 的电阻圆相交。
  3. 交点到负载的旋转角度决定了串联传输线的长度。
  4. 在交点处，阻抗变为 1+jx，用并联枝节（或串联电容/电感）抵消电抗部分。

虽然现代射频设计软件（如 ADS、AWR、HFSS）已经内置了自动匹配和 Smith 圆图功能，但理解 Smith 圆图的几何原理仍然至关重要——它让你能够直观地判断阻抗变换趋势、识别匹配瓶颈、估算带宽。

🔑 Smith圆图背后优雅的复变函数理论
Smith 圆图本质上是一个共形映射（Conformal Mapping）：将复 Γ 平面的单位圆内部映射到归一化阻抗 z 的右半平面。映射函数就是 z = (1+Γ)/(1−Γ) 这个简单的双线性变换。
这个映射的优雅之处在于：
1. 等电阻圆和等电抗圆在 Γ 平面中仍然是圆（Möbius变换保圆性）
2. 变换是保角的——圆之间的夹角在映射前后保持不变
3. 圆图上的"旋转"操作对应传输线方程中 e^{−2jβl} 因子——物理意义清晰
可以说，Smith 圆图不是工程师随便画出来的辅助工具，而是传输线理论推导出来的必然几何结果。

波导

当频率高到毫米波或太赫兹频段时，即使是同轴电缆也因损耗过大和尺寸过小而不实用。此时需要用波导（Waveguide）来传输电磁能量。波导本质上是中空的金属管（或填充介质的金属管），电磁波在其中以特定的模式（Mode）传播。

矩形波导的主要模式

矩形波导（宽边 a，窄边 b）支持两大类模式：

  • TE模式（Transverse Electric）：电场沿传播方向分量为零。
  • TM模式（Transverse Magnetic）：磁场沿传播方向分量为零。

模式用下标 mn 标记（如 TE₁₀、TE₂₀、TM₁₁），m 和 n 表示场在 x 和 y 方向的半周期数。TE₁₀ 是矩形波导的主模（最低截止频率的模式）。

关键参数——截止频率（Cut-off Frequency）：

f_c(mn) = (c/(2π√μ_rε_r)) √((mπ/a)² + (nπ/b)²)

对于 TE₁₀ 模：f_c = c/(2a√μ_rε_r)。例如宽边 a=22.86mm 的 WR-90 波导（X频段），f_c ≈ 6.56GHz。工作频率通常选在 f_c 到 2f_c 之间（单模传输范围）。

💡 为什么波导是"空心"的？——不需要内导体！
矩形波导的内部是空心的（或填充介质），没有中心导体。很多人第一次看到会很困惑：没有导体，电流回流路径在哪里？
答案在于波导的工作模式：波导壁上的感应电流（表面电流）承担了"回流"的功能。对于 TE₁₀ 模，电流在宽壁上沿着纵向流动，在窄壁上来回跨越。这些电流是由波导内的电磁场在金属表面上感应产生的。
空心波导的优点：
• 没有介质损耗（空气ε_r≈1）
• 没有中心导体的欧姆损耗
• 功率容量极大（可以传输数兆瓦的峰值功率）
因此，雷达发射机、卫星通信上行链路、粒子加速器等高功率射频系统无一例外地使用波导。

微带线与实际PCB

微带线（Microstrip）是电路板上最常用的平面传输线结构——一条窄带金属走线、一个介质层、一个完整参考地平面。

微带线的电场和磁场部分在空气中、部分在介质中，因此用一个有效介电常数 ε_eff 来描述：

ε_eff = (ε_r + 1)/2 + (ε_r − 1)/2 · 1/√(1 + 12h/w)

其中 w 是线宽，h 是介质厚度。对于 50Ω 微带线的线宽公式（近似）：

# 微带线特性阻抗计算（Hammerstad-Jensen公式）
        import numpy as np

        def microstrip_Z0(w, h, t, epsilon_r):
            """
            计算微带线特性阻抗 Z0
            参数:
            w: 线宽 (mm)
            h: 介质厚度 (mm)
            t: 铜箔厚度 (mm)
            epsilon_r: 相对介电常数
            """
            # 修正线宽（考虑铜箔厚度）
            w_eff = w + (t/np.pi) * (1 + np.log(2*h/t)) if t > 0 else w

            # 有效介电常数
            epsilon_eff = (epsilon_r + 1)/2 + (epsilon_r - 1)/2 * \
                          (1 + 12*h/w_eff)**(-0.5)

            # 特性阻抗（宽线近似 w/h > 1 和窄线近似 w/h < 1 统一公式）
            F = 1 + 42.3 / (epsilon_r**0.7 + 0.3)
            Z0 = 120 * np.pi / (w_eff/h + F + (w_eff/h + 1.393 + 0.667*np.log(w_eff/h + 1.444)) / \
                 np.sqrt(epsilon_eff)) * 0.5

            return Z0, epsilon_eff

        # 典型FR4参数: epsilon_r = 4.5, h = 0.8mm, t = 0.035mm
        # 50欧姆线宽约为1.45mm（需要迭代或查找表精确）
        w = 1.45
        Z0, eff = microstrip_Z0(w=w, h=0.8, t=0.035, epsilon_r=4.5)
        print(f"线宽 {w}mm: Z0 = {Z0:.1f} Ω, ε_eff = {eff:.3f}")
        # 输出: 线宽 1.45mm: Z0 ≈ 50.2 Ω, ε_eff ≈ 3.482

PCB设计实用提示

  • FR4 vs 高频基板：FR4（ε_r≈4.2–4.8, 损耗角正切≈0.02）廉价但损耗大，适合GHz以下；Rogers 4350B（ε_r≈3.48, tanδ≈0.0037）适用于5G毫米波；PTFE（特氟龙，ε_r≈2.1, tanδ≈0.0002）用于低损耗高频。
  • 50Ω微带线宽估算：FR4标准板厚1.6mm下约3.0mm；0.8mm板厚下约1.5mm；Rogers 4350B（h=0.254mm）下约0.55mm。
  • 拐角处理：直角拐角引入等效电容和阻抗不连续，高频走线用45°斜角或弧形弯。
  • 过孔：过孔的寄生电感和电容在毫米波频段不可忽略，尽量少用过孔。
