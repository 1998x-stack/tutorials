第18章 敏感问题抽样

大家好，我是你们的讲师。今天我们来探讨抽样调查中最棘手也最有趣的话题之一——敏感问题抽样。什么是敏感问题？吸毒、性行为、逃税、堕胎、政治献金……任何让受访者感到尴尬、羞耻、有风险或可能带来法律后果的问题，都是敏感问题。

想想看，如果你在街头被访问员拦住，对方直接问："先生，请问您上个月吸过毒吗？" 你会怎么回答？即使你吸过，大概率也会说"没有"。这不仅仅是"不诚实"的问题——这是一种强烈的社会期望偏差（Social Desirability Bias），人们在面对敏感问题时倾向于给出一个"安全"或"被社会认可"的回答，而非真实答案。

这里有一个差距大得惊人的真实案例。在某城市进行吸毒率调查时，研究者采用了两种方法。第一种是传统的匿名自填问卷——直接问："您在过去一年中是否使用过海洛因？" 结果为大约1%的人承认。然后研究者用另一种方法——我们本章要讲的随机化应答技术（RRT）——对同一群体的另一个独立样本进行测量，结果约为6%。相差了整整6倍！这6倍差值去哪里了？它们消失在"社会期望"的黑洞里了——那些人在匿名问卷中也选择了隐瞒，因为他们不信任调查的保密性。当一种更为强大的隐私保护机制出现时，真相才浮出水面。

核心洞察：在敏感问题调查中，受访者的"不配合"不一定出于恶意。很多时候，这是一种理性的自我保护——当真实回答可能带来负面后果时，隐瞒是合理的选择。因此，解决问题的思路不是什么"我们要让受访者更诚实"，而是设计一种机制，让诚实回答对受访者来说没有风险。这就是下面所有方法的共同哲学基础。

随机化应答技术（Randomized Response Technique, RRT）

RRT的核心创意非常巧妙：通过一个随机装置（如硬币、骰子、有编号的球）来决定受访者回答哪个问题，而访问员不知道随机装置的结果。这样，访问员看到的是一个"是/否"的回答，但无法知道这个回答是针对敏感问题还是非敏感问题的。受访者因此获得了"否认的遮蔽"——即使回答了"是"，也可以解释为"我只是倒霉抽到了那个无关的问题"。

Warner模型（1965）

这是RRT的经典模型，由统计学家Stanley Warner于1965年提出。受访者被要求用一枚不透明的硬币（或随机数生成器）来决定回答哪个问题：

  • 问题A（敏感）："我使用过海洛因。"
  • 问题B（相反）："我没有使用过海洛因。"

受访者掷硬币，如果正面朝上（概率p）则回答问题A，反面朝上（概率1-p）则回答问题B。访问员只记录"是"或"否"，不知道掷硬币的结果。

# Warner模型估计与方差计算

# 已知参数
p = 0.7  # 回答问题A的概率（p通常取0.6-0.8为宜）
n = 1000  # 样本量
yes_observed = 380  # 观察到回答"是"的人数

# 估计敏感行为比例 π
# E(λ) = p·π + (1-p)(1-π)
# 解得：π_hat = (λ - (1-p)) / (2p - 1)
lambda_hat = yes_observed / n
pi_hat = (lambda_hat - (1 - p)) / (2 * p - 1)

# 方差
var_pi = (pi_hat * (1 - pi_hat) / n) + (p * (1 - p)) / (n * (2*p - 1)**2)
se = var_pi ** 0.5

print(f"估计使用率: {pi_hat:.3f}")
print(f"标准误: {se:.3f}")
# 注意：方差中含有 p(1-p)/(2p-1)^2 项，分母会放大方差
# 这是RRT的代价——隐私保护以精度损失为代价

Simmons模型（亦称为"无关问题模型"）

Simmons模型是对Warner模型的改进。它不再使用敏感问题的反面，而是用一个已知概率的无关问题替代：

  • 问题A（敏感）："我使用过海洛因。"
  • 问题B（无关）："我的生日在1-4月之间吗？"（已知概率≈1/3）

由于无关问题的真实比例是已知的（或可以独立估计），估计敏感比例所需的计算更简单，且统计效率更高。Simmons模型比Warner模型更常用。

特征 Warner模型 Simmons模型
问题设置 敏感问题 + 其反面 敏感问题 + 无关问题
无关问题概率 自动为(1-π) 需要已知（如生日概率）
受访者信任度 中等（反面问题仍让人不安） 更高（无关问题明确无害）
统计效率 较低 较高
方差公式 有(2p-1)²项放大方差 放大效应较小

项目计数技术（Item Count Technique / List Experiment）

RRT的缺点是操作稍显复杂——受访者需要理解随机装置的原理，这对于文化程度较低的群体可能是个障碍。项目计数技术（也被称为列表实验，List Experiment）是一种更简洁的替代方案。

操作原理：

  • 将受访者随机分为两组：控制组和实验组
  • 控制组收到一份包含k个非敏感项的清单，要求回答"以上几项适用于您？"（只说数量，不说哪几项）
  • 实验组收到同一份清单但加上了敏感项，同样只报告数量

例如，想要调查是否为LGBTQ+群体，你可以这样设计：

控制组（仅非敏感项）：
  我出生于1980年之前
  我在农村长大
  我每周运动超过3小时
  我养宠物
  → 回答一个数字（0-4）

实验组（加上敏感项）：
  我出生于1980年之前
  我在农村长大
  我每周运动超过3小时
  我养宠物
  **我是LGBTQ+群体成员**
  → 回答一个数字（0-5）

估计：实验组的平均计数 - 控制组的平均计数 = 敏感项的比例
列表实验的一个关键设计细节：非敏感项目（控制项）的设计非常重要。它们不能是极度普遍或极度罕见的事件——因为如果几乎所有人都适用或不适用所有控制项，那么加上敏感项后几乎没有变化空间。好的控制项应该让每个受访者大约适用2-3项，这样敏感项的加入才会产生可测量的计数差异。此外，控制项应涵盖不同的主题领域，避免「地板效应」或「天花板效应」。

加性噪声模型（Additive Noise Model）

这种方法的思路更加直接：在受访者的真实回答上添加一个已知分布的随机噪声，然后分析人员在去噪后得到总体估计。它最初由Warner之后的学者们进一步发展，现在也被应用于差分隐私（Differential Privacy）领域——这是现代数据隐私保护的前沿技术。

在调查中，最简单的实现方式：

  • 受访者拿到一个骰子，私下掷骰子
  • 如果掷出1-5，如实回答敏感问题
  • 如果掷出6，必须回答"是"（无论真实情况如何）
  • 访问员只记录"是/否"的回答

分析时，观察到"是"的比例减去噪声部分（1/6）再除以真实回答的概率（5/6），就得到敏感行为真实比例的估计。

受訪者的理解门槛：我在实际项目中曾使用过RRT方法，发现一个重要的实操经验——很多受访者无法理解这套复杂的随机化机制。他们虽然看到掷了硬币，但不理解"硬币结果不告诉访问员"如何保护隐私。有些人甚至认为"如果我掷到正面，岂不成了我要承认自己吸毒？" 这提醒我们：RRT方法实施前，必须安排充分的前置说明，最好用通俗的类比（比如投票的保密性）来帮助受访者理解机制。如果受访者不信任这套机制，方法失效。

匿名调查方法

在走向更现代的方法时，有一个问题值得认真思考：传统的RRT和列表实验增加的操作复杂度是否值得？在某些情境下，简单的匿名调查可能就足够了。

匿名程度 方法 对诚实度的影响 适用场景
高 无人值守自填问卷+密封信封 显著提高诚实度 学校、机构内部调查
中 在线匿名问卷（无IP追踪） 中等提高 网络调查、大规模研究
低 面对面访问+访问员记录 诚实度最低 非敏感内容的官方统计
特殊 随机化应答技术（RRT） 最高水平保护 极端敏感问题

研究综述表明：对于中等敏感的问题（如政治倾向、宗教信仰），完善的匿名措施足以获得相当高的诚实率。但对于高度敏感的问题（吸毒、非法行为、性行为异常），RRT或列表实验仍然明显优于简单的匿名自填问卷。

本章小结

敏感问题抽样向我们揭示了一个深刻的道理：调查误差不仅仅来自技术层面，也来自人类心理和社会规范层面。RRT、列表实验、加性噪声模型等方法，本质上都是在设计一种「让诚实安全」的沟通机制。然而，这些方法也不是免费的午餐——它们以降低统计效率（增大方差）为代价来换取更高的诚实度。此外，受访者对机制的理解和信任是方法有效的前提。在实际应用中，需要根据问题的敏感程度、目标人群的特征、操作的可行性来综合选择方法。

到目前为止，我们讨论的所有方法都假设样本中的每个个体有某种"概率"被选中。但下一章我们将打开一个新的视角——不等概率抽样（Unequal Probability Sampling），其中每个单位被选中的概率可以不同，而且我们将学会如何通过加权来获得无偏估计。这不仅是抽样理论的重要一环，也是第20章小域估计的基础工具。
