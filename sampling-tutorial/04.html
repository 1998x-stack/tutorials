第4章 分层抽样（上）——原理与比例分配

大家好，我是你们的讲师。上一章我们学习了简单随机抽样，并讨论了它的一个主要缺点：如果总体高度异质（heterogeneous），SRS需要很大的样本量才能达到足够的精度。举个例子：要估计全国居民的平均收入，SRS需要几千个样本——因为收入这东西，少的每月两千，多的每月几万甚至几十万，方差极大。

那有没有什么办法可以用更少的样本达到同样的精度？有——这就是分层抽样（Stratified Sampling）。它的核心思想是：把总体分成若干个相对同质的子组（层），在每一层内独立抽样。因为层内的个体差异变小了，所以总的抽样误差就会变小。

本章我们学习分层抽样的基本原理和最简单的比例分配方法。下一章再深入学习最优分配和奈曼分配等更高级的内容。

分层抽样的基本原理

核心思想

分层抽样的逻辑可以用一句话概括：组内同质、组间异质（Homogeneous within, Heterogeneous between）。

具体来说，假设我们把总体划分成H层（Strata），每层的规模为Nₕ（h = 1, 2, ..., H），且 ΣNₕ = N。在每一层内，我们独立地抽取一个大小为nₕ的样本，且各层之间的抽样相互独立。然后，各层的估计结果按层的大小加权合并，得到总体估计。

为什么分层能提高效率？我们可以从方差的角度来理解。总体方差σ²可以分解为两部分：

σ² = σ²组间 + σ²组内

组间方差反映的是不同层均值之间的差异，组内方差反映的是层内个体的差异。在SRS中，总的抽样误差来自总体方差σ²的全部。而在分层抽样中，由于我们在每一层内独立抽样，组间方差被完全消除了——因为不同层的差异已经被分层的过程"吸收"掉了——剩下的只有组内方差。

关键洞察：分层抽样的效率提升来自于"锁定（Lock in）了已知的差异"。如果你知道不同组之间的均值差异很大（比如不同地区的收入差异），那么用SRS的话，这种差异会变成抽样误差的一部分；而用分层抽样，这种差异被"固定"下来，不再影响抽样的波动性。分层越有效（即层内越同质、层间越异质），效率提升就越明显。

分层变量选择

分层抽样的效果很大程度上取决于分层变量（Stratification Variable）的选择。好的分层变量需要满足以下条件：

  • 与目标变量高度相关：分层变量应该能在很大程度上解释目标变量的变异性。比如，估计收入时按"教育水平"分层（收入与教育正相关），就比按"眼睛颜色"分层有效得多。
  • 易于获取和操作：分层变量的值必须在抽样前就能获得——因为你需要在抽样之前就知道每个个体属于哪一层。常见的分层变量包括性别、年龄段、地区、户籍类型、企业规模等。
  • 分类明确、互不重叠：每个个体只能属于一个层，不然就会产生"交叉分层"的混乱。

在实践中，常见的分层变量包括：

  • 地理分层：省、市、县、城市/农村——地域通常与大量社会经济指标高度相关
  • 人口学分层：性别、年龄段、教育水平、职业类别
  • 规模分层：企业按员工数量分层、学校按在校生人数分层
  • 行业分层：按产业或行业类别分层

比例分配

当你确定好如何分层之后，下一个问题是：每层应该分配多少样本？最直观的方法就是比例分配（Proportional Allocation）：每层的样本量与该层在总体中的规模成正比。

nₕ = n × (Nₕ / N)

也就是说，如果某层占总人口的30%，那就把总样本量的30%分配给它。这种方法的优点是：

  • 简单直观：谁都不会觉得"大层多抽、小层少抽"有什么不合理
  • 自加权（Self-weighting）：在比例分配下，每个样本个体的最终权重相同（等于N/n），简化了后续的数据处理
  • 无偏估计：总体均值就是各层样本均值的加权平均，权重即为层占比

比例分配下总体均值μ的估计为：

x̄st = Σ (Nₕ/N) × x̄ₕ

其中 x̄ₕ 是第h层的样本均值。这个估计量的方差为：

Var(x̄st) = Σ (Nₕ/N)² × (σₕ²/nₕ) × (Nₕ - nₕ)/(Nₕ - 1)

忽略FPC时（各层抽样比较小），简化为：

Var(x̄st) ≈ (1/n) × Σ (Nₕ/N) × σₕ²

而SRS的方差为 σ²/n。由于 σ² = σ²组间 + Σ(Nₕ/N)σₕ²，分层抽样的方差比SRS少了 σ²组间/n 这一项——这就是效率提升的来源。

维度 SRS 分层抽样（比例分配）
方差公式 σ²/n Σ(Nₕ/N)·σₕ² / n
包含组间方差 是，组间方差计入抽样误差 否，组间方差被分层消除
效率 基准 组间差异越大，效率提升越显著
样本加权 等权 自加权（比例分配下）
子总体估计 无法控制子总体样本量 每层本就是子总体，可控制精度

层界的确定

当你使用连续变量（如年龄、收入）进行分层时，需要决定"从哪儿切一刀"——即确定层界（Strata Boundaries）。这不是随便定个整数倍数就能解决的。

统计学家发展出一种称为累积平方根法（Cumulative Root Frequency Method）的技术来确定最优层界。其步骤如下：

  • 将分层变量的取值范围划分为非常细的小区间（如年龄按每1岁一组）
  • 统计每个小区间的频数 fᵢ（或频数密度）
  • 计算每个小区间的平方根 √fᵢ
  • 将这些平方根值累加
  • 将累加和等分为H份（H为层数），切点即为最优层界

这个方法的直觉比较微妙：它试图在层内样本量尽可能均衡和层内方差尽可能小之间取得平衡。实际操作中，大多数统计软件（如R的 strSample 包、Python的sampling库）都有现成的函数可以计算最优层界。

def cumulative_root_frequency(hist_values, n_strata):
    """
    累积平方根法确定最优层界
    hist_values: 各小区间的频数列表
    n_strata: 目标层数
    """
    import numpy as np
    sqrt_f = np.sqrt(hist_values)
    cum_sum = np.cumsum(sqrt_f)
    total = cum_sum[-1]
    # 等分累积和
    cut_points = [total * i / n_strata for i in range(1, n_strata)]
    boundaries = []
    for cp in cut_points:
        idx = np.searchsorted(cum_sum, cp)
        boundaries.append(idx)
    return boundaries

# 示例：某市居民年龄分布（每组1岁，共100组）
np.random.seed(42)
age_dist = np.random.exponential(scale=35, size=10000)
hist, _ = np.histogram(age_dist, bins=100, range=(0, 100))
bds = cumulative_root_frequency(hist, 4)
print(f"4层最优层界: {bds}岁")  # 例如 [23, 42, 62] 等

真实案例：公司满意度调查

几年前，我的一位朋友在一家中型互联网公司做员工满意度调查。公司约2000名员工，分布在不同部门：技术部（约800人）、产品部（约300人）、市场部（约250人）、运营部（约350人）、行政人事部（约300人）。

HR部门最初打算做SRS，简单粗暴地抽200人发问卷。但我的朋友建议他们按部门分层。为什么？因为不同部门的满意度差异可能非常大：

  • 技术部的满意度可能集中在"研发资源是否充足"
  • 市场部的满意度或与"业绩压力"直接相关
  • 行政人事部的满意度可能受到"薪酬天花板"的影响

如果不分层，SRS可能抽到技术部偏多而市场部偏少（或反过来），导致总体满意度估计的误差增大。更严重的是，如果用SRS，某些小部门（如行政人事部）可能只被抽到很少的人，甚至完全没被抽中——这样就没法分析各部门之间的差异了。

最终他们采用了比例分配的分层抽样：每层抽10%的员工。技术部抽80人，产品部抽30人，市场部抽25人，运营部抽35人，行政部抽30人，总共200人。结果发现：

  • 总体满意度为3.8/5.0，但部门之间有显著差异——技术部和行政部在3.5左右，而产品部高达4.2
  • 如果做SRS，行政部可能只被抽到15人左右，估计误差会大得多
  • 分层后，每个部门的样本量都足够精确估计各自的满意度
一个容易忽视的问题：分层抽样并不是"免费的午餐"。虽然分层可以降低方差，但前提是分层变量必须与目标变量确实相关。如果你按照一个与满意度无关的变量（比如星座）来分层——每组内的差异和总体差异几乎一样大，分层不会带来任何效率提升。更糟糕的是，在极端情况下，如果分层变量选择不当，分层抽样甚至会引入额外的偏差。我见过有人把企业按"注册年份"分层来估计盈利能力——结果几乎等于没分层，白白增加了操作的复杂度。

本章小结

分层抽样的核心思想是"分而治之"——把异质的总体分成相对同质的子组，在组内独立抽样，然后加权合并。这种方法可以消除组间方差，从而在相同样本量下获得比SRS更小的抽样误差。比例分配是最简单、最常用的样本分配方法，每层的样本量与该层在总体中的比例一致。

比例分配虽然简单直观，但它并不是效率最高的分配方式。如果你知道不同层的方差差异很大（比如某些层的个体差异大、某些层个体差异小），那么把更多的样本分配到方差大的层中，可以进一步提高精度。这就是我们下一章要学习的最优分配（Optimal Allocation）和奈曼分配（Neyman Allocation）。
