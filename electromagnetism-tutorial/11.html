开场：一个装订工人的伟大发现

迈克尔·法拉第（Michael Faraday, 1791-1867）——一个只上过小学的装订店学徒，后来成为19世纪最伟大的实验物理学家之一。他的故事本身就令人振奋。

1820年奥斯特发现电流产生磁后，法拉第就开始思考一个"对称"的问题：既然电能生磁，那磁能否生电？他花了十年时间反复尝试，终于——1831年8月29日——他用一个铁环和两个线圈实现了历史性的突破。当其中一个线圈的电流变化时，另一个线圈中产生了瞬时电流。

法拉第的洞察：关键在于"变化"。恒定磁场不会产生感应电流，只有变化的磁场才能产生电流。这就是电磁感应（Electromagnetic Induction）——它催生了发电机、变压器、感应电机，彻底改变了人类文明。

法拉第的十年追寻

法拉第的实验装置非常简洁：一个软铁环，左右两侧分别绕制线圈 A 和 B。线圈 A 连接电池和开关，线圈 B 连接灵敏电流计。

  • 闭合开关的瞬间：电流计指针偏转一下，随即归零。
  • 断开开关的瞬间：电流计指针反向偏转一下，随即归零。
  • 开关保持闭合（恒定电流）：电流计毫无反应。

法拉第立即意识到：感应电流只在线圈 A 中电流变化（即磁场变化）的瞬间出现。当磁场稳定后，感应消失。

这就引出了法拉第电磁感应定律的积分形式：

ε = −dΦB / dt

其中 ε 是感应电动势（单位：V），ΦB = ∫S B · dS 是穿过回路所围曲面的磁通量。

微分形式（更深刻地揭示了场的本质）：

∇ × E = −∂B / ∂t

这是麦克斯韦方程组的核心之一：变化的磁场产生一个非保守的涡旋电场。

动生电动势

当导体在磁场中运动时，导体内的自由电子受到洛伦兹力，沿导体方向移动，形成电动势——这就是动生电动势（motional EMF）。

对于在磁场中运动的直导体：ε = ∫ (v × B) · dl

滑轨上的导体棒

经典模型：两根平行导轨（间距 L），一根导体棒以速度 v 在导轨上滑动，整个系统置于垂直向外的均匀磁场 B 中。

动生电动势：ε = B L v

感应电流：I = ε / R = B L v / R

力平衡：棒受安培力 F = I L B = B² L² v / R 阻碍运动——这正是能量守恒的体现：外界需要做功来克服安培力，机械能转化为电能。

法拉第圆盘发电机

法拉第本人设计的第一个发电机——一个铜圆盘在磁极间旋转，电刷从盘心和盘缘引出电流。虽然电压很低，但它是所有现代发电机的鼻祖。

特征 动生电动势 感生电动势
产生原因 导体在恒定磁场中运动 变化的磁场激发涡旋电场
物理机制 洛伦兹力（qv × B） 感应电场（∇ × E = −∂B/∂t）
适用公式 ε = ∫(v × B)·dl ε = −dΦB/dt
能否用同一公式 ε = −dΦ/dt 也适用 ε = −dΦ/dt 也适用
能量转换 机械能 → 电能 磁场能 → 电能
示例 发电机、麦克风 变压器、感应加热

感生电动势与涡旋电场

感生电动势的物理本质比动生电动势更深刻。当磁场随时间变化时，即使没有导体存在，也会在空间中激发出一个非保守的涡旋电场。

这一点与静电场截然不同：

  • 静电场：由电荷产生，是保守场（∇ × E = 0），电场线从正电荷出发到负电荷终止。
  • 感应电场：由变化磁场产生，是非保守场（∇ × E = −∂B/∂t ≠ 0），电场线是闭合的环线。

这个涡旋电场是"无源"的——它不由电荷产生，不始于正电荷也不终止于负电荷。它的"源"是变化的磁场。

🔑 核心洞察
感应电场的一个绝妙应用是电子感应加速器（betatron）。
在 betatron 中，一个环形真空室置于交变磁场中。变化的磁场产生涡旋电场，加速其中的电子。同时，磁场本身还扮演着"约束"的角色——洛伦兹力使电子沿圆形轨道运动。
关键条件是：轨道处的磁场 B 应等于轨道内平均磁场的½（即"2:1条件"），才能保证电子轨道半径保持不变。1940年，克尔斯特（Kerst）建造了第一台 betatron，将电子加速到了 2.3 MeV。

楞次定律

法拉第定律中的负号（ε = −dΦ/dt）就是楞次定律（Lenz's Law）的数学表达。

楞次定律：感应电流的方向总是使其产生的磁场阻碍引起感应电流的磁通量变化。

这本质上是能量守恒的体现。如果感应电流不是阻碍而是促进磁通量变化，那么只要微小的扰动就会导致磁通量无限增大——这显然不可能。

楞次定律在工程中的体现：

  • 涡流制动：磁铁靠近运动导体时，导体中感应涡流，涡流产生的磁场阻碍运动——用于高铁制动、电表阻尼。
  • 感应加热：交变磁场在导体内产生涡流，涡流的焦耳热加热导体——用于 IH 电饭煲、淬火。
  • 金属探测器：交变磁场在金属中感应涡流，涡流产生的次生磁场被探测线圈检测。
💡 经典演示：跳跃的铝环
法拉第的原始实验至今仍是课堂上的最佳演示：
将一个铝环套在插入交流电路的铁芯螺线管上方。接通电源后，铝环跳起！
为什么？螺线管中的交变电流产生交变磁场，铝环中感应出涡流。楞次定律说，感应电流的方向要使自身受到的合力远离螺线管（即阻碍磁通量的增加）。于是铝环受到了向上的电磁力——它跳起来了。
如果铝环上开一个缝，涡流无法形成，铝环就不会跳起。这直观地证明了"涡流"的存在。

电磁感应的工程应用

发电机

发电机是目前全球电力的主要来源。无论是火力、水力、风力还是核能，归根结底都是通过机械能驱动线圈在磁场中旋转，产生交变感应电动势。

最简单的交流发电机：一个矩形线圈在均匀磁场中以角速度 ω 旋转。磁通量 Φ = B A cos(ωt)，感应电动势 ε = −dΦ/dt = B A ω sin(ωt) —— 正弦交流电。

变压器

一对绕在公共磁芯上的线圈。一次线圈接交变电源，在磁芯中产生交变磁通；二次线圈中感应出电动势。

理想变压器：V₂ / V₁ = N₂ / N₁（电压比等于匝数比），I₂ / I₁ = N₁ / N₂（电流反比）。

变压器使得电力远距离传输成为可能——高压输电大幅降低线路损耗（P = I²R），而变压器在两端完成升压和降压。

感应加热（IH 电饭煲）

IH 电饭煲的加热原理：线圈产生高频交变磁场（≈ 20-40 kHz），磁场穿透锅底（铁磁材料），在锅底感应出涡流和磁滞损耗，两者共同将锅加热。

优势：热效率高（> 90%，传统电阻加热约 70%）、火力可控、安全（无明火、线圈不发热）。

Python 模拟：法拉第感应

import numpy as np
import matplotlib.pyplot as plt

# 模拟参数
R = 0.1          # 回路电阻 (Ω)
L = 0.01         # 回路电感 (H) — 使瞬态更平滑
B0 = 1.0         # 磁场幅值 (T)
A = 0.01         # 回路面积 (m²)
f = 50           # 频率 (Hz)
dt = 1e-5        # 时间步长 (s)
T = 0.1          # 总时长 (s)
N = int(T / dt)

t = np.linspace(0, T, N)
Phi = B0 * A * np.cos(2 * np.pi * f * t)  # 磁通量
EMF = -np.gradient(Phi, dt)               # 感应电动势 = -dΦ/dt

# LR 电路暂态求解
I = np.zeros(N)
for i in range(1, N):
    dI = (EMF[i] - R * I[i-1]) / L * dt
    I[i] = I[i-1] + dI

plt.figure(figsize=(10, 8))
plt.subplot(3, 1, 1)
plt.plot(t * 1000, Phi * 1000)
plt.ylabel('磁通量 Φ (mWb)')
plt.title('法拉第感应定律数值模拟')

plt.subplot(3, 1, 2)
plt.plot(t * 1000, EMF)
plt.ylabel('感应电动势 ε (V)')

plt.subplot(3, 1, 3)
plt.plot(t * 1000, I * 1000)
plt.xlabel('时间 (ms)')
plt.ylabel('感应电流 I (mA)')

plt.tight_layout()
plt.show()

本章小结

  • 法拉第电磁感应定律：ε = −dΦB/dt。变化的磁场产生电动势——这是发电机和变压器的物理基础。
  • 动生电动势由导体在磁场中运动产生（洛伦兹力机制）；感生电动势由变化磁场激发涡旋电场产生。
  • 楞次定律（负号的含义）：感应效应总是阻碍磁通量的变化——能量守恒的必然推论。
  • 电磁感应的工程应用：发电机、变压器、感应电机、感应加热、涡流制动……没有电磁感应就没有现代电力系统。
