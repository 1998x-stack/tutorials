切线问题：导数的诞生

大家好，我是你们的讲师。如果你问一个学完微积分的人"你觉得微积分里哪个概念最重要"，十有八九的回答是导数。导数就像是微积分世界里的"速度计"——它告诉我们每一件事物在每一瞬间的变化有多快。

但你知道吗？导数的诞生其实源于一个非常实际的问题：如何求一条曲线在某一点的切线斜率？古希腊的阿波罗尼奥斯和阿基米德已经会求圆和圆锥曲线的切线，但面对任意曲线（比如 y = sin x 或 y = e^x），古人束手无策了两千年。直到17世纪，牛顿和莱布尼茨分别独立地给出了答案——而这个答案，就是我们今天要学习的导数。

这个故事告诉我们一个深刻的道理：数学中那些最伟大的发现，往往来自最朴素的问题。切线斜率——听起来像是初中几何的内容——却成了整个微积分的起点之一。本章我们就从"如何求切线的斜率"这个朴素问题出发，一步步揭开导数的神秘面纱。

从平均变化率到瞬时变化率

平均变化率：割线的斜率

假设你开着一辆车，记录下它在不同时刻的位置。如果 t = 1 秒时在 s = 10 米处，t = 3 秒时在 s = 50 米处，那么这段时间内的平均速度就是 (50 - 10) / (3 - 1) = 20 米/秒。

用数学语言表达：对于函数 y = f(x)，当自变量从 x0 变化到 x0 + Δx（Δx 读作"delta x"，表示 x 的改变量），函数值的改变量是 Δy = f(x0 + Δx) - f(x0)。平均变化率就是：

平均变化率 = Δy / Δx = [f(x0 + Δx) - f(x0)] / Δx

几何上，这个比值就是经过点 (x0, f(x0)) 和 (x0 + Δx, f(x0 + Δx)) 的割线（secant line）的斜率。你在中学就学过：两点确定一条直线，斜率等于纵坐标之差除以横坐标之差。

瞬时变化率：让Δx趋于0

但平均速度只告诉我们"整体"跑多快，没办法回答"在 t = 1 秒那一瞬间，车速是多少"这个问题。为了得到瞬时速度，我们需要让时间间隔 Δt 无限缩小——从 1 秒缩到 0.1 秒，再到 0.01 秒，再到 0.0001 秒……当 Δt → 0 时，平均速度的极限就是瞬时速度。

这是一个典型的极限过程——正是在第7章中我们反复练习的那类计算。导数的定义就是：让 Δx → 0，求平均变化率的极限：

f'(x0) = lim_{Δx→0} [f(x0 + Δx) - f(x0)] / Δx

这个极限（如果存在）就是函数 f 在点 x0 处的导数（derivative）。

导数有多种等价的记号，你要习惯它们：

记号 读法 提出者 使用场景
f'(x0) f prime of x0 拉格朗日 函数形式，最常用
dy/dx|_{x=x0} dy over dx 莱布尼茨 微分形式，链式法则方便
y'(x0) y prime 拉格朗日 与 f' 类似
df/dx|_{x=x0} d f over d x 莱布尼茨 强调变量关系
Df(x0) D f 欧拉 微分算子形式
用极限理解导数
导数定义的本质是一个极限——平均变化率在 Δx→0 时的极限。如果你在第3-7章扎实地掌握了极限的计算，那么导数的定义对你来说只是在"老朋友"的肩膀上往前走了一步。反之，如果极限计算还没熟练，建议回头翻翻第7章的极限综合实战。

左导数与右导数

与极限类似，导数也有"从哪边逼近"的问题。左导数是 Δx → 0^- 时平均变化率的极限，右导数是 Δx → 0^+ 时的极限。函数在某点可导的充要条件是：左导数 = 右导数。

让我们看一个经典例子：绝对值函数 f(x) = |x| 在 x = 0 处的可导性。

左导数：f'_-(0) = lim_{h→0^-} [|0 + h| - |0|] / h
                 = lim_{h→0^-} |h| / h
                 = lim_{h→0^-} (-h) / h    （因为 h < 0 时 |h| = -h）
                 = -1

右导数：f'_+(0) = lim_{h→0^+} [|0 + h| - |0|] / h
                 = lim_{h→0^+} |h| / h
                 = lim_{h→0^+} h / h        （因为 h > 0 时 |h| = h）
                 = 1

左导数是 -1，右导数是 +1，两者不相等——所以 f(x) = |x| 在 x = 0 处不可导。这个尖点就是所谓的"不可导点"。

一个真实的面试题
我曾经看到一个大厂算法岗的面试题，第一问就是"f(x) = |x - 3| 在 x = 3 处是否可导？"很多候选人脱口而出"可导"，因为在 x = 3 两侧函数值的差是一样的。但实际上，按照左导数等于右导数的判断标准，它和 |x| 在 0 处完全一样，是不可导的。面试官正是用这道题来考察候选人的基本功是否扎实——当你觉得问题"太简单了"的时候，往往就是最需要警惕的时候。

导数的几何意义：切线方程

切线的定义

在中学数学里，你学到的"圆的切线"定义是：与圆只有一个交点的直线。但这个定义对一般曲线不适用——比如 y = x^3 在 x = 0 处，x 轴就是它的切线，但 x 轴与曲线相交于 (0,0)，并且还会在别处再次相交吗？不，但这里的关键是：切线的现代定义不是"只有一个交点"，而是割线的极限位置。

当你让 Δx → 0 时，割线绕着点 (x0, f(x0)) 旋转，越来越逼近一个固定的位置——这个极限位置就是切线（tangent line）。而导数 f'(x0) 就是这条切线的斜率。

切线方程

已知斜率和一点，你马上就能写出切线方程：

切线方程：y - f(x0) = f'(x0) · (x - x0)

也就是说，在点 x0 附近，函数值可以用切线来近似：

f(x) ≈ f(x0) + f'(x0) · (x - x0),  当 x → x0 时

这个简单的近似公式是整个微积分的核心思想之一：用直线（线性函数）来近似曲线。在你学习第13章泰勒公式时，会发现这实际上就是泰勒公式的一阶近似。

可导与连续的关系

这是微积分里一个极其重要的关系，很多初学者会搞混，咱们一次性说清楚。

定理：如果一个函数在某点可导，那么它在该点一定连续。

证明其实很简洁：如果 f'(x0) 存在，那么当 Δx → 0 时，

Δy = [f(x0 + Δx) - f(x0)]
    = [Δy / Δx] · Δx
    → f'(x0) · 0 = 0   （当 Δx → 0 时）

所以 lim_{Δx→0} f(x0 + Δx) = f(x0)，即函数在 x0 处连续。

但反过来呢？连续不一定可导！

最经典的例子就是前面的 f(x) = |x| ——它在 x = 0 处连续（因为 lim |x| = 0 = |0|），但左右导数不相等，所以不可导。连续是"没有断点"，可导是"没有尖点"——这是两个不同的光滑层次。

关系 成立吗？ 反例
可导 ⇒ 连续 ✅ 成立 可导必然连续
连续 ⇒ 可导 ❌ 不成立 f(x) = |x| 在 x=0
不连续 ⇒ 不可导 ✅ 成立（逆否命题） ——
不可导 ⇒ 不一定不连续 ✅ 成立 f(x) = |x| 在 x=0
一个常见的"想当然"错误
有些同学在做了大量光滑函数（多项式、指数函数等）的求导练习后，会产生一种错觉："所有连续函数的图像都是光滑曲线，所以都应该是可导的。"但事实远非如此——维尔斯特拉斯构造了一个处处连续但处处不可导的函数（维尔斯特拉斯函数）。这个函数的图像在任何尺度下都是锯齿状的，无论你怎么放大，都找不到一条光滑的切线。它告诉我们：连续和可导之间的差距，比大多数人想象的要大得多。

用定义求几个简单函数的导数

在进入第9章的求导法则之前，让我们先用定义（差商极限）求几个简单函数的导数。这有助于你真正理解导数的含义，而不是机械地套用公式。

例1：常数函数 f(x) = C

f'(x) = lim_{h→0} [C - C] / h = lim_{h→0} 0 / h = 0

常数函数的导数为0——这是直观的：一条水平线的斜率处处是0。

例2：线性函数 f(x) = kx + b

f'(x) = lim_{h→0} [k(x+h) + b - (kx + b)] / h
       = lim_{h→0} (kh) / h
       = k

线性函数的导数等于斜率 k，且与 x 无关——直线的斜率处处相等，合情合理。

例3：二次函数 f(x) = x^2

f'(x) = lim_{h→0} [(x+h)^2 - x^2] / h
       = lim_{h→0} [x^2 + 2xh + h^2 - x^2] / h
       = lim_{h→0} (2xh + h^2) / h
       = lim_{h→0} (2x + h)
       = 2x

f(x) = x^2 的导数是 2x。注意这里的关键技巧：展开 (x+h)^2 后，x^2 被消掉了，剩下的项都有公因子 h，约掉后令 h → 0 即可。这个过程体现了导数定义计算的核心套路：展开 → 抵消 → 约掉 h → 求极限。

例4：f(x) = 1/x

f'(x) = lim_{h→0} [1/(x+h) - 1/x] / h
       = lim_{h→0} [(x - (x+h)) / (x(x+h))] / h
       = lim_{h→0} [-h / (x(x+h))] / h
       = lim_{h→0} -1 / [x(x+h)]
       = -1 / x^2

注意，这里 x ≠ 0——反比例函数在 0 处根本无定义，自然不可导。

例5：f(x) = sin x（极限形式）

sin'(x) = lim_{h→0} [sin(x+h) - sin x] / h
         = lim_{h→0} [sin x cos h + cos x sin h - sin x] / h
         = lim_{h→0} [sin x (cos h - 1) / h + cos x (sin h / h)]

利用 lim_{h→0} sin h / h = 1（第4章的重要极限）和
    lim_{h→0} (cos h - 1) / h = 0（可推导），得到：

sin'(x) = cos x

这个结果非常漂亮——正弦函数的导数竟然是余弦函数。

一个小窍门
用定义求导时，如果遇到分式，先通分再化简；如果遇到根号，用有理化（乘以共轭）；如果遇到三角，用和差化积公式。这些"预处理"操作的共同目标只有一个：把 h 从分母中消掉，这样才能安全地令 h → 0。

本章小结

本章我们完成了从极限到导数的关键跨越。你学到了：导数就是平均变化率在 Δx → 0 时的极限，也就是切线斜率；左导数和右导数必须相等才能说"可导"；可导一定连续，但连续不一定可导；以及如何用定义求简单函数的导数。

现在你手中已经握着微积分最核心的工具——导数的定义。从下一章开始，我们不再每次都从定义出发了——第9章将教你一套高效的求导法则，让你能够快速求出任意初等函数的导数。想象一下：不用每次求 sin x 的导数都做一次极限计算，而是直接写出 cos x——这就是标准求导法则带来的效率提升。
