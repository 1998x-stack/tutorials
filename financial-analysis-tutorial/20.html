第20章 衍生品定价

大家好，我是你们的讲师。上一章我们聊了多因子模型——用一组系统性的风险因子来解释股票收益的差异。但金融世界的版图远不止股票这一种资产。如果我说有一种金融产品，它的价格不是由"公司好不好"决定的，而是由"大家觉得未来会怎样"决定的——你会不会觉得这很神奇？这就是衍生品（Derivatives）的世界。

衍生品的名字来源于它的本质：其价值是从另一种"基础资产"的价格中衍生出来的。股票期权、股指期货、商品期货、利率互换、信用违约互换……这些产品构成了一个规模远超股票市场的巨大市场。根据国际清算银行的数据，全球衍生品场外市场的名义本金规模超过600万亿美元——是全球GDP的6倍多。

为什么衍生品如此重要？因为它们提供了两样基础资产无法直接提供的东西：杠杆和保险。通过期权，你可以用很小的权利金控制大量市值的股票，同时风险是有限的；通过期货，你可以锁定未来的价格，规避原材料价格上涨的风险。但同时，衍生品也是双刃剑——使用不当可以造成毁灭性的损失，巴林银行的倒闭和李森的故事就是最经典的教训。本章我们将深入衍生品定价的核心逻辑。

20.1 期货与期权基础

在进入定价模型之前，我们需要先理清衍生品的两个基本类型：期货和期权。它们最核心的区别在于收益结构的线性与非线性。

期货：线性收益

期货合约（futures contract）的本质是：双方约定在未来某个时间，以约定的价格买卖某项资产。如果你是买方（多头），那么当标的资产价格上涨时，你赚钱；下跌时，你亏钱。收益是线性的——盈亏与标的价格变动呈1:1的关系（忽略基差等细节）。

期货的特点：不需要全额付款，只需缴纳一定比例的保证金（通常5%-15%），天然带有杠杆。在中国，股指期货（IF、IC、IH）是机构投资者对冲风险的核心工具；商品期货（螺纹钢、铁矿石、原油等）则是实体企业管理价格风险的重要工具。

期权：非线性收益

期权合约（option contract）的本质是：买方支付一笔"权利金"（premium），获得了在未来以特定价格买卖某项资产的权利，而不是义务。

几个关键术语：

  • 行权价（Strike Price / K）——合约约定的买卖价格。
  • 到期日（Expiration Date / T）——合约的有效期限。
  • 权利金（Option Premium）——买方为获取这个权利支付的价格。
  • 实值/平值/虚值（ITM/ATM/OTM）——取决于标的价格与行权价的关系。
状态 看涨期权（Call） 看跌期权（Put）
实值 ITM 标的价格 > 行权价 标的价格 < 行权价
平值 ATM 标的价格 ≈ 行权价 标的价格 ≈ 行权价
虚值 OTM 标的价格 < 行权价 标的价格 > 行权价

期权的关键特征就是收益的不对称性：买方的最大亏损是已支付的权利金，但潜在收益（理论上）是无限的；卖方的最大收益是收到的权利金，但潜在亏损是无限的。正是这种不对称性，使得期权成为一种强大的风险管理和投机工具。

新手提示：很多刚接触期权的同学会犯一个错误——把期权当期货做。期权不是简单的"看涨就买Call"。"时间价值"的流逝是期权买方的"隐形敌人"——即使股价朝着你预期的方向移动，但如果涨得不够快、不够多，你可能到期时仍然是亏损的。这就是所谓的"方向对了还亏钱"的尴尬局面。

20.2 Black-Scholes模型

1973年，费希尔·布莱克（Fischer Black）和迈伦·舒尔斯（Myron Scholes）发表了那篇注定载入史册的论文，提出了期权定价的Black-Scholes模型（简称BS模型）。罗伯特·默顿（Robert Merton）在同一时期独立地做出了类似的贡献。这项成就后来获得了诺贝尔经济学奖（1997年），遗憾的是布莱克当时已去世，未能一同获奖。

BS模型的核心公式如下：

C = S₀·N(d₁) - K·e⁻ʳᵀ·N(d₂)

其中：

  • d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
  • d₂ = d₁ - σ√T
  • C ——看涨期权价格
  • S₀ ——当前标的价格
  • K ——行权价
  • r ——无风险利率
  • T ——距到期日时间（以年为单位）
  • σ ——波动率（标准差）
  • N(·) ——标准正态分布的累积分布函数

d₁和d₂的经济学含义

公式中的d₁和d₂不只是一个数学符号，它们有非常直观的经济学含义：

  • N(d₁) ——在风险中性世界里，期权到期时处于实值的概率，经过"标的资产数量"的调整。它也是期权Delta的近似值（见20.3节）。更通俗地说，它衡量的是"股价超过行权价的概率权重"。
  • N(d₂) ——在风险中性世界里，期权到期时处于实值的概率。乘以K·e⁻ʳᵀ（行权价的现值），就是"预计要支付的行权价的现值"。

所以BS公式可以非常直观地理解为：期权的价值 = 预期收到的东西 - 预期付出的东西，只是这两者都是在风险中性概率下计算的期望值。

隐含波动率（Implied Volatility）

BS公式中有五个输入参数（S, K, r, T, σ），其中前四个都可以从市场上直接观察到，唯独波动率σ是未知的。但有趣的是，我们可以反过来用——将市场价格代入BS公式，反解出σ。这个反解出来的波动率就是隐含波动率（IV）。

隐含波动率可以理解为市场对"未来波动"的一致预期。如果某支股票的IV突然飙升，说明市场普遍认为该股票未来将有大波动——无论向上还是向下。所以IV有时被称为市场的"恐惧指数"。

关键洞见：BS模型最重要的贡献不是用一个公式计算出了期权的"正确价格"，而是提供了一个统一的语言框架来讨论期权的价格——将价格分解为标的价格、时间和波动率三个维度。在这个框架下，交易员可以清晰地表达他们的观点："我认为隐含波动率被高估了"或者"我赌波动率会上升"。

20.3 希腊字母体系

希腊字母（Greeks）是期权交易员最重要的风险管理工具。它们衡量的是期权价格对不同输入因素的敏感度。下表总结了五个核心希腊字母：

希腊字母 符号 衡量什么 方向提示 对冲应用
Delta Δ 标的价格每变动1元，期权价格变动多少 Call: 0~1, Put: -1~0 Delta对冲（中性策略基础）
Gamma Γ 标的价格每变动1元，Delta变动多少 ATM附近最大 Gamma Scalping（波动率交易）
Theta Θ 每过去一天，期权价格损失多少 买方为负，卖方为正 时间价值收割策略
Vega ν 隐含波动率每变动1%，期权价格变动多少 到期前均为正 波动率交易的核心指标
Rho ρ 无风险利率每变动1%，期权价格变动多少 Call为正，Put为负 利率敏感型头寸管理

希腊字母可以组合使用，构建出各种复杂的风险管理策略。例如：

  • Delta中性（Delta Neutral）——通过买卖标的资产对冲Delta，使得持仓组合对股价的方向性变动"免疫"，只保留Gamma和Vega的风险暴露。这是期权做市商最基础的持仓管理策略。
  • Gamma Scalping——在Delta中性的前提下，利用Gamma的"凸性"效应，从小幅价格波动中盈利。标的价格上涨时，Gamma使得Delta变成正值，需要卖出股票对冲（获利）；标的价格下跌时，Gamma使得Delta变成负值，需要买入股票对冲（又获利）。
踩坑警示：很多期权新手犯过的最昂贵的错误是"只关注Delta，忽视Gamma"。一个临近到期的平值期权Gamma极高——标的价格稍微一动，Delta就会大幅变化。如果你以为Delta=0.5就意味着"50%的概率挣钱"而买入大量仓位，结果标的价格反向波动2%，你的Delta可能已经变成了0.8（反向继续亏大）或者0.2（方向对了但仓位太小）。Gamma是期权的"加速度"，它可以让盈利加速，也可以让亏损加速。

下面是一个利用Python实现BS定价和希腊字母计算的示例：

import numpy as np
from scipy.stats import norm

def bs_call_price(S, K, T, r, sigma):
    """
    Black-Scholes 欧式看涨期权定价
    S: 标的价格, K: 行权价, T: 到期时间(年)
    r: 无风险利率, sigma: 波动率
    """
    d1 = (np.log(S / K) + (r + 0.5 * sigma ** 2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)

    call = S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)
    return call

def bs_greeks(S, K, T, r, sigma):
    """计算五个希腊字母"""
    d1 = (np.log(S / K) + (r + 0.5 * sigma ** 2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)

    delta = norm.cdf(d1)           # Call Delta
    gamma = norm.pdf(d1) / (S * sigma * np.sqrt(T))
    theta = -(S * norm.pdf(d1) * sigma) / (2 * np.sqrt(T)) \
            - r * K * np.exp(-r * T) * norm.cdf(d2)
    vega = S * norm.pdf(d1) * np.sqrt(T)
    rho = K * T * np.exp(-r * T) * norm.cdf(d2)

    return {'Delta': delta, 'Gamma': gamma,
            'Theta': theta, 'Vega': vega, 'Rho': rho}

# 示例：当前股价100，行权价105，距到期30天，利率3%，波动率25%
S, K, T, r, sigma = 100, 105, 30/365, 0.03, 0.25
price = bs_call_price(S, K, T, r, sigma)
greeks = bs_greeks(S, K, T, r, sigma)
print(f"Call Price: {price:.4f}")
for g, v in greeks.items():
    print(f"{g}: {v:.4f}")

20.4 波动率曲面与套利

如果BS模型是完美准确的，那么对于同一标的、同一到期日的所有期权，反解出的隐含波动率应该相同。但实际市场告诉我们一个截然不同的故事。

波动率微笑与偏斜

在1987年股灾之前，交易员们确实认为"波动率是一个常数"。但股灾之后，市场发现了一个令人不安的现象：深度虚值看跌期权的隐含波动率显著高于平值期权的隐含波动率——市场愿意为"尾部风险保护"支付更高的溢价。将不同行权价的IV画在图上，会形成一条"微笑"（smile）或"偏斜"（skew）曲线。

  • 波动率微笑（Volatility Smile）——深度实值和深度虚值的IV都高于平值，呈现U型。常见于外汇期权市场。
  • 波动率偏斜（Volatility Skew）——深度虚值Put的IV明显高于深度虚值Call的IV，呈现偏斜形态。这是股票期权市场的典型特征，反映了市场对崩盘风险的持续担忧。

期限结构

不但不同行权价的IV不同，不同到期日的IV也不同。将不同期限的IV画出来，就是波动率的期限结构（term structure）。在正常市场中，远期IV通常高于近期IV（contango），因为远期的不确定性更高。但在危机时期，近期IV可能飙升到远超远期的水平（backwardation）。

实战应用：将不同行权价和不同到期日的IV放在一张三维图上，就构成了波动率曲面（Volatility Surface）。它是对BS模型"波动率恒定"假设的否定，也是专业期权交易员的"天气预报图"。波动率曲面的形状变化本身就是重要的交易信号——在2008年金融危机前夕，标普500期权的波动率偏斜急剧陡峭化，这是一个极其可靠的预警信号。

波动率套利策略

理解了IV的规律之后，交易员可以构建各种波动率套利策略：

  • 做多波动率（Long Vol）——买入跨式组合（Straddle，同时买入Call和Put），赌市场会有大波动。如果IV被低估，或者即将有重大事件（财报、政策会议），这种策略有吸引力。
  • 做空波动率（Short Vol）——卖出跨式组合，赌市场会保持平静。这是一种"收集时间价值"的策略，但风险在于：黑天鹅事件可以让一次做空波动率的亏损抹去之前数月的盈利。
  • 波动率套利（Vol Arbitrage）——同时买入和卖出不同行权价或不同期限的期权，利用IV之间的"异常"差距获利。比如，如果近月IV远低于远月IV但你觉得不合理，可以卖出远月期权同时买入近月期权。
核心洞见：记住，衍生品定价的核心不是"预测未来"，而是"管理风险"。BS模型、希腊字母、波动率曲面——这些工具的价值在于，它们将交易员的"主观判断"转化为可以量化和对冲的"风险暴露"。一个好的衍生品交易员，不是在赌博市场涨跌，而是在管理Delta、Gamma、Vega的暴露组合。

衍生品为金融世界增添了"维度"——它让投资者可以独立地对方向、时间、波动率分别下注。在下一章中，我们将进入固定收益分析的世界。如果说衍生品是金融工具箱里的"手术刀"，那么固定收益分析就是"听诊器"——它帮你诊断一家公司、一个国家的"财务健康状况"。从债券定价到收益率曲线，从久期到信用利差，你会发现固定收益的分析体系同样是精妙而严谨的。
