第4章 供给与需求

大家好，我是你们的讲师。学完贸易和比较优势，你可能在想：现实中谁来协调这些交易？答案是——市场。而市场的"两只手"就是供给与需求。经济学里最经典的一句话是："你甚至可以让鹦鹉学会经济学——只要教它说'供给与需求'。"这一章我们就来彻底搞懂这句话的含义。

本章学习目标：
  • 理解市场的概念与竞争类型
  • 掌握需求曲线与供给曲线的逻辑
  • 区分"变动"与"变动量"
  • 求解市场均衡并预测变化方向

4.1 故事：2020年口罩价格暴涨

故事时间：2020年初新冠疫情突然爆发，口罩瞬间成了"战略物资"。平时一包医用外科口罩不到10元，短短几天内涨到了50元甚至100元以上。有人大骂"发国难财"，有人连夜建生产线，有人到处找口罩。为什么口罩价格涨了这么多？答案全在供给与需求之中。

需求端：疫情让几乎每个人都需要戴口罩，需求曲线向右移动。供给端：正逢春节假期，工厂停工、物流停运，供给曲线几乎没有移动，甚至在短期内向左移动。结果是——均衡价格飙升。等到各工厂恢复生产、产能上来了（供给曲线右移），价格才逐步回落。

4.2 市场与竞争

一个市场是由某种商品的买者和卖者组成的群体。经济学根据参与者数量把市场分为几类：

  • 完全竞争市场：许多买者和卖者，产品同质，没人能影响价格（如农产品市场）。
  • 垄断：只有一个卖者（如本地自来水公司）。
  • 寡头：几个大卖者（如智能手机芯片市场）。
  • 垄断竞争：许多卖者，产品有差异（如餐馆）。

我们这一章先聚焦完全竞争市场——它是理解其他市场结构的基准。

4.3 需求曲线

需求定律（Law of Demand）

其他条件不变时，价格越高，需求量越小；价格越低，需求量越大。这个负相关关系几乎适用于所有商品。

需求变动 vs 需求量变动

这是初学者最容易混淆的地方。

  • 需求量变动：由价格变化引起——沿同一条需求曲线移动。
  • 需求变动：由价格以外的因素引起——整条需求曲线移动。

什么会导致需求曲线移动？我们称之为需求变化因素：

因素变化方向例子
收入增加正常品需求↑，低档品需求↓收入涨了，更多人去餐厅吃饭（正常品）
偏好改变需求同向变化健身风潮兴起，运动装备需求↑
替代品价格替代品涨价→本商品需求↑牛肉涨价→猪肉需求↑
互补品价格互补品涨价→本商品需求↓汽油涨价→汽车需求↓
预期影响当前需求预期房价上涨→现在买房
买者数量买者越多→需求越大出生率高的城市，学区房需求↑

4.4 供给曲线

供给定律（Law of Supply）

其他条件不变时，价格越高，供给量越大；价格越低，供给量越小。正相关关系源自边际成本递增——生产越多，额外多生产一单位的成本越高，所以需要更高的价格来激励。

供给变动 vs 供给量变动

  • 供给量变动：由价格变化引起——沿同一条供给曲线移动。
  • 供给变动：由价格以外的因素引起——整条供给曲线移动。
因素变化方向例子
技术改进成本下降→供给↑自动化生产线使手机产量大增
投入品价格投入涨价→供给↓钢铁涨价→汽车供给↓
预期影响当前供给预期房价跌→开发商捂盘改为抛售
卖者数量卖者越多→供给越大网约车平台开放注册→出行供给↑

4.5 市场均衡

均衡（equilibrium）发生在需求量等于供给量时的价格水平。此时市场"出清"——没有短缺也没有过剩。

  • 过剩（surplus）：价格高于均衡价，供给量 > 需求量。卖家降价清仓。
  • 短缺（shortage）：价格低于均衡价，需求量 > 供给量。买家抬价抢购。

市场力量会自动推动价格回归均衡——这就是亚当·斯密所说的"看不见的手"。

# 求解市场均衡价格与数量
# 假设需求函数: Qd = 100 - 2P
# 假设供给函数: Qs = -20 + 4P
# 均衡条件: Qd = Qs

def find_equilibrium():
    # 100 - 2P = -20 + 4P
    # 120 = 6P
    P_eq = 120 / 6  # P = 20
    Q_eq = 100 - 2 * P_eq  # Q = 60
    return P_eq, Q_eq

P_star, Q_star = find_equilibrium()
print(f"均衡价格: P* = {P_star}")
print(f"均衡数量: Q* = {Q_star}")

# 比较静态分析：假设收入增加使需求变为 Qd' = 140 - 2P
def new_equilibrium():
    # 140 - 2P = -20 + 4P
    # 160 = 6P
    P_eq = 160 / 6
    Q_eq = 140 - 2 * P_eq
    return P_eq, Q_eq

P2, Q2 = new_equilibrium()
print(f"收入增加后: P* = {P2:.1f}, Q* = {Q2:.1f}")
print(f"结论: 需求增加 → 价格上涨 + 数量增加")

4.6 比较静态分析

比较静态分析是经济学家预测市场变化的基本工具：

  • 需求↑ + 供给不变 → 价格↑，数量↑
  • 需求↓ + 供给不变 → 价格↓，数量↓
  • 供给↑ + 需求不变 → 价格↓，数量↑
  • 供给↓ + 需求不变 → 价格↑，数量↓
  • 需求和供给同时变化 → 方向需具体分析
踩坑提醒：很多人在分析市场时，会不知不觉把"需求变动"和"需求量变动"混为一谈。做分析题时第一步要问自己：这个事件影响的是供给端还是需求端？它移动的是曲线，还是沿着曲线移动？记住这个区分，你就能避开90%的初学者陷阱。

4.7 本章小结

这一章我们搭建了微观经济学最核心的分析框架：供给-需求模型。这个模型看似简单，但极其强大——从口罩价格到房价、从猪肉到石油，几乎任何市场现象都可以用它来分析。

下一章，我们将在供给-需求基础上增加一个维度——弹性。弹性帮我们回答：降价到底能多卖多少？涨价到底能多赚多少？敬请期待第5章！
