第13章 广义逆矩阵与最小二乘

在中学数学中，逆矩阵的概念简单而精致：如果 $A$ 是 $n \times n$ 且可逆，那么 $A^{-1}$ 存在，且 $Ax = b$ 的解就是 $x = A^{-1}b$。

但现实世界中的矩阵往往是"长方形"的——样本数多于特征数（超定方程），或特征数多于样本数（欠定方程）。在这种情况下，逆矩阵不存在。那怎么办？答案就是广义逆矩阵（pseudoinverse）。

13.1 Moore-Penrose伪逆的四条公理

对任意矩阵 $A \in \mathbb{R}^{m \times n}$，它的 Moore-Penrose 伪逆 $A^+ \in \mathbb{R}^{n \times m}$ 是唯一满足以下四个条件（Penrose 条件）的矩阵：

  1. $A A^+ A = A$（伪逆是 $A$ 的"弱逆"）
  2. $A^+ A A^+ = A^+$（反过来也成立）
  3. $(A A^+)^T = A A^+$（$A A^+$ 是对称的）
  4. $(A^+ A)^T = A^+ A$（$A^+ A$ 也是对称的）

注意，前两个条件定义了广义逆的基本性质——它只需要在 $A$ 的值域上表现得像逆矩阵。后两个条件保证了伪逆在某种意义上是"唯一的"和"最小范数的"。

如果 $A$ 是 $m \times n$ 的列满秩矩阵（$m > n$, $\operatorname{rank}(A) = n$），伪逆有简单的公式：

$$A^+ = (A^T A)^{-1} A^T$$

如果 $A$ 是行满秩矩阵（$m < n$, $\operatorname{rank}(A) = m$），则：

$$A^+ = A^T (A A^T)^{-1}$$

当 $A$ 是方阵且可逆时，伪逆退化为通常的逆：$A^+ = A^{-1}$。

13.2 通过SVD构造伪逆

用 SVD 构造伪逆是最优雅也最通用的方法。给定 $A$ 的 SVD：

$$A = U \Sigma V^T$$

则伪逆为：

$$A^+ = V \Sigma^+ U^T$$

其中 $\Sigma^+$ 是通过将 $\Sigma$ 对角线上的非零奇异值取倒数并转置得到的：如果 $\Sigma = \operatorname{diag}(\sigma_1, \dots, \sigma_r, 0, \dots, 0)$，则 $\Sigma^+ = \operatorname{diag}(1/\sigma_1, \dots, 1/\sigma_r, 0, \dots, 0)$。（注意 $\Sigma$ 是 $m \times n$ 而 $\Sigma^+$ 是 $n \times m$。）

这个构造方法深刻地揭示了伪逆的几何意义：

  • $U^T$ 将向量投影到左奇异向量坐标系中
  • $\Sigma^+$ 在非零奇异值方向上做缩放（取倒数），在零奇异值方向上不做任何事情（置零）
  • $V$ 将结果映射回原始坐标

换句话说，伪逆通过 SVD 将"除以零"的问题替换为"忽略零方向"——对于没有信息的零空间方向，我们不传递任何信息。

import numpy as np

def pseudo_inverse_svd(A, rcond=1e-15):
    """使用SVD计算Moore-Penrose伪逆"""
    U, s, Vt = np.linalg.svd(A, full_matrices=False)
    # 截断过小的奇异值
    max_s = s[0]
    s_inv = np.array([1/si if si > rcond * max_s else 0 for si in s])
    return (Vt.T * s_inv) @ U.T

# 验证
A = np.array([[1, 2], [3, 4], [5, 6]], dtype=float)
A_pinv = pseudo_inverse_svd(A)
print("A+:\n", A_pinv)

# 与NumPy内置对比
print("numpy.linalg.pinv:\n", np.linalg.pinv(A))

# Penrose条件验证
print("Condition 1 (A A+ A = A):", np.allclose(A @ A_pinv @ A, A))
print("Condition 2 (A+ A A+ = A+):", np.allclose(A_pinv @ A @ A_pinv, A_pinv))
print("Condition 3 (A A+ symmetric):", np.allclose(A @ A_pinv, (A @ A_pinv).T))
print("Condition 4 (A+ A symmetric):", np.allclose(A_pinv @ A, (A_pinv @ A).T))

13.3 最小二乘与最小范数解的统一

伪逆最美妙的地方在于，它提供了一个统一框架来处理所有线性系统：

情形一：超定系统（$m > n$，且 $\operatorname{rank}(A) = n$）

方程数量多于未知数，通常没有精确解。我们寻求最小二乘解：使 $\|Ax - b\|_2$ 最小化的 $x$。伪逆给出的解：

$$x = A^+ b = (A^T A)^{-1} A^T b$$

这个解是唯一的，且等价于 QR 分解给出的解。

情形二：欠定系统（$m < n$，且 $\operatorname{rank}(A) = m$）

未知数多于方程数，有无穷多解。我们寻求最小范数解：在所有满足 $Ax = b$ 的解中，使 $\|x\|_2$ 最小的那个。伪逆给出的解：

$$x = A^+ b = A^T (A A^T)^{-1} b$$

情形三：秩亏系统（$\operatorname{rank}(A) < \min(m, n)$）

既有解空间的正交性丢失，又有零空间非平凡。伪逆自动给出：在所有最小二乘解中范数最小的那个。

统一结论：对任意线性系统 $Ax = b$，伪逆解 $x = A^+ b$ 是：
  1. 如果 $Ax = b$ 有解，则是其中 $\|x\|_2$ 最小的解
  2. 如果 $Ax = b$ 无解，则是使 $\|Ax - b\|_2$ 最小的解中 $\|x\|_2$ 最小的
换句话说，$A^+$ 统一了"最小二乘"和"最小范数"两个概念。

13.4 Tikhonov正则化：从伪逆到岭回归

伪逆虽然优美，但在处理病态问题时有一个致命问题：非常小的奇异值会导致 $1/\sigma_i$ 变得极大，从而放大噪声。一个 $\sigma_i = 10^{-12}$ 的奇异值会导致伪逆的相应方向放大了 $10^{12}$ 倍的输入噪声——这通常是灾难性的。

Tikhonov 正则化（统计学习中也叫岭回归，ridge regression）的解决方案是：不取 $1/\sigma$，而取 $\sigma / (\sigma^2 + \lambda)$。正则化的最小二乘问题为：

$$\min_x \|Ax - b\|_2^2 + \lambda \|x\|_2^2$$

其解为：

$$x_\lambda = (A^T A + \lambda I)^{-1} A^T b$$

如果 $A$ 的 SVD 是 $A = U \Sigma V^T$，则有简洁形式：

$$x_\lambda = \sum_{i=1}^r \frac{\sigma_i}{\sigma_i^2 + \lambda} (u_i^T b) v_i$$

对比伪逆的 $1/\sigma_i$ 和正则化的 $\sigma_i/(\sigma_i^2 + \lambda)$：

  • 当 $\sigma_i \gg \lambda$ 时，$\sigma_i/(\sigma_i^2 + \lambda) \approx 1/\sigma_i$，正则化几乎不影响大的奇异值
  • 当 $\sigma_i \ll \lambda$ 时，$\sigma_i/(\sigma_i^2 + \lambda) \approx \sigma_i/\lambda \to 0$，正则化"压制"了小的奇异值，抑制了噪声放大
import numpy as np
import matplotlib.pyplot as plt

# 构造一个病态问题
np.random.seed(42)
n = 50
# 生成奇异值呈指数衰减的矩阵
U, _ = np.linalg.qr(np.random.randn(n, n))
Vt, _ = np.linalg.qr(np.random.randn(n, n))
s = np.logspace(0, -12, n)  # 奇异值从 1 到 1e-12
A = U @ np.diag(s) @ Vt

x_true = np.random.randn(n)
b = A @ x_true + 1e-10 * np.random.randn(n)  # 微小噪声

# 伪逆解（数值上不稳定）
x_pinv = np.linalg.pinv(A) @ b
error_pinv = np.linalg.norm(x_pinv - x_true)

# 正则化解
lam = 1e-6
x_reg = np.linalg.solve(A.T @ A + lam * np.eye(n), A.T @ b)
error_reg = np.linalg.norm(x_reg - x_true)

print(f"伪逆误差: {error_pinv:.4f}")
print(f"正则化误差: {error_reg:.4f}")
正则化参数 $\lambda$ 的选择：

$\lambda$ 过小则正则化效果不足，过大则引入了太多偏差。常用的选择方法：

  • L-curve（L曲线）：在不同的 $\lambda$ 下画 $\|Ax_\lambda - b\|$ 与 $\|x_\lambda\|$ 的曲线，取拐点
  • 交叉验证：将数据划分为训练集和验证集，选择验证误差最小的 $\lambda$
  • 广义交叉验证（GCV）：不需要划分数据，直接计算 GCV 得分
在实际应用中，简单尝试 $\lambda = 10^{-6}, 10^{-4}, 10^{-2}$ 往往就能看到显著改善。
踩坑实录：正规方程 $A^T A$ 的数值灾难

我曾见过一个物理实验数据分析的项目：研究人员有 200 个样本，30 个特征，想通过线性回归拟合一个统计模型。他用正规方程 $(A^T A)^{-1} A^T b$ 求解。结果完全无法物理意义——回归系数的符号与实验预期相反。

问题出在哪？数据中有三个特征强相关（相关系数超过 0.999），导致 $A^T A$ 的条件数高达 $10^{12}$。正规方程中的 $(A^T A)^{-1}$ 计算完全被舍入误差淹没。

解决方案：改用 SVD 伪逆（或者直接加一个小的 $\lambda$ 做岭回归）。结果系数符号正确，物理意义清晰。

这个教训是统计计算中的经典案例：永远不要用正规方程解病态的最小二乘问题。用 SVD、QR 或者加正则化。在机器学习中，这就是为什么 $n \ll p$ 的高维回归需要岭回归、Lasso 或弹性网。

本章小结

广义逆矩阵将逆矩阵的概念从方阵推广到任意矩阵。SVD 提供了构造伪逆的最优雅途径，而伪逆统一了超定和欠定系统的求解。Tikhonov 正则化在伪逆的基础上增加了一个"平滑参数"，在数值稳定性和解的质量之间取得了平衡——这在现代机器学习的语境中就是岭回归。
