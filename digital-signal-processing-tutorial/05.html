第5章 Z变换

想象一下，你要判断一个差分方程描述的滤波器是稳定的，或者你要分析一个系统的频率响应——在时域里掰扯半天可能会非常痛苦。有没有一种方法可以把这些操作"代数化"？答案是：Z变换。

Z变换是离散时间信号处理的"复频域显微镜"。它把时域的序列映射到复平面（z平面）上，让卷积变成乘法、差分方程变成代数方程、稳定性判断变成看零极点位置——一切变得井井有条。毫不夸张地说，Z变换和傅里叶变换是DSP工程师最重要的两个数学工具。

5.1 Z变换的定义与物理意义

双边Z变换的定义是：

        X(z) = Σ_{n=-∞}^{∞} x[n] · z^{-n}

        其中 z = re^{jω} 是一个复变量：
        - r 是幅度半径 (|z|)
        - ω 是数字角频率 (相位)

当 r = 1（即 |z| = 1），z = e^{jω}，此时Z变换退化为离散时间傅里叶变换（DTFT）。所以Z变换是DTFT的推广——它将频率分析从单位圆延伸到了整个复平面。

💡 直觉理解：Z变换的本质是"把序列看成一系列指数信号的加权和"。每个z值对应一个复指数信号 zⁿ = rⁿ·e^{jωn}，而X(z)度量的是序列与这个复指数的"相似度"。用这个视角来看待信号，许多操作的本质就一清二楚了。

Z变换本身是一个幂级数（Laurent级数）。和所有级数一样，它只在一个特定的z值范围内收敛——这个范围就是收敛域（Region of Convergence, ROC）。

5.2 收敛域（ROC）：不说ROC就是耍流氓

这是一个初学Z变换最容易忽视的问题。同一个X(z)表达式，不同的ROC对应的序列完全不同。我们来看一个经典例子：

        X(z) = 1 / (1 - az^{-1}) = z / (z - a)

        如果 ROC 是 |z| > |a| → 右边序列：x[n] = aⁿ·u[n]
        如果 ROC 是 |z| < |a| → 左边序列：x[n] = -aⁿ·u[-n-1]

        看到没有？同样的X(z)，ROC不同，逆变换结果天差地别！

不同序列类型的ROC特征总结如下：

序列类型 ROC特征 例子
右边序列 |z| > R（圆外区域），R为最远极点半径 因果序列 aⁿ·u[n] → |z| > |a|
左边序列 |z| < R（圆内区域），R为最近极点半径 反因果序列 -aⁿ·u[-n-1] → |z| < |a|
双边序列 环形区域 R₁ < |z| < R₂ x[n] = aⁿ·u[n] - bⁿ·u[-n-1] → |a| < |z| < |b|
有限长序列 除0和∞外的整个z平面 x[n]只在有限区间非零
⚠️ 踩坑实录：早年在做一个数字均衡器（graphic equalizer）时，我设计了一组IIR滤波器，直接读取H(z)的系数做实现。测试时发现低频段完全错误——信号发散。排查了整整两天，最后发现是滤波器的一个极点在单位圆外（|p| ≈ 1.01），系统不稳定。为什么设计时没发现？因为我在MATLAB中用零极点图看了，但有一个极点的实部转成系数时被四舍五入到了错误的方向。从那以后我养成了一个习惯：每次设计完滤波器，第一件事就是用zplane()画零极点图，然后手动检查每个极点的模值是否小于1。

5.3 Z变换的重要性质

Z变换之所以强大，是因为它能将时域的操作转化为z域的简单代数运算：

性质 时域 Z域 ROC变化
线性 a·x₁[n] + b·x₂[n] a·X₁(z) + b·X₂(z) ROC交集
时移 x[n - k] z^{-k}·X(z) ROC不变（可能增删0或∞）
卷积定理 x[n] * h[n] X(z)·H(z) ROC交集
频移（调制） z₀ⁿ·x[n] X(z/z₀) ROC缩放|z₀|倍
时间反转 x[-n] X(1/z) ROC取倒数
初值定理 x[0] = lim_{z→∞} X(z) （因果序列可用） -
终值定理 lim_{n→∞} x[n] = lim_{z→1} (1-z^{-1})X(z) （需收敛） -

其中最最重要的就是卷积定理：时域卷积 = z域乘积。有了这条性质，你就不再需要做繁琐的卷积求和了——在z域做一次多项式乘法，再逆变换回去，完事。这也是后面用FFT加速卷积的理论基础（z域乘积 → 频域乘积 → FFT）。

5.4 逆Z变换：三种方法

知道了X(z)想要找回x[n]？有三种方法。按"需要的手工程度"从高到低排列：

① 留数法（围线积分法）——最正统但最复杂

        x[n] = (1/2πj) · ∮ X(z) · z^{n-1} dz

        使用留数定理计算围线积分。
        理论意义大于实际工程意义——大部分时候我们不用它。

② 部分分式展开法——最常用

将X(z)展开成简单有理分式的和，每个分式对应一个基本序列的Z变换：

        例如将 X(z) = (1 + 2z^{-1}) / (1 - 0.5z^{-1})(1 - 0.3z^{-1})
        展开为：
        X(z) = A/(1-0.5z^{-1}) + B/(1-0.3z^{-1})

        求A、B后，查表得：
        x[n] = A·(0.5)ⁿ·u[n] + B·(0.3)ⁿ·u[n]   （若ROC为|z|>0.5）

③ 长除法（幂级数展开法）——直观但容易出错

直接用多项式除法把X(z)展开成z的幂级数，然后直接读出x[n]的系数。适合求前几项，但不适合求封闭形式。

        X(z) = 1 / (1 - 0.5z^{-1})

        做除法：1 + 0.5z^{-1} + 0.25z^{-2} + 0.125z^{-3} + ...

        所以 x[0]=1, x[1]=0.5, x[2]=0.25, x[3]=0.125, ...
        即 x[n] = (0.5)ⁿ·u[n]
⌨️ 工程实践：在实际的DSP开发中，你几乎不会手动计算逆Z变换——MATLAB的 iztrans() 或 residuez() 函数会帮你做。但理解三种方法的原理，能让你在调试时快速判断结果是否合理。

5.5 系统函数 H(z) 与零极点分析

现在我们把Z变换应用到系统分析上。对于一个由差分方程描述的LTI系统：

        Σ_{k=0}^{N} a_k · y[n-k] = Σ_{k=0}^{M} b_k · x[n-k]

        两边做Z变换（利用时移性质）：
        Y(z) · Σ a_k·z^{-k} = X(z) · Σ b_k·z^{-k}

        系统函数：
        H(z) = Y(z)/X(z) = (Σ b_k·z^{-k}) / (Σ a_k·z^{-k})

        因式分解形式：
        H(z) = G · (Π (z - q_k)) / (Π (z - p_k))

        其中 q_k 是零点（H(z)=0），p_k 是极点（H(z)→∞）

零极点图就是把这些零点和极点画在复平面上的一张图。在一张图上你可以瞬间读出系统的大量信息：

  • 稳定性：所有极点必须在单位圆内（|p_k| < 1）。只要有任何一个极点在单位圆上或圆外，系统就是不稳定或临界稳定的。
  • 因果性：如果系统是因果的，ROC必须是圆外区域 |z| > max|p_k|。
  • 幅频响应大致形状：靠近单位圆的零点对应"陷波"（频谱的谷），靠近单位圆的极点对应"谐振峰"（频谱的峰）。零极点的角度决定了频率位置。
💡 零极点速成法：假设你看到一个零极点图，想快速判断滤波器类型：靠近z=1的极点 → 低通；靠近z=-1的极点 → 高通；共轭极点对在单位圆内靠近e^{±jω₀} → 带通/带阻（峰值在ω₀处）。这就是可视化零极点分析的价值——不用算任何数学公式，一眼看出系统特性。

从系统函数到频率响应只需要一步：令 z = e^{jω}（在单位圆上采样），就得到了频率响应 H(e^{jω})。这正是下一章的主题。

5.6 本章小结

  • 理解了Z变换的定义和物理意义——它是DTFT在复平面的推广，z = re^{jω}；
  • 掌握了ROC的四种形态（右边、左边、双边、有限长）及其与序列类型的对应关系，理解了"不说ROC就是耍流氓"的深刻含义；
  • 学会了Z变换的主要性质，重点掌握卷积定理——时域卷积 = z域乘积；
  • 掌握了三种逆Z变换方法：留数法、部分分式展开法、长除法；
  • 理解了系统函数H(z)和零极点图，以及如何从零极点判断稳定性、因果性和频率响应形状。

Z变换把我们从时域的"蛮力运算"中解放了出来，但真正的频率分析旅程才刚刚开始。下一章频域分析，我们要系统梳理四大傅里叶变换的家族关系，并重点学习DTFT——它是连接Z域和频域的桥梁，也是理解DFT和FFT的必经之路。
