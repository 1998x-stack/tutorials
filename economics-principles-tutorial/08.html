8.1 预算约束线：你能买什么

假设你每个月有5000元可支配收入，只用来买两样东西：聚餐（每顿100元）和买书（每本50元）。那么你能买多少？这取决于你在这两样之间的分配。如果你全用来聚餐，可以吃50顿；全用来买书，可以买100本。如果各花一半，就是25顿+50本。经济学用一条简洁的直线来描述这种权衡——预算约束线（Budget Constraint）。

预算约束线的一般形式是：P₁X₁ + P₂X₂ = I，其中P是价格，X是数量，I是收入。它的斜率为 -P₁/P₂，表示为了多消费一单位商品1，必须放弃多少单位商品2。当收入增加时，整条线向外平移；当某种商品的价格变化时，线的斜率发生旋转。

💡 思考题：如果聚餐价格从100元涨到120元，而你的收入不变，预算约束线会发生什么变化？答案是它会绕着纵轴截距（买书的数量）向内旋转——因为买书的价格没变。

8.2 无差异曲线：你想要什么

预算约束线告诉我们"能买什么"，而无差异曲线（Indifference Curve）告诉我们"想要什么"。无差异曲线是消费者偏好（Preferences）的可视化表达——同一条曲线上的所有消费组合带来的满足程度（效用）完全相同。

无差异曲线有三个关键性质：

  • 向右下倾斜：如果少消费一种商品，必须多消费另一种才能保持效用不变
  • 凸向原点：边际替代率递减——当你已经有很多书时，你愿意用很多书换一顿饭；但当你没饭吃时，你就不愿意换了
  • 永不相交：如果两条无差异曲线相交，逻辑上会产生矛盾的偏好排序

边际替代率（Marginal Rate of Substitution, MRS）衡量消费者愿意用一种商品替代另一种商品的比例。在图形上，它就是无差异曲线上某一点的斜率的绝对值。MRS递减意味着随着X消费量增加，每多消费一单位X愿意放弃的Y越来越少——这完全符合直觉。

8.3 消费者最优选择：切点条件

把预算约束线和无差异曲线放在一起，消费者的最优选择就在它们相切的那一点。在这一点上：MRS = P₁/P₂，即主观的边际替代率等于客观的价格比率。换句话说，消费者对两种商品的"内心评价"恰好等于市场的"价格评价"。

用数学语言来说，消费者在约束条件下最大化效用函数U(X,Y)，拉格朗日乘子法是最标准的解法：

# 消费者效用最大化（拉格朗日乘子法）
# 最大化 U(X,Y) = X^α * Y^(1-α)
# 约束条件 Px*X + Py*Y = I

from sympy import symbols, diff, solve

X, Y, λ, Px, Py, I, α = symbols('X Y λ Px Py I α')

# 拉格朗日函数
L = X**α * Y**(1-α) - λ * (Px*X + Py*Y - I)

# 一阶条件
dL_dX = diff(L, X)  # α * X^(α-1) * Y^(1-α) - λ*Px = 0
dL_dY = diff(L, Y)  # (1-α) * X^α * Y^(-α) - λ*Py = 0
dL_dλ = diff(L, λ)  # -(Px*X + Py*Y - I) = 0

# 解得最优消费量
# 从一阶条件可得: MRS = (α/(1-α)) * (Y/X) = Px/Py
# 最优解: X* = α*I/Px,  Y* = (1-α)*I/Py

print(f"最优X消费量: X* = {α}*I/Px")
print(f"最优Y消费量: Y* = (1-{α})*I/Py")

这个结果的直觉含义很漂亮：消费者将收入的α比例花在X商品上，将(1-α)花在Y商品上，α恰好是效用函数中X的指数——也就是消费者对X的"偏好强度"。

8.4 收入效应与替代效应：涨价之后你变了多少？

当一种商品的价格下降时，消费者会多买它——这我们都知道。但经济学家要追问：这个"多买"中有多少是因为它相对变便宜了（替代效应），有多少是因为你的实际购买力提高了（收入效应）？

替代效应（Substitution Effect）：价格下降使该商品相对于其他商品更便宜，消费者会用它替换其他商品。这是沿着无差异曲线的移动。

收入效应（Income Effect）：价格下降使你的实际收入增加，从而改变你的消费组合。这是无差异曲线层级的移动。

商品类型价格下降时的替代效应价格下降时的收入效应总效应
正常品（Normal Good）消费量增加消费量增加消费量增加
低档品（Inferior Good）消费量增加消费量减少（但替代效应更大）消费量增加
吉芬品（Giffen Good）消费量增加消费量减少（且收入效应超过替代效应）消费量减少

注意吉芬品（Giffen Good）是一个极端情况——价格下降反而导致需求减少。这在现实中极少见，最著名的例子是爱尔兰马铃薯饥荒时期：马铃薯价格上涨，但穷人买不起肉，反而被迫买更多马铃薯（因为马铃薯仍然是他们最便宜的卡路里来源），因此马铃薯的需求量反而上升了。不过要注意，吉芬品和"奢侈品"不是同一个概念——吉芬品一定是低档品，但低档品不一定是吉芬品。

8.5 行为经济学：为什么双十一你会买一堆用不着的东西

传统的消费者选择理论假设人是理性的——我们有明确的偏好、掌握完美信息、随时能做最优决策。但行为经济学研究发现，真实人类的行为和"理性人"假设之间存在着系统性的偏差。

想想你今年双十一做了什么。满200减30、跨店满减、预付定金翻倍、限时秒杀……这一套组合拳下来，你可能买了三箱卫生纸、一台永远不会用的空气炸锅、以及五件穿了一次就再也没碰过的衣服。事后你问自己：我当时到底在想什么？

行为经济学家从以下几个角度解释了这类现象：

  • 有限理性（Bounded Rationality）：人脑的处理能力有限，面对复杂计算时会走"捷径"（启发式），而这些捷径有时会带来系统性偏误。比如"锚定效应"——原价399现价199让你觉得很划算，但399本身可能就是个随意定的价。
  • 心理账户（Mental Accounting）：消费者在心里把钱分到不同的"账户"里。"意外之财"会被轻易花掉，但"辛苦赚来的钱"会更谨慎。双十一的"满减"就是利用了"反正已经省了"的心理账户错觉。
  • 现状偏向（Status Quo Bias）：人们倾向于保持当前的选择。这就是为什么免费试用期结束后，很多人会继续付费订阅——因为取消订阅需要"做出改变"。
🧠 行为经济学小贴士：理性消费者模型并不"错误"——它提供了一个有用的基准。就像物理学先从真空中自由落体学起，然后才引入空气阻力。行为经济学不是要推翻消费者选择理论，而是丰富它、修正它，让模型更接近真实世界。

8.6 从消费者到生产者

消费者选择理论是微观经济学的基石——它告诉我们需求曲线为什么向右下倾斜，价格变化如何影响消费行为，以及理性假设在什么情况下会失效。从下一章开始，我们将视角从消费者转向生产者。企业如何决定生产多少？成本结构如何影响供给决策？这就是第9章要讲的内容——生产成本。
