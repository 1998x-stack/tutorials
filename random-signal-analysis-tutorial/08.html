开篇：信号的时间平移不变性

大家好，我是你们的讲师。在上一章中，我们学习了随机信号的相关函数与微积分运算。那时候我们注意到一个有趣的问题：随机信号的自相关函数 R(t₁, t₂) 依赖于两个时间点，这让分析变得相当复杂——如果信号有 1000 个采样点，理论上我们需要计算 1000×1000 个相关值。有没有什么简化方法？

答案就是平稳性（stationarity）。如果一个随机信号的统计特性不随时间原点的平移而改变，那么它的均值就是一个常数，自相关函数只依赖于两个时间点之间的差 τ = t₂ - t₁。这样一来，信号分析的复杂度从 O(N²) 降到了 O(N)，而且频域分析（下一章的功率谱密度）才有了坚实的数学基础。平稳性看似是一个很强的假设，但它在工程中却是最常用的出发点——就像物理学中先假设无摩擦再引入摩擦力一样合理。

平稳性：时间的平移不变性

直觉理解

考虑这样一个场景：你在一台稳定的机器旁边测量振动信号。如果你今天测量一小时和明天同一时间测量一小时，数据的统计特性（均值、方差、相关性结构）应该是相似的，这就是平稳性的直觉——信号的统计特性不随时间原点改变。换句话说，系统已经进入了"稳态"，没有了启动瞬态或趋势变化。

用数学语言来说：对于任意的时间偏移 τ₀，如果随机信号 X(t) 的统计特性与 X(t + τ₀) 完全相同，那么 X(t) 就是平稳的。注意，这里说的是"统计特性"相同，而不是样本函数相同——单个样本当然在随时间变化，但它的"统计规律"是不变的。

为了让你更直观地理解，想象你在海边观察海浪。虽然每一个波浪的高度、形状都不同，但你观察一小时和观察下一小时的波高分布应该是基本一致的——这就是海浪的平稳性。相反，如果你从早上八点到九点观察海面（涨潮），波高的均值从 1 米逐渐升到 2 米，这就不是平稳的。

关键区分："平稳"不等于"没有波动"。平稳信号可以有剧烈的波动，只要这种波动的统计规律不变。白噪声是平稳的（均值 0、方差 σ² 恒定），但它仍然在剧烈振荡。另一角度：非平稳信号的统计特性是时间 t 的函数，而平稳信号的统计特性只依赖于时间差 τ。

严平稳（SSS）

定义与数学表述

严格平稳（Strict-Sense Stationary, SSS）的定义非常直接：对于任意正整数 n、任意时间点 t₁, t₂, ..., tₙ 以及任意时间偏移 τ₀，随机信号 X(t) 的 n 维联合分布满足：

F_X(x₁,...,xₙ; t₁,...,tₙ) = F_X(x₁,...,xₙ; t₁+τ₀,...,tₙ+τ₀)

也就是说：信号的任意有限维分布在时间平移下不变。这意味着什么呢？一阶分布不依赖于 t（均值恒定），二阶分布只依赖于时间差 τ（自相关只依赖于 τ），三阶、四阶……所有阶的分布都有同样的平移不变性。

严平稳是一个极其完整的平稳概念——它要求信号的所有统计特性都是时不变的。但正因为这个定义太过完美，在工程实践中几乎无法验证。你能想象自己需要验证一个信号的所有有限维分布在时间平移下都不变吗？这就像要检查一座房子的每一块砖头——理论上是这样，实际上你只能抽样检查。

理解要点：SSS 是理论家喜爱的定义，因为它给出了一个干净的数学框架。但对工程师来说，SSS 的主要价值在于——它告诉我们在什么意义上"信号可以无限观测"：如果你有一个 SSS 信号，你可以在任何一个时间段观测它，得到的统计结论都可以推广到其他时间段。不过在实践中，我们几乎从不检查 SSS，而是使用更方便的 WSS。

宽平稳（WSS）

定义与工程意义

宽平稳（Wide-Sense Stationary, WSS）是工程中最常用的平稳定义。它的条件比 SSS 弱得多，只涉及前两阶矩：

  • 均值恒定：E[X(t)] = μ_X，是一个与时间 t 无关的常数。
  • 方差有限：E[|X(t)|²] < ∞，信号的总功率有限。
  • 自相关只依赖于时延：R_X(t₁, t₂) = R_X(t₂ - t₁) = R_X(τ)，其中 τ = t₂ - t₁。

这三个条件就是 WSS 的全部要求。注意它不要求高阶矩的平稳性，也不要求分布的具体形式——这使得 WSS 几乎成了所有实际信号处理算法的基础假设。

为什么 WSS 如此重要？因为这三个条件恰好是功率谱密度存在所需要的。我们下一章将要学习的维纳-辛钦定理（Wiener-Khinchin Theorem）告诉我们：WSS 信号的自相关函数与功率谱密度构成一对傅里叶变换对。如果信号不满足 WSS，这个关系就不成立，频域分析就失去了根基。

高斯情况：SSS ⇔ WSS

有一个非常重要的特例：如果 X(t) 是高斯随机过程，那么 WSS 等价于 SSS。为什么？因为高斯过程的任意有限维分布完全由均值向量和协方差矩阵决定。如果一阶矩和二阶矩是时不变的，那么所有高阶矩（它们都由前两阶矩确定）也一定是时不变的。换句话说，对高斯信号而言，"宽平稳"自动意味着"严平稳"。

这个性质在实际工作中非常有用：你只需要确认一个高斯信号的均值和自相关满足 WSS 条件，就可以放心地把它当作严平稳信号来处理，不需要担心高阶矩的问题。

注意：这个等价关系只对高斯过程成立。对于非高斯信号，WSS 完全不保证 SSS。例如一个 WSS 的二元信号（取值 ±1），它的三阶矩可能随时间变化，但二阶矩却不変。不要因为方便就默认非高斯信号的 WSS 等价于 SSS！

联合平稳与循环平稳

联合平稳（Jointly WSS）

在实际工程中，我们经常同时处理多个随机信号——例如一个通信系统同时有发送信号和接收信号，一个控制系统同时有参考信号和误差信号。这时候除了各自的平稳性，还需要考虑两个信号之间的互相关特性。

两个随机信号 X(t) 和 Y(t) 被称为联合宽平稳，如果它们各自都是 WSS 的，并且它们的互相关函数只依赖于时间差 τ：

R_{XY}(t₁, t₂) = E[X(t₁) Y(t₂)] = R_{XY}(τ)，其中 τ = t₂ - t₁。

联合平稳性在系统分析中至关重要。回想一下第 12 章将要讲的内容：当一个随机信号通过线性系统时，输入输出之间的互相关函数反映了系统对信号的影响。如果输入是 WSS 的且系统是线性时不变的（LTI），那么输入输出就是联合平稳的——这个性质让我们可以用简单的卷积运算来分析系统。

循环平稳（Cyclostationary）

现实世界中，很多随机信号的统计特性并不是恒定的，而是呈现周期性变化。最典型的例子就是数字通信中的调制信号。想象一下 BPSK 调制：信号的符号以固定的符号周期 T_s 切换，每个符号的幅度可以建模为一个随机变量。这时信号的统计特性不是时不变的，而是以 T_s 为周期变化的——这就叫做循环平稳。

循环平稳信号的数学定义：均值 m_X(t) 和自相关 R_X(t₁, t₂) 对时间偏移具有周期性，即存在 T > 0 使得：

m_X(t + T) = m_X(t) 且 R_X(t₁ + T, t₂ + T) = R_X(t₁, t₂)

循环平稳分析在工程中有重要的应用：在认知无线电（Cognitive Radio）中，可以利用信号的循环平稳特性来检测是否存在主用户信号——因为噪声通常不是循环平稳的，而通信信号是。这种"循环平稳检测"比传统的能量检测在低信噪比下表现更好。

实用技巧：处理循环平稳信号的一个常用技巧是"循环平均"——在每个周期内的相同相位位置做平均，就可以把循环平稳信号转换为平稳信号。这本质上就是接收机中"符号同步"和"匹配滤波"的数学基础。

如何判断信号是否平稳？

在工程实践中，判断平稳性通常不是通过严格的统计检验，而是通过领域知识和对信号产生机制的理解：

  • 物理系统：如果系统已经处于稳态（没有启动瞬态、没有时变参数），通常可以假设 WSS。
  • 趋势检验：检查信号是否有明显的趋势项（如线性增长或衰减）。有趋势项 → 非平稳。
  • 分段统计：将信号分成多段，比较每段的均值和方差。如果变化显著 → 怀疑非平稳。
  • 时间-频率分析：用短时傅里叶变换（STFT）观察频谱随时间的变化。稳定 → 可近似为 WSS。

踩坑故事：为非平稳信号穿上"平稳的外衣"

讲一个我在研究生阶段参与的课题组项目。师兄毕业论文的核心内容是用功率谱估计来分析某种脑电信号（EEG）的频域特征，进而推断不同认知任务下的大脑活跃区域。EEG 信号被分成 10 秒的片段，每一段都被假定为 WSS，然后直接用周期图法估计功率谱。

实验结果在组会上展示时看起来很不错——不同任务下的 α 波（8-12 Hz）和 β 波（13-30 Hz）的功率比例出现了明显的差异，似乎验证了实验假设。然而一位审稿人提出了尖锐的问题："你们的 EEG 数据做过平稳性检验吗？"

师兄当时就懵了——没人做过这个检验。在审稿人的要求下，我们重新审视了数据。用分段方差检验发现，很多 10 秒片段内的 EEG 信号并非平稳：受试者在实验过程中可能出现眨眼、肌肉紧张、注意力波动等非平稳因素，这些都会显著改变信号的能量分布。更有意思的是，之前观察到的"显著差异"中，有一部分根本不是认知任务导致的，而是受试者在不同任务阶段的疲劳程度不同造成的——也就是说，我们一直在把非平稳信号的功率谱变化错误地归因于实验变量。

教训：把非平稳信号当作平稳信号做功率谱估计，得到的结果可能完全不可靠。功率谱密度的数学定义依赖于平稳性假设，如果不验证这个假设就直接用 FFT 做谱分析，相当于在流沙上盖房子。在工程实践中，遇到未知信号时请首先问三个问题：(1) 信号的产生机制是否支持平稳性？(2) 是否有趋势项或周期性成分？(3) 分段时间窗是否足够短以保证近似平稳？记住：平稳性不是一个可以"默认"的假设，而是一个需要你主动验证的前提条件。

本章小结

这一章我们学习了随机信号分析中最重要的概念之一——平稳性。严平稳（SSS）要求所有有限维分布在时间平移下不变，是理论上的黄金标准；宽平稳（WSS）只要求均值和自相关满足时不变条件，是工程实践中最实用的假设。我们还讨论了高斯过程中 SSS 与 WSS 的等价性、联合平稳的处理方法以及循环平稳这个更贴近工程实际的概念。最后通过一个 EEG 信号分析的真实案例，警示了忽视平稳性假设可能带来的严重后果。

下一章我们将进入随机信号频域分析的核心——功率谱密度与维纳-辛钦定理。你会发现，WSS 假设的价值在频域中得到了最充分的体现：正是由于自相关只依赖于 τ，它才能与功率谱密度构成傅里叶变换对。请记住：平稳性是"桥"，而维纳-辛钦定理是那座桥上最重要的通行证。
