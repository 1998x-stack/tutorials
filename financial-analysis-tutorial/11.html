第11章 相对估值法

大家好，我是你们的讲师。上一章我们深入研究了DCF——估值理论中最严谨的"重型武器"。但坦率地说，在真实的研究工作中，DCF的使用频率远不如另一种方法：相对估值法（Relative Valuation）。

打开任何一份券商研报，你大概率会看到一张"可比公司估值表"——列出几家同行业的公司，对比它们的PE、PB、PS等乘数，然后给出目标公司的合理估值区间。这就是相对估值法的核心逻辑：资产的价值不是孤立存在的，而是通过与其他可比资产的比较来确定的。

相对估值法之所以如此流行，原因有三：第一，它简单直观——PE=15倍比WACC=9.47%好理解多了；第二，它贴近市场——市场本身就是通过比较来定价的；第三，它容易沟通——跟客户说"这公司的PE比同行低30%"比解释DCF模型轻松得多。但简单不等于容易——用好相对估值法，你需要深刻理解每个乘数的含义和局限。

11.1 市盈率PE深度解析

PE（Price to Earnings Ratio，市盈率）是使用最广泛、也最容易被误解的估值指标。它的公式简单得不能再简单：

PE = 股价 / 每股收益（或市值 / 净利润）

PE的含义直观来说是"回本年限"——如果一家公司每年赚1块钱，你以15块的股价买入，需要15年回本（假设利润不变）。当然现实中利润会变化，所以PE只是一个估值参考。

三种PE口径

PE类型 分母（每股收益E） 优点 缺点
静态PE 最近一个完整财年的实际利润 数据确定，无可争议 可能过时（去年的利润不能代表现在）
滚动PE（TTM PE） 最近四个季度利润之和 时效性最好，反映最新业绩 受季节性因素干扰
前瞻PE（Forward PE） 分析师对未来12个月的利润预测 前瞻性，反映市场预期 预测可能不准，偏差大

一般来说，滚动PE（TTM PE）是默认选项——它既避免了静态PE的滞后性，又不像前瞻PE那样依赖不确定的预测。但如果你发现一家公司的滚动PE突然飙升，要问的第一个问题是：是股价涨了（估值扩张），还是利润跌了（盈利下降）？这两种情况对应完全不同的信号。

PE的适用与不适用场景

PE最好用在盈利稳定、可预测性强的公司——消费品牌、医药、公用事业等。但在以下场景中PE会严重失真：

  • 亏损公司——PE为负数，没有意义。
  • 强周期性公司——钢铁、化工在周期顶部利润极高导致PE极低（看似"便宜"），但恰恰是卖出时机；周期底部亏损PE无限大（看似"贵"），反而是买入时机。
  • 高增长公司——高PE可能被高增长所消化，需要用PEG来做修正（见11.4节）。
  • 一次性收益/损失——出售资产、重组费用等非经常性项目会扭曲利润。需要调整"经调整净利润"来计算PE。
踩坑警示：2021年初，某明星分析师在研报中宣称某白酒公司"PE仅为30倍，历史低位，强烈买入"。但实际情况是——公司的利润由于2020年特殊时期的压货政策被人为拔高，TTM PE看似"低"是因为分母（利润）处于非正常高位。当2021年渠道库存消化、利润回归正常后，实际PE瞬间跳到50倍以上。这就是典型的"利润周期顶部的低PE陷阱"——看起来便宜，其实是在周期或利润的顶峰。

11.2 PB / PS / EV乘数

除了PE之外，相对估值法中还有几个重要的乘数，各自对应不同的适用场景。

乘数 公式 含义 主要适用行业 注意点
PB 市值 / 净资产 每元净资产的市场定价 银行、保险、房地产 净资产含商誉需调整；轻资产公司不适用
PS 市值 / 营业收入 每元营收的市场定价 亏损公司、成长早期公司 不能反映盈利能力和成本结构
EV/EBITDA 企业价值 / 息税折旧摊销前利润 不受资本结构影响的经营回报倍数 重资产、高折旧行业 分子用EV而非市值，消除了杠杆影响
EV/EBIT 企业价值 / 息税前利润 比EV/EBITDA更保守（扣除折旧） 资本密集度中等的制造业 比EV/EBITDA更准确地反映维护性资本支出
EV/Sales 企业价值 / 营业收入 PS的"企业价值版" 收购兼并场景 类似PS，但考虑了杠杆差异

关于PB有一个非常重要的关系——ROE-PB框架。理论上，一家公司的合理PB应该由它的ROE和股权成本决定：

合理PB = (ROE - g) / (Ke - g)

这个公式告诉我们一个关键逻辑：ROE越高，PB应该越高；股权成本越低，PB越高。如果你发现一家公司ROE很高但PB很低——那可能就是被低估的信号。

11.3 可比公司选择

相对估值法的核心操作是"找可比公司"（Comparable Companies / Comps）。但"可比"不是随便找几家同行业公司就行。严格来说，可比公司应该在以下几个维度上相似：

  • 行业与业务模式——同行业是基础，但还不够。比如同样是"互联网公司"，腾讯（社交+游戏+广告+金融）和拼多多（电商）之间的可比性就非常有限。
  • 规模——大公司和小公司不应该直接比较PE。头部企业通常享受流动性溢价和确定性溢价，PE可以比小公司高出不少。
  • 增长率——增长快的公司理应享受更高的估值倍数。这就是PEG存在的原因。
  • 资本结构——杠杆不同的公司，用PE比较会失真。这也是EV/EBITDA乘数存在的意义——它排除杠杆的影响。
实操建议：建立一个Comps表时，除了列出每家公司的PE、PB、PS等估值乘数，还应该列出关键的财务特征——收入增长率、毛利率、净利率、ROE、资产负债率、市值规模。只有充分理解了每家公司的"基本面画像"，才能判断估值的差异是合理的（因为基本面不同）还是不合理的（可能存在定价错误）。通常建议选择6-12家可比公司，太少则样本不足，太多则可能混入了不具可比性的公司。

11.4 PEG与估值修正

PE的最大缺陷是它没有考虑增长。一家PE=30倍但每年增长40%的公司，和一家PE=10倍但零增长的公司，哪个更"贵"？直觉上，前者可能更"便宜"——如果它能维持高增长，几年后利润增长几倍，30倍的PE很快就会被消化。

为了解决这个问题，彼得·林奇（传奇基金经理，麦哲伦基金13年29%年化回报）推广了PEG指标：

PEG = PE / 增长率

其中增长率通常用未来3-5年的盈利增长率预测（取整数百分比，如20%增长用20）。林奇的规则是：

  • PEG < 1 —— 被低估，买入信号
  • PEG ≈ 1 —— 估值合理
  • PEG > 1.5 —— 高估，谨慎

例如：某科技公司PE=40倍，分析师预计未来3年利润年增长30%，则PEG=40/30=1.33——略高于1，可能略微高估但不算离谱。而一家零售公司PE=12倍，增长率5%，PEG=12/5=2.4——看起来PE很低，但相对于增长其实很"贵"。

核心洞见：PEG是一个非常实用的"快捷"指标，但它有一个暗含的假设——增长率是可持续的。如果一家公司的高增长是靠补贴用户、烧钱换规模实现的，未来增长随时可能断崖式下跌，那么当前的PEG再低也不应该买入。永远记住：任何估值指标的价值，都不会超过你所依赖的假设的准确度。

一个值得反思的泡沫故事

2020-2021年的A股"核心资产"行情，是理解相对估值法局限性的绝佳案例。当时的市场逻辑是这样的：这些公司（茅台、恒瑞、海天、美的等）是"核心资产"，确定性高、ROE高，应该享受"确定性溢价"——于是市场把它们的PE推高到了前所未有水平。茅台TTM PE在2021年2月达到70倍，恒瑞医药达到100倍，海天味业超过100倍。

从相对估值角度看，这些公司跟自己的历史PE比，已经到了历史最高分位（>99%）。但当时市场上流行的论调是："这次不一样——核心资产是稀缺的，高估值可以用增长消化。"最终的结果呢？从2021年高点算起，茅台最大回撤约45%，恒瑞医药回撤超过60%，海天味业回撤超过65%。

这个故事的教训很深刻：相对估值法无法帮助你判断市场是否整体过热。当所有可比公司都处在历史高位时，比较本身就没有"锚"了。这就是为什么专业投资者即使主要用相对估值法，也会定期用DCF来"校准"——确保自己没有在潮水退去时还在裸泳。

以下是用Python获取A股数据并计算PE/PEG的示例：

import akshare as ak
import pandas as pd

# 获取白酒行业多家公司的估值数据
stock_list = ['600519', '000858', '000568', '002304']
names = ['贵州茅台', '五粮液', '泸州老窖', '洋河股份']

data = []
for code, name in zip(stock_list, names):
    # 获取财务数据
    info = ak.stock_financial_abstract_ths(symbol=code)
    # 这里简化处理 - 实际需获取具体财务指标
    # 获取实时行情
    quote = ak.stock_zh_a_spot_em()
    row = quote[quote['代码'] == code]

    if not row.empty:
        pe = float(row['市盈率-动态'].values[0])
        # 假设增长率（教学演示用，实际需基于预测）
        growth = 0.15
        peg = pe / (growth * 100)
        data.append({
            '公司': name,
            'PE(TTM)': round(pe, 1),
            'PEG(估算)': round(peg, 2)
        })

df = pd.DataFrame(data)
print(df)

运行这段代码可以快速查看同行业公司的PE和PEG对比。你可能会惊讶地发现——即使在同一行业，不同公司的PE差异也可能很大，而这些差异背后，往往隐藏着市场对各自增长前景和竞争优势的不同判断。

掌握了相对估值法，你就掌握了市场上最常用的定价语言。但估值终究是"基本面"的范畴——它回答的是"这家公司值多少钱"的问题。从下一章开始，我们将转向一个完全不同的分析维度：技术分析。技术分析不关心"价值"，只关心"价格和趋势"——它是在战场上短兵相接时的武器。
