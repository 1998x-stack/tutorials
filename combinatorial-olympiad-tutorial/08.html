第八章：抽屉原理（一）

大家好，我是你们的讲师。今天我们要讲组合数学中一个"最简单又最狡猾"的工具——抽屉原理（Pigeonhole Principle）。说它简单，是因为它的核心思想连小学生都能听懂：把10个苹果放进9个抽屉里，至少有一个抽屉有至少2个苹果。说它狡猾，是因为在竞赛中，你能不能认出"苹果"和"抽屉"，以及怎么构造它们，往往决定了你能不能解出这道题。

我第一次接触抽屉原理是在高一那年，老师讲了一个题："证明：在任意5个整数中，必有3个数的和是3的倍数。"我当时觉得这不可能——5个数太随意了，怎么可能保证？结果老师用抽屉原理三句话证完。那一刻我才明白，组合数学里真正的高手不是算得快，而是会"制造抽屉"。

抽屉原理还有一个更酷的名字——Dirichlet 原理。是的，就是那位在数论领域大名鼎鼎的德国数学家 Dirichlet。他在1834年第一次明确提出这个原理，用来证明数论中的逼近定理。所以别看它朴素，它的力量远超你的想象。

8.1 抽屉原理的三种表述

抽屉原理有三种常用的表述形式，分别适用于不同场景。我们逐一来看。

8.1.1 简单形式（第一抽屉原理）

表述一：如果把 n+1 个物体放入 n 个抽屉中，那么至少有一个抽屉中含有至少 2 个物体。

这是最基本的形式，适用于"存在性"问题——证明在某类划分下必然有重合。

例1：同月出生——证明在13个人中，至少有2个人在同一个月出生。

抽屉是12个月份，人是13个，每个"人"对应他出生的月份。13个人放入12个月份抽屉，由抽屉原理，至少有一个抽屉有至少2个人，即至少有2个人在同一个月出生。

就这么简单。但竞赛题不会这么直白——它们会把这个原理包装起来。

8.1.2 一般形式（第二抽屉原理）

表述二：如果把 m 个物体放入 n 个抽屉中，那么至少有一个抽屉中含有至少 ⌈m/n⌉ 个物体。

这里的 ⌈x⌉ 表示向上取整（ceil），即不小于 x 的最小整数。当 m = n+1 时，⌈(n+1)/n⌉ = 2，退化为简单形式。

这个一般形式的关键在于⌈m/n⌉ 这个数的构造。很多问题的突破口就是计算这个天花板值。

例2：某次数学竞赛有100道选择题，每题5个选项。证明：至少有多少名考生参加考试，才能保证有2个人的答案完全相同？

分析：每道题有5种选择，100道题，所以一份答卷的可能情况有 5100 种。把"不同的答卷类型"看作抽屉，考生看作物体。要保证有一个抽屉至少有2个物体，物体数必须大于抽屉数。所以至少需要 5100 + 1 名考生。

但如果题目改成"保证有3个人的答案完全相同"呢？这时就需要至少 2 × 5100 + 1 名考生。一般地，要保证有 k 个人答案相同，需要 (k-1) × 抽屉数 + 1 个物体。

这个"乘 (k-1) 加 1"的模式非常重要——它是抽屉原理反过来的表述：如果每个抽屉最多放 k-1 个物体，那么 n 个抽屉最多放 n(k-1) 个物体。一旦超过这个数，就必然有抽屉达到 k 个。

8.1.3 平均值形式（第三抽屉原理）

表述三：如果 n 个正数的平均值为 A，那么其中至少有一个数 ≥ A，也至少有一个数 ≤ A。

这个形式看似平凡，但它的威力在于——你不需要知道每个数的具体值，只要知道平均值，就能断言最大最小值的存在范围。

竞赛中，平均值形式常常与"反证法"配合使用：假设所有数都小于某个值，推出与总和不符的矛盾。

8.2 ⌈m/n⌉ 的巧妙构造——如何制造抽屉

抽屉原理最难的永远不是定理本身，而是如何在具体问题中识别出"苹果"和"抽屉"。下面我总结几种最常用的构造方法。

8.2.1 按数值范围构造抽屉

当题目涉及一组数字，要求证明存在某两个数的差或和满足特定条件时，常常按数值范围来划分抽屉。

例3：从1到200的自然数中任意选出101个数。证明：其中一定存在两个数，使得一个数是另一个数的倍数。

分析：这是一个经典题。我们需要构造抽屉，使得每个抽屉中的任意两个数互为倍数关系。怎么做？

把每个数写成 2k × 奇数 的形式（即提取所有因子2后剩下的奇数部分）。1到200中，奇数部分只可能是 1,3,5,...,199，共有100个奇数。

这100个奇数就是100个抽屉。每个数放入它对应的奇数抽屉中。现在选了101个数，由抽屉原理，至少有两个数落在同一个奇数抽屉。设它们分别是 2a × t 和 2b × t（t 是同一个奇数），那么较小的那个整除较大的那个（因为 2min(a,b) × t 整除 2max(a,b) × t）。

证毕。 关键在于发现了"提取2的幂次后看奇数部分"这个构造。

8.2.2 按余数类构造抽屉

这是竞赛中最常见的抽屉构造方式——把整数按模某个数的余数分类。

模数抽屉数典型应用
模22个（奇/偶）奇偶性分析、整除问题
模33个（0/1/2）和/差为3的倍数问题
模nn个任意n+1个整数中必有差为n的倍数
模某数的平方多个勾股数、平方差问题

例4：证明：任意5个整数中，必定有3个数的和是3的倍数。

分析：每个数按模3的余数分为0、1、2三类。我们考虑三种情况：

  • 情况一：如果有至少3个数属于同一余数类（比如都是模3余1），那么这3个数的和是 1+1+1 ≡ 0 (mod 3)，满足条件。
  • 情况二：如果每个余数类最多有2个数，且0、1、2每一类至少有一个数——那么从每类中各取一个数，它们的和是 0+1+2 ≡ 0 (mod 3)，也满足条件。
  • 情况三：如果某个余数类没有数——那么5个数只能分布在2个余数类中。由抽屉原理，至少有一个类有 ⌈5/2⌉ = 3 个数，这就回到了情况一。

所有情况都成立，故原命题得证。

这个分情况讨论的方法非常重要——它不是简单地套用抽屉，而是先利用抽屉确定范围，再分类讨论。

8.2.3 按区间划分构造抽屉

当问题涉及"两个数比较接近"时，可以将数值范围等分成若干区间作为抽屉。

例5：在区间 [0,1] 内任取 5 个点。证明：其中至少有两个点的距离不超过 0.25。

分析：把区间 [0,1] 等分成 4 个子区间：[0, 0.25), [0.25, 0.5), [0.5, 0.75), [0.75, 1]。每个子区间长度都是 0.25。5 个点放入 4 个区间，由抽屉原理，至少有一个区间包含至少 2 个点。这个区间内任意两点距离 ≤ 区间长度 = 0.25。

核心技巧：要证明"存在两个数之差不超过 d"，就把范围切成多个长度为 d 的小区间。物体数 = 点数，抽屉数 = 区间数。物体的数量只要大于区间数，结论就成立了。

8.3 图形中的抽屉原理

几何中的抽屉问题往往更加隐蔽——你需要将几何图形划分成若干区域作为抽屉。

8.3.1 三角形内点问题

例6：在边长为1的等边三角形内任意放5个点。证明：其中至少有两个点的距离不超过 0.5。

分析：将等边三角形各边中点连接，把大三角形分成4个边长为0.5的小等边三角形（每个面积是大三角形的1/4）。5个点放入4个小三角形，由抽屉原理，至少有两个点落在同一个小三角形中。同一个小三角形的任意两点距离 ≤ 小三角形的边长 = 0.5。

这个"分割"技巧非常实用。遇到三角形内的点问题，首先考虑用平行于底边的线分割，或者从顶点到底边的线段分割。

8.3.2 正方形五点问题

例7：在边长为1的正方形内任意放9个点。证明：其中至少有三个点构成的三角形面积不超过 1/8。

分析：将正方形分成4个 0.5×0.5 的小正方形。9个点放入4个小正方形，由抽屉原理，至少有一个小正方形包含至少 ⌈9/4⌉ = 3 个点。这三个点都在一个 0.5×0.5 的正方形内，它们构成的三角形面积 ≤ 小正方形面积的一半 = 0.5²/2 = 1/8。

这个题的关键是：三角形面积不超过它所在矩形面积的一半——这是平面几何的基本事实。看到了吗？抽屉原理负责"找到三个点在一起"，几何性质负责"估计面积范围"——两个工具配合使用。

⚠️ 踩坑实录：有一年全国高中数学联赛的预选题，出了这么一道："在边长为1的正方形内任意放5个点，证明必有两点的距离不超过 √2/2。"很多考生立刻想到把正方形分成4个小正方形，然后说5点放入4格，由抽屉原理，至少有一格有两点，这两点距离 ≤ 小正方形对角线 = √2/2。

这个答案看似正确但有漏洞！因为点可以落在小正方形的边界上——如果两点恰好在相邻两个小正方形的边界上（比如在同一条边上），它们可能落在不同的"格子"里，但距离依然可以很小。这就需要更精细的分析。正确的解法应该考虑：即使按边界划分，如果点落在相邻格的边界上，也要将其视为特殊情况处理。或者使用另一种划分方式——将正方形按对角线分成4个三角形区域，这样边界上的点自然归属唯一。

这个教训告诉我们：考试中用图画划分抽屉时，一定要确保每个点唯一确定一个抽屉。如果有歧义（比如点在边界上），你的证明就会有漏洞。

8.4 Erdős-Szekeres 经典定理

这是抽屉原理历史上最著名的应用之一，由组合数学大师 Erdős 和 Szekeres 在1935年证明。

Erdős-Szekeres 定理：任意 n²+1 个不同的实数中，必存在一个长度为 n+1 的单调子序列（要么递增，要么递减）。

这个定理的漂亮之处在于它用抽屉原理证明了一个看似和"抽屉"毫无关系的问题。

证明思路：对于每个数 ai，定义两个量：

  • Li = 以 ai 结尾的最长递增子序列的长度
  • Di = 以 ai 结尾的最长递减子序列的长度

现在考虑 n²+1 个数对应的有序对 (Li, Di)。

关键观察：对于任意 i ≠ j，如果 ai < aj 且 i < j，则 Lj ≥ Li + 1（因为可以在 ai 的递增子序列后面接上 aj），所以 (Li, Di) ≠ (Lj, Dj)。同理，如果 ai > aj，则 Dj ≥ Di + 1，所以两对也不等。

因此，所有 n²+1 个有序对 (Li, Di) 互不相同。

假设这个序列中既没有长度为 n+1 的递增子序列，也没有长度为 n+1 的递减子序列，那么每个 Li ∈ {1,2,...,n}，每个 Di ∈ {1,2,...,n}，所以 (Li, Di) 最多有 n × n = n² 种可能。但这里有 n²+1 个不同的有序对——由抽屉原理，不可能！

所以假设不成立，原命题成立。

Erdős-Szekeres 定理的抽屉结构：
│  物体    │   n²+1 个有序对 (Lᵢ, Dᵢ)
│  抽屉    │   n² 种可能的取值 (1..n)×(1..n)
│  结论    │   必有两个相同 → 矛盾 → 假设不成立
│           │   → 存在长度为 n+1 的单调子序列

思路的精髓：这个证明的伟大之处在于，它本来要证"存在长的单调子序列"，但反过来假设"不存在"，然后给每个数贴上一个"标签"（Lᵢ, Dᵢ），发现如果有 n²+1 个数，标签的多寡就产生了矛盾。这种反证法 + 抽屉原理 + 巧妙标记的组合打法，在竞赛中经常出现。记住它。

8.5 "两个人在同一个月出生"型问题

这类问题是在说：当物体数量超过抽屉数量时，"存在两个东西有某种共同属性"。我总结了几种最常见的变体：

类型物体抽屉最少物体数
同月出生人12个月份13
同余整数模n的余数n+1
同种颜色球颜色种类颜色数+1
同分同校考生（分数×学校数）乘积+1
同生日人366天（闰年）367

例8（经典生日问题）：至少需要多少人，才能保证至少有3个人的生日在同一天？（不考虑闰年）

一年有365天。每个抽屉（天）最多放2个人，最坏情况下是 2 × 365 = 730 个人，仍然没有3个人的生日相同。第731个人无论生日是哪天，都会让那天的至少一个人数变成3。所以答案是在 2 × 365 + 1 = 731。

推广：要保证至少有 k 个人的生日在同一天，需要 (k-1) × 365 + 1 人。

例9（更隐蔽的版本）：从 1,2,...,2n 中取 n+1 个数，证明其中必有一个数整除另一个数。

这个经典题用我们 8.2.1 节的方法：每个数写成 2k × 奇数，n+1 个数对应 n+1 个奇数，但奇数范围是 1 到 2n-1 中只有 n 个奇数。由抽屉原理，两个奇数相同 → 小的整除大的。

"存在两个..."问题的解题套路：

  1. 找"共同属性"：结论要证明"存在两个 X具有某种关系"
  2. 设计抽屉：将"可能的状态"或"值的分类"作为抽屉
  3. 计算数量：物体数 > 抽屉数 → 必有重合
  4. 验证重合确实蕴含结论：抽屉中的两个物体之间的关系正好就是题目要求的"存在关系"

8.6 本章小结与过渡

本章我们学习了抽屉原理的三种表述形式，掌握了如何通过数值范围、余数类和区间划分等方法来构造抽屉。我们在几何图形中看到了抽屉原理的应用，领略了 Erdős-Szekeres 定理的优美证明，最后总结了"存在两个..."型问题的通用解法。

抽屉原理的精髓不在于定理本身——那不过是 ⌈m/n⌉ 而已——而在于你能否看出"什么东西做抽屉最合适"。这需要大量的练习和敏锐的观察力。

下一章我们来学习抽屉原理的进阶——嵌套抽屉、面积抽屉、以及与 Ramsey 理论的联系。你会发现，一旦掌握了构造嵌套抽屉的技巧，"存在性"问题几乎无所不能。
