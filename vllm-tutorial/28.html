28.3 进阶学习路径

学完这门课，你已经具备了生产级 vLLM 部署和调优的能力。如果想进一步深入：

  • 源码级理解：读 nano-vLLM（1200 行），然后读 vLLM 源码中的 scheduler 和 model_runner
  • 底层优化：学习 CUDA 编程、Triton kernel 编写，理解 PagedAttention kernel 的实现细节
  • 系统设计：研究 vLLM Production Stack 的多模型管理、LMCache、前缀感知路由
  • 学术追踪：SOSP/OSDI/MLSys 会议中关于推理系统的论文

最后的建议：技术更迭很快，但原理不会过时。PagedAttention 的"分页思想"源自 1960 年代的 OS，到今天依然是最优解。学好原理，再看什么新框架都只是"旧思想的新组合"。祝你在 LLM 推理系统的道路上越走越远！
