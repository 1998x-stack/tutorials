第10章 相对估值法(下)

大家好，我是你们的讲师。上一章我们讲了PE、PEG和PB——这些是最常用的估值指标，但它们的适用范围其实有限。比如，一家亏损的科技公司PE是负数怎么估？一家高杠杆的钢铁企业PB很低就是便宜吗？并购交易中需要不受资本结构影响的估值倍数又该用哪个？

这一章我们就来补充几个重要的估值工具：PS（市销率）、EV/EBITDA（企业价值倍数）、DDM（股利折现模型），以及一个越来越被实务界青睐的方法——退出倍数法（Exit Multiple Method, EMM）。这些工具各自有特定的适用场景，学会它们，你的估值工具箱就更完整了。

10.1 PS：未盈利企业的估值工具

PS（Price-to-Sales Ratio，市销率）= 总市值 / 营业收入。当一家公司没有盈利（EPS为负），PE就失效了——这时候PS就派上了用场。

PS的核心逻辑是：虽然你现在不赚钱，但你的收入终有一天会转化为利润。 只要收入在增长，亏损可能是阶段性的（比如因为研发投入或市场扩张）。

但PS有一个隐含前提：你必须同时看毛利率。 如果一家公司PS=5倍，毛利率只有10%，那它要实现盈利非常困难。相反，如果毛利率高达70%，即使当前亏损，一旦费用率下降，盈利空间就会大幅释放。

这也是PS最终"收敛于PE"的逻辑：

PS = PE × 净利润率（Net Profit Margin）

推导：
PS = 市值 / 收入 = (市值 / 净利润) × (净利润 / 收入) = PE × 净利润率

这意味着，如果一家公司行业平均PE是20倍，净利润率是5%，那么合理的PS大约是20×5%=1倍。如果它的PS远高于1倍，要么市场预期净利润率会大幅提升（比如亏损的科技公司），要么就是被高估了。

举个实际案例。2023年的拼多多（PDD），PS约6倍，但净利润率约25%，对应的隐含PE = 6/0.25 = 24倍。这个PE对于年增长30%+的公司来说并不贵。而同期一些亏损的SaaS公司，PS高达10-20倍但净利润率接近0——隐含PE趋近于无穷大，估值就非常脆弱。

10.2 EV/EBITDA：并购交易的通用语言

EV/EBITDA（企业价值倍数）是并购交易和跨市场比较中最重要的指标，没有之一。

EV（企业价值）= 总市值 + 总负债 - 现金及等价物
EBITDA = 息税前利润（EBIT）+ 折旧 + 摊销

EV/EBITDA相比PE有两大优势：

  • 不受资本结构影响：PE受负债水平影响大——两家盈利完全相同的公司，负债高的那家PE可能更低（因为高利息减少了净利润）。但EV/EBITDA使用的是企业价值和EBITDA，不受利息和资本结构的影响，更适合跨公司比较。
  • 不受折旧政策影响：不同公司对固定资产的折旧年限和折旧方法不同，这会人为地扭曲净利润。EBITDA加回了折旧和摊销，消除了这个干扰。在重资产行业（钢铁、化工、电信）尤其有用。

计算示例：

公司A：市值500亿，负债200亿，现金50亿，EBITDA 80亿
EV = 500 + 200 - 50 = 650亿
EV/EBITDA = 650 / 80 = 8.1倍

公司B：市值400亿，负债100亿，现金80亿，EBITDA 70亿
EV = 400 + 100 - 80 = 420亿
EV/EBITDA = 420 / 70 = 6.0倍

结论：公司B的相对估值更便宜（6.0倍 < 8.1倍）

使用EV/EBITDA时要注意：EBITDA不包括资本开支（CAPEX），如果两家公司资本开支差异很大，EV/EBITDA会有误导性。一个更好的变种是EV/EBIT（企业价值/息税前利润），它考虑了折旧（即资本消耗），但未考虑摊销。

10.3 DDM与分红率

DDM（Dividend Discount Model，股利折现模型）是DCF的一个特例——只不过未来现金流被简化为了"分红"。公式很简单：

DDM估值 = 每股分红 / (股权成本 - 增长率)

这个公式有一个明显的局限——它只适用于稳定派发分红的公司。对于大多数成长型公司来说，它们不分红或分红极低，DDM就不适用。

那DDM什么时候最有用呢？答案是——银行股。

银行的业务模式决定了分红是股东回报的核心方式（因为银行需要维持资本充足率，不能大规模回购股份）。而且银行的盈利波动相对小，分红政策稳定，非常适合用DDM来估值。

举例：假设工商银行每股分红0.35元，股权成本10%，分红年增长率3%，那么DDM估值为：

DDM估值 = 0.35 / (10% - 3%) = 0.35 / 0.07 = 5.0元

如果当时工行股价是4.5元，那么DDM显示它被低估了约10%。这就是一个买入信号。

10.4 退出倍数法（EMM）：终结"永续幻觉"

还记得第8章我们说的DCF终值陷阱吗？终值占估值80%以上，而这个比例大部分来自"永续增长"的假设——但大多数公司活不到永续那一天。

退出倍数法（Exit Multiple Method, EMM） 就是为了解决这个问题而生的。EMM不做永续假设，而是设定一个3-5年的持有期，在持有期结束时按一个合理的市场倍数（比如15倍PE或8倍EV/EBITDA）"退出"。

EMM估值 = Σ FCFt / (1 + r)ᵗ (t=1到n) + 退出价值 / (1 + r)ⁿ

其中：
FCFt = 第t年的自由现金流
退出价值 = 第n年的预测净利润 × 目标PE倍数
n = 持有期（通常3-5年）

EMM相比传统DCF的优势是明显的：

  • 不需要永续增长率假设——这是DCF最大的误差来源
  • 退出倍数更直观——你可以问自己："3年后市场给这家公司15倍PE合理吗？"这比"公司永续增长3%合理吗"更容易判断
  • 与市场实际接轨——私募股权和并购交易中，退出倍数法是标准做法

根据2024年的一项行业调查，亚太地区已有超过40%的买方分析师在估值报告中同时使用DCF和EMM，而在北美这个比例更高。EMM正在以惊人的速度替代DCF作为首要估值工具。

10.5 不同行业的估值方法适配

没有一种估值方法放之四海而皆准。下表总结了各行业最适合的估值方法：

行业类型 首选方法 辅助方法 适用逻辑
消费（白酒、食品、家电） PE / DCF PEG / EMM 现金流稳定，盈利可预测
金融（银行、保险、券商） PB / DDM PE 重资产，分红稳定，PE受周期扰动
科技（软件、互联网、半导体） PS / EV/EBITDA PEG / EMM 早期亏损或低盈利，重增长
周期（钢铁、有色、化工） PB EV/EBITDA 盈利波动大，PE不靠谱
重资产（电信、电力、交通） EV/EBITDA DCF / PB 折旧比重高，资本结构复杂
地产 NAV（净资产价值法） PB 核心价值在土地和项目储备

踩坑故事：用PE给亏损科技股估值，错失良机

这是我职业生涯中交过的最大一笔"学费"之一。2020年，我关注到一家云计算SaaS公司，它的收入连续三年翻倍增长，客户留存率高达120%（净留存率，即老客户增购超过流失）。但我当时的估值框架只有PE，打开年报一看——净利润是负的。PE是负数，没法用。于是我就直接把它从股票池里删除了。

结果这家公司后面三年涨了10倍。它后来盈利了，但更重要的是，如果当时我用PS来估值——它当时的PS是8倍，而可比公司平均PS是15倍——我会发现它其实很便宜。亏损只是因为研发投入太大，而不是因为业务不行。等到它盈利的那一天，市场早已把股价推上了天。

这个教训让我养成了一个习惯：永远不要因为PE是负数就把一家公司排除在外。 亏损不等于没价值，关键是要用对估值工具。PS、EV/EBITDA、甚至EMM，都是为这样的场景准备的。

核心原则：估值方法的选择，本质上是对公司商业模式的理解。没有万能的方法，只有适合特定场景的工具。做估值之前，先问自己三个问题：①这家公司赚钱了吗？②它的资产重不重？③它的增长靠什么驱动？回答了这三个问题，你就知道该用什么方法了。

小结与预习

这一章我们完成了相对估值法的另一半：PS特别适合亏损但有收入的企业，但要结合毛利率使用；EV/EBITDA是并购和跨市场比较的通用语言，不受资本结构和折旧政策影响；DDM适用于稳定派息的公司（尤其是银行股）；退出倍数法EMM是我们推荐的终值替代方案，它不做永续假设，更贴近市场实际。最后我们讨论了不同行业应该选择什么估值方法，以及选错方法的惨痛教训。

到这里，我们的"传统估值方法"部分就全部讲完了。下一章我们进入一个更有挑战性的话题——成长股估值。对于科技公司、创新型企业，传统的PE/PB/PS很难给出满意的答案，我们需要一套全新的估值框架。
