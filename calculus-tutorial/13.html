用多项式"模拟"整个世界

大家好，我是你们的讲师。在之前的章节里，我们反复看到这样一个思想：用直线（一阶线性函数）来近似曲线。微分 dy = f'(x)dx 就是这种思想的体现——在微小范围内，用切线代替曲线。但如果你问一个工程师："用直线近似够吗？"他大概率会摇头。因为很多问题是非线性的，直线近似误差太大。

于是问题来了：如果直线不够，我们能不能用抛物线（二次函数）来近似？如果抛物线还不够，能不能用三次、四次……甚至 n 次多项式来近似？答案是肯定的——这正是泰勒公式（Taylor's formula）要做的事。

我对泰勒公式的感情非常深。大二那年上数值分析课，教授打开 MATLAB，画出了 sin x 和它的各阶泰勒展开——当展开到第 5 项时，近似曲线几乎完美地贴合了正弦曲线；到第 9 项时，肉眼已经看不出任何误差。那一刻我真实地感受到了数学公式的"魔力"。泰勒公式让我明白了一件事：世界上的复杂函数，其实都可以用简单的多项式"模拟"出来——这不正是机器学习里"用简单模型拟合复杂函数"的数学基础吗？

为什么需要多项式逼近

多项式是"最简单的函数"之一：

  • 求导简单：每次降幂，n 次多项式求 n+1 次导就是 0
  • 求值简单：只需要加、减、乘、除——计算机轻松完成
  • 积分简单：反导也是多项式

相比之下，sin x、e^x、ln x 这些函数，如果你让计算机直接计算，它其实也是用多项式近似来算的。泰勒公式就是这些近似的理论基础。

逼近阶数 近似函数 精度特点 典型例子
0 阶 常数 f(x0) 点附近是平的 f(x) ≈ f(0)
1 阶（线性） 切线 f(x0) + f'(x0)(x-x0) 有方向，无弯曲 微分近似
2 阶（二次） 抛物线 能拟合凹凸性 f(x) ≈ f(0) + f'(0)x + f''(0)x²/2
n 阶 n 次多项式 精度极高（误差为 (x-x0)^(n+1) 阶） 泰勒展开到 n 阶

泰勒公式的系数推导

假设我们要在 x0 附近用 n 次多项式 Pn(x) 来逼近函数 f(x)。设：

Pn(x) = a0 + a1(x-x0) + a2(x-x0)² + a3(x-x0)³ + ... + an(x-x0)^n

我们希望 Pn(x) 和 f(x) 在 x0 处的各阶导数尽可能匹配：

Pn(x0) = f(x0)        → a0 = f(x0)
Pn'(x0) = f'(x0)       → 1!·a1 = f'(x0)    → a1 = f'(x0)/1!
Pn''(x0) = f''(x0)     → 2!·a2 = f''(x0)   → a2 = f''(x0)/2!
Pn'''(x0) = f'''(x0)   → 3!·a3 = f'''(x0)  → a3 = f'''(x0)/3!
...
Pn^(n)(x0) = f^(n)(x0) → n!·an = f^(n)(x0) → an = f^(n)(x0)/n!

由此得到泰勒公式：

f(x) = f(x0) + f'(x0)(x-x0) + f''(x0)/2! · (x-x0)² + ...
         + f^(n)(x0)/n! · (x-x0)^n + Rn(x)

其中 Pn(x) 是泰勒多项式（Taylor polynomial），Rn(x) 是余项。

拉格朗日余项与皮亚诺余项

余项刻画了用多项式近似代替原函数的误差。两种最常见的余项形式：

皮亚诺余项

Rn(x) = o((x-x0)^n)   当 x → x0 时

用文字说：余项是比 (x-x0)^n 更高阶的无穷小。皮亚诺余项简单好用，适用于求极限——因为你只需要知道余项趋于零的速度足够快即可。

拉格朗日余项

Rn(x) = f^(n+1)(ξ) / (n+1)! · (x-x0)^(n+1)

其中 ξ 介于 x 和 x0 之间。

拉格朗日余项给出了误差的具体定量形式——它和第12章拉格朗日中值定理的思想一脉相承（实际上，当 n=0 时，泰勒公式就是拉格朗日中值定理！）。拉格朗日余项适用于需要估计具体误差大小的情况——比如在数值计算中判断近似是否满足精度要求。

两种余项的选择
- 求极限：用皮亚诺余项就够了，简单直接
- 近似计算 + 误差估计：必须用拉格朗日余项，因为它能给出误差的上界
- 一个简单的记忆方法：拉格朗日余项里有个 ξ（未知的中间点），所以能给出"最大可能的误差"

麦克劳林展开：泰勒在零点处的特例

当 x0 = 0 时，泰勒公式称为麦克劳林公式（Maclaurin series）。这是最常用的形式——因为大多数函数在 0 附近的性质最"干净"。

几个最重要的麦克劳林展开

1. e^x = 1 + x + x²/2! + x³/3! + ... + x^n/n! + ...
   （指数函数的展开系数是阶乘的倒数）

2. sin x = x - x³/3! + x^5/5! - x^7/7! + ... + (-1)^k · x^(2k+1)/(2k+1)! + ...
   （正弦展开只有奇次项，正负交替）

3. cos x = 1 - x²/2! + x⁴/4! - x⁶/6! + ... + (-1)^k · x^(2k)/(2k)! + ...
   （余弦展开只有偶次项，正负交替）

4. ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ... + (-1)^(n-1) · x^n/n + ...
   （注意收敛区间是 (-1, 1]）

5. (1+x)^α = 1 + αx + α(α-1)x²/2! + ... + [α(α-1)...(α-n+1)]/n! · x^n + ...
   （二项式展开的推广，也叫牛顿广义二项式定理）
函数 展开到第 4 项 备注
e^x 1 + x + x²/2 + x³/6 所有阶导数 = e^x
sin x x - x³/6 + x^5/120 只有奇次项
cos x 1 - x²/2 + x⁴/24 只有偶次项
ln(1+x) x - x²/2 + x³/3 注意收敛区间
1/(1-x) 1 + x + x² + x³ 等比级数，|x|<1
如何高效记忆这些展开
我的记忆方法是"找关系"：
1. e^x 展开的系数是 1, 1/2!, 1/3!, 1/4!... ——分母是阶乘
2. sin x 是 e^x 的"奇次项版本"——把偶次项去掉，正负交替
3. cos x 是 e^x 的"偶次项版本"——把奇次项去掉，正负交替
4. 事实上，e^(ix) = cos x + i·sin x（欧拉公式）——从复数角度看，cos 和 sin 就是 e^(ix) 展开的实部和虚部！

一旦你记住了这个统一点，就不需要单独记忆 sin x 和 cos x 的展开了。

用泰勒公式求极限和近似计算

求极限

泰勒公式在求极限时的威力巨大——尤其是遇到 0/0 型的未定式时。它的核心优势在于：把复杂函数展开成多项式，极限就变成了多项式极限。

例：求 lim_{x→0} (sin x - x) / x³

直接用 sin x 的展开：sin x = x - x³/6 + o(x³)

代入：
(sin x - x) / x³ = (x - x³/6 + o(x³) - x) / x³
                 = (-x³/6 + o(x³)) / x³
                 = -1/6 + o(1) → -1/6

所以极限 = -1/6。  ✅

如果用洛必达法则，需要连续三次求导——泰勒展开一步到位，这就是效率的差距。

近似计算

例：用泰勒展开计算 √e 的近似值，精确到 0.001。

√e = e^(1/2)，用 e^x 的展开：e^x = 1 + x + x²/2! + x³/3! + ...

令 x = 0.5：
e^(0.5) = 1 + 0.5 + 0.5²/2 + 0.5³/6 + 0.5⁴/24 + ...

计算每项：
1 = 1
0.5 = 0.5
0.25/2 = 0.125
0.125/6 ≈ 0.02083
0.0625/24 ≈ 0.002604
下一项：0.03125/120 ≈ 0.000260 < 0.001

所以加到第 5 项：
√e ≈ 1 + 0.5 + 0.125 + 0.02083 + 0.00260 = 1.64843...

实际值 √e ≈ 1.64872...，误差仅 0.0003，满足精度要求！
泰勒展开的常用"坑"
使用泰勒展开求极限时，最关键的是展开到合适的阶数：

- 展开阶数不够：分子分母中的关键项被消掉，导致无法判断极限值
- 展开阶数太高：徒增无意义的计算量

一个好用的规则：看分母的次数。如果分母是 x^k，那么分子至少展开到 x^k 项。比如分母是 x³，分子就要展到 x³ 那一项（包括）。

另外，在复合函数中使用泰勒展开时要小心——比如对 sin(e^x - 1) 展开，需要先展开内层 e^x - 1，再代入外层展开，每一步都要保持相同的阶数。

泰勒展开的阶数选择原则

在实际使用中，如何确定展开到哪一项？下面几条"黄金法则"可以帮你快速判断：

法则1（极限问题）：如果分母（或分子中作为比较基准的项）是 x^k，展开到 x^k 项即可。如果是 (x-x0)^k，同理。

法则2（近似计算）：如果需要精度 ε，找到最小的 n 使得 |f^(n+1)(ξ)|/(n+1)! · |x-x0|^(n+1) < ε。用拉格朗日余项来估计。

法则3（奇偶性判断）：如果在 0 附近展开，可以利用函数的奇偶性来预判哪些项一定为 0。奇函数在 0 处的泰勒展开只有奇次项（如 sin x），偶函数只有偶次项（如 cos x）。

法则4（乘积型极限）：如果分子是两个函数相减，比如 e^x - 1 - x，要展开到两项不能完全抵消的阶数。例如 e^x = 1 + x + x²/2 + o(x²)，所以 e^x - 1 - x = x²/2 + o(x²)——至少展到二次项。

综合例子：求 lim_{x→0} [e^x - 1 - x - x²/2] / x³

e^x = 1 + x + x²/2 + x³/6 + o(x³)
e^x - 1 - x - x²/2 = x³/6 + o(x³)

所以极限 = (x³/6 + o(x³)) / x³ = 1/6  ✅

如果只把 e^x 展到 x² 项，e^x - 1 - x - x²/2 = 0，极限变成 0/0——无法判断。
再次验证了法则4：展开到不能完全抵消的阶数。
泰勒展开的"速查表"用法
在考试或实际计算中，你不需要每次都从零开始推导泰勒展开。把本节列出的 5 个常见展开背下来（e^x, sin x, cos x, ln(1+x), (1+x)^α），然后复杂的函数通过变量代换（比如令 u = x²）转化为这些基本展开的组合。这就像掌握了基本单词后，可以组出任何句子。

本章小结

泰勒公式是微积分中最优美的公式之一：它告诉我们任意光滑函数都可以用多项式来逼近，系数由函数在该点的各阶导数决定。拉格朗日余项给出了误差的定量估计，皮亚诺余项则更适合极限计算。五个基本函数的麦克劳林展开——e^x、sin x、cos x、ln(1+x)、(1+x)^α——是你需要熟练掌握的工具。

如果说拉格朗日中值定理是用一条直线近似函数，那么泰勒公式就是把这条直线升级成了一个 n 次多项式——精度提升了几个数量级。下一章，我们将把所有这些求导工具用于实战——导数的应用：单调性、极值、凹凸性、渐近线和函数作图。你会发现，我们前面积累的所有知识，都可以用来解决一个很酷的问题：画出任意函数的图像。
