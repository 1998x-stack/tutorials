11.1 垄断的来源：为什么只有一个卖家？

如果说完全竞争是"所有人都是价格接受者"，那么垄断（Monopoly）就是"唯一卖家，价格我自己定"。垄断并非偶然——它总有某种进入壁垒（Barriers to Entry）在保护其地位。经济学家识别出三种主要来源：

一、自然垄断（Natural Monopoly）

当一家企业能以低于多家企业的平均成本来服务整个市场时，就形成了自然垄断。典型例子是电网——为一条街道铺设两套电网是巨大的资源浪费。自来水公司、天然气管道、铁路轨道都属于此类。自然垄断的典型特征是巨大的固定成本和极低的边际成本，导致一家企业比多家企业更有效率。

二、政府授权垄断

政府通过专利、版权、特许经营等制度人为创造的垄断。专利制度就是这样的一把"双刃剑"——它给予发明者20年的独占权，以激励创新；但在此期间，消费者要支付垄断价格。药品行业是典型：一款创新药的成本中，研发占比极高而制造成本很低，专利垄断让药企能收回研发投入。

三、资源控制

当一家企业控制了某种关键资源时，就形成了垄断。最经典的例子是20世纪中叶的戴比尔斯（De Beers）——它控制了全球约90%的钻石原石供应，通过控制投放量来维持钻石的高价。另一个例子是稀土：中国一度控制了全球90%以上的稀土供应，形成了事实上的资源垄断。

⚠️ 注意区分：垄断不一定是非法的。自然垄断通常被政府监管而非禁止。利用专利获得的临时垄断是法律鼓励的。但通过掠夺性定价、捆绑销售、排他性交易等手段维持垄断地位——这些行为在大多数国家是违法的。

11.2 垄断定价：MR < P 的秘密

垄断企业最核心的区别于完全竞争企业的地方在于：垄断者面对的需求曲线就是整个市场的需求曲线——它是向右下倾斜的。这意味着：

  • 要卖出更多产品，垄断者必须降低所有单位的价格（不仅仅是新增的那一单位）
  • 因此，边际收益（MR）小于价格（P），MR曲线位于需求曲线下方

在垄断利润最大化的产量Q*处，MR = MC。但定价时，消费者支付的价格P*在需求曲线上对应Q*——这个价格远高于MC。这就产生了垄断者的"超额利润"——垄断租金（Monopoly Rent）。

# 垄断利润最大化计算
# 假设垄断者面临的需求: P(Q) = 100 - 2Q  (向下倾斜的需求曲线)
# 成本函数: TC(Q) = 20 + 10Q + Q²

def monopoly_equilibrium():
    # 总收益: TR = P*Q = (100 - 2Q)*Q = 100Q - 2Q²
    # 边际收益: MR = d(TR)/dQ = 100 - 4Q
    # 边际成本: MC = d(TC)/dQ = 10 + 2Q
    # 利润最大化: MR = MC
    # 100 - 4Q = 10 + 2Q  =>  6Q = 90  =>  Q* = 15
    Q_star = 15
    P_star = 100 - 2 * Q_star  # P* = 70
    TR = P_star * Q_star       # TR = 1050
    TC = 20 + 10*Q_star + Q_star**2  # TC = 20 + 150 + 225 = 395
    profit = TR - TC            # π = 655

    # 对比: 如果P=MC（完全竞争的情况）
    # 100 - 2Q = 10 + 2Q  => Q_comp = 22.5, P_comp = 55
    Q_comp = 22.5
    P_comp = 100 - 2 * Q_comp
    CS_monopoly = 0.5 * (100 - P_star) * Q_star  # 消费者剩余
    CS_competition = 0.5 * (100 - P_comp) * Q_comp

    print(f"垄断产量: Q* = {Q_star}")
    print(f"垄断价格: P* = {P_star}")
    print(f"垄断利润: π = {profit}")
    print(f"垄断价格比完全竞争高: {P_star - P_comp:.1f}")
    print(f"消费者剩余损失: {CS_competition - CS_monopoly:.1f}")
    return Q_star, P_star, profit

monopoly_equilibrium()
# 输出:
# 垄断产量: Q* = 15
# 垄断价格: P* = 70
# 垄断利润: π = 655
# 垄断价格比完全竞争高: 15.0
# 消费者剩余损失: 168.75

11.3 垄断的无谓损失

垄断之所以被认为"不好"，不是因为它赚了钱——而是因为它造成了效率损失。与完全竞争相比，垄断造成了两方面的问题：

比较维度完全竞争垄断
价格P = MC（边际成本定价）P > MC（加价定价）
产量有效率的产量 Qe低于有效率的产量 Qm < Qe
消费者剩余较大较小（部分变为生产者剩余，部分损失）
生产者剩余正常利润超额利润
总剩余最大化（有效率）小于最大值（有无谓损失）
无谓损失无有（DWL，三角形区域）

无谓损失（Deadweight Loss, DWL）是垄断的核心成本。它代表了那些"本来应该被生产的交易（因为消费者愿意支付的价格高于生产成本）但因为垄断的限产而没有实现"的价值。这部分价值既没有被消费者获得，也没有被生产者获得——它是纯粹的社会损失。

11.4 价格歧视：垄断者的精明算盘

如果垄断者能对不同的消费者收取不同的价格，这种策略称为价格歧视（Price Discrimination）。价格歧视分为三级：

  • 一级价格歧视（Perfect Price Discrimination）：对每个消费者收取其"保留价格"——即他愿意支付的最高价格。此时垄断者获得了全部消费者剩余，但产量恢复到有效率水平（Qe），因此无谓损失消失了。现实中很罕见，二手车市场的讨价还价有些类似。
  • 二级价格歧视（Quantity Discrimination）：按购买数量定价。比如"买三送一"、批发价低于零售价、电信套餐的阶梯定价。这利用了不同消费者对数量的不同需求弹性。
  • 三级价格歧视（Group Pricing）：按消费者群体定价。学生票、老年票、早鸟票、不同国家的不同价格——这些都属于三级价格歧视。前提是不同群体的需求弹性不同，且群体之间不能转卖。
💡 生活中的价格歧视：为什么电影票分学生票和全价票？因为学生的需求弹性大（对价格敏感），而上班族的需求弹性小（约会不能随便换日子）。电影院通过价格歧视从不同群体中获取更多消费者剩余。有趣的是，价格歧视在增加企业利润的同时，往往也增加了总产量——在某些情况下它甚至能增进社会福利。

11.5 故事：标准石油公司的拆分

19世纪末，约翰·洛克菲勒的标准石油公司（Standard Oil）通过一系列精明的商业操作——包括秘密收购、与铁路公司达成运费回扣协议、低于成本的掠夺性定价——逐步控制了美国90%以上的石油精炼能力。标准石油不仅是一个垄断者，它还是一个托拉斯（Trust）——通过信托协议控制多家公司的股权，实质性地消除了竞争。

标准石油的垄断带来了什么后果？它确实降低了成本（通过规模经济和标准化），但它也利用了市场力量提高价格、挤压竞争对手、控制运输渠道。农民和小企业主抗议油价的垄断，媒体曝光了标准石油的不正当竞争手段。

1890年，美国国会通过了《谢尔曼反托拉斯法》（Sherman Antitrust Act），这是全球第一部现代反垄断法。1911年，美国最高法院裁定标准石油公司违反了谢尔曼法，下令将其拆分为34家独立的公司。这些"后代"公司中，有些今天你依然认识——埃克森美孚、雪佛龙、BP（前标准石油的海外业务）。

标准石油的拆分开创了一个重要的先例：市场力量不能通过不正当手段获得或维持。从那以后，美国的反垄断执法经历了多次周期性调整——时而激进（如AT&T在1984年的拆分），时而保守（如微软在2000年代初的反垄断案最终和解）。但《谢尔曼法》的基本原则至今仍在指导着全球竞争政策。

11.6 垄断与竞争之间

垄断是市场结构的另一个极端——从完全竞争的"很多人、价格接受者"到垄断的"一个人、价格制定者"。但现实世界中的大多数市场，恰恰处于这两个极端之间。下一章我们要讨论的垄断竞争（Monopolistic Competition），就是介于两者之间的市场结构——有很多卖家的同时，每家都有自己的"小垄断"力量。我们下一章见。
