开篇：有多少"相关"是真实的？

大家好，欢迎继续学习典型相关分析。上一章我们理解了CCA的数学原理——如何找到一组线性组合使两组变量的相关性最大化。但是，当我们从样本数据中计算出一个典型相关系数 ρ=0.75 时，这个值到底意味着什么？它是真实存在的总体相关，还是仅仅是抽样误差导致的假象？应该保留几对典型变量来解读？如何用这些结果讲一个有意义的故事？

这一章我们来回答这些实际问题。CCA的显著性检验和结果解释，是区分"会用CCA"和"用好CCA"的分水岭。我们从检验方法开始，逐步深入到完整案例和可视化。

典型相关系数的显著性检验

CCA中最常用的显著性检验方法是威尔克斯 Λ (Wilks' Lambda) 检验。它的基本思路是：检验前 k 个典型相关系数是否显著不为零。这是一个序贯检验过程 (sequential testing)。

对于 m = min(p, q) 对典型相关系数 ρ₁ ≥ ρ₂ ≥ ... ≥ ρₘ，我们要依次检验：

H₀⁽¹⁾: ρ₁ = ρ₂ = ... = ρₘ = 0 （所有典型相关均为零——两组变量不相关）
如果拒绝 H₀⁽¹⁾，继续检验：
H₀⁽²⁾: ρ₂ = ... = ρₘ = 0 （去掉第一对后，剩余的均为零）
H₀⁽³⁾: ρ₃ = ... = ρₘ = 0
...以此类推

Wilks Λ 统计量定义为：
Λₖ = ∏ᵢ₌ₖᵐ (1 - ρᵢ²)

巴特利特 (Bartlett) 给出了近似 χ² 变换：
χ²ₖ = -[n - k - ½(p + q + 1)] · ln(Λₖ)
自由度 df = (p - k + 1)(q - k + 1)

这个序贯检验从 Λ₁ 开始。如果 p 值 < 0.05，拒绝 H₀⁽¹⁾，说明至少第一对典型相关是显著的。然后检验 Λ₂，依此类推。当某一步的 p 值 > 0.05 时停止，从这一步起的后续典型相关对都不显著。

步骤Λ 值χ²自由度p值结论
1（检验所有ρ）0.3268.524< 0.001至少ρ₁显著
2（去掉ρ₁后）0.6724.1150.062ρ₂及之后均不显著
3（去掉ρ₁,ρ₂后）0.897.280.515-

在这个例子中，第一对典型相关显著（p < 0.001），但去掉第一对之后，剩余的相关系数整体不显著（p = 0.062）。因此我们只保留第一对典型变量进行解释。

维度选择：保留多少对典型变量？

显著性检验是选择典型变量对数的首要依据，但不是唯一的。在实践中，我们还考虑以下标准：

统计显著性标准

如上所述的 Wilks Λ 序贯检验，p < 0.05 的保留。这是最客观的标准。

实际显著性标准

有时候，第二对典型相关系数 ρ₂ = 0.30，虽然由于样本量较大而统计显著（p = 0.01），但 ρ₂² = 0.09，意味着这对典型变量只解释了9%的共同变异。在实际解释中意义不大。通常只保留 ρ² > 0.10 的对。

可解释性标准

这是最重要的实践标准：你能给这对典型变量一个有意义的命名吗？如果典型载荷的模式很混乱——部分变量载荷为正、部分为负，且没有合理的理论解释——即使统计显著也建议放弃。毕竟CCA的输出是要讲给业务方听的。

实践建议：先看统计显著性做初筛，然后用交叉验证检验稳定性（在子样本上重复CCA，看典型载荷是否一致），最后问自己一个问题："我能用两三句话向业务同事解释这对典型变量的含义吗？"如果不能，就别保留。

典型结构分析

确定了保留的典型变量对数后，下一步是解读它们。典型结构分析主要包括三个层次。

层次一：典型载荷 (Canonical Loadings)

典型载荷是原始变量与自身组典型变量的相关系数。例如，假设第一对典型变量 u₁ 的载荷为：

u₁ = 0.52·生活质量 + 0.48·预期寿命 - 0.03·犯罪率 + 0.41·教育资源

载荷解读：
- 生活质量和预期寿命在 u₁ 上有较大的正载荷（>0.40）
- 教育资源的载荷中等（0.41）
- 犯罪率的载荷接近于零

结论：u₁ 主要代表了"社会福利与健康水平"这个潜在维度

层次二：交叉载荷 (Cross-Loadings)

交叉载荷衡量原始变量与另一组典型变量的相关性。如果X组的生活质量变量在 v₁（Y组的典型变量）上有0.45的交叉载荷，说明"社会福利与健康水平"这个维度与Y组变量的典型组合中度相关。

层次三：冗余指数

上一章已经介绍过，冗余指数回答："Y组的典型变量能解释X组多少变异？"这是判断CCA实际价值的最实用指标。如果冗余指数低于10%，即使典型相关系数很高，CCA的实际解释力也有限。

完整案例：经济指标与健康指标的关系

让我们通过一个完整案例来串起所有概念。假设我们有20个国家的数据，X组包含3个经济指标（人均GDP、就业率、教育支出占比），Y组包含3个健康指标（预期寿命、婴儿存活率、医疗保险覆盖率）。

第一步：运行CCA

# R代码示例
library(CCA)
X <- data.frame(GDP = c(...), 就业率 = c(...), 教育支出 = c(...))
Y <- data.frame(预期寿命 = c(...), 婴儿存活率 = c(...), 医保覆盖 = c(...))

cc_result <- cc(X, Y)
cc_result$cor  # 典型相关系数
# [1] 0.892 0.412 0.156

# Wilks Lambda 检验
library(CCP)
p.asym(cc_result$cor, nrow(X), ncol(X), ncol(Y))
# 输出：第一对 p < 0.001，第二对 p = 0.087，第三对 p = 0.642

第二步：保留第一对典型变量

典型系数：
a₁ (经济→u₁): GDP=0.61, 就业率=0.35, 教育支出=0.52
b₁ (健康→v₁): 预期寿命=0.58, 婴儿存活率=0.33, 医保覆盖=0.47

典型载荷：
经济变量对 u₁ 的载荷:  GDP=0.89, 就业率=0.52, 教育支出=0.74
健康变量对 v₁ 的载荷:  预期寿命=0.91, 婴儿存活率=0.55, 医保覆盖=0.68

交叉载荷（经济变量对 v₁）：
GDP=0.79, 就业率=0.46, 教育支出=0.66

冗余指数：
Rd(经济|v₁) = (0.89²+0.52²+0.74²)/3 × 0.892² = 0.539 × 0.796 = 0.429
→ 健康变量的典型变量 v₁ 能解释经济变量组 42.9% 的变异

第三步：解读

u₁ 可以被命名为"经济发展水平"——因为GDP、教育支出载荷高。v₁ 可以被命名为"国民健康水平"。典型相关系数 ρ=0.892，说明经济发展水平和国民健康水平之间有着很强的正相关。冗余指数42.9%也相当可观——健康水平指标能解释近一半的经济指标变异。这个分析为国家层面的政策评估提供了量化依据。

结果可视化

可视化是传达CCA结果的最有效方式。以下是几种常用图形：

典型变量散点图 (Canonical Variate Plot)

以 u₁ 为横轴、v₁ 为纵轴做散点图，每个点代表一个样本（如一个国家的数据）。如果第一典型相关很强，散点应该沿着对角线方向聚集。这种图能直观地展示两组变量的整体关联模式。

载荷条形图

用条形图展示每个原始变量在典型变量上的载荷大小，便于快速识别哪些变量的贡献最大。典型载荷 > 0.40 通常被认为是"有意义的"。

双标图风格展示

将典型载荷和样本得分叠加在同一张图上：用箭头表示原始变量（箭头的方向代表与典型变量的相关方向，长度代表载荷大小），用点表示样本的典型变量得分。这种展示方式同时呈现了"变量关系"和"样本分布"。

可视化提示：在做CCA散点图时，不妨给每个样本点添加标签（如国家名、行业名、区域名）。例如在经济-健康CCA中，你可能会发现北欧国家聚集在右上角（高经济-健康得分），而某些发展中国家聚集在左下角——这种模式本身就是有价值的洞察，有助于后续的标杆分析和政策对标研究。

踩坑故事：小样本下的不稳定载荷

再分享一个我看到的CCA误用案例。某研究团队收集了30个样本、12个X变量和8个Y变量的数据，跑了CCA后得到第一典型相关系数 ρ=0.85（p=0.002），看起来很漂亮。他们根据典型载荷解读说："X组的第3、7、11号变量与Y组的第2、5号变量之间存在强相关模式。"

然而，当我对他们数据进行留一法交叉验证（每次去掉一个样本重新计算CCA）时，发现典型载荷的变动非常大——同一个变量的载荷有时从0.72降到0.15。原因很简单：30个样本要估计20个变量的协方差结构（12+8个变量），协方差矩阵的自由度严重不足，估计极不稳定。

教训：CCA对样本量非常敏感。经验规则是：样本数 n 至少应该是变量总数 (p+q) 的5到10倍。在 p+q=20 的情况下，至少需要100-200个样本。小样本下得到的典型载荷和相关系数虽然可能统计显著，但往往是"纸牌屋"——换一个样本就变了。如果你的样本量有限，考虑使用正则化CCA (regularized CCA) 或减少变量数后再分析。永远记得对稳定的典型载荷做交叉验证，不要让"幸运"的p值欺骗了你。

本章小结

本章我们深入了CCA的实践层面。我们先学习了Wilks Λ检验和Bartlett近似χ²进行序贯显著性检验，然后讨论了维度选择的三个标准——统计显著性、实际显著性和可解释性。接着我们进入了典型结构分析的三层次框架（载荷、交叉载荷、冗余指数），通过一个完整的经济-健康案例串起了CCA的分析全流程。在可视化方面，我们介绍了典型变量散点图、载荷条形图和双标图风格展示。最后，我们用一个关于小样本的踩坑故事提醒大家：典型载荷的稳定性对样本量高度敏感，交叉验证是必不可少的质量检查。

下一章，我们将换一个视角——从两组连续变量的CCA，转向分类数据的分析方法：对应分析 (Correspondence Analysis)。对应分析可以看作是CCA在分类数据上的推广，它通过将列联表转化为低维空间中的"行点"和"列点"，让我们能直观地看到两个分类变量之间的关联模式。准备好了吗？我们对应分析见。
