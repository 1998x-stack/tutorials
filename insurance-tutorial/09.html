第9章 年金保险

我有个客户王阿姨，2010年退休时手里有50万存款。她把这笔钱分成两份：25万存银行定期，25万买了某保险公司的一款养老年金保险。2024年我们见面时，银行的25万已经花得差不多了（日常开销+孙子上学补贴），而年金那25万还在源源不断地给她"发工资"——每个月固定到账1500元，已经领了14年，预计还能领到90岁。王阿姨说："银行存款是越花越少，心里发慌。年金是每个月都来，心里踏实。"

这个故事完美诠释了年金的本质：把一笔钱变成一辈子的现金流。长寿是福，但"人活着钱没了"却是这个时代最大的养老焦虑。年金保险（Annuity Insurance）就是为应对这种焦虑而生的金融工具。

年金保险的概念与分类

年金保险（Annuity）是指以被保险人生存为给付条件，按约定周期（年、半年、季、月）分期给付保险金，直至被保险人身故或合同期满的人身保险。简单说——你一次性或分期交一笔钱给保险公司，保险公司承诺从某个时间开始，定期给你"发工资"，一直发到你去世或合同期满。

即期年金 vs 延期年金

维度 即期年金（Immediate Annuity） 延期年金（Deferred Annuity）
保费缴纳方式 一次性缴清（趸缴） 可趸缴也可分期缴
开始给付时间 缴费后立即开始（通常1个月或1年后） 缴费后经过一段约定期间才开始（如10年后）
积累期 无 有（保费在积累期内增值）
适合人群 临近退休或已退休，手头有一笔资金 中年人提前规划养老，利用时间积累收益

举个例子：张阿姨65岁退休，用100万买了即期年金，从次月开始每月领5000元，领到去世。李经理45岁时买了一份延期年金，分10年交费，从60岁开始按月领取——他给自己留了15年的积累期，让保费有更多时间增值。

个人年金 vs 团体年金

个人年金是个人与保险公司直接签订的年金合同，灵活度较高，你可以自由选择领取方式、领取年限、是否含保证给付期等。而团体年金是企业或组织为员工统一购买的，通常作为企业补充养老计划的载体，保费和条款由雇主和保险公司协商，员工个人自主权较小但管理成本低。

确定年金与生命年金

这一组分类是年金保险最精妙的地方——区别在于给付期限的确定方式：

  • 确定年金（Annuity Certain）：按约定期间给付，无论被保险人是否存活。比如"10年确定年金"，不管被保险人第5年是否去世，10年的年金都会给付（如身故则给受益人）。
  • 生命年金（Life Annuity）：以被保险人生存为条件，活着就给付，身故即停止。你的寿命越长，领得越多。

市场上最常见的是保证型生命年金——它是两者的结合：保证至少给付N年（比如10年、20年），如果被保险人在保证期内身故，剩下的年金由受益人领取；如果超过保证期后仍然健在，继续领取直至身故。

核心概念：生命年金的本质是长寿风险（Longevity Risk）的转移。你担心活得太久钱不够花，保险公司通过集合大量被保险人的资金，用那些较早去世的人的"剩余"来补贴长寿者的"超额"——这就是"以命换命"的保险机制。精算上，生命年金的给付金额高于确定年金（同等保费下），因为它包含了"长寿者补贴"的再分配效应。

变额年金与分红年金

变额年金

传统年金的给付金额是固定的，但通货膨胀会侵蚀其购买力。30年前的1000元是一笔不小的收入，今天可能只够吃一顿饭。变额年金（Variable Annuity）就是为了解决这个问题而设计的——给付金额与投资账户的收益挂钩，投资做得好就多领，做得不好就少领。

变额年金的保费进入独立投资账户，由客户选择投资标的（股票基金、债券基金、混合基金等）。领取时，每期的给付金额根据账户价值重新计算。这种设计既有机会让养老金跟上通胀，也承担了投资风险——市场不好时领取金额可能会下降。

分红年金

分红年金介于固定年金和变额年金之间——它有一个固定的基础给付金额，保险公司每年根据经营状况分配红利，红利可以增加给付金额或累积生息。与变额年金不同，分红年金的保底部分是确定的，客户不承担投资损失的风险，只是红利部分不确定。

年金类型 vs 风险收益特征

固定年金：低风险，低收益（确定给付）
        │
分红年金：中低风险，中低收益（保底+不确定红利）
        │
变额年金：中高风险，中高收益（完全市场驱动）
        │
投连年金：高风险，高收益（市场驱动+无保底）

年金保险的定价

年金因子与生命表

年金保险的定价核心是一个叫做年金因子（Annuity Factor）的数值。它表示：在当前利率和死亡率假设下，每1元保费可以购买到的未来每年的年金金额。用公式表示：

年金因子 = Σ [ₜPₓ / (1 + i)ᵗ]  (t从0到ω-x-1)

其中：
  ₜPₓ = x岁的人在t年后仍然生存的概率
  i   = 预定利率
  ω  = 生命表极限年龄
  Σ  = 求和符号

这个公式背后是两条核心假设：

  • 生存概率——来自生命表。人口死亡率越低（寿命越长），年金因子越大，同等保费下每期给付金额越少（因为保险公司需要支付更长时间）
  • 预定利率——利率越高，保险公司通过投资赚的钱越多，同等保费下给付金额可以更高

第四套生命表对年金的影响

2023年底，中国保险业发布了第四套经验生命表（2020-2023）。与前一套相比，新的生命表预期寿命进一步延长——男性平均寿命从79.5岁延长到81.3岁，女性从84.6岁延长到86.7岁。这意味着什么？

行业影响：生命表更新后，年金保险的定价面临巨大压力——因为预期寿命延长意味着保险公司需要多付好几年的养老金。据测算，仅此一项就可能导致同等保费的年金给付金额下降5%-8%。这就是为什么各家保险公司在2024年集体调整了年金产品的领取金额。如果你正在考虑买年金险，行业内有句忠告：生命表越新，年金的"性价比"越低。早买永远比晚买划算。

养老金第三支柱

个人养老金制度

中国的养老保障体系是"三支柱"结构：

支柱 名称 覆盖范围 现状
第一支柱 基本养老保险 城镇职工+城乡居民 覆盖10亿人，但替代率持续下降（目前约40%）
第二支柱 企业年金+职业年金 机关事业单位+部分企业 覆盖约7000万人，覆盖面窄
第三支柱 个人养老金+商业养老保险 个人自愿参与 2022年起步，快速发展中

个人养老金制度于2022年11月正式启动，在全国36个城市试点。核心机制是：个人在银行开立个人养老金账户，每年最多可存入12000元，这笔钱可以购买指定的金融产品（包括储蓄、基金、理财、保险四类），享受税收递延优惠——存入时不计入当年应纳税所得额，领取时按3%的优惠税率征收。

税优政策与商业养老保险

对工薪阶层来说，个人养老金最大的吸引力在于节税。我们来算一笔账：

假设某人年应纳税所得额处于25%的税率档：
  每年存入：12,000元
  当年节税：12,000 × 25% = 3,000元
  领取时缴税：假设账户增值到15,000元，领取缴税15,000 × 3% = 450元
  净节税收益：3,000 - 450 = 2,550元/年

税率越高的人，节税效果越明显。45%税率档的人每年最多可节税5400元。但也要注意：对于本来就不需要缴税的低收入群体，个人养老金反而没有节税优势（领取时反而要交3%的税）。

实践建议：个人养老金账户中的保险产品主要是专属商业养老保险和年金保险。选择建议如下——如果离退休不足10年，优先考虑确定给付型年金，养老金"落袋为安"最重要；如果还年轻（30-40岁），可以考虑结算利率较高的专属商业养老保险，利用长期积累获取更好收益。但无论哪种，个人养老金的核心优势是节税+强制储蓄，不要把它当作投资工具，而应看作是补充养老金的"压舱石"。

养老年金市场趋势

2025-2026年，中国养老金融市场正经历深刻变革。根据国家金融监管总局的数据，截至2025年底，商业养老保险保费收入突破1.2万亿元，年增长率超过20%。几个关键趋势值得关注：

第一，个人养老金制度全面扩围。2025年起，个人养老金从36个试点城市推广至全国，参与人数预计在2026年底突破8000万人。账户产品种类也在扩容——除了最初的四大类产品，新增了商业养老金、养老基金等多种选择。

第二，险企的"养老+保险"生态布局加速。以中国太保为例，其"太保家园"养老社区已在全国13个城市落地，形成"保险支付+养老服务"的闭环模式——客户购买养老年金达到一定保费规模，可以获得养老社区的入住权。这种模式将年金保险从单纯的金融产品升级为"一站式养老解决方案"。

第三，利率下行对年金定价形成持续压力。随着中国10年期国债收益率进入"2时代"，保险公司投资端收益下降，年金产品的定价利率和结算利率都在下调。2024年监管将普通型年金产品的预定利率上限从3.5%下调至3.0%，这意味着新发产品的收益率会进一步降低。

趋势总结：年金保险正在从"有则买之"的锦上添花，变成"不得不买"的养老刚需。第一支柱替代率持续走低、第二支柱覆盖面有限、人均寿命持续延长、老龄化加速——这四个因素叠加，让个人商业养老金成为应对"长寿风险"的核心工具。对保险从业者来说，年金险的销售将从"卖产品"转向"做养老规划"，这是行业升级的大方向。

年金保险给我们提供了一个"对抗长寿"的金融保障——活得越久领得越多，这本身就是对幸福晚年的一种激励机制。但人的一生中，不确定的不仅是寿命，还有健康。下一章我们就进入健康保险，聊聊医疗险、重疾险、护理险这些与每个人健康息息相关的险种。
