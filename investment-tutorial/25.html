第25章 交易策略设计

2019年，我认识的一位量化私募基金经理做了一件"蠢事"：他花了大半年时间，开发了一个非常复杂的多因子模型，用上了LSTM神经网络、高频数据、舆情分析，回测曲线漂亮得令人窒息——年化收益40%、最大回撤只有8%、夏普比率2.8。他信心满满地实盘上线，结果三个月后，策略亏损了15%。更讽刺的是，同期他团队里一个交易员用最简单的"20日均线上穿60日均线"策略，赚了12%。

这个故事告诉我们两件事：第一，复杂 ≠ 有效；第二，策略设计的关键不在于工具的炫酷程度，而在于你是否理解了策略赚钱的底层逻辑——它赚的是谁的钱？在什么市场环境下有效？什么情况下会失效？本章将带你系统地梳理交易策略的"兵器库"，从经典的趋势跟踪到前沿的AI辅助选股，帮你建立一套完整的策略设计与评估框架。

25.1 策略分类：各门各派的"武功秘籍"

交易策略千千万，但万变不离其宗。从最底层逻辑来看，所有策略都可以归入以下几类：

趋势跟踪：顺势而为

趋势跟踪（Trend Following）是历史最悠久、最"朴素"的策略流派。它的哲学很简单：趋势是你的朋友（The trend is your friend）。核心假设是市场价格存在惯性——涨的会继续涨，跌的会继续跌。

  • 均线策略：最简单的实现方式。当短期均线上穿长期均线时买入（金叉），下穿时卖出（死叉）。经典参数组合有5日/20日、20日/60日、60日/120日。不同参数适应不同周期的趋势。
  • MACD：指数平滑异同移动平均线，是均线的"升级版"。通过计算快慢线的差值和信号线来判断趋势的强弱和转折。实际操作中，MACD柱由负转正是买入信号，柱体背离则预示趋势可能反转。
  • 海龟交易法则：著名的经典趋势跟踪系统，使用唐奇安通道（Donchian Channel）突破作为入场信号。核心是"突破N日高点买入，跌破N日低点卖出"，配合加仓规则和止损规则。
# 海龟交易法则简化版（唐奇安通道突破）
import pandas as pd
import numpy as np

def turtle_trading_strategy(df, entry_period=20, exit_period=10, atr_period=20):
    """
    海龟交易法则简化实现
    entry: 价格突破过去N日最高点买入
    exit: 价格跌破过去M日最低点卖出
    stop: 2倍ATR止损
    """
    data = df.copy()

    # 唐奇安通道
    data['entry_high'] = data['收盘'].rolling(entry_period).max()
    data['exit_low'] = data['收盘'].rolling(exit_period).min()

    # ATR（平均真实波幅）
    data['prev_close'] = data['收盘'].shift(1)
    data['tr'] = np.maximum(
        data['最高'] - data['最低'],
        np.maximum(
            abs(data['最高'] - data['prev_close']),
            abs(data['最低'] - data['prev_close'])
        )
    )
    data['atr'] = data['tr'].rolling(atr_period).mean()

    # 信号生成
    data['signal'] = 0
    buy_condition = data['收盘'] > data['entry_high'].shift(1)
    data.loc[buy_condition, 'signal'] = 1

    # 无持仓时突破买入，有持仓时跌破止损或通道低点卖出
    position = 0
    entry_price = 0
    data['position'] = 0

    for i in range(entry_period, len(data)):
        if position == 0 and data['signal'].iloc[i] == 1:
            position = 1
            entry_price = data['收盘'].iloc[i]
            data.loc[data.index[i], 'position'] = 1
        elif position == 1:
            stop_price = entry_price - 2 * data['atr'].iloc[i]
            if data['收盘'].iloc[i] < data['exit_low'].iloc[i] or data['收盘'].iloc[i] < stop_price:
                position = 0
                data.loc[data.index[i], 'position'] = 0
            else:
                data.loc[data.index[i], 'position'] = 1

    return data
趋势跟踪的"命门"：趋势策略最怕的是"震荡市"（Range-bound Market）。在横盘整理行情中，频繁的金叉死叉信号会导致"两边打脸"——每次买入就跌、卖出就涨。2023年A股的很多时间里就处于这种状态。识别市场环境比优化策略参数更重要：当波动率偏低、均线缠绕时，降低趋势策略的仓位甚至空仓，就是最好的策略。

均值回归：跌多了会涨，涨多了会跌

均值回归（Mean Reversion）的哲学与趋势跟踪正好相反。它假设价格围绕一个"合理价值"波动，当价格偏离太远时，会像弹簧一样被拉回来。均值回归策略在震荡市中表现优异，但在强趋势市中会亏损严重。

  • RSI（相对强弱指标）：当RSI低于30（超卖）时买入，高于70（超买）时卖出。RSI的周期常用14天，但不同的市场需要调整参数。
  • 布林带（Bollinger Bands）：以移动平均线为中轨，加减两倍标准差为上下轨。价格触及下轨时买入，触及上轨时卖出。布林带的收缩和扩张还能反映波动率的变化——"布林带收窄"往往预示着即将出现大行情。
真实案例：2024年有位做"ETF网格交易"的散户朋友，用布林带均值回归策略在中证500 ETF上做了一年，收益12%。他的做法很简单：把资金分成20份，每跌1%买入一份，每涨1%卖出一份，坚决不做"追涨"。他的成功之处不是策略有多精妙，而是严格纪律——在2024年1月极端下跌中他持续买入，然后在4月的反弹中分批卖出。这种"机械执行"的能力，其实比策略本身更稀缺。

25.2 因子策略：系统化的选股框架

如果你觉得技术指标太"短线"，想要更系统化的、有理论支撑的选股框架，因子策略（Factor Investing）是你的不二之选。因子就是那些能够系统性地解释股票收益差异的共同特征。

四大经典因子

因子逻辑常用指标A股表现
价值因子买入被低估的股票PE、PB、PS、PCF长期有效，但2020-2024年连续跑输成长
动量因子买入过去涨得好的股票过去6-12个月收益率A股短周期动量显著，长周期动量弱
质量因子买入盈利能力强的公司ROE、毛利率、资产负债率A股中非常有效，受机构资金青睐
低波因子买入波动较小的股票过去1年日收益率标准差A股低波效应不如美股显著

因子的复合与择时

单一因子往往有"失效期"。比如2020-2021年，价值因子持续跑输成长因子，如果只押注价值因子会非常痛苦。因此实际中通常采用"多因子复合"的方式，把几个低相关的因子结合起来，平滑收益曲线。一个经典的复合方式是：同时使用价值+质量+低波三个因子，等权打分，选出综合排名前20%的股票构建组合。

# 多因子选股：简单打分法
def multi_factor_scoring(df, factors):
    """
    多因子打分选股
    df: 包含各因子值的DataFrame（每行一只股票）
    factors: 因子列名列表
    返回：每个股票的综合得分
    """
    scores = pd.DataFrame(index=df.index)
    scores['股票'] = df['股票']

    for factor in factors:
        # 标准化处理：将因子值转为0-1之间的分位数
        rank = df[factor].rank(method='average')
        normalized = (rank - rank.min()) / (rank.max() - rank.min())
        scores[factor] = normalized

    # 等权打分
    factor_cols = [f for f in factors]
    scores['总分'] = scores[factor_cols].mean(axis=1)

    # 排序
    scores = scores.sort_values('总分', ascending=False)

    return scores

25.3 事件驱动：抓住"催化剂"

基本面研究告诉你"买什么"，交易策略告诉你"什么时候买"。事件驱动策略就是在"催化剂"出现时进行交易。

  • 财报超预期：当上市公司公布的业绩超出市场预期时，股价通常在短期内会有显著的正向反应。研究表明，A股"业绩超预期"效应在财报公布后的5-10个交易日内最为显著。
  • 分红公告：高分红预期的股票在除权除息日前常有"抢权行情"，但需警惕"填权"和"贴权"的分化。
  • 指数调整：当沪深300、中证500等指数调整成分股时，被调入的股票通常有短期上涨（被动基金调仓买入），被调出的则有下跌压力。这个"指数效应"在A股中非常稳定。
  • 并购套利：当一家公司宣布收购另一家公司后，目标公司的股价会向收购价靠拢，但存在交易失败的风险。并购套利的收益来自这个价差，风险来自交易失败。
"踩坑"教训：我见过最惨的事件驱动案例是2023年某游资在"业绩预增公告"出来后追高买入，结果净利润暴增是因为一次性资产处置收益（卖了一套北京的房子），而非主营业务改善。股价在高位横盘两天后暴跌40%。千万要区分"一次性收益"和"持续性改善"——扣非净利润才是衡量公司真实盈利能力的关键指标。

25.4 策略评估体系：别被回测曲线"骗"了

这是最容易被忽视，但也最重要的一节。一个漂亮的回测曲线背后可能藏着无数个"坑"。评估策略是否可靠，至少需要从以下几个维度审视：

夏普比率、索提诺比率、卡玛比率

指标公式关注点
夏普比率(策略收益 - 无风险利率) / 总波动率收益与总风险的权衡
索提诺比率(策略收益 - 无风险利率) / 下行波动率只惩罚下跌风险，忽略上涨波动
卡玛比率年化收益率 / 最大回撤收益与最差情景的权衡

三个指标各有侧重。如果你的策略经常小亏但偶尔"踩雷"暴跌，索提诺比率会帮你发现问题——下行波动率大时索提诺比率会显著低于夏普。如果你关心"最坏情况下亏多少"，卡玛比率更直观——一般卡玛比率大于1算不错，大于2算优秀。

回撤分析：最真实的"压力测试"

回撤（Drawdown）是策略从峰值跌落到谷底的幅度。分析回撤不能只看"最大回撤"这一个数字，还要看：回撤持续了多久（回撤修复期）、回撤发生在什么市场环境下、策略在回撤期间的交易行为是否合理。如果一个策略最大回撤只有5%，但修复期需要3年，那它的实际风险可能比一个最大回撤15%但2个月就修复的策略更大。

滚动夏普：检验策略的稳定性

用全周期算一个夏普比率是"作弊"行为——它掩盖了策略在不同时期的巨大表现差异。正确的做法是计算"滚动夏普"（Rolling Sharpe），比如每12个月算一次夏普比率，观察其时间序列。如果滚动夏普忽高忽低、甚至长时间为负，说明策略的"Alpha"并不稳定。

# 滚动夏普比率计算
def rolling_sharpe(returns, window=252, risk_free=0.0):
    """
    计算滚动夏普比率
    returns: 日收益率序列
    window: 滚动窗口（默认252个交易日）
    risk_free: 无风险利率（默认0）
    """
    rolling_return = returns.rolling(window).mean() * 252 - risk_free
    rolling_vol = returns.rolling(window).std() * np.sqrt(252)
    rolling_sharpe = rolling_return / rolling_vol

    # 统计结果
    result = pd.DataFrame({
        '滚动夏普': rolling_sharpe
    })

    return result

25.5 策略容量与交易成本

很多个人投资者容易忽略一个关键问题：你的策略能容纳多少钱？交易成本会吃掉多少收益？

策略容量：你的资金规模合适吗？

每个策略都有其"容量天花板"。一个基于小市值股票的因子策略，在资金规模从1000万增长到10亿时，其超额收益可能会被交易成本完全吞噬。原因很简单：你买卖的规模越大，对市场价格的冲击越大。高换手率的策略容量天然小于低换手率的策略。

冲击成本和滑点

冲击成本（Impact Cost）是指你的交易本身推动价格朝不利方向移动的幅度。滑点（Slippage）是指你的成交价和预期价格之间的差异。在回测中，很多漂亮的收益数据在扣除了千分之一甚至万分之五的滑点成本后，就变得平庸了。

回测的"潜规则"：做回测时，永远假设你的成交价比信号发出时的价格差0.1%-0.3%（视流动性和策略类型而定）。如果在这个假设下策略收益仍然显著为正，才值得进一步研究。如果你发现某个策略的收益几乎全部来自"开盘价买入、收盘价卖出"的价差，那多半是回测中的"未来函数"在作祟。

换手率与持有期

换手率（Turnover Rate）直接影响交易成本。一个年换手率200%的策略（即每年把整个组合换两遍），假设单边交易成本为千分之一，每年就有0.4%的成本损耗。对于年超额收益只有2-3%的策略来说，这是致命的。低换手率策略（如季度调仓的因子策略）天然更有"容量优势"。

25.6 2025-2026年策略风向

站在2025年年中看未来，交易策略领域有几个不可忽视的趋势：

AI辅助选股：从"手工挖掘"到"AI赋能"

大语言模型（如GPT-4、Claude）在金融文本分析上的能力已经相当惊人。2025年，越来越多的量化机构开始用LLM自动分析财报电话会议记录、提取管理层语调、识别关键风险因素。传统的因子策略正在被"另类数据+AI"的范式所取代。我们将在下一章详细讨论这个主题。

ESG因子配置：从"道德选择"到"风险因子"

ESG投资最初被视为"道德偏好"，但越来越多的研究表明ESG评分高的公司在极端市场事件中的抗跌能力更强。2025年，ESG因子正被纳入主流的多因子模型中，成为继价值、动量、质量、低波之后的"第五因子"。

本章要点回顾：
  • 趋势跟踪（均线、MACD、海龟法则）在趋势市中有效，均值回归（RSI、布林带）在震荡市中有效
  • 四大经典因子：价值、动量、质量、低波，多因子复合比单一因子更稳定
  • 事件驱动策略包括财报超预期、分红公告、指数调整、并购套利
  • 评估策略不能只看夏普比率，还要看索提诺比率、卡玛比率、回撤修复期和滚动夏普稳定性
  • 策略容量和交易成本决定了一个策略能否"实盘存活"，回测中务必预留滑点损耗
  • 2025-2026年AI辅助选股和ESG因子配置是两个最重要的策略演进方向
