第12章 多速率信号处理 (一)

想象你正在设计一个软件无线电（SDR）接收机。天线捕捉到的信号在GHz频段，经过混频下变频后得到一个几十MHz带宽的中频信号。你用了采样率高达122.88MHz的ADC来数字化它。但问题是——你最终想要解调的可能只是一个25kHz窄带FM信号。整条信号处理链如果全部在122.88MHz下运行，FFT和滤波器的MIPS消耗将高得离谱，功耗和发热根本不可接受。

怎么办？一个最自然的想法是：在保证不丢失信息的前提下，把采样率降下来。25kHz的FM信号，理论上只需要50kHz以上的采样率就够了。从122.88MHz降到50kHz，降了超过2000倍——这就是多速率信号处理（Multirate Signal Processing）的核心舞台。

💡 核心洞察：多速率信号处理是一个"用数学技巧省计算量"的领域。它的核心思想很简单：如果一个信号的带宽很窄，你就不需要用很高的采样率来处理它。降采样之后再处理，计算量可以降低几个数量级。在本章中，你将学到三种核心操作——抽取、内插和多相分解，以及它们在工程中的经典应用。

12.1 为什么要改变采样率？

在进入数学推导之前，我们先想清楚"为什么要多速率"这个问题。工程上有几个非常实际的理由：

  • 降低计算量：这是最直接的原因。处理器的运算量正比于采样率——采样率降到1/M，FFT和滤波的计算量也降到约1/M。对于一个降2000倍的SDR系统来说，计算量差异是巨大的。
  • 匹配不同子系统：一个通信系统中，不同模块可能工作在不同的采样率上。ADC工作在122.88MHz，基带处理工作在50kHz，音频输出在48kHz——这之间的桥梁就是采样率转换。
  • 数据压缩：如果信号的有效带宽远低于采样率，降采样就是最有效的无损压缩。
  • 多标准兼容：一个软件无线电平台可能需要处理不同带宽的信号（FM广播200kHz、LTE 20MHz、Wi-Fi 80MHz），多速率结构是唯一可行的方案。

12.2 抽取（Decimation）

抽取就是降低采样率。最简单的抽取就是M倍下采样——每M个样本保留一个，丢弃其余M-1个：

y[n] = x[Mn]    （M倍下采样）

但直接下采样有一个严重问题：频谱混叠。回顾第4章的采样定理：如果信号中包含高于f_s/(2M)的频率分量，这些分量在下采样后会产生不可恢复的混叠。因此，下采样之前必须经过抗混叠低通滤波，截止频率为π/M（归一化频率）。

完整的抽取系统框图：

x[n]  →  [抗混叠LPF: 截止π/M]  →  [↓M 下采样]  →  y[n]
(f_s)         (f_s)                     (f_s/M)

频域视角：抗混叠滤波器把信号带宽限制到π/M以内，然后M倍下采样将频谱扩展M倍，得到一个在0~π范围内占满的频谱。如果省略了滤波步骤，超过π/M的频率分量就会"折叠"回低频区域，产生混叠——听起来就像信号被"折叠"了。

⚠️ 踩坑实录：很多年前我做地震数据采集系统，传感器输出信号带宽0-100Hz，原始ADC采样率是1kHz。为了降低数据量，我直接做了5倍抽取（保留每第5个样本），没有加抗混叠滤波。结果频谱中出现了明显的"假峰"——50Hz工频干扰被混叠到了10Hz附近，和真实的地震波信号混在一起。团队花了近两周分析这个"10Hz异常信号"，最后才发现是工频干扰混叠造成的。从此之后，我做任何抽取操作前都要画一遍混叠检查图。

12.3 内插（Interpolation）

内插是抽取的逆操作——提高采样率。L倍内插的流程是：

  1. 上采样（Upsampling）：在原始样本之间插入L-1个零，得到y_u[n]。此时采样率变为L·f_s，但频谱中包含原始信号频谱的L个镜像副本。
  2. 抗镜像低通滤波：截止频率为π/L的低通滤波器，滤除所有镜像副本，只保留0~π/L范围内的原始频谱。
y_u[m] = { x[m/L],  m = 0, ±L, ±2L, ...
         { 0,       otherwise

    频域：Y_u(e^{jω}) = X(e^{jωL})   （频谱压缩L倍，产生L-1个镜像）

补零上采样的频谱为什么会压缩？从时域理解：补零使得时间轴"拉伸"了L倍，所以频域自然就"压缩"了L倍——压缩意味着产生了L-1个高频镜像副本。

完整的内插系统框图：

x[n]  →  [↑L 上采样(补零)]  →  [抗镜像LPF: 截止π/L]  →  y[n]
(f_s)        (L·f_s)                  (L·f_s)             (L·f_s)

注意一个常见的误解：补零上采样不会引入额外信息，它只是在两个样本之间"插入"了零值，然后通过低通滤波器把这些零变成合理的插值。所以一个好的内插器，本质上是一个好的低通滤波器。

抽取和内插作为多速率信号处理的两大基本操作，它们的对比可以总结如下：

对比项 抽取（Decimation） 内插（Interpolation）
采样率变化 降低（f_s → f_s/M） 提高（f_s → L·f_s）
核心操作 M倍下采样（丢弃样本） L倍上采样（补零）
前置/后置滤波器 前置抗混叠LPF（截止π/M） 后置抗镜像LPF（截止π/L）
频谱变化 频谱扩展M倍 频谱压缩L倍，产生镜像
信息损失 有（丢弃样本不可恢复） 无（补零不增加信息）
主要风险 混叠（Aliasing） 镜像（Imaging）

12.4 有理数采样率转换：先内插再抽取

如果需要的采样率转换因子不是整数，而是一个有理数L/M呢？比如从44.1kHz的CD音频转换到48kHz的数字音频（音频工程中常见的重采样）。

做法很直观：先内插L倍，再抽取M倍。只要中间级的两个滤波器合并成一个低通滤波器，截止频率为π / max(L, M)，采样率变成(L/M)·f_s。

x[n] → [↑L] → [LPF: 截止π/max(L,M)] → [↓M] → y[n]

对于44.1kHz→48kHz的重采样：L=160，M=147。中间采样率为44.1×160 = 7.056MHz，滤波器工作在这样高的频率下，需要几百甚至上千阶FIR。实际实现中会用多相分解来大幅降低计算量（见下一节）。

这正是你手机里每个音频播放器都在做的事情。你每次听歌，音频驱动都在后台做着有理数采样率转换——而你从未感知到它的存在。

12.5 Nobel恒等式

Nobel恒等式是多速率信号处理中最"优雅"的一组恒等式。它们告诉我们：滤波和上下采样的顺序在某些条件下可以交换，从而大幅降低计算量。

恒等式1（对抽取）：
    x[n] → [H(z)] → [↓M]   等价于    x[n] → [↓M] → [H(z^M)]

恒等式2（对内插）：
    x[n] → [↑L] → [H(z)]   等价于    x[n] → [H(z^L)] → [↑L]

直观理解：如果在抽取之前先滤波，滤波器工作在高速率下（f_s），计算量很大。如果先抽取再滤波，滤波器工作在低速率下（f_s/M），计算量减为1/M。但代价是滤波器变为了"多相位"形式H(z^M)——它的实现需要多相分解。

Nobel恒等式的实际意义：永远先做速率转换，再做滤波，只要你能处理对应的滤波器形式。这也是下一节多相分解的数学基础。

12.6 多相分解（Polyphase Decomposition）

多相分解是多速率信号处理中最有实用价值的技术。它的核心思想是：把一个滤波器H(z)分解为M个"子滤波器"（多相分量），每个子滤波器工作在低采样率下。

假设我们有一个FIR滤波器H(z) = Σ h[n]·z^{-n}，长度为N。我们把h[n]按M倍抽取的方式分成M组：

E_k(z^M) = Σ h[nM + k] · z^{-nM},  k = 0, 1, ..., M-1

H(z) = Σ_{k=0}^{M-1} z^{-k} · E_k(z^M)

结合Nobel恒等式，先抽取再滤波的多相结构就变成了：

x[n] → 分成M路子序列 → [↓M] → [E_0(z)] → ┐
                            [↓M] → [E_1(z)] → ├→ 求和 → y[n]
                            ...               │
                            [↓M] → [E_{M-1}(z)] → ┘

每个E_k(z)都是一个工作在低采样率（f_s/M）下的子滤波器，长度约为N/M。原来需要N次乘加/样本（高速率），现在只需要N/M次乘加/样本（低速率）——直接节省了M倍的计算量。

🔧 工程案例：一个实际的例子：假设你有一个256阶FIR做10倍抽取的抗混叠滤波。直接结构（先滤波再抽取）需要每秒做256×f_s次乘加。多相分解后，每个子滤波器约25.6阶（256/10），工作频率f_s/10，总计算量约为25.6×f_s/10×10 = 25.6×f_s——节约了90%的计算量。在嵌入式DSP芯片上，这个差距直接决定了你能不能把算法"塞进"实时处理的时间预算内。

12.7 本章小结

这一章我们做了以下几件事：

  • 通过SDR接收机的案例，论证了多速率信号处理的工程必要性；
  • 详细讲解了抽取（Decimation）——从下采样到抗混叠滤波，并通过一个地震信号的踩坑故事强调了滤波的重要性；
  • 详细讲解了内插（Interpolation）——上采样（补零）和抗镜像滤波，澄清了"补零不增加信息"的常见误解；
  • 介绍了有理数因子L/M的采样率转换（先内插再抽取），以44.1kHz→48kHz音频重采样为例；
  • 介绍了Nobel恒等式——滤波和上下采样的顺序交换规则；
  • 重点讲述了多相分解（Polyphase Decomposition）——将滤波器分解为M个子滤波器，实现在低采样率下运行，以90%的计算量节省案例收尾。

多相分解是这一章最重要的知识点。它把"降采样节省计算"这件事从"可能"变成了"可落地"。下一章我们将在此基础上，深入到具体的多速率系统构件——CIC滤波器、半带滤波器和滤波器组。
