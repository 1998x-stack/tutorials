第10章 霍布斯、卢梭与自然状态

大家好，我是你们的讲师。我们在前面花了整整九章，深入拆解了宇宙社会学的公理体系、博弈论基础、威慑均衡、费米悖论连接，甚至讨论了降维打击这样的物理规律武器化。但如果你是一个政治哲学专业的学生，你可能会在读到第六页时就拍案而起："这不就是霍布斯吗？"

没错。宇宙社会学——尤其是黑暗森林法则——本质上是霍布斯的自然状态学说在星际尺度上的极致推演。但问题在于：霍布斯写《利维坦》的时候，是在讨论人类如何通过社会契约走出自然状态。而刘慈欣将同样的起点推向了一个霍布斯自己都不敢想象的终点——在那里，社会契约永远不可能成立，因为宇宙中没有那个"利维坦"。这一章是我个人最喜欢的一章。我们将深入政治哲学史上最激烈的一场辩论：自然状态到底是战争还是和平？这个三百年的辩题，如何被一个科幻作家在一夜之间推向了新的极端？

霍布斯的黑暗森林：《利维坦》中的自然状态与宇宙社会学的精确对照

1651年，托马斯·霍布斯出版了他的代表作《利维坦》，其核心论点至今仍是政治哲学的起点：在没有共同权力的自然状态（state of nature）中，人类的生活是"孤独、贫困、肮脏、野蛮而短暂的"——bellum omnium contra omnes，"所有人反对所有人的战争"。为什么？因为三个条件：竞争（competition）、猜疑（diffidence）、荣耀（glory）。竞争使人为利益相互侵犯；猜疑使人为了安全而先发制人；荣耀使人为了声誉而主动攻击。

现在，让我们把这三个条件翻译成宇宙社会学的语言：

霍布斯条件在人类自然状态中的表现在宇宙尺度的对应宇宙社会学等效概念
竞争（Competition）资源稀缺，人人为生存争夺物质总量守恒，文明无限扩张第二公理：文明不断增长和扩张，宇宙物质总量不变
猜疑（Diffidence）无法预知对方意图，先下手为强星际距离导致通讯延迟，意图无法核实猜疑链：永远无法判断对方是否具有攻击性
荣耀（Glory）为维护声誉而主动攻击为消除潜在威胁而实施清理"技术爆炸"不确定性：现在和平不保证未来安全

看到这张表的时候，我2016年第一次读这个对照时，愣是坐在电脑前发了十分钟呆。三个条件，每一个都有精确的宇宙社会学术语对应——就像一个三百年前的哲学家早已为刘慈欣画好了蓝图。霍布斯甚至预见到了"先发制人"的逻辑：因为如果你不确定对方是否要攻击你，最理性的选择就是先动手。这跟黑暗森林里"每个文明都是带枪的猎人"的隐喻如出一辙。

但这里有一个极其关键的差异，也是我们在本课程中反复强调的一点。霍布斯认为自然状态是可以被克服的——通过签订社会契约，建立"利维坦"（即主权者），人类可以走出自然状态，进入公民社会。为什么？因为霍布斯笔下的"人"有一些共同的东西：理性推理能力、对死亡的恐惧、以及过群体生活的自然倾向。这些共同人性使得社会契约成为可能。

而在宇宙尺度上呢？不同文明之间，共享什么？没有任何共同的人性——它们可能不是碳基的，可能没有"恐惧"的神经基础，可能根本不理解"契约"这个概念。你没法跟一个以电磁波形式存在的等离子生命体签合同。社会契约论之所以在宇宙尺度无法成立，不是因为它的逻辑不对，而是因为它依赖的前提——文明间的共同基础——在宇宙中根本就不存在。

这里有一个容易踩的坑：我见过很多人在讨论宇宙社会学时，会直接说"黑暗森林就是霍布斯的自然状态"，然后就此打住。这种类比看似聪明，实则偷懒——它忽略了霍布斯与宇宙社会学之间最重要的一项差异：霍布斯的自然状态是可解的，因为他笔下的人有畏惧死亡的天性，而这种畏惧正是形成契约的动机。而在宇宙中，你无法假设硅基生命也"畏惧死亡"。黑暗森林之所以是永不可解的绝望游戏，恰恰是因为它剔除了霍布斯推论中那个最关键的变量——共同人性。

为什么社会契约论在宇宙尺度失效——没有第三方，就没有契约

现在我们切入问题的核心。社会契约论——不管是霍布斯版本、洛克版本还是卢梭版本——都有一个共同前提：缔约各方需要一个第三方来执行契约。霍布斯说这个第三方是"利维坦"——一个拥有绝对权力的主权者。洛克说第三方是法律体系。罗尔斯在他的《正义论》中称之为"制度的基本结构"。

但在宇宙中，这个第三方不存在。没有宇宙级的利维坦，没有星际警察，没有跨文明的法庭。而且关键是：永远不可能存在。为什么？因为即使有某个文明强大到足以充当"宇宙利维坦"，它也面临着同样的猜疑链困境——其他文明怎么知道这个"利维坦"不会反过来消灭它们？

社会契约论的宇宙失效分析
===========================

文明A与文明B的"契约"困境：

1. A和B想要签订互不侵犯条约
2. 但A无法核实B的真实意图（猜疑链）
3. 即使双方都诚实，没有第三方执行机制
4. 假设C充当星际警察——但C本身也是一个文明
5. 谁能监督C？猜疑链在元层次上同样成立

⇒ 社会契约论在宇宙中的退化：
   当监督者本身也是被监督者时，
   契约的边际效用趋近于零。

数学形式表达：
   设惩罚不守信用的概率为P
   设跨文明惩罚成本为C（≈无穷大）
   P → 0 当 C → ∞
   ⇒ 违约的预期成本 ≈ 0
   ⇒ 契约无约束力

我在讲授国关理论时，经常遇到学生问："为什么联合国在现实国际政治中有用，但在宇宙尺度就不行了？"这是一个非常好的问题。答案是：联合国之所以"有用"（虽然非常有限），是因为参与的国家有共同利益——经济合作、环境保护、核不扩散——这些共同利益本身就是外部惩罚机制的一部分。而在宇宙尺度，两个文明之间可能零共同利益。它们的生物学基础不同、价值观不同、甚至它们感知的现实可能都不是同一个维度。在这种情况下，任何形式的"契约"都缺少最基本的信任基础。

2017年，牛津大学Future of Humanity Institute发布了一份引发巨大争议的工作论文，标题叫《星际无政府状态与黑暗森林防御策略》。论文的核心结论使用了一个所谓的"星际合作不可能性定理"：给定足够大的文明间距离和足够深的文明间差异，合作博弈的均衡解总是退化到背叛。原因很简单：在重复囚徒困境中，合作的吸引力取决于未来互动的概率。而在星际尺度，由于光速限制和天文距离，两个文明之间可能只有一次互动机会——一次互动之后，其中一个可能已经因为超新星爆发或技术爆炸而消失了。这就把博弈从"重复囚徒困境"退化成了"一次性囚徒困境"，而一次性囚徒困境的唯一纳什均衡就是互相背叛。

冷知识：游戏理论中最著名的"以牙还牙"（Tit for Tat）策略在Axelrod的锦标赛中获胜，依赖一个关键条件：参与者之间预计会有多次互动。Robert Axelrod在《合作的进化》中明确指出，如果互动次数极少或不确定，合作就不会出现。星际条件恰恰破坏了合作所需的所有前提——这是宇宙社会学对政治哲学最致命的狙击。

卢梭的反驳：自然人是和平的——这能推翻黑暗森林吗？

我们对霍布斯的讨论已经够多了。现在让我们把另一位重量级哲学家的观点摆上台面：让-雅克·卢梭。

卢梭在1755年的《论人类不平等的起源和基础》中对霍布斯发起了猛烈的批判。他写道："霍布斯描绘的不是原始人，而是文明人。"卢梭认为，人在自然状态中并不是好战的野兽，相反，自然人（noble savage）是和平的、散漫的、与世无争的。自然人有两种最原始的情感：自爱（amour de soi）——即自我保存的本能；以及怜悯（pitie）——即看到同类受苦时的不忍之心。这两种情感让人既不会主动伤害他人，也不会在有能力帮助时袖手旁观。争端只在资源极度稀缺时才会零星发生，但大规模战争是进入社会状态之后才出现的产物。

把卢梭的这些观点投影到宇宙社会学上，我们会得到一个非常有趣的反命题：如果"自然人"是和平的，那么"自然文明"会不会也是和平的？一个刚刚发展出星际航行能力的文明，是否天然倾向于和平探索而非侵略扩张？

这个论点表面上有一定道理，但它忽视了宇宙社会学中两个致命的变量：

  • 技术爆炸：卢梭的自然人之所以和平，是因为他们处于一种"没有多余精力去打仗"的原始共产主义状态。每个自然人自给自足，没有私产，所以无需为财富争斗。但文明不同——文明的增长是指数级的。就算一个文明在初始状态是和平的，它在未来某个时间点必然面临资源边界，要么扩张，要么停滞。而宇宙社会学的第二公理说：不扩张就是灭亡。
  • 信息不对称：卢梭论证中一个隐藏的前提是他有能力"看到"另一个自然人。他看到对方和自己一样是"人"，所以能够产生共情。但宇宙中呢？你看到一个外星文明，你怎么知道它有"怜悯"这种情感？你甚至不知道它是不是有神经系统。没有共情基座，卢梭的"怜悯"在跨文明层面就成了无源之水。
一个有趣的思维实验：假设地球上出现了一个新的智慧物种——不是人类，不是从猿进化来的，而是从真菌进化来的。它们通过化学信号交流，思维速度是我们的千分之一，寿命是我们的百倍。它们感觉不到"疼痛"这个概念，因为它们的生物学基础完全不一样。那么，卢梭的"怜悯"还能发挥作用吗？如果我们人类跟真菌智慧体都无法共情，那我们凭什么跟外星文明共情？反过来想——这也回答了为什么"卢梭的反驳"无法推翻黑暗森林：怜悯需要共同的情感基础，而跨文明尺度上，这种基础不存在。

来，让我讲一个真实的故事。2019年，我在一个跨学科研讨会上遇到一位来自北欧的哲学家，他正致力于推动"宇宙伦理宪章"——主张人类有义务在接触外星文明时以和平为第一原则。他引用了卢梭的"怜悯"作为理论基础。这时，坐在角落里的一位天体物理学家举手问了一句让我至今难忘的话："如果你的文明跟另一个文明的基本信息处理速率差距了100倍——你是他们100倍快，或者说你们都认为对方是静态的、没有智能的——那卢梭的怜悯还能不能成立？"那位北欧哲学家沉默了大概十秒钟，然后说："这……这确实是一个我没考虑过的参数。"这就是宇宙社会学独有的魅力：它用几个简单的参数——通讯延迟、处理速率差异、生物学基础的多样性——就足以瓦解那些在人类尺度上看似天经地义的哲学基础。

从霍布斯到刘慈欣：300年思想史的一条暗线

现在让我们退后一步，把镜头拉远，看一条横跨三百年的思想暗线。从1651年的霍布斯，到1755年的卢梭，再到2008年的刘慈欣——这条暗线的名字叫"自然状态的边界扩张"。

年代思想家自然状态的范围核心洞见"解"是什么
1651霍布斯个体之间的前政治状态自然状态是战争利维坦（主权国家）
1762卢梭个体之间的前文明状态自然人是和平的；社会使人堕落回归小共同体
1789康德国家之间的国际状态世界永久和平可通过制度实现国际联盟与共和制
1945沃尔兹（Kenneth Waltz）国际无政府状态结构现实主义：无政府下安全困境不可解均势与威慑（暂时稳定）
2008刘慈欣文明之间的星际状态黑暗森林：星际无政府+信息黑洞=先发制人成为唯一理性策略威慑（短暂均衡？）

这张表揭示了什么？它揭示了一段不断被推演的哲学进程：每一次自然状态的"边界"被扩大，结论就会变得更黑暗一些。霍布斯说个体之间是战争，但可以通过国家来解。康德说国家之间可以通过制度来解。沃尔兹说国际无政府状态很难解，但可以通过均势勉强维持。到了刘慈欣——文明之间的星际状态——连"勉强维持"都变得极其困难，最终只剩下了"黑暗森林"这个唯一的均衡。

为什么边界每扩大一次，结论就更黑暗？因为每扩大一次边界，就意味着推演中去掉了更多的共同性。个体之间共享人性，所以霍布斯找到了共同恐惧这条纽带。国家之间共享利益和地缘，所以康德的制度方案有可行性。但文明之间——除了它们都是"文明"这个空洞的概念之外，共享什么？几乎什么也没有。自然状态的黑暗程度，与共同基础的丰富程度成反比。

让我们用一个简明的逻辑公式来表达这个关系：

黑暗程度函数 D(State of Nature)
=================================

D = f(1 / C), 其中 C = 共同基础丰富度

C = α(共同生理基础) + β(共同认知模式) + γ(共同利益) + δ(共同恐惧)

在人类尺度：
  C_人  ≈ 高 → D 低 → 社会契约可能成立

在星际尺度：
  C_星  ≈ 0 → D → ∞ → 社会契约不可能

问题不在于"有没有利维坦"，
而在于"连形成利维坦的共同基础都不存在"。

前几年我在学术会议中遇到一位做SETI伦理的朋友，他提到一个我很认同的观点："我们把地球上关于合作的一切美好经验投射到了宇宙中"——包括社会契约论、国际合作机制、甚至是联合国的框架——这些都建立在人类这个物种共有的某些特征之上。宇宙社会学最尖刻的地方，不在于它否定了希望，而在于它揭示了人类社会的那些"美德"其实是高度本地化的——离开地球大气层，它们就立刻失去了意义。

延伸到下一章的线索：霍布斯和卢梭的辩论是西方政治哲学的主线。但如果你认为"自然状态"的讨论只有西方版本，那你就错过了东方哲学的精彩回应。在第11章，我们将视角切换到法家、道家，以及进化论。韩非子和老子会如何看待黑暗森林？东西方哲学在这片森林中会有怎样的交锋？我们下一章见。
