当统计特性不再是常数

大家好，我是你们的讲师。回顾第21章和第22章，我们学习了两种经典的最优滤波方法：维纳滤波和卡尔曼滤波。它们的共同点是：需要知道信号和噪声的统计特性（自相关函数、协方差矩阵等），或者至少知道系统模型。如果这些信息完全已知且不随时间变化，维纳滤波就已经是最优线性滤波器了——理论上没有更好的线性方法。

但现实世界很少给你这么好的条件。想象一下：你在一个嘈杂的咖啡馆里打电话，背景噪声不是平稳的——上一秒是咖啡机研磨声，下一秒是旁边桌的聊天声，再下一秒是门铃响。噪声的统计特性在秒级甚至毫秒级尺度上变化。你不可能提前知道这些噪声的自相关函数，更不可能用一个固定的滤波器来应对所有场景。

这就需要自适应滤波——一种能够在线调整自身系数的滤波器，它通过不断观察输入数据和误差信号来自动适应环境的统计变化。本章将介绍两种最经典的自适应算法：LMS（最小均方）和 RLS（递归最小二乘），然后展示它们在噪声抵消、信道均衡和回声消除中的精彩应用。

LMS 算法：简单而实用

从维纳滤波到随机梯度下降

维纳滤波器的解是维纳-霍夫方程：R·w = p，其中 R 是输入自相关矩阵，p 是输入与期望的互相关向量。这个解需要知道 R 和 p——而我们通常不知道。LMS 算法的天才之处在于：它用瞬时值代替统计期望，从而把求解方程组的问题变成了一个简单的递归更新。

具体来说，LMS 算法的迭代公式极其简洁：

# LMS 算法（每个时间步执行一次）
y[n] = w[n]ᵀ · x[n]              # 滤波器输出
e[n] = d[n] - y[n]                # 误差信号
w[n+1] = w[n] + μ · e[n] · x[n]  # 系数更新

其中：

  • x[n] = [x[n], x[n-1], ..., x[n-N+1]]ᵀ 是输入向量
  • w[n] 是滤波器系数向量（长度为 N）
  • d[n] 是期望响应（参考信号）
  • μ 是步长因子（学习率）
  • e[n] 是误差信号

这个更新公式其实就是一个随机梯度下降（Stochastic Gradient Descent, SGD）——朝着降低瞬时误差平方的方向调整系数。它不需要计算任何统计量，每次只用当前一个样本的信息来更新，因此计算量只有 O(N)——跟滤波器的阶数成正比。

步长 μ 的选择艺术

LMS 算法中唯一需要你手工设定的参数就是步长 μ。这个参数的选择直接影响算法的性能：

  • 收敛条件：0 < μ < 2/λ_max，其中 λ_max 是输入自相关矩阵的最大特征值。一个更保守的实用条件：0 < μ < 2/(N · P_in)，其中 P_in 是输入信号的平均功率。
  • 大步长（μ 较大）：收敛速度快，能快速跟踪环境变化，但稳态误差（失调）大，系数在最优值附近剧烈抖动。
  • 小步长（μ 较小）：稳态误差小，系数收敛后精度高，但收敛慢，对突发环境变化反应迟钝。
步长选择的工程经验：在实际项目中，我通常先把 μ 设置得相对较大（比如 0.01），观察误差曲线的下降速度，等算法收敛后再逐步降低 μ 以获得更低的稳态误差。有些系统甚至采用"变步长"策略——收敛初期大步长加速，进入稳态后自动切换为小步长。这种变步长 LMS 在声学回声消除中非常常见。

RLS 算法：快速收敛的代价

递归最小二乘的基本原理

LMS 算法的优点是简单、鲁棒，但它的收敛速度高度依赖于输入信号自相关矩阵的特征值散布（条件数）。如果输入信号是白噪声，特征值全相等，LMS 收敛很快；如果输入信号是高度有色（如语音信号），特征值散布很大，LMS 收敛极慢——可能到天荒地老。

RLS 算法（递归最小二乘）从另一个角度出发：它不是最小化瞬时误差平方的期望，而是最小化从开始到现在所有误差的加权平方和：

# RLS 的代价函数
J[n] = Σ λ^(n-k) · e[k]²     (k = 1 to n)

其中 λ 是遗忘因子（0 < λ ≤ 1）
λ 越小 → 越关注近期数据（快跟踪）
λ 越大 → 越利用历史数据（稳估计）

RLS 的更新公式比 LMS 复杂得多，它需要维护一个逆相关矩阵 P[n] 并递归更新：

# RLS 算法（每个时间步执行一次）
e[n] = d[n] - w[n-1]ᵀ · x[n]    # 先验误差
k[n] = P[n-1]·x[n] / (λ + x[n]ᵀ·P[n-1]·x[n])  # 增益向量
w[n] = w[n-1] + k[n] · e[n]     # 系数更新
P[n] = (P[n-1] - k[n]·x[n]ᵀ·P[n-1]) / λ      # 逆相关矩阵更新

RLS 的核心优势在于：它的收敛速度不受输入信号特征值散布的影响——对于高度有色的语音信号，RLS 可能在几十步内收敛，而 LMS 可能需要成千上万步。这个差距在声学回声消除等应用中至关重要。

LMS 与 RLS 的取舍

但是，RLS 的高性能是有代价的：计算量 O(N²)。对于 N=128 阶的滤波器（声学回声消除的典型值），RLS 每一步需要约 16000 次乘加运算，而 LMS 只需要 128 次。在嵌入式系统或实时应用中，这个差距非常致命。

选择建议：在计算资源充足且收敛速度至关重要时，选 RLS。在实时嵌入式系统或计算资源受限时，选 LMS。如果既想要 RLS 的速度又想要 LMS 的计算量，可以考虑"中间路线"：仿射投影算法 (APA) 或共轭梯度法。另外，NLMS（归一化 LMS）通过将步长除以输入功率来缓解特征值散布问题，在工程中非常实用。

经典应用：自适应滤波的三大战场

自适应噪声抵消

自适应噪声抵消（Adaptive Noise Cancellation, ANC）是自适应滤波最经典的应用之一。它的配置非常巧妙：

  • 主输入：期望信号 s[n] + 噪声 v₀[n]（信号和噪声混在一起）
  • 参考输入：与 v₀[n] 相关、但与 s[n] 不相关的噪声 v₁[n]（通过另一个传感器采集）
  • 自适应滤波器：将参考输入滤波，试图"模拟"出 v₀[n] 的副本
  • 误差输出：主输入减去滤波后的参考输入 → s[n] + (v₀[n] - v̂₀[n])

当自适应滤波器收敛时，v̂₀[n] ≈ v₀[n]，误差输出 ≈ s[n] ——噪声被抵消了！这个方案的一个著名应用是胎儿心电提取：母亲腹部电极采集的信号包含胎儿心电 + 母亲心电 + 噪声，而母亲胸部电极采集的信号主要包含母亲心电。以胸部信号为参考输入，自适应滤波可以从腹部信号中消去母亲心电分量，留下清晰的胎儿心电信号。

自适应均衡

在数字通信中，信号经过多径信道传输后会产生码间干扰（ISI）——当前符号受到前后符号的干扰。自适应均衡器的作用是对接收信号进行逆滤波，补偿信道效应，恢复原始发送符号。

典型的训练模式：发送端先发送一段已知的训练序列，接收端的均衡器以这个已知序列为期望响应进行自适应训练。训练完成后切换到决策导向模式——用检测器的输出作为"伪期望信号"继续跟踪信道的变化。这种训练+跟踪的两阶段策略就是大多数调制解调器和 WiFi 接收机的工作方式。

回声消除

如果你用过免提电话或视频会议，你一定遇到过回声——自己说话的声音从对方那边传回来，延迟后再次听到。这是因为扬声器放出的声音被麦克风再次拾取，形成了一个声学回路。

回声消除器的思路是：利用自适应滤波器估计回声路径的冲击响应（从扬声器到麦克风的声学传递函数），然后从麦克风信号中减去估计出的回声。由于回声路径会因房间内的人员走动、物体移动而改变，滤波器必须持续自适应——这正是 LMS/NLMS 算法大显身手的场景。

有趣的事实：典型的声学回声路径冲击响应可能有几百甚至上千毫秒长——这意味着自适应滤波器的阶数需要达到几千（采样率 16kHz 时，500ms 对应 8000 阶）。这么大的阶数，RLS 的 O(N²) 计算量完全不现实，因此声学回声消除几乎清一色使用 NLMS 算法及其变种。这告诉我们：在工程中，"最优"的算法不一定是"最好"的算法——计算可行性才是第一位的。

Python 实战：LMS 自适应噪声抵消

下面我们用 Python 实现一个完整的 LMS 自适应噪声抵消系统，模拟胎儿心电提取的简化版本。

import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
fs = 500                     # 采样率 500 Hz
T = 5                        # 时长 5 秒
t = np.arange(0, T, 1/fs)
N = len(t)

# 1. 模拟信号源
# "母亲心电"：用一个 1.2Hz 的窄带信号近似
f_maternal = 1.2
maternal_ecg = 0.5 * np.sin(2*np.pi*f_maternal*t)
# "胎儿心电"：用 4Hz 的窄带信号近似
f_fetal = 4.0
fetal_ecg = 0.3 * np.sin(2*np.pi*f_fetal*t + 0.5)

# 2. 模拟传感器采集
# 腹部信号（主输入） = 胎儿心电 + 母亲心电 + 噪声
noise_abd = 0.1 * np.random.randn(N)
primary = fetal_ecg + maternal_ecg + noise_abd

# 胸部信号（参考输入） = 母亲心电（经过一个未知通道）+ 噪声
# 模拟通道：一个简单的 5 抽头 FIR 滤波器
h = np.array([0.8, 0.3, -0.2, 0.1, -0.05])
ref_signal = np.convolve(maternal_ecg, h, mode='same')
ref_signal += 0.05 * np.random.randn(N)

# 3. LMS 自适应滤波器
M = 8                         # 滤波器阶数
mu = 0.02                     # 步长
w = np.zeros(M)               # 滤波器系数初始化
x_buf = np.zeros(M)           # 输入缓冲区

output = np.zeros(N)
error = np.zeros(N)

for n in range(N):
    x_buf = np.roll(x_buf, 1)
    x_buf[0] = ref_signal[n]
    y = np.dot(w, x_buf)                    # 滤波输出
    e = primary[n] - y                      # 误差 = 估计的胎儿心电
    w = w + 2 * mu * e * x_buf              # LMS 更新
    output[n] = y
    error[n] = e

# 4. 绘制结果
plt.figure(figsize=(12, 10))
plt.subplot(411); plt.plot(t, primary);
plt.title('腹部信号（主输入：胎儿心电 + 母亲心电 + 噪声）')
plt.subplot(412); plt.plot(t, ref_signal);
plt.title('胸部信号（参考输入：母亲心电经信道）')
plt.subplot(413); plt.plot(t, output);
plt.title('自适应滤波器输出（估计的母亲心电）')
plt.subplot(414); plt.plot(t, error, label='消噪后');
plt.plot(t, fetal_ecg, '--', alpha=0.7, label='真实胎儿心电');
plt.title('误差信号 = 估计的胎儿心电')
plt.legend(); plt.xlabel('时间 (秒)')
plt.tight_layout()
plt.show()

print(f"最终滤波器系数: {w}")

运行这段代码，你会看到：自适应滤波器在初始阶段有一个"学习过程"（误差信号开始时较大），大约 1-2 秒后收敛，随后误差信号几乎完全跟踪上了真实的胎儿心电波形。这说明自适应滤波器成功地"学会"了母亲心电从胸部到腹部的传输路径，并从腹部信号中抵消了它。

踩坑故事：输入功率波动导致的 LMS 发散

我曾参与过一个助听器自适应反馈抑制系统——助听器的麦克风和扬声器靠得很近，扬声器放大的声音会被麦克风再次拾取，形成刺耳的啸叫（反馈）。自适应滤波器的作用是估计这个反馈路径并实时抵消。

系统在实验室里的安静环境下测试时，一切正常。但到了真实用户场景——佩戴者在嘈杂的餐厅里用餐——问题出现了：自适应滤波器突然发散，输出信号爆炸式增长，比原来的反馈啸叫还响，用户被吓了一大跳。

排查后发现，原因在于 LMS 的步长 μ 是在实验室里凭借安静环境下的输入功率设定的。到了餐厅环境，麦克风拾取的信号功率大了好几倍（因为周围人声、碗碟碰撞声等），而 LMS 的收敛条件 μ < 2/(N · P_in) 中的 P_in 变大后，原来设定的 μ 就不再满足收敛条件了。这导致了 LMS 系数的发散——系数越跳越大，滤波器输出也跟着爆炸。

教训：标准 LMS 对输入信号的功率变化非常敏感。解决方案是使用归一化 LMS（NLMS）——将步长除以输入信号的瞬时功率（再加一个小常数防止除零）： 
# NLMS 更新公式
w[n+1] = w[n] + μ · e[n] · x[n] / (||x[n]||² + ε)
NLMS 的步长自动适应输入功率变化，在实际工程中几乎总是优于标准 LMS。如果你在用自适应滤波，请默认选择 NLMS而不是 LMS——除非你有非常特殊的理由不这么做。从那以后，我在所有自适应滤波项目中都默认使用 NLMS，再也没有发生过类似发散事故。

本章小结

本章我们学习了自适应滤波的核心思想：当信号统计特性未知或时变时，让滤波器在线学习最优系数。LMS 算法以 O(N) 的超低计算量实现了这个目标，但收敛速度受信号特征影响；RLS 算法牺牲计算量换来了更快的收敛速度。三种经典应用——噪声抵消、信道均衡、回声消除——展示了自适应滤波在通信、生物医学和消费电子中的广泛用途。

从第20章的估计理论到第21-22章的维纳/卡尔曼滤波，再到第23-24章的功率谱估计，最后到本章的自适应滤波——我们走过了"已知统计 → 未知统计"、"离线 → 在线"、"平稳 → 时变"的方法演进路径。这些工具构成了随机信号分析的完整工具箱。
