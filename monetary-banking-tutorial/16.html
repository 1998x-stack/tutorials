引子：你手里为什么要有钱？

问一个看似简单的问题：你为什么不把所有资产都换成股票、基金、房产或黄金，而是要留一部分现金在手里？

答案似乎很直白：因为你要花钱啊。买菜要钱、坐车要钱、交房租要钱……但你有没有想过，现代金融体系下，你几乎可以用信用卡、借记卡、手机支付完成绝大多数交易。理论上你只需要在还款日前卖掉一点资产就行了——为什么还要持有"不生息"的现金呢？

这个问题就是货币需求理论的核心。它探讨的不是"货币供给从哪来"（那是上一章的内容），而是"经济主体为什么想要持有货币"。你、你的公司、你的国家——为什么宁愿放弃利息收益也要保留一部分购买力以货币形态存在？

这一章，咱们要梳理从古典到现代的四条主要理论思路，看看经济学家们是怎么回答这个问题的。

古典货币需求理论

古典货币需求理论有两个代表性表述，都诞生于20世纪初，但从不同的角度切入：

费雪交易方程（MV=PT）

美国经济学家欧文·费雪在1911年提出了著名的交易方程：

MV = PT

其中，M是货币数量，V是货币流通速度，P是物价水平，T是交易总量（实际产出）。

费雪的思路非常"工程师"——他把货币看成是经济交易的"润滑剂"。一定时期内，社会中发生的交易总量是PT，完成这些交易需要的货币量是M，而每一块钱在单位时间内"转手"的次数就是V。所以MV必须等于PT，这是恒等式。

但费雪的贡献不只是列出这个恒等式。他做了一个关键假设：V和T在短期内是稳定的。V取决于支付习惯（发薪周期、支付技术等），不会频繁变动；T由充分就业下的实际产出决定，短期内也不变。那么，结论就出来了：M的变化将直接导致P的同比例变化——这就是古典货币数量论的灵魂。

剑桥方程（M=kPY）

几乎同时期，英国剑桥大学的马歇尔和庇古从另一个角度思考了这个问题。他们问的不是"货币怎么周转"，而是"人们为什么愿意持有货币"——这是个微观经济学的视角。

剑桥方程写成：

M = kPY

其中，k是"马歇尔k"——人们愿意以货币形式持有的收入比例。Y是实际收入。PY就是名义收入。

如果费雪方程和剑桥方程在数学上可以推导互通（k = 1/V），为什么还要单独提剑桥方程？因为视角不同：

  • 费雪方程强调的是货币作为交易媒介——关注的是货币的"流速"和"流量"。
  • 剑桥方程强调的是货币作为资产（财富储存手段）——关注的是人们"想持有多少货币余额"这个存量决策。

剑桥方程暗含了一个更丰富的分析框架：k不是常数，它取决于持有货币的机会成本（利率）、对未来收入的预期、对物价变动的预期等。这就为后来的凯恩斯革命埋下了伏笔。

// 费雪方程 vs 剑桥方程：数学推导与对比
// ========================================

// 费雪交易方程
// MV = P × T
// 其中 T = 交易总量（包括中间品交易）

// 剑桥方程
// M = k × P × Y
// 其中 Y = 最终产出（GDP概念）

// 关联推导：
// 如果只考虑最终交易，T ≈ Y，则：
// MV = PY  →  M = (1/V) × PY
// 即 k = 1/V
// 剑桥的 k 就是费雪 V 的倒数

// 关键区别：
// 费雪：V 由制度因素决定，短期内不变
// 剑桥：k 由人们的资产选择决定，可能变化

// 政策含义差异：
// 费雪 → MV = PT → 如果M↑，P↑，V和T不变
// 剑桥 → M = kPY → 如果M↑，P↑或者k↓或者Y↑
// 剑桥方程给了货币变动更多"出口"

凯恩斯流动性偏好理论

1936年，凯恩斯在《就业、利息和货币通论》中提出了对货币需求的革命性解释——流动性偏好理论。凯恩斯认为，人们持有货币（而不是其他资产）的原因有三个：

交易动机：为了日常交易而持有货币。你今天要买菜、交水电费、还信用卡，这些支出和收入之间有时差，所以你得在手里留一些现金。凯恩斯认为，交易性货币需求与收入水平正相关——收入越高，交易规模越大，需要的货币越多。

预防动机：为了应对意外支出而持有货币。你可能会生病、失业、突然需要修车……未来是不确定的，手里留点现金能让你睡得更安稳。预防性货币需求也主要与收入正相关。

投机动机：为了捕捉投资机会而持有货币。这是凯恩斯最原创性的贡献。凯恩斯注意到，当利率很低时，债券价格很高，人们预期利率未来可能上升（债券价格下跌），所以宁愿持有现金而不去买债券，等待更好的买入时机。投机性货币需求与利率负相关——利率越低，持有货币的机会成本越小，投机需求越大。

凯恩斯将三种动机合并，得到货币需求函数：

M / P = L(Y, r)

其中，实际货币余额需求M/P与收入Y正相关，与利率r负相关。

⚠️ 凯恩斯的革命性： 在凯恩斯之前，古典理论认为货币需求只与收入（或交易量）有关，基本不考虑利率的影响。凯恩斯引入了"投机动机"，建立了货币需求与利率的负相关关系——这意味着，货币政策可以通过影响利率来影响货币需求，进而影响实体经济。这个洞见是后续所有货币政策理论的基石。

凯恩斯还提出了一个极端的可能性——流动性陷阱。当利率降到极低时，所有人都会预期利率上升，于是"投机性货币需求"趋于无穷大。换句话说，无论央行投放多少货币，都会被人们以现金形式持有没有，利率不会再下降，货币政策完全失效。这个概念在理解日本"失去的三十年"时非常关键。

弗里德曼现代货币数量论

到了1950-60年代，米尔顿·弗里德曼在吸收凯恩斯理论的基础上，对古典货币数量论做了现代化改造，提出了现代货币数量论。

弗里德曼的货币需求函数比凯恩斯的复杂得多：

M / P = f(Yp, rb - rm, re - rm, π^e - rm, ...)

其中，Yp是永久收入（而非当期收入），rb、re是债券和股票的预期收益率，π^e是预期通胀率，rm是货币本身的收益率。

弗里德曼对货币需求有三个核心判断：

1. 永久收入假说：决定货币需求的主要不是当期收入，而是"永久收入"——你预期自己长期平均能赚多少钱。当期收入的波动（比如疫情导致的临时降薪）不会显著影响货币需求，只有永久收入的变化才会。这导致货币需求函数非常稳定。

2. 货币需求对利率不敏感：弗里德曼认为，货币本身也提供"收益"（比如银行提供的便利服务、安全性），当债券利率上升时，人们可以通过调整持有的货币"质量"（比如在存款和现金之间转换）来应对，所以货币需求对利率变化不敏感。这个结论和凯恩斯正好相反。

3. 货币需求函数是稳定的：既然货币需求主要取决于永久收入（稳定变量），对利率又不敏感，那么货币需求就是一个可以预测的稳定函数。这意味着——货币供给的变化会直接反映到价格和名义收入上。这就是弗里德曼著名的论断"通货膨胀永远是货币现象"的理论基础。

💡 货币主义 vs 凯恩斯主义： 如果你对比这两个流派，核心分歧在于：凯恩斯认为货币需求不稳定（受"动物精神"和预期影响），所以货币政策的传导机制是利率→投资→产出；弗里德曼认为货币需求稳定，所以货币供给本身直接决定名义收入——不需要经过利率这个中间变量。这一分歧直接影响了两个学派对"央行应该怎么做"的看法。

货币流通速度

在讨论了各种货币需求理论之后，咱们来看一个实证指标——货币流通速度（V）。它是将理论和现实连接起来的关键桥梁。

V = GDP / M2

这个指标衡量的是：每1元货币在一年中参与了GDP创造的"周转次数"。如果V=2，意味着1元钱在一年内被花了两次；如果V=0.5，意味着1元钱在一年内只被花了半次（即大量货币沉淀下来没有参与交易）。

从实证数据来看，中美两国的货币流通速度都呈现长期下降的趋势：

  • 中国：1990年代的V约为1.2左右；到2025年，V已降至约0.4-0.5。这意味着M2的增长速度远超GDP增速，大量货币被"窖藏"起来而没有进入流通。
  • 美国：从1960年代的约1.8降至2020年代的约1.1。2008年金融危机后，V的下降尤为明显。

货币流通速度下降意味着什么？如果V在下降，即使M2增速很快，GDP增长也可能不及预期。这正是2022-2023年中国出现的情况：M2增速维持在10%以上，但GDP实际增速只有5%左右，名义GDP增速约6-7%，中间的"缺口"就是被货币流通速度下降吸收了。

从理论角度看，货币流通速度的下降反映了：

  • 金融深化：随着金融体系发展，货币的"窖藏"需求增加（人们持有更多金融资产而不是消费）。
  • 不确定性上升：经济前景不明时，预防性储蓄增加，货币流动放慢。
  • 收入分配恶化：高收入群体的边际消费倾向低，他们的货币沉淀效应更强。

下表总结了四种货币需求理论的核心对比：

理论 代表人物 核心公式 决定货币需求的主要因素 货币需求对利率是否敏感 政策含义
费雪交易方程 费雪 MV=PT 交易量T（收入） 否 货币供给→物价
剑桥方程 马歇尔、庇古 M=kPY 名义收入、k值 隐含可能是 k的稳定性是关键
凯恩斯流动性偏好 凯恩斯 M/P=L(Y,r) 收入+利率+预期 是（投机动机） 利率传导机制、流动性陷阱
弗里德曼现代货币数量论 弗里德曼 M/P=f(Yp, rb-rm, ...) 永久收入 否（极不敏感） 货币供给固定增长规则

踩坑故事：日本"失去的三十年"

理论总是抽象的。咱们来看一个真实的、极端的案例——日本。

1990年代初，日本资产泡沫破裂，股市崩盘、房地产暴跌。此后日本陷入了长达三十年的低增长、低通胀（甚至通缩）的泥潭。在这三十年中，日本央行做了几乎所有能做的事：

  • 把政策利率从8%一路降到0%（1999年就实现了"零利率"）。
  • 实施多轮量化宽松（2001年全球第一个搞QE的央行）。
  • 大量购买国债、ETF、REITs。
  • 推出收益率曲线控制（YCC，2016年）。
  • 把央行资产负债表扩张到GDP的130%以上（全球最高）。

但结果呢？三十年来，日本平均通胀率只有0.2%左右，经济增长长期在1%以下徘徊。货币需求理论告诉我们——即使在零利率下，日本的货币需求几乎是"无限"的。企业和居民对经济前景极度悲观，根本不想借钱消费或投资，所有的流动性都被以存款或现金的形式"吸收"了。

有一位日本央行官员曾在2013年接受采访时感叹："我们就像在推一根绳子。无论我们向银行系统注入多少流动性，它都流不到实体经济中去。银行不缺钱，企业不缺钱——他们缺的是信心。"

这正是凯恩斯流动性陷阱的完美写照。在流动性陷阱中，货币需求的利率弹性趋近于无穷大——利率已降到零，但人们仍然无限度地持有货币，任何额外的货币供给都被"吞噬"了，无法刺激实体经济活动。

日本央行前行长白川方明在回忆录中写道：2013年日本实施"量化质化宽松"（QQE）后，央行购买了大量政府债券，但银行只是把卖债获得的资金存在央行的准备金账户上，几乎没有增加贷款。货币乘数从1990年的约10倍降到了2010年后的约2-3倍。这不是银行不愿意放贷，而是企业和居民不愿意借贷——面对持续通缩和人口萎缩，投资预期回报率是负数，谁还会借钱？

⚠️ 教训： 日本案例告诉我们，货币供给和货币需求是硬币的两面。不论央行印多少钱，如果公众和企业对货币的需求无限大（因为悲观、因为不确定性、因为负的预期回报率），货币供给扩张的效果就会大打折扣。这也是为什么理解货币需求理论对评价货币政策至关重要——你永远要问：钱印出来了，有人需要它吗？

理解了货币供给和货币需求，咱们就有了分析货币市场的完整架构。但央行具体是怎么操作的呢？它用哪些"工具箱"里的工具来影响货币供给和利率？这些工具在中国和在美国又有什么不同？下一章，咱们进入货币政策工具与操作，看看央行的"十八般武艺"。
