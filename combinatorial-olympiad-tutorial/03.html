第3章：二项式定理与多项式

大家好，我是你们的讲师。上一章我们学了组合数和一大堆恒等式——你可能觉得它们在"硬记"，彼此之间似乎没什么关联。今天我们就来揭开它们之间的联系面纱。二项式定理就是那把钥匙。

二项式定理 (a+b)^n 的展开式早在初中就学过，但你可能只把它当作一个代数公式来对待。今天我要告诉你的是：二项式定理本质上是一个组合恒等式——它只是用多项式语言重新表述了组合计数。一旦你看穿了这一层，组合数学和代数之间的防火墙就被打通了，之前那些要"背"的恒等式全都可以自然而然地推导出来。

3.1 (a+b)^n 展开的组合意义

先回到最基本的：

(a+b)^n = (a+b)(a+b)...(a+b)  共 n 个因子

展开这个式子时，我们在做什么？从每个因子 (a+b) 中，我们要选 a 或选 b 作为贡献。最后得到的每一项形如 a^{n-k} × b^k——其中 k 是选择 b 的因子个数。

那么，有多少种方式得到 a^{n-k}b^k 呢？就是从 n 个因子中选出 k 个来取 b（剩下的 n-k 个自动取 a），方案数正是 C(n,k)。所以：

(a+b)^n = ∑_{k=0}^{n} C(n,k) × a^{n-k} × b^k

这个视角揭示了二项式定理的本质——展开式的系数就是组合数。每一个组合恒等式，只要写成二项式系数的形式，都可以从二项式展开中"读"出来。

🔥 关键洞察：从此以后，再看到 (a+b)^n 不要只想到"代数公式"。要想到"n个因子的乘法分配律"和"选b的k个位置的组合计数"——这两者是同一枚硬币的两面。

3.2 赋值法求系数和

二项式定理中最强大的技巧之一就是赋值法——给 a 和 b 赋予特定的数值，从而"提取"出想要的系数组合。

3.2.1 经典赋值

赋值 所得结果 含义
a=b=1 ∑ C(n,k) = 2^n 全部系数之和
a=1, b=-1 ∑ (-1)^k C(n,k) = 0 交错系数和为零
a=2, b=1 ∑ 2^{n-k} C(n,k) = 3^n 加权系数和
a=1, b=2 ∑ 2^k C(n,k) = 3^n 对称的加权和

这四个赋值覆盖了联赛中最常见的一类题目。例如 2018 年全国高中数学联赛就考过这样一道题：

"已知 (1+x)^n = ∑_{k=0}^{n} a_k x^k，求 ∑_{k=0}^{n} 3^k a_k 的值。"

这本质上就是把 x=3 代入 (1+x)^n，得 (1+3)^n = 4^n。在考场上，这就是 10 秒钟的事——但前提是你对赋值法足够熟练。

3.2.2 求导赋值

当系数的前面带着 k 或 k² 这样的因子时，光赋值还不够，需要借助求导：

对 (1+x)^n = ∑ C(n,k) x^k 两边求导：
n(1+x)^{n-1} = ∑ k·C(n,k)·x^{k-1}

令 x=1，得 ∑ k·C(n,k) = n·2^{n-1}

再次求导（或先乘以 x 再求导），可得：
∑ k²·C(n,k) = n(n+1)·2^{n-2}

这就证明了我们在第二章中给出的两个加权和恒等式。不需要背，会求导就行。

3.2.3 积分赋值

当系数以 1/(k+1) 的形式出现时，可以考虑积分：

∫₀¹ (1+x)^n dx = ∫₀¹ ∑ C(n,k) x^k dx = ∑ C(n,k)/(k+1)

左边 = [(1+x)^{n+1}/(n+1)]₀¹ = (2^{n+1} - 1)/(n+1)

所以 ∑ C(n,k)/(k+1) = (2^{n+1} - 1)/(n+1)

赋值、求导、积分——这三个工具结合起来，几乎可以处理任何形式的组合求和。

💡 实战建议：在解决一道组合求和问题时，先观察求和项的形式：

  • 纯 C(n,k) —— 直接赋值
  • k·C(n,k) —— 求导一次
  • k²·C(n,k) —— 求导两次或使用组合技巧
  • C(n,k)/(k+1) —— 积分
  • C(n,k)·m^k —— 令 x=m 代入 (1+x)^n

3.3 多项式定理的推广

二项式定理处理的是两个变量的展开。如果是三个、四个甚至更多个变量呢？

3.3.1 多项式定理（Multinomial Theorem）

(x₁ + x₂ + ... + x_m)^n
= ∑_{k₁+k₂+...+k_m = n} [n!/(k₁! k₂! ... k_m!)] × x₁^{k₁} x₂^{k₂} ... x_m^{k_m}

这里的系数 n!/(k₁!k₂!...k_m!) 称为多项式系数，记作 C(n; k₁, k₂, ..., k_m) 或 nCk₁,k₂,...,k_m。

它的组合解释也很有趣：把 n 个不同的物品分成 m 组，第 i 组恰好有 k_i 个物品，且各组内部不排序——这就是多项式系数的计数意义。堆叠因子 n! 是全排列，而每个 k_i! 是除以组内的顺序重复。

3.3.2 三项式特例

(a+b+c)^n = ∑_{i+j+k=n} C(n; i, j, k) × a^i b^j c^k

其中 C(n; i, j, k) = n! / (i! j! k!)

例如 (a+b+c)^2 展开：系数分别为 C(2; 2,0,0)=1, C(2; 0,2,0)=1, C(2; 0,0,2)=1, C(2; 1,1,0)=2, C(2; 1,0,1)=2, C(2; 0,1,1)=2。所以你得到 a²+b²+c²+2ab+2ac+2bc。

3.3.3 多项式系数的几组恒等式

和二项式系数一样，多项式系数也有自己的恒等式体系：

∑_{k₁+...+k_m=n} C(n; k₁, ..., k_m) = m^n           (全部系数之和)
C(n; k₁, ..., k_m) = C(n-1; k₁-1, k₂, ..., k_m) + ... + C(n-1; k₁, ..., k_m-1)   (Pascal型递推)

第一个恒等式的解释：把 n 个不同的物品放入 m 个不同的盒子（可以空），总共有 m^n 种方法。左边是对各盒子的物品数 (k₁,...,k_m) 求和，每种的放法数就是多项式系数 C(n; k₁,...,k_m)。

3.4 用二项式定理证明组合恒等式

现在我们展示二项式定理作为"恒等式生成器"的强大威力。以下三个例子都是联赛/CMO级别的经典题型。

3.4.1 例1：两个二项式的卷积

等式两边都是两个二项式乘积的展开系数，利用 Vandermonde 卷积：

(1+x)^m (1+x)^n = (1+x)^{m+n}
[∑ C(m,i) x^i] × [∑ C(n,j) x^j] = ∑ C(m+n, k) x^k

比较 x^k 的系数：
∑_{i=0}^{k} C(m,i) C(n, k-i) = C(m+n, k)

这就给出了第二章中 Vandermonde 卷积恒等式的一个纯代数证明。

3.4.2 例2：证明 ∑ (-1)^k C(n,k) C(2k, k) = 0

这看起来是一个不太容易直接构造组合模型的恒等式。我们用生成函数法（其实还是二项式定理的延伸）：

∑_{k=0}^{n} (-1)^k C(n,k) C(2k, k) 4^{n-k}
= 4^n ∑ (-1)^k C(n,k) C(2k,k) / 4^k

然后利用 C(2k,k) 的生成函数：∑_{k≥0} C(2k,k) x^k = 1/√(1-4x)。但这是后续章节的内容——先留个悬念。

3.4.3 例3：封闭形式求和

求 ∑_{k=0}^{n} (-1)^k C(n,k) / (k+1)

解法：考虑 ∫₀¹ (1-x)^n dx
= ∫₀¹ ∑ (-1)^k C(n,k) x^k dx
= ∑ (-1)^k C(n,k) / (k+1)

而直接积分 ∫₀¹ (1-x)^n dx = 1/(n+1)

所以 ∑ (-1)^k C(n,k)/(k+1) = 1/(n+1)

太漂亮了——一个看起来复杂的求和，通过积分和展开，3行就给出了答案。

🔥 核心方法总结：用二项式定理证明恒等式的套路是：

  1. 写出二项式展开：如 (1+x)^n = ∑ C(n,k) x^k
  2. 对两边进行某种操作：赋值、求导、积分、乘另一个展开式
  3. 比较两边同一 x^k 项的系数
  4. 系数相等 → 恒等式成立

这个模式之所以强大，是因为它把组合问题转化成了代数问题——而代数是有章可循的。

3.5 实战中的踩坑与进阶技巧

3.5.1 综合题型：联赛真题分析

来看一道 2015 年全国高中数学联赛加试中的组合问题：

"设 n 是正整数，证明：∑_{k=0}^{n} C(2n, 2k) = 2^{2n-1}"。

看起来平淡无奇，但很多同学第一步就写错了——直接把 C(2n,2k) 代入 (1+x)^{2n} 展开，然后发现 x=1 和 x=-1 消不掉奇数项。正确做法是：

(1+1)^{2n} = ∑_{k=0}^{2n} C(2n, k) = A + B
(1-1)^{2n} = ∑_{k=0}^{2n} (-1)^k C(2n, k) = A - B

其中 A = ∑ C(2n, 2k)（偶数项和），B = ∑ C(2n, 2k+1)（奇数项和）

两式相加：2A = 2^{2n} + 0，所以 A = 2^{2n-1}

要点是：利用 (1+1)^n 和 (1-1)^n 这对组合来分离奇偶项。这个方法可以推广到分离模 3、模 4 等更复杂的情况——只需引入单位根（复数根）：

令 ω = e^{2πi/3} = -1/2 + √3i/2
(1+1)^n + (1+ω)^n + (1+ω²)^n = 3 × ∑_{k≡0 (mod 3)} C(n,k)

这就是"三项和法分离模3"的技巧。在 CMO 级别的题目中，这种技巧经常被用到。

⚠️ 踩坑实录：有一年我辅导的学生参加省队选拔，遇到了这样一道题："求 ∑_{k=0}^{n} C(n,k)² 的值。"

这个学生看到 C(n,k)² 就懵了，硬套公式算了半天没算出来。其实这题就是 Vandermonde 卷积的特殊情况：

∑ C(n,k)² = C(2n, n)。

他为什么没看出来？因为他把注意力放在了"平方"上，忘了平方就是 C(n,k) × C(n,k)——而 (1+x)^n (1+x)^n = (1+x)^{2n} 展开时，x^n 的系数正是 ∑ C(n,k) C(n, n-k) = ∑ C(n,k)²。

所以教训是：看到 C(n,k)²、C(n,k)C(m,k) 这类乘积，第一时间想能不能用 Vandermonde 卷积。不是所有组合恒等式都需要从零推——"识别模式"本身就是一个重要技能。

3.5.2 一个有趣的现实类比

二项式定理在现实中有很多应用。比如在密码学中，一个 n 位的二进制密钥的总可能组合数是 2^n——这正是 (1+1)^n 展开。在遗传学中，一对杂合子父母（Aa × Aa）的后代基因型分布为 AA:Aa:aa = 1:2:1——系数就是 C(2,k) 的序列 1,2,1。在概率论中，二项分布 P(X=k) = C(n,k) p^k (1-p)^{n-k} 正是 (p+(1-p))^n 展开的各项。所以二项式定理不仅仅是一个代数公式——它是理解离散世界的数学语言。

3.6 本章小结与过渡

今天我们打通了组合数与多项式之间的桥梁。我们从 (a+b)^n 的组合意义出发，学习了赋值法、求导法、积分法等在组合和求值中的运用，了解了多项式定理及其系数，并通过大量实例展示了二项式定理在组合恒等式证明中的核心地位。

学完这一章，你应该能感受到：组合数学不是孤立的"计数技巧"，而是一套和代数、分析紧密相连的数学语言。下一章我们要讲容斥原理——当计数问题中出现"至少"、"恰好"、"不满足"等条件时，容斥原理就是最有力的武器。它和二项式定理的结合会碰撞出更精彩的火花。我们第四章见。
