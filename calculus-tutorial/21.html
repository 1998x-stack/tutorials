从一元极值到多元极值

大家好，我是你们的讲师。欢迎回到微积分课堂。

回想一下一元函数中我们怎么找极值的：先求导数 f'(x) = 0 找到驻点（stationary point），再用二阶导数 f''(x) 的符号判断是极大值、极小值还是拐点。整个过程简洁、优雅，像一条流水线。

到了多元函数，情况复杂了很多。一个二元函数 z = f(x, y) 的"极值"概念本身就更有层次——一个点可以在 x 方向是"峰"、在 y 方向是"谷"，形成所谓鞍点（saddle point），就像马鞍的形状。你站在鞍点上，沿马背方向看是"下坡"，沿马腹方向看是"上坡"。这种在不同方向上一阶偏导全为零但既非极大也非极小的点，在一元函数里不存在——这是多元极值独有的丰富性。

我对鞍点的第一次深刻印象来自大学数学竞赛。当时的题目是这样的：求 f(x, y) = x² - y² 的极值。我算出两个偏导为零的点只有 (0, 0)，然后想当然地认为偏导为零就是极值点——结果错了。老师在课堂上用一张马鞍图展示：z = x² - y² 在 (0, 0) 处沿 x 方向是极小，沿 y 方向是极大，两者互相"拉扯"，实际上那里既不是极大也不是极小。全场鸦雀无声，大家都被这个现象震撼到了。

这就是本章的核心：如何正确判定多元函数的极值。我们会从无条件极值（无约束的自由优化）开始，再到条件极值（带约束的优化），最后介绍拉格朗日乘数法——一个在工程、经济、机器学习中无处不在的强大工具。

无条件极值：驻点与必要条件

对于二元函数 z = f(x, y)，极值的必要条件和一元的逻辑完全一致：在极值点处，所有一阶偏导数都必须为零。

如果 (x₀, y₀) 是 f(x, y) 的极值点（且 f 在该点可微），则：
        ∂f/∂x (x₀, y₀) = 0
        ∂f/∂y (x₀, y₀) = 0

满足这两个方程的点称为函数的驻点（stationary point）或临界点（critical point）。它的几何意义是：在驻点处，曲面的切平面是水平的（即与 xy 平面平行）。这是因为切平面的法向量是 (∂f/∂x, ∂f/∂y, -1)，当两个偏导都为零时，法向量为 (0, 0, -1)，指向正下方，所以切平面水平。

但和一元的 f'(x) = 0 一样，这只是必要条件而非充分条件。驻点可能是极小值点、极大值点，也可能是鞍点。如何区分？我们需要引入二阶信息。

步骤回顾：找多元极值的基本流程
1. 求解方程组 f_x = 0, f_y = 0，找到所有驻点
2. 对每个驻点，用二阶偏导数构造 Hessian 矩阵
3. 判断 Hessian 矩阵的"正定性"或计算判别式 AC - B²
4. 结合实际问题确定是极大、极小还是鞍点
5. 如有边界条件，还需检查边界上的最值

二阶判别：Hessian 矩阵与 AC - B²

一元函数中我们靠 f''(x) 的符号进行二阶判别。二元函数中，二阶偏导数不止一个——我们有 f_xx、f_yy、f_xy（即 ∂²f/∂x²、∂²f/∂y²、∂²f/∂x∂y）。它们被组织成一个 2×2 矩阵，称为Hessian 矩阵（Hessian matrix）：

H = [ f_xx   f_xy ]
            [ f_yx   f_yy ]

这个矩阵是二阶导数的多元推广。它的特征值正负决定了驻点的性质。但在实际计算中，我们通常不直接求特征值，而是使用一个等价的判别量：AC - B²（这个命名来源于常见的记号 A = f_xx、B = f_xy、C = f_yy）。

判别法则如下：

条件 结论 几何意义
AC - B² > 0 且 A > 0 极小值点 曲面在该点处向上弯，像一个碗
AC - B² > 0 且 A < 0 极大值点 曲面在该点处向下弯，像一个倒扣的碗
AC - B² < 0 鞍点 一方向上弯另一方向下弯，像马鞍
AC - B² = 0 无法确定 需要更高阶的判别方法

这个 AC - B² 到底是什么？它就是 Hessian 矩阵的行列式 det(H)。所以你也可以这样记：

  • det(H) > 0 且 f_xx > 0 → 极小值
  • det(H) > 0 且 f_xx < 0 → 极大值
  • det(H) < 0 → 鞍点
  • det(H) = 0 → 需要更高阶分析

让我们通过一个完整的例子来走一遍流程。

求 f(x, y) = x³ - y³ + 3xy 的极值。

        第一步：求驻点
        f_x = 3x² + 3y = 0  →  y = -x²
        f_y = -3y² + 3x = 0 →  x = y²

        代入得 x = (-x²)² = x⁴ → x(x³ - 1) = 0
        → x = 0 或 x = 1

        · x = 0 → y = 0 → 驻点 (0, 0)
        · x = 1 → y = -1 → 驻点 (1, -1)

        第二步：求二阶偏导
        f_xx = 6x,  f_yy = -6y,  f_xy = 3

        第三步：对每个驻点判别
        在 (0, 0)：A = 0, B = 3, C = 0
        AC - B² = 0 - 9 = -9 < 0 → 鞍点

        在 (1, -1)：A = 6, B = 3, C = 6
        AC - B² = 36 - 9 = 27 > 0, 且 A > 0 → 极小值点
        f(1, -1) = 1 + 1 - 3 = -1 为极小值
常见错误：忘记检查 AC - B² = 0 的情况
当 AC - B² = 0 时，Hessian 判别法失效。例如 f(x, y) = x⁴ + y⁴ 在 (0, 0) 处显然有极小值，但计算可得 A = C = B = 0，AC - B² = 0——你必须用多项式展开或直接观察来判断。

反过来，f(x, y) = x⁴ - y⁴ 同样有 AC - B² = 0，但在 (0, 0) 处实际上是鞍点。所以 AC - B² = 0 时一定要小心，不能盲目下结论。

条件极值的引入

刚才讨论的"无条件极值"——在整个定义域内找极值——在实际中反而比较少见。更多的情况是：在某些约束条件下找极值。比如：

  • 用固定长度的篱笆围出最大的矩形面积（周长固定，面积最大化）
  • 在预算固定的情况下最大化消费者效用（budget constraint）
  • 设计一个圆柱形罐头，给定容积使表面积最小（减少材料成本）

这些问题的共同点是：目标函数 f(x, y) 要在约束条件 g(x, y) = 0 下求极值。你不能在整个 xy 平面内自由选择，只能沿着一条曲线（约束条件）移动。

从几何上看，约束条件 g(x, y) = 0 通常是一条曲线。你要在这条曲线上找到函数 f(x, y) 的最高（或最低）点。简单粗暴的做法是：从约束方程中解出一个变量（比如 y = h(x)），代入目标函数化为一元函数再求导。但这种方法在约束条件复杂或变量多的时候难以适用——于是我们需要更强大的工具。

拉格朗日乘数法的原理与几何解释

拉格朗日乘数法（Lagrange multiplier method）是解决条件极值问题的标准方法。它的核心思想极其巧妙：在条件极值点处，目标函数的梯度与约束函数的梯度平行。

为什么？想象你在一条山路上行走（约束条件就是山路），你想找到山路的最高点。在山路的最高点处，你的运动方向与等高线相切——此时 ∇f（指向 f 增长最快的方向）与山路的切线垂直。而约束曲线的梯度 ∇g 也垂直于约束曲线本身。所以 ∇f 和 ∇g 方向相同或相反，即存在一个 λ 使得：

∇f = λ ∇g

这个 λ 被称为拉格朗日乘数（Lagrange multiplier）。

实际操作中，我们构造一个拉格朗日函数（Lagrangian）：

L(x, y, λ) = f(x, y) - λ · g(x, y)

然后求解方程组：

∂L/∂x = 0  →  f_x - λ g_x = 0
        ∂L/∂y = 0  →  f_y - λ g_y = 0
        ∂L/∂λ = 0 →  g(x, y) = 0

三个方程解三个未知数 (x, y, λ)。就是这么简洁。

来看一个经典例子：用长度为 L 的篱笆围一个矩形区域，求最大面积。

目标函数：f(x, y) = xy  （面积）
        约束条件：2x + 2y = L，即 g(x, y) = 2x + 2y - L = 0

        构造 Lagrangian：
        L(x, y, λ) = xy - λ(2x + 2y - L)

        求偏导：
        ∂L/∂x = y - 2λ = 0  →  y = 2λ
        ∂L/∂y = x - 2λ = 0  →  x = 2λ
        ∂L/∂λ = -(2x + 2y - L) = 0 → 2x + 2y = L

        由 x = y = 2λ 代入约束：2(2λ) + 2(2λ) = 8λ = L → λ = L/8
        所以 x = y = L/4

        结论：正方形时面积最大，最大面积 = (L/4)² = L²/16

有趣的是，拉格朗日乘数 λ 本身也有实际意义——它衡量约束条件的"松紧程度"。λ 的绝对值表示约束条件放松一单位时，目标函数值会增加多少。在上面的例子中，λ = L/8，意味着篱笆长度 L 每增加 1 单位，最大面积增加 L/4（因为 dA/dL = L/8 × 2 + L/8 × 2 = L/4……严格来说需要进一步推导，但直觉上是成立的）。

拉格朗日乘数法的几何直觉
画两张图叠在一起：一张是目标函数 f(x, y) 的等高线图（contour lines），另一张是约束曲线 g(x, y) = 0。在条件极值点处，等高线恰好与约束曲线相切。

为什么相切？因为如果它们相交（而不相切），沿着约束曲线你可以走到更高（或更低）的等高线上。只有在相切的位置，你才无法"再进一步"——进退两难，恰好就是极值。而两条曲线相切的数学条件就是它们的法向量平行，即 ∇f ∥ ∇g。

实际最值问题

我们来看看拉格朗日乘数法如何应用于更实际的优化问题。

最短距离问题

求原点到曲线 x² + xy + y² = 1 的最短距离。

距离的平方为 f(x, y) = x² + y²（最小化距离等价于最小化距离的平方，计算更方便）。约束条件为 g(x, y) = x² + xy + y² - 1 = 0。

L(x, y, λ) = x² + y² - λ(x² + xy + y² - 1)

        ∂L/∂x = 2x - λ(2x + y) = 0    ...(1)
        ∂L/∂y = 2y - λ(x + 2y) = 0    ...(2)
        ∂L/∂λ = -(x² + xy + y² - 1) = 0  ...(3)

        由(1)(2)消去 λ（或用矩阵形式求解），可得 x² = y²。
        代入(3)得 3x² = 1 → x = ±1/√3, y = ±1/√3
        d² = x² + y² = 2/3 → d = √(2/3) ≈ 0.816

原点到该椭圆的最短距离约为 0.816 个单位。

最大体积问题

一个长方体盒子无盖，用 12 m² 的材料，求最大体积。

设长宽高分别为 x、y、z，体积 V = xyz，材料面积约束（底面 + 四个侧面）：xy + 2xz + 2yz = 12。

构造 L = xyz - λ(xy + 2xz + 2yz - 12)，解方程组可得 x = y = 2，z = 1，V_max = 4 m³。（详细计算留给读者作为练习。）

多个约束的拉格朗日乘数法
当有多个约束条件 g₁ = 0, g₂ = 0, ..., g_k = 0 时，拉格朗日函数变为：
L = f - λ₁g₁ - λ₂g₂ - ... - λ_kg_k
每个约束对应一个独立的 λ。解方程组时，未知数个数为 n + k（n 个变量 + k 个乘数）。这在实际工程问题中非常常见——比如一个机械系统同时受到几何约束和力约束。

多元极值在经济学中的应用

微积分在经济学中的应用极为广泛，而拉格朗日乘数法更是经济学的核心数学工具之一。我们来看两个经典例子。

效用最大化

消费者的效用函数为 U(x, y) = xa yb（Cobb-Douglas 效用函数），其中 x 和 y 是两种商品的消费量。预算约束为 p_x · x + p_y · y = I（收入 I 全部用于消费）。

用拉格朗日乘数法：

L = xa yb - λ(p_x · x + p_y · y - I)
        ∂L/∂x = a xa-1 yb - λ p_x = 0
        ∂L/∂y = b xa yb-1 - λ p_y = 0
        ∂L/∂λ = -(p_x · x + p_y · y - I) = 0

        解得 x* = (a/(a+b)) · (I/p_x),  y* = (b/(a+b)) · (I/p_y)

这个结果的意义极其深刻：消费者将收入的比例 a/(a+b) 花在商品 x 上，比例 b/(a+b) 花在商品 y 上。如果 a > b，说明消费者更偏好商品 x，会花更大比例的收入在上面。这个简单的模型是微观经济学中消费理论的基石。

成本最小化

生产函数为 Q = Kα Lβ（Cobb-Douglas 生产函数），K 是资本投入，L 是劳动力投入。总成本 C = rK + wL（r 是资本价格，w 是工资）。给定产量 Q₀，如何最小化成本？

这也是条件极值问题——约束条件为 Kα Lβ = Q₀，目标函数为 C = rK + wL。求解结果将告诉你：企业应该在资本和劳动力之间如何分配资源，才能以最低成本达到目标产量。

经济学问题 目标函数 约束条件 拉格朗日乘数 λ 的意义
效用最大化 效用 U(x, y) 预算 p·x = I λ = 收入的"边际效用"（每多一块钱的效用增加）
成本最小化 成本 C(K, L) 产量 Q = Q₀ λ = 产出量的"边际成本"
利润最大化 利润 π(Q) 市场需求曲线 λ = 市场竞争的"影子价格"

经济学中 λ 有一个特殊的名字——影子价格（shadow price）。它告诉你约束条件放松一单位时，目标函数的变化量。这是一个极其实用的概念：企业在做决策时，不仅要看"现在是否最优"，还要看"如果约束条件变化，我的收益能提升多少"——后者就是通过 λ 来衡量的。

说到经济应用，我的一位朋友在读经济学博士时跟我讲过一个故事。他做了一项关于最低工资对就业影响的研究，需要在一个多约束条件下优化社会福利函数。他手动推导了整整两周的拉格朗日方程组，结果因为一个符号错误，所有的 λ 都算反了——约束放松时福利竟然"下降"，这显然违背经济直觉。他排查了三天，才发现在构造 Lagrangian 时把 f - λg 写成了 f + λg，导致所有 λ 的符号都反转了。

这个故事告诉我们两件事：第一，拉格朗日乘数的符号很重要——它直接决定了你对经济现象的解读是否正确；第二，当你的计算结果违背直觉时，不要强行解释，而是要回头检查基础数学——很多时候问题出在最简单的地方。

多元函数的极值是微分学的"高潮"——它将导数从"看变化快慢"推向"找最优解"，为工程优化、经济决策、机器学习模型训练提供了数学基础。下一章我们将转向多元微积分的另一半：重积分（multiple integrals）。如果说微分是"撕碎"（把整体拆成局部变化），那么积分就是"拼合"（把局部累加为整体）。我们会看到，二重积分和三重积分如何在平面区域和空间区域上完成这种"拼合"，以及如何通过坐标变换来简化复杂的积分计算。
