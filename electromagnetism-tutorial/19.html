第19章 趋肤效应与涡流

高频电流只走导体的表面——这不是设计选择，是麦克斯韦方程组决定的物理事实。当开关电源频率从几十千赫兹提升到兆赫兹级别时，原本"够用"的粗铜线突然变得像细铁丝一样电阻剧增；当手机处理器跑到GHz频段时，芯片内部互连线实际上只有表面一个原子层那么厚的区域在导电。要理解这一切，我们必须深入趋肤效应和涡流的物理本质。

趋肤效应的物理机制

趋肤效应源于电磁感应的自洽作用。当交流电流通过导体时，电流产生的交变磁场会在导体内部感应出电场（法拉第定律），而这个感应电场的方向遵循楞次定律——它总是阻碍引起它的磁场变化。

考虑一根圆柱导体通入交流电流。导体中心的电流产生的磁通量穿过导体内部区域，时变的磁通量在导体内诱导出涡旋电场。在导体中心附近，感应电场的方向与原始电流方向相反，因此抵消了中心区域的电流；而在表面附近，感应电场的方向与原始电流方向相同，增强了表面电流。净效果是：电流密度从表面向中心呈指数衰减。

数学上，对于半无限大导体（x > 0），由麦克斯韦方程组可推导出电流密度的分布：

J(x) = J0 · e−x/δ · cos(ωt − x/δ)

其中 x 是到表面的距离，δ 就是趋肤深度。电流密度在深度 δ 处衰减到表面值的 1/e（约 37%）。注意电流不仅有幅度衰减，还有相位滞后——这表明深层电流"跟不上"表面电流的变化。

关键洞察：趋肤效应不是表面效应，而是内部抵消效应。不是电流被"推"到表面，而是内部电流被感应电场抵消了。直流时整个截面均匀导电，频率越高，内部抵消越彻底。

趋肤深度

趋肤深度的公式可以从麦克斯韦方程组严格导出：

δ = √(2 / (ωμσ)) = 1 / √(πfμσ)

其中 ω = 2πf 是角频率，μ = μ₀μᵣ 是磁导率，σ 是电导率。这个公式揭示了一个重要关系：趋肤深度与频率的平方根成反比。频率越高，电流越集中于表面。

以铜（σ = 5.96×10⁷ S/m，μᵣ ≈ 1）为例：

60 Hz≈ 8.5 mm美标电力系统
400 Hz≈ 3.3 mm航空电源、雷达
1 kHz≈ 2.1 mm音频变压器

100 kHz≈ 0.21 mmDC-DC 变换器

下面是一个用于计算趋肤深度的 Python 函数：

import numpy as np

def skin_depth(f, sigma, mu_r=1, mu_0=4*np.pi*1e-7):
    """
    计算趋肤深度

    参数:
        f: 频率 (Hz)
        sigma: 电导率 (S/m)
        mu_r: 相对磁导率 (默认 1)
        mu_0: 真空磁导率 (4π × 10⁻⁷ H/m)

    返回:
        delta: 趋肤深度 (m)
    """
    mu = mu_r * mu_0
    delta = 1.0 / np.sqrt(np.pi * f * mu * sigma)
    return delta

# 示例：铜在不同频率下的趋肤深度
sigma_Cu = 5.96e7
frequencies = [50, 60, 1e3, 100e3, 1e6, 1e9]
for f in frequencies:
    d = skin_depth(f, sigma_Cu) * 1e3  # 单位：mm
    print(f"{f:10.0f} Hz:  δ = {d:.3f} mm" if f < 1e6
          else f"{f:10.0f} Hz:  δ = {d*1000:.2f} μm")

高频导体的工程影响

趋肤效应最直接的后果是：交流电阻远大于直流电阻。直流时整个截面参与导电，交流时只有厚度为 δ 的表层有效。一根半径 5mm 的铜棒，直流电阻 RDC 很小；但在 1MHz 时，有效导电截面从 πr² 缩减到约 2πr·δ，交流电阻变成：

RAC / RDC ≈ r / (2δ) （当 δ ≪ r）

对于 5mm 半径的铜棒在 1MHz（δ≈65μm）：RAC / RDC ≈ 38 倍！这意味着高频电流的损耗远高于直流预估。

工程应对措施：

  • 空心导体：既然电流只在表层，高频应用（如射频谐振腔）直接用镀银铜管或空心管，节省材料且散热更好。
  • 绞线（Litz wire）：将多根细漆包线绞合，每根线的直径小于趋肤深度，迫使电流均匀分布于各股。
  • 电力系统的分裂导线：高压输电线采用 2~8 根分裂导线，每根间距约 30~50cm，等效地增大了导线"表面"。
  • 镀银或镀金：高频 PCB 和连接器常见镀银处理，利用银略高于铜的电导率（仅提升 6%），但更重要的是防氧化——氧化层会大大增加表面电阻。

涡流

涡流（eddy current）是整体导体在变化磁场中感应出的闭合电流回路。与趋肤效应不同，涡流存在于所有导体内，不只是载流导体本身——即使一块孤立的金属片放在交变磁场中，内部也会产生涡流。

涡流的基本物理：法拉第定律 → 感应电场 → 导体中形成电流回路。涡流遵循焦耳定律产生热量：

Peddy ∝ f² · B² / ρ

其中 f 是变化频率，B 是磁通密度峰值，ρ 是电阻率。注意涡流损耗与频率的平方成正比——频率翻倍，损耗变成四倍！这是高频磁性元件设计的核心挑战。

涡流损耗的三次方关系：更精确的公式是 Peddy ∝ f²·B²·d²/ρ，其中 d 是导体厚度。厚度翻倍，损耗变成四倍——这意味着把铁芯切成薄片可以显著降低涡流损耗。

减少涡流的经典方法：

  • 变压器铁芯叠片：将铁芯切成 0.2~0.5mm 厚的硅钢片，片间涂绝缘漆。涡流被限制在每片内，总损耗大大降低。硅钢中添加 3% 的硅还能提高电阻率 ρ，进一步减少涡流。
  • 铁氧体磁芯（ferrite core）：铁氧体材料的电阻率高达 1~10⁶ Ω·m（硅钢只有 4×10⁻⁷ Ω·m），因此涡流损耗极小，适合 MHz 频段。代价是饱和磁通密度低（约 0.3~0.5 T），不如硅钢（约 1.5~2.0 T）。
  • 磁粉芯：将铁磁粉末与绝缘粘结剂混合压制而成，粉末尺寸小到足以抑制涡流。

趋肤与邻近效应的应用

趋肤效应和涡流并非只有害处——许多重要技术正是利用这些效应工作的。

感应加热

将工件置于高频交变线圈中，工件内感应出的涡流产生焦耳热。频率选择取决于目标加热深度：

  • 高频（100~500 kHz）：趋肤深度小，用于表面淬火，使齿轮、轴承等零件表面硬化而内部保持韧性。
  • 中频（1~10 kHz）：穿透较深，用于透热加热（锻造前的整体预热）。
  • 工频（50/60 Hz）：穿透最深，用于熔炼炉（如中频感应炉炼钢）。

金属探测器

发射线圈产生交变磁场，金属物体中的涡流产生二次磁场被接收线圈检测。脉冲式金属探测器（PI）利用涡流的衰减特性区分不同金属——铁磁材料涡流衰减慢，非铁磁材料衰减快。

邻近效应

当两根载有交流电流的导线靠近时，彼此的磁场会改变对方的电流分布，这就是邻近效应。在变压器绕组中，邻近效应甚至比趋肤效应更显著——相邻匝导线中的相反方向电流会迫使电流集中在匝间的狭窄区域。

工程实践：Litz 线由多根相互绝缘的细漆包线绞合而成，每根线的直径通常为 0.05~0.1mm，远小于工作频率下的趋肤深度。Litz 线同时对抗趋肤效应和邻近效应，广泛用于高频变压器和电感绕制中。

高频PCB中的趋肤

当频率进入 GHz 射频和毫米波领域，PCB 导体的表面特性变得至关重要——因为电流只存在于表层几个微米甚至亚微米的范围内。

常规 PCB 使用电解铜箔（ED copper），其与 FR4 基材接触的一面经过粗化处理以增强附着力，粗糙度（Rz）通常在 5~15 μm。在 1GHz 时趋肤深度约 2 μm，电流完全在粗糙表面内流动——这意味着电流路径比理想平面更长，电阻显著增加。

高频 PCB 的应对策略：

  • 反转铜箔（RTF）：粗糙度降至 3~5 μm。
  • 超光滑铜箔（VLP/HVLP）：粗糙度低至 1~2 μm，专为 10 GHz 以上设计。
  • 表面镀层选择：ENIG（化学镀镍浸金）的表面镀金层虽然导电性略差于铜，但防氧化特性使其优于裸铜。在毫米波频段，更推荐使用 OSP（有机保护膜）或直接镀银。
  • 低损耗基材：Rogers 4000 系列、Megtron 6 等低损耗材料不仅介电损耗低，其铜箔粗糙度也更可控。
高频设计警告：当你设计一个 5 GHz 的微带线时，电流在铜箔表面最外层的 2 μm 内流动。如果铜箔表面被氧化或粗糙度过大，导体损耗可能比理论值高出 50%！在毫米波设计中，铜箔粗糙度是仅次于介质损耗的第二大信号损耗来源。
真实案例：某开关电源设计团队在 500 kHz 的 LLC 谐振变换器中使用实心铜线绕制变压器，测得效率仅 91%。检查发现 2 mm 直径的铜线在 500 kHz 下趋肤深度仅 0.09 mm——有效导电面积只有线径的不到 10%！换用由 0.05 mm 细漆包线绞合而成的 Litz 线后，变压器铜损降低 70%，整机效率跃升至 96%。这个真实案例生动说明了：高频磁性元件的绕组选择不是材料问题，是物理问题。

从趋肤效应到涡流，我们看到同一个物理本质的两种表现。这些效应深刻影响从电力变压器到纳米级芯片互连的所有电磁系统。下一章我们将转向系统层面的电磁兼容性设计，那里趋肤效应和涡流的知识将成为诊断 EMI 问题的关键工具。
