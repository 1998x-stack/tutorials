从描述到推断：打开多元假设检验的大门

大家好，欢迎继续学习。前两章我们打好了数学基础，学习了多元正态分布的核心性质。这些工具是用来做什么的？归根结底，是要回答这样的问题："A 组和 B 组在这 10 个指标上有没有显著差异？""这个新药的疗效是否优于安慰剂？""不同年级的学生在多个测试维度上表现是否相同？"这些问题在现实中被问了无数次，而答案的获取方法，正是多元假设检验 (multivariate hypothesis testing)——本章的核心内容。

有人会说："这些检验我逐个变量做 t 检验不就行了？"好问题。想象一个场景：你有一个包含 20 个变量的临床试验数据。如果每个变量单独做 t 检验（α=0.05），20 次检验中至少有一次假阳性的概率是 1 - 0.95²⁰ ≈ 64%。什么概念？你的"显著性发现"有近三分之二的概率是假的。多元假设检验的核心价值，就是用一个全局检验同时考虑所有变量，将整体犯第一类错误的概率控制在预定水平上。

单样本均值向量检验

假设我们有一个 p 维总体 X ~ Np(μ, Σ)，以及 n 个独立样本。我们想检验：

H₀: μ = μ₀  （均值向量等于指定值）
H₁: μ ≠ μ₀  （至少有一个分量不同）

在一元统计中，我们用的是 t 统计量：t = (x̄ - μ₀) / (s/√n)。在多元统计中，Hotelling T² 分布是 t 检验的自然推广：

T² = n (x̄ - μ₀)ᵀ S⁻¹ (x̄ - μ₀)

其中 x̄ 是样本均值向量，S 是样本协方差矩阵

仔细看这个公式：它把一元 t 统计量的平方 (x̄ - μ₀)²/(s²/n) 推广到了 p 维空间。T² 本质上就是样本均值到目标值的马氏距离乘以 n。如果 H₀ 成立，T² 服从自由度为 p 和 n-1 的 Hotelling T² 分布，它与 F 分布的关系为：

T² ~ [p(n-1)/(n-p)] · F(p, n-p)

关键点：

  • T² 越大，观测到的均值向量与 μ₀ 的差异越大
  • T² 考虑了变量的相关结构（通过 S⁻¹）——如果两个变量高度相关，T² 会自动"调整"，不会重复计算
  • 当 p=1 时，T² 退化为一元 t 统计量的平方——这是自洽的
直观理解：T² 检验相当于在 p 维空间中问："样本均值点 x̄ 到目标点 μ₀ 的距离，在考虑数据的变异和相关性之后，是否大到足以被认为不是抽样误差？"——它把所有变量的信息整合成了一个统计量。

两样本均值向量检验

更常见的情况是比较两个总体的均值向量是否相等。假设两个独立样本：

样本1：X₁₁, ..., X₁n₁ ~ N(μ₁, Σ)
样本2：X₂₁, ..., X₂n₂ ~ N(μ₂, Σ)

假设：两个总体的协方差矩阵相同（齐性假设）

两样本 Hotelling T² 统计量为：

T² = (n₁n₂/(n₁+n₂)) (x̄₁ - x̄₂)ᵀ S_pl⁻¹ (x̄₁ - x̄₂)

其中 pooled 协方差矩阵：
S_pl = [(n₁-1)S₁ + (n₂-1)S₂] / (n₁+n₂-2)

注意 S_pl 是两组协方差矩阵的加权平均，权数是各自的自由度。这是"假设两组协方差相同"下的最优估计。

与 F 分布的关系：T² ~ [p(n₁+n₂-2)/(n₁+n₂-p-1)] · F(p, n₁+n₂-p-1)

如果两组协方差不相等，情况就复杂了——这是多元版的Behrens-Fisher 问题。一种近似方法是使用 James 检验 或修正的自由度近似。

场景检验统计量分布（原假设下）
单样本，已知 Σχ² = n(x̄-μ₀)ᵀΣ⁻¹(x̄-μ₀)χ²(p)
单样本，未知 ΣT² = n(x̄-μ₀)ᵀS⁻¹(x̄-μ₀)[p(n-1)/(n-p)]·F(p,n-p)
两样本，Σ₁=Σ₂T² = (n₁n₂/(n₁+n₂))(x̄₁-x̄₂)ᵀS_pl⁻¹(x̄₁-x̄₂)[p(n₁+n₂-2)/(n₁+n₂-p-1)]·F(p, n₁+n₂-p-1)
两样本，Σ₁≠Σ₂近似 T²近似 F，修正自由度

多样本均值比较：MANOVA

当你有 k 个组（k ≥ 2）要同时比较时，就需要多元方差分析 (MANOVA)。它是一元 ANOVA 的多元推广，但比 ANOVA 多了一个关键维度：不仅要关心组间均值是否有差异，还要考虑多个因变量之间的相关性。

MANOVA 的核心思想是将总变异分解为组间变异和组内变异，但在多元框架下，这些"变异"不是标量而是矩阵：

总平方和与叉积矩阵：T = W + B

其中：
W = Σᵢ Σⱼ (xᵢⱼ - x̄ᵢ)(xᵢⱼ - x̄ᵢ)ᵀ  （组内）
B = Σᵢ nᵢ (x̄ᵢ - x̄)(x̄ᵢ - x̄)ᵀ         （组间）

MANOVA 的检验统计量有多个变种，它们从不同角度比较 W 和 B：

统计量公式特点
Wilks ΛΛ = |W| / |B+W|最常用；相当于似然比检验统计量
Pillai's traceV = tr(B(B+W)⁻¹)对偏离假设最稳健；推荐
Roy's largest rootθ = λ_max(BW⁻¹)统计功效最高但不够稳健
Lawley-Hotelling traceU = tr(BW⁻¹)介于 Wilks 和 Roy 之间
选择建议：对于实际数据分析，我通常推荐 Pillai's trace。它对正态偏离和协方差矩阵异质性的容忍度最高（即最稳健），特别当样本量不是很大时。R 的 manova() 和 Python 的 statsmodels.multivariate.manova 都会默认输出所有这些统计量，你无需手动选择——观察它们是否给出一致的结论即可。

协方差矩阵检验：Box's M 检验

前面两样本 Hotelling T² 和 MANOVA 都假设各组协方差矩阵相等。这个假设是否成立？Box's M 检验就是用来回答这个问题的。

零假设：H₀: Σ₁ = Σ₂ = ... = Σk。检验基于各组的协方差矩阵与 pooled 协方差矩阵的"差异"：

M = [(n-k)ln|S_pl| - Σᵢ (nᵢ-1)ln|Sᵢ|]

其中 S_pl 是 pooled 协方差矩阵，Sᵢ 是第 i 组的样本协方差矩阵

Box's M 统计量近似服从 F 分布或 χ² 分布（取决于修正版本）。

重要提醒：Box's M 检验对正态性假设非常敏感——如果数据偏离正态，Box's M 可能在小样本下拒绝 H₀（即使协方差真的相等），在大样本下不拒绝 H₀（即使不相等）。我的建议是：在大样本下（各组 nᵢ > 20）不要过于依赖 Box's M 的 p 值，而是结合可视化（如箱线图、散点图矩阵）来判断各组离散程度是否相似。如果发现明显的异质性，考虑使用对异方差稳健的检验方法或用 James 检验替代 Hotelling T²。

Profile 分析

Profile 分析 (profile analysis) 是均值比较的一个特殊而实用的形式。想象一个场景：一家公司测量了员工在入职时、培训后、工作半年后、工作一年后的满意度评分（4 个时间点），并分为"培训组"和"对照组"。我们关心的是：两组的"满意度曲线"是否有差异？

Profile 分析回答三个层层递进的问题：

  1. 平行性 (parallelism)：两组的曲线是否平行？即组别与时间点的交互作用是否显著？检验统计量仍然是 Wilks Λ 或 Pillai's trace。
  2. 重合性 (coincidence)：如果曲线平行，它们是否重合？即组别的主效应是否显著？
  3. 水平性 (level)：如果曲线重合，所有时间点的均值是否相同？即时间点的主效应是否显著？

三个检验是层级的：只有平行性不拒绝，才继续检验重合性；只有重合性不拒绝，才继续检验水平性。这种层级结构避免了多重比较问题。

# Profile 分析的 R 代码示意
# 假设 data 包含：group (A/B), time1-time4 四个满意度评分

# 检验平行性（交互作用）
result_parallel <- manova(cbind(time1, time2, time3, time4) ~ group,
                          data = data)
summary(result_parallel, test = "Pillai")

# 如果平行性成立，检验重合性
# 将四个时间点的均值作为新的因变量
data$mean_score <- rowMeans(data[, c("time1","time2","time3","time4")])
t.test(mean_score ~ group, data = data)

Profile 分析的应用场景包括：心理学中的重复测量设计、教育评估中的多时间点测试、市场调研中的产品属性评分等。

实战案例：不同品种植物的多指标测量

让我们用一个经典数据集来看这些方法如何落地。这个案例来自 Fisher 1936 年的鸢尾花 (Iris) 数据——可能是统计学史上最著名的数据集。它包含了 3 个鸢尾花品种各 50 个样本，每个样本测量 4 个变量：花萼长度、花萼宽度、花瓣长度、花瓣宽度。

问题：三个品种在这 4 个测量指标上是否存在显著差异？

分析步骤：

  1. 描述性统计：计算每个品种的均值向量和协方差矩阵
  2. 正态性检验：检查每组数据是否近似多元正态（用 Mardia 检验）
  3. 协方差齐性检验：用 Box's M 检验三个组的协方差矩阵是否相等
  4. MANOVA：用 Wilks Λ 或 Pillai's trace 检验品种间的整体差异
  5. 事后分析：如果整体检验显著，找出哪些变量贡献了差异
import pandas as pd
import numpy as np
from sklearn.datasets import load_iris
from statsmodels.multivariate.manova import MANOVA

# 加载数据
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)
df['species'] = iris.target_names[iris.target]

# MANOVA
manova = MANOVA.from_formula(
    'sepal_length + sepal_width + petal_length + petal_width ~ species',
    data=df)
print(manova.mv_test())
# 输出：Wilks Λ = 0.023, F ≈ 199, p < 2×10⁻¹²
# 结论：三个品种之间有极其显著的差异

# Box's M 检验（可用 pingouin 库）
# import pingouin as pg
# box_m = pg.box_m(data=df.drop('species', axis=1),
#                  groups=df['species'])
# print(box_m)
结果解读：Wilks Λ = 0.023 是一个非常小的值，说明组内变异（W）远小于总变异（B+W）——即组间差异非常大。F 统计量约为 199，p 值小到几乎为零。我们可以很自信地说：这三个鸢尾花品种在四个测量指标上有显著差异。后续的判别分析（第17章）正是基于这种差异来建立分类规则的。

踩坑故事：多重比较的陷阱

这个故事发生在我刚接触数据分析时。一个生物系的同事拿着基因敲除小鼠的实验数据来找我：实验组和对照组各 10 只小鼠，每只测量了 15 个血液生化指标。他的分析方法是——逐一做 t 检验，然后圈出 p < 0.05 的指标作为"显著差异"。结果有 4 个指标"显著"。

我问他："你做多重比较校正了吗？"他一脸茫然。于是我帮他算了一笔账：15 次独立 t 检验，每次 α=0.05，至少出现一次假阳性的概率 = 1 - 0.95¹⁵ ≈ 54%。他的 4 个"显著"指标中，大概有一半是假的。后来我们用 Hotelling T² 做一个全局检验，p = 0.08——不显著。再用 Bonferroni 校正后，15 个指标没有一个能通过校正后的阈值。

教训：在多元假设检验中，永远不要用"逐变量 t 检验 + 未校正 p 值"的策略来发表结论。要么用全局检验（Hotelling T²、MANOVA），要么用适当的校正方法（Bonferroni、FDR）。否则你的论文中那些标注着"*"的显著结果，很可能是统计噪音而不是真实信号。

本章小结与衔接

这一章我们从单样本 Hotelling T² 检验开始（它是 t 检验的多元推广），到两样本检验（需要协方差齐性假设），再到 MANOVA 处理多样本均值比较问题（Wilks Λ、Pillai's trace 等四种统计量），然后讨论了 Box's M 检验用于协方差矩阵齐性检验，最后介绍了 Profile 分析这一实用的重复测量方法。贯穿始终的核心思想是：一次性检验所有变量，控制整体第一类错误率。

到此为止，我们完成了多元统计推断的理论核心：从数据生成（多元正态）到估计（MLE）到检验（Hotelling T²、MANOVA）。从下一章开始，我们将转换视角——在进入 PCA、因子分析、聚类等强大的方法之前，还有一步必不可少的工作：数据预处理和可视化。数据不会自动以漂亮的矩阵形式出现——它可能包含缺失值、异常值、不同量纲和不满足假设的分布。如何"清理"数据、用什么方法"探索"数据，直接决定了后续分析的成败。
