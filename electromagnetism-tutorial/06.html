大家好，我是你们的讲师。今天我想先问一个看似简单、实则深刻的问题：静电能量储存在哪里？直觉可能会告诉你：能量储存在电荷上，就像水储存在水库里一样。但法拉第和麦克斯韦告诉我们：不——能量储存在电场中。这个观念的转变是物理学史上的一次革命，它直接催生了电磁波的概念：如果能量在电场中，那么它就能以光速在空间中传播。这就是为什么手机信号不需要电线——信息以电磁波的形式携带着能量穿过空气。

电荷系的静电能

将一群点电荷从无限远（彼此之间无相互作用）聚拢到指定位置，外力克服静电力做的功转化为系统的静电势能（Electrostatic Energy）。对于两个点电荷 q₁ 和 q₂，相距 r：

U = (1/4πε₀) · q₁q₂ / r

对于 N 个点电荷的系统：

U = ½ Σi=1N qi V(ri)

其中 V(ri) 是除 qi 外其他所有电荷在 qi 位置产生的电势。为什么要乘以 ½？因为求和 Σ qiVi 将每一对电荷之间的相互作用算了两次——你可以自己检验两个点电荷的情况：Σ q₁V₁ + q₂V₂ = q₁ · (q₂/4πε₀r) + q₂ · (q₁/4πε₀r) = 2 × U12，所以需要 ½ 因子来修正。

对于连续电荷分布，求和变为积分：

U = ½ ∫ ρ V dV

这个公式在实际计算中很实用——已知电荷分布和电势分布就能算出总静电能。

电容器

电容器（Capacitor）是储存静电能量的器件，也是电子电路中最基本的无源元件之一。电容的定义是：

C = Q / V

其中 Q 是一个极板上的电荷量（另一个极板为 −Q），V 是两极板之间的电势差。电容的单位是法拉（F），1 F = 1 C/V。通常我们遇到的电容器值在皮法（pF, 10⁻¹² F）到毫法（mF, 10⁻³ F）之间。

平行板电容器

两块面积为 A、间距为 d 的平行导电极板，中间填充相对介电常数为 εr 的介质：

C = ε₀ εr A / d

增大面积 A、减小间距 d、使用高 εr 的介质，都能提高电容值。

圆柱电容器

同轴圆柱电容器（如电缆），内筒半径 a、外筒半径 b、长度 L：

C = 2πε₀ εr L / ln(b/a)

球形电容器

两个同心球壳，内球半径 a、外球半径 b：

C = 4πε₀ εr · ab / (b − a)

🔑 核心洞察：能量在场中，不在电荷上
如果能量储存在电荷上，那么当两个电容器的极板靠在一起但电场消失时，能量应该还在——但事实是能量消失了（做了功、发热了）。只有当能量被理解为储存在电场中时，一切才自洽。
我们来看一个思想实验：一个孤立带电导体球，其静电能是 U = Q²/(8πε₀R)。这就是将电荷从无穷远搬来所需的功。但我们可以不使用电荷这个量，而用电场来表达：U = (ε₀/2) ∫ E² dV。这个积分遍及全空间——它说能量分布在电场存在的整个空间中，而不是集中在电荷上。当我们用 E² 来表达能量时，能量自然地"弥散"在场中。这为后来电磁波携能传输奠定了理论基础。

电场能量密度

能量密度的概念是电磁学中最优美的观念之一。在真空中，静电能量密度为：

uE = ½ ε₀ E²

在介质中，考虑到极化的贡献：

uE = ½ D · E

总静电能为能量密度在全空间的积分：

U = ∫V ½ D · E dV

这个公式与 U = ½ ∫ ρV dV 完全等价（可以通过矢量恒等式和边界条件证明），但从物理观念上有着天壤之别：一个说能量在电荷上，一个说能量在电场中。后者才是正确的物理图景。

让我们验证一下：对于平行板电容器，E = V/d（均匀电场），体积为 A·d，所以：

U = uE · 体积 = (½ ε₀ E²)(A·d) = ½ ε₀ (V/d)² A d = ½ (ε₀ A/d) V² = ½ CV²

这正是电容器的储能公式 U = ½ CV²。两种方法殊途同归——但"能量在场中"的观念更接近物理本质。

电容器的串并联

这是电路中最基本但也最容易搞混的知识点之一。

  • 串联（Series）：1/Ceq = 1/C₁ + 1/C₂ + ... + 1/CN
  • 并联（Parallel）：Ceq = C₁ + C₂ + ... + CN

关键要记住：电容的串并联规律与电阻正好相反。电阻串联是相加，电容串联是倒数相加；电阻并联是倒数相加，电容并联是相加。为什么？因为电阻上的电压和电流关系是 V = IR（加性的是电阻），而电容上的电荷和电压关系是 Q = CV（加性的是电容）。串联时各电容电荷相等（Q 相同），总电压是各电压之和：V = V₁ + V₂ = Q/C₁ + Q/C₂ = Q(1/C₁ + 1/C₂)。

💡 常见错误：搞混串并联公式
我见过太多学生在考场上把电容的串联公式写成 Ceq = C₁ + C₂（那是并联！）。一个简单的记忆方法："串联倒着加，并联正着加"。或者反过来想——串联的结果一定比最小的那个还小（因为 1/Ceq > 1/Cmin），并联的结果一定比最大的还大。如果你算出来的串联 Ceq 大于任何一个单独电容，那就是算错了。

电容网络与储能应用

电容器在现代电子系统中无处不在。以下是几类常见的电容及其典型应用：

类型 典型容量范围 耐压 主要特点 典型应用
陶瓷电容（MLCC） 1 pF ~ 100 µF 6.3 V ~ 几千V 体积小、高频特性好 旁路、耦合、谐振
电解电容（铝/钽） 0.1 µF ~ 几 F 6.3 V ~ 数百V 容量大、有极性、漏电流大 电源滤波、储能
薄膜电容 1 nF ~ 几十 µF 100 V ~ 几千V 稳定性好、ESR 低、无极 高压、电力电子
超级电容 1 F ~ 数千 F 2.5 V ~ 3 V（单节） 能量密度高、功率密度高 短时储能、UPS、能量回收

超级电容器（Supercapacitor / EDLC）是近年来发展迅速的储能技术。它利用电极表面的双电层效应和/或赝电容效应来存储电荷，容量可达数千法拉。与电池相比，超级电容的功率密度高（可以快速充放电）、循环寿命长（数十万次），但能量密度低（只有锂电池的 1/5~1/10）。

⚠ 真实故事：超级电容公交车的启示
2010年前后，中国多个城市投入了超级电容公交车的示范运营。这种公交车每到一站快速充电 30 秒，然后行驶到下一站。超级电容的高功率密度特性使它非常适合这种"频繁启停、短途行驶"的场景——充电快、循环寿命长、充放电效率高。
但运营中发现了一个问题：超级电容的能量密度太低，导致续航里程有限（通常不到 10 公里），而且电容的串联均压问题让系统可靠性打了折扣。最终纯电动公交车（锂电池+慢充）和快充锂电池成为了主流方案。
这个故事的启示是：电容和电池各有优劣，关键是把正确的技术用在正确的场景。电容适合短时间大功率的脉冲需求；电池适合长时间持续供电。在电动汽车的再生制动能量回收中，两者往往协同工作——电容吸收短时高功率的制动能量，电池提供平稳的续航能量。 
# RC 电路充放电时间常数计算
import numpy as np
import matplotlib.pyplot as plt

R = 10e3    # 10 kΩ
C = 100e-6  # 100 µF
V0 = 5.0    # 5V
tau = R * C  # 时间常数 τ = 1 秒

t = np.linspace(0, 5*tau, 500)

# 充电：V(t) = V0 * (1 - exp(-t/RC))
V_charge = V0 * (1 - np.exp(-t/tau))

# 放电：V(t) = V0 * exp(-t/RC)
V_discharge = V0 * np.exp(-t/tau)

plt.figure(figsize=(8, 5))
plt.plot(t, V_charge, 'b-', label='充电')
plt.plot(t, V_discharge, 'r--', label='放电')
plt.axvline(x=tau, color='gray', linestyle=':', label=f'τ = {tau*1000:.0f} ms')
plt.axhline(y=V0*(1-1/np.e), color='gray', linestyle=':')
plt.xlabel('时间 (s)')
plt.ylabel('电压 (V)')
plt.title('RC 电路充放电曲线')
plt.legend()
plt.grid(True)
plt.show()
# 在 t = τ 时，充电电压达到 0.632·V0

进一步探索：电介质上的力

当电介质板部分插入平行板电容器时，它会被"吸入"电容器中——这是因为电场对介质中的极化电荷施加了力，产生一个将介质拉入电容器的净力。计算这个力可以用虚功法（Virtual Work Method）：假设介质板移动一小段距离 dx，计算电容能量变化 dU，则力 F = −dU/dx。

另一种方法是利用麦克斯韦应力张量（Maxwell Stress Tensor）——它是电磁场对物质作用力的统一描述框架。虽然超出了本章的范围，但值得知道的是：你在工程中处理电磁力时（比如 MEMS 中的静电驱动、高压开关中的电动力），应力张量是最强大的工具。它把体力和面力统一在同一种数学语言中。

从静电能到电流

静电能量的研究让我们认识到：电场本身携带着能量，电容是这个能量的存储容器。但到目前为止，我们讨论的都是静止的电荷——它们不会运动。从下一章开始，我们要让电荷动起来。

当你把电容器的两个极板用导线连接起来，储存的能量就会释放——电荷通过导线移动，形成电流。但是等一等：在导线中，电荷移动的速度非常慢（漂移速度只有 mm/s 量级），为什么灯一打开就亮了呢？为什么电场能长距离传播？这引出了下一章的主题——恒定电流与电路：欧姆定律的微观本质是什么？基尔霍夫定律如何从电磁场理论中导出？以及那个让你第一次接触到"指数"数学之美的 RC 电路充放电。
