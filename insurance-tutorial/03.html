第3章 保险基本原则（上）

大家好，我是你们的讲师。这一章和下一章，我们进入保险学理论中最核心的部分——保险基本原则。一共五个原则：最大诚信、保险利益、近因、损失补偿、代位求偿与分摊。这些原则是保险合同的"宪法"，几乎所有保险纠纷的判决依据都可以追溯到这些原则上来。

在开始之前，我讲一个我亲身经历的案例。几年前一位朋友买了份重疾险，投保时体检报告显示有甲状腺结节，但代理人问他"有没有什么健康问题"时，他觉得"结节算不上什么病"就没说。两年后他被确诊为甲状腺癌，向保险公司申请理赔，保险公司调取了他投保前两年的体检记录，以"未如实告知"为由拒赔。朋友非常愤怒："我交了两年保费，现在真得病了你们不赔？"从情理上确实令人同情，但从保险法的角度，保险公司的主张有充分依据。这个案例涉及的就是我们今天要讲的两个原则——最大诚信原则和保险利益原则。

最大诚信原则

最大诚信原则的含义

最大诚信原则（Utmost Good Faith, Uberrimae Fidei）是保险合同的基石。它要求保险合同的双方当事人在订立和履行合同过程中，都必须以最大的诚意、全面地、真实地披露和告知与合同有关的重要事实，不得有任何隐瞒或欺骗。

注意这里的"最大"二字。普通的买卖合同也讲诚信，但保险合同的诚信要求更高——因为保险合同是典型的信息不对称合同。投保人最了解自己的风险状况（你的身体好不好、你的房子电路老不老），而保险公司只能通过你的告知来评估风险。如果投保人隐瞒或歪曲信息，保险公司的风险定价就会完全失效，最终所有诚实投保人的保费都会被推高。

告知义务

告知义务是最大诚信原则在投保环节的核心体现。投保人在订立保险合同前，必须向保险人如实告知与保险标的有关的重要事实。

从国际比较看，告知义务有两种立法模式：

模式 要求 代表国家/地区 特点
无限告知 投保人主动、全面地告知所有重要事实，无论保险人是否询问 英国、澳大利亚（传统） 对投保人要求较高，常用于海上保险
询问回答 投保人只需如实回答保险人提出的问题，未询问的事项视为非重要事项 中国、日本、德国 对投保人更友好，降低告知负担

我国《保险法》第十六条采用询问回答主义："订立保险合同，保险人就保险标的或者被保险人的有关情况提出询问的，投保人应当如实告知。"也就是说，保险公司问了你就得如实回答，保险公司没问的——原则上你不需要主动披露。但实务中保险公司的健康告知问卷非常详尽，基本上把你需要告知的事项都涵盖了。

那什么是"重要事实"？标准是：足以影响保险人决定是否承保或者提高保险费率的事实。比如重疾险中，既往的癌症病史、严重高血压、糖尿病等显然是重要事实；而偶尔的感冒发烧、轻度近视通常不认为是重要事实。

保证条款

与告知义务相关的另一个重要概念是保证（Warranty）。保证是投保人或被保险人对某一事项的作为或不作为、或者某一事实状态的存在与否作出的承诺。保证分为两类：

  • 明示保证（Express Warranty）：在保单中明确写出的保证条款。例如车险保单中明确约定："被保险车辆不得用于竞赛、测试。"
  • 默示保证（Implied Warranty）：虽然没有明写，但法律或惯例认为当然存在的保证。例如海上保险中，被保险船舶必须适航（Seaworthiness）、不得绕航。

违反保证的后果非常严重——哪怕违反保证与损失之间没有因果关系，保险公司也可以解除合同、拒绝赔偿。比如船舶保险人约定"被保险船舶不得驶入北大西洋冬季水域"，结果船主违反了这条保证进入了该水域，虽然事故原因是轮机故障而非恶劣天气，保险公司仍然可以拒赔。

常见误解：很多人以为"只要我最后确实得病了，保险公司就应该赔"。但保险法的逻辑是：没有如实告知本身就是一种违约行为，违约的后果与最后损失是否发生没有直接关系。用个比喻——你申请贷款时说谎了收入，后来确实还不上钱了，银行会因为你"确实还不上"就不追究你之前的欺诈吗？不会。保险也是同理。当然，实务中法院也会根据未告知事项与保险事故之间是否存在"因果关系"来综合判断，但最安全的做法永远是：问什么，答什么，不隐瞒。

违反最大诚信的后果

根据我国《保险法》第十六条，违反如实告知义务的后果分几种情况：

投保人未如实告知 → 判断主观故意/重大过失

故意不告知：
  → 保险人有权解除合同
  → 解除前发生的保险事故，不赔不退保费

重大过失不告知：
  → 保险人有权解除合同
  → 未告知事项 *对* 保险事故的发生有严重影响 → 不赔但退保费
  → 未告知事项 *对* 保险事故的发生无影响 → 应该赔付

注意：保险人的解除权自知道解除事由之日起30日内行使，
超过30日不行使则解除权消灭。自合同成立之日起超过2年的，
保险人不得解除合同（不可抗辩条款）。

这个"两年不可抗辩条款"是保护投保人的重要制度——保险人不能在合同生效两年后以未如实告知为由解除合同。但要注意：如果投保人是故意欺诈（比如带病投保、伪造体检报告），即使过了两年，法院也可能支持保险公司拒赔。不可抗辩条款保护的是"过失性未告知"，不是"恶意欺诈"。近年来法院的审判趋势是：对故意未告知且影响重大的案件，即使过了两年也支持保险公司拒赔。

保险利益原则

保险利益的定义

保险利益（Insurable Interest）是指投保人或被保险人对保险标的具有的法律上承认的利益。简单说就是：被保险人会不会因为保险标的受损而遭受经济损失？如果会，就说明有保险利益；如果不会，就没有保险利益。

保险利益原则的核心功能是防止赌博和道德风险。如果任何人都可以为别人的财产或生命投保，那保险就变成了赌博——张三可以为李四的房子买火险，然后盼着李四家着火。这正是18世纪英国《赌博法案》（Gambling Act 1745）出台的背景：当时很多人为与自己毫无利益关系的船舶投保海上保险，本质就是在赌船会不会沉没。

财产保险的保险利益

财产保险的保险利益来源非常广泛，只要对财产具有合法的经济权益即可，包括：

  • 所有权：房产所有人对房屋有保险利益
  • 占有权：承租人对租赁的房屋有保险利益
  • 抵押权：银行对抵押的房产有保险利益（在其债权范围内）
  • 保管责任：仓储公司对客户存放的货物有保险利益
  • 合同权益：承包商对已完工但未交付的工程有保险利益

财产保险的保险利益有一个独特的时间要求：损失发生时必须存在保险利益，但投保时不要求存在。比如一家贸易公司为在途货物投保货运险——投保时货还在海上没到港，公司还没取得所有权，但这是允许的，只要在货物发生损失时公司已经取得了所有权（或承担了风险）。

人身保险的保险利益

人身保险的保险利益规则不同。我国《保险法》第三十一条规定，投保人对以下人员具有保险利益：

关系类型 范围
本人 任何人对自己都有保险利益
近亲属 配偶、子女、父母
其他亲属/有扶养关系的人 与投保人有抚养、赡养或者扶养关系的家庭其他成员、近亲属
劳动关系 用人单位对与其有劳动关系的劳动者
被保险人同意 被保险人同意投保人为其订立合同，视为有保险利益

人身保险的保险利益要求在投保时必须存在，但后续如果保险利益关系发生变化（比如离婚），已经成立的人身保险合同通常不受影响——这一点和财产保险完全不同。

对比分析：财产保险的保险利益要求"损失发生时存在"；人身保险的保险利益要求"投保时存在"。为什么有这个差异？因为财产保险的目的是补偿损失，如果没有保险利益就没有损失，自然不需要补偿。而人身保险合同往往具有储蓄和投资性质（如终身寿险、年金），如果因为婚姻关系变化就导致保单失效，对被保险人（受益人）可能不公平。这种差异体现了财产保险"补偿性"和人身保险"给付性"的本质区别。

保险利益的特殊问题

抵押权中的保险利益

在房屋抵押贷款中，银行（抵押权人）和借款人（抵押人）对同一房产都有保险利益。银行对房产的保险利益以其贷款余额为限。实务中，银行通常会要求借款人为抵押房产购买财产保险，并将银行列为"受益人"或"损失赔偿受领人"。这样如果房产发生火灾等损失，保险赔款会优先用于偿还银行贷款，剩余部分归借款人。

股权中的保险利益

股东对公司的财产是否有保险利益？这个问题在实务中经常有争议。严格来说，股东对公司财产没有直接的保险利益——因为公司是独立法人，公司财产不等于股东财产。公司损失导致股价下跌，但这种损失是间接的、不确定的，不构成保险法意义上的保险利益。所以股东不能以个人名义为公司财产投保。

但股东可以对自己的股权购买保险。比如一家创业公司的核心创始人可以为其股权购买"关键人物保险"——如果创始人因意外丧失工作能力，保险公司以约定价格收购其股权，保障创始人家属的权益。

期待利益

期待利益（Expectancy）是指基于现有利益而将来可能获得的利益，是财产保险中一种特殊的保险利益。典型的例子是预期利润——一家工厂购买了营业中断险，如果厂房因火灾停产，保险公司不仅赔偿厂房损失，还赔偿停产期间的预期利润损失。这里的"预期利润"就是期待利益。

期待利益虽然尚未实现，但只要它有合理的经济基础、可以被客观估算，就可以成为保险利益的客体。这也是为什么企业财产险通常会包含"利润损失险"（Business Interruption Insurance）的附加条款。

最大诚信的司法实践

如实告知的司法认定标准

近年来，随着保险纠纷数量增加，法院对如实告知义务的认定越来越精细化。主要的争议焦点包括：

争议焦点一：什么是"明知"？
  → 投保人说自己"不知道"有某种疾病即可免责？
  → 司法实践：以"正常人应知"为标准。体检报告明确提示异常
    且医生建议复查的，投保人不能以"没仔细看"为由主张不知情。

争议焦点二：询问方式是否有效？
  → 线上投保时的"健康告知弹窗"是否构成有效询问？
  → 多数法院认为：投保人勾选"以上告知均真实完整"即构成
    已履行询问义务，除非投保人能证明系统故障或界面误导。

争议焦点三：因果关系抗辩
  → 未告知事项与保险事故之间是否有因果关系？
  → 重疾险：未告知甲状腺结节，后来确诊甲状腺癌——有因果关系
  → 未告知甲状腺结节，后来确诊心脏病——无因果关系，应当赔付

2025年典型案例

2025年初，北京金融法院审理了一起广受关注的保险纠纷案。投保人刘某在2022年通过互联网投保了一份百万医疗险，投保时健康告知询问"过去2年内是否有住院或手术经历"。刘某2021年因胆结石做过腹腔镜手术，但投保时他认为"这是小手术，不算什么"就勾选了"否"。2024年刘某因急性胰腺炎住院治疗，产生医疗费用12万元，保险公司以未如实告知为由拒赔。

法院审理后认为：胆结石手术属于"手术经历"，保险公司询问明确，投保人未如实回答违反了告知义务。但法院同时查明，刘某本次住院的急性胰腺炎与之前的胆结石手术没有因果关系。根据保险法司法解释，未告知事项与保险事故的发生无影响的，保险人应当承担赔偿责任。最终判决保险公司赔偿12万元。

这个案例说明了三点：一是互联网保险的健康告知同样具有法律效力；二是违反告知义务不一定导致全盘拒赔，因果关系很重要；三是法院在保护投保人利益和尊重保险精算逻辑之间寻求平衡。但必须强调：这不是鼓励大家不如实告知——如果你故意隐瞒严重疾病且后续因此出险，拒赔是大概率结果。

教学提醒：实务中我见过太多"我以为不算病"的案例。投保人的"我以为"和保险法的"重要事实"之间往往有差距。正确的做法是：拿不准的，如实说。保险公司核保人员会根据医学知识判断这个情况是否影响承保决策。大多数小问题（如轻度脂肪肝、乳腺增生）核保结果都是"标准体承保"或者"除外承保"，完全不影响你获得保障。但如果你不说，将来理赔时就可能被认定为"未如实告知"。

好，第三章先到这里。我们讲了保险基本原则的前两个——最大诚信原则和保险利益原则。最大诚信原则要求双方诚实披露，保险利益原则防止赌博和道德风险。这两个原则是所有保险合同合法有效的基础。
