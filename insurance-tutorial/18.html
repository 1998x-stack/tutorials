第18章 保险资金运用

2015年，宝能系旗下的前海人寿在短短几个月内动用数百亿保险资金举牌万科A股，震动整个资本市场。而后安邦保险、华夏人寿等多家保险公司纷纷效仿，在二级市场掀起了一阵"险资举牌潮"。这些保险公司利用销售高现金价值保单快速获取保费收入，再将资金投入股市举牌上市公司——史称"野蛮人入侵"。这场风波最终导致监管部门出台了史上最严格的险资投资监管规定，多位保险公司高管被调查，数家公司的业务结构被强制整改。

这个故事揭示了一个核心问题：保险资金运用是一把双刃剑——用好了，保险公司可以给客户创造更多收益、降低保费负担；用歪了，可能把整个保险公司的偿付能力拖垮，甚至威胁金融系统稳定。这一章，我们就来拆解这门"含着金汤匙"的学问。

保险资金运用概述

资金来源

保险公司可以运用的资金，主要来自三个渠道：

资金来源 性质 规模与特点
资本金 股东投入的自有资金 相对较小，是"压舱石"，承担最终损失吸收功能
准备金 为未到期责任和未决赔款预提的资金 规模最大，是资金运用的主体。寿险公司的责任准备金占总资产的80%以上
保费收入 当期净保费扣除运营成本后的可用资金 现金流来源，但具有不确定性——新业务增速波动会影响可用资金量

截至2024年底，中国保险行业总资产约31万亿元，可运用资金规模接近28万亿元。这相当于A股总市值的约三分之一——体量巨大。保险资金因此被称为资本市场最重要的"长期机构投资者"之一。

运用三原则

保险资金运用有一条铁律——优先级永远不变：

  • 安全性（Safety）——第一位。保险资金是"负债性资金"，不是可以亏损的自有资金。如果投资亏了，未来赔款可能付不出来。寿险保单的兑现可能是二三十年后的事，但正因如此，安全第一。
  • 流动性（Liquidity）——第二位。需要随时准备赔付和给付。财产险的赔款波动大，流动性要求更高；寿险的给付周期相对可预测，流动性要求略低。
  • 收益性（Profitability）——第三位。在保证安全和流动的前提下，追求合理的收益。这不是说收益不重要——实际上，投资收益已经成了现代保险公司利润的最大来源之一，特别在利率下行、承保利润薄的时代。

关键数字：中国保险业平均综合投资收益率（2020-2024年）约4.5%-5.5%之间。看似不高，但考虑一下——28万亿的5%就是1.4万亿的投资收益。而保险行业的承保利润（即保费减去赔款和费用后的利润）在大多数年份其实是负数或微利状态——保险公司主要靠投资收益"养活"承保业务。这就是为什么保险业有句老话："承保亏损、投资补。"

投资渠道与配置

大类资产配置

中国保险资金的投资渠道经历了从"极度受限"到"逐步放开"的历程。目前可投资的大类资产包括：

资产类别 配置占比（行业均值） 特点
银行存款 约10% 安全性最高，但收益低（1.5%-2.5%），主要是流动性管理需要
债券（国债/地方债/金融债/信用债） 约45-50% 最大配置类别。国债和地方债占比最高，信用债需关注评级和久期
股票和基金 约10-15% 受比例监管限制（权益类投资合计不超过上季度末总资产的30%）
不动产 约5-8% 商业地产、写字楼、物流仓储等，以长期持有收租为主
另类投资 约25-30% 私募股权（PE）、基础设施（如高速公路、地铁、清洁能源）、债权投资计划

另类投资的崛起

近年来，另类投资在保险资金配置中的地位越来越重要。原因很简单：传统债券和存款的收益率随着利率下行不断走低，保险公司的负债成本（保单预定利率）却下降得慢。"利差损"压力下，保险公司不得不寻找更高收益的资产——另类投资就扮演了这个角色。

比如大型保险集团会设立自己的另类投资子公司（平安旗下的平安不动产、平安资本；太保旗下的太保私募等），直接投资基础设施项目——高速公路收费权、光伏电站、数据中心、长租公寓等。这些项目能提供5%-8%的长期稳定回报，且期限与保险负债高度匹配。

保险公司资产配置决策树（简化）：

输入：负债特征（久期/利率敏感性/流动性要求）
   ↓
第一步：确定战略资产配置（SAA）
   ├─ 根据负债久期确定债券组合的基准久期
   ├─ 根据偿付能力资本要求确定权益类比例上限
   └─ 根据现金流预测确定流动性资产最低配置
   ↓
第二步：进行战术资产配置（TAA）
   ├─ 基于市场判断进行短期偏离（例如"当前债券收益率处于低位，适度降低久期"）
   └─ 设置偏离限制（相对SAA的偏离不超过±5%）
   ↓
第三步：执行与再平衡
   ├─ 选定具体资产和投资标的
   ├─ 定期检查实际配置比例
   └─ 当偏离超过阈值时进行再平衡交易

资产负债管理（ALM）

久期匹配

资产负债管理（Asset-Liability Management, ALM）是保险公司投资管理的核心方法论。它解决的根本问题是：不要等客户来领钱的时候，发现钱投在还没到期的资产里拿不出来，或者市场不好亏了。

最核心的工具是久期匹配（Duration Matching）。久期（Duration）是债券价格对利率变化的敏感度的衡量指标，也是"加权平均回本时间"。如果保险公司的负债平均久期是12年（意味着保单未来12年内平均会兑现），那么资产组合的久期也应该在12年左右。这样，即使利率发生变化，资产和负债的价值变化会大致同步——这叫免疫策略（Immunization Strategy）。

现金流测试

除了久期匹配，保险公司还要做动态现金流测试——在多种利率情景下模拟未来几十年的现金流，看够不够支付赔款和给付。中国偿二代体系要求保险公司定期进行现金流压力测试，至少覆盖以下情景：

  • 利率上升300bp（base points，基点）
  • 利率下降300bp
  • 股票市场下跌30%
  • 保费收入下降一定比例

现实世界的ALM难题：中国寿险行业当前面临的最大ALM挑战是"长债短投"与"短债长投"并存。一方面，市场上长期国债（20年、30年）的供给不足，寿险公司的长久期负债找不到足够的长久期资产来匹配——这是"长债短投"；另一方面，部分公司为了追求收益，用短期的万能险资金去投长期的另类资产——这是"短债长投"，极可能导致流动性危机。两个方向都是风险，前者是再投资风险，后者是流动性风险。真正的ALM高手，是在负债端和资产端之间找到那个微妙的平衡点。

偿付能力对投资的约束

偿二代下的风险因子

偿付能力监管体系对保险资金运用构成了硬约束。中国的偿二代（C-ROSS, China Risk Oriented Solvency System）对不同资产类别设定了不同的风险因子（Risk Factor）——风险越高的资产，占用的偿付能力资本越多。

举个具体的例子：同样投资100元——

  • 投资国债：风险因子接近于0，占用偿付能力资本几乎为0
  • 投资AAA级信用债：风险因子约0.04，占用资本4元
  • 投资沪深300股票：风险因子约0.31，占用资本31元（衍生品对冲后可以降低）
  • 投资未上市股权：风险因子约0.48，占用资本48元

这个机制的核心逻辑是：偿付能力充足率 = 实际资本 / 最低资本。如果你持有的高风险资产多了，最低资本要求就高，偿付能力充足率就降。如果充足率跌破监管红线（综合充足率100%），公司就不能再开展新业务了。这就形成了一种"市场纪律"——逼着保险公司在高收益和高资本占用之间做出理性选择。

权益投资比例限制

除了通过偿付能力资本要求间接调控，监管还设定了直接的权益投资比例上限。根据现行规定，保险公司权益类资产的投资余额合计不得超过上季度末总资产的30%。对于偿付能力充足率更高的公司，上限可以适当提升（最高到45%）；对于充足率不足的公司，上限会被强制收紧。

2015年的"险资举牌潮"之后，监管还专门针对保险公司对上市公司的"重大股权投资"（即持有一家上市公司5%以上股份）出台了更加严格的审批要求——禁止保险机构与非保险机构一致行动人共同收购上市公司，禁止使用杠杆资金进行股权投资等。

踩坑故事：2020年，某寿险公司投资部门为了追求高收益，通过多层嵌套的资管产品大幅配置了房地产企业发行的信用债。2021年房地产市场下行，多家房企违约，该公司的信用债持仓出现大幅亏损。更要命的是，由于多层嵌套的资管产品涉及"期限错配"——底层资产是长期限的房地产信托，上层是短期限的保险产品——导致流动性危机。偿付能力充足率从200%跌到不足120%，被监管部门要求限期整改并限制新业务。这个案例的教训是：保险资金运用最危险的不是"亏损"本身，而是"流动性和期限错配"。只要你熬得住时间，债券总有到期日；但如果你等不起，被迫在低点卖出，那就是永久的实亏。

保险资金运用的监管

监管体系

中国保险资金运用的监管由国家金融监督管理总局（原银保监会，2023年改革后并入金融监管总局）负责。核心监管规章包括：

  • 《保险资金运用管理办法》——基础性规章，规定了保险资金运用的基本原则、投资范围和禁止性行为
  • 《关于加强和改进保险资金运用比例监管的通知》——设定各类资产的投资比例上限和集中度要求
  • 《关于保险资金投资股权和不动产有关问题的通知》——对另类投资的具体规定

大类资产比例监管

监管部门对保险公司的各类投资设置了"四道红线"似的比例约束：

保险资金运用比例监管核心指标：

┌────────────────────────────────┬───────────┐
│          监管指标              │  上限     │
├────────────────────────────────┼───────────┤
│ 权益类资产（股票+基金+股权）    │  ≤ 30%   │
│ 不动产类资产                   │  ≤ 30%   │
│ 其他金融资产（资管产品等）       │  ≤ 25%   │
│ 境外投资                      │  ≤ 15%   │
│ 单一上市公司股票               │  ≤ 5%    │
│ 单一保险资产管理产品            │  ≤ 10%   │
│ 流动性资产（现金+存款+货币基金） │  ≥ 5%    │
└────────────────────────────────┴───────────┘

2025年投资新趋势

站在2025年这个时间点，保险资金运用领域正在经历几个深刻的趋势性变化：

长期股权投资与"耐心资本"

2024年以来，监管部门持续鼓励保险资金作为"耐心资本"参与国家战略和实体经济。保险公司开始加大长期股权投资的配置力度——不是二级市场买卖股票，而是直接持有上市公司或非上市公司的大比例股权，参与公司治理，获取长期分红收益。中国平安持有工商银行、招商银行等大型银行股，中国人寿投资中石化、中国联通等央企——这些"长期底仓"每年贡献稳定的分红收入。

新会计准则影响

2023年起，中国上市保险公司开始分步实施新金融工具会计准则（IFRS 9）。新准则对保险资金投资最大的影响在于：过去计入"可供出售金融资产"的股票投资，价格波动可以计入权益（不影响利润），现在大部分需要按照"公允价值计量且变动计入当期损益"（FVTPL）来处理——这意味着股市波动会直接体现在季度利润中。这导致保险公司投资部门的考核方式发生了根本变化——从"追求长期总回报"到更加关注"利润波动管理"。

绿色投资与ESG

保险资金积极响应国家"双碳"战略，绿色投资规模快速增长。截至2024年底，保险资金在绿色领域的投资余额超过1.5万亿元，主要投向清洁能源（风、光、核）、绿色交通（新能源充电桩、电动车）、绿色建筑等领域。一些保险公司已经将ESG（环境、社会和治理）评分纳入了资产配置的决策流程——对ESG评分低于一定门槛的企业，自动限制投资敞口。

未来展望：保险资金运用正在从"被动配置"向"主动管理"转型。传统的"买入持有到期的债券组合+少量股票"模式，正在让位于更加多元化的配置——私募股权、夹层投资、基础设施REITs（房地产投资信托基金）、S基金（私募股权二级市场基金）等新资产类别不断加入保险投资组合。同时，资产负债管理能力正在成为保险公司核心竞争力的重要组成部分——能够精准匹配负债的资产配置能力，比短期的"赌方向"式的投资收益更值钱。

保险资金运用讲完了。这一章我们从保险公司的"钱袋子"出发，聊了资金来源、投资渠道、资产负债管理、监管约束和未来趋势。你会发现，保险公司的投资部门和一家普通的基金公司有着本质区别——保险投资有"负债"这个锚，所有的投资决策都必须回到底层负债的特征上面。
