第12章 Gauss 求积公式

大家好，我是你们的讲师。前面几章我们讨论的求积公式都有一个共同特点：节点是预先固定的（等距或指定位置），我们只优化权重。但如果我们能同时优化节点位置和权重呢？想象一下，如果 $n+1$ 个节点可以在区间内自由移动，那我们就有了 $2(n+1)$ 个自由度。用这些自由度，理论上我们可以让公式对 $2n+1$ 次多项式精确成立！这就是 Gauss 求积公式的核心思想——它是数值积分的"天花板"。

我第一次接触 Gauss 求积时大为震撼——$n=3$ 的 Gauss 公式只需要 3 次函数求值就能精确积分 5 次多项式，而梯形公式（2 次求值）只能精确积分线性函数。这效率差距简直是指数级的！后来我做有限元分析，单元刚度矩阵的数值积分全部用 Gauss 求积，因为每减少一次函数求值，整个仿真就能快不少。在工程计算中，函数求值往往是最耗时的环节，Gauss 求积的价值就在这里。

12.1 Gauss 求积的核心思想

考虑带有权函数 $w(x) \ge 0$ 的一般求积公式：

∫_a^b f(x) w(x) dx ≈ Σ_{i=0}^n w_i f(x_i)

在 Newton-Cotes 公式中，节点 $x_i$ 是固定等距的，我们只确定权重 $w_i$。这给了 $n+1$ 个自由度，所以代数精度最多为 $n$（或 $n+1$，某些对称情况）。

而 Gauss 求积将节点 $x_i$ 也视为自由参数，总共有 $2(n+1)$ 个自由度。高斯证明：通过适当选择节点和权重，可以使公式达到 $2n+1$ 次代数精度。这是求积公式的"理论上界"——不可能用 $n+1$ 个节点达到更高的代数精度。

那么，最优节点是什么？答案是：以权函数 $w(x)$ 正交的 $n+1$ 次正交多项式的零点。

重大 insight：Gauss 求积将两个看似无关的问题——数值积分和正交多项式——巧妙地联系了起来。正交多项式的零点不仅是多项式根，更是最优求积节点。这体现了数学中"不同领域的深层结构相互联系"的美妙之处。

12.2 Gauss-Legendre 求积

最常用的 Gauss 求积是 Gauss-Legendre 求积——区间 $[-1,1]$，权函数 $w(x)=1$。节点是 Legendre 多项式 $P_{n+1}(x)$ 的零点。

$n$节点 $x_i$权重 $w_i$代数精度
1$0$$2$1
2$\pm 1/\sqrt{3} \approx \pm 0.57735$$1$ (每个)3
3$0$, $\pm \sqrt{3/5} \approx \pm 0.77460$$8/9$, $5/9$ (每个)5
4$\pm 0.33998$, $\pm 0.86114$$0.65215$, $0.34785$7

Python 中计算 Gauss-Legendre 节点和权重非常方便：

import numpy as np
from numpy.polynomial.legendre import leggauss

# 计算 n 点 Gauss-Legendre 的节点和权重
n = 5  # 5 个节点，代数精度 = 2*5 - 1 = 9
x, w = leggauss(n)
print("节点:", x)
print("权重:", w)

# 在 [a, b] 上应用 Gauss-Legendre 公式
def gauss_legendre_quad(f, a, b, n):
    """n 点 Gauss-Legendre 求积"""
    # 节点和权重在 [-1, 1] 上
    x_ref, w_ref = leggauss(n)
    # 映射到 [a, b]
    half = (b - a) / 2
    mid = (a + b) / 2
    x_mapped = half * x_ref + mid
    # 积分变换 ∫_a^b f(x) dx = (b-a)/2 ∫_{-1}^1 f((b-a)/2 * t + (a+b)/2) dt
    result = half * np.sum(w_ref * f(x_mapped))
    return result

# 测试 f(x) = e^x 在 [0, 1] 上
f = np.exp
exact = np.exp(1) - 1
for n in [1, 2, 3, 4, 5]:
    approx = gauss_legendre_quad(f, 0, 1, n)
    print(f"n={n}: {approx:.12f}, 误差={abs(approx-exact):.2e}")

惊人的效率：用 $n=5$ 的 Gauss-Legendre 求积（仅 5 次函数求值）计算 $\int_0^1 e^x dx$，误差已经可以达到 $10^{-12}$ 量级。而复合梯形公式要达到相同精度大约需要 $10^6$ 次函数求值！这就是 Gauss 求积的效率优势。

12.3 其他 Gauss 型求积

不同区间和权函数的组合对应不同的正交多项式族，从而得到不同形式的 Gauss 求积公式：

名称区间权函数 $w(x)$正交多项式典型应用
Gauss-Legendre$[-1, 1]$$1$Legendre $P_n(x)$一般数值积分
Gauss-Chebyshev$[-1, 1]$$1/\sqrt{1-x^2}$Chebyshev $T_n(x)$端点奇异的积分
Gauss-Laguerre$[0, \infty)$$e^{-x}$Laguerre $L_n(x)$半无限区间积分
Gauss-Hermite$(-\infty, \infty)$$e^{-x^2}$Hermite $H_n(x)$概率论、量子力学

选择指南：如果你的被积函数包含 $e^{-x}$ 因子（比如原子物理中的波函数重叠积分），用 Gauss-Laguerre 求积可以把指数因子吸收到权函数中，剩下的部分用多项式很好地逼近。类似地，如果被积函数在端点有 $1/\sqrt{1-x^2}$ 型的奇异性，Gauss-Chebyshev 可以让这部分的奇异性被权函数"吃掉"，数值计算变得非常稳定。

12.4 Gauss vs Newton-Cotes

最后我们来做一个全面的对比：

对比维度Gauss 求积Newton-Cotes 求积
节点选择自由（正交多项式零点）固定（等距）
$n+1$ 个节点的代数精度$2n+1$$n$（或 $n+1$）
精度效率极高（相同节点数最优）较低
嵌套性不可嵌套（节点不重合）部分可嵌套（如 Romberg）
实现复杂度中等（需查节点和权重）简单
适用场景高精度、函数求值昂贵快速估算、教学演示

踩坑提醒——嵌套性问题：Gauss 求积的一个"缺点"是节点不能嵌套。当你用 $n$ 点 Gauss 公式算完发现精度不够，想用 $n+1$ 点 Gauss 公式再算时，之前 $n$ 个节点的函数值全部浪费了——新的节点位置完全不同。而 Newton-Cotes 的复合梯形公式可以通过递推公式 $T_{2n} = T_n/2 + h\sum f_{\text{new}}$ 利用之前的计算结果。所以如果精度要求不高但需要灵活控制，复合梯形 + Richardson 外推（Romberg）可能更实用。

本章我们学习了数值积分的"天花板"——Gauss 求积公式。通过同时优化节点和权重，用最少的函数求值次数达到了最高的代数精度。Gauss 求积与正交多项式紧密关联，这为我们下一章的学习埋下了伏笔。下一章我们将暂时离开积分，转向数值微分——用离散数据近似计算导数。这看起来像是"反问题"，但有趣的是，Richardson 外推在数值微分中同样大有用武之地！
