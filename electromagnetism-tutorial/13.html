开场：一个悖论催生的革命

1861年，詹姆斯·克拉克·麦克斯韦（James Clerk Maxwell, 1831-1879）正在撰写《论物理力线》。当他仔细审视安培环路定理 ∮B·dl = μ₀ Ienclosed 时，发现了一个令人不安的矛盾。

考虑一个正在充电的电容器。取一个环绕电容器一根引线的闭合回路。根据安培环路定理，B 的环量应该等于穿过回路所围曲面的电流。但问题来了：

  • 如果取一个穿过导线的曲面——曲面上有电流通过，Ienclosed = I。
  • 如果取一个穿过电容器极板之间的曲面——曲面上没有电流，Ienclosed = 0。

Ampère 定律给出的 B 竟然依赖于曲面的选择！这对于任何物理定律都是不可接受的——一个正确的物理定律必须与曲面的选择无关。

麦克斯韦的解决方案是什么？他做出了一记惊世骇俗的天才之笔：变化的电场产生的效果相当于一种"电流"。

位移电流

麦克斯韦将安培环路定理中的电流 Ienclosed 替换为总电流——传导电流 + 位移电流：

Itotal = Iconduction + Idisplacement

其中位移电流（displacement current）定义为：

Id = ε₀ dΦE / dt  或  Jd = ε₀ ∂E / ∂t

在电容器极板之间，虽然没有实际的电荷迁移（没有传导电流），但电场在随时间变化（充电时 E 增大），这个变化产生了位移电流。注意：位移电流不是真实的电流——它是变化电场的等效磁效应。

🔑 核心洞察
"位移电流不是电流——它是麦克斯韦最天才的洞察。"
很多人第一次学到这里会困惑：既然不是真实的电荷流动，为什么叫"电流"？
原因在于它在产生磁场方面等同于电流。变化的电场激发了磁场，就像真实的电流激发磁场一样。这实际上是电磁对称性的体现：
— 变化的磁场 → 激发电场（法拉第定律，∇×E = −∂B/∂t）
— 变化的电场 → 激发磁场（麦克斯韦修正，∇×B = μ₀ε₀ ∂E/∂t）
这二者的对称性是电磁波存在的数学根源。

修正后的安培-麦克斯韦定律：

∮C B · dl = μ₀ (IC + ε₀ dΦE/dt)

微分形式：∇ × B = μ₀ J + μ₀ ε₀ ∂E / ∂t

麦克斯韦方程组（完整形式）

加上位移电流后，电磁学完成了最后的统一。以下是完整的麦克斯韦方程组：

编号 名称 积分形式 微分形式 物理意义
1 电场高斯定理 ∮S D · dS = Qf ∇ · D = ρf 电荷是电场的源（发散源）
2 磁高斯定理 ∮S B · dS = 0 ∇ · B = 0 不存在磁单极子
3 法拉第定律 ∮C E · dl = −dΦB/dt ∇ × E = −∂B/∂t 变化磁场激发电场
4 安培-麦克斯韦定律 ∮C H · dl = If + dΦD/dt ∇ × H = Jf + ∂D/∂t 电流和变化电场激发磁场

在真空中（无源、无介质），方程简化为：

  • ∇ · E = 0
  • ∇ · B = 0
  • ∇ × E = −∂B / ∂t
  • ∇ × B = μ₀ ε₀ ∂E / ∂t

除了一个负号和一个常数，电场和磁场几乎完全对称！这种对称性正是电磁波存在的数学基础。

对称性与电磁对偶

观察真空中的麦克斯韦方程组：电场和磁场互换（E → cB, B → −E/c）后，方程形式保持不变（在无源情况下）。这种电磁对偶性（Electromagnetic Duality）是物理学中最深刻的对称之一。

如果存在磁单极子（磁荷 ρm 和磁流 Jm），这种对偶就会变成完美对称——不过目前物理实验尚未发现磁单极子的存在。

⚠ 常见误区澄清
一些教科书会说："位移电流就是变化的电场。"这么说容易产生误解。
实际上，位移电流密度 Jd = ε₀ ∂E/∂t 并不是电场本身，而是电场的时间变化率。它只在电场变化时不为零——恒定电场不产生位移电流。
还有一个常见的陷阱：位移电流在电介质中还有极化项贡献：Jd = ε₀ ∂E/∂t + ∂P/∂t（其中 P 是极化强度），这才是完整的形式。

从麦克斯韦到光

麦克斯韦在完成方程组后不久，做了一个改变世界的推导。他从真空中的四个方程出发，推导出了电磁场的波动方程：

对 ∇ × E = −∂B/∂t 取旋度：

∇ × (∇ × E) = −∂(∇ × B)/∂t = −μ₀ ε₀ ∂²E/∂t²

利用矢量恒等式 ∇ × (∇ × E) = ∇(∇·E) − ∇²E，且 ∇·E = 0：

∇²E = μ₀ ε₀ ∂²E/∂t²

这正是标准的波动方程！从波动方程中可以读出电磁波的传播速度：

c = 1 / √(μ₀ ε₀) = 2.9979 × 10⁸ m/s

麦克斯韦当时震惊了：这个速度与当时测量的光速（3.14 × 10⁸ m/s，设备精度有限）高度吻合。他随即宣布：

"光是一种电磁波。"

这是19世纪物理学最伟大的预言——它统一了电学、磁学和光学，将光学降格为电磁学的一个子领域。

💡 数值验证：从 μ₀ 和 ε₀ 计算光速
既然 c = 1/√(μ₀ ε₀)，我们可以用手边的值验证一下：
μ₀ = 4π × 10⁻⁷ H/m（精确值，这是定义）
ε₀ ≈ 8.854187817 × 10⁻¹² F/m（由 μ₀ 和 c 导出）
c = 1 / √(4π × 10⁻⁷ × 8.854187817 × 10⁻¹²) ≈ 2.99792458 × 10⁸ m/s
这与现代测量值完全一致！μ₀ 和 ε₀ 这两个分别来自磁学和电学的常数，居然在麦克斯韦方程组中共同决定了光速的大小——这种巧合只能用"统一"来解释。

Python 验证：从 μ₀ 和 ε₀ 到光速

import numpy as np

# 真空磁导率 (精确定义值)
mu0 = 4 * np.pi * 1e-7   # H/m

# 真空介电常数 (定义值, 由光速导出)
epsilon0 = 8.8541878128e-12   # F/m

# 计算光速
c_calculated = 1 / np.sqrt(mu0 * epsilon0)
c_known = 299792458  # 标准值 m/s

print(f"从 μ₀ 和 ε₀ 计算的光速: {c_calculated:.2f} m/s")
print(f"已知光速 (标准值):      {c_known:.2f} m/s")
print(f"相对误差: {abs(c_calculated - c_known) / c_known * 100:.2e} %")
print()

# 更直观：电磁波的电场和磁场分量
# 在自由空间传播的平面电磁波
# E = E₀ cos(kz - ωt) x̂
# B = (E₀/c) cos(kz - ωt) ŷ
E0 = 1.0        # V/m
c = c_known
f = 100e6       # 100 MHz
omega = 2 * np.pi * f
k = omega / c
z = np.linspace(0, 3*c/f, 1000)  # 3个波长
t = 0

E = E0 * np.cos(k * z - omega * t)
B = (E0 / c) * np.cos(k * z - omega * t)

print(f"频率 f = {f/1e6:.0f} MHz")
print(f"波长 λ = {c/f:.2f} m")
print(f"波数 k = {k:.4f} rad/m")
print(f"电场幅值 E₀ = {E0} V/m")
print(f"磁场幅值 B₀ = E₀/c = {E0/c:.2e} T")

# 验证 E/B = c
print(f"\nE/B 比值 (应为光速): {E0 / (E0/c):.2f} m/s")
🔑 核心洞察
麦克斯韦做出"光是电磁波"这个预言时，他只有31岁。20多年后的1887年，赫兹用实验首次验证了电磁波的存在——他用一个火花隙产生了电磁波，并检测到了它的反射、折射和偏振性质。
但赫兹验证时，麦克斯韦已去世9年（1879年，享年仅48岁）。他没能亲眼看到自己的理论被证实，也没能预见电磁波将彻底改变人类文明——从无线电、电视到手机和Wi-Fi。
费曼曾评价："从长远来看人类的历史——比如说一万年以后回头看——19世纪最重要的事件无疑是麦克斯韦发现了电磁学定律。与这个重要的科学事件相比，同期的美国内战只是地域性的琐事。"

本章小结

  • 位移电流：变化的电场产生磁效应——Id = ε₀ dΦE/dt，是麦克斯韦对安培环路定理的关键修正。
  • 麦克斯韦方程组：四个方程完整描述了经典电磁学——电场高斯定理、磁高斯定理、法拉第定律、安培-麦克斯韦定律。
  • 电磁对称性：真空中电场和磁场高度对称——变化电场激发磁场，变化磁场激发电场。
  • 光即电磁波：c = 1/√(μ₀ ε₀) = 2.9979×10⁸ m/s——电磁波速度恰好等于光速。
  • 麦克斯韦方程组是物理学的巅峰之一——它统一了电、磁、光三大领域。

麦克斯韦方程组是电磁学的巅峰。掌握了它，下一步我们看看场在介质界面上如何行为——电磁场的边界条件，这是衔接理想模型与现实世界的桥梁。
