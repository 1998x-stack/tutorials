第8章 资产组合理论

2008年的一个深秋夜晚，哈佛大学捐赠基金的CIO（首席投资官）坐在办公室里，盯着屏幕上的一片血红。这支管理着超过360亿美元的基金，当年亏损了27%。不仅仅是哈佛——耶鲁亏了25%，斯坦福亏了30%。这些世界最顶尖的投资机构，拥有最聪明的人才和最先进的模型，为什么在同一时间遭受了如此惨重的损失？

答案就藏在一个古老的投资问题里："把所有鸡蛋放在一个篮子里"和"把鸡蛋分散在不同的篮子里"，到底哪个更优？这看似简单的问题，直到1952年才被一位25岁的年轻人用数学给出了答案。他就是哈里·马科维茨（Harry Markowitz），他提出的现代资产组合理论（Modern Portfolio Theory, MPT）彻底改变了投资世界的面貌。这一章，我们就来揭开它的面纱。

本章核心：资产组合理论告诉我们，分散化是投资中唯一的"免费午餐"。通过合理配置不同资产，你可以在不降低预期收益的情况下降低风险，或者在不增加风险的情况下提高预期收益。这听起来像是魔法，但背后有坚实的数学支撑。

8.1 马科维茨均值-方差模型

模型的三大假设

任何理论模型都有假设条件，马科维茨的模型也不例外。理解这些假设，你才能明白什么时候该用理论、什么时候该相信直觉：

  • 投资者是理性的：所有投资者都是风险厌恶的，在给定的风险水平下追求收益最大化，或在给定的收益水平下追求风险最小化。
  • 收益服从正态分布：资产的收益率可以用均值（期望收益）和方差（风险）两个参数完全刻画。
  • 投资者只关注均值和方差：投资决策只基于预期收益和风险，不考虑流动性、税收等其他因素。

这些假设听起来很严格，但正是这种简化让我们能够用数学语言来描述组合选择问题。

两资产组合的数学表述

先看最简单的情况：只考虑两种资产。假设你投资了股票A和债券B，权重分别为 w_A 和 w_B（w_A + w_B = 1），那么组合的预期收益和方差分别为：

# 两资产组合的数学公式
# 预期收益：加权平均
E(R_p) = w_A * E(R_A) + w_B * E(R_B)

# 方差：取决于两种资产的相关性
σ²_p = w_A² * σ²_A + w_B² * σ²_B + 2 * w_A * w_B * Cov(R_A, R_B)

# 其中协方差 Cov(R_A, R_B) = ρ_AB * σ_A * σ_B
# ρ_AB 是资产A和B的相关系数，取值范围[-1, 1]

这个公式中藏着一个关键洞见：组合的风险不仅取决于单个资产的风险，更取决于资产之间的相关性。如果你选择两只完全正相关（ρ=1）的股票，分散化几乎没有意义。但如果你选择相关性较低的资产（比如股票和债券，历史上ρ≈0.2-0.4），就可以显著降低组合的整体风险。

关键洞察：当ρ = -1时（完全负相关），理论上你可以构建一个零风险组合！但在现实中，几乎找不到完全负相关的资产。不过这个思想告诉我们——找到低相关甚至负相关的资产，是构建高效组合的核心策略。

8.2 可行集与有效前沿

当你把两种资产以不同比例组合时，所有可能的（风险，收益）点就构成了可行集（Feasible Set）。可行集的左边界就是大名鼎鼎的有效前沿（Efficient Frontier）。

有效前沿上的每个点代表一个"帕累托最优"的组合——在给定风险下收益最高，或在给定收益下风险最低。位于有效前沿之下的组合都是"次优"的。

最小方差组合与全局最小方差

在有效前沿上，最左边的点被称为全局最小方差组合（Global Minimum Variance Portfolio, GMVP）。这是所有可能组合中风险最低的那个。它的权重可以通过求导得出：

# 两资产全局最小方差组合的权重
# 假设：σ_A = 20%, σ_B = 10%, ρ = 0.3

σ_A = 0.20; σ_B = 0.10; ρ = 0.3
Cov = ρ * σ_A * σ_B

# 最小方差组合权重
w_A_min = (σ²_B - Cov) / (σ²_A + σ²_B - 2*Cov)
w_B_min = 1 - w_A_min

# 计算结果：w_A_min ≈ 0.18, w_B_min ≈ 0.82
# 即：82%的债券 + 18%的股票组合风险最低

这个数学结果告诉我们一个反直觉的结论：即使股票的预期收益远高于债券，追求最低风险的投资者也应该持有少量股票——因为股票和债券的低相关性可以提供分散化收益。

组合类型股票权重债券权重预期收益标准差
全股票100%0%10%20%
全债券0%100%4%10%
GMVP18%82%5.1%9.2%
均衡组合60%40%7.6%13.1%

注意GMVP的标准差（9.2%）甚至低于全债券组合（10%）！这就是分散化的神奇之处——1+1 < 2，组合的风险小于各部分风险之和。

8.3 无差异曲线与最优组合

有效前沿给出了所有"技术上的最优组合"，但具体选哪个点，取决于投资者的风险偏好。在经济学中，我们用无差异曲线（Indifference Curve）来刻画投资者对风险和收益的权衡态度。

每一条无差异曲线代表给投资者带来相同满意度的所有（风险，收益）组合。曲线的斜率反映了投资者的风险厌恶系数（Risk Aversion Coefficient, A）：

# 效用函数
U = E(R) - 0.5 * A * σ²

# 其中：
# U = 投资者的效用（Utility）
# E(R) = 组合预期收益
# A = 风险厌恶系数（A > 0 表示风险厌恶）
# σ² = 组合方差

# 例如：A=3的投资者，面对一个E(R)=8%, σ=15%的组合
U = 0.08 - 0.5 * 3 * (0.15)² = 0.08 - 0.03375 = 4.625%

A越大，投资者越厌恶风险，要求的风险溢价越高。典型的A值范围在2-5之间。机构投资者（如养老金）的A值通常较高（约4-5），而对冲基金的A值较低（约2-3）。

最优组合就是无差异曲线与有效前沿的切点。在这个点上，投资者在现有的技术条件下获得了最高的效用。

踩坑实录：2000年互联网泡沫期间，很多投资者自认为"风险承受能力很强"，但实际上他们混淆了"风险偏好"和"风险错觉"。当纳斯达克暴跌78%时，那些自诩"A=2"的投资者纷纷在底部割肉——真实的风险厌恶系数在熊市中会急剧上升。这就是所谓的风险偏好时变性，我们将在第12章行为金融学中详细讨论。

8.4 无风险资产与资本配置线

上面讨论的组合只包含风险资产。现在，我们引入无风险资产（Risk-free Asset）——通常用短期国债（如美国T-bill或中国国债）来代表。这极大地扩展了投资者的选择空间。

当你可以在无风险资产和风险组合之间自由配置时，所有可能的组合构成了一条直线，即资本配置线（Capital Allocation Line, CAL）：

# 资本配置线公式
E(R_c) = R_f + [(E(R_p) - R_f) / σ_p] * σ_c

# 其中：
# R_f = 无风险利率（如中国10年期国债收益率 ≈ 1.8%）
# E(R_p) = 风险组合的预期收益
# σ_p = 风险组合的标准差
# σ_c = 最终组合的标准差

# CAL的斜率 = (E(R_p) - R_f) / σ_p，又称夏普比率（Sharpe Ratio）

最优风险组合就是CAL斜率最大的那个风险组合——也就是夏普比率（Sharpe Ratio）最高的组合。所有理性投资者都会选择同一个最优风险组合（无论他们的风险偏好如何），区别只在于他们在无风险资产和这个最优风险组合之间的配置比例。这个重要的性质被称为分离定理（Separation Theorem）。

分离定理的含义极其深刻：资产配置（无风险vs风险组合）和产品选择（风险组合内部构成）可以分开决策。前者取决于你的风险偏好，后者只取决于市场条件。

8.5 实际应用：从理论到实践

理论讲完了，让我们看看怎么在实际中使用。构建一个马科维茨最优组合需要以下步骤：

第一步：估计预期收益

这是最困难也最容易被低估的一步。历史平均收益并不等于未来预期收益。常用的方法包括：资本资产定价模型（CAPM）估算、分析师一致预期、隐含收益率（从估值反推）等。

第二步：估计协方差矩阵

你可以用历史数据计算样本协方差矩阵，但这会带来大量估计误差。更稳健的方法包括：单因子模型简化、收缩估计（Shrinkage Estimation）、指数加权移动平均（EWMA）等。

踩坑实录：一家国内券商资管在2015年构建了一个"数学上完美"的组合优化模型。模型输出的权重高度集中在几只看似"低风险高收益"的股票上（权重超过80%）。结果2015年股灾中这几只股票恰好是同行业的，相关性在危机中急剧上升到0.9以上，组合单日回撤超过15%。教训是：样本外相关性会显著高于样本内相关性，尤其在危机时期。永远不要盲目信任优化器的极端输出。

第三步：加入约束条件优化

在实践中，不加约束的马科维茨优化会产生极端且不可行的权重。常见的约束包括：

  • 权重非负（不允许做空）：大多数公募基金和养老金不允许做空
  • 行业集中度限制：单个行业不超过20%
  • 个股上限：单只股票不超过5%
  • 换手率约束：减少交易成本

Python实践示例

# 使用Python实现均值-方差优化
import numpy as np
from scipy.optimize import minimize

# 假设三只股票的预期收益和协方差矩阵
mu = np.array([0.12, 0.08, 0.05])  # 预期收益
cov = np.array([[0.04, 0.008, 0.002],
                [0.008, 0.0225, 0.004],
                [0.002, 0.004, 0.01]])

# 优化目标：在给定收益下最小化方差
def portfolio_var(weights, cov):
    return weights.T @ cov @ weights

def target_return(weights, mu, target):
    return np.sum(weights * mu) - target

# 设置目标收益为9%
target_r = 0.09
n = len(mu)
init_guess = np.ones(n) / n
bounds = tuple((0, 0.5) for _ in range(n))  # 单只权重不超过50%
constraints = [
    {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
    {'type': 'eq', 'fun': lambda w: target_return(w, mu, target_r)}
]

result = minimize(portfolio_var, init_guess,
                  args=(cov,), method='SLSQP',
                  bounds=bounds, constraints=constraints)
print("最优权重:", result.x)
# 输出示例: [0.45, 0.35, 0.20]
# 夏普比率 = 0.68
实用建议：在实际工作中，不要直接使用"原始"的有效前沿。建议采用Black-Litterman模型作为替代方案——它结合了市场均衡收益和投资者主观观点，输出更加稳定、更符合直觉的权重配置。此外，等权重组合（1/N策略）虽看似"没技术含量"，但在很多实证研究中表现优于复杂的优化模型——这就是"少即是多"在投资中的体现。
方法优点缺点适用场景
均值-方差优化数学严谨，理论优美对输入敏感，输出极端学术研究、参考基准
Black-Litterman稳定、融合主观观点实现复杂、参数难调大型机构资产配置
风险平价分散化好，不依赖收益预测可能牺牲收益养老金、FOF组合
等权重（1/N）简单、稳健、成本低忽略个体差异个人投资者、定投
本章要点回顾：
  • 马科维茨均值-方差模型证明了分散化投资的数学基础——组合风险取决于资产间的相关性而非单个资产风险之和
  • 有效前沿上的组合在给定风险下收益最高，或给定收益下风险最低
  • 全局最小方差组合（GMVP）是所有可能组合中风险最低的那个
  • 引入无风险资产后，资本配置线（CAL）和分离定理将资产配置与品种选择分离
  • 实践中要注意协方差矩阵估计的稳健性，使用约束优化避免极端权重

马科维茨的理论奠定了现代投资组合管理的基石。但这里有一个天然的延伸：如果所有投资者都使用同样的方法配置资产，那么资产的均衡价格应该是多少？这个问题引向下一章——资本资产定价模型（CAPM）。CAPM将告诉我们，一项资产的预期收益到底应该由什么决定。
