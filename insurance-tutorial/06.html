第6章 保险费率与精算原理

大家好，我是你们的讲师。前面五章我们一直在聊保险的法律和制度层面，这一章终于进入保险最"硬核"的部分——精算科学。

很多人觉得精算很神秘——精算师也确实被戏称为"保险界最聪明的人"。但精算的核心思想并不复杂：计算风险的概率和成本，然后定出一个合理的价格。说白了，就是给风险"标价"。

我就从这个角度切入：假设你是一家保险公司的老板，有人来找你投保。你需要回答两个问题：第一，这个人出事的概率有多大？第二，如果出事了，你要赔多少钱？两个问题都回答了，保费自然就出来了。精算就是让这个过程从"拍脑袋"变成"按公式"。

大数定律与保险

大数定律的精算基础

大数定律（Law of Large Numbers）是保险业存在的数学根本。用最简洁的话说：个体风险不可预测，群体风险可以预测。

抛一枚公平硬币，你无法预测下一次是正面还是反面。但随着抛掷次数增加，正面出现的比例会越来越稳定地趋近50%。同样道理，你无法预测某一个人明年的住院概率——但如果有100万人，你就可以相当精确地预测出明年大概会有多少人住院、平均住院费用是多少。

精算学家用公式来表达这个原理：

设 X₁, X₂, ..., Xₙ 为 n 个独立同分布的风险单位的损失金额
每个风险单位的期望损失为 E[X] = μ，方差为 Var[X] = σ²

则 n 个风险单位的平均损失为 X̄ = (X₁ + X₂ + ... + Xₙ)/n

大数定律告诉我们：当 n → ∞ 时，X̄ → μ
即样本均值会无限接近期望值

实际损失率与期望损失率的偏差与 √n 成反比：
P(|X̄ - μ| < ε) ≥ 1 - σ²/(nε²)  (切比雪夫不等式)

这意味着：承保的同质风险越多，实际赔付率就越接近预期赔付率，
保险公司面临的不确定性就越小。

这条定律决定了保险公司必须追求规模——没有足够的保单数量，精算定价就失去了统计基础，风险定价就会失准。

大数定律的局限性

大数定律有两个关键假设：独立性和同分布。在现实中，这两个假设常常不能完全满足。地震发生时，一个城市的所有保单同时发生损失——不独立。不同年龄、不同职业的人死亡率差异很大——不同分布。所以精算师需要在这些"不完美"条件下做调整，引入风险分类、巨灾模型、信度理论等工具来改进定价精度。

数学洞察：为什么保险公司对"稀有风险"定价困难？比如太空旅行保险——目前全球只有几百人上过太空，样本量太小，大数定律根本不适用。这种情况下，精算师只能使用"信度理论（Credibility Theory）"，把少量实际数据与先验信息（行业数据、模型推演）进行加权平均。样本量越少，先验信息的权重越大。这本质上是贝叶斯统计在保险定价中的应用。

保费构成

一份保单的保费由几个部分构成：

组成部分 占比（典型） 说明
纯保费（Pure Premium） 60%-75% 用于支付预期赔款，是保费的核心。纯保费 = 期望损失频率 × 期望损失金额
安全附加（Risk Loading） 5%-10% 用于应对实际损失超过预期的情况，相当于"缓冲垫"
附加保费（Expense Loading） 15%-25% 用于支付佣金、管理费、核保理赔成本等经营费用
利润附加（Profit Loading） 2%-5% 保险公司的预期利润，包括风险资本回报

用公式表示：毛保费 = 纯保费 + 安全附加 + 附加保费 + 利润附加

纯保费的计算思路是：如果你承保了1000栋房子，每栋值100万，历史数据显示每年约有1栋着火全损。那么纯保费就是 1/1000 × 100万 = 1000元/栋。但这只是"平均"——实际可能一年烧2栋（赔200万），也可能一年都不烧。为了应对这种波动，需要加安全附加。通常使用标准差的一定倍数作为安全附加。

财产险费率厘定

财产险费率厘定有四种基本方法，从简单到复杂：

分类法（Class Rating）

把风险单位分成几个大类，同一类内执行统一费率。比如车险中按"家用轿车""商务车""货车"分类，每类同一个基准费率。这个方法简单、操作成本低，但缺点是同一类内不同个体的风险差异被忽略了。

表定法（Schedule Rating）

在分类法的基础上，根据每个风险的具体特征做加减调整。例如企业财产险的基准费率是0.1%，但根据消防设施（减费）、建筑材质（加费或减费）、周边环境（加费）等因素调整。表定法考虑了更多个体差异，更公平。

经验法（Experience Rating）

根据被保险人过去的实际损失经验来调整费率。如果你的车过去三年都没出过事故——费率下调；如果每年出两次险——费率上调甚至拒保。经验法最适合"大客户"或自留风险水平较高的保单，因为个体数据有统计意义。

追溯法（Retrospective Rating）

最复杂的定价方式：先按标准费率收保费，保险期满后根据实际发生的损失来确定最终保费。损失小—退费；损失大—加费。追溯法常见于大型企业的团体保险方案，把被保险人的风险管理积极性与保费直接挂钩。

四种费率厘定方法的对比：

方法       数据来源           公平性    复杂度    适用场景
分类法     行业/群体数据       低        低        个人标准化产品
表定法     分类法+个体特征     中       中低      中小企业财产险
经验法     个体历史损失数据    高        中        大企业、团体保险
追溯法     当期实际损失数据    最高      高        超大型项目

寿险费率厘定

生命表

寿险精算最核心的工具是生命表（Life Table / Mortality Table）。生命表记录了不同年龄、不同性别的人的死亡率，精算师根据这个表来计算寿险保费。生命表的数据来源是人口普查、人寿保险经验数据等。

生命表的核心指标包括：

  • qx：x岁的人在一年内死亡的概率
  • lx：x岁时的生存人数（假设初始10万人）
  • dx：x岁到x+1岁之间的死亡人数
  • ex：x岁时的平均余命（预期剩余寿命）

预定利率与预定费用率

除了死亡率，寿险定价还需要考虑两个重要的假设：

  • 预定利率（Assumed Interest Rate）：保险公司预期能用保费投资赚取的年化收益率。利率越高，保费越低（因为未来的投资回报可以抵消一部分保费）。
  • 预定费用率（Assumed Expense Rate）：保险公司预期的经营管理费用占保费的比例。费用率越高，保费越高。

三差分析

寿险公司的利润来源可以用"三差"来概括：

名称 含义 盈利来源
死差（Mortality Margin） 实际死亡率与预定死亡率的差异 实际死亡人数比预定少 → 死差益
费差（Expense Margin） 实际费用率与预定费用率的差异 实际经营成本比预定低 → 费差益
利差（Interest Margin） 实际投资收益率与预定利率的差异 实际投资回报比预定高 → 利差益

行业趋势：近年来，中国寿险业的利润结构中，利差益的占比越来越高（超过80%），死差益和费差益的贡献在下降。这意味着寿险公司越来越像"资产管理公司"——不是靠"少死人"赚钱，而是靠"投资能力强"赚钱。这也带来了新的风险：如果市场利率持续下行，寿险公司的利差可能从"益"变成"损"（利差损）。这正是2024-2025年很多保险公司面临的挑战：过去承诺了较高的预定利率，但当前市场利率大幅下降，导致巨大的利差损风险。

风险分类与费率差异化

风险因子的选择

精算定价的核心挑战是：如何把风险群体细分，使每个群体的费率与其真实风险水平匹配。常用的风险因子包括：

  • 寿险/健康险：年龄、性别、吸烟状况、职业、家族病史、体检指标
  • 车险：年龄、驾龄、车型、行驶里程、违章记录、地区
  • 财产险：建筑结构、消防设施、地区、行业、安保措施

风险分类需要平衡公平性和可操作性。分类太粗——低风险人群补贴高风险人群，导致低风险者流失；分类太细——操作成本高且可能面临"歧视"指控。监管机构通常会对可使用的风险因子做出限制，比如禁止使用种族、宗教等作为定价因子。

费率监管

费率不是保险公司想定多少就定多少。我国对保险费率实行审批制和备案制相结合：

  • 审批制：关系社会公众利益的险种（如交强险）、依法实行强制保险的险种、新开发的人寿保险险种——费率需报监管机构审批
  • 备案制：其他险种的费率在销售前报监管机构备案即可

监管机构有权要求保险公司调整不合理的费率，以保护消费者利益和维护市场秩序。

2025年生命表更新对费率的影响

2025年初，中国精算师协会发布了第四套经验生命表，取代了2016年的第三套。这次生命表更新有几个显著变化：

  • 预期寿命延长：男性平均预期寿命从第三套的76.7岁提高到78.3岁，女性从82.2岁提高到84.1岁
  • 高龄段死亡率下降明显：70岁以上人群死亡率较十年前下降了约12%
  • 低龄段死亡率变化不大：但意外死亡（特别是年轻人）有所增加

这些变化对寿险产品费率产生了直接影响：

  • 终身寿险：寿命延长意味着保险公司赔付时间更晚、保费积累期更长，理论上保费可以下降5%-8%
  • 定期寿险：死亡率的整体改善使保费有下调空间，但低龄意外死亡增加部分抵消了降幅
  • 年金保险：寿命延长意味着保险公司需要支付更长时间的年金，保费需要上调约10%-15%

踩坑故事：2023年有一位客户在银行买了一款"终身养老年金"，银行理财经理宣传说"活得越久领得越多，稳赚不赔"。客户没注意到的大问题是：这款年金的定价假设是基于第三套生命表的，而2025年第四套生命表明示寿命在延长——这意味着保险公司的年金支付义务比预期的更重。虽然已经销售的合同不会受影响（固定给付），但这件事说明了一个精算底层逻辑：长寿也是一种风险——对个人是"活得太久钱不够花"的长寿风险，对保险公司是"活得比预计久导致超预期赔付"的长寿风险。所以你买年金时，不妨问自己一个问题：保险公司有没有低估我的寿命？如果有，那就是你"赚"到了长期稳定的现金流。

好，第六章讲到这里。我们从大数定律开始，了解了保费的构成、财产险费率厘定的四种方法、寿险费率厘定的三差分析，最后结合2025年生命表更新讨论了精算假设对费率的实际影响。精算是一门"用过去预测未来"的科学——所以它永远不完美，但总是工具箱里最好的工具。
