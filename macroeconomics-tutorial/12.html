第12章 总需求-总供给模型

你有没有好奇过一个问题：为什么经济衰退时物价通常会下跌，但有时候也会出现"经济停滞+物价上涨"这种奇怪组合？如果IS-LM模型已经能分析产出和利率了，为什么还需要引入价格水平？

答案在于：IS-LM模型假设价格水平P固定不变——这在短期分析中是一个合理的简化，但一旦我们要分析通胀、通缩、以及经济的长期走势，就必须把P当成一个内生变量来对待。总需求-总供给模型（AD-AS模型）就是IS-LM的自然延伸：它以价格水平为纵轴、总产出为横轴，在一个更广阔的框架中统一分析短期的波动和长期的增长。

1. 总需求AD曲线：从IS-LM到AD

总需求AD（Aggregate Demand）曲线描述的是：在产品市场和货币市场同时均衡的前提下（即IS-LM均衡），总产出与价格水平之间的关系。AD曲线向下倾斜——价格水平越低，均衡产出越高。

为什么AD向下倾斜？主要有三种解释：

财富效应（庇古效应）：价格水平下降 → 你持有的货币和存款的实际购买力上升 → 你感觉自己"变富了" → 增加消费 → 总需求增加。这个效应得名于剑桥经济学家庇古（Arthur Pigou）。

利率效应（凯恩斯效应）：价格水平下降 → 同样的名义货币供给M可以购买更多东西（实际货币供给M/P增加）→ LM曲线右移 → 利率下降 → 投资增加 → 总需求增加。这是从IS-LM推导AD的核心机制。

汇率效应（蒙代尔-弗莱明效应）：价格水平下降 → 本国商品相对于外国商品更便宜 → 出口增加、进口减少 → 净出口增加 → 总需求增加。这个机制在开放经济中格外重要。

从IS-LM推导AD：

推导思路：
IS曲线与价格水平P无关，因为IS中不包含P
LM曲线包含P：M/P = kY - h·r

当P下降时：
  M/P上升（实际货币供给增加）→ LM右移 → 在IS不变的情况下，均衡产出Y增加
  在(P, Y)平面上，P↓ 对应 Y↑ → AD曲线向下倾斜

更一般地说，AD曲线的推导就是：
  对每一个价格水平P，找到对应的IS-LM均衡产出Y
  将这些(P, Y)点连起来就是AD曲线

AD曲线移动的因素（AD右移）：
  · 扩张性财政政策（G↑或T↓）：IS右移 → 每个P对应的Y都增加 → AD右移
  · 扩张性货币政策（M↑）：LM右移 → 每个P对应的Y都增加 → AD右移
  · 消费者信心上升（自主性消费a↑）：IS右移 → AD右移
  · 技术进步预期（自主性投资I₀↑）：IS右移 → AD右移
效应名称 传导机制 主要关联市场
财富效应（庇古效应） P↓ → 实际财富↑ → C↑ → Y↑ 产品市场
利率效应（凯恩斯效应） P↓ → M/P↑ → r↓ → I↑ → Y↑ 货币市场→产品市场
汇率效应 P↓ → 本币贬值 → NX↑ → Y↑ 外汇市场→产品市场

2. 短期总供给SRAS：为什么短期AS向上倾斜

总供给（AS）描绘的是：在给定价格水平下，企业愿意生产的总产出。最令人困惑的问题是：

在长期，AS是垂直的（产出由资本、劳动、技术决定，与价格无关）。但在短期，AS是向上倾斜的——价格水平越高，企业生产的产出越多。为什么？

主要有三种理论解释短期的价格-产出正相关：

工资刚性理论（Sticky-Wage Theory）：名义工资在短期内无法充分调整。如果物价上涨，但工资没有同步上涨（受劳动合同约束），企业的实际劳动成本下降，利润空间扩大，于是增加雇佣、扩大生产。产出的增加是暂时的——等到劳动合同重新谈判、工资上调到位后，产出会回落。

价格黏性理论（Sticky-Price Theory）：不是所有企业都能立即调价。菜单成本（打印新价签、更新系统的费用）导致企业只有在价格变动幅度足够大时才调整。当物价总水平上升时，那些尚未调价的企业相对变得更便宜，吸引了更多需求，因此不得不扩大生产。

错觉理论（Misperceptions Theory）：企业可能混淆"自己产品价格上升"和"整体价格水平上升"。如果企业发现自己产品的价格涨了，但没意识到所有东西都在涨，就会误以为是需求在增加，从而扩大生产（卢卡斯，1972）。

关键洞察：短期AS的斜率取决于价格调整速度
所有三个理论的核心都在于：价格和工资在短期内不能"完美"调整。这种不完美使得名义变量（价格水平）能够影响实际变量（产出和就业）。如果价格是完美灵活的（如古典经济学假设），短期AS就是垂直的，名义变量对实际变量没有影响——这就是著名的"古典二分法"（Classical Dichotomy）和"货币中性"（Money Neutrality）。

3. 长期总供给LRAS：垂直的潜在产出线

与向上倾斜的短期AS不同，长期总供给（LRAS）是一条垂直的直线。LRAS的位置由经济体的潜在产出（Potential Output，也称充分就业产出Y*）决定。潜在产出取决于三个因素：

资本存量K：机器设备、厂房、基础设施的总量。资本越多，生产能力越强。

劳动力L：就业人口的数量和质量（人力资本）。人口增长和教育水平决定了劳动力的供给。

技术A：全要素生产率。同样的资本和劳动投入，技术越好产出越高。这是长期增长的最终引擎。

长期总供给由生产函数决定：Y* = A · F(K, L)

在AD-AS坐标系中：
  横轴：实际产出Y
  纵轴：价格水平P
  LRAS：垂直于Y*的一条直线

短期AS会逐渐"漂移"到LRAS的位置：
  · 如果实际产出Y > Y*（经济过热）→ 工资和价格将上升 → SRAS左移 → 回到Y*
  · 如果实际产出Y < Y*（经济萧条）→ 工资和价格将下降 → SRAS右移 → 回到Y*

这个"漂移"过程可能需要几个季度到几年，取决于工资和价格的调整速度

4. 需求冲击与供给冲击

AD-AS模型最有价值的应用是分析不同类型的经济冲击。经济学家将冲击分为两大类：

需求冲击（Demand Shock）：导致AD曲线移动的事件。例如：消费者信心下降（AD左移）、政府增支（AD右移）、央行降息（AD右移）、出口市场萎缩（AD左移）。

供给冲击（Supply Shock）：导致AS曲线移动的事件。例如：石油价格暴涨（SRAS左移）、自然灾害（SRAS左移）、技术进步（SRAS右移）、劳动力市场改革（LRAS右移）。

两者的区别在政策含义上至关重要。应对需求冲击，政策是有效的——AD左移（衰退）可以通过扩张性政策来抵消。但应对供给冲击就棘手了：

冲击类型 影响 产出 物价 最令人头疼的情形
负向需求冲击 AD左移 ↓（衰退） ↓（通缩压力） 通缩加剧衰退预期，形成恶性循环
正向需求冲击 AD右移 ↑（繁荣） ↑（通胀压力） 经济过热，央行被迫加息
负向供给冲击 SRAS左移 ↓（衰退） ↑（通胀） 滞胀（stagflation）——产出下降+物价上升，政策两难
正向供给冲击 SRAS右移 ↑（繁荣） ↓（通缩） 这是"好"的通缩——技术进步带来的，如2000年代的IT革命
政策两难：当滞胀发生时你怎么办？
假设经济突然面临负向供给冲击（如油价暴涨）。物价上涨、产出下降。央行面临两难：如果为了抑制通胀而加息（AD左移），衰退会加深；如果为了刺激增长而降息（AD右移），通胀会恶化。左手打右手，怎么选都不对。

这正是1970年代各国央行面临的噩梦。当时全球经历了两次石油危机，美联储主席保罗·沃尔克在1979年做出了一个痛苦的决定：优先抗通胀——把联邦基金利率提到了惊人的20%。美国经济在1980-82年经历了严重的衰退（失业率超过10%），但通胀最终被制服了。沃尔克的选择在短期内造成了巨大的痛苦，但从长期来看为1980年代的经济繁荣奠定了基础。

5. 两次石油危机的对照：1973 vs 2026

踩坑故事：1973年OPEC禁运——现代滞胀的教科书案例
1973年10月，第四次中东战争爆发。阿拉伯石油输出国组织（OAPEC）宣布对美国等支持以色列的国家实施石油禁运。国际油价从每桶3美元暴涨到12美元——翻了四倍。

对全球经济来说，这意味着一次巨大的负向供给冲击。SRAS曲线猛烈左移：美国GDP在1974年下降了0.5%，1975年又下降了0.2%；同时通胀率从1972年的3.2%飙升到1974年的11%（CPI峰值）。滞胀（stagflation）这个生造词就是在这个时期诞生的，用来描述"stagnation（停滞）+ inflation（通胀）"的可怕组合。

各国政府最初的反应是凯恩斯式的——增加支出、降低利率来刺激需求。结果呢？通胀更高了，产出并没有多少恢复。直到货币主义者弗里德曼的分析被广泛接受，各国才开始转向"优先控制通胀"的政策框架。这个过程花了近十年，代价极其惨痛。

现在回到2026年。中东地缘政治持续紧张，能源价格在2025-2026年经历了新一轮大幅波动。布伦特原油价格一度突破每桶120美元——名义上接近历史高位。供给冲击的分析框架再次被激活。

但今天的全球经济和1973年至少有四个本质不同：

第一，能源强度下降。1973年，每单位GDP消耗的石油比现在高出约60%。今天的经济"轻"了很多，服务业占比更大、能源效率更高。同样幅度的油价上涨对经济的冲击已经大幅减弱。

第二，央行独立性增强。1973年各国央行受政治压力影响大，不敢大幅加息抗通胀。今天各国央行（尤其是美联储和欧央行）已经建立了"通胀目标制"的纪律，政策可信度更高。

第三，能源结构多元。1973年石油在全球一次能源中占比近50%。今天可再生能源、天然气、核能的占比大幅上升，单一石油冲击的影响力下降。

第四，政策工具箱更丰富。除了传统的财政货币工具，各国还积累了战略石油储备（SPR）、价格上限机制、补贴转型等多种手段来对冲供给冲击。

历史不会简单重复，但韵律相似：1973年的教训告诉我们，供给冲击造成的滞胀远比需求冲击难以应对。2026年我们面临的能源冲击虽然规模可能不小，但全球经济抵御冲击的"免疫力"已经大大增强。不过，供给冲击分析框架在今天的价值依然不变——它提醒我们：当物价和就业同时恶化时，政策制定者必须在短期镇痛（忍受更高的通胀以维持增长）和长期止痛（忍受衰退以消灭通胀）之间做出痛苦的选择。

这一章，我们完成了从IS-LM到AD-AS的升级。你学会了AD曲线的三大向下倾斜效应、SRAS向上倾斜的三种解释、LRAS垂直的深层原因，以及如何区分和应对需求冲击与供给冲击。这是短期宏观经济分析的"终极框架"。下一章，我们将深入一个我们在IS-LM中过于简化的假设——消费函数。你将会发现，凯恩斯的简单消费函数只是冰山一角，弗里德曼、莫迪利安尼和行为经济学家们为我们提供了更丰富的消费理论。
