大家好，我是你们的讲师。前面三章咱们分别看了盈利能力、偿债能力和营运能力——这些都是"静态"的分析。但投资投的是未来——一家公司过去再优秀，如果失去了成长性，它的价值就会大打折扣。反过来，一家当前不怎么赚钱但成长性极佳的公司，反而可能创造惊人的回报。

但成长也是一把双刃剑。A股市场上，投资者对"高增长"有极强的偏好，给高增长公司的估值往往远超合理水平。而一些公司也深谙此道——他们会想尽办法在报表上"制造"增长，哪怕这种增长是不可持续的、甚至是虚假的。

这一章，咱们来系统学习怎么分析成长能力——分辨真实的增长和堆砌的增长，以及增长是否与估值匹配。

营收增长率——有机增长还是并购增长？

营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%，这是最基础的成长指标。但这里有一个关键问题：营收增长是"长出来的"还是"买出来的"？

有机增长是指公司通过自身经营实现的增长——比如市场份额扩大、产品提价、新产品推出。这种增长是可持续的，反映了公司真正的竞争力。

并购增长是指公司通过收购其他公司来实现的营收合并增长。这种增长不一定反映公司核心竞争力的提升——收购来的业务能不能整合好、会不会产生商誉减值，都是很大的未知数。

如何区分？看两个地方：一是年报中的"同一控制下企业合并"和"非同一控制下企业合并"相关数据，看营收增长中有多少来自并购；二是看"商誉"科目的变化——如果商誉大幅增加，说明公司在大量进行并购。

实战技巧：计算"剔除并购后的有机增长率"：有机增长率 = (本期内生营收 - 上期营收) / 上期营收。如果一家公司报告的营收增长率为30%，但其中25%来自并购标的的并表，那么它的有机增长率只有5%。这个"5%"才是真实竞争力的体现。

另外还要关注量的增长 vs 价的增长。如果营收增长主要靠涨价，它是有天花板的；如果靠销量增长（市场份额扩大），持续性更强。年报中的"量价分析"可以帮你判断。

利润增长率——质量比速度更重要

净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%，这是投资者最关注的指标。但利润增长率是最容易被操纵的——通过会计政策变更、资产减值转回、非经常性损益等手段，公司可以让净利润增长率"看起来很好"。

分析利润增长率时，一定要把利润拆解开：

  • 扣非净利润增长率：剔除"非经常性损益"后的净利润增长率，这是核心盈利能力的真实体现。
  • 营业利润增长率：利润表上的"营业利润"增长率，剔除了营业外收支的影响。
  • EBIT增长率：息税前利润的增长率，剔除了资本结构和税率的影响，更适合做横向比较。

一个值得关注的现象：利润增长率持续高于营收增长率。这种情况通常意味着利润率在提升，这本身是好事。但如果利润增长率的优势不是来自毛利率提升或费用率下降，而是来自"会计手段"（比如折旧年限延长、坏账计提比例降低、研发费用资本化），那就需要高度警惕了。

警戒信号：当一家公司的扣非净利润增长率连续3年低于报表净利润增长率时，说明它在持续依赖非经常性损益"美化"利润。这种公司长期来看几乎一定会出现问题。

可持续增长率（SGR）——不要超出自身能力地增长

可持续增长率（Sustainable Growth Rate, SGR）是一个非常重要但经常被投资者忽略的指标。它回答了这样一个问题：在不改变资本结构、不发行新股的前提下，公司仅靠留存收益能支撑多高的增长率？

SGR = ROE × (1 - 分红率)

更精确的公式：SGR = (净利润 / 净资产) × (留存收益 / 净利润) = 留存收益 / 净资产

可持续增长率计算示例：

某公司2023年数据：
净利润 = 10亿
净资产 = 50亿
ROE = 10 / 50 = 20%
分红 = 3亿（分红率30%）
留存收益 = 10 - 3 = 7亿

SGR = ROE × (1 - 分红率) = 20% × 70% = 14%
——或者——
SGR = 留存收益 / 期初净资产 = 7 / 50 = 14%

结论：在不改变资本结构的前提下，
该公司最多能以14%的速度增长。

如果公司实际增长率超过14%：
→ 要么资产负债率会上升（多借钱）
→ 要么需要增发新股（稀释股东权益）
→ 要么增长速度不可持续（必然回调）

SGR是判断增长"真实性"和"可持续性"的有力工具。如果一家公司的实际增长率长期显著高于SGR，说明它的增长是靠提高杠杆来实现的——这是不可持续的。一旦信贷环境收紧，这种高增长就会戛然而止。

增长的估值匹配——PEG比率

高增长不等于好投资。如果一家公司的估值已经充分反映了它的增长预期，那么即使它真的实现了高增长，投资者也可能赚不到钱。这个问题的核心是：增长是否已经体现在价格中？

PEG = 市盈率（PE） / 盈利增长率（g），PEG比率解决了"PE单独看没有高低基准"的问题。一般来说：

  • PEG < 1：可能被低估（增长高于估值反映的预期）
  • PEG = 1：合理估值（增长正好匹配估值）
  • PEG > 1：可能被高估（增长不足以支撑当前估值）

但PEG也有局限性：它假设增长率可以持续，而实际上高增长很难长期维持。一家公司当年的增长率是100%，但大概率第二年会大幅回落。所以我一般用3-5年的平均预期增长率来计算PEG，而不是用单一年份的数据。

增长特征 好信号 警示信号 危险信号
营收增长 有机增长为主，量价齐升 增长主要靠并购 营收增长但DSO大幅上升

综合判断：我用"增长质量评分"系统——营收增速、利润增速、SGR匹配度、现金流匹配度和PEG各占20分，满分100分。低于60分的高增长公司，即使增长数字再好看，也要谨慎对待。因为低质量的增长比没有增长更危险——它往往以暴雷收场。

真实案例：并购堆出来的"增长"

A股上市公司M公司是典型的"并购驱动增长"的故事。2015-2018年，M公司通过连续并购7家标的公司，营收从10亿增长到了60亿，年复合增长率高达56%。利润也从1亿增长到了6亿，看起来是一支标准的"成长股"，市场给出了50倍PE的估值。

但仔细一看问题就暴露了：M公司的有机增长率（剔除并购并表影响）不到8%——这意味着它的核心业务几乎没增长。商誉余额从0飙升到了35亿，相当于净资产的80%。并且被并购的7家公司中有4家的业绩承诺期刚结束，业绩立即"变脸"——第一年还能勉强完成承诺，第二年就大幅下滑。

2019年，M公司发布了商誉减值公告，一次性计提商誉减值18亿，全年巨亏12亿。股价从45元跌到了6元，市值蒸发了87%。那些买入"M公司增长故事"的投资者，实际买入的是一堆难以整合的收购标的和高悬的商誉巨雷。

核心教训：区分有机增长和并购增长是成长能力分析的第一课。一家公司如果靠并购堆起来的"增长"超过50%，而有机增长不到10%，那它本质上不是一个成长型公司，而是一个"收购整合型"公司。这两种模式的估值逻辑完全不同。
