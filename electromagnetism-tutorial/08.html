开场：奥斯特的偶然与必然

1820年的一个春日，哥本哈根大学的物理教授汉斯·克里斯蒂安·奥斯特（Hans Christian Orsted）正在给学生们做一场关于电流热效应的演示实验。当一根通有电流的导线靠近桌上的小磁针时，一个细微但确凿的偏转发生了——磁针动了一下。

整个欧洲物理学界沸腾了。两千年来，电和磁被认为是两种截然不同的现象。吉尔伯特（1600年）在《论磁》中断言电和磁没有关系，牛顿和库仑也从未想过两者之间可能存在联系。但奥斯特的实验第一次证明：电流可以产生磁场，电与磁是统一的。

消息传到巴黎的法国科学院，安培在一周内就重复并扩展了奥斯特的实验，提出了电流之间相互作用力的定量规律。安培甚至大胆地断言：所有磁现象本质上都是电流（运动的电荷）产生的。这个洞见至今仍是磁学的基石。

🔑 核心洞察
奥斯特实验中的磁针偏转其实非常微弱——助手甚至认为那是实验误差。但奥斯特坚持反复验证。科学发现需要敏锐的观察力，更需要不轻易否定"异常数据"的坚持。正如巴斯德所言："在观察领域中，机遇只偏爱有准备的头脑。"

磁感应强度 B

正如我们用电场强度 E来描述电场的强弱和方向，我们用磁感应强度 B来描述磁场的强弱和方向。但两者的引入方式有很大不同。

电场强度是通过点电荷受力定义的：F = qE。而磁场对运动电荷的力更复杂——它不仅依赖于电荷量和位置，还依赖于电荷的速度。这就是洛伦兹力的磁部分：

F = q(E + v × B)

当电场为零时，F = qv × B。从这个定义可以提取出 B 的正式定义：磁感应强度 B 是这样一个矢量场，使得运动电荷所受的磁力为 qv × B。

物理量 符号 单位 定义方式
电场强度 E N/C 或 V/m F = qE
磁感应强度 B T（特斯拉） F = qv × B

磁感应强度的单位是特斯拉（Tesla, T）：1 T = 1 N·s/(C·m) = 1 kg/(s²·A)。还有一些标志性的磁场数值供参考：

  • 地球表面磁场 ≈ 50 μT（微特斯拉）
  • 一块普通冰箱贴 ≈ 5 mT
  • 电磁铁（实验室）≈ 1-2 T
  • 超导磁铁（MRI）≈ 3 T（临床）到 7 T（科研）
  • 最强的脉冲磁场 ≈ 100 T（以毫秒计，否则设备会炸毁）
💡 右手定则速记
判断磁场方向最常用的方法是右手定则：
直导线：右手握住导线，拇指指向电流方向，四指弯曲的方向即磁场环绕方向。
螺线管：右手握住螺线管，四指弯曲指向电流环绕方向，拇指指向内部磁场方向。
这是电磁学中最基础的直觉工具——如果你觉得叉积运算太绕，先用右手定则感受方向，再用数学精确计算。

毕奥-萨伐尔定律

毕奥-萨伐尔定律（Biot-Savart Law）是磁学的"库仑定律"。它给出了任意电流分布产生磁场的通用计算方法。

考虑一段长度为 dl、载有电流 I 的导线微元。该微元在场点 r 处产生的磁感应强度微元为：

dB = (μ₀ / 4π) · (I dl × r̂) / r²

其中：

  • μ₀ = 4π × 10⁻⁷ T·m/A —— 真空磁导率（permeability of free space）
  • dl —— 电流微元的方向（即电流方向）
  • r̂ —— 从电流微元指向场点的单位矢量
  • r —— 电流微元到场点的距离

完整的磁场通过对所有电流微元积分得到：

B(r) = (μ₀ / 4π) ∫ (I dl' × (r − r')) / |r − r'|³

下面是毕奥-萨伐尔定律与库仑定律的对比表——对称性之美令人惊叹：

特性 库仑定律（电场） 毕奥-萨伐尔定律（磁场）
源 电荷 dq 电流微元 I dl
表达式 dE = (1/4πε₀)(dq r̂)/r² dB = (μ₀/4π)(I dl × r̂)/r²
方向 沿 r̂（径向） 垂直于 dl 和 r̂（叉积方向）
基本常数 ε₀ = 8.854×10⁻¹² F/m μ₀ = 4π×10⁻⁷ H/m
作用类型 中心力（沿连线方向） 非中心力（垂直于运动方向）

经典电流分布的磁场

无限长直导线

这是最简单的应用。设无限长直导线沿 z 轴放置，电流方向沿 +z，距离导线垂直距离 r 处的磁感应强度为：

B = (μ₀ I) / (2π r) · φ̂

磁场方向环绕导线（右手定则），大小随距离增大而减小。注意这与点电荷电场随 1/r² 衰减不同——无限长线源产生的场是 1/r 衰减。

圆环电流轴线上

半径为 R 的圆环载有电流 I，在轴线上距圆心 z 处的磁场为：

B(z) = (μ₀ I R²) / (2 (R² + z²)^(3/2)) · ẑ

在圆心处（z=0）：B = μ₀ I / (2R)。在远处（|z| >> R）：B ≈ μ₀ I R² / (2|z|³) —— 衰减很快。

无限长螺线管

理想情况下（无限长、紧密绕制），螺线管内部磁场均匀，外部为零：

B内部 = μ₀ n I

其中 n 是单位长度的匝数。这是一条非常重要的公式——它表明只要螺线管足够长，内部磁场与轴向位置无关。

洛伦兹力与带电粒子运动

我们已经知道运动电荷在磁场中受力 F = qv × B。一个关键的物理特征：磁场力总是垂直于速度，因此不做功——磁场可以改变带电粒子的运动方向，但不能改变其速率（动能不变）。

回旋运动

当带电粒子以速度 v 垂直于均匀磁场 B 进入时，粒子做匀速圆周运动。洛伦兹力提供向心力：

q v B = m v² / r

解得回旋半径（陀螺半径）：r = m v / (q B)

回旋角频率（cyclotron frequency）：ωc = q B / m

注意：回旋频率只依赖于荷质比和磁场强度，与粒子速度无关。这一性质在回旋加速器（cyclotron）中得到了巧妙利用。

霍尔效应

当载流导体置于垂直于电流方向的磁场中时，洛伦兹力使运动电荷向导体侧面偏转，从而在两侧积累电荷，形成横向电场——这就是霍尔效应（Hall Effect）。

平衡时，洛伦兹力与电场力相等：q EH = q vd B → EH = vd B

霍尔电压：VH = EH · d = vd B d = (I B) / (n q d)

霍尔系数：RH = EH / (J B) = 1/(n q)。通过测量霍尔系数可以判断载流子的类型（正/负）和浓度——这是半导体器件表征的基本手段。

💡 应用：霍尔传感器
基于霍尔效应的磁场传感器在汽车电子中无处不在——曲轴位置传感器、车速传感器、电子节气门、无刷直流电机的转子位置检测。霍尔传感器的最大优点是非接触测量，寿命长且不受灰尘污染影响。

Python 数值计算：任意电流路径的磁场

毕奥-萨伐尔定律的强大之处在于它可以处理任意形状的载流导线。对于复杂路径，解析解通常不存在，但数值积分轻而易举：

import numpy as np
import matplotlib.pyplot as plt

mu0 = 4*np.pi * 1e-7   # 真空磁导率 (T·m/A)

def magnetic_field(current, path, point):
    """
    用毕奥-萨伐尔定律计算任意电流路径在给定点产生的磁场。

    参数:
        current: 电流大小 (A)
        path: 路径点阵列, shape (N, 3) — 依次连接这些点
        point: 场点坐标, shape (3,)

    返回:
        B: 磁感应强度矢量, shape (3,)
    """
    B = np.zeros(3)
    N = len(path)

    for i in range(N - 1):
        dl = path[i+1] - path[i]        # 电流微元矢量
        r_mid = 0.5 * (path[i] + path[i+1])  # 微元中点
        R = point - r_mid                # 从微元到场点的矢量
        r = np.linalg.norm(R)
        if r == 0:
            continue
        R_hat = R / r
        # 毕奥-萨伐尔定律
        dB = (mu0 * current / (4*np.pi)) * np.cross(dl, R_hat) / r**2
        B += dB

    return B

# 示例：正方形线圈（边长0.1m）在中心处的磁场
L = 0.1
sq = np.array([[-L/2, -L/2, 0],
               [ L/2, -L/2, 0],
               [ L/2,  L/2, 0],
               [-L/2,  L/2, 0],
               [-L/2, -L/2, 0]])  # 闭合路径

B_center = magnetic_field(1.0, sq, np.array([0.0, 0.0, 0.0]))
print(f"B at center = {B_center} T")
print(f"|B| = {np.linalg.norm(B_center):.2e} T")

这段代码展示了毕奥-萨伐尔定律数值积分的核心思路：将连续路径离散化为小段，对每段计算 dB，然后求和。精度取决于离散化的细密程度。

🔑 核心洞察
将毕奥-萨伐尔定律与库仑定律对比，你会发现一个深刻的对称性：
静电场：电荷产生径向电场（源向），散度非零，旋度为零。
静磁场：电流产生环向磁场（旋涡向），散度为零，旋度非零。
电荷是电场的"源"，电流是磁场的"涡旋源"。这种互补关系在麦克斯韦方程组中以数学的形式得到完美统一——我们将在第13章见证这一巅峰。

本章小结

  • 磁场由运动电荷（电流）产生；磁感应强度 B 由洛伦兹力定义：F = qv × B。
  • 毕奥-萨伐尔定律是计算任意电流分布磁场的通用工具，相当于磁学中的库仑定律。
  • 经典构型的磁场公式（无限长直导线 B ∝ 1/r，螺线管内部均匀 B = μ₀ n I）是高频考点。
  • 洛伦兹力垂直于速度、不做功，带电粒子在均匀磁场中做回旋运动——回旋频率 ωc = qB/m。
  • 霍尔效应利用洛伦兹力测量磁场和载流子特性，是半导体器件的重要表征手段。
