第6章 机器人机构学基础

大家好，我是你们的讲师。上一章我们学会了如何用数学工具描述一个刚体在空间中的位姿，以及如何在坐标系之间做变换。现在，我们把目光从"单个刚体"转向"多个刚体的组合"——机器人本身就是由多个连杆通过关节连接而成的机构系统。这一章的主题是机构学（Mechanism），它研究的是：这些连杆和关节如何连接、能产生哪些运动、有哪些限制，以及在什么情况下会失效。

我见过很多做机器人算法的同学，一头扎进控制代码和深度学习模型里，对机械结构完全不关心。结果一到真机部署就出问题——算法在仿真里跑得好好的，到了真机上机械臂的运动范围超出了关节限位、末端精度远低于预期、甚至在某个位形下机器人"锁死"了（奇异位形）。这些问题的根源都在机构学。相信我，理解你的机器人的机构，和理解你的控制算法一样重要。

6.1 自由度与约束

自由度（Degree of Freedom, DOF）是机构学中最基本的概念。一个物体在三维空间中自由运动时有6个自由度（3个平移 + 3个旋转）。当它通过关节与其他物体连接时，关节施加约束，自由度减少。

Grübler-Kutzbach公式给出了计算机构自由度的方法：

# 空间机构自由度公式
M = 6*(N - 1 - J) + Σ_{i=1}^J f_i

# 其中：
# M  = 机构自由度
# N  = 连杆数量（包括基座）
# J  = 关节数量
# f_i = 第i个关节的自由度

# 平面机构（纯平面运动）的简化版本：
M = 3*(N - 1 - J) + Σ_{i=1}^J f_i

# 举例：标准六轴工业机械臂
# N = 7（基座 + 6个连杆）, J = 6（6个旋转关节）
# 每个旋转关节 f_i = 1
# M = 6*(7 - 1 - 6) + 6*1 = 6  ✓ 六自由度

这里有一个容易混淆的点：过约束（Over-constrained）和欠约束（Under-constrained）。过约束意味着存在冗余约束（即去掉某个约束不影响自由度），这在精密并联机构中很常见——但需要非常高的制造精度才能正常工作。欠约束意味着机构在某些方向上没有约束力——比如一个没有锁死机构的气球，它在空间中的位置是不可控的。

让我分享一个关于自由度的惨痛教训。我在大学期间参加RoboMaster比赛，我们设计了一个云台系统，由两个电机驱动：一个控制pitch（俯仰），一个控制yaw（偏航），没有roll（横滚）自由度——因为我们觉得"只需要两轴就够了"。结果在赛场上，当机器人快速转弯时，惯性导致相机画面严重倾斜（roll方向的扰动），而我们的云台无法补偿这个自由度——打出去的弹丸全部偏了。如果我们当时应用Grübler公式认真分析一下系统所需的自由度，就不会犯这个错误。后来我们紧急改装，在云台底座加了一个roll轴电机，才勉强逆转了局势。这个教训让我一直铭记：DOF不是越多越好，而是"刚好够用"最好。

关键认知：自由度计算的本质是"你能独立控制的运动数量"。六轴机械臂的6个自由度刚好能控制末端在三维空间中的完整位姿（3位置+3姿态）。如果任务只需要位置（比如点胶），5个自由度的SCARA就足够了——用6轴反而是浪费。反过来，如果任务需要避开障碍物同时保持末端姿态，你需要冗余自由度（7轴）。选择多少自由度，取决于任务的最小需求。

6.2 关节类型与连杆

机器人的运动能力最终由关节的类型和连杆的几何参数决定。常见的关节类型有：

  • 旋转关节（Revolute Joint, R）：绕一个轴旋转，1个自由度。工业机械臂中最常见的关节类型，类似人类的大臂、小臂关节。
  • 移动关节（Prismatic Joint, P）：沿一个轴直线运动，1个自由度。常见于SCARA机器人和一些直角坐标系机器人（Gantry）。
  • 球关节（Spherical Joint, S）：绕三个轴旋转，3个自由度。类似人类的肩关节。在机器人中通常用三个旋转关节串联来实现。
  • 圆柱关节（Cylindrical Joint, C）：绕一个轴旋转 + 沿该轴平移，2个自由度。
  • 万向节（Universal Joint, U）：绕两个正交轴旋转，2个自由度。

连杆（Link）是连接两个关节的刚体部分。一个连杆的参数包括长度、质量、质心位置、惯性张量等。在运动学中，我们关心的是连杆的"几何"——相邻两个关节轴线的空间关系（用DH参数描述）。在动力学中，我们还需要连杆的质量和惯性参数。

连杆和关节构成运动链（Kinematic Chain）。运动链有两种基本形式：开链（Open Chain）和闭链（Closed Chain）。绝大多数工业机械臂是开链——从基座到末端，关节串联，没有闭环。而并联机构（如Stewart平台）是多条运动链并行连接，形成闭环。

6.3 串联与并联机构

串联机构（Serial Manipulator）和并联机构（Parallel Manipulator）代表了机器人机构设计的两个极端，各有显著的优缺点。

串联机构（如经典的六轴机械臂）的特点是：关节串联，工作空间大，灵活性好，但精度较低（误差从基座到末端逐级累积），刚度较差（悬臂结构在末端负载下会产生较大变形）。典型应用包括弧焊、搬运、喷涂等。

并联机构（如Stewart平台、Delta机器人）的特点是：多条运动链平行连接，刚度极高，精度很高（误差被平均化），但工作空间小，运动范围受限。Stewart平台（6个线性驱动器并联）广泛用于飞行模拟器和高精度定位台。Delta机器人（三臂并联）在食品分拣、电子装配等高速轻载场景表现出色。

下表对比了两种机构的特性：

特性 串联机构 并联机构
工作空间 大（可达臂展的球状空间） 小（约为机构尺寸的一部分）
刚度 低（悬臂结构，末端刚度最差） 高（多链分担负载，整体刚度高）
精度 中等（误差逐级累积） 高（误差被多条链平均化）
负载/自重比 低（臂体自重大，有效负载小） 高（驱动器可固定于基座）
惯性 大（大臂带动小臂，惯量大） 小（驱动器近基座，动部件轻）
运动学解算 正向容易，逆向复杂 正向复杂，逆向相对容易
代表机型 FANUC CRX、KUKA KR Stewart平台、ABB FlexPicker

知识点：近年来，混合机构（Hybrid Mechanism）开始流行——将串联和并联混合使用。例如，协作机器人KUKA iiwa使用串联结构但每个关节内置扭矩传感器，实现了"串联的灵活性 + 力控的精度"。而一些新一代人形机器人的腿部采用了"串并混联"——髋关节用并联机构提供高刚度，膝关节和踝关节用串联机构提供灵活性。这种混合设计正在成为行业趋势。

6.4 工作空间与灵巧度

工作空间（Workspace）是指机器人末端可到达的所有位置集合。它分为两种：

  • 可达工作空间（Reachable Workspace）——末端可以到达的所有位置点的集合。不考虑姿态，只要能达到就算。
  • 灵巧工作空间（Dexterous Workspace）——末端可以在任意姿态下到达的所有位置点集合。这是机器人真正"好用的"空间。

灵巧工作空间通常远小于可达工作空间。举个例子，六轴机械臂在靠近其基座的位置，虽然末端能到达（可达），但可实现的姿态范围非常有限（不灵巧）。而在工作空间中心区域，末端可以在很大范围的姿态下到达同一点——这就是灵巧空间。

可操作性（Manipulability）是衡量机器人在特定位形下运动能力的量化指标。Yoshikawa定义了可操作性指数：\(w = \sqrt{\det(\mathbf{J}(\mathbf{q}) \mathbf{J}(\mathbf{q})^T)}\)，其中 \(\mathbf{J}\) 是Jacobian矩阵。这个几何意义是Jacobian矩阵的奇异值之积——可以理解为"末端速度空间椭球的体积"。体积越大，机器人在该位形下的运动能力越强，越接近"灵巧"状态。当 \(w = 0\) 时，机器人处于奇异位形。

可操作性椭球（Manipulability Ellipsoid）是一个更直观的工具。将Jacobian的SVD得到的奇异值 \(\sigma_1, \sigma_2, \ldots, \sigma_6\) 作为椭球的轴长，轴的方向由对应的左奇异向量给出。这个椭球告诉你：在当前的关节位形下，末端在哪个方向上运动最容易（长轴方向），哪个方向上最困难（短轴方向）。在轨迹规划时聪明的做法是尽量让椭球的长轴指向所需运动方向——这样可以最小化关节速度和力矩。

6.5 奇异位形

奇异位形（Singularity）是机器人机构学中最重要也最头疼的概念。当机器人处于奇异位形时，Jacobian矩阵 \(\mathbf{J}\) 的秩降低，即某些方向上末端失去运动能力（或反转：在某些方向上，微小末端运动需要极大的关节速度）。

奇异位形分为两大类：

边界奇异位形（Boundary Singularity）发生在机器人完全伸展或完全收缩时。此时末端位于工作空间的边界上，只能单向运动（朝工作空间内部），无法朝边界外运动。这在物理上很直观——你的手臂完全伸直时，你无法再"拉直"一点。

内部奇异位形（Interior Singularity）发生在工作空间内部，通常是由于两个或多个关节的轴线共线。一个经典例子是PUMA 560机械臂在"肘部伸直"（upper arm和forearm成一直线）时的奇异——此时肩关节和肘关节的旋转轴重合，末端失去一个方向上的运动能力。

我用一个个人经历来说明奇异位形的危害。在一次汽车零部件焊接项目中，焊接机械臂在某条焊缝的中点附近突然剧烈抖动，焊点全部偏位，导致一批零件报废。排查发现，机器人示教时设置的路径恰好经过了一个内部奇异点。当机器人接近奇异点时，Jacobian的条件数急剧增大，微小焊接误差在逆运动学求解时被放大，导致关节速度指令剧烈振荡。解决方案很简单——在奇异点附近额外添加一个途经点，让路径平滑地绕过奇异区域。从此之后，我在任何轨迹规划任务中都会先做奇异点检测：计算路径上每个点的Jacobian条件数，如果超过阈值就自动插入路径修正。

避坑指南：处理奇异位形有三种主流策略。(1) 回避策略——在轨迹规划阶段就绕开奇异区域，优点是安全可靠，缺点是可能限制工作空间。(2) 阻尼最小二乘（DLS）——在逆运动学求解时加入阻尼项 \(\mathbf{J}^T(\mathbf{J}\mathbf{J}^T + \lambda^2\mathbf{I})^{-1}\)，在奇异点附近牺牲精度换取稳定性，这是实践中用的最多的方法。(3) 奇异分离——将运动分解为"奇异方向"和"非奇异方向"，只在非奇异方向上求解，奇异方向上的分量由冗余关节或机构吸收。第10章我们会详细讨论这些方法。
