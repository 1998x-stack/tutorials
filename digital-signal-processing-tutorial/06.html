第6章 频域分析

你有没有想过一个问题：为什么我们听到的音乐是"哆来咪"而不是一片混沌？为什么Wi-Fi能在同一根天线上同时传输几十个手机的数据？答案都藏在频率这个维度里。

时域信号告诉我们"某个时刻的幅度是多少"，就像电影的单帧画面。频域分析则告诉我们"信号里有哪些频率成分、各自多强"——这相当于看整个电影的"色彩频谱统计"。两者是同一枚硬币的两面，但频域揭示的信息在时域里往往完全看不出来。这一章，我们就系统整理傅里叶变换家族的谱系，并重点学习离散时间傅里叶变换（DTFT）。

6.1 四大傅里叶变换：一张图理清家族关系

在网上经常有人问："FT、FS、DTFT、DFT到底什么关系？" 如果你也有这个困惑，下面这张表一次性给你讲清楚：

变换名称 缩写 时域 频域 适用范围
傅里叶级数 FS 连续、周期 离散、非周期 周期信号的谐波分解
傅里叶变换 FT 连续、非周期 连续、非周期 模拟信号的频谱分析
离散时间傅里叶级数 DTFS 离散、周期 离散、周期 周期序列的频谱
离散时间傅里叶变换 DTFT 离散、非周期 连续、周期 非周期序列的频谱
离散傅里叶变换 DFT 离散、有限长 离散、有限长 计算机可计算的频谱
💡 记忆方法：一个域里周期/离散，另一个域里就离散/周期。如果一个信号在时域是周期的，它的频域就是离散的（只有基波和谐波）；如果一个信号在时域是离散的，它的频域就是周期的。时域离散 → 频域周期（2π周期）——这一点对DTFT和DFT至关重要。

在上表中，DTFT是我们这一章的主角，DFT是下一章的主角。两者的关系是：DFT = DTFT在频域上的采样。你可以理解成DTFT是一条连续曲线，DFT就是这条曲线上均匀取的一些点。

6.2 离散时间傅里叶变换（DTFT）

DTFT的定义如下：

        正变换： X(e^{jω}) = Σ_{n=-∞}^{∞} x[n] · e^{-jωn}

        逆变换： x[n] = (1/2π) ∫_{-π}^{π} X(e^{jω}) · e^{jωn} dω

这里有几个关键点：

  • X(e^{jω}) 是ω的连续函数：ω取-π到π之间的任意实数值。对于有限长序列，DTFT是一条光滑曲线。
  • X(e^{jω}) 是以2π为周期的周期函数：这是"时域离散导致频域周期"的必然结果。验证：X(e^{j(ω+2π)}) = Σ x[n]·e^{-j(ω+2π)n} = Σ x[n]·e^{-jωn}·e^{-j2πn} = X(e^{jω})，因为e^{-j2πn}=1。
  • 存在条件：序列必须绝对可和，即 Σ |x[n]| < ∞。对于能量信号（有限能量），这个条件自然满足。对于周期信号（如sin(ω₀n)），DTFT在常规意义下不存在，需要引入冲激函数（Dirac delta）来描述。
🔧 直观理解：DTFT的公式可以理解为——把信号x[n]与各频率的复指数 e^{jωn} 做"相似度匹配"。X(e^{jω})的值越大，说明信号在频率ω处的分量越强。这和你在Spotify上看到的频谱可视化器是完全一样的原理。

6.3 频率响应：系统在频域里的"指纹"

从上一章我们知道，LTI系统的冲激响应h[n]包含了系统的全部时域信息。在频域中，对应的概念是频率响应 H(e^{jω})：

回忆Z变换的系统函数：H(z)。令 z = e^{jω}，就得到频率响应：

        H(e^{jω}) = H(z) |_{z=e^{jω}}

        它还可以表示为：
        H(e^{jω}) = |H(e^{jω})| · e^{j·∠H(e^{jω})}

        - 幅频响应 |H(e^{jω})|：各频率分量的增益
        - 相频响应 ∠H(e^{jω})：各频率分量的相位偏移
        - 群延迟 τ_g(ω) = -d(∠H(e^{jω}))/dω：相位对频率的变化率

群延迟是一个稍微进阶但非常重要的概念。它度量的是不同频率分量通过系统后的"延迟差"。如果群延迟不是常数，信号的不同频率成分会"到达"输出端的时间不同——这会造成相位失真（比如方波通过系统后上升沿变"斜"）。

系统类型 幅频特性 相频特性 典型应用
低通滤波器 低频≈1，高频≈0 通常内尽量线性 去噪、抗混叠
高通滤波器 低频≈0，高频≈1 通常内尽量线性 直流去除
带通滤波器 某一频段≈1，其余≈0 通常内尽量线性 信道选择
全通滤波器 全频段|H|=1 非线性（用于相位补偿） 相位均衡、延迟补偿

6.4 DTFT的性质

DTFT和Z变换有很多相似的性质，主要是因为DTFT就是Z变换在单位圆上的特例：

性质 时域 DTFT
周期性 - X(e^{j(ω+2π)}) = X(e^{jω})
线性 a·x₁[n] + b·x₂[n] a·X₁(e^{jω}) + b·X₂(e^{jω})
时移 x[n-k] e^{-jωk}·X(e^{jω})
频移 e^{jω₀n}·x[n] X(e^{j(ω-ω₀)})
卷积 x[n] * h[n] X(e^{jω})·H(e^{jω})
调制 x[n]·w[n] (1/2π) X(e^{jω}) ⨂ W(e^{jω})
Parseval定理 Σ |x[n]|² (1/2π) ∫ |X(e^{jω})|² dω

对称性是一个容易忽视但非常实用的性质。对于实序列x[n]，其DTFT满足：

        共轭对称性：X(e^{-jω}) = X*(e^{jω})

        这意味着：
        - 幅度谱是偶函数：|X(e^{jω})| = |X(e^{-jω})|
        - 相位谱是奇函数：∠X(e^{jω}) = -∠X(e^{-jω})

        所以实序列的频谱只需要看0~π（正频率部分）就够了，
        另一半是冗余的。这就是为什么频谱图通常只画一半。

Parseval定理说的是信号在时域的能量等于在频域的能量。这个性质在工程中非常实用——比如你想知道某个频段内的能量占比，只需要对|X(e^{jω})|²在该频段积分即可。

6.5 典型序列的DTFT

看看几个常见序列的DTFT长什么样。这些结果在后续章节中会反复用到：

        ① 矩形脉冲： x[n] = 1, 0 ≤ n ≤ L-1

           X(e^{jω}) = e^{-jω(L-1)/2} · sin(ωL/2) / sin(ω/2)

           这个函数叫做 Dirichlet 核（也叫周期 sinc 函数）。
           它的主瓣宽度 = 4π/L，旁瓣峰值 ≈ -13dB。

        ② 指数衰减序列： x[n] = aⁿ·u[n], |a| < 1

           X(e^{jω}) = 1 / (1 - ae^{-jω})

        ③ 单位冲激： x[n] = δ[n]

           X(e^{jω}) = 1  （平坦的白色频谱——所有频率能量相同）
⚠️ 踩坑实录：一个做车载音频的朋友遇到了一个奇怪的"嗡嗡"声问题。他们在播放测试音（1kHz正弦波）时，偶尔会听到一个微弱的2kHz泛音。起初以为是扬声器失真，换了扬声器问题依旧。最后检查发现是DSP内部的一个固定点运算——在做正弦波生成时，查询表（LUT）的索引计算有截断误差，导致输出的正弦波在零点附近产生了非线性失真，引入了2次谐波。频谱上看，除了1kHz的主峰外，2kHz处有一个小峰（大约-40dB）。这个问题的教训是：频域分析不仅是用来"观察"信号的工具，它更是诊断系统缺陷的听诊器。学会看频谱图，等于拥有了"信号X光"的能力。

6.6 本章小结

  • 梳理了四大傅里叶变换的家族谱系：FS → FT → DTFS → DTFT → DFT，掌握了"一个域周期则另一域离散"的记忆规律；
  • 理解了DTFT的定义和存在条件（绝对可和），注意它的频域是连续且2π周期的；
  • 学会了频率响应H(e^{jω})的概念——它由幅频响应、相频响应和群延迟构成；
  • 掌握了DTFT的主要性质，特别是共轭对称性（实序列频谱只需看一半）和Parseval定理（时频能量守恒）；
  • 分析了典型序列的DTFT，包括矩形脉冲的Dirichlet核和指数衰减序列的有理函数形式。

DTFT给了我们连续且完整的频谱图像，但它有一个"致命"问题：X(e^{jω})是ω的连续函数，计算机无法直接处理连续变量。我们需要一个可计算的版本——离散傅里叶变换（DFT）。下一章我们会看到，DFT本质上就是对DTFT在频域进行采样，让频谱变得"离散"且"有限"，从而可以在计算机上精确计算。
