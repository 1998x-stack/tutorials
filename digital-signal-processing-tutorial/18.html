第18章 时频分析

让我们停下来想一想。到目前为止，我们学习了两种看待信号的方式：

  • 时域：看信号幅度随时间的变化，直观但信息有限；
  • 频域（FFT / AR谱 / MUSIC）：看信号包含哪些频率成分，但时间信息完全丢失。

FFT把时间轴"压扁"了——它假设信号在整个观测窗口内是平稳的（即频率成分不随时间变化）。但现实世界中绝大多数信号是非平稳的：语音信号中每个音节的频谱在快速变化、音乐中不同乐器的音符此起彼伏、雷达回波的多普勒频率随时间改变、心电信号（ECG）的波形随心跳周期变化。

让我给你讲个故事。2017年我帮一个团队调试工业设备的故障诊断系统。设备旋转时会产生振动信号，传感器采集到的数据看起来杂乱无章。用FFT一看，频谱上好几个峰，但完全对应不上已知的机械故障频率。后来我们把信号做了时频分析——结果一目了然：某个频率分量只在设备启动的头两秒出现，然后消失了。那是轴承的"磨合期"特征。如果只看FFT，这个瞬态特征被整个时间窗口平均掉了，完全看不出。维修团队替换轴承后，时频谱上那个瞬态分量果然消失了。

💡 核心认知：时频分析的终极目标就是回答一个问题——"在某个时刻 \(t\)，信号的频率成分是什么？"它把一维的时间信号映射到二维的时频平面上，让你同时看到时间信息和频率信息。

18.1 短时傅里叶变换（STFT）：加窗的智慧

时频分析最直观的思路是：既然FFT要求信号平稳，那我们就把信号切成一帧一帧的小段，假设每一帧内部是平稳的，对每一帧做FFT。这就是短时傅里叶变换（Short-Time Fourier Transform, STFT）。

数学上，STFT定义为：

\(X(\tau, f) = \int_{-\infty}^{\infty} x(t) w(t - \tau) e^{-j2\pi f t} dt\)

其中 \(w(t)\) 是窗函数，\(\tau\) 是窗的位置。在离散域中：

\(X[m, k] = \sum_{n=0}^{L-1} x[n + mH] w[n] e^{-j2\pi kn/L}\)

其中 \(L\) 是帧长（FFT点数），\(H\) 是帧移（hop size，相邻帧之间的采样点间隔）。

STFT面临的根本矛盾是——时间分辨率与频率分辨率不可兼得。这就是海森堡不确定性原理在时频分析中的体现：

\(\Delta t \cdot \Delta f \geq \frac{1}{4\pi}\)

  • 长窗（\(L\) 大）：频率分辨率好（FFT的 \(\Delta f = f_s/L\) 小），但时间分辨率差（不知道频率变化发生的精确时刻）；
  • 短窗（\(L\) 小）：时间分辨率好，能精确定位频率变化的时刻，但频率分辨率差（FFT的bin很宽）。
🔧 工程实践：STFT的参数选择取决于应用场景。对于语音信号处理，常用的参数是：帧长 \(L = 20-30\text{ms}\)（对应采样率 \(f_s=16\text{kHz}\) 时 \(L=320-480\)），帧移 \(H = L/2\) 或 \(L/4\)。对于音乐分析，可能需要更长的帧（40-60ms）以分辨低频音符。窗函数通常选用Hamming或Hann窗以降低频谱泄漏。这个参数组合不是随便选的——它经过了数十年的工程验证。

18.2 语谱图：看得见的声音

STFT的幅度平方 \(|X[m, k]|^2\) 在时频平面上以图像形式展示出来，就是语谱图（Spectrogram）。横轴是时间，纵轴是频率，颜色深浅表示能量强度。

如果你从来没有看过语谱图，我强烈建议你找一段自己的录音，用Python做一张看看。你会惊讶地发现：语音中每个元音对应着几条水平的亮线（共振峰，formants——声道的谐振频率），而辅音则表现为竖直的条纹或散沙状的宽带噪声。

// Python代码：生成语谱图
import numpy as np
import matplotlib.pyplot as plt
from scipy import signal

# 参数设置
fs = 16000          # 采样率 16kHz
framelen = 0.025    # 帧长 25ms
hop = 0.010         # 帧移 10ms

# 读取音频数据 (假设 x 是一维numpy数组)
nperseg = int(framelen * fs)
noverlap = nperseg - int(hop * fs)

# 计算STFT
f, t, Zxx = signal.stft(x, fs, window='hann',
                        nperseg=nperseg, noverlap=noverlap)

# 计算语谱图（dB scale）
spectrogram = 20 * np.log10(np.abs(Zxx) + 1e-10)

# 显示
plt.pcolormesh(t, f, spectrogram, shading='gouraud')
plt.ylabel('频率 (Hz)')
plt.xlabel('时间 (s)')
plt.title('语谱图 Spectrogram')
plt.colorbar(label='强度 (dB)')
plt.show()

语谱图在语音识别、音乐信息检索、动物声学分析、地震信号处理中有广泛的应用。比如，Whisper（OpenAI的语音识别模型）的输入特征就是基于语谱图的Log-Mel谱——本质上是对语谱图做了梅尔刻度映射和取对数操作。

18.3 小波变换：让窗口自己"长大"和"缩小"

STFT有一个"先天缺陷"：窗长 \(L\) 是固定的。这意味着无论分析高频还是低频成分，使用的时窗宽度都一样。但物理信号的时频特性往往希望——低频成分需要宽的时间窗以获得好的频率分辨率，高频成分需要窄的时间窗以精确定位时间。

小波变换（Wavelet Transform）正是为了解决这个问题而生的。它用一组伸缩和平移的小波基函数来替代STFT的固定窗正弦基函数。

连续小波变换（CWT）

CWT的定义为：

\(W(a, b) = \frac{1}{\sqrt{a}} \int_{-\infty}^{\infty} x(t) \psi^*\left(\frac{t-b}{a}\right) dt\)

其中 \(\psi(t)\) 是母小波（mother wavelet），\(a\) 是尺度参数（控制伸缩，\(a\) 越大对应低频），\(b\) 是平移参数（控制时间位置）。

CWT在参数 \((a, b)\) 平面上连续变化——理论上给出的是过完备的表示（计算量巨大）。在实际应用中，我们通常使用离散小波变换（DWT），它把尺度和平移参数离散化为2的幂次，大大降低了计算量。

离散小波变换与Mallat算法

离散小波变换的工程实现基于Mallat多分辨率分析（Multi-resolution Analysis, MRA）框架。Mallat算法的本质是：通过一对镜像滤波器组将信号逐级分解。

// Mallat DWT分解算法
// 对长度为N的信号x进行J级小波分解

初始化: c_0 = x  (原始信号)
for j = 1 to J:
    // 1. 通过低通滤波器h0 + 下采样2 -> 近似系数（低频）
    c_j = downsample(conv(c_{j-1}, h0), 2)

    // 2. 通过高通滤波器h1 + 下采样2 -> 细节系数（高频）
    d_j = downsample(conv(c_{j-1}, h1), 2)

// 输出: c_J (最粗的近似系数), d_1,...,d_J (各级细节系数)
// 总系数个数仍为N (无损分解)
// 重构: 镜像过程，用合成滤波器 + 上采样

这个过程将一个信号分解为低频"近似"部分和高频"细节"部分，然后在下一级只对近似部分继续分解。这就像一个多倍放大镜——低频用宽窗口看得粗略但频率精确，高频用窄窗口看得精细但频率模糊。

DWT有两个深远的影响：第一，它启发了JPEG 2000图像压缩标准（用CDF 9/7小波）；第二，Mallat算法的滤波器组结构与后续深度学习中的卷积神经网络有着惊人的相似性。

特性 STFT / 语谱图 连续小波变换 (CWT) 离散小波变换 (DWT)
时间分辨率 固定（由窗长决定） 可变（低频差、高频好） 可变（低频差、高频好）
频率分辨率 固定（由窗长决定） 可变（低频好、高频差） 可变（低频好、高频差）
计算效率 高（基于FFT） 低（连续积分） 高（滤波器组，O(N)）
逆变换 完美重建（ISTFT） 存在（但复杂） 完美重建（滤波器组对称）
主要用途 语音、音乐、通用 地质/生物信号分析 压缩、去噪、编码

18.4 Wigner-Ville分布：最高分辨率，但小心交叉项

STFT和小波变换都属于线性时频表示——它们对信号做线性变换，不存在交叉项干扰。但线性变换受不确定性原理的限制，时频分辨率不可能同时很高。

如果我们愿意放弃线性特性呢？双线性时频分布（也叫Cohen类）可以给出更高的时频分辨率。其中最著名的就是Wigner-Ville分布（WVD）：

\(W_x(t, f) = \int_{-\infty}^{\infty} x\left(t + \frac{\tau}{2}\right) x^*\left(t - \frac{\tau}{2}\right) e^{-j2\pi f \tau} d\tau\)

WVD有一些非常吸引人的性质：

  • 时频分辨率极高——理论上可以达到不确定性原理的下界，比STFT和小波都要高；
  • 满足边缘特性——对频率积分得到瞬时功率，对时间积分得到能量谱密度；
  • 对线性调频（chirp）信号——WVD的时频表示是一条完美的斜线，没有任何模糊。
⚠️ 踩坑实录：WVD有一个致命的缺点——交叉项干扰（cross-term interference）。当信号包含多个频率分量时，WVD会在两个分量的中点处产生一个"幽灵分量"。比如一个信号包含频率 \(f_1\) 和 \(f_2\) 两个正弦波，WVD会在 \((f_1+f_2)/2\) 处出现一个虚假的交叉项，而且这个交叉项的幅度可以达到真实分量的两倍！我第一次在雷达信号上试WVD时，看到时频谱上的"第三个目标"兴奋了半天，后来才发现是交叉项的幻觉。这个教训告诉我：WVD的图不能裸看，必须结合先验知识判读。

为了解决交叉项问题，研究人员提出了各种加窗WVD（称为伪WVD或平滑WVD），通过在时域和频域分别加窗来抑制交叉项——但代价是牺牲了时频分辨率。这就是所谓的"时频分析中的不确定性"——没有任何一种时频分布能在所有指标上做到最优。

18.5 应用选择指南

面对这么多时频分析工具，到底该怎么选？我根据自己的工程经验给你一个参考：

你关心的问题 推荐工具 原因
语音信号的共振峰变化？ STFT语谱图 参数成熟（25ms帧长），物理意义直观
机械振动中的瞬态冲击（轴承故障）？ CWT 或 DWT 小波能匹配冲击信号的局部特性
信号压缩或去噪？ DWT Mallat算法高效，小波阈值去噪经典
线性调频信号参数估计（雷达）？ WVD 或 分数阶傅里叶变换 chirp信号在WVD中表现为完美直线
多分量信号分离？ STFT + 掩码 (masking) 没有交叉项，便于手动/自动分离
生物医学信号（ECG/EEG）分析？ CWT 时频分辨率可调，多频段同时观察
💡 进阶方向：时频分析在2020年代最新的发展包括：同步压缩变换（Synchrosqueezing Transform, SST）——它"重排"时频系数使得能量更集中；以及基于深度学习的时频表示——用神经网络学习最优的时频核函数。这些内容已经超出了本课程的初级-中级范围，但如果你有兴趣，可以参考第26章AI+DSP融合中的相关知识。

18.6 本章小结

  • STFT（语谱图）是最常用的时频分析工具，通过加窗FFT滑动实现，时间与频率分辨率受不确定性原理制约——长窗频率好、短窗时间好；
  • 小波变换实现了变分辨率分析——低频宽窗口、高频窄窗口。CWT适合精细分析，DWT（Mallat算法）适合工程实现，JPEG 2000是其典型应用；
  • Wigner-Ville分布提供了极高时频分辨率，但双线性结构带来交叉项干扰——使用时必须警惕"幽灵分量"；
  • 不同场景选择不同的工具：语音用STFT、冲击检测用小波、chirp分析用WVD、压缩去噪用DWT。
