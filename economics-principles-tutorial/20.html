20.1 你的100元还值100元吗？

大家好，我是你们的讲师。我有个问题想问你：你还记得你小时候一碗牛肉面多少钱吗？我上大学的时候学校门口的兰州拉面只要5块钱一碗。现在你再去看看——没有15块钱根本下不来。这就是通货膨胀（Inflation）最直观的体验：钱不值钱了。

但通胀不仅仅是"东西变贵了"那么简单。它深刻影响着每一个人的财富——你的存款、你的工资、你的房贷利率、甚至你退休后的养老金购买力。理解通胀，是理解现代宏观经济的核心。本章我们来系统地学一学：通胀是怎么衡量的？它为什么发生？它让谁受益、让谁受损？

20.2 CPI — 消费者物价指数

怎么衡量物价上涨了多少？经济学家发明了消费者物价指数（Consumer Price Index, CPI）。CPI的基本思想是：选择一个典型的"消费篮子"（Basket of Goods），看买这个篮子的花费每年变化了多少。

CPI的五个步骤

  1. 确定消费篮子：通过调查居民消费习惯，确定一个典型的商品和服务组合。比如一个城市家庭平均每个月买多少大米、多少汽油、付多少房租。
  2. 获取价格数据：统计调查员去超市、菜市场、网上商城收集每样商品的价格。
  3. 计算篮子总花费：用基期价格计算一次，用当期价格计算一次。
  4. 计算CPI：CPI = (当期篮子花费 / 基期篮子花费) × 100。基期的CPI总是100。
  5. 计算通胀率：通胀率 = (本期CPI - 上期CPI) / 上期CPI × 100%。

下面这张表展示了中国和美国CPI篮子组成的巨大差异——这反映了两国消费结构的本质不同：

消费类别中国CPI权重美国CPI权重差异解读
食品烟酒约30%约13%恩格尔系数：中国人仍花更多收入在"吃"上
居住约23%约33%美国居住成本（房租+自有住房等价租金）权重更大
交通通信约13%约15%占比接近，但中国公共交通更便宜
教育文化娱乐约11%约6%中国家庭在子女教育上投入更大
医疗保健约9%约8%美国医疗"单价"极高但保险覆盖降低直接支出感
衣着约6%约3%中国服装消费重视程度较高
生活用品及服务约5%约5%家用电器、日常用品等
其他约3%约17%美国包括烟草、个人护理、金融服务等更细分类
💡 权重的意义：如果猪肉在中国CPI中权重高，那么猪肉涨价就会显著推高中国通胀。2020年非洲猪瘟导致猪肉价格翻倍，直接拉高中国CPI约2个百分点——这就是为什么央行和政策制定者需要密切关注CPI构成。

20.3 CPI的偏差——统计局低估了通胀吗？

你有没有这种感觉——官方公布的通胀率看起来不高（比如中国近年CPI多在0-3%之间），但你自己感觉生活成本涨得很厉害？这不是你的错觉。CPI存在一些系统性的偏差（Biases），导致它可能低估了真实的物价上涨：

替代偏差（Substitution Bias）

CPI篮子里的商品权重是固定的——比如说猪肉占2%。但实际上，当猪肉涨价太多时，消费者会转而吃鸡肉。CPI的"固定篮子"没有反映这种替代行为，所以高估了猪肉涨价对生活成本的影响。换句话说：CPI假定消费者是"傻的"——涨价了也不换。

质量偏差（Quality Bias）

一部iPhone 15比iPhone 6贵很多，但它的性能也好了不知多少倍。CPI把价格差异全算作"涨价"了，但事实上你买的是更好的产品。一部分涨价应该算作"质量提升"而非"通胀"。同样，10年前的一台5000元的电脑和今天一台5000元的电脑——今天这台性能可能强了10倍。CPI没有完美地剔除这种质量变化。

新商品偏差（New Goods Bias）

当一种全新商品出现（比如智能手表、移动支付服务），它一开始并不在CPI的篮子里。等它被纳入篮子时，它的价格早已大幅下降过了（想想一开始的智能手环多少钱）。新商品带来的生活成本降低，CPI捕捉不到。

⚠️ 重要理解：因为上述偏差，很多经济学家认为CPI每年高估了约0.5-1个百分点的真实通胀。这听起来不大，但累积几十年，影响巨大——因为它直接影响社保养老金每年的涨幅计算。

20.4 名义利率 vs 实际利率

假设你把10,000元存银行，一年期定期利率是3%。一年后你得到了10,300元——你赚了吗？那要看通胀是多少。

如果这一年通胀率是5%，那么你的10,300元实际购买力反而下降了。这就是费雪方程（Fisher Equation）要解决的问题：

实际利率 ≈ 名义利率 — 通胀率

  • 名义利率（Nominal Interest Rate）：银行告诉你的利率，不加通胀调整。
  • 实际利率（Real Interest Rate）：剔除通胀后，你真正的购买力增长。

用费雪方程：如果名义利率3%，通胀率5%，那么实际利率 = 3% - 5% = -2%。你的存款实际在缩水！这就是为什么长期高通胀对一个国家的储户来说是"隐形税收"——你的钱放在银行，躺在那，但一天比一天不值钱。

20.5 通胀的再分配效应——谁赢谁输？

通胀不是"所有人都均匀地少买5%的东西"。它像一场静悄悄的财富再分配——有人赢，有人输：

群体通胀下的处境原因
债务人（借钱的）✅ 赢家用贬值后的钱还债——实际债务负担减轻
债权人（放贷的）❌ 输家收回的钱购买力下降——实际收益为负
固定收入者（如退休人员）❌ 输家养老金固定，但物价上涨了
工资谈判能力强者✅ 可能赢工资能跟上通胀甚至跑赢通胀
现金持有者❌ 输家现金的购买力每天都在缩水
房产持有者✅ 可能赢资产价格随通胀上涨，房贷实际负担减轻
🔑 一句话总结：通胀惩罚储蓄者，奖励借贷者；惩罚拿固定工资的人，奖励能"重新定价"的人（比如企业主、炒股炒房者、有议价权的专业人士）。

20.6 现实故事一：2022年全球通胀浪潮

2022年，全球经历了一波几十年来最猛的通胀浪潮。但不同国家的表现截然不同，给宏观经济学提供了一次绝佳的"自然实验"：

  • 美国：CPI一度冲到9.1%（2022年6月）。原因？疫情后的天量刺激政策（直升机撒钱）、供应链断裂、俄乌战争推高能源粮食价格。美联储被迫启动了40年来最激进的加息，基准利率从0%一路拉到5%以上。
  • 欧元区：同样被能源危机重创。德国CPI一度超过8%，欧元区整体在2022年10月达到10.6%的高点。欧洲央行被迫结束负利率时代。
  • 中国：2022年中国CPI最高只有2.8%。为什么中国通胀这么低？因为中国经济当时处于需求不足的状态——房地产下行拖累内需，出口因全球经济放缓走弱。中国面临的问题不是通胀，而是通缩压力。

这个故事说明了一个关键道理：通胀本质上是一个"太多货币追逐太少商品"的现象。美国是需求过热+供给冲击，中国是需求疲软+供给过剩——同一个地球，两种截然不同的通胀故事。

20.7 现实故事二：委内瑞拉的恶性通胀

如果说2022年美国的9%通胀算"猛"的话，那委内瑞拉的"通胀"你用什么词来形容？2018年委内瑞拉的通胀率——撑住——超过了1,000,000%。是的，百万分之一百万。这就是恶性通胀（Hyperinflation）。

在恶性通胀下，经济学课本里的"钱"变成了笑话：

  • 一包卫生纸的价格在早上和下午可以差一倍。
  • 去餐馆吃饭，菜还没上完，菜单的价格就已经更新了。
  • 人们把一捆一捆的钞票塞进塑料袋去超市——真正的"钞票不如钞票值钱"（钞票的重量都比它的购买力大）。
  • 有报道称，人们用玻利瓦尔钞票当作墙纸来贴——因为一卷真正的墙纸比一捆同样面积的钞票还贵。

恶性通胀的根源几乎总是政府疯狂印钞。委内瑞拉政府长期靠印钱来填补财政赤字——当政府不断"印钱"来支付账单，市场上的货币数量暴增，每单位货币的价值就急速下降。最终，人们对本币完全失去信心，经济体系近乎崩溃，造成大规模人道危机。

⚠️ 恶性通胀的经典公式：政府财政赤字 → 央行为政府融资（印钞）→ 货币供给暴增 → 物价飞涨 → 需要更多货币 → 印更多钱 → 物价涨幅反超印钞速度 → 人们抛弃货币 → 经济美元化/物物交换。一旦进入这个循环，极难打破。

20.8 用代码算一算通胀

理论说多了，我们来点实际的。下面这个Python函数模拟了CPI计算和通胀率的计算过程：

def calculate_cpi_and_inflation(prices_base, prices_current, quantities):
    """
    prices_base: 基期价格 {商品名: 价格}
    prices_current: 当期价格 {商品名: 价格}
    quantities: 消费篮子 {商品名: 数量}
    """
    cost_base = sum(prices_base[g] * quantities[g] for g in quantities)
    cost_current = sum(prices_current[g] * quantities[g] for g in quantities)
    cpi = (cost_current / cost_base) * 100
    return cpi

def inflation_rate(cpi_current, cpi_previous):
    return ((cpi_current - cpi_previous) / cpi_previous) * 100

# 设定消费篮子
basket = {"大米": 50, "猪肉": 5, "汽油": 30, "理发": 2}

# 基期价格 (2023年)
base_prices = {"大米": 3, "猪肉": 25, "汽油": 7, "理发": 30}

# 各年价格
yearly_prices = {
    2023: base_prices,
    2024: {"大米": 3.2, "猪肉": 28, "汽油": 8, "理发": 35},
    2025: {"大米": 3.3, "猪肉": 30, "汽油": 7.5, "理发": 38},
}

cpis = {}
for year, prices in yearly_prices.items():
    cpis[year] = calculate_cpi_and_inflation(base_prices, prices, basket)
    print(f"{year}年 CPI: {cpis[year]:.1f}")

print()
for year in [2024, 2025]:
    rate = inflation_rate(cpis[year], cpis[year - 1])
    print(f"{year}年通胀率: {rate:.2f}%")

# 输出:
# 2023年 CPI: 100.0
# 2024年 CPI: 110.9
# 2025年 CPI: 114.6
# 2024年通胀率: 10.90%
# 2025年通胀率: 3.36%
💡 实操建议：你可以试着把你自己的消费记录做成一个"个人CPI"。记三个月账，看看你的个人通胀率和官方CPI的差距——往往会有惊喜（或惊吓）的发现。

20.9 本章小结

  • CPI是衡量生活成本的核心指标，但存在替代偏差、质量偏差和新商品偏差
  • 费雪方程告诉我们：实际利率 = 名义利率 - 通胀率
  • 通胀是一场隐形的财富再分配——债主受损，借债人受益
  • 温和通胀（2-3%）是现代央行追求的目标；恶性通胀则是经济毁灭的灾难
