4.1 天线如何产生辐射？

大家好，我是你们的讲师。我们已经学习了Maxwell方程组和电磁波在空间中的传播行为。现在问题来了：电磁波最开始是怎么产生的？天线的那根金属线里到底发生了什么神奇的事情，让原本在电路里规规矩矩流动的电子，突然向空间"甩出"了电磁波？

要理解辐射，我们必须回到一个基本的物理事实：只有加速运动的电荷才产生辐射。让我们通过三个场景来建立直觉：

  • 静止电荷：产生静电场，但能量不会"跑远"——场随着距离只是缓慢衰减（1/r²），但没有任何能量辐射出去。
  • 匀速运动的电荷：产生恒定电流和静磁场——仍然没有辐射。这就是为什么一根通直流电的导线不会发射电磁波。
  • 加速（或减速）运动的电荷：终于产生了辐射！当电荷来回振荡时，电场和磁场的变化相互激发，形成脱离导线飞向远方的电磁波。

从Maxwell方程组的视角看：变化的电流（即振荡的电荷运动）通过位移电流∂D/∂t在空间中建立起时变电磁场，这个场以有限速度（光速）向外传播——这就是辐射。换句话说，天线就是把导线上传输的导行波（guided wave）转化为自由空间中传播的辐射波（radiating wave）的换能器。

核心思想：天线不是"产生"电磁波的能量——它只是把电路中的交变能量"耦合"到自由空间中。从电路角度看，天线呈现为输入阻抗Z_in；从场角度看，天线建立了一个特定的空间场分布。这两个视角通过天线理论紧密联系在一起。

一个经典的思想实验有助于理解：想象一段直导线，两端各有一个金属球，中间断开一个小缝。在缝上施加交变电压——电子在两个球之间来回运动。由于电荷在导线两端聚集和疏散，周围的空间中形成了一个时变的电场和磁场。当交变频率足够高、导线尺寸与波长可比时，大部分能量不再返回电路，而是"甩"向了远方——这就是天线辐射的基本原理。

4.2 赫兹电偶极子的场分布

赫兹电偶极子（Hertzian electric dipole）——也称为电流元（current element）——是天线理论中最基本的辐射单元。它是一段极短的载流导线（长度dl << λ），导线上电流均匀分布（幅度和相位处处相同）。虽然真实的偶极子天线电流分布并不均匀（后面我们会看到半波振子），但赫兹偶极子是理解和推导更复杂天线辐射特性的基础。

设电流元沿z轴放置，中心在原点，长度为dl，载有均匀时谐电流I = I₀ e^(jωt)。通过求解磁矢位（magnetic vector potential）A的波动方程，可以得到完整的电场和磁场分布。

在球坐标系（r, θ, φ）中，完整的场分量为：

磁场（只有φ分量）：
Hφ = (I₀ dl · e^(-jkr) / (4π)) · (jk/r + 1/r²) · sinθ

电场（两个分量）：
Er = (I₀ dl · η₀ · e^(-jkr) / (2π)) · (1/r² - j/kr³) · cosθ
Eθ = (I₀ dl · η₀ · e^(-jkr) / (4π)) · (jk/r + 1/r² - j/kr³) · sinθ

观察这些表达式中的距离因子，我们发现场由三项组成：

  • 静电场项（1/r³）：只在极近的距离内显著，对应电荷的库仑场。
  • 感应场项（1/r²）：近-中场区域主导，对应电磁感应效应。
  • 辐射场项（1/r）：远场区域主导——这是真正"飞出去"的电磁波！

在远场区（r >> λ），只有1/r项保留：

远场近似：
Eθ = jη₀ · (I₀ dl · k · e^(-jkr) / (4πr)) · sinθ
Hφ = Eθ / η₀

注意到远场中：E和H相互垂直、同相、幅度比为η₀，传播方向为径向r̂——这正是我们第2章讲过的均匀平面波的特性！所以在离天线足够远的地方，电磁波看起来就像一个局部均匀的平面波。

数学小贴士：为什么辐射场是1/r衰减？想象一个以天线为中心的球面，辐射功率穿过球面的总通量要守恒（能量守恒）。球面积=4πr²，通量密度（坡印廷矢量）正比于|E|²/η₀，所以|E|² ∝ 1/r²，即|E| ∝ 1/r。这就是物理直觉：功率密度随距离平方衰减，因此场幅度随距离线性衰减。

4.3 近场区、辐射近场区与远场区的划分

天线的周围空间并非"一刀切"的远场区。根据不同距离上场行为的差异，通常划分为三个区域：

电抗近场区（Reactive Near-Field, r < 0.62√(D³/λ)）

这是最靠近天线的区域。在这里，1/r²和1/r³项占主导，电抗性能量（不向外辐射的能量）储存在天线附近。电场和磁场之间的相位差接近90°——就像电感和电容中的无功功率一样。在天线设计中，任何靠近天线的物体（包括人体、外壳、安装支架）都会严重扰动这个区域的场分布，导致天线失谐、效率下降。这就是iPhone 4"天线门"的根本原因——手掌进入了电抗近场区。

辐射近场区（Fresnel区, 0.62√(D³/λ) < r < 2D²/λ）

在这个区域，辐射场已经占据主导，但场的角向分布（方向图）仍然随距离变化。Fresnel区的概念在天线测量中非常重要——如果你想精确测量天线的远场方向图，测试距离必须大于2D²/λ（远场条件），否则测量结果会与真实远场方向图有偏差。如果受限于场地条件只能在Fresnel区测量，需要使用近场-远场变换（NF-FF transformation）技术。

远场区（Fraunhofer区, r > 2D²/λ）

远场区中，方向图不再随距离变化，场表现为从天线发出的球面波的局部平面波近似。天线设计中几乎所有的性能参数——方向图、增益、极化——都是在远场条件下定义的。在天线测量中，远场条件r > 2D²/λ是最基本的测试要求。

场区 距离范围 主导场分量 能流方向 典型应用/注意事项
电抗近场区 r < 0.62√(D³/λ) 1/r³ 静电场项 径向-横向往复（无功） 天线阻抗受周围环境影响；人体感应
辐射近场区 (Fresnel) 0.62√(D³/λ) < r < 2D²/λ 1/r² 感应场 + 1/r 辐射场 主要是径向向外 近场测量；方向图随距离变化
远场区 (Fraunhofer) r > 2D²/λ 1/r 辐射场 纯径向向外 标准天线测量；方向图稳定
踩坑故事：我参与过的一个5G基站天线项目中，测试团队在一个小型暗室里测量一款1.8米长的面板天线在2.6 GHz频段的方向图。按计算，远场条件要求 r > 2×(1.8)²/0.115 ≈ 56米——而暗室长度只有30米。他们在Fresnel区测出的方向图副瓣电平比真实值高了约2 dB，导致项目验收一度无法通过。最终我们不得不采用柱面近场扫描+NF-FF变换来获得准确的远场方向图。所以请记住：远场条件不是一个建议——它是一个硬性的数学要求。

4.4 远场方向图与辐射电阻

归一化方向图

从赫兹偶极子的远场表达式可以看出，场强Eθ与sinθ成正比——即在垂直于偶极子轴线的方向（θ=90°）辐射最强，沿轴线方向（θ=0°或180°）辐射为零。这个空间分布就是天线的辐射方向图（radiation pattern）。

归一化方向图函数：F(θ, φ) = E(θ, φ) / E_max

对于赫兹偶极子：F(θ) = sinθ。这是一个典型的"甜甜圈"形状方向图：在赤道面（H面，θ=90°平面）是全向的，在子午面（E面，φ=常数平面）是8字形的。

天线方向图通常用E面（电场矢量所在平面）和H面（磁场矢量所在平面）两个主平面上的切面来描述。对于沿z轴放置的偶极子：

  • E面（φ=常数平面）：包含z轴，方向图呈"8"字形。F(θ) = sinθ
  • H面（θ=90°平面）：垂直于z轴，方向图是圆（全向）。F(φ) = 1

辐射电阻

天线辐射出去的功率可以用一个等效电阻上的焦耳热损耗来类比——这个等效电阻就是辐射电阻（radiation resistance）Rrad。

定义：Rrad = 2Prad / |I₀|²

对于赫兹偶极子：

Rrad = 80π² (dl/λ)² (Ω)

这个公式的意义非常深刻：

  • 辐射电阻与(dl/λ)²成正比——这意味着天线的电尺寸（相对于波长）决定了它的辐射能力。
  • 对于 dl = 0.01λ 的极短偶极子：Rrad ≈ 0.08 Ω ——而它的欧姆电阻（铜损）可能有几Ω。这样一来，大部分输入功率都损耗在导线上而不是辐射出去——这就是短天线辐射效率极低的原因。
  • 这就是为什么手机天线必须利用整个手机壳体作为辐射体（增加"有效电长度"），而一个单独的微小PCB贴片根本不足以有效辐射。
工程直觉：辐射电阻是天线最重要的指标之一。一个设计良好的半波偶极子的辐射电阻约73 Ω，大约等于标准传输线的特性阻抗（50 Ω或75 Ω），因此可以直接匹配。而一个电小环天线的辐射电阻可能只有0.1 Ω量级——这就是为什么电小天线必须使用高Q匹配网络（并且带宽很窄）的原因。

4.5 对偶原理与磁流元

天线理论中有一个非常优雅的对偶原理（duality principle）。它指出：如果我们将电场与磁场互换、电流与磁流互换、介电常数与磁导率互换，那么Maxwell方程组的形式保持不变。利用这一对称性，我们可以从已知的电偶极子解直接得到磁流元（magnetic current element）的解。

电偶极子 (Electric Dipole) ⇔ 磁流元 (Magnetic Current Element)
E ⇔ H
H ⇔ -E
ε ⇔ μ
I (电流) ⇔ Im (磁流)
直线导线 ⇔ 缝隙 (slot)

对偶原理的直接工程应用是缝隙天线（slot antenna）的理论分析：在金属导体板上开一个窄缝，当缝隙被适当激励时，它将向空间辐射电磁波。利用对偶原理，缝隙天线的远场方向图可以直接从互补的偶极子天线得到——只需要将电场和磁场互换，并将电流元换为磁流元（缝隙上的等效磁流）。

具体来说：一个长度为L的半波缝隙（谐振缝隙）在一个无限大理想导电平面上开槽，其方向图与一个半波偶极子的方向图相同，但极化方向旋转了90°（E/H互换）。这就是巴比涅原理（Babinet's principle）在电磁学中的体现。

工程应用：缝隙天线广泛用于机载雷达和卫星通信——因为它们可以与飞行器蒙皮共形安装，不产生额外的风阻。波导缝隙天线阵是机载火控雷达的标准配置，例如AN/APG-68雷达的平板缝隙阵天线。此外，智能手机金属边框上开缝作为天线辐射体的设计也基于同样的原理——这正是为什么几乎所有金属机身的手机都需要在天线位置预留非金属"白带"或"注塑条"的原因。

下面我们用Python计算赫兹偶极子的远场方向图和辐射电阻：

import numpy as np

def hertzian_dipole_field(I0, dl, lam, r, theta):
    """
    计算赫兹偶极子的远场 (r >> lam)
    参数:
        I0    : 电流幅度 (A)
        dl    : 偶极子长度 (m)
        lam   : 波长 (m)
        r     : 观测点距离 (m)
        theta : 观测点极角 (rad)
    返回:
        E_theta : 电场 θ 分量 (V/m)
        H_phi   : 磁场 φ 分量 (A/m)
    """
    eta0 = 377
    k = 2 * np.pi / lam

    E_theta = 1j * eta0 * I0 * dl * k / (4 * np.pi * r) * np.exp(-1j * k * r) * np.sin(theta)
    H_phi = E_theta / eta0
    return E_theta, H_phi

def radiation_resistance_dipole(dl, lam):
    """赫兹偶极子辐射电阻"""
    return 80 * np.pi**2 * (dl / lam)**2

# 示例计算
I0 = 1.0        # 电流 1A
dl = 0.01       # 长度 0.01λ
lam = 0.3       # 对应 1 GHz
r = 10 * lam    # 远场距离

print(f"偶极子长度: dl = {dl*lam*100:.1f} cm (λ={lam*100:.0f} cm)")
print(f"辐射电阻: R_rad = {radiation_resistance_dipole(dl, lam):.4f} Ω")
print()

# 计算不同角度上的场强
theta_vals = np.array([0, 30, 45, 60, 90])
for th_deg in theta_vals:
    th = np.deg2rad(th_deg)
    E_th, H_ph = hertzian_dipole_field(I0, dl, lam, r, th)
    power_density = 0.5 * abs(E_th)**2 / 377
    print(f"θ = {th_deg:3d}°: |E| = {abs(E_th)*1000:.2f} mV/m, "
          f"功率密度 = {power_density*1e6:.2f} μW/m²")

# 验证总辐射功率 = 积分坡印廷矢量
total_power = radiation_resistance_dipole(dl, lam) * I0**2 / 2
print(f"\n总辐射功率 (由R_rad计算): {total_power*1000:.4f} mW")

# 通过球面积分验证
P_rad_int = 0
for th_deg in range(0, 180, 1):
    th = np.deg2rad(th_deg + 0.5)
    E_th, _ = hertzian_dipole_field(I0, dl, lam, r, th)
    # 球面微元上的功率：dP = S_avg * r² sinθ dθ dφ, φ方向积分2π
    S_avg = 0.5 * abs(E_th)**2 / 377
    dP = S_avg * r**2 * np.sin(th) * np.deg2rad(1) * 2 * np.pi
    P_rad_int += dP

print(f"球面积分验证辐射功率: {P_rad_int*1000:.4f} mW")
print(f"两者一致!")
