第11章 有效市场假说

2019年8月，国内一位知名的"价值投资大V"在社交媒体上高调宣布：他满仓买入某只ST股票，理由是"市场错了，这个公司至少值50亿"。一个月后，这只股票因为连续20个交易日收盘价低于1元而退市，他的粉丝们损失惨重。

这个大V的悲剧源于两个极端的信念：他既相信市场是有效的（价格会最终回归"真实价值"），又相信自己的判断比市场更聪明。这种矛盾心理，是几乎所有投资者的共同困扰。事实上，有效市场假说可能是金融学中最"反人性"的理论——它告诉你：你可能并不比市场更聪明。

核心问题：股价已经反映了所有可用信息吗？如果答案是"是"，那么主动投资就是一场零和博弈甚至负和博弈（扣除费用后）。如果答案是"否"，那么"打败市场"就是可能的。这可能是投资学中最重要的问题，没有之一。

11.1 有效市场的三种形式

1970年，Eugene Fama在一篇经典论文中将有效市场分为三个层次：

弱式有效市场（Weak-Form EMH）

当前价格已经完全反映了历史价格和交易量信息。这意味着：技术分析（K线、均线、MACD等）无法持续获得超额收益。如果你看见一只股票已经连涨三天就追进去——从弱式有效角度看，你并没有比别人更多的信息。

实验证据：长期以来，学术界的共识是主要股票市场（美股、A股）是弱式有效的。但2020年代后的高频数据研究发现了微弱的"可预测性"——在毫秒级别，过去的价格对未来有预测能力。但这种可预测性很快被高频交易者套利掉，对普通投资者没有实际意义。

半强式有效市场（Semi-Strong-Form EMH）

当前价格已经反映了所有公开信息——财报、新闻、公告、分析师报告等。在半强式有效市场下，基本面分析（看财报、算估值）也无法持续跑赢市场。因为一旦信息被公开，价格会在极短时间内完成调整。

事件研究（Event Study）是检验半强式有效性的经典方法。如果市场是半强式有效的，那么盈利公告发布后的价格调整应该是"瞬间完成"的。但大量实证发现：

  • 盈余公告后漂移（PEAD, Post-Earnings Announcement Drift）：盈利超预期的公司在公告后3-6个月内仍然持续跑赢——这是对半强式有效最著名的否定证据之一。
  • 中国A股的异象：A股市场的PEAD效应更为显著，且存在"利好出尽是利空"的反转模式——说明半强式有效在A股可能更弱。

强式有效市场（Strong-Form EMH）

当前价格已经反映了所有信息，包括内幕信息。在强式有效市场下，连公司CEO都不可能在自家股票上获得超额收益——因为股价已经把CEO知道但还没公开的所有信息都反映进去了。

这个形式几乎在所有市场上都被否定。大量的研究表明：内幕交易者确实能获得超额收益，这也是监管机构花大力气打击内幕交易的原因。2024年，中国证监会公布的行政处罚案件中，内幕交易占比仍然超过30%。

市场形式信息集谁不能打败市场？实证支持
弱式历史价格和交易量技术分析者较好
半强式所有公开信息基本面分析者有争议
强式所有信息（含内幕）所有人被否定

11.2 实证证据：支持与反对

支持EMH的证据

随机游走（Random Walk）：大量统计检验发现，短期股价变动近似于随机游走——今天的涨跌几乎无法预测明天的涨跌。如果市场不是有效的，你会预期某种"模式"的存在，但数据中找不到一致的模式。

共同基金业绩：标准普尔2024年的SPIVA报告显示，过去15年中，超过85%的主动管理型美国大盘基金跑不赢标普500指数。在中国，2020-2025年期间，公募偏股基金的中位数收益率也未能持续跑赢沪深300全收益指数。如果市场不是有效的，为什么专业的基金经理（应该是最聪明的投资者）整体上跑不赢指数呢？

Renaissance Medallion基金的"反例"：你可能会说："那文艺复兴科技公司的大奖章基金呢？它年化收益超过66%！"这确实是一个著名的反例。但请注意：大奖章基金在1998年就停止对外募集资金了，只管理创始人和员工的资金。当它对外募资的RIEF基金（同样使用其交易策略）在2005-2010年的表现就远不如大奖章基金本身。这说明：有效的策略一旦规模化，其超额收益就会被迅速侵蚀——这恰恰是EMH预测的。

深度思考：EMH的支持者会说："你看到的所谓'打败市场'的例子，要么是运气（幸运的猴子也能打出莎士比亚），要么是承担了更高风险。"反对者会说："那么如何解释Vanguard创始人John Bogle统计的——在1980-2005年间，没有任何一只主动管理基金能连续15年跑赢指数？"这个争论持续了半个世纪，至今没有定论。

11.3 市场异象

如果市场是有效的，以下现象就不应该存在：

规模效应（Size Effect）

Banz在1981年发现，美国股市中小盘股的长期收益率显著高于大盘股。这个效应在1970-1980年代非常强，但在1990年代后减弱了——可能是被学者发现后，大量资金涌入套利，套利空间被抹平了。这被称为"异象的消失"，是EMH的有力支持证据之一。

1月效应（January Effect）

小盘股在1月份的表现显著好于其他月份。一种解释是：投资者在12月为了避税或美化组合而卖出亏损股票，1月买回。但随着市场越来越有效，1月效应在美股上已经基本消失。在中国A股，有类似的"春节效应"——春节前后几周市场往往上涨。

动量与反转（Momentum and Reversal）

Jegadeesh和Titman在1993年发现，买入过去6-12个月涨幅最大的股票、卖出跌幅最大的股票可以产生显著的超额收益。但有趣的是，在更长的持有期（3-5年）中，这个效应会反转——前期涨幅最大的股票在后期反而表现更差（长期反转效应）。

# 动量策略的简单实现（Python示例）
import pandas as pd
import numpy as np

def momentum_strategy(returns, lookback=12, holding=3, top_pct=0.1):
    """
    动量策略回测框架（简化版）
    - lookback: 过去几个月计算动量
    - holding: 持有几个月
    - top_pct: 选取前百分之多少的股票
    """
    # 计算过去lookback个月的累计收益（动量分数）
    momentum_scores = (1 + returns).rolling(lookback).apply(
        lambda x: x.prod(), raw=True) - 1

    # 每月末选取动量分数最高的top_pct股票
    n_stocks = len(returns.columns)
    n_select = int(n_stocks * top_pct)

    # 持有holding个月
    # 这里简化处理：每月调仓，等权买入选中的股票
    portfolio_return = pd.Series(index=returns.index)

    for month in range(lookback, len(returns) - holding + 1):
        rankings = momentum_scores.iloc[month].rank(ascending=False)
        selected = rankings[rankings <= n_select].index
        # 等权持有，计算未来holding个月的收益
        forward_returns = returns.iloc[month+1:month+1+holding][selected]
        portfolio_return.iloc[month+1:month+1+holding] = (
            forward_returns.mean(axis=1).values
            if len(selected) > 0 else 0)

    return portfolio_return

# 注意：这个简化框架未考虑交易成本和冲击成本

11.4 2025-2026年EMH大辩论

近年来，EMH的争论不仅没有平息，反而在2025-2026年迎来了新一轮高潮：

沃顿"相关性忽视"研究

2025年，沃顿商学院发表了一项轰动性的研究，题目是《Correlation Neglect in Financial Markets》。研究发现，投资者在评估多资产组合时，系统性地忽略资产之间的相关性——他们只关注每只个股的"故事"，而忽略了整体组合的冗余风险。更有趣的是，这种"相关性忽视"产生了可预测的定价模式：行业内相关性高的股票更容易被错误定价，因为投资者低估了它们的共同风险。

这个发现的重要意义在于：它提供了一个行为金融学框架下的可交易模式——你不必假设所有投资者都理性，只需要找到系统性的错误定价并套利。这比EMH的"完全理性"假设要温和得多，但也不像极端行为金融学那样认为市场完全无效。

行为金融学的反击

2026年，行为金融学阵营发表了多项研究，对EMH发起了新的挑战：

  • 社交媒体的信息污染：研究发现，Reddit的WallStreetBets论坛上的情绪指标可以在10分钟级别预测部分小盘股的走势——这显然违背了弱式有效。
  • 注意力驱动的定价：在信息过载时代，投资者的注意力是一种稀缺资源。被高关注度的股票往往被过度定价，随后出现均值回归。
  • 不可分散的情绪风险：2025年的一项研究表明，投资者情绪存在一个无法被套利的"系统性成分"——即使机构投资者知道情绪导致的错误定价，也不敢贸然反向操作，因为"情绪可能持续更久"（凯恩斯的名言再次被验证）。

11.5 适应性市场假说（AMH）

面对EMH和行为金融学之间几十年的僵局，MIT的Andrew Lo在2004年（并在2017年出版的《适应性市场》中详细阐述）提出了一个折中框架——适应性市场假说（Adaptive Market Hypothesis, AMH）。

AMH的核心思想是：市场不是"有效"或"无效"的二元状态，而是随着市场参与者的构成和外部环境的变化而动态调整的。它借鉴了进化生物学中的几个概念：

  • 进化：市场参与者不是完全理性的，他们通过试错学习。成功的策略被保留，失败的策略被淘汰。
  • 适应：市场有效性不是一成不变的，而是在不同市场状态下变化。在稳定时期，市场更有效；在危机时期，市场有效性下降（套利机会增加）。
  • 生态位：不同的策略在市场的不同"生态位"中生存。一个在牛市中表现优异的策略，在熊市中可能失效。
维度EMH行为金融学AMH（折中）
理性假设完全理性有限理性、非理性进化适应
市场效率总是有效总是无效取决于市场状态
套利机会不存在存在但有限周期性地出现和消失
投资者行为固定偏好系统性偏差从经验中学习、进化
对主动投资的含义应该买指数基金可以利用非理性在某些时期有效、在某些时期无效
AMH对投资者的实用启示：
  • 市场有效性是随时间变化的——在危机中（如2008、2020年），套利空间增大；在牛市顶端，有效性最高（所有信息已被挖掘）。
  • 超额收益的来源不是"发现错误定价"，而是在市场无效时期提供流动性并承担逆向风险。
  • 投资者的风险厌恶不是恒定的——在牛市中风险承受能力"看起来"增强了，但实际上是资产价格上涨带来的幻觉。
  • 策略需要根据市场环境动态调整：没有永远有效的策略，只有不断进化的投资者。
给投资者的最终建议：无论你相信EMH还是行为金融学，有一个结论是确定的——大多数人的大多数交易是零和的。如果你没有信息优势、分析优势或行为优势（控制情绪的能力），那么买指数基金并长期持有是最理性的选择。而如果你确实有这些优势，也请保持谦逊——连文艺复兴科技这样的顶级机构，也会面临策略失效的时期。

EMH是一个让人又爱又恨的理论。它给了我们一个思考市场定价效率的基准框架，同时也揭示了人类认知的边界。但正如AMH所指出的，市场参与者不是完美的理性计算器，而是有血有肉、带着偏见和情感的人。这在下一章——行为金融学中，我们将系统地解剖这些偏见和情感，看看它们如何在市场中制造系统性的定价偏差。
