开篇：时域的卷积 vs. 频域的乘法

大家好，我是你们的讲师。上一章我们在时域中推导了随机信号通过 LTI 系统的三个核心公式：均值、互相关、自相关的传递关系。这些公式虽然精确，但涉及的内容比较复杂——特别是双重卷积，手动计算起来相当繁琐。

现在让我们换到频域视角。还记得在确定性信号分析中，时域的卷积对应频域的乘法：y(t) = h(t) ∗ x(t) ⟺ Y(ω) = H(ω) · X(ω)。到了随机信号的领域，同样的简化发生了——只不过我们处理的不是频谱 X(ω) 本身（它不存在），而是功率谱密度 S_X(ω)。

本章我们将看到：频域公式 S_Y(ω) = |H(ω)|² · S_X(ω) 是随机信号通过 LTI 系统分析的"王牌"——简单到令人难以置信，但又极其强大。

输出功率谱（频域分析）

S_Y(ω) = |H(ω)|² · S_X(ω)

上一章我们推导了时域中的输出自相关：

R_Y(τ) = R_X(τ) ∗ h(τ) ∗ h(-τ)

对这个等式两边同时做傅里叶变换：

  • 左边：R_Y(τ) 的傅里叶变换是 S_Y(ω)（维纳-辛钦定理）。
  • 右边：R_X(τ) 的傅里叶变换是 S_X(ω)；h(τ) 的傅里叶变换是 H(ω)；h(-τ) 的傅里叶变换是 H*(ω)（因为 h(t) 是实函数，H*(ω) = H(-ω)）。

时域卷积 ⟹ 频域相乘，所以：

S_Y(ω) = S_X(ω) · H(ω) · H*(ω) = |H(ω)|² · S_X(ω)

这就是频域分析的核心公式：输出功率谱密度 = 输入功率谱密度 × 系统幅频响应的平方。注意：相位响应 H(ω) 的相角在 |H(ω)|² 中消失了！这是因为功率谱只关心功率的分布，不关心相位——巧合的是，随机信号的功率谱也确实不包含相位信息。

深刻理解：为什么是幅频响应的"平方"？因为功率与幅度的平方成正比。如果系统将输入信号的幅度放大 A 倍，输出功率放大 A² 倍。所以 S_Y(ω) = |H(ω)|² · S_X(ω) 的本质是"每个频率上的输入功率被系统在该频率上的功率增益缩放"。这个公式的正确性依赖于维纳-辛钦定理——这就是为什么我们花了整整第 9 章来为这一刻做准备。

总输出功率

将 S_Y(ω) 在全频带积分并除以 2π，就得到输出信号的总平均功率：

E[Y²(t)] = (1/2π) ∫_{-∞}^{∞} S_Y(ω) dω
          = (1/2π) ∫_{-∞}^{∞} |H(ω)|² · S_X(ω) dω

如果输入是白噪声（S_X(ω) = N₀/2），那么：

E[Y²(t)] = (N₀/2) · (1/2π) ∫_{-∞}^{∞} |H(ω)|² dω

这个公式告诉我们：白噪声通过 LTI 系统的输出功率，完全由系统的总能量（|H(ω)|² 的积分）决定。这就引出了下面这个概念。

等效噪声带宽

定义与动机

等效噪声带宽（Equivalent Noise Bandwidth, ENB）是一个极其有用的工程概念。它的动机是：我们想用一个理想矩形滤波器来等效替代实际滤波器，使得在输入为白噪声时，两者输出相同的噪声功率。

定义：等效噪声带宽 B_eq 是这样一个矩形滤波器的带宽——该矩形滤波器的增益等于实际滤波器在中心频率处的增益 |H(ω₀)|²（对于低通滤波器，中心频率在 ω = 0 处），矩形宽度为 B_eq：

B_eq = (1 / |H(ω₀)|²) · (1/2π) ∫_{-∞}^{∞} |H(ω)|² dω

对于低通滤波器，ω₀ = 0；对于带通滤波器，ω₀ 是中心频率。

可以这样理解：

  • ∫ |H(ω)|² dω 是系统对白噪声的总功率增益
  • |H(ω₀)|² 是系统在通带内的最大功率增益
  • 它们的比值就是等效矩形带宽

为什么不等同于 3dB 带宽

等效噪声带宽 B_eq 和 3dB 带宽 B_{3dB} 是两个不同的概念：

  • 3dB 带宽：|H(ω)|² 下降到最大值一半（-3dB）处的频率宽度。
  • 等效噪声带宽：考虑了滤波器在整个频带上的响应特性，是一个"功率等效"的带宽度量。

对于一阶 RC 低通滤波器：B_{3dB} = 1/(2πRC) Hz（在 ω 域中 ω_{3dB} = 1/RC），而等效噪声带宽 B_eq = π/(2RC) rad/s = 1/(4RC) Hz。注意 B_eq > B_{3dB}，因为实际滤波器的滚降是缓慢的，在高频仍有残留的噪声功率通过。

对于高阶巴特沃斯（Butterworth）滤波器，B_eq 逼近 B_{3dB}：阶数越高，滚降越陡，等价矩形越接近 3dB 带宽。

工程速查：常见滤波器的 B_eq / B_{3dB} 比值：一阶 RC：1.57；二阶 Butterworth：1.11；三阶 Butterworth：1.05；四阶 Butterworth：1.03。这个比值通常介于 1.0 到 1.6 之间。在设计接收机时，如果你用 3dB 带宽来估计噪声功率，可能会低估 10%-50%。

白噪声通过 LTI 系统

输出特性

白噪声通过 LTI 系统是最常见也最简单的应用场景。设输入白噪声的 PSD 为 S_X(ω) = N₀/2，则：

  • 输出 PSD：S_Y(ω) = (N₀/2) · |H(ω)|²——输出 PSD 的形状完全由 |H(ω)|² 决定。
  • 输出自相关：R_Y(τ) = (N₀/2) · ϕ_h(τ)，其中 ϕ_h(τ) = h(τ) ∗ h(-τ) 是冲激响应的自相关。
  • 输出功率：σ²_Y = (N₀/2) · ∫_{-∞}^{∞} ϕ_h(τ) dτ = (N₀/2) · (1/2π) ∫ |H(ω)|² dω。

一个重要的观察：即使输入白噪声是"完全不相关"的（R_X(τ) ∝ δ(τ)），输出信号 Y(t) 也是相关的——因为在同一个 t₁ 和 t₂ 时刻的输出涉及了输入中的重叠部分。系统引入了"记忆"，将不相关的输入转换成了相关的输出。这也解释了为什么物理世界中几乎不存在真正不相关的随机过程——任何观测设备都有有限的带宽，观测到的信号必然带有时间相关性。

噪声整形与白化滤波器

噪声整形（Noise Shaping）

噪声整形是指用滤波器将白噪声转换为具有特定频谱形状的有色噪声的过程。这个过程在许多工程领域中都有重要应用：

  • Σ-Δ 模数转换器：通过噪声整形将量化噪声的功率从低频推到高频，然后用数字低通滤波器滤除，从而在信号频带内获得极高的信噪比。
  • 音频抖动（Dither）：在音频量化中加入经过整形的噪声，使量化噪声的频谱避开人耳最敏感的频段。
  • 测试信号生成：生成具有指定 PSD 的随机信号用于系统测试——方法就是先将高斯白噪声通过一个整形滤波器。

噪声整形的数学实现非常直接：给定目标 PSD S_target(ω)，设计一个成形滤波器 H_shape(ω) 使得 |H_shape(ω)|² = S_target(ω) / (N₀/2)。然后将白噪声通过这个滤波器，输出就是具有目标 PSD 的有色噪声。

白化滤波器（Whitening Filter）

白化滤波器是噪声整形的逆操作：输入是有色噪声，输出是白噪声。如果一个有色噪声的 PSD 为 S_X(ω) > 0，那么白化滤波器的频率响应为：

H_w(ω) = 1 / √S_X(ω)

这样 S_Y(ω) = |H_w(ω)|² · S_X(ω) = 1，实现了"白化"。

白化滤波器在通信接收机中扮演着关键角色：当接收信号受到有色噪声干扰时，先通过白化滤波器将噪声变为白噪声，然后就可以使用针对白噪声优化的经典检测算法。这就是"白化匹配滤波"的基本思想。

一个有趣的对称性：噪声整形和白化滤波器是一对互逆操作。在信息论中，这种"用滤波器调控噪声谱"的思想与"注水定理"（Water-filling Theorem）有深刻的联系——最优的功率分配策略是在信道质量好的频率上"多注水"（多分配功率），在信道质量差的频率上"少注水"。噪声整形相当于在频谱上"挖坑"或"填平"。

踩坑故事：噪声带宽不等于3dB带宽

这个故事来自我参与过的一个卫星通信接收机设计项目。接收机的中频（IF）部分使用了一个多级带通滤波器链来抑制带外噪声。在计算系统的总噪声系数时，团队中的一位工程师使用了各级滤波器的 3dB 带宽来估算进入基带的噪声功率总量。

他计算的结果显示：系统在给定带宽内的信噪比（SNR）可以达到 12 dB，满足设计指标。然而在实际硬件测试中，测得的 SNR 只有 9.8 dB。差了 2.2 dB——这个差距对于通信链路来说是致命的，因为链路预算中已经没有余量了。

经过详细排查，问题出在噪声带宽的估算上。他误用了 3dB 带宽（B_{3dB}）来替代等效噪声带宽（B_eq）。实测的滤波器链是由多个单级调谐电路级联构成的，每一级的滚降都比较缓（近似于一阶带通），因此 B_eq/B_{3dB} 比例约为 1.2 左右。误用 B_{3dB} 导致低估了约 20% 的噪声功率，折算到 dB 就是大约 0.8 dB 的误差。更严重的是，还有一级滤波器的中心频率由于元件容差发生了偏移——本来应该在对准信号的频率，但实际偏了半个带宽，导致噪声通过更多而信号通过更少——这部分又额外损失了约 1.4 dB。

教训：噪声带宽不等同于 3dB 带宽！等效噪声带宽 B_eq 是"功率等效"的概念，3dB 带宽是"幅度半功率点"的概念。两者有本质区别。精确计算噪声功率时，必须使用 ∫|H(ω)|²dω 的积分来计算，而不是用 B_{3dB} 近似。如果你的滤波器滚降不陡（如低阶 Butterworth、Chebyshev、椭圆滤波器等），B_eq 与 B_{3dB} 的差距可能达到 20%-50%。特别是在级联系统中，每一级的误差会累积放大。一个可靠的做法：始终计算 ∫|H_total(ω)|² dω 的精确值，而不是用带宽近似。

本章小结

这一章我们从频域角度完成了随机信号通过 LTI 系统的分析。核心公式 S_Y(ω) = |H(ω)|² · S_X(ω) 让复杂的双重卷积变成了简单的频域乘法，极大地简化了分析过程。我们引入了等效噪声带宽（ENB）的概念——它用一个矩形滤波器来等效实际滤波器的噪声特性，但需要注意的是 ENB 与 3dB 带宽并不等同。我们还讨论了白噪声通过 LTI 系统的输出特性，以及噪声整形与白化滤波器这一对互逆操作在工程中的广泛应用。最后通过卫星接收机噪声估算的案例警示了混用带宽概念的代价。

下一章我们将把连续时间的理论推广到离散时间域——学习平稳序列与离散系统分析。你会发现，所有的公式都有离散版本，而且由于数字系统的灵活性，离散域的分析往往比连续域更加强大和易用。
