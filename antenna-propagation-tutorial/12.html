12.1 地面反射的双射线模型

大家好，我是你们的讲师。在上一章中，我们推导了自由空间传播的Friis方程——它假设天线之间是"真空"般的理想空间，没有任何反射或散射物体。但现实中的地球表面——无论是柏油马路、水泥屋顶还是波光粼粼的湖面——都会反射无线电波。

当发射天线和接收天线都在地面以上时，存在两条传播路径：一条是直达波（direct ray），另一条是地面反射波（ground-reflected ray）。两条路径的电磁波在接收端叠加——形成干涉。这就是双射线模型（two-ray model）。

如果两条射线的相位差为0°（同相），它们相加的场强是直达波的2倍（功率是4倍），比自由空间多6 dB；如果相位差为180°（反相），场强相互抵消（在理想反射下，功率为0）。这就是信号随位置波动的根本原因。

两条路径的路径长度差Δd可以近似为（当d >> h₁, h₂时）：

Δd ≈ 2h₁h₂ / d

其中h₁和h₂分别是发射和接收天线的高度，d是水平距离。这个近似背后的几何：从发射点到接收点的直达路径长度≈d，而反射路径长度≈d + 2h₁h₂/d（在远场条件下）。

当Δd = λ/2时（即相位差π），接收功率出现第一个零点（null）。这个零点对应的距离是：

dnull = 4h₁h₂ / λ

在远距离（d >> 4h₁h₂/λ），相位差趋近于0，但是——等等——理想地面反射有一个180°的相位反转（反射系数≈ -1），所以两路信号总是反相相加。结果就是：

Pr ∝ 1 / d⁴

也就是说，在远距离区域，接收功率随距离的四次方衰减——每增加一倍距离，功率下降12 dB，而不是自由空间的6 dB！

一个感性的生活场景：为什么你在家里走来走去时，WiFi信号强度会忽上忽下？通常的解释是"距离变化"——但实际上，几米的位移不可能带来20 dB的波动。真正的原因是地面反射形成的干涉：不同位置的直达波和反射波的相位关系不同，在某些位置它们相加增强，在另一些位置相互抵消。当你在房间里走动时，你实际上是在遍历空间中的干涉条纹。这就是为什么信号强度曲线在空间中的变化像正弦波一样起伏。

需要特别指出的是，双射线模型假设地面是一个无限大的理想导电平面。实际地面的反射情况要复杂得多——反射系数取决于极化、频率、地面电参数和入射角。这正是下一节的内容。

12.2 地面反射系数的极化特性

当电磁波入射到两种介质的交界面上时，反射波的幅度和相位由菲涅尔反射系数（Fresnel reflection coefficients）决定。对于地面来说，水平极化（TE波）和垂直极化（TM波）的反射系数不同：

ΓTE = (η₂cosθᵢ - η₁cosθₜ) / (η₂cosθᵢ + η₁cosθₜ)
ΓTM = (η₂cosθₜ - η₁cosθᵢ) / (η₂cosθₜ + η₁cosθᵢ)

其中η₁和η₂分别是空气和地面的波阻抗，θᵢ是入射角，θₜ是折射角（由Snell定律决定）。

一个特别重要的工程现象是布鲁斯特角（Brewster angle）：当入射角满足某种条件时，垂直极化（TM）波的反射系数为零——反射波完全消失。这个现象只发生在平行极化波上（即电场平行于入射面）。对地面而言，垂直极化波的布鲁斯特角大约在60°-80°之间（取决于地面的介电常数）。

这个性质的实际含义：为什么电视广播（VHF/UHF）爱用水平极化，而手机通信（蜂窝网络）用垂直极化？因为垂直极化的地面反射系数有时会很小（接近布鲁斯特角时），可以减少多径干扰。但事情没这么简单——在移动通信中，手持设备的姿态变化会导致极化旋转，所以没有绝对的"最优极化"。两种极化在实际工程中各有优缺点：水平极化的地面反射更强、覆盖更均匀；垂直极化更适合近距离、非视距场景。

极化选择的一个实战教训：国内某机场部署了一套新的UHF频段地空通信系统，使用垂直极化天线。试运行期间发现，距离跑道远端（约30 km）的信号稳定性差，经常出现语音断续。经排查发现：机场周边有大面积的农田和灌区，在垂直极化下的反射系数很低（近似布鲁斯特角条件），导致反射波很弱，但地表波衰减严重，两者叠加后反而在特定距离处形成深度零点。更换为水平极化天线后，虽然地面反射更强，但多径得以稳定合并，整体链路稳定性显著改善。

12.3 粗糙表面的散射

以上讨论假设地面是光滑的镜面（specular reflection）。但实际地面——草地、农田、碎石路、建筑屋顶——都是粗糙的。

瑞利粗糙度准则（Rayleigh roughness criterion）给出了判断表面是否"光滑"的条件：

hrms < λ / (8 cos θᵢ)

其中h_rms是表面高度起伏的均方根值，θᵢ是入射角（相对于法线）。如果h_rms小于这个阈值，反射主要是相干（镜面）反射；否则，非相干（漫）散射开始占主导。

粗糙表面上的总散射场可以分为两部分：

  • 相干分量（coherent component）：能量沿镜面反射方向传播，其幅度随粗糙度增加而减小
  • 非相干分量（incoherent component）：能量散射到各个方向，形成扩散背景

工程中，在双射线模型中使用修正的反射系数来近似粗糙度的影响：

Γrough = Γsmooth · exp[-2(khrmscosθᵢ)²]

这个指数因子称为粗糙度衰减因子（roughness reduction factor）。当h_rms很小时，因子≈1；当h_rms大时，因子迅速趋近于0——也就是说，粗糙表面几乎不产生相干反射。

一个直观例子：水面在微风下（h_rms约1-3 mm）对C波段（λ≈5 cm）是"光滑"的——你在海边远眺，可以看到夕阳在水面的镜面倒影。但到了毫米波（λ≈1 mm），同样水面就变成了"粗糙"表面——倒影变得模糊不清。这就是为什么毫米波雷达看水面时回波强度远低于镜面反射的预测。

12.4 地表波传播

除了直达波和反射波之外，还有一种特殊的传播机制——地表波（surface wave / ground wave）。它是由Sommerfeld（1909年）和Norton（1937年）在理论上建立的。

地表波的能量被"束缚"在地面与空气的交界面附近，沿着地面传播。它的物理机制如下：

  • 垂直极化的电波入射到有耗地面时，会在空气中产生一个沿地面传播的波（称为"Norton surface wave"）
  • 波前倾斜（wave tilt）：地表波的等相位面不再垂直于传播方向，而是向前倾斜——能量不断从空气中注入地面，被地面吸收
  • 水平极化波不能形成地表波——因为水平极化的电场与地面平行，会在导电地面中产生短路电流，迅速衰减

地表波的一个关键特征是它的衰减速度与频率密切相关：

  • LF（30-300 kHz）和MF（300-3000 kHz）频段：地表波可以传播数百到数千公里。这是为什么AM广播（中波，530-1700 kHz）在夜间可以收听到很远城市的电台。
  • HF（3-30 MHz）：地表波衰减增大，但仍可用于几十到几百公里的距离。
  • VHF以上（>30 MHz）：地表波衰减极大，通常无法作为有效的传播机制。
有趣的故事：1912年RMS Titanic号沉没时，求救信号使用的是中波（MF）频段的Marconi无线电报。Titanic的无线电报务员Jack Phillips和Harold Bride使用火花隙发射机在500 kHz频率上发送了CQD（后改为SOS）求救信号。正是由于地表波在MF频段可以传播数百公里，Carpathia号（距Titanic约93公里）才收到了求救信号并赶来救援。如果Titanic使用的是VHF通信（如果当时有的话），由于地表波不能传播，求救信号可能根本传不出去。

12.5 地波传播的工程计算

地波传播的工程计算比自由空间复杂得多——它涉及地面电参数、频率、极化、距离的复杂函数关系。幸运的是，国际电信联盟（ITU）提供了标准化的工程工具。

ITU-R P.368标准提供了地波传播的曲线图，覆盖频率从10 kHz到30 MHz，包括不同地面类型（海水、湿地、干地）的曲线。这些曲线是以"场强 vs 距离"的形式给出的，发射功率为1 kW，天线为短单极子。

对于混合路径（mixed-path）传播——比如信号先从陆地、然后穿越海湾、再到陆地——Millington方法（1949年）提供了预测方法。核心思想是：

  • 把路径按地面类型分成若干段
  • 对每种地面类型分别查ITU-R P.368曲线得到场强
  • 用Millington的"不连续性修正"来处理不同地面交界处的场强突变
Millington方法的局限性：Millington方法基于大量实测数据的经验拟合，在大多数情况下准确度可接受（±3 dB）。但它假设地面是水平分层的，且忽略地形起伏。在复杂地形（多山、城市密集区）下误差会显著增大。现代的ITU-R P.1546和P.526模型提供了更精确的传播预测。

下面是一个简化版本的地波场强计算Python实现：

import numpy as np

def ground_wave_field_strength(f, d, sigma, epsilon_r, P_t=1000):
    """
    计算地波传播的场强 (简化模型)
    基于ITU-R P.368的近似解析公式

    参数:
        f        : 频率 (Hz)
        d        : 距离 (m)
        sigma    : 地面电导率 (S/m)
        epsilon_r: 地面相对介电常数
        P_t      : 发射功率 (W)，默认1 kW
    返回:
        E        : 场强 (dBμV/m)
    """
    c = 3e8
    lam = c / f
    eta0 = 120 * np.pi  # 自由空间波阻抗

    # 数值距离 p (Sommerfeld数值距离的简化)
    # 对于垂直极化地波
    p = (np.pi * d) / (lam) * np.sqrt((epsilon_r - 1j * sigma/(2*np.pi*f*8.85e-12))) / \
        (np.sqrt(epsilon_r - 1j * sigma/(2*np.pi*f*8.85e-12)) - 1)
    # 实际数值距离更复杂，这里仅做演示

    # 自由空间场强 (作为参考)
    E0 = np.sqrt(30 * P_t) / d
    E0_dBuV = 20 * np.log10(E0 * 1e6)

    # 简化衰减函数 (A(p)经验近似)
    A = 1.0 / (1 + 0.3 * p + 0.02 * p**2)

    E = E0 * A
    E_dBuV = 20 * np.log10(E * 1e6)
    return E_dBuV

# 示例对比：不同频率下的地波衰减
frequencies = [500e3, 1e6, 5e6, 10e6]  # 500 kHz ~ 10 MHz
d = 100000  # 100 km
sigma = 0.005  # 干地
er = 5

print("频率       | 100 km 场强 (dBμV/m)")
print("-" * 35)
for f in frequencies:
    E = ground_wave_field_strength(f, d, sigma, er)
    freq_str = f"{f/1e3:.0f} kHz" if f < 1e6 else f"{f/1e6:.1f} MHz"
    print(f"{freq_str:8s} | {E:.1f}")
# 输出示例:
# 频率       | 100 km 场强 (dBμV/m)
# -----------------------------------
# 500 kHz    | 57.3
# 1.0 MHz    | 44.1
# 5.0 MHz    | 18.6
# 10.0 MHz   | 6.2

# 结论: 频率越高，地波衰减越严重
工程总结：地波传播的几个要点要牢记：
  • LF/MF频段（<3 MHz）是地波传播的主力频段。AM广播、航空导航信标（NDB）、海上遇险安全频率（GMDSS）都在使用地波。
  • 垂直极化是地表波的唯一选择——水平极化不能形成地表波。
  • 地波衰减随频率迅速增加：频率每增加10倍，有效传播距离大约缩减到1/10。
  • 海水上的地波衰减最小，干地次之，城市密集区衰减最大。

这一章我们看到了地面如何影响电波传播：双射线模型告诉我们信号在空间中的起伏振荡，极化特性决定了反射系数的差异，粗糙表面的散射削弱了镜面反射，而地表波提供了一种沿着地面"爬行"的特殊传播方式。

反射和地表波之外，电波还有另一种翻越障碍物的方式——绕射（diffraction）。当遇到山峰、建筑物等尖锐障碍物时，电波不会简单地被挡住——它会像光遇到刀刃一样"弯折"进入阴影区。在下一章中，我们将学习惠更斯-菲涅尔绕射原理、刃峰绕射模型、球面绕射和多径衰落——这些知识将把你从"视线传播工程师"升级为"复杂环境传播工程师"。
