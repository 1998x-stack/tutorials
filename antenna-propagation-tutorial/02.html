2.1 从库仑定律到麦克斯韦的飞跃

大家好，我是你们的讲师。今天我们要攻克整个天线与电波传播课程中最为核心的一章——Maxwell方程组。如果说天线是无线系统的心脏，那么Maxwell方程组就是描述这颗心脏搏动的物理定律。

在19世纪中叶之前，电学和磁学是两个独立的学科。库仑定律描述了电荷之间的静电力，安培定律描述了电流产生磁场，法拉第定律则揭示了变化的磁场产生电场。但James Clerk Maxwell敏锐地意识到，这些定律之间存在一个不对称性：既然变化的磁场能产生电场（法拉第电磁感应），那么反过来——变化的电场是否也应该能产生磁场？

1861年至1865年间，Maxwell在已有的安培定律中发现了数学上的不一致性。考虑一个正在给电容器充电的电路：在电容器极板之间，没有传导电流通过，但磁场却实实在在地存在。Maxwell引入了位移电流（displacement current）的概念——变化的电通量密度D相当于一种电流密度——从而补全了安培定律，使整个方程组达到了数学上的自洽。这个修正的意义远比修补一个小bug深远得多：它预言了电磁波的存在，并且把光统一进了电磁理论中。

Maxwell的伟大洞察：如果变化的磁场产生电场，变化的电场也产生磁场，那么二者相互激发、交替产生，就能以波的形式在空间中传播——这就是电磁波。这一思想实验如此简洁，却又如此深刻，彻底改变了人类文明的轨迹。

Maxwell方程组的积分形式如下，它描述了电磁场在宏观尺度上的行为：

高斯定律（电场）：∮S D · dS = ∫V ρv dV
——电场线从正电荷发出，终止于负电荷。电荷是电场的源。

高斯磁定律：∮S B · dS = 0
——自然界中不存在"磁单极子"。磁感线是闭合的，没有起点和终点。

法拉第电磁感应定律：∮C E · dl = -d/dt ∫S B · dS
——变化的磁场在闭合回路中感应出电动势（电场）。这是发电机、变压器的基础。

安培-麦克斯韦定律：∮C H · dl = ∫S (J + ∂D/∂t) · dS
——电流和变化的电场都产生磁场。位移电流∂D/∂t就是Maxwell的天才修正。

记忆口诀：高斯电场看电荷，高斯磁场无单极；法拉第变磁生电，安培麦氏流生磁。

2.2 麦克斯韦方程组的微分形式

积分形式虽然物理意义直观，但在分析连续分布的场问题时并不方便。利用矢量分析中的散度定理（Gauss's theorem）和斯托克斯定理（Stokes's theorem），我们可以将面积分和线积分转化为体积分和面积分，从而得到微分形式。

散度定理：∮S F · dS = ∫V (∇ · F) dV
斯托克斯定理：∮C F · dl = ∫S (∇ × F) · dS

应用这两个定理，我们得到微分形式的Maxwell方程组：

∇ · D = ρv   （电场的散度等于电荷密度）
∇ · B = 0   （磁场的散度恒为零）
∇ × E = -∂B/∂t   （电场的旋度等于负的磁感应强度变化率）
∇ × H = J + ∂D/∂t   （磁场的旋度等于电流密度加位移电流密度）

微分形式的优雅之处在于：四个方程用四个简单的矢量偏微分方程就描述了电磁场的全部行为。左侧的散度∇·描述了场在一点上的"发散"程度，旋度∇×描述了场的"旋转"程度。

对比维度 积分形式 微分形式
数学工具 面积分、线积分 散度、旋度
适用范围 宏观、大尺度边界 逐点描述、连续媒质
物理意义 通量和环量与源的关系 场的空间变化率与源的关系
方便处理 边界条件、对称问题 数值计算、波动方程推导
典型应用 高斯定律求电场 有限元法(FEM)求解

2.3 波动方程与电磁波的预言

现在，让我们做一件Maxwell在1865年做过的事情：将四个方程合并，推导出电磁波的存在。

在自由空间（无源区域）中，ρv = 0，J = 0。对法拉第定律的旋度方程两边取旋度，并代入安培-麦克斯韦定律：

∇ × (∇ × E) = ∇ × (-∂B/∂t) = -∂/∂t (∇ × B) = -μ₀ε₀ ∂²E/∂t²

利用矢量恒等式 ∇ × (∇ × E) = ∇(∇ · E) - ∇²E，且 ∇ · E = 0（自由空间无电荷），得到：

∇²E - μ₀ε₀ ∂²E/∂t² = 0

这就是经典的三维波动方程！它的形式与弦振动方程完全一致，只是波速变成了：

c = 1 / √(μ₀ε₀)

当Maxwell代入当时的真空介电常数ε₀和磁导率μ₀的测量值时，他得到了一个惊人的结果：c ≈ 3.1 × 10⁸ m/s——这与当时已知的光速测量值惊人地一致！Maxwell由此做出了物理学史上最伟大的论断之一：光就是电磁波。

历史瞬间：据说当Maxwell得出这个结果时，他几乎不敢相信自己的眼睛。他在《电磁通论》（A Treatise on Electricity and Magnetism, 1873）中写道："我们很难避免得出这样的结论：光是由电磁现象在媒质中的横波振动构成的。"——这是人类历史上首次用理论将光学和电磁学统一起来。

波动方程的一个基本解是均匀平面波（uniform plane wave）：

E(r, t) = E₀ e^(j(ωt - k · r))

其中E₀是振幅矢量，ω是角频率，k是波矢量（大小为k = 2π/λ，方向为传播方向），r是位置矢量。这个解描述了一个以速度c沿k方向传播的正弦电磁波。电场E、磁场H和传播方向k三者相互正交，构成右手螺旋关系。

关于符号约定：我们在本课程中使用 e^(jωt) 的时间因子约定，这是天线和微波工程中最常用的形式。在光学中更常用 e^(-iωt) 的约定——注意这会导致相位表达方式的差异，但物理本质完全相同。

2.4 时谐电磁场的相量表示法

在绝大多数天线和电波传播问题中，我们关心的是时谐（time-harmonic）信号——即单一频率的正弦稳态信号。对这类问题，相量（phasor）表示法能极大简化分析。

相量法的核心思想：对于时谐场 E(r, t) = Re[Ẽ(r) e^(jωt)]，时间导数 ∂/∂t 变为 jω 乘法。这样一来，Maxwell方程组中的偏微分方程就变成了代数方程：

∇ × Ẽ = -jωμH̃
∇ × H̃ = J̃ + jωεẼ

更妙的是，复介电常数（complex permittivity）允许我们用一个复数来同时描述介质的介电特性和导电损耗：

εc = ε' - jε'' = ε₀(εr' - jεr'')

损耗角正切（loss tangent）定义为：tanδ = ε'' / ε' = σ / (ωε')

tanδ是衡量介质损耗的关键参数。空气的tanδ ≈ 0，而FR4 PCB板材的tanδ ≈ 0.02——这意味着在毫米波频段，FR4的介质损耗会非常显著，这是为什么5G毫米波电路需要使用低损耗材料（如Rogers 4350B，tanδ ≈ 0.0037）的原因。

在相量表示法下，均匀平面波的两个核心参数是：

波数（wave number）：k = ω√(με) = 2π/λ
——描述单位长度上相位的变化量，是空间角频率。

本征阻抗（intrinsic impedance）：η₀ = √(μ₀/ε₀) ≈ 377 Ω
——在自由空间中，电场与磁场的比值。这意味着对于均匀平面波，|E|/|H|始终等于377 Ω。这个数值在天线设计和链路预算中频繁出现。

那么在实际介质中呢？η = √(μ/ε) = η₀√(μr/εr)。例如，在水中（εr ≈ 81），η ≈ 377/9 ≈ 42 Ω——这就是为什么水下的电磁波传播特性与空气中截然不同。

踩坑故事：我曾见过一个天线设计团队，在仿真中使用自由空间的本征阻抗377 Ω来计算微带贴片天线的输入阻抗，结果仿真与实测始终对不上。原因很简单：微带天线的辐射边位于介质基板（εr ≈ 4.4的FR4）和空气的交界面上，实际的辐射阻抗应该使用有效介电常数εeff修正后的η值。这个疏忽让团队浪费了整整两周来排查问题。最终发现：一个377 Ω的默认值，在不同的介质环境中并不适用。

2.5 均匀平面波的传播特性

均匀平面波是最简单的电磁波形式，但它揭示了电磁波传播的所有基本特性。让我们一一考察：

波长与相速度

波长：λ = vp / f = 2π / k
——同一时刻，相邻两个同相位点之间的距离。

相速度：vp = ω / k = 1 / √(με)
——等相位面移动的速度。在真空中vp = c，在介质中vp = c/√εr。

群速度：vg = dω/dk
——能量和信息传递的速度。在无色散介质中vg = vp；在色散介质中两者不同。

一个重要的直觉训练：当电磁波从空气进入水时，频率不变（频率是源的属性），波速降低为真空中的1/9，波长相应缩短为真空中的1/9。这意味着一个900 MHz的手机信号在水中波长为：λ = (3×10⁸/9) / (900×10⁶) ≈ 3.7 cm，在空气中则是33 cm。所以潜艇通信必须使用极低频（ELF, 30-300 Hz），此时波长可达数千公里，才能穿透海水。

坡印廷矢量与功率流

电磁波的功率流由坡印廷矢量（Poynting vector）描述：

S = E × H  （瞬时坡印廷矢量，单位：W/m²）

对于时谐场，时间平均坡印廷矢量为：

Savg = (1/2) Re[Ẽ × H̃*]

对于均匀平面波：Savg = |E₀|²/(2η) · k̂ ——功率完全沿传播方向流动，没有无功成分。

物理直觉：坡印廷矢量告诉你"能量往哪里流"。天线辐射的总功率就是对包围天线的封闭曲面进行坡印廷矢量积分。在天线测量中，我们通过近场扫描得到E和H的幅度和相位，然后用坡印廷定理计算出远场方向图。

趋肤深度

当电磁波进入导电媒质（如铜、铝）时，很快被衰减。幅度衰减为表面值的1/e（约37%）时的穿透深度称为趋肤深度（skin depth）：

δ = 1 / √(πfμσ)

对于铜（σ = 5.8×10⁷ S/m, μ = μ₀）：

  • 50 Hz（工频）：δ ≈ 9.3 mm —— 电力线缆可以用较粗的铜导线
  • 1 MHz（AM广播）：δ ≈ 66 μm —— 需要镀银或镀锡
  • 1 GHz（手机通信）：δ ≈ 2.1 μm —— 仅表面导电
  • 28 GHz（5G毫米波）：δ ≈ 0.4 μm —— 需要极其光滑的表面

这就是为什么高频电路使用微带线（microstrip）而非粗导线，且印制板表面镀金或镀银的原因——高频电流只分布在导体表面极薄的一层内，内部导体几乎"浪费"了。

让我们用Python写一个简单的均匀平面波场计算：

import numpy as np

def plane_wave_field(f, z, t, E0=1.0):
    """
    计算均匀平面波的电场和磁场瞬时值
    参数:
        f   : 频率 (Hz)
        z   : 传播方向上的位置 (m)
        t   : 时间 (s)
        E0  : 电场振幅 (V/m)
    返回:
        Ex  : 电场 x 分量 (V/m)
        Hy  : 磁场 y 分量 (A/m)
    """
    c = 3e8
    eta0 = 377.0            # 自由空间本征阻抗 (Ω)
    omega = 2 * np.pi * f   # 角频率
    k = omega / c           # 波数 (rad/m)

    # 假设电场沿 x 方向极化，沿 +z 方向传播
    # E(z,t) = x̂ · E0 · cos(ωt - kz)
    Ex = E0 * np.cos(omega * t - k * z)
    # H(z,t) = ŷ · E0/η0 · cos(ωt - kz)
    Hy = E0 / eta0 * np.cos(omega * t - k * z)

    return Ex, Hy

# 示例：1 GHz 信号沿 z 轴传播，在 t=0 时计算 z=0~3λ 的波形
f = 1e9
lam = 3e8 / f
z_points = np.linspace(0, 3*lam, 301)
Ex, Hy = plane_wave_field(f, z_points, t=0)

# 计算平均坡印廷矢量幅度
S_avg = 0.5 * (1.0**2) / eta0
print(f"频率: {f/1e9:.1f} GHz")
print(f"波长: {lam*100:.1f} cm")
print(f"平均功率密度: {S_avg*1000:.2f} mW/m²")
print(f"在 z = {lam:.4f} m 处:")
print(f"  E 场峰值: {Ex.max():.2f} V/m")
print(f"  H 场峰值: {Hy.max():.4f} A/m")
print(f"  E/H 比值: {Ex.max()/Hy.max():.1f} Ω (应等于 {eta0} Ω)")

跑一下这段代码，你会发现电场与磁场的比值始终是377 Ω——这不是巧合，而是自由空间本征阻抗的直接体现。记住：在自由空间中，电场和磁场是"锁定"的，不能独立变化。

本章我们从库仑定律出发，跟随Maxwell的思维轨迹，一步步构建了完整的电磁场理论框架。我们看到了Maxwell如何凭借深刻的物理直觉——位移电流假说——实现了电磁学的统一，并推导出了波动方程，预言了电磁波的存在。我们还学习了相量表示法这一强大的分析工具，以及均匀平面波的传播特性——波长、相速度、群速度、坡印廷矢量和趋肤深度。
