开场：天线——电磁波的"释放之门"

在第17章中，我们学习了如何用传输线和波导将电磁能量引导到指定位置。但传输线中的电磁波是"束缚波"——能量被约束在导体周围，无法辐射到自由空间。而通信的最终目标恰恰是将信号能量从一个地方无线地传送到另一个地方。这个时候就需要天线（Antenna）登场了。

天线的本质是一个阻抗变换器和模式转换器：它把传输线中的 TEM 波（束缚在导体附近）转换为自由空间中的球面波（辐射到远区），同时完成特性阻抗 Z₀ 到自由空间波阻抗 η₀ 的过渡。

从赫兹1887年实验中的简单偶极子开始，到如今手机中集成的数十个天线单元，天线技术走过了130多年的历程。本章将从辐射的物理本质出发，系统学习天线的基本参数、常见天线类型、链路预算和相控阵原理。

推迟势

要理解电磁辐射，首先要回到辐射的源头——时变电流分布。第8章我们学过静磁场的矢势：A(r) = (μ₀/4π)∫(J(r')/R)dV'。但这是静态形式，它假设电流变化的信息是瞬时传递的。

在时变情况下，信息传播速度是有限的（光速 c）。源点的电流变化需要经过时间 R/c 才能影响到场点。因此，静态矢势需要加入推迟时间（Retarded Time）t' = t − R/c：

A(r, t) = (μ₀/4π) ∫ (J(r', t − R/c) / R) dV'

这就是推迟矢势（Retarded Vector Potential）。推迟标势的形式完全对称：

V(r, t) = (1/(4πε₀)) ∫ (ρ(r', t − R/c) / R) dV'

从推迟势出发，可以通过 B = ∇×A 和 E = −∇V − ∂A/∂t 得到辐射电场和磁场。推迟势虽然是计算辐射场的基础，但在工程中直接用它求解复杂的天线结构比较困难——我们通常先求解电流分布，然后用数值方法计算辐射场。

🔑 核心洞察：为什么时变电流会辐射？
静态情况：匀速运动的电荷产生恒定磁场，没有辐射。
时变情况：电荷做加速运动或电流随时间变化，产生互相耦合的时变 E 和 B——在远区形成自持传播的电磁波。
电磁辐射的物理本质是：加速度产生电磁波。
这个结论不仅对自由电荷成立，对偶极子中的振荡电流同样成立。赫兹偶极子中的电子在两端之间来回振荡，经历不断的方向变化（加速度），因此源源不断地辐射能量。

赫兹偶极子

赫兹偶极子（Hertzian Dipole）是最简单的辐射体——一段长度为 dl（远小于波长）、载有均匀时谐电流 I(t)=I₀cos(ωt) 的短导线。虽然它在工程上不实用（辐射效率太低），但它是天线理论的"氢原子"——理解它就能理解一切复杂天线的辐射机理。

辐射场

在球坐标系中（偶极子沿 z 轴放置），赫兹偶极子的辐射场为：

  • E_θ = j (η₀ I₀ dl k / (4πr)) sinθ e^{−jkr}
  • H_φ = j (I₀ dl k / (4πr)) sinθ e^{−jkr}
  • E_r 和 H_r 在远区可忽略（与 1/r² 成正比衰减）

关键观察：

  • 远场 E_θ 和 H_φ 与 1/r 成正比——这就是辐射波的特征（能量能够传播到无穷远）。
  • 场幅度与 sinθ 成正比——辐射有方向性，在偶极子轴向（θ=0,π）辐射为零，在赤道面（θ=90°）最强。
  • E 和 H 同相、相互正交、且垂直于 r 方向——远区是 TEM 波。

辐射方向图

赫兹偶极子的方向图呈甜甜圈（donut）形状。归一化方向函数为 F(θ) = sinθ。在 E 平面（包含偶极子的平面，即 yz 面）上，方向图是 8 字形；在 H 平面（赤道面，即 xy 面）上，方向图是完美的圆形（全向）。

辐射功率和辐射电阻

远区坡印廷矢量沿径向积分，得到总辐射功率：

P_rad = 80π² (I₀ dl / λ)²

将辐射功率类比为电阻消耗的功率 P = I²R_rad，得到辐射电阻：

R_rad = 80π² (dl/λ)²

对于 dl = λ/50，R_rad ≈ 0.32Ω——这个电阻远小于导线的欧姆电阻（约几Ω）。这就是短偶极子效率极低的原因：大部分输入功率被欧姆损耗消耗掉了，而不是辐射出去。

天线基本参数

评价天线性能需要一套标准化的参数体系：

参数 符号 定义/含义 典型值
方向图 F(θ,φ) 辐射场幅度随空间角度的分布 各向同性、全向、定向
方向性系数 D 最大辐射方向的功率密度与平均值的比值 偶极子≈1.64(2.15dBi)
增益 G 方向性系数×效率：G = η_e·D 偶极子≈2.15dBi
半功率波束宽度 HPBW 主瓣功率降至一半时的角度宽度 偶极子E面≈78°
输入阻抗 Z_in 天线端口的输入阻抗 = R_rad+R_loss + jX 半波振子≈73+j42.5Ω
带宽 BW 满足 VSWR < 2 的频率范围 贴片天线≈3–5%
极化 — 天线辐射波的极化方式（线/圆/椭圆） 取决于天线结构
有效面积 A_eff 接收天线从入射波中收取功率的有效面积 A_eff = G·λ²/(4π)

常见天线类型

半波偶极子（Half-Wave Dipole）

半波偶极子由两根总长为 λ/2 的导线构成（每根 λ/4），在中心馈电。它是最实用的基本天线：

  • 电流分布近似正弦：I(z) = I₀ cos(πz/λ)，在端点为零、在中心最大。
  • 输入阻抗：Z_in ≈ 73 + j42.5 Ω。去掉电抗部分（缩短约5%长度）可得到纯阻性 73Ω——与 75Ω 同轴电缆匹配良好。
  • 增益：D ≈ 1.64（2.15 dBi）。
  • HPBW：E面约78°，H面全向。

单极天线（Monopole over Ground）

在地平面上垂直放置一根 λ/4 导体，利用地平面的镜像效应可等效为自由空间中的 λ/2 偶极子。输入阻抗为偶极子的一半（≈36Ω），增益约 5.15 dBi（实际取决于地平面大小）。广泛应用于车载天线、基站天线。

八木-宇田天线（Yagi-Uda）

由一个有源振子（驱动单元）、一个反射器和多个引向器组成。利用寄生单元的耦合和谐振，实现较高的方向性增益（典型值 10–15 dBi）。这是家庭电视接收最常见的天线类型。

微带贴片天线（Microstrip Patch）

一块矩形金属贴片在介质基板上方，下面有参考地平面。特征：

  • 外形扁平、可贴装在任意曲面上。
  • 增益中等（5–8 dBi），带宽窄（3–5%）。
  • 易于阵列化和圆极化设计。
  • 广泛应用：手机、GPS、WiFi路由器、卫星通信。

抛物面天线（Parabolic Dish）

利用抛物面的几何聚焦特性，将馈源喇叭发出的球面波转换为平面波束。增益与口径面积成正比：G ≈ 4πA/λ² · η_aperture（口径效率通常 50–70%）。

💡 天线放置的位置效应
天线附近的物体会显著改变其辐射特性——这就是所谓的天线耦合和外壳效应。
• 金属物体靠近天线会：改变谐振频率、降低辐射效率、扭曲方向图
• 人体组织（含水率高、介电常数大）靠近天线会：吸收大量辐射能量（SAR问题）、严重失谐
• 地平面尺寸影响：有限地平面会使单极天线方向图向上倾斜（波束倾斜效应）
这就是为什么手机天线设计必须将天线与外壳、电池、屏幕、摄像头模组、手掌握持位置一起仿真优化。单独计算天线在自由空间中的性能没有任何意义——"天线"从来不是单独工作的，它永远是"天线+环境"这个系统的一部分。

Friis传输公式

在无线通信系统中，我们需要知道：给定发射功率和天线增益，接收端能收到多少功率？这就是 Friis传输公式（Friis Transmission Equation）解决的问题。

P_r / P_t = G_t · G_r · (λ / (4πR))²

其中：

  • P_t：发射功率，P_r：接收功率
  • G_t：发射天线增益，G_r：接收天线增益
  • R：发射天线与接收天线之间的距离
  • λ：工作波长

(λ/(4πR))² 是自由空间路径损耗（Free-Space Path Loss, FSPL）。用对数表示为：

FSPL(dB) = 20log₁₀(R) + 20log₁₀(f) + 92.45

（R 的单位为 km，f 的单位为 GHz）

工程单位转换：

  • P_r(dBm) = P_t(dBm) + G_t(dBi) + G_r(dBi) − FSPL(dB) − 附加损耗
  • dBm：功率相对于1mW的分贝值。0dBm = 1mW，30dBm = 1W。
  • dBi：增益相对于各向同性辐射体的分贝值。
  • dBd：增益相对于半波偶极子的分贝值。dBi = dBd + 2.15。

示例：WiFi路由器（P_t=20dBm, G_t=2dBi）在 2.4GHz、距离100米处、接收天线增益 0dBi：

FSPL = 20log₁₀(0.1) + 20log₁₀(2.4) + 92.45 = −20 + 7.6 + 92.45 ≈ 80 dB

P_r = 20 + 2 + 0 − 80 = −58 dBm ≈ 1.6 nW

这就是现代WiFi接收芯片灵敏度（通常 −80 到 −95 dBm）的设计依据——即使经过多堵墙的衰减，依然有足够的余量保证可靠的通信。

相控阵入门

相控阵天线（Phased Array）是一种通过控制天线阵列各单元的馈电相位来实现无机械运动波束扫描的技术。它是现代雷达和5G毫米波通信的核心。

阵列因子

考虑 N 个相同的天线单元沿 x 方向等间距 d 排列，每个单元的馈电相位依次相差 Δφ。那么阵列的总方向图可以表示为：

F_total(θ) = F_element(θ) · AF(θ)

其中 AF 是阵列因子（Array Factor）：

AF(θ) = sin[N(ψ/2)] / [N sin(ψ/2)]

ψ = kd·sinθ + Δφ

这是方向图乘积定理（Pattern Multiplication Principle）——阵列总方向图等于单元方向图乘以阵列因子。

波束扫描

通过调节相邻单元的相位差 Δφ，主波束指向角 θ₀ 满足：

sinθ₀ = −Δφ / (kd)

Δφ = −kd·sinθ₀

这意味着：只要改变移相器的相位，主波束就可以在空间中快速扫描，不需要任何机械转动。

⚠ 相控阵的"幽灵"——栅瓣
阵列因子的周期性结构可能导致在非期望方向上出现幅度与主瓣相同的波束——称为栅瓣（Grating Lobes）。栅瓣的出现条件是：元件间距 d 在某个扫描角下使得相邻单元的相位差达到 2π 的整数倍。
防止栅瓣出现的条件（最大扫描角 θ_max）：
d / λ < 1 / (1 + |sinθ_max|)
对于不扫描（θ_max=0）的阵列：d < λ（间距可以小于波长）
对于扫描到 ±60° 的阵列：d < λ / (1 + sin60°) = λ / 1.866 ≈ 0.536λ
这就是几乎所有相控阵系统都采用 λ/2 间距的原因——它保证了从端射到端射的整个前半空间无栅瓣。

5G毫米波相控阵实例

在5G NR FR2频段（24–52 GHz），手持设备需要使用相控阵实现波束赋形（Beamforming）。一个典型的手机毫米波模组包含：

  • 4×4 或 8×8 的贴片天线阵列（间距约 λ/2 ≈ 3mm @ 28GHz）
  • 每个天线单元后接一个移相器和放大器（称为"通道"）
  • RFIC（射频集成电路）集成所有通道，通过数字接口控制各通道相位
  • 波束宽度约 10–20°，扫描范围 ±45° 到 ±60°

Python示例：偶极子方向图极坐标绘制

import numpy as np
        import matplotlib.pyplot as plt

        # 半波偶极子 E面方向图
        theta = np.linspace(0, 2*np.pi, 361)
        # 归一化方向函数
        F = np.abs(np.cos(np.pi/2 * np.cos(theta)) / np.sin(theta))
        # 处理 theta=0,pi 处的奇点
        F = np.nan_to_num(F, nan=0.0)

        plt.figure(figsize=(6, 6))
        ax = plt.subplot(111, projection='polar')
        ax.plot(theta, F, linewidth=2, label='半波偶极子 E面')
        ax.set_theta_zero_location('N')
        ax.set_theta_direction(-1)
        ax.set_thetamin(0)
        ax.set_thetamax(180)
        ax.set_title('半波偶极子 E面方向图', va='bottom')
        ax.legend(loc='upper right')
        plt.tight_layout()
        plt.show()

        # 计算关键参数
        # HPBW 查找（半功率点 ≈ 0.707 幅度）
        idx_half = np.where(np.abs(F - 0.707) < 0.01)[0]
        if len(idx_half) >= 2:
            HPBW = np.rad2deg(theta[idx_half[-1]] - theta[idx_half[0]])
            print(f"HPBW ≈ {HPBW:.1f}°")
        # 方向性系数估算
        D = 2 / (np.trapz(F**2 * np.sin(theta), theta))
        print(f"D ≈ {D:.2f} ({10*np.log10(D):.2f} dBi)")
🔑 工程现实：手机天线的"方寸之舞"
智能手机是现代天线工程最极致的挑战。在不到 150×70mm 的空间内，需要集成以下天线：
• 主天线（蜂窝：2G/3G/4G/5G NR Sub-6GHz）——通常利用金属边框
• 分集天线——提高接收信噪比
• MIMO天线——4×4或8×8 MIMO需要多个天线单元
• GPS/WiFi/蓝牙天线——2.4/5/6GHz
• 毫米波相控阵模组——28/39GHz
• NFC线圈天线——13.56MHz（近场）
• 无线充电线圈
这些天线必须在指甲盖大小的空间中共存，同时满足：效率 > 40%、隔离度 > 10dB、SAR（人体比吸收率）合规。
更棘手的是，用户的手握持姿势会显著改变天线的辐射环境。苹果的"天线门"（iPhone 4）就是握持导致天线短路、信号骤降的著名案例。现代手机天线设计必须从第一天就把"手"作为天线的一部分来仿真——这就是所谓的人头+人手模型（SAM Head + Hand Phantom）测试。
