第二十章 单自由度体系强迫振动

上一章我们研究了"结构自己唱歌"——自由振动。但现实中的结构从不会"自娱自乐"，它们总是被各种外力"强迫着跳舞"：风吹、地震、车辆、机械振动。当外力的节奏恰好和结构的"舞步"合拍时，一场危险的共振就可能上演。

1999年，韩国首尔的三丰百货店在正常营业中轰然倒塌，导致502人遇难。事故的直接原因是结构设计问题和违规改建，但在调查过程中人们发现了一个有趣却令人不安的细节：在倒塌前的几分钟里，商场内的顾客和员工都感受到了异常剧烈的"晃动"，货架上的商品纷纷掉落。事后分析表明，空调机组的运行频率与大楼某阶自振频率接近，产生了持续的动力放大效应。虽然这并非倒塌的主因，但振动的异常本应是提前疏散的信号——可惜没有人意识到这一点。

这个故事告诉我们：理解强迫振动、识别共振信号，不仅是理论考试要考的内容，更是关乎生命的工程能力。本章我们就来深入学习单自由度体系在强迫振动下的行为，包括简谐荷载、一般荷载下的杜哈梅积分，以及工程中如何利用隔振技术"驯服"有害振动。

简谐荷载作用下的强迫振动

简谐荷载（Harmonic Load） 是最简单、也是最基本的一种动荷载形式：p(t) = p₀·sin(θt)，其中p₀是荷载幅值，θ是荷载频率。

为什么简谐荷载这么重要？有三个原因：第一，很多真实的动荷载（如旋转机械的离心力、波浪力）本身就是简谐或接近简谐的；第二，任意周期荷载可以通过傅里叶级数展开为一系列简谐分量的叠加；第三，简谐荷载的响应解揭示了动力放大和共振的本质规律——这是理解所有复杂动荷载响应的基础。

运动方程及其解

    m·ü(t) + c·u̇(t) + k·u(t) = p₀·sin(θt)

    除以m，令 ω² = k/m，ζ = c/(2mω)：
    ü(t) + 2ζω·u̇(t) + ω²·u(t) = (p₀/m)·sin(θt)

这个微分方程的全解由两部分组成：

    u(t) = u_h(t) + u_p(t)

    齐次解 u_h(t)：对应自由振动，随时间衰减（有阻尼时）
    特解 u_p(t)：对应稳态强迫振动，与荷载同频率但可能有相位差

工程中我们最关心的是稳态响应（Steady-State Response）——因为齐次解部分在阻尼作用下会很快衰减到可忽略的程度。稳态解的形式为：

    u_p(t) = C·sin(θt - φ)

    其中：
    C = p₀/k · 1/√[(1-β²)² + (2ζβ)²]    —— 稳态振幅
    φ = arctan[2ζβ/(1-β²)]                 —— 相位角
    β = θ/ω                                 —— 频率比（Frequency Ratio）
理解相位角φ：相位角φ表示响应滞后于荷载的角度。当β<1（荷载频率低于自振频率）时，φ接近0°，响应与荷载基本同步。当β=1（共振）时，φ=90°——此时荷载最大时速度最大而非位移最大。当β>1（荷载频率高于自振频率）时，φ接近180°，响应与荷载反相——即荷载向下时结构向上运动。

动力放大系数

我们定义动力放大系数（Dynamic Magnification Factor, DMF） 为：

    DMF = 最大动位移 / 静位移 = 1 / √[(1-β²)² + (2ζβ)²]

    其中静位移 u_st = p₀/k

DMF是结构动力学中最重要的无量纲参数之一。它告诉我们：同样大小的荷载，作用在结构上产生的动位移是静位移的多少倍？

下面给出不同阻尼比下DMF随频率比β的变化：

频率比 β ζ=0（无阻尼） ζ=0.02 ζ=0.05 ζ=0.10 ζ=0.20 说明
0.0 1.000 1.000 1.000 1.000 1.000 静力区
0.5 1.333 1.327 1.317 1.299 1.266 准静力区
0.9 5.263 4.898 4.124 3.194 2.216 接近共振
1.0 ∞（发散） 25.00 10.00 5.000 2.500 共振点
1.5 0.800 0.799 0.797 0.793 0.781 越过共振
2.0 0.333 0.333 0.333 0.333 0.332 惯性区

从这个表格中，我们可以提炼出几个极其重要的工程结论：

  • 共振（Resonance）：当β=1时，DMF达到最大值。对于无阻尼体系，DMF无限大（理论上振幅会随时间线性增长到无穷大）；对于实际结构，DMF≈1/(2ζ)。一个阻尼比5%的结构在共振时的DMF≈10——这意味着同样的力作用在结构上，动位移是静位移的10倍。
  • 阻尼的"削峰"作用：阻尼在远离共振时作用不大（β=0.5或β=2时不同阻尼比的DMF几乎没有差别），但在共振附近阻尼的"削峰"效果非常显著。这就是为什么抗震设计中阻尼比是核心参数。
  • 三个频段分区：β<0.5为准静力区（动力效应不明显），0.5≤β≤1.5为共振区（DMF显著放大），β>1.5为惯性区（DMF小于1，结构"跟不上"荷载的变化）。

共振现象分析

共振是结构动力学中最重要也最危险的现象。我们来深入看看共振前后的振动特征。

共振的本质：当荷载频率θ等于结构自振频率ω时，外力不断把能量输入结构，而阻尼不断耗散能量。当输入=输出时，振幅稳定在一个有限值——对于无阻尼体系，能量只进不出，振幅无限增长直到结构破坏。

    共振时的振幅：
    C_res = p₀/k · 1/(2ζ) = u_st / (2ζ)

    或者用DMF表达：
    DMF_max ≈ 1/(2ζ)      （当ζ较小时）

    共振时相位角φ = 90°

共振建立的"时间"效应：共振不是瞬间达到最大振幅的——它需要一个建立过程。对于低阻尼结构，共振振幅的增长比较慢。这意味着：如果一个结构在共振条件下工作但只持续了很短时间（比如一阵突风），它可能还来不及达到理论上的最大振幅。这就是为什么有些结构能"侥幸"躲过共振破坏——不是因为共振不危险，而是因为"共振时间不够长"。

工程启示：如果你在设计中发现结构的自振频率与某种常见荷载的频率接近，你有几个"调频"选项：（1）增大结构刚度（提高ω），让β减小；（2）增加质量（降低ω，但要注意这会同时增加地震力）；（3）增加阻尼（虽然不改变频率，但可大幅降低共振峰值）；（4）改变荷载频率（如调整机器转速）。其中第（1）和第（3）是最常用的方案。

一般荷载下的杜哈梅积分

实际的动荷载很少是完美的简谐函数——地震地面加速度是剧烈的不规则抖动，风荷载是随机的脉动，爆炸荷载是瞬间的冲击。对于这些任意形式的荷载，我们需要更强大的工具——杜哈梅积分（Duhamel Integral）。

基本思想

杜哈梅积分的思想非常巧妙：把一个任意荷载p(t)看作一系列瞬时冲量的叠加，每个微冲量p(τ)·dτ在τ时刻作用在结构上，产生一个微小的速度增量，然后结构按自由振动"释放"这个响应。把所有微响应从0到t时刻叠加起来，就得到了t时刻的总响应。

    单位脉冲响应函数 h(t-τ)（无阻尼）：
    h(t-τ) = (1/(mω)) · sin[ω(t-τ)]

    杜哈梅积分（无阻尼）：
    u(t) = (1/(mω)) ∫₀ᵗ p(τ)·sin[ω(t-τ)] dτ

    考虑阻尼：
    u(t) = (1/(mω_D)) ∫₀ᵗ p(τ)·e^(-ζω(t-τ))·sin[ω_D(t-τ)] dτ

    其中 ω_D = ω·√(1-ζ²)，为阻尼自振频率。

这个积分看起来复杂，但它蕴含着深刻的思想：一旦你知道了一个结构在单位脉冲下的响应（即结构的"个性"），你就可以通过叠加积分得到它在任意荷载下的响应。这就像你知道了乐器的音色（脉冲响应），就能预测它演奏任何曲子的声音。

典型荷载的杜哈梅积分结果

荷载类型 p(t)表达式 最大响应（无阻尼） 工程应用
阶跃荷载 p₀·H(t) 2·u_st 突加恒载
矩形脉冲 p₀, 0<t<t₁ 取决于t₁/T 爆炸荷载简化
三角脉冲 p₀·(1-t/t₁) 小于矩形脉冲 更真实的爆炸模拟
半正弦脉冲 p₀·sin(πt/t₁) 约1.5~1.7·u_st 冲击荷载

关键观察：对于短脉冲荷载（脉冲持续时间t₁远小于结构自振周期T），最大响应主要由"冲量"决定，而不是由荷载的峰值决定。也就是说：一个持续0.1秒、峰值100kN的脉冲，和一个持续0.2秒、峰值50kN的脉冲，如果它们的总冲量相同（在荷载-时间曲线下面积相同），引起的结构响应可能是非常接近的。这就是为什么在抗爆设计中，"冲量"比"峰值压力"更重要的原因。

隔振原理与工程应用

现在我们从"如何防止振动破坏"转向"如何利用动力学的原理来控制振动"——这就是隔振（Vibration Isolation）。

隔振有两种基本场景：一是保护结构或精密设备免受地基振动的干扰（被动隔振），二是减少设备运转时传给基础的振动力（主动隔振）。

隔振的基本原理

隔振的核心指标是隔振效率（隔离系数），定义为传递力与激振力的比值（主动隔振）或传递位移与基础位移的比值（被动隔振）。

    力的传递率 TR（Transmissibility）：
    TR = F_transmitted / F_applied

    TR = √[1 + (2ζβ)²] / √[(1-β²)² + (2ζβ)²]

    当 β < √2 时，TR > 1 —— 隔振系统反而放大了振动！
    当 β > √2 时，TR < 1 —— 隔振系统有效
    当 β → ∞ 时，TR → 0 —— 理想隔振（但实际不可能达到）

这里有一个反直觉的重要结论：只有当频率比β>√2时，隔振系统才真正起到"隔振"作用。如果β<√2，隔振器不仅不隔振，反而会放大振动！这就解释了为什么隔振设计的关键是"让隔振器的自振频率远低于干扰频率"。

工程中的隔振实例

1. 建筑基底隔振（Base Isolation）

这是高层建筑抗震中最成熟的技术之一。在建筑底部安装叠层橡胶支座（LRB，Lead Rubber Bearing），使整个建筑的自振周期从0.5~1秒延长到2~4秒。而地震的主要能量集中在0.1~0.8秒的短周期段——这样建筑物的自振频率就远低于地震的"主导频率"，β远大于√2，地震能量被有效隔离。

2. 精密设备隔振

半导体工厂的光刻机、医院的核磁共振仪——这些设备对环境振动极其敏感。通常的做法是在设备下方安装空气弹簧隔振平台，将平台的自振频率降到1~2Hz（甚至更低），而干扰振动（来自行人、车辆、暖通设备）通常在10Hz以上——β值足够大，隔振效果显著。

3. 桥梁支座隔振

高阻尼橡胶支座被广泛应用于地震高烈度区的桥梁。它一方面延长了结构周期（降低地震力），另一方面通过橡胶材料的滞回耗能（增加阻尼）来减小响应。

    典型基底隔振系统示意：

        上部结构（质量m）
        ┌─────────────────────┐
        │                     │
        │    ├──┬──┬──┬──┤    │
        └────┴──┴──┴──┴──────┘
              │  │  │  │
        ┌─────┴──┴──┴──┴──────┐
        │    隔振支座（弹簧+阻尼器）  │
        └─────────────────────────┘
              地基（地面运动输入）

        隔振支座将上部结构与地面"软连接"，
        使结构自振周期避开地震主频段。
踩坑故事：一个真实的"隔振翻车"案例。某高层建筑在楼顶安装了一台大型冷却塔，设计时工程师在冷却塔支架下方安装了弹簧隔振器。他们选用的隔振器自振频率为5Hz，而冷却塔运行时的主频率恰好也在5Hz左右——β=1！结果冷却塔一开机，整个顶层的振动比不装隔振器时还要剧烈数倍。最后不得不重新设计，将隔振器的自振频率降到1.5Hz（β≈3.3），问题才得以解决。这个故事的教训是：隔振器的频率选择不是随意的——你必须知道干扰频率，然后"故意"让隔振器的自振频率远低于它。在β>√2之前，隔振器不叫"隔振器"，它叫"放大器"。

本章小结与下章预告

在这一章中，我们学习了单自由度体系强迫振动的核心内容：

  • 简谐荷载下的稳态响应——振幅、相位角和动力放大系数DMF，理解β、ζ、DMF三者之间的定量关系；
  • 共振现象——当β=1时能量输入最大，DMF≈1/(2ζ)决定了结构能承受的最大动响应；
  • 杜哈梅积分——将任意荷载的响应表示为脉冲响应的叠加，是动力响应分析的理论基石；
  • 隔振原理——频率比β>√2是隔振有效的必要条件，工程中通过延长结构自振周期来实现隔振。
