开场：从静场到波的飞跃

从第1章到第14章，我们一直在研究静态或准静态的电磁场。电场由静止电荷产生，磁场由恒定电流产生。但如果电荷开始加速运动，或者电流随时间变化，会发生什么？答案是——电磁波。

1864年，麦克斯韦完成了物理学史上最伟大的理论综合之一。他将法拉第的电磁感应定律推广（加入位移电流项），得到了一组完整的方程组。当他试图从真空中消去电场或磁场时，他惊讶地发现：电场和磁场各自满足波动方程，其传播速度 c = 1/√(μ₀ε₀)。代入数值后得到 c ≈ 3×10⁸ m/s，恰好等于当时已知的光速！麦克斯韦由此断言：光就是一种电磁波。这是人类第一次从纯理论计算中推导出光速，也是物理学史上最激动人心的时刻之一。

从这一章开始，我们将进入电磁波动力学的广阔世界。平面波虽然不是最一般的解，但它直观地展示了电磁波的核心特征：横波性、偏振性、能流传播。理解平面波是理解反射、折射、波导、天线以及一切射频/微波工程的基础。

波动方程的推导

我们从真空中（ρ=0, J=0）的麦克斯韦方程组出发：

  • ∇ · E = 0
  • ∇ · B = 0
  • ∇ × E = −∂B/∂t
  • ∇ × B = μ₀ε₀ ∂E/∂t

对方程(3)两边取旋度（∇×）：∇ × (∇ × E) = −∂(∇ × B)/∂t。利用矢量恒等式 ∇ × (∇ × E) = ∇(∇·E) − ∇²E，并代入 ∇·E=0：

−∇²E = −∂(∇ × B)/∂t

再将(4)式代入：∇²E = μ₀ε₀ ∂²E/∂t²

完全对称地，对 B 做同样的推导可得：∇²B = μ₀ε₀ ∂²B/∂t²

这就是三维向量波动方程。最经典的一维标量形式是：∂²u/∂z² = (1/v²)∂²u/∂t²，其中波速 v = 1/√(με)。在真空中，这就是光速 c。

🔑 核心洞察
波动方程的出现完全归功于麦克斯韦方程组的"对称化"——安培环路定理中加入的位移电流项 ∂D/∂t。
没有位移电流，∇ × B = μ₀J，取旋度后只会得到 ∇²B = μ₀∇×J，形不成波动方程。
换句话说：位移电流是电磁波存在的数学前提。麦克斯韦纯粹从数学美学的角度加入这一项，却预言了整个无线电时代。

平面波解

平面波是波动方程最简单、最重要的解，也是最理想的近似。所谓"平面波"，是指等相位面为平面的波。考虑沿 +z 方向传播、电场沿 x 方向偏振的平面波：

E(z,t) = E₀ cos(kz − ωt) x̂

B(z,t) = B₀ cos(kz − ωt) ŷ

其中 ω = 2πf 是角频率，k = 2π/λ 是波数。代入波动方程可得色散关系：ω = ck（在真空中）。

由法拉第定律 ∇×E = −∂B/∂t 可以推导出电场和磁场的幅度关系：

B₀ = E₀/c

在非磁性介质（μ≈μ₀）中，更通用的形式是 E₀/H₀ = η，其中 η = √(μ/ε) 为波阻抗，在真空中 η₀ ≈ 377Ω。

平面波的关键性质：

  • 横波性（TEM）：E ⊥ B ⊥ k，三个矢量相互正交，属于"横电磁波"（Transverse Electromagnetic Wave）。
  • 同相位：在真空中，E 和 B 同相振荡（相位差为零），同步达到最大值和最小值。
  • 瞬时关系：|E| = c|B|，即在空间中任一点任一时刻，电场和磁场的幅度满足比例关系。

坡印廷矢量与能量流

电磁波携带着能量在空间中传播。描述这一能量流动的关键物理量是坡印廷矢量（Poynting Vector）：

S = E × H

坡印廷矢量的单位是 W/m²，表示单位时间内通过单位面积的能量（能量流密度）。方向就是电磁波传播的方向。

对于平面波 E = E₀cos(kz−ωt)x̂，H = H₀cos(kz−ωt)ŷ：

S = E₀H₀ cos²(kz−ωt) ẑ

由于电磁波的频率极高（例如可见光 ~10¹⁴ Hz），在绝大多数工程应用中我们关心的是时间平均值：

⟨S⟩ = ½ E₀H₀ = E₀²/(2η)

坡印廷定理（Poynting Theorem）是能量守恒定律在电磁场中的体现：

−∂u/∂t = ∇·S + J·E

其中 u = ½(εE² + μH²) 是电磁场能量密度。这个公式告诉我们：某区域内电磁能量的减少，要么以坡印廷矢量的形式流出了边界，要么被焦耳热消耗掉了。

💡 工程启示：坡印廷矢量与功率传输
在分析同轴电缆、微带线等传输线时，电磁能量并不是通过导线内部的电子传输的，而是通过导线之间的介质中的坡印廷矢量流传输的！导线的作用只是引导坡印廷矢量的方向。这也是为什么射频设计中"场"的视角比"路"的视角更本质。
另一个有趣的事实：变压器铁芯中虽然磁场很强，但电场接近零，坡印廷矢量很小——所以变压器不是通过铁芯传输能量的！

电磁波的极化

极化（Polarization）描述的是电场矢量 E 在空间固定点随时间变化的轨迹。这是电磁波区别于声波（纵波）的关键性质之一。

线极化

电场矢量始终沿一条直线振荡。例如我们上文的平面波 E=E₀cos(kz−ωt)x̂ 就是 x 方向线极化波。任意方向的线极化波可以分解为 x 和 y 两个正交分量的叠加：E = (E_x x̂ + E_y ŷ)cos(kz−ωt)，两个分量相位相同。

圆极化

当两个正交分量的幅度相等但相位差 ±90° 时，电场矢量末端画出一个圆：

E(z,t) = E₀[cos(kz−ωt)x̂ + sin(kz−ωt)ŷ] → 右旋圆极化（RHCP）

E(z,t) = E₀[cos(kz−ωt)x̂ − sin(kz−ωt)ŷ] → 左旋圆极化（LHCP）

椭圆极化

当两个正交分量幅度不等、相位差不等于 0 或 ±90° 时，电场矢量末端画出一个椭圆。椭圆极化是最一般的情况，线极化和圆极化是它的特例。

极化的数学描述可用琼斯矢量（Jones Vector）表示。例如 x 方向线极化可写作 [1, 0]ᵀ，右旋圆极化可写作 [1, −j]ᵀ/√2。

极化类型 x分量与y分量的关系 琼斯矢量 典型应用
水平线极化 E_y=0 [1, 0]ᵀ 地面电视广播
垂直线极化 E_x=0 [0, 1]ᵀ 移动通信基站
±45°线极化 E_x=±E_y, 同相 [1, ±1]ᵀ/√2 WiFi天线分集
右旋圆极化 E_x=E_y, 相位差−90° [1, −j]ᵀ/√2 卫星通信、GPS
左旋圆极化 E_x=E_y, 相位差+90° [1, +j]ᵀ/√2 卫星通信
💡 极化匹配——天线设计中不可忽视的关键
发射天线和接收天线的极化必须匹配，否则会产生严重的极化失配损耗。垂直极化天线接收水平极化信号时，理论上接收功率为零（正交极化隔离）。这也是为什么卫星通信常使用圆极化——卫星姿态变化不影响信号接收，因为圆极化波对接收天线的朝向不敏感。而GPS卫星使用右旋圆极化（RHCP），所以GPS接收天线也是RHCP的。如果你用线极化天线去接收GPS信号，会有约3dB的极化损耗。

电磁波在导电介质中的传播

在实际工程中，电磁波很少在真空中传播，更多是在有损耗的介质（如土壤、海水、混凝土、生物组织）中传播。导电介质中，麦克斯韦方程组的解会发生质的变化。

复介电常数

引入复介电常数 ε̃ = ε − jσ/ω。这时波数 k 变成复数：k̃ = ω√(μ ε̃) = β − jα。平面波解变为：

E = E₀e^{−αz}cos(ωt − βz)x̂

其中 α 称为衰减常数（Np/m），β 为相位常数（rad/m）。指数衰减因子 e^{−αz} 意味着波的振幅随传播距离指数下降。

损耗角正切

定义 tanδ = σ/(ωε)，用于区分介质类型：

  • tanδ ≪ 0.01：良介质（低损耗），如纯水、玻璃、聚四氟乙烯（PTFE）。
  • tanδ > 1：良导体，如铜(σ≈5.8×10⁷)、铝、金。趋肤深度 δ_s = 1/α = √(2/(ωμσ))。
  • 中间区域：有耗介质，如湿土壤、混凝土。

趋肤效应（Skin Effect）的"雏形"在这里已经显现：高频电流趋向于在导体表面流动，因为内部电磁波已经被指数衰减掉了。我们将在第19章深入讨论。

电磁波谱

电磁波的频率跨越了令人难以置信的范围——从极低频（ELF）的通讯频段到伽马射线。不同频段的电磁波在产生方式、传播特性和工程应用上截然不同。

ELF（极低频） 3–30 Hz 10³–10⁴ km 电力线、地磁脉动 潜艇通讯
VHF（甚高频） 30–300 MHz 1–10 m FM广播、电视 广播、对讲机
UHF（特高频） 300–3000 MHz 10–100 cm 手机、WiFi、微波炉 移动通信、无线网络
SHF（超高频） 3–30 GHz 1–10 cm 雷达、5G毫米波 雷达、卫星通信

人类的眼睛只能感知可见光这一狭窄的窗口。但电磁学告诉我们，所有频段的电磁波都遵循同一套麦克斯韦方程组——它们的区别仅仅是频率不同。这使得电磁学具有惊人的统一性。

Python示例：平面电磁波可视化

import numpy as np
        import matplotlib.pyplot as plt

        # 参数设置
        c = 3e8            # 光速 (m/s)
        f = 1e9            # 频率 1 GHz
        lam = c / f        # 波长 0.3 m
        k = 2 * np.pi / lam # 波数
        omega = 2 * np.pi * f # 角频率

        z = np.linspace(0, 2*lam, 1000)  # 空间范围 0~2λ
        t = 0                              # 取 t=0 瞬间

        # 电场 (x方向) 和 磁场 (y方向) 的瞬时分布
        E = np.cos(k*z - omega*t)          # E_x
        H = np.cos(k*z - omega*t)          # H_y (归一化)

        plt.figure(figsize=(10, 4))
        plt.plot(z/lam, E, label='E_x (电场)', linewidth=2)
        plt.plot(z/lam, H, '--', label='H_y (磁场)', linewidth=2)
        plt.axhline(0, color='gray', linewidth=0.5)
        plt.xlabel('z/λ')
        plt.ylabel('归一化幅度')
        plt.title('平面电磁波在空间中的瞬时分布 (t=0)')
        plt.legend()
        plt.grid(alpha=0.3)
        plt.tight_layout()
        plt.show()

运行这段代码可以看到，E 和 H 在空间上同相振荡，且二者正交。改变 t 值可以观察到波沿 +z 方向传播的动画效果。

🔑 实践洞察：WiFi信号的穿墙衰减
很多人以为WiFi穿墙信号弱只是因为"墙太厚"，其实材料的电磁特性才是决定性因素。
以2.4GHz WiFi为例：
• 空气：衰减≈0 dB/m（几乎不衰减）
• 木质隔墙：衰减≈1–2 dB
• 砖墙：衰减≈4–8 dB
• 混凝土墙：衰减≈10–20 dB
• 钢筋混凝土墙：衰减≈20–35 dB（钢筋的导电性导致强烈反射和吸收！）
关键在于材料的介电常数 ε_r 和损耗角正切 tanδ。混凝土的 ε_r ≈ 6–8，tanδ 较大，波进入后波长急剧缩短且快速衰减。更致命的是，墙体中的钢筋网会反射电磁波，在前方形成驻波零点。这就是为什么在钢筋混凝土结构的大型建筑中，仅仅隔一堵墙，WiFi信号就可能完全消失。
