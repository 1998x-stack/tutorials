第20章 电磁兼容（EMC）基础

你的手机和 WiFi 路由器共享 2.4 GHz 频段却不互相干扰，医疗植入设备与安检门安全共存，一辆车内有 100+ 个 ECU 还能正常工作——这些不是运气，而是电磁兼容（EMC）工程师的工作成果。EMC 研究的是：如何让电子设备在预期的电磁环境中正常执行功能，同时不向环境释放过量的电磁能量。

EMC三要素

任何电磁干扰（EMI）问题都可以抽象为三个基本元素的组合：

  • 源（Source/Emitter）：产生干扰能量的设备或电路。例如开关电源的 MOSFET 开关节点、数字时钟的谐波、雷电放电、电机换向火花。
  • 耦合路径（Path/Coupling）：干扰从源传播到受害者的通道。可以是导线传导、空间辐射、电容性耦合或电感性耦合。
  • 受害者（Victim/Receptor）：受到干扰影响的设备或电路。例如传感器的小信号放大器、无线接收机、数字逻辑电路。
EMC 设计的核心思想：只要打断三要素中的任意一个——抑制源、阻断路径、保护受害者——EMC 问题就能解决。最好的工程实践是同时在所有三个层面进行治理，而不是只依赖单一措施。

耦合机制

干扰从源到达受害者有四种基本耦合方式。正确识别耦合方式是解决 EMC 问题的第一步。

传导发射（Conducted Emission）

干扰通过导线（电源线、信号线、地线）从一个电路传导到另一个电路。频率范围通常为 150 kHz ~ 30 MHz。开关电源中的开关纹波沿着电源线传导，影响同一电源总线上其他设备的工作。

辐射发射（Radiated Emission）

干扰通过电磁波在空间中传播。频率通常在 30 MHz 以上，因为低频时波长太长，有效辐射所需的"天线"尺寸过大。数字电路的高次谐波、时钟信号的基频及其谐波都是常见的辐射源。

近场耦合

当源与受害者的距离小于 λ/2π（约 1/6 波长）时，近场效应占主导。近场耦合分为两种：

  • 电容耦合（E 场耦合）：存在于高阻抗电路中。两导体之间的寄生电容 C 提供了耦合路径。干扰幅度与频率和电压变化率（dV/dt）成正比：I = C · dV/dt。典型场景：数字信号线平行于高阻抗模拟输入线。
  • 电感耦合（H 场耦合）：存在于低阻抗电路中。两导体间的互感 M 将源回路的电流变化耦合到受害者回路。感应电压 V = M · dI/dt。典型场景：大功率感性负载（继电器、电机）的电流突变耦合到附近的敏感信号线。

差模与共模

从信号完整性的角度，EMI 噪声可分为差模（Differential Mode, DM）和共模（Common Mode, CM）两大类。这个区分极为重要，因为两者的产生机理、传播路径和抑制措施完全不同。

差模噪声存在于信号线与回线之间，是信号电流的"副产品"。差模电流在两条导线中以相反方向流动——它本身就是有用信号的一部分。差模辐射正比于回路面积，因此减小回路面积是抑制差模辐射的关键。

共模噪声是 EMC 问题的头号敌人。共模电流在两条导线上以相同方向流动，通过寄生电容或杂散耦合到地，形成一个巨大的回路天线。一条 10 cm 的走线上流过 10 μA 的共模电流，在 100 MHz 处可能辐射出 40 dBμV/m 的场强——远超 FCC Class B 限制！

共模辐射为何危险？差模噪声的回路面积小，且镜像电流趋向于抵消远场辐射。共模噪声则不然——它的"回路"包含了整个设备与地之间的巨大环路，效率相当于一个偶极子天线。这就是为什么 10 μA 的共模电流可能比 100 mA 的差模电流产生更强辐射的原因。

共模扼流圈（Common Mode Choke）是最常用的共模抑制器件。它将两根导线同向绕在一个磁芯上：差模电流在两线中方向相反，磁场相互抵消，等效电感极小；共模电流方向相同，磁场叠加，表现为高阻抗。理想共模扼流圈对差模信号完全透明，对共模噪声呈现高阻抗。

屏蔽效能

屏蔽是用金属或磁性材料封闭敏感电路或干扰源，阻止电磁场进出。屏蔽效能的定量描述为：

SE (dB) = R + A + B

其中 R 是反射损耗（reflection loss），A 是吸收损耗（absorption loss），B 是多重反射修正项（当 A > 10 dB 时可忽略）。

屏蔽机制因场类型而异：

  • 电场屏蔽：主要靠反射。高电导率材料（铜、铝）中的自由电子在电场作用下重新分布，产生反向电场抵消入射波。接地良好的金属外壳对电场屏蔽非常高效——往往只用一层薄铜箔就能达到 60 dB 以上的屏蔽效能。
  • 磁场屏蔽：
    • 低频磁场（< 100 kHz）：需要高磁导率材料（μ-metal、坡莫合金）提供磁短路路径。μ-metal 的 μᵣ 可达 10⁵，但容易被机械应力或焊接改变性能，且不能用于需要饱和的强磁场中。
    • 高频磁场（> 100 kHz）：利用涡流排斥磁场。铜或铝在 1 MHz 以上的磁场屏蔽效果很好——因为涡流产生的反向磁场抵消了入射磁场。
屏蔽设计要点：屏蔽效能受限于屏体上最大的开孔或缝隙。一个 10 cm 的缝隙会将理想屏蔽体在 300 MHz（λ ≈ 1 m）处的 SE 从 60 dB 拉到几乎为零。这就是为什么高频屏蔽体要求缝隙宽度小于 λ/20，且使用导电衬垫或波导通风窗。

接地与滤波

接地和滤波是 EMC 设计中最基础也是最重要的两个手段。

接地策略

接地（Grounding）的核心目的不是"接大地"，而是为电流提供低阻抗的回流路径。

  • 单点接地：所有电路在地的一点连接。适合低频（< 1 MHz），避免地环路。
  • 多点接地：各电路就近接地，形成低阻抗地平面。适合高频（> 10 MHz），减少接地阻抗。
  • 混合接地：1~10 MHz 范围内，结合单点和多点方式的优点。
  • 地环路：当两个设备在不同电位处接地时，地电位差在信号回路中产生共模电流。解决方法：断开地环路（使用光耦、隔离变压器、共模扼流圈），或确保足够小的接地阻抗。

滤波元件

  • 铁氧体磁珠（Ferrite Bead）：低频时呈低阻抗（小电感），高频时磁芯损耗增加呈高阻抗（纯电阻），用于吸收高频噪声。注意：铁氧体磁珠在直流偏置电流过大会饱和，失去抑制效果。
  • X 电容：跨接在电源线之间（Line-Neutral），用于抑制差模噪声。必须是安全认证的安规电容（X1/X2 等级），失效模式为开路。
  • Y 电容：跨接在电源线与地之间（Line-Ground, Neutral-Ground），用于抑制共模噪声。必须是 Y 等级安规电容，失效模式为开路，且容量受漏电流限制（通常 < 4.7 nF）。

PCB 设计的 20H 规则：电源层相对于地层内缩 20H（H 是层间介质厚度），可以将边缘辐射降低约 70%。这是因为边缘辐射场的边缘效应在电源层缩入后急剧减弱。注意：只有高速多层 PCB 才需要严格遵循 20H 规则——对于低速板，节省空间比减少辐射更重要。

下面是一个计算屏蔽效能的 Python 函数：

import numpy as np

def shielding_effectiveness(f, t, sigma, mu_r=1, mu_0=4*np.pi*1e-7):
    """
    计算平面波垂直入射时金属平板的屏蔽效能

    参数:
        f: 频率 (Hz)
        t: 材料厚度 (m)
        sigma: 电导率 (S/m)
        mu_r: 相对磁导率
        mu_0: 真空磁导率 (H/m)

    返回:
        SE_total: 总屏蔽效能 (dB)
        R_loss: 反射损耗 (dB)
        A_loss: 吸收损耗 (dB)
    """
    mu = mu_r * mu_0
    omega = 2 * np.pi * f
    delta = np.sqrt(2 / (omega * mu * sigma))  # 趋肤深度

    # 本征阻抗: 自由空间 Z0 ≈ 377 Ω
    Z0 = 377
    # 材料的波阻抗
    Zm = np.sqrt(1j * omega * mu / (sigma + 1j * omega * 8.85e-12))
    Zm_mag = np.abs(Zm)

    # 反射损耗 (dB)
    R_loss = 20 * np.log10(np.abs(Z0 + Zm)**2 / (4 * Z0 * np.abs(Zm)))

    # 吸收损耗 (dB)
    A_loss = 8.686 * (t / delta)  # 8.686 = 20 * log10(e)

    # 多重反射修正 B (当 A > 10 dB 时可忽略)
    if A_loss > 10:
        B = 0
    else:
        B = 20 * np.log10(np.abs(1 - ((Zm - Z0)/(Zm + Z0))**2 * np.exp(-2*t/delta)))

    SE_total = R_loss + A_loss + B
    return SE_total, R_loss, A_loss

# 示例: 0.5mm 铜屏蔽 @ 1 MHz
sigma_Cu = 5.96e7
SE, R, A = shielding_effectiveness(1e6, 0.5e-3, sigma_Cu)
print(f"1 MHz 铜板 (0.5 mm): SE = {SE:.1f} dB")
print(f"  反射损耗: {R:.1f} dB, 吸收损耗: {A:.1f} dB")
真实案例：某创业公司的物联网网关产品首次送检 EMC，辐射发射测试在 200 MHz 处超标 12 dB。工程师花了两周排查，最终发现问题出在 PCB 地平面的一个 3 cm×1 cm 的缝隙——这个缝隙是为了绕开一个走线区域而无意中形成的，却意外地起到了缝隙天线的作用。在缝隙两侧每 5 mm 增加一个缝合电容（stitching capacitor 10 nF），辐射量降低了 15 dB，顺利通过测试。事后统计显示：超过 50% 的电子产品首次 EMC 测试会失败，其中绝大多数问题出在 PCB 布局和接地。

EMC 设计不是产品开发后期可以"修补"的问题——它必须从原理图设计、PCB 布局、结构设计阶段就开始考虑。深入理解本章介绍的耦合机制、差模共模、屏蔽和接地原理，将为后续第 25 章的 EMC 设计实战打下坚实基础。下一章我们将进入电磁场的数值计算方法——这些方法正是分析复杂 EMC 问题的核心工具。
