第20章 多因子模型实战

大家好，我是你们的讲师。上一章我们聊了量化选股的入门概念——因子投资、回测和过拟合。这一章我们直接上实战，来拆解多因子模型最核心的那点事。

你可能听说过"因子"这个词在很多场合被提到——"价值因子""动量因子""低波因子"——但真正把它们落地到A股市场上，你会发现很多坑：同一个因子用不同的方式计算，结果天差地别；有些因子在美股有效，来了A股就变废；因子拥挤到一定程度，超额收益一夜之间消失。这一章我们就要解决这些问题——从因子构建到效果评价，从因子合成到拥挤度监控，完整地走一遍多因子模型的工作流。

20.1 五大经典因子的A股表现

全球量化研究对因子做了几十年的探索，从中提炼出了一批"经久不衰"的经典因子。在A股市场上，下面五个因子是研究最充分、表现最稳定的：

  • 价值因子（HML）：买入低估值股票，卖出高估值股票。常用的估值指标有PE、PB、PS、PCF等。在A股历史上，PB因子的表现优于PE因子。
  • 成长因子：买入高成长性股票。常用指标有营收增速、净利润增速、EPS增长率等。2025年A股中，成长因子和资金流因子是表现最突出的因子类别，累计超额超过12%。
  • 质量因子：买入高质量公司。核心指标是ROE（净资产收益率），此外还有毛利率、资产负债率、盈利稳定性等。ROE因子在A股中长期有效，尤其与估值因子结合使用时。
  • 动量因子：买入过去一段时间涨幅较好的股票（动量效应）。A股市场上的动量效应偏短期——20日-60日动量比较有效，而12个月的长期动量效应不明显（与美股不同）。
  • 低波因子：买入低波动率的股票。这个因子听起来反直觉——低风险的股票居然有更高的风险调整后收益？但它在全球市场都经过验证。A股中低波因子的IC长期为正。

下面是一张这五大因子在A股近10年的表现对比表：

要点解读：上表中的年化超额收益是指做多因子分值最高组、做空最低组的"多空组合"收益，不代表纯多头收益。在A股实际投资中，多数机构是做多高分组（不构建空头头寸），所以实际可获得的多头超额收益大约是表格中数值的一半。另外需要注意，因子的表现在不同年份有显著的轮动性——没有哪个因子年年都有效。

20.2 因子构建的完整流程

因子构建不是简单地把数据拉出来排个序。要让一个因子"干净可用"，需要经过四个步骤：

  1. 去极值（Winsorization）：剔除极端值的影响。比如某只股票的PE突然飚到10000倍，这往往是计算问题或微利股导致的，需要将其替换为合理的边界值。
  2. 中性化（Neutralization）：剔除其他因素的干扰。最常用的是行业中性化和市值中性化——因为不同行业的PE天然不同，大盘股和小盘股的特征也不同，如果不做中性化，你得到的因子效果可能是来自行业或市值的差异，而非因子本身的预测力。
  3. 标准化（Standardization）：把不同量纲的因子值转换到同一尺度，通常使用Z-score方法——使因子的均值为0、标准差为1。
  4. 因子合成：把多个单因子按照一定权重合成为一个综合因子。等权合成是最常用的方法，也有用IC加权、IR加权、机器学习加权等方式。

下面是一段因子中性化的示例代码：

import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

def factor_neutralization(factor_data, industry_dummies, market_cap):
    """
    对因子进行行业和市值中性化
    factor_data: 因子原始值（Series）
    industry_dummies: 行业哑变量（DataFrame）
    market_cap: 对数市值（Series）
    """
    # 1. 去极值：MAD方法（中位数绝对偏差）
    median = factor_data.median()
    mad = (factor_data - median).abs().median()
    upper = median + 5 * mad
    lower = median - 5 * mad
    factor_winsor = factor_data.clip(lower, upper)

    # 2. 中性化回归：因子值 ~ 行业哑变量 + 对数市值
    X = pd.concat([industry_dummies, market_cap], axis=1)
    X = X.fillna(0)
    y = factor_winsor.fillna(0)

    model = LinearRegression()
    model.fit(X, y)
    y_pred = model.predict(X)

    # 残差 = 剔除行业和市值影响后的纯净因子
    factor_neutral = y - y_pred

    # 3. 标准化为Z-score
    factor_zscore = (factor_neutral - factor_neutral.mean()) / factor_neutral.std()

    return factor_zscore

# 使用示例
# clean_factor = factor_neutralization(pe_data, industry_onehot, log_market_cap)

这个过程很关键，我见过太多新手一上来就拿着原始因子值去排序选股，结果发现表现很好——但仔细一查，好是因为选到的全是小盘股。做中性化就是要把因子剥离干净，确保我们赚的是这个因子真正的"能力"，而不是其他因素的"搭便车"。

20.3 如何评价一个因子好不好

因子构建好了，怎么判断它是不是"好因子"？业界有几个标准评价指标：

  • IC（Information Coefficient，信息系数）：因子值与未来收益的相关系数。IC的绝对值越大，因子预测力越强。IC为正表示因子值越大未来收益越高（正向因子），为负则是反向因子。一般IC均值在0.03以上就是不错的好因子。
  • Rank IC：因子排序与收益排序的秩相关系数（Spearman相关系数）。相比普通IC，Rank IC对极端值不敏感，更稳健。
  • ICIR（IC Information Ratio）：IC的均值除以IC的标准差。这个指标衡量的是因子预测的稳定性——即使一个因子IC均值不错，但如果波动很大（忽高忽低），跨期使用效果也会很差。通常ICIR大于0.5就算可用，大于1.0就算优秀的因子。

另外还有一个重要的角度是因子换手率。如果一个因子频繁切换持仓，交易成本会非常高。好的因子应该在保持预测力的同时，有较低的换手率。

评价因子的经验阈值： IC均值 > 0.03 → 好因子；IC均值 0.01~0.03 → 可用因子；IC均值 < 0.01 → 需要进一步改进。
ICIR > 1.0 → 优秀；ICIR > 0.5 → 可用；ICIR < 0.3 → 稳定性存疑。
但需要注意：这些阈值在不同市场周期中会有变化，熊市中因子普遍"变弱"是正常现象。

20.4 因子拥挤度：发现超额正在消失

因子投资的"悲剧"在于：当一个因子被越来越多的人知道并使用，它的超额收益就会逐渐消失。这就是因子拥挤度（Factor Crowding）的问题。

因子拥挤度的监测指标主要有三个：

  • 头部机构持仓重合度：大量机构同时持有哪些股票？当某类因子（比如成长因子）的逻辑被广为人知后，资金集中涌入，导致这部分股票的估值被推高到不合理水平。
  • 因子估值价差：因子的多空组合估值差。当价差过大时，说明因子的"便宜货"已经被买贵了。
  • 因子反转频率：因子收益从正转负的频率。拥挤度过高时，因子收益的可持续性下降，经常出现"急涨后急跌"的现象。

根据2025年的统计数据，A股市场上约40%的机构使用类似的多因子策略，因子同质化问题相当严重。这意味着当市场风格切换时，大量的机构在同一时间调仓，导致因子踩踏（Factor Crash）——所有使用相似因子的策略同时失效。

踩坑故事：因子拥挤导致动量崩溃

2020年底到2021年初，A股市场上最流行的因子是"动量+成长"。当时核心资产的上涨如日中天，几乎所有量化私募都在使用这两个因子。数据显示，当时全市场动量因子的拥挤度达到了历史极值——动量因子多空组合的估值差处于99%分位数。

结果2021年春节后发生了什么？核心资产暴跌，茅指数在短短几周内下跌超过20%。动量因子在那次风格切换中遭遇了史诗级别的回撤——多空组合在2月份的月度收益是-18%。很多私募产品的净值在那一个月里就跌去了全年的涨幅。

更糟糕的是，当所有机构同时试图砍掉仓位时，流动性的枯竭加剧了下跌速度——这正是因子拥挤导致踩踏的经典场景。事后复盘发现，那些提前监测到拥挤度指标并降低了因子暴露的机构，在这次下跌中受到的冲击要小得多。

教训：因子拥挤度应该作为量化策略的风控指标，当拥挤度进入高位区间时，降低该因子的权重是明智的选择。

20.5 因子择时与动态加权

既然每个因子在不同的市场环境下表现不同，那有没有办法"预测"哪些因子接下来会表现更好？这就是因子择时（Factor Timing）要解决的问题。

目前主流的因子择时方法有几种：

  • 基于宏观状态的因子择时：在不同的宏观环境下（扩张/衰退/高通胀/低通胀），不同因子的历史表现有差异。比如成长因子在扩张期表现好，低波因子在衰退期表现好。
  • 基于因子自身状态的择时：因子估值价差低时未来表现好（均值回复），因子拥挤度高时降低权重。
  • 机器学习因子择时：用PLS（偏最小二乘法）等模型来预测因子未来一期的收益。国泰海通证券在2026年的研究中，使用PLS模型预测A股因子收益，取得了不错的效果。

近年来，一个重要的理念变化是从"为域选因子"到"为因子找域"。什么意思呢？传统方法是先确定一个选股域（比如沪深300成分股），然后在这个域内找表现最好的因子。新思路是反过来——先确定某个因子有其固有的"舒适区"，然后找到这个因子最适用的域。比如研发支出这个因子，在大盘股里效果一般，但在科技属性强的小盘股里表现非常突出。

"为因子找域"比"为域选因子"更合理——因为因子没有绝对的好坏，只有是否适合当前的环境和标的池。

小结与预习

这一章我们深入到了多因子模型的实战层面。我们学习了五大经典因子在A股的表现，因子构建的四步流程（去极值→中性化→标准化→合成），评价因子的核心指标（IC/ICIR），以及因子拥挤度的监测方法。最后我们讨论了因子择时和"为因子找域"的新型方法论。

经典因子很重要，但超额收益的探索永无止境。下一章我们来看一些更新的、更有趣的前沿因子——研发因子、业绩透支度POR、资金流因子、分析师预期因子等等。这些因子在近年的研究中表现出了相当不错的Alpha，而且它们的挖掘思路会给你带来很多启发。
