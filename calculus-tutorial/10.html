当 y 不再孤独

大家好，我是你们的讲师。在前两章里，我们学的所有求导都有一个共同点：函数都是显式写成的 y = f(x) 形式——y 在等式左边孤零零地站着，右边是 x 的表达式。但现实世界并不总是这么"配合"。

想象一个场景：你在物理课上遇到了一个运动的质点，它的 x 坐标和 y 坐标都是时间 t 的函数——x = cos t，y = sin t。这时候你问"轨迹上任一点的切线斜率是多少？"这个问题不是 y = f(x) 的形式可以回答的——因为 y 不是 x 的直接函数，而是通过 t 这个"中介"间接关联的。这就是本章要解决的问题。

我曾帮过一个物理系的朋友推导一个机械臂的运动轨迹，表达式里同时混着 x 和 y，根本解不出 y = f(x) 的形式。当时我用隐函数求导法，几分钟就搞定了——他惊讶地说："原来不需要解出 y 也能求导！"事实也正是如此：在很多时候，保持隐式反而比显式更方便。

显函数 vs 隐函数

先搞清楚基本概念：

显函数（explicit function）：y = f(x)，比如 y = x^2 + 3x + 1。y 被显式地表示为 x 的函数。

隐函数（implicit function）：F(x, y) = 0，比如 x^2 + y^2 = 1。y 和 x 的关系隐藏在方程中，没有显式解出 y。

要注意：有些隐函数可以化成显函数（比如 x^2 + y^2 = 1 可以化成 y = ±√(1 - x^2)），但有很多根本解不出来——比如 sin(xy) + e^y = x^3。对这类方程，隐函数求导法是唯一的选择。

类型 形式 例子 能否显化
显函数 y = f(x) y = x^3 + 2x - 1 本身就是显式
可显化的隐函数 F(x,y) = 0 x^2 + y^2 = 1 可解出 y = ±√(1-x^2)
不可显化的隐函数 F(x,y) = 0 y - ε sin y = x 无法用初等函数显化

隐函数求导的基本方法

隐函数求导的核心思想很简单：方程两边同时对 x 求导，然后把 y 看作 x 的函数。也就是说，每当遇到含有 y 的项，就把它当作复合函数来处理——先对 y 求导，再乘以 dy/dx。

步骤详解

例：求 x^2 + y^2 = 1 上某点处的导数 dy/dx。

第1步：方程两边同时对 x 求导
d/dx (x^2) + d/dx (y^2) = d/dx (1)

第2步：逐项求导
2x + d/dx (y^2) = 0

第3步：处理含有 y 的项——把 y^2 看作 y(x) 的复合函数
d/dx (y^2) = 2y · dy/dx

第4步：整理出 dy/dx
2x + 2y · dy/dx = 0
dy/dx = -x / y

结果中居然还含有 y！这是隐函数求导的一个特点——导数表达式里通常同时含有 x 和 y。这在几何上很容易理解：对于圆 x^2 + y^2 = 1，给定同一个 x 值，上半圆（y > 0）和下半圆（y < 0）的切线斜率是相反的——这正是公式中分母 y 的符号在起作用。

更复杂的例子

求 sin(xy) + e^y = x^3 中 dy/dx 的表达式。

第1步：两边对 x 求导
cos(xy) · d/dx(xy) + e^y · dy/dx = 3x^2

第2步：用乘法法则处理 d/dx(xy)
d/dx(xy) = (1)·y + x·dy/dx = y + x·dy/dx

第3步：代入
cos(xy) · (y + x·dy/dx) + e^y · dy/dx = 3x^2

第4步：合并含 dy/dx 的项
y·cos(xy) + x·cos(xy)·(dy/dx) + e^y·(dy/dx) = 3x^2
dy/dx · [x·cos(xy) + e^y] = 3x^2 - y·cos(xy)

第5步：解出 dy/dx
dy/dx = [3x^2 - y·cos(xy)] / [x·cos(xy) + e^y]
隐函数求导的要点
永远记住：当你对 y 的某项求导时，先对 "y 本身" 求导，再乘上 dy/dx。例如：
d/dx(y^3) = 3y^2 · dy/dx
d/dx(sin y) = cos y · dy/dx
d/dx(e^y) = e^y · dy/dx
这其实就是链式法则的直接应用——y 是 x 的函数。

对数求导法

对数求导法（logarithmic differentiation）是隐函数求导的一个巧妙应用。当函数是幂指函数形式（比如 y = x^x）或连乘除形式（比如 y = (x+1)^3·(x+2)^4 / (x+3)^5）时，先取对数再求导，能大幅简化计算。

幂指函数求导

求 y = x^x（x > 0）的导数。

第1步：两边取自然对数
ln y = ln(x^x) = x · ln x

第2步：两边对 x 求导（注意 y 是 x 的函数，左边要用隐函数求导）
(1/y) · dy/dx = ln x + x · (1/x) = ln x + 1

第3步：解出 dy/dx
dy/dx = y · (ln x + 1) = x^x · (ln x + 1)

这个结果是令人印象深刻的——x^x 的导数在公式表里可找不到，但通过对数求导法，我们轻松地得到了结果。

连乘除函数求导

求 y = (x+1)^3 · (x+2)^4 / (x+3)^5 的导数。

第1步：取对数
ln y = 3·ln(x+1) + 4·ln(x+2) - 5·ln(x+3)

第2步：对 x 求导
(1/y)·dy/dx = 3/(x+1) + 4/(x+2) - 5/(x+3)

第3步：解出 dy/dx
dy/dx = y · [3/(x+1) + 4/(x+2) - 5/(x+3)]

如果不取对数，这个函数要用除法法则配合乘法法则，计算量会大得多。对数的一个"魔法"就是：把乘法变成加法，把除法变成减法，把幂变成倍数——这些变换让求导瞬间变简单。

一个真实的竞赛题
我大学时参加数学竞赛，遇到一道题让求 y = (sin x)^(tan x) 的导数。赛场上有不少人直接开始用链式法则硬算，结果越算越乱。实际上，这题只需要两步：取对数 → 隐函数求导。前后不到一分钟就能出结果。这件事让我深刻体会到：解题不是比谁力气大，而是比谁会转化问题。

参数方程求导

一阶导数

参数方程（parametric equations）用参数 t 同时表示 x 和 y：

x = φ(t)
y = ψ(t)

那么 dy/dx（切线斜率）如何求？核心思想是：dy/dx = (dy/dt) / (dx/dt)。只要 t 的导数分别存在且 dx/dt ≠ 0。

例：圆的参数方程 x = cos t, y = sin t, t ∈ [0, 2π)

dx/dt = -sin t
dy/dt = cos t

dy/dx = (dy/dt) / (dx/dt) = cos t / (-sin t) = -cot t

注意 t = 0 或 t = π 时，sin t = 0，dx/dt = 0，导数为无穷大——
这对应圆上左右两端的竖直切线。

参数方程的二阶导数陷阱

这里有个非常容易出错的地方。你可能会想当然地认为：

d^2y/dx^2 = d/dt(dy/dx) / dx/dt   ← 正确公式！
           ≠ d^2y/dt^2 / d^2x/dt^2   ← 错误！千万不要这样算！

原因是：dy/dx 是 t 的函数，我们需要对 x 求二阶导，所以要用链式法则再除一次 dx/dt。

例：x = t^2, y = t^3，求 d^2y/dx^2

一阶导数：
dx/dt = 2t
dy/dt = 3t^2
dy/dx = (3t^2) / (2t) = (3/2)t,  t ≠ 0

二阶导数：
d^2y/dx^2 = d/dt(dy/dx) / dx/dt
          = d/dt[(3/2)t] / (2t)
          = (3/2) / (2t)
          = 3 / (4t)
参数方程二阶导数的经典错误
这是班上学生最常见的错误之一。我见过一个同学在作业里写：d^2y/dx^2 = (d^2y/dt^2) / (d^2x/dt^2)，然后算出一大堆错误结果。记住：二阶导数是先求出 dy/dx（作为 t 的函数），再对 t 求导，最后除以 dx/dt——这个"除以 dx/dt"的步骤绝不能省略。

一个帮助记忆的方式：d^2y/dx^2 可以理解为 d/dx (dy/dx)。既然 dy/dx 是 t 的函数，而 d/dx = (d/dt) / (dx/dt)，所以就有了上面的公式。

相关变化率问题

相关变化率（related rates）是隐函数求导在现实问题中的直接应用。核心思路是：两个物理量（比如圆的半径和面积）通过一个方程相关联，当一个量变化时，另一个量也以某种速率变化。我们通过对方程求导来建立两个变化率之间的关系。

经典问题：吹气球

问题：一个球形气球以 100 cm^3/s 的速率充气。
问当半径 r = 5 cm 时，半径的增长速率是多少？

球体积公式：V = (4/3)πr^3

已知 dV/dt = 100，求当 r = 5 时的 dr/dt。

两边对 t 求导（r 是 t 的函数）：
dV/dt = 4πr^2 · dr/dt

代入已知：
100 = 4π · 5^2 · dr/dt
100 = 100π · dr/dt
dr/dt = 1/π ≈ 0.318 cm/s

这个结果表明：当半径是 5 cm 时，半径以大约 0.318 cm/s 的速率增长。有趣的是，随着半径变大，同样的充气速率下，半径的增长速率会越来越慢——因为等体积的空气需要"覆盖"更大的表面积。

相关变化率的解题步骤

这类问题的通用解法可以概括为以下步骤：

  1. 理解问题：明确哪些量在变化，哪些是已知的变化率，哪些是要找的变化率
  2. 建立方程：找出所有相关量之间的几何或物理关系
  3. 隐式求导：方程两边对时间 t 求导（所有变量都视为 t 的函数）
  4. 代入数值：代入特定时刻的已知值，解出未知变化率
  5. 检查单位：确保结果单位正确
一个翻车的面试题
我在做 TA 的时候，有一次给新生出了一道相关变化率题："一架飞机在 2 km 的高空以 400 km/h 的速度水平飞行，经过雷达站正上方时，雷达与飞机的距离变化率是多少？"

有一个同学的答案让我哭笑不得——他算出了 400 km/h。但他没意识到，飞机刚刚经过正上方那一瞬间，距离的变化率是 0（因为距离在这个时刻达到最小值，最小值的导数为 0）。他犯的错误是忽略了"正上方"这个关键条件，直接用勾股定理求导时漏掉了关键步骤。这个故事告诉我们：相关变化率问题中，"先列方程再求导"的顺序绝对不能颠倒。

本章小结

这一章我们大大扩展了求导的"作战范围"。隐函数求导让我们能够处理那些无法写成 y = f(x) 形式的方程；对数求导法巧妙地化解了幂指函数和连乘除函数的求导难题；参数方程求导处理了用参数 t 间接描述的函数关系；而相关变化率则展示了隐函数求导在现实问题中的巨大威力。

下一章我们将对导数做进一步的深化——进入高阶导数与微分的领域。你会发现：仅仅求一次导往往不够，对位移求导得到速度，再求导得到加速度——这就是二阶导数最直观的物理意义。而微分则给我们提供了一个全新的视角：dy = f'(x)dx，它不仅是计算工具，更是一种思维方式。
