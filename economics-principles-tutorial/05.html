第5章 弹性及其应用

大家好，我是你们的讲师。上一章我们学会了用供给-需求模型分析市场价格。但你可能会问：降价10%能带来多少额外销量？涨价会不会反而让总收入下降？要回答这些问题，我们需要引入一个极其实用的概念——弹性（elasticity）。

本章学习目标：
  • 理解需求价格弹性及其决定因素
  • 掌握弹性与总收益的关系
  • 了解供给弹性、收入弹性与交叉弹性
  • 用弹性分析农业政策等现实问题

5.1 故事：谷贱伤农

故事时间：在中国农村有一句老话叫"丰年谷贱伤农"——大丰收的年份，农民收入反而下降。2008年，中国粮食产量创下历史新高，但许多粮农发现卖粮收入不如往年。为什么粮食多了反而赚得少了？这个问题的核心就是需求价格弹性。

粮食是典型的需求缺乏弹性商品——人们不会因为粮食便宜就多吃三碗饭。当供给曲线右移（丰收），价格大幅下跌，但需求量增加幅度很小。结果：价格跌幅 > 销量增幅 → 总收益下降。

5.2 需求价格弹性（Price Elasticity of Demand）

定义：需求量变动百分比 ÷ 价格变动百分比。

# 中点法计算弹性（midpoint method）
# 避免"从A到B"与"从B到A"结果不同的尴尬

def elasticity_midpoint(p1, p2, q1, q2):
    """
    中点法弹性公式：
    Ed = |(Q2-Q1)/[(Q1+Q2)/2] / (P2-P1)/[(P1+P2)/2]|
    """
    p_change = (p2 - p1) / ((p1 + p2) / 2)
    q_change = (q2 - q1) / ((q1 + q2) / 2)
    return abs(q_change / p_change)

# 例：苹果从5元涨到7元，购买量从100降到60
ed = elasticity_midpoint(5, 7, 100, 60)
print(f"需求价格弹性 = {ed:.2f}")
# 输出: 需求价格弹性 ≈ 1.50（富有弹性）

# 例：食盐从2元涨到3元，购买量从50降到48
ed2 = elasticity_midpoint(2, 3, 50, 48)
print(f"食盐需求价格弹性 = {ed2:.2f}")
# 输出: 食盐需求价格弹性 ≈ 0.14（缺乏弹性）

弹性的决定因素

  • 替代品多少：替代品越多，弹性越大（如水果）。替代品越少，弹性越小（如胰岛素）。
  • 必需品 vs 奢侈品：必需品弹性小（如水电），奢侈品弹性大（如珠宝）。
  • 市场范围：市场范围越小，弹性越大（如"蒙牛牛奶"比"所有牛奶"更有替代品）。
  • 时间跨度：长期弹性 > 短期弹性（消费者慢慢调整习惯）。

5.3 弹性与总收益

这是一个极其重要的实际应用。企业定价决策的核心就是弹性分析。

弹性值类型降价效果涨价效果
|Ed| > 1富有弹性总收益↑（销量增幅 > 价格降幅）总收益↓
|Ed| = 1单位弹性总收益不变总收益不变
|Ed| < 1缺乏弹性总收益↓总收益↑（价格增幅 > 销量降幅）
踩坑提醒：很多创业者一上来就打"价格战"——以为降价就能赚更多。错了！如果你的产品需求缺乏弹性（比如是一种独特的专业软件），降价反而会让总收益下降。反过来，奢侈品品牌反复涨价却越赚越多——因为它们的客户群体对价格不敏感。理解弹性，才能做出聪明的定价决策。

5.4 供给价格弹性

供给价格弹性衡量的是供给量对价格变化的敏感程度。

  • 完全无弹性（Es = 0）：无论价格怎么变，供给量不变。如：今天晚上的演唱会座位。
  • 缺乏弹性（0 < Es < 1）：供给调整慢。如：短期内的农产品。
  • 富有弹性（Es > 1）：供给调整快。如：长期内的大多数工业品。

供给弹性的关键决定因素是时间。时长短，供给弹性小（工厂来不及扩产）；时间长，供给弹性大（可以建新厂、招新人）。

5.5 收入弹性与交叉弹性

收入弹性（Income Elasticity of Demand）：衡量需求量对收入变化的敏感程度。

  • Ei > 0 → 正常品（收入涨，买得更多）
  • Ei < 0 → 低档品（收入涨，买得更少，如方便面）

交叉价格弹性（Cross-Price Elasticity of Demand）：衡量A商品需求量对B商品价格变化的敏感程度。

  • Exy > 0 → 替代品（牛肉涨价，猪肉需求量上升）
  • Exy < 0 → 互补品（汽油涨价，汽车需求量下降）

5.6 弹性在农业政策中的应用

回到"谷贱伤农"的问题。由于粮食的需求弹性很小（通常 |Ed| < 0.3），市场机制会导致农民"增产不增收"。这在世界各国都是一个政治敏感问题。政府的应对措施包括：

  • 价格支持：设定最低收购价，政府按支持价收购过剩粮食。
  • 直接补贴：不管市场价格如何，直接给农民发补贴（WTO允许的"绿箱政策"）。
  • 限制耕种面积：人为减少供给以维持价格。

理解弹性，你就理解了农业政策背后的经济学逻辑——不是政府"乱干预"，而是市场在特定弹性结构下会失灵。

5.7 本章小结

弹性是一个看似简单但威力无穷的工具。它能帮你回答很多实际商业决策问题：该降价还是涨价？竞争对手的价格变动会影响你多少？收入的增长会如何改变市场需求？
