大家好，我是你们的讲师。前面六章我们一直在研究静止的电荷。但现在，是时候让它们动起来了——不过别急，我们讨论的还只是恒定电流（Steady Current），即电荷在慢慢地、匀速地流动。就像你在高速公路上开车，虽然是运动状态，但如果速度恒定，物理规律和静止时的差别也没有那么大。真正的"加速度"（变化的电流）要等到电磁感应那一章才会出现。

电流与电流密度

电流（Electric Current）定义为通过某一截面的电荷量随时间的变化率：

I = dq / dt

单位是安培（A），1 A = 1 C/s。电流是一个标量——尽管我们常说电流"从正极流向负极"，但这指的只是一种约定方向（正电荷运动的方向）。实际上在金属导体中，运动的是电子（负电荷），所以电子的运动方向与电流方向相反。

更精细的描述需要电流密度（Current Density）J，它是一个矢量：

J = n q vdrift

其中 n 是载流子数密度（单位：m⁻³），q 是每个载流子的电荷量，vdrift 是漂移速度（Drift Velocity）。通过任意截面 S 的电流为：

I = ∫S J · dS

电流密度 J 才是描述电荷输运的"真主角"——它是微观量，不依赖于截面形状；而 I 是积分量，取决于你选择多大的截面。

连续性方程（Continuity Equation）是电荷守恒的数学表达：

∇ · J = −∂ρ / ∂t

对恒定电流来说，∂ρ/∂t = 0，所以 ∇·J = 0——这意味着电流线是连续的，没有源也没有汇。换句话说，在恒定电流下，流入任意闭合曲面的电流等于流出的电流，电荷不会在任何一点积累。

欧姆定律的微观本质

中学就学过的欧姆定律 V = IR，在微观层面有着丰富的物理内涵。微分形式的欧姆定律是：

J = σ E

其中 σ 是电导率（Conductivity），单位 S/m（西门子/米）。这个式子告诉我们：电流密度正比于电场强度，比例系数就是电导率。为什么？

在金属导体中，自由电子像气体一样在晶格之间做无规则热运动。加上外电场 E 后，电子在热运动的基础上获得一个"定向漂移"的增量。这个漂移速度由电场力和碰撞阻力平衡决定：

vdrift = −(eτ / me) E

其中 τ 是弛豫时间（Relaxation Time），即电子两次碰撞之间的平均自由时间。代入 J = nqvdrift 可得电导率：

σ = n e² τ / me

这就是德鲁德模型（Drude Model）的结果。虽然简单，但它成功解释了金属导电的基本规律。

银 (Ag) 1.59 × 10⁻⁸ 6.30 × 10⁷ +0.0038 最佳导体（昂贵）

这里有一个非常反直觉的事实：电子的漂移速度极其缓慢，通常只有 mm/s 量级（在 1 A/mm² 的电流密度下）。但电子的热运动速度约为 10⁶ m/s。也就是说，电子像一窝疯狂振动的蜜蜂，整体极其缓慢地朝一个方向"飘"去。那你可能会问：为什么我一按开关，灯立刻就亮了？

🔑 核心洞察：电场传播速度 vs 电子漂移速度
开关一打开，灯立刻亮——但这并不是电子从开关处"跑"到了灯泡处。实际上，是电场的传播速度（接近光速 ≈ 3 × 10⁸ m/s）沿着导线传播，推动了导线中所有的电子几乎同时开始运动。就像一条水管，你打开水龙头时，水几乎是立刻流出来的——但这不是因为水从水厂瞬间流到了你家，而是因为管道中已经充满了水，阀门一开压力波就传到了整个系统。
电子的漂移速度 ≈ 0.1 mm/s（比蜗牛还慢）
电场的传播速度 ≈ 3 × 10⁸ m/s（光速）
这两个数字相差 12 个数量级！如果你以为"电子从电源运动到灯泡"来解释"灯亮了"，那就大错特错了。

焦耳定律与功率耗散

电流通过导体时，电子与晶格不断碰撞，将定向动能转化为晶格振动（热能）。这就是焦耳热（Joule Heating）的微观机制。宏观上的焦耳定律为：

P = I² R = V² / R

单位：瓦特（W），1 W = 1 J/s。单位体积内的功率耗散（功率密度）为：

p = J · E = σ E²

焦耳热的应用无处不在——电热水器、白炽灯、电炉都是利用电阻发热。但更多的时候，焦耳热是我们要避免的负面效应：CPU 芯片因为焦耳热需要散热、输电线路的焦耳损耗能导致数兆瓦的功率浪费。

保险丝的原理正是利用焦耳热：当电流超过额定值，保险丝（低熔点合金）的温度达到熔点，熔化断开电路，保护后方设备。保险丝材料的典型选择是铅锡合金，熔点约 200°C，电阻率适中。

基尔霍夫定律

基尔霍夫定律是电路分析的基石，它包括两条定律：

基尔霍夫电流定律（KCL）

流入任意节点的总电流等于流出该节点的总电流。

数学表达：Σ Iin = Σ Iout，或 Σ I = 0（取流入为正值）。本质上是电荷守恒在恒定电流下的体现——电流线连续，不可能在节点处积累电荷。

基尔霍夫电压定律（KVL）

沿任意闭合回路，各元件电压降的代数和为零。

数学表达：Σ Vk = 0。本质上是静电场保守性（∮E·dl = 0）在电路中的体现。注意基尔霍夫电压定律来源于电场为保守场，在时变电磁场中（当有变化的磁场穿过回路时），需要修正为包含感应电动势的形式。

💡 KVL 符号规则——最容易犯错的地方
应用 KVL 时最容易出错的不是方程本身，而是符号约定。我建议使用这个一致的方法：
(1) 先选择回路绕行方向（顺时针或逆时针）。
(2) 沿绕行方向，如果从一个元件的"+"端经过元件走到"−"端，该元件的压降记为 +V；反之记为 −V。
(3) 对于电源：对于理想的电压源，从"−"到"+"经过电源时记为 −V（电压升），从"+"到"−"记为 +V（电压降）。
最稳妥的做法是：任何时候都坚持"电压降 = 正"的约定，然后根据电流参考方向与回路方向是否一致来判断电阻上的压降符号。多做几道题后就会形成直觉，一开始宁可写慢一点、检查符号，也不要图快。

RC 电路暂态

将电阻 R 和电容 C 串联，加上直流电压源 V₀，就构成了 RC 电路。当开关闭合时，电容开始充电，其端电压随时间变化：

VC(t) = V₀ (1 − e−t/RC)  （充电过程）

当电容已充满电（VC = V₀）后，断开电源并将电容两端短路放电：

VC(t) = V₀ e−t/RC  （放电过程）

乘积 τ = RC 称为时间常数（Time Constant）。τ 的物理含义是：在充电过程中，电容电压从 0 上升到 0.632V₀ 所需的时间（1 − e⁻¹ ≈ 0.632）；在放电过程中，电压下降到 0.368V₀ 所需的时间。

  • t = τ 时：充电到 63.2%，放电到 36.8%。
  • t = 3τ 时：充电到 95.0%，放电到 5.0%。
  • t = 5τ 时：充电到 99.3%，放电到 0.7%。

工程上一般认为经过 5τ 后，电路进入"稳态"——虽然严格来说，指数函数永远达不到精确的终值，但实际上 99.3% 和 100% 已经没有区别了。

# RC 电路暂态仿真
import numpy as np
import matplotlib.pyplot as plt

R = 1e3      # 1 kΩ
C = 1e-6     # 1 µF
V0 = 5.0     # 5V
tau = R * C  # τ = 1 ms

t = np.linspace(0, 5e-3, 500)  # 0 ~ 5ms

# 充电过程
V_charge = V0 * (1 - np.exp(-t/tau))

# 在 2.5ms 处开始放电
idx_switch = len(t) // 2
t_discharge = t[idx_switch:] - t[idx_switch]
V_discharge = V_charge[idx_switch] * np.exp(-t_discharge/tau)

plt.figure(figsize=(8, 4))
plt.plot(t[:idx_switch], V_charge[:idx_switch], 'b-', label='充电')
plt.plot(t[idx_switch:], V_discharge, 'r-', label='放电')
plt.axhline(y=V0*(1-1/np.e), color='gray', linestyle=':', label=f'0.632V₀ (τ={tau*1e6:.0f}µs)')
plt.axvline(x=tau, color='gray', linestyle=':')
plt.xlabel('时间 (s)')
plt.ylabel('电容电压 Vc (V)')
plt.title('RC 电路暂态响应')
plt.legend()
plt.grid(True)
plt.show()

print(f"时间常数 τ = R×C = {R}×{C*1e6:.0f}µF = {tau*1e6:.0f} µs")
print("充电到 5τ 后认为进入稳态")
⚠ 工程案例：PCB 上的 IR Drop
在设计一块大功率 PCB 时，工程师按照原理图检查了所有信号连接，认为设计没有问题。但板子打样回来后在远端芯片处发现供电电压只有 4.7 V 而不是设计的 5 V。检查后发现：从电源入口到远端芯片的铜箔走线长 15 cm，宽度仅 0.3 mm（35 µm 铜厚），估算电阻约 25 mΩ。远端芯片瞬态电流高达 3 A，压降 V = IR = 75 mV——但问题不止于此，还有共用的地回路。
实际上整个回路的总 IR Drop 达到了 300 mV，导致远端芯片供电不足。这个案例告诉我们：欧姆定律虽然简单，但在大电流下铜箔的 IR Drop 不可忽视。解决方案是增加走线宽度（或使用多层板中的电源平面）、缩短走线长度，以及在设计阶段做 DC-IR Drop 仿真。
顺便说一句，PCB 上常见的 1 oz（盎司）铜厚约为 35 µm，其方块电阻约为 0.5 mΩ/□。你可以估算：一条宽 1 mm、长 10 cm 的走线，电阻约为 0.5 × (100/1) = 50 mΩ。

恒定电流的场论视角

从电磁场的角度看，恒定电流电路中的电场满足：

  • ∇ × E = 0：电场仍然是保守场（静电场条件成立，因为磁场不变化）。
  • ∇ · J = 0：电流连续，无源无汇。
  • J = σE：本构关系。

这三个条件共同决定了电路中的电流和电压分布。实际上，整个电路理论（KCL、KVL）可以通过在特定拓扑结构（导线 + 节点）上对麦克斯韦方程组做集中参数近似（Lumped Element Approximation）推导出来。这个近似成立的条件是：电路尺寸远小于电磁波波长。对于低频电路（如 50 Hz 工频），波长是 6000 km，任何实际电路都满足这个条件。但对于 GHz 级的射频电路，集中参数近似不再成立，必须用分布参数电路（传输线理论）来处理——那是第 17 章的内容。

从电流到磁场

好了，我们花了 7 章篇幅来讨论电——从静止的电荷到运动的电荷（恒定电流）。现在我们已经有了足够的电学基础，可以去拜访电磁学的另一半了：磁。

奥斯特在 1820 年的一次课堂演示中偶然发现：通电导线旁边的指南针发生了偏转。这个发现震惊了当时的科学界——电和磁不是互不相关的现象，而是同一个硬币的两面。电流产生磁场——磁场的源就是运动的电荷。
