第19章 量化选股入门

大家好，我是你们的讲师。前面我们用了整整18章的篇幅，系统地学习了基本面分析、技术分析、投资组合理论和行为金融学。从这一章开始，我们将进入一个全新的世界——量化投资。

你可能觉得"量化"离散户很遥远，那是华尔街用几百亿资金玩的游戏。但事实上，过去十年量化投资在中国的普及速度远超大多数人的想象。截至2025年，A股市场的量化交易量占比已经达到30%左右，而早在2021年就曾突破40%。更重要的是，曾经需要大型IT团队才能搭建的量化系统，如今一个普通投资者用一台笔记本电脑加一些开源库就能跑起来。这不是天方夜谭——这门课接下来的7章，就是要带你从零到一，搭建属于自己的量化分析框架。

19.1 什么是量化选股

先给一个定义。量化选股（Quantitative Stock Selection），就是用数据和数学模型来驱动投资决策，而不是凭感觉、凭消息、凭"我觉得它要涨"。

听起来很玄乎？换个说法你就明白了。传统的非量化投资决策流程是这样的：

  • 看到一条利好新闻 → 觉得这家公司不错 → 看一眼K线图好像要突破 → 买入
  • 朋友推荐了一只股票 → 上网搜了搜好像确实有题材 → 买入
  • 这只股票跌了10% → 心里发慌 → 割肉

量化投资的流程则是这样的：

  • 定义一个选股逻辑（比如"低市盈率+高ROE"）→ 用历史数据回测 → 评估效果 → 如果有效，形成规则 → 严格执行
  • 市场条件变化了 → 回测新的逻辑 → 比较哪个更优 → 切换策略

这两者的本质区别是什么？纪律性。非量化决策充满了情绪和随机性，量化决策则是规则驱动的、可重复的、可验证的。

量化选股的优势总结起来有三个：

  • 纪律性：模型说买就买，说卖就卖，不掺杂"再等等看"的情绪干扰
  • 广度：一个量化模型可以同时覆盖全市场5000多只股票，人脑做不到这一点
  • 可复现：同一套规则在不同时间段的收益表现可以回测评估，策略的好坏是可检验的

说到量化投资，就绕不开一个人——詹姆斯·西蒙斯（James Simons）。这位数学教授出身的对冲基金经理，在1988年创立了大奖章基金（Medallion Fund），在1988年到2018年的30年间创造了约66%的年化收益率（扣除管理费前）。这个数字有多惊人？如果巴菲特的年化20%是"股神"，那西蒙斯的66%已经超越了"神"的范畴。不过需要说明的是，大奖章基金主要依靠高频交易和极其复杂的数学模型，普通投资者不可能复制。但西蒙斯证明了：数学和模型确实可以在金融市场中找到赚钱的规律。

19.2 因子投资的逻辑

因子（Factor）是量化选股最基础的概念。因子就是能够解释股票收益率差异的某种特征。

举个例子：假设你发现"市盈率低的股票平均比市盈率高的股票表现更好"，那么"市盈率"就是一个因子。再比如"过去一个月涨得好的股票接下来一个月继续涨"——"过去一个月的收益率"也是一个因子，通常叫"动量因子"。

因子投资的逻辑很简单：超额收益不是随机的，而是可以由某些系统性的因子来解释的。如果把股票的收益率拆开来看：

股票收益率 = 市场收益（Beta） + 因子收益（Alpha因子） + 个股特异性收益

这里的Alpha就是相对于市场的超额收益，而Beta就是市场整体的涨跌。因子投资的目标，就是找到能够稳定产生Alpha的那些特征。

从经济学角度，因子溢价有两种解释：

  • 风险补偿解释：某些因子之所以能产生超额收益，是因为它们承担了某种系统性风险。比如"低波动因子"——低波动股票之所以表现好，是因为投资者的"彩票偏好"让高波动股票被高估了。
  • 行为偏差解释：某些因子利用了市场参与者的系统性行为偏差。比如"价值因子"——投资者对成长股过度乐观导致其被高估，对价值股过度悲观导致其被低估。

实践中，一个优秀的量化策略往往不是靠单一因子，而是多因子组合。因为单个因子总会有"失灵期"，多个低相关性因子组合在一起，可以在不同市场环境下保持相对稳定的表现。

为了更直观地理解量化和主动投资的区别，我们来看一张对比表：

维度 量化投资 主动投资（非量化）
决策依据 数据、模型、统计检验 基本面研究、经验判断
覆盖广度 全市场5000+只股票 通常覆盖50-100只
决策方式 规则化、系统化 个案分析、主观判断
情绪影响 极低（由模型执行） 较高（容易受恐惧贪婪影响）
可复现性 完全可复现、可回测 难以复现（每次决策逻辑不同）
策略容量 通常较大（尤其低频策略） 取决于个股流动性
适用场景 规律性、重复性机会 深度研究、特殊情况
典型代表 西蒙斯、Two Sigma 巴菲特、彼得·林奇

关键认知：量化和主动投资不是非此即彼的关系。最好的投资者往往把两者结合起来——用量化框架做广度筛选，用主动研究做深度决策。这种"量化+主动"的融合模式，在业内被称为"系统化基本面投资"（Systematic Fundamental Investing）。

19.3 回测：量化的实验室

如果说量化策略是一座大楼，那么回测（Backtesting）就是这座大楼的结构力学测试。在没有投入真金白银之前，你需要先在历史数据上验证你的策略是否有效。

一个完整的回测流程包括以下步骤：

  1. 数据准备：获取历史行情数据（开盘价、收盘价、成交量等）和财务数据（PE、PB、ROE等）
  2. 信号生成：根据你的选股规则，在每个调仓日生成买入/卖出信号
  3. 组合构建：根据信号构建投资组合，确定每只股票的权重
  4. 绩效评估：计算策略在回测期内的收益率、波动率、最大回撤、夏普比率等指标

下面是一个非常简化的因子回测伪代码，展示了核心逻辑：

# 简化版因子回测框架伪代码
def factor_backtest(data, factor_name, holding_days=20):
    """
    data: DataFrame, 包含每只股票每个时点的因子值和收益率
    factor_name: 因子列名
    holding_days: 持有期天数
    """
    results = []

    # 在每个调仓日（每月末）
    for rebalance_date in get_monthly_end_dates(data):
        # 1. 获取当前时点的因子值
        current_data = data.loc[rebalance_date]

        # 2. 按因子值排序，分为10组（十分位组）
        current_data['decile'] = pd.qcut(
            current_data[factor_name], 10,
            labels=False, duplicates='drop'
        )

        # 3. 做多最高组（第9组），做空最低组（第0组）
        long_stocks = current_data[current_data['decile'] == 9].index
        short_stocks = current_data[current_data['decile'] == 0].index

        # 4. 计算持有期内的组合收益
        forward_returns = get_forward_returns(
            data, rebalance_date, holding_days
        )

        long_return = forward_returns.loc[long_stocks].mean()
        short_return = forward_returns.loc[short_stocks].mean()
        spread_return = long_return - short_return

        results.append({
            'date': rebalance_date,
            'long_return': long_return,
            'short_return': short_return,
            'spread_return': spread_return
        })

    # 5. 汇总评估
    df_results = pd.DataFrame(results)
    annual_return = df_results['spread_return'].mean() * 12
    annual_vol = df_results['spread_return'].std() * sqrt(12)
    sharpe = annual_return / annual_vol
    max_drawdown = calculate_max_drawdown(df_results['spread_return'])

    return {
        'annual_return': annual_return,
        'annual_vol': annual_vol,
        'sharpe_ratio': sharpe,
        'max_drawdown': max_drawdown,
        'monthly_returns': df_results
    }

但是，回测远没有看起来那么简单。回测的陷阱很多，新手几乎一定会踩，下面是最常见的几个：

  • 幸存者偏差（Survivorship Bias）：回测中只使用了当前还在上市的股票，忽略那些已经退市的股票。这会高估策略的表现——因为你只看到了"活下来"的股票。
  • 前视偏差（Look-ahead Bias）：在回测中使用了你当时不可能知道的信息。比如用2024年12月31日发布的年报数据来做2024年6月的选股决策——这在现实中是不可能的，因为年报发布有延迟。
  • 数据窥探偏差（Data Snooping）：反复在同样的数据上试验不同的参数组合，直到找到一个"表现最优"的。这种方式找到的策略在样本外几乎必然失效。
  • 参数过度优化（Overfitting）：把参数调整到完美拟合历史数据，但丧失了泛化能力。

19.4 过拟合：量化最大的敌人

如果说量化选股有一个"第一公敌"，那非过拟合（Overfitting）莫属。

过拟合在机器学习里是指模型在训练数据上表现很好，但在新数据上表现很差。在量化投资中，过拟合意味着你的策略在回测中看起来很完美，但一到实盘就露馅了。

踩坑故事：回测年化50%，实盘亏钱的经典案例

2017年前后，国内量化私募刚刚兴起。有一家当时颇有名气的私募团队，开发了一套"基于技术指标的多因子策略"。回测做到了惊人的年化50%，夏普比率超过3，最大回撤不到10%。团队信心满满地发行了产品，自营资金加客户资金一共投入了5个多亿。

结果怎么样？实盘前三个月就亏了8%。团队一开始以为是执行问题，各种调试优化。等到第六个月，产品净值已经跌了15%。而同期回溯同一策略在最新的样本外数据上——年化收益变成了-5%。

问题出在哪？事后复盘发现了三个致命错误：第一，回测中使用了未来函数，计算均线时无意中包含了未来数据；第二，策略依赖了大量参数调整，在2008-2016年数据上反复"调参数"直到完美，实际上是在拟合那个特定历史区间的噪音；第三，完全没有考虑交易成本，实盘后发现频繁换手导致交易成本吞噬了大部分收益。

这个故事的教训：任何策略如果不是用严格的样本外测试验证过，它在回测中展示的收益率约等于零。量化界有一句经典的话——"回测是最大回撤最低的时期，也是收益率最高的时期。"（笑）

19.5 从回测到实盘的鸿沟

即使你的回测做得天衣无缝，从回测到实盘之间仍然存在一条巨大的鸿沟。主要的实盘摩擦成本包括：

  • 交易成本：佣金、印花税、冲击成本。回测中往往假设以收盘价无成本交易，实盘中做不到。
  • 滑点（Slippage）：你的订单提交时的价格和成交价格之间的差异，尤其是在快速变动的市场中。
  • 流动性限制：涨跌停时无法交易，停牌股票无法买卖，小市值股票冲击成本高。
  • 资金容量：一个在小资金下表现很好的策略，资金量上去后收益会被冲击成本吃掉。

业内有一个经验法则：回测夏普比率打八折，才是真实的预期夏普。如果你的回测夏普是2.0，预期实盘夏普大概在1.6左右。如果回测夏普只有1.0，打八折后只有0.8，那这个策略的性价比就存疑了。

为什么是八折？这个数字来自于大量私募实盘数据的统计。回测中可以完美控制的各种微观结构摩擦，在实盘中都会变成实实在在的成本。有人做过更严格的计算：A股市场单边交易成本（佣金+印花税+冲击成本）平均在0.1%-0.3%之间，一个每年换手50倍的策略，仅交易成本就会吃掉5%-15%的收益。

核心启示：在评估一个量化策略时，不要把回测数据当作"预期收益"，而是当作"理想情况下的收益上限"。真正的考验永远在实盘。这也是为什么量化团队通常会用"模拟盘"（Paper Trading）来作为中间环节——在不投入真金白银的情况下，看看策略在实时市场中的表现。

小结与预习

这一章我们打开了量化投资的大门。我们讨论了：什么是量化选股及其纪律性/广度/可复现的优势；因子投资的基本逻辑——因子是超额收益的驱动基因；回测作为量化实验室的基本流程和常见陷阱；以及过拟合这个量化最大的敌人。最后我们谈到了从回测到实盘之间不可忽视的鸿沟。
