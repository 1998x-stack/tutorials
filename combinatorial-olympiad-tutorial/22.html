第二十二章：染色问题

大家好，我是你们的讲师。欢迎来到我们"专题深入"部分的最后一章。到今天为止，我们已经学了计数、递推、抽屉原理、母函数、图论、Ramsey理论。现在我们要学一个看似简单、但在竞赛中极其强大的技巧——染色法。

你可能从小学奥数就接触过染色——比如给棋盘涂黑白两色来证明某些覆盖问题不可能。但竞赛中的染色法远不止如此。它实际上是一种构造不变量的方法：通过给对象赋予颜色，把原本模糊的"能不能"问题，转化为颜色数量的代数关系，从而用奇偶性或模运算来得到矛盾。

说得直白一点：染色法的本质是制造一个可以计算的"陷阱"——让目标结构在染色下必然满足某种等式，而试图构造的结构却会破坏这个等式。

22.1 棋盘染色：三种基本模式

棋盘染色是染色法中最经典、最直观的一类问题。下面三种模式几乎覆盖了所有棋盘染色题。

22.1.1 黑白交替（国际象棋棋盘染色）

这是最基础的染色方式——像国际象棋棋盘一样，按"黑-白-黑-白"交替染色。通常，相邻的格子颜色不同。

例1：一个8×8棋盘被剪掉对角上的两个格子（一黑一白），是否能用31个2×1的多米诺骨牌完全覆盖？

分析：8×8棋盘有32黑和32白。每个多米诺骨牌恰好覆盖一黑一白。如果剪掉一黑一白，剩下的棋盘有31黑31白。表面上看，31个多米诺覆盖31黑31白——数量和颜色都匹配，似乎可行？

但问题的关键在于：剪掉的如果是对角上的两个格子——比如左上角和右下角——这两个格子同色！因为左上角(1,1)为黑色，右下角(8,8)坐标和为16也是黑色。剪掉两个同色格，剩下的是32黑30白（或30黑32白）。每个多米诺覆盖一黑一白，所以可覆盖的黑色数必须等于白色数——但现在不平衡了，所以不可能完全覆盖。

关键点：棋盘染色把"覆盖问题"转化为"颜色数量是否相等"的问题。如果两种颜色的格子数不同，用固定覆盖一黑一白的骨牌就不可能完全覆盖。这是染色法最简单也最有效的应用。

22.1.2 模k周期染色

当2-染色（黑白两色）不够用时，可以考虑模k染色。比如给每个格子赋予它在某种模数下的余数作为颜色。

例2：一个n×n棋盘，能否用1×3的L型三格骨牌（形如"↴"）完全覆盖？

分析：对n×n棋盘按(i + 2j) mod 3进行染色。每个1×3的L型骨牌在不同方向上覆盖的三个格子，其(i+2j) mod 3的值分别是0、1、2各一个（经过验证）。因此，每种颜色的格子数必须相等，棋盘才能被覆盖。这就要求n²能被3整除，即n是3的倍数。可以证明，当n是3的倍数时确实可以覆盖。

22.1.3 按坐标和染色

这是最灵活的一种染色方式：给坐标(i,j)赋予某种函数值f(i,j)作为颜色。常见的函数包括 i+j、i、j、i×j 等。

例3：证明用2×2的正方形骨牌不能恰好覆盖一个(m×n)棋盘，其中m和n都是奇数。

分析：用坐标和的奇偶性染色(i+j mod 2)。2×2的骨牌总是覆盖2黑2白。但m和n都是奇数时，棋盘有 mn 个格子，是奇数。而2×2骨牌覆盖4个格子——如果完全覆盖，格子总数应为4的倍数。但更精细的分析是：奇数×奇数的棋盘中，黑白格子数相差1（具体哪种颜色多一个取决于起始格的颜色）。设黑色有B个，白色有W个，则|B-W| = 1。但每个2×2骨牌覆盖的B和W数量相等，所以所有骨牌覆盖的B和W也相等，矛盾。

方法论：选择什么染色方式？通常遵循以下原则：

  1. 骨牌/覆盖件的形状——如果每个覆盖件覆盖k个格子，尝试模k染色
  2. 覆盖件的"步长"——如果覆盖件跨过一定距离的格子，尝试按坐标和模某个数染色
  3. 对称性要求——如果问题有旋转对称性，尝试用复数或模长作为颜色

22.2 染色与不变量结合

染色法最强大的时候，是它将染色后的某种数值保持不变，而操作会改变这个数值，从而产生矛盾。这就是"染色不变量"的思想。

定理 22.1（染色不变量的基本原理）

给每个对象赋予一个数值（"颜色值"），使得每次操作不改变所有对象数值之和的某种模性质。如果初始状态和目标状态的数值之和不匹配，则不可能通过操作完成转换。

22.2.1 经典案例：15拼图

著名的15拼图（15-puzzle）中，通过滑动方块，能否从初始排列到达目标排列？答案是否定的——大约一半的排列是不可达的。

染色不变量分析：将方格按国际象棋棋盘染色，并定义排列的奇偶性（逆序数的奇偶性）与空格位置的奇偶性之和。每次滑动方格都会同时改变排列的奇偶性和空格位置的奇偶性，所以这个和是不变量。如果初始和目标状态的这个和不相等，就不可能到达。

22.2.2 IMO 1995 经典染色题

IMO 1995 第4题（经典）

一个 p×p 的棋盘被分割成 p² 个单位正方形。在每一步中，你可以选择一个 r×r 或 s×s 的正方形子棋盘（r,s 为正整数，r≠s），并翻转它的所有格子的颜色（白色变黑色，黑色变白色）。初始状态下的所有格子都是白色。问：是否可能经过若干步操作，使得最终只有正中心的那个格子是黑色的？

分析：我们定义一个不变量。给坐标为(i,j)的格子赋予一个权重 w(i,j) = aibj（模某个数），其中 a, b 的选择使得对 r×r 和 s×s 的子正方形，其内部权重之和为零。用数学语言表达：

设 p 为大于 max(r,s) 的质数，在有限域 F_p 上定义：
w(i,j) = α^i · β^j

其中 α, β 是 F_p 中满足下式的元素：
(1 + α + α² + ... + α^{r-1})(1 + β + β² + ... + β^{r-1}) = 0
(1 + α + α² + ... + α^{s-1})(1 + β + β² + ... + β^{s-1}) = 0

这两个条件等价于选择 α, β 分别为 r 次和 s 次单位根。
因为 r 和 s 都不等于 p，这样的 α, β 在 F_p 的扩域中存在。

更具体地说：考虑模 2。每个操作翻转一个子正方形内的所有格子——等价于每个格子的颜色值（白=0,黑=1）加上该正方形内的"特征函数"模2。关键是怎么判断这个操作是否可能实现目标状态。

更具体地说：考虑模 2。每个操作翻转一个子正方形内的所有格子——等价于每个格子的颜色值（白=0,黑=1）加上该正方形内的"特征函数"模2。关键是怎么判断这个操作是否可能实现目标状态。

事实上，这个问题的完整解法需要用到多项式理论和代数方法——它比表面看起来要深刻得多。最终结论是：当且仅当 r 和 s 都是奇数时，目标状态才可能达到。

这个问题的染色不变量构造非常巧妙：取一个足够大的质数 p > r,s，在有限域 Fp 上考虑多项式 P(x,y) = (1+x+...+xr-1)(1+y+...+yr-1) 和 Q(x,y)。最终通过比较多项式系数来得到结论。

22.3 边染色与Ramsey型结论

上一章我们深入讨论了Ramsey理论，其中"边染色"就是最核心的工具。这里我们再补充几个与边染色相关的竞赛技巧。

例4：给定n个点的完全图，每一条边被染成红色或蓝色。证明：图中要么存在一个顶点，它关联的所有边中至少有一个颜色占了全部边的2/3以上。

分析：这个结论实际上是Turán型定理的一个简单推论。对任意顶点v，设其红边数为dr(v)，蓝边数为db(v)，dr(v)+db(v)=n-1。如果每个顶点的红边数和蓝边数都严格小于2(n-1)/3（即(n-1)/3 < dr(v) < 2(n-1)/3），则所有顶点的红边总和在特定范围内。但总的红边数∑dr(v) = 2×(红边总数)。通过算两次可以推出矛盾。

这个例子告诉我们：边染色也可以和"算两次"原理结合——通过从"边的角度"和"点的角度"分别计算同色边的数目，列出不等式来得到所需结论。

📖 踩坑实录：有一次我带学生做一道染色题："一个n×n棋盘，每个格子可以涂红、蓝、绿三种颜色之一。如果任意2×2子正方形中的四种颜色都不全相同（即有两种颜色出现，每种至少一个），问n最大是多少？"

一个非常聪明的学生很快找到了n=5时的构造，并试图证明n≥6时不可能。他用的是"按列染色分析"方法——假设n=6，考虑第一列的6个格子。因为2×2的限制，相邻行的颜色关系受到严格约束。他分析了所有可能的情况后发现——实际上n≥6是可能的！他的"不可能性证明"漏掉了一个关键的染色模式。

他的教训是：染色问题中，构造和证明"不可能"同样重要。有时候构造比证明更难——你找到了一个构造，确认了临界值；有时候证明更难——你需要排除所有可能的染色方案。在考场上，如果时间有限，先试着构造一下，看看"可能"的最大值是多少，再尝试证明大于这个值不可能。两条腿走路比一条腿快。

22.4 IMO 2011 染色问题赏析

来看一道近年的IMO真题：

IMO 2011 第2题

设 S 是平面上的有限点集，其中任意三点不共线。将 S 中的某些点染成红色，其余点染成蓝色。如果一条以 S 中的点为端点的线段，其两端点颜色不同，则称其为"好线段"。求最好的染色方案中，好线段数量的最大值的最小值——即无论如何染色，好线段至少有多少条？

分析思路：

设总共有r个红点，b个蓝点，r+b=n。好线段的条数就是红点和蓝点之间连线的总数，即r×b。

但问题没有这么简单——上面的计数是对固定的染色方案而言的。题目问的是："最好"的染色方案中，好线段数量的最小值是多少？换句话说，无论你怎么选择哪些点染红、哪些点染蓝，好线段数都不会低于某个数M，求这个M的最大可能值。

实际上，这个问题的关键是要证明：对任意染色方案，好线段数至少为______。而我们需要找出让这个"至少"最大的n。

这个问题的详细解法超出了本课的范围，但我想通过它说明一点：染色题往往不是孤立的技巧问题，它经常和极值图论、计数论证、算两次等组合工具配合使用。如果你只会"黑白交替"一个技巧，那IMO级别的染色题你多半做不出来。

22.5 染色法的套路总结

经过这么多例子，我们来总结一下——什么时候该想到染色？

问题特征 染色模式 典型案例
用某种形状覆盖棋盘 棋盘黑白交替或模k染色 多米诺覆盖、L形骨牌覆盖
操作改变格子颜色/状态 染色不变量（赋值法） 15拼图、翻转游戏
图上关系/认识关系问题 边染色（红蓝两色） Ramsey型问题
想证明某种配置不可能 构造冲突的奇偶性条件 棋盘剪角问题
数值分配/公平划分问题 加权染色（赋予不同权值） IMO 1995, IMO 2011

染色法的本质：染色不是目的，而是手段。你染色的目的，是制造一个可以用代数或计数方法处理的不变量。当你面对一个问题感到无从下手时，问自己：我能给这些对象染上颜色，使得每次操作前后某个量保持不变吗？如果某个量在操作中保持不变——而目标状态又和初始状态的这个量不同——那目标就是不可达的。
