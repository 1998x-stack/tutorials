第17章 量化投资基础

你是否有过这样的经历：看到一个"策略"在过去5年实现了年化30%的收益，但自己一跟投就开始亏钱？或者，你听说过量化投资、因子模型、统计套利这些词，但觉得它们离你很遥远，是"只有华尔街精英才能玩的东西"？

量化投资听起来很"高大上"，但它的核心思想其实很简单：用系统化的方法替代情绪化的决策。一个量化策略本质上就是一套"如果A条件满足，就做B操作"的规则——只不过这套规则是基于历史数据的统计分析得出的，而不是"我觉得明天会涨"的主观判断。

本章将从因子投资、统计套利、回测方法论、过拟合防范、评估指标和Python工具六个方面，带你进入量化投资的世界。我们不会写复杂的数学模型——核心是理解量化思维的逻辑。

西蒙斯大奖章基金的传奇

如果要选一个量化投资史上最传奇的人物，那一定是詹姆斯·西蒙斯（James Simons）。这位传奇的数学家、前美国国家安全局密码破译专家，在1988年创立了文艺复兴科技公司的大奖章基金（Medallion Fund）。从1988年到2018年，大奖章基金实现了年化66%的费前收益率（扣除费用后约39%），这是人类投资史上无可争议的最佳业绩纪录。

大奖章基金的秘密是什么？不是基本面分析，不是内幕消息——而是从海量的市场数据中发现统计上的"规律"（也即市场"异象"），然后用算法来交易这些规律。西蒙斯的成功证明了：如果方法得当，量化交易可以持续战胜市场。需要注意的是，大奖章基金的策略严格保密，至今无人完全复制其成功。

普通投资者的"量化第一步"：你不必成为数学家才能做量化投资。从最简单的策略开始——比如"每月初买入低波动率排名前10的股票，持有一个月后换仓"——这就是一个可测试的量化策略。关键在于：写下来，回测，执行，复盘。

17.1 因子投资

因子投资（Factor Investing）是量化投资最基础、最成熟的领域。一个"因子"就是一个能够解释股票收益的特征变量。学术研究发现，某些因子在长期能够持续产生正的风险溢价。

五大经典因子

因子定义逻辑A股表现
价值因子（Value）低市盈率、低市净率的股票"便宜"的股票有安全边际，市场对负面信息反应过度2017年前有效，近年效果减弱
规模因子（Size）小市值股票小公司有更高的增长潜力和风险溢价显著但波动大，2021年后阶段性失效
动量因子（Momentum）过去6-12个月涨幅高的股票趋势具有惯性，信息缓慢扩散短周期（1-3个月）动量明显，长周期动量不稳定
质量因子（Quality）高ROE、低杠杆、稳定的盈利"好公司"有更低的破产风险和更高的长期回报长期有效，尤其在熊市中表现突出
低波因子（Low Volatility）历史波动率最低的股票低波动股票被机构投资者"过度避险"，实际回报却更高（"低波异象"）A股中低波因子表现稳定，夏普比率高

多因子组合

单一因子的表现在不同市场环境下可能差异很大。比如2021年前价值因子在A股表现优异，但2021-2023年随着"赛道投资"的兴起，成长因子大幅跑赢价值因子。实务中，量化机构会将多个因子组合成一个"多因子评分模型"：

# 多因子评分模型示例
        import pandas as pd
        import numpy as np

        # 假设我们有10只股票的因子数据
        stocks = pd.DataFrame({
            'code': ['000001', '000002', '000003', '000004', '000005'],
            'pe': [8, 15, 30, 12, 50],        # 市盈率（值越低越好）
            'momentum_6m': [0.05, 0.12, -0.08, 0.20, 0.30],  # 6个月动量
            'roe': [0.15, 0.12, 0.08, 0.20, 0.05],            # ROE
            'volatility': [0.25, 0.18, 0.35, 0.22, 0.40]      # 波动率
        })

        # 因子标准化（转换为0-1之间的排名分位）
        for col in ['pe', 'momentum_6m', 'roe', 'volatility']:
            # 注意：pe和volatility是值越小越好，取1-排名分位
            if col in ['pe', 'volatility']:
                stocks[f'{col}_score'] = 1 - stocks[col].rank(pct=True)
            else:
                stocks[f'{col}_score'] = stocks[col].rank(pct=True)

        # 综合得分（等权）
        score_cols = [c for c in stocks.columns if c.endswith('_score')]
        stocks['total_score'] = stocks[score_cols].mean(axis=1)

        # 选出得分最高的2只股票
        selected = stocks.nlargest(2, 'total_score')
        print("选中的股票:", selected['code'].tolist())
        # 输出: 选中的股票: ['000002', '000004']

17.2 统计套利

统计套利（Statistical Arbitrage, StatArb）是基于统计学原理的套利策略。它不像传统套利那样追求"无风险"——而是利用统计规律发现资产之间的定价偏离，押注于这种偏离最终会回归。

配对交易（Pairs Trading）

配对交易是统计套利最经典的策略，由摩根士丹利的量化团队在1980年代开发。核心思想是：找到两只历史上价格走势高度相关的股票，当它们的价差（Spread）偏离到统计上的极端水平时，做多被低估的一只、做空被高估的一只，押注价差回归均值。

# 配对交易的简化示例
        import numpy as np
        import pandas as pd
        from statsmodels.tsa.stattools import coint

        # 模拟两只相关股票A和B的价格序列
        np.random.seed(42)
        t = 500  # 500个交易日

        # 共同趋势
        common = np.cumsum(np.random.normal(0, 0.01, t))
        # 股票A = 共同趋势 + 噪声
        stock_a = 100 + common + np.random.normal(0, 0.5, t)
        # 股票B = 共同趋势 * 1.5 + 噪声（B的波动更大）
        stock_b = 150 + common * 1.5 + np.random.normal(0, 1.0, t)

        # 协整检验
        # 如果两只股票存在协整关系，说明价差是平稳的
        score, pvalue, _ = coint(stock_a, stock_b)
        print(f"协整检验 p-value: {pvalue:.4f}")

        if pvalue < 0.05:
            # 计算价差（价差 = B - beta * A）
            beta = np.cov(stock_a, stock_b)[0,1] / np.var(stock_a)
            spread = stock_b - beta * stock_a
            spread_z = (spread - np.mean(spread[-100:])) / np.std(spread[-100:])

            # 开仓信号：z-score超过 ±2 时开仓
            if spread_z[-1] > 2:
                print("信号：做空价差（卖B买A）")
            elif spread_z[-1] < -2:
                print("信号：做多价差（买B卖A）")
            else:
                print("价差在正常范围内，不交易")
        else:
            print("两只股票不存在协整关系，不做配对交易")

均值回归策略

均值回归（Mean Reversion）是另一类重要的统计套利策略，基于"资产价格在短期内会回归其均值"的观察。最典型的应用是布林带策略（Bollinger Bands）和反转策略（Reversal Strategy）。但需要注意的是，均值回归在趋势明显的市场（如2023年AI主线行情）中会持续亏损——这提醒我们：没有任何一个量化策略是"万能钥匙"。

17.3 回测方法论

回测（Backtesting）是量化投资中最关键也最容易"骗自己"的环节。一个好的回测框架应该能真实反映策略在"实战"中的表现。

回测框架的核心要素

  • 数据清洗：处理停牌、退市、涨跌停、复权价格（前复权还是后复权？）
  • 交易成本：包括佣金、印花税（A股当前为0.05%）、滑点（建议至少用0.1-0.2%的双边成本）
  • 冲击成本：大资金成交时对价格的冲击——这也是为什么很多"看起来完美"的回测实盘就失效的元凶
  • 成分股变动：指数调仓日，策略是否提前"知道"了未来加入的股票？

存活偏差（Survivorship Bias）

回测中最隐蔽的陷阱之一。如果你只用"当前还在交易"的股票来回测，就忽略了那些已经退市的股票——而这些退市股票通常是表现最差的。结果：回测收益被系统性高估。正确的做法是使用历史全样本来回测，包括那些后来退市的股票。

前瞻偏差（Look-Ahead Bias）

在回测中"提前知道"了未来信息。比如使用财务数据时，你需要在财报公布日后才能使用该数据。如果用了一季度末的财务数据但假设在季度末就进行了交易——实际上财报要到4月底才披露完毕——这就产生了前瞻偏差。

经典踩坑：我曾见过一个"完美的量化策略"——在2015-2022年实现了年化45%的收益，回撤不到10%。仔细分析后发现：策略用到了"当日收盘价"来生成次日开盘的交易信号（即未来函数）。修正这个偏差后，年化收益直接从45%掉到了8%。这就是量化回测中"看似完美，实则虚假"的典型案例。

17.4 过拟合陷阱

过拟合（Overfitting）是量化投资中"万恶之源"。你不断调整参数、增加过滤条件、尝试不同的因子组合，最终得到了一份看起来"完美"的回测报告——但一到实盘就"见光死"。

多重检验偏差

你从100个候选因子中挑选了回测中表现最好的3个，然后兴奋地宣布"找到了一个有效的策略"。但统计学告诉我们：如果从100个完全随机的因子中选3个，有接近95%的概率会选出统计上"显著"的因子——纯粹因为运气。这就是著名的多重检验偏差（Multiple Testing Bias），也叫"数据挖掘偏差"。

# 多重检验偏差的模拟
        import numpy as np

        def simulate_false_strategy():
            """模拟100个完全随机的因子，看看能找出几个"显著"的"""
            np.random.seed(42)
            n_factors = 100
            n_periods = 120  # 10年月度数据
            significant = 0

            for i in range(n_factors):
                # 生成完全随机的收益率（均值为0的噪声）
                returns = np.random.normal(0, 0.05, n_periods)
                # t检验：看均值是否显著不为0
                t_stat = np.mean(returns) / (np.std(returns) / np.sqrt(n_periods))
                if abs(t_stat) > 1.96:  # 5%显著性水平
                    significant += 1

            return significant

        # 运行1000次模拟
        results = [simulate_false_strategy() for _ in range(1000)]
        avg_significant = np.mean(results)
        print(f"平均发现"显著"因子的数量: {avg_significant:.1f}")
        # 输出结果通常在5个左右——全是假的

防范过拟合的方法

  • 样本外测试：将数据分为训练集（前70%）和测试集（后30%），只能在测试集上"验证"一次
  • 交叉验证：在时间序列上做滚动交叉验证，确保策略在不同时间段都有效
  • 减少参数数量：一个有5个自由参数的策略，比只有1个参数的策略过拟合风险高得多
  • 经济逻辑检验：任何策略必须有经济含义——如果它有效但"不知道为什么有效"，大概率是过拟合
  • 夏普比率裁剪：实盘夏普比率通常是回测夏普比率的40-60%。如果回测夏普是3.0，请做好实盘只有1.2-1.8的准备

17.5 策略评估指标

判断一个量化策略的好坏不能只看"收益率"。以下是一组更完整的评估指标体系：

指标公式含义优秀标准
夏普比率(Rp - Rf) / σp每承担一单位总风险获得的超额回报>1.0 良好，>2.0 优秀
最大回撤从峰值到谷底的最大跌幅策略在最差情况下会亏多少<20%（股票策略），<5%（套利策略）
Calmar比率年化收益 / 最大回撤承受最大亏损后能获得多少收益>2.0 良好，>3.0 优秀
信息比率(Rp - Rb) / TE每承担一单位跟踪误差获得的超额回报（衡量主动管理能力）>0.5 良好，>1.0 优秀
胜率盈利交易次数 / 总交易次数一笔交易赚钱的概率趋势策略<50%也正常，高频策略>60%
盈亏比平均盈利 / 平均亏损赚的时候赚多少，亏的时候亏多少>2.0 良好
实战中的"霍华德·马克斯法则"：橡树资本的霍华德·马克斯（Howard Marks）曾提出一个尖锐的观点——"如果某个策略只有在回测中表现完美，而在实盘中表现糟糕，那它从来就不是一个好策略，只是回测设计者的自我欺骗。"判断策略质量的最好方法不是看回测曲线，而是：打开"交易记录"看每一笔交易的逻辑。如果10笔中有8笔"赚钱的理由"经不起推敲，这个策略大概率是过拟合的产物。

17.6 Python量化环境

最后，简单介绍一下目前主流的Python量化工具生态：

数据处理：NumPy & Pandas

这是量化分析最基本的"基础设施"。NumPy提供高效的数组运算，Pandas提供时间序列数据处理能力。基本上所有量化策略的代码都始于：

import numpy as np
        import pandas as pd
        import matplotlib.pyplot as plt
        import akshare as ak  # A股数据源

        # 下载贵州茅台的历史数据
        df = ak.stock_zh_a_hist(symbol="600519", start_date="20200101",
                                adjust="qfq")  # 前复权
        df['Date'] = pd.to_datetime(df['日期'])
        df.set_index('Date', inplace=True)
        df['returns'] = df['收盘'].pct_change()
        print(df[['收盘', 'returns']].head())

回测框架

  • Backtrader：最受欢迎的开源回测框架。支持多数据源、多时间周期、佣金模型、滑点模型。学习曲线中等，文档完善。
  • Qlib：微软开源的中国AI量化平台，内置了机器学习模型库（LSTM、Transformer等），支持特征工程和自动因子挖掘。对有一定编程基础的投资者来说非常强大。
  • Zipline：Quantopian的经典回测引擎，虽然Quantopian已关闭，但Zipline仍是学习回测框架设计的优秀参考。
新手建议：不要一开始就搭建复杂的回测管道。先用Pandas手动实现一个最简单的策略回测（比如"每月初买入中证500 ETF，持有到月底卖出"），理解回测的每一个步骤。等理解了基本的回测逻辑后，再用Backtrader这样的框架来"上规模"。很多量化新手犯的错误是：花3个月搭了一个"专业级回测平台"，结果只测了3个策略就放弃了——从简单开始，迭代改进。

量化投资不是"点金术"，而是一套系统化的投资方法论。它最大的价值不是创造"完美的交易策略"，而是帮你建立纪律性、可复制的投资决策流程。记住一个简单的真理：量化策略最强大的敌人不是市场，而是你自己的贪婪和恐惧——代码不会恐慌，但写代码的人会。下一章，我们将讨论一个最近几年最受关注也最富争议的投资话题——ESG与可持续投资。你会了解为什么"投资向善"不仅是一种情怀，也可能是一条通往超额收益的路径。
