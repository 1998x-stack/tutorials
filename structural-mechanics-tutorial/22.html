第二十二章 能量原理与近似方法

在整个结构力学的学习中，我们始终在做一件事：建立方程并求解。力法以力为未知量，需要平衡+协调；位移法以位移为未知量，需要平衡+物理+几何。但你是否想过——有没有一种更"根本"的方式来看待结构行为？不需要列平衡方程、不需要画隔离体图，而是直接从一个标量（能量）出发，推导出结构的控制方程？

答案是肯定的。这就是能量原理（Energy Principles）。

如果你学过分析力学，你一定熟悉"拉格朗日方程"和"哈密顿原理"——它们用能量来重新表述力学定律，优雅而统一。结构力学中的能量原理是分析力学在弹性结构中的具体应用。更重要的是，能量原理构成了有限元法（FEM）的理论基石。我们之前在第13~14章学的矩阵位移法，本质上就是有限元法的一个特例——用能量原理来系统化地建立单元刚度矩阵和总体刚度矩阵。

本章我们就来系统学习能量原理——从应变能和势能的概念出发，到最小势能原理，再到里兹法这个强大的近似求解工具，最后看看能量原理如何"演化"为有限元法。这一章将帮助你建立"从上往下看"结构力学的视野——当你掌握了能量原理，你会发现前面学过的很多方法都可以从一个统一的角度重新理解。

应变能与外力势能

在开始之前，我们需要清楚地定义两个核心能量概念。

应变能（Strain Energy）

当结构在外力作用下发生变形时，外力做功转化为结构内部的应变能——就像你拉一根橡皮筋，弹力势能储存在橡皮筋内。对于线弹性结构，应变能等于外力在加载过程中所做的功。

对于不同类型的变形，应变能的计算公式不同：

变形类型 应变能公式 说明
轴向拉压 U = ∫(N²/(2EA)) dx N为轴力，EA为抗拉刚度
弯曲 U = ∫(M²/(2EI)) dx M为弯矩，EI为抗弯刚度
剪切 U = ∫(αV²/(2GA)) dx V为剪力，α为剪应力分布系数
扭转 U = ∫(T²/(2GJ)) dx T为扭矩，GJ为抗扭刚度

对于一般杆系结构，总应变能是各部分之和：

    U = (1/2) ∫ (N²/(EA) + M²/(EI) + αV²/(GA) + T²/(GJ)) dx

    如果用位移表示（对梁的弯曲）：
    U = (1/2) ∫ EI·(d²u/dx²)² dx

外力势能（Potential Energy of External Forces）

外力势能是外力在结构变形过程中所做的功的负值。对于保守力系（即外力大小不随变形路径而变），外力势能的计算很简单：

    V = - Σ Pᵢ·uᵢ - ∫ q(x)·u(x) dx

    其中：
    Pᵢ —— 集中力
    uᵢ —— 集中力作用点的位移（沿力方向）
    q(x) —— 分布荷载

注意，外力势能有负号——这是因为"系统势能"的定义是：当外力使结构变形时，结构的势能减少（外力做正功，势能降低）。

总势能（Total Potential Energy）

定义一个弹性结构的总势能 Π为应变能与外力势能之和：

    Π = U + V

这个看似简单的标量函数，是整个能量法的核心。后面所有的方法都从这里出发。

最小势能原理

最小势能原理（Principle of Minimum Potential Energy） 是结构力学中最基本的变分原理之一。它说：

"在所有满足位移边界条件的可能位移场中，真实的位移场使结构的总势能Π取最小值。"

用数学语言表达：

    δΠ = δ(U + V) = 0

    其中δ表示"变分"——即对位移函数做微小变化（满足边界条件）时
    总势能的一阶变分为零（取极值），且该极值是最小值。

这个原理的优美之处在于：它把结构力学问题——求解偏微分方程（平衡方程）——转化为一个标量函数的优化问题。你不再需要从微元体出发列平衡方程，而只需要构造总势能函数，然后令它取最小值。

物理直觉：大自然倾向于让系统处于"最低能量"状态——水往低处流是重力势能最小，弹簧停在平衡位置是弹性势能最小。弹性结构也是一样：在外荷载的作用下，结构变形到某一形态，这个形态一定是使总势能最小的那个形态。如果你猜了一个位移场，计算出的总势能比真实解的总势能高——那你的猜测一定偏离了真实解。这就是能量法能给出"上限"的物理原因。

与虚功原理的关系

你可能会问：最小势能原理和第7章学的虚功原理是什么关系？

答案是：它们是等价的。 对于线弹性结构，虚功原理（Principle of Virtual Work）可以推导出最小势能原理，反之亦然。虚功原理说"虚功之和为零"，最小势能原理说"总势能变分为零"——数学上它们是可以互推的。但从概念上讲：

  • 虚功原理是"力"的视角——它关心的是虚位移上各力做的功；
  • 最小势能原理是"能量"的视角——它关心的是结构整体的势能状态。

两个视角各有优势。虚功原理在推导具体公式（如图乘法）时更直接，最小势能原理在建立近似解法（如里兹法）时更方便。

一个记忆技巧：虚功原理可以看作"微分的"能量原理——它描述的是任意一小步虚位移上的功的平衡。最小势能原理是"积分的"能量原理——它描述的是整个变形过程中能量的全局最优。所以最小势能原理是一个全局性原理，而虚功原理是一个局部性原理。

里兹法的基本原理与应用

里兹法（Ritz Method） 是最小势能原理的直接应用，也是最古老的近似解法之一。它的思想简单而巧妙：

我们不直接去寻找使势能最小的精确位移函数u(x)——那可能需要求解复杂的微分方程——而是"猜"一个带有待定参数的近似位移函数，然后调整这些参数使总势能最小。

实施步骤

    步骤1：选择试探函数（Trial Function）
            ũ(x) = a₁·φ₁(x) + a₂·φ₂(x) + ... + a_N·φ_N(x)

            其中 φ₁, φ₂, ..., φ_N 是满足位移边界条件的基函数
            a₁, a₂, ..., a_N 是待定参数（广义坐标）

    步骤2：将试探函数代入总势能表达式
            Π(a₁, a₂, ..., a_N) = U(aᵢ) + V(aᵢ)

    步骤3：对每个参数求偏导，令其为零（最小化条件）
            ∂Π/∂a₁ = 0, ∂Π/∂a₂ = 0, ..., ∂Π/∂a_N = 0

    步骤4：求解线性方程组，得到参数 a₁, a₂, ..., a_N

    步骤5：代回试探函数，得到近似位移场 ũ(x)

简单来说：里兹法把"寻找函数"的问题转化成了"寻找一组数"的问题——这是一个从无限维到有限维的"降维打击"。

简支梁受均布荷载的里兹法示例

考虑一个简支梁，跨度L，抗弯刚度EI，受均布荷载q₀作用。用里兹法求跨中挠度。

    选择试探函数（满足简支梁边界条件：u(0)=u(L)=0, u''(0)=u''(L)=0）：
    ũ(x) = a₁·sin(πx/L) + a₃·sin(3πx/L)

    注：这里只取对称项（因为荷载对称，反对称项不需要）

    应变能：
    U = (1/2)∫₀ᴸ EI·(ũ'')² dx
      = (EI/2)[a₁²∫(π²/L²·sin(πx/L))² dx + a₃²∫(9π²/L²·sin(3πx/L))² dx]
      = (EIπ⁴/(4L³))·(a₁² + 81·a₃²)

    外力势能：
    V = -∫₀ᴸ q₀·ũ(x) dx
      = -q₀·[a₁∫sin(πx/L)dx + a₃∫sin(3πx/L)dx]
      = -q₀·(2L/π)·(a₁ + a₃/3)

    总势能：
    Π(a₁, a₃) = (EIπ⁴/(4L³))(a₁² + 81a₃²) - (2q₀L/π)(a₁ + a₃/3)

    令 ∂Π/∂a₁ = 0, ∂Π/∂a₃ = 0：
    (EIπ⁴/(2L³))·a₁ - (2q₀L/π) = 0  →  a₁ = 4q₀L⁴/(EIπ⁵)
    (EIπ⁴·81/(2L³))·a₃ - (2q₀L/(3π)) = 0  →  a₃ = 4q₀L⁴/(243·EIπ⁵)

    跨中挠度（x=L/2）：
    ũ(L/2) = a₁·sin(π/2) + a₃·sin(3π/2) = a₁ - a₃
            = 4q₀L⁴/(EIπ⁵)·(1 - 1/243)
            ≈ 0.01307·q₀L⁴/(EI)

    精确解（材料力学）：
    u_exact(L/2) = 5q₀L⁴/(384EI) ≈ 0.01302·q₀L⁴/(EI)

    误差 < 0.4%！只用两个参数就达到了极高的精度。
试探函数项数 跨中挠度系数（×q₀L⁴/EI） 与精确解的误差 说明
1项（a₁·sin(πx/L)） 0.01307 +0.4% 基本够用，偏大（上限）
2项（+a₃·sin(3πx/L)） 0.01302 <0.01% 极高精度
精确解 0.01302（5/384） — 精确的理论解
踩坑故事：多年前一位研究生用里兹法分析一个悬臂板的振动问题。他选择了一组多项式作为试探函数，这些多项式单独看都满足几何边界条件（悬臂端位移为零）。但他的前两项函数φ₁(x)和φ₂(x)之间几乎是线性相关的（一个是他写的(x/L)²，另一个是(x/L)²+0.001(x/L)³）。结果他的刚度矩阵几乎是奇异的——方程组解出来的系数a₁和a₂巨大且正负抵消，表面上挠度曲线看起来合理，但中间参数已经"爆炸"了。这就是著名的"数值病态"问题。教训是：选择试探函数时，基函数之间要尽量"正交"（线性无关），否则计算结果对数值误差极其敏感。这也解释了为什么有限元法中统一使用多项式插值函数（形函数），而不是随意挑选的函数——形函数在单元层面保证了良好的数值性质。

从能量原理到有限元法

现在，让我们做一个重要的思想跨越——从里兹法到有限元法。

里兹法有一个关键限制：试探函数必须是全局定义的，即一个函数要在整个结构域上都有定义，并且满足所有的位移边界条件。对于形状复杂的结构（如带孔洞的板、不规则形状的拱坝），构造一个全局试探函数极其困难，甚至不可能。

有限元法的天才之处在于：它把整个结构域离散（Discretize）成若干个小的、形状规则的单元（Element），在每个单元内用简单的插值函数（通常是低阶多项式）作为试探函数，然后通过"组装"过程把这些局部试探函数连接起来。

回顾我们第13~14章学的矩阵位移法：

    从能量角度重新理解矩阵位移法：

    1. 每个单元的应变能：U_e = (1/2)·u_eᵀ·k_e·u_e
       其中 k_e = ∫ Bᵀ·D·B dV 是单元刚度矩阵
       B是应变-位移矩阵，D是材料本构矩阵

    2. 总应变能：U = Σ U_e = (1/2)·Uᵀ·K·U
       其中K是总体刚度矩阵（由各单元刚度矩阵组装而成）

    3. 总势能：Π = (1/2)·Uᵀ·K·U - Pᵀ·U

    4. 最小势能原理：∂Π/∂U = 0  →  K·U = P

    这正是矩阵位移法的基本方程！

这个推导过程清楚表明：矩阵位移法就是"离散化的最小势能原理"。你第13~14章学的"直接刚度法"（单元分析→坐标变换→组装→求解），本质上就是最小势能原理在计算机上的系统化实现。

思想演进路线： 虚功原理 → 最小势能原理 → 里兹法（全局试探函数）→ 有限元法（单元试探函数+组装） ↑ ↑ ↑ 19世纪 20世纪初 1950s-1960s 理论奠基 近似计算方法 计算机时代的革命 这条路线展示了结构力学从"解析"到"数值"的演进轨迹。能量原理在其中扮演了"桥梁"的角色——它让结构力学从手工求解微分方程，走向了计算机上的矩阵运算。

我们用一个对比表来总结从里兹法到有限元法的演化：

对比维度 里兹法 有限元法
试探函数定义域 整个结构域（全局） 每个单元内部（局部）
基函数性质 需满足所有位移边界条件 仅需简单多项式（满足单元间连续性）
复杂形状适应性 差（不规则域几乎不可行） 好（用大量小单元逼近复杂形状）
未知量 广义坐标（a₁, a₂, ...） 结点位移（u₁, u₂, ...）
方程装配 无（一次形成全局方程） 单元刚度矩阵→组装→总体方程
计算机实现 困难（基函数依赖具体问题） 容易（标准化、模块化）
一个形象的比喻：如果把求解结构问题比作做一床被子——里兹法像是用一整块巨大的布料，你要根据被子的形状精确裁剪（全局试探函数），非常考验手艺；有限元法像是用小碎布（单元）拼接，虽然每块布很小，但只要拼接得足够多、足够密，同样能做出完美的被子，而且不需要高超的裁剪技术。计算机就是那个"拼布工人"——它不怕重复劳动，只怕没有标准化的工序。

能量法的更多应用

能量原理的应用远不止于此。它在以下几个领域也有突出的贡献：

结构稳定性分析

我们在第16章学过压杆稳定。其实欧拉临界荷载也可以通过能量法推导——当压杆屈曲时，弯曲应变能的增量刚好等于轴力势能的减少量（即外力做功）。通过令总势能的二阶变分为零，可以确定临界荷载。

固有频率估计

我们在第21章已经见过瑞利法——它就是基于能量守恒来估计自振频率的。瑞利商公式本质上就是能量法的直接应用。

弹塑性分析

对于弹塑性结构（第17章），能量原理也是重要的分析工具。极限分析中的"上限定理"和"下限定理"本质上都是基于能量考虑。

本章小结与下章预告

在这一章中，我们学习了能量原理与近似方法的系统内容：

  • 应变能与外力势能——两种基本的能量形式及其计算；
  • 最小势能原理——真实位移场使总势能取最小值，将结构问题转化为标量优化问题；
  • 里兹法——选择试探函数、代入势能、求偏导为零，三步得到近似解；
  • 从能量原理到有限元法——里兹法+离散化=有限元法，矩阵位移法是最小势能原理的系统化实现。
