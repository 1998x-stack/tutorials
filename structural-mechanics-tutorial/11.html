引言：当刚架"侧移"的时候

同学们好，上一章我们学习了位移法的基本框架——转角位移方程、基本体系、无侧移刚架的计算。但你们可能已经意识到了：现实中的框架结构几乎没有"无侧移"的。一栋高层建筑在风荷载下，整个楼面会产生水平位移——这就是侧移（Sway）。

1985年，墨西哥城遭遇了一场8.0级大地震。在震后的调查中，工程师们发现了一个令人震惊的现象：大量建筑的倒塌不是因为强度不够，而是因为过大的侧移导致了P-Δ效应（即竖向荷载在侧移后的结构中产生的附加倾覆力矩）——柱子先是弯曲、然后失稳、最后整栋楼像叠罗汉一样一层层塌下去。墨西哥城地震中倒塌的许多建筑，如果按照弹性范围内的强度校核都是"够的"——问题出在侧移导致的几何非线性效应被忽略了。

这个故事告诉我们：侧移不是一个小问题——它是结构在水平荷载下的第一响应，也是决定结构安全的关键指标。在位移法中，处理有侧移的刚架（不论是水平荷载还是不对称竖向荷载引起的侧移）是必须掌握的核心技能。

本章我们将深入位移法的三个进阶话题：对称性的利用、直接平衡法、以及含斜杆刚架的处理。最后，我们会对力法和位移法做一次全面的对比分析。

11.1 位移法中的对称性利用

与力法一样，位移法同样可以充分利用对称性来减少未知量。而且——位移法中的对称性利用往往更直接、更强大。

对称与反对称的"取半"规则

在位移法中，取半结构的规则与力法相同：

  • 对称荷载：对称轴截面无转角（\(\theta=0\)），但可以有垂直于对称轴的位移。取半结构时用定向支座模拟。
  • 反对称荷载：对称轴截面无垂直于对称轴的位移（\(u=0\)），但可以有转角。取半结构时用链杆支座模拟。

用位移法分析对称结构的关键好处是：取半结构后，基本未知量的数量大幅减少——原来两个结点的转角变成一个，原来两个结点的水平位移变成一个或消失。

例：对称刚架在对称水平荷载下

      P ←──┬────┬──→ P
            │    │
          ══╪════╪══
            │    │
            │    │
          ══╪════╪══
            │    │
          ──┴────┴──

  结构和荷载均对称 → 对称轴截面上：
    无转角（θ=0）
    无水平位移（被对称约束住）
    可以有竖向位移

  取半结构后：
      P ←──┬──║
            │
          ══╪══║
            │
            │
          ══╪══║
            │
          ──┴──
  基本未知量从6个降为3个！
对称性利用的顺序：在位移法中，我的建议是"先对称、后计算"——拿到一个结构，第一件事就是看对称性。如果是对称的，立刻取半结构。这和你是否用位移法无关——无论用什么方法，对称性都能让工作量减半。

11.2 直接平衡法

上一章我们介绍了用"基本体系+典型方程"的位移法求解流程。但实际上，位移法还有一种更直接的方式——直接平衡法（Direct Equilibrium Method）。这种方法不需要引入附加约束和典型方程，而是直接对结点和杆件列平衡方程。

直接平衡法的步骤

  1. 确定基本未知量（结点转角和独立线位移）；
  2. 对每个有转角自由度的刚结点，用力矩平衡方程 \(\sum M = 0\)；
  3. 对每个有线位移自由度的方向，用截面平衡方程（一般为\(\sum F_x = 0\)或\(\sum F_y = 0\)，视位移方向而定）.
  4. 联立求解位移；
  5. 回代求杆端力。

与"基本体系法"相比，直接平衡法省去了画单位变形图和求刚度系数\(r_{ij}\)的步骤——你直接列方程，一步到位。对于有侧移刚架，直接平衡法特别方便，因为侧移方向的平衡方程本质上是"楼层剪力平衡"。

直接平衡法示例：单层单跨刚架，水平力 P

  P →     ────┬────
               │
               │
               │
             ──┴──

  基本未知量：柱顶转角 θ（1个） + 柱顶水平位移 Δ（1个）= 2个

  转角平衡方程（柱顶结点）：
    M_{上柱左} + M_{上柱右} + M_{横梁} = 0

  水平力平衡（取柱顶以上部分为隔离体）：
    柱底剪力之和 = P
    F_{Q1} + F_{Q2} = P

  写出每根杆件的杆端力表达式（含θ和Δ），
  代入上述2个方程可得解。
为什么直接平衡法重要？因为它更接近"直觉"——你对结点的力矩平衡有直观感受，对楼层的剪力平衡也有直观感受。而基本体系法虽然系统性强，但有时会让人迷失在"刚度系数r_{ij}"的技术细节中。在工程实践中，很多有经验的工程师在做初步手算时都是用直接平衡法，因为它快、直观、不容易出错。

有侧移刚架的剪力平衡方程

对于有侧移刚架，除了结点的力矩平衡，还需要建立楼层剪力平衡方程。取某一层以上的所有结构为隔离体，水平方向的剪力平衡方程为：

\(\sum F_{Q\text{柱}} = \sum F_{H\text{水平外力}}\)

每一根柱子对楼层的剪力贡献，可以通过该柱的杆端弯矩来表达：

\(F_{Q\text{柱}} = -\frac{M_{上} + M_{下}}{h}\)

其中 \(M_{上}\) 和 \(M_{下}\) 分别是柱上端和下端的弯矩（顺时针为正），\(h\) 为柱高。这个关系来自柱隔离体的力矩平衡。

11.3 含斜杆刚架的处理

含斜杆的刚架在位移法中需要特别处理——因为斜杆的变形与水平位移、竖向位移都有关系，转角位移方程需要针对斜杆进行坐标变换。

基本思路

斜杆的处理分两步：

  1. 将斜杆两个端点的线位移 \(\Delta_x\)、\(\Delta_y\) 分解为沿杆轴方向（轴向变形）和垂直于杆轴方向（弯曲变形）的两个分量；
  2. 将垂直于杆轴方向的分量代入转角位移方程计算杆端弯矩。

也就是说，你需要先建立结点的整体位移和斜杆局部位移之间的几何关系——这实际上就是一个坐标变换（Coordinate Transformation）的过程。这是后面矩阵位移法的核心思想

例：含斜杆的刚架

          ────┬────
         /    │
        /     │
       /      │
      /       │
     /        │
    /─────────┘
   △         △

  柱是垂直的，但屋顶有斜杆。

  分析过程：
  1. 柱顶的水平位移 Δ 引起斜杆两端的
     垂直线位移分量 = Δ · sinα
     （α是斜杆与水平面的夹角）

  2. 斜杆的转角位移方程中，Δ_{AB} 项要用
     这个垂直分量代入

  3. 斜杆的轴力也会对结点的线位移平衡
     有贡献——需要额外考虑
踩坑故事：我曾经在分析一个坡屋面厂房时，在斜杆的处理上犯了一个低级错误。我把斜杆当作水平杆来处理，直接用水平位移代入转角位移方程——完全忘了斜杆的垂直线位移分量需要乘以 \(\cos\alpha\)。结果算出来的斜杆弯矩比正确值大了40%，导致我把斜梁的截面选得过厚。后来被审图公司打回来重算，不但浪费了时间，还让甲方对设计团队的能力产生了质疑。这个教训让我记住了：含斜杆的结构，每一步都要问自己——这个位移方向是和杆件垂直还是平行？

含斜杆结构的简化策略

如果斜杆的轴向刚度很大（EA很大），可以考虑忽略轴向变形——假设斜杆长度不变。这样，结点之间的线位移就建立了固定的几何关系，独立线位移的数量会减少。

另外，如果斜杆与水平方向的夹角非常小（如坡度 < 1:10），可以近似按水平杆处理——但这只适用于初步估算，正式计算中还是应该按照实际角度处理。

11.4 力法与位移法对比

到现在为止，我们学完了两种分析超静定结构的基本方法：力法和位移法。我们来做一个全面的对比。

对比项 力法（Force Method） 位移法（Displacement Method）
基本未知量 多余约束力（力） 结点位移（位移）
基本体系 去掉多余约束的静定结构 施加附加约束使结点位移为零
建立方程的依据 变形协调条件 静力平衡条件
方程物理意义 位移为零（或已知值） 附加约束反力为零
核心系数 柔度系数 δ（位移/力） 刚度系数 r（力/位移）
系数间关系 [r] = [δ]⁻¹ [δ] = [r]⁻¹
超静定次数敏感度 高度敏感——次数越高越复杂 不敏感——只与结点位移数量有关
温度/支座移动 直接处理，自然 需要额外处理固端力
程序化难度 高（基本体系选择不唯一） 低（步骤统一，适合编程）
手算适用场景 低次超静定（1-3次） 中低自由度结构（2-6自由度）
如何选择？一个实用的经验法则：
• 超静定次数 ≤ 3 且杆件不多 → 用力法
• 结点位移数 ≤ 6 且侧移不明显 → 用位移法
• 超静定次数高但结点少 → 用位移法
• 需要计算温度/支座移动 → 力法更自然
• 度数高、结点多 → 用矩阵位移法（第13-14章）

一句话总结：力法适合"少约束"的结构，位移法适合"少结点"的结构，两者都难以处理时——上矩阵位移法。

11.5 位移法的适用性与局限

位移法相比力法有一个巨大的优势：它的自由度数量只取决于结点位移，与超静定次数无关。这意味着——一个100次超静定的结构，如果它只有5个结点，位移法就只需要解5个方程！这是位移法在计算机时代占据主导地位的根本原因。

但位移法也有它的局限：

  • 温度变化的处理不够直观——需要用固端弯矩来等效温度效应，不如力法直接；
  • 支座移动的处理——与温度变化类似，也需要转化为等效荷载；
  • 手算工作量大——对于有侧移刚架，画变形图和计算刚度系数比较繁琐；
  • 正负号容易出错——位移法的正负号规则与日常习惯不完全一致。

尽管如此，位移法仍然是结构力学中承上启下的关键方法——上承力法（同一问题的对偶视角），下启矩阵位移法和有限元法（计算机化的结构分析）。可以说，掌握了位移法，你就掌握了现代结构分析的"基因"。

本章我们完成了位移法的进阶学习：

  • 对称性在位移法中的利用——取半结构，减少未知量；
  • 直接平衡法——不依赖基本体系，直接列位移法方程；
  • 含斜杆刚架的处理——坐标变换与分量分解；
  • 力法与位移法的全面对比——各自的特点、优缺点、适用场景。

学到这里，我们已经掌握了两种"精确法"——力法和位移法。但手算的局限性摆在那里：自由度一多，计算量就让人望而却步。下一章我们将学习一种更"巧妙"的方法——力矩分配法（Moment Distribution Method）。这是1932年由Hardy Cross提出的一种渐进方法——它不需要解联立方程，而是通过反复"分配-传递-再分配"的迭代过程逼近精确解。对于连续梁和无侧移刚架，力矩分配法在手算时代是真正的"王牌"方法。我们下一章见。
