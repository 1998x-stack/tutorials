第十九章 结构动力学概论

不知道你有没有在电影里见过这样的场景——一座大桥在风中微微颤动，行人走在上面感觉脚下像踩着一层薄冰。你可能觉得这只是导演制造紧张气氛的手法，但事实上，这种"抖动"是真实存在的，而且它是结构工程师必须严肃对待的核心问题。

在前18章中，我们一直默认一个假设：荷载是"安静地"作用在结构上的——它们的大小不随时间变化，或者变化得足够缓慢，以至于结构内部的加速度可以忽略不计。这就好比你慢慢地把一摞书放在书架上，书架几乎感觉不到冲击。但现实世界中的荷载往往是"冲动"的：地震来时地面剧烈摇晃，狂风吹过时建筑左右摇摆，火车驶过桥梁时桥面上下跳动。这些情况下，结构的惯性力（inertia force）不再是配角，而是主角。这就是结构动力学要解决的问题。

本章是结构动力学部分的开篇。我们从最基础的概念入手：什么是动力荷载？什么是动力自由度？然后建立单自由度体系（SDOF，Single Degree of Freedom）的运动方程，并深入讨论它的自由振动行为——包括无阻尼和有阻尼两种情况。请记住，单自由度体系虽然简单，但它揭示了振动现象的几乎所有本质规律。搞懂了单自由度，就拥有了理解复杂振动问题的"密钥"。

动力荷载与静力荷载的本质区别

要理解结构动力学，我们首先要回答一个看似简单的问题：什么样的荷载算"动荷载"？

你可能会说：随时间变化的荷载就是动荷载。这个答案只对了一半。准确地说，动荷载（Dynamic Load） 是那些变化速度快到足以引起结构显著加速度的荷载。关键在于"显著"二字——一个很慢的变化，即使荷载值在变化，也不会产生明显的动力效应。

举个直观的例子：你站在体重秤上，如果你慢慢地蹲下再站起，秤上的读数会有变化，但变化很平缓。但如果你在秤上猛地蹦一下，读数会瞬间冲到远高于你体重的数值——这就是动力效应。那个瞬间的超额数值，就是惯性力的体现。

因此，判断一个荷载是否需要按动荷载处理，不是看它"变没变"，而是看它的变化速度与结构自振周期的关系。如果荷载的变化周期远大于结构的自振周期（一般大于3~5倍），那么即使荷载在随时间变化，也可以近似按静荷载处理。反之，如果荷载变化周期接近或小于结构自振周期，就一定要按动荷载分析。

荷载类型 典型持续时间/频率 是否按动荷载处理 说明
结构自重 永久作用 否 变化极慢，零动力效应
楼面活荷载 小时~天级 通常否 变化慢，可视为静载
风荷载 1~60秒（脉动分量） 是 脉动分量频率接近结构自振频率
地震作用 0.1~2秒（主要能量频段） 是 典型动力荷载，必须动力分析
爆炸冲击 毫秒级 是 超短时间，巨大冲击力
车辆过桥 0.5~5秒 视情况 需考虑车速与桥跨关系
重要结论：结构动力学关心的是"动"之本身——即结构的振动响应，而不是荷载的绝对值大小。一个很小的动荷载，如果它的频率恰好接近结构的自振频率，也可能引发巨大的响应（这就是我们下一章要讲的共振）。反之，一个很大的荷载如果施加得非常缓慢，结构可能平平安安、毫无波澜。

动力自由度

我们之前在第2章学过"自由度"的概念——那是几何组成分析中的自由度，指的是刚体运动自由度。结构动力学中也有一个"自由度"的概念，但含义不同。

动力自由度（Dynamic Degree of Freedom） 是指：为了完整地描述结构体系中所有质量的运动位置，所需要的最少独立坐标（位移分量）的数量。

两者的关键区别在于：几何组成分析中的自由度关注的是刚体运动（不考虑变形），而动力自由度关注的是质量在变形状态下的位移。换句话说，动力自由度是"质量可以怎么动"的问题。

几个典型例子

  • 单层框架——假设楼盖是刚性的（平面内无限刚性），质量集中在楼盖处，那么楼盖只能水平平动，动力自由度为1（单自由度体系）。
  • 三层框架——每层楼盖都能水平平动，三个质量块各有一个水平位移，动力自由度为3。
  • 简支梁——质量均匀分布在梁上。理论上梁上有无穷多个点，每个点都可以竖向位移，所以它是无限自由度体系。但在实际分析中，我们通常把质量集中到有限个点上（集中质量法），简化为有限自由度。
动力自由度示例：

        单层框架（SDOF）          三层框架（3-DOF）
            ┌───┐                    ┌───┐
            │ m │→ u(t)              │ m3│→ u3(t)
            └───┘                    └───┘
              │                        │
            ┌───┐                    ┌───┐

集中质量法与一致质量法

在结构动力学中，对于连续体（如梁、板），我们有两种常用的质量建模方法：

  • 集中质量法（Lumped Mass Model）——把分布质量集中到有限个结点上。简单直观，但会损失一些频率信息（高阶频率偏差较大）。
  • 一致质量法（Consistent Mass Model）——通过形函数对质量进行一致分布（和刚度矩阵的形函数相同）。精度更高，但计算更复杂。

在本书的框架内，我们主要使用集中质量法，因为它概念清晰、手算方便。

单自由度体系的运动方程

现在，让我们进入结构动力学最核心的部分——建立运动方程（Equation of Motion）。

单自由度体系（SDOF System）是最简单的振动系统，但它的运动方程揭示了所有振动问题的本质。一个典型的SDOF体系由三个要素构成：质量（Mass, m）、刚度（Stiffness, k） 和 阻尼（Damping, c）。

你可以把它想象成一个弹簧-质量-阻尼系统：一个质量为m的物块，通过一根刚度为k的弹簧和一个阻尼系数为c的阻尼器连接在基础上。当外力p(t)作用在物块上时，物块会产生位移u(t)。这个系统的运动方程就是描述u(t)如何随时间演化的数学表达式。

建立运动方程的两种方法

在结构力学中，建立运动方程主要有两种方法：刚度法和柔度法。

1. 刚度法（Stiffness Method）

刚度法基于达朗贝尔原理（d'Alembert's Principle）：在质量上施加惯性力，将动力问题转化为"静力平衡"问题。惯性力 \( f_I = -m\ddot{u}(t) \)，方向与加速度相反。于是，根据力的平衡：

    f_I + f_D + f_S = p(t)

    其中：
    f_I = m·ü(t)    —— 惯性力（Inertia Force）
    f_D = c·u̇(t)    —— 阻尼力（Damping Force）
    f_S = k·u(t)    —— 弹性恢复力（Elastic Restoring Force）
    p(t)            —— 外荷载（External Load）

    代入得：
    m·ü(t) + c·u̇(t) + k·u(t) = p(t)

这就是单自由度体系运动方程的标准形式。它是一个二阶常系数线性微分方程（ODE）。

2. 柔度法（Flexibility Method）

柔度法从位移的角度出发：质量在惯性力、阻尼力、弹性恢复力和外荷载的共同作用下产生的位移，等于各力单独作用下产生的位移之和。用柔度系数δ（刚度k的倒数：δ = 1/k）表示：

    u(t) = δ·[p(t) - m·ü(t) - c·u̇(t)]

    两边乘以k得：
    m·ü(t) + c·u̇(t) + k·u(t) = p(t)

    可见刚度法和柔度法得到的是同一个方程。

刚度法更适合框架结构（因为层间刚度容易计算），而柔度法在一些特殊情况下（如悬臂梁端部质量）更直观。两种方法最终导出相同的运动方程。

无阻尼自由振动

如果你把一个单摆拉开然后松手，它会来回摆动——这就是自由振动。在结构动力学中，自由振动（Free Vibration） 是指结构在没有外荷载作用（p(t)=0）且不考虑阻尼（c=0）的情况下，由初始位移或初始速度引发的振动。

运动方程简化为：

    m·ü(t) + k·u(t) = 0

    或者写成：
    ü(t) + ω²·u(t) = 0

    其中 ω² = k/m，ω 称为自振圆频率（单位：rad/s）。

这个方程的解是简谐函数：

    u(t) = A·cos(ωt) + B·sin(ωt)

    或者写成：
    u(t) = C·cos(ωt - φ)

    其中，常数A和B由初始条件（初始位移u₀和初始速度v₀）确定：
    A = u₀
    B = v₀/ω

    振幅 C = √(A² + B²)
    初相位 φ = arctan(B/A)

自振频率与周期

无阻尼自由振动的核心参数是自振频率（Natural Frequency） 和 自振周期（Natural Period）：

参数名称 符号 公式 单位 物理意义
自振圆频率 ω ω = √(k/m) rad/s 每秒振动的弧度数
自振频率（工程频率） f f = ω / (2π) Hz（赫兹） 每秒振动的次数
自振周期 T T = 2π/ω = 1/f s（秒） 完成一次完整振动所需时间

这里有一个极其重要的结论：自振频率只取决于结构自身的质量m和刚度k，与外荷载无关！ 你不可能通过改变荷载来改变结构的自振频率。这就像秋千的摆动周期只取决于绳长（和重力加速度），而不管推秋千的人用了多大力。

直观理解：刚度越大，频率越高（"硬"的结构振得快）；质量越大，频率越低（"重"的结构振得慢）。一个轻巧的钢制天线可能自振频率高达10Hz以上，而一座重型混凝土大坝的自振频率可能只有0.5Hz左右。这个直觉在你后续做结构设计时非常重要——通过调整刚度与质量的比值，你可以"调"出想要的频率，避开动荷载的主要能量频段。

有阻尼自由振动

实际结构中的振动不会永远持续下去——如果你敲击一个音叉，声音会逐渐减弱直到消失。这个"衰减"的过程，就是阻尼的作用。

阻尼（Damping） 是结构在振动过程中耗散能量的机制。阻尼的来源多种多样：材料内部的摩擦（内耗）、构件连接处的摩擦、空气阻力、地基土的能量辐射等等。在结构动力学中，我们通常使用黏滞阻尼模型（Viscous Damping Model），即阻尼力与速度成正比。

运动方程与求解

考虑阻尼的自由振动方程：

    m·ü(t) + c·u̇(t) + k·u(t) = 0

    假设解的形式 u(t) = e^(st)，代入得特征方程：
    m·s² + c·s + k = 0

    解得特征根：
    s₁,₂ = -ζω ± ω√(ζ² - 1)

    其中：
    ζ = c / (2√(mk)) = c / c_cr   —— 阻尼比（Damping Ratio）
    c_cr = 2√(mk) = 2mω           —— 临界阻尼系数（Critical Damping Coefficient）

根据阻尼比ζ的取值，振动呈现三种截然不同的状态：

阻尼状态 条件 振动行为 工程实例
无阻尼 ζ = 0 等幅简谐振动（永不衰减） 理想情况，实际不存在
欠阻尼 0 < ζ < 1 振幅逐渐衰减的振动 绝大多数工程结构（ζ≈0.02~0.10）
临界阻尼 ζ = 1 最快回到平衡位置，不振动 精密仪器、某些阻尼器设计
过阻尼 ζ > 1 缓慢回到平衡位置，不振动 某些特种减振装置

对于工程结构，我们最关心的是欠阻尼（Underdamped） 情况。它的解为：

    u(t) = e^(-ζωt) · [u₀·cos(ω_D·t) + (v₀+ζωu₀)/ω_D · sin(ω_D·t)]

    其中 ω_D = ω·√(1-ζ²)  —— 阻尼自振频率（Damped Natural Frequency）

    当ζ较小时（如0.05），ω_D ≈ ω，即阻尼对频率的修正很小。

对数衰减率

在工程中，我们常常需要通过实测的振动衰减曲线来反算结构的阻尼比。这就用到了对数衰减率（Logarithmic Decrement） 的概念。

假设我们从振动曲线上量取了相邻两周波的振幅u_n和u_{n+1}：

    对数衰减率 δ = ln(u_n / u_{n+1}) = 2πζ / √(1-ζ²)

    对于小阻尼（ζ < 0.2），可近似为：
    δ ≈ 2πζ

    因此：
    ζ ≈ δ / (2π)

也就是说，你只需要在实测的振动时程曲线上量出相邻两个波峰的幅值，取自然对数后除以2π，就得到了阻尼比。这是现场测试中估算阻尼的最常用方法。

踩坑故事：我读研究生时参与过一栋20层钢结构的现场动力测试。我们在楼顶用激振器施加激励，然后记录结构的自由振动衰减曲线。第一次测试结束后，我兴冲冲地用对数衰减率算出了阻尼比——ζ≈0.005。导师看了一眼说："这个数偏小了。"我不服气，又算了一遍，还是0.005。导师带我去看了现场：原来我选取的衰减段正好是激振器刚停止的瞬间，那时电梯还在运行、水管里的水还在流动——这些"非结构阻尼"让衰减曲线出现了不规则的波动。后来我们在凌晨2点重新做了一次测试，关闭了所有非必要设备，算出的阻尼比是ζ≈0.02，这才是钢结构的真实阻尼比。这个教训是：实测数据中的"噪声"不是错误信号，它可能携带着你不知道的物理信息。识别和排除这些干扰，才是真正的工程能力。

工程结构的典型阻尼比

在结构设计中，阻尼比是一个重要的参数——它直接影响地震作用的大小（阻尼越大，地震响应越小）。以下是一些典型结构的阻尼比范围：

结构类型 阻尼比ζ（小变形） 阻尼比ζ（大变形/弹塑性） 说明
焊接钢结构 0.5%~1% 2%~5% 接头摩擦小，阻尼低
螺栓连接钢结构 1%~2% 3%~7% 连接处摩擦耗能
钢筋混凝土结构 2%~5% 5%~10% 混凝土微裂缝耗能
预应力混凝土结构 1%~2% 3%~6% 裂缝少，阻尼较低
砌体结构 3%~8% 8%~15% 砂浆接缝摩擦耗能大
木结构 4%~10% 10%~20% 节点连接和木材内摩擦
温馨提示：从上面的表格可以看出，阻尼比并不是一个固定的材料属性——它随变形幅值而变化。在小变形（风振、微震）下阻尼比较小，在大变形（强震、弹塑性阶段）下阻尼比显著增大。这就是为什么抗震规范中，弹性分析时取较小阻尼比，而弹塑性分析时取较大阻尼比。

本章小结与下章预告

在这一章中，我们学习了结构动力学最基础的内容：

  • 动力荷载与静力荷载的本质区别——关键在于荷载变化速度与结构自振周期的关系；
  • 动力自由度的概念——描述质量运动所需的最少独立坐标数；
  • 单自由度体系的运动方程——刚度法与柔度法两种推导思路，殊途同归；
  • 无阻尼自由振动——自振频率ω=√(k/m)是结构最重要的动力特性之一；
  • 有阻尼自由振动——阻尼比ζ决定了振动衰减的快慢，对数衰减率是实测阻尼的利器。
