从平面到空间的跳跃

大家好，我是你们的讲师。欢迎回到微积分课程。

回忆一下，我们之前十几章的所有工作都发生在一个平面上——二维的 xy 坐标系里。函数 y = f(x) 是一条曲线，定积分 ∫ab f(x) dx 是曲线下的面积。二维世界很美好，但现实世界是三维的。飞机不是在纸面上飞行的，导弹不是在二维平面里追踪目标的，MRI 扫描重建出来的不是一张切片而是一个立体结构。

从本章开始，我们将正式踏入三维空间。你会发现，很多在二维中熟悉的概念都会有对应的"三维版本"：直线变成平面，斜率变成梯度，面积变成体积。要理解这些概念，我们首先需要一套描述空间的语言——这就是向量（vector）和空间解析几何（analytic geometry in space）。

我有个朋友在本科做机器人路径规划的项目，一开始他只用了二维坐标描述机器人的位置。等到项目需要处理机械臂的关节角度和末端执行器的姿态时，他发现自己完全不会描述"朝向"——二维里一个角度 θ 就够用了，三维里你还需要俯仰角、偏航角、翻滚角，一套完整的 Euler angles。他终于意识到：从二维到三维，不是简单地加一个坐标轴，而是整个几何思维方式都需要升级。

这一章我们将会：先建立向量的运算体系（这是"语言"本身），然后用向量来写出空间中的平面和直线方程，最后认识常见的二次曲面。这套工具将为你后续的多元函数微积分打下坚实的几何基础。

向量的点乘与叉乘

向量（vector）是有大小和方向的量，记作 a = (a₁, a₂, a₃) 或 a = a₁i + a₂j + a₃k，其中 i、j、k 分别是 x、y、z 轴正方向的单位向量（unit vector）。

向量的加减法和数乘很简单——对应分量加减或缩放即可。真正有趣的是两种乘法：点乘（dot product）和叉乘（cross product）。它们名字相似，但几何意义截然不同。

点乘：衡量两个向量的"对齐程度"

设 a = (a₁, a₂, a₃)、b = (b₁, b₂, b₃)，它们的点积定义为：

a · b = a₁b₁ + a₂b₂ + a₃b₃ = |a| |b| cos θ

其中 θ 是两个向量之间的夹角。点乘的结果是一个标量（scalar），因此也被称为"标量积"。

点乘的几何意义非常直观：

  • 当 a · b > 0，夹角小于 90°，两向量"大致同向"
  • 当 a · b = 0，夹角等于 90°，两向量垂直（orthogonal）
  • 当 a · b < 0，夹角大于 90°，两向量"大致反向"

点乘最常用的场景之一：投影（projection）。向量 a 在 b 方向上的投影长度为 (a · b) / |b|。在物理中，力 F 沿位移 s 方向所做的功就是 W = F · s——你只关心力在位移方向上的分量，与位移垂直的那部分力不做功。

记忆技巧：点乘的坐标形式为什么是"对应分量相乘再相加"？
因为单位向量之间有正交关系：i·i=1，i·j=0。
所以 (a₁i+a₂j+a₃k)·(b₁i+b₂j+b₃k) 展开后，交叉项全部为零，只剩下 a₁b₁ + a₂b₂ + a₃b₃。你现在知道线性代数里"正交基"的重要性了吧？

叉乘：生成垂直于两个向量的新向量

叉乘是三维空间独有的运算。设 a = (a₁, a₂, a₃)、b = (b₁, b₂, b₃)，它们的叉积定义为：

a × b = (a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁)

叉乘的结果是一个向量（vector），因此也被称为"向量积"。这个结果向量具有两个关键性质：

  • 方向：垂直于 a 和 b 所在的平面，遵循右手定则（right-hand rule）——右手四指从 a 弯向 b，拇指指向就是 a × b 的方向
  • 大小：|a × b| = |a||b| sin θ，即 a 和 b 张成的平行四边形的面积

叉乘的几何应用极其广泛：

  • 法向量：给定平面上两个不共线的向量，叉乘得到该平面的法向量
  • 力矩：M = r × F，力臂与力的叉积
  • 面积：三角形的面积 = (1/2)|AB × AC|
常见错误：叉乘不满足交换律！
a × b = - (b × a)。方向上完全相反。
这意味着交换顺序会得到一个方向相反的向量。在计算力矩、法向量时，顺序搞错会导致符号错误——整个系统的受力分析就会出问题。我第一次做电磁学作业时就在这个上面吃了大亏，计算通电导线在磁场中的受力时把叉积顺序搞反了，结果所有力的方向都画反了。

平面方程的三种形式

在三维空间中，平面的数学描述有三种等价形式。掌握这三种形式之间的转换，是解决空间几何问题的基本功。

点法式

给定平面上一点 P₀(x₀, y₀, z₀) 和平面的法向量（normal vector）n = (A, B, C)，平面上任意一点 P(x, y, z) 满足向量 P₀P 与 n 垂直：

n · (P - P₀) = 0
        → A(x - x₀) + B(y - y₀) + C(z - z₀) = 0

这就是平面的点法式方程（point-normal form）。它的几何意义非常清晰：法向量 n 完全确定了平面的"朝向"，而点 P₀ 确定了平面在空间中的位置。

一般式

把点法式展开、合并常数项，就得到：

Ax + By + Cz + D = 0

这就是平面的一般式方程。（A, B, C）就是平面的法向量。判断两个平面是否平行或垂直，只需看它们的法向量是否平行或垂直即可。

截距式

如果平面与三个坐标轴分别相交于 (a, 0, 0)、(0, b, 0)、(0, 0, c)（其中 a、b、c 均不为零），可以写成：

x/a + y/b + z/c = 1

这就是截距式方程（intercept form）。它的好处是一目了然：眼神一扫就知道平面在三个轴上的截距。在快速画草图时，截距式非常方便。

形式 方程 已知条件 最适用场景
点法式 A(x - x₀) + B(y - y₀) + C(z - z₀) = 0 一点 + 法向量 已知法向量的情况
一般式 Ax + By + Cz + D = 0 三个不共线的点 计算机图形学中最常用
截距式 x/a + y/b + z/c = 1 三个坐标轴截距 快速画图、几何直观

空间直线方程

描述空间中的一条直线，最自然的方式是用一个点和一个方向向量。

参数式

给定直线上一点 P₀(x₀, y₀, z₀) 和方向向量 s = (m, n, p)，直线上任意一点可写为：

(x, y, z) = (x₀, y₀, z₀) + t · (m, n, p),  t ∈ ℝ

        即：
        { x = x₀ + mt
        { y = y₀ + nt
        { z = z₀ + pt

这就是直线的参数式方程（parametric form）。参数 t 从 -∞ 到 +∞ 跑一遍，整条直线就被扫出来了。在物理中，t 通常对应时间——你知道一个质点的初始位置和速度向量，就能写出它的轨迹。

对称式

从参数式中消去参数 t（假设 m、n、p 均不为零）：

(x - x₀)/m = (y - y₀)/n = (z - z₀)/p

这是直线的对称式方程（symmetric form），也叫"点向式"。它看起来更简洁，但使用上不如参数式灵活。如果某个方向分量为零，对称式中对应的分母就会消失（此时分子也必须为零）。

一个实用的几何事实
两条直线在空间中的关系有四种可能：平行、相交、异面（skew）、重合。异面直线是三维空间特有的——在二维平面上两条直线要么平行要么相交，但三维中它们可以既不平行也不相交，像两条在不同高度上交错的高架桥。判断异面直线之间的距离是一类经典的空间几何问题。

常见二次曲面识别

在二维平面中，二次曲线包括椭圆、双曲线、抛物线。在三维空间中，"二次曲线"升级为"二次曲面"（quadric surfaces）——方程 Ax² + By² + Cz² + Dxy + Exz + Fyz + ... = 0 所描述的曲面。

以下是微积分中最常见的几种二次曲面：

球面（Sphere）

(x - x₀)² + (y - y₀)² + (z - z₀)² = R²

球心在 (x₀, y₀, z₀)，半径为 R。这是最简单、最对称的二次曲面。任何方向看都一样。

椭球面（Ellipsoid）

x²/a² + y²/b² + z²/c² = 1

三个半轴长度分别为 a、b、c。当 a = b = c 时退化为球面。椭球面是"压扁的球"——沿不同方向半径不同。地球本身就是一个扁椭球体（赤道半径略大于极半径）。

抛物面（Paraboloid）

z = x²/a² + y²/b²      （椭圆抛物面，开口向上）
        z = - (x²/a² + y²/b²)   （椭圆抛物面，开口向下）

想象把一个抛物线绕对称轴旋转一圈，就得到了旋转抛物面。卫星电视接收天线的形状就是一个旋转抛物面——因为它能把平行入射的信号汇聚到焦点上。

双曲面（Hyperboloid）

x²/a² + y²/b² - z²/c² = 1    （单叶双曲面）
        - x²/a² - y²/b² + z²/c² = 1   （双叶双曲面）

单叶双曲面的形状像一个"马鞍"的立体版——它是直纹面（ruled surface），即由直线扫成的曲面。冷却塔的经典造型就是单叶双曲面，这种结构在工程上兼具强度和稳定性。

如何快速识别二次曲面？
看方程中平方项的系数符号和次数：
· 三个平方项系数均为正 → 椭球面（或球面）
· 两个平方项正、一个负 → 单叶双曲面
· 一个平方项正、两个负 → 双叶双曲面
· 一个是一次项 → 抛物面（一次项在哪根轴，开口就沿哪根轴）

空间曲线在坐标面上的投影

在工程和物理问题中，经常需要把空间曲线"压"到某个坐标平面上。例如，一个三维空间中运动的质点，它的轨迹在 xy 平面上的投影就是它的"俯视图"。

设空间曲线由两个曲面的交线给出：

F(x, y, z) = 0
        G(x, y, z) = 0

要得到它在 xy 平面上的投影，从两个方程中消去 z，得到 H(x, y) = 0。投影曲线就是方程组 { H(x, y) = 0, z = 0 }。

例如，圆柱面 x² + y² = R² 与平面 z = y 的交线是一条空间椭圆。从两个方程中消去 z（已经消去了），在 xy 平面上投影得到 x² + y² = R²——即一个圆。这条空间曲线投影到 xy 平面上就是圆柱的底面圆。

投影技术在实际中有广泛的应用：

  • 在工程制图中，用三视图（正视图、俯视图、侧视图）来表示三维物体
  • 在计算机图形学中，3D 场景通过投影变换渲染到 2D 屏幕上
  • 在物理模拟中，三维的受力分析可以投影到特定方向上进行简化

关于投影，我不得不说起我大学时期的一个"血的教训"。有一次做一道曲面积分的题目，题目给的曲面是空间中的一条曲线绕轴旋转而成的。我急于计算，对着参数方程一顿猛算，算了整整三页草稿纸，答案还是不对。后来助教看了一眼说："你为什么不先把这个旋转曲面的母线投影到 xy 平面上看看？"我试了一下——原来那个曲面在 xy 平面的投影就是一个简单的圆盘，绕 z 轴对称，用柱坐标只需三行就计算完毕。那三页草稿纸就这么白白浪费了。

这个故事告诉我们：在三维空间几何中，选择合适的视角（即选择合适的投影面）往往比蛮力计算更重要。很多时候你只需要退一步，把三维问题降维成一个二维问题，一切就豁然开朗了。

空间解析几何和向量为多元函数微积分提供了必不可少的"坐标系"。本章建立起来的向量运算（点乘、叉乘）、平面和直线方程、二次曲面识别，在接下来的偏导数、重积分、曲线/曲面积分中会反复出现。下一章我们将正式开始讨论多元函数的偏导数——你将会看到，当一个函数依赖于多个变量时，"变化率"这个概念变得远比一元函数丰富和有趣。
