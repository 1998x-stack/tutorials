从曲线到图像：导数的全面实战

大家好，我是你们的讲师。经过前面六章的积累，我们已经掌握了从导数定义到求导法则、从隐函数求导到泰勒公式的完整工具链。现在，是时候把这些工具投入实战了。

导数的应用贯穿了从工程优化到金融分析的各个领域。一家航空公司想知道如何定价才能使利润最大——这是求极值。一个建筑师想知道拱形屋顶是否会出现应力集中的拐点——这是研究凹凸性。一个数据科学家想了解模型误差随着训练轮次的变化趋势——这是分析单调性。所有这些问题的数学内核，就是本章要讲的内容。

让我讲一个我亲历的故事。本科时参加一个商业案例竞赛，题目是"为一个连锁咖啡店优化纸杯尺寸"。大多数人从市场调研入手，而我从数学入手：假设纸杯是圆锥台形，用导数求出给定容积下表面积最小的形状——这直接决定了原材料成本。最终我们组用这个"数学解法"拿到了最佳方案奖。评委说："其他人都给了市场方案，只有你们给了数学方案。"这就是导数极值应用的力量——它不是一个抽象的数学题，而是实实在在能帮企业省钱、帮工程师优化设计的工具。

用一阶导数判定单调性

这是导数最直接的应用之一。我们已经在第12章中值定理的推论中见过它：

单调性判定定理：设 f(x) 在区间 I 上可导，则：

  • 如果 f'(x) > 0 对所有 x ∈ I 成立，则 f 在 I 上严格递增
  • 如果 f'(x) < 0 对所有 x ∈ I 成立，则 f 在 I 上严格递减
  • 如果 f'(x) = 0 对所有 x ∈ I 成立，则 f 在 I 上为常数
例：求 f(x) = x³ - 3x + 1 的单调区间。

f'(x) = 3x² - 3 = 3(x² - 1) = 3(x+1)(x-1)

令 f'(x) = 0，得驻点 x = -1 和 x = 1。

划分区间：
(-∞, -1)：f'(-2) = 3(4-1) = 9 > 0 → 单调递增
(-1, 1)：f'(0) = 3(0-1) = -3 < 0 → 单调递减
(1, +∞)：f'(2) = 3(4-1) = 9 > 0 → 单调递增

所以 f(x) 在 (-∞, -1) 和 (1, +∞) 上递增，在 (-1, 1) 上递减。

极值的判定

第一充分条件

如果 x0 是 f 的驻点（f'(x0) = 0），那么在 x0 两侧：

  • f' 从正变负 → x0 是极大值点
  • f' 从负变正 → x0 是极小值点
  • f' 不变号 → x0 不是极值点（是拐点）

以上面的 f(x) = x³ - 3x + 1 为例：

  • x = -1：左边 f' > 0（递增），右边 f' < 0（递减），正变负 → 极大值，f(-1) = -1 + 3 + 1 = 3
  • x = 1：左边 f' < 0（递减），右边 f' > 0（递增），负变正 → 极小值，f(1) = 1 - 3 + 1 = -1

第二充分条件

如果 f'(x0) = 0 且 f''(x0) ≠ 0，那么：

  • f''(x0) < 0 → x0 是极大值点（函数在 x0 处凸向下，即上凸）
  • f''(x0) > 0 → x0 是极小值点（函数在 x0 处凸向上，即下凸）

用上面的例子：f''(x) = 6x

  • f''(-1) = -6 < 0 → 极大值 ✅
  • f''(1) = 6 > 0 → 极小值 ✅

第二充分条件比第一充分条件使用起来更方便——只需计算一个二阶导数值，不需要分析一阶导数在两侧的符号。但它也有局限性：当 f''(x0) = 0 时，这个方法失效，需要回到第一充分条件。

条件 判定方式 适用情况 缺点
第一充分条件 f' 在 x0 两侧的符号变化 所有驻点 + 不可导点 需要分析两侧符号
第二充分条件 f''(x0) 的符号 f'(x0)=0 且 f''(x0)≠0 f''(x0)=0 时失效
不可导点的极值
极值不仅可能出现在驻点（f'=0），也可能出现在不可导点。经典例子：f(x) = |x| 在 x=0 处不可导，但它确实是极小值点。所以找极值时，既要检查驻点，也要检查不可导点。

操作流程：① 找出所有可能的极值点（驻点 + 不可导点）→ ② 对每个点用充分条件判定。

凹凸性与拐点

凹凸性的判定

二阶导数的符号决定了函数图像的弯曲方向：

  • f''(x) > 0 → 函数是凹的（concave up），图像像碗形 ∪
  • f''(x) < 0 → 函数是凸的（concave down），图像像倒碗形 ∩

注意：不同教材对"凹凸"的定义可能相反，所以理解"二阶导数的正负"比记住"哪个叫凹凸"更重要。你只需要记住：f'' > 0 时切线斜率在增大（图像向上弯曲），f'' < 0 时切线斜率在减小（图像向下弯曲）。

例：讨论 f(x) = x⁴ - 6x² + 4 的凹凸性。

f'(x) = 4x³ - 12x
f''(x) = 12x² - 12 = 12(x² - 1)

令 f''(x) = 0，得 x = -1 和 x = 1。

(-∞, -1)：f''(-2) = 12(4-1) = 36 > 0 → 凹
(-1, 1)：f''(0) = 12(0-1) = -12 < 0 → 凸
(1, +∞)：f''(2) = 12(4-1) = 36 > 0 → 凹

拐点的判定

拐点（inflection point）是函数图像上凹凸性发生改变的点。如果 f''(x0) = 0 且 f'' 在 x0 两侧变号，那么 (x0, f(x0)) 是一个拐点。

上面的例子中，x = -1 和 x = 1 都是拐点，因为 f'' 在它们两侧都改变了符号。实际上：

  • f(-1) = (-1)⁴ - 6(1) + 4 = 1 - 6 + 4 = -1，拐点 (-1, -1)
  • f(1) = 1 - 6 + 4 = -1，拐点 (1, -1)
f''(x0)=0 并不一定是拐点
就像 f'(x0)=0 不一定是极值点一样，f''(x0)=0 也不一定是拐点。反例：f(x) = x⁴，f''(x) = 12x²，在 x=0 处 f''(0)=0，但 f''(x) > 0 对所有 x≠0 成立，符号没有变化——所以 (0,0) 不是拐点。

因此，拐点的判定规则与极值类似：不仅要检查二阶导数为零的点，还要看二阶导数在两侧的符号是否真的改变了。

渐近线的求法

渐近线（asymptote）描述的是函数图像在无穷远处的"趋向"行为。有三种类型：

水平渐近线

如果 lim_{x→+∞} f(x) = L 或 lim_{x→-∞} f(x) = L，那么 y = L 是水平渐近线。

例：f(x) = (2x² + 1) / (x² - 1)
lim_{x→±∞} (2x² + 1) / (x² - 1) = 2
所以 y = 2 是水平渐近线。

垂直渐近线

如果 lim_{x→x0⁺} f(x) = ±∞ 或 lim_{x→x0⁻} f(x) = ±∞，那么 x = x0 是垂直渐近线。通常出现在分母为零的点。

例：f(x) = (2x² + 1) / (x² - 1)
分母 x² - 1 = (x+1)(x-1) = 0 → x = -1 和 x = 1
lim_{x→1⁺} f(x) = +∞, lim_{x→1⁻} f(x) = -∞
所以 x = 1 和 x = -1 都是垂直渐近线。

斜渐近线

当水平渐近线不存在时，函数可能在无穷远处趋近于一条斜线 y = kx + b：

k = lim_{x→∞} f(x)/x
b = lim_{x→∞} [f(x) - kx]

例：f(x) = (x² + 1) / (x - 1)

k = lim_{x→∞} (x² + 1) / [x(x-1)] = lim_{x→∞} (x² + 1) / (x² - x) = 1

b = lim_{x→∞} [(x² + 1)/(x-1) - x]
  = lim_{x→∞} [(x² + 1 - x(x-1)) / (x-1)]
  = lim_{x→∞} [(x² + 1 - x² + x) / (x-1)]
  = lim_{x→∞} (x + 1) / (x - 1) = 1

所以斜渐近线为 y = x + 1。
渐近线的快捷判断
对于有理函数（多项式之比）：
- 分子次数 < 分母次数 → 水平渐近线 y=0
- 分子次数 = 分母次数 → 水平渐近线 y = 首项系数比
- 分子次数 = 分母次数 + 1 → 有斜渐近线
- 分子次数 ≥ 分母次数 + 2 → 无渐近线（有更高阶的"弯曲"趋向）

垂直渐近线通常出现在分母为零且分子不为零的点。如果分子也为零，需要检查极限是否为无穷大。

函数作图的完整流程

现在，我们把所有工具组合起来，看如何完整地绘制一个函数的图像。

函数作图六步法：

  1. 定义域与基本性质：求定义域，判断奇偶性（对称性）、周期性
  2. 渐近线：求水平和垂直渐近线、斜渐近线
  3. 一阶导数分析：求 f'(x)，找驻点，确定单调区间
  4. 二阶导数分析：求 f''(x)，找拐点候选点，确定凹凸区间
  5. 特殊点：求与坐标轴交点（x 截距和 y 截距），标注极值点和拐点的坐标
  6. 连线成图：根据以上信息描绘图像，注意单调性和凹凸性的过渡
综合示例：作图 f(x) = x³ / (x² - 1)

第1步：定义域
分母 x² - 1 ≠ 0 → x ≠ ±1
定义域：(-∞, -1) ∪ (-1, 1) ∪ (1, +∞)
奇偶性：f(-x) = (-x)³/(x²-1) = -x³/(x²-1) = -f(x) → 奇函数，关于原点对称

第2步：渐近线
垂直渐近线：x = -1 和 x = 1
斜渐近线：k = lim x³/[x(x²-1)] = 1, b = lim [x³/(x²-1) - x] = 0
          所以 y = x 是斜渐近线

第3步：一阶导数
f'(x) = [3x²(x²-1) - x³·2x] / (x²-1)² = (3x⁴-3x²-2x⁴)/(x²-1)²
      = (x⁴ - 3x²) / (x²-1)² = x²(x²-3)/(x²-1)²

f'(x)=0 → x=0, x=±√3

单调性：
(-∞,-√3)：f' > 0 → 递增
(-√3,-1)：f' < 0 → 递减
(-1,0)：f' < 0 → 递减
(0,1)：f' < 0 → 递减
(1,√3)：f' < 0 → 递减
(√3,+∞)：f' > 0 → 递增

第4步：二阶导数（略，直接给出结果）
f''(x) = 2x(x²+3)/(x²-1)³
f''(x)=0 → x=0

凹凸性：
(-∞,-1)：f'' < 0 → 凸
(-1,0)：f'' > 0 → 凹
(0,1)：f'' < 0 → 凸
(1,+∞)：f'' > 0 → 凹
拐点：x=0 处 f'' 改变符号，且 f(0)=0，拐点 (0,0)

第5步：特殊点
与 x 轴交点：x=0 → (0,0)
极值点：x=-√3 极大值，f(-√3) = -3√3/2
        x=√3 极小值，f(√3) = 3√3/2
拐点：(0,0)

第6步：连线（根据以上信息可画出完整图像）

完成这六步后，你就拥有了这个函数的"完整画像"。你会发现，用导数分析过的函数图像不再是凭感觉画的——每一条曲线、每一个转折、每一条渐近线都有精确的数学依据。

一个真实的"做题翻车"故事
我当 TA 时有一次批改作业，题目是画 f(x) = x/(x²-1) 的图像。有个同学画得非常认真——坐标轴、刻度、曲线——但图像完全是错的。错在哪里？他把 x=1 和 x=-1 这两条垂直渐近线之间的部分画成了一个"拱形"，而实际上函数在这个区间里是单调递减的，根本没有"拱"。

问题出在他跳过了"一阶导数分析"这一步，直接凭感觉画了。函数作图的价值就在于此：它强迫你系统地分析函数，而不是依赖直觉。很多"看起来应该是这样"的感觉，在导数面前不堪一击。

最大值最小值的实际应用

这是导数在现实生活中最重要的应用之一。求闭区间 [a, b] 上连续函数的最值：

  1. 找出所有临界点（驻点 + 不可导点）
  2. 计算所有临界点和端点处的函数值
  3. 比较这些值，最大的是最大值，最小的是最小值
优化问题：用一面墙（足够长）和 100 米篱笆围成一个矩形区域。
如何围使面积最大？

设垂直于墙的边长为 x 米，平行于墙的边长为 y 米。
篱笆总长：2x + y = 100 → y = 100 - 2x
面积 S = x · y = x(100 - 2x) = 100x - 2x²
定义域：x > 0 且 y > 0 → 0 < x < 50

S'(x) = 100 - 4x = 0 → x = 25
S''(25) = -4 < 0 → 极大值（也是最大值）

所以 x = 25, y = 50, 最大面积 S = 25 × 50 = 1250 平方米。  ✅

这种优化问题在现实中无处不在：如何用最少的材料造出最大的容器？如何定价使利润最大？如何分配预算使营销效果最好？它们本质上都是在求解导数为零的方程。

优化问题的解题要点
1. 确定变量：把实际问题中的变量用字母表示
2. 建立目标函数：写出你要最大化或最小化的量的表达式
3. 确定约束条件：变量之间的限制关系，用来消元
4. 确定定义域：实际问题中变量往往有范围限制
5. 求导数并令其为零：得到临界点
6. 验证：用第二充分条件或端点值判断是最大还是最小

本章小结

本章整合了前面七章的所有求导知识，将它们应用于分析函数性质这个核心任务上。一阶导数决定单调性和极值候选点，二阶导数决定凹凸性和拐点，渐近线描述了无穷远处的行为，六步作图法将所有这些信息整合成一幅完整的函数图像。最后，最值优化问题展示了导数知识在现实世界中的直接应用——从工程设计到商业决策，导数无处不在。
