第25章 · 素材活学活用

大家好，我是你们的讲师。前两章我们干了件大事——建了素材体系，装了时政弹药。现在你手里应该有几十张素材卡，每张卡上都是一个经典案例+一条金句+几个关键数据。你可能觉得自己已经"弹药充足"了。但我要告诉你一个扎心的真相：素材背下来不等于会用，会用不等于用得妙。

这一章我们解决什么问题？就是"怎么把死素材用活"。素材活学活用的核心能力，就是"一材多用"和"跨话题迁移"——同样一个案例，换个角度看，就能论证完全不同的主题。掌握了这个能力，你手里的30个案例，在考场上就能发挥出90个案例的效果。

一、"华为5G"用遍所有题——翻车现场

先讲一个典型的反面案例。几年前我有个学生，特别勤奋，积累了大约50个案例。但他有一个"致命习惯"——无论考什么主题，他都用"华为5G"这个案例。考科技创新用华为5G、考自主创新用华为5G、考数字经再用华为5G、考中美贸易还用华为5G……只是每次换一个说法而已。

有一次模考，题目是"基层治理现代化"。他的大作文第三段是这么写的：

"在基层治理中，我们也要学习华为5G的技术创新精神。华为5G凭借自主研发，在全球通信领域取得了领先地位，这启示我们在基层治理中也要勇于创新……"

我批改的时候看到这段，差点没笑出声。华为5G和基层治理有什么关系？强行扯在一起，整个论证逻辑非常牵强。而且更致命的是——他的整篇文章里只有这一个企业案例，阅卷老师在连续看到三遍"华为5G"之后，直接在批注栏写了四个字："素材匮乏"。

这个故事的教训是：素材多不等于素材好，更不等于会用。真正的高手不是"一个案例用到底"，而是"一个案例灵活变，多个案例交替用"。

阅卷人的反感：同一个案例在一篇800-1000字的文章里出现三次以上，阅卷人一定会产生"这人没别的案例了"的印象。更严重的是，如果你每次都把同一个案例硬塞进不同的主题里，会让整篇文章看起来非常"模板化"。记住：素材是为论点服务的，不是你背了几条就一直往外倒的。

二、一材多用：一个案例的四种"变身"

什么叫一材多用？我以"浙江千万工程"这个经典案例为例，展示它是如何在四个完全不同的主题下"变身"的：

使用主题 素材改写版本（论证段落） 改写要点
乡村振兴 "浙江'千万工程'实施20年来，从'千村示范、万村整治'起步，推动农村人居环境发生历史性变革。全省农村生活污水治理率从不足30%提升至90%以上，农村居民人均可支配收入从5431元增长到4.8万元，生动诠释了乡村振兴的实践路径。" 侧重农村建设成果和农民增收——突出"乡村振兴"语境
生态文明 "浙江'千万工程'以环境整治为切入点，全省累计建成美丽乡村特色精品村2000余个。安吉余村从靠山吃山的开矿村变成绿水青山的旅游村，正是'两山'理念的生动实践。实践证明，生态保护与经济发展不是对立的，而是相互促进的。" 侧重环境整治和"两山"理念——突出"生态文明"语境
基层治理 "浙江'千万工程'的成功，靠的是基层党组织的有力引领和群众的广泛参与。全省推行'网格化管理、组团式服务'，将治理触角延伸到每一个村庄。从政府'独角戏'到干群'大合唱'，千万工程走出了一条共建共治共享的基层治理新路。" 侧重组织动员和治理模式——突出"基层治理"语境
共同富裕 "浙江'千万工程'最宝贵的经验，是让发展成果惠及每一个乡村、每一位农民。全省城乡居民收入比从2.45:1缩小至1.9:1，为全国最低之一。从'输血'到'造血'，千万工程证明：共同富裕不是平均主义，而是在高质量发展中实现共享。" 侧重收入差距缩小和成果共享——突出"共同富裕"语境

看到区别了吗？同一个"浙江千万工程"案例，在四个主题下呈现了四个完全不同的面貌。案例的"内核"没有变——都是关于浙江农村的变化——但"外壳"完全变了：选取的数据不同、使用的措辞不同、强调的意义不同。这就是"一材多用"的精髓：同一个故事，讲给不同的人听，要换不同的讲法。

练习方法：选一个你最熟悉的案例（比如"华为芯片突围""浙江千万工程""淄博烧烤"等），拿出一张纸，写出它最少能论证的4个不同主题。写不出来？说明你对这个案例的理解还不够深——你需要重新审视这个案例到底"哪里好""好在哪""还能用在哪儿"。一个好的案例至少要在3个不同主题下运用自如，才算真正"消化"了。

三、素材的"变装"技巧：跨话题迁移

除了"一个案例多主题"，更高阶的能力是"跨话题迁移"——把一个领域的案例迁移到另一个看似不相关的领域来论证。这需要你对案例有更深层的理解。

举个例子。很多同学觉得"淄博烧烤"只是"消费热点"的案例，只能用在经济领域。但你看我这样用它：

  • 论证"服务型政府"——"淄博烧烤爆火的背后，是当地政府快速响应：开通烧烤专列、24小时巡逻、严控价格、连夜修路。这不是偶然，而是一座城市从'管理思维'向'服务思维'转变的缩影。"
  • 论证"社会治理"——"淄博烧烤出圈后，全城市民自发让路于客、志愿者上街疏导、商家承诺不涨价。这种政府引导、社会协同、公众参与的治理模式，是社会治理现代化的生动实践。"
  • 论证"文化自信"——"淄博烧烤的文化内核不是'吃'，而是'烟火气'——中国人独有的社交方式和市井文化。它之所以能火，正是因为触动了人们对传统文化的集体记忆和情感共鸣。"

同样一个"淄博烧烤"，我让它出现在经济、政治、社会、文化四个领域的论证中。关键诀窍是什么？就是"找到案例与主题之间的逻辑连接点"——淄博烧烤和政府服务之间的连接点是"政府做了什么"，淄博烧烤和基层治理之间的连接点是"多方参与"。

"变装"三部曲：第一，找出案例最关键的2-3个核心要素（是什么）；第二，分析这些要素与目标主题之间的逻辑关系（为什么）；第三，用目标主题的"话语体系"重新表达这个案例（怎么说）。三步走完，这个案例就"变装"成功了。

四、避免生搬硬套：素材的"减法"智慧

讲完了怎么活用素材，我必须讲一个同样重要的反面——什么时候素材是加分项，什么时候是扣分项。

素材不是用得越多越好。申论大作文的段落一般在4-5段，每段一个论证层次。你只需要在每个论证层次中选最合适的1-2个案例就够了。但很多同学恨不得每段塞三个案例——结果段落变成了"案例堆砌"，没有分析、没有逻辑、没有深度。

"减法"原则：宁可少用一个恰到好处的案例，也不要堆砌三个牵强的案例。一个案例+100字的分析论证，效果好于三个案例+只有20字的点评。阅卷人想看到的是"你通过案例得出了什么结论"，而不是"你背了多少案例"。

什么时候素材是扣分项？三个信号：

  • 信号一：案例和论点之间没有逻辑关系。比如你论证"乡村振兴的重要性"，硬扯到了"华为5G的创新精神"。阅卷人的反应是：这人在凑字数。
  • 信号二：案例描写过多，挤占了分析空间。一段300字，你花了200字在讲案例的来龙去脉，只剩下100字来论证——这种情况下案例写得再好也没用，因为阅卷人读完发现你"没有论证"。
  • 信号三：案例使用频率过高。整篇文章翻来覆去就那两三个案例，表面上看你有素材，实际上暴露了你的素材库太浅。

下面我给出一个"硬塞素材"和"素材融入论证"的对比，让你直观感受什么叫做"恰到好处"：

【反面示例 · 硬塞素材】
┌────────────────────────────────────────────────────────┐
│  实现乡村振兴需要产业支撑。浙江''千万工程〞就是      │
│  一个很好的例子，它从2003年开始实施，到2023年         │
│  已经20年了，全省农村环境发生了巨大变化，农民         │
│  收入也提高了。另外还有贵州''村超〞，2023年火爆      │

│  电商直播年销售额突破5亿元。这些案例都说明，         │
│  产业兴旺是乡村振兴的关键。（238字）                  │
│                                                        │
│  → 问题：三个案例平铺直叙，"罗列式"写作，             │
│    没有任何分析论证，阅卷人读完只看到"在列案例"       │
└────────────────────────────────────────────────────────┘

【正面示例 · 素材融入论证】
┌────────────────────────────────────────────────────────┐
│  产业振兴是乡村振兴的"牛鼻子"。浙江''千万工程〞      │

│  什么〞。（241字）                                      │
│                                                        │
│  → 优点：只用了一个案例，但深度挖掘，对比正反，       │

└────────────────────────────────────────────────────────┘

看明白了吗？正面示例只用了"浙江千万工程"这一个案例，但它做到了三件事：第一，点明了案例的"底层逻辑"；第二，通过正反对比深化了论证；第三，落脚到了自己的核心观点上。反面示例用了三个案例，但每个都只是"抛锚"——扔出来就完了，没有任何后续加工。

五、建立你的"20个万能案例"弹药库

最好的策略不是追求"我有100个案例"，而是"我有20个案例，每一个都能当5个用"。下面我教你如何打造自己的"弹药库"：

选材标准：覆盖五大领域，每个领域选4个案例。经济（新能源汽车、数字经济、民营经济、中欧班列）、政治（枫桥经验、接诉即办、放管服改革、数字政府）、文化（国潮/故宫文创、文旅融合、非遗保护、游戏出海）、社会（千万工程/乡村振兴、共同富裕、"一老一小"服务、基层矛盾调解）、生态（双碳/新能源、长江禁渔、库布其治沙、垃圾分类）。这只是建议，你可以根据你的兴趣和熟悉程度来调整。

深度加工：每个案例，你至少要写出它能论证的3个不同主题。比如杭州"城市大脑"——可以论证科技创新、数字政府、城市治理、服务型政府、数据赋能等多个主题。如果你发现一个案例只能论证1个主题，说明你对这个案例的理解还不够深——再去查查资料，把案例吃透。

分层管理：把20个案例分成三个梯队。第一梯队（5个）——你最熟悉、用得最顺的"王牌案例"，每个至少能论证4个以上主题。第二梯队（10个）——比较熟悉，每个能论证2-3个主题。第三梯队（5个）——作为"备选"，防止考场上遇到非常偏门的主题。平时重点练习第一梯队，逐步把第二梯队往前推。

定期彩排：每两周做一次"模拟调用"的练习。随便找一个申论真题的作文题，限时5分钟，从你的20个案例中快速选出最适合这个题目的3-4个案例。这个练习不写全文，只做"选材"训练，熟能生巧后，考场上选材会快得多。

一句话记住本章精华：素材活用的关键不是"背了多少"，而是"消化了多少"。一个案例反复用、多角度用、深入分析用，比十个案例走马观花式地抛出来，效果好得多。你用过的每一个案例，才能真正成为你的"武器"。

六、本章小结与第六部分预告

这一章我们讲了素材活学活用的四个核心技巧：一材多用——同一个案例适配不同主题；跨话题迁移——找出案例的"逻辑连接点"；素材减法——宁缺毋滥，一个深入胜过三个浅出；建立20个万能案例武器库，每个案例至少能论证3个主题。

从第23章到第25章，我们完成了整个"素材积累与时政热点"模块的全部内容。你的素材体系建好了，热点准备好了，怎么灵活运用也学会了。这三章给你的是"子弹怎么造、怎么装、怎么打"——但真正的战场，还需要你在真题中去练。
