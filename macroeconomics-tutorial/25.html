第25章 DSGE模型与现代方法

2020年3月，新冠疫情刚刚扩散到全球，各国央行面临一个前所未有的困境：经济活动骤然停止，金融市场剧烈动荡，传统经验全部失效。美联储主席鲍威尔没有拍脑袋决策，他身后有一个庞大的建模团队，正在用DSGE模型模拟不同的政策方案——大规模资产购买、紧急降息到零、重启多种流动性工具。这些模型不是完美的，但它们是决策者手中为数不多的"思想实验平台"。在极度的不确定性中，一个逻辑自洽的理论框架比纯粹的经验判断更能让人保持清醒。

大家好，我是你们的讲师。在第二十三和第二十四章，我们分别讨论了经济周期理论和预期理论。现在是时候把这两条线汇聚到一起了。这一章要介绍的DSGE模型（动态随机一般均衡模型），是现代宏观经济学的主导范式。它是理性预期革命、卢卡斯批判和微观基础运动的最终产物。从央行到财政部，从国际货币基金组织到顶级投行，DSGE模型是政策分析和经济预测的标准化工具。

1. 从IS-LM到DSGE的演进：为什么宏观需要"微观基础"

回顾一下我们在前面章节学过的IS-LM模型。它有两条曲线：IS代表产品市场均衡（投资=储蓄），LM代表货币市场均衡（货币需求=货币供给）。这个模型在几十年内一直是宏观经济分析的标准框架。它简洁、直观，用手绘就能分析财政和货币政策的效果。

但IS-LM有一个根本性的问题：它不是从个体决策推导出来的。IS曲线为什么向右下倾斜？因为它假设利率越低投资越多——但这个"假设"是从哪里来的？企业的投资行为背后涉及利润最大化、跨期选择、不确定性。LM曲线为什么向右上倾斜？因为它假设收入越高货币需求越多——但个人和企业如何决定持有多少货币？这涉及交易动机、预防动机和投机动机的最优权衡。

在卢卡斯批判（1976）之前，经济学家不太在意这些。只要模型能拟合历史数据、能做出大致合理的预测，那就是个好模型。但卢卡斯告诉大家：没有微观基础的模型，在政策环境发生变化时会彻底失效。因为模型的"参数"不是自然常数，而是受政策影响的人为选择的结果。

正是这一认知催生了现代宏观经济学的方法论转向：构建模型必须"从底层做起"——从家庭的效用函数、企业的生产函数出发，推导出它们的最优决策，然后加总到整个经济层面。这样的模型才经得起卢卡斯批判的检验。DSGE模型就是这个方法论转向的最终产物。

维度 IS-LM模型（传统） DSGE模型（现代）
微观基础 无——行为方程是直接假设的 有——从效用最大化和利润最大化推导
预期 简单外推或忽略 理性预期（前瞻性）或学习预期
动态 静态或比较静态分析 跨期优化，完整动态路径
随机性 确定性的"冲击"分析 随机冲击驱动波动（技术、偏好、政策等）
一般均衡 部分市场均衡（产品+货币） 所有市场同时出清（含劳动市场、债券市场）
政策评估 约化形式参数估计 结构参数不受政策变化影响（卢卡斯批判免疫）

DSGE这个缩写里的每个字母都有含义：Dynamic（动态——涉及跨期决策），Stochastic（随机——包含不确定性冲击），General（一般——所有市场联动），Equilibrium（均衡——所有行为一致）。可以说，DSGE是宏观经济学二百年来最野心勃勃的建模尝试——它试图在一个框架中同时涵盖增长、周期、政策和预期。

2. 新凯恩斯DSGE的标准结构

目前学术界和央行使用最广泛的DSGE框架是新凯恩斯DSGE（New Keynesian DSGE）。它是RBC模型（强调微观基础和一般均衡）和新凯恩斯主义（强调价格黏性和市场摩擦）的融合体。一个标准的NK-DSGE模型由以下四个核心构件组成：

构件一：家庭（Household）——效用最大化。代表性家庭在预算约束下，选择消费、劳动供给和储蓄（购买债券），以最大化一生效用的贴现值。这个部分推导出两个关键方程：欧拉方程（决定消费的跨期配置——今天的消费取决于对未来利率和通胀的预期）和劳动供给方程（决定在消费和休闲之间的权衡）。

构件二：企业（Firm）——利润最大化。代表性企业在约束条件下选择价格和产量。但新凯恩斯DSGE的"企业"不是完全竞争的——它们具有一定的市场势力（垄断竞争），而且面临着价格调整成本（Calvo定价——每期只有一部分企业能调整价格）。正是这个设定引入了"价格黏性"，使得货币政策能够对实体经济产生影响。

构件三：央行（Central Bank）——泰勒规则。央行遵循一个明确的利率规则来响应通胀和产出缺口的变化。最经典的设定是泰勒规则（Taylor Rule）：当通胀高于目标或产出高于潜在水平时加息，反之降息。央行通过这种行为规则"封闭"了模型——它决定了货币政策如何对冲击做出反应。

构件四：市场出清（Market Clearing）。所有市场（产品市场、劳动市场、债券市场）同时达到均衡。家庭的总供给 = 企业的总需求，没有"多余的"产出也没有"不足"的供给。这个条件让整个模型在逻辑上自洽。

新凯恩斯DSGE的三个核心方程（简化版线性形式）：

① 动态IS曲线（来自欧拉方程）：
   x_t = E_t[x_{t+1}] - (1/σ)(i_t - E_t[π_{t+1}] - r_t^n)

   含义：今天的产出缺口(x_t)取决于预期的未来产出缺口
   和实际利率(i_t - E_t[π_{t+1}])相对于自然利率(r_t^n)的偏离

② 新凯恩斯菲利普斯曲线（来自企业定价决策）：
   π_t = βE_t[π_{t+1}] + κ·x_t + u_t

   含义：今天通胀(π_t)取决于未来通胀预期和当前产出缺口
   κ衡量价格黏性的程度——黏性越大，κ越小

③ 泰勒规则（央行行为）：
   i_t = φ_π·π_t + φ_x·x_t + ε_t

   含义：央行根据通胀和产出缺口调整利率
   φ_π > 1（泰勒原则）确保货币政策能稳定通胀

三个方程联立求解，就得到整个经济的均衡动态路径
理解这三个方程之间的"对话"
这三个方程构成了一个完整的反馈系统。想象一个负面需求冲击：

① 动态IS：冲击降低了预期未来产出 → 企业减少今天的投资和雇佣 → 产出缺口为负 → 经济进入衰退
② NKPC：负的产出缺口给通胀带来下行压力 → 通胀走低甚至通缩风险
③ 泰勒规则：央行看到通胀低于目标 → 降低政策利率 → 实际利率下降 → 刺激消费和投资 → 经济逐渐回到潜在水平

你看，三个方程就像三个齿轮——一个转动，另外两个跟着动。模型就是这个"齿轮系统"的数学表达。

3. DSGE的三类冲击与脉冲响应

DSGE模型不是用来解释"稳态"的——它的核心用途是分析冲击发生后经济如何动态调整。一个标准的NK-DSGE模型通常包含三类冲击：

第一类：技术冲击（Technology Shock）。这是从RBC模型继承来的。技术水平A_t的意外变动会影响总供给——正面技术冲击（如AI技术的突破）提高生产效率，降低边际成本，从而降低通胀、刺激产出。负面技术冲击（如石油禁运）则相反。

第二类：偏好冲击（Preference Shock）。消费者的"耐心程度"或对消费vs储蓄的偏好发生了意外变化。典型例子：疫情暴发时，人们因为不确定性急剧增加而大幅提高储蓄率，即使利率没有变化——这就是一个负面的偏好冲击（当期消费的边际效用下降）。

第三类：货币政策冲击（Monetary Policy Shock）。央行在泰勒规则之外的"意料之外"的加息或降息。比如2008年美联储突然降息75个基点（远超泰勒规则的建议），那多出来的部分就是政策冲击。这种冲击会通过影响实际利率传导到实体经济。

模型对这些冲击的响应，通常用脉冲响应函数（Impulse Response Function，IRF）来展示。IRF描述了：给定一个标准差大小的冲击，关键经济变量（产出、通胀、利率、就业等）在未来若干期内如何偏离它们的稳态值。

冲击类型 对产出的影响 对通胀的影响 对利率的影响 典型例子
正向技术冲击 上升（持久） 下降 下降（央行应对低通胀） 互联网技术的普及
负向偏好冲击 下降（暂时） 下降 下降（央行宽松应对） 疫情中的"预防性储蓄"
紧缩性货币政策冲击 下降（暂时） 下降 先升后降（冲击自身衰减） 美联储2022年的激进加息

读IRF要抓住三个关键：方向（是正还是负？）、幅度（多大？）、持久性（多久恢复？）。一类好的DSGE模型，它的IRF应该在定性上符合经济学直觉，在定量上匹配真实数据的"经验脉冲响应"。如果模型说"加息会导致通胀上升"，那这个模型就有问题——不是模型错了就是代码写错了。

4. 动手实践：用Dynare求解最简单的RBC模型

理论讲完了，我们动手做一件具体的事。我们用一个名为Dynare的软件包来求解一个最简单的RBC模型。Dynare是MATLAB/Octave的一个工具箱，专门用于求解、模拟和估计DSGE模型。全世界的央行和学术机构都在用它。

下面这个代码块展示了一个标准的RBC模型（没有价格黏性，没有货币）的Dynare代码。它只有不到20行，但包含了整个DSGE建模的核心流程。

// 最简单的RBC模型——Dynare代码
// 包含：家庭效用最大化 + 企业利润最大化 + 一般均衡

var y c k i l w r a;          // 定义内生变量
varexo e_a;                    // 定义外生冲击（技术冲击）

parameters beta delta alpha rho sigma;

// 参数校准
beta  = 0.99;   // 贴现因子（反映耐心程度）
delta = 0.025;  // 资本折旧率（每季度2.5%）
alpha = 0.33;   // 资本收入份额（劳动收入占67%）
rho   = 0.95;   // 技术冲击的持久性
sigma = 0.01;   // 技术冲击的标准差

// 模型方程（一阶条件+资源约束）
model;
    // 欧拉方程：消费的跨期选择
    1/c = beta*(1/c(+1))*(1+r(+1)-delta);
    // 劳动供给：消费和休闲的权衡
    w = (1-alpha)*exp(a)*(k(-1)/l)^alpha;
    // 劳动需求：劳动边际产出=工资
    w = (1-alpha)*exp(a)*(k(-1)/l)^alpha;
    // 资本回报率
    r = alpha*exp(a)*(k(-1)/l)^(alpha-1);
    // 生产函数：柯布-道格拉斯
    y = exp(a)*(k(-1)^alpha)*(l^(1-alpha));
    // 资本积累：投资=储蓄
    k = (1-delta)*k(-1) + i;
    // 资源约束：总产出=消费+投资
    y = c + i;
    // 技术冲击过程：AR(1)
    a = rho*a(-1) + e_a;
end;

// 稳态值计算
initval;
    k = 9;
    l = 0.33;
    // ...（其余初值由Dynare自动计算）
end;

// 执行模拟
stoch_simul(order=1, irf=40) y c i l k a;

Dynare把这个代码输入进去，会自动完成以下工作：

  1. 计算模型在"无冲击"情况下的稳态（长期均衡值）
  2. 在稳态附近对模型做对数线性化（把非线性方程变成线性方程）
  3. 求解线性理性预期方程组（使用Blanchard-Kahn条件）
  4. 生成脉冲响应函数（你指定什么冲击→它模拟40期的影响）
  5. 可以进一步做二阶矩匹配（模拟数据的方差、自相关，与真实数据对比）

不到20行代码，你就建立了一个完整的宏观经济模型。这就是DSGE框架的魅力——一旦你掌握了"微观基础+一般均衡+动态优化"的建模语言，你可以不断往模型里添加新的模块：加入政府（财政政策）、加入开放经济（进出口、汇率）、加入金融摩擦（金融加速器）、加入异质性家庭（后面会讲的HANK模型）。

给学习者的实践建议
想亲手跑一跑DSGE模型？你需要以下环境：
· 安装 MATLAB（商业软件）或免费开源的 GNU Octave
· 安装 Dynare（dynare.org，免费，支持Windows/Mac/Linux）
· 下载一个标准的RBC模型.mod文件（Dynare官网有很多示例）
· 打开终端，进入文件目录，输入 dynare rbc_model.mod

几分钟后，你就会看到脉冲响应图出现在屏幕上。亲眼看到"一条技术冲击通过欧拉方程传导到消费和投资，最终影响产出"的动态过程，比任何纸上谈兵都更有教育意义。

5. DSGE的前沿与争议

DSGE模型并非没有批评者。事实上，在2008年全球金融危机之后，DSGE模型遭受了猛烈的批评。时任英国女王伊丽莎白二世访问伦敦经济学院时提出了一个著名的问题："为什么经济学家没有预测到这场危机？"这个问题直接打在了DSGE模型的痛处。

批评者指出了DSGE模型的几大缺陷：

  • "代表性主体"的荒谬性。标准DSGE假设经济中有一个"代表性家庭"——这等于说所有人的行为可以平均为一个典型个人。但现实中，高收入者和低收入者的消费行为天差地别；贷款人和储蓄者的行为也完全不同。用一个"平均人"来代表所有人，会掩盖大量重要信息。
  • 金融部门的缺失。危机前的DSGE模型几乎不包含金融中介。银行只是"面纱"——贷款和存款完全等价。这就好比一辆车没有方向盘和刹车——你根本看不到金融加速器、信贷冻结、明斯基时刻这些现象。
  • 线性近似的局限。标准的DSGE求解方法在稳态附近做线性化。这在"小冲击"下没问题，但2008年那样的大萧条级别的冲击已经远远偏离了线性近似的有效范围。
  • 预测能力存疑。DSGE模型在解释历史数据方面表现出色，但真实世界的预测能力经常不如简单的VAR（向量自回归）模型。

面对这些批评，DSGE模型也在进化。以下是最重要的几个前沿方向：

前沿领域 英文缩写 解决了什么问题 关键人物/机构
异质性代理人新凯恩斯 HANK 引入异质性家庭（有储蓄者和借贷者），研究不平等与宏观政策的互动 Kaplan, Moll, Violante (2018)
金融摩擦DSGE Financial DSGE 包含银行、金融加速器、信贷约束，可以模拟金融危机 Gertler & Karadi (2011)
非线性DSGE Nonlinear DSGE 使用全局方法求解模型，不再依赖线性近似，可处理大冲击 Fernandez-Villaverde等
气候DSGE Climate DSGE 将气候变化和碳税纳入一般均衡框架 Nordhaus (2018 Nobel), Golosov等
CBDC分析 CBDC-DSGE 模拟央行数字货币对银行体系和货币政策的传导影响 各国央行研究部门

DSGE模型最关键的反思是什么？我认为是模型与现实之间关系的重新定义。在战前，宏观经济学家倾向于认为"模型就是真理"——你建一个模型，就以为弄明白了真实的机制。金融危机教会我们的教训是：模型永远是地图，不是实景。地图的作用是帮你找到方向，而不是把它和实际地形混为一谈。

今天顶尖的央行和宏观经济研究机构在建模时，普遍采用多模型策略（Multiple-Model Strategy）——他们同时运行几个不同的模型（DSGE、VAR、半结构式模型），观察它们给出的信号是否一致。如果所有模型都指向同一个方向，那么政策建议就是可靠的。如果模型之间打架了，那就说明模型设定可能遗漏了某些关键机制——需要回到基础理论去思考。

从第八到第十二章，我们学习了IS-LM和AD-AS——它们是宏观经济学的基本分析工具，简单但有效。从第二十一到第二十二章，我们学习了经济增长理论——索洛模型和内生增长理论讨论的是长期趋势。从第二十三到第二十五章，我们学习了经济周期和现代宏观方法——这是关于经济波动的短期理论和前沿模型。

至此，我们已经完成了从古典到现代、从简单到复杂、从长期到短期的完整知识体系建设。接下来的三章，我们将把这些工具应用到真实世界中——第26章深度分析中国宏观经济，第27章讨论全球宏观经济热点，第28章是一个综合实战案例，你将扮演一次央行行长的角色。前路精彩，别停下脚步。

好的，第二十五章的内容就到这里了。我们从DSGE的起源（卢卡斯批判后对微观基础的追求）开始，剖析了新凯恩斯DSGE的四块基石（家庭、企业、央行、市场出清），解读了三类核心冲击的脉冲响应，亲手写了一段Dynare代码来跑一个RBC模型，最后讨论了DSGE的争议和前沿方向。希望你通过这一章建立的不仅仅是对DSGE的认知，更是一种"从第一性原理出发构建经济分析框架"的思维方式。

下一章，我们将把前面二十五章的所有知识用到最有趣的案例上——中国宏观经济深度分析。我们会用学过的所有工具来解构中国经济的增长密码、结构转型、债务问题和政策演变。准备好迎接最精彩的部分吧。
