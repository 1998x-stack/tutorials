大家好，我是你们的讲师。今天我们要讲一个既优美又反直觉的定理——渐进均分性（Asymptotic Equipartition Property, AEP）。

AEP的名字听起来很学术化，但它的核心直觉其实可以用一句话概括："绝大多数概率质量集中在一个很小的典型集里。"

如果你抛一枚硬币1000次，你会得到什么样的序列？直觉告诉你，大约500次正面、500次反面——这很有道理。但问题是：具体是哪500次是正面？ 所有可能的序列中，恰好500正500反的序列数量有多大？总共有C(1000,500) ≈ 2.7×10²⁹⁹种。听上去很多，但要知道所有可能序列（2¹⁰⁰⁰）的数量更多——大约是10³⁰¹。也就是说，"几乎均衡"的序列只占所有可能序列的极小一部分，但它们的概率之和却接近1。

这很反直觉：大部分序列（那些正反面数量严重不均的）几乎不会出现，而少数"典型"序列几乎占据了所有的概率。AEP定理精确地描述了这一现象，并给出了它的数学形式。

大数定律与典型序列

AEP的数学基础是弱大数定律（Weak Law of Large Numbers）。大数定律说：当试验次数n足够大时，样本均值趋近于期望值。

应用到信息论场景：考虑一个独立同分布（i.i.d.）的随机序列 X₁, X₂, ..., Xₙ ~ P(x)。对每个序列，我们可以计算其经验对数概率：

-(1/n) log p(X₁, X₂, ..., Xₙ) = -(1/n) ∑ log p(X_i)

由于 X_i 是i.i.d. 的，-(1/n) log p(X₁, ..., Xₙ) 是一组独立随机变量 -log p(X_i) 的样本均值。根据大数定律，这个均值收敛到它的期望——也就是熵 H(X)：

-(1/n) log p(X₁, ..., Xₙ) → H(X) （在概率意义下）

换句话说，对于长序列，它的"平均惊讶程度"基本就等于熵。这个收敛关系就是AEP定理的核心。

由此我们可以定义典型序列（typical sequence）：一个序列 (x₁, ..., xₙ) 被称为相对于分布P的"典型序列"，如果它的经验对数概率接近真实熵。形式上：

| -(1/n) log p(x₁,...,xₙ) - H(X) | < ε

等价地，典型序列的概率大约为 2^{-nH(X)}。

这个定义的精妙之处在于：所有典型序列都以近似相等的概率出现（都是 ≈ 2^{-nH(X)}），所以叫"渐进均分"——概率均匀分布在这些典型序列上。

AEP定理的陈述与证明

下面我们给出AEP定理的严格陈述：

设 X₁, X₂, ..., Xₙ 是 i.i.d. 离散随机变量序列，服从概率分布 p(x)。则：

-(1/n) log p(X₁, X₂, ..., Xₙ) → H(X) （依概率收敛）

也就是说，对任意 ε > 0，存在 N 使得当 n > N 时：

P( | -(1/n) log p(X₁,...,Xₙ) - H(X) | < ε ) > 1 - ε

基于这个收敛性质，定义典型集（typical set）A_εⁿ：

A_εⁿ = { (x₁,...,xₙ) : | -(1/n) log p(x₁,...,xₙ) - H(X) | < ε }

A_εⁿ 具有以下三个关键性质：

  1. 大概率：当 n 足够大时，P(A_εⁿ) > 1 - ε。几乎所有概率质量都在典型集中。
  2. 大小近似为 2^{nH(X)}：|A_εⁿ| ≤ 2^{n(H(X)+ε)}，且对足够大的 n，|A_εⁿ| ≥ (1-ε)2^{n(H(X)-ε)}。
  3. 元素概率近似相等：对每个 (x₁,...,xₙ) ∈ A_εⁿ，有 2^{-n(H(X)+ε)} ≤ p(x₁,...,xₙ) ≤ 2^{-n(H(X)-ε)}。

性质1和性质2的结合是AEP最强大的结论：典型集只占所有可能序列的极小比例，但却包含了几乎所有的概率。下一节我们具体感受一下这个数字对比。

典型集的"少即是多"原理

AEP的核心反直觉之处就在这里。让我们用具体数字感受一下：

n（抛硬币次数） 总序列数 2ⁿ 典型集大小 ≈ 2^{nH} 典型集占比 典型集中概率
10 1,024 ≈ 256 25% ≈ 98%
100 ≈ 1.27×10³⁰ ≈ 1.13×10³⁰ ≈ 89% → 1
1,000 ≈ 1.07×10³⁰¹ ≈ 1.43×10³⁰⁰ ≈ 13% → 1
10,000 ≈ 10³⁰¹⁰ ≈ 10³⁰⁰⁰ ≈ 0.001% → 1

看到n=10,000那一行了吗？典型集大小只占总序列的千万分之一都不到，但却包含了几乎所有的概率。这就像什么呢？想象一个有10¹⁰人参加的音乐会，其中99.999%的人只挤在一个不到千分之一的角落里——而其他地方几乎空无一人。这就是AEP描述的图景。

对于公平硬币（H=1 bit），典型序列是那些正面数量接近n/2的序列。对于n=1000，典型集大小 ≈ 2¹⁰⁰⁰·H = 2¹⁰⁰⁰。但等等——公平硬币的H=1，所以典型集大小 ≈ 2¹⁰⁰⁰？这不等于总序列数了吗？

这里有一个微妙之处：对于公平硬币，所有序列都是等概率的（2^{-1000}），所以-(1/n) log p 对所有序列都精确等于1 = H。也就是说，公平硬币的所有序列都是"典型"的——因为它们都是等概率的。AEP告诉我们：当H=log|X|时（均匀分布），所有序列都是典型的——这时典型集就是整个空间。

但对于非均匀分布，情况就精彩了。比如一个偏倚硬币（p=0.9），H=0.469。那么典型序列是那些正面（概率0.9的事件）约占90%、反面约占10%的序列。所有序列中，只有约 2^{0.469n} 个是典型的，而总共有2ⁿ个序列。当n=1000时，典型集 ≈ 2⁴⁶⁹，而全体序列 ≈ 2¹⁰⁰⁰。比例是 2^{-531} ≈ 10^{-160}！

这就是"少即是多"的精髓：我们只需要编码典型序列就足够了——因为非典型序列几乎从不出现。无损压缩需要的码字数量约等于典型集的大小，即大约 2^{nH(X)}，所以平均每个符号只需要约 H(X) 个比特。这正是香农第一定理（信源编码定理）的直觉核心！

高概率集与最小集

典型集并不是唯一能"抓住大多数概率"的集合。事实上，对于任何给定的δ>0，我们可以找到许多不同的集合B_δⁿ，使得P(B_δⁿ) ≥ 1-δ。典型集的特点是：

  • 它是"近似最小"的高概率集——任何高概率集的大小都不会显著小于典型集
  • 它的元素概率近似相等——这是AEP的"渐进均分"含义

更精确地说：设 B_δⁿ 是满足 P(B_δⁿ) ≥ 1-δ 的任意集合。那么：

|B_δⁿ| ≥ (1-δ) 2^{n(H(X)-ε)}

对于足够大的n和任意ε>0成立。这意味着，任何高概率集的大小下限都是 ≈ 2^{nH(X)}。换句话说，你不能通过精心挑选"大部分概率"的集合来显著压缩。

这个结论直接导出了无损压缩的理论极限：要无损地表示n个i.i.d.信源输出，至少需要约 n·H(X) 个比特。少于这个数，你必然丢失信息。这就是香农第一定理的核心内容——我们下一章会详细讨论。

AEP是Shannon第一定理（无损压缩极限）的直觉核心： 我们只需要给典型序列分配码字——而非所有可能序列！典型集的大小约为2^{nH(X)}，因此用n·H(X)个比特就可以唯一标识所有典型序列。非典型序列的概率太小了，即使完全不编码它们，对平均码长的影响也在允许的误差范围内。这就像你设计停车场时，只考虑正常大小的车辆——超长卡车虽然偶尔出现，但你不必给它们预留车位。

用Python模拟一下AEP收敛：

import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

def simulate_aep(p_heads, n_trials=1000, max_n=5000):
    """
    模拟抛硬币，验证 -(1/n) log p(序列) 收敛到 H(p)

    参数:
        p_heads: 正面概率
        n_trials: 蒙特卡洛试验次数
        max_n: 最大序列长度
    """
    # 理论熵
    p = np.array([p_heads, 1-p_heads])
    H = -np.sum(p * np.log2(p))

    # 对不同的n进行计算
    ns = [10, 50, 100, 200, 500, 1000]

    for n in ns:
        # 生成n_trials个长度为n的序列
        # 每个序列，计算 -1/n * log p(序列)
        # p(序列) = p_heads^k * (1-p_heads)^(n-k)
        # 其中k是正面次数
        ks = np.random.binomial(n, p_heads, n_trials)
        log_probs = ks * np.log2(p_heads) + (n - ks) * np.log2(1 - p_heads)
        neg_avg_log = -log_probs / n

        mean_est = np.mean(neg_avg_log)
        std_est = np.std(neg_avg_log)

        print(f"n={n:5d}: 平均 -(1/n)log p = {mean_est:.4f} (H={H:.4f}), 标准差={std_est:.4f}")

print("=== 偏倚硬币 p=0.9 ===\n")
simulate_aep(0.9)

print("\n=== 公平硬币 p=0.5 ===\n")
simulate_aep(0.5)

从输出可以看到：随着n增大，-(1/n)log p(序列)越来越接近H，且方差越来越小。这就是AEP的收敛过程——对于n=1000，偏差已经非常小了。

注意，对于偏倚硬币，收敛到H=0.469，而对于公平硬币，收敛到H=1.000。n越大，-(1/n)log p(序列)偏离H的概率越小——这就保证了典型集的概率趋于1。

这一章我们学习了渐进均分性（AEP），它是连接信息论基本概念和实际压缩理论的桥梁。

关键 takeaways：

  • AEP：-(1/n) log p(X₁,...,Xₙ) → H(X)（依概率收敛）
  • 典型集 A_εⁿ 包含几乎所有概率，大小 ≈ 2^{nH(X)}，元素概率 ≈ 2^{-nH(X)}
  • 典型集只占所有可能序列的极小部分（对非均匀分布），但却抓住了"主要矛盾"
  • 任何高概率集的大小至少为 ≈ 2^{nH(X)}，这意味着 n·H(X) 是无损压缩的理论极限

AEP给了我们一个惊人的启示：只需要大约 2^{nH(X)} 个码字就能"几乎无损"地表示n次独立的信源输出。这就是Shannon第一定理的核心——下章我们正式进入信源编码的世界。
