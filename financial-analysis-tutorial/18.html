第18章 行为金融学

大家好，我是你们的讲师。在开始本章之前，我先请你做一个思想实验。我有两个选项：A. 我直接给你5000元；B. 你抛一枚硬币，正面朝上我给你10000元，反面朝上你一分钱也拿不到。你会选哪个？

绝大多数人在面对这个选择题时会选A——稳稳地拿5000元。现在换一个问题：A. 你确定损失5000元；B. 你抛一枚硬币，正面朝上你损失0元，反面朝上你损失10000元。你选哪个？

有趣的是，同样的绝大多数人在面对"损失"版本时会选B——宁愿冒险赌一把，也不愿意接受确定的损失。这就是行为金融学最核心的发现：人类在"盈利"和"亏损"两个区域内的决策逻辑是完全不对称的。而这个不对称性，正是金融市场中许多"非理性"现象的心理根源。

传统金融学假设"理性人"——市场参与者总是理性的、风险厌恶的、按照期望效用最大化做决策。但如果你做过一天真实的交易就会知道：这个假设和现实之间的距离，大概相当于"理想气体"和"真实气体"之间的距离——作为理论起点可以理解，但直接套用会出大问题。

18.1 认知偏差图谱

行为金融学在过去四十年间发现了几十个影响投资决策的认知偏差。以下是其中对投资者影响最大的几个：

偏差名称 定义 交易中的表现 应对方法
过度自信 高估自身知识、判断或能力的准确度 过度交易、仓位过重、拒绝止损 交易日志复盘、每笔交易记录预测理由
锚定效应 过度依赖最先获得的信息做决策 盯着买入价不肯卖、用历史高点做目标价 不看持仓成本做决策、关注当前基本面
损失厌恶 同等金额的损失带来的痛苦远大于盈利的快乐 死扛亏损头寸、过早卖出盈利头寸 预设止损和止盈、机械执行
确认偏差 倾向于寻找支持自己已有观点的信息 只看多方的研报、忽视警示信号 主动找反面论据、做"魔鬼代言人"
羊群效应 追随大多数人的行为而非独立判断 追涨杀跌、跟风热门板块 独立思考、逆向投资训练
近因效应 过度关注近期事件而忽略长期趋势 大涨后追入、大跌后恐慌卖出 拉长决策时间框架、参考长期均值
自我诊断：坦白说，这些偏差我一个不少全都犯过。有一次我买了某只消费股，跌了15%之后——不是因为公司基本面变了，纯粹是因为"不愿意亏着卖"——硬扛了半年。结果股价又跌了30%。如果我当初理性地计算一下机会成本，这笔错误决策带来的隐形损失比表面上的浮亏要大得多。后来我给自己定了一条铁律：如果今天给你一笔现金，你还会买这只股票吗？如果答案是否，立刻卖出。

18.2 前景理论与损失厌恶

2002年，心理学家丹尼尔·卡尼曼（Daniel Kahneman）获得了诺贝尔经济学奖——尽管他一天经济学都没学过。他和长期合作者阿莫斯·特沃斯基（Amos Tversky，可惜在1996年去世，未能共同获奖）提出的前景理论（Prospect Theory），是行为金融学最重要的理论基石。

前景理论的核心可以用一个简单的图景来概括：想象一条S形的价值函数曲线。在"盈利"区域（曲线的右半部分），曲线是凹的——也就是说，从0到100元的快乐增加，比从1000元到1100元的快乐增加要大得多（边际效用递减）。在"亏损"区域（曲线的左半部分），曲线是凸的——而且比右半部分陡峭得多。

这个不对称的陡峭程度告诉我们一个惊人的事实：一笔1000元的亏损带来的痛苦，大约相当于1000元盈利带来的快乐的2到2.5倍。换句话说，要想弥补一笔亏损带来的心理伤害，你需要的不是等额的盈利，而是两倍以上的盈利。

核心洞见：损失厌恶不仅仅是"不喜欢亏钱"这么简单——它直接导致了投资者两个最常见的错误行为：① 过早卖出盈利股（因为害怕盈利变成亏损，想"落袋为安"）；② 过久持有亏损股（因为不愿意把浮亏变成实亏）。这两个行为叠加在一起的效果极其致命——你在限制盈利的同时放大了亏损。用围棋术语来说，这叫"恶手叠恶手"。

前景理论还有一个重要的概念：参照点依赖（Reference Dependence）。人们判断"盈利"还是"亏损"并不是基于绝对金额，而是基于某个参照点——最常见的参照点就是"买入价"。这就是为什么投资者会做出"10元的股票跌到8元不卖，跌到6元更不卖"的非理性行为——因为只要不卖，亏损就只是"浮亏"，一旦卖出，就变成了"实亏"——而人的心理对"实亏"的厌恶远大于"浮亏"。

下面用Python简单模拟前景理论的价值函数曲线——当损失厌恶系数为2时，亏损区域比盈利区域陡峭一倍以上：

import numpy as np
import matplotlib.pyplot as plt

# 前景理论价值函数参数
alpha = 0.88    # 敏感度递减系数（<1 表示边际递减）
lam = 2.25      # 损失厌恶系数（亏损区域的斜率大约是盈利区域的2.25倍）

# 生成x轴数据（从-10到10的盈亏值）
x = np.linspace(-10, 10, 1000)

# 前景理论价值函数
v = np.where(x >= 0,
             x ** alpha,                 # 盈利区域：凹函数
             -lam * (-x) ** alpha)        # 亏损区域：凸函数且更陡峭

# 在x=0处盈亏从"等价"变为"损失更痛"
print(f"盈利+1 的价值: {1 ** alpha:.3f}")
print(f"亏损-1 的价值: {-lam * 1 ** alpha:.3f}")
print(f"亏损痛感倍率: {lam:.2f} 倍")
# 输出验证：亏损1元的心理成本是盈利1元快乐值的2.25倍

运行这段代码你会发现，同样的1元钱，亏损时的心理冲击（-2.25）是盈利时的愉悦感（+1.00）的两倍多。这个不对称性，就像一把隐藏在每个人脑子里的"反向杠杆"——它让你在牛市中赚得不够多，在熊市中亏得格外痛。

18.3 市场异象与行为解释

传统金融学的"有效市场假说"（Efficient Market Hypothesis, EMH）认为：市场价格已经反映了所有可得信息，任何试图获取超额收益的努力都是徒劳的。然而，现实中的金融市场充满了EMH无法解释的"异象"（anomalies）。行为金融学为这些异象提供了合理的解释。

动量效应（Momentum Effect）——过去几个月表现好的股票，在未来几个月倾向于继续表现好；表现差的股票倾向于继续表现差。Jegadeesh和Titman（1993）的开创性研究发现，买入过去6个月涨幅最高的股票、卖出涨幅最低的股票，可以产生年化超过12%的超额收益。行为学的解释是：投资者对新信息的反应不足（underreaction），导致价格调整缓慢，形成趋势延续。

反转效应（Reversal Effect）——与动量效应相反，从3-5年的长期维度看，过去表现差的股票反而会后来居上。这对应了投资者的过度反应（overreaction）：当市场对一个利好或利空消息反应过度后，价格会在更长时间维度上"回归均值"。

一月效应（January Effect）——历史上，每年1月份小盘股的收益率系统性高于其他月份。可能的解释是：投资者在12月为了"税收损失收割"（tax-loss harvesting）卖出亏损股票，导致价格被压低，然后在1月买回，推动价格上涨。随着市场效率的提高，这一效应在近年已经明显减弱。

处置效应（Disposition Effect）——投资者倾向于过早卖出盈利股票、过久持有亏损股票。这是前景理论+心理账户（mental accounting）的直接体现。研究发现，个人投资者的处置效应比机构投资者更显著——这在一定程度上解释了为什么"散户跑不赢大盘"。

市场异象 表现 行为学解释 对应的策略含义
动量效应 强者恒强（6-12个月维度） 反应不足（anchoring） 趋势跟踪策略的基础
反转效应 均值回归（3-5年维度） 过度反应（overreaction） 逆向投资策略的基础
一月效应 1月小盘股超额收益 税收损失收割 + 窗口粉饰 12月买小盘、1月卖
处置效应 卖盈持亏的非对称行为 损失厌恶 + 心理账户 反向交易散户的行为流
学习建议：理解市场异象的价值不在于"套利"——因为这些异象大部分都已经在学术文献中被充分讨论了，套利空间往往很小（而且可能在你交易前就消失了）。真正的价值在于：当你看到价格行为与你的预期不符时，试着从行为学的角度去理解"市场在想什么"——而不是立刻归咎于"市场错了"。

18.4 如何克服行为偏差

知道了这么多偏差的存在，问题来了：知道了就能避免吗？很不幸，答案通常是"不能"。这一点卡尼曼本人在《思考，快与慢》中也承认——即使是他这样的行为经济学大师，在个人投资决策中照样会犯过度自信和锚定效应的错误。

好消息是：虽然我们无法"治愈"自己的认知偏差，但可以通过系统化的方法来降低偏差的影响。

1. 交易日志——这是我认为最有价值的单一步骤。每一次交易，都要记录：买入决策的逻辑、预期的持有期限、目标价和止损价、当时的情绪状态。定期复盘时你会发现，那些让你赚钱的交易和让你亏钱的交易，在决策模式上有惊人的规律。

2. 系统化规则——把交易规则变成不可违背的"算法"。例如：- 任何单笔亏损超过总资金的2%，无条件平仓；- 持有时间超过3个月且浮亏超过10%，必须重新评估基本面，如果不能给出买入的理由，无条件平仓。规则越具体越好，判断空间越小越好。

3. 预承诺机制（Pre-commitment）——在"冷静期"设定好规则，然后在"冲动期"强制自己遵守。比如：提前在交易系统中设置好止损单，而不是等到市场大跌时再做决定。设置好定投计划并授权自动扣款，而不是每月手动判断"现在该不该买"。

4. 算法辅助决策——骑手在高强度比赛中容易判断失误，所以汽车制造商发明了ABS、ESP等电子辅助系统。同样的道理，你可以用程序化的信号来辅助——不是完全替代你的判断，而是在你被情绪冲昏头脑的时候给你一个"冷冰冰的提醒"。

踩坑警示：我要特别提醒一点：不要因为学了几种偏差就觉得自己比别人聪明。这是"过度自信偏差"的一个更高级的变种——"我知道自己有偏差，所以我比那些不知道的人更高明"——而这个想法本身就是过度自信。行为金融学最有价值的地方是教人谦卑，而不是教人傲慢。

GameStop：散户起义还是羊群狂欢？

2021年初发生的GameStop（游戏驿站）事件，是行为金融学教科书级别的案例。GameStop是一家实体游戏零售商，在数字下载时代的冲击下，其基本面持续恶化——2019年亏损超过7亿美元，股价从2015年的40美元跌至2020年的3美元以下。

但是，Reddit论坛WallStreetBets板块的一群散户注意到了另一个事实：GameStop是被华尔街机构大量做空的股票——做空比例一度超过流通股的140%（因为有多次借券操作）。于是他们开始了一场历史性的"轧空"（short squeeze）行动。

事情的经过是这样的：散户们在论坛上组织起来，大规模买入GameStop的看涨期权和正股。买入推动股价上涨，上涨迫使做空机构追加保证金、被迫买入股票平仓，平仓进一步推高股价——形成了一个自我强化的正反馈循环。股价从2021年初的18美元，在短短两周内飙升至483美元（盘中最高），涨幅超过2500%。

从行为金融学的角度来看，GameStop事件完美展示了多个偏差的同时作用：

  • 羊群效应——散户们看到论坛上的"战绩"后蜂拥而入，很多人根本不了解GameStop的基本面
  • 过度自信——参与者相信自己能"在最高点卖出"，但绝大多数人在股价回落时都没有离场
  • 锚定效应——股价冲到483美元时，很多人以这个价格作为"目标价"，一直持有等待"再回483"，结果股价最终跌回了20美元左右
  • 确认偏差——散户们只关注对多方有利的信息（做空比例仍在高位、机构被迫平仓），忽视了基本面和高估值风险
核心洞见：GameStop事件最值得反思的不是"散户大战机构"的叙事，而是：当一群人因为"共同的敌人"而聚集在一起时，集体非理性的程度会被极大放大。在这场游戏中，真正赚钱的是那些在散户入场之前就布局的早期参与者，以及在高位做空波动率的对冲基金（它们通过卖出看涨期权赚取了巨额权利金）。而那些在股价400美元以上冲进去的散户，大部分承担了最后的损失。在金融市场里，"跟着大家一起冲"很少是一个好策略。

行为金融学的核心价值，是让我们对自己的决策多一分审视、对市场多一分敬畏。它不是说"市场永远是错的"——恰恰相反，它告诉我们，市场有时候看起来"错误"的行为，背后可能有着深刻的心理学根源。理解这些根源，不是让你去"战胜市场"，而是让你更清楚地知道自己为什么容易输给市场。
