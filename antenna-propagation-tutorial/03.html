3.1 什么是极化？直观理解

大家好，欢迎回到课堂。想象你在一个平静的湖面上扔下一块石头，水面泛起一圈圈涟漪——水面上每一个点都在上下振动，振动方向是垂直的。现在再取一根绳子，一端固定，你用手上下抖动另一端——绳子上产生一个波，每个质点的振动方向仍然是垂直的。但如果你的手改为左右抖动呢？振动的方向就变成了水平。

对于电磁波，情况类似但多了一个维度。电磁波中的电场矢量E的振动方向——在垂直于传播方向的平面上的轨迹——就定义了波的极化（polarization）。由于在远场中电场和磁场始终正交且幅度成比例（|E| = η|H|），我们通常用电场方向来描述极化。

为什么要关心极化？因为它是天线与传播之间一个常被忽视却极其关键的匹配维度。让我讲一个真实的故事：

GPS卫星的右旋圆极化：全球定位系统（GPS）的卫星全部使用右旋圆极化（Right-Hand Circular Polarization, RHCP）信号。为什么不用简单的线极化？原因在于法拉第旋转效应（Faraday rotation）——当线极化波穿过电离层时，地球磁场会使极化面发生旋转。GPS信号从20,200公里高的轨道到达地面，需要在电离层中穿行数百公里，极化面的旋转角度难以精确预测，可能高达数十度。如果GPS卫星使用线极化，地面的线极化接收天线一旦与旋转后的极化面不匹配，就会产生巨大的极化失配损耗。而使用圆极化则完全没有这个问题——无论电离层怎么旋转，RHCP信号始终保持RHCP特性。

知识延伸：GPS的L1频段（1575.42 MHz）使用RHCP，而L2频段（1227.60 MHz）也使用RHCP。实际上，绝大多数卫星通信和深空探测任务都选择圆极化——NASA的深空网络（DSN）与旅行者号、火星探测器的通信全部使用圆极化。原因就是消除法拉第旋转的影响。

从数学上看，极化就是电场矢量末端的轨迹。设电磁波沿+z方向传播，电场由两个正交分量组成：

E(z, t) = Ex cos(ωt - kz + φx) ẑ + Ey cos(ωt - kz + φy) ŷ

在固定z处观察，Ex和Ey随时间变化，其矢端轨迹决定了极化类型。Ex和Ey的幅度比和相位差Δφ = φy - φx是决定极化形状的两个关键参数。

3.2 线极化、圆极化与椭圆极化

线极化（Linear Polarization）

当两个分量的相位相同（Δφ = 0°）或相反（Δφ = 180°）时，合成电场矢量在一条直线上振动——这就是线极化。如果只有Ex分量（Ey = 0），那就是水平极化；如果只有Ey分量（Ex = 0），那就是垂直极化；如果Ex = Ey且同相，那就是45°斜极化。

我们日常生活中最常见的线极化源：广播电台的发射天线。AM广播（530-1700 kHz）通常使用垂直极化（发射塔本身就是单极子天线），而FM广播（88-108 MHz）通常使用水平极化（偶极子水平放置）。这就是为什么FM收音机天线（通常呈T形）需要水平展开才能获得最佳接收的原因。

圆极化（Circular Polarization）

当Δφ = ±90°且|Ex| = |Ey|时，矢量末端轨迹是一个完美的圆。根据旋转方向：

  • 右旋圆极化（RHCP）：Δφ = -90°（Ey 滞后 Ex 90°）。顺着传播方向看，电场矢量顺时针旋转。GPS、卫星电视使用RHCP。
  • 左旋圆极化（LHCP）：Δφ = +90°（Ey 超前 Ex 90°）。顺着传播方向看，电场矢量逆时针旋转。

椭圆极化（Elliptical Polarization）

最一般的情况：Δφ ≠ 0°或90°，或者|Ex| ≠ |Ey|。矢端轨迹是椭圆。椭圆极化可以用两个关键参数描述：

  • 轴比（Axial Ratio, AR）：椭圆长轴与短轴之比，AR ≥ 1。AR = 1（0 dB）是理想圆极化，AR → ∞ 是线极化。通常AR ≤ 3 dB可视为"工程上的圆极化"。
  • 倾斜角（Tilt Angle）：椭圆长轴与x轴的夹角。
踩坑故事：多年前我参与一个卫星通信地面终端的测试项目，天线声称是"圆极化"，但实测AR高达6 dB。这意味着它实际上更接近椭圆极化，导致与卫星的RHCP信号之间有约1.5 dB的额外失配损耗。问题出在馈电网络的相位误差上——两根馈线的长度差导致了超过30°的相位偏差。这个教训告诉我们："宣称"圆极化和"做到"圆极化之间，差的是一个精密的馈电网络和严格的相位控制。最终我们用微带线长度补偿彻底解决了问题。
极化类型 相位差 Δφ 振幅比 Ey/Ex 轴比 AR 典型应用
水平线极化 0° 或 180° 0 ∞ FM广播、WiFi、基站天线
垂直线极化 0° 或 180° ∞ ∞ AM广播、移动手机
右旋圆极化 -90° 1 1 (0 dB) GPS、卫星通信、深空探测
左旋圆极化 +90° 1 1 (0 dB) 部分卫星电视、RFID
椭圆极化 一般值 任意 > 1 实际天线（非理想）

3.3 极化匹配与极化失配损耗

在无线链路中，发射天线和接收天线的极化方向必须"对上"才能有效传输能量。这个"对上的程度"用极化匹配因子（Polarization Matching Factor）或极化损耗因子（Polarization Loss Factor, PLF）来量化：

PLF = |êtx · êrx*|²

其中êtx和êrx分别是发射和接收天线的极化单位矢量（可以是复数形式以描述椭圆极化）。PLF的取值范围是0到1，对应的dB值为：

极化失配损耗 (dB) = -10 log₁₀(PLF)

我们来看几个典型场景：

  • 完美匹配（同极化）：êtx = êrx，PLF = 1，损耗 = 0 dB。例如两个垂直极化天线对准。
  • 完全失配（交叉极化）：êtx ⊥ êrx，PLF = 0，损耗 = ∞ dB。理论上收不到任何信号。
  • 线极化接收圆极化：PLF = 1/2，损耗 = 3 dB。手机（线极化天线）接收GPS（圆极化）信号，天然损失3 dB。
  • RHCP接收LHCP：PLF = 0，完全失配。RHCP天线几乎收不到LHCP信号。
工程直觉：在布置WiFi网络时，如果你把AP的天线垂直放置，那么你的手机天线也应该尽量与垂直对齐。但由于人手握持手机会自然倾斜，实际上存在一定程度的极化失配——这就是为什么有时候"转一下手机"信号会变好。MIMO系统利用多极化（如±45°双极化面板天线）来减轻这个问题：两个正交极化路径提供了极化分集。

特别值得注意的是，地面反射可以改变极化。一个垂直极化的信号在从地面反射后，其水平分量和垂直分量的反射系数不同（Fresnel反射系数），导致合成极化状态发生改变。这就是为什么在移动通信中，到达接收天线的信号常常是椭圆极化的——即使发射天线是完美的线极化。

3.4 介质分界面上的电磁波行为

当电磁波从一种介质传播到另一种介质时，在分界面上会同时发生反射和透射（折射）。描述这一行为的理论基础是电磁场的边界条件（boundary conditions），它们直接来自Maxwell方程组的积分形式：

  • 电场切向分量连续：E1t = E2t
  • 磁场切向分量连续（无表面电流时）：H1t = H2t
  • 电位移矢量法向分量连续（无表面电荷时）：D1n = D2n
  • 磁感应强度法向分量连续：B1n = B2n

应用这些边界条件，我们可以推导出著名的斯涅尔定律（Snell's Law）：

反射定律：θr = θi
折射定律：n₁ sinθi = n₂ sinθt，其中 n = √(εrμr) 是折射率

在分析反射和折射问题时，我们需要将入射波分解为两种正交的极化分量：

  • TE极化（Transverse Electric，也称垂直极化或s-极化）：电场矢量垂直于入射面（即平行于分界面）。入射面：由入射波矢和界面法线确定的平面。
  • TM极化（Transverse Magnetic，也称平行极化或p-极化）：磁场矢量垂直于入射面（即电场矢量平行于入射面）。

这两种极化的反射和透射行为完全不同——这就是下一节Fresnel公式要讲的内容。

记忆技巧：TE = "E垂直于入射面"（E Transverse to plane of incidence），TM = "M垂直于入射面"（M/磁场 Transverse）。两种极化定义在不同文献中可能叫法不一——在光学中通常称为s-极化（TE）和p-极化（TM）——重要的是理解物理本质，而不是死记名称。

3.5 反射系数与透射系数的Fresnel公式

Augustin-Jean Fresnel在1823年推导出了描述电磁波在两种介质分界面上反射和透射行为的公式。虽然Fresnel当时用的是光的弹性以太理论，但Maxwell方程组后来从电磁学角度完美验证了他的结果。

TE极化（垂直极化/s-极化）的Fresnel公式：

反射系数：ΓTE = (η₂ cosθi - η₁ cosθt) / (η₂ cosθi + η₁ cosθt)
透射系数：τTE = (2 η₂ cosθi) / (η₂ cosθi + η₁ cosθt)

TM极化（平行极化/p-极化）的Fresnel公式：

反射系数：ΓTM = (η₂ cosθt - η₁ cosθi) / (η₂ cosθt + η₁ cosθi)
透射系数：τTM = (2 η₂ cosθi) / (η₂ cosθt + η₁ cosθi)

其中η₁和η₂是两种介质的本征阻抗，θi和θt是入射角和透射角（由Snell定律联系）。

布儒斯特角（Brewster Angle）

Fresnel公式最引人注目的结论是：当入射角满足特定条件时，TM极化的反射系数会变为零——也就是说，反射波中完全不含TM分量，只有TE分量！这个特殊角度称为布儒斯特角（Brewster angle）：

tan θB = n₂ / n₁

这意味着：如果以布儒斯特角入射一个圆极化波或任意极化的波，反射波将是纯TE极化。这个性质有非常实用的工程应用：

  • 偏振太阳镜：利用布儒斯特角反射，滤除水面或路面反射的眩光（自然光中的水平极化分量）。
  • 天线罩设计：通过使天线罩壁的入射角接近布儒斯特角，可以最大限度地减少反射损耗。
  • 激光器中的布鲁斯特窗：使激光在通过窗口时TM分量无损耗通过。

全反射与临界角

当电磁波从光密介质（n₁大）入射到光疏介质（n₂小）时，如果入射角大于某个临界角（critical angle），那么透射波将不复存在——能量全部反射：

sin θc = n₂ / n₁

这个现象称为全内反射（Total Internal Reflection, TIR）。虽然全反射在光学中更常见（光纤通信就是基于这个原理），但在微波领域也有重要应用：

  • 介质谐振器天线：利用高介电常数材料内部的多次全反射形成谐振模式。
  • 地表波传播：当电磁波以掠入射角从大气射向地面时，在大地表面附近形成表面波（Zenneck波）。
故事时间：1999年，NASA的火星极地着陆者号（Mars Polar Lander）在进入火星大气层时与地面失去了联系。事后调查发现，一个可能的原因是飞船的下行链路天线使用了线极化，而大气层中的尘埃和冰晶引起的散射改变了信号极化状态，导致地面站的圆极化天线接收到了极度失配的信号。虽然最终原因没有定论，但这个案例深刻说明了：在复杂的传播环境中，极化效应不是一个微不足道的细节——它可能直接决定任务成败。之后的火星探测器（如好奇号、毅力号）全部使用圆极化天线。

下面我们用Python计算Fresnel反射系数并找出布儒斯特角：

import numpy as np

def fresnel_reflection(theta_i_deg, eps_r1=1, eps_r2=4):
    """
    计算两种介质分界面的Fresnel反射系数
    参数:
        theta_i_deg : 入射角 (度)
        eps_r1      : 介质1的相对介电常数 (默认: 空气)
        eps_r2      : 介质2的相对介电常数 (默认: εr=4, 如玻璃/塑料)
    返回:
        Gamma_TE, Gamma_TM : TE和TM极化的反射系数
    """
    theta_i = np.deg2rad(theta_i_deg)
    n1 = np.sqrt(eps_r1)
    n2 = np.sqrt(eps_r2)

    # Snell定律求透射角
    sin_theta_t = n1/n2 * np.sin(theta_i)
    # 若入射角大于临界角，sin_theta_t > 1 → 全反射
    if sin_theta_t > 1:
        return 1+0j, 1+0j  # 全反射

    theta_t = np.arcsin(sin_theta_t)

    # Fresnel反射系数（假设 μr1 = μr2 = 1）
    Gamma_TE = (n1*np.cos(theta_i) - n2*np.cos(theta_t)) / \
               (n1*np.cos(theta_i) + n2*np.cos(theta_t))
    Gamma_TM = (n2*np.cos(theta_i) - n1*np.cos(theta_t)) / \
               (n2*np.cos(theta_i) + n1*np.cos(theta_t))

    return Gamma_TE, Gamma_TM

# 计算并打印不同入射角下的反射系数
eps2 = 4  # 例如: PCB介质 (FR4 约 4.4)
theta_range = np.arange(0, 90, 5)
print("θ_i(°) | Γ_TE      | Γ_TM      | |Γ_TE|² | |Γ_TM|²")
print("-" * 55)
for th in theta_range:
    gte, gtm = fresnel_reflection(th, eps_r2=eps2)
    print(f"{th:6d} | {gte.real:+.3f}{gte.imag:+.3f}j | "
          f"{gtm.real:+.3f}{gtm.imag:+.3f}j | "
          f"{abs(gte)**2:.4f} | {abs(gtm)**2:.4f}")

# 寻找布儒斯特角 (TM反射系数为零)
for th in np.arange(0, 90, 0.1):
    _, gtm = fresnel_reflection(th, eps_r2=eps2)
    if abs(gtm) < 0.001:
        print(f"\n布儒斯特角 ≈ {th:.1f}° "
              f"(理论值: {np.rad2deg(np.arctan(np.sqrt(eps2))):.1f}°)")
        break

本章我们学习了电磁波的极化——这个看似抽象的概念在实际系统中至关重要。从GPS的圆极化选择到WiFi信号的极化失配，从手机握持姿态导致的信号变化到反射引起的极化改变，极化贯穿了天线设计与电波传播的方方面面。我们还探讨了电磁波在介质分界面上的反射与透射行为——边界条件、Snell定律、Fresnel公式、布儒斯特角和全反射——这些知识是理解雷达散射、天线罩设计、电磁兼容以及后续第12章"地面反射传播"的基础。
