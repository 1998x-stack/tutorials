第19章 不等概率抽样

大家好，我是你们的讲师。到目前为止，我们讨论的所有概率抽样方法——简单随机抽样、分层抽样、系统抽样、整群抽样——都有一个隐含假设：总体中的每个单元入选样本的概率是相等的。但现实世界中有太多场景，等概率假设既不现实也不高效。

想象一下你要对全国企业进行一次经济调查。如果采用简单随机抽样，那么一个小杂货店和一个年营收百亿的集团公司被抽中的概率完全相同。你抽到的样本里大概率塞满了小微企业——它们数量多但贡献的经济总量很小，而真正决定经济走向的头部企业很可能一个都没抽到。这样的样本能用来估计全国工业总产值吗？显然不行。这就是不等概率抽样（Unequal Probability Sampling）要解决的核心问题：让入选概率与单元的大小（或某个辅助变量）成正比，使得样本在关键指标上更具代表性。

直觉理解：不等概率抽样的哲学是——"重要的单元应该更有可能被选入样本"。这与我们的日常经验一致：审计师审查大额交易更仔细，医生对高危病人做更频繁的检查，交通部门在繁忙路口设置更多的监控。在抽样中，我们给大单位更高的入选概率，然后通过加权来获得无偏的总体估计。

πPS抽样（Probability Proportional to Size）

πPS抽样的核心思想非常直接：总体中第i个单元的入选概率πi与它的规模度量（size measure）xi成正比。这里的"规模"可以是企业员工数、学校的学生数、医院的床位数、城市的人口数等。

假设总体有N个单元，每个单元的规模为xi，总规模为X = Σxi。如果我们抽取n个样本，在不放回πPS抽样中，每个单元的入选概率为：

πPS抽样——入选概率

        第i个单元的入选概率 π_i = n × x_i / X

        其中：
        x_i = 第i个单元的规模度量（如员工数）
        X = Σx_i，所有单元的规模之和
        n = 目标样本量

        示例：一个企业有5000名员工，该行业总员工数为500000
        如果n=100，则π_i = 100 × 5000 / 500000 = 1.0
        这家企业几乎肯定会被抽中（π_i = 1 意味着必入样）

注意一个重要的边界条件：当某个单元的规模大到使π_i>1时，该单元应被必选入样（certainty selection），然后从总体中移除，剩余单元重新计算入选概率。

PPS系统抽样（Systematic PPS Sampling）

PPS在实际中最常用的实现方式是累积规模法（Cumulative Size Method），它本质上是将系统抽样的思想推广到不等概率场景。

操作步骤非常直观：

  1. Step 1：将总体中的所有单元按某种顺序排列（通常按规模排序或按地理区域排序）
  2. Step 2：计算每个单元的累积规模
  3. Step 3：在区间[0, X/n]内随机生成一个起点r（其中X是总规模，n是样本量）
  4. Step 4：选取累积规模达到 r, r+X/n, r+2X/n, ..., r+(n-1)X/n 的那些单元
实际案例：我参与过的一个全国工业企业调查，就是用PPS系统抽样。我们把全国规上工业企业按员工数排序，用累积规模法每50万人抽取一家企业。结果年营收超过100亿的超大型企业几乎全被抽中，中型企业按概率部分入样，而小微企业如果所在规模段足够密集也有可能入选。最终样本虽然只有2000家企业，但覆盖了全国工业总产值的约40%。如果用等概率抽样，要达到同样的覆盖率估计需要上万样本。
方法 入选概率 是否放回 适用场景
简单随机抽样 等概率 放回/不放回 同质性总体
πPS抽样 与规模成正比 不放回 单元规模差异大
PPS系统抽样 近似与规模成正比 不放回 大规模名录，高效实施
有放回PPS 与规模成正比 放回 简单估计，但样本可能有重复

Hansen-Hurwitz估计（有放回PPS）

当你在不等概率抽样中使用有放回（with replacement）抽样时，Hansen和Hurwitz在1943年提出了一个非常优雅的估计量。它的核心思想是：每个被抽中单元的观测值除以其被选中概率，然后用这些"加权值"的平均来估计总体总量。

# Python：Hansen-Hurwitz估计示例
        import numpy as np

        # 假设我们从6家企业中抽取n=3个样本（有放回，PPS）
        # 企业员工数（规模度量）
        sizes = np.array([200, 50, 5000, 300, 800, 150])
        total_size = sizes.sum()  # = 6500

        # 每个单元每次抽取的入选概率
        p_i = sizes / total_size

        # 抽取样本（模拟）
        np.random.seed(42)
        sample_idx = np.random.choice(6, size=3, replace=True, p=p_i)
        # 假设我们调查了这些企业的产值
        y_i = np.array([1500, 8500, 2200])  # 万元

        # Hansen-Hurwitz 总量估计
        N = 6   # 总体单元数
        n = 3   # 样本量
        # 估计总体总量：τ_hat = (1/n) * Σ(y_i / p_i)
        tau_hat = (1/n) * sum(y_i / p_i[sample_idx])

        print(f"估计总体总量: {tau_hat:.0f} 万元")
        # 注意：方差估计也有相应的公式，此处省略

Horvitz-Thompson估计（一般不等概率）

如果说有放回PPS中的Hansen-Hurwitz估计很重要，那Horvitz-Thompson（HT）估计在抽样理论中的地位则更加核心。HT估计器适用于任何不放回的不等概率抽样设计，而且被证明是总体总体的无偏估计，只要所有包含概率（inclusion probability）均为正数。

HT估计的公式惊人的简洁：

Horvitz-Thompson估计

        总体总量 τ 的HT估计：
        τ_hat_HT = Σ(y_i / π_i)     (对所有样本中的i求和)

        其中 π_i 是第i个单元的包含概率（入选概率）

        这本质上是一种"倒概率加权"（Inverse Probability Weighting）：
        - 如果π_i=0.5，意味着这个单元代表2个总体单元，权重w_i=2
        - 如果π_i=0.1，意味着这个单元代表10个总体单元，权重w_i=10
        - 如果π_i=1.0，意味着这个单元只代表自己，权重w_i=1

        HT估计的方差（严格形式）：
        Var(τ_hat_HT) = ΣΣ (π_ij - π_i·π_j) · (y_i/π_i) · (y_j/π_j)
        其中 π_ij 是第i和第j个单元同时被选入样本的联合包含概率
HT估计的哲学意义：Horvitz-Thompson估计可以说是抽样理论中最重要的一个公式。它的美妙之处在于——无论你的抽样设计多么复杂，只要你知道每个单元的包含概率π_i，你就可以通过简单地用1/π_i加权来得到总体总量的无偏估计。这个"设计加权"（design weight）的思想贯穿了整个抽样理论，也直接衔接了我们在第10-11章讲过的抽样权重。有了HT估计，你就可以自如地处理任意的不等概率设计了。

实际故事：企业调查中的PPS应用

让我分享一个我在实际项目中踩过的坑。某次做区域经济调查，我们需要对全市企业进行抽样。我们的注册名录上有大约10万家企业。当时我采用了简单随机抽样，抽取了1000家。结果你猜怎么着？样本里95%以上是员工不足10人的小微企业，年营收在100万以上的只有不到20家。当我们用简单平均估算全市企业平均营收时，结果严重偏低——因为我们几乎没抽到那些真正创造大部分产值的大企业。

后来改用了PPS抽样，按员工数为规模度量。同一笔预算、同样的1000个样本，但这次样本涵盖了全市半数以上的大型企业。用HT估计重新计算后，全市企业平均营收的估计值比之前高了将近3倍——更接近官方统计的真实值。仅此一项设计改进，就让估计的偏差降低了约60%。

不等概率抽样中的三个常见陷阱：

1. 规模度量的质量：PPS的效果高度依赖规模度量的准确性。如果你用5年前的企业员工数来度量当前规模，那些已经大幅扩张或收缩的企业就会产生偏差。一定要使用最新的辅助数据。

2. 极端大单元：当某个单元的规模远大于其他所有单元时（比如一个行业巨头占了全行业50%的规模），它会以π≈1的概率被抽中。这不是问题，但如果该单元的响应很低或数据质量差，它对总体估计的影响会非常大。必选单元应作为"确保调查"的对象。

3. 不放回PPS的方差估计：不放回PPS的联合包含概率π_ij计算起来非常复杂（尤其是PPS系统抽样），这使得HT方差估计在实践中难以计算。常用的替代方案是使用"简化方差估计"或"重抽样方法"（如刀切法、Bootstrap）来近似方差。

不等概率抽样的实践总结

把前面所有内容整合起来，可以总结出不等概率抽样的几个核心要点：

  • 什么时候用：当总体单元的规模差异很大（如企业调查、医院调查、学校调查），且你手头有可靠的规模度量时
  • 收益：在同样样本量下，PPS可以大幅提高效率（尤其是对总量和均值的估计），因为它自动为"大单元"分配了更高的代表性
  • 代价：设计更复杂、方差估计更困难、对辅助数据的质量要求更高
  • 核心工具：HT估计器是通用的无偏估计框架；PPS系统抽样是最常用的实现方式
估计量 适用设计 无偏性 方差估计难度
Hansen-Hurwitz 有放回PPS 无偏 简单
Horvitz-Thompson 一般不放回不等概率 无偏 较复杂（需π_ij）
比率估计 需辅助信息 近似无偏（大样本） 中等

本章小结

不等概率抽样是抽样理论中一个重要的拓展——它承认"有些单元比别的单元更重要"，并利用这个信息来优化设计。我们从πPS抽样和PPS系统抽样出发，介绍了Hansen-Hurwitz估计和Horvitz-Thompson估计这两个核心工具。HT估计器的简洁与普适性使其成为抽样理论中最优雅的成果之一：只要知道入选概率，加权即可无偏。但要注意，不等概率抽样的实施成本和对辅助数据质量的要求都不低，需要在实践中权衡利弊。

不等概率抽华为我们解决了很多实际问题，但还有一个更棘手的问题：当某些子总体的样本量非常小——比如一个县只有几十个样本——即使使用了最佳设计，直接估计也不可靠。这就是下一章的主题：小域估计（Small Area Estimation），它旨在用间接但聪明的方法来解决"样本量不足"的困境。
