第7章 风险与收益

大家好！恭喜你走到了第7章——我们终于来到了投资学最核心、最基础、最重要的概念：风险与收益。

你有没有在基金销售页面看到过这样的宣传语？"近一年收益率35%！"旁边用蚂蚁一样的字体写着"过往业绩不预示未来表现"。你有没有想过：为什么收益这么高的基金，第二年却能亏40%？你有没有想过——我们常说的"收益"到底应该怎么算才算正确？

这一章我们将回答所有这些问题。更重要的是：我们要建立一套量化收益和风险的语言。没有这套语言，你就无法理解后面所有的投资模型——CAPM、APT、有效市场假说、现代投资组合理论。所以请静下心来，这一章的知识将成为你后续所有分析的基石。

一个真实的故事：日本"渡边太太"的教训

2001-2005年间，日本出现了一个特殊的投资群体——"渡边太太"（Mrs. Watanabe）。这些日本家庭主妇发现日本银行的存款利率接近0%，而澳大利亚和新西兰的利率高达5-7%。于是她们借入日元（利率几乎为零），兑换成澳元和纽元，存入高息账户——这就是著名的套息交易（Carry Trade）。

最初几年，一切顺利。渡边太太们每年轻松获得5%以上的"收益"。很多人甚至辞掉了工作，全职做套利。她们看到的是每年稳定的5%，她们没有看到的风险是什么？

  • 汇率风险：日元兑澳元可能升值，汇兑损失吞噬利息收益
  • 利率风险：如果日本央行突然加息，套息的基础（低利率日元）就崩塌了
  • 尾部风险：2008年金融危机期间，全球风险资产被抛售，日元作为避险货币暴涨。渡边太太们不但利息没了，本金还因为汇率升值损失了30-50%。
渡边太太给我们的教训：不要只看收益，要问"收益从哪里来？"高收益的背后一定有某种形式的风险暴露。如果你看不出风险在哪里，那风险就是你自己——因为你可能在不了解的情况下承担了你不愿意承担的风险。

7.1 收益度量

持有期收益率（HPR, Holding Period Return）

最基本的概念：你持有一只股票一段时间，一共赚了（或亏了）百分之几。

# 持有期收益率计算
# 你买入某股票，价格100元，持有1年后卖出，价格120元
# 持有期间获得了2元股息

HPR = (卖出价 - 买入价 + 股息) / 买入价
HPR = (120 - 100 + 2) / 100 = 22%

年化收益率

如果你持有期不是整1年怎么办？这里有一个关键的区别：

  • 算术平均收益率：把每年的收益率简单相加后求平均。优点是计算简单，缺点是不够准确——它无法反映"复利"的实际情况。
  • 几何平均收益率：把每年的收益率连乘后开n次方。这是真正的年化复合收益率，也是你在基金宣传中看到的最常用指标。
# 算术平均 vs 几何平均 — 差异巨大！
# 某基金连续3年的收益率：
# 第1年：+100%（翻倍）
# 第2年：-50%（腰斩）
# 第3年：+50%

# 算术平均 = (100% - 50% + 50%) / 3 = 33.3% ❌
# 看起来很不错！

# 几何平均（实际年化）：
# 收益因子：2.0, 0.5, 1.5
# 总收益率 = 2.0 × 0.5 × 1.5 - 1 = 50%
# 年化收益率 = (2.0 × 0.5 × 1.5)^(1/3) - 1 = 14.5% ✓
# 这才是真实年化

# 惊人结论：即使算术平均很高，如果你的波动率太大，
# 几何平均（你的真实回报）会远低于算术平均。
# 这叫做"波动率拖累"（Volatility Drag）。
波动率拖累的实战意义：如果你第一年赚50%、第二年亏30%，你的年化收益率只有2.5%——而不是10%的算术平均！这也是为什么"控制回撤"比"追求高收益"更重要的原因。一只每年稳定赚15%的基金，长期下来远胜于"一年翻倍、一年腰斩"的基金。

7.2 风险度量

在投资学中，"风险"不是"损失的可能性"这么模糊的概念，而是有精确定义的数学量。

方差与标准差

最经典的风险度量：波动率。你计算出资产收益率的"离散程度"——波动越大，风险越大。

指标公式解释含义
方差（Variance）收益率与均值之差的平方的平均值衡量收益率偏离均值的程度
标准差（Std Dev）方差的平方根与收益率单位一致，更直观
下行风险（Downside Risk）只计算亏损时的波动投资者更关心下跌，而非上涨
最大回撤（Max Drawdown）从最高点到最低点的最大跌幅你实际可能承受的最大亏损

最大回撤是普通投资者最应该关注的指标——因为它直接决定了你是否会在市场底部恐慌卖出。如果一个产品历史最大回撤30%，在发生30%回撤时你还能保持冷静吗？如果你不能，那你就不应该买这个产品。

7.3 无风险利率

在投资学理论中，"无风险利率"（Risk-Free Rate, Rf）是所有定价模型的起点。它代表了你不承担任何风险就能获得的收益——理论上是你进行任何投资的机会成本。

在实践中，最常用的无风险利率代理变量是短期国债收益率：

  • 美国：3个月期国债收益率（T-bill rate）
  • 中国：1年期国债收益率或银行间同业拆借利率（SHIBOR）

但这里有一个争议：国债真的"无风险"吗？在发达国家，国债违约风险确实接近于零。但在新兴市场，国债可能存在违约风险（如2022年的俄罗斯国债违约）。而且，即使国债不违约，它还有通胀风险和汇率风险——如果你是用美元计价的投资者，买中国国债就不是"无风险"的。这在第21章（国际投资）中会进一步讨论。

2025年无风险利率环境

到了2025-2026年，全球无风险利率的格局呈现显著分化：

国家/地区10年期国债收益率（2025年6月）实际利率（扣除CPI）
美国4.2-4.5%约1.5-2.0%
中国1.8-2.0%约1.0-1.2%
日本1.0-1.2%约0.5-0.8%
德国（欧元区）2.5-2.8%约0.5-1.0%

7.4 风险溢价

既然你可以"无风险"地获得Rf的收益，为什么还要去投资股票、债券或其他风险资产？因为——风险溢价。

风险溢价（Risk Premium）是投资风险资产所要求的额外回报，以补偿投资者承担的风险。最核心的概念是股权风险溢价（Equity Risk Premium, ERP）：投资股票相对于无风险国债的超额收益。

历史数据

  • 美国（1926-2025）：标普500相对于国债的年化ERP约为5-6%。也就是说，股票平均每年跑赢国债5-6个百分点。
  • 中国（2005-2025）：沪深300相对于国债的年化ERP约为3-5%。但波动极大——某些年份股票收益高出30%，某些年份低出40%。
  • 全球（1900-2025）：学术研究发现，全球主要发达市场的ERP在3-5%之间。新兴市场的ERP更高（因为风险更大），但数据期较短，统计上不够稳定。
实操指引：如果你在做长期投资规划（比如为退休做准备），使用4-5%的股权风险溢价假设是合理的。但千万不要把过去3年股票的高收益外推到未来——"过去3年年化20%"不意味着未来3年也能这样。投资学中有个概念叫均值回归（Mean Reversion）：高收益之后往往跟着低收益（或亏损），反之亦然。做投资规划时一定要"谦卑"地看待历史收益。

7.5 通胀调整

你今年赚了10%，看起来很美好。但物价也涨了3%——你的实际购买力只增加了7%。这个道理很简单：名义收益（Nominal Return）vs 实际收益（Real Return）。

核心公式来自经济学家Irving Fisher（欧文·费雪）：

# Fisher效应：名义利率 = 实际利率 + 预期通胀率
# 更精确的公式： (1 + 名义利率) = (1 + 实际利率) × (1 + 通胀率)

# 示例：
# 名义收益率 = 8%
# 通胀率 = 3%
# 实际收益率 = (1 + 0.08) / (1 + 0.03) - 1 = 4.85%
# （近似算法：8% - 3% = 5%，但精确计算要用上面的公式）

# 为什么这个区别很重要？
# 假设你投资了30年，年化名义收益率为8%
# 名义终值: 1.08^30 = 10.06倍
# 但如果通胀率是3%，实际终值: (1.08/1.03)^30 = 4.18倍
# 超过一半的"收益"其实只是通胀！
高通胀环境下的投资：2022年美国通胀率一度超过9%，即使名义上你赚了10%，实际收益率几乎为零。在高通胀环境中，股票和房地产通常能保值，而长期债券（固定票息）的实际购买力会被通胀严重侵蚀。这也是为什么2022年美股的"恐慌"并非单纯因为股市下跌，而是因为股债双杀——股票和债券都在跌，投资者无处可逃。

7.6 税收影响

投资者最终拿到手的不是"税前收益"，而是"税后收益"。不同国家的税制差异巨大，直接影响了投资策略的选择。

资本利得税

  • 美国：短期资本利得（持有<1年）按普通所得税率征税（最高37%），长期资本利得（持有>1年）税率优惠（最高20%）。这个制度本质上鼓励长期持有。
  • 中国：个人投资者在A股市场买卖股票的资本利得暂不征收资本利得税（这是中国股市发展的重大税收优惠政策）。但未来政策可能变化，需保持关注。

股息税

  • 中国：持股超过1年，股息收入免税；持股1个月至1年，税率10%；持股不足1个月，税率20%。
  • 美国：合格股息（Qualified Dividend）按长期资本利得税率征税（最高20%），普通股息按普通收入税率（最高37%）。

税收递延账户

中国有个人养老金账户（每年可税前扣除1.2万元），美国有401(k)和IRA账户。这些账户的核心机制是税收递延（Tax Deferral）——投资收益在取出之前不征税，让复利的"雪球"滚得更大。

账户类型存入时取出时适合人群
传统型（税前）免税（抵扣当年税基）按普通收入税率征税当前税率高、退休后税率低的人
Roth型（税后）不抵扣税基本金和收益均免税当前税率低、预期未来税率上升的人
本章要点回顾：
  • 持有期收益率是最基本的收益度量，年化收益率必须使用几何平均而非算术平均
  • 风险可以用方差/标准差（波动率）、下行风险和最大回撤来量化
  • 无风险利率是投资定价的起点，通常以短期国债收益率为代理变量
  • 股权风险溢价（ERP）是股票相对于无风险利率的超额回报，美国约5-6%，中国约3-5%
  • 实际收益 = 名义收益调整通胀后的结果，高通胀环境下固定收益资产的实际回报会严重缩水
  • 税收制度影响投资的税后收益，利用税收递延账户可以显著提升长期复利效果
