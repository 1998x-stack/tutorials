开篇：从降维到发现"潜变量"

大家好，欢迎进入因子分析 (Factor Analysis, FA) 的世界。在过去的几章中，我们深入学习了 PCA——一种通过线性组合构造新变量来"压缩"数据的方法。PCA 做的是数据简化：从 p 个变量中提取 k 个主成分，用它们尽可能好地"代表"原始数据。但这里有一个微妙的问题：这些主成分真的"存在"吗？

因子分析提出了一个完全不同的哲学视角：观测变量之间的相关性，是由少数几个不可观测的"潜在因子"(latent factors) 驱动的。比如说，你为什么会在语文、数学、英语、物理这四门考试中表现一致？因为存在一个"综合学习能力"的潜变量在背后起作用。PCA 会说："我可以把四科成绩压缩成两个主成分。"FA 会说："不，你的四科成绩之所以相关，是因为它们共享一个潜在因子——学习能力。"

这一章，我们将建立因子分析的基本概念框架。先别着急上手做分析——打好概念基础比马上跑代码重要得多。

因子分析的动机

为什么说 PCA 和 FA 的哲学不同？让我们用一个具体例子来说明。

假设你设计了一份包含 10 个题目的心理学量表，测量"工作满意度"。你发现题目 1-3（关于薪酬、福利、晋升）之间的相关性很高，题目 4-7（关于同事关系、上司支持、团队氛围）之间的相关性也很高，但这两组题目之间的相关性较低。这暗示可能存在两个潜在因子："物质满意度"和"人际满意度"。因子分析的目标就是从数据中提取出这些潜变量。

PCA 与 FA 的根本区别在于它们如何处理方差：

维度主成分分析 (PCA)因子分析 (FA)
目标解释总方差解释变量之间的协方差（相关性）
模型无概率模型，只是线性变换有概率模型：X = μ + LF + ε
误差项无显式误差项有显式的特殊因子（误差）ε
关注点预测/重建数据发现潜在结构
可解释性载荷通常不稀疏旋转后通常可解释
典型应用降维、可视化、数据压缩量表开发、心理学、社科研究

这个对比中最重要的区别是：PCA 没有"误差"概念——它假设所有方差都应该被解释（总方差 = 解释方差）；而 FA 假设每个变量由两部分构成——一部分来自公共因子（shared），另一部分是每个变量独有的（unique/error）。FA 只关心"公共部分"。

正交因子模型

因子分析的核心是正交因子模型 (Orthogonal Factor Model)。它的数学形式如下：

正交因子模型：

X = μ + L F + ε

其中：
X: p × 1 的观测随机向量（p 个变量）
μ: p × 1 的均值向量
L: p × m 的因子载荷矩阵（m < p 是因子数）
F: m × 1 的公因子向量（不可观测）
ε: p × 1 的特殊因子向量（不可观测）

假设条件：
1. E(F) = 0, Cov(F) = I（公因子正交且标准化）
2. E(ε) = 0, Cov(ε) = Ψ = diag(ψ₁, ψ₂, ..., ψₚ)（特殊因子互不相关）
3. Cov(F, ε) = 0（公因子与特殊因子不相关）

协方差结构（核心公式）：
Σ = L Lᵀ + Ψ

也就是说，观测变量的方差-协方差矩阵
被分解为两个部分：
- 公共部分：L Lᵀ（由 m 个公因子解释）
- 唯一部分：Ψ（各变量独有的方差）

这个模型乍看有点抽象，但我们可以把它想象成一个"信号 + 噪声"的分解：每个观测变量 Xᵢ 的变异可以分解为"被公因子解释的部分"（信号）和"该变量独有的部分"（噪声）。公因子 F 是潜变量——它们看不见摸不着，但通过影响观测变量之间的相关性而被间接发现。

理解 FA 模型的关键：FA 是在问"如果去掉每个变量独有的那部分变异，剩下的共同变异可以用几个维度来描述？"这与 PCA 的问题"如何用几个线性组合的变量来最好地重构原始数据？"有本质的区别。

公因子与特殊因子

理解因子分析的关键在于区分两个核心概念：共同度 (communality) 和 特殊方差 (uniqueness/specific variance)。

对于第 i 个变量 Xᵢ，在正交因子模型中：

Var(Xᵢ) = (lᵢ₁² + lᵢ₂² + ... + lᵢₘ²) + ψᵢ
         = hᵢ²                   + ψᵢ
         = 共同度 (communality)   + 特殊方差 (uniqueness)

其中：
hᵢ² = Σⱼ₌₁ᵐ lᵢⱼ² —— 共同度，表示变量 Xᵢ 的方差中被公因子解释的部分
ψᵢ —— 特殊方差，表示变量 Xᵢ 的方差中不能被公因子解释的部分

共同度是一个 0 到 1 之间的值（如果数据标准化了）。共同度高（比如 0.8）说明该变量与公因子的关系紧密——它的变异主要来自公因子；共同度低（比如 0.2）说明该变量大部分是"噪音"——公因子对它没什么解释力，它主要由独特因素决定。

在实际分析中，共同度可以帮助我们发现"问题变量"：如果一个变量的共同度非常低（< 0.3），你可能需要思考——这个变量是否与整体概念不相关？是否测量误差太大？是否应该考虑删除它？

共同度 h²含义行动建议
≥ 0.7公因子对该变量解释力强保留，该变量是良好的指标
0.4 - 0.7中等解释力保留，但需关注
0.2 - 0.4解释力较弱考虑是否删除
< 0.2几乎无解释力强烈建议删除

方差分解

上面的 Var(Xᵢ) 分解是针对单个变量的。现在我们来看整体的方差分解。

在标准化数据下，全部 p 个变量的总方差 = p（因为每个变量方差为 1）。FA 将这个总方差分解为：

总方差 = Σᵢ₌₁ᵖ hᵢ² + Σᵢ₌₁ᵖ ψᵢ
       = 总共同度 + 总特殊方差

更常用的是看每个公因子解释了多大比例的总方差：

第 j 个公因子解释的总方差比例 = (Σᵢ₌₁ᵖ lᵢⱼ²) / p

其中 Σᵢ₌₁ᵖ lᵢⱼ² 是第 j 列载荷的平方和，
也就是该因子贡献的总方差（在正交模型下等于特征值）。

这个分解与 PCA 的方差贡献率非常相似——但有一个关键区别：PCA 的"总方差"是完整的总方差（含独特方差），而 FA 的"总共同度"排除了特殊方差。所以 FA 的因子解释方差比例通常是"占共同度方差的比例"，而不是"占总方差的比例"。

这也解释了为什么 FA 的结果中，"累计方差解释率"往往比 PCA 低——因为 FA 不承诺解释独特方差，只解释变量之间的共享方差。

因子载荷的性质

因子载荷 (factor loadings) 是连接观测变量与潜在因子的桥梁。在正交因子模型中，载荷 lᵢⱼ 有非常明确的统计含义：

  • lᵢⱼ 是变量 Xᵢ 与因子 Fⱼ 的协方差（当变量标准化后，就是相关系数）
  • lᵢⱼ 的平方表示变量 Xᵢ 的方差中被因子 Fⱼ 解释的比例
  • 载荷矩阵 L 的每一行给出该变量在所有 m 个因子上的载荷，每一列给出该因子在所有 p 个变量上的载荷

载荷矩阵的结构直观地反映了数据的内在结构：

理想的"简单结构"载荷矩阵（标准化数据）：

        因子1（人际）  因子2（物质）  共同度
题目1        0.82         0.12       0.69
题目2        0.76         0.08       0.58
题目3        0.79         0.15       0.65
题目4        0.11         0.85       0.73
题目5        0.06         0.78       0.61
题目6        0.18         0.81       0.69

在这个理想化的载荷矩阵中：
- 每个变量只在少数因子上有高载荷（本例中每个变量只在一个因子上高）
- 每个因子只在部分变量上有高载荷
- 这种结构称为"简单结构"(simple structure)，
  是 Thurstone 在 1940 年代提出的因子分析理想形态

真实的载荷矩阵没有那么干净——变量可能在多个因子上都有中等载荷。这就需要因子旋转来"纯化"载荷模式，我们将在下一章详细讨论。

适用性条件

与 PCA 一样，FA 也不是对所有数据都适用。在开始 FA 之前，你应该检查以下条件：

相关矩阵必须适合因子分析

如果变量之间几乎没有相关性（相关矩阵接近单位矩阵），因子分析就没有意义——因为没有"共同变异"可以被提取。这正是我们之前见过的 Bartlett 球形检验的用途。此外，KMO 检验同样适用于 FA，评判标准也与 PCA 相同。

样本量要求

FA 对样本量有较高要求。一般准则：

样本量评估备注
n ≥ 300良好大多数情况足够
n ≥ 500很好估计稳定
n ≥ 1000极好因子结构可靠
n < 100不推荐估计严重不稳定

除了绝对样本量，还有一个"变量与样本量比例"的经验法则：每个变量至少需要 5-10 个样本。即如果你有 20 个变量，至少需要 100-200 个样本。

变量类型

FA 最适合连续变量。对于二分类或有序分类变量，需要使用专门的变体（如项目响应理论 IRT 或分类因子分析）。如果必须对分类变量用 FA，建议使用基于多分格相关 (polychoric correlation) 的因子分析。

因子数的先验知识

与 PCA 不同，FA 要求你预先指定因子数 m。有时候理论会告诉你（比如某个心理学量表设计时有 3 个维度），有时候你需要通过数据分析（scree 图、平行分析）来估计。我们下一章会讨论具体的因子数选择方法。

FA vs PCA 选择的简易指南：如果你的目标是"压缩数据"（比如把 100 个变量压缩成 20 个新变量用于建模），用 PCA。如果你的目标是"发现潜变量结构"（比如弄清楚是什么潜在维度驱动了消费者的品牌感知），用 FA。简单说——数据压缩选 PCA，结构发现选 FA。

本章小结与衔接

本章我们建立了因子分析的基本概念框架。我们从 PCA 与 FA 的哲学差异出发——PCA 解释总方差，FA 解释协方差——然后学习了正交因子模型 X = μ + LF + ε 的数学形式及其假设条件。接着我们深入讨论了共同度和特殊方差的含义、方差分解的方式，以及因子载荷作为变量与因子之间桥梁的统计性质。最后，我们列出了 FA 的适用性条件——样本量、变量类型、KMO 和 Bartlett 检验。

有了概念基础之后，下一章我们将进入"动手"环节：因子分析的两个关键步骤——因子估计（如何从数据中提取因子）和因子旋转（如何让因子结果更容易解释）。从主成分法到极大似然法，从方差最大旋转到斜交旋转——下一章的内容将让你的 FA 工具箱丰富起来。
