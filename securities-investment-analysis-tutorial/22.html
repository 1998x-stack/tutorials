第22章 行业估值与轮动

大家好，我是你们的讲师。如果你问一个资深的基金经理："选股和选行业，哪个更重要？"他很可能会告诉你——行业配置。大量的研究证明，投资收益的80%以上来自行业和风格的配置决策，而不是个股选择。你花了大半年研究出来的"绝世好股"，如果选错了一个在下跌周期中的行业，照样亏钱。

这一章我们来解决两个核心问题：第一，如何判断一个行业是"贵"还是"便宜"？第二，如何在不同的行业之间做轮动，抓住每个阶段的领涨板块？

22.1 行业估值的相对坐标系

不同行业的PE（市盈率）直接比较是没有意义的。银行业的PE常年只有5-8倍，而半导体行业的PE可以高达50-100倍。这不代表银行比半导体"便宜"——它们的商业模式、成长性、风险结构完全不同。

PB-ROE矩阵是解决这个问题的一个有效工具。这个矩阵把行业放在两个维度上：

  • 横轴：ROE（净资产收益率）——衡量行业的盈利能力
  • 纵轴：PB（市净率）——衡量市场的估值水平

矩阵的四个象限是：

  • 高ROE + 高PB：盈利能力强、市场给了高溢价。典型的是消费、医药中的优质赛道。
  • 高ROE + 低PB：盈利能力强但市场低估的"价值洼地"。典型的是银行、地产（在景气低谷时）。
  • 低ROE + 高PB：盈利能力一般但市场在"讲故事"。典型的是概念炒作期的科技板块。
  • 低ROE + 低PB：盈利能力弱且市场不看好。典型的是周期低谷的钢铁、煤炭等传统行业。

踩坑故事：2021年追涨消费行业高位接盘

2021年春节前，A股的消费板块（尤其是白酒）涨得让人眼红。贵州茅台的PE从2020年初的30多倍一路飙到2021年2月的70多倍。当时市场上流传着一句话——"只要是在消费赛道里，买什么都能赚钱。"

我的朋友小张就是在这时候冲进去的。他原本是做IT的，对消费行业并不了解，但看到身边的朋友都赚钱了，实在忍不住在2021年1月全仓杀入了某二线白酒股，买入时PE已经超过100倍。他当时的理由是："白酒的赛道好，长期持有肯定没问题。"

结果春节后市场风格骤变，消费板块暴跌。那波调整持续了整整一年多，他买入的白酒股从高点跌了超过60%。直到2023年才勉强回本—而同期新能源和AI板块已经翻了好几倍。

教训：再好的行业，买贵了也是一场灾难。消费赛道当然没问题，但100倍PE买入意味着你需要等100年才能靠利润收回投资。行业景气度再高，也要关注估值——这是行业投资中最基本的纪律。

A股目前有54个申万细分行业，它们的估值水平在全球化视角下也值得关注。截至2026年上半年，A股消费板块的PE相对全球主要市场偏高（溢价约20%-30%），而制造业和科技板块的PE则与纳斯达克和日韩市场相近。对于有全球配置视角的投资者来说，这种跨市场的估值对比提供了很有价值的参考坐标。

22.2 景气投资的四个阶段演变

行业轮动的核心驱动力是景气度。所谓景气投资，就是投资于那些景气度正在改善或处于高位的行业。A股市场的景气投资逻辑在过去十年经历了四个阶段的演变：

  • 阶段一（2016-2017）：高ROE定景气。这个阶段的市场主线是"价值回归"。在供给侧改革的推动下，龙头公司的盈利能力大幅改善，市场用ROE来衡量行业景气——ROE越高，越受追捧。这个阶段表现最好的是白酒、家电、保险等"核心资产"。
  • 阶段二（2020-2021）：高增速G定景气。疫情后的流动性宽松催生了成长股行情。市场不再盯着ROE，而是看增速G——谁的净利润增长最快，谁就是"最靓的仔"。新能源、半导体、CXO等高增速板块成为领涨主力。
  • 阶段三（2022-2024）：景气投资遭遇争议。高增速板块在2022年遭遇重创，很多投资者开始质疑"景气投资到底还有没有效"。这个阶段市场的共识被打破，不同的声音开始涌现。
  • 阶段四（2025年起）：△g（增速改善）定景气。市场逐渐形成了新的共识——真正重要的是"增速的变化方向"而不是"增速的绝对值"。一个行业从增速10%提升到20%（△g为正）比一个行业从50%降到30%（△g为负）更值得投资，即使后者的绝对值更高。这个框架更注重边际变化，也更符合均值回复的逻辑。

这四个阶段的演变告诉我们：没有一成不变的景气投资框架。市场对"景气"的定义在持续进化，投资者的认知也必须随之更新。

22.3 五维行业配置框架

2026年，浙商证券提出了一套系统的五维行业配置框架，在实践中表现出了良好的效果。这五个维度分别是：

  1. 分析师Alpha Capture：通过分析师的盈利预测调整方向，捕捉行业内正在发生的景气变化。
  2. 交易拥挤度：衡量行业是否过于"拥挤"，避免追高。
  3. 量价评分：技术面的信号，包括相对强弱、趋势跟随等。
  4. 风格适配度：当前市场风格（大盘/小盘、价值/成长）与该行业的历史适配度。
  5. 中观景气：行业层面的高频数据，如开工率、价格指数、库存周期等。

这个框架的优势在于多信号的综合判断。单一维度的信号可能出错（比如分析师也有看走眼的时候），但五个维度的信号同时指向同一个行业时，胜率就大幅提升了。

22.4 行业轮动策略搭建

有了评估框架，我们来看看如何实际搭建一个行业轮动策略。主流的行业轮动方法有三种：

  • 动量轮动：买入过去一段时间表现最好的行业，卖出表现最差的。理论依据是趋势延续。适合趋势性强的市场环境。
  • 反转轮动：买入过去一段时间表现最差的行业，卖出表现最好的。理论依据是均值回复。适合震荡市。
  • 景气轮动：基于行业的基本面景气度（产能利用率、库存周期、盈利增速等）做配置。理论依据是基本面驱动股价。适合中长期投资。

下面是这三种轮动策略在A股的对比：

策略类型 核心理念 持仓周期 年化超额 最大回撤 胜率 最佳市场环境
动量轮动 强者恒强 1-3个月 约8%-12% 约18% 约55% 趋势行情
反转轮动 跌多了会涨 1-6个月 约5%-8% 约12% 约52% 震荡行情
景气轮动 基本面驱动 6-12个月 约10%-15% 约10% 约60% 经济周期清晰

在实际操作中，多数机构并不是"单选"某种轮动策略，而是把三种方法结合起来。比如用景气轮动确定中长期配置方向，用动量轮动确定短期买卖时机，用反转轮动做风险控制——当某个行业涨得太快时，自动降低其权重。

下面是行业动量轮动的一个简化计算示例：

def industry_momentum_strategy(
    industry_returns,     # DataFrame, 各个行业的日度收益率
    lookback=60,          # 动量回看期（交易日）
    top_n=5,             # 选择前N个行业
    rebalance_freq=20,   # 调仓频率（交易日）
    max_deviation=0.04   # 行业偏离度约束（4%）
):
    """
    行业动量轮动策略
    """
    import numpy as np
    import pandas as pd

    rebalance_dates = industry_returns.index[::rebalance_freq]
    weights_record = []

    for i, date in enumerate(rebalance_dates):
        if i == 0:
            continue

        # 计算每个行业的过去N日累计收益
        lookback_start = rebalance_dates[i-1]
        period_returns = (1 + industry_returns.loc[
            lookback_start:date
        ]).prod() - 1

        # 排名并选择前N个行业
        rank = period_returns.rank(ascending=False)
        selected = rank[rank <= top_n].index

        # 等权配置（做多前N个行业）
        weights = pd.Series(0, index=industry_returns.columns)
        weights[selected] = 1.0 / top_n

        # 行业偏离度约束：单个行业权重不超过基准的4%
        # 假设基准权重为 1/num_industries
        base_weight = 1.0 / len(industry_returns.columns)
        weights = weights.clip(
            lower=base_weight - max_deviation,
            upper=base_weight + max_deviation
        )
        weights = weights / weights.sum()  # 重新归一化

        weights_record.append(weights)

    return pd.DataFrame(weights_record, index=rebalance_dates[1:])

22.5 拥挤度预警：防止踩踏

行业轮动最大的风险不是"选错了方向"，而是"所有人都选了同一个方向"——拥挤度踩踏。

行业拥挤度的监测通常从四个维度进行：

  • 成交占比：该行业的成交额占全市场成交额的比例。当某个行业成交占比创下历史新高时，通常意味着资金已经极度集中。
  • 换手率：行业的整体换手率。换手率过高说明短期交易过热。
  • 超额收益斜率：行业相对大盘的累计超额收益在短期内的上升斜率。斜率过于陡峭往往不可持续。
  • 主动基金超配比例：公募基金相对基准指数超配该行业的比例。当所有基金都在超配时，后续的增量资金就有限了。

当一个行业在这四个维度上都进入了高位区间时，就发出了"拥挤度预警"。在拥挤度高位买入行业，接下来的超额收益大概率是负的。

实用建议：行业轮动策略中，设置一个"拥挤度过滤器"是简单的风控手段——当某个行业的拥挤度综合评分进入历史前10%分位时，无论动量信号多强，都降低该行业的配置权重至基准水平以下。这个简单的规则，可以帮你避开大多数"踩踏式下跌"。

简单来说：拥挤度是行业轮动的"刹车片"，没有刹车片的车是不该上路的。

小结与预习

这一章我们深入到了行业配置的领域。我们学到了：不能用PE直接跨行业比较，PB-ROE矩阵是更好的定位工具；景气投资的逻辑从ROE到G再到△g不断进化；五维行业配置框架提供了系统化的行业评价方法；动量/反转/景气三种轮动策略各有适用场景；最后，拥挤度预警是行业轮动必不可少的风控环节。
