第2章 基本概念

大家好，我是你们的讲师。在正式进入各种抽样方法之前，我们必须先建立一套通用术语体系。想象一下：你做了一次调查，报告里写了"总体均值估计为52.3，抽样误差±2.1，置信度95%"。如果读者不理解"总体均值"和"置信度"的精确含义，这份报告就是一堆没有意义的数字。

更重要的是，这些概念之间有一个逻辑链条：总体决定了抽样框，抽样框决定了样本是否能代表总体；参数是你想知道的真相，统计量是你用样本算出来的近似值；抽样误差是你为省钱省力付出的代价，而非抽样误差则是你犯的"额外错误"。理解这条链条，才能看懂后续所有方法的设计逻辑。

在这一章中，我们还会通过一个真实的历史案例——1948年美国总统大选民调翻车事件——来展示这些概念是如何在实践中起作用的。让我们从最基本的开始。

总体与样本

目标总体与调查总体

目标总体（Target Population）是你真正想研究的对象整体。比如"中国所有18岁以上的成年人"、"某城市所有注册小微企业"、"全国所有三级甲等医院的住院患者"。但问题是，你几乎不可能精准地接触到目标总体中的每一个个体。

因此在实际操作中，我们会退而求其次，定义一个调查总体（Survey Population）——也就是你实际有机会抽取的个体集合。比如通过电话号码簿来联系居民、通过工商注册名录来联系企业。调查总体和目标总体之间的差距，就是涵盖不全（Undercoverage）。

涵盖不全是一个极其隐蔽但又极其危险的偏差来源。想想看：如果你通过固定电话进行民意调查，那些只有手机没有座机的年轻人就被排除在外了；如果你通过微信推送问卷，不上微信的老年群体就被排除在外了。这种偏差不会随着样本量的增大而消失——事实上，样本越大，你只是把错误的结果算得更精确而已。

抽样框

抽样框（Sampling Frame）是你实际用于抽取样本的名单或系统。它可以是名录（如企业注册列表、学生花名册）、地图区域（如行政区划图）、或电话号码数据库。

一个好的抽样框需要满足三个条件：完整（覆盖所有调查总体中的个体）、准确（信息不过时、无重复）、可访问（每个个体有明确的联系方式）。但现实中，完美的抽样框几乎不存在。常见的问题包括：

  • 涵盖不全：抽样框漏掉了一部分总体。例如，用QQ用户列表调查中国网民（漏掉了不用QQ的网民）。
  • 涵盖过全：抽样框包含了不属于总体的个体。例如，企业名录中包含已经倒闭的企业。
  • 重叠：同一个个体在抽样框中出现多次。这在多级抽样框中尤为常见。
  • 聚类：抽样框中的个体不是独立存在的，而是嵌套在更高层单元中。例如学生嵌套在班级中、居民嵌套在社区中。

参数与统计量

在统计学中，我们严格区分两组符号：一组描述总体，一组描述样本。

概念 总体参数 样本统计量
均值 μ = (1/N) Σyᵢ x̄ = (1/n) Σyᵢ
方差 σ² = (1/N) Σ(yᵢ - μ)² s² = (1/(n-1)) Σ(yᵢ - x̄)²
比例 π p̂ = x/n
标准差 σ s
总量 τ = N × μ τ̂ = N × x̄

你可能会问：为什么样本方差的分母是 n-1 而不是 n？这是因为样本方差用 x̄ 代替了 μ，而 x̄ 本身就是从样本中估计出来的，这"消耗"了一个自由度（Degree of Freedom）。用 n-1 做分母可以保证 E(s²) = σ²，也就是说 s² 是 σ² 的无偏估计（Unbiased Estimator）。

无偏性是什么含义？ 无偏不是说你每一次抽样的估计值都恰好等于总体参数，而是说如果你重复抽样无数次（每次都独立抽取新的样本），这些估计值的平均值会等于总体参数。你可以把它理解为一个射击选手——他打出的子弹可能偶尔偏离靶心，但长期平均落在靶心上。有偏的估计就像准星歪了的枪，弹着点虽然密集但始终偏向一边。

在抽样理论中，我们关注三个最重要的估计量性质：

  • 无偏性（Unbiasedness）：E(θ̂) = θ。长期平均下来，估计值不偏不倚。
  • 一致性（Consistency）：随着样本量 n 增大，θ̂ 越来越接近 θ。大样本下估计是可靠的。
  • 有效性（Efficiency）：在所有无偏估计中，方差最小。这意味着估计的波动最小。

抽样误差与非抽样误差

这是抽样理论中最重要的划分之一。抽样的总误差（Total Error）可以分解为两个部分：

总误差 = 抽样误差 + 非抽样误差

抽样误差（Sampling Error）

抽样误差是因为只调查了一部分个体而非全体而产生的误差。即使你执行了完美的概率抽样，只要你没有调查每一个人，就必然存在抽样误差。抽样误差的大小取决于：

  • 样本量：样本量越大，抽样误差越小
  • 总体变异程度：总体内个体差异越大，抽样误差越大
  • 抽样设计：不同的抽样方法（简单随机、分层、整群等）影响抽样误差的大小

好消息是：抽样误差是可以计算和控制的。因为它来自随机化过程，我们可以用概率论的工具来量化它。

非抽样误差（Non-sampling Error）

非抽样误差是除抽样之外所有其他原因导致的误差。它不会随着样本量的增大而自动减小——在某些情况下甚至会增大。常见类型包括：

  • 无应答误差（Nonresponse Error）：被抽中的人不愿意参与。
  • 测量误差（Measurement Error）：问卷设计不当、访员诱导、受访者记忆偏差等。
  • 处理误差（Processing Error）：数据录入、编码、清理过程中的错误。
  • 覆盖误差（Coverage Error）：抽样框不完整导致的偏差。

区分这两种误差至关重要。统计学家用精度（Precision）来描述抽样误差的大小，用偏差（Bias）来描述非抽样误差的大小。而总的误差可以用均方误差（Mean Square Error, MSE）来度量：

MSE = Bias² + Variance

这个公式揭示了一个重要的权衡：有时候稍微引入一点偏差，可以大幅降低方差，从而降低总误差。这就是为什么有些抽样方法（如比率估计、回归估计）虽然是有偏的，但依然被广泛使用——它们在偏差和方差之间找到了更好的平衡。

关键洞察：很多人以为抽样调查的误差主要来自"抽样的随机性"，但实际上对于大多数大规模调查而言，非抽样误差才是总误差的主要来源。一次精心设计的抽样（比如用SRS抽了1000人），其抽样误差通常只有±3%左右；但如果没有应答率只有20%，而且无应答者的特征与应答者系统性地不同，那偏差可能高达±10%甚至更多。这就是为什么好的调查不仅关注抽样设计，更关注数据质量控制。

置信度与置信区间

当你看到"某调查结果显示，52%的受访者支持政策A，抽样误差为±3%，置信度为95%"这样的表述时，你能准确理解它的含义吗？

很多人的直觉理解是："总体支持率有95%的概率落在49%~55%之间。"——这是错的。

正确的理解是：如果重复进行同样的调查100次（每次独立抽取新的样本），那么其中大约95次的置信区间会包含真实的总体支持率。换句话说，置信度（Confidence Level）描述的是方法的可靠性，而不是某个特定区间的概率。

对于均值的置信区间，其基本公式为：

CI = x̄ ± zα/2 × SE(x̄)

其中 SE(x̄) = s / √n 是样本均值的标准误（Standard Error），zα/2 是标准正态分布的临界值（95%置信度下取1.96，近似为2）。

这意味着：95%置信区间的宽度大约是 4 倍的标准误。标准误每缩小一半，需要把样本量扩大为原来的4倍——这就是抽样中的"边际效用递减"规律。

import numpy as np

# 模拟一个场景：总体均值μ=50，总体标准差σ=15
np.random.seed(42)
population = np.random.normal(50, 15, size=100000)
true_mean = population.mean()

def confidence_interval(sample, conf_level=0.95):
    """计算样本均值的置信区间"""
    n = len(sample)
    x_bar = sample.mean()
    se = sample.std(ddof=1) / np.sqrt(n)
    z = 1.96  # 95%置信度
    return x_bar - z * se, x_bar + z * se

# 重复抽样1000次，看看有多少CI包含了真值
coverage_count = 0
n_simulations = 1000
for i in range(n_simulations):
    sample = np.random.choice(population, size=100, replace=False)
    ci_low, ci_high = confidence_interval(sample)
    if ci_low <= true_mean <= ci_high:
        coverage_count += 1

print(f"真实均值: {true_mean:.2f}")
print(f"覆盖率: {coverage_count/n_simulations*100:.1f}%")
# 输出应接近95% —— 这就是置信度的含义
常见的置信区间误解：很多人看到"95%置信区间为[48.2, 52.8]"，就解读为"总体均值有95%的概率落在这个区间内"。但从频率学派的观点看，总体均值是一个固定值而非随机变量，它要么在区间内，要么不在——不存在概率的说法。正确的频率学派解读是：这个区间是由一个"在95%的情况下能覆盖真值"的方法计算出来的。当然，在贝叶斯统计中确实有"可信区间"（Credible Interval）的概念可以做出概率解释，但那是不一样的框架。在日常沟通中，你可以把"95%置信度"理解为"我们对这个区间包含真值有相当强的信心"。

案例回顾：1948年美国总统大选

现在让我们用刚学的概念来分析一个经典案例。1948年，美国总统大选在民主党候选人、时任总统哈里·杜鲁门和共和党候选人托马斯·杜威之间展开。选前几乎所有民调都预测杜威将轻松获胜。著名的《芝加哥论坛报》甚至在选举结果出来前就印刷了"杜威击败杜鲁门"的号外头版——这张照片至今仍是新闻史上的经典翻车案例。

当时最具影响力的民调机构是乔治·盖洛普（George Gallup）创立的美国民意研究所。盖洛普使用了配额抽样（Quota Sampling）方法：调查员按照年龄、性别、收入、地域等人口特征设定配额，然后在配额范围内自由选择受访者。

表面上看，这似乎很科学——样本在人口学特征上"看起来"和总体一致。但问题出在哪里？我们来拆解：

  • 抽样框偏差：调查主要通过面对面访问进行，但调查员倾向于选择那些看起来"友善、有知识、住在方便到达的区域"的受访者。这导致样本中中产阶级和共和党支持者的比例偏高。
  • 无应答偏差：在1948年，民主党的支持者（尤其是低收入群体和农村居民）更不愿意接受调查或不太容易被调查员接触到。
  • 事后调整失败：盖洛普试图用配额来纠正偏差，但配额变量（年龄、性别等）不足以完全消除政治倾向上的选择性偏差——因为即使在同一性别和年龄组内，是否愿意接受调查与政治倾向仍然相关。

结果：盖洛普预测杜威获得49.5%的选票、杜鲁门44.5%；实际结果是杜鲁门49.6%、杜威45.1%。偏差高达近10个百分点。而1948年之后，民调行业进行了深刻反思，逐步从配额抽样转向了概率抽样，成为现代民意调查的基石。

从1948年学到的教训：这个案例完美地展示了偏差（Bias）与方差（Variance）的区别。盖洛普的配额抽样如果重复多次，结果可能相当一致（低方差）——但系统性地偏离了真实值（高偏差）。用均方误差的语言来说：一个高偏差、低方差的估计器，远不如一个无偏但稍有波动的估计器，因为偏差不会随着样本量的增大而消失。概率抽样的核心价值恰恰在于消除了选择偏差，让随机化成为"客观性"的数学保证。
