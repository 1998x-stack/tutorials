1.1 为什么学习电波传播与天线

大家好，我是你们的讲师。在开始我们这段为期28章的旅程之前，我想先问一个问题：你上一次主动思考"信号是怎么到我手机上的"是什么时候？

对大多数人而言，电磁波是隐形的——它无处不在，却从不被看见。每当你在咖啡馆连上WiFi、用导航APP找到目的地、刷短视频流畅得毫无卡顿，你都站在一个看不见的工程奇迹之上：电磁波在空间中精确地传播，而你手中的天线恰好把它抓住了。

从宏大的尺度看：GPS卫星在20,200公里的高空以1.5 GHz的L波段向地面广播信号，传播路径上的自由空间损耗高达180 dB以上，地面接收机却能从噪声中提取出纳瓦级的信号，精准计算出你的位置。从小小的日常看：你的蓝牙耳机在10米范围内稳定传输音频，靠的是一个微型的PCB天线和精心设计的链路预算。

核心认知：天线与电波传播是无线系统的心脏和血管。没有天线，再先进的编码方案、再强大的芯片也只是"有嘴说不出话"——信号永远停留在基带处理器里，无法跃入空中。

但要做好天线并不容易。一个经典的教训来自2010年的iPhone 4 "天线门"（Antennagate）事件。

那一年，苹果发布了iPhone 4，其工业设计为之一新：不锈钢边框被巧妙地用作蜂窝天线的辐射体。然而用户很快发现，当手掌紧握手机左下角的缝隙时——即所谓的"死亡握持（Death Grip）"——信号条会迅速下降甚至断连。苹果最初的回应是"你握错了"，引发舆论哗然。最终，乔布斯不得不在新闻发布会上承认："我们不是完美的，手机也不是完美的。"苹果被迫提供免费的Bumper保护套来缓解问题。

这起事件的本质是什么？是一个天线阻抗匹配被人体耦合改变的问题。人手靠近天线近场区域时，引入了额外的电容和损耗，使天线失谐——驻波比急剧上升，辐射效率断崖式下跌。这个案例告诉我们：天线设计绝不仅仅是理论仿真，还涉及用户环境、人体交互、机械结构的多学科耦合。即使是在全球最富盛名的公司，一个看似微小的天线布局失误也可能导致数亿美元的召回损失和品牌声誉受损。

如果你觉得天线门只是一个消费电子领域的特例，那么请想一想：基站天线的风载计算是否会影响铁塔的安全？卫星天线的热变形是否会改变波束指向？车载雷达天线与保险杠材料的电磁兼容性如何验证？这些问题的背后，都是我们这门课要讨论的核心内容。

1.2 从赫兹到6G：电波传播研究简史

赫兹的火花：电磁波从纸面到现实（1887）

1865年，苏格兰物理学家James Clerk Maxwell用20个方程组统一了电学与磁学，预言了电磁波的存在，并计算出其传播速度恰好等于光速——从而推断出光本身就是电磁波。然而，Maxwell英年早逝（1879年，年仅48岁），生前未能亲眼看到他的理论被实验证实。

1887年，德国卡尔斯鲁厄大学的年轻教授Heinrich Hertz在实验室里完成了改变世界的一个实验：他用一个火花隙振荡器作为发射源，在数米外用一个环形接收器检测到了电磁波。Hertz不仅证实了电磁波的存在，还证明了它具有反射、折射、偏振等波动性质，完全符合Maxwell的预言。

有趣的是，当有人问Hertz电磁波有什么用时，他回答："它毫无用处。这只是一个实验，证明理论是对的。"——Hertz并不知道，他的火花隙实验开启了人类无线通信的大门。为了纪念他，频率的单位被命名为赫兹（Hz）。

知识延伸：Hertz使用的原始发射天线就是最早的半波偶极子（half-wave dipole）的雏形。今天，半波偶极子仍然是天线理论和工程中最基础的参考天线。

马可尼跨大西洋通信：意外的天波（1901）

1895年，年仅21岁的意大利发明家Guglielmo Marconi改进了Hertz的装置，实现了超过2公里的无线电报通信。短短6年后，1901年12月12日，Marconi在加拿大纽芬兰的圣约翰斯收到了从英格兰康沃尔的波尔杜（Poldhu）横跨大西洋传来的摩尔斯电码信号——字母"S"。两地相距约3,500公里！

当时物理学界的共识是：电磁波像光一样沿直线传播，而地球是弯曲的——因此无线电波不可能传播到地平线以外。Marconi的实验结果让整个学术界陷入困惑。直到1920年代，Edward Appleton和他人通过实验发现了电离层（ionosphere）的存在：距离地面约60-1,000公里的高层大气被太阳紫外线电离，形成了能够反射短波无线电信号的带电粒子层。原来，Marconi的信号并不是沿地面弯曲传播的，而是被电离层反射到了大西洋对岸！

正是这个"意外的天波反射"，让人类第一次见识到了电波传播的复杂性——也催生了后来整个短波通信产业和电离层物理学科。

二战雷达：天线技术的加速器（1930s-1945）

第二次世界大战是天线技术发展最快的时期。为了探测敌机，英国、美国和德国竞相开发雷达（RADAR: Radio Detection And Ranging）系统。雷达需要精确控制波束指向、高增益、低副瓣——这对天线设计提出了前所未有的要求。

这一时期诞生了多项划时代的天线发明：

  • 八木-宇田天线（Yagi-Uda antenna）：日本东北大学的八木秀次和宇田新太郎在1926年发明，二战中被广泛用于雷达和通信。一个有源振子加上多个无源引向器和反射器，就能形成高度定向的波束。
  • 喇叭天线（Horn antenna）与抛物面反射器（Parabolic reflector）：用于微波雷达，通过大口径实现高增益和窄波束。
  • 波导缝隙天线阵（Waveguide slot array）：在波导壁上开槽辐射电磁波，是机载雷达的理想选择。

二战后，这些技术迅速转为民用，成为广播电视、微波通信、射电天文学的基础。

卫星通信时代（1957-至今）

1957年，苏联发射了人类第一颗人造卫星Sputnik，其搭载的简单天线向全世界广播了"哔哔"声——这是人类首次从太空向地面发送无线电信号。1962年，美国发射了Telstar，实现了首次跨大西洋电视信号中继。Telstar使用了一个螺旋天线和一个喇叭天线，工作于约6 GHz的上行和约4 GHz的下行频段。

1973年，美国海军发射了第一颗GPS导航卫星，使用L波段（1.2/1.5 GHz）的天线向地面广播精确定位信号。此后，GPS从军用走向民用，彻底改变了交通运输、测绘、物流、户外运动等行业。

今天的卫星通信正在经历第二次革命：Starlink（SpaceX）、OneWeb等低轨卫星星座使用相控阵天线（phased array antenna）实现数千颗卫星与地面终端的动态波束跟踪。每一颗Starlink卫星上装有相控阵天线，而用户终端则是一个"平板天线"——本质上也是一个小型相控阵。

蜂窝通信演进：1G → 5G → 6G

蜂窝通信的历史就是一部天线与传播技术的进化史。我们用一张表来快速回顾：

代际 商用时期 频段 天线技术 数据速率
1G (AMPS/TACS) 1980s 800-900 MHz 单极子/偶极子 模拟语音
2G (GSM) 1990s 900/1800 MHz 面板天线（单极化） 9.6-64 kbps
3G (UMTS) 2000s 2.1 GHz 双极化面板天线 384 kbps - 2 Mbps
4G (LTE) 2010s 700 MHz - 2.6 GHz MIMO (2×2, 4×4) 100 Mbps - 1 Gbps
5G 2020s sub-6 GHz + mmWave Massive MIMO, 波束赋形 1-20 Gbps
6G (研究阶段) 2030s 预计 sub-THz / THz RIS, 超表面, 可重构天线 > 100 Gbps

特别值得一提的是Massive MIMO（大规模多输入多输出）——这是5G基站天线的核心技术。一个5G基站天线阵列可以包含64个或128个天线单元，通过数字波束赋形（digital beamforming）同时对准多个用户。这彻底改变了传统"一根天线覆盖一个扇区"的设计范式。

1.3 天线技术的演进脉络

天线的发展历史可以追溯到Hertz的第一根偶极子，至今经历了130多年的演进。下表梳理了关键的里程碑：

年代 天线类型 关键人物 应用领域
1887 赫兹偶极子 (Hertzian dipole) Heinrich Hertz 实验验证电磁波
1895 加载单极子 (Loaded monopole) Marconi, Popov 无线电报
1926 八木-宇田天线 (Yagi-Uda) 八木秀次, 宇田新太郎 短波通信、雷达
1930s 喇叭天线 (Horn antenna) Southworth, Barrow 微波通信、射电天文
1937 抛物面天线 (Parabolic dish) John Kraus 等人 雷达、卫星通信
1950s 螺旋天线 (Helical antenna) John Kraus 卫星、遥测
1970s 微带贴片天线 (Microstrip patch) Munson, Howell 航天、移动通信
1980s 相控阵 (Phased array) 多国军方 雷达、电子战
1990s 分集与扇区天线 移动通信厂商 蜂窝基站
2010s MIMO 与 Massive MIMO Marzetta (Bell Labs) 4G/5G 基站
2020-2025 超表面/可重构智能表面 (RIS) 全球学术界 6G预研、智能传播环境
2025-2026 ELDAA, Pinching天线, SW-MTS 学术界与产业界 6G候选技术

让我们简要解读几个近期的前沿热点：

  • ELDAA（Extremely Large Distributed Array Antenna，超大规模分布式阵列天线）：将天线单元分布在广阔的地理空间上，通过协作波束赋形实现超高空间复用度和能量效率。
  • Pinching antennas（"捏合"天线）：一种新型的介质波导馈电天线概念，通过改变介质波导的表面结构来"捏出"辐射点，实现低成本的连续可调天线阵列。
  • RIS（Reconfigurable Intelligent Surface，可重构智能表面）：在平面上集成大量可调相位的亚波长单元，主动调控电磁波的反射方向——相当于"智能镜面"，让无线信号绕开障碍物。
  • SW-MTS（Surface Wave-Modulated Travelling-wave Structure，表面波调制行波结构）：利用表面波沿结构的传播和逐渐辐射来实现宽带宽角扫描。
趋势总结：天线技术正在从"被动辐射器"向"主动智能辐射表面"演化。一百多年前Hertz的简单偶极子，正在进化为成千上万个单元协同工作的分布式智能系统。这是一个激动人心的时代。

1.5 必备基础与工具准备

数学基础

本课程涉及的数学以大学理工科二年级水平为基础。以下是必需掌握的内容以及推荐的复习资源：

  • 矢量分析与场论：梯度、散度、旋度、高斯散度定理、斯托克斯定理。推荐复习教材：《高等数学》下卷"矢量分析与场论"章节。
  • 复数运算：极坐标与直角坐标转换，复数的指数形式、乘除、共轭。这是相量表示法的核心。
  • 傅里叶变换：时域与频域的关系，傅里叶变换对的基本性质。天线理论中用于计算远场方向图。
  • 一阶/二阶常微分方程和偏微分方程：用来推导波方程和边值问题。
复习建议：如果你面对矢量微积分或傅里叶变换感觉生疏，不用慌张。每当我们引入新的数学工具时，都会给出一个简短的"数学小贴士"（Math Box）来快速回顾。你只需要对基本概念有印象即可——随着课程推进，你会越来越熟练。

软件工具

本课程使用以下工具进行工程计算和仿真。每个工具我们都会在相应章节中逐步介绍：

  • Python（推荐3.10+）：我们将用Python进行链路预算计算、天线阵列方向图合成、传播模型拟合等。课程附带的代码库将使用numpy和matplotlib。
  • CST Studio Suite 或 HFSS：这是两款业界标准的电磁全波仿真软件。第23章会详细介绍仿真设置。如果暂时没有正版授权，可以考虑以下替代方案：
    • ANSYS HFSS（学生版免费，有单元数限制）
    • FEKO / Altair Feko（学术版可申请）
    • 开源替代：OpenEMS（GPL协议，功能有限但可用于教学）
  • MATLAB（可选）：天线工具箱（Antenna Toolbox）和相控阵工具箱（Phased Array System Toolbox）非常好用——如果你已有许可证的话。

一个简单的Python示例：我们来写一个计算自由空间路径损耗（Free Space Path Loss, FSPL）的函数——这将是全课程中最常用到的公式之一：

import numpy as np

def fspl(d, f):
    """
    计算自由空间路径损耗 (dB)
    参数:
        d : 距离 (m)
        f : 频率 (Hz)
    返回:
        L : 路径损耗 (dB)
    """
    c = 3e8  # 光速 (m/s)
    L = 20 * np.log10(d) + 20 * np.log10(f) + 20 * np.log10(4 * np.pi / c)
    return L

# 示例：5G 毫米波 28 GHz, 距离 100 m
loss = fspl(100, 28e9)
print(f"28 GHz, 100 m 自由空间路径损耗: {loss:.1f} dB")
# 输出: 28 GHz, 100 m 自由空间路径损耗: 101.4 dB
关于dB：如果你对dB（分贝）的概念不太熟悉，我们会在第11章详细讲解。现在你只需要知道，dB是对数单位，每增加3 dB表示功率加倍，每降低3 dB表示功率减半。

推荐参考书

天线与电波传播领域的经典教材非常多。以下四本是我个人最推荐的，你可以根据需要选读：

  • Constantine A. Balanis — "Antenna Theory: Analysis and Design"（第4版/第5版）。天线领域的"圣经"，从理论到工程面面俱到，公式推导严谨，适合随手翻阅。
  • Robert E. Collin — "Antennas and Radiowave Propagation"。数学严谨，特别在电波传播部分写得非常深刻，适合进阶阅读。
  • John D. Kraus & Ronald J. Marhefka — "Antennas for All Applications"。Kraus是螺旋天线的发明者，这本书风格亲切、插图丰富，适合入门。
  • 张瑜 — 《电磁波空间传播》。国内编写的传播类教材，中文亲切，案例接地气，适合与英文教材互补。

建议初学者以Balanis的教材为主线，遇到推导难题时查阅Collin，想快速了解应用时翻看Kraus。而张瑜的《电磁波空间传播》在移动通信信道建模方面有很好的中文讲解。

常见误区提醒：很多初学者刚一上来就扎进公式推导，忽略了物理直觉的培养。我的建议是：先理解物理图像，再消化数学公式。比如，在学习偶极子辐射之前，先想象一下"电荷在导线上来回振动，能量如何从导线表面流向远方"——这个物理画面比任何公式都重要。

在第1章中，我们回顾了电波传播研究和天线技术的发展历程——从Hertz火花隙实验的朴素开端，到Marconi的跨大西洋奇迹，再到今天Massive MIMO和6G超表面天线百花齐放的局面。我们看到天线技术如何从一门实验艺术，逐步演变为建立在Maxwell方程组基础上的精密工程学科。更重要的是，我们见证了人类无线通信能力从零到星球尺度的飞跃。

从下一章开始，我们将回归本源：用严格的数学语言描述电磁波。第2章将带领大家深入Maxwell方程组——一切天线与电波传播理论的基石。我们将从积分形式推导到微分形式，用相量表示法简化时谐场的分析，并推导出关键的波方程。准备好迎接电磁世界的数学之美吧。
