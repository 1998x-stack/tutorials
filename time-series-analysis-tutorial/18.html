第18章 状态空间模型与卡尔曼滤波

大家好！如果你把之前学过的所有时序模型——ARIMA、指数平滑、季节分解——放在一起审视，你会发现一个有趣的共性：每个模型都在做同一件事——将观测数据和隐藏状态分开，然后试图从观测中推断隐藏状态的变化规律。ARIMA 的"隐藏状态"是过去残差的加权和，指数平滑的"隐藏状态"是时变的水平、趋势和季节成分。

状态空间模型（State-Space Model）将这种共性提炼为一个统一的数学框架。而卡尔曼滤波（Kalman Filter）是求解这个框架的最核心的算法——它不仅高效，而且能在每一步给出不确定性的度量。从阿波罗登月舱的导航系统，到今天所有自动驾驶汽车的位置估计，卡尔曼滤波是工程学中最成功的时间序列算法之一。

本章核心目标：理解状态空间模型的"状态-观测"架构，掌握卡尔曼滤波的预测-更新递归流程，认识 ARIMA 和指数平滑作为状态空间模型特例的统一视角。

18.1 状态空间模型的基本架构

状态空间模型由两个方程构成：

状态方程（State Equation）——描述隐藏状态如何随时间演变：

α_t = T·α_{t-1} + R·η_t,    η_t ~ N(0, Q)

观测方程（Observation Equation）——描述观测数据如何由状态生成：

y_t = Z·α_t + ε_t,    ε_t ~ N(0, H)

其中 α_t 是不可观测的状态向量（如时变趋势、季节性因子），y_t 是我们看到的实际数据。T、Z、R 是转移矩阵，Q 和 H 是噪声协方差矩阵。

这个框架的强大之处在于它的灵活性：改变 Z 和 T 的结构，就能表示不同模型：

模型状态向量状态方程含义
局部水平模型α_t = [水平]水平带噪声的随机游走
Holt-Wintersα_t = [水平, 趋势, 季节]水平、趋势、季节的时变演化
ARIMA(p,d,q)α_t = [AR滞后, MA滞后]AR 和 MA 分量的状态表示
动态回归α_t = [β₀, β₁, ..., β_k]回归系数随时间变化

18.2 卡尔曼滤波：预测-更新循环

卡尔曼滤波是状态空间模型的核心推理算法。它递归地执行一个两步循环：

第一步：预测（Predict）

根据上期状态预测本期状态和方差：
α_{t|t-1} = T·α_{t-1|t-1}
P_{t|t-1} = T·P_{t-1|t-1}·T' + R·Q·R'

预测观测值：
ŷ_t = Z·α_{t|t-1}

第二步：更新（Update）

当新观测 y_t 到达后，修正状态估计：
v_t = y_t - ŷ_t             （预测误差/创新）
K_t = P_{t|t-1}·Z'·F^{-1}   （卡尔曼增益）
α_{t|t} = α_{t|t-1} + K_t·v_t
P_{t|t} = (I - K_t·Z)·P_{t|t-1}

这个循环的精妙之处在于：卡尔曼增益 K_t 自动平衡了"模型的信任"和"数据的信任"。当观测噪声大时，K_t 变小，滤波器更相信模型；当状态更新噪声大时，K_t 变大，滤波器更相信新观测。

直观理解：卡尔曼增益就像你的"学习率"——但它不是固定的，而是根据当前的不确定性动态计算出来的。不确定性高的时候步子迈大些（多学新数据），不确定性低的时候步子迈小些（信任模型自身）。

18.3 卡尔曼平滑

卡尔曼滤波是在线算法——只使用到当前时刻的信息。如果我们已经拿到了全部数据（离线场景），可以做卡尔曼平滑（Kalman Smoothing），利用未来信息回望过去的状态估计，得到更精确的结果。

# 使用 statsmodels 的状态空间框架
import statsmodels.api as sm
import pandas as pd

# 假设有一个时间序列 y
# 用局部水平模型（Random Walk + Noise）拟合
model = sm.tsa.UnobservedComponents(y, 'local level')
result = model.fit()

# 滤波状态（只用到当前及之前信息）
filtered = result.filtered_state

# 平滑状态（用到全部信息）
smoothed = result.smoothed_state

# 预测
forecast = result.forecast(12)
print("未来12期预测:", forecast)

18.4 状态空间模型的统一视角

状态空间模型最重要的贡献是提供了统一框架。之前学过的几乎所有经典模型都能表示为状态空间模型的特殊形式：

  • ARIMA(p,d,q) 可以写成状态空间形式，状态向量包含 AR 和 MA 分量。这意味着我们可以用卡尔曼滤波来估计和预测 ARIMA，而不需要用传统的 Box-Jenkins 方法
  • 指数平滑（ETS） 本质上就是一个约束后的状态空间模型，其中的水平、趋势和季节成分就是状态变量
  • 动态线性模型（DLM） 允许回归系数随时间变化，状态空间框架让时变参数的估计变得直接而优雅
# 用状态空间形式表示 ARIMA(1,1,1)
# 这等价于标准 ARIMA，但用卡尔曼滤波估计
arima_ss = sm.tsa.SARIMAX(y, order=(1,1,1))
result_ss = arima_ss.fit()

# 查看状态序列
print("状态变量:", result_ss.filtered_state.shape)

# 进行比较：传统 ARIMA 估计
arima_trad = sm.tsa.ARIMA(y, order=(1,1,1))
result_trad = arima_trad.fit()
print("系数对比:", result_ss.params, result_trad.params)

18.5 真实故事：阿波罗导航与卡尔曼滤波

1960年代，美国阿波罗计划面临一个极其棘手的工程问题：宇航员需要精确知道自己在大空中的位置，但飞船上没有 GPS，唯一的办法是用惯性测量单元（IMU）和星体追踪器获取噪声巨大的观测数据。

当时在 NASA 工作的数学家 Rudolf Kalman 提出了他的滤波算法，核心思想正好解决了这个问题：将飞船的动力学模型（已知物理规律）和传感器观测（噪声数据）巧妙地融合起来，每一步都给出最优的位置估计和不确定度。

1969年，阿波罗 11 号使用卡尔曼滤波在太空中计算轨道，成功将宇航员送上了月球。想象一下：你在距离地球 38 万公里的太空中，面对一堆噪声数据，用笔记本上的矩阵运算就能精确知道自己的位置——这就是卡尔曼滤波的魔力。

今天，从手机 GPS、无人机导航到自动驾驶汽车，每一个需要传感器融合的系统背后都有卡尔曼滤波在默默工作。

⚠ 应用提醒：卡尔曼滤波假设噪声服从高斯分布、系统是线性的。如果非线性程度较高，需要使用扩展卡尔曼滤波（EKF）或无迹卡尔曼滤波（UKF）。此外，如果噪声分布严重偏离高斯，卡尔曼滤波的估计可能不是最优的。

18.6 状态空间模型的实践优势

相比单独构建 ARIMA 或指数平滑模型，使用状态空间框架有四个显著优势：

优势说明
缺失数据天然处理卡尔曼滤波遇到缺失观测时，跳过更新步骤、继续预测即可——不需要插值
时变参数回归系数、季节性因子等可以随时间变化，而不是固定不变
不确定性量化每一步都给出状态估计的方差——知道"我们有多确定"
模块化设计趋势、季节、回归、AR 组件可以自由组合
# 构建一个包含趋势+季节+回归分量的状态空间模型
# 这相当于 ETS + 外生变量的统一模型
model = sm.tsa.UnobservedComponents(
    y,
    level='local linear trend',   # 时变趋势
    seasonal=12,                   # 月度季节性
    exog=x_reg                     # 外生回归变量
)
result = model.fit()
print(result.summary())
要点总结：状态空间模型是一个统一的建模框架，卡尔曼滤波是其核心推理引擎。ARIMA、指数平滑都是状态空间模型的特殊形式。卡尔曼滤波的预测-更新循环是工程史上最优雅的递归算法之一。
