📡 第2章 信号与频谱基础

大家好，我是你们的讲师。先问一个"灵魂拷问"：你家WiFi路由器上是不是有两个频段——2.4GHz和5GHz？你有没有发现，2.4GHz穿墙能力强，但速度慢；5GHz速度快，但隔一堵墙信号就掉了两格？这是为什么？要回答这个问题，我们需要先搞明白一个核心概念——频谱。

信号与频谱是整个通信工程的"普通话"。从模拟电话到6G太赫兹通信，从FM收音机到Massive MIMO波束赋形，所有通信技术最终都可以归结为"如何在频谱上高效、可靠地传输信息"。本章我们将从最基础的信号分类开始，一路讲到频谱规划与香农极限——这些是后面每一章都要反复用到的底层知识。

2.1 信号的分类

信号是信息的载体。在通信系统中，我们按照不同维度对信号进行分类：

  • 连续信号 vs 离散信号：连续信号在时间和幅度上都连续（如麦克风采集的声波），离散信号仅在离散时间点上有取值（如CD音频44.1kHz采样后的数字序列）。现实中大多数原始信号是连续的，但现代通信系统几乎全部使用离散信号进行处理——这就是为什么ADC（模数转换器）是通信系统里的关键器件。
  • 周期信号 vs 非周期信号：正弦波是典型的周期信号，语音则是非周期信号。有趣的是，任何非周期信号都可以看作无穷多个周期信号的叠加——这正是傅里叶分析的核心思想。
  • 确定信号 vs 随机信号：确定信号可以用数学表达式精确描述（如sin(2πft)），随机信号则只能用统计特性来描述（如信道噪声）。通信系统中，我们发送的是确定信号，但它经过信道后叠加了随机噪声——接收端的任务就是从随机信号中恢复出确定信号。

小知识：奈奎斯特采样定理告诉我们——要以无损的方式从离散样本中恢复出连续信号，采样频率必须≥信号最高频率的两倍。这就是为什么CD音频用44.1kHz采样（覆盖20Hz-20kHz人耳范围），而5G系统的采样率则高达数百MHz——因为5G的载波频率是GHz级别。

2.2 时域与频域：一枚硬币的两面

想象一下你在钢琴上同时弹奏C大调和弦——C、E、G三个音。你的耳朵听到的是一个丰满的"和声"，而不是三个独立的音——这是"时域"的感知方式。但如果用频谱分析仪去看，你会看到三个清晰的尖峰：262Hz（C4）、330Hz（E4）、392Hz（G4）——这就是"频域"的呈现方式。

时域告诉我们信号随时间如何变化，频域告诉我们信号由哪些频率分量组成。同一个信号，从两个不同的角度观察，看到的是完全不同的信息。就像一枚硬币的两面——都是真实的，但只看一面永远无法理解全貌。

在通信工程中，绝大多数设计都是在频域完成的：信道带宽决定了能传多快，子载波间隔决定了抗多径能力，频谱分配决定了谁能用哪段频率——这些都是频域的概念。但你最终在示波器上看到的、在天线上辐射出去的，都是时域信号。通信工程师必须具备在时域和频域之间自由切换的能力。

2.3 傅里叶分析：通信工程的"数学望远镜"

傅里叶变换是连接时域和频域的桥梁。它的数学表达式如下：

时域 → 频域：
X(f) = ∫ x(t)·e^(-j2πft) dt   (从 -∞ 到 +∞)

频域 → 时域（逆变换）：
x(t) = ∫ X(f)·e^(j2πft) df   (从 -∞ 到 +∞)

先别被积分符号吓到。直观理解是：傅里叶变换把一段信号"拆开"，看它里面包含了哪些频率分量，每个分量的幅度和相位是多少。就像用三棱镜把白光分解成彩虹——白光（时域信号）经过三棱镜（傅里叶变换）就变成了七色光谱（频谱）。

在实际工作中，我们很少手动算傅里叶变换——都是用计算机的FFT（快速傅里叶变换）算法。下面是用Python验证的一个例子：

import numpy as np
import matplotlib.pyplot as plt

# 参数设置
fs = 1000  # 采样率 1000 Hz
t = np.arange(0, 1, 1/fs)  # 1秒时长

# 生成一个复合信号：50Hz + 120Hz 正弦波
x = 0.7 * np.sin(2 * np.pi * 50 * t) + 1.2 * np.sin(2 * np.pi * 120 * t)

# FFT
X = np.fft.fft(x)
freqs = np.fft.fftfreq(len(x), 1/fs)

# 只取正频率部分，显示幅度谱
idx = np.where(freqs >= 0)
plt.plot(freqs[idx], np.abs(X[idx]))
plt.xlabel('频率 (Hz)')
plt.ylabel('幅度')
plt.title('FFT 幅度谱')
plt.grid(True)
plt.show()

# 输出：在 50Hz 和 120Hz 处出现两个峰值，
# 幅度分别为 350 和 600（与输入幅度成正比）

这个例子的核心在于：时域中看似复杂的波形，变换到频域后就是两个清晰的峰值。通信工程中大量使用这种思路——在时域难以处理的问题（比如多径干扰），变换到频域后可能变得非常简单。

关键洞察：OFDM（正交频分复用）——4G/5G的核心技术——本质上就是通过IFFT（逆快速傅里叶变换）在发射端一次生成大量正交子载波，再通过FFT在接收端解调。没有FFT算法的高效实现，OFDM就不可能商用。这就是"一个数学算法改变了一个产业"的最佳案例。

2.4 频谱资源：通信世界的"土地"

频谱是通信行业最宝贵的资源，就像房地产行业的土地——位置（频率）决定了价值，而总量是有限的。全球频谱由ITU（国际电信联盟）统筹管理，各国频谱监管机构（中国的工信部、美国的FCC）负责具体分配。

3GPP为5G NR定义了多个频段范围（FR1和FR2），下面是中国当前5G频段的具体分配情况：

运营商 频段 频率范围 带宽 组网方式 特点
中国移动 n41 2515-2675 MHz 160 MHz SA 覆盖与容量均衡，主力频段
中国移动 n79 4800-4900 MHz 100 MHz SA 大容量，热点补盲
中国电信 n78 3400-3500 MHz 100 MHz SA 与中国联通共建共享
中国联通 n78 3500-3600 MHz 100 MHz SA 与中国电信共建共享
中国广电 n28 703-733 / 758-788 MHz 30 MHz×2 SA 700MHz"黄金频段"，覆盖广

注意到频率越高、带宽越大的规律了吗？这就是物理定律决定的：高频段有更宽的连续频谱可用（所以5GHz WiFi比2.4GHz快），但高频段传播损耗更大、穿墙能力更差（所以5GHz穿墙不如2.4GHz）。通信工程中的每一个设计决策，本质上都是在与物理定律博弈。

概念辨析：频谱 vs 带宽。很多初学者混用这两个词，但它们有严格区别。频谱是指电磁波的频率分布范围（如"5GHz频段"），带宽是指一段频谱的宽度（如"100 MHz带宽"）。打个比方：频谱就像是"道路的位置"（在北京三环），而带宽就像是"道路的宽度"（8车道）。位置决定了覆盖特性，宽度决定了容量上限。

2.5 香农定理：信道容量的终极边界

1948年，克劳德·香农发表了一篇划时代的论文——《通信的数学理论》。其中给出了一个简单到令人震惊的公式，定义了在噪声信道中无差错传输的最大数据速率：

C = B · log₂(1 + S/N)

其中，C是信道容量（单位bit/s），B是信道带宽（单位Hz），S/N是信噪比（Signal-to-Noise Ratio，无量纲）。这个公式告诉我们三件事：

  • 带宽越大，容量越大（线性关系——增加带宽直接提升容量）
  • 信噪比越高，容量越大（对数关系——信噪比提升到一定程度后，收益递减）
  • 容量有上限——无论你用多复杂的编码和调制，都不可能超过香农极限

下面这张表展示不同信噪比下的香农容量极限：

SNR (dB) S/N (线性比) log₂(1+S/N) 每Hz容量 (bit/s/Hz) 100 MHz带宽下容量
-5 0.32 0.40 0.40 40 Mbps
0 1.0 1.00 1.00 100 Mbps
5 3.16 2.06 2.06 206 Mbps
10 10.0 3.46 3.46 346 Mbps
15 31.6 5.03 5.03 503 Mbps
20 100 6.66 6.66 666 Mbps
30 1000 9.97 9.97 997 Mbps

从表中可以清晰看到：当SNR从20dB提升到30dB（功率提升10倍），每Hz容量只从6.66增加到9.97——这就是对数收益递减的体现。而5G系统中，256QAM需要的SNR大约在22-25dB，这正是香农极限附近的"吃力不讨好"区域。

香农定理的工程指导意义：香农定理为通信系统设计提供了一个"天花板"。如果你的系统距离香农极限还很远（比如差了3dB以上），说明调制编码方案有很大优化空间；如果已经逼近香农极限（比如LDPC码在0.5dB以内），那再提升编码增益就非常困难了，需要从其他维度（如增加带宽、更多天线）来提升容量。5G-Advanced和6G的研究中，一个重要方向就是"如何逼近甚至突破传统香农极限"——比如通感一体化、智能超表面（RIS）等新技术，本质上都是在挑战这个经典框架。

2.6 踩坑故事：频谱规划不当导致同频干扰

2020年某地市5G商用初期，运营商A和运营商B在相邻的区县各自部署了5G基站。A用的是n41频段（2.6GHz），B用的是n78频段（3.5GHz）。按说两个频段间隔900MHz，应该互不干扰才对。

但在边界区域，用户频繁投诉5G掉线、速率极低。路测团队带着扫频仪一测，发现问题出在基站天线的带外杂散发射上——A的n41基站发射机在3.5GHz附近产生了谐波分量，正好落在B的n78接收频段内。相当于A基站在"大声说话"时，在高频段产生了"回音"，干扰了B基站的"听众"。

排查结论是：基站射频前端滤波器的带外抑制指标不达标。最终解决方案是加装额外的腔体滤波器，但此时已经过去了三周，边界区域的用户体验和品牌口碑都受到了影响。

这个案例告诉我们：频谱规划不仅仅是"分配频率"这么简单，还需要考虑发射机的杂散指标、滤波器性能、天线隔离度等工程细节。一个看起来"纸上完美"的频谱分配方案，到实际部署中可能会因为一个滤波器指标而翻车。

好了，这一章我们在信号与频谱的世界里走了一圈：从什么是信号、时域频域怎么看，到傅里叶变换的物理意义，再到频谱资源分配和香农极限——这些是通信工程最基本但最重要的概念。下一章，我们将学习如何把这些信号"搬"到高频载波上传输——也就是模拟调制与数字调制。调制是通信系统的"发动机"，理解了它，你就明白了信号是如何"飞"起来的。
