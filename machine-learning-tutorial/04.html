4.1 "学习"就是一个优化问题

有没有想过一个问题：当我们说"模型在学习"的时候，到底在发生什么？答案是：模型在学习 = 在找一组最优参数，让损失函数最小化。这本质上就是一个优化问题。理解优化，你就理解了"学习"的内在机制。

我最早学ML的时候，把各种优化器当成"黑盒子"——fit一下就完了。直到有一次训练一个自定义损失函数，模型死活不收敛，我才被迫去理解梯度到底在往哪个方向走。结果发现我的损失函数设计有问题——它在很多区域梯度是0，优化器根本不知道该往哪走。

4.2 梯度下降：一切优化的起点

想象你蒙着眼睛站在一座山上，想要走到山谷最低点。你不知道山的全貌，但你能感觉到脚下地面的倾斜方向。你每次往最陡的下坡方向走一小步——这就是梯度下降。

数学上，参数更新规则就是：

θ_new = θ_old − η · ∇L(θ)

其中 η 是学习率（步长），∇L(θ) 是损失函数在当前参数处的梯度（最陡上升方向），前面加个负号就是下降方向。

三种梯度下降变体

方法每次更新用多少数据优点缺点
Batch GD全部数据集收敛稳定，梯度准确大数据集太慢，显存放不下
Stochastic GD1个样本极快，可在线学习梯度方差大，收敛震荡
Mini-batch GD小批量（32-256）平衡速度和稳定性需要调batch size
import numpy as np

def gradient_descent(X, y, lr=0.01, epochs=1000, batch_size=32):
    n, d = X.shape
    w = np.zeros(d)
    for epoch in range(epochs):
        # 随机采样一个mini-batch
        idx = np.random.choice(n, batch_size, replace=False)
        X_batch, y_batch = X[idx], y[idx]
        # 计算梯度（线性回归的MSE梯度）
        grad = 2/batch_size * X_batch.T @ (X_batch @ w - y_batch)
        w -= lr * grad
    return w

4.3 学习率：最重要的超参数

学习率 η 决定每一步跨多大。这是所有超参数中最重要、最需要调优的一个。

  • 太大：一步跨过最优点，在两边来回震荡，甚至发散（loss变成NaN）
  • 太小：收敛如蜗牛，可能在到达最优点之前训练就结束了
  • 刚好：快速且稳定地收敛
⚠️ 血泪教训：有一次我训练ResNet，loss一直在震荡就是不下降。我调了各种东西——换优化器、加BN、调整网络结构——花了两天。最后发现就是学习率设成了默认的0.1而不是0.001。从那以后，我训练任何模型第一件事就是确认学习率。建议你养成习惯：先用一个小学习率（如1e-4）跑几步确保loss在降，再逐步上调。

学习率调度策略

策略做法适用场景
Step Decay每隔N个epoch，lr乘以0.1通用，简单有效
Cosine Annealinglr按余弦曲线从大到小深度学习训练标配
Warmup前几个epoch lr从0线性增长Transformer训练必备
ReduceLROnPlateauloss不降时自动降低lr不确定训练时长时使用

4.4 凸优化：为什么线性回归一定有最优解？

线性回归的MSE损失函数是凸函数——只有一个全局最小值，没有局部最小值。这是一个非常重要的性质：意味着你用梯度下降一定能找到全局最优，而且不需要担心初始参数的位置。

非线性模型（比如神经网络）的损失函数是非凸的——有无数个局部最小值。理论上这很可怕，但实际上对于大型神经网络，大多数局部最小值都"够好"了，真正的困难不是局部最小值，而是鞍点（梯度为零但不是极值点的地方）。在高维空间中，鞍点远比局部最小值常见。

🔑 直觉：在100维空间中，一个点是局部最小值需要所有100个方向的二阶导数都大于0，这个概率是 (1/2)^100 ——极小。而鞍点只需要有些方向向上、有些方向向下。所以深度学习真正的挑战是逃离鞍点，动量法（Momentum）和Adam就是为此而生的。

4.5 现代优化器：SGD的进化史

纯SGD有一个致命问题：在所有方向上用同一个学习率，但参数尺度差异很大时，在一个小山谷里来回震荡，就是走不出去。

优化器核心改进PyTorch调用
SGD + Momentum加入动量项，像滚下山坡的小球有惯性SGD(momentum=0.9)
AdaGrad每个参数有自适应学习率Adagrad()
RMSprop用指数移动平均平滑学习率RMSprop()
AdamMomentum + RMSprop，标配Adam(lr=1e-3)
AdamWAdam + 解耦权重衰减，更优AdamW(lr=1e-3, weight_decay=0.01)
💡 实践建议：2026年了，如果你不知道该用哪个优化器，就用 AdamW。这是目前深度学习训练的标配——它是Adam的正确实现版本，把权重衰减（L2正则化）从自适应学习率中解耦出来。需要调的主要参数就是学习率（1e-4 到 1e-3）和weight_decay（0.01 到 0.1）。

4.6 本章小结

优化理论教会我们一件事：模型训练的本质是在高维空间中寻找损失函数的最小值。理解梯度下降的直觉、知道学习率的重要性、会选择合适的优化器——这三样东西就能让你在训练模型时不再"瞎调"。下一章，我们开始真正动手：数据预处理与探索性分析。
