开篇：用最少的材料，撑起最大的空间

大家好，我是你们的讲师。今天我们来聊一种充满"智慧"的结构——桁架（Truss）。

如果你坐过火车经过南京长江大桥，或者看过任何一座大跨度的铁路桥，你一定见过桁架——那些由无数三角形小格子组成的钢铁骨架，在阳光下像一张巨大的渔网。但你有没有想过一个问题：为什么桥梁工程师宁愿用成百上千根细小的杆件拼成一张"网"，也不直接用一根大尺度的实腹梁？

答案藏在"效率"二字里。一根简支梁在承受跨中荷载时，截面上部的受压区和下部的受拉区之间，存在一个"中性轴"区域——那里的应力几乎为零。换句话说，实腹梁的大部分材料并没有发挥最大作用。而桁架把这些材料集中到受拉和受压的路径上，形成了清晰的"力流"（force flow）——所有杆件要么受拉，要么受压，没有弯矩。这让材料的利用效率达到了极致。

一个经典的数据对比是：对于跨度30米的铁路桥，实腹钢板梁的自重约为每延米10-12吨，而同等跨度的钢桁梁自重只有每延米4-5吨。这就是桁架的威力——用一半的材料，完成同样的任务。

桁架的基本假设

在结构力学中，我们分析的是理想桁架（Ideal Truss）。所谓"理想"，包含以下三个基本假设：

  1. 所有结点都是光滑铰结点——各杆在结点处可以自由相对转动，不传递弯矩。这意味着每根杆件只承受轴力（拉力或压力）。
  2. 所有杆件都是直杆——轴线为直线。
  3. 荷载只作用在结点上——如果杆件上有分布荷载（如自重），需按等效原则分配到两端结点。这一假设保证了杆件内部无弯矩。

在这三个假设下，每根桁架杆件的内力只有轴力（Axial Force），且轴力沿杆长不变。这就是所谓的"二力杆"（two-force member）——两端承受沿杆轴线方向的一对平衡力。

一个重要的"但书"：实际桁架并非完全满足上述假设。例如，钢桁架的结点通常是用节点板（gusset plate）加螺栓或焊接连接的，并非理想铰；杆件自重是沿杆长分布的，而不是集中在结点上的。这些"非理想因素"会在杆件中引起次弯矩（secondary moment）。但在大多数情况下，次弯矩对整体内力的影响不超过5%，所以在初步设计中完全可以忽略。

零杆的快速判别

零杆（Zero-Force Member）是指在特定荷载工况下内力为零的杆件。识别零杆可以大幅简化桁架分析的工作量——你不需要对零杆进行计算。

零杆的判别遵循以下三条规则：

  • 规则一（L形结点）：两根不共线的杆件在无荷载的铰结点处相交，则这两根杆件均为零杆。
  • 规则二（T形结点）：三根杆件在无荷载的铰结点处相交，其中两根共线，则第三根杆件（即"支路"上的杆件）为零杆。
  • 规则三（K形结点对称）：在对称桁架承受对称荷载时，对称轴上K形结点的两侧斜杆内力大小相等、符号相反。若此时无水平荷载作用，则两侧斜杆可判定为零杆。
零杆判别示例：

        (1) L形结点：
            A——B          A和B是两根不共线杆件，
            |             结点处无外荷载→两根均为零杆
            C

        (2) T形结点：
            A——B——C        杆DB与AB、BC不共线
                |         结点B无外荷载→DB为零杆
                D

        (3) 对称性：
            对称桁架、对称荷载
            对称轴处的斜杆可能为零杆
实用建议：拿到一个桁架题目后，别急着列方程。先用30秒扫一遍——找到所有L形和T形结点，把零杆标记出来。往往你标完零杆后，需要计算的杆件就少了一半。这是一个非常实用的"先减后算"策略。

结点法求解桁架内力

结点法（Method of Joints）是求解桁架内力的最基本方法。它的思路是以结点为隔离体，列出每个结点的平衡方程，逐个求解杆件内力。

结点法的具体步骤：

  1. 先求出桁架的支座反力（整体平衡）；
  2. 标记出零杆；
  3. 从一个只有两根未知杆件的结点开始求解（通常从支座结点开始）；
  4. 每个结点列两个平衡方程：∑Fx = 0 和 ∑Fy = 0；
  5. 将求出的杆力传递到相邻结点，直到所有杆件求解完毕。

关键技巧：始终假定未知杆力为拉力（背离结点）。如果计算结果为正，则该杆受拉（tension）；结果为负，则该杆受压（compression）。这个"统一假定法"可以避免符号混乱。

踩坑故事：我曾亲眼见过一位同事用结点法分析一个体育馆的屋盖桁架，在计算斜腹杆时搞反了受拉和受压的方向。他假定某根斜腹杆"受压"并以此列方程，计算结果是正的——他于是得出结论"这根杆受压"。但事实上，由于他假定的方向反了，"正"的结果恰恰说明它受拉。这个错误在后续的截面验算中被发现，但导致了整个屋盖半数杆件的重新设计。教训就是：统一假定"背离结点为拉力"，让正负号替你说话，不要试图"猜"杆件的受力方向。

截面法

截面法（Method of Sections）是结点法的"大杀器"——当你不关心全部杆件内力，只想求某几根关键杆件的内力时，截面法比结点法快得多。

截面法的思路是：用一个截面截断三根杆件（因为平面内只能列三个平衡方程），取一侧为隔离体，列出三个方程求解。

截面法的关键技巧：

  • 截面选择——截面最好只截断三根杆件（刚好三个未知数，三个平衡方程可解）。如果截面截断了四根或更多杆，但某些杆的力可以通过特殊条件（如零杆、对称性）确定，也可以。
  • 矩心选择——利用"∑M = 0"时，选择两根未知杆件的交点为矩心，可以消去这两根杆的力，直接解出第三根杆的力。
  • 投影方程——当矩心法不便时，利用∑Fy = 0或∑Fx = 0求解。
截面法示例：

        求下图桁架中杆AB的内力：

              A
             / \
            /   \
           /     \
          /       \
         B─────────C
         △         △

        步骤1：沿截面I-I截断包括AB在内的三根杆件
        步骤2：取左侧（或右侧）为隔离体
        步骤3：以AB与另一杆的交点为矩心，列∑M = 0
        步骤4：解出F_AB
对比维度 结点法 截面法
研究对象 单个结点 部分桁架（隔离体）
方程数 每个结点2个（∑Fx=0, ∑Fy=0） 每截面3个（∑Fx=0, ∑Fy=0, ∑M=0）
求解能力 一次最多解2根未知杆 一次最多解3根未知杆
适用范围 全部杆件内力 特定杆件内力（快速求解）
计算效率 杆件少时效率高 求局部杆件时效率高

联合法应用

在实际工程中，我们通常结合使用结点法和截面法，发挥各自的优势：

  • 用结点法求所有杆件的内力——适合杆件数量不多的桁架（< 20根杆）；
  • 用截面法快速求关键杆件的内力——适合大跨度桁架的初步设计或校核。

一个典型的"联合法"策略是：先用截面法求出某几根关键杆的内力，然后用结点法校验。这种方法在工程师的日常工作中非常常见——截面法给出"大方向"，结点法给出"细节确认"。

高级思维：桁架分析的最高境界不是"会算"，而是"会看"。一个有经验的工程师看到一张桁架图，能在5秒内画出近似内力分布图——哪里受拉最大，哪里受压最大，哪里可能是零杆。这种能力来自大量练习和对力流路径的直觉。建议你多做一些这样的训练：给定一个桁架和荷载，不列方程，只凭定性分析画出内力分布，然后拿计算验证——你会惊讶于自己的进步速度。
