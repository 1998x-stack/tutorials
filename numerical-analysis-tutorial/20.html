第20章 非线性方程求根(一)

大家好，我是你们的讲师。从小到大，我们学过解一元二次方程 ax²+bx+c=0 的求根公式。但你有没有想过，如果方程是 x⁵ - 3x + 1 = 0 怎么办？或者是 e⁻ˣ = sin x 呢？这些都称为非线性方程，而绝大多数非线性方程是没有解析公式的。

在工程和科学计算中，非线性方程求根是最常见的问题之一。从计算化学反应的平衡浓度，到金融中的隐含波动率，再到计算机图形学中的光线与曲面求交——如果你能写出一个高效可靠的非线性方程求根算法，你的代码会比90%的程序员更"聪明"。本章我们就从最基础的方法讲起。

20.1 二分法 (Bisection Method)

二分法是所有求根方法中最朴素、最安全的一种。它的思想非常简单：如果连续函数f(x)在区间[a,b]两端取异号，即f(a)·f(b) < 0，那么根据介值定理，区间内至少存在一个根。接下来不断将区间一分为二，保留有根的那一半，直到区间长度小于容差。

def bisection(f, a, b, tol=1e-12, max_iter=100):
    if f(a) * f(b) >= 0:
        raise ValueError("f(a) and f(b) must have opposite signs")

    for k in range(max_iter):
        c = (a + b) / 2
        fc = f(c)

        if abs(fc) < tol or (b - a) / 2 < tol:
            return c, k + 1

        if f(a) * fc < 0:
            b = c
        else:
            a = c

    return (a + b) / 2, max_iter

二分法的优点

二分法最大的好处是保证收敛——只要初始区间满足f(a)·f(b) < 0，它就一定收敛。而且每次迭代误差至少减半，所以需要的迭代次数可以事先知道：要达到容差ε，需要n = log₂((b-a)/ε)步。对于初始区间长度为1、容差10⁻¹²的问题，仅需大约40步。这种可靠性使其成为其他求根算法失败时的绝佳"备胎"。

但二分法也有明显的缺点：收敛速度慢（线性收敛，且收敛因子为½），而且需要已知一个异号区间。如果一个函数所有的根都在同一侧（比如f(x)=x²），二分法就完全无法工作。另外，对于有重根的情况（f(x) = (x-1)²在x=1处有重根），f(a)·f(b)不会改变符号，二分法也失效。

20.2 不动点迭代法

不动点迭代的思路完全不同。我们将方程f(x)=0改写成等价形式x = g(x)，然后从一个初始猜测x₀出发，反复迭代：

xk+1 = g(xk)

如果序列{xk}收敛到x*，且g连续，则x* = g(x*)，这正是我们想要的根。

但这里有一个关键问题：同一个f(x)=0可以写成无数种不同的不动点形式，有些收敛，有些发散。比如x² - 2 = 0可以写成：

  • x = (x + 2/x)/2 → 收敛（这就是著名的Newton法）
  • x = 2/x → 不收敛（对任何非零初值都会振荡）
  • x = x² + x - 2 → 发散（除非x₀恰巧是根）

不动点收敛定理

设x*是g的不动点。如果存在x*的一个邻域U，使得对任意x∈U，有|g'(x)| ≤ L < 1（L是Lipschitz常数），则对任意初值x₀∈U，不动点迭代收敛到x*。

推论：如果g'(x*) ≠ 0，则收敛是线性的；如果g'(x*) = 0，则收敛至少是二次的。

不动点迭代的几何直观可以通过"蛛网图"(cobweb diagram)来理解：在平面直角坐标系中画出直线y=x和曲线y=g(x)。从(x₀, x₀)出发，向上/下连接到曲线上的(x₀, g(x₀))，再水平连接到直线上的(g(x₀), g(x₀))，重复这个过程。如果|g'| < 1，蛛网会螺旋向内收敛到交点；如果|g'| > 1，蛛网则会向外发散。

20.3 收敛阶的定义与判断

在数值分析中，我们用一个精确的数学定义来衡量迭代法收敛的快慢：

收敛阶 (Order of Convergence)

如果存在常数q（0 < q < 1）和正整数p ≥ 1，使得

limk→∞ |ek+1| / |ek|p = C > 0

其中ek = xk - x*，则称迭代法具有p阶收敛。p=1时称为线性收敛，p=2时称为二次收敛。

这个定义的意义在于：高收敛阶意味着误差在每一步被更快地缩小。具体来说：

  • 线性收敛 (p=1)：每步误差乘以常数C，即|ek| ≈ Ck|e₀|。如果C=0.5，需要约10步将误差缩小到千分之一。
  • 二次收敛 (p=2)：每步误差平方，即|ek| ≈ C^{2k-1}|e₀|2k。仅需约5步就能将误差从10⁻¹缩小到10⁻³²！
方法收敛阶需要导数保证收敛？
二分法线性 (p=1)否是（条件满足时）
不动点迭代线性 (p=1)否仅当|g'|<1
Newton法二次 (p=2)是需初值足够好
弦截法p≈1.618否需初值足够好

要确定一个迭代法的收敛阶，一个常用的技巧是Taylor展开。例如对于不动点迭代xk+1=g(xk)，将g在x*处展开：

ek+1 = g'(x*)ek + ½g''(x*)ek² + ...

如果g'(x*) ≠ 0，则|ek+1|/|ek| → |g'(x*)|，所以是线性收敛，且渐近常数C = |g'(x*)|。如果g'(x*) = 0且g''(x*) ≠ 0，则|ek+1|/|ek|² → ½|g''(x*)|，即二次收敛。

20.4 加速方法

线性收敛的迭代法虽然可靠，但速度太慢。能否将一个线性收敛序列"加速"成更高收敛阶的序列呢？答案是肯定的。

Aitken的Δ²加速：设想一个线性收敛序列xk → x*，即xk+1 - x* ≈ C(xk - x*)。通过消去常数C，我们可以得到估计：

x* ≈ xk - (Δxk)² / Δ²xk

其中Δxk = xk+1 - xk，Δ²xk = xk+2 - 2xk+1 + xk。这个变换可以反复应用——即使原始序列是线性收敛，加速后的序列可能具有二次收敛速度。

Steffensen方法进一步把Aitken加速嵌入到不动点迭代中，无需计算导数：

def steffensen(g, x0, tol=1e-12, max_iter=100):
    x = x0
    for k in range(max_iter):
        y = g(x)
        z = g(y)
        # Aitken 加速
        denom = z - 2*y + x
        if abs(denom) < 1e-15:
            return x, k+1
        x_new = x - (y - x)**2 / denom

        if abs(x_new - x) < tol:
            return x_new, k+1
        x = x_new
    return x, max_iter


def fixed_point_iteration(g, x0, tol=1e-12, max_iter=100):
    x = x0
    for k in range(max_iter):
        x_new = g(x)
        if abs(x_new - x) < tol:
            return x_new, k+1
        x = x_new
    return x, max_iter

踩坑故事：被"慢收敛"拖垮的期权定价引擎

几年前我帮一个量化团队审查他们的期权定价系统。他们用Black-Scholes模型反推隐含波动率，核心就是一个非线性方程f(σ) = Cmarket - CBS(σ) = 0。他们直接用了简单的不动点迭代，写成σ = g(σ)的形式，g的导数在目标σ附近约0.85。表面上看迭代是收敛的，但系统在生产环境每天处理数十万笔期权，每次需要迭代约50步才能达到所需精度——高峰时期服务延迟严重。我检查后发现这是一个典型的线性收敛场景，Aitken加速可以将步数降到约10步。更关键的是，切换到Newton法（虽然需要vega——即BS公式对σ的导数）可以实现二次收敛，通常3-5步完成。最终他们采用了Newton法加二分法做保护（防止Newton因初值不好而发散），系统延迟降低了5倍。

这个小故事告诉我们一个重要的工程原则：当对性能有高要求时，"能用"不等于"够好"。理解收敛阶，选择合适的算法，有时比简单地优化代码细节带来的收益大得多。

让我们来对比一下这三种方法的实际表现。假设我们要求解f(x) = cos(x) - x = 0，精确根约为0.739085：

方法初值到达10⁻¹²容差的步数最终误差
固定点迭代0.5约50步9.5×10⁻¹³
Aitken加速0.5约9步6.2×10⁻¹³
Steffensen0.5约5步3.1×10⁻¹³

从表中可以看出，加速方法的优势极为明显。Steffensen方法本质上把每步需要计算两次g(x)的代价换来了二次收敛的速度，性价比很高。

最后做个总结：我们今天学习了三种基础求根方法——简单可靠的"二分法"、理解起来最直观的"不动点迭代法"、以及其"加速版本"。这些都是理解更高级方法（Newton法、弦截法等）的基石。
