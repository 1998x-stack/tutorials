7.1 为什么要用阵列？

大家好，欢迎回到课堂。前面几章我们学习了各种单天线——偶极子、单极子、环、缝隙、八木。它们各有特色，但都有一个共同特点：单个天线的能力是有限的。

让我们回顾一下：一个半波偶极子的最大增益只有2.15 dBi，HPBW约78°——它的"视野"太宽了，能量分散到了几乎一半的空间中。如果你想在10公里之外可靠地接收到信号，你需要把能量像手电筒一样聚焦成一个窄束——这需要的增益可能是20、30甚至40 dBi。

仅靠单个天线，要达到这么高的增益是极其困难的：你需要口径面积极大的抛物面天线（比如直径数米的碟形天线）。但抛物面天线是机械笨重的、不可电子扫描的、制造公差极高的。

天线阵列（antenna array）提供了一个优雅的解决方案：将多个（几十、几百甚至成千上万个）天线单元按一定几何布局排列，通过控制每个单元的激励幅度和相位，合成出高度指向性的波束。

故事时间：二战雷达的推动力：1940年代初，英国为了抵御德国空军的夜间轰炸，迫切需要能够快速扫描天空的雷达。但机械旋转一个巨大的抛物面天线太慢了——跟不上高速轰炸机的机动。MIT辐射实验室（Radiation Lab）的科学家们开发了世界上第一个实用的相控阵天线——通过一个称为"波导缝隙阵列"的设计，实现了电子波束扫描。这个技术后来成为现代相控阵雷达的基石。今天，一个宙斯盾驱逐舰上的AN/SPY-1相控阵雷达包含超过4,000个天线单元，可以在微秒级的时间尺度上改变波束指向，同时跟踪数百个目标。

阵列的四大核心优势：

  • 更高的增益：N个单元的理论最大增益提高约10 log₁₀(N)倍。
  • 波束扫描（beam steering）：通过电子方式改变各单元的激励相位，波束可以瞬间指向不同方向——无需机械转动。
  • 方向图形状控制（pattern shaping）：通过选择不同的幅度加权（如Taylor加权、Chebyshev加权），可以控制副瓣电平、波束宽度等。
  • 自适应零陷（adaptive nulling）：在干扰方向形成零陷，抑制干扰信号——这是电子战和现代通信系统的重要能力。

7.2 方向图乘积定理

阵列天线理论中最重要的原理就是方向图乘积定理（pattern multiplication principle）：

阵列的总方向图 = 单元方向图 × 阵列因子

Ftotal(θ, φ) = Felement(θ, φ) · AF(θ, φ)

其中Felement是单个天线单元的方向图（假设所有单元完全相同且同向放置），AF是阵列因子（array factor）——仅由单元的位置和激励系数决定的函数。

这个定理的成立前提是：

  • 所有天线单元完全相同
  • 所有单元的指向方向一致
  • 互耦效应可以忽略（或者已经包含在单元方向图中）

物理信息：单元方向图提供了"包络"，阵列因子提供了"精细结构"。最终的方向图是两者的乘积。想象一下：如果单元方向图在某个方向上有零点（比如偶极子的端射方向），那么无论阵列因子有多大，总方向图在该方向仍然是零。

工程启示：设计阵列天线时，单元的选择决定了方向图的"外轮廓"，而阵列布局决定了内部的结构细节。如果你需要宽角扫描（如±60°），单元本身必须有足够宽的波束覆盖——否则扫描到边缘时增益会急剧下降（称为"扫描损耗"）。

7.3 均匀线阵的分析

我们从最简单、最基础的阵列构型开始——均匀线阵（Uniform Linear Array, ULA）。

假设N个相同的天线单元沿x轴等间距排列，间距为d。每个单元的激励幅度相同（均匀加权），但相邻单元之间的相位差为β（即第n个单元的激励为Iₙ = I₀ e^(j(n-1)β)）。

阵列因子的定义为所有单元辐射场的复数和（忽略单元方向图）：

AF = Σn=1N e^(j(n-1)(k·d·sinθ + β))

设 ψ = kd·sinθ + β，则上式是一个等比级数求和：

AF = [1 - e^(jNψ)] / [1 - e^(jψ)]

取绝对值并归一化：

|AF| = |sin(Nψ/2) / sin(ψ/2)|

这个简洁的闭合表达式就是均匀线阵的阵列因子。它的形状由三个参数决定：

参数 符号 对方向图的影响
单元数 N N越大：主瓣越窄（≈ 1/N），增益越高（≈ 10log₁₀N）。副瓣数量增加，但第一副瓣电平固定为 -13.5 dB（均匀加权时）。
单元间距 d d增大：主瓣变窄，但可能出现栅瓣（grating lobe）。d减小：波束变宽，但不会出现栅瓣。最佳折中：d ≈ λ/2。
相位差 β β控制主瓣指向：θ₀ = arcsin(-β/kd)。β = 0 → 侧射（broadside），β = -kd → 端射（endfire）。

关于副瓣电平：均匀加权（即所有单元等幅激励）的ULA，第一副瓣电平固定为 -13.5 dB，与N无关。这在很多应用中是不够低的（例如雷达通常要求 -30 dB以下的副瓣以避免虚警）。要降低副瓣，必须使用幅度锥削（amplitude tapering）——也就是让中心单元的幅度大、边缘单元的幅度小。常用的锥削函数有Taylor加权、Chebyshev加权（可精确控制最大副瓣电平）、Hamming加权等。幅度锥削的代价是主瓣略微变宽（约1.2-1.5倍）。

踩坑故事：一个5G基站天线项目在初始设计时采用了均匀加权的8×8阵列，仿真增益达到了27 dBi，客户非常满意。但实际测试时发现副瓣电平只达到 -11 dB（理论值 -13.5 dB），略高于指标要求的 -13 dB。项目验收时测试工程师发现有两路功放模块的增益不一致，导致幅度加权偏离了均匀分布——上层4行和下层4行之间出现了约1.5 dB的幅度差。这个"意外锥削"反而使副瓣变差了。最终我们更换了更精确的功放模块并加入了微调衰减器才解决问题。教训：阵列的幅度一致性非常重要——馈电网络的幅度误差直接影响副瓣性能。

7.4 波束扫描与栅瓣条件

波束扫描

在ULA中，通过改变相邻单元之间的渐进相位β，主瓣方向随之改变。关系很简单：

θ₀ = arcsin(-β / kd)

其中k = 2π/λ是波数。当β由负变正时，主瓣从一侧扫描到另一侧。在实际系统中，β由移相器（phase shifter）或数字波束赋形实现。

波束扫描会导致两个效应：

  • 波束展宽：随着扫描角增大，投影口径减小，主瓣宽度按1/cosθ₀增大。当扫描到60°时，波束宽度变为侧射时的2倍。
  • 增益下降：同样因为投影口径减小，加上单元方向图的包络效应，增益随扫描角下降。通常3 dB扫描角范围在±60°左右。

栅瓣条件

栅瓣（grating lobe）是天线的"幽灵"——它们是阵列因子中除了主瓣之外的其他极大值，指向非期望的方向。栅瓣是有害的：它们会浪费能量、造成角度模糊（雷达无法区分目标是来自主瓣方向还是栅瓣方向）、增加干扰。

栅瓣出现的数学条件：当|sin(Nψ/2) / sin(ψ/2)|的分子和分母同时为零时，函数达到极大值。这发生在ψ = ±2mπ（m=0,1,2,...）时。m=0对应主瓣。为了避免m=±1栅瓣出现在可见空间（|sinθ| ≤ 1）中，需要：

避免栅瓣条件：d/λ < 1 / (1 + |sinθ₀|)

对于侧射（θ₀ = 0°）：d < λ（实际上通常取d = λ/2作为安全边界）
对于全半球扫描（|θ₀| < 90°）：d < λ/2
对于有限扫描范围：d可以适当增大

工程经验：ULA的单元间距通常取λ/2，这是一个经过深思熟虑的折中——它既保证了全半球扫描时不出现栅瓣，又不会使阵元间距过小（过密会增加互耦和成本）。但在一些特殊场景中也会有意选择更大的间距：例如卫星通信地面终端如果只需要在±10°范围内扫描，间距可以增大到1.5λ甚至2λ，以减少所需单元数量、降低成本。

直觉训练：想象一组等间距排列的人，每个人手里举着一个手电筒。如果所有人同时把手电筒指向正前方，光线叠加形成一个明亮的"主瓣"。但如果你让相邻的人偏转一个微小角度——这相当于给每个手电筒增加一个相位——光线就会朝另一个方向汇聚。然而，当间距太大时（大于λ/2），除了这个"主瓣"方向之外，还有其他方向上的光线也恰好同相叠加——这就是"栅瓣"。

7.5 均匀面阵与圆阵简介

均匀面阵（Uniform Planar Array）

将线性阵列在另一个维度上扩展，就得到平面阵列（planar array）。最常见的矩形平面阵列的阵列因子是两个正交方向上线性阵列因子的乘积：

AF(θ, φ) = AFₓ(θ, φ) · AFᵧ(θ, φ)

其中AFₓ和AFᵧ分别是沿x方向和y方向的线性阵列因子。这意味着：

  • 平面阵列可以在俯仰和方位两个方向上独立控制波束——这是实现二维波束扫描的能力。
  • 矩形阵列的波束在两个主平面上分别由两个方向的单元数和间距决定。
  • 如果两个方向的间距和单元数相同，波束的横截面大致是圆形/椭圆形的。

平面阵列的增益近似为：G ≈ 4π · (Nₓ · Nᵧ · dₓ · dᵧ) / λ² · η。其中Nₓ, Nᵧ是行列数，dₓ, dᵧ是行列间距，η是口径效率。

圆阵（Circular Array）

与线性阵列和平面阵列不同，圆阵（circular array）的单元均匀排列在一个圆周上。圆阵有几个独特的优势：

  • 方位角全向性：圆阵的波束可以在360°方位角上等性能地扫描——没有"死区"。而线性阵列无法覆盖端射方向附近（增益急剧下降）。
  • 适合DoA估计：圆阵的方位角分辨能力均匀，非常适合无线电测向和到达角（Angle of Arrival, AoA）估计。
  • 适用于共形安装：圆柱体上的圆阵适合安装在导弹、火箭等轴对称载体上。

圆阵的缺点是阵列因子没有闭合形式的简单表达式（需要用到Bessel函数），分析比ULA复杂。此外，圆阵在俯仰方向的波束控制能力较弱。

前沿动态：华为的5G Massive MIMO基站天线（如AAU5613）采用了192个天线单元的平面阵列（12行×16列，双极化），在64T64R配置下可以实现±60°的二维波束扫描。未来的6G基站可能使用1024甚至4096个单元的"超大规模MIMO（XL-MIMO）"——这将把天线阵列技术推向新的高度。

下面我们用Python绘制ULA的阵列因子，展示不同参数对方向图的影响：

import numpy as np

def ula_array_factor(N, d_over_lam, theta_deg, beta=0):
    """
    计算均匀线阵的阵列因子
    参数:
        N           : 单元数
        d_over_lam  : 间距 (以波长λ为单位)
        theta_deg   : 观测角 (度), 0°=侧射方向
        beta        : 相邻单元间渐进相位 (rad)
    返回:
        AF_norm     : 归一化阵列因子 (线性, 最大值=1)
    """
    theta = np.deg2rad(theta_deg)
    kd = 2 * np.pi * d_over_lam
    psi = kd * np.sin(theta) + beta

    # 使用闭合公式 |AF| = |sin(Nψ/2) / sin(ψ/2)|
    AF = np.sin(N * psi / 2) / np.sin(psi / 2)
    # 处理 ψ → 0 时的极限 (主瓣方向)
    AF = np.where(np.abs(np.sin(psi/2)) < 1e-10, N, AF)

    AF_dB = 20 * np.log10(np.abs(AF) / N + 1e-10)
    return AF_dB

# 参数扫描分析
theta_plot = np.linspace(-90, 90, 901)

print("均匀线阵 (ULA) 方向图参数分析")
print("=" * 50)
print()

# 案例1: 不同单元数 N 的影响
print("案例1: N对方向图的影响 (d=0.5λ, β=0)")
N_list = [4, 8, 16, 32]
for N in N_list:
    af = ula_array_factor(N, 0.5, theta_plot)
    # 找到 HPBW (近似)
    hp_idx = np.where(af >= -3)[0]
    if len(hp_idx) > 0:
        HPBW = theta_plot[hp_idx[-1]] - theta_plot[hp_idx[0]]
    else:
        HPBW = 0
    # 第一副瓣电平
    mainlobe = np.where(af >= -1)[0]
    if len(mainlobe) > 0:
        outside = af.copy()
        outside[mainlobe[0]:mainlobe[-1]+1] = -100
        SLL = outside.max()
    else:
        SLL = -100
    print(f"  N={N:2d}: HPBW ≈ {HPBW:.1f}°, 第一副瓣 = {SLL:.1f} dB, "
          f"峰值增益≈ {10*np.log10(N):.1f} dBi")

print()

# 案例2: 不同间距 d 的影响
print("案例2: d对方向图的影响 (N=8, β=0)")
d_list = [0.25, 0.5, 0.75, 1.0]
for d in d_list:
    af = ula_array_factor(8, d, theta_plot)
    hp_idx = np.where(af >= -3)[0]
    HPBW = theta_plot[hp_idx[-1]] - theta_plot[hp_idx[0]] if len(hp_idx) > 0 else 0
    # 检查栅瓣
    grating = np.max(af[theta_plot > 30]) > -5 if d > 0.5 else False
    print(f"  d={d:.2f}λ: HPBW ≈ {HPBW:.1f}°, 栅瓣={grating}")

print()

# 案例3: 波束扫描
print("案例3: 波束扫描 (N=16, d=0.5λ)")
scan_angles = [0, 30, 60]
for theta0 in scan_angles:
    beta = -2 * np.pi * 0.5 * np.sin(np.deg2rad(theta0))
    af = ula_array_factor(16, 0.5, theta_plot, beta=beta)
    hp_idx = np.where(af >= -3)[0]
    HPBW = theta_plot[hp_idx[-1]] - theta_plot[hp_idx[0]] if len(hp_idx) > 0 else 0
    peak_idx = np.argmax(af)
    print(f"  θ₀={theta0:2d}°: β={beta:.2f}, HPBW ≈ {HPBW:.1f}°, "
          f"实际峰值在 θ={theta_plot[peak_idx]:.1f}°")

print()
print("关键发现:")
print("1. N增大 → 主瓣变窄, 增益升高, 但第一副瓣保持在 -13.5 dB")
print("2. d > λ 时会出现栅瓣, 工程上取 d = λ/2 作为标准间距")
print("3. 波束扫描时 HPBW 按 1/cosθ₀ 展宽, 增益相应下降")

本章我们从"为什么需要阵列"这个根本问题出发，学习了天线阵列的核心理论。方向图乘积定理将问题优雅地分解为单元方向图和阵列因子的乘积——这是阵列分析的基石。均匀线阵（ULA）的闭合表达式|sin(Nψ/2)/sin(ψ/2)|让我们直观地理解了N、d、β三个参数对方向图的影响。我们还认识了栅瓣的危害和避免条件、波束扫描的原理和局限性，并简要介绍了平面阵列和圆阵的拓展。

第8章我们将从通用的阵列理论回归到一种具体而重要的天线类型——微带天线（microstrip/patch antenna）。微带天线是当今无线通信系统中应用最广泛的天线类型之一：它低剖面、轻重量、易于制造、可集成。我们将学习微带天线的传输线模型和腔体模型，设计一个矩形贴片天线，并讨论馈电方式、带宽拓展和多频段设计。微带阵列是第7章阵列理论和第8章微带天线技术的完美结合——我们将在后续章节中再回来看这个交汇点。
