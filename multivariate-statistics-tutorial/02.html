数学脚手架：多元统计的线性代数基础

大家好，欢迎回到多元统计分析课程。在开始之前，我想先问一个问题：你学线性代数的时候，有没有觉得"这些东西到底有什么用"？特征值、特征向量、矩阵分解——考试的时候背得滚瓜烂熟，一考完就忘干净了。如果你有过这样的体验，请放心，你不是一个人。事实上，绝大多数人在学线代时都缺少一个关键的视角——线性代数是用来描述数据变换的语言，而多元统计分析，恰恰就是用这套语言来描述多维数据的科学。

本章是整门课程的"数学脚手架"。我们会快速过一遍矩阵运算的核心概念，把重点放在那些在后面章节中反复出现的内容上：特征值分解、协方差矩阵的谱分解、SVD、以及随机向量的统计性质。如果你已经有不错的线代基础，可以把这一章当作温习；如果你觉得线代学得不够扎实，请仔细阅读每一节——它们是你理解后续所有方法的基石。

矩阵基本运算回顾

我们先从最基本的矩阵运算开始。假设读者已经接触过矩阵的概念，下面这张速查表可以帮助你快速唤醒记忆：

运算记号定义备注
加法C = A + Bcij = aij + bij要求 A、B 维数相同
数乘C = λAcij = λ · aij每个元素都乘以 λ
矩阵乘法C = Am×nBn×pcij = Σk=1n aikbkjA的列数必须等于B的行数
转置AT(AT)ij = aji行变列、列变行
逆矩阵A−1A−1A = AA−1 = I只对方阵；A必须满秩
行列式det(A) 或 |A|标量，衡量矩阵的"体积缩放"A 奇异当且仅当 det(A)=0
迹tr(A)对角线元素之和tr(AB) = tr(BA)

在多元分析中，有几个性质会反复用到：

  • (AT)T = A 和 (AB)T = BTAT（转置的"反转律"）
  • (AB)−1 = B−1A−1（逆的"反转律"）
  • tr(ABC) = tr(CAB) = tr(BCA)（迹的循环置换不变性）
  • det(A−1) = 1/det(A)

特别要强调 迹 (trace) 这个运算。你可能觉得迹没什么了不起——不就是对角线和吗？但在多元统计中，迹有着深刻的统计含义：如果你把数据矩阵的协方差矩阵记为 S，那么 tr(S) 就是所有变量方差的总和，也就是数据的总变异量。当我们做 PCA 时，前 k 个主成分"解释的方差比例"，本质上就是前 k 个特征值之和除以 tr(S)。

特征值与特征向量

对于一个方阵 A，如果存在非零向量 v 和标量 λ 使得：

A v = λ v

则 λ 是 A 的特征值 (eigenvalue)，v 是对应的特征向量 (eigenvector)。

这个等式说的几何含义是什么？你可以把 A 看作一个线性变换——它对空间里的向量进行拉伸、旋转、挤压。但特征向量 v 是一个特殊的方向：当 A 作用在 v 上时，它只是把 v 拉长（或缩短）了 λ 倍，方向没有改变。换句话说，特征向量是变换的"不变方向"。

几何直觉：想象你把一块橡皮泥捏扁。橡皮泥上画的网格线大部分都会扭曲，但总有一些线方向不变——它们就是"特征方向"。特征值则告诉你沿这个方向被拉伸（λ > 1）还是压缩（0 < λ < 1）了多少。如果 λ = 0，意味着这个方向被完全压扁了——矩阵在这个方向上是奇异的。

如果 A 是一个 对称矩阵 (symmetric matrix)——即 A = AT——那么它的特征值全部是实数，且特征向量可以选为正交的。这个性质在多元统计中至关重要，因为协方差矩阵 Σ 就是对称的。对称矩阵的谱分解 (spectral decomposition) 为：

A = Q Λ Qᵀ

其中 Λ = diag(λ₁, λ₂, ..., λₚ) 是特征值对角矩阵
Q = [v₁, v₂, ..., vₚ] 是正交矩阵 (QᵀQ = I)
Q 的每一列是对应的特征向量

谱分解的意义在于：它把一个矩阵拆解成了"旋转（Qᵀ）→ 缩放（Λ）→ 旋转回来（Q）"三个步骤。当我们后面讲 PCA 时，你会发现 PCA 本质上就是在做协方差矩阵的谱分解。

矩阵的几种重要分解

在多元分析中，除了谱分解，还有几种分解方法非常常用：

Cholesky 分解

如果 A 是对称正定矩阵（所有特征值 > 0），则存在唯一的下三角矩阵 L，使得：

A = L Lᵀ

Cholesky 分解在多元正态分布的模拟中特别有用：如果你要生成服从 N(μ, Σ) 的随机向量，可以先生成标准正态向量 z ~ N(0, I)，然后做变换 x = μ + L z，其中 L 是 Σ 的 Cholesky 因子。这样一步就搞定了多元随机数的生成。

QR 分解

任意 n × p 矩阵 X（n ≥ p）可以分解为 X = Q R，其中 Q 是 n × p 列正交矩阵（QᵀQ = I），R 是 p × p 上三角矩阵。QR 分解在最小二乘回归中用于数值稳定的求解。

奇异值分解（SVD）

任意 m × n 矩阵 A 都可以分解为：

A = U Σ Vᵀ

其中 U 是 m × m 正交矩阵（左奇异向量）
Σ 是 m × n 对角矩阵（奇异值 σ₁ ≥ σ₂ ≥ ... ≥ σᵣ > 0）
V 是 n × n 正交矩阵（右奇异向量）

SVD 是多元分析中最重要的矩阵分解，没有之一。为什么？因为 PCA 可以完全等价地通过 SVD 来实现——你不需要对协方差矩阵做特征值分解，而是直接对中心化后的数据矩阵做 SVD，右奇异向量就是主成分方向，奇异值的平方就是特征值。SVD 的数值稳定性更好，而且可以直接处理 n < p 的高维情况（而协方差矩阵在这种情况下是奇异的）。

实战建议：在实际编码中，如果你要做 PCA，推荐用 SVD（scikit-learn 的 PCA 实现底层就是 SVD）。不要自己去算协方差矩阵再算特征值分解——数值误差大，而且多了一步中间计算。SVD 一步到位。

随机向量

现在我们把概率论引入矩阵框架。一个随机向量 (random vector) X 是一个由 p 个随机变量组成的列向量：

X = (X₁, X₂, ..., Xₚ)ᵀ

描述随机向量的分布，需要两个基本量：

均值向量 (mean vector)：

μ = E[X] = (E[X₁], E[X₂], ..., E[Xₚ])ᵀ
   = (μ₁, μ₂, ..., μₚ)ᵀ

协方差矩阵 (covariance matrix)：

Σ = Cov(X) = E[(X - μ)(X - μ)ᵀ]

   = ⎡ Var(X₁)   Cov(X₁,X₂)  ...  Cov(X₁,Xₚ) ⎤
     ⎢ Cov(X₂,X₁)  Var(X₂)   ...  Cov(X₂,Xₚ) ⎥
     ⎢    ⋮          ⋮        ⋱      ⋮       ⎥
     ⎣ Cov(Xₚ,X₁) Cov(Xₚ,X₂) ...  Var(Xₚ)   ⎦

协方差矩阵的性质极其重要，请一定记住：

  • Σ 是 对称 的：Σij = Σji
  • Σ 是 半正定 的：对任意非零向量 a，有 aᵀΣa ≥ 0
  • 对角线元素 σii = Var(Xi) 是方差
  • 非对角线元素 σij = Cov(Xi, Xj) 衡量两个变量的线性关系
  • 如果 Σ 是正定的（可逆），则数据没有冗余维度

此外还有两个相关的概念：互协方差矩阵 (cross-covariance matrix) 用于衡量两个随机向量之间的协方差：Cov(X, Y) = E[(X - μX)(Y - μY)ᵀ]；相关矩阵 (correlation matrix) P 则是将协方差矩阵标准化后的结果：Pij = σij / (σiiσjj)1/2，取值范围在 [-1, 1]。

重要区分：协方差矩阵和相关矩阵携带的信息不同。协方差矩阵包含量纲信息——如果一个变量以"万元"为单位，另一个以"元"为单位，前者的方差会看起来大很多，但这不代表它"更重要"。相关矩阵消除了量纲的影响，纯粹反映线性关系的强弱。在后续的 PCA 和因子分析中，你将面临一个重要选择：到底用协方差矩阵还是相关矩阵？这个选择会直接影响分析结果，我们会在第8章详细讨论。

二次型与线性变换

二次型 (quadratic form) 是指形如 aᵀA a 的表达式，其中 A 是对称矩阵。在多元统计中，二次型无处不在——

  • 马氏距离 (Mahalanobis distance)：D² = (x - μ)ᵀΣ−1(x - μ)，本质上就是一个二次型。它度量的是从点 x 到分布中心 μ 的"统计距离"，考虑了变量的相关性。
  • 样本协方差矩阵：S = (1/(n-1)) Σi=1n (xi - x̄)(xi - x̄)ᵀ，每一项都是外积（rank-1 矩阵），所有项之和就是二次型。
  • 残差平方和：在回归分析中，εᵀε = Σ εi² 也是二次型。

关于线性变换，考虑一个随机向量 X 和常数矩阵 A（m × p）和常数向量 b（m × 1）：

Y = A X + b

则：
E[Y] = A μ + b
Cov(Y) = A Σ Aᵀ

这个公式极其重要——它告诉我们，对随机向量做线性变换后，新的均值向量和协方差矩阵如何计算。注意协方差变换公式 AΣAᵀ 的结构：它"包裹"了 Σ，左侧用 A，右侧用 Aᵀ。这个模式在后续章节中会出现无数次——比如在 PCA 中计算主成分的方差、在因子分析中分解协方差结构、在 CCA 中寻找最大相关方向等。

实战：用NumPy验证矩阵运算

光说不练是学不好多元统计的。下面我们用一个简短的 Python 例子，验证上面讲的理论：

import numpy as np

# 生成随机数据：100个样本，3个变量
np.random.seed(42)
n, p = 100, 3
X = np.random.randn(n, p) * np.array([1, 5, 2])  # 不同尺度

# 样本均值向量
mean_vec = np.mean(X, axis=0)
print("均值向量:", mean_vec)

# 样本协方差矩阵（注意 ddof=1 用无偏估计）
S = np.cov(X, rowvar=False)  # rowvar=False 表示每列是一个变量
print("协方差矩阵:\n", S)

# 样本相关矩阵
corr = np.corrcoef(X, rowvar=False)
print("相关矩阵:\n", corr)

# 验证：相关矩阵 = 对角标准差^-1 * 协方差矩阵 * 对角标准差^-1
std_inv = np.diag(1 / np.std(X, axis=0, ddof=1))
corr_from_cov = std_inv @ S @ std_inv
print("从协方差矩阵计算的相关矩阵:\n", corr_from_cov)
print("验证成功:", np.allclose(corr, corr_from_cov))
关键点：注意代码中的 rowvar=False——这是用 NumPy 做多元分析最常见的坑之一。NumPy 的 np.cov 默认把行当作变量（每行一个变量），但我们的数据矩阵通常是每列一个变量。忘记设置 rowvar=False 会导致协方差矩阵维度完全错误。我在一次讲座中亲眼看到一位博士生在这个细节上花了两个小时排查问题——所以请一定记住这个参数。

踩坑故事：矩阵求逆的陷阱

最后跟大家分享一个实战中的教训。几年前我在做一个基因表达数据的分析项目，数据集是典型的"高维小样本"：20000个基因的表达水平，只有50个样本。我需要计算样本间的马氏距离来做聚类。公式很简单：D² = (x - μ)ᵀΣ−1(x - μ)。我写好了代码，自信满满地跑起来——结果程序报错："Singular matrix"。协方差矩阵不可逆。

我当时想：这怎么可能？数据明明是我从实验室拿来的，又不是凭空生成的。后来才意识到：当 p > n 时，样本协方差矩阵的秩最多为 n-1，不可能是满秩的。换句话说，你根本就没有足够的信息来估计 20000 × 20000 的协方差矩阵——参数数量远大于样本量。这让我第一次切身感受到了维度灾难 (curse of dimensionality) 的威力。

教训：在做多元分析时，永远要检查 n 和 p 的关系。如果 p > n，经典的协方差矩阵求逆方法会失效。这时候你可能需要用正则化方法（如收缩估计、图套索）、降维（先做 PCA 再分析）或者改用不需要求逆的距离度量（如欧氏距离）。这个教训我们在第26章"高维数据的挑战与对策"中会深入探讨。

本章小结与衔接

这一章的内容可能有点多，但请相信，每一块都是后面章节的"必备品"。我们复习了矩阵基本运算（特别是迹和行列式的统计含义），深入理解了特征值和特征向量的几何本质（变换的不变方向），学习了谱分解、Cholesky 分解、QR 分解和 SVD 四种重要的矩阵分解（其中 SVD 是 PCA 的基石），定义了随机向量的均值向量和协方差矩阵（这两个就是多元数据的"骨架"），探讨了二次型和线性变换在统计中的表现形式，最后用 NumPy 验证了协方差矩阵和相关矩阵的计算，并记住了 rowvar=False 这个关键参数。

有了矩阵和随机向量的基本工具，下一章我们将在这些框架上搭建第一个正式的多元统计模型——多元正态分布。你会看到，上一节的线性变换公式 Y = AX + b 在这里扮演了多么核心的角色：正态分布经线性变换后还是正态分布，所有条件分布也都是正态的，而且协方差矩阵的逆决定了条件期望的系数——这些性质是整个多元推断大厦的砖瓦。
