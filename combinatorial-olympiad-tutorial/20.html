第二十章：图论基础（二）

大家好，欢迎回到图论的世界。上一章我们打下了图论的"地基"——图的定义、握手引理、树、二分图。这些是最底层的工具，就像我们工具箱里的螺丝刀和锤子。今天我们要讲的东西就更"致命"了——匹配、Hall定理、欧拉与哈密顿图、竞赛图。这些是竞赛中直接拿来"得分"的武器。

每次带竞赛集训，到了图论这个部分我总是很兴奋，因为你会发现图论的美妙之处：很多看起来和生活经验完全无关的数学结论，竟然能精准地解决非常实际的问题。比如Hall定理——它可以告诉你一群男生和女生能不能实现"每个人都找到舞伴"的愿望。这不就是现实版的"非诚勿扰"吗？

20.1 匹配与覆盖

20.1.1 基本概念

设 G = (V, E) 是一个无向图。

  • 匹配（matching）：边集 M ⊆ E，其中任意两条边没有公共顶点。
  • 最大匹配（maximum matching）：包含边数最多的匹配。
  • 完美匹配（perfect matching）：覆盖所有顶点的匹配（每个顶点恰好是一条匹配边的端点）。
  • 饱和顶点（saturated vertex）：匹配中某条边的端点。
  • 顶点覆盖（vertex cover）：顶点集 C ⊆ V，使得每条边至少有一个端点在 C 中。

匹配和覆盖之间存在一个经典的对偶关系：

定理 20.1（Konig定理）

在二分图中，最大匹配的边数等于最小顶点覆盖的顶点数。

这个定理的证明不复杂，但很重要——它建立了"边"和"点"之间的数量等式。利用这个定理，很多看起来关于"选边"的问题可以转化为"选点"的问题，反之亦然。

20.1.2 Hall婚配定理

这是图论中最重要的定理之一，也是竞赛中出场率极高的结论。

定理 20.2（Hall婚配定理——Hall's Marriage Theorem）

设二分图 G = (X, Y, E)。G 存在一个饱和 X 中所有顶点的匹配（即从 X 到 Y 的完美匹配）当且仅当对 X 的任意子集 S，有：

|N(S)| ≥ |S|

其中 N(S) 是 S 中所有顶点的邻居集合（在 Y 中）。

这个条件称为Hall条件。名字"婚配定理"来自一个经典比喻：X 是一群男生，Y 是一群女生，边表示两人认识。问：能否让每个男生都和他认识的女生配对（一夫一妻）？Hall条件说：对于任意一组男生，他们认识的女生总数不能少于这组男生的人数——这是必要条件，也是充分条件。

证明要点：必要性显然——如果存在饱和X的匹配，则对任意 S ⊆ X，S 中每个男生至少匹配一个不同的女生，所以 |N(S)| ≥ |S|。

充分性的证明稍复杂，通常用归纳法或用Berge引理（增广路定理）。这里给出一个简洁的证明思路：

反证法。假设满足Hall条件但不存在饱和X的匹配。
取一个最大匹配M，由于M不饱和X，存在未匹配顶点 x₀ ∈ X。
定义集合A为从x₀出发通过交错路径（alternating path）
能到达的X中顶点集合，B为能到达的Y中顶点集合。

可以证明|B| = |A| - 1（因为从x₀到这些点的交错路径在Y中终止）。
同时，由于M的最大性，B中没有未匹配的Y顶点（否则有增广路）。
因此N(A) ⊆ B，|N(A)| ≤ |B| < |A|，违反Hall条件。
矛盾。证毕。

Hall定理的应用非常广泛。来看一个经典问题：

例1：一副扑克牌（52张，去掉大小王）被随机分成13堆，每堆4张。证明：可以从每堆中各选一张牌，使得选出的13张牌恰好包含全部13种面值（A, 2, 3, ..., 10, J, Q, K）。

分析：构造二分图。左侧X = {13堆}，右侧Y = {13种面值}。如果某堆包含某种面值的牌，就在对应的堆和面值之间连一条边。我们需要一个饱和X的匹配——即每堆选一种不同的面值。

验证Hall条件：对任意 k 堆的集合 S，它们总共包含 4k 张牌，但面值只有 13 种。如果这些堆认识的面值总数 < k，意味着 S 中的 4k 张牌分散在少于 k 种面值中——每种面值最多出现4次（因为一副牌每种面值恰好有4张），所以 4k ≤ 4|N(S)|，即 |N(S)| ≥ k = |S|。因此Hall条件成立，存在完美匹配。

20.1.3 一个踩坑故事

📖 踩坑实录：有一年集训队做模拟题，题目是："一个班级有n个男生和n个女生。每个男生恰好认识k个女生，每个女生恰好认识k个男生。是否一定可以让每个男生都和他认识的女生配对？"

超过半数的同学回答"是"——因为他们认为Hall条件显然成立：任意一组 S 个男生，每个人认识 k 个女生，所以 S 中男生认识的女生总人次为 k|S|。但认识的人次和认识的人数不是一回事——多个男生可能认识同一个女生。所以 |N(S)| 可能远小于 k|S|。

实际上这个问题并没有那么简单。仅仅知道每个人认识相同数量的人，不能保证 |N(S)| ≥ |S| 对所有 S 成立。反例？考虑 n=3，每个男生认识2个女生。如果3个男生都认识同样的2个女生，那么 |N(S)| = 2 < 3 = |S|，Hall条件被违反。所以"是"的结论是错误的。

这个故事的教训是：Hall条件必须对 X 的所有子集都成立，不能只凭直觉认为"看起来差不多"就下结论。很多学生在考场上也是这样——草草验证了一两个特殊情况就以为充分条件是显然成立的，结果丢了分。

20.2 欧拉图与哈密顿图

这两类图的问题源远流长，分别来自两个著名的历史问题：

  • 欧拉图（Eulerian graph）：起源于哥尼斯堡七桥问题（1736年）。欧拉证明了：走过七桥各一次且返回起点的路线不存在。
  • 哈密顿图（Hamiltonian graph）：起源于哈密顿提出的"环球旅行"问题（1859年）。寻找一条经过所有顶点恰好一次的回路。

它们的区别用一句话概括：欧拉图关心边的遍历，哈密顿图关心点的遍历。

性质 欧拉图 哈密顿图
经过对象 边各一次 顶点各一次
充要条件 连通且所有顶点度数为偶数 没有简单的充要条件
判定难度 极易（看度数即可） NP完全（极难）
竞赛价值 中等（结论直给） 极高（需要创造性）

20.2.1 欧拉图的判定

定理 20.3：一个连通图是欧拉图（存在经过每条边恰好一次并返回起点的回路）当且仅当所有顶点的度数均为偶数。

这个定理的证明是"双向"的：必要性很容易——如果存在欧拉回路，每次进入一个顶点和离开它都会贡献2度，所以所有顶点度数为偶数。充分性用构造法——从任一顶点出发走一条不重复边的闭迹，如果还没覆盖所有边，从剩余边构成的子图中找一个与已走闭迹相交的顶点再走一条闭迹，合并即可。

20.2.2 哈密顿图的充分条件

与欧拉图不同，哈密顿图没有一个简单的充要条件。但竞赛中经常用到两个充分条件：

定理 20.4（Dirac定理，1952）

如果简单图 G 有 n ≥ 3 个顶点，且每个顶点的度数至少为 n/2，则 G 是哈密顿图（含有哈密顿回路）。

定理 20.5（Ore定理，1960）

如果简单图 G 有 n ≥ 3 个顶点，且对任意不相邻的顶点 u, v，有 deg(u) + deg(v) ≥ n，则 G 是哈密顿图。

注意，这两个条件是充分的但不是必要的——很多哈密顿图的顶点度数很小，仍然存在哈密顿回路。例如一个简单的 n-边形（Cn），每个顶点度数为2，但显然存在哈密顿回路（就是它本身）。

Dirac定理和Ore定理在竞赛中的应用场景通常是：题目给了你某些"度数条件"，让你证明存在一条经过所有顶点的回路。这时候就要立刻想到这两个定理。

20.3 竞赛图（Tournament）

20.3.1 竞赛图的定义

竞赛图（tournament）是完全有向图——每对不同的顶点之间恰好有一条有向边。它模拟了一个循环赛的赛果：每两名选手之间都比赛了一场，并且分出胜负（没有平局）。

竞赛图在组合竞赛中是一个极好的命题素材，因为它的性质非常丰富且优美。

20.3.2 Landau定理——得分序列定理

竞赛图中，每个顶点的出度（outdegree）表示该选手赢了多少场比赛。把所有出度从小到大排序得到序列 s1 ≤ s2 ≤ ... ≤ sn，称为这个竞赛图的得分序列（score sequence）。

定理 20.6（Landau定理，1953）

非递减非负整数序列 (s1, s2, ..., sn) 是某个 n 顶点竞赛图的得分序列当且仅当对每个 k = 1, 2, ..., n-1，有：

∑_{i=1}^{k} s_i ≥ C(k, 2)

且 ∑_{i=1}^{n} s_i = C(n, 2)。

解读这个条件：前 k 个得分之和至少等于 k 个顶点之间的内部比赛场数（因为内部的比赛总要有人赢）。等号仅在前 k 名选手之间互相全输给了后 n-k 名之类的情形下成立。

20.3.3 Redei定理——Hamilton路径

定理 20.7（Redei定理，1934）

每个竞赛图中都存在一条哈密顿路径——即经过所有顶点恰好一次的有向路径。

这个定理非常优美：完全有向图中，无论胜负关系多么混乱，你一定可以找到一条贯穿所有选手的"排名链"。也许A打败了B，B打败了C，但C打败了A——然而Redei定理告诉我们，仍然存在一种排列 v1, v2, ..., vn，使得 vi → vi+1（vi赢了vi+1）。

证明用归纳法：n=1,2时显然。假设对n-1顶点成立，考虑n顶点竞赛图。删除顶点v，由归纳假设剩下部分有哈密顿路径 P = v1 → v2 → ... → vn-1。如果v指向v1，则 v → v1 → v2 → ... → vn-1 即为所求。如果vn-1指向v，则同理。否则找到第一个满足 v → vi 的 i，此时 vi-1 → v（因为如果 vi-1 → v，那么i就不是第一个），从而 v1 → ... → vi-1 → v → vi → ... → vn-1 即为哈密顿路径。

20.3.4 竞赛图的其他常用性质

除了上述两个核心定理，竞赛图还有以下常用性质：

  • 传递竞赛图：如果竞赛图是传递的（即 a→b 且 b→c 蕴含 a→c），则它唯一对应一个全序排列，且得分序列为 (0, 1, 2, ..., n-1)。
  • King定理：每个竞赛图中都存在一个"King"——一个通过不超过两条有向边可以到达任何其他顶点的顶点。（想象一下，在社交网络中，你朋友的朋友几乎覆盖了所有人。）
  • 强连通性：强连通的竞赛图被称为"强竞赛图"，它一定包含哈密顿回路（不只路径，还有回路）。这也是一个常考的结论。

例2（真题改编）：n名选手进行循环赛（无平局）。证明：可以将他们排成一列 v1, v2, ..., vn，使得 vi 赢了 vi+1（即存在Hamilton路径）。

这正是Redei定理的直接结论。但考场上你不能只说"由Redei定理得"——因为定理本身可能不是教材中的标准内容，你需要把归纳法证明的过程写出来。这就是为什么要理解定理的证明而非仅仅记住结论。

20.4 本章总结

这一章的信息量很大，我们覆盖了四个重点专题：

  • 匹配与Hall定理——判断二分图是否存在完美匹配的利器，应用场景包括配对问题、拉丁方填充等
  • 欧拉图与哈密顿图——欧拉图有简单的充要条件（偶数度），哈密顿图只有充分条件（Dirac/Ore）
  • 竞赛图与Redei定理——完全有向图必然存在Hamilton路径
