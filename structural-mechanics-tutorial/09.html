引言：对称是结构工程师最好的朋友

同学们好，上一章我们掌握了力法的基本武器——确定超静定次数、建立力法方程、求解多余未知力。但如果你的结构有五六次甚至十几次超静定，你打算怎么用手算？答案是——你需要取巧。

在实际工程中，绝大部分重要结构都是对称的。国家大剧院的椭球壳、央视总部大楼的环状结构、几乎所有的高层建筑平面——你画一条中轴线，左右几乎就是镜像。对称不是设计师的审美癖好；对称带来结构受力均匀、减小扭转、方便施工——而且，对称还能让我们的力学分析工作量减少一半甚至更多。

我读研究生的时候，导师给我一个任务：分析一个7跨连续刚构桥在施工阶段的受力。我数了一下，超静定次数高达15次。如果用常规力法，列出15×15的柔度矩阵再手算求解——哪怕用计算器按，一个星期也搞不完。导师看了一眼，只说了一句话："你先把荷载分解成对称和反对称，各算各的。"分解完之后，对称部分7次、反对称部分8次——但每个子问题还可以继续利用对称性，最终每个子问题只需要解2-3个方程。不到半天就算完了。这件事教会我一个道理：对称性不是"锦上添花"的技巧，而是超静定结构分析的"核武器"。

本章我们接着力法的主题，深入探讨三个重要场景：对称性的利用、温度与支座移动的处理、以及超静定结构的位移校核。

9.1 对称结构的简化分析

对称结构的基本概念

所谓对称结构（Symmetric Structure），是指结构的几何形状、支座条件、截面刚度（EI、EA）均关于某轴对称。只有三个方面都对称，才能称为对称结构。

对于一个对称结构，任意荷载都可以分解为对称荷载（Symmetric Load）和反对称荷载（Antisymmetric Load）：

荷载 = 对称部分 + 反对称部分

对称荷载作用于对称结构时：结构的变形、弯矩图、轴力图是对称的，剪力图是反对称的。
反对称荷载作用于对称结构时：结构的变形、弯矩图、轴力图是反对称的，剪力图是对称的。

利用对称性取半结构

对称性利用最强大的一种方式就是取半结构（Half-structure Method）——只要荷载也是对称或反对称的，我们就可以只分析半个结构，在对称轴处施加适当的边界条件。

取半结构的规则如下：

荷载类型 对称轴处截面特性 半结构支座模拟 说明
对称荷载 无转角（θ=0）、有水平位移 定向支座（滑动支座） 对称轴截面只有竖向位移和水平位移，没有转动
反对称荷载 无竖向位移、有转角 链杆支座（竖向约束） 对称轴截面可以转动，但竖向位移为零

我们来看一个具体的例子：一个对称三跨连续梁，受对称均布荷载q作用。

原结构：三跨连续梁（对称）
  q ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
  ┌──────────────────────────────────┐
  │                                  │
  △        △              △        △
  A        B              C        D
  ├── L ──┼── L ──┼── L ──┼── L ──┤

由于结构对称、荷载对称：
  → 在中点C处，转角为零
  → 取半结构：AC跨，C端用定向支座模拟

半结构：
  q ↓↓↓↓↓↓↓↓↓↓↓↓
  ┌──────────────────┐
  │                  │
  △        △        ║
  A        B        C (定向支座)
  ├── L ──┼── L ──┤
  → 超静定次数从3次降为1次！
对称性利用的关键认识：取半结构后，超静定次数大幅降低。上面的例子从3次降为1次——力法方程从3个联立方程变成了1个一次方程。这不是"锦上添花"，而是把"不能手算"变成了"可以手算"。对于高次超静定结构，是否善于利用对称性，决定了你是"能解"还是"不能解"。

对称性的进一步利用：基本体系的对称化

即使无法取半结构（比如荷载不是严格对称或反对称），我们仍然可以通过对称性来简化力法方程。方法是：将多余未知力分组为对称组（Symmetric Group）和反对称组（Antisymmetric Group）。此时，柔度矩阵会呈现分块对角化的特点——对称组的系数与反对称组的系数互不影响（\(\delta_{ij} = 0\)，当i和j分属不同组时）。这相当于把一个大方程组分解为两个小方程组，计算量大大降低。

9.2 温度变化时的力法计算

力法的一个独特优势是：它可以直接处理温度变化引起的结构内力。这对于静定结构来说不会产生内力（只有变形），但对于超静定结构——温度变化会产生温度应力（Thermal Stress）。

在超静定结构中，多余约束限制了结构因温度变化而产生的自由变形，从而产生了内力。这就是为什么夏天的高铁轨道在无缝线路中会产生巨大的温度应力；也是为什么大跨度桥梁需要设置伸缩缝。

温度变化的力法方程

力法典型方程在温度作用下的形式与荷载作用完全相同，只是自由项 \(\Delta_{iP}\) 要替换为温度变化引起的位移 \(\Delta_{it}\)：

\(\delta_{ij} X_j + \Delta_{it} = 0\)

其中 \(\Delta_{it}\) 是基本体系中，仅由温度变化引起的沿 \(X_i\) 方向的位移。计算方法来自第7章的内容：

\(\Delta_{it} = \sum \alpha t_0 \int \bar{F}_N \mathrm{d}s + \sum \frac{\alpha \Delta t}{h} \int \bar{M} \mathrm{d}s\)

其中：

  • \(\alpha\)——材料的线膨胀系数（钢材约1.2×10⁻⁵/°C，混凝土约1.0×10⁻⁵/°C）
  • \(t_0\)——轴线处的温度变化值
  • \(\Delta t\)——截面上下缘的温度差
  • \(h\)——截面高度
  • \(\bar{F}_N\) 和 \(\bar{M}\)——单位未知力在基本体系中引起的轴力和弯矩
一个真实的工程教训：1998年，北京某大型体育馆的环形屋面在夏季施工合龙（温度约30°C），到冬季气温降至-10°C时，屋面出现了大面积裂缝。设计时计算了恒载、活载、风载，唯独低估了温度应力的影响——40°C的温差在超静定的环形钢桁架中产生了巨大的温度应力，远超预期。最终修复花费了数百万元。这个案例告诉我们：在超静定结构设计中，温度作用不可忽视，尤其对于大跨度、大体积结构。

9.3 支座移动时的力法计算

与温度变化类似，支座移动（Support Settlement）也是超静定结构产生内力的重要原因。对于静定结构，支座移动只产生刚体位移，不产生内力；但对于超静定结构，多余约束阻止了结构的自由位移，从而产生了附加内力。

支座移动的力法方程变为：

\(\delta_{ij} X_j + \Delta_{ic} = -\bar{R}_{ic} \cdot c\)

或者更标准的形式：

\(\delta_{ij} X_j + \Delta_{ic} = -c_i\)

其中 \(c_i\) 是原结构在第i个多余约束方向的已知位移（支座沉降量）。右边不再自动为零——它等于支座移动的实际值。

在求解过程中，\(\Delta_{ic}\) 表示基本体系中，实际支座移动引起的沿 \(X_i\) 方向的位移。它可以通过单位荷载法计算。

例：一端固定、一端铰支梁，中间支座发生沉降 Δ

  原结构：
  ┌──────────────────────┐
  │                      │
  ╞═══════════════△══════╡
 固定端            ↓Δ   活动铰

  1次超静定结构，中间支座沉降 Δ。

  基本体系：去掉中间支座，代以 X₁

  力法方程：
  δ₁₁ X₁ + Δ₁c = -Δ

  其中 Δ₁c 是实际支座移动在基本体系中
  引起的沿 X₁ 方向的位移。由于基本体系中
  已经去掉了这个支座，所以支座移动在
  基本体系中不引起任何内力——但会引起位移。

  通常我们取基本体系中没有任何约束沉降，
  则 Δ₁c = 0，方程简化为：

  δ₁₁ X₁ = -Δ
  X₁ = -Δ / δ₁₁
一个重要的工程应用：在软土地区（如上海、天津），高层建筑的不均匀沉降是设计中的核心问题。一座超静定框架结构，如果某个柱基沉降了2cm，会在上部结构中产生可观的附加弯矩。这就是为什么沉降观测是高层建筑施工中的强制性要求——通过监测数据，可以反向验证设计假定的正确性，必要时采取措施。

9.4 超静定结构的位移校核

当我们用力法解出了超静定结构的内力后，还需要进行位移校核（Displacement Check）。这不仅是为了验证计算结果的正确性，更重要的是——结构设计规范对变形有严格的限值要求（如《混凝土结构设计规范》规定：受弯构件的挠度不应超过l₀/250）。

位移计算的两种方法

方法一：利用虚功原理（单位荷载法）

这是通用方法。把已经求出的最终弯矩图M与单位荷载下的弯矩图相乘即可。但注意——在超静定结构中，单位荷载需要施加在原结构上（而不是基本体系上），因为我们需要的是原结构的真实变形。

\(\Delta = \int \frac{M \cdot \bar{M}}{EI} \mathrm{d}s + \int \frac{k F_Q \cdot \bar{F}_Q}{GA} \mathrm{d}s + \int \frac{F_N \cdot \bar{F}_N}{EA} \mathrm{d}s\)

方法二：利用力法计算结果直接计算

位移也可以由力法的柔度系数间接得到——但这个方法不如单位荷载法直观，不常用。

校核的基本原则

一个好的校核应该满足以下条件：

  • 平衡校核——最终内力图必须满足结点的静力平衡条件。取任何隔离体，\(\sum F_x = 0\)、\(\sum F_y = 0\)、\(\sum M = 0\)。
  • 变形协调校核——原结构中被去掉的约束处的位移应该等于零（或给定的支座移动值）。这是用力法求解时已经满足的条件，但可用于检查计算是否有误。
  • 对称性校核——如果结构和荷载都是对称的，内力图也应该满足相应的对称/反对称关系。若不对称，说明计算有误。
校核类型 检查什么 用的公式
平衡校核 结点是否满足力的平衡 \(\sum F = 0\)、\(\sum M = 0\)
变形协调校核 多余约束处位移是否为给定值 \(\delta_{ij}X_j + \Delta_{iP} = \) 已知位移
对称性校核 内力图形状是否合理 目视检查
一个实用的校核策略：我的经验是，每次用力法算完一个结构，先别急着往下走，花5分钟做三件事：（1）在关键结点处取一个隔离体，验算内力平衡；（2）用图乘法算一下"去掉约束处"的位移，检查是否满足边界条件；（3）如果结构对称，检查弯矩图的对称性。这三步能在90%的情况下帮你发现计算错误。要知道——在工程设计中，一个有数值错误但"看起来合理"的弯矩图，比一个明显有错的弯矩图更危险，因为它可能骗过审图人员的眼睛。

9.5 力法实用技巧总结

结合两章的内容，我们总结一下力法分析的实用要点：

  1. 宁少勿多——选择基本体系时，去掉的约束不要太多，只要能变成静定结构就行。去掉不必要的约束只会增加方程数量。
  2. 先看对称——计算之前先看结构和荷载是否对称。如果能取半结构，一定要取；即使不能，也要尽量分组利用对称性。
  3. 图乘优先——计算柔度系数时，能用图乘法就不要用积分——图乘法快一个数量级。
  4. 留意非荷载因素——温度变化和支座移动在超静定结构中会产生内力，不能忽略。
  5. 校核不可少——每解完一个结构，至少做一次平衡校核。

本章小结与下章预告

本章我们深入学习了力法的三个进阶话题：

  • 对称结构的简化分析——取半结构法和对称/反对称分组法，这是降低超静定次数的利器；
  • 温度变化时的力法计算——温度应力的产生机理和计算框架；
  • 支座移动时的力法计算——基础沉降对超静定结构的影响分析；
  • 超静定结构的位移校核——平衡校核、变形协调校核和对称性校核。

力法的学习到此基本完成。但力法有一个"天生"的局限：超静定次数越高，计算量越大。下一章我们将学习另一种完全不同的方法——位移法（Displacement Method）。与力法"以力为未知量"的思路相反，位移法"以位移为基本未知量"。你会看到，这两种方法就像一对"双生子"——一个从力出发，一个从位移出发，最终到达同一目的地。而位移法的思路，对我们后面学习矩阵位移法和有限元法至关重要。下一章第10章：位移法（一），我们不见不散。
