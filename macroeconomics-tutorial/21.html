第21章 经济增长：索洛模型

1953年，一位年轻的美国经济学家搭乘火车穿越印度次大陆。窗外是绵延不绝的贫穷村庄：赤脚的孩子在尘土中玩耍，骨瘦如柴的农民在烈日下劳作，牛车在泥土路上缓慢前行。这个场景和繁华的西方世界形成了惊人的对比。这位经济学家就是罗伯特·索洛（Robert Solow）。他在这次旅途中受到震撼，决心回答一个经济学中最根本的问题：为什么有些国家富裕，有些国家贫穷？三年后，他发表了一篇改变经济增长研究方向的论文——《对经济增长理论的贡献》，并因此在1987年获得了诺贝尔经济学奖。

大家好，欢迎来到第21章经济增长：索洛模型。在前面的二十章中，我们讨论的主要是经济短期波动——通货膨胀、失业、货币政策、财政政策等。但从这一章开始，我们把时间尺度拉长到几十年甚至上百年，来讨论一个更根本的问题：一个国家的经济规模和生活水平，在长期中是如何决定的？增长的源泉到底是什么？为什么有些国家能在几十年间从一贫如洗变成发达国家（韩国、新加坡），而另一些国家却停滞不前？现代经济增长理论的开端，就是索洛模型。

1. 为什么有些国家富、有些国家穷

先看几组数据，感受一下国家间贫富差距的"尺度"：

  • 卢森堡的人均GDP（按购买力平价计算）超过12万美元/年。
  • 美国大约在7.6万美元/年。
  • 中国约1.9万美元/年（按PPP计算，约1.2万美元按名义汇率）。
  • 印度约8000美元/年。
  • 布隆迪（非洲最穷国家之一）约800美元/年。

从卢森堡到布隆迪，差距超过100倍。这意味着一个卢森堡新生儿一生的收入水平，相当于100个布隆迪新生儿的总和。而这种差距不是天生的——在200年前工业革命之前，所有国家的人均收入大体相当（大约都在1000-2000美元/年，按今天的美元计）。差距是过去200年经济增长的"分化"造成的。

2. 索洛模型的核心机制

索洛模型从一个简单的总量生产函数开始：Y = F(K, L)。它假设一个国家的总产出Y取决于两个投入要素：资本存量K（机器、厂房、基础设施）和劳动力L（就业人数）。为了方便分析，索洛把它转换成了人均形式：

基本生产函数（人均形式）：
  y = f(k)

其中：
  y = Y/L = 人均产出
  k = K/L = 人均资本（也叫"资本-劳动比"）

假设生产函数满足"稻田条件"（Inada Conditions）：
  · f(0) = 0（没有资本就没有产出）
  · f'(k) > 0（资本越多，产出越多）
  · f''(k) < 0（资本的边际产出递减——资本越多，每单位新增资本的贡献越小）

经典的柯布-道格拉斯形式：y = k^α（其中0 < α < 1，通常α ≈ 1/3）

现在问一个关键问题：人均资本k是如何随时间变化的？

资本存量会因两件事而变动：

（1）投资增加资本。人们将一部分收入储蓄起来，储蓄转化为投资，投资增加了资本存量。人均投资 = s × y = s × f(k)，其中s是储蓄率（0到1之间）。

（2）折旧减少资本。机器会老化、厂房会磨损。每年有固定比例δ的资本折旧（报废）。人均折旧 = δ × k。

因此，人均资本的净变化 = 投资 - 折旧：

Δk = s·f(k) - δ·k

这就是索洛模型的"核心方程"：
  · 如果 s·f(k) > δ·k：k在增加（资本深化）
  · 如果 s·f(k) < δ·k：k在减少（资本浅化）
  · 如果 s·f(k) = δ·k：k不变——达到稳态！
理解"稳态"（Steady State）

这是索洛模型中最关键的概念。稳态不是"停止不动"，而是"人均指标不再变化"。在稳态下，k = k*（稳态人均资本），y = f(k*)是稳态人均产出。

如果经济初始时k低于k*，那么Δk > 0，经济在"追赶"稳态。在追赶过程中，人均产出增长较快（因为资本深化）。但越接近稳态，增长速度越慢（因为边际报酬递减）。

一旦达到稳态，总产出Y仍然随着劳动力数量L的增长而增长（比如说每年增长1%），但人均产出y不再增长。这意味着——只靠资本积累，不能带来人均收入的永久增长。

3. 索洛模型的图解

让我们用图形来理解索洛模型的核心逻辑。虽然本章是文字版，但我们可以用表格来模拟图形中的关系：

1 1.00 0.20 0.10 +0.10 资本深化

从这个模拟可以看出两个重要的结论：

结论一：高储蓄率国家更富裕。假设另一个国家A的储蓄率是s=0.3（比我们的s=0.2高）。它的稳态人均资本更高，稳态人均产出也更高。所以高储蓄率确实能带来更高的生活水平。这解释了为什么东亚国家（中国、新加坡、韩国）的高储蓄率是其经济崛起的重要原因。

结论二：但高储蓄率不能带来永续增长。当经济达到新稳态后，人均产出就不增长了。要想人均产出持续增长，必须依靠技术进步——这是索洛模型最重要的结论。

"资本积累的局限性"——一个直观的理解

想象一个农村家庭，只有一把锄头（资本）。多买一把锄头，两个人可以轮流用，产出增加很明显（边际产出高）。再加一把锄头——第三个人也能干活了。但当你有一百把锄头，家庭成员每人只有一双手、一双脚，再多锄头也没人用了——每多一把锄头的贡献趋近于零。

一个国家也是同样的道理：你可以不停地建工厂、买机器，但工人一天只有24小时、只有这么多双手。当资本相对于劳动已经足够丰富时，继续增加资本的回报会越来越低。要想持续提高人均产出，必须让同样数量的劳动变得更"高效"——这就是技术进步和人力资本的作用。

4. 技术进步——增长的终极来源

既然资本积累不能带来永续增长，那么增长的真正动力是什么？索洛模型的答案是技术进步（Technological Progress）。把技术进步引入生产函数后，模型变成了：

Y = A · F(K, L)

其中A是"全要素生产率"（Total Factor Productivity, TFP），也叫"索洛残差"。

在"有效劳动"形式下：
  y_eff = f(k_eff)

其中：
  · y_eff = Y/(A·L) ——每单位"有效劳动"的产出
  · k_eff = K/(A·L) ——每单位"有效劳动"的资本
  · A以固定速率g增长：ΔA/A = g（技术进步率）

在稳态下，k_eff和y_eff不变，但：
  · 人均产出y = Y/L = A · y_eff → 以速率g增长
  · 总产出Y → 以速率(g + n)增长（n是人口增长率）

核心结论：长期人均增长的唯一来源是技术进步。

但这里有一个"吊诡"的地方：在索洛模型中，技术进步是外生的——它被假设以固定的速率g"从天而降"，模型本身不解释技术进步的来源。这就是后面"内生增长理论"要解决的问题（第22章）。

索洛残差（Solow Residual）是一个极其重要的概念。它指的是"经济增长中不能被资本和劳动投入的增长所解释的部分"——也就是说，它是经济增长的"余值"。计算方法是：

ΔY/Y = α·ΔK/K + (1-α)·ΔL/L + ΔA/A

其中ΔA/A就是索洛残差。如果我们知道GDP增长了多少、资本和劳动增长了多少，扣掉后剩下的就是TFP增长——大致可以理解为"技术进步+制度改善+效率提升"的综合效果。

下面我们用索洛残差的方法来核算中国1980-2020年的经济增长：

时期 GDP年均增速 资本贡献 劳动贡献 TFP（索洛残差） TFP占比
1980-1990 9.3% 4.0% 2.5% 2.8% 30%
1990-2000 10.5% 5.0% 1.5% 4.0% 38%
2000-2010 10.8% 6.0% 1.0% 3.8% 35%
2010-2020 6.8% 4.5% 0.3% 2.0% 29%
1980-2020 9.3% 4.9% 1.3% 3.1% 33%

数据来源：基于Wu(2015)、Penn World Table等估算。这里的"资本贡献"中包含了资本深化和资本存量的增长，其中大量依赖固定资产投资，"劳动贡献"中包含了劳动力数量增长和人力资本改善，"TFP"包括了市场化改革、技术进步、对外开放红利和效率改善。

从中国的索洛分解中学到什么？

· 中国经济增长的三分之一左右来自TFP（技术进步+制度红利），三分之二来自资本和劳动投入的增长。
· 资本贡献是中国增长的最大引擎——高储蓄率+高投资率是中国模式的核心特征。
· 但劳动贡献在持续下降（人口老龄化、劳动力增长放缓），这是所谓的"人口红利消退"。
· 2010年之后TFP贡献也在下降——这可能意味着改革红利在减弱，技术追赶的空间在缩小。

一个令人担忧的趋势：如果TFP增长持续低迷（2%以下），同时劳动人口负增长，即使维持高投资率，中国GDP增速也可能回落到3-4%左右的水平。这就是为什么提出"高质量发展"——从要素驱动转向创新驱动。

5. 趋同与分化

索洛模型有一个重要预测：穷国应该比富国增长得更快，因为穷国的资本存量低（k小），资本的边际产出高，每一单位新增投资带来的增长效果更大。如果各国在其他条件（储蓄率、人口增长率、技术进步率）相同的情况下，穷国会"追赶"富国，最终所有国家的人均收入会趋同。这就是条件趋同（Conditional Convergence）。

但在现实世界中，情况要复杂得多：

绝对趋同的失败。如果我们只看最穷和最富国家组——非洲和欧洲——我们没有看到趋同。1950年，非洲的人均收入大约是欧洲的10%，70年后，这个比例不但没有提高，反而略有下降。这说明"穷国自动追赶富国"在全世界范围内是不成立的。

条件趋同的证据。但如果我们把国家按照储蓄率、人口增长率、制度质量等因素分组，在"条件相同"的组内，确实发现了趋同的证据。也就是说：一个国家的稳态水平是由其结构性特征决定的，低于自己稳态的国家会增长得更快，但低于别国稳态不等于低于所有国家的平均水平。

"俱乐部趋同"。一些实证研究发现，世界经济增长呈现"俱乐部趋同"（Club Convergence）的特征——国家分成了几个"俱乐部"（富裕俱乐部、中等俱乐部、贫困俱乐部），每个俱乐部内部的趋同是明显的，但俱乐部之间的差距在扩大而不是缩小。这更接近现实世界的图景。

东亚奇迹：有条件的趋同故事

1960年，韩国的人均GDP与加纳大致相当（不到1000美元）。两国都是刚刚独立不久、以农业为主、资源禀赋相似。但随后的50年，两个国家走出了完全不同的发展轨迹：

· 韩国：高储蓄率（s>30%）、大力投资教育、出口导向、政局相对稳定、制度逐步改善 → 年均增长6%以上 → 2010年人均GDP超过2万美元
· 加纳：政治不稳、制度薄弱、储蓄率低、贸易保护、投资不足 → 年均增长不到2% → 2010年人均GDP不到2000美元

这充分说明了"条件"的重要性。索洛模型说的"趋同"是有条件的——需要具备相似的储蓄率、人口增长率、制度质量、教育水平等前提条件。如果一个穷国制度腐败、内战频发、储蓄率极低，它不但不会趋同，反而可能进一步掉队。

"条件趋同"的政策含义是：想让穷国变富，光给钱是不够的。需要帮助它们改善制度、增加教育投入、提高储蓄率和投资效率——相当于把它们的"稳态"不断提升。
使用索洛模型时要注意的"坑"：索洛模型虽然是增长理论的基石，但它有重要的局限性。最核心的一点是：它把技术进步当作"上帝赐予的礼物"（外生的），不解释技术进步本身是怎么来的。对于中国这样的国家来说，改革开放前三十年的技术进步很大程度上是"引进、消化、吸收、再创新"——购买国外先进设备、学习国外技术和管理经验。这在索洛模型中表现为TFP的增长，但模型本身没法告诉你：如何系统性地促进创新？为什么有的国家创新能力强而有的不行？这些问题需要"内生增长理论"来回答——这正是我们下一章的主题。

好，第21章就到这里。我们从"为什么国家贫富差距如此之大"这个问题出发，系统学习了索洛模型的三大核心逻辑：资本积累驱动增长但边际递减、储蓄率决定稳态水平但不决定长期增长率、技术进步才是长期人均增长的唯一源泉。我们还用索洛残差的方法核算了中国经济增长的来源，并讨论了"趋同"的不同形态。

下一章，我们将走出索洛模型的框架，进入内生增长理论——不再把技术进步当作"外生变量"，而是从经济体系内部解释技术进步和创新是如何产生的。我们还会讨论人力资本、制度质量和"中等收入陷阱"等更贴近现实的话题。
