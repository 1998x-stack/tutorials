引言：不用解方程的方法

同学们好，我们前面学了两大类方法：力法和位移法。它们都可以精确求解超静定结构，但都有一个共同的"痛点"——需要解联立方程。对于1-2次超静定或1-2个位移自由度，手算没问题。但自由度一多（比如4个以上），手算解方程本身就成了一个不小的工程。

有没有一种方法——不需要解联立方程，只需要做一系列简单的算术运算，就能逐步逼近精确解？答案是肯定的。这就是由美国结构工程师哈迪·克罗斯（Hardy Cross）在1932年提出的力矩分配法（Moment Distribution Method），也被称为"Cross Method"。

关于Hardy Cross本人有一个很有意思的故事。Cross是一位非常低调的教授，他在伊利诺伊大学香槟分校教书。他发明的力矩分配法最初发表在一篇并不起眼的论文中。但在接下来的30年里，这个方法成为了全球结构工程师手算连续梁和刚架的标准工具——因为它太实用了：你不需要列出任何方程组，只需要在结点上反复做"分配"和"传递"两个操作，结果就会越来越精确。直到今天，虽然计算机已经普及，力矩分配法仍然是每个结构工程师应该理解的基本技能——它培养你对弯矩如何"流动"的结构直觉。

2010年，我参与了一个旧桥加固项目。那座桥是一座5跨连续梁桥，我们需要在桥下增加临时支撑，然后在支撑点上顶升梁体进行支座更换。总工要求我们用两种方法独立计算顶升力：一种是用SAP2000建模分析，另一种就是手算——用力矩分配法。"计算机可能会骗人，"他说，"但力矩分配法每一步都写在纸上，错了就看得出来。"两种方法的计算结果对比如预期一样吻合，但总工在手算过程中发现了一个SAP2000模型中边界条件设置错误——一个支座被错误地设成了固定端。力矩分配法救了我们的设计一次。这个经历让我深刻地认识到：力矩分配法不是一个过时的古董——它是理解结构内力分布的"透视镜"。

12.1 力矩分配法的基本概念

力矩分配法本质上是一种迭代的位移法（Iterative Displacement Method）。它的核心思想是：

  • 先把所有结点"锁住"（施加附加刚臂），不让它们转动；
  • 计算荷载作用下的固端弯矩，以及结点上的"不平衡力矩"；
  • 然后"放松"一个结点，让不平衡力矩按各杆的"转动刚度"比例分配给相邻杆件；
  • 分配后的力矩还会"传递"到杆件的另一端；
  • 一次放松结束后再锁住这个结点，去放松下一个结点；
  • 重复这个过程，直到所有结点的不平衡力矩小到可以忽略。

力矩分配法只适用于无侧移结构（连续梁和无侧移刚架）。对于有侧移的结构，力矩分配法需要结合侧移修正——这就变成了"弯矩分配法"的高级版，超出了本科范围。

力矩分配法的三个关键参数：（1）转动刚度（Stiffness Factor）——每个杆件抵抗结点转动的能力；（2）分配系数（Distribution Factor）——结点上各杆分摊不平衡力矩的比例；（3）传递系数（Carry-over Factor）——杆件一端的力矩传递到另一端的比例。记住这三个参数，你就掌握了力矩分配法的"三板斧"。

12.2 转动刚度与分配系数

转动刚度（Stiffness Factor）

转动刚度 \(S_{AB}\) 的定义是：当A端产生单位转角时，在A端需要施加的弯矩大小。它反映了杆件对结点转动的"抵抗"程度。

不同远端约束条件下的转动刚度：

远端约束 转动刚度 \(S_{AB}\) 说明
远端固定 \(4i\) 标准刚接框架中最常见
远端铰支 \(3i\) 远端为铰接时
远端滑动（定向） \(i\) 远端为定向支座时
远端自由 \(0\) 悬臂端

其中 \(i = EI/l\) 称为杆件的线刚度（Linear Stiffness）。

分配系数（Distribution Factor）

当一个结点上有n根杆件交汇，结点处的不平衡力矩 \(M_{unbal}\) 会按照各杆转动刚度的比例分配到每根杆件上。

\[ \mu_{Aj} = \frac{S_{Aj}}{\sum S_{Ak}} \]

其中 \(\mu_{Aj}\) 是杆件Aj在A结点上的分配系数。分配到杆件Aj近端（A端）的弯矩为：

\(M_{Aj}^{dist} = \mu_{Aj} \times (-M_{unbal})\)

注意分配的是不平衡力矩的相反数——因为放松结点的目的是消除不平衡力矩，所以要加一个与之相反的力矩。

例：一个三杆交汇的刚结点

            C
            │
            │ i₂ = 2
     ───────┤
   B  i₁=3  │
            │
            │
            D  i₃ = 1

  各杆线刚度不同 → 转动刚度不同
  假设所有远端固定：

  S_{AB} = 4 × 3 = 12
  S_{AC} = 4 × 2 = 8
  S_{AD} = 4 × 1 = 4
  ΣS = 24

  分配系数：
  μ_{AB} = 12/24 = 0.5
  μ_{AC} = 8/24 = 0.333
  μ_{AD} = 4/24 = 0.167
  校核：0.5 + 0.333 + 0.167 = 1.0 ✓
直觉理解：分配系数就像是"分蛋糕"时各人拿到的比例——线刚度越大的杆件越"僵硬"，越能"抢"到更多的不平衡力矩。想想看：你在一个弹簧群中转动一个结点——弹簧越硬，你转动它需要的力就越大，因此它"分摊"到的弯矩也就越大。这很合理，对吗？

12.3 传递系数

在力矩分配法中，当我们把分配弯矩加到杆件的A端时，这个弯矩的一部分会"传递"到杆件的B端。传递系数（Carry-over Factor）\(C_{AB}\) 定义为：

\(C_{AB} = \frac{M_{BA}}{M_{AB}}\)

不同远端约束的传递系数：

远端约束 传递系数 \(C\) 说明
远端固定 1/2 一半的分配弯矩传到远端
远端铰支 0 铰支座不能传递弯矩
远端滑动 -1 传递后改变符号

对于等截面直杆，远端固定时传递系数恒为1/2。这个"一半"的数值来自于转角位移方程——当A端有单位转角而B端固定时，B端会产生A端弯矩一半的弯矩。

12.4 单结点分配

单结点分配是力矩分配法的基础——结构中只有一个可以转动的刚结点。这是最简单的场景，只需要一次分配和一次传递就能得到精确解。

单结点分配示例：两跨连续梁

   q = 20 kN/m
  ┌──────────────────────┬──────────┐
  │                      │          │
  △──────────────────────┼──────────△
  A                      B    P=50kN C
  ────── 6m ──────├── 4m ──┤

  AB跨：均布荷载 q = 20 kN/m，l = 6m
  BC跨：集中荷载 P = 50 kN（距B端4m），l = 4m

  第一步：计算固端弯矩
  M_{AB}^F = -ql²/12 = -20×36/12 = -60 kN·m
  M_{BA}^F = +ql²/12 = +60 kN·m
  M_{BC}^F = -Pa²b/l² ... 计算得 -37.5 kN·m
  M_{CB}^F = +Pab²/l² ... 计算得 -12.5 kN·m

  第二步：计算分配系数（结点B）
  假设EI相同，i_{BA}=EI/6，i_{BC}=EI/4
  S_{BA} = 4i_{BA} = 4EI/6 = 0.667EI
  S_{BC} = 4i_{BC} = 4EI/4 = 1.0EI
  μ_{BA} = 0.667/(0.667+1.0) = 0.4
  μ_{BC} = 1.0/(0.667+1.0) = 0.6

  第三步：结点B的不平衡力矩
  M_{B,unbal} = M_{BA}^F + M_{BC}^F = 60 + (-37.5) = 22.5 kN·m

  第四步：分配（放松结点B）
  M_{BA}^{dist} = μ_{BA} × (-22.5) = -9 kN·m
  M_{BC}^{dist} = μ_{BC} × (-22.5) = -13.5 kN·m

  第五步：传递到远端
  M_{AB}^{carry} = -9 × 0.5 = -4.5 kN·m
  M_{CB}^{carry} = -13.5 × 0.5 = -6.75 kN·m

  第六步：最终杆端弯矩
  M_{AB} = -60 + (-4.5) = -64.5 kN·m
  M_{BA} = 60 + (-9) = 51 kN·m
  M_{BC} = -37.5 + (-13.5) = -51 kN·m
  M_{CB} = -12.5 + (-6.75) = -19.25 kN·m

  校核：M_{BA} + M_{BC} = 51 - 51 = 0 ✓ 结点B平衡了！
初学者容易忽略的一个细节：放松结点时分配的力矩是"不平衡力矩的相反数"——而不是不平衡力矩本身。通俗地说：结点上现有的弯矩和不平衡，你要加一个"反方向"的力矩把它掰回到平衡位置。如果忘了这个负号，你的分配力矩恰好会把不平衡力矩翻倍，结果越来越发散而不是收敛。

12.5 多结点逐次分配

实际工程中的连续梁通常有多个中间支座——也就是多个结点需要放松。多结点分配不能一次完成，需要逐次迭代（Successive Approximation）。

多结点分配的基本流程

  1. 锁定所有结点，计算各跨的固端弯矩；
  2. 放松一个结点（比如从左边开始），分配并传递；然后重新锁住该结点；
  3. 放松下一个结点，分配并传递；再锁住；
  4. 重复上述过程，依次轮流放松所有结点；
  5. 每一轮分配后，结点上的不平衡力矩都会减小；
  6. 当所有结点上的不平衡力矩都小到可以忽略时（通常2-3轮后），停止迭代；
  7. 把所有轮的分配值、传递值和初始固端弯矩叠加，得到最终弯矩。

一个关键观察：当你放松一个结点时，分配力矩会传递到相邻结点——这就破坏了相邻结点的平衡。所以你需要反复"放松-分配-锁住-下一个"的循环，直到所有结点都接近平衡。

收敛速度：力矩分配法的收敛速度非常快。这是因为传递系数是1/2——每经过一个结点，传递过去的不平衡力矩就减半。一般情况下，做3轮分配后，结果就已经非常接近精确解了（误差通常在1%以内）。这就是为什么在手算时代，力矩分配法如此受欢迎——你花15分钟做3轮手算，精确度就足以满足工程设计的要求。 
多结点迭代中的关键观察：

  第1轮（放松结点B）：
    → B的不平衡力矩分配后，
      C会收到从B传递过来的额外弯矩
    → C现在有了新的不平衡力矩

  第2轮（放松结点C）：
    → C的不平衡力矩分配后，
      B会收到从C传递回来的额外弯矩
    → B又有了新的小量不平衡力矩

  第3轮（再次放松B）：
    → 这次B的不平衡力矩已经比上一轮小很多
    → 传递到C的就更小了

  收敛特性：
  |ΔM^{(n+1)}| ≈ 0.5 × |ΔM^{(n)}|
  逐次减半，3轮后不到初始值的1/8
实用的手算表格格式：在我的学生时代，老师教我们用一种"上-下-箭头"的表格来做力矩分配——上面写分配系数，中间写各轮分配值，箭头表示传递方向。我至今觉得这种表格是最清晰的手算组织方式。建议你在做习题时也采用这种格式——它能让你清晰地追踪每一轮分配，减少错误。

12.6 力矩分配法与位移法的联系

力矩分配法并不是一种"独立"的方法——它实际上是位移法的迭代形式。

在位移法中，我们列出所有结点的力矩平衡方程，然后一次性求解联立方程组。在力矩分配法中，我们每次只放松一个结点——相当于"高斯-赛德尔迭代法（Gauss-Seidel Method）"在结构力学中的物理实现。

具体来说：

  • 位移法的未知量是结点转角；力矩分配法中的"分配"操作，本质上就是在调整结点转角。
  • 位移法方程组的系数矩阵是对角占优的，所以高斯-赛德尔迭代收敛很快——这从数学上解释了为什么力矩分配法收敛迅速。
  • 如果有计算机，你用位移法直接求解就行了；如果手算，力矩分配法避免了联立方程的求解过程。
力矩分配法的现代启示：Hardy Cross在1932年提出的这个方法，在今天看来依然有重要的教育意义。在计算机时代，我们很容易陷入"一键出结果"的思维模式——点击"运行"，然后就相信屏幕上显示的应力云图。但力矩分配法强迫你"看见"内力传递的每一步：力矩如何从一跨传递到另一跨，如何逐次衰减。培养这种直觉，正是结构力学教育的核心目标之一。

本章小结与下章预告

本章我们学习了力矩分配法的完整体系：

  • 基本概念——力矩分配法是一种渐进的迭代法，不需要解联立方程；
  • 转动刚度与分配系数——\(S = 4i\)（远端固定）、\(S = 3i\)（远端铰支），分配系数是各杆转动刚度的比例；
  • 传递系数——远端固定时C=1/2；
  • 单结点分配——一次分配加传递即可得精确解；
  • 多结点逐次分配——需要2-3轮迭代收敛到工程精度；
  • 与位移法的联系——力矩分配法是位移法的高斯-赛德尔迭代实现。

力矩分配法是手算时代的巅峰。但从下一章开始，我们将进入结构力学的另一个世界——矩阵位移法（Matrix Displacement Method）。这是位移法的计算机化版本，也是现代有限元分析软件的底层算法。你会发现，有了位移法的基础，矩阵位移法的概念水到渠成——只是把"手算解方程"变成了"让计算机解矩阵方程"。下一章第13章：矩阵位移法（一），我们将学习单元刚度矩阵的建立、坐标变换和整体刚度矩阵的集成——这是有限元分析的"原始基因"。准备好进入计算力学的大门了吗？我们下一章见。
