3.1 机器学习就是"不确定性"的艺术

如果你写过传统软件，你习惯了确定性：输入A，永远得到B。但机器学习不同——同一个模型对同一张图，可能这次的预测是"猫"（置信度87%），下次还是"猫"但是82%。ML的本质是处理不确定性。概率统计就是用来量化这种不确定性的数学语言。

讲个真实例子。我之前做一个二分类模型预测用户是否流失，模型输出的概率是0.6。产品经理问："所以这个用户会流失？"我说："60%的概率会。"他很不理解——能不能给个确定的答案？这就是ML与传统编程在思维上的根本区别：我们输出的是概率分布，不是确定答案。

3.2 随机变量与概率分布

随机变量（Random Variable）就是把随机事件的结果映射为数值。比如抛硬币，正面→1，反面→0。在ML中，模型的输出、误差项、先验参数，都是随机变量。

分布类型特点ML中的应用
伯努利分布二元结果（0或1）二分类问题的标签
二项分布n次伯努利试验成功次数测试集上正确预测的个数
高斯（正态）分布钟形曲线，自然界最常见线性回归的误差项、权重先验
多项式分布多类别单次试验多分类softmax输出
泊松分布单位时间内事件发生次数用户点击次数、到达率建模

高斯分布尤其重要。很多ML算法（线性回归、高斯朴素贝叶斯、GMM）都假设数据服从高斯分布——这被称为"正态性假设"。使用前记得检查一下你的数据是否符合。

⚠️ 经典踩坑：直接用原始数据训练而不看分布，是新手最常犯的错误。如果特征分布严重偏态（比如收入数据，少数人收入极高），很多基于正态假设的模型（如线性回归的p值、置信区间）会给出误导性结论。正确做法是先用 sns.histplot() 看一下分布，必要时做 log 变换或 Box-Cox 变换。

3.3 条件概率与贝叶斯定理

条件概率 P(A|B) 表示"在B已经发生的条件下，A发生的概率"。贝叶斯定理就是条件概率的一个恒等式，但它深刻改变了我们对"学习"这件事的理解：

P(θ|D) = P(D|θ) × P(θ) / P(D)

翻译成人话：后验 ∝ 似然 × 先验。在见到数据D之前，你对参数θ有一个"先验信念"P(θ)；看到数据之后，你更新了这个信念，得到"后验"P(θ|D)。这就是贝叶斯学派的核心思想——学习就是不断更新信念的过程。

朴素贝叶斯分类器（第12章）就建立在这个定理之上。虽然它的"特征独立"假设在现实中几乎不成立，但在文本分类（垃圾邮件过滤）中效果好得惊人——这也是ML中有趣的一点：一个"理论上错误"的模型在实践中可能非常好用。

3.4 最大似然估计 MLE 与最大后验估计 MAP

大多数ML模型的训练目标都是"找到一组参数，使得在这些参数下，观测到当前数据的概率最大"——这就是最大似然估计（MLE）。

  • MLE：argmax P(D|θ) —— 只相信数据，不看先验
  • MAP：argmax P(θ|D) ∝ P(D|θ)P(θ) —— 在数据基础上融入先验知识

MAP相当于MLE加了一个"正则化项"——你熟悉的L2正则化（Ridge）本质上就是假设参数服从高斯先验的MAP估计！这就是为什么数学很重要——理解了推导，你会发现L2和L1正则化不是随便加上的"技巧"，而是有深刻概率含义的必然结果。

3.5 统计检验在特征选择中的应用

做特征工程时，我们经常需要回答："这个特征和目标变量真的有关系吗？还是只是随机波动？"这时候就需要统计检验了。

检验方法适用场景零假设 H₀
卡方检验两个类别变量是否独立两个变量独立
t检验两组均值是否有显著差异两组均值相等
ANOVA多组均值是否有差异所有组均值相等
互信息任意两个变量的依赖程度（无假设检验，纯信息论）
from sklearn.feature_selection import chi2, mutual_info_classif
from scipy.stats import f_oneway

# 方法1：卡方检验（适用于非负特征 + 分类目标）
chi2_scores, p_values = chi2(X, y)
# p值 < 0.05 的特征被认为与目标显著相关

# 方法2：互信息（能捕捉非线性关系）
mi_scores = mutual_info_classif(X, y)
# 互信息 > 0 说明特征和目标之间存在任意形式的依赖

# 方法3：ANOVA F检验
f_scores, p_values = f_oneway(*[X[y==c, i] for c in np.unique(y)])
💡 实践建议：做特征选择时，不要只看p值。p值 < 0.05 只说明"这个关系不太可能是随机产生的"，不代表"这个关系很强也很重要"。一定要结合效应量（effect size）和业务理解来做判断。在大数据集上，p值几乎永远是显著的——因为样本量大了什么都能显著。这时候更应该关注特征的重要性排名和实际的预测贡献。

3.6 本章小结

概率统计在ML中的核心作用是：描述不确定性、量化证据强度、做出信息最优的决策。理解贝叶斯定理的直觉、知道什么时候用什么分布、会用统计检验辅助特征选择——这三样东西你从本章带走就足够了。下一章我们进入优化理论，看看模型到底是怎么"学习"的。
