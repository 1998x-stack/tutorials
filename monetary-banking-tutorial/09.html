银行面临的四大风险——银行人的"生死劫"

上一章咱们讲了商业银行怎么赚钱。但你有没有想过一个问题：银行的钱是别人的（存款），却用来做高风险的事情（放贷、投资），如果出了问题怎么办？

银行经营的本质是在风险和收益之间走钢丝。咱们先来看看银行面临的最核心的四大风险，这是每个银行从业人员入职培训第一堂课就会讲的内容：

1. 信用风险——借钱不还的风险

这可能是最直观的风险。银行把钱借给企业或个人，对方到期还不上钱了，银行的贷款就变成了不良贷款。信用风险是银行最古老、最基础的风险。咱们常说"银行的命根子是信贷质量"，一点不夸张。衡量信用风险的指标包括不良贷款率（NPL ratio）、拨备覆盖率等。

2. 利率风险——利差变薄的风险

银行的资产负债期限天然不匹配——短期借款、长期贷款。当利率发生变化时，资产和负债的利率调整幅度不一致，银行的净息差（NIM）就会波动。2022-2023年美联储暴力加息，很多美国中小银行的NIM急剧恶化，硅谷银行的倒闭就是利率风险的极端体现。

3. 流动性风险——钱不够还的风险

银行最怕什么？挤兑。大量存款人同时来取钱。银行手里没有足够的现金满足提款需求，被迫低价变卖资产，引发更大恐慌。流动性风险往往不是孤立出现的——它通常由信用风险或利率风险触发，然后像多米诺骨牌一样连锁反应。

4. 操作风险——"人祸"的风险

巴塞尔协议对操作风险的定义是"由不完善或有问题的内部程序、人员、系统或外部事件所造成损失的风险"。翻译成人话就是：员工犯错、系统故障、欺诈、法律纠纷等。2012年瑞银的"流氓交易员"事件导致23亿美元亏损，就是操作风险的典型案例。

💡 风险之间的关系： 这四大风险不是孤立的。信用风险恶化→借款人还不上钱→银行流动性紧张→被迫高息借钱→NIM恶化（利率风险）→恐慌蔓延→挤兑（流动性风险恶化）→更多资产贱卖→信用风险和利率风险进一步恶化。这就是为什么银行风险需要系统性监管而不是单点管理。

巴塞尔协议体系——全球银行业的"宪法"

你可能听说过"巴塞尔协议"这个词。它不是什么建议或指南，而是全球银行业监管的基石。巴塞尔委员会（总部设在瑞士巴塞尔）从1974年成立以来，陆续推出了三代协议，每一代都在刷新全球银行业的风控标准。

Basel I（1988年）——从"一刀切"开始

核心要求：银行资本充足率（资本/风险加权资产）不低于8%。这是一个划时代的里程碑——它让全球第一次有了统一的银行资本标准。但Basel I对风险的区分太粗糙了：所有企业贷款不管风险高低，风险权重都是100%。这显然不合理。

Basel II（2004年）——"三大支柱"框架

Basel II建立起一个更精巧的框架，包含三大支柱：第一支柱：最低资本要求（细化了信用风险、市场风险、操作风险的计量方法）；第二支柱：监管检查（监管机构有权要求银行持有更多资本）；第三支柱：市场纪律（要求银行披露风险信息，让市场来监督）。

Basel III（2010年至今）——金融危机后的"亡羊补牢"

2008年金融危机暴露了Basel II的严重不足。Basel III应运而生，核心思路就是：多、更高、更严。咱们来看一张表：

资本类型 Basel II要求 Basel III要求 变化
核心一级资本充足率（CET1） 2% 4.5% 大幅提高
一级资本充足率 4% 6% 提高
总资本充足率 8% 8%（不含缓冲） 不变但增加缓冲
资本留存缓冲（CCB） 无 2.5% 新增
逆周期资本缓冲（CCyB） 无 0-2.5% 新增
系统重要性银行附加资本（G-SIB） 无 1%-3.5% 新增
杠杆率要求 无 ≥3% 新增
流动性覆盖率（LCR） 无 ≥100% 新增
净稳定资金比例（NSFR） 无 ≥100% 新增

看完这张表你就明白了：Basel III不是简单的"小修小补"，而是对银行监管体系的全面升级。核心变化可以总结为：资本更多（从8%到最高10.5%+缓冲）、维度更全（资本+流动性+杠杆率）、力度更大（系统重要性银行还要额外加码）。

🔑 关键理解： 为什么要这么严格地要求资本充足率？因为资本是银行吸收亏损的"缓冲垫"。假设一家银行有100亿资产，其中98亿是存款（负债），2亿是资本金（所有者权益）。如果资产损失了3亿，2亿的资本金被全部吃掉还不够，存款人的钱就会受损。但如果资本金有10亿，3亿的亏损完全扛得住，存款人毫发无伤。资本充足率要求的本质是：让银行股东先承担风险，而不是让储户（纳税人）买单。

中国银行监管框架

中国银行业的监管体系经历了多次改革。2023年，中国金融监管体制进行了一次重大调整——在原有银保监会（中国银行保险监督管理委员会）基础上组建了国家金融监督管理总局（简称金融监管总局），统一负责除证券业以外的金融监管。这次改革进一步强化了"功能监管"和"穿透式监管"的理念。

除了资本充足率（严格执行Basel III标准），中国还有一个独特的宏观审慎评估体系——MPA考核（Macro Prudential Assessment，宏观审慎评估）。央行从2016年起对商业银行进行MPA考核，涵盖资本和杠杆、资产负债、流动性、定价行为、资产质量、跨境融资、信贷政策执行等七大方面，每一项都有打分标准。考核结果分成A/B/C三档，直接关系到银行的法定存款准备金利率和业务准入。

对于系统重要性银行（中国有20家左右被认定为国内系统重要性银行，D-SIB），监管还有附加要求：额外的资本缓冲、更严格的压力测试、更详细的处置计划（"生前遗嘱"）。

咱们再来看看拨备覆盖率的计算——这是衡量银行信用风险防御能力的重要指标：

// 拨备覆盖率计算
// 定义：贷款损失准备 / 不良贷款余额 × 100%
// 监管要求：≥150%（中国）

const bankData = {
  totalLoans: 5000,           // 贷款总额5000亿
  normal: 4800,               // 正常类贷款
  specialMention: 120,        // 关注类贷款
  substandard: 45,            // 次级类贷款
  doubtful: 25,               // 可疑类贷款
  loss: 10,                   // 损失类贷款
  loanLossReserve: 150        // 已计提贷款损失准备
};

// 不良贷款 = 次级 + 可疑 + 损失
const npl = bankData.substandard + bankData.doubtful + bankData.loss;
// 不良率
const nplRatio = npl / bankData.totalLoans * 100;
// 拨备覆盖率
const provisionCoverage = bankData.loanLossReserve / npl * 100;

console.log(`不良贷款余额: ${npl}亿`);
console.log(`不良贷款率: ${nplRatio.toFixed(2)}%`);
console.log(`拨备覆盖率: ${provisionCoverage.toFixed(2)}%`);
console.log(`监管要求: ≥150%`);
console.log(`是否达标: ${provisionCoverage >= 150 ? '是' : '否'}`);

// 输出:
// 不良贷款余额: 80亿
// 不良贷款率: 1.60%
// 拨备覆盖率: 187.50%
// 监管要求: ≥150%
// 是否达标: 是

1.6%的不良率、187%的拨备覆盖率，这是中国银行业2025年的平均水平——看起来相当稳健。但平均数掩盖了很多结构性问题，比如城商行和农商行的资产质量明显弱于国有大行。

⚠️ 踩坑故事：包商银行事件（2019年）——大股东占款引发的信用危机

2019年5月24日，央行和银保监会宣布：因包商银行出现"严重信用风险"，对其实行接管，为期一年。这是中国近20年来首例银行被接管事件，整个金融市场为之震动。

包商银行的问题出在哪？简单说——大股东"明天系"把银行当成了自己的"提款机"。明天系通过复杂的关联交易和隐秘的股权结构，从包商银行大量占款，资金用于购买明天系自身的资产。这种"大股东掏空银行"的行为持续多年，导致包商银行的资产黑洞越滚越大。

接管后的清产核资发现，包商银行的损失高达上千亿元。最终处理方案是：对个人存款全额保障（5000万以下的个人存款本息全部兑付）；对机构债权按比例保障——5000万以上的机构债权，最高只能兑付90%，有些甚至只兑付了70%。

包商银行的教训是双重的：第一，对银行来说，公司治理不是"软约束"——大股东和关联交易的风险可以摧毁一家银行。银保监会后来出台了《银行保险机构大股东行为监管办法》，对关联交易进行严格限制。第二，对市场来说，"刚兑信仰"被打破——金融机构之间借钱再也不是"稳赚不赔"了。2019年后，银行间的同业存单发行利率出现了明显的分化，大银行和中小银行的信用利差显著拉大。

包商银行事件是中国银行业风险出清的标志性事件之一。它告诉我们：银行监管不是"多管闲事"，而是在保护每一个普通储户的钱袋子。在包商银行的案例中，正是因为监管及时介入接管，才避免了更大范围的金融风险扩散——5000万以下的个人存款全部得到了保障。
