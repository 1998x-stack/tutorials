第十八章：不等式与估计

大家好，我是你们的讲师。在组合数学中，我们很少能获得精确值——绝大多数情况下，我们只能给出一个估计："至少有多少""至多有多少""大约是多少"。这就像画一个靶子——有时候你知道靶心在哪里（精确值），但更多时候你只能画出一个圆，知道答案在这个圆里就足够了。

不等式与估计能力，是区分一个"会做组合题的学生"和一个"能稳定拿分的学生"的关键。因为联赛中的组合大题，几乎从不会让你算出一个精确的数字——而是让你证明某个量的上界或下界。如果你不会估计，你甚至连题目在问什么都搞不清楚。

本章我们来系统学习组合估计的三大类方法，以及不等式放缩的标准套路。

18.1 三类基本的组合估计方法

组合估计从方法论上可以分成三个大类：计数估计、概率估计和归纳估计。每类都在不同的场景下发挥作用。

方法 核心思想 适用场景 典型工具
计数估计 通过直接计数或双计数获得上下界 子集族大小、边数、集合元素数 二项式系数不等式、双计数
概率估计 利用随机变量的期望与方差给出界 随机结构的存在性、平均值的下界 马尔可夫不等式、切比雪夫不等式
归纳估计 递推关系 + 不等式归纳 递归结构、分治算法分析 主定理、递推式放缩

18.1.1 计数估计：一个简单例子

例1：从 {1,2,...,n} 中选出 k 个数，要求这 k 个数中任意两个数的差至少为 d。最多能选多少个？

上界：将选出的数从小到大排列为 a₁ < a₂ < ... < a_k。则 aᵢ ≥ a₁ + (i-1)d。由于 a_k ≤ n，所以 a₁ + (k-1)d ≤ n，即 k ≤ (n - a₁)/d + 1 ≤ n/d + 1。但这是比较粗糙的上界。

更紧的上界：把 1 到 n 分成若干组：{1,2,...,d}, {d+1,...,2d}, ... 每组最多选 1 个数，因此 k ≤ ⌈n/d⌉。

构造达到：取 1, 1+d, 1+2d, ... 即可达到 ⌈n/d⌉。所以答案是 ⌈n/d⌉。

这个例子展示了计数估计的基本思路：通过构造"分组"来获得上界，再通过一个具体的选取方案来证明这个上界是紧的。这里的"分组"其实就是抽屉原理的一种应用。

18.2 二项式系数的不等式链

二项式系数是组合数学中最频繁出现的不等式对象。掌握关键的不等式链，可以在复杂的组合问题中快速获得需要的估计。

18.2.1 基本不等式

最常用的二项式系数不等式：

① 单调性：C(n, k) < C(n, k+1) 当 k < (n-1)/2
          C(n, k) > C(n, k+1) 当 k > (n-1)/2
          中间项 C(n, ⌊n/2⌋) 最大

② 对称性：C(n, k) = C(n, n-k)

③ 部分和上界：Σ_{i=0}^{k} C(n, i) ≤ (n+1)^k   (粗糙估计)

④ 指数上界：C(n, k) ≤ n^k / k!

⑤ 精确上界：C(n, k) ≤ (en/k)^k
   这是通过 Sterling 公式的变体得到的，在概率方法中尤其常用

18.2.2 指数上界的应用

C(n, k) ≤ (en/k)^k 这个不等式在概率方法中极其常用。它允许我们处理 C(n, k) 和 2^{-C(k,2)} 的乘积——比如在推导 Ramsey 数下界时：

C(n, k) × 2^{1 - C(k,2)} < 1
→ (en/k)^k × 2^{1 - k(k-1)/2} < 1
→ 取对数：k ln(en/k) + (1 - k(k-1)/2) ln 2 < 0
→ 当 k 较大时，主导项是 -k²/2 · ln 2
→ 因此 n ~ k × 2^{k/2} × √e 时不等式成立
→ 即 R(k,k) > k × 2^{k/2} / e

这就是前面第15章使用过的估计。没有这个二项式系数的不等式链，我们无法从概率论证转化为具体的数值下界。

18.3 Stirling 公式与阶的估计

当 n 很大时，n! 的直接计算几乎不可能，但 Stirling 公式给出了一个精确的渐近估计：

Stirling 公式：n! ~ √(2πn) × (n/e)ⁿ

更精确地，有双边不等式：

√(2πn) × (n/e)ⁿ × e^{1/(12n+1)} < n! < √(2πn) × (n/e)ⁿ × e^{1/(12n)}

在竞赛中，我们几乎不需要使用这么精确的版本。通常只用到它的"弱化版"：

竞赛中足够使用的 Stirling 变体：

① (n/e)ⁿ < n! < nⁿ

② n! = Θ(√n · (n/e)ⁿ)
   （即 n! 的增长阶是 √n × (n/e)ⁿ）

③ 对阶乘的比值：
   C(2n, n) = (2n)! / (n! · n!) ~ 4ⁿ / √(πn)

④ 对 log n! 的估计：
   log n! = n log n - n + O(log n)

其中最有用的是③：C(2n, n) ~ 4ⁿ / √(πn)。这个结果在组合概率的各类估计中频繁出现——比如随机游走的返回概率、Catalan 数的渐近等。

竞赛中的 Stirling：在实际比赛中，Stirling 公式的直接考察非常少（因为它涉及的分析超出了高中范围）。更常见的是你需要在推导过程中对阶乘或组合数进行定性的阶估计，比如判断"当 n 充分大时，哪个量增长更快"。在这种情况下，记住 "C(2n, n) 的增长速度 ≈ 4ⁿ / √(πn)" 就足够了。

18.4 不等式放缩的标准套路

在组合估计中，放缩是一门艺术——放得太松，界没有意义；放得太紧，可能证不出来。以下是几种标准的放缩套路：

18.4.1 保留主要项

当和式中有多项时，找出增长最快的那一项，用它来估计整个和的大小。

例：估计 S = Σ_{i=0}^{n} C(n, i) 的大小。
已知 Σ C(n, i) = 2ⁿ，这是精确值不需要估计。
但如果只求和到 k < n/2：
S_k = Σ_{i=0}^{k} C(n, i)
可以用最大项 C(n, k) 来估计：C(n, k) ≤ S_k ≤ (k+1) × C(n, k)
当 k 较小时，S_k ~ C(n, k)（因为最大项占主导）

18.4.2 放大分母/缩小分子

对于形如 a/b 的量，如果要证明它 ≤ 某个数，可以放大分子或缩小分母；如果要证明它 ≥ 某个数，则缩小分子或放大分母。这看似简单，但在具体操作中经常出错。

18.4.3 积分放缩法

对于单调函数 f(x)，和式 Σ_{i=a}^{b} f(i) 可以用积分来逼近：

如果 f 单调递增：
∫_{a-1}^{b} f(x) dx ≤ Σ_{i=a}^{b} f(i) ≤ ∫_{a}^{b+1} f(x) dx

如果 f 单调递减，不等号方向相反。

例：估计 Σ_{i=1}^{n} 1/i（调和级数部分和）
由积分放缩：ln(n+1) ≤ Σ_{i=1}^{n} 1/i ≤ 1 + ln n

18.4.4 利用对称性削峰填谷

当需要估计一个二项式系数时，利用"中间最大"的性质，把较小的系数用较大的替代，从而简化分析：

Σ_{i=0}^{n} C(n, i)² = C(2n, n)  ✓ （恒等式）
如果我们要估计 Σ_{i=0}^{k} C(n, i) 且 k 较小：
C(n, i) ≤ C(n, k) 对所有 i ≤ k 成立
所以 Σ_{i=0}^{k} C(n, i) ≤ (k+1) × C(n, k)

18.5 上下界不匹配时的处理策略

在竞赛中经常会遇到这种情况：你用某种方法得到了一个上界，用另一种构造得到了一个下界（构造示例），但这两个界之间存在差距——上界比下界大很多。这说明要么上界不够紧，要么构造没有达到最优，或者两者都有。

面对这种情况，以下策略值得尝试：

情况 可能原因 应对策略
上界远大于下界 放缩过于粗糙 / 构造非最优 寻找更精细的计数方法 / 改良构造
上界等于下界 极有可能是正确答案 验证极端情况 n=1,2,3 是否匹配
下界超过上界 一定算错了！ 逐行检查推导和构造

⚠️ 踩坑实录：有一年省队选拔赛，题目是："设 A 是 {1,2,...,n} 的子集族，A 中任意两个集合的交集大小至少为 1。证明 |A| ≤ 2ⁿ⁻¹。"

这个结论看起来非常合理——所有包含元素 1 的子集共有 2ⁿ⁻¹ 个，它们两两相交（因为有公共元素 1），所以界是紧的，构造已经给出。我的一个学生非常自信地写下了证明：

"假设 |A| > 2ⁿ⁻¹。对每个子集 S，考虑它的补集 Sᶜ。由于 |A| > 2ⁿ⁻¹，由抽屉原理，A 中一定包含一对互补的子集 (S, Sᶜ)。但 S 和 Sᶜ 的交集为空，与条件矛盾。所以 |A| ≤ 2ⁿ⁻¹。"

完美！看起来滴水不漏。但仔细想想——他说"由抽屉原理，A中一定包含一对互补的子集"，这成立吗？2ⁿ个子集，互补配对得2ⁿ⁻¹对。|A| > 2ⁿ⁻¹，确实由抽屉原理，A 中一定包含至少一对互为补集的子集。正确！

但评卷老师给了他一处分：他忘了考虑空集！空集和任何集合的交集都是空集——如果 A 中包含了空集，那 |A| 就不能超过 1。所以他应该在前面加上一句"假设 A 中不含空集"（否则结论不成立：A = {∅, {1}} 但 n=2 时 |A|=2 > 2²⁻¹=2？不对，2>2不成立。实际上空集的情况需要单独讨论）。

这个教训是：在组合证明中，永远不要忘记空集、全集的特殊情况。边界情况往往是证明的盲点。一个真正的竞赛高手在写完证明后，一定会专门检查"n 最小的时候""边界的元素""空集和全集"这些特殊情形。

18.6 本章小结与过渡

本章我们学习了组合不等式与估计的三大类方法：计数估计、概率估计和归纳估计。重点掌握了二项式系数的不等式链（尤其是 (en/k)ᵏ 形式的指数上界）、Stirling 公式在阶乘估计中的应用，以及不等式放缩的几种标准套路——保留主要项、放大分母缩小分子、积分放缩法、对称性削峰填谷。最后我们讨论了上下界不匹配时的处理策略，并通过一个踩坑案例复习了边界条件的重要性。
