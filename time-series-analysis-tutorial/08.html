第8章 自回归模型 (AR)

大家好！从这一章开始，我们正式进入时间序列建模的核心地带——ARMA家族。如果把时间序列分析比作做菜，前几章我们在认识食材（时序的基本概念）和切菜手法（分解、平滑、频谱分析），现在终于要开始下锅了。而这第一道主菜，就是自回归模型（Autoregressive Model, AR）。

想象这样一个场景：你是一家连锁超市的库存经理，需要预测明天的鸡蛋销量。你发现一个规律——如果昨天卖了500盒，今天往往会卖520盒左右；如果昨天只卖了300盒，今天也很少超过350盒。换句话说，今天的销量和昨天的销量之间存在某种线性关系。这种"用过去预测未来"的思路，就是自回归模型的本质。

8.1 自回归的思想：用昨天的自己预测今天

"自回归"这个名字很形象——"自己对自己做回归"。一般的线性回归是用其他特征（X）来预测 Y，而自回归是用 Y 的过去值来预测 Y 的当前值。一个 AR(p) 模型的数学表达式如下：

Y_t = c + φ₁Y_{t-1} + φ₂Y_{t-2} + ... + φ_pY_{t-p} + ε_t

其中 c 是常数项，φ₁ 到 φ_p 是自回归系数，ε_t 是白噪声（均值为零、方差恒定的随机误差）。p 是模型的阶数，表示我们用了多少个过去值来预测当前值。

最简单的 AR(1) 模型就是：Y_t = c + φ₁Y_{t-1} + ε_t。今天的值等于昨天的值乘以某个系数，再加上一个随机扰动。

记忆技巧："自回归"就是"自己回归自己"——把时间序列在时间轴上往后平移，然后用平移后的序列做普通线性回归。你可以把 Y_t 理解为因变量，把 Y_{t-1}, Y_{t-2}, ..., Y_{t-p} 理解为自变量。

8.2 AR(1) 的两种命运：收敛与发散

AR(1) 模型看起来简单，但它揭示了一个至关重要的问题：序列是"回归到均值"还是"越跑越远"？

假设我们有一个 AR(1) 模型：Y_t = 0 + 0.9 Y_{t-1} + ε_t。如果昨天是 100，今天大约是 90 加上一个随机数，明天大约是 81 加上误差，后天大约是 73……以此类推，序列会逐渐"回归"到均值 0 附近。这就是平稳的 AR(1)。

但如果 φ₁ = 1.05 呢？昨天是 100，今天是 105，明天是 110.25，后天是 115.76……序列会越涨越高，最终发散到无穷。这就是非平稳的 AR(1)，也叫"随机游走带漂移"的变体。

φ₁ 的范围 行为 平稳性
|φ₁| < 1 冲击随时间衰减，序列回归均值 平稳
|φ₁| = 1 冲击永久留存，单位根过程 非平稳（随机游走）
|φ₁| > 1 冲击放大，序列发散 非平稳（爆炸性）
踩坑故事：一位数据分析师为了预测某只股票的价格，直接对价格序列拟合了 AR(2) 模型，得到的 φ₁ + φ₂ = 0.98。他对预测结果非常有信心，结果实际值不到三个月就严重偏离了预测区间。原因很简单——股票价格接近随机游走（φ₁ ≈ 1），AR 模型在非平稳数据上会产生虚假的回归结果。正确的做法是先用差分（第11章会讲）把非平稳序列转为平稳序列，再建模。

8.3 Yule-Walker 方程：求解 AR 系数

如何估计 AR 模型的参数 φ₁, φ₂, ..., φ_p？最经典的方法是 Yule-Walker 方程。它利用一个关键事实：AR 过程的自相关函数（ACF）和模型系数之间存在一一对应的关系。

对于 AR(p) 模型，Yule-Walker 方程如下（以 p=2 为例）：

ρ₁ = φ₁ + φ₂ρ₁

ρ₂ = φ₁ρ₁ + φ₂

其中 ρ₁ 和 ρ₂ 是滞后 1 和滞后 2 的理论自相关系数。用样本数据计算出 ρ₁ 和 ρ₂，就可以解出 φ₁ 和 φ₂。对于一般的 AR(p)，这涉及一个 p×p 的线性方程组，可以用矩阵形式简洁表示。

8.4 特征根与平稳性条件

对于 AR(p) 模型，平稳性的判断比"|φ₁| < 1"要复杂得多。通用的方法是用特征方程（Characteristic Equation）。

将 AR(p) 模型写成滞后算子形式：

(1 - φ₁L - φ₂L² - ... - φ_pL^p) Y_t = ε_t

对应的特征方程是：1 - φ₁z - φ₂z² - ... - φ_pz^p = 0

AR(p) 过程平稳的充要条件是：特征方程的所有根的模都大于 1（这些根落在单位圆之外）。也就是说，如果任何一个根的模 ≤ 1，序列就是非平稳的。这个条件等价于说 φ₁ + φ₂ + ... + φ_p < 1，但实际判断时必须检查所有根，不能只看系数之和。

关键洞察：为什么特征根要在单位圆外？因为特征根对应的是"冲击传播的速度"。根的模大于 1 意味着冲击会随着时间衰减（就像往平静的湖面扔石子，波纹会越来越小）。根的模等于 1 意味着冲击永久存在（扔一个石子，波纹永远不消散），这就是单位根过程。根的模小于 1 意味着冲击反而被放大（越扔波纹越大），这在实际数据中很少见。

8.5 PACF：定阶的秘密武器

AR 模型的阶数 p 怎么确定？这是我们遇到的第一个"模型选择"问题。偏自相关函数（Partial Autocorrelation Function, PACF）就是为这个问题而生的。

PACF 衡量的是"剔除了中间所有滞后项的影响之后，Y_t 和 Y_{t-k} 之间的纯相关性"。这个概念和普通线性回归中的偏回归系数完全一致——想象你对 Y_t 做回归，自变量依次加入 Y_{t-1}, Y_{t-2}, ...，那么 Y_{t-k} 的回归系数就是 PACF 在滞后 k 的值。

AR(p) 模型有一个非常漂亮的性质：它的 PACF 在滞后 p 之后截尾（spike cut off），即 PACF(k) 在 k > p 时理论上为零。这个性质就像指纹一样唯一——看到 PACF 在滞后 3 之后突然截断，你几乎可以确定数据来自 AR(3)。

滞后 k PACF 值（AR(2) 理论值） 含义
1 显著非零 Y_t 与 Y_{t-1} 直接相关
2 显著非零 Y_t 与 Y_{t-2} 直接相关
3 ≈ 0（截尾） Y_t 与 Y_{t-3} 无直接关系
4 ≈ 0 同上

在实际应用中，我们观察样本 PACF 图，看哪些滞后的 PACF 显著超出 95% 置信带（通常用 ±2/√T 作为阈值，T 是样本量）。最后一个显著 PACF 的滞后阶数就是 p 的候选值。

8.6 Python 实操：拟合 AR 模型

下面我们用 statsmodels 来演示 AR 模型的拟合过程。假设我们生成了一个 AR(2) 过程：

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.ar_model import AutoReg
from statsmodels.graphics.tsaplots import plot_pacf, plot_acf
from statsmodels.tsa.stattools import acf, pacf

# 生成 AR(2) 数据: Y_t = 2 + 0.6*Y_{t-1} + 0.3*Y_{t-2} + eps_t
np.random.seed(42)
n = 500
phi1, phi2 = 0.6, 0.3
c = 2.0

Y = np.zeros(n)
eps = np.random.normal(0, 1, n)
Y[0], Y[1] = 0, 0
for t in range(2, n):
    Y[t] = c + phi1*Y[t-1] + phi2*Y[t-2] + eps[t]

# 画 PACF 图——应该看到 lag 1 和 lag 2 显著
fig, ax = plt.subplots(figsize=(10, 4))
plot_pacf(Y, lags=20, ax=ax, method='ywm')
plt.show()

# 拟合 AR(2) 模型
model = AutoReg(Y, lags=2, trend='c')
result = model.fit()
print(result.summary())

# 验证系数恢复效果
print(f"估计的常数: {result.params[0]:.3f} (真实: {c})")
print(f"估计的 φ1: {result.params[1]:.3f} (真实: {phi1})")
print(f"估计的 φ2: {result.params[2]:.3f} (真实: {phi2})")

输出中你会看到系数估计值和它们的 p 值。如果模型正确指定，p 值通常远小于 0.05，表示这些滞后项确实是显著的。

8.7 AR(p) 的矩阵形式与最小二乘估计

Yule-Walker 方法虽然经典，但现代实践中更常用的是最小二乘估计（OLS）。将 AR(p) 写成矩阵形式：

Y = Xφ + ε

其中 Y 是 (T-p)×1 的向量（从第 p+1 个观测开始），X 是 (T-p)×(p+1) 的矩阵（包含常数项列和 p 个滞后列），φ 是 (p+1)×1 的参数向量。

OLS 估计量就是标准的 φ̂ = (X'X)^{-1}X'Y。这种方法在样本量足够大时与 Yule-Walker 等价，而且更容易处理额外的回归项（如趋势或季节性虚拟变量）。但要注意：OLS 要求误差项是白噪声，且滞后项不能与误差项相关。如果模型正确指定，这个条件自动满足。

8.8 AR 模型的优缺点

优点：

  • 直观——用过去预测未来，符合直觉
  • 参数少——AR(3) 只有 3 个系数加一个常数，易于解释
  • 有完善的理论体系——Yule-Walker、PACF 定阶、特征根检验一应俱全
  • 在平稳数据上预测效果可靠

缺点：

  • 只捕捉线性关系，无法处理非线性模式
  • 要求数据平稳（或经过变换后平稳），否则产生虚假回归
  • 长期预测能力弱——多步预测时，误差会通过递推累积
  • 对异常值敏感——一个离群点会影响后续所有预测

好了，AR 模型的核心内容就是这些。我们用 AR 模型学会了如何从过去的观测值中提取预测信息，但现实世界中还有另一种力量在起作用——不可预测的冲击和误差。下一章我们要讲的移动平均模型（MA），正是专门为这种"随机冲击"建模的。MA 模型和 AR 模型就像一对孪生兄弟，理解它们各自的特点和关系，是掌握 ARMA 家族的第一步。
