第20章 期货与衍生品策略

上一章我们聊了期权，那是关于"权利"的游戏。这一章我们进入期货的世界——这里是"义务"的领域。如果你买入一份期货合约，你就有义务在到期时按约定价格交割（或进行现金结算）。没有"选择不行权"的说法。这种机制让期货成为最纯粹的对冲工具，也是最大的杠杆赌场。

期货市场每年承载着数百万亿美元的名义交易量，覆盖从大豆到国债、从原油到比特币的几乎所有可交易资产。而对冲基金和机构投资者使用互换（Swap）、信用违约互换（CDS）等场外衍生品进行着更为复杂的风险管理。我们这一章的目标是：理解这些工具的逻辑，知道它们如何被用来管理风险——以及当风险失控时会发生什么。

Archegos的"杠杆屠杀"：2021年的衍生品灾难

2021年3月，一个叫Bill Hwang的基金经理用短短几天时间，让全球顶级投行损失超过100亿美元。他是怎么做到的？答案是：总收益互换（Total Return Swap, TRS）。

Archegos Capital通过TRS这种场外衍生品，以极少的保证金撬动了巨大的杠杆——据估算，实际杠杆率高达5-10倍。更可怕的是，Bill Hwang集中持有了少数几只股票（ViacomCBS、Discovery等），且都是方向相同的多头暴露。当ViacomCBS股价在2021年3月从100美元跌到45美元时，Archegos的TRS头寸被强制平仓。由于头寸太大，平仓本身又进一步压低了股价，形成了死亡螺旋。

Credit Suisse和Nomura分别承担了55亿和29亿美元的损失。而Bill Hwang后来被指控欺诈和市场操纵，面临数十年的刑期。

核心教训：Archegos案揭示了一个古老的真理——杠杆是放大器，不是创造器。它放大收益，也放大亏损。当你使用高杠杆交易衍生品时，你不是在和市场对赌，而是和时间赛跑。问题是，时间总是赢家。

20.1 套期保值策略

套期保值（Hedging）是期货最古老也最正统的用途——转移价格风险。想象你是中国南方航空公司，每年需要进口大量航空燃油。如果油价上涨10%，你的利润可能减少几十亿。你不需要投机油价，你只需要锁定成本。

空头对冲（Short Hedge）

卖出期货合约来保护现货多头。适用于生产商、库存持有者——他们持有实物商品，担心价格下跌。比如：一家铜矿企业预计三个月后产出1000吨铜，担心铜价下跌，于是卖出铜期货。如果铜价真的下跌，期货的盈利弥补现货的亏损。

多头对冲（Long Hedge）

买入期货合约来锁定未来购买价格。适用于需要未来采购原材料的实体。回到南航的例子：买入原油期货锁定未来几个月的燃油成本。

基差风险（Basis Risk）

理论上完美的对冲几乎不存在，因为现货价格和期货价格的变动幅度并不总是完全一致。基差 = 现货价格 - 期货价格。基差的变动意味着对冲是不完美的。这是套期保值者面临的核心风险。

# 基差风险示例
# 假设你持有1000桶原油，用卖出期货对冲
# 当前现货价格 = 80美元/桶
# 当前期货价格 = 82美元/桶
# 基差 = 80 - 82 = -2美元

# 一个月后：
# 现货价格 = 75美元/桶（下跌5美元）
# 期货价格 = 76美元/桶（下跌6美元）
# 基差 = 75 - 76 = -1美元

# 现货亏损：-5美元/桶
# 期货盈利：+6美元/桶
# 净效果：+1美元/桶（超完美对冲，因基差缩窄）
# 但如果基差扩大（比如期货只跌了4美元），净效果会是亏损
最佳对冲比率：使得组合方差最小的对冲比率可以通过回归得出：h* = ρ × (σ_s / σ_f)，其中ρ是现货与期货收益率的相关系数，σ_s和σ_f分别是现货和期货的标准差。当期货波动大于现货时，我们需要少于1:1的比率。

20.2 基差交易

基差交易者（Basis Trader）不像套保者那样关注绝对价格，他们关注的是基差的变动方向。核心的基差交易逻辑是：

收敛逻辑：理论上，到期日期货价格会收敛于现货价格（否则存在无风险套利机会）。如果你认为当前基差过大（期货相对现货过贵或过便宜），你可以构建一个头寸来赌基差回归。

期现套利：当期货价格显著高于现货价格时，可以买入现货、卖出期货，持有至到期获取基差收敛的收益。这被称为"正向套利"。反之，当期货价格低于现货时，卖出现货、买入期货构成"反向套利"。

基差状态套利方向操作风险
期货升水（Contango）正向套利买现货 + 卖期货融资成本、仓储费
期货贴水（Backwardation）反向套利卖现货 + 买期货借券成本、流动性

20.3 互换利率与IRS估值

利率互换（Interest Rate Swap, IRS）是全球最大的衍生品市场，名义本金超过500万亿美元。最基本的IRS是固定vs浮动互换：一方同意按固定利率支付利息，另一方同意按浮动利率（如LIBOR、SHIBOR）支付利息。

IRS的主要使用场景是利率风险管理：

  • 一家公司发行了浮动利率债券，但希望锁定固定利率支出 → 支付固定、收取浮动
  • 一家保险公司持有固定收益资产，但预期利率上升 → 支付浮动、收取固定

估值方法：IRS的估值就是比较两支"虚拟债券"的价值。固定利率端可视为一支固定利率债券，浮动利率端可视为一支浮动利率债券（其价格在重定价日回归面值）。互换的价值 = 固定端现值 - 浮动端现值。

# IRS估值的简化示例
# 名义本金：1亿元，期限：5年
# 固定利率：3.5%（年付）
# 浮动利率：当前SHIBOR 3M（每3个月重定价）

# 固定端现值 = Σ [CF_i / (1 + r_i)^t_i]
# 其中CF_i是第i期的固定利息支付，r_i是贴现率

# 浮动端现值 ≈ 面值（因为浮动利率在重定价日重置为市场利率）
# 所以互换价值 ≈ 固定端现值 - 面值
实操提示：2023年，LIBOR已正式退出历史舞台，被SOFR（美国）、SONIA（英国）、TONA（日本）等无风险参考利率（RFRs）取代。这意味着过去基于LIBOR的所有IRS合约都需要进行"后备条款"（Fallback Provisions）的修改。如果你在工作中接触到IRS合约，一定要确认使用的参考利率是哪一种。

20.4 CDS信用保护

信用违约互换（Credit Default Swap, CDS）本质上是针对债券违约的"保险"。CDS的买方定期支付保费（称为"利差"），卖方承诺在债券发生信用事件时赔偿买方损失。

CDS定价：CDS的保费（利差）反映了市场对该债券违约概率的评估。假设某公司5年期CDS利差为200个基点（即每年2%的名义本金），这意味着市场隐含的违约概率大约在某个区间。粗糙的近似：年化违约概率 ≈ 利差 / (1 - 回收率)。

信用事件：包括破产、债务加速到期、拒付/延期支付、债务重组等。国际互换与衍生品协会（ISDA）对每种信用事件都有严格定义。

合成CDO：CDS的衍生品——将一组公司债券的CDS打包成不同层级（Tranche），分别出售给不同风险偏好的投资者。这就是2008年金融危机中扮演核心角色的产品。当大量次级抵押贷款同时违约时，那些被评级机构评为AAA级的CDO高层级也未能幸免。

2008年的深刻教训：AIG（美国国际集团）卖出了超过4000亿美元的CDS保护，几乎完全没有做对冲。当次贷危机爆发、大量CDS需要赔付时，AIG面临倒闭。美国政府最终出手救助，总金额超过1800亿美元。这个故事告诉我们：在CDS市场上，"保费"虽小，"理赔"可能大到无法承担。

20.5 杠杆与保证金管理

期货交易的核心机制是保证金制度——用少量资金撬动大额仓位。理解保证金是期货风险管理的起点。

初始保证金（Initial Margin）：开仓时需要存入的最低资金。通常为合约价值的5-15%。例如，一手沪深300股指期货合约价值约120万元，初始保证金约12万元（10%比率）。

维持保证金（Maintenance Margin）：持仓过程中允许的最低保证金水平。通常为初始保证金的70-80%。当账户权益低于维持保证金时，就会收到追加保证金通知（Margin Call）。

追加通知：收到Margin Call后，你需要在限定时间内补足资金至初始保证金水平。如果未能及时补足，经纪商有权强制平仓。这正是期货交易中最残酷的环节——你可能会在流动性不足时被迫以极差的价格平仓。

20.6 衍生品监管

2008年金融危机后，全球衍生品监管发生了根本性变革。

Dodd-Frank法案（2010年）：这是美国自大萧条以来最全面的金融改革法案，对衍生品市场的主要影响包括：

  • 标准化场外衍生品（如CDS）必须在中央对手方清算（Central Counterparty Clearing, CCP）进行清算，而非双边交易
  • 交易需向交易信息库（Trade Repositories）报告
  • 提高资本金和保证金要求
  • 限制银行的自营交易（Volcker Rule）

中国证监会监管框架：中国的期货和衍生品市场由证监会统一监管，主要法律依据是2022年8月1日施行的《中华人民共和国期货和衍生品法》。这是中国首部专门针对期货和衍生品的法律，亮点包括：

  • 明确区分"期货"和"衍生品"的法律概念
  • 引入"中央对手方"制度
  • 为场外衍生品提供法律基础
  • 加强对跨境衍生品交易的监管
  • 对操纵市场、内幕交易等违法行为设定了更严厉的处罚
监管趋势总结：全球衍生品监管的核心方向是"将场外推向场内"——通过中央清算、强制报告、提高资本要求来降低系统性风险。对投资者而言，这意味着更高的交易成本，但更低的交易对手风险。这是一笔值得支付的"保险费"。

期货和场外衍生品构成了现代金融体系的"骨架"。它们让风险可以被定价、分割和转移，使得实体经济能够更好地管理不确定性。但正如Archegos和AIG的故事所警示的——这些工具本身是中性的，其效果取决于使用者的风控能力和道德底线。
