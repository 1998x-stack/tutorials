开篇：2025年的存款和贷款，你的钱在经历什么？

大家好，我是你们的讲师。2025年有一组数据让很多中国老百姓睡不着觉：银行一年期存款利率降到了1.1%，而CPI（消费者物价指数）持续在0%附近徘徊，甚至偶尔跌到负值。你存10万块在银行，一年后利息只有1100块。如果物价不涨，你的购买力倒是没缩水，但如果算上机会成本——你把钱花掉能获得的效用——实际上是亏损的。

与此同时，房贷利率从2023年的4%以上降到了2025年的3%附近。如果你在2023年买房，贷了200万、30年期，等额本息月供是9546元。如果你等到2025年才贷，同样是200万、30年期，月供降到了8432元——每月少了1100多块。

这一降一升之间，利率正在深刻重塑每一个家庭的财务状况。但利率为什么会有这些变化？它由什么决定？未来的走势又该怎么判断？

这一章，咱们就从利息的本质出发，搭建一套分析利率的完整框架。

利息的本质

利息到底从哪来？这个问题看起来简单，却是经济学史上争论最久的问题之一。

主流的解释有三种：

  • 时间偏好（Time Preference）：人们天生偏好"现在"而非"未来"——现在拥有100块的快乐，比一年后拥有100块的快乐更多。所以要让人们放弃现在的消费，必须给予补偿，这个补偿就是利息。用经济学的话说，利息是"推迟消费的报酬"。
  • 机会成本：你把钱借给别人，就不能自己用这笔钱投资或消费。利息是对这种"机会成本"的补偿。
  • 风险补偿：借钱有可能收不回来。利息中有一部分就是针对信用风险的"保费"——风险越高，利率越高。

古典经济学家欧文·费雪（Irving Fisher）在1930年的著作《利息理论》中给出了一个至今仍有影响力的表述：利息是"现在之财"与"未来之财"交换的溢价。简单粗暴地理解——利息就是"时间"的价格。

跨学科视角： 伊斯兰金融（Sharia Finance）禁止收取利息（Riba），认为金钱本身不应产生金钱。在伊斯兰金融体系中，银行贷款采取"利润共享"（Mudarabah）或"成本加利润"（Murabahah）的模式，而不是传统意义上的"利息"。这种模式在马来西亚、沙特、阿联酋等国广泛存在，全球伊斯兰金融资产规模超过4万亿美元。了解这一点很重要——说明"利息"并非天然的、普世的金融概念，而是一种特定的制度安排。

名义利率与实际利率——费雪方程

这是利率分析中最基本、也最重要的一个区分。

  • 名义利率（Nominal Interest Rate）：银行告诉你"年利率3.5%"，这就是名义利率——你没有考虑任何其他因素看到的数据。
  • 实际利率（Real Interest Rate）：名义利率减去通胀率后的真实回报。如果名义利率是3.5%，通胀率是2%，那么实际利率就是1.5%。

为什么这个区分如此重要？因为真正影响你购买力的是实际利率，而不是名义利率。

费雪方程（Fisher Equation）精确地表达了这三者之间的关系：

// 费雪方程：名义利率 = 实际利率 + 预期通胀率
// 精确版本：(1 + 名义利率) = (1 + 实际利率) × (1 + 预期通胀率)
// 简化版本（低通胀时近似成立）：i ≈ r + π^e

function 计算实际利率(名义利率, 预期通胀率) {
    // 精确计算
    let 实际利率精确 = (1 + 名义利率) / (1 + 预期通胀率) - 1;

    // 简化计算（近似）
    let 实际利率近似 = 名义利率 - 预期通胀率;

    return {
        精确: 实际利率精确,
        近似: 实际利率近似,
        误差: Math.abs(实际利率精确 - 实际利率近似)
    };
}

// 场景1：正常通胀环境
console.log("场景1：正常通胀");
let r1 = 计算实际利率(0.035, 0.02);
// 精确: 1.47%, 近似: 1.50%, 误差: 0.03%

// 场景2：高通胀环境
console.log("场景2：高通胀");
let r2 = 计算实际利率(0.15, 0.10);
// 精确: 4.55%, 近似: 5.00%, 误差: 0.45%
// 注意：在高通胀时，简化版本的误差变大！

// 场景3：通缩环境
console.log("场景3：通缩（物价下跌）");
let r3 = 计算实际利率(0.015, -0.01);
// 精确: 2.53%, 近似: 2.50%
// 通缩时，实际利率 > 名义利率！
费雪方程的核心启示： 投资者和企业决策应该基于实际利率，而不是名义利率。高名义利率不一定"紧缩"——如果通胀更高，实际利率可能是负的（2008年金融危机后很多国家的实际利率长期为负）。低名义利率不一定"宽松"——如果通胀更低甚至通缩，实际利率可能是正的（这正是2023-2024年中国面临的情况）。

经典利率决定理论

利率由什么决定？不同的经济学派给出了不同的答案。咱们来对比三种最经典的理论。

理论 代表人物 核心观点 利率由什么决定 主要局限
古典利率理论 亚当·斯密、马歇尔、费雪 利率由"储蓄"和"投资"的供求决定 储蓄（供给）= 投资（需求）→均衡利率 忽略了货币因素和收入变化
凯恩斯流动性偏好理论 约翰·梅纳德·凯恩斯 利率是放弃流动性的报酬；人们持有货币的三大动机（交易、预防、投机） 货币供给 = 货币需求（L1+L2）→均衡利率 忽略了实际因素（储蓄、投资）
可贷资金理论 罗伯逊、俄林 利率由可贷资金的供求决定，综合了实际因素和货币因素 可贷资金供给（储蓄+货币供给增加+...）= 可贷资金需求（投资+货币窖藏+...） 理论较复杂，不易直观理解

古典利率理论

古典经济学家认为，利率是由"节俭"（储蓄的供给）和"生产力"（投资的需求）共同决定的。如果人们更节俭（储蓄增加），利率下降，刺激企业增加投资。最终储蓄等于投资，利率达到均衡。

这个理论有一个重要的政策含义：政府不需要干预利率。如果市场是自由的，利率会自动调节到"储蓄=投资"的均衡点。

凯恩斯的流动性偏好理论

凯恩斯在1936年的《就业、利息和货币通论》中提出了完全不同的观点。他认为，利率不是"等待"或"节俭"的报酬，而是放弃流动性（持有现金）的报酬。

凯恩斯说，人们为什么想持有不生息的现金？因为三个动机：

  • 交易动机：日常买东西需要现金
  • 预防动机：应对突发情况需要现金
  • 投机动机：预期债券价格下跌时，持有现金等待抄底

利率就是让人们放弃这些现金、购买债券的"诱惑力"。所以利率由"货币供给"和"货币需求"决定，而不是由"储蓄"和"投资"决定。

凯恩斯还提出了一个非常重要的概念：流动性陷阱（Liquidity Trap）。当利率低到一定程度，所有人都预期利率只会上升（债券价格只会下跌），于是不管央行增加多少货币供给，人们都会持有现金而不去投资——货币政策彻底失效。这个概念在解释日本"失去的三十年"和中国2023-2025年的低通胀环境时非常有价值。

可贷资金理论

这个理论是前两种理论的综合。它认为利率既受"实际因素"（储蓄、投资）影响，也受"货币因素"（货币供给、货币窖藏）影响。可贷资金的供给包括储蓄和货币供给增量，可贷资金的需求包括投资和货币窖藏。

今天的主流经济学教材（如曼昆的《宏观经济学》）实际上综合了这三种理论——用IS-LM模型（第19章详细介绍）来同时分析商品市场和货币市场。

中国利率市场化进程

讲完理论，咱们来看看中国的实际情况。中国的利率体系经历了一个从"政府定价"到"市场决定"的漫长过程。

在改革开放前，中国的利率完全由政府决定。人民银行既是央行又是商业银行，利率几十年不变。1993年，十四届三中全会提出"利率市场化"的改革方向。此后经历了近30年的渐进改革：

  • 1996年：放开银行间同业拆借利率（CHIBOR）——这是利率市场化的第一步
  • 2004年：放开贷款利率上限和存款利率下限
  • 2013年：全面放开贷款利率管制，取消贷款利率下限
  • 2015年：放开存款利率上限，理论上完成了"名义上的利率市场化"
  • 2019年8月：LPR（贷款市场报价利率）改革——这是最关键的一步

LPR改革的核心是建立"MLF利率 → LPR → 贷款利率"的传导机制。每月20日，18家报价行根据MLF利率（中期借贷便利利率）和自身资金成本，报出LPR。央行通过调整MLF利率来引导LPR变化，进而影响实体经济贷款利率。

2025-2026年，LPR经历了多次下调：一年期LPR从3.1%降至约2.7%，五年期以上LPR从3.6%降至3.0%。每一次LPR调整都引发了市场的巨大关注。

LPR VS 基准利率： 改革前，银行贷款利率参照央行公布的"贷款基准利率"（行政定价）。改革后，参照LPR（市场报价）。这两者的根本区别是：基准利率由央行直接定，LPR由银行根据市场情况报价。LPR改革后，央行的角色从"运动员"变成了"裁判员"——不再直接定利率，而是通过MLF利率来引导利率。

存款利率市场化也在稳步推进。2022年，央行建立了存款利率市场化调整机制，银行可以根据市场利率变化自主调整存款利率。2024-2025年，主要商业银行多次下调存款利率，一年期定期存款利率从1.5%降到了1.1%左右。

踩坑故事：名义利率下降但实际利率上升——2023-2024年的利率悖论

2023-2024年，中国的利率环境出现了一个让很多人困惑的"悖论"：央行一直在降息（名义利率下降），但企业的融资成本"感受"却在上升。

怎么回事？咱们来看数据。

2023年初，一年期LPR是3.65%，CPI同比涨幅是2.0%左右——实际利率大约是1.65%。到了2024年下半年，一年期LPR降到了3.1%，但CPI涨幅降到了0%附近，甚至出现负值（通缩）。实际利率变成了3.1% - 0% = 3.1%。

也就是说：虽然名义利率下降了0.55个百分点，但实际利率反而上升了1.45个百分点。

对于企业来说，实际融资成本在大幅上升。一个做制造业的朋友告诉我："2023年初我们贷款的实际成本（扣除通胀）大概是1.7%左右，还能承受。到了2024年底，实际成本升到了3%以上，很多扩产计划都暂停了。"

这就引出了一个非常重要的政策洞见：在通缩环境中，即使央行不断降息，实际利率也可能不降反升，货币政策的宽松效果会被通缩"吞噬"。这也是为什么日本央行在1990年代泡沫破裂后，即使把利率降到零也未能有效刺激经济——因为通缩让实际利率居高不下。

关键教训： 不要只看名义利率！在做投资决策时，一定要考虑通胀预期。如果判断未来是通缩环境，即便名义利率很低，实际利率也可能很高，借钱并不划算。反之，如果判断未来通胀上升，即便名义利率偏高，实际利率可能很低甚至为负，借钱反而是"占便宜"。

从利率到利率结构

这一章咱们从利息的本质出发，学习了名义利率与实际利率的区分（费雪方程），对比了三种经典的利率决定理论（古典、凯恩斯、可贷资金），也梳理了中国LPR改革的进程。核心结论是：利率不是单一的数字，而是由时间偏好、通胀预期、风险、流动性等多种因素共同决定的复杂价格信号。

但还有一个问题咱们没有回答：为什么同样是人民币，同样是低风险的国债，1年期国债和10年期国债的利率不一样？为什么企业债的利率比国债高？为什么有时候短期利率高于长期利率（收益率曲线倒挂）？

下一章，咱们就来深入探讨利率结构与收益率曲线——从期限结构到风险结构，从预期理论到流动性溢价，帮你看透利率背后的全部信息。
