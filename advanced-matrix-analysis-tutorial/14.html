第14章 矩阵扰动理论

假设你费尽心力计算了一个矩阵的特征值，却因为数据采集仪器的微小误差——矩阵的某个元素变化了 $10^{-8}$——导致结果面目全非。这种情况会发生吗？

答案是：对于某些矩阵，会的。而且后果可能极其严重。矩阵扰动理论（Perturbation Theory）就是研究这个问题的数学工具：当输入的矩阵发生微小变化时，特征值、特征向量、奇异值等输出量会如何变化？

14.1 条件数与敏感性分析

条件数（condition number）是衡量问题"好不好解"的定量指标。对线性方程组 $Ax = b$，条件数定义为：

$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

常见的条件数是用 2-范数定义的：$\kappa_2(A) = \sigma_{\max} / \sigma_{\min}$，即最大奇异值与最小奇异值的比值。

条件数的意义：如果 $A$ 有 $\delta A$ 的扰动，$b$ 有 $\delta b$ 的扰动，则解的相对误差有上界：

$$\frac{\|\delta x\|}{\|x\|} \leq \frac{\kappa(A)}{1 - \kappa(A) \frac{\|\delta A\|}{\|A\|}} \left( \frac{\|\delta A\|}{\|A\|} + \frac{\|\delta b\|}{\|b\|} \right)$$

这个公式看起来很复杂，但核心信息很简单：条件数放大了输入误差。$\kappa(A) \approx 10^3$ 意味着你可能会丢失 3 位有效数字；$\kappa(A) \approx 10^8$ 意味着你几乎得不到任何有效的数字。

一个直观的理解：

条件数衡量了"近奇异性"的程度。$\kappa(A) \gg 1$ 表示 $A$ 接近奇异（某些方向上的变换非常弱）。如果 $\kappa(A) = \infty$，$A$ 就是奇异的——这意味着存在非零的 $x$ 使得 $Ax = 0$，系统失去唯一解。

Hilbert 矩阵 $H_{ij} = 1/(i + j - 1)$ 是著名的病态矩阵：$n=10$ 时条件数已经达到 $10^{13}$ 量级。如果你用一个 $10 \times 10$ 的 Hilbert 矩阵做线性回归，截断误差会将结果完全淹没。

14.2 Weyl定理：特征值扰动界

Weyl 定理是矩阵扰动理论中最经典的结果之一。它告诉我们：对称矩阵的特征值对扰动的敏感性是被严格约束的。

Weyl定理：

设 $A$ 和 $B$ 都是 $n \times n$ 的对称矩阵（或 Hermite 矩阵），它们的特征值分别按升序排列为 $\lambda_1 \leq \lambda_2 \leq \dots \leq \lambda_n$ 和 $\mu_1 \leq \mu_2 \leq \dots \leq \mu_n$。则：

$$\max_i |\mu_i - \lambda_i| \leq \|B - A\|_2$$

换句话说，特征值的最大偏离不超过扰动矩阵的 2-范数。

这个定理的重要性在于：对称矩阵的特征值是"良态"的。你扰动矩阵的范数有多大，特征值的变化就不会超过这个范围。没有更糟的情况了——这是最优的界。

证明思路利用了谱定理和 Courant-Fischer 极小极大定理。核心是把特征值问题转化为 Rayleigh 商的极值问题，然后利用 Rayleigh 商对矩阵的 Lipschitz 连续性。

数值验证：

import numpy as np

n = 10
np.random.seed(42)

# 构造对称矩阵
A = np.random.randn(n, n)
A = A + A.T  # 对称化

# 计算特征值
lam = np.sort(np.linalg.eigvalsh(A))

# 加扰动
E = 1e-6 * np.random.randn(n, n)
E = E + E.T  # 对称扰动
B = A + E

mu = np.sort(np.linalg.eigvalsh(B))

# Weyl 定理的界
perturb_norm = np.linalg.norm(E, 2)
max_deviation = np.max(np.abs(mu - lam))

print(f"扰动范数 ||E||_2: {perturb_norm:.6e}")
print(f"特征值最大变化: {max_deviation:.6e}")
print(f"Weyl 界满足: {max_deviation:.6e} <= {perturb_norm:.6e}?",
      max_deviation <= perturb_norm + 1e-15)

14.3 Davis-Kahan sinΘ定理：特征向量扰动

Weyl 定理告诉我们特征值的稳定性。但特征向量呢？很不幸，特征向量可以非常敏感——即使对称矩阵也是如此。

Davis-Kahan sin$\Theta$ 定理（1970）给出了特征向量子空间之间夹角的正弦的界。设 $A$ 和 $B = A + E$ 是对称矩阵，考虑 $A$ 的特征值分组：一组特征值 $\Lambda_1$ 和一组 $\Lambda_2$，它们之间有 gap $\delta = \min_{\lambda \in \Lambda_1} |\lambda - \mu|$ 对 $\mu \in \Lambda_2$。

设 $P_1$ 是 $A$ 在 $\Lambda_1$ 对应的特征空间上的正交投影，$Q_1$ 是 $B$ 在对应特征空间上的正交投影。则：

$$\|\sin \Theta(P_1, Q_1)\|_F \leq \frac{\|E\|_F}{\delta}$$

这个定理的直觉很清晰：

  • 分子是扰动的强度
  • 分母是特征值间隔（gap）——间隔越大，特征向量越稳定

这意味着：当两个特征值非常接近时，对应的特征向量对扰动极度敏感。这在实际中有重要含义——例如在谱聚类中，如果两个聚类非常接近，它们的特征向量很容易"混在一起"。

14.4 Bauer-Fike定理：非对称矩阵的扰动分析

对于非对称矩阵，情况要复杂得多。Weyl 定理不再适用——非对称矩阵的特征值可以对扰动极端敏感。

Bauer-Fike定理：

设 $A \in \mathbb{C}^{n \times n}$ 可对角化，$A = X \Lambda X^{-1}$。对任意扰动 $E$，$A + E$ 的每个特征值 $\mu$ 都在 $A$ 的某个特征值 $\lambda$ 的圆盘内：

$$|\mu - \lambda| \leq \kappa_p(X) \|E\|_p$$

其中 $\kappa_p(X) = \|X\|_p \|X^{-1}\|_p$ 是 $X$ 的条件数。

这个定理的关键在于 $\kappa(X)$。对于对称矩阵，$X$ 可以选为正交矩阵，$\kappa(X) = 1$，这就是 Weyl 定理的另一种形式。但对于非对称矩阵，$\kappa(X)$ 可以非常大——这意味着特征值可以对一个微小扰动产生巨大变化。

经典踩坑：Wilkinson多项式的教训

矩阵扰动理论中最著名的一个警示是 Wilkinson 多项式。考虑多项式：

$$p(x) = (x-1)(x-2)\cdots(x-20) = \prod_{k=1}^{20} (x - k)$$

它的根是 $1, 2, \dots, 20$，完美地分开且互不相等。现在把这个多项式展开，然后对 $x^{19}$ 的系数施加一个 $2^{-23} \approx 10^{-7}$ 的扰动——也就是说，将多项式的某个系数改变不到千万分之一。

结果呢？Wilkinson 在 1960 年代用实际计算发现：这个微小扰动完全摧毁了多项式的根。原来在 20 附近的根变成了 $20.85 \pm 2.81i$ 这样的复根。一个 $10^{-7}$ 的系数扰动让部分根移动了几个单位！

这个现象的核心原因是什么？多项式对应的伴随矩阵（companion matrix）是一个非对称矩阵，其特征向量矩阵的条件数极大。Bauer-Fike 定理中的 $\kappa(X)$ 达到了天文数字——因此即使微小的系数扰动也会造成特征值的巨大偏移。

这个发现在数值分析历史上是一个分水岭。它告诉整个领域：用特征多项式求特征值这条路是走不通的。永远不要这样做。

14.5 特征值的条件数

单个特征值的条件数可以定义为：

$$s(\lambda) = \frac{1}{|\langle y, x \rangle|}$$

其中 $x$ 和 $y$ 分别是特征值 $\lambda$ 的右特征向量和左特征向量，满足 $Ax = \lambda x$ 和 $y^* A = \lambda y^*$（即 $A^* y = \bar{\lambda} y$），且 $\|x\| = \|y\| = 1$。

对于对称矩阵，左特征向量和右特征向量是相同的（可以选择 $y = x$），所以 $|\langle y, x \rangle| = 1$，条件数 $s(\lambda) = 1$。这再次印证了对称特征值的良态性。

对于非对称矩阵，$x$ 和 $y$ 可以几乎正交，使得 $|\langle y, x \rangle|$ 非常小，条件数 $s(\lambda)$ 变得极大。非正规矩阵（non-normal matrix）如 Jordan 块，其左、右特征向量在特征值接近重根时会趋于共线但方向相反，导致条件数爆表。

特征值对扰动的敏感性分析：

import numpy as np

# 非对称矩阵：特征值良好但特征向量病态
A = np.array([[1, 1000],
              [0, 2]], dtype=float)

# 对比：对称矩阵
B = np.array([[1, 1e-10],
              [1e-10, 2]], dtype=float)

for name, M in [("非对称 A", A), ("对称 B", B)]:
    eigvals = np.linalg.eigvals(M)
    print(f"\n{name} 特征值: {eigvals}")

    # 加扰动
    for eps in [1e-8, 1e-6, 1e-4]:
        E = eps * np.random.randn(2, 2)
        eigvals_pert = np.linalg.eigvals(M + E)
        deviation = np.linalg.norm(eigvals_pert - eigvals)
        print(f"  扰动 {eps:.0e} -> 特征值偏移: {deviation:.6e}")

你会看到，对角元素相差 1 的 $2 \times 2$ 矩阵，上角元 1000 的耦合使特征向量极端倾斜，小扰动导致了可观的漂移；而近对称矩阵即使在 8 个数量级的耦合下，特征值也几乎不变。

对实践的启示：
  1. 对称性是黄金：如果问题允许，尽量使用对称矩阵。对称特征值的条件数恒为 1，这是你能得到的最佳保证。
  2. 检查条件数：在求解任何 $Ax = b$ 之前，先检查 $\kappa(A)$。如果 $\kappa(A) > 10^8$，你的结果可能一位有效数字都没有。
  3. 非正规矩阵要小心：如果一个矩阵是"非正规的"（$A^T A \neq A A^T$），它的特征值可能对扰动极度敏感。这时 SVD 比特征值更可靠。
  4. 预处理：如果条件数太大，考虑缩放行/列（equilibration）、添加正则化项，或者换一种数值格式（如高精度浮点）。

本章小结

矩阵扰动理论回答了数值线性代数中最核心的问题之一："我的计算结果有多可靠？"条件数、Weyl 定理、Davis-Kahan 定理和 Bauer-Fike 定理为我们提供了定量分析的工具。对称矩阵的特征值是良态的（Weyl 定理），而 Wilkinson 多项式的教训提醒我们：永远不要通过特征多项式来计算特征值，永远不要低估非对称矩阵的敏感性。

至此，矩阵分解与谱理论部分就结束了。下一章我们将转向矩阵分析应用的另一个重要方向——矩阵微分与矩阵优化，这是机器学习和数据科学中不可或缺的数学工具。
