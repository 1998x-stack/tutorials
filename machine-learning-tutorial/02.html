2.1 为什么ML工程师需要线性代数？

先讲个我自己的踩坑故事。刚学机器学习的时候，我用sklearn训练了一个线性回归模型，R²到了0.92，感觉美滋滋。结果同事问我："你的设计矩阵的秩是多少？有没有多重共线性？"我当场就愣住了——"设计矩阵"是什么？"秩"又是什么？后来才发现，我的几个特征之间高度相关（VIF超过10），导致模型系数非常不稳定，换一批数据效果直接崩了。

这就是为什么我们需要线性代数：在ML中，数据就是矩阵，模型就是矩阵运算，优化就是解方程。你不一定要会手算100阶行列式，但你必须理解向量、矩阵、特征分解这些概念在算法中到底在干什么。这一章我们只学ML真正用得到的部分。

2.2 标量、向量、矩阵、张量

这四个概念构成了ML中所有数据的基本容器：

概念维度ML中的例子NumPy表示
标量 (Scalar)0维一个样本的损失值、学习率np.float32(3.14)
向量 (Vector)1维一个样本的特征、一个词的embeddingnp.array([1,2,3])
矩阵 (Matrix)2维整个数据集（样本×特征）、权重Wnp.array([[1,2],[3,4]])
张量 (Tensor)3维+一批RGB图片（B×H×W×C）np.ones((32,224,224,3))

在深度学习中，"张量"（Tensor）这个词你会天天见。PyTorch的核心数据结构就叫torch.Tensor。本质上它就是一个多维数组——一个批量32张224×224的彩色图片就是一个4维张量。

2.3 矩阵乘法：ML的"呼吸"

如果你只能从线性代数里带走一个概念，那就是矩阵乘法。一个全连接神经网络层做的事情就是这个：

# 一层神经网络，用NumPy实现
import numpy as np

batch_size, input_dim, output_dim = 64, 128, 10
X = np.random.randn(batch_size, input_dim)   # 输入 (64, 128)
W = np.random.randn(input_dim, output_dim)    # 权重 (128, 10)
b = np.random.randn(output_dim)               # 偏置 (10,)

# 前向传播：就是一个矩阵乘法 + 广播加法
logits = X @ W + b   # 结果: (64, 10)

注意维度匹配规则：(m × n) @ (n × p) → (m × p)。如果维度对不上，程序就报错。实际开发中，"维度不匹配"几乎是最常见的bug。

💡 面试技巧：被问到"Transformer中Q、K、V的维度是多少？"你不需要死记，只要记住 Q @ K^T → 注意力矩阵，然后推导维度就行。比如Q是 (seq_len, d_k)，K^T是 (d_k, seq_len)，乘出来就是 (seq_len, seq_len) 的注意力矩阵——逻辑自洽。

2.4 特征值与特征向量：PCA的灵魂

一个矩阵A作用在一个向量v上，大多数情况下v的方向会改变。但如果存在某个特殊的v，使得A乘以v之后只是把v拉长/缩短了（方向不变），这个v就是特征向量，拉伸的倍数就是特征值：

A · v = λ · v

在机器学习中，特征分解最经典的应用就是PCA（主成分分析）。PCA的本质就是：计算协方差矩阵的特征向量，取最大的k个特征值对应的特征向量作为"主成分"，然后将数据投影到这些主成分上实现降维。我们在第14章会详细展开。

2.5 奇异值分解 SVD：推荐系统的基础

SVD是对任意矩阵的分解（不需要是方阵），它将一个 m×n 矩阵分解为三个矩阵的乘积：

A = U · Σ · V^T

其中Σ是一个对角矩阵，对角线上是"奇异值"。在推荐系统中，SVD用于矩阵分解协同过滤：把用户-物品评分矩阵分解为用户隐向量矩阵和物品隐向量矩阵，然后用来预测缺失的评分。Netflix Prize竞赛的获奖方案核心就是这个思想。

🔑 为什么SVD在ML中这么重要？因为它能找到数据的"低秩近似"——用更少的维度捕捉数据的主要变化。这本质上也是深度学习在做的事情：把高维输入压缩成有意义的低维表示。理解了SVD，你就理解了"表示学习"的数学直觉。

2.6 实践：用NumPy直观感受线性代数

别光看公式，动手算一算：

import numpy as np

# 创建一个"数据集"：100个样本，每个5个特征
X = np.random.randn(100, 5)
print(f"数据形状: {X.shape}")          # (100, 5)

# 协方差矩阵：5×5，描述特征之间的线性关系
cov = np.cov(X.T)
print(f"协方差矩阵形状: {cov.shape}")  # (5, 5)

# 特征分解
eigenvalues, eigenvectors = np.linalg.eig(cov)
print(f"特征值: {eigenvalues}")
print(f"第一个特征向量解释了 {eigenvalues[0]/eigenvalues.sum():.1%} 的方差")

# SVD
U, S, Vt = np.linalg.svd(X, full_matrices=False)
print(f"奇异值: {S}")
print(f"用前2个奇异值可以还原 {S[:2].sum()/S.sum():.1%} 的信息")
⚠️ 常见陷阱：np.linalg.eig 返回的特征向量是列向量形式，即 eigenvectors[:, i] 是第i个特征值对应的特征向量。另外，np.cov 默认每行是一个变量，所以需要 X.T 来让每列作为一个变量。搞反了结果就是错的而且很难debug。

2.7 本章小结

线性代数在ML中的角色可以总结为一句话：数据是矩阵，模型运算是矩阵乘法，优化是解方程，降维是特征分解。你不用成为数学大师，但每次看到X @ W + b的时候，脑子里要立刻浮现出维度之间的流动。
