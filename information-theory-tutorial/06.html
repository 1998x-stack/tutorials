大家好，我是你们的讲师。经过前面几章的积累，我们已经掌握了信息论的核心概念：熵、互信息、KL散度。但这些概念不仅仅是几个孤立的公式——它们背后有一套优美的数学性质网络。理解这些性质，你才能真正"玩转"信息论。

就像学围棋——知道"棋子可以落在交叉点上"只是开始，理解"金角银边草肚皮"这样的深层规律才算入门。同样，知道H = -∑ p log p只是开始，理解熵的极值性质、条件作用的影响、以及最大熵原理，才算真正理解了熵。

我自己的体会是，每次复习这些性质，都会有新的发现。初学者觉得它们只是抽象的定理，但在工程中它们有非常具体的含义——比如"最大熵原理"直接告诉你，为什么机器学习中正则化项那么像高斯分布的熵。

熵的极值性质

熵有两个最基础的极值性质：

性质1：下界 H(X) ≥ 0，等号成立当且仅当X是确定性的（即某个x的p(x)=1，其余为0）。

这个性质非常直观：不确定性不可能为负。如果一个随机变量是确定性的——比如一台永远抛出正面的魔术硬币——那么它的熵就是0，因为完全不存在不确定性。

但注意，这个性质仅对离散随机变量成立。连续随机变量的微分熵（differential entropy）可以为负！我们将在第20章讨论这个有趣的区别。

性质2：上界 H(X) ≤ log |X|，其中|X|是随机变量X的取值个数。等号成立当且仅当X服从均匀分布。

这个性质的证明要用到Jensen不等式和KL散度的非负性。简洁来说：

log |X| - H(X) = D_KL(p || Uniform) ≥ 0

所以 H(X) ≤ log |X|。也就是说，在所有"取值个数相同"的分布中，均匀分布的熵最大。

这个性质有一个很实用的直觉：均匀分布是最"不确定"的分布。如果你对一个系统一无所知，最保守的假设就是假设它服从均匀分布——因为任何偏离均匀的信息都会降低熵、增加你的"知识量"。

举个例子：一个6面骰子，每个面概率都是1/6的熵是log₂6 ≈ 2.585 bits。但如果这个骰子被动了手脚，"1点"的概率是1/2，其余各面平分剩下的1/2，那么熵会降低。计算一下：-0.5·log₂0.5 - 5·(0.1·log₂0.1) = 0.5 + 1.661 = 2.161 bits，确实小于2.585。

条件作用不增熵

这是信息论中非常重要但常被误解的一条性质：

H(X|Y) ≤ H(X)

也就是说，知道Y之后，X的平均不确定性不会增加。换句话说，"附加信息不会增加困惑"——至少平均而言不会。

这个性质的证明也非常简洁：

H(X) - H(X|Y) = I(X;Y) ≥ 0

由于互信息非负，所以 H(X|Y) ≤ H(X)。

但这里有一个非常微妙的陷阱——初学者几乎都会在这里栽跟头。这个性质说的是平均意义上的条件熵 H(X|Y) = E_Y[H(X|Y=y)]。它并不保证对每个具体的 y 都有 H(X|Y=y) ≤ H(X)。

一个反直觉的例子： 假设你有一个二元随机变量X（取值0或1，等概率），和一个随机变量Y：当X=0时，Y总是0；当X=1时，Y以50%概率为0、50%概率为1。那么，给定Y=1时，X只能是1——H(X|Y=1)=0，小于H(X)=1。但给定Y=0时，X可能是0或1——H(X|Y=0)实际上可能会大于H(X)（具体计算留作练习）。

常见陷阱："条件作用不增熵"中的"平均"二字至关重要！H(X|Y=y)（给定某个具体y值的条件熵）可能大于H(X)，但E_Y[H(X|Y=y)] = H(X|Y) ≤ H(X)总是成立的。打个比方：某个具体的朋友可能让你对一件事更困惑，但总体来说，多一个信息来源不会让你更困惑。

这个性质在实际工作中意味着什么？

  • 在数据分析中，增加特征永不会增加不确定性（平均而言）——这为特征工程提供了理论支撑
  • 在信道通信中，收到信号后，对发送符号的不确定性（条件熵）小于或等于发送符号本身的熵——接收信号"有用"
  • 在机器学习中，条件熵 H(标签|特征) ≤ H(标签)——好的特征应该降低对标签的不确定性

熵的链式法则与独立性

我们在第3章介绍过链式法则，这里做更深入的展开。

链式法则：H(X,Y) = H(X) + H(Y|X)

这意味着联合熵可以分解为"先知道X的熵"加上"知道X后Y剩余的熵"。这就像是把联合不确定性拆分成了两个步骤。

如果X和Y独立，那么 H(Y|X) = H(Y)，因此 H(X,Y) = H(X) + H(Y)。独立随机变量的联合熵就是各自熵之和——这和我们直觉一致：X的信息不提供关于Y的任何信息，所以"知识"不能互相抵消。

对于n个随机变量的链式法则：

H(X₁, X₂, ..., Xₙ) = ∑_{i=1}^{n} H(X_i | X₁, ..., X_{i-1})

这个公式的重要性在于：它建立了多变量序列的编码策略。我们可以逐个编码每个符号，每一步都利用已经编码过的符号来减少不确定性。事实上，所有实用的压缩算法都在做这件事——利用已经看到的历史数据来预测下一个符号的概率。

结合"条件作用不增熵"，我们可以得到：

H(X₁, X₂, ..., Xₙ) ≤ ∑_{i=1}^{n} H(X_i)

等号成立当且仅当所有X_i相互独立。也就是说，变量之间的依赖关系减少了联合熵——因为一部分信息是重复的。

Python演示条件熵：

import numpy as np

# 演示 H(X|Y) ≤ H(X)
# 构造一个简单的联合分布

# X: 是否下雨 (0=晴, 1=雨)
# Y: 是否带伞 (0=没带, 1=带了)
# 构造一个合理的联合分布：下雨时更可能带伞

# 联合分布 P(X,Y)
p = np.array([
    [0.35, 0.15],  # X=0(晴): Y=0(没带) 0.35, Y=1(带) 0.15
    [0.10, 0.40],  # X=1(雨): Y=0(没带) 0.10, Y=1(带) 0.40
])

# 边缘分布
p_x = p.sum(axis=1)  # [0.5, 0.5]
p_y = p.sum(axis=0)  # [0.45, 0.55]

# H(X)
h_x = -np.sum(p_x * np.log2(p_x))
print(f"H(X) = {h_x:.4f} bits")  # 1 bit（均匀）

# H(Y)
h_y = -np.sum(p_y * np.log2(p_y))
print(f"H(Y) = {h_y:.4f} bits")

# H(X|Y) = ∑ p(y) * H(X|Y=y)
h_x_given_y = 0
for y in range(2):
    p_x_given_y = p[:, y] / p_y[y]
    h_x_given_y_specific = -np.sum(p_x_given_y * np.log2(p_x_given_y + 1e-12))
    h_x_given_y += p_y[y] * h_x_given_y_specific
    print(f"  H(X|Y={y}) = {h_x_given_y_specific:.4f}")

print(f"H(X|Y) = {h_x_given_y:.4f} bits")
print(f"验证 H(X|Y) ≤ H(X): {h_x_given_y:.4f} ≤ {h_x:.4f} = {h_x_given_y <= h_x}")

# 输出：
# H(X) = 1.0000 bits
# H(Y) = 0.9928 bits
#   H(X|Y=0) = 0.8755
#   H(X|Y=1) = 0.8985
# H(X|Y) = 0.8881 bits
# 验证 H(X|Y) ≤ H(X): 0.8881 ≤ 1.0000 = True
# 注意：两个具体的 H(X|Y=y) 都小于 H(X)
# 但换个分布可能 H(X|Y=某个特定y) > H(X)

最大熵原理

现在我们进入信息论中最优雅的概念之一——最大熵原理（Maximum Entropy Principle）。

最大熵原理告诉你：当你在约束条件下对未知分布做估计时，应该选择熵最大的那个分布。因为熵最大的分布是最"无偏"的——它不引入除了已有约束之外的任何额外假设。

这听起来像哲学，但它有非常实际的数学结果：

已知约束 最大熵分布 概率形式 应用场景
无约束（仅有支撑集） 均匀分布 p(x) = 1/N 等概率假设
已知期望 E[X] = μ 指数分布 p(x) ∝ e^{-λx} 等待时间、寿命数据
已知均值和方差 高斯（正态）分布 p(x) ∝ e^{-(x-μ)²/(2σ²)} 自然界最常见的分布
支撑集在[a,b]内 均匀分布 U[a,b] p(x) = 1/(b-a) 区间约束
已知期望和支撑集[0,∞) 指数分布 p(x) = λe^{-λx} 泊松过程的间隔时间

为什么高斯分布如此普遍？最大熵原理给了我们一个深刻的理解：在所有具有相同均值和方差的分布中，正态分布的熵最大。换句话说，如果自然界只知道一个随机变量的均值和方差，那么它会选择正态分布——因为它是最"随机"、最"无偏"的分布。

这就是中心极限定理（大量独立随机变量之和趋近于正态分布）的信息论解释：当信息被反复求和时，除了均值和方差，其他细节都被"洗掉"了，所以最终分布就是给定这两个约束下的最大熵分布。

真实故事： 在统计力学中，吉布斯（Gibbs）分布正是基于最大熵原理推导出来的。给定平均能量约束下，系统的平衡态分布就是熵最大的分布——热力学第二定律说孤立系统的熵趋于增大，直到达到约束下的最大值。所以，信息论的最大熵原理和热力学的第二定律是同一枚硬币的两面。

最大熵原理的实用价值：在机器学习中，我们经常需要对未知分布建模。最大熵原理提供了一个原则性的方法：在满足观察到的统计量（如某些特征均值）的约束下，选择最"均匀"的分布。L1正则化（拉普拉斯先验）和L2正则化（高斯先验）都可以从最大熵角度理解——它们对应着对不同先验约束的最大熵解。

这一章我们深入探索了熵的四大性质：极值性质（非负性和上界）、条件作用不增熵、链式法则的推广、以及最大熵原理。

关键 takeaways：

  • 离散熵H(X) ≥ 0，且H(X) ≤ log|X|（均匀分布取等号）
  • 条件作用不增熵 H(X|Y) ≤ H(X)——但记住是平均意义上！
  • 链式法则 H(X₁,...,Xₙ) = ∑ H(X_i|X₁,...,X_{i-1})，独立时简化为 ∑ H(X_i)
  • 最大熵原理：在给定约束下选择熵最大的分布，这是最"无偏"的选择

理解了大数定律和典型集，你就能直观感受为什么无损压缩的极限是熵。
