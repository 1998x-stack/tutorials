开篇：你账户里的数字，到底算不算"钱"？

大家好，我是你们的讲师。先问一个看似简单的问题：你打开手机银行，看到余额显示"100,000元"，这10万块钱跟你钱包里的10张100元纸币，是同一回事吗？

从购买力的角度看，好像是的——你都能拿去买东西。但从货币统计的角度看，它们完全是两种不同的东西。钱包里的现金属于M0，而活期存款属于M1，定期存款和其他更广义的资产则属于M2。

2025年一季度，中国人民银行公布的数据显示，中国M2规模突破了310万亿元。这个数字一出来，立刻引发了热议。有人说"印了这么多钱，通胀要来了"，有人说"M2这么高但物价没涨，说明钱都沉淀在房地产和理财里了"。谁说得对？

要回答这个问题，你首先得搞清楚：M0、M1、M2到底是什么？它们之间的区别为什么重要？这一章，咱们就来拆解货币的职能和计量体系。

货币的五大职能

经济学家经过几百年的讨论，对货币的职能达成了基本共识。主流观点认为，货币在经济社会中发挥五大职能：

职能 通俗解释 核心要求 举例
价值尺度
（Unit of Account）		 货币是"标尺"，用来衡量一切商品和服务的价值 必须本身价值稳定，否则尺子就不准 一台iPhone标价7999元，一辆汽车20万元
流通手段
（Medium of Exchange）		 货币是交易的"中间人"，解决物物交换的双重巧合难题 必须被广泛接受，易于携带和分割 你用100元买水果，水果店收下
支付手段
（Means of Payment）		 货币用于清偿债务、支付工资、纳税等非即时交易 必须是社会公认的最终支付工具 你还房贷、发工资、交税
贮藏手段
（Store of Value）		 货币可以把购买力从现在"存"到未来 必须价值相对稳定，不能快速贬值 你存10万块在银行，明年还能买差不多价值的东西
世界货币
（World Money）		 货币在国际贸易、国际借贷中发挥上述职能 必须是国际社会广泛接受的硬通货 美元作为石油定价货币、各国央行储备货币

这五大职能不是并列的，而是有层次关系的。最根本的职能是流通手段——货币之所以成为货币，首先是因为它能用来交换东西。如果一样东西不能用来买东西，它就算再有价值尺度或贮藏手段的功能，也不能算真正的货币。

思考题： 比特币具备哪些货币职能？它可以在某些平台上直接购物（流通手段），可以作为投资品（贮藏手段——虽然波动很大），但很少被用来标价（价值尺度不完善）。在美国，2024年比特币现货ETF获批后，机构投资者开始将其视为"数字黄金"——但美联储主席鲍威尔明确说过"比特币不是货币，它是投机性资产"。你怎么看？

顺便提一下"支付手段"和"流通手段"的区别。这两个看起来差不多，但经济学家硬是给分开了——流通手段是"一手交钱、一手交货"的即时交易，支付手段是"先拿货、后付款"的延期清偿。简单说，流通手段对应现货交易，支付手段对应信用交易。

货币层次的划分——M0、M1、M2

好，现在我们知道货币有五大职能了。但一个现实问题是：一个社会里到底"有多少钱"？是只算现金，还是把银行存款也算上？股票算不算？

这就涉及货币的层次划分。各国央行都会根据流动性（即变现的难易程度）将货币划分为不同层次。流动性越高的资产，越接近"货币"。

层次 中国定义 美国定义（美联储） 流动性
M0 流通中的现金（纸币+硬币） 流通中的现金（纸币+硬币） 最高
M1 M0 + 企业活期存款 M0 + 活期存款（含旅行支票） 高
M2 M1 + 准货币（定期存款、储蓄存款、货币市场基金、居民存款等） M1 + 储蓄存款、小额定期存款、零售货币市场基金 中
M3等 不常用，2022年起不再公布 M2 + 大额定期存款、机构货币市场基金、回购协议 较低

中美两国的M2口径有显著差异。中国的M2口径更宽——居民储蓄存款全部算在M2里（因为中国人习惯存钱）；而美国M2的居民储蓄部分有金额上限。所以直接比较中美M2的绝对值没有太大意义。更重要的是看各国M2的增速趋势。

关键概念： M0是"基础货币"，是央行直接创造的；M1是"狭义货币"，反映现实的购买力（企业和个人随时可以动用的钱）；M2是"广义货币"，既包括现实的购买力，也包括潜在的购买力（定期存款等）。还有一个概念叫M2-M1剪刀差（即准货币增速 - 狭义货币增速），它能透露很多关于经济预期的信息——后面咱们会细讲。

货币统计的实际应用

理解了M0/M1/M2的定义，咱们来看看这些数字在实际中怎么用。毕竟，学货币银行学不是背定义，而是为了读懂经济信号。

中国M2超过310万亿意味着什么？

2025年，中国M2余额超过310万亿元人民币，大概是GDP（约130万亿元）的2.4倍，也就是货币化率（M2/GDP）约为240%。这个比例在全球主要经济体中是最高的——美国大约为80%，日本约为200%，欧元区约为120%。

高货币化率意味着什么？有好几种解读：

  • "货币超发论"：M2增速远超GDP增速，说明货币供给过度，长期看有通胀压力。
  • "储蓄沉淀论"：中国人爱存钱，M2中大量是居民存款（约130万亿），这些钱并没有进入消费领域，所以物价不上涨。
  • "资产沉淀论"：货币没有进入商品市场，而是流入了房地产市场、理财市场等资产领域。

哪种解释对？其实没有唯一答案。现实往往混合了以上几种因素。但这正是货币分析魅力的地方——同样的数据，不同角度能看到不同的故事。

M1-M2剪刀差的信号意义

在货币分析中，有一个指标比M2绝对值更重要——M1-M2同比增速剪刀差。

逻辑是这样的：

  • 如果M1增速 > M2增速（剪刀差为正）：说明企业活期存款增长快，企业愿意持有活期资金，准备投资和扩大生产。这是经济活跃的信号。
  • 如果M1增速 < M2增速（剪刀差为负）：说明企业更倾向于定期存款或理财，不愿意马上花钱。这是经济预期偏弱、投资意愿不足的信号。

2023-2024年，中国的M1-M2剪刀差持续为负且不断走阔，M1增速甚至出现过负值。这反映了企业和居民的投资消费意愿不足，资金在金融体系内"空转"。直到2025年货币政策转向"适度宽松"后，M1增速才开始回升。

// 货币乘数简化计算模型
// 基础概念：M2 = 基础货币 × 货币乘数

// 假设：
let 基础货币 = 100;      // 央行投放的100亿基础货币
let 法定准备金率 = 0.10;  // 10%的存款准备金率
let 现金漏损率 = 0.05;    // 公众持有5%的现金
let 超额准备金率 = 0.03;   // 银行持有3%的超额准备金

// 货币乘数公式
function 计算货币乘数(法定准备金率, 现金漏损率, 超额准备金率) {
    let 货币乘数 = (1 + 现金漏损率) /
                   (法定准备金率 + 现金漏损率 + 超额准备金率);
    return 货币乘数;
}

let 当前乘数 = 计算货币乘数(0.10, 0.05, 0.03);
// 结果 ≈ (1.05) / (0.18) ≈ 5.83倍

let 理论M2 = 基础货币 * 当前乘数;
// 结果 ≈ 100 * 5.83 = 583亿

console.log(`基础货币: ${基础货币}亿`);
console.log(`货币乘数: ${当前乘数.toFixed(2)}倍`);
console.log(`理论M2: ${理论M2.toFixed(0)}亿`);

// 如果降准到8%：
let 降准后乘数 = 计算货币乘数(0.08, 0.05, 0.03);
// 结果 ≈ (1.05) / (0.16) = 6.56倍
console.log(`降准后货币乘数: ${降准后乘数.toFixed(2)}倍`);

踩坑故事：只看M2，忽略了剪刀差

咱们来讲一个真实发生的事。老李是一家小型制造企业的老板，2023年初他看到新闻说"央行M2增速保持在12%以上"，心想："央行在放水，经济要回暖了，我得赶紧扩大产能。"于是他贷款了500万元，买了一批新设备。

但他忽略了一个关键的细节：虽然M2增速很高，但M1增速只有3%，M1-M2剪刀差持续为负。这说明什么？钱虽然很多，但都躺在定期存款里，企业和居民不愿意花钱。经济信心不足，需求并未真正恢复。

结果老李的产能扩大后，发现下游订单并没有增加，产品卖不出去，新设备的产能利用率不到40%。库存积压、现金流紧张，最后被迫裁员20%。

如果老李当初看了M1-M2剪刀差数据，而不是只看M2增速，他就不会这么乐观地扩张。这就是只盯着单一指标、忽略了结构性信息的代价。

实战教训： 分析货币政策环境，不要只看M2的总量，更要看M1与M2的结构和增速差。一个健康的货币环境，应该是M1和M2同步增长，且M1增速不低于M2增速。如果M2很高但M1很低，说明钱在"空转"而不是在"流通"。

货币计量在数字时代的新挑战

传统的M0/M1/M2划分体系在21世纪面临越来越多的挑战。数字人民币、余额宝、微信零钱通、货币基金——这些新型"准货币"到底应该算在哪个层次？

中国央行在2022年调整了M2统计口径，将货币市场基金（如余额宝对应的天弘基金）纳入了M2统计。美国美联储也曾多次调整货币层次定义。这说明货币计量是一个动态演进的过程——随着金融创新，货币的边界越来越模糊。

有一个有趣的问题：数字人民币（e-CNY）属于M0还是M1？答案是M0——因为它本质上是现金的数字化，是央行直接发行的负债，不计息。但数字人民币的"智能合约"功能——比如定向消费、可编程支付——又让它超越了传统M0的范畴。这种"模糊性"正是未来货币体系要面对的新课题。

延伸阅读： 如果你对货币层次的精确划分感兴趣，可以去央行官网下载每月的《货币供应量统计表》。注意看M2的"其他存款性公司"科目，那里包含了大量的结构性存款、同业存单等创新工具。你会发现，"什么是货币"这个看似简单的问题，在实践中远比教科书复杂得多。

从职能到制度

这一章咱们学习了货币的五大职能——价值尺度、流通手段、支付手段、贮藏手段、世界货币——以及M0/M1/M2的货币层次划分。核心结论是：货币不是一种东西，而是一个光谱——从现金到存款到货币基金，越往右走"货币性"越弱，但它们都在不同程度上发挥着货币的职能。

好了，现在咱们知道了货币的起源（第2章）和职能（第3章），但货币的"玩法"还远不止这些。历史上，人类设计了各种制度来管理货币——金本位、布雷顿森林体系、浮动汇率……这些制度是怎么演变的？为什么1971年美元与黄金脱钩是一个转折点？为什么欧元区会经历债务危机？

下一章，咱们就来聊聊货币制度的演变——从金本位到牙买加体系，从特里芬难题到最优货币区理论，带你看清国际货币体系的底层逻辑。
