从曲边梯形面积引入定积分

大家好，我是你们的讲师。

这三章以来，我们一直在学习不定积分的计算方法。但有一个问题始终悬而未决：不定积分（一堆函数）和"求曲线下的面积"（一个数值）之间到底是什么关系？在大多数教材中，定积分被放在不定积分之前介绍，但我觉得先学不定积分再学定积分更容易理解——因为你可以带着"我已经知道怎么求原函数"的底气，来面对定积分这个最终问题。

定积分起源于一个古老的几何问题：如何计算曲边梯形的面积？直边梯形的面积很好算——梯形面积公式 (a+b)h/2 小学就学过了。但曲边梯形呢？它的上底是一条弯曲的曲线，不是一条直线。比如下图所示的函数曲线 y = f(x) 下方、x = a 和 x = b 之间、x 轴上方围成的区域，面积怎么算？

面对这种"不规则"的形状，古代数学家的思路非常直接：用规则的图形来逼近不规则的图形。把曲边梯形切成很多个窄窄的竖条，每个竖条的顶部近似看成一条水平线——这样每个竖条就变成了一个矩形。把所有矩形的面积加起来，就得到了曲边梯形面积的近似值。

这个方法听起来很朴素，但它引出了一个关键问题：切得越细，近似越精确。如果我们把区间无限细分，让每个矩形的宽度趋近于零，那么这些矩形面积之和的极限，就是曲边梯形面积的精确值。这个极限——就是定积分！

黎曼和与定积分的精确定义

上面说的"切细条求和取极限"的过程，数学上被形式化为黎曼和（Riemann sum）。具体来说：

  1. 将区间 [a, b] 分成 n 个子区间，分点为 a = x₀ < x₁ < x₂ < ... < x_n = b
  2. 在第 i 个子区间 [x_{i-1}, x_i] 上任取一点 ξ_i，以 f(ξ_i) 为高、Δx_i = x_i - x_{i-1} 为宽作矩形
  3. 黎曼和：S_n = Σ_{i=1}^{n} f(ξ_i) · Δx_i
  4. 取极限：如果当所有子区间的最大长度 λ = max Δx_i → 0（即 n → ∞）时，黎曼和 S_n 趋近于一个确定的极限值 I，那么称 f(x) 在 [a, b] 上黎曼可积，记作：
∫_a^b f(x) dx = lim_{λ→0} Σ_{i=1}^{n} f(ξ_i) · Δx_i

        其中：
        ∫_a^b  — 定积分号，a 是下限（lower limit），b 是上限（upper limit）
        f(x)   — 被积函数
        dx     — 积分变量（"宽度微元"）
        几何意义：f(x) 在 [a, b] 上与 x 轴围成的代数面积

关于定积分的定义，有一个非常关键的细节："代数面积"。意思是，如果曲线在 x 轴下方，那么对应矩形的高 f(ξ_i) 是负数，这部分面积就是负的。所以定积分给出的是 x 轴上方和下方面积的代数和，而不是绝对面积。如果你想算"曲线和 x 轴之间围成的实际总面积"，需要把函数取绝对值 |f(x)| 再积分。

一个身边的例子：想象你记录了你家水龙头的出水流速（升/分钟）随时间的变化曲线。如果某段时间水流速为正（放水），那段时间曲边梯形的面积就是你接到的水量。如果某段时间水流速为负（漏水），对应面积是负的。定积分给出的是净出水量（放水量减去漏水量）。但如果你想知道总共流过了多少水（不管方向），就要对流速的绝对值积分。这个"方向性"在物理中无处不在——功可以是正功和负功，电荷量可以有正负，净变化和总变化是两回事。

定积分的性质

定积分拥有一些非常有用的性质，熟练掌握它们可以大幅简化计算：

性质 公式 直观理解
线性性 ∫_a^b [cf(x) ± d·g(x)] dx = c∫_a^b f(x) dx ± d∫_a^b g(x) dx 常数可提，和差可拆
区间可加性 ∫_a^b f(x) dx = ∫_a^c f(x) dx + ∫_c^b f(x) dx 大区域面积等于子区域面积之和
积分上下限互换 ∫_a^b f(x) dx = -∫_b^a f(x) dx 互换上下限，结果变号
零区间 ∫_a^a f(x) dx = 0 宽度为零，面积为零
保号性 若 f(x) ≥ g(x) 在 [a,b] 上成立，则 ∫_a^b f(x) dx ≥ ∫_a^b g(x) dx 函数越大，曲线下的面积越大
绝对值不等式 |∫_a^b f(x) dx| ≤ ∫_a^b |f(x)| dx 和的绝对值 ≤ 绝对值的和
积分中值定理 存在 ξ ∈ [a,b] 使得 ∫_a^b f(x) dx = f(ξ)·(b-a) 曲边梯形存在一个等价矩形
记忆技巧：定积分的性质不需要死记硬背。几乎所有性质都可以从"面积求和"的几何图像中直观推导出来。比如"区间可加性"——整个区域的面积当然等于左半块加右半块；"保号性"——上面的曲线围出的面积当然大于等于下面的曲线。用几何直觉驱动记忆，比单纯背公式有效得多。

牛顿-莱布尼茨公式：沟通微分与积分的桥梁

如果定积分只能通过黎曼和的定义来算，那它的实用价值会大打折扣——谁会愿意每次算一个面积都去做无穷求和呢？幸好，牛顿和莱布尼茨各自独立发现了一个堪称微积分史上最重要的公式：

牛顿-莱布尼茨公式（微积分基本定理）：

        如果 F(x) 是 f(x) 在 [a, b] 上的一个原函数（即 F'(x) = f(x)），那么：

        ∫_a^b f(x) dx = F(b) - F(a)

        也就是说：定积分 = 原函数在上限处的值 - 原函数在下限处的值。

这个公式的伟大之处在于，它把两个看似毫不相干的问题联系在了一起：

  • 微分（求导）：关注函数在局部的变化率——切线斜率
  • 定积分：关注函数在整体的累积效果——曲线下面积

牛顿-莱布尼茨公式告诉我们：局部变化率的累积等于整体变化量。这个发现让人类第一次拥有了系统的工具来计算面积、体积、弧长、功、流量等各类累积量——只需要找到原函数，代入上下限相减即可。

来看一个具体的例子：

计算 ∫_0^1 x² dx

        先求不定积分：
        ∫ x² dx = x³/3 + C

        取 F(x) = x³/3（C 可以取任意值，但在相减时会消掉）

        代入牛顿-莱布尼茨公式：
        ∫_0^1 x² dx = F(1) - F(0) = 1³/3 - 0³/3 = 1/3

        用黎曼和验证（取 n 个等宽子区间，右端点）：
        矩形宽度 Δx = 1/n，第 i 个矩形高度为 (i/n)²
        S_n = Σ (i/n)² · (1/n) = 1/n³ · Σ i² = 1/n³ · n(n+1)(2n+1)/6
        = (2n² + 3n + 1)/(6n²)
        lim_{n→∞} S_n = 2/6 = 1/3 ✓

对比一下：用黎曼和定义来计算，你需要做求和、化简、求极限，过程繁琐且容易出错；但用牛顿-莱布尼茨公式，你只需要做不定积分和代入上下限——三步搞定。这就是为什么这个公式被称为"微积分基本定理"：它让微积分从一个少数数学家才能驾驭的理论变成了每个理工科学生都能掌握的实用工具。

容易忽略的细节：使用牛顿-莱布尼茨公式时，被积函数 f(x) 必须在 [a, b] 上连续。如果函数在区间内有间断点，就不能直接使用这个公式。例如 f(x) = 1/x 在 [-1, 1] 上就有 x = 0 这个间断点，此时 F(b) - F(a) = ln|1| - ln|-1| = 0 - 0 = 0，但曲线下的实际面积并不是零（实际上积分发散）。这个错误我见过好几个人犯过——他们机械地套用了公式，却忽略了公式的前提条件。

变上限积分函数及其求导

牛顿-莱布尼茨公式中，下限 a 是固定的，上限 b 是变量。如果我们把上限换成变量 x，就得到了一个以 x 为自变量的变上限积分函数（integral with variable upper limit）：

Φ(x) = ∫_a^x f(t) dt

这个函数有一个极其优美的性质：它的导数就是被积函数本身，即 Φ'(x) = f(x)。

换句话说，Φ(x) 就是 f(x) 的一个原函数！这就是微积分基本定理的另一个版本——有时候被称为"微积分第一基本定理"：

微积分第一基本定理：
        如果 f(x) 在 [a, b] 上连续，则变上限积分函数 Φ(x) = ∫_a^x f(t) dt
        在 [a, b] 上可导，且 Φ'(x) = f(x)。

这个定理的几何解释非常直观：Φ(x) 表示从 a 到 x 的累积面积。当 x 向右移动一小步 Δx 时，面积增加了一窄条，这窄条的高度近似为 f(x)，宽度为 Δx。所以面积变化率 ΔΦ/Δx ≈ f(x)，取极限正是 Φ'(x) = f(x)。

变上限积分函数的求导在解题中有广泛应用：

例1：求 d/dx [∫_0^x sin(t²) dt] = sin(x²)

        例2（复合上限）：求 d/dx [∫_0^{x²} e^t dt]
        令 u = x²，则 d/dx [∫_0^u e^t dt] = e^u · du/dx = e^(x²) · 2x

        例3（上下限均为变量）：求 d/dx [∫_{x}^{x²} t² dt]
        = d/dx [∫_a^{x²} t² dt - ∫_a^{x} t² dt]
        = (x²)² · 2x - x² · 1
        = 2x^5 - x²

这类题目在考研和竞赛中非常常见。关键规律是：对于 d/dx ∫_{u(x)}^{v(x)} f(t) dt = f(v(x))·v'(x) - f(u(x))·u'(x)。记住这个公式，可以快速破解大部分变上限积分求导问题。

定积分的换元与分部积分法

我们已经学过的换元积分法和分部积分法，不仅可以用于不定积分，也完全可以应用于定积分——但需要做一些小调整。

定积分的换元法

定积分换元的核心区别是：积分上下限也要跟着换。

∫_a^b f(x) dx = ∫_{u(a)}^{u(b)} f(φ(t)) · φ'(t) dt

        设 x = φ(t)，则：
        • dx = φ'(t) dt
        • 当 x = a 时，t = φ^(-1)(a)（即原下限对应的新下限）
        • 当 x = b 时，t = φ^(-1)(b)（即原上限对应的新上限）
例：∫_0^1 √(1 - x²) dx

        令 x = sin θ，则 dx = cos θ dθ
        当 x = 0 时，θ = 0；当 x = 1 时，θ = π/2

        原式 = ∫_0^{π/2} √(1 - sin²θ) · cos θ dθ
        = ∫_0^{π/2} cos²θ dθ
        = ∫_0^{π/2} (1 + cos 2θ)/2 dθ
        = [θ/2 + sin 2θ/4]_0^{π/2}
        = π/4

        这正好是半径为 1 的四分之一圆的面积。✓
定积分换元的一个优势：与不定积分不同的是，定积分换元后不需要再把结果代回原来的变量！因为你已经同步更新了积分上下限，直接计算新积分即可。这往往能省去不少代数运算。所以当你用换元法算定积分时，记得同时换上下限，不要在最后又费劲把 θ 代回 x。

定积分的分部积分法

形式与不定积分基本相同，但增加了代入上下限的计算：

∫_a^b u·dv = [u·v]_a^b - ∫_a^b v·du

        其中 [u·v]_a^b = u(b)·v(b) - u(a)·v(a)
例：∫_0^1 x·e^x dx

        设 u = x，dv = e^x dx，则 du = dx，v = e^x

        ∫_0^1 x·e^x dx = [x·e^x]_0^1 - ∫_0^1 e^x dx
        = (1·e¹ - 0·e⁰) - (e¹ - e⁰)
        = e - (e - 1)
        = 1

这是一个有趣的结果——∫_0^1 x·e^x dx = 1。记住这个数值，以后在很多概率统计问题中会反复见到它（例如指数分布期望值的计算）。

定积分的分部积分法特别适合处理那些需要多次分部积分的场合。例如计算 ∫_0^π x² sin x dx，只需要连续做两次分部积分，每次降低 x 的幂次，代入上下限后就能得到精确值。这个技巧在傅里叶级数和信号处理中非常常用——你会在后续课程中反复遇到。

关于定积分和不定积分关系的最后思考：现在你已经看到了整个积分学的全景。不定积分是一族函数（原函数 + C），定积分是一个数值（原函数在上下限的差值）。连接它们的桥梁就是牛顿-莱布尼茨公式。这个公式的伟大之处在于，它告诉我们一个深刻的道理：一个函数的局部变化信息（导数）完全决定了它的整体累积行为（积分）。这也正是微积分能够如此强大的原因——通过研究瞬间的变化率，我们可以预测长时段的总效果。从物理学的牛顿运动定律（微分形式）推导出动量定理和能量守恒（积分形式），从经济学的边际成本推导出总成本——无不是这个思想的体现。

到这里，我们完成了微积分学的一元函数微分和积分部分的核心内容。从第19章开始，我们将进入多元函数微积分的领域——空间解析几何、偏导数、重积分、曲线积分与曲面积分。在多元世界里，导数和积分变得更加丰富和复杂：方向导数、梯度、散度、旋度、格林公式、高斯公式、斯托克斯公式——每一个都是人类智慧的结晶。而你所学的所有一元微积分知识，都是多元微积分的基石。
