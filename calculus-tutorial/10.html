并不是所有函数都"显式"地告诉你 y 等于什么

大家好，我是你们的讲师。到目前为止，我们处理的函数都是显函数——形式为 y = f(x)，等号左边只有 y，右边是关于 x 的表达式。这种形式很友好：你想知道 x 处的函数值，直接代入计算就行。你想求导数，对表达式直接套公式就行。

但现实世界中，很多关系并不是这样"直白"的。比如圆的方程 x² + y² = 1——你没法写成 y = 某个只含 x 的表达式（除非分两段写 y = ±√(1 - x²)）。再比如一个描述行星轨道的方程，或者经济学中描述供需平衡的方程，它们往往把 x 和 y 混在一起，形成一个隐函数 F(x, y) = 0。

更复杂的情况是：x 和 y 本身都不是直接的自变量，它们都依赖于第三个变量——比如时间 t。这就形成了参数方程：x = x(t), y = y(t)。

对于这些"不那么直白"的函数形式，我们之前学的求导法则还能用吗？答案是肯定的，但需要一些技巧性的调整。本章我们就来掌握这些技巧。

显函数 vs 隐函数

先明确一下概念：

类型 形式 举例 特点
显函数 y = f(x) y = x² + 3x + 1 y 直接由 x 表达
隐函数 F(x, y) = 0 x² + y² = 1 x 和 y 混合在方程中
参数方程 x = x(t), y = y(t) x = cos t, y = sin t x 和 y 都是 t 的函数

这三种形式可以互相转换（在一定条件下），但有时候直接转换非常麻烦甚至不可能。隐函数求导法和参数方程求导法就是为了在这些"别扭"的形式下仍能高效地求出导数。

隐函数求导：两边对 x 求导

隐函数求导的核心思想只有一句话：对方程两边同时对 x 求导，把 y 视为 x 的函数。

这听起来简单，但执行起来有一个关键点：遇到 y 的项时，要用链式法则——因为 y 是 x 的函数，所以对 y² 求 x 的导数时，结果是 2y · dy/dx。

例 1：求圆 x² + y² = 1 上某点的切线斜率 dy/dx。

方程两边对 x 求导：
        d/dx (x²) + d/dx (y²) = d/dx (1)

        注意：d/dx (y²) = d/dy (y²) · dy/dx = 2y · dy/dx  ← 链式法则！

        所以：2x + 2y · dy/dx = 0
             dy/dx = -x / y

结果漂亮极了：圆上任意一点 (x, y) 处的切线斜率为 -x/y。比如在 (0, 1) 处斜率为 0（水平切线），在 (1, 0) 处斜率为无穷大（垂直切线）——这和圆的几何性质完全吻合。

注意：隐函数求导的结果中通常同时包含 x 和 y。这是正常的，在某些情况下甚至必须这样——因为隐函数可能无法显式地解出 y = f(x)。

例 2：求方程 y³ + y = x 所确定的隐函数的导数。

两边对 x 求导：
        3y² · dy/dx + dy/dx = 1
        dy/dx · (3y² + 1) = 1
        dy/dx = 1 / (3y² + 1)

注意：这个方程中 y 无法用初等函数解出来（它是三次方程，虽然理论上可解，但表达式非常复杂）。但隐函数求导法完全不需要解出 y，直接对两边求导就得到了 dy/dx 的表达式——这就是隐函数求导的威力。

隐函数求导的检查方法
求完导之后，可以做一个检查：如果原方程可以解出显式形式（比如圆的方程 x² + y² = 1 可以解出 y = √(1 - x²)），那么用显函数求导法验证结果。

在圆上取上半部分 y = √(1 - x²)，求导得：
dy/dx = (1/2)(1 - x²)^(-1/2) · (-2x) = -x / √(1 - x²) = -x / y
和隐函数求导的结果一模一样。验证通过！

对数求导法：幂指函数的救星

对数求导法是隐函数求导的一个绝妙应用。它的思路是：先取对数，把乘法变成加法，把幂指变成乘积，然后隐函数求导。

这种方法特别适合处理两类函数：

  1. 多个函数相乘除的表达式（比如 f(x) = u₁(x)·u₂(x)·u₃(x) / (v₁(x)·v₂(x))）
  2. 幂指函数（比如 f(x) = x^sin x）

例：求 f(x) = x^sin x 的导数（x > 0）。

两边取自然对数：
        ln f(x) = sin x · ln x

        两边对 x 求导（注意左边是隐函数求导！）：
        f'(x) / f(x) = cos x · ln x + sin x · (1/x)

        两边同乘 f(x)：
        f'(x) = x^(sin x) · [cos x · ln x + sin x / x]

如果不取对数，你能直接对这个函数求导吗？它的底数是 x，指数是 sin x——既不是幂函数（指数是常数），也不是指数函数（底数是常数）。这就是幂指函数（power-exponential function），只能用对数求导法或第13章将要学习的对数恒等式变形来处理。

对数求导法操作步骤
1. 令 y = f(x)，两边取 ln
2. 化简右边（把乘法变加法，幂变乘积）
3. 两边对 x 求导（左边是 y'/y，右边直接求导）
4. 两边同乘 y，得到 y'

参数方程的一阶导数：dy/dx = (dy/dt)/(dx/dt)

现在考虑参数方程形式：x = x(t), y = y(t)，其中 t 是参数。我们想求的是 y 对 x 的导数 dy/dx，而 y 和 x 都是 t 的函数。

直觉告诉我们：dy/dx 应该等于 (dy/dt) / (dx/dt)。这不是近似，而是精确的公式（前提是 dx/dt ≠ 0）。

参数方程的一阶导数
如果 x = x(t), y = y(t) 均可导，且 x'(t) ≠ 0，则

dy/dx = (dy/dt) / (dx/dt) = y'(t) / x'(t)

例：已知椭圆的参数方程 x = a cos t, y = b sin t，求 dy/dx。

dx/dt = -a sin t
        dy/dt = b cos t

        dy/dx = (b cos t) / (-a sin t) = -(b/a) · cot t

当 t = π/2 时（椭圆的上顶点），cot t = 0，dy/dx = 0——水平切线。当 t = 0 时（椭圆的右顶点），cot t → ∞，dy/dx → -∞——垂直切线。几何意义一目了然。

参数方程二阶导数的经典陷阱

这是本章最重要的部分之一，也是几乎所有学生都会掉进去的坑。

很多同学想当然地认为：参数方程的二阶导数就是 d²y/dx² = y''(t) / x''(t)。这是完全错误的！

正确的推导如下：

一阶导数是：dy/dx = y'(t) / x'(t)  ——记作 φ(t)

        二阶导数是：d²y/dx² = d/dx (dy/dx) = d/dx (φ(t))

        注意：φ(t) 是 t 的函数，但我们需要对 x 求导！
        所以要用链式法则：dφ/dx = (dφ/dt) · (dt/dx) = (dφ/dt) / (dx/dt)

        因此：d²y/dx² = φ'(t) / x'(t)
                       = [d/dt (y'(t)/x'(t))] / x'(t)
                       = [y''(t)x'(t) - y'(t)x''(t)] / [x'(t)]³

我把这个结果用表格总结如下：

导数阶数 常见误区 正确答案
一阶 dy/dx — y'(t) / x'(t)
二阶 d²y/dx² y''(t) / x''(t) [y''(t)x'(t) - y'(t)x''(t)] / [x'(t)]³
一个发生在考试中的真实案例
几年前我带过一个考研的学生，他的数学基础很不错，求导公式背得滚瓜烂熟。有一次模拟考试，有一道题是参数方程求二阶导，他用了 3 分钟就"算"完了——y''(t)/x''(t)，觉得十分简单。

结果发下试卷，这道10分的题得了0分。他来找我，一脸委屈地问："老师，参数方程二阶导为什么不是直接除？"

我给了他一个简单的直观解释：dy/dx 本身已经是 t 的函数了（它等于 y'(t)/x'(t)）。你要求这个函数再对 x 求导，而 x 本身也是 t 的函数。这就是为什么需要用链式法则把 t 作为桥梁，而不能简单地用 y''/x''。两者之间差了一个"对 t 求导后再除以 x'(t)"的步骤。

这个坑你一定要记住——它每年都会出现在考场上，区别只是以不同的形式出现。前人的血泪教训，就是你省下的时间。

相关变化率问题实例

隐函数求导和参数方程求导的高级应用之一是相关变化率（related rates）问题。这类问题的典型场景是：已知某个量的变化率（比如梯子滑落时上端下滑的速度），求另一个相关量的变化率（比如梯子下端远离墙的速度）。

经典问题：一架长度为 L 的梯子靠在一面垂直的墙上。梯子下端以速度 vₓ 水平拉开。问当梯子下端距离墙为 x 时，梯子上端下滑的速度是多少？

解法：

设梯子下端距离墙为 x(t)，上端距离地面为 y(t)。
        由勾股定理：x² + y² = L²

        对时间 t 求导（隐函数求导！）：
        2x · dx/dt + 2y · dy/dt = 0

        解得：dy/dt = -(x/y) · dx/dt

        已知 dx/dt = vₓ（拉开速度），所以：
        dy/dt = -(x / √(L² - x²)) · vₓ

        负号表示 y 在减小（梯子上端下滑），符合物理直觉。

这个问题的关键在于：梯子长度 L 是常数，x 和 y 都随时间变化，它们之间的关系由隐函数 x² + y² = L² 给出。而我们并不需要解出 x(t) 和 y(t) 的具体表达式——直接对关系式两边求导即可。

相关变化率问题的解题框架
1. 找出问题中各个量之间的关系方程
2. 对方程两边同时对时间 t 求导（隐函数求导）
3. 代入已知的数值（注意正负号——增大为正，减小为负）
4. 求出未知的变化率

本章我们学习了三种特殊形式的求导方法：隐函数求导（两边对 x 求导，y 视为 x 的函数）、对数求导法（取对数后转化为隐函数）、参数方程求导（dy/dx = (dy/dt)/(dx/dt)）。我们还特别强调了一个极易踩坑的陷阱：参数方程的二阶导数绝不是 y''/x''，而是一个涉及链式法则的公式。

从下一章开始，我们将从"一阶导数"走向"高阶"。第11章的主题是高阶导数与微分——当我们对一个函数反复求导多次后，会看到哪些有趣的规律？微分 dy = f'(x)dx 又是什么意思？这些内容将为我们理解泰勒公式和微分中值定理铺平道路。
