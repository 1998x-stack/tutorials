商业银行的起源与功能——从金匠到金融枢纽

你有没有想过，银行这个看似现代的东西，其实几百年前就有了？而且它的起源非常接地气——跟金匠有关。17世纪的伦敦，商人们把黄金存放在金匠那里保管，金匠出具一张"金匠券"作为凭证。有意思的事情发生了：人们发现拿着金匠券直接交易更方便，不用每次都去取黄金出来。于是，金匠券开始在市面上流通——这就是最早的银行券，也就是纸币的前身。

更关键的是，聪明的金匠发现：不可能所有存黄金的人同时来提取。所以金匠只需要保留一部分黄金作为准备金（比如20%），剩下的可以借出去赚利息。这个"部分准备金制度"，就是现代商业银行的雏形。

从这个起源可以看出，银行最核心的几项功能已经天然地摆在桌面上：

  • 信用中介——银行把资金从盈余方（存款人）引导到短缺方（借款人），这是银行最根本的功能。没有这个中介，有钱的人和缺钱的人只能靠"熟人借贷"，资金配置效率极低。
  • 支付中介——银行为客户办理支付结算，让资金安全、便捷地从一方转移到另一方。你刷微信支付、转支付宝，背后都是银行在完成清算。
  • 信用创造——银行通过贷款可以"创造"出新的存款，这就是货币乘数效应。咱们在第2章其实已经接触过这个概念的雏形。
  • 金融服务——除了存贷汇，银行还提供保管、代理、咨询等综合服务。
💡 小贴士： 你有没有注意到，所有银行门口挂的招牌都叫"XX银行"，而不是"XX存钱处"？这反映了银行的核心定位：它是一个"中介"，而不是一个"仓库"。银行不是把你的钱锁在保险柜里等你来取，而是把你的钱拿去放贷、投资，让资金流动起来。这就是现代金融体系运转的基础。

资产业务（钱去哪了）

银行的资产业务，说白了就是银行把吸收来的钱用到了什么地方。这直接决定了银行能不能赚钱、会不会亏钱。咱们来看三类核心资产：

贷款——银行最大的资产

贷款是银行最传统、也是规模最大的资产业务。在中国商业银行的总资产中，贷款通常占到50%-60%甚至更高。按照客户类型，可以分成：

  • 对公贷款：给企业的贷款，包括流动资金贷款（用于日常经营）、固定资产贷款（用于买设备建厂房）、项目融资（用于特定大型项目）等。特点是金额大、期限灵活、利率通常高于零售贷款。
  • 零售贷款：给个人的贷款，包括房贷（占比最大）、消费贷、信用卡透支等。特点是单笔金额小、分散度高、违约相关性相对较低。
  • 贴现：银行买入企业持有的未到期商业票据，实质上是短期贷款的一种形式。

投资——银行的"副业"

银行闲置的资金会配置到债券市场上，主要买国债、地方债、政策性银行债（这些称为"利率债"，风险低），以及少量信用债（企业债，风险相对高一些）。在中国，银行持有了大部分国债和地方债，是债券市场最大的机构投资者。

同业资产——银行之间的"拆借"

银行之间也会互相借钱。当一家银行资金充裕时，会把钱拆借给资金紧张的同行。还包括存放中央银行的超额准备金——这部分虽然利息很低，但提供了最高的流动性。

咱们来看一张典型的商业银行资产负债表简化模板，一图就能理解资产和负债的对应关系：

资产（钱去哪了） 占比（约） 负债+所有者权益（钱从哪来） 占比（约）
贷款及垫款 55% 吸收存款 65%
债券投资 20% 同业负债 15%
存放央行（准备金） 10% 发行债券（含同业存单） 8%
同业资产 8% 其他负债 7%
其他资产 7% 所有者权益（净资产） 5%

这张表非常重要。你看，存款是银行最大的资金来源（约65%），而贷款是最大的资金运用（约55%）。银行通过"低买高卖"赚钱——用较低的利率吸收存款（付息），用较高的利率发放贷款（收息），中间的差额就是利润来源，专业术语叫净息差（NIM）。

负债业务（钱从哪来）

银行的负债业务是筹集资金的业务。没有资金来源，银行就没法放贷投资，所以负债业务是前提和基础。

存款——最稳定、最便宜的资金

存款是银行的"生命线"。中国商业银行大约65%的负债来自存款。存款分为几类：

  • 活期存款：随时可以存取，利率最低（目前年息0.1%-0.35%），但对银行来说是最便宜的"原材料"。
  • 定期存款：约定了存期和利率，银行可以锁定这笔资金的使用期限。利率比活期高，但也更贵。
  • 大额存单（CD）：面向机构和个人发行的记账式大额存款凭证，起点高（个人20万起），可以在二级市场转让，流动性好。利率比普通定期高。

同业负债——银行之间的"拆借"

银行向其他银行或非银行金融机构借入的资金。包括同业拆入、同业存放、卖出回购等。同业负债的特点是期限短（几天到几个月）、对利率敏感。如果一家银行过度依赖同业负债，在流动性紧张时会非常被动——这也是很多银行出事的原因之一。

发行债券——主动负债

银行可以通过在银行间市场发行债券来主动筹集资金，包括金融债、次级债、二级资本债等。跟存款不同，发债是"主动负债"——银行可以根据自己的资金需求决定发多少、什么时候发。

咱们来算一笔账，理解银行是怎么赚钱的：

// 净息差(NIM)计算示例
// NIM = (利息收入 - 利息支出) / 计息资产平均余额

const bank = {
  loanBalance: 1000,       // 贷款余额1000亿
  avgLoanRate: 0.045,     // 平均贷款利率4.5%
  depositBalance: 800,    // 存款余额800亿
  avgDepositRate: 0.018,  // 平均存款利率1.8%
  interbankBalance: 200,  // 同业负债200亿
  avgInterbankRate: 0.025 // 同业负债利率2.5%
};

// 利息收入
const interestIncome = bank.loanBalance * bank.avgLoanRate;
// 利息支出
const interestExpense = (bank.depositBalance * bank.avgDepositRate)
                      + (bank.interbankBalance * bank.avgInterbankRate);
// 净息差
const nim = (interestIncome - interestExpense) / bank.loanBalance;

console.log(`利息收入: ${interestIncome}亿`);
console.log(`利息支出: ${interestExpense}亿`);
console.log(`净息差(NIM): ${(nim * 100).toFixed(2)}%`);

// 输出:
// 利息收入: 45亿
// 利息支出: 19.4亿
// 净息差(NIM): 2.56%

NIM是中国商业银行最主要的利润来源。2025年，由于央行降息，商业银行的净息差持续收窄，已经降到1.5%左右的水平——这意味银行的盈利能力面临巨大压力。

资产负债管理（ALM）

资产负债管理是银行内部最核心的管理工作之一。简单说，就是管理好"借来的钱"和"贷出去的钱"之间的匹配关系，确保银行既赚钱又安全。ALM有三大核心关注点：

期限匹配

存款大多是短期的（活期随时可取，定期的平均期限也就1-2年），而贷款却有很多是长期的（房贷长达30年）。这种"短借长贷"的期限错配是银行与生俱来的特征。期限匹配管理的核心就是要确保：到期的负债有足够的现金流来偿还，不会出现"钱到期了但收不回来"的情况。

利率缺口

银行的资产和负债对利率的敏感度不同。如果利率上升，资产和负债的价值都会变化，但变化的幅度可能不一样。利率缺口管理就是要衡量和控制这种风险。举例：如果银行持有大量固定利率的长期贷款（资产），而存款利率随市场上升（负债成本增加），那么银行的NIM就会收窄——这就是利率风险。

流动性管理

流动性是银行的"生命线"。流动性管理的核心是确保银行任何时候都有足够的资金来满足客户提款和到期债务的偿还需求。流动性管理指标包括流动性覆盖率（LCR）、净稳定资金比例（NSFR）等。

⚠️ 踩坑故事：硅谷银行倒闭（2023年）——期限错配+利率上升的教科书级案例

2023年3月，美国硅谷银行（SVB）在48小时内从"正常经营"到"倒闭被接管"，成为2008年金融危机后美国最大的银行倒闭事件。发生了什么？

硅谷银行的负债端主要是科技公司存款——这些存款在过去几年高速增长，但有个致命问题：它们大多是活期存款，随时可能被提走。而资产端呢？硅谷银行把大量资金投向了长期美国国债和抵押贷款支持证券（MBS）。在2022年之前低利率环境下，这些债券的价格稳定，看起来是"安全"的配置。

但2022年美联储开始暴力加息——从0%一路加到5%以上。债券价格暴跌（债券价格与利率反向变动），硅谷银行持有的长期债券浮亏巨大。如果持有到期，这些亏损还只是"浮亏"（会计上记为可供出售金融资产的未实现损失）。问题是——存款人开始恐慌，大规模提款。硅谷银行不得不提前卖出债券来应付提款需求，于是"浮亏"变成了"实亏"，亏损金额180亿美元直接击穿了银行的资本金。

这就是典型的期限错配危机：借的是"活期"（短期资金），投的是"长期"（长期债券），利率一涨，两头夹击。如果一个普通的储户在2023年3月8日把100万美元存在硅谷银行，3月10日银行倒闭后，超过25万美元的部分（FDIC保险上限）可能到现在还没完全拿回来。

硅谷银行的教训告诉我们：资产管理不是简单的"吸收存款、发放贷款"，而是要在安全性、流动性和盈利性之间找到一个精妙的平衡。银行的管理者必须时刻追问三个问题：我的钱够不够付给储户？（流动性）我的贷款能不能收回来？（信用风险）利率变化会不会让我的利差消失？（利率风险）

🔑 知识点总结： 商业银行的业务可以用一句话概括——"以负债为起点、以资产为终点、以ALM为全过程管理"。负债业务筹集资金（存款、同业、发债），资产业务运用资金（贷款、投资、同业），资产负债管理确保整个过程中银行不"翻车"。理解了这三块，你就理解了商业银行经营的基本逻辑。
