14.1 信道建模的必要性与挑战

大家好，我是你们的讲师。经过前四章的学习，我们已经掌握了自由空间传播、地面反射、地表波、绕射和散射等单一传播机制。现在的问题是：这些机制在实际的无线信道中是如何共存和相互作用的？

一个城市中的移动通信场景——基站天线在楼顶，你在路上边走边刷手机——信号经历了以下复杂的物理过程：

  • 直达波被建筑物遮挡了（绕射损失）
  • 旁边的玻璃幕墙反射了一路信号过来（多径）
  • 地面也反射了一路（双射线模型）
  • 路边的行道树引起了一定的散射
  • 你移动时，所有路径的长度都在变化（多普勒效应）

如果我们试图用公式精确描述这种场景，我们将陷入无穷无尽的数学复杂性中。这就是我们需要信道模型（channel model）的原因——它把复杂的物理传播现象凝练成一组易于处理的数学参数，使得：

  • 不同通信系统的性能可以公平地对比（所有厂家使用同一信道模型做仿真）
  • 网络规划时可以预测覆盖范围（不需要在每个城市都做实测）
  • 系统设计时能评估各种技术的收益（均衡器、MIMO、HARQ等）
一个昂贵的教训：2000年代初，3G（UMTS）在欧洲开始部署时，运营商们发现实际覆盖范围远小于预期——在某些城市，一个基站的覆盖半径只有规划值的60%。原因是3G商用部署前使用的信道模型是在郊区/开阔地环境下开发的，没有充分反映城市峡谷中的穿透损耗和阴影效应。运营商被迫增加基站数量，3G的部署成本因此大幅上升。这个教训深刻影响了后来4G/5G的信道模型开发——3GPP花了大量精力进行全球范围的无线信道测量活动，以确保模型能覆盖各种场景。

14.2 大尺度路径损耗模型

大尺度模型描述的是信号电平的平均趋势——在数十到数百波长的尺度上（即数米到数百米），信号强度的变化。

最基本的模型是对数距离路径损耗模型（log-distance path loss model）：

PL(d) = PL(d₀) + 10n · log₁₀(d/d₀)

其中d₀是参考距离（通常为1 m或100 m），PL(d₀)是参考距离处的路径损耗（通过测量或自由空间公式求得），n是路径损耗指数（path loss exponent）。

不同的传播环境，n值差别很大：

  • 自由空间：n = 2.0（Friis公式）
  • 开阔郊外：n = 2.0 - 2.5
  • 典型城市：n = 2.7 - 3.5
  • 密集城区（高楼林立）：n = 3.5 - 4.5
  • 室内视距：n = 1.6 - 2.0（波导效应使损耗小于自由空间）
  • 室内非视距：n = 3.0 - 4.0
  • 工厂/仓库（密集金属结构）：n = 3.0 - 6.0

除了简单的对数距离模型，工程中还广泛使用各种经验模型。下表总结了最常见的几个：

模型名称 频率范围 适用场景 精度 特点
Okumura (1968) 150-1920 MHz 城市/郊区宏蜂窝 ±5 dB 第一个系统的城市传播模型，基于东京实测
Hata (1980) 150-1500 MHz 城市/郊区宏蜂窝 ±5 dB Okumura的参数化公式版本，手动计算方便
COST-231 Hata (1999) 1500-2000 MHz 城市/郊区宏蜂窝 ±5 dB Hata扩展到2 GHz，支持3G规划
COST-231 Walfisch-Ikegami 800-2000 MHz 城市微蜂窝/峡谷 ±6 dB 考虑街道峡谷和屋顶绕射
3GPP TR 38.901 (2018+) 0.5-100 GHz 5G NR宏/微/室内 ±8 dB 支持毫米波、3D MIMO、O2I穿透
使用经验模型的正确姿势：经验模型是通过大量测量数据拟合得到的，它们适用于与测量环境相似的地形。把日本东京拟合的Okumura模型直接用于青藏高原，误差可想而知。当使用任何经验模型时，都必须用本地实测数据来进行校准——即调整修正因子来匹配本地传播环境。这是网络规划工程师的基本功。

14.3 阴影衰落

即使在同一距离上，不同位置的接收信号强度也会有明显的差异——这是因为路径上的大型障碍物（如山丘、大型建筑群）造成了不同程度的遮挡和绕射损耗。这种变化称为阴影衰落（shadow fading）。

阴影衰落的一个重要特征是它的分布——大量实测数据表明，阴影衰落（用dB表示）服从正态分布：

PL(d) = PL(d₀) + 10n · log₁₀(d/d₀) + Xσ

其中Xσ ~ N(0, σ²)是零均值、标准差为σ的正态随机变量（单位是dB）——这就是对数正态阴影（log-normal shadowing）。

标准差σ取决于环境类型：

  • 室内环境：σ ≈ 6-8 dB
  • 城区环境：σ ≈ 8-10 dB
  • 郊区环境：σ ≈ 6-8 dB
  • 乡村开阔地：σ ≈ 4-6 dB

覆盖率计算：已知路径损耗模型和阴影衰落标准差，我们可以计算小区边缘覆盖概率——即小区边缘处（距离d₀给定）的信号强度超过灵敏度阈值的概率。如果路径损耗中值为PL₀，灵敏度门槛为P_th(dBm)，发射功率为P_t(dBm)，那么：

Pcov = Q((P_th + PL₀ - P_t) / σ)

其中Q(·)是标准正态分布的尾概率函数。这个公式是网络覆盖规划的核心——它告诉我们在给定站点配置下，多远的距离内能保证90%或95%的覆盖概率。

网络规划的启示：假设目标是小区边缘覆盖概率85%，路径损耗指数n=3.5，阴影衰落标准差σ=8 dB。那么小区的有效覆盖半径大约是"中值路径损耗=灵敏度+8 dB"对应的距离。这意味着如果我们用中值路径损耗（即不考虑阴影）来规划，覆盖半径会被高估约20-30%。这就是为什么3G早期覆盖出了问题的根源之一——当时用了过于乐观的传播模型。

14.4 小尺度衰落与多普勒效应

在大尺度模型之上，还有小尺度衰落（small-scale fading）——在半个波长尺度内（在2 GHz下仅7.5 cm）信号幅度的剧烈波动。这是多径干涉的直接后果。

瑞利衰落（Rayleigh fading）：当接收信号中没有主导的直达路径时（NLOS环境），多径信号包络的统计分布服从瑞利分布：

p(r) = (r/σ²) · exp(-r²/(2σ²)), r ≥ 0

瑞利衰落是移动信道中最"恶劣"的情况——信号幅度有很高的概率跌落到深零点（比中值低20-40 dB）。

莱斯衰落（Rice fading）：当接收信号中有一个主导的直达路径时（LOS环境），包络服从莱斯分布：

p(r) = (r/σ²) · exp(-(r² + A²)/(2σ²)) · I₀(Ar/σ²), r ≥ 0

其中A是主导径的幅度，I₀(·)是零阶修正贝塞尔函数。莱斯分布的K因子定义为K = A²/(2σ²)（主导径功率与散射径功率之比）。K越大，信道越接近AWGN信道（K=∞时完全无衰落）；K=0时还原为瑞利衰落。

多普勒效应：当接收机（或发射机/散射体）移动时，接收信号的频率会发生偏移：

fd = (v/λ) · cosθ

其中v是相对运动速度，θ是运动方向与信号到达方向之间的夹角。

多普勒频移导致了时间选择性衰落——信道冲激响应随时间变化。描述信道变化快慢的参数是相干时间（coherence time）：

Tc ≈ 0.423 / fd,max

当信号符号周期远小于Tc（慢衰落），信道在一个符号内基本不变；当符号周期接近或超过Tc（快衰落），信道在符号内发生变化，解调变得困难。

实例计算：一辆时速120 km的汽车在2.1 GHz（4G基站）下行驶，最大多普勒频移f_d = 120/3.6 ÷ (3e8/2.1e9) ≈ 233 Hz。相干时间T_c ≈ 0.423/233 ≈ 1.8 ms。LTE的时隙长度是0.5 ms，远小于1.8 ms——所以信道在一个时隙内近似不变，时域均衡相对简单。但如果车时速500 km（高铁），f_d ≈ 972 Hz，T_c ≈ 0.44 ms——此时信道在0.5 ms的时隙内已经变化明显了。这就是为什么高铁通信需要特殊的信道估计技术。

14.5 3GPP/ITU标准信道模型

从3G到5G再到6G，3GPP和ITU制定了一系列标准信道模型，确保所有设备厂家使用同样的"测试环境"来评估系统性能。

3GPP TR 38.901（5G NR信道模型）

这是目前最全面的信道模型标准，覆盖频段从0.5 GHz到100 GHz（即从sub-1 GHz覆盖到毫米波）。它支持以下场景：

  • UMa（Urban Macro）：城市宏蜂窝，基站高于屋顶
  • UMi（Urban Micro）：城市微蜂窝，基站低于屋顶
  • RMa（Rural Macro）：乡村宏蜂窝，开阔环境
  • InH（Indoor Hotspot）：室内热点，如商场、机场、展厅

38.901使用几何随机统计模型（Geometry-based Stochastic Channel Model, GSCM）——它不像前面提到的物理模型那样"从头算起"，而是根据场景参数随机生成散射体簇（cluster），每个簇中有多个具有特定时延、功率、角度和极化的射线。

模型提供两种输出格式：

  • CDL（Clustered Delay Line，簇延迟线）：非几何模型，用于SISO系统和链路级仿真，直接给出多径的时延和幅度。
  • TDL（Tapped Delay Line，抽头延迟线）：CDL的简化版本，用于快速链路仿真。
6G信道建模的新挑战：面向2030年代的6G通信将在更高的频段（sub-THz, 100-300 GHz）和更极端的场景（空地一体化、智能反射面）下工作。这些对信道模型提出了新要求：
  • 近场效应：在超大规模阵列（ELAA）下，天线口径可达数十个波长，瑞利距离（Fraunhofer距离）远超传统假设，信道处于辐射近场区
  • 空间非平稳性：不同天线单元"看到"的散射体集合不同，传统的宽平稳假设不再成立
  • 分子吸收：在THz频段，大气分子（尤其是水蒸气）引起显著的频率选择性吸收

最后，让我们用Python生成瑞利衰落信道的时域波形——这是每个无线通信工程师入门时都会写的第一段信道仿真代码：

import numpy as np

def generate_rayleigh_fading(fs, T, fc, v, N=100):
    """
    生成瑞利衰落信道的时域包络 (Jakes模型)

    参数:
        fs : 采样率 (Hz)
        T  : 总时长 (s)
        fc : 载波频率 (Hz)
        v  : 移动速度 (m/s)
        N  : 散射体数量
    返回:
        t   : 时间矢量 (s)
        envelope : 瑞利衰落包络 (线性幅度)
        phase    : 衰落相位 (rad)
    """
    c = 3e8
    f_d_max = v * fc / c  # 最大多普勒频移
    t = np.arange(0, T, 1/fs)

    # Jakes模型: 使用N个等功率的正弦波来模拟多普勒谱
    # 各正弦波频率分布在[-f_d_max, f_d_max]范围内
    alpha = 2 * np.pi * np.random.rand(N)       # 随机初始相位
    beta  = 2 * np.pi * np.random.rand(N)       # 随机到达角
    phi   = 2 * np.pi * np.random.rand(N)       # 随机相位

    # 计算衰落复数信道
    h = np.zeros(len(t), dtype=complex)
    for n in range(N):
        f_n = f_d_max * np.cos(alpha[n])
        h += np.exp(1j * (2 * np.pi * f_n * t + beta[n])) / np.sqrt(N)

    # 归一化到平均功率=1
    h = h / np.sqrt(np.mean(np.abs(h)**2))

    return t, np.abs(h), np.angle(h)

# 参数设置
fs = 1000      # 1 kHz采样
T  = 2.0       # 2秒信号
fc = 2.1e9     # 2.1 GHz
v  = 30 / 3.6  # 30 km/h ≈ 8.33 m/s

t, env, phase = generate_rayleigh_fading(fs, T, fc, v)

# 统计特性分析
print(f"载波频率: {fc/1e9:.1f} GHz")
print(f"移动速度: {v*3.6:.1f} km/h")
print(f"最大多普勒频移: {v*fc/3e8:.1f} Hz")
print(f"相干时间 T_c ≈ {0.423/(v*fc/3e8)*1000:.2f} ms")
print(f"包络均值: {np.mean(env):.3f}")
print(f"包络均方根: {np.std(env):.3f}")
print(f"最大衰落深度: {-20*np.log10(np.min(env)):.1f} dB")
# 输出示例:
# 载波频率: 2.1 GHz
# 移动速度: 30.0 km/h
# 最大多普勒频移: 58.3 Hz
# 相干时间 T_c ≈ 7.26 ms
# 包络均值: 0.884
# 包络均方根: 0.520
# 最大衰落深度: 32.4 dB

# 结果验证: 瑞利衰落中，包络的中值约为0.832
# 有1%的概率深度超过20 dB — 这就是为什么无线通信需要分集和纠错
信道仿真需要避免的陷阱：上面的Jakes模型只是一个教学演示。实际仿真时有几个坑需要注意：
  • Jakes模型要求N足够大（>50）才能有好的统计效果，太小的话衰落统计特性会不准确
  • 如果移动速度=0（静态场景），多普勒频移为0，信道是时不变的——但实际静态信道中即使没有运动，环境变化（如树叶晃动、人走动）也会引起缓慢的信道变化
  • 真实的5G链路仿真需要使用3GPP 38.901的CDL/TDL模型的参数文件，而不是简单的瑞利分布——因为CDL模型还包含了各径的时延、角度、极化模式

这是"电波传播"部分的最后一章。我们完成了从自由空间到标准信道模型的完整演进：自由空间Friis公式为一切提供了基准，双射线模型引入了地面反射干涉，刃峰绕射打开了阴影区，散射和多径带来了统计特性，最后——所有这一切被凝练成3GPP/ITU的标准信道模型，让全球的通信工程师可以用同样的语言来讨论无线信道的性能。
