6.1 半波振子天线

大家好，我是你们的讲师。如果说天线世界里有且只有一个"明星"，那一定就是半波振子（half-wave dipole）。它是天线中最经典、最基础、最广泛使用的类型——几乎所有天线教材的开篇案例，所有天线测量中的参考标准，几乎所有天线工程师的第一个设计项目。

半波振子的总长度2l = λ/2，沿z轴放置，中心馈电。它的电流分布近似为：

I(z) = I₀ cos(k₀z)，其中 -λ/4 < z < λ/4

这是一个驻波分布：两端（z = ±λ/4）电流为零（开路端），中心（z = 0）电流最大。这种分布与赫兹偶极子的均匀电流假设不同——实际上，只有当天线长度远小于波长时（即短偶极子），均匀电流假设才成立。半波振子上的电流沿天线呈余弦分布。

对这个电流分布进行积分（使用磁矢位法），得到远场方向图：

Eθ ∝ cos(π/2 · cosθ) / sinθ

这个方向图在E面（包含天线轴线的平面）呈"8"字形——比赫兹偶极子（sinθ）的方向图略窄一些（因为天线更长，电流分布不同）。在H面（垂直于天线轴线的平面）仍然是全向的。

半波振子的关键性能规格：

  • 最大方向性系数：D₀ = 1.64 = 2.15 dBi
  • 输入阻抗：Zin ≈ 73 + j42.5 Ω。注意这里有正电抗（感性），需要通过适当缩短振子长度来调谐到谐振（使Xin = 0）。实际中，半波振子的物理长度约为0.47λ~0.48λ（而非精确的0.5λ），这就是所谓缩短效应（end effect）。
  • 辐射效率：铜线制作的半波振子效率通常 > 95%。
  • 半功率波束宽度（HPBW）：E面约78°，H面360°（全向）。

为什么半波振子是通用参考天线（0 dBd）？

在天线增益测量中，经常使用"相对于半波偶极子的增益"——单位是dBd。0 dBd = 2.15 dBi。也就是说，如果一个天线标称"增益 6 dBd"，对应的绝对增益是 6 + 2.15 = 8.15 dBi。选择半波振子作为参考是因为：它容易制作、方向图简单、性能稳定、辐射效率接近100%。

历史背景：Heinrich Hertz在1887年证实电磁波存在时使用的就是偶极子天线的雏形——虽然当时他还没有精确地使用λ/2的尺寸。到了20世纪30年代，半波振子已经标准化为天线测量的参考天线。今天，你在任何一个天线暗室中都能看到作为标准增益参考的半波振子。

6.2 单极子天线与镜像原理

如果把一个偶极子从中间切开，把下半部分替换为无限大理想导电平面——就得到了单极子天线（monopole antenna）。这个导电平面起到了镜像（image）作用：根据镜像原理，导电平面下方的虚拟镜像偶极子替代了被移除的下半部分。

因此，一个长度为h的直立单极子（h = λ/4，即四分之一波长单极子）在导电平面上的表现与一个λ/2偶极子在自由空间中的表现相同——但只能在上半空间辐射。

关键参数对比：

  • 输入阻抗：Zin ≈ 36.5 + j21.25 Ω——正好是半波振子的一半（因为镜像使输入电压减半而电流相同）。
  • 方向性系数：D₀与半波振子相同（2.15 dBi），但增益是相对于上半空间的。由于能量只辐射到半球空间中，等效增益看起来比同等的偶极子更高。
  • 方向图：上半空间与半波振子相同，下半空间为零（理想导电平面的遮挡）。
日常生活中的单极子：你几乎每天都在使用单极子天线——车载FM天线（约75 cm长，对应FM广播88-108 MHz的λ/4）、AM广播发射塔（塔本身就是单极子，高度可达几百米，对应中波的λ/4）。甚至你的手机内部也包含单极子或其变体——手机底部的"白带"区域往往是单极子天线的辐射体。

重要实践：接地平面的大小。理想的单极子需要无限大导电平面，但现实中只能使用有限尺寸的接地板。接地板尺寸不足时，天线的阻抗和方向图都会偏离理想值。一个经验规则：接地板半径至少应大于λ/4，否则天线性能会显著下降。这就是为什么车载天线需要安装在车身金属顶部的中央位置，而不是边缘。

6.3 环天线

环天线（loop antenna）是将导线弯曲成环形（圆形、方形、三角形等）的天线。根据环的周长相对于波长的大小，环天线分为两大类：

电小环（Electrically Small Loop, 周长 << λ）

当环的周长C远小于波长（C/λ < 0.1）时，环上的电流近似均匀（幅度和相位处处相同），天线表现为一个磁偶极子（magnetic dipole）——与电偶极子对偶。它的方向图呈"甜甜圈"状：环平面内全向，垂直于环平面的方向上有零点。

电小环的辐射电阻非常小：

Rrad = 20π² (C/λ)⁴ (Ω)

这是一个与(C/λ)⁴成正比的量。对于C = 0.1λ的环，Rrad ≈ 0.02 Ω——远小于导线的欧姆电阻。这意味着电小环的辐射效率极低（可能只有百分之几），除非采取特殊措施。

铁氧体加载环天线：在环上套一个高磁导率的铁氧体磁芯（μr >> 1），可以显著增大环天线的等效磁偶极矩，从而大幅提升有效高度和接收灵敏度。这就是为什么AM收音机（530-1700 kHz）内部使用一个绕在铁氧体棒上的线圈作为天线——铁氧体将空间中的磁场集中到线圈中，使得一个仅几厘米长的天线就能有效接收波长数百米的AM信号。

电大环（Electrically Large Loop, 周长 ≈ λ）

当环的周长接近一个波长时（如C = λ的谐振环），电流分布不再均匀，而是出现驻波。此时环天线的方向图与电小环完全不同，最大辐射方向在环平面内。电大环的输入阻抗较高（约100 Ω），辐射效率较好，但带宽较窄。

踩坑故事：我曾经参与一个HF频段（3-30 MHz）的短波通信接收天线项目，使用电小环天线接收远距离电台信号。在实验室测试时一切正常，但部署到野外后接收效果极差。排查发现，环天线对附近的金属物体（围栏、管道）非常敏感——它们在环的近场区内耦合了显著的感应电流，完全改变了环天线的谐振频率和方向图。这个教训：电小环天线的近场区很大，对周边环境非常敏感。部署时必须远离金属结构。

6.4 缝隙天线与巴比涅原理

想象一块无限大的理想导电平板，上面开了一个窄缝。如果在缝的两侧施加激励（通过波导或传输线馈电），这个缝隙就会向空间辐射电磁波。这就是缝隙天线（slot antenna）的基本结构。

分析缝隙天线有一个非常巧妙的工具——巴比涅原理（Babinet's principle）。巴比涅原理最初来自光学：一个不透明屏上的开口的衍射图样，与互补屏（即开口处为不透明、其他部分透光）的衍射图样相同，但振幅相位互补。1946年，H.G. Booker将其推广到电磁学中，形成了Booker-Babinet原理：

一个理想导电屏上的缝隙天线的辐射场，与该缝隙的互补结构（即相同形状的金属导体条带）的辐射场是对偶的——E与H互换。

具体来说：一个长度为L = λ/2的半波谐振缝隙在无限大导电平面上的辐射，与一个半波偶极子的辐射方向图完全相同——但电场和磁场的角色互换，即极化方向旋转了90°。

如果半波偶极子的方向图在E面呈"8"字形、在H面呈全向，那么半波缝隙的方向图就是：在H面呈"8"字形、在E面呈全向——极化方向垂直于缝隙的长边。

工程应用：

  • 波导缝隙天线阵：在矩形波导的宽壁上沿中心线或偏置开一系列缝隙，通过精确控制缝隙的位置和倾角来控制每个缝隙的激励幅度和相位，进而合成所需的远场方向图。这是机载火控雷达（如F-16的AN/APG-68）的标准天线方案。
  • 智能手机金属边框天线：现代智能手机的金属中框上必须保留"注塑条"或"白带"——这些地方实际上是缝隙天线的缝隙。金属边框作为导电屏，塑料条下方的馈电结构在缝隙处激励辐射。换句话说，你手机上的"天线白带"本质上是一个精心设计的缝隙天线。
为什么缝隙天线如此重要？因为它是"共形天线"的雏形——天线不需要突出于载体表面，而是与载体蒙皮平齐。对于飞行器、导弹、卫星等空气动力学要求严苛的平台，缝隙天线几乎是唯一的选择。

6.5 八木-宇田天线

1926年，日本东北大学的八木秀次（Hidetsugu Yagi）和他的学生宇田新太郎（Shintaro Uda）合作发明了一种结构简单但方向性很强的天线——后被称为八木-宇田天线（Yagi-Uda antenna）。有趣的是，八木在1928年将这项发明带到美国申请了专利，而宇田的名字在专利中被遗漏了——所以在西方通常简称为"八木天线"，但在日本和许多工程圈中，正确的名称是"八木-宇田天线"以纪念两人的贡献。

标准的八木-宇田天线由一组平行排列的振子组成：

  • 有源振子（driven element）：通常是一半波振子（或折合振子），由馈线直接激励。
  • 反射器（reflector）：位于有源振子后方（约0.2-0.3λ间距），比有源振子稍长（通常比半波振子长约5%）。它"挡住"了向后方向辐射的能量。
  • 引向器（directors）：位于有源振子前方（间距约0.2-0.4λ），数量不等（从1个到十几个），比有源振子稍短（通常比半波振子短约5%）。它们将能量"引导"向前方。

工作原理是寄生耦合（parasitic coupling）：无源振子（反射器和引向器）上的电流是由有源振子辐射的场感应产生的。通过精确控制各振子的长度和间距，可以操控各寄生振子上感应电流的幅度和相位，使前方方向的场同相叠加而增强，后方方向的场反相相消而减弱。

单元数 典型增益范围 典型HPBW 带宽 (VSWR<2) 典型应用
2（1个有源+1个反射器） 5 - 7 dBi 50 - 60° 3 - 5% 室内TV天线
3（+1个引向器） 7 - 9 dBi 40 - 50° 3 - 5% 点对点通信
5（+3个引向器） 10 - 12 dBi 30 - 40° 2 - 4% 电视接收、业余无线电
10（+8个引向器） 13 - 15 dBi 20 - 30° 1 - 3% 远距离点对点、射电天文
15+（多引向器） 15 - 18 dBi 15 - 25° < 2% 专业通信、EMC测量
八木-宇田天线的优势与局限：结构简单、成本低廉、方向性好。但带宽窄（因为所有寄生振子的谐振频率都是精确设计的）、对机械公差敏感（毫米波频段尤其突出）。在UHF电视广播频段（470-862 MHz），八木-宇田天线几乎统治了家用接收天线市场数十年。但在今天的5G基站中，它已被宽带贴片阵列取代——因为5G需要更宽的带宽（几百MHz）和波束赋形能力（电调而非机械转向）。

下面我们用Python计算半波振子的阻抗并与其他天线对比：

import numpy as np

def dipole_impedance(L_over_lambda):
    """
    计算中心馈电偶极子的输入阻抗（近似公式，适用于 L ≈ 0.5λ）
    基于诱导EMF法（induced EMF method）的数值拟合
    参数:
        L_over_lambda : 偶极子总长度 (以λ为单位)
    返回:
        R_in, X_in : 输入电阻和电抗 (Ω)
    """
    # 半波振子 (L=0.5λ) 的理论值
    if abs(L_over_lambda - 0.5) < 0.001:
        return 73.1, 42.5

    # 通用的正弦电流分布近似
    kL = 2 * np.pi * L_over_lambda
    # 使用诱导EMF法公式（略去详细推导，直接给出数值实现）
    # 这里用简化的多项式近似
    R_in = 20 * (kL)**2 * (1 - 0.2 * np.sin(kL))
    X_in = 30 * (np.sin(kL) - np.cos(kL) / (kL))
    return R_in, X_in

# 计算不同长度偶极子的阻抗
print("偶极子长度 (λ) | R_in (Ω) | X_in (Ω) | 备注")
print("-" * 50)
for L in [0.1, 0.25, 0.4, 0.48, 0.5, 0.52, 0.6, 1.0]:
    R, X = dipole_impedance(L)
    note = ""
    if abs(L - 0.5) < 0.01:
        note = "← 半波振子"
    elif abs(L - 1.0) < 0.01:
        note = "← 全波振子 (高阻抗)"
    print(f"     {L:.2f}         | {R:6.1f}  | {X:6.1f}  | {note}")

# 关键发现
print("\n重要结论:")
print("1. 半波振子 (0.5λ): R_in ≈ 73 Ω, X_in ≈ +42 Ω (感性)")
print("2. 半波振子缩短至约 0.47-0.48λ 时可实现谐振 (X_in = 0)")
print("3. 全波振子 (1.0λ): 阻抗极高, 不适合直接馈电")
print("4. λ/4 单极子: Z_in ≈ 36.5 Ω (半波振子的一半)")

# 计算各种天线的辐射电阻
lam = 1.0
print("\n\n天线类型对比:")
print("-" * 60)
# 半波振子
print(f"半波偶极子 (λ/2):      R_rad = 73.1 Ω")
# 短偶极子
for dl in [0.01, 0.05, 0.1]:
    R_short = 80 * np.pi**2 * (dl)**2
    print(f"短偶极子 (dl={dl}λ):    R_rad = {R_short:.4f} Ω")
# 电小环
for C in [0.05, 0.1, 0.2]:
    R_loop = 20 * np.pi**2 * C**4
    print(f"电小环 (C={C}λ):       R_rad = {R_loop:.6f} Ω")

本章我们学习了五种最基本的天线类型：半波振子——天线世界的"标准语言"；单极子——利用镜像原理将偶极子"对折"到导电平面上，广泛应用于移动通信和广播；环天线——从电小环到电大环的不同行为，铁氧体加载的巧妙应用；缝隙天线——基于巴比涅原理的共形天线解决方案；八木-宇田天线——简单结构实现高指向性的经典设计。

这些基础天线是天线工程的"积木"。第7章我们将把它们组合成阵列——通过排列多个天线单元并控制它们的激励幅度和相位，实现更高的增益、更窄的波束和波束扫描的能力。天线阵列是几乎所有现代无线系统（5G Massive MIMO、相控阵雷达、卫星通信）的基础，是天线技术从"单打独斗"走向"协同作战"的关键跃迁。
