引子：央行行长的"工具箱"

想象一下，你刚被任命为中国人民银行行长。当前中国经济面临三重压力：需求收缩、供给冲击、预期转弱。你的任务是——通过货币政策来稳增长、促就业、保持物价稳定。

你手里有哪些"武器"可用？它们各自的效果如何？有什么副作用？什么时候该用什么？

这一章，咱们就来打开央行的"工具箱"，一个一个地梳理其中的工具。从传统的"三大法宝"到中国特色的结构性工具，再到最前沿的利率走廊操作——看完这一章，你再看到"央行宣布降准0.5个百分点"的新闻时，就能立刻理解这意味着什么。

一般性货币政策工具

央行的"三大法宝"是所有货币政策工具的基石，也是各国央行通用的"标准武器"。

1. 公开市场操作

公开市场操作（Open Market Operations, OMO）是央行最灵活、最常用的工具。操作方式很简单：央行在金融市场上买卖国债（或其他合格证券），从而调节银行体系的准备金数量。

  • 央行买入国债 → 向银行注入准备金 → 增加基础货币 → 扩张货币供给
  • 央行卖出国债 → 从银行回收准备金 → 减少基础货币 → 收缩货币供给

在中国，央行的公开市场操作主要通过"逆回购"和"正回购"进行。逆回购是央行向一级交易商（主要是大型银行）购买证券并约定未来卖回，本质上是短期资金投放；正回购则是相反操作。2025年，中国的逆回购操作以7天期为主，利率（7天逆回购利率）已经成为中国最重要的政策利率。

公开市场操作的优点很突出：主动性强（央行掌握操作主动权）、灵活性高（可以每天操作、调整规模）、精准性好（可以定向针对特定市场或期限）。缺点是：需要发达的国债市场作为操作基础，而且对"量"的影响部分取决于银行的配合程度。

2. 存款准备金率

调整存款准备金率是"三大法宝"中最具冲击力的工具。它直接改变货币乘数：降准→货币乘数放大→货币供给增加；升准→货币乘数缩小→货币供给减少。

中国的存款准备金制度有一个特点：差别准备金率。大型银行（工农中建等）的准备金率比中小银行高。2025年，经过多轮降准后，大型银行的法定准备金率约为9.5%，中小型银行约为6.5%。这个水平在全球来看仍属中等偏上（欧洲央行约1%，美联储约0%），意味着中国仍有降准空间。

每次降准0.5个百分点，大约释放长期流动性5000-6000亿元。但降准的缺点是"钝器"——它影响所有银行，无法定向；而且信号意义强，容易引发市场过度解读。

3. 再贷款/再贴现

当商业银行流动性不足时，可以向央行申请借款——这就是再贷款。如果商业银行持有未到期的票据（商业汇票），可以将其卖给央行获得资金——这就是再贴现。

再贷款/再贴现兼有"量"和"价"的双重功能：央行可以通过调整再贷款/再贴现的利率（再贷款利率/再贴现率）来影响商业银行的借款成本；也可以通过调整额度来影响可借款的数量。

在中国，再贷款工具近年来经历了重大"升级"——从传统的流动性支持工具演变为结构性的产业支持工具。这就引出了咱们下一节的讨论。

中国特色的结构性工具

传统"三大法宝"的问题是：它们只能影响"总量"。不管是降准还是降息，释放的流动性会流向哪里？央行无法控制。资金可能流入房地产而不是制造业，可能流向大企业而不是小微企业，可能进入金融市场"空转"而不是支持实体经济。

这就是结构性货币政策工具登场的背景。中国央行在这方面是世界的先行者——最早大规模使用结构性工具的央行之一。

咱们来看看几种最重要的中国结构性工具：

  • MLF（中期借贷便利）：2014年创设，期限3个月至1年，是央行向商业银行提供中期资金的工具。MLF利率已经成为中国最重要的中期政策利率，是LPR（贷款市场报价利率）定价的"锚"。
  • SLF（常备借贷便利）：2013年创设，期限1-90天，是央行向商业银行提供短期资金的工具。SLF利率是利率走廊的上限（我们下文会详述）。
  • PSL（抵押补充贷款）：2014年创设，主要面向政策性银行（国开行、进出口银行、农发行），用于支持棚户区改造、水利工程、保障房建设等重点领域。2025-2026年，PSL再次大规模用于支持"三大工程"（保障性住房、城中村改造、平急两用基础设施）。
  • 支农支小再贷款：专门用于支持"三农"和小微企业的再贷款工具。2025年额度合计约3万亿元，利率比MLF低约1个百分点。
  • 科技创新再贷款：2022年创设，2025年总规模已达8000亿元，用于支持科技型中小企业融资。利率仅1.75%，远低于市场利率。

为什么叫"结构性"工具？因为它们不是"普降甘霖"，而是"精准滴灌"——设定特定的发放对象、用途要求、利率优惠，引导资金流向政策希望支持的领域。

⚠️ 关键认知： 总量工具解决"水多水少"的问题，结构性工具解决"水往哪流"的问题。2008年后全球央行的经验教训是：光放水不够，还要解决"传导"问题。中国央行在结构性工具上的创新，已经被许多新兴市场国家借鉴。

利率走廊机制

利率走廊（Interest Rate Corridor）是现代央行调控市场利率的核心制度。它不直接控制利率水平，而是通过设定一个"走廊"来引导市场利率在目标区间内波动。

咱们用中国版利率走廊来说明：

  • 天花板——SLF利率：商业银行如果缺钱，可以直接向央行申请SLF借款，利率是SLF利率。所以市场利率不可能显著高于SLF利率——否则银行就会直接找央行借钱。
  • 地板——超额准备金利率（IOER）：商业银行如果有闲钱，最低可以存在央行获得超额准备金利率。所以市场利率不可能显著低于IOER——否则银行就会把多余资金存回央行套利。
  • 中枢——DR007（存款类机构7天质押式回购利率）：这是央行密切关注的市场利率指标。央行通过公开市场操作（逆回购），使DR007围绕7天逆回购利率波动。

你可以把利率走廊想象成一个"培养皿"：SLF利率是盖子（上限），IOER是底板（下限），而市场利率（DR007）就是培养皿中的微生物种群——央行不需要直接控制每个微生物的数量，只要控制好温度和湿度（设置好走廊），微生物自然会在适宜的环境中生长。

2025年，中国的利率走廊大致是：SLF利率约3.0%（上限），7天逆回购利率约1.5%（政策利率中枢），超额准备金利率约0.35%（下限）。DR007基本在1.4%-1.7%之间波动，显示出非常平稳的运行态势——这说明利率走廊在发挥作用。

2025-2026年中国货币政策实操

说了这么多工具，在实际操作中它们是怎么组合使用的？咱们以2025-2026年的中国货币政策为例来看。

2024年底，央行货币政策定调从"稳健"转向"适度宽松"，这是14年来的首次重大转变。进入2025年，央行迅速推出一系列宽松操作：

  • 降准0.5个百分点：2025年1月，央行宣布全面降准0.5个百分点，释放长期流动性约1万亿元。这不是"挤牙膏"式的操作——一次性0.5个百分点，信号意义极强。
  • 降息10个基点（bp）：2025年一季度，7天逆回购利率从1.60%降至约1.50%，MLF利率从1.90%降至约1.80%。LPR随之下降：1年期LPR从3.0%降至2.7%左右，5年期以上LPR从3.6%降至3.0%左右。
  • 结构性工具余额持续增长：到2025年底，结构性货币政策工具总余额约5.9万亿元，同比增加约8000亿元。科技创新再贷款、碳减排支持工具、保障性住房再贷款等工具持续扩容。

这套政策组合拳的效果如何？2025年中国GDP增速约5.0%（达到预期目标），CPI在0.5%左右（仍然偏低）。货币政策成功"托底"了经济，但没有引发通胀——这说明政策的"度"把握得还可以。

// LPR（贷款市场报价利率）形成机制详解
// ======================================

// LPR = MLF利率 + 银行加点
// 其中：
//    MLF利率 = 央行中期政策利率（当前约1.80%）
//    银行加点 = 由18家报价行根据自身资金成本、市场供求等决定

// 计算示例（2026年Q1数据）：
const MLF_RATE = 1.80;  // %
const bankSpread = 0.90; // 银行加点幅度 (bp转换后为0.90%)

// 1年期LPR
const LPR_1Y = MLF_RATE + bankSpread;  // = 2.70%

// 5年期以上LPR（房贷锚）
// 通常在1年期LPR基础上加60-100bp
const LPR_5Y = LPR_1Y + 0.30;  // = 3.00%

// 实际贷款利率 = LPR + 银行对客户的信用加点
// 优质企业：LPR - 30bp = 2.70% - 0.30% = 2.40%
// 普通企业：LPR + 50bp = 2.70% + 0.50% = 3.20%
// 个人房贷：LPR_5Y + 0bp = 3.00%（首套优质客户）

// 2025年LPR下调历程：
// 2024年底：1Y=3.10%, 5Y=3.60%
// 2025年Q1：1Y=2.90%, 5Y=3.30% (首次降息10bp)
// 2025年Q2：1Y=2.70%, 5Y=3.00% (二次降息20bp)
// ↓↓↓

踩坑故事：2015年股灾——央行与监管的"协奏失败"

2015年夏天，中国股市经历了一场惊心动魄的过山车。上证指数从2014年的2000点一路飙升至2015年6月的5178点，然后在短短两个月内暴跌至2850点——超过40万亿市值蒸发。

这场危机背后有一个鲜为人知的货币政策故事。

2015年6月中旬，股市已经开始暴跌。证券监管部门（当时的证监会）希望央行降准降息、释放流动性来"托市"。但央行当时的考虑是：第一，货币政策应该关注实体经济而不是股市；第二，股市泡沫本身就是前期过度宽松引发的，现在应该让市场自己调整。

于是，在6月27日（周六），央行宣布的是"定向降准"——只降准0.5个百分点，而且是"定向"给"三农"和小微企业，不是全面降准。市场解读为央行不愿意大力救市，信心进一步崩溃。

更严重的是，在7月初的危急时刻，证监会通过窗口指导要求券商大量买入股票"维稳"，但央行并没有同步释放足够的流动性来支持这些买入。结果，券商自己也因为买入股票而资金紧张，反而加剧了市场的流动性枯竭。

等到7月8日央行终于宣布"无限量"提供流动性支持时，市场已经经历了最惨烈的一波杀跌。许多融资盘被强制平仓，大批投资者血本无归。

2015年股灾的教训是多方面的，但从货币政策角度而言，核心教训是：货币政策与监管政策必须协调配合。如果央行只想管自己的"通胀目标"而忽视金融稳定，当金融风险爆发时，最终央行还是要出手——而且出手成本更高、效果更差。

⚠️ 教训： 2015年之后，中国建立了宏观审慎评估体系（MPA），央行被赋予了更多的金融稳定职能。2018年国务院机构改革后，银监会和保监会合并，宏观审慎监管进一步强化。到2025年，央行还会定期参加国务院金融稳定发展委员会的会议，确保货币政策和监管政策的协调。这些制度变革的种子，都种在2015年的那场"教训"里。

好了，咱们已经弄清楚了央行手里的"工具箱"有什么、怎么用。但工具使用后，效果是怎么传导到实体经济的呢？央行降准降息之后，企业的融资成本真的会降低吗？降低的传导链条有多长？中间会不会"堵车"？

这些问题，就是咱们下一章——货币政策传导与效应——要回答的。
