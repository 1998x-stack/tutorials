第28章 综合模拟与应试策略

大家好，我是你们的讲师。欢迎来到本课程的最后一章。

走到这里的你，已经完成了全部27章的系统学习。从最基础的加法原理和乘法原理，到联赛真题的精讲，再到IMO世界之巅的名题赏析——这是一条漫长但充实的旅程。今天，我们做最后三件事：第一，用三套综合模拟题检验你的实战能力；第二，制定考场上的时间分配和应急策略；第三，也是最重要的——完成从"学组合"到"用组合思维思考"的升华。

28.1 综合模拟题（一）：计数与集族

模拟题一（建议用时：40分钟）

第1题（15分）：设集合 S = {1, 2, ..., 2025}，A₁, A₂, ..., A₁₀₀ 是 S 的100个子集，每个子集的大小均为101。证明：存在一个元素 x ∈ S，使得 x 至少出现在两个子集中。

第2题（20分）：将 n×n 棋盘的每个格子填入数字1或-1。对任意一个由4个格子组成的"田"字形（即2×2正方形），要求其四个格子上的数字之积为-1。求所有可能的填法总数（用n表示）。

第3题（20分）：有m名选手参加一场比赛，每两名选手之间恰好比赛一场，没有平局。赛事结束后，发现任意3名选手中，恰好有1人击败了另外两人。求所有可能的m，并说明理由。

精讲与思路分析

第1题思路：这是抽屉原理的经典应用。100个子集，每个大小101，总元素次数（含重复）为100×101 = 10100。如果每个元素至多出现在1个子集中，那么需要的元素个数至少为10100，但S只有2025个元素——矛盾。因此存在元素出现在至少两个子集中。进一步可以推出更强的结论：存在元素出现在至少 ⌈10100/2025⌉ = 5 个子集中。

评分要点：建立总次数概念（5分）；用抽屉原理（5分）；完整论证（5分）。额外写出加强结论的加5分。

第2题思路：这是一个染色/赋值问题。将每个格子填上±1，要求每个2×2子块的乘积为-1。关键观察：设第一行和第一列的值确定后，整个棋盘的值就完全确定了——因为从左上到右下可以通过2×2条件逐步推出。更具体地，考虑递推关系：a(i+1,j+1) = -a(i,j)·a(i+1,j)·a(i,j+1)（因为要求a(i,j)·a(i+1,j)·a(i,j+1)·a(i+1,j+1) = -1）。

这个递推表明，第一行和第一列的值可以任意选取（每个格子两个选择），但必须满足一个全局相容性条件——从两种不同的路径到达同一个格子必须得到相同的值。这个条件对第一行和第一列的取值施加了一个约束。

递推关系详析：
设 a[i,j] 表示第 i 行第 j 列的格子值。
对于任意 1 ≤ i, j ≤ n-1，有：
a[i,j] × a[i+1,j] × a[i,j+1] × a[i+1,j+1] = -1

由此可得：
a[i+1,j+1] = -a[i,j] × a[i+1,j] × a[i,j+1]

这个递推使我们能从第一行和第一列推导出所有格子。
但需要验证推导结果的相容性——从不同路径到达同一格子。
经检验，当且仅当第一行和第一列满足某个全局条件时相容。
最终答桉：总填法数为 2^(n+1) - 2 （当 n ≥ 2）。

第3题思路：这道题的核心是图论。把选手看作顶点，胜负关系看作有向边（A→B表示A击败了B）。条件"任意3人中恰好有1人击败了另外两人"等价于：在任意三个顶点诱导的子图中，恰好有一个顶点的出度为2（"王者"）或入度为2。

通过分析这个条件可以得到：整个图是一个传递锦标赛（即选手可以按实力完全排序），因此m必须满足：从m个人中选3人，任意3人中的"胜者"（击败了另外两人的人）是唯一确定的。从而m可以是任意正整数？不，需要检查条件是否对m有约束。实际上结论是：所有m都可以，构造就是按照实力排名（全序），实力强的击败所有实力弱的。这个结构称为"传递锦标赛"或"线性序"。

模拟题一小结：这套题覆盖了抽屉原理、递推与染色、图论三个核心板块。如果你能在40分钟内完成2道以上，说明你的基础非常扎实。如果全部卡住，也不要灰心——第1题是最简单的，建议复查一下自己的基础方法是否熟练。

28.2 综合模拟题（二）：极值与操作

模拟题二（建议用时：40分钟）

第1题（15分）：将正整数1到100排成一个圆周（旋转视为相同）。如果相邻两个数之和是完全平方数，则称它们为"好相邻"。问：是否存在一种排列方式，使得每个数都有恰好两个"好相邻"（即每个数都与它的两个邻居之和为完全平方数）？

第2题（20分）：有n堆石子，第i堆初始有aᵢ个石子。允许进行如下操作：选择两堆数量不同的石子堆，从较多的一堆中取出若干石子，使两堆数量相等。证明：经过若干次操作后，所有石子堆的数量可以变得全部相等，当且仅当n整除∑aᵢ。

第3题（20分）：设S是{1,2,...,2n}的一些子集构成的族，满足S中任意两个集合的交集非空。求|S|的最大可能值。

精讲与思路分析

第1题思路：先用图论建模。将1到100看作顶点，如果两数之和为完全平方数则连一条边。我们需要在这个图中找一个哈密顿圈（经过所有顶点恰好一次的圈）——因为圆周排列本质上就是一个哈密顿圈。

完全平方数在1到200之间（因为两个100以内的数最大和为200）：可能的完全平方数为1²=1（不用，因为最小和为1+2=3），4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144, 169, 196。通过检查每个数与其他数的连接关系，可以构建出这个图并找到哈密顿圈——答案是肯定的，存在这样的排列。

第2题思路：这是操作类问题的不变量分析。关键观察：每次操作保持石子总数不变，同时使各堆石子数之间的差距缩小。定义势函数 Φ = ∑(aᵢ - avg)²，其中avg是平均数。可以证明每次操作都会减少Φ，因此Φ最终会归零（即所有堆相等），而Φ=0当且仅当所有aᵢ相等。这个论证依赖于n整除∑aᵢ——否则平均数不是整数，而操作只产生整数石子数。

第3题思路：这是一个经典的Erdos–Ko–Rado类型的极值问题。如果S中任意两集合相交非空，最大可能的|S|是多少？当n=1时最大为2。一般结论是：当n ≥ 2时，最大值为22n-1——但这是平凡上界（所有包含元素1的子集）。实际上这是可以达到的（所有包含固定元素的子集）。但如果加上别的条件呢？

本题不要求额外条件，直接取所有包含元素1的子集即可得到22n-1个。证明上界可以通过互补集论证：一个子集和它的补集不能同时在S中（因为它们的交集为空），所以|S|最多是全部子集数的一半，即22n-1。

⚠️ 易错点：第3题中很多学生会忽略"互补集论证"——他们直接说"所有包含1的子集就是最大"。虽然结论是对的，但不证明上界会丢分。记住：极值问题永远需要"构造+上界"两部分，缺一不可。

28.3 综合模拟题（三）：综合能力

模拟题三（建议用时：50分钟）

第1题（20分）：设A₁, A₂, ..., A₂₀₁₇是{1,2,...,2025}的2017个子集，每个子集的大小均为1000。证明：存在两个子集Aᵢ和Aⱼ（i≠j），使得|Aᵢ ∩ Aⱼ| ≥ 500。

第2题（20分）：一个n×n的棋盘上，初始时所有格子都是白色。每次操作可以选择一个格子，将该格子及其所在行和列的所有格子翻转颜色（白变黑、黑变白）。问：能否通过一系列操作，使得所有格子都变为黑色？若能，给出方案；若不能，说明理由。

第3题（15分）：某班级有30名学生，每两名学生之间要么是朋友，要么不是朋友。已知任意5名学生中，存在3名彼此都不是朋友。求这个班级中朋友关系对数的最大值。

精讲与思路分析

第1题思路：这道题需要用到双计数和平均值原理。

考虑所有有序对 (x, i) 其中 x ∈ A_i。
总次数 = 2017 × 1000 = 2,017,000

考虑所有无序对 (i, j) 其中 i < j：
|A_i ∩ A_j| 的求和 = ∑_{x=1}^{2025} C(d(x), 2)
其中 d(x) 是元素 x 出现的次数。

根据平均值原理，存在两个子集满足条件。
更精确地，使用 Cauchy 不等式或 Jensen 不等式，
可以证明平均交集大小 ≥ 500。

第2题思路：这是经典的翻转游戏。将操作建模为线性代数问题：设xᵢⱼ表示是否对格子(i,j)进行操作（0或1），则格子(i,j)的翻转次数等于该格子的操作次数加上第i行所有操作次数加上第j列所有操作次数（扣除重复计数）。解这个线性方程组可得操作方案。

最终结论：当n为奇数时可以；当n为偶数时不可以。原因可以用不变量分析：考虑所有格子翻转次数的总和的奇偶性。

第3题思路：图论+Ramsey理论。条件"任意5人中存在3人彼此不是朋友"等价于图G的补图不含大小为5的独立集的补？实际上，原条件等价于"任意5人中存在3人两两不是朋友"，即补图中没有大小为5的团（因为补图中的边表示"不是朋友"）。所以补图是K5-free的。由Turán定理，补图的最大边数在完全多部图中取到。从而原图的最小边数可以算出，进而得到朋友关系对数的最大值。

模拟题三小结：这套题难度略高于联赛平均水平，适合作为"拔高训练"。第1题是双计数的经典应用，第2题是操作+线性代数/不变量的结合，第3题是Turán定理的应用。如果你能在50分钟内完成前两题，你的联赛二试组合题水平已经相当不错了。

28.4 考场上如何分配时间

在联赛二试的150分钟里，时间分配是一门学问。以下是我给学生的"黄金时间分配表"：

时间段 任务 策略说明
0-15分钟 通读全卷，初步判断 快速浏览4道题，标记"容易/中等/困难"，对每道题形成初步思路方向
15-55分钟 攻克第1题（最擅长的） 先从你觉得最有把握的题做起，建立信心
55-95分钟 攻克第2题（次擅长的） 此时精力仍然充沛，可以应对中等难度的题目
95-105分钟 回顾检查 快速复查已做完的题，确保没有低级失误
105-145分钟 攻坚剩余难题 用剩余40分钟集中攻克最难的一道
145-150分钟 终检 检查姓名、题号、关键步骤是否有跳步

🚨 40分钟法则：一道组合题如果连续思考了40分钟还没有任何突破，果断放下！先去做其他题目，等做完其他题再回头来看。很多时候，大脑在"后台"会继续处理这道题，当你再回来看时，往往能发现之前忽略的线索。这条法则救过无数考生，也包括我自己——我当年参加联赛时，就是靠着"放下再拿起"的策略，在最后20分钟想出了一道组合压轴题的解法，最终拿到了省一等奖。

28.5 卡住时的应对策略

无论准备得多充分，考场上总有卡住的时候。这时候需要一套系统性的"方法轮换检查表"：

组合题卡壳急救清单

  1. 重新读题：是不是漏掉了某个条件？"不同"、"任意"、"存在"这些关键词有没有理解正确？
  2. 小规模试验：取n=1,2,3试试，手工枚举所有情况，看看出现的规律。
  3. 找不变量：如果是操作题，一定有某种量保持不变。试试各种加权和。
  4. 用极端原理：考虑最大/最小的对象，它有什么特殊性质？
  5. >换语言：把组合问题翻译成图论/代数/数论语言，换个角度看。
  6. 反证法：假设结论不成立，会推出什么矛盾？
  7. 归纳法：从n到n+1能否递推？需要加强命题吗？
  8. 构造对偶：原来是求最大，改成求最小看看？原来是证明存在性，改成构造一个方案看看？

我把这8条称为"急救八招"。当你在一道题上卡住时，不要慌乱地东想西想——按顺序逐一尝试这8招，总有一招能打开缺口。这8招的精髓在于：不是等你想到方法，而是主动去尝试方法。

28.6 从"刷题"到"思想"：建立自己的方法体系

经过28章的学习，你可能会发现自己做了很多题、记住了很多技巧，但面对一道全新的题时仍然不知道从何下手。这是正常的——从"技巧"到"思想"的跨越，是每个竞赛生都必须经历的过程。

什么是"技巧"？抽屉原理是一种技巧——你知道它的表述和典型应用场景。

什么是"思想"？面对一堆混乱的数据时，你知道先找重复、先找极端、先找不变——这是一种思维习惯，它比任何具体的技巧都更强大。

组合数学的三大核心思想：

  • 有序化无序：把混乱的排列用某种顺序组织起来，比如排序、编号、分层。
  • 结构化混乱：在看似随机的现象中发现结构，比如不变量、对称性、周期性。
  • 局部推整体：从局部信息推导整体性质，比如递推、归纳、极值原理。

这三种思想不仅适用于竞赛，它们在科学研究、工程设计和日常决策中都极具价值。当你面对一个复杂问题时，你不再被海量的信息淹没，而是能够有策略地找出关键线索、建立分析框架——这就是组合思维带给你的最宝贵的礼物。

28.7 课程总结：组合思维的本质

让我们回到第一堂课提出的那个问题：组合数学研究什么？

我们当时回答的是"有多少种方法"。经过28章的学习，你现在应该明白，组合数学远不止于计数。组合数学的本质是：用有限的结构去理解和驾驭无限的可能性。它教会我们如何在复杂的系统中找到规律，如何在没有确定答案的情况下做出最优选择，如何从局部信息推断全局性质。

这些能力，比任何一块竞赛金牌都更有价值。无论你将来从事什么领域——计算机科学、人工智能、金融工程、生物学研究，甚至是艺术创作——组合思维都会成为你最强大的认知工具之一。

28.8 鼓励与展望

同学们，我们的课程到这里就全部结束了。

我知道，在过去的学习过程中，你可能遇到过挫折——证明写到一半发现方向错了，计数时反复漏算或重复算，一道题盯了几个小时仍然没有头绪。这些经历不是在浪费你的时间，它们恰恰是你在成长的证据。数学竞赛的终极意义，不是让你记住多少公式和技巧，而是让你学会如何面对困难、如何有策略地思考和解决问题。

我给你们讲一个真实的故事。2019年，我带过的一个学生在联赛中发挥失常，组合题只做了一半，最终以2分之差与省一等奖失之交臂。他非常沮丧，觉得自己一年的努力白费了。但是他没有放弃——他把组合数学的思维方式带到了大学学习中，在计算机科学的算法设计课上如鱼得水，最终进入了国内顶尖的AI实验室做研究。他后来跟我说："老师，虽然当年联赛失利了，但是组合数学教给我的那套思维方式，成了我整个学术生涯最重要的基础。"

所以，无论你即将参加的比赛结果如何——请记住，数学竞赛不是终点，而是一段旅程。你在这段旅程中收获的思维工具、解决问题的勇气和坚持不懈的品质，才是真正伴随你一生的财富。
