从一重积分到多重积分

大家好，我是你们的讲师。欢迎回到微积分课程。

回想一下，定积分 ∫ab f(x) dx 做了什么？它把一个区间 [a, b] 切成无限多个小段，每段长度 dx，在每个小段上取函数值 f(x)，然后把这些"小矩形"的面积全部加起来。最终得到的是曲线下方的面积。

现在我们把思路升级一下。如果我想求的不是曲线下的面积，而是曲面下的体积呢？比如说，一个山坡覆盖了一个矩形区域，每个点 (x, y) 处的高度是 f(x, y) —— 我想知道这个山坡有多少"土方量"。这就是二重积分（double integral）要解决的问题：

V = ∬D f(x, y) dA

其中 D 是 xy 平面上的一个区域，dA 是面积微元。它的思想仍然和一重积分一样：分割 → 近似 → 求和 → 取极限。只不过这次我们把区域 D 切成许多小方块（而不是小区间），每个小方块的面积是 dA，近似为一个柱体的体积 f(x, y) dA，再把所有柱体的体积加起来取极限。

我在大二学二重积分时，始终觉得"曲顶柱体的体积"这个概念太抽象了——曲面下的体积到底怎么切、怎么加？直到有一次去参观一个建筑工地，工程师用全站仪测量了一片地基的每个格点的高程数据，然后直接用"每个小格子面积 × 高程"加起来算出土方量。那一刻我恍然大悟：这不就是二重积分吗？不过我们数学课上用的是 Riemann 和 取极限，工地上用的是有限个格点的数值近似，但核心思想完全一样。

二重积分的定义与性质

严格来说，二重积分定义为 Riemann 和的极限：将区域 D 划分为 n 个小区域 ΔD_k，每个小区域的面积为 ΔA_k，在 ΔD_k 中任取一点 (ξ_k, η_k)，则：

∬D f(x, y) dA = limmax diam(ΔD_k)→0 Σk=1n f(ξ_k, η_k) · ΔA_k

如果这个极限存在，称 f 在 D 上可积（integrable）。和一元情形类似，连续函数在有界闭区域上一定是可积的。

二重积分的基本性质与定积分一一对应：

  • 线性性：∬ (af + bg) = a∬ f + b∬ g
  • 区域可加性：∬_{D₁∪D₂} f = ∬_{D₁} f + ∬_{D₂} f（当 D₁ 与 D₂ 不重叠时）
  • 保序性：若 f ≤ g，则 ∬ f ≤ ∬ g
  • 绝对值不等式：|∬ f| ≤ ∬ |f|
  • 中值定理：存在一点 (ξ, η) ∈ D 使得 ∬ f = f(ξ, η) · Area(D)

直角坐标下化累次积分

二重积分怎么算？答案只有一个：把它化成两个一重积分依次计算，这叫累次积分（iterated integral）。

X 型区域

如果区域 D 可以描述为 a ≤ x ≤ b，y₁(x) ≤ y ≤ y₂(x)（即上下边界是 x 的函数），那么：

∬D f(x, y) dA = ∫x=ab [ ∫y=y₁(x)y₂(x) f(x, y) dy ] dx

先对 y 积分（视 x 为常数），得到关于 x 的函数，再对 x 积分。

Y 型区域

如果区域 D 可以描述为 c ≤ y ≤ d，x₁(y) ≤ x ≤ x₂(y)（即左右边界是 y 的函数），那么：

∬D f(x, y) dA = ∫y=cd [ ∫x=x₁(y)x₂(y) f(x, y) dx ] dy

先对 x 积分，再对 y 积分。

来看一个具体例子：计算 ∬D xy dA，其中 D 是由 y = x、y = 0、x = 1 围成的三角形区域。

按 X 型（先 y 后 x）：
        D: 0 ≤ x ≤ 1, 0 ≤ y ≤ x

        ∬ xy dA = ∫01 [ ∫0x xy dy ] dx
        = ∫01 [ x · (y²/2)|0x ] dx
        = ∫01 (x · x²/2) dx
        = (1/2) ∫01 x³ dx
        = (1/2) · (1/4) = 1/8
先对谁积分？看区域形状
如果区域的上下边界能用 x 的显式函数表示（y₁(x) ≤ y ≤ y₂(x)），优先选择 X 型（先 y 后 x）。
如果区域的左右边界能用 y 的显式函数表示（x₁(y) ≤ x ≤ x₂(y)），优先选择 Y 型（先 x 后 y）。
如果两种方式都可以，选择积分表达式更简单的那种。

积分次序的选择

有时按一种次序积分非常困难（甚至无法用初等函数表示），但交换次序后却非常简单。这就是交换积分次序（changing order of integration）的技巧。

经典例子：计算 I = ∫01 ∫y1 ex² dx dy。

先看原来的次序：先对 x 从 y 到 1 积分，再对 y 从 0 到 1 积分。但是 ex² 的原函数不是初等函数——这条路走不通。

交换次序：原来的积分区域是 0 ≤ y ≤ 1，y ≤ x ≤ 1。等价地描述为 0 ≤ x ≤ 1，0 ≤ y ≤ x。所以：

I = ∫01 ∫0x ex² dy dx
        = ∫01 [ y · ex² ]|0x dx
        = ∫01 x · ex² dx
        = (1/2) ex² |01
        = (e - 1)/2

瞧，交换次序后问题变得如此简单！关键就是画出积分区域，然后从另一个方向重新描述它。

交换积分次序的步骤
1. 根据原积分限画出积分区域 D
2. 从另一个方向重新描述 D 的边界（X 型 ↔ Y 型）
3. 写出新的累次积分表达式
4. 计算

注意：交换次序时积分限不是简单互换——你需要重新确定边界函数！

极坐标变换与雅可比行列式

当积分区域是圆形、环形或扇形时，直角坐标的描述非常麻烦（边界涉及平方根和反三角函数）。这时候使用极坐标（polar coordinates）会大幅简化计算。

极坐标变换

x = r cos θ,   y = r sin θ
        r ≥ 0,  0 ≤ θ ≤ 2π

面积微元从直角坐标的 dA = dx dy 变为极坐标下的 dA = r dr dθ。多出来的一个因子 r 来自坐标变换的雅可比行列式（Jacobian determinant）：

J = ∂(x, y)/∂(r, θ) = | ∂x/∂r  ∂x/∂θ |
                           | ∂y/∂r  ∂y/∂θ |
                       = | cos θ   -r sin θ |
                         | sin θ    r cos θ |
                       = r(cos²θ + sin²θ) = r

雅可比行列式在坐标变换中扮演着至关重要的角色——它告诉你坐标变换前后"面积缩放"的比例。在极坐标下，远离原点的地方一个"小扇形"的实际面积大于 dr dθ，因为弧长 r dθ 随着 r 增大而变大了。这个 r 因子就是补偿这个缩放效应的。

极坐标下的二重积分公式：

∬D f(x, y) dx dy = ∬D' f(r cos θ, r sin θ) · r dr dθ

经典例子：计算 ∬D e-(x² + y²) dA，其中 D 是整个 xy 平面。

用极坐标变换：
        ∬ℝ² e-(x² + y²) dx dy = ∫02π ∫0∞ e-r² · r dr dθ

        先对 r 积分（令 u = r²，du = 2r dr）：
        ∫0∞ e-r² r dr = 1/2

        再对 θ 积分：
        ∫02π (1/2) dθ = π

        所以整个广义积分的值为 π。由此可得高斯积分：
        ∫-∞∞ e-x² dx = √π

这个结果在概率论和统计学中至关重要——它是正态分布归一化常数的基础。如果没有极坐标变换，这个看起来很简单的积分在直角坐标下根本无法直接计算。

坐标系 变量 面积/体积微元 适合场景
直角坐标 (x, y) dx dy 矩形区域、多项式函数
极坐标 (r, θ) r dr dθ 圆形、扇形、含 x² + y² 的函数
柱坐标 (r, θ, z) r dr dθ dz 旋转体、圆柱形区域
球坐标 (ρ, φ, θ) ρ² sin φ dρ dφ dθ 球体、锥体区域

三重积分的概念与计算

既然二重积分是"曲顶柱体的体积"，那么三重积分（triple integral）就是"四维空间中曲顶柱体的超体积"——虽然四维不可想象，但我们可以从物理角度理解：如果 f(x, y, z) 表示空间某点的密度，那么 ∭Ω f dV 就是物体 Ω 的总质量。

M = ∭Ω ρ(x, y, z) dV

其中 ρ 是密度，dV = dx dy dz 是体积微元。

三重积分的计算和二重积分类似——化成三次累次积分：

对于 Ω: a ≤ x ≤ b, y₁(x) ≤ y ≤ y₂(x), z₁(x, y) ≤ z ≤ z₂(x, y)
        ∭Ω f dV = ∫ab ∫y₁(x)y₂(x) ∫z₁(x,y)z₂(x,y) f(x, y, z) dz dy dx

从最内层开始，逐层向外积分。每一次积分时，外层变量都被视为常数。

例如，计算 ∭Ω x dV，其中 Ω 是由 z = 0、z = 1 - x - y 和坐标平面围成的四面体。

Ω: 0 ≤ x ≤ 1, 0 ≤ y ≤ 1 - x, 0 ≤ z ≤ 1 - x - y

        ∭ x dV = ∫01 ∫01-x ∫01-x-y x dz dy dx
        = ∫01 ∫01-x x(1 - x - y) dy dx
        = ∫01 x [ (y - xy - y²/2) ]01-x dx
        = ∫01 x · [(1-x) - x(1-x) - (1-x)²/2] dx
        = ∫01 x · (1-x)²/2 dx
        = (1/2) ∫01 (x - 2x² + x³) dx
        = (1/2) · (1/2 - 2/3 + 1/4) = 1/24
三重积分常见错误：积分限依赖关系搞错
三重积分的积分限是有依赖层次的：最内层的 z 限可以依赖 x 和 y，中间层的 y 限可以依赖 x，最外层的 x 限必须是常数。如果你把依赖关系写反了——比如在 z 的积分限里写出了 x 和 y 之外的自由变量——那计算结果一定是错的。

我有个同学在一次考试中，因为把三重积分的次序搞反了，导致计算一个球的体积得到了负数。他死活检查不出错误，最后发现是 z 的积分上限写成了 x 的函数，但球对称情形下应该是一个不含 x 的表达式。记住：从最内层到最外层，自由变量越来越少。

柱坐标与球坐标变换

和二重积分类似，三重积分也可以通过坐标变换来简化。常用的两种坐标系是柱坐标（cylindrical coordinates）和球坐标（spherical coordinates）。

柱坐标

柱坐标 = 极坐标 + z 轴：

x = r cos θ,  y = r sin θ,  z = z
        dV = r dr dθ dz

柱坐标适用于具有旋转对称性的区域——比如圆柱、圆锥、旋转抛物面。判断标准：如果被积函数或积分区域中含有 x² + y²（即到 z 轴的距离），柱坐标很可能是好选择。

球坐标

球坐标用三个参数描述空间点：到原点的距离 ρ、俯仰角 φ（与 z 轴正方向的夹角）、方位角 θ（在 xy 平面上的投影与 x 轴的夹角）：

x = ρ sin φ cos θ
        y = ρ sin φ sin θ
        z = ρ cos φ

        ρ ≥ 0,  0 ≤ φ ≤ π,  0 ≤ θ ≤ 2π
        dV = ρ² sin φ dρ dφ dθ

体积微元中的 ρ² sin φ 就是球坐标变换的雅可比行列式。注意：球坐标下的体积微元不是简单的 dρ dφ dθ，而是随着 ρ 和 φ 变化——离原点越远，实际体积越大。

球坐标适用于球对称的区域——球体、球壳、锥体。判断标准：如果被积函数或积分区域中含有 x² + y² + z²（即到原点的距离），球坐标几乎是唯一的选择。

经典例子：计算半径为 R 的球体的体积。

        球坐标下球体 Ω: 0 ≤ ρ ≤ R, 0 ≤ φ ≤ π, 0 ≤ θ ≤ 2π

        V = ∭Ω 1 dV = ∫02π ∫0π ∫0R ρ² sin φ dρ dφ dθ

        = ∫02π dθ · ∫0π sin φ dφ · ∫0R ρ² dρ
        = 2π · 2 · (R³/3)
        = (4/3)πR³

这就是大家熟知的球体积公式。注意三个积分是独立的（因为积分限都是常数），所以可以拆成三个独立积分的乘积——这正是球坐标的威力所在。

如何选择坐标系？一张决策表
· 区域是矩形/长方体 → 直角坐标
· 区域是圆/圆柱/旋转体 → 柱坐标
· 区域是球/锥体 → 球坐标
· 被积函数含 x² + y² → 柱坐标
· 被积函数含 x² + y² + z² → 球坐标
· 被积函数是多项式 → 直角坐标（因为坐标变换反而会引入根号和三角函数，使积分更复杂）

关于坐标变换，我印象最深的是做一道三重积分的题目：计算被平面截取的球体部分的质心。当时我用直角坐标尝试了三次，每次到中间一步积分就复杂到无法继续。后来一位师兄提醒我："你为什么不试试球坐标？球体天然就是球坐标的天下。"我换了球坐标，原来复杂的积分边界变成了简单的常数限，被积函数也变成了初等函数的乘积——十五分钟就解出来了。而之前用直角坐标折腾的三个小时，全是在和带根号的积分限搏斗。

这个故事告诉我们一个深刻的教训：在多元积分中，选择合适的坐标系比积分技巧本身重要十倍。选对了坐标系，你的积分限简洁、被积函数简单、计算流畅；选错了坐标系，你会在代数泥潭中越陷越深。

重积分是多元微积分中"积分"这一半的核心。它把一元定积分从"线"推广到了"面"和"体"，让我们能够计算二维区域上的累积量和三维空间中的总质量。下一章我们将进一步推广积分——从平面区域和空间区域推广到曲线和曲面上。你将看到，积分的舞台可以是一条弯曲的路径（曲线积分）或一张扭曲的曲面（曲面积分），而数学工具也随之变得更加优雅和强大。
