第13章 寡头与博弈论

13.1 寡头：少数人的游戏

想象一下，某个城市只有三家航空公司——你的对手不是成百上千家小企业，而是两三个和你一样的大块头。你做任何决策都必须考虑：他们会怎么反应？这就是寡头（Oligopoly）市场的本质——少数企业主导市场，彼此之间战略相互依存（Strategic Interdependence）。

寡头市场的特征很明显：

  • 少数企业：通常2-10家大型企业占据大部分市场份额。行业集中度（CR4或CR8）很高。
  • 进入壁垒高：需要巨额资本、技术专利、规模经济或政府许可才能进入。
  • 产品可以是同质的：钢铁、石油、水泥等行业的产品几乎没有差异。
  • 也可以是有差异的：汽车、手机、家电——品牌和功能各不相同。
  • 合谋与竞争并存：企业之间既可能合作（形成卡特尔提高价格），也可能激烈竞争（打价格战）。

典型寡头行业：航空（中国国航、东航、南航）、电信（移动、联通、电信）、汽车（丰田、大众、通用）、搜索引擎（谷歌、百度）。在这些市场中，一家企业的降价行为会引发连锁反应——这就是为什么分析寡头必须用博弈论（Game Theory）这个工具。

13.2 博弈论基础

博弈论是研究"多个决策者相互影响时的策略选择"的数学框架。每一个博弈都有三个基本要素：

  • 参与者（Players）：做出决策的主体。在寡头市场中就是那几家大企业。
  • 策略（Strategies）：参与者可以选择的行动方案。比如"降价"或"不降价"。
  • 支付（Payoffs）：每种策略组合带来的收益。通常是利润。

描述一个博弈的最直观工具是支付矩阵（Payoff Matrix）。来看一个最简单的例子：假设航空A和航空B都在考虑要不要降价。如果都降价，各自赚50亿；都不降价，各自赚80亿（因为票价高）；A降价而B不降价，A赚120亿而B只赚20亿（A抢走了B的旅客）。反之亦然。

这个支付矩阵可以用表格来表示：

航空公司B
不降价降价
航空公司A不降价A: 80亿, B: 80亿A: 20亿, B: 120亿
降价A: 120亿, B: 20亿A: 50亿, B: 50亿

现在问题来了：如果你是航空公司A的CEO，你会怎么选？

13.3 囚徒困境与纳什均衡

上面这个博弈的推理过程是这样的：从A的角度看，如果B不降价，我降价能赚120亿（vs 80亿）；如果B降价，我降价能赚50亿（vs 20亿）。无论B怎么选，我降价都是最优的——也就是说，"降价"是A的占优策略（Dominant Strategy）。同理，B也会选择降价。最终（降价，降价）成为均衡——各赚50亿。

但看到了吗？如果双方都选择不降价，各赚80亿，对所有人都更好。然而理性的个体决策导致了集体的次优结果。这正是囚徒困境（Prisoner's Dilemma）的核心——个⼈理性与集体理性的冲突。

纳什均衡（Nash Equilibrium）是一个更一般化的概念：在纳什均衡中，给定对手的选择，没有人有动机单方面改变自己的策略。在囚徒困境中，（降价，降价）就是唯一的纳什均衡——任何偏离都会让自己受损。

💡 纳什均衡的直观理解：想象四个人打麻将，每个人都想赢。纳什均衡就是所有玩家都"没法在不换牌的情况下改变打法来让自己赢更多"的一个状态。你不是在打你最理想的牌，你是在打"考虑到别人会怎么出牌后"的最优牌。

下面用代码来系统识别纳什均衡：

# 纳什均衡识别
# 用囚徒困境的支付矩阵来演示

import itertools

# 定义博弈
players = ['A', 'B']
strategies = {'A': ['Cooperate', 'Defect'],
              'B': ['Cooperate', 'Defect']}

# 支付: (A的收益, B的收益)
#        B合作       B背叛
# A合作  (3, 3)     (0, 5)
# A背叛  (5, 0)     (1, 1)
payoffs = {
    ('Cooperate', 'Cooperate'): (3, 3),
    ('Cooperate', 'Defect'):    (0, 5),
    ('Defect', 'Cooperate'):    (5, 0),
    ('Defect', 'Defect'):       (1, 1),
}

def is_nash_equilibrium(strategy_profile):
    """检查给定的策略组合是否为纯策略纳什均衡"""
    sA, sB = strategy_profile
    # 检查A是否有动机偏离
    for alt_sA in strategies['A']:
        if alt_sA != sA:
            # 比较当前收益 vs 单方面改变策略后的收益
            current = payoffs[(sA, sB)][0]
            alternative = payoffs[(alt_sA, sB)][0]
            if alternative > current:
                return False, f"A可以单方面改为{alt_sA}获得更高收益"
    # 检查B是否有动机偏离
    for alt_sB in strategies['B']:
        if alt_sB != sB:
            current = payoffs[(sA, sB)][1]
            alternative = payoffs[(sA, alt_sB)][1]
            if alternative > current:
                return False, f"B可以单方面改为{alt_sB}获得更高收益"
    return True, "是纳什均衡"

# 检查所有可能的策略组合
for profile in itertools.product(strategies['A'], strategies['B']):
    is_eq, reason = is_nash_equilibrium(profile)
    payoff = payoffs[profile]
    eq_mark = "✓ NE" if is_eq else "✗"
    name = f"(A={profile[0]}, B={profile[1]})"
    print(f"{name}  支付={payoff}  {eq_mark}  {reason}")
# 输出:
# (A=Cooperate, B=Cooperate)  支付=(3, 3)  ✗  A可以单方面改为Defect获得更高收益
# (A=Cooperate, B=Defect)     支付=(0, 5)  ✗  A可以单方面改为Defect获得更高收益
# (A=Defect, B=Cooperate)     支付=(5, 0)  ✗  B可以单方面改为Defect获得更高收益
# (A=Defect, B=Defect)        支付=(1, 1)  ✓ NE  是纳什均衡

13.4 合谋与卡特尔：OPEC的囚徒困境

面对囚徒困境，企业自然会想："我们能不能坐下来谈好，都别降价？"这种合谋（Collusion）行为——当它正式化时称为卡特尔（Cartel）——就是寡头企业试图克服囚徒困境的手段。

全球最著名的卡特尔是OPEC（石油输出国组织）。OPEC成立于1960年，目前有13个成员国，控制着全球约40%的石油产量和80%的已探明石油储量。OPEC的核心机制就是配额制度——每个成员国分配一个产量上限，通过集体限产来维持油价。

但OPEC面临一个永恒的囚徒困境：

  • 如果所有成员遵守配额：油价高，大家都赚得多——这是合作的最优结果。
  • 但每个成员都有动机偷偷超产：稍微多产一点，在油价不变的情况下，自己的收入就增加了。
  • 如果所有成员都超产：供应过剩，油价暴跌，大家都受损。

这就是为什么OPEC的配额协议经常被违反。沙特阿拉伯经常抱怨其他成员国"作弊"，而伊拉克、尼日利亚等国家则经常超产。每次OPEC会议都是合作与背叛的博弈——各方唇枪舌剑，最终达成一个"都能勉强接受但没人严格遵守"的协议。

2020年的价格战是典型案例：沙特和俄罗斯在减产谈判中谈崩，双方都大幅增产，导致油价暴跌至历史低位（甚至出现负油价）。直到疫情冲击下的油价崩溃让双方都意识到"再打下去谁也受不了"，才重新回到谈判桌前达成减产协议。

⚠️ 现实中的合谋是否违法？在大多数国家，企业之间达成价格协议或分配市场是违法的。美国《谢尔曼法》明确禁止"限制贸易的合谋行为"。2015-2019年，多家汽车零部件制造商因组成价格卡特尔被全球多国罚款超过50亿美元。反垄断执法的一个重要任务就是发现和打破企业之间的合谋行为。

13.5 重复博弈：囚徒困境可以被克服吗？

一次性的囚徒困境中，"背叛"是占优策略。但如果同样的企业要长期反复互动呢？在这种情况下，重复博弈（Repeated Game）提供了另一种可能——"以牙还牙"（Tit-for-Tat）策略可以促使合作出现。

以牙还牙策略很简单：第一回合合作，之后每回合模仿对手上一回合的行动。这意味着：

  • 如果你一直合作，我也一直合作，双方都获得较好的长期收益。
  • 如果你背叛了一次，我下一次就背叛——但你可以在之后再通过合作"修复"关系。
  • 长期合作的收益可能超过短期背叛的好处，特别是当未来很重要时（贴现率足够低）。

这解释了为什么OPEC虽然没有一个强制执法机构，但大多数时候成员国还是大致遵守配额——因为大家知道，今天的背叛会引发明天的报复，而石油是要长期卖的。这也解释了为什么寡头市场中的企业往往不会打毁灭性的价格战——他们明白今天的降价会引发竞争对手明天的报复，最终谁都赚不了钱。

13.6 从博弈到要素市场

寡头和博弈论为我们提供了一整套分析企业战略互动的工具。但与现实中大多数市场一样，企业的决策不仅涉及产品定价——它们还要决定雇佣多少人、使用多少资本、如何组织生产。这正是下一章的主题：生产要素市场。工资是如何决定的？为什么CEO的收入是普通工人的几百倍？劳动市场背后的经济学原理是什么？我们下一章见。
