第4章：容斥原理

大家好，我是你们的讲师。前面的三章我们学的是"正着数"——直接计算满足条件的情况数。但现实中的很多计数问题，直接数满足条件的太难了，反而是数"不满足条件的"更简单。比如：有多少个排列使得没有人坐在自己的座位上？有多少个1到100之间的数与30互素？这些问题用正面计数简直是噩梦，但用"反过来数"的思路就异常简单。这就是容斥原理大显身手的舞台。

容斥原理的核心思想就是一句话：加多了，减回来；减多了，再加回去。听起来像废话？别急——当这个思想用集合的语言精确表达出来时，它就成了组合数学中最强大的计数工具之一。

4.1 从两个集合到三个集合的递进

4.1.1 两个集合的容斥

先看最简单的形式。对两个集合 A、B：

|A ∪ B| = |A| + |B| - |A ∩ B|

道理很简单：把 A 和 B 的元素数加起来，A ∩ B 中的元素被重复算了一次，所以要减掉一次。

也可以反过来：

|A ∩ B| = |A| + |B| - |A ∪ B|

这种"反过来"的形式有时更有用——如果我们知道全集、知道各集合的大小和它们的并，就能求出交集。

4.1.2 三个集合的容斥

对三个集合 A、B、C：

|A ∪ B ∪ C| = |A| + |B| + |C|
             - |A∩B| - |A∩C| - |B∩C|
             + |A∩B∩C|

模式：加所有单个、减所有两两交、加所有三三交。

🔥 Venn图理解：画三个相互交叠的圆。数并集时——

  • 先加三个圆各自的大小（加了3次核心重叠区）
  • 再减三对两两重叠区（每块被减了1次，而核心区被减了3次）
  • 最后加回核心重叠区（它目前被加了3-3=0次，需要加回1次）

得到的结果：每个区域恰好被计数1次。加多了减，减多了加——就是这个道理。

4.1.3 一般公式

对 n 个集合 A₁, A₂, ..., A_n：

|⋃_{i=1}^{n} A_i| = ∑|A_i| - ∑|A_i ∩ A_j| + ∑|A_i ∩ A_j ∩ A_k| - ... + (-1)^{n-1}|A₁∩A₂∩...∩A_n|

或者用更紧凑的写法：

|⋃ A_i| = ∑_{k=1}^{n} (-1)^{k-1} ∑_{1≤i₁<i₂<...<i_k≤n} |A_{i₁} ∩ A_{i₂} ∩ ... ∩ A_{i_k}|

这个公式的正确性可以用数学归纳法证明，也可以用二项式定理证明——每种证法都很有启发性。但我认为最重要的不是背公式，而是理解它的两个使用步骤：

  1. 定义好"坏"集合：设 A_i 为"第 i 个条件被违反"的集合
  2. 计算交集大小：|A_{i₁} ∩ ... ∩ A_{i_k}| 是多少——通常它只依赖于 k 而非具体下标的组合（对称性）

当问题是对称的时候（即所有交集的大小只依赖于交集的个数），容斥的计算会大大简化。

4.2 经典错排问题 D_n

错排问题是容斥原理最经典的应用之一——"全错位排列"（也被称为"derangement"）是指 1,2,...,n 的一个排列 π，使得 π(i) ≠ i 对所有的 i 成立。也就是说：没有一个人坐在自己的座位上。

4.2.1 错排数的容斥推导

设 A_i 表示"第 i 个人坐在自己座位上"的那些排列的集合。我们要求的是"不在任何 A_i 中"的排列数目。注意：

  • A_i 的交集大小：固定其中 k 个人的座位，剩下 n-k 个人可以任意排列，所以任意 k 个集合的交集大小为 (n-k)!。
  • 由于对称性，任何 k 个集合的交集大小都相同，而这样的 k-集合有 C(n,k) 个。

由容斥原理：

D_n = n! - C(n,1)(n-1)! + C(n,2)(n-2)! - ... + (-1)^n C(n,n) 0!

将组合数展开：
C(n,k)(n-k)! = n!/(k!(n-k)!) × (n-k)! = n!/k!

所以：
D_n = n! × [1 - 1/1! + 1/2! - 1/3! + ... + (-1)^n/n!]

这就是著名的错排公式。实际上 D_n 非常接近 n!/e——因为 e^{-1} = 1 - 1/1! + 1/2! - 1/3! + ...，所以 D_n = ⌊n!/e + 0.5⌋（四舍五入到最接近的整数），这个逼近在 n≥1 时成立。

可以看到 D_n 和 n!/e 已经非常接近了。容斥原理在这里优雅地给出了一个有封闭形式逼近的精确公式。

4.2.2 错排的递归关系

错排数还有一个优美的递推关系（可以用容斥或直接组合推导）：

D_n = (n-1)(D_{n-1} + D_{n-2})

且 D₁=0, D₂=1。

这个递推也可以这样理解：考虑元素 1 被放在了第 k 个位置（k ≠ 1）：

  • 如果元素 k 恰好在第 1 个位置，那么剩下 n-2 个元素需要错排（D_{n-2} 种）
  • 如果元素 k 不在第 1 个位置，那么这 n-1 个元素（1,2,...,k-1,k+1,...,n）需要排到 n-1 个位置（第 1 个位置和除 k 之外的各位置），且恰好是一个错排问题（D_{n-1} 种）

k 有 n-1 种选择，故 D_n = (n-1)(D_{n-1} + D_{n-2})。

💡 两种思路的对比：容斥原理给出了错排的封闭公式，而递推给出了它的"生长规律"。在竞赛中——如果你需要直接计算某个 n 的 D_n，用公式更快；如果你要证明 D_n 的某个性质（比如奇偶性），用递推更方便。两种工具都要掌握。

4.3 欧拉函数 φ(n) 的组合推导

欧拉函数 φ(n) 表示 1 到 n 之间与 n 互素的整数的个数。你可能在数论中见过它的公式，但你知道吗——这个公式也可以通过容斥原理组合推导出来。

设 n 的素因数分解为 n = p₁^{a₁} p₂^{a₂} ... p_k^{a_k}，即 n 有 k 个不同的素因子。

设 A_i = { m | 1 ≤ m ≤ n, p_i 整除 m }，即所有能被 p_i 整除的数的集合。

那么 φ(n) 就是"不在任何 A_i 中"的数的个数，即全集大小减去 |A₁ ∪ A₂ ∪ ... ∪ A_k|：

φ(n) = n - |A₁ ∪ A₂ ∪ ... ∪ A_k|

由容斥原理：
φ(n) = n - ∑|A_i| + ∑|A_i ∩ A_j| - ∑|A_i ∩ A_j ∩ A_k| + ...

而 |A_i| = n/p_i
|A_i ∩ A_j| = n/(p_i p_j)
以此类推。

所以：
φ(n) = n - ∑ n/p_i + ∑ n/(p_i p_j) - ∑ n/(p_i p_j p_k) + ...
      = n × [1 - ∑ 1/p_i + ∑ 1/(p_i p_j) - ∑ 1/(p_i p_j p_k) + ...]
      = n × ∏_{p|n} (1 - 1/p)

这个推导展示了容斥原理和数论的深刻联系。

4.4 禁位排列问题

错排是"每个人都不坐自己的座位"——这是一个特定的禁止位置。但更一般地，我们可以禁止任意位置的组合。这种问题称为禁位排列（或"有限制排列"）。

4.4.1 有禁区的排列

在 n×n 的棋盘上，放 n 个车（rook），使得没有两个车在同一行或同一列。如果没有任何限制，这就是 n! 种放法。但如果我们禁止某些位置放车（比如某些格子不能放），问题就变成了禁位排列。

设棋盘上有 m 个"禁止"格子。我们需要计算：有多少种放 n 个车的方式，使得没有任何车被放在禁止格子上？

这正是容斥的用武之地：

设 B_i = "第 i 个禁止格子上有车" 的排列集合

总数 = n! - ∑|B_i| + ∑|B_i ∩ B_j| - ...

但要注意：两个禁止格子如果在同一行或同一列，它们不能同时被放车，这种情况下的交集大小为 0。所以计算时需要对"兼容"的禁止格子组合求和。这个问题可以用"棋子多项式"系统求解，但在联赛中通常只需处理小规模的情况。

4.4.2 例题：夫妻入座问题

这是禁位排列的一个经典变种：有 n 对夫妻参加宴会，圆桌上有 2n 个座位，要求每对夫妻不能相邻而坐。有多少种坐法？

这个问题的完整解答要用到容斥原理——对"有多少对夫妻相邻"进行分类讨论。最终的公式是：

答案 = ∑_{k=0}^{n} (-1)^k C(n,k) × 2^k × (2n-1-k)! × (2n)/(2n-k)

这里不展开详细推导，但你可以感受到——当问题的不利条件不独立（夫妻关系导致依赖）时，容斥原理就成了为数不多的有效工具。

4.5 一个真实的踩坑故事

我来说一个自己当年参加竞赛时的教训。那是一道省队选拔题，题目是这样的：

"用 1,2,3,4,5,6 组成一个六位数，要求每位数字恰好出现一次。有多少个这样的六位数，使得数字 i 都不在第 i 位（i=1,2,3,4,5,6）？"

这不就是错排 D₆ 吗？我心中窃喜。错排公式 D₆ = 6! (1 - 1/1! + 1/2! - 1/3! + 1/4! - 1/5! + 1/6!) = 720 × (1 - 1 + 1/2 - 1/6 + 1/24 - 1/120 + 1/720) = 720 × (265/720) = 265。我很快就写完了。

但旁边一个同学写了完全不同的答案。我们一讨论才发现——我审题出错了！题目说的是"数字 i 都不在第 i 位"，但它是六位数，所以第一位是十万位，第六位是个位。而我的理解是"数字1不在第一位，数字2不在第二位……"，这正是错排，没错啊？

然后监考老师走过来，提醒了一句："六位数，首位可以是0吗？"

我恍然大悟！题目没有说首位不能为0——但六位数的首位不能是0！也就是说，当数字 6 出现在第一位时，这个"六位数"实际上变成了一个五位数（因为首位是0时在数值上不算）。这道题真正的限制比标准错排多了一个：数字 6 不能出现在第一位——因为如果 6 出现在第一位，那第一位就是 0（0 不在给定数字中），所以这其实是一个"禁位排列"问题，而非单纯的错排。

⚠️ 踩坑实录：我最终在这道题上丢了几乎所有分，不是因为容斥原理想错了，而是因为没注意到题目中的隐含条件。这个教训让我明白：用容斥原理之前，先确保你理解了"全集"到底是什么。全集定义错了，后面的一切都是徒劳。

从此以后，每次做容斥原理的题目，我都会在草稿纸上先写下三行：

  1. 全集 U 是什么？（所有可能的情况）
  2. "坏"集合 Aᵢ 是什么？（哪些情况我不想要）
  3. 各交集 |Aᵢ ∩ Aⱼ| 是否对称？（能不能用组合数简化）

这"容斥三问"帮我避免了很多低级失误。

4.6 容斥原理的更多应用

4.6.1 有上下界的整数解

第五章我们会详细讲隔板法。但当一个方程要求 x_i 既有下界又有上界时——比如 x₁+x₂+x₃=20，且 1≤x₁≤5, 2≤x₂≤8, 0≤x₃≤10——容斥原理和隔板法的结合就成了标准解法。

思路：先处理下界（变量替换），再用容斥处理上界违反的情况。这是联赛和CMO的高频考点。

4.6.2 素数分布

用容斥原理可以计算"不超过 N 且与 N 互素的数的个数"（即欧拉函数），这我们已经做了。更进一步，可以用来计算"不超过 N 的素数个数"的近似——这就是素数定理的组合证明思路之一。

4.6.3 染色问题中的容斥

用 k 种颜色给 n 个区域染色，要求相邻区域不同色。这可以用容斥原理：先不管限制全涂（k^n 种），减去有相邻同色的情况，再加回有两对相邻同色的……。最终得到的结果就是"色多项式"——图论中的经典概念。

🔥 容斥原理的核心思维模式：

很多初学者觉得容斥原理难，是因为他们总想"正面直接计数"。但容斥原理的真正优势在于——把"每个 A_i 都不能出现"转化为"允许一些 A_i 出现，但用加减符号修正"。后者看似增加了复杂度，但因为它将"整体约束"转化为"局部约束的自由组合"，反而在很多情况下可以逐层击破。

记住：当我们对"完全满足所有条件"束手无策时，试着先放弃一些条件——这就是容斥原理的精髓。

4.7 本章小结与过渡

今天我们学习了容斥原理从两个集合到 n 个集合的递进、经典错排问题 D_n 的两种推导（容斥公式和递推公式）、欧拉函数的容斥推导、禁位排列问题，以及容斥原理在整数解、素数分布和染色问题中的扩展应用。

容斥原理是组合数学中最具"策略性"的工具之一——它不是机械的公式套用，而是需要你精心设计"坏"集合。好的"坏"集合能让计算变得对称而简洁；差的"坏"集合则会让容斥层层嵌套，复杂度爆炸。下一章我们将学习隔板法与整数解，它和容斥原理几乎是最佳搭档——当你学会了它们俩的组合用法，联赛中的组合计数题你将再无畏惧。我们第五章见。
