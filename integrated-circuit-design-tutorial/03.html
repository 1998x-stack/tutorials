第3章 MOS晶体管原理

大家好，我是你们的讲师。上一章我们从半导体物理出发，理解了PN结、掺杂和MOS电容器。现在我们要把这些底层知识串联起来，去认识那颗真正推动数字时代前进的核心元件——MOS晶体管。

你每天写Verilog的时候，"assign"和"always"语句最终都会被EDA工具翻译成由成千上万个MOS晶体管组成的电路。不理解这个"积木"是什么，你就很难理解为什么某些电路的延迟会比预期大，为什么功耗在某些条件下会失控，为什么工艺从28nm迁移到7nm时你的设计方法需要改变。所以这一章，我们要把MOSFET吃透。

3.1 MOSFET基本结构

四端器件

一个MOSFET（Metal-Oxide-Semiconductor Field-Effect Transistor）有四个端子：源极（Source）、漏极（Drain）、栅极（Gate）和体（Body/Bulk）。在数字电路中，体通常连接到固定电位（NMOS接GND，PMOS接VDD），所以我们经常说"MOSFET是压控电流源"——栅极电压控制源漏之间的电流。

MOSFET的核心工作机理是场效应：栅极施加电压时，在栅氧化层下方的硅表面形成一个电场，这个电场在源和漏之间"感应"出一个导电沟道——就和上一章我们讲的MOS电容器的反型层是一个道理。

NMOS vs PMOS：

  • NMOS：P型衬底，N+源漏。栅极加正电压时，在P型衬底表面感应出N型反型层（电子沟道）。载流子是电子。
  • PMOS：N型衬底，P+源漏。栅极加负电压时，在N型衬底表面感应出P型反型层（空穴沟道）。载流子是空穴。

电子的迁移率大约是空穴的2-3倍，所以同样尺寸下NMOS的驱动能力更强。这也是为什么在CMOS电路中，PMOS通常需要做更大的宽长比（W/L）来与NMOS匹配——典型比例是PMOS的宽度是NMOS的2-3倍。

沟道的形成

当VGS（栅源电压）超过阈值电压VTH时，栅氧化层下方的半导体表面发生反型，形成导电沟道，将源极和漏极连接起来。此时如果在源漏之间施加电压VDS，就会产生从漏到源的电流ID。

记住这个类比：MOSFET就像一根水龙头水管。栅极电压是"旋钮"——你拧到一定程度（超过VTH），水开始流；拧得越大，水流越粗（电流越大）。但水能流多快还受到"水管粗细"（沟道宽度W）和"水龙头到出水口的水压差"（VDS）的限制。这个类比不是完美的，但对于初学者理解MOSFET的"压控"特性已经足够。

3.2 电流-电压特性

线性区与饱和区

MOSFET的I-V特性可以分为三个区域——截止区、线性区和饱和区：

  • 截止区（Cut-off Region）：VGS < VTH，沟道未形成，ID ≈ 0（实际上有亚阈值漏电流，见下节）。
  • 线性区（Linear/Triode Region）：VGS > VTH 且 VDS < VGS - VTH（即VDS较小）。沟道连续，电流与VDS近似呈线性关系。这时沟道像一个由栅压控制的电阻：VGS越大，"电阻"越小。
  • 饱和区（Saturation Region）：VGS > VTH 且 VDS ≥ VGS - VTH（即VDS较大）。当VDS增大到一定程度时，漏端的沟道被"夹断"（pinch-off），电流不再随VDS显著增大——进入饱和状态。

MOSFET漏电流的基本方程（长沟道模型，也称Shichman-Hodges模型）如下：

截止区 (V_GS < V_TH):
  I_D = 0

线性区 (V_GS > V_TH, 且 V_DS < V_GS - V_TH):
  I_D = μ · C_ox · (W/L) · [(V_GS - V_TH) · V_DS - (1/2) · V_DS²]

饱和区 (V_GS > V_TH, 且 V_DS >= V_GS - V_TH):
  I_D = (1/2) · μ · C_ox · (W/L) · (V_GS - V_TH)² · (1 + λ · V_DS)

其中：
  μ       = 载流子迁移率 (cm²/V·s)
  C_ox    = 单位面积栅氧化层电容 = ε_ox / t_ox
  W/L     = 晶体管宽长比
  V_TH    = 阈值电压
  λ       = 沟道长度调制系数（饱和区的有限输出电阻）
  (1 + λ·V_DS) = 沟道长度调制修正项

跨导与输出电阻

两个对于电路设计非常重要的参数：

  • 跨导 gm（transconductance）：gm = ∂ID/∂VGS。它衡量的是"栅压变化1V能引起多大漏电流变化"——通俗地说，就是晶体管的"放大能力"。在饱和区，gm = μ·Cox·(W/L)·(VGS - VTH)。
  • 输出电阻 ro（output resistance）：ro = ∂VDS/∂ID。在饱和区，由于沟道长度调制效应，ID随VDS略有增加，所以输出电阻是有限的——ro ≈ 1/(λ·ID)。

对于模拟IC设计师来说，gm和ro的乘积——本征增益（intrinsic gain）Av = gm · ro——是衡量一个晶体管放大能力的重要指标。在短沟道器件中，λ变大，ro下降，本征增益随之降低，这也是先进工艺对模拟设计越来越不友好的原因之一。

3.3 亚阈值特性

你可能以为当VGS < VTH时，MOSFET就完全关断了，电流为零。真实情况远非如此——亚阈值区域（subthreshold region）是现代低功耗芯片设计中最重要的战场之一。

亚阈值摆幅

当VGS低于VTH时，MOSFET并没有完全关闭，而是存在一个指数衰减的"亚阈值电流"（也叫弱反型电流）。这个电流近似为：

ID,sub ∝ eq·(VGS - VTH) / (n·kT)

衡量亚阈值特性的关键指标是亚阈值摆幅SS（Subthreshold Swing），定义为使亚阈值电流变化10倍所需的栅压变化量：

SS = (∂log10(ID) / ∂VGS)−1 = n · (kT/q) · ln(10) = n · 60mV (在300K时)

理想情况下，n ≈ 1，SS ≈ 60mV/dec —— 这是室温下的物理极限，无法突破（除非使用隧穿晶体管等新型器件）。实际器件中，n > 1，所以SS通常在70~100mV/dec之间。SS越小，晶体管开关越陡峭，关断特性越好，静态功耗越低。

漏电流组成

在深亚微米工艺中，晶体管的"关断"状态存在多种漏电流分量：

  • 亚阈值漏电 Isub：VGS < VTH时的弱反型电流。这是最主要的漏电来源。
  • 栅极漏电 Igate：栅氧化层太薄（~1nm量级），量子隧穿导致栅极电流。HKMG（高k金属栅极）技术通过物理厚度更大的高k材料显著降低了这一漏电。
  • GIDL（Gate-Induced Drain Leakage）：栅-漏交叠区的强电场导致的漏极隧穿电流。在低功耗设计中尤其需要注意。
  • PN结反向漏电：源/漏与衬底之间的PN结在反向偏置下的漏电流。
一个真实的5nm芯片设计教训：2022年参与的一个AI加速器项目，我们在5nm工艺节点下设计了一个大规模的近阈值计算阵列（near-threshold computing）。在仿真的"理想世界"里，整个阵列在0.5V供电下完美工作，功耗极低。但流片回来后，待机功耗比仿真值高了约30%。最后定位到的原因是：我们低估了亚阈值漏电的工艺角（process corner）分布——在FF（Fast-Fast）工艺角、125°C高温条件下，亚阈值漏电比典型条件下高了超过10倍。而我们的电源门控（power gating）策略没有充分考虑到这个极端情况。最终不得不通过增加电源门控晶体管的尺寸来修复——这又增加了面积。这件事的教训是：在先进工艺下，亚阈值漏电不是"小问题"，而是需要和时序、功耗一起作为一级设计约束来对待。

3.4 短沟道效应

当我们把MOSFET的沟道长度从几十微米缩小到几十纳米时，很多在长沟道模型中可以被忽略的效应变得不可忽视。这些统称为短沟道效应（Short Channel Effects, SCEs）。

  • 速度饱和（Velocity Saturation）：在短沟道中，沟道电场极高（可达105 V/cm），载流子的漂移速度不再随电场线性增长，而是趋近于饱和速度vsat（硅中约107 cm/s）。结果就是：饱和电流与(VGS-VTH)的关系从平方律退化为近似线性关系，gm降低。
  • 阈值电压滚降（VTH Roll-off）：沟道长度越短，源漏耗尽层对沟道的"侵蚀"越显著，导致VTH下降。同样宽度下，短沟道器件的VTH明显低于长沟道器件。如果设计不当，沟道非常短的器件可能在VGS=0时就关不断。
  • DIBL（Drain-Induced Barrier Lowering）：漏极电压通过降低源端势垒来"帮助"载流子注入沟道，导致VTH随VDS增大而降低。DIBL是短沟道器件中最棘手的效应之一，直接影响模拟电路的输出电阻和数字电路的关断特性。
  • 热载流子效应（Hot Carrier Effect）：在高电场下，载流子获得足够高的能量，注入栅氧化层或破坏Si-SiO₂界面，导致阈值电压漂移和跨导退化。这是影响器件长期可靠性的重要因素。
  • 穿通（Punchthrough）：当源极和漏极的耗尽层在沟道中"相遇"时，无论栅压如何，源漏之间都会存在大电流。这是晶体管的硬失效——通常通过增加沟道掺杂或使用halo/pocket注入来防止。

3.5 FinFET与GAAFET

当工艺节点进入20nm以下，平面MOSFET的短沟道效应变得无法控制——特别是亚阈值摆幅恶化、DIBL严重、漏电失控。产业界需要一个根本性的结构变革。

FinFET的登场

2011年，Intel在22nm节点首次商用推出了FinFET（鳍式场效应晶体管），这是一个划时代的结构创新。与传统平面晶体管不同，FinFET的沟道从"水平躺倒"变成了"竖直站立"——一个形似鱼鳍（fin）的硅片从衬底上凸起，栅极从三面包裹这个鳍片（因此也称为Tri-gate晶体管）：

  • 三面栅极控制取代了平面器件的单面栅极控制，栅控能力大幅增强。
  • 沟道变"薄"（鳍的宽度，典型值5-8nm），有效遏制了短沟道效应。
  • DIBL和SS显著改善——即使沟道长度缩短到十几纳米，漏电仍然可控。
  • 每个"鳍"提供固定的驱动电流（约数百μA），设计师通过并联鳍数来调整驱动能力。

GAAFET：下一代

FinFET在7nm和5nm节点继续演进，但到了3nm以下，FinFET也接近了物理极限——当鳍片之间的间距缩小到极致时，栅极对沟道的控制又开始减弱。产业界开始转向GAAFET（Gate-All-Around FET）：

  • Samsung在2022年率先量产的3nm GAE（Gate-All-Around Early）工艺中采用了纳米片（Nanosheet）结构——栅极从四面完全包裹水平堆叠的纳米片沟道。
  • 相比FinFET，GAAFET提供了更好的栅控能力（SS更低、DIBL更小），允许在保持低漏电的前提下进一步缩小沟道长度。
  • 纳米片的宽度可以调节（不同于FinFET的"固定鳍宽"），设计师有了更多的性能-功耗权衡自由度。
  • 未来的演进方向包括ForkFET（叉片FET，间距更小）、CFET（互补FET，在同一位置堆叠NMOS和PMOS），密度将继续提升。

一个值得思考的事实：从Intel 22nm的FinFET到Samsung 3nm的GAAFET，中间经历了11年。而平面MOSFET从1960年代到2011年用了将近50年。这说明先进工艺节点的迭代速度虽然在放缓，但结构创新的"大招"还在不断释放。如果你在2025-2026年设计芯片，大概率会用FinFET（7nm-5nm）或GAAFET（3nm-2nm），理解这些结构的物理原理对你的设计决策至关重要。

3.6 本章小结

这一章我们系统地梳理了MOSFET从结构到原理、从长沟道模型到短沟道效应、从平面器件到FinFET再到GAAFET的完整知识体系。你理解了为什么MOSFET是一个"压控电流源"，学会了I-V方程中每个符号的物理意义，看到了亚阈值漏电为什么成为先进工艺的头号敌人，也知道了为什么产业界会在20nm以下从平面转向三维晶体管结构。

现在你握着这些知识，我们终于可以进入一个真正让人兴奋的部分——用这些东西搭建CMOS数字电路。下一章，我们将从最简单的CMOS反相器开始，学习如何用NMOS和PMOS组合在一起构建逻辑门，然后深入理解延迟、功耗和噪声容限——这些才是芯片设计中每天都要面对的工程问题。
