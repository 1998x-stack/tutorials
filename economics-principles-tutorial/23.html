第二十三章：总需求与总供给

大家好，我是你们的讲师。在前几章里，我们一直在讨论长期经济增长——什么因素让一个经济体在几十年尺度上变得更富裕。但如果你看新闻，经济学家讨论的往往是短期问题：这个季度GDP增长为什么放缓？失业率为什么突然上升？这把我们带到了一个全新的领域——短期经济波动。

这一章我们要引入宏观经济学中最重要的分析框架之一：AD-AS模型（总需求-总供给模型）。它就像是宏观经济学的"供需图"——只不过把微观里的"某种商品的价格和数量"，换成了"整体价格水平和总产出"。

23.1 经济波动的特征

首先，我们需要理解短期经济波动的三个基本特征。第一个特征是不规则性——经济衰退的发生时间没有固定的周期。不像一年有四季，经济不是每四年或每十年必然衰退一次。第二个特征是不可预测性——绝大多数衰退都是在事后才被确认的，极少有经济学家能准确预测衰退的时点。第三个特征是传播的广泛性——当衰退发生时，它几乎会波及经济体的所有部门，从制造业到服务业，从就业到消费。

关于预测：美联储前主席艾伦·格林斯潘有一个著名观点——经济学家预测衰退的能力并不比气象学家预测地震强多少。在过去的50次美国衰退中，只有不到10%在发生前被主流经济学家明确预测到。不是经济学家不聪明，而是经济系统本质上是一个复杂适应系统，充满了反馈回路和非线性效应。

那么，什么工具可以帮我们分析这些波动？答案是——AD-AS模型。

23.2 总需求曲线（Aggregate Demand, AD）

总需求（Aggregate Demand）是一个经济体中所有最终商品和服务的需求总和。从国民收入核算恒等式出发：

Y = C + I + G + NX

消费（C）、投资（I）、政府购买（G）和净出口（NX）共同构成了总需求。AD曲线描绘的是在每一价格水平下，经济体愿意购买的商品和服务总量。它是一条向右下倾斜的曲线——价格水平越低，总需求量越大。

为什么AD曲线向右下倾斜？和微观经济学里的需求曲线不同，这里的原因要微妙得多。主要有三种效应：

第一，财富效应（Wealth Effect）。当整体价格水平下降时，你手里持有的货币和银行存款的实际购买力上升了——你"感觉"更富有了，于是消费更多。这被称为"庇古效应"（Pigou Effect），以经济学家阿瑟·庇古命名。

第二，利率效应（Interest Rate Effect）。价格水平下降意味着人们需要持有的名义货币量减少。多余的货币会被存入银行或用于购买债券，这压低了利率。更低的利率刺激了企业和家庭的借贷与投资，从而增加了总需求。这个传导链条是：P↓ → 货币需求↓ → 利率↓ → I↑和C↑ → Y↑。

第三，汇率效应（Exchange Rate Effect）。当本国价格水平下降时，本国利率也会下降（如上所述）。本国利率相对外国利率的下降，会导致本币在外汇市场上贬值。本币贬值意味着出口商品更便宜、进口商品更贵，因此净出口增加，总需求上升。传导链条：P↓ → 利率↓ → 本币贬值 → NX↑ → Y↑。

关键区分：以上三种效应解释的是"沿着AD曲线的移动"——价格水平变动导致总需求量的变化。这和"AD曲线的整体平移"是两回事。AD曲线平移是由价格水平之外的因素引起的——比如消费者信心变化、财政政策变动、或者国外经济形势变化。

23.3 总供给曲线（Aggregate Supply, AS）

总供给描绘的是经济体在不同价格水平下愿意生产的商品和服务总量。关键在于：在长期和短期内，总供给曲线的形状完全不同。

长期总供给曲线（LRAS）是垂直的。在长期，一个经济体的产出由其实际生产要素决定——劳动、资本、自然资源和技术水平。价格水平对实际产出没有影响。LRAS的位置对应的是潜在产出（Potential Output）或自然产出水平（Natural Rate of Output）。当经济在LRAS上运行时，失业率等于自然失业率——经济学家称之为"充分就业"状态。

短期总供给曲线（SRAS）是向右上倾斜的。为什么短期内有正斜率？这里有三种主要的理论解释：

  • 黏性工资理论（Sticky-Wage Theory）：名义工资在短期内难以调整。当价格水平上升时，名义工资不变，这意味着实际工资下降，企业愿意雇佣更多工人、生产更多产品。
  • 黏性价格理论（Sticky-Price Theory）：并非所有企业都能即时调整价格。当整体价格水平上升时，那些没有及时调价的企业相对变得便宜，吸引了更多需求，从而扩大生产。
  • 错觉理论（Misperceptions Theory）：生产者可能将整体价格水平的上升误认为是自己产品相对价格的上升，从而增加供给。这个理论由米尔顿·弗里德曼提出。

重要提醒：这三种理论都承认一个核心事实——在长期，所有这些"黏性"和"错觉"都会消失。工资会重新谈判，价格会全部调整，生产者会看清真实情况。这就是为什么长期总供给曲线是垂直的，而短期总供给曲线向右上倾斜。两者并不矛盾——它们回答的是不同时间维度上的问题。

23.4 AD-AS模型中的冲击与稳定

当AD曲线或AS曲线受到外部冲击而发生移动时，经济就会偏离其长期均衡，产生波动。我们可以区分两种类型的冲击：

需求冲击（Demand Shock）：AD曲线移动。例如，消费者信心骤降导致消费减少（AD左移），短期内产出下降、价格水平下降。在长期，由于预期调整，SRAS会向右移动，经济回归潜在产出水平——但这个过程可能需要数季甚至数年。

供给冲击（Supply Shock）：AS曲线移动。例如，油价暴涨导致生产成本上升（SRAS左移），短期内产出下降但价格水平上升——这就是滞胀（Stagflation）的典型特征。政策制定者面临两难：如果刺激总需求来增加产出，会导致更严重的通胀；如果紧缩来抑制通胀，会导致更深的衰退。

故事：2008年全球金融危机

2008年金融危机是一个典型的需求冲击故事。次贷危机爆发后，美国房屋价格崩盘，家庭财富蒸发超过10万亿美元。消费者信心崩溃，消费断崖式下跌；同时，银行体系濒临崩溃，信贷冻结，企业投资骤停。结果是AD曲线大幅左移。2008年第四季度到2009年第一季度，美国GDP以年化约6%的速度萎缩——这是大萧条以来最严重的衰退。

为什么经济没有"自我纠正"？理论上，如果工资和价格完全灵活，AD左移后很快就能恢复。但现实中，名义工资具有向下黏性——企业不愿意减薪，工人也不接受减薪。当AD左移时，调整更多通过"数量"（裁员）而不是"价格"（减薪）来实现。这就是为什么衰退伴随着大面积失业，而且恢复缓慢。美联储和财政部不得不通过大规模货币宽松和财政刺激来"推"AD曲线向右移动。

数据点：2007年到2009年，美国失业率从4.6%飙升到10.0%。即使到了2014年、也就是危机爆发六年后，失业率仍然在6%以上。AD-AS模型清楚地展示了为什么"总需求管理"在衰退期如此重要——市场自我纠正的机制在短期内远没有教科书描述的那么顺畅。

故事：2020年COVID-19双重冲击

2020年的新冠疫情带来了一种罕见的双重冲击。一方面，封锁措施和企业停工导致生产能力骤降——工厂关闭、餐厅停业、航空公司停飞——这是巨大的负向供给冲击（SRAS左移）。另一方面，恐慌和不确定性导致消费骤降、投资停滞——这是同样巨大的负向需求冲击（AD左移）。两个冲击叠加，造成了现代经济史上最急剧但也是最短暂的经济收缩。

2020年第二季度，美国GDP年化环比下降31.4%。但到了第三季度就反弹了33.4%——V型复苏。为什么这么快？因为双重冲击中的"供给冲击"主要是疫情封锁导致的，当封锁解除、生产恢复时，供给侧的反弹也非常迅速。同时，各国政府史无前例的财政刺激（美国CARES法案2.2万亿美元）和货币宽松（美联储"无限QE"）帮助支撑了总需求。

23.5 AD-AS均衡模型示例

以下是一个简化的AD-AS均衡数值模型。它可以帮助你理解冲击如何在模型中传导：

=== AD-AS 均衡模型 ===

参数设定：
  自发消费 (C₀) = 200
  边际消费倾向 (MPC) = 0.75
  投资 (I) = 150
  政府购买 (G) = 200
  税收 (T) = 150
  净出口 (NX) = 50
  潜在产出 (Y*) = 2000

AD曲线方程：
  Y = C₀ + MPC × (Y - T) + I + G + NX
  Y = 200 + 0.75(Y - 150) + 150 + 200 + 50
  Y = 200 + 0.75Y - 112.5 + 400
  Y - 0.75Y = 487.5
  0.25Y = 487.5
  Y = 1950（短期均衡产出）

LRAS = Y* = 2000（长期潜在产出）

分析：
  - 短期均衡产出(1950) < 潜在产出(2000)
  - 存在产出缺口 = 2000 - 1950 = 50（衰退缺口）
  - 在长期，预期调整将使SRAS向右移动
  - 或者：政府可增加G或减税来刺激AD

如果政府购买增加50（G = 250）：
  Y = 200 + 0.75(Y-150) + 150 + 250 + 50
  0.25Y = 537.5
  Y = 2150  →  超过潜在产出（通胀压力）

23.6 稳定政策的争议

既然AD-AS模型表明政府可以通过需求管理来平滑经济波动，那是不是应该积极使用？这里存在深刻的学术分歧，也是宏观经济学最经典的辩论之一。

干预派认为，市场自我纠正的速度太慢，成本太高。凯恩斯的名言"长期来看，我们都死了"正是这个意思。如果经济陷入衰退，通过货币政策或财政政策来稳定总需求是必要的。2008年和2020年的经验都支持这一立场。

自由放任派则认为，政策干预存在内在的问题：时滞（认识滞后、决策滞后、效果滞后）和政策制定者的信息不足。弗里德曼等经济学家主张"规则而非相机抉择"——比如设定一个固定的货币增长率，让市场自己调整。

现实中，大多数现代经济体采取了一种折中路线：在极端情况下积极干预，在正常时期依靠自动稳定器（比如失业保险、累进税制）来温和缓冲波动。中央银行普遍采用"通胀目标制"来为政策提供规则框架，同时又保留了相机抉择的空间。

好了，这一章我们搭建了理解经济波动的核心框架——AD-AS模型。下一章我们将深入讨论这个模型中最重要的政策工具之一：货币政策与央行。理解了央行如何影响总需求，你就掌握了宏观经济政策的一半精髓。
