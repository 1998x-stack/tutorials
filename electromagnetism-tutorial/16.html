开场：界面即世界

上一章我们学习的是电磁波在无限大均匀介质中的平面波传播——这是一个非常理想的模型。但在现实世界中，电磁波几乎总是在不同介质的交界面上传播的。水面为什么会反光？光纤为什么能传光？隐形战机为什么对雷达波"隐身"？这些问题的答案都指向同一个物理过程——电磁波在界面上的反射与折射。

从原理上说，反射和折射根植于第14章讨论的边界条件——在两种介质的界面上，电场和磁场的切向分量连续。这一看似简单的条件，在时变电磁场中会推导出极其丰富的结果：Snell定律、Fresnel反射系数、Brewster角、全内反射、驻波……

本章不仅理论优美，而且具有巨大的工程价值——从光纤通信到雷达隐身，从太阳能电池抗反射层到偏光墨镜，处处都是反射与折射的应用。

法向入射

最简单也最基础的情形：平面波从介质1（η₁）垂直入射到介质2（η₂）的平坦界面。在 z=0 处，一部分波反射回介质1，一部分透射进介质2。

定义反射系数和透射系数：

Γ = E_r / E_i = (η₂ − η₁) / (η₂ + η₁)

τ = E_t / E_i = 2η₂ / (η₂ + η₁)

其中 E_i、E_r、E_t 分别代表入射、反射、透射电场的幅度（切向分量）。注意 Γ 和 τ 之间的关系：1 + Γ = τ，这是由边界条件（电场切向连续）保证的。

与反射系数相关的几个重要工程参数：

  • 功率反射系数（反射率）：R = |Γ|²，表示入射功率中被反射的比例。
  • 功率透射系数（透射率）：T = 1 − R（无耗介质），由能量守恒保证。
  • 驻波比（VSWR, Voltage Standing Wave Ratio）：SWR = (1 + |Γ|) / (1 − |Γ|)，射频工程中最常用的匹配质量指标。

几个典型案例：

  • 波从空气（η₀≈377Ω）垂直入射到水（η_water≈η₀/√ε_r≈377/√81≈42Ω）：Γ = (42−377)/(42+377) ≈ −0.8，R = 0.64。64%的功率被反射！这就是平静水面会产生强烈镜面反射的原因——从斜上方看你几乎看不见水下的东西。
  • 波从空气入射到玻璃（η_glass≈377/√2.25≈251Ω）：Γ ≈ (251−377)/(251+377) ≈ −0.2，R ≈ 0.04。一扇玻璃窗约反射4%的入射光，这个数字虽然小，但它足以让你在暗房中看清窗外的夜景（室内亮、室外暗时，这4%的反射足以形成清晰的镜像）。

斜入射与Snell定律

当平面波以入射角 θ_i（相对于法线）斜入射到界面时，几何关系变得丰富起来。核心思想是相位匹配（Phase Matching）：在界面上任何一点，入射波、反射波、透射波的相位必须一致，否则边界条件无法在整个界面上处处满足。

相位匹配条件直接导出了两个基本定律：

  • 反射定律：θ_r = θ_i（入射角等于反射角）。
  • Snell折射定律：n₁ sinθ_i = n₂ sinθ_t。

其中折射率 n = c/v = √(ε_r μ_r)。对于非磁性材料（μ_r ≈ 1），n ≈ √ε_r。

Snell定律的工程含义非常深刻：

  • 当波从光疏介质进入光密介质（n₁ < n₂），θ_t < θ_i——折射线偏向法线。
  • 当波从光密介质进入光疏介质（n₁ > n₂），θ_t > θ_i——折射线偏离法线。当 θ_i 足够大时，θ_t 达到 90°，此时发生全内反射（Total Internal Reflection）。

💡 光纤通信的核心——全内反射
光纤由纤芯（n₁≈1.46）和包层（n₂≈1.44）构成。当光在纤芯中以大于临界角 θ_c = arcsin(n₂/n₁) 的角度传播时，在纤芯-包层界面发生全内反射，光被"囚禁"在纤芯中传播数百公里而不泄露。
从 Snell 定律来看，光纤的本质是人为制造了一个 n₁ > n₂ 的波导结构。光纤的数值孔径 NA = √(n₁²−n₂²) ≈ sinθ_a，其中 θ_a 是光纤能"捕获"入射光的最大锥角。
现代通信光纤的损耗可以低至 0.15 dB/km（在1550nm波段），这意味着光在光纤中传播100公里后仍有约3%的功率余量。这是全世界互联网基础设施的物理基石。

Fresnel公式

Snell定律只告诉我们折射光的方向，但反射和透射的幅度各是多少？这正是 Fresnel 公式要回答的问题。斜入射时需要区分两种极化方式：

TE极化（s-极化）

电场方向垂直于入射面（即平行于界面，德语 senkrecht = 垂直）。反射系数：

Γ_TE = (η₂ cosθ_i − η₁ cosθ_t) / (η₂ cosθ_i + η₁ cosθ_t)

TM极化（p-极化）

电场方向平行于入射面（德语 parallel）。反射系数：

Γ_TM = (η₂ cosθ_t − η₁ cosθ_i) / (η₂ cosθ_t + η₁ cosθ_i)

当 θ_i = 0 时（法向入射），两个公式退化为同一个：Γ = (η₂−η₁)/(η₂+η₁)，与上一节一致。

Fresnel公式中蕴含着一个极其重要的现象——半波损失：当波从光疏介质射向光密介质（n₁ < n₂）时，Γ 为负值，意味着反射波电场与入射波电场方向相反（相位翻转180°）。这就是光学中著名的"半波损失"现象的本质。

Python示例：Fresnel反射系数随入射角的变化

import numpy as np
        import matplotlib.pyplot as plt

        # 介质参数：空气→玻璃 (n1=1, n2=1.5)
        n1, n2 = 1.0, 1.5
        theta_i = np.linspace(0, 90, 901)  # 入射角（度）
        theta_i_rad = np.deg2rad(theta_i)
        theta_t_rad = np.arcsin(n1/n2 * np.sin(theta_i_rad))

        # 非磁性介质：η ∝ 1/n
        # TE极化
        r_TE = (n1*np.cos(theta_i_rad) - n2*np.cos(theta_t_rad)) / \
               (n1*np.cos(theta_i_rad) + n2*np.cos(theta_t_rad))
        # TM极化
        r_TM = (n2*np.cos(theta_i_rad) - n1*np.cos(theta_t_rad)) / \
               (n2*np.cos(theta_i_rad) + n1*np.cos(theta_t_rad))

        plt.figure(figsize=(8, 5))
        plt.plot(theta_i, r_TE, label='TE (s-polarization)', linewidth=2)
        plt.plot(theta_i, r_TM, label='TM (p-polarization)', linewidth=2)
        plt.axhline(0, color='gray', linewidth=0.5)
        plt.axvline(np.rad2deg(np.arctan(n2/n1)), color='red',
                    linestyle=':', label=f'Brewster角 = {np.rad2deg(np.arctan(n2/n1)):.1f}°')
        plt.xlabel('入射角 (deg)')
        plt.ylabel('反射系数 Γ')
        plt.title('Fresnel反射系数 vs 入射角 (空气→玻璃)')
        plt.legend()
        plt.grid(alpha=0.3)
        plt.tight_layout()
        plt.show()
🔑 Brewster角的工程魔法
从 Fresnel 公式中可以看出：TM极化的反射系数 Γ_TM 在某个特定入射角下等于零——这个角就是Brewster角 θ_B = arctan(n₂/n₁)。
在 Brewster 角入射时，TM极化的电磁波完全没有反射，全部透射！利用这一原理：
• 偏光墨镜：水面/路面的反射光主要是水平极化（s-分量），偏光镜片利用 Brewster 效应滤除这部分强反射光，让你看清水面下的鱼或路面细节
• 偏振分光镜：在 Brewster 角放置玻璃堆，可以将一束自然光分离为完全偏振的 TM 光和 TE 光
• 激光腔：激光器中常用 Brewster 窗口，使 TM 偏振光无损耗通过，TE 偏振光被反射出腔外，从而获得线偏振激光输出

全反射与倏逝波

当光从光密介质（n₁）射向光疏介质（n₂ < n₁），且入射角大于临界角 θ_c = arcsin(n₂/n₁) 时，发生全内反射（TIR）。此时：

  • 反射系数 |Γ| = 1——所有能量被反射回介质1。
  • 透射波不再以传统方式传播——它的传播常数在法线方向变为纯虚数，意味着波在介质2中沿法线方向指数衰减。
  • 这个衰减波称为倏逝波（Evanescent Wave），振幅沿法线方向按 e^{−κz} 衰减，衰减长度约为波长量级。

全反射并不像字面意思那样"完全没有场进入介质2"——实际上有一个衰减极快的倏逝波穿透到介质2中（穿透深度约 λ/10）。如果在这个穿透距离内放置另一个高折射率介质，倏逝波会"隧穿"过去，形成受抑全内反射（FTIR）——这正是光纤耦合器和光学触摸屏的工作原理。

干涉与多层介质

当两个或多个界面的反射波叠加时，会产生干涉现象。干涉的相长或相消取决于各反射波之间的相位差。利用这个原理，我们可以设计多层介质薄膜来实现特定的反射/透射特性。

四分之一波长阻抗变换层

在两种介质之间插入一层厚度为 λ/4、折射率为 n_m 的中间层，可以实现零反射条件：

n_m = √(n₁ · n₂)

这就是抗反射（AR）镀膜的原理。在玻璃上镀一层 MgF₂（n≈1.38），使得 n_m ≈ √(1×1.5)≈1.22——MgF₂很接近这个理想值。单层 AR 镀膜可以将玻璃表面的反射率从约4%降到1%以下。

布喇格反射镜

交替堆叠两种不同折射率的 λ/4 厚薄膜，形成布喇格反射镜（Bragg Mirror / Distributed Bragg Reflector, DBR）。每个界面产生的小反射在特定带宽内相干叠加，可以实现接近100%的反射率（甚至超过金属反射镜）。应用包括：VCSEL激光器谐振腔、光纤光栅、光学滤波器。

⚠ 雷达隐身——从阻抗匹配到宽带吸收
战机的雷达隐身并非"不反射雷达波"，而是将回波能量尽可能吸收或偏转到非威胁方向。Salisbury Screen是最简单的方案：一块导电地板 + λ/4 厚的介质层 + 表面阻性薄层。当电磁波入射时，表面电阻层中的感应电流被转换为焦耳热，实现谐振吸收。
但单层 Salisbury Screen 的带宽极窄（约10-20%相对带宽）。现代隐身技术使用多层渐变阻抗结构（Jaumann Absorber 或 Dallenbach layer），通过从空气到金属板逐层渐变介电常数，使反射系数在极宽频带（如2-18GHz）内保持很低。「隐身涂料」的本质就是这样的多层阻抗渐变结构——这也是为什么 B-2 轰炸机的涂层维护成本极其昂贵，因为这种多层结构一旦受损，吸波性能就严重下降。

本章总结与展望

本章我们探讨了电磁波在介质界面上的行为，这是连接理想平面波与现实世界的桥梁：

  • 法向入射：反射系数 Γ 和驻波比 VSWR，衡量匹配质量的几个关键参数。
  • Snell定律：从相位匹配条件导出，揭示了折射角与折射率的关系。
  • Fresnel公式：TE/TM 两种极化方式的反射系数，定量描述反射和透射幅度。
  • Brewster角与全反射：两种重要的极端情形，偏光墨镜和光纤通信的物理基础。
  • 多层介质干涉：通过 λ/4 膜层实现抗反射或高反射，广泛应用于光学镀膜和激光器。

反射与折射讨论的核心是"自由空间中的电磁波遇到界面会发生什么"。而下一章我们将转向一个有界结构——传输线与波导。当电磁波被导体结构"约束"在特定路径中传播时，它将展现出与自由空间波截然不同的特性：特性阻抗、色散、截止频率、模式……这些概念是射频和微波工程的核心语言。
