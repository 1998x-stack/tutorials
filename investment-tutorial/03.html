第3章 债券

大家好！上一章我们讲了股票——高收益、高波动的"热血青年"。这一章我们来看它的反面——债券，那个看起来有点"无聊"但每个投资组合都离不开的"稳重中年人"。

债券（Bond）本质上就是一张借条。你借钱给政府或公司，他们给你一张凭证，上面写着："我借了你多少钱、利息是多少、什么时候还你。"就是这么简单。但千万别小看这张"借条"——全球债券市场的规模（约140万亿美元）远远超过股票市场（约110万亿美元）。世界上最大的资产管理公司BlackRock的核心仓位不是股票，而是债券。

很多人觉得债券"收益太低、没啥意思"，这种想法在2025年之前可能是对的——中国10年期国债收益率从2014年的4.5%一路跌到2025年的1.8%。但2025年下半年到2026年，随着全球经济格局的变化，债券市场的机会正在重新浮现。一位资深交易员跟我说过："股市好做的时候没人看债券，但每次熊市一来，债券就是你的救命稻草。"

摩根大通的"伦敦鲸"：用债券赌翻船的交易员

2012年，摩根大通有一位绰号"伦敦鲸"的交易员Bruno Iksil，他的工作是交易公司债券的信用违约互换（CDS）。他做了一笔巨额的信用保护卖出交易，赌那些公司债券不会违约。起初一切顺利，他每月给银行带来上亿美元的利润。

但2012年5月，市场突然逆转——一批公司债的价格暴跌，"伦敦鲸"的仓位瞬间出现数十亿美元的浮亏。更糟的是，他押注的仓位实在太大，大到只要他试图平仓，市场上的其他玩家就会联合起来"围猎"他。最终摩根大通在短短六周内亏损了62亿美元——这是华尔街历史上最大的交易亏损之一。CEO Jamie Dimon后来在国会听证会上说："我们犯了所有可能犯的错误。"

教训：债券看起来"无聊"、"安全"，但你如果以为它没有风险，那就大错特错了。利率风险、信用风险、流动性风险——每一种都可以让你血本无归。"伦敦鲸"的故事告诉我们一个朴素的真理：任何金融工具的"风险"不是你感觉它有没有风险，而是你真正理解它有多大的不确定性。

3.1 债券的核心要素

一只债券有四个基本要素，你读懂了这四个数字，就理解了这只债券的全部：

  • 面值（Face Value / Par Value）：债券到期时发行人偿还的本金金额。通常是100元人民币或1,000美元。
  • 票息率（Coupon Rate）：每年支付的利息占面值的百分比。如果面值100元、票息率3%，你每年能拿到3元利息。
  • 到期日（Maturity Date）：发行人承诺还本付息的日期。1年以内的叫短期债券（T-bills），1-10年的叫中期票据（Notes），10年以上的叫长期债券（Bonds）。
  • 收益率（Yield）：你实际获得的年化回报率。注意——收益率不等于票息率！因为债券的市场价格在变化，你在二级市场买入的价格可能高于或低于面值。
关键概念：票息率是固定的，但收益率是变动的。当你在市场上以90元买入一张面值100元、票息3%的债券，你的当期收益率是 3/90 = 3.33%，高于票息率。如果以110元买入，收益率则降到 3/110 = 2.73%。

3.2 债券类型

债券的世界远比股票更丰富多彩：

类型发行人风险等级中国典型收益（2025）
国债（Treasury Bonds）中央政府接近无风险10年期约1.8%
地方债（Municipal Bonds）地方政府低风险10年期约2.0-2.3%
金融债银行/金融机构低-中风险3年期约2.2-2.8%
企业债（Corporate Bonds）普通公司中-高风险3年期约3-8%
可转债（Convertible Bonds）上市公司中风险票息约0.5-2%（主要看转股价值）

可转债

可转债（Convertible Bond）是一种"下有保底、上不封顶"的债券。它兼具债券和股票的特性：在股价低迷时，你持有债券，收取固定利息；在股价上涨时，你可以按约定价格将债券转换成股票，分享股价上涨的收益。2021年，可转债市场在A股大放异彩——有的可转债一年内涨了300%。但注意：可转债的价格也可能跌破面值，尤其是当正股持续下跌时。

3.3 收益率曲线

把不同期限的国债收益率画在图上，你就得到了一条收益率曲线（Yield Curve）。这是债券市场最重要的"温度计"。

# 2025年6月中国国债收益率曲线（示例数据）
# 期限      收益率
短期（3个月）   1.50%
中期（2年）     1.65%
中期（5年）     1.72%
长期（10年）    1.80%
长期（30年）    2.05%

# 这是一条正常的"向上倾斜"收益率曲线
# 期限越长，收益率越高（因为风险越大）

# 如果变成"倒挂"——短期收益率高于长期——
# 这通常是经济衰退的前兆信号。
# 2023年美国收益率曲线深度倒挂，2024年美国GDP确实降温了。

期限结构理论

为什么收益率曲线通常向上倾斜？有三种理论解释：

  • 预期理论：长期利率是未来短期利率预期的平均。如果你预期未来利率上升，长期收益率就会高于短期。
  • 流动性偏好理论：投资者偏好短期债券（流动性更好），因此长期债券必须提供更高的收益率来补偿流动性溢价。
  • 市场分割理论：不同期限的债券市场有不同的供需关系，比如保险资金喜欢买长期债券，货币基金只买短期。

3.4 久期与凸性

如果你管理一个债券组合，你最关心的问题是：利率变了1%，我的组合会亏多少钱？

久期（Duration）

久期是利率风险的度量工具。简单说：久期越长，债券价格对利率变化越敏感。例如，一只久期为5年的债券，当市场利率上升1%，其价格大约下降5%。

久期的实用价值：如果你预期利率即将上升，你应该缩短久期（卖出长期债券、买入短期债券），以减少价格下跌的损失。反之，如果你预期利率下降，应该拉长久期以获取最大的价格上涨收益。

凸性（Convexity）

久期是一个线性近似，但债券价格与收益率的关系实际上是凸的（Convex）——也就是说，利率下降时价格上升的幅度大于利率上升时价格下跌的幅度。这种不对称性就是凸性。

实战中的应用：退休基金和保险公司常用免疫策略（Immunization）来管理利率风险——通过匹配资产和负债的久期，使利率变化对组合净值的影响最小化。想象一下，如果某养老基金10年后要支付1亿元的退休金，它就需要构建一个久期为10年的债券组合，这样即使利率变化，组合的价值变化也能与负债的现值变化互相抵消。

3.5 信用评级

你买了一家公司发的债券，怎么知道它能不能按时还钱？这就轮到信用评级机构登场了。

三大评级机构

穆迪（Moody's）、标普（S&P）和惠誉（Fitch）是全球三大评级机构。它们的评级体系稍有不同，但逻辑一样：

等级穆迪标普/惠誉含义
投资级Aaa, Aa, A, BaaAAA, AA, A, BBB违约风险低，适合保守型投资者
投机级（高收益/垃圾债）Ba, B, Caa, Ca, CBB, B, CCC, CC, C违约风险高，但收益率也高
违约-D已经违约

评级每下调一个等级，利差（Spread，即该债券收益率与同期限国债收益率之差）通常会扩大50-100个基点（0.5%-1%）。这就是"评级下调——利差扩大——价格下跌"的连锁反应。

真实的"踩坑"故事：2020年，国内有一位投资者张先生买了一只"AAA"评级的地产企业债，收益率6.5%，觉得"AAA怎么会违约"？结果2021年恒大暴雷，这只债券的价格从100元暴跌到15元。张先生后来才知道：AAA评级只是评级机构的评估意见，不是担保。更关键的是，评级机构的收入来自被评级公司——这种利益冲突在行业内是公开的秘密。2008年金融危机时，大量"AAA"评级的次贷产品全部违约——评级机构的公信力至今没有恢复。
本章要点回顾：
  • 债券是固定收益证券，核心四要素：面值、票息率、到期日、收益率
  • 主要债券类型包括国债、地方债、企业债和可转债，风险收益特征各不相同
  • 收益率曲线是市场利率预期的晴雨表，倒挂是衰退预警信号
  • 久期度量利率风险，凸性度量价格对利率变化的非线性响应
  • 信用评级帮助判断债券的违约风险，但不可盲目迷信

债券讲完了。你可能觉得债券"稳是稳，但收益太低了"——确实，个人投资者直接买债券的性价比不如买基金。下一章我们将进入基金的世界，看看如何用别人的专业能力帮你赚钱，以及ETF为什么是过去十年最伟大的金融创新。走吧！
