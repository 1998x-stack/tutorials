第19章 国民收入核算

19.1 欢迎来到宏观经济学世界

大家好，我是你们的讲师。从本章开始，我们将进入一个全新的领域——宏观经济学（Macroeconomics）。如果说微观经济学关心的是"一棵树"——单个消费者、单个企业、单个市场——那么宏观经济学关心的是"整片森林"：整个国家的经济总量如何运转、为什么有些国家富裕而有些贫穷、为什么会发生经济危机。

宏观经济学有一个核心问题：我们怎么知道一个国家的经济表现是好是坏？你不能靠"感觉"说"哎我觉得今年经济不行"。经济学家需要一把共同的尺子。这把尺子就是本章要讲的主题——国民收入核算（National Income Accounting）。

🔑 关键概念：国民收入核算是一套衡量经济总产出的会计体系。它就像给整个经济体做体检——GDP就是那个"体温读数"。没有准确的核算，就没有合理的宏观政策。

19.2 什么是GDP？

GDP（Gross Domestic Product，国内生产总值）是宏观经济学中最重要的概念。它的官方定义是：在给定时期内，一个国家境内生产的全部最终产品与服务的市场价值。我们来拆解这个定义的每一个关键词：

  • 市场价值：用货币衡量，把苹果和橘子加起来。1个iPhone + 1斤大米没法直接加，但用它们的价格就可以。
  • 最终产品：只计算最终卖给消费者的产品，不计中间产品（比如小麦做成面包，只算面包的价值，不算小麦）。这避免了重复计算。
  • 境内："国内"的"内"字是关键——不管是中国企业还是外资企业，只要在中国境内生产，都计入中国的GDP。
  • 给定时期：通常是一年或一个季度。GDP是流量（Flow），不是存量——就像水龙头的出水量，而不是浴缸里的水量。

GDP的核算主要有三种方法，支出法、收入法和生产法。最常用的是支出法（Expenditure Approach），也是我们下面重点讲的。

19.3 GDP的四个组成部分

用支出法来看，GDP可以拆成四个大块——在宏观经济学里，这是一个你必须牢记于心、刻在脑子里的恒等式：

Y = C + I + G + NX

  • C — 消费（Consumption）：居民购买的商品和服务。吃穿住行、医疗教育、看电影喝奶茶——这些都是消费。消费通常占GDP的最大部分，在中国约占总量的55%左右。
  • I — 投资（Investment）：注意！宏观经济学里的"投资"跟你在股市"炒股票"是两码事。这里的投资是指实物资本形成——企业买机器建厂房、居民买新房、以及存货变动。简单说，是那些能增加未来生产能力的支出。
  • G — 政府购买（Government Purchases）：政府购买的商品和服务，包括国防、基础设施建设、公务员薪资等。注意：转移支付（Transfer Payments）——比如养老金、失业救济——不计入G，因为这些不是政府购买产品，只是钱从一个口袋转到另一个口袋。
  • NX — 净出口（Net Exports）：出口（Exports）减去进口（Imports）。出口增加GDP，进口减少GDP。因为C+I+G已经包含了一部分进口产品但在中国境内消费的部分，减去进口才得到真正的"中国生产"。

19.4 名义GDP vs 实际GDP

如果GDP从100万亿涨到了105万亿，那经济真的增长了5%吗？不一定——可能只是因为物价涨了5%，而实际生产的东西一点没多。这就是名义GDP（Nominal GDP）和实际GDP（Real GDP）的区别。

  • 名义GDP：用当期价格计算。它同时反映了产量变化和价格变化。
  • 实际GDP：用基期不变价格计算。它只反映产量的变化，剔除了价格因素。

我们用一组简化的数据来说明——假设一个经济体只生产两种产品：面包和手机：

年份面包产量面包价格手机产量手机价格名义GDP实际GDP(2023为基期)
20231000个10元50部2000元110,000元110,000元
20241100个12元55部2200元134,200元121,000元
20251200个13元60部2300元153,600元132,000元

看2024年：名义GDP从11万涨到13.42万，增长了22%；但实际GDP只从11万涨到12.1万，增长10%。多出来的12个百分点就是物价上涨造成的。所以在新闻报道中提到的"GDP增速"，通常指的都是实际GDP增速。

19.5 GDP平减指数

如果你想把名义GDP和实际GDP之间的差距变成一个单一的"物价指数"，那就是GDP平减指数（GDP Deflator）：

GDP平减指数 = (名义GDP / 实际GDP) × 100

用上表的数据：2024年的GDP平减指数 = (134,200 / 121,000) × 100 ≈ 110.9。这意味着从2023到2024年，整体物价水平上涨了约10.9%。GDP平减指数是一个衡量所有最终产品和服务的价格水平的指标，它比下一章要讲的CPI衡量范围更广、更全面。

def calculate_gdp(c, i, g, exports, imports):
    """支出法计算GDP"""
    nx = exports - imports  # 净出口
    gdp = c + i + g + nx
    return gdp

def gdp_deflator(nominal_gdp, real_gdp):
    """计算GDP平减指数"""
    return (nominal_gdp / real_gdp) * 100

# 中国2024年数据（万亿元）
C = 55.1   # 消费
I = 28.3   # 投资
G = 15.2   # 政府购买
EX = 25.0  # 出口
IM = 24.8  # 进口

gdp_nominal = calculate_gdp(C, I, G, EX, IM)
gdp_real = 96.5  # 假设以不变价格计算的实际GDP
deflator = gdp_deflator(gdp_nominal, gdp_real)

print(f"名义GDP: {gdp_nominal:.1f} 万亿元")
print(f"实际GDP: {gdp_real:.1f} 万亿元")
print(f"GDP平减指数: {deflator:.1f}")
# 输出:
# 名义GDP: 98.8 万亿元
# 实际GDP: 96.5 万亿元
# GDP平减指数: 102.4

19.6 GDP的局限性——GDP不是一切

GDP是一个了不起的发明——它让我们有了一个统一的标准来衡量经济规模。但如果我们把GDP当作"社会福利"的唯一衡量标准，那就大错特错了。GDP存在以下几大盲区：

地下经济与非法活动

你去街边小摊买的煎饼果子，老板没开发票——这笔交易没被记录，不在GDP里。更不用说毒品交易、赌博等非法经济。据国际货币基金组织（IMF）估算，印度和尼日利亚的地下经济规模可能达到GDP的30-50%。在中国的某些灰色地带——比如房屋装修不需要发票的现金交易——也大量存在漏报的情况。

家务劳动与照顾工作

妈妈在家带小孩、做家务——这些工作如果请保姆来做，是要付钱的，那是GDP。但妈妈自己做就不算GDP。这是一个严重的性别盲区——联合国估计，全球无偿家务和照顾工作的价值约占全球GDP的10-40%。

环境成本与资源消耗

砍掉一片原始森林卖木材——GDP增加了。但森林的生态价值、生物多样性损失——GDP完全不考虑。这就是为什么经济学家提出了绿色GDP的概念，要扣减环境退化成本。

幸福感和生活满意度

GDP增长了，但你大概率没觉得自己变富了——这背后有一个很现实的原因。我们来看一个真实的故事。

⚠️ 现实故事：GDP增长 vs 中位数收入

从1990年到2020年，美国的实际GDP增长了超过100%，但中位数家庭收入（Median Household Income）只增长了约30%。为什么？因为增长的蛋糕很大一部分流向了高收入群体——富人收入增长远快于普通人。这就是GDP增长不等于共同富裕的典型例证。

中国的数据也呈现类似的结构性特征：2024年中国GDP突破98万亿元，稳居世界第二。但同时，劳动报酬在国民收入中的占比长期偏低，居民可支配收入增速经常慢于GDP增速。这个问题在经济学上被称为"GDP总量和居民感受之间的温差"。

关键在于：GDP衡量的是生产了多少，而不是谁得到了多少。以后看到GDP增长的数据，记得再追问一句：增长的红利分配公平吗？

一个另类方案：国民幸福总值 GNH

不丹，一个喜马拉雅山下的佛教王国，在全世界率先做了一个"反潮流"的事——放弃以GDP为核心，而是推行国民幸福总值（Gross National Happiness, GNH）。不丹第四任国王吉格梅·辛格·旺楚克在1972年宣布："国民幸福比国民生产总值更重要。"

GNH包括四大支柱、九个领域：

  • 四大支柱：可持续和公平的社会经济发展、环境保护、文化保护与促进、善治
  • 九个领域：心理幸福感、时间使用、社区活力、文化多样性、生态韧性、生活水平、健康、教育、治理

当然，GNH的衡量非常困难，不丹自己的经济发展也面临诸多挑战。但它的意义在于提醒我们：经济的最终目的是让人过得更好，而不是让数字更大。

💡 课后思考：还有一些替代性的指标体系值得了解——人类发展指数（HDI）（联合国开发计划署编制，GDP+教育+寿命）、真实发展指数（GPI）（扣减环境成本，加上家务劳动）。如果你感兴趣，可以查查你所在国家的最新排名。

19.7 本章小结

本章我们学习了宏观经济学最基础的度量工具——国民收入核算。记住以下几个核心要点：

  • GDP是衡量经济总产出的核心指标，用支出法可拆分为 C + I + G + NX
  • 名义GDP包含了价格变化，实际GDP剔除了价格因素
  • GDP平减指数是衡量总体物价水平的指标之一
  • GDP有很多局限性——它不衡量地下经济、家务劳动、环境破坏和公平分配

GDP是一个工具，不是目标。它就像体检时秤上的体重——很重要，但它不能告诉你你的胆固醇、血压、心率。下一章我们要学习的是另一个重要的宏观指标——生活成本与通胀。你的100元存款，在通胀面前，到底能买到什么？我们第20章见！
